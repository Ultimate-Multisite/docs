# Superdav AI Agent

Superdav AI Agent हा प्रगत WordPress AI सहाय्यक आहे, जो Gratis AI Agent ला theme-जाणकार कौशल्ये, सुधारित इंटरनेट शोध क्षमता, site builder वैशिष्ट्ये आणि वैकल्पिक स्वयंचलन एकत्रीकरणांनी विस्तारित करतो. आवृत्ती 1.18.0 Google Calendar वाचन साधने, संपर्क आणि उपस्थित mapping, मानवी मंजुरी gates, स्मरणपत्र deduplication, TextBee SMS सूचना, व्यवस्थापित Superdav सेवा provisioning, आणि विश्वासू developer कार्यप्रवाहांसाठी स्वतंत्रपणे वितरित Advanced companion package जोडते.

## प्रमुख वैशिष्ट्ये {#key-features}

- **Theme-जाणकार कौशल्ये** — तुमच्या सक्रिय WordPress theme आणि स्थापित plugins च्या आधारे मार्गदर्शन आणि क्षमता आपोआप जुळवते
- **अनेक इंटरनेट शोध प्रदाते** — अधिक समृद्ध शोध परिणाम आणि प्रगत संशोधन क्षमतांसाठी Brave Search आणि Tavily यांपैकी निवडा
- **Site builder संपर्क forms** — तुमच्या स्थापित form plugin वापरून chat interface मधून थेट पूर्ण कार्यक्षम contact forms तयार करा
- **सुधारित site builder** — अधिक चांगल्या plugin शोध आणि त्रुटी पुनर्प्राप्तीसह सुधारित बहु-टप्पा site build plans
- **अंतर्भूत theme कौशल्ये** — Block Themes, Classic Themes, Kadence Blocks, आणि Kadence Theme साठी विशेष मार्गदर्शन
- **सुधारित WooCommerce integration** — अधिक चांगली विश्वासार्हता आणि compatibility साठी मूळ WooCommerce APIs वापरण्यासाठी refactor केलेले
- **स्वयंचलित provider detection** — plugins आणि themes सक्रिय किंवा निष्क्रिय केल्यावर उपलब्ध plugins आणि themes आपोआप ओळखते आणि refresh करते
- **सानुकूल logo निर्मिती** — स्वयंचलित namespace-safe sanitisation सह सानुकूल SVG logos तयार करा आणि embed करा
- **Palette contrast validation** — तुमच्या theme ला लागू करण्यापूर्वी WCAG accessibility compliance साठी colour pairs तपासा
- **Photo-enhanced discovery** — design direction वर प्रभाव टाकण्यासाठी Theme Builder discovery interview दरम्यान photos upload करा
- **Hospitality menus** — restaurants, cafes, आणि hospitality businesses साठी संरचित food आणि beverage menu pages तयार करा
- **Desktop आणि mobile previews** — design-direction निवडीदरम्यान desktop आणि mobile devices वर तुमच्या design चे preview पहा
- **Tier 1 site scraping** — competitor आणि inspiration websites चे विश्लेषण करण्यासाठी sd-ai-agent/site-scrape आता default उपलब्ध आहे
- **Google Calendar read tools** — automations आगामी schedule context बद्दल विचार करू शकावेत म्हणून configured calendars आणि events वाचा
- **Contact आणि attendee mapping** — follow-up work निर्माण करण्यापूर्वी calendar participants ना ज्ञात site users आणि contacts शी match करा
- **Human approval gates** — messages पाठवण्यापूर्वी, work resume करण्यापूर्वी किंवा changes करण्यापूर्वी संवेदनशील automations review साठी pause करा
- **Reminder records आणि deduplication** — आधीच पाठवलेली reminders track करा, जेणेकरून repeat automation runs त्याच recipient ला दोनदा notify करणार नाहीत
- **TextBee SMS notifications** — SMS provider सक्षम असताना TextBee द्वारे configured text-message notifications पाठवा
- **Managed Superdav service provisioning** — manual endpoint configuration ऐवजी automatic setup सह समर्थित sites ना hosted Superdav endpoints शी connect करा
- **Advanced companion package** — filesystem, database, WP-CLI, REST dispatcher, plugin builder, git snapshot, user-management, आणि benchmark workflows साठी स्वतंत्रपणे वितरित trusted-developer tools

## आवश्यकता {#requirements}

- WordPress 6.0 किंवा उच्च
- PHP 8.1 किंवा उच्च
- WordPress Multisite (network-activated) किंवा single-site स्थापना
- सक्रिय AI provider connector plugin (Anthropic, OpenAI, किंवा compatible)
- Gratis AI Agent v1.9.0 किंवा उच्च (Superdav AI Agent हा Gratis AI Agent विस्तारित करतो)

## स्थापना {#installation}

1. plugin files `/wp-content/plugins/superdav-ai-agent` येथे upload करा
2. WordPress मधील **Plugins** screen द्वारे plugin सक्रिय करा
3. Gratis AI Agent सुद्धा सक्रिय असल्याची खात्री करा
4. **Gratis AI Agent → Settings** येथे जा आणि गरजेनुसार तुमचा internet search provider, managed Superdav service connection, Google Calendar, किंवा TextBee credentials configure करा

WordPress.org package मध्ये मुख्य Superdav AI Agent वैशिष्ट्ये समाविष्ट आहेत. Advanced companion package स्वतंत्रपणे तयार आणि वितरित केले जाते, अशा environments साठी जिथे trusted developer tools स्पष्टपणे स्थापित आणि authorised केलेले असतात.

## झटपट सुरुवात {#quick-start}

सक्रिय केल्यावर, Superdav AI Agent तुमची Gratis AI Agent स्थापना theme-जाणकार कौशल्ये आणि अतिरिक्त क्षमतांनी आपोआप समृद्ध करतो. WordPress admin sidebar मधील **Gratis AI Agent** panel उघडा आणि नवीन वैशिष्ट्ये वापरण्यास सुरुवात करा.

**उदाहरण सूचना:**

```
Contact page वर contact form जोडा.
```

```
नवीनतम WordPress security best practices साठी web शोधा.
```

```
सक्रिय theme च्या block patterns वापरून hero section तयार करा.
```

```
उद्याचे configured Google Calendar events review करा आणि attendees साठी reminders draft करा.
```

```
मी message approve केल्यानंतर SMS reminder पाठवा.
```

## Theme-जाणकार कौशल्ये {#theme-aware-skills}

Superdav AI Agent तुमचा सक्रिय theme आणि स्थापित plugins आपोआप ओळखतो, आणि योग्य theme-जाणकार कौशल्य assistant च्या context मध्ये load करतो. उपलब्ध कौशल्यांमध्ये समाविष्ट आहे:

| कौशल्य | लागू होते |
|---|---|
| **Block Themes** | WordPress block-based (Full Site Editing) architecture वापरणारे themes |
| **Classic Themes** | PHP templates आणि classic editor वापरणारे पारंपरिक WordPress themes |
| **Kadence Blocks** | Kadence Blocks plugin सक्रिय असलेल्या sites |
| **Kadence Theme** | सक्रिय theme म्हणून Kadence theme असलेल्या sites |

प्रत्येक कौशल्यावरील सविस्तर मार्गदर्शनासाठी [Theme-जाणकार कौशल्ये](../../user-guide/configuration/theme-aware-skills) पहा.

## इंटरनेट शोध प्रदाते {#internet-search-providers}

Superdav AI Agent v1.10.0 इंटरनेट शोध queries साठी Brave Search चा पर्याय म्हणून **Tavily** जोडते. दोन्ही providers real-time search results देतात आणि **Gratis AI Agent → Settings → Advanced** मध्ये configure करता येतात.

| प्रदाता | वैशिष्ट्ये | सेटअप |
|---|---|---|
| **Brave Search** | जलद, गोपनीयता-केंद्रित शोध परिणाम | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | अधिक समृद्ध शोध परिणाम, प्रगत संशोधन क्षमता | [Tavily API](https://tavily.com/) |

कॉन्फिगरेशन सूचनांसाठी [इंटरनेट शोध](../../user-guide/configuration/internet-search) पहा.

## कॅलेंडर, मंजुरी आणि सूचना कार्यप्रवाह {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 सामग्री आणि साइट-बांधणी कार्यांच्या पलीकडे ऑटोमेशन कार्यप्रवाह विस्तारित करते:

- **Google Calendar साधने** कॅलेंडर क्रेडेन्शियल्स उपलब्ध असताना agent ला कॉन्फिगर केलेली कॅलेंडर्स आणि इव्हेंट तपशील वाचू देतात.
- **संपर्क आणि उपस्थित सहाय्यक** स्मरणपत्रे किंवा फॉलो-अप कार्ये तयार करण्यापूर्वी इव्हेंट सहभागींचे WordPress वापरकर्ते किंवा ज्ञात संपर्कांशी मॅपिंग करतात.
- **मानवी मंजुरी गेट्स** अधिकृत वापरकर्ता प्रस्तावित कृतीचे पुनरावलोकन करेपर्यंत ऑटोमेशनला थांबण्याची परवानगी देतात.
- **स्मरणपत्र नोंदी** ऑटोमेशन पुन्हा प्रयत्न केले गेले किंवा वारंवार शेड्यूल केले गेले तेव्हा डुप्लिकेट कॅलेंडर किंवा सूचना स्मरणपत्रे टाळतात.
- **TextBee SMS समर्थन** TextBee प्रदाता कॉन्फिगर केलेला असताना मजकूर-संदेश सूचना सक्षम करते.

संवेदनशील संदेश पाठवण्यापूर्वी अजूनही मानवी पुनरावलोकन आवश्यक असलेल्या वेळापत्रक-जागरूक सहाय्य, उपस्थित फॉलो-अप आणि स्मरणपत्र कार्यप्रवाहांसाठी ही साधने वापरा.

## साइट बिल्डर संपर्क फॉर्म {#site-builder-contact-forms}

Site Builder agent आता चॅट इंटरफेसमधून थेट संपर्क फॉर्म तयार करू शकतो. फक्त Site Builder ला संपर्क फॉर्म तयार करण्यास सांगा, आणि तो आपोआप तुमचा स्थापित form plugin (Contact Form 7, WPForms, Fluent Forms किंवा Gravity Forms) ओळखेल आणि वापरण्यास-सज्ज फॉर्म तयार करेल.

तपशीलवार मार्गदर्शनासाठी [Site Builder संपर्क फॉर्म](../../user-guide/administration/site-builder-contact-forms) पहा.

## Theme Builder वैशिष्ट्ये {#theme-builder-features}

Superdav AI Agent v1.16.0 मधील Theme Builder मध्ये शक्तिशाली नवीन क्षमता समाविष्ट आहेत:

### फोटो अपलोडसह डिस्कव्हरी मुलाखत {#discovery-interview-with-photo-upload}

Theme Builder डिस्कव्हरी मुलाखतीत आता फोटो अपलोड टप्पा समाविष्ट आहे जिथे तुम्ही दृश्य प्रेरणा देऊ शकता. तुमच्या ब्रँडचे, स्पर्धक साइट्सचे किंवा डिझाइन प्रेरणेचे फोटो अपलोड करा, ज्यामुळे Theme Builder ला तुमची डिझाइन दिशा समजण्यास मदत होईल.

तपशीलवार मार्गदर्शनासाठी [डिस्कव्हरी मुलाखत](./theme-builder/discovery-interview.md) पहा.

### डेस्कटॉप आणि मोबाइल पूर्वावलोकनांसह डिझाइन दिशा {#design-direction-with-desktop-and-mobile-previews}

Theme Builder तुमची पूर्ण theme तयार करण्यापूर्वी तुमच्या डिझाइनचे डेस्कटॉप आणि मोबाइल डिव्हाइसवर पुनरावलोकन करा. अनेक प्रतिसादक्षम ब्रेकपॉइंट्सवर रिअल-टाइम पूर्वावलोकने पहा.

तपशीलवार मार्गदर्शनासाठी [डिझाइन दिशा](./theme-builder/design-direction.md) पहा.

### आतिथ्य मेनू {#hospitality-menus}

Theme Builder आता रेस्टॉरंट्स, कॅफे, बार आणि इतर आतिथ्य व्यवसायांसाठी संरचित अन्न आणि पेय मेनू पृष्ठे तयार करू शकतो. मेनूमध्ये किंमत, वर्णने, आहारविषयक माहिती आणि अॅलर्जन इशारे समाविष्ट असतात.

तपशीलवार मार्गदर्शनासाठी [आतिथ्य मेनू](./theme-builder/hospitality-menus.md) पहा.

## कॉन्फिगरेशन {#configuration}

सर्व सेटिंग्ज WordPress admin मधील **Gratis AI Agent → सेटिंग्ज** अंतर्गत आहेत. Superdav AI Agent खालील कॉन्फिगरेशन पर्याय जोडते:

| सेटिंग | वर्णन |
|---|---|
| **इंटरनेट शोध प्रदाता** | Brave Search आणि Tavily यांपैकी निवडा (पर्यायी) |
| **Tavily API Key** | Tavily इंटरनेट शोधासाठी API key (Tavily निवडले असल्यास) |
| **व्यवस्थापित Superdav सेवा** | स्वयंचलित endpoint प्रोव्हिजनिंगसह समर्थित साइट्सना hosted Superdav सेवेशी जोडा |
| **Google Calendar क्रेडेन्शियल्स** | agent कॅलेंडर्स आणि इव्हेंट्स वाचू शकेल यासाठी कॅलेंडर प्रवेश कॉन्फिगर करा |
| **TextBee SMS प्रदाता** | मंजूर SMS सूचना कार्यप्रवाहांसाठी TextBee क्रेडेन्शियल्स कॉन्फिगर करा |

## प्रगत companion पॅकेज {#advanced-companion-package}

Superdav AI Agent v1.18.0 विश्वासार्ह developer कार्यप्रवाहांसाठी स्वतंत्रपणे वितरित Advanced companion पॅकेज जोडते. या क्षमता WordPress.org core पॅकेजमध्ये समाविष्ट नाहीत. agent ला developer tooling मध्ये प्रवेश देण्याचा administrators ना स्पष्ट विश्वास असलेल्या साइट्सवरच companion पॅकेज स्थापित करा.

Advanced साधनांमध्ये filesystem operations, database access, WP-CLI, REST dispatcher workflows, plugin builder utilities, git snapshots, user-management helpers आणि benchmark tooling समाविष्ट आहेत. ही साधने विश्वासार्ह administrators पुरती मर्यादित ठेवा आणि production साइट्सवर त्यांना सक्षम करण्यापूर्वी approval-gate कॉन्फिगरेशनचे पुनरावलोकन करा.
