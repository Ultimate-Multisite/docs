# बदलनोंद {#changelog}

## 1.18.0 — 2026-06-29 रोजी प्रकाशित {#1180--released-on-2026-06-29}

### नवीन {#new}

- **Google Calendar साधने** — वेळापत्रक-जागरूक स्वयंचलनांसाठी कॉन्फिगर केलेली कॅलेंडर आणि कार्यक्रम वाचा
- **संपर्क मॅपिंग आणि सहभागी सहाय्यक** — कॅलेंडर सहभागींची साइट वापरकर्ते आणि संपर्कांशी जुळणी करा
- **मानवी मंजुरी गेट्स आणि स्मरण नोंदी** — पुनरावलोकनासाठी स्वयंचलने थांबवा आणि दुबार सूचना टाळा
- **TextBee SMS प्रदाता** — TextBee द्वारे कॉन्फिगर केलेल्या मजकूर-संदेश सूचना पाठवा
- **प्रगत सहचर पॅकेज** — WordPress.org बिल्डपासून स्वतंत्रपणे वितरित केलेली विश्वसनीय-विकसक फाइलसिस्टम, डेटाबेस, WP-CLI, REST डिस्पॅचर, प्लगइन बिल्डर, git स्नॅपशॉट, वापरकर्ता-व्यवस्थापन आणि बेंचमार्क साधने जोडा

### सुधारित {#improved}

- **व्यवस्थापित Superdav सेवा सेटअप** — समर्थित साइट्ससाठी होस्ट केलेले सेवा समाप्तिबिंदू आणि स्वयंचलित कनेक्शन तरतूद जोडा
- **प्रकाशन पॅकेजिंग** — स्वतंत्र कोर आणि प्रगत ZIPs तयार करा, दोन्ही GitHub वर प्रकाशित करा आणि WordPress.org ला फक्त कोर पॅकेज पाठवा

### दुरुस्त {#fixed}

- **AI विनंती विश्वसनीयता** — मॉडेल निवड, विनंती कालमर्यादा, डीफॉल्ट सेटअप मॉडेल, तर्क-मजकूर हाताळणी आणि अवैध साधन कॉल्ससाठी पुन्हा प्रयत्न मार्गदर्शन सुधारा
- **कॅलेंडर आणि स्मरण मजबुतीकरण** — Google Calendar टोकन्स आणि स्मरण डीडुप्लिकेशन बळकट करा
- **ऑनबोर्डिंग आणि मंजुरी पुन्हा सुरू करणे** — फ्रंटएंड ऑनबोर्डिंग लाँचेस आणि पुष्टी केलेली क्षमता पुन्हा सुरू करणे दुरुस्त करा
- **WordPress.org पॅकेजिंग पुनरावलोकन समस्या** — कोर प्रकाशनासाठी पॅकेजिंग पुनरावलोकन अभिप्राय हाताळा

## 1.16.0 — 2026-05-20 रोजी प्रकाशित {#1160--released-on-2026-05-20}

### नवीन {#new-1}

- **Logo SVG तयार करण्याची क्षमता** — Theme Builder आता नेमस्पेस-सुरक्षित स्वच्छतेसह कस्टम लोगो SVGs तयार करून एम्बेड करू शकतो
- **डिस्कव्हरी मुलाखतीत फोटो अपलोड** — Theme Builder डिस्कव्हरी मुलाखतीत आता अधिक समृद्ध डिझाइन संदर्भासाठी फोटो अपलोड पायरी समाविष्ट आहे
- **Palette Contrast तपासण्याची क्षमता** — थीमला लागू करण्यापूर्वी WCAG अनुपालनासाठी रंग जोड्या तपासा
- **आतिथ्य मेनू** — Theme Builder आता आतिथ्य व्यवसायांसाठी संरचित अन्न आणि पेय मेनू पृष्ठे तयार करू शकतो
- **डेस्कटॉप आणि मोबाइल पूर्वावलोकन रेंडरिंग** — डिझाइन-दिशा निवडीदरम्यान तुमचे डिझाइन डेस्कटॉप आणि मोबाइल उपकरणांवर पूर्वावलोकित करा
- **नेव्हिगेशन लेबल पॅरामीटर** — Create Menu क्षमता आता पृष्ठ शीर्षकापासून वेगळ्या स्वतंत्र `navigation_label` ला समर्थन देते
- **Tier 1 साधन उपलब्धता** — sd-ai-agent/site-scrape आता Theme Builder मध्ये डीफॉल्टनुसार उपलब्ध Tier 1 साधन आहे

### दुरुस्त {#fixed-1}

- **AI Client कॅशे** — आता क्रॉस-विनंती सातत्यासाठी ट्रान्झिएंट्सद्वारे समर्थित, दीर्घकाळ चालणाऱ्या एजंट कार्यांवर डेटा गमावणे टाळते
- **प्लगइन ओळ कृती दुवे** — स्पष्टतेसाठी दुरुस्त आणि पुनर्नामित

## 1.10.0 — 2026-05-05 रोजी प्रकाशित {#1100--released-on-2026-05-05}

### नवीन {#new-2}

- **Tavily इंटरनेट शोध** — Brave Search सोबत अधिक समृद्ध इंटरनेट शोध परिणामांसाठी Tavily ला शोध प्रदाता म्हणून जोडा
- **थीम-जागरूक अंगभूत कौशल्ये** — Block Themes, Classic Themes, Kadence Blocks आणि Kadence Theme कौशल्य मार्गदर्शिका आता प्लगइनसोबत येतात
- **साइट बिल्डर संपर्क फॉर्म क्षमता** — चॅट इंटरफेसमधून थेट कोणत्याही पृष्ठावर संपर्क फॉर्म जोडा

### सुधारित {#improved-1}

- **WooCommerce एकत्रीकरण पुनर्रचित** — अधिक चांगल्या विश्वसनीयता आणि सुसंगततेसाठी आता मूळ WooCommerce APIs वापरते
- **प्रदाता यादी आपोआप रिफ्रेश होते** — कोणतेही प्लगइन सक्रिय किंवा निष्क्रिय झाल्यावर

### दुरुस्त {#fixed-2}

- **navigate-to क्षमता** — काही प्रशासन पृष्ठांवरील अनंत रीलोड लूप दुरुस्त केला
- **list-posts क्षमता** — आता श्रेणी आणि टॅग नावे योग्यरित्या स्लग्समध्ये रूपांतरित करते
- **WP-CLI आदेश** — मागील पुनर्रचनेनंतर गहाळ नेमस्पेस उपनावे पुनर्संचयित केली
- **कार्यक्रम स्वयंचलन** — ज्या साइट्सवर स्वयंचलन तक्ते अद्याप तयार झालेले नाहीत त्या सौम्यपणे हाताळते
- **memory-save क्षमता** — आता प्रणाली सूचना बिल्डरमध्ये योग्य नेमस्पेस उपसर्ग वापरते
- **स्केलर साधन परिणाम** — आता AI कडे परत करण्यापूर्वी योग्यरित्या गुंडाळले जातात
- **वापर आकडेवारी** — जुन्या आवृत्त्यांमधून अपग्रेड करताना आता लेगसी क्षमता की स्वरूप योग्यरित्या हाताळते
