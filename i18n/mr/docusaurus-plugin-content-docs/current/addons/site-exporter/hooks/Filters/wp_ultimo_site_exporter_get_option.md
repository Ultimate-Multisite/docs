# Filter: wp_ultimo_site_exporter_get_option

हे नेटवर्क ऑप्शनचे स्लगिफाइड व्हॅल्यू (slugfied network option value) परत करते.

## पॅरामीटर्स {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $option_value | `mixed` | नेटवर्क ऑप्शनचे व्हॅल्यू. |
| $option_name | `string` | ऑप्शनचे नाव. |
| $default | `mixed` | डीफॉल्ट व्हॅल्यू. |

### Since {#since}

- 1.9.6
### Source {#source}

[`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-helper.php#L126) मध्ये लाइन 126 वर परिभाषित केले आहे.

## Returns {#returns}
हे स्लगिफाइड नेटवर्क ऑप्शनचे व्हॅल्यू परत करते.
