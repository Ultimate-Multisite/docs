# साईट एक्सपोर्टर बदलांचा इतिहास

Version: 1.0.6 - रिलीझ झाले 2026-05-11
* नवीन: साईट एक्सपोर्ट आता एक स्वतःहून सुरू होणारे index.php बंडल करतो, ज्यामुळे ZIP फाईल वेगळ्या प्लगइन इन्स्टॉलेशनशिवाय नवीन होस्टवर (fresh host) इन्स्टॉल करता येते.
* नवीन: नेटवर्क एक्सपोर्टमुळे प्रशासकांना (administrators) साईट एक्सपोर्ट ॲडमिन पेजवरून सर्व सबसाईट्स एकाच आर्काइव्हमध्ये एक्सपोर्ट करता येतात.

Version: 1.0.5 - रिलीझ झाले 2026-05-05
* दुरुस्ती: `get_config` सारख्या अनमॉक केलेल्या मेथड्सना (unmocked methods) हाताळण्यासाठी WP_CLI polyfill stub मध्ये `__callStatic` जोडले आहे, ज्यामुळे चाचणी बूटस्ट्रॅप अयशस्वी होणे (test bootstrap failures) रोखले जाते.

Version: 1.0.4 - रिलीझ झाले 2025-11-25

* दुरुस्ती: Ultimate Multisite 2.4 सोबतची सुसंगतता (compatibility) सुधारली.

Version: 1.0.2 - रिलीझ झाले 2025-09-28

* प्रीफिक्स (prefix) ultimate-multisite असे बदलले; टेक्स्ट डोमेन (text domain) अपडेट केले; आणि व्हर्जन वाढवले.

### Version 1.0.1 - रिलीझ झाले 2023-08-09

* सुधारित: आता इम्पोर्टर फाईलचा योग्य फाइल पाथ (file path) मिळवण्याची खात्री केली जाते.
* दुरुस्ती: इम्पोर्ट दरम्यान त्रुटी (errors) टाळण्यासाठी ऑब्जेक्ट कॅशे (object cache) अक्षम केला.
* दुरुस्ती: इम्पोर्टर प्लगइन डाउनलोड लिंक काम न करण्याच्या समस्येचे निराकरण केले.

### Version 1.0.0 - रिलीझ झाले 2022-12-23

* दुरुस्ती: एक्सपोर्ट फॉर्म पॉपअप लोड होत नव्हता;
* दुरुस्ती: एक्सपोर्ट पेजवर Javascript फाईल्स लोड होत नव्हत्या;
* अंतर्गत: नवीन बिल्ड प्रक्रिया;

### Version 1.0.0-beta.4 - रिलीझ झाले 2022-01-13

* अंतर्गत: हुक्स (hooks) आणि फिल्टर्स (filters) जनरेटर जोडले;
* अंतर्गत: डेव्हलपरसाठी कार्यक्षमतेत सुधारणा करण्यासाठी (developer quality of life) WP Ultimo स्टब्स जोडले;
