# Ultimate AI Connector for Compatible Endpoints बदलांची नोंद (Changelog)

## Version 2.0.0 — Released on 2026-04-24

* **नवीन (New):** मल्टी-प्रदाता समर्थन (Multi-provider support) — अनेक AI endpoints कॉन्फिगर करा आणि प्रदात्यांमध्ये स्वयंचलित फॉलबॅकसह (automatic fallback) विनंत्या रूट करा.
* **सुधारणा (Fix):** प्रत्येक प्रदात्यासाठी योग्य provider IDs, registration URLs आणि HTTP filter scoping सह मल्टी-प्रदाता SDK एकत्रीकरण (integration).
* **सुधारणा (Fix):** नवीन प्रदाता कार्ड आता जोडल्यावर आपोआप विस्तारते (auto-expands); प्लगइन अपडेटवर स्क्रिप्ट कॅशे बस्टिंग (script cache busting) लागू केले.
* **सुधारणा (Fix):** eval() साठी डायनॅमिक प्रदाता क्लास नेमस्पेस (namespace) आणि नवीन जोडलेल्या प्रदात्यांसाठी ऑटो-एक्सपँड बिहेवियर (auto-expand behaviour).
* **सुधारणा (Fix):** WordPress 6.9+ सुसंगततेसाठी स्थिर Card/CardBody/CardHeader/CardDivider घटकांचा (components) वापर केला (आता हे प्रायोगिक राहिलेले नाहीत).
* **सुधारणा (Fix):** प्रदाता ड्रॅग-टू-रीऑर्डर (drag-to-reorder) साठी अपरिभाषित DragHandle ऐवजी युनिकोड ग्रिप आयकॉन (unicode grip icon) वापरले.
* **सुधारणा (Fix):** प्रत्येक पेज लोडवर ट्रिगर होणाऱ्या ब्लॉकिंग HTTP विनंतीला (blocking HTTP request) काढून टाकले.
* **सुधारित (Improved):** GitHub Actions वर्कफ्लो (workflows) Node.js 24 पर्यंत अपग्रेड केले.
