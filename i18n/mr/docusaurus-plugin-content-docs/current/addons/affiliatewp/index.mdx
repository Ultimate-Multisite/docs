# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

तुमच्या Ultimo Network चा ग्राहक आधार वाढवण्यासाठी शक्तिशाली AffiliateWP चा वापर करा!

## Installation

1. 'ultimate-multisite-git ' ही फाईल '/wp-content/plugins/' डिरेक्टरीमध्ये अपलोड करा
2. WordPress मधील 'Plugins' मेनूद्वारे प्लगइन सक्रिय (Activate) करा
3. प्लगइन सेट करण्यासाठी स्टेप-बाय-स्टेप Wizard चे अनुसरण करा

## Usage Guide

[AffiliateWP](https://affiliatewp.com/) हे तुमच्या ऑनलाइन व्यवसायासाठी एफिलिएट प्रोग्राम तयार करताना एक मुख्य उपाय (go-to solution) बनले आहे. Ultimate Multisite: AffiliateWP हे एक पूल (bridge) आहे जे तुम्हाला तुमच्या प्रीमियम नेटवर्कसाठीही तेच काहीतरी तयार करण्याची क्षमता देईल.

## Getting Started

सर्वात आधी, तुम्हाला **[AffiliateWP](https://affiliatewp.com/) सक्रिय (active) असणे आवश्यक आहे**! जर तुमच्याकडे **ते प्लगइन नसेल जे यासोबत एकत्रीकरण (integrate) करते**, तर तुम्ही आमचे ॲड-ऑन वापरू शकणार नाही. जर तुमचा विचार असेल की तुम्ही तुमच्या क्लायंटला ते त्यांच्या साइटवर वापरू देणार नाही, तर AffiliateWP फक्त तुमच्या मुख्य साइटवर सक्रिय असावे.

तसेच, तुम्हाला **Ultimate Multisite** आणि **Ultimate Multisite: AffiliateWP** दोन्ही **संपूर्ण नेटवर्कवर (network-wide) सक्रिय** करण्याची आवश्यकता असेल.

## Adding Ultimate Multisite as an AffiliateWP Integration

तुमच्या मुख्य साइटवर, 'Affiliates Settings' येथे जा आणि 'Integrations' टॅबवर, Ultimate Multisite चेक करा. यामुळे [AffiliateWP](https://affiliatewp.com/) ला Ultimate Multisite साइन-अप फ्लोमधून भेटवस्तू (visits) आणि रूपांतरण (conversions) ट्रॅक करण्यास सुरुवात करण्यास सांगेल.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_खाली Ultimate Multisite बॉक्स तपासा आणि सेव्ह करा_

बस, इतकेच करावे लागते. रेफरल लिंक्समधून येणाऱ्या भेटी (visits) ट्रॅक केल्या जातील आणि जेव्हा तो वापरकर्ता नंतर त्याचा पहिला पेमेंट करतो, तेव्हा रूपांतरण (conversion) तयार केले जाईल आणि रूपांतरणसाठी जबाबदार एफिलिएटकडे कमिशन बाजूला ठेवले जाईल.

मात्र, फक्त नवीन तयार केलेल्या वापरकर्त्याकडून मिळालेले पहिले पेमेंटच ट्रॅक केले जाईल. आवर्ती रेफरल पेमेंट (recurring referral payments) सक्षम करण्यासाठी, तुम्हाला [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ची आवश्यकता असेल.

## Enabling Recurring Referrals

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) स्थापित केल्यानंतर, 'Affiliates Settings' मधील 'Recurring Referrals' टॅबवर जा आणि आवर्ती रेफरल्सचा वापर सक्षम करा.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_हे खूप सोपे आहे: Recurring Referrals सक्षम करा आणि तुम्ही तयार आहात!_

पुन्हा, इतकेच करावे लागते!

आता, जेव्हा रेफर केलेल्या खात्यावर (referred account) नवीन पेमेंट प्राप्त होते, तेव्हा जबाबदार एफिलिएटला आपोआप त्याचे कमिशन मिळेल!
