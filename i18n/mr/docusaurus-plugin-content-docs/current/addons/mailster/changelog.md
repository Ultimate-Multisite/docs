# Mailster बदलांचा इतिहास

### 1.0.0
* पहिली आवृत्ती (Initial release)
* साइनअप किंवा पेमेंटवर आपोआप सबस्क्रायबर सिंक (Automatic subscriber sync on signup or payment)
* उत्पादन-विशिष्ट लिस्ट असाइनमेंट (Product-specific list assignments)
* GDPR अनुपालनासाठी चेकबॉक्स-आधारित ऑप्ट-इन (Checkbox-based opt-in for GDPR compliance)
* डबल ऑप्ट-इन सपोर्ट (Double opt-in support)
* सर्वसमावेशक फील्ड मॅपिंग (Comprehensive field mapping)
* तपशीलवार लॉगिंग (Detailed logging)
