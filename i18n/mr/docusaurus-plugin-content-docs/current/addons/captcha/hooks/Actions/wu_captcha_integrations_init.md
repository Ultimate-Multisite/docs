# Action: wu_captcha_integrations_init

जेव्हा सर्व इंटिग्रेशन्स (integrations) सुरू होतात, तेव्हा हे ॲक्शन फायर होते.

थर्ड-पार्टी इंटिग्रेशन्स (Third-party integrations) येथे 'हुक' (hook) करू शकतात आणि स्वतःची इंटिग्रेशन जोडू शकतात.
