# Filter: wu_cap_captcha_config

Cap Captcha कॉन्फिगरेशन फिल्टर करा.

डेव्हलपर्सना Cap Captcha सेटिंग्ज कस्टमाइझ करण्याची परवानगी देतो.

## पॅरामीटर्स

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Cap कॉन्फिगरेशन ॲरे. |
| $level | `string` | सुरक्षा स्तर (fast, medium, max). |
