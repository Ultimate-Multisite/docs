# Filter: networks_user_is_network_admin

वापरकर्त्याला कोणत्या नेटवर्कचे व्यवस्थापन (administrator) आहे, हे फिल्टर करते.

## पॅरामीटर्स {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $ | `array\|bool` | नेटवर्क आयडींची यादी किंवा वापरकर्त्यासाठी कोणतेही नेटवर्क नसल्यास false. |
| $ | `int` | ज्या वापरकर्त्यासाठी नेटवर्क परत करायचे आहे त्याचा युजर आयडी. |
