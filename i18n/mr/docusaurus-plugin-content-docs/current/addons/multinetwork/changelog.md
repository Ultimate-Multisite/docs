# मल्टी-नेटवर्क बदलांचा इतिहास {#multi-network-changelog}

### 1.3.0 {#130}
* नवीन: नेटवर्क टेम्पलेट पूर्वावलोकन करणारा (Network template previewer) — खरेदी करण्यापूर्वी लाइव्ह पॅनेलसह नेटवर्क टेम्पलेट्स ब्राउझ करा आणि पूर्वावलोकन करा.
* नवीन: चेकआउटवर नेटवर्क टेम्पलेट निवडक (Network template picker) — ग्राहक साइनअप चेकआउट प्रक्रियेदरम्यान नेटवर्क टेम्पलेट निवडू शकतात.
* नवीन: क्रॉस-नेटवर्क मॅजिक लिंक SSO — ग्राहक मॅजिक-लिंक प्रमाणीकरण (authentication) द्वारे उप-नेटवर्क्समध्ये सहजपणे प्रवेश करू शकतात.
* नवीन: नेटवर्क मीडिया क्लोनिंग — टेम्पलेटमधून नेटवर्क क्लोन करताना साइट मीडिया योग्यरित्या डुप्लिकेट होतो.
* दुरुस्ती: मल्टी-टेनन्सी अॅडऑनशिवाय आता नेटवर्क क्लोनिंग कार्य करते; क्लोन केल्यानंतर सुपर ॲडमिन विशेषाधिकार (privileges) पुन्हा दिले जातात.
* दुरुस्ती: नेटवर्क निर्मिती आता WooCommerce आणि सर्व पेमेंट गेटवेसह सुसंगत आहे.
* दुरुस्ती: फॉलबॅक डोमेन निर्मितीमध्ये आता पोर्ट नंबर समाविष्ट केला आहे.
* दुरुस्ती: टेम्पलेट पिकर CSS सर्व थीमवर योग्य रेंडरिंगसाठी एनक्यू (enqueued) केलेल्या स्टाइलशीटमध्ये हलवला आहे.
* दुरुस्ती: साइट क्वेरी स्कोपमध्ये (site query scope) undefined `network_id` की विरुद्ध संरक्षण (Guard) जोडले.
* दुरुस्ती: कॉन्टेक्स्ट स्विचिंगपूर्वी नेटवर्क परमिशन तपासणीमुळे `wpdb` टेबल प्रॉपर्टी त्रुटी (errors) टाळल्या जातात.
