# सपोर्ट तिकीट बदलनोंद {#support-tickets-changelog}

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* सुधारणा: `vendor/` डिरेक्टरी Git ट्रॅकिंगमधून काढली (जी आधीच .gitignore मध्ये कव्हर आहे), ज्यामुळे रिपॉझिटरीचा आकार कमी होईल.
* सुधारणा: WordPress 7.0 पर्यंत चाचणी केली.

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* दुरुस्ती: नल (null) व्हॅल्यूज स्वीकारल्या जातील, जेथे तिकीट मॉडेल सेटर्समध्ये नल व्हॅल्यूज असू शकतात.
* दुरुस्ती: योग्य मल्टीसाइट सपोर्टसाठी सर्व तिकीट ग्लोबल नेटवर्क-व्यापी टेबलमध्ये साठवले जातील.
* दुरुस्ती: स्टाफ फील्ड्स लपवली आणि ग्राहकच्या नवीन तिकीट फॉर्मवर (new-ticket form) ब्लँक पेज रीडायरेक्टची समस्या दुरुस्त केली.
* दुरुस्ती: मेटाडेटा स्टोरेजसाठी `undefined add_meta()` कॉल योग्य `update_meta()` ने बदलले.
* दुरुस्ती: नोंदणीकृत नसलेल्या क्षमता तपासणीला (unregistered capability check) योग्य `wu_view_all_support_tickets` ने बदलले.
* दुरुस्ती: तिकीट स्टेटस, असाइनमेंट आणि क्विक-एडिट ॲक्शनसाठी गहाळ AJAX हँडलर्स जोडले.
* दुरुस्ती: तिकीट व्ह्यूमध्ये स्टाफ रिस्पॉन्स डिटेक्शनसाठी मेथडचे नाव दुरुस्त केले.
* दुरुस्ती: डुप्लिकेट उत्तर हँडलर्स एकत्र केले आणि नॉनस ॲक्शनची नावे संरेखित केली.
* दुरुस्ती: `[wu_submit_ticket]` शॉर्टकोडसाठी गहाळ फ्रंटएंड व्ह्यू जोडले.
* दुरुस्ती: गहाळ `user_id` कॉलम जोडला आणि `Support_Ticket::get_user_id()` मेथड दुरुस्त केली.
* दुरुस्ती: प्रायोरिटी फिल्टर ऑप्शन टॅगमधील अतिरिक्त डबल-कोट काढून टाकले.
* दुरुस्ती: सुपर ॲडमिनसाठी नेटवर्क ॲडमिन तिकीट व्यवस्थापन पॅनेल जोडले.
* सुधारणा: ॲडमिन CSS एकाच बाह्य स्टाइलशीटमध्ये एकत्र केले.
* सुधारणा: सबसाइट ॲडमिन मेनूमधून सेटिंग्ज सबमेनू काढले.
* सुधारणा: सपोर्ट तिकीट पेजेसवरच फ्रंटएंड ॲसेट्स सशर्तपणे लोड केले जातील.
* सुधारणा: जेव्हा Bedrock रूट ॲडलोडरने आधीच डिपेंडन्सी लोड केल्या असतील, तेव्हा प्लगइन ऑटोलोडरला वगळले जाईल.

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* जोडले: ॲडमिन आणि फ्रंटएंडसाठी पूर्ण तिकीट पाहण्याची कार्यक्षमता.
* जोडले: तिकीट उत्तर सबमिशनसाठी AJAX हँडलर.
* जोडले: योग्य फॉर्म हाताळणीसह तिकीट उत्तर कार्यक्षमता सपोर्ट.
* जोडले: ॲडमिनमध्ये तिकीट सबमिशन आणि उत्तरांसाठी योग्य सूचना (notice) डिस्प्ले.
* जोडले: ग्राहक स्वतः सबमिट केलेल्या तिकीटशी आपोआप सध्याच्या वापरकर्त्याचे (current user) कनेक्शन.
* जोडले: ग्राहकांना तिकीट मालकीवर ओव्हरराईड करण्यापासून रोखण्यासाठी सुरक्षा सुधारणा.
* जोडले: गहाळ मदतनीस फंक्शन्स (wu_format_date, wu_user_can_view_ticket, इत्यादी).
* जोडले: योग्य फाइल अटॅचमेंट डाउनलोड आणि हाताळणी.
* जोडले: तिकीट उत्तरांसाठी आणि स्टेटस बदलांसाठी ईमेल नोटिफिकेशन सिस्टीम.
* दुरुस्त केले: तिकीट उत्तर फॉर्ममध्ये आता आवश्यक तिकीट ID समाविष्ट आहे.
* दुरुस्त केले: ॲडमिन तिकीट व्ह्यूमध्ये सूचना (notice) दिसण्यात येणाऱ्या समस्या.
* दुरुस्त केले: सपोर्ट तिकीट फंक्शन्समध्ये सिंटॅक्स त्रुटी.
* सुधारणा: उत्तरांसाठी आणि अटॅचमेंट्ससाठी योग्य क्वेरी आणि स्कीमा क्लासेससह डेटाबेस संरचना.
* सुधारणा: तिकीट मॉडेलसाठी ॲट्रिब्यूट्सऐवजी वास्तविक प्रॉपर्टीजकडे (real properties) स्थलांतर.
