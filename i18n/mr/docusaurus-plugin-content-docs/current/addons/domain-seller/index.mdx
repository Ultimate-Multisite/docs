# डोमेन विक्रेता

**Multisite Ultimate: डोमेन विक्रेता** तुमचे WordPress Multisite नेटवर्क एका शक्तिशाली डोमेन विक्री प्लॅटफॉर्ममध्ये रूपांतरित करते. ग्राहकांना साइन-अप दरम्यान किंवा त्यांच्या Dashboard मधून थेट कस्टम डोमेन नोंदणी करू द्या, रिअल-टाइम उपलब्धता तपासणी आणि पारदर्शक किंमतींसह.

### डोमेन विक्रेता का?

डोमेन नोंदणी ही कोणत्याही WaaS (सेवा म्हणून वेबसाइट) प्लॅटफॉर्मसाठी सर्वाधिक मागणी असलेल्या अपसेल्सपैकी एक आहे. तुमच्या ग्राहकांना तृतीय-पक्ष नोंदणीकर्त्याकडे पाठवण्याऐवजी, तुम्ही थेट डोमेन देऊ शकता — ग्राहकांना अडथळामुक्त अनुभव देताना तुमच्या नेटवर्कमध्ये अधिक महसूल ठेवून.

### मुख्य वैशिष्ट्ये

#### अनेक डोमेन नोंदणीकर्त्यांचे समर्थन
* **OpenSRS** — पूर्ण नोंदणी, नूतनीकरण आणि व्यवस्थापन
* **Namecheap** — WhoisGuard आणि सँडबॉक्स चाचणीसह संपूर्ण एकत्रीकरण
* **HostAfrica** — Domains Reseller API द्वारे डोमेन शोध, किंमत, नोंदणी, हस्तांतरण, DNS आणि ID संरक्षण
* **Openprovider** — पुनर्विक्रेता किंमत, नोंदणी, नूतनीकरण, हस्तांतरण, DNS, WHOIS गोपनीयता आणि TLD समक्रमण
* **Hostinger** — सामायिक Hostinger एकत्रीकरण टोकन वापरून डोमेन शोध आणि नोंदणी
* **विस्तारक्षम आर्किटेक्चर** — Integration Registry पॅटर्नवर बांधलेले, त्यामुळे अधिक प्रदाते जोडणे सोपे होते
* **अंगभूत कनेक्शन चाचणी** — प्रत्यक्ष वापरात आणण्यापूर्वी प्रदाता क्रेडेन्शियल्स आणि कनेक्टिव्हिटीची पडताळणी करा

#### मार्गदर्शित सेटअप विझार्ड
* **एक-क्लिक ऑनबोर्डिंग** — प्रदाता कॉन्फिगरेशन, TLD आयात आणि डोमेन उत्पादन निर्मितीमध्ये तुम्हाला मार्गदर्शन करते
* **स्वयंचलित TLD समक्रमण** — तुमच्या जोडलेल्या प्रदात्यांकडून सर्व उपलब्ध TLDs आणि घाऊक किंमती आयात करते
* **डीफॉल्ट उत्पादन निर्मिती** — योग्य मार्कअप डीफॉल्ट्ससह वापरण्यास तयार डोमेन उत्पादन तयार करते

#### लवचिक डोमेन किंमत
* **तीन मार्कअप मोड** — टक्केवारी मार्कअप, निश्चित मार्कअप किंवा प्रति TLD निश्चित किंमत
* **TLD-विशिष्ट उत्पादने** — वेगवेगळ्या TLD गटांसाठी स्वतंत्र डोमेन उत्पादने तयार करा (.com/.net, .io, देश कोड्स इ.)
* **प्रास्ताविक किंमत** — नवीन नोंदणी आकर्षित करण्यासाठी सवलतीची पहिल्या वर्षाची किंमत द्या
* **रिअल-टाइम किंमत** — बुद्धिमान कॅशिंगसह प्रदात्यांकडून थेट घाऊक किंमत
* **कूपन समर्थन** — Multisite Ultimate सवलत कोड्ससह पूर्ण सुसंगतता

#### WHOIS गोपनीयता संरक्षण
* **प्रति-उत्पादन कॉन्फिगरेशन** — WHOIS गोपनीयता Disabled, Always Included किंवा Customer Choice म्हणून सेट करा
* **ग्राहक ऑप्ट-इन** — Customer Choice वर सेट केल्यावर, चेकआउट चेकबॉक्स ग्राहकांना गोपनीयता संरक्षण जोडू देतो
* **कॉन्फिगर करण्यायोग्य किंमत** — गोपनीयता अॅड-ऑनसाठी प्रति-वर्ष किंमत सेट करा किंवा ते मोफत द्या
* **नोंदणीकर्ता एकत्रीकरण** — नोंदणीच्या वेळी WhoisGuard (Namecheap) किंवा WHOIS गोपनीयता (OpenSRS) स्वयंचलितपणे सक्षम करते

#### एकत्रित चेकआउट अनुभव
* **तीन डोमेन मोड** — ग्राहक उपडोमेन (मोफत), नवीन डोमेन नोंदणी किंवा त्यांच्या आधीपासून मालकीच्या विद्यमान डोमेनचे मॅपिंग यांपैकी निवडतात
* **रिअल-टाइम डोमेन शोध** — पृष्ठ रीलोड न करता AJAX-संचालित उपलब्धता तपासणी
* **स्मार्ट सूचना** — पसंतीचे डोमेन घेतलेले असल्यास पर्यायी TLD सूचना
* **पारदर्शक किंमत** — नोंदणी किंमत, नूतनीकरण किंमत आणि पर्यायी WHOIS गोपनीयता शुल्काचे स्पष्ट प्रदर्शन
* **नोंदणीकर्ता संपर्क फील्ड्स** — चेकआउटदरम्यान आवश्यक नोंदणीकर्ता माहिती (नाव, पत्ता, फोन) इनलाइन गोळा करते
* **स्वयंचलितपणे तयार केलेला साइट URL** — निवडलेल्या डोमेनवरून साइटचा पत्ता स्वयंचलितपणे मिळवला जातो

#### स्वयंचलित नूतनीकरण प्रणाली
* **सदस्यत्वाशी जोडलेली नूतनीकरणे** — ग्राहकाचे सदस्यत्व सक्रिय असताना डोमेन स्वयंचलितपणे नूतनीकरण होतात
* **स्मार्ट शेड्यूलिंग** — डोमेन कालबाह्यता तारखांवर आधारित नूतनीकरण जॉब्स रांगेत ठेवले जातात
* **पुन्हा प्रयत्न लॉजिक** — अयशस्वी नूतनीकरण प्रयत्नांसाठी बुद्धिमान पुनर्प्रयत्न
* **ईमेल सूचना** — आगामी नूतनीकरणे, यशस्वी नूतनीकरणे, अपयश आणि कालबाह्यता यांसाठी कॉन्फिगर करण्यायोग्य अलर्ट्स

#### ग्राहक DNS व्यवस्थापन
* **स्व-सेवा DNS** — ग्राहक त्यांच्या Dashboard मधून DNS रेकॉर्ड्स (A, AAAA, CNAME, MX, TXT) जोडू, संपादित आणि हटवू शकतात
* **डोमेन स्थिती** — रिअल-टाइम नोंदणी स्थिती, कालबाह्यता तारखा आणि नेमसर्व्हर माहिती
* **स्वयंचलित डोमेन मॅपिंग** — नोंदणीकृत किंवा विद्यमान डोमेन ग्राहकाच्या साइटशी स्वयंचलितपणे मॅप केले जातात

#### प्रशासक साधने
* **मॅन्युअल डोमेन नोंदणी** — प्रशासक पॅनेलमधून ग्राहकांच्या वतीने डोमेन नोंदणी करा
* **डोमेन मॉनिटरिंग** — सर्व ग्राहक डोमेन, स्थिती आणि कालबाह्यता तारखा एका पृष्ठावरून पहा
* **प्रदाता-विशिष्ट लॉगिंग** — प्रत्येक नोंदणीकर्त्यासाठी स्वतंत्र लॉग चॅनेल्स (उदा., domain-seller-namecheap.log)
* **नोंदणी तपशील** — मुख्य डोमेन संपादन पृष्ठावर नोंदणीकर्ता माहिती आणि नोंदणी डेटा पहा

### समर्थित प्रदाते

* **OpenSRS** — नोंदणी, नूतनीकरण, DNS व्यवस्थापन आणि WHOIS गोपनीयतेसह पूर्ण एकत्रीकरण
* **Namecheap** — नोंदणी, नूतनीकरण, बॅच उपलब्धता तपासणी आणि मोफत WhoisGuard सह पूर्ण एकत्रीकरण
* **HostAfrica** — नोंदणी, नूतनीकरण, हस्तांतरण, DNS व्यवस्थापन, EPP कोड मिळवणे, नोंदणीकर्ता लॉक आणि ID संरक्षण
* **Openprovider** — नोंदणी, नूतनीकरण, हस्तांतरण, DNS व्यवस्थापन, EPP कोड मिळवणे, नोंदणीकर्ता लॉक, WHOIS गोपनीयता आणि पुनर्विक्रेता TLD समक्रमण
* **Hostinger** — नोंदणी, डोमेन शोध, नेमसर्व्हर अद्यतने, नोंदणीकर्ता लॉक आणि WHOIS गोपनीयता; नूतनीकरणे Hostinger ऑटो-रिन्यूद्वारे हाताळली जातात
* **GoDaddy** — नोंदणी आणि नूतनीकरण
* **ResellerClub** — नोंदणी, नूतनीकरण आणि DNS व्यवस्थापन
* **NameSilo** — नोंदणी आणि नूतनीकरण
* **Enom** — नोंदणी, नूतनीकरण आणि DNS व्यवस्थापन

### तांत्रिक तपशील

* **आवश्यक:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **आर्किटेक्चर:** Domain_Selling_Capability इंटरफेससह Integration Registry पॅटर्न
* **ऑटोलोडिंग:** PSR-4 क्लास मॅपिंगसह Jetpack ऑटोलोडर
* **मानके:** WordPress कोडिंग मानके, स्ट्रिक्ट टायपिंग, प्रिपेअर्ड स्टेटमेंट्स

## स्थापना

1. **Multisite Ultimate v2.4.12+** स्थापित आणि नेटवर्क-सक्रिय केलेले असल्याची खात्री करा
2. plugin `/wp-content/plugins/ultimate-multisite-domain-seller/` येथे अपलोड करा
3. WordPress Network Admin मधील **Plugins** मधून plugin नेटवर्क-सक्रिय करा
4. **Setup Wizard** पहिल्या सक्रियतेवर आपोआप सुरू होतो — पुढील चरणांसाठी पायऱ्या अनुसरा:
   * तुमचा domain registrar कनेक्ट करा
   * कनेक्शनची चाचणी करा
   * उपलब्ध TLDs आणि घाऊक किंमत आयात करा
   * किंमत markup सह तुमचा पहिला domain product तयार करा
5. **Ultimate Multisite > Checkout Forms** मध्ये तुमच्या checkout form मध्ये **Domain Selection** field जोडा

## वारंवार विचारले जाणारे प्रश्न

### Multisite Ultimate ची कोणती आवृत्ती आवश्यक आहे?

आवृत्ती 2.4.12 किंवा त्याहून अधिक. addon WP Ultimo v1.x किंवा 2.4.12 पेक्षा कमी Ultimate Multisite आवृत्त्यांशी सुसंगत नाही.

### कोणते domain providers समर्थित आहेत?

नऊ registrars समर्थित आहेत: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, आणि **Enom**. registrar APIs वेगवेगळ्या क्रिया उघड करत असल्यामुळे क्षमता provider नुसार बदलतात; provider-दर-provider आढाव्यासाठी setup guide पहा. addon ची Integration Registry रचना custom providers जोडणे सोपे करते — [developer guide](/developer/integration-guide/custom-registrar) पहा.

### मी WHOIS privacy protection देऊ शकतो का?

होय. प्रत्येक domain product मध्ये तीन मोडसह WHOIS Privacy setting असते: Disabled, Always Included, किंवा Customer Choice. Customer Choice वर सेट केल्यास, ग्राहकांना checkout दरम्यान किंमतीसह (किंवा Namecheap च्या WhoisGuard साठी "Free") checkbox दिसतो. registration वेळी privacy flag registrar कडे आपोआप पाठवला जातो.

### मी वेगवेगळ्या TLDs साठी वेगवेगळी pricing सेट करू शकतो का?

होय. वेगवेगळ्या TLD समूहांसाठी स्वतंत्र domain products तयार करा. प्रत्येक product चा स्वतःचा markup type (टक्केवारी, निश्चित markup, किंवा निश्चित किंमत), समर्थित TLDs, आणि introductory pricing असू शकते.

### ग्राहकांना कोणते checkout पर्याय दिसतात?

Domain Selection field ग्राहकांना तीन tabs देते: **Subdomain** (तुमच्या नेटवर्कवर मोफत subdomain वापरा), **Register New Domain** (नवीन domain शोधा आणि register करा), आणि **Existing Domain** (त्यांच्या मालकीचा आधीच असलेला domain map करा). तुम्ही सक्षम केलेले modesच दिसतील.

### ते automatic renewals ला समर्थन देते का?

होय. domains auto-renew करण्यासाठी सेट करता येतात आणि ते ग्राहकाच्या membership status शी जोडलेले असतात. renewal payments ग्राहकाच्या विद्यमान payment gateway द्वारे प्रक्रिया केले जातात.

### ग्राहक त्यांचे DNS records व्यवस्थापित करू शकतात का?

होय. ग्राहकांना त्यांच्या dashboard मध्ये A, AAAA, CNAME, MX, आणि TXT records जोडण्यासाठी, संपादित करण्यासाठी आणि हटवण्यासाठी DNS management interface मिळतो. DNS management OpenSRS, ResellerClub, Enom, HostAfrica, आणि Openprovider सोबत उपलब्ध आहे. hosted domains साठी Hostinger DNS core Hostinger domain-mapping integration द्वारे हाताळले जाते.

### setup wizard आहे का?

होय. setup wizard पहिल्या सक्रियतेवर आपोआप चालतो आणि registrar कनेक्ट करणे, TLDs आयात करणे, आणि तुमचा पहिला domain product तयार करणे यांत तुम्हाला मार्गदर्शन करतो. तुम्ही addon settings मधून wizard पुन्हा चालवू शकता.
