# Domain Seller बदलांचा इतिहास

Version 1.1.0 - रिलीझ झाले: 2026-05-08
- नवीन: ResellerClub, Enom, आणि OpenSRS रजिस्ट्रारसाठी DNS रेकॉर्ड तयार करण्याची सुविधा (add_dns_record) लागू केली.
- सुधारणा: डिफॉल्ट DNS रेकॉर्ड पार्सर आता {DOMAIN} आणि {SITE_URL} टोकन्सला स्वीकारतो.
- सुधारणा: डोमेन निवड चेकआउट फील्डचे स्लग्स (slugs) नेम्सपेस केले आहेत, जेणेकरून site_url सोबत टक्कर होणार नाही.

* नवीन: NameSilo रजिस्ट्रार इंटिग्रेशन.

* सुधारणा: install-wp-tests.sh मध्ये MySQL flags योग्यरित्या शब्द-विभाजन (word-split) करतात.

* **सुधारित:** सेटिंग्ज पेजवर सर्वत्र चांगले वर्णन (search TLDs, renewals, DNS, notifications).

Version 1.0.1 - रिलीझ झाले: 2026-02-27

* **नवीन:** बल्क किंमत व्यवस्थापनासाठी TLD इम्पोर्ट टूल.
* **नवीन:** डोमेन उत्पादनांसाठी परिचय किंमत (Introductory pricing) समर्थन.
* **नवीन:** Cypress सह E2E चाचणी सूट (E2E test suite).
* **नवीन:** डोमेन जीवनचक्र सूचनांसाठी ईमेल टेम्पलेट्स.
* **नवीन:** admin डोमेन नोंदणी मोडलवर रजिस्ट्रंट पत्त्याचे फील्ड, जे सेटिंग्जमधून आपोआप भरले जाते.
* **नवीन:** डोमेन व्यवस्थापन इंटरफेस, ज्यामध्ये रेकॉर्ड जोडणे, संपादित करणे आणि हटवण्याचे समर्थन आहे.
* **नवीन:** आपोआप डोमेन मॅपिंगसह "Bring your own domain" चेकआउट पर्याय.
* **नवीन:** चेकआउट दरम्यान डोमेन नावावरून साइट URL आपोआप तयार करणे.
* **नवीन:** सेटिंग्जमध्ये डिफॉल्ट नेमसर्व्हर आणि DNS रेकॉर्ड कॉन्फिगरेशन.
* **नवीन:** मुख्य डोमेन संपादन पृष्ठावर डोमेन नोंदणी तपशील आणि DNS व्यवस्थापन.
* **नवीन:** सेटअप विजार्ड आपोआप योग्य डिफॉल्टसह एक डिफॉल्ट डोमेन उत्पादन तयार करतो.
* **नवीन:** सर्व कॉन्फिगर केलेल्या प्रदात्यांमध्ये (providers) क्रॉनद्वारे (cron) दैनिक आपोआप TLD सिंक.
* **नवीन:** प्रति-उत्पादन कॉन्फिगरेशनसह WHOIS गोपनीयता संरक्षण (always on, customer choice, किंवा disabled).
* **नवीन:** किंमत प्रदर्शन आणि डार्क मोड समर्थनसह WHOIS गोपनीयता चेकबॉक्स.
* **नवीन:** मॅन्युअल डोमेन नोंदणीसाठी Register Domain admin पेज.
* **नवीन:** Ultimate Multisite update server द्वारे आपोआप प्लगइन अपडेट्स.
* **नवीन:** उत्पादनांच्या सूची टेबलमध्ये डोमेन उत्पादन प्रकार फिल्टर टॅब, जांभळ्या बॅज स्टायलिंगसह.
* **नवीन:** डोमेन चेकआउट फॉर्मवर रजिस्ट्रंट संपर्क फील्ड (नाव, पत्ता, शहर, राज्य, पिन कोड, देश, फोन).
* **नवीन:** रजिस्ट्रार API कॉल करण्यापूर्वी रजिस्ट्रंट फील्ड व्हॅलिडेशनसह स्पष्ट त्रुटी संदेश.
* **नवीन:** डोमेन नोंदणी घटनांसाठी प्रदाता-विशिष्ट लॉग चॅनेल (उदा. domain-seller-namecheap.log).
* **नवीन:** मुख्य नोंदणी/साइनअप चेकआउट फॉर्मवर रजिस्ट्रंट संपर्क फील्ड (डोमेन नोंदणी करताना दाखवले जाते).
* **सुधारित:** डोमेन सर्च चेकआउट फील्डला एकत्रित डोमेन निवड फील्डने बदलले, जे सबडोमेन, रजिस्टर आणि विद्यमान डोमेन टॅबला समर्थन देते.
* **सुधारित:** डोमेन उत्पादन सेटिंग्ज कोर विजेट सिस्टमद्वारे उत्पादन संपादन पृष्ठावर इनलाइन रेंडर होतात.
* **सुधारित:** ग्राहक डोमेन माहिती स्वतंत्र मेटाबॉक्सऐवजी कोर डोमेन मॅपिंग विजेटमध्ये जोडली जाते.
* **सुधारित:** TLD इम्पोर्ट विजार्ड सर्व प्रदात्यांकडून एका क्लिकमध्ये सिंक करण्यासाठी सोपे केले.
* **सुधारित:** Namecheap डोमेन उपलब्धता जलद शोधासाठी बॅच API कॉल वापरते.
* **सुधारित:** Namecheap किंमत API योग्य पॅरामीटर्स आणि प्रतिसाद पार्सिंग वापरते.
* **सुधारित:** TLD स्टोरेज एकाच नेटवर्क पर्यायामध्ये केंद्रीकृत केले.
* **सुधारित:** DNS बदल, ट्रान्सफर आणि कॉन्फिगरेशन लागू करण्यासाठी डोमेन ॲक्टिव्हिटी लॉगिंग.
* **सुधारित:** बॅच व्हॅलिडेशनसह OpenSRS साठी पूर्ण TLD सिंक.
* **सुधारित:** पृष्ठांकित API विनंत्यांसह Namecheap साठी पूर्ण TLD सिंक.
* **सुधारित:** जुन्या प्रदाता क्लासेसला Integration Registry पॅटर्नने बदलले.
* **सुधारित:** DNS आणि ट्रान्सफर कॉन्फिगरेशनसह सेटिंग्ज पॅनेल.
* **सुधारित:** फोन नंबर आपोआप +CC.NNN रजिस्ट्रार फॉरमॅटमध्ये स्वरूपित होतात.
* **सुधारित:** फोन फील्ड व्हॅलिडेशन सबमिशन करण्यापूर्वी स्वरूपण वर्ण काढून टाकते.
* **सुधारित:** व्हर्जन आवश्यकता Ultimate Multisite 2.4.12 वर वाढवली, स्पष्ट सूचनासह.
* **सुधारित:** CI वर्कफ्लो ॲडऑन आणि कोर प्लगइन दोघांसाठी योग्य चेकआउट वापरतो.
* **सुधारित:** prepare_registrant_info() चेकआउट-सेव्ह केलेल्या वापरकर्ता मेटामधून वाचते, बिलिंग पत्त्याच्या बॅकअपसह.
* **दुरुस्त:** चेकआउट दरम्यान लॉग-इन नसलेल्या वापरकर्त्यांसाठी डोमेन सर्च AJAX अयशस्वी होत होते.
* **दुरुस्त:** चेकआउट दरम्यान लॉग-इन नसलेल्या वापरकर्त्यांसाठी डोमेन किंमत AJAX अयशस्वी होत होते.
* **दुरुस्त:** WP-CLI कमांड चालवताना Spyc क्लास पुनर्रचना (redeclaration) घातक त्रुटी देत होती.
* **दुरुस्त:** Namecheap सँडबॉक्स API टाइमआउट खूप कमी होता.
* **दुरुस्त:** डोमेन सर्च 'Select' बटण मजकूर हिरव्या बॅकग्राउंडवर दिसत नव्हता.
* **दुरुस्त:** संपर्क माहिती गहाळ असल्यामुळे "RegistrantFirstName is Missing" त्रुटीसह डोमेन नोंदणी अयशस्वी होत होती.
* **दुरुस्त:** चेकआउट वेळी साइट अस्तित्वात नसतानाही डोमेन रेकॉर्ड blog_id=0 सह तयार होत होता.
* **दुरुस्त:** डिफॉल्ट TLD सेटिंग स्ट्रिंग म्हणून परत येत होती, पार्स केलेल्या ॲरेऐवजी.
* **काढले:** स्वतंत्र डोमेन व्यवस्थापन admin पेज — आता कोर डोमेन पृष्ठांद्वारे हाताळले जाते.
