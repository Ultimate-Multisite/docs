# Action: wu_domain_seller_register_integrations

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | इंटिग्रेशन रजिस्ट्री. |

### Since {#since}

- 2.1.0

[`inc/class-domain-seller.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-domain-seller/blob/main/inc/class-domain-seller.php#L293) या फाईलमध्ये लाइन २९३ वर परिभाषित केले आहे.
