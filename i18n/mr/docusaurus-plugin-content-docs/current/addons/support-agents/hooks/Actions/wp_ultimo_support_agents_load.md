# Action: wp_ultimo_support_agents_load

जेव्हा सर्व अवलंबित्व (dependencies) लोड होतात, तेव्हा हे ट्रिगर होते.

यामुळे प्लगइन डेव्हलपर्सना नवीन कार्यक्षमता (functionality) जोडता येते. उदाहरणार्थ, नवीन होस्टिंग प्रदात्यांना (Hosting providers) सपोर्ट जोडणे इत्यादी.
