# Support Agents {#support-agents}

संपूर्ण नेटवर्कचे व्यवस्थापन स्वतः करणे हे सोपे काम नाही. जसजसे तुमचे नेटवर्क वाढू लागेल, तसतसे तुम्हाला सपोर्टची वाढती मागणी जाणवेल. तुमचा कामाचा भार कमी करण्यासाठी मदत करणारे लोक आणणे चांगले राहील. मात्र, पूर्ण ॲक्सेस/परवानगी देणे धोकादायक असू शकते आणि तुमच्या नेटवर्कला धोका पोहोचू शकते. इथेच हे Support Agent add-on उपयुक्त ठरते.

हे add-on तुम्हाला Support Agent नावाचा एक नवीन प्रकारचा WordPress user तयार करण्याची सुविधा देते आणि त्या वापरकर्त्याला कोणत्या परवानग्या (permissions) आहेत यावर पूर्ण नियंत्रण देते. परवानगी देणे हे टॉगल बटण (toggle button) ऑन करण्याइतके सोपे आहे आणि तुम्हाला ती काढून टाकायची असल्यास, ते बंद (off) करणे देखील तितकेच सोपे आहे.

## Features {#features}

- WordPress आणि Ultimate Multisite च्या परवानग्यांची (permissions) तपशीलवार यादी.
- सहजपणे ॲक्सेस (access) सक्षम (enable) आणि अक्षम (disable) करा.
- अमर्यादित एजंट खाती (Agent accounts) तयार करा.

## Installation {#installation}

1. 'ultimate-multisite-support-agents' ही फाईल '/wp-content/plugins/' या डिरेक्टरीमध्ये अपलोड करा.
2. WordPress मधील 'Plugins' मेनूद्वारे प्लगइन सक्रिय (activate) करा.
3. प्लगइन सेट करण्यासाठी स्टेप-बाय-स्टेप विझार्ड (Wizard) फॉलो करा.
