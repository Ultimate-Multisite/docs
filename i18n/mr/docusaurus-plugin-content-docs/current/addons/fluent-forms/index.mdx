# Fluent Forms

हा अॅड-ऑन (add-on) Ultimate Multisite मध्ये उत्पादने (products), सदस्यत्व (memberships) किंवा साइट्सवर (sites) Fluent Forms वर मर्यादा (limits) घालण्याची सोय उपलब्ध करतो.
Fluent Forms मध्ये कस्टम पोस्ट प्रकारांऐवजी (custom post types) कस्टम टेबल्स (custom tables) वापरले जातात, त्यामुळे सामान्य मर्यादा घालण्याचे पर्याय (normal limiting options) Fluent Forms सोबत काम करत नाहीत.

## Features {#features}

- Fluent Forms किंवा Conversational Fluent Forms साठी मर्यादा (limits) जोडा.
- जेव्हा ग्राहक मर्यादेपर्यंत पोहोचतील, तेव्हा त्यांना त्यांचा प्लॅन अपग्रेड करण्यासाठी एक मैत्रीपूर्ण संदेश (friendly message) मिळेल.
