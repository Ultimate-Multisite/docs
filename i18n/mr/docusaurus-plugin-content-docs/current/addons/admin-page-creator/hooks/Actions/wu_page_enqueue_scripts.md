# Action: wu_page_enqueue_scripts

येथे एक कस्टम ॲक्शन जोडली आहे, जेणेकरून आपल्याला आवश्यकतेनुसार स्क्रिप्ट्स आणि स्टाइल्स (scripts and styles) लोड करण्यासाठी एक जागा मिळेल आणि ते फक्त योग्य पानांवरच लोड होतील याची खात्री होईल.
