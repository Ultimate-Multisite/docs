# भाषा निवडक (Language Selector) {#language-selector}

Multisite Ultimate: साइन-अपवर भाषा निवडक

साइन-अप प्रक्रियेत (sign-up flow) भाषा निवडण्यासाठी एक फील्ड जोडा.

## इन्स्टॉलेशन (Installation) {#installation}

१. 'ultimate-multisite-language-selector' ही फाईल '/wp-content/plugins/' या डिरेक्टरीमध्ये अपलोड करा.
२. तुमच्या WordPress Network Admin मधील 'Plugins' मेनूद्वारे प्लगइन ॲक्टिव्हेट करा.

## वापर मार्गदर्शिका (Usage Guide) {#usage-guide}

या मोफत ॲड-ऑनमुळे तुम्ही तुमच्या वापरकर्त्यांना (users) साइन-अप प्रक्रियेत आणि लॉगिन पेजवर कोणती भाषा पाहायला आवडेल, हे निवडू देता.

[तुमचे लायसन्स सक्रिय केल्यानंतर](../../user-guide/getting-started/installing-ultimate-multisite.md) तुम्हाला तुमच्या लायसन्सद्वारे उपलब्ध असलेले Ultimate Multisite ॲड-ऑन इन्स्टॉल करता येतील.

तुमच्या Ultimate Multisite सेटिंग्ज पेजवर जा आणि नंतर, डाव्या बाजूच्या मेनूमध्ये, _Add-ons_ लेबलवर क्लिक करा.

![Add-ons page](/img/admin/addons-page.png)

_Language Selector इन्स्टॉल करण्यासाठी Add-ons पेजवर जा_

तिथे तुम्हाला आम्ही दिलेले सर्व ॲड-ऑन मिळतील.

मोफत ॲड-ऑन आमच्या सर्व ग्राहकांना उपलब्ध आहेत, त्यांनी कोणत्या सबस्क्रिप्शन प्लॅनसाठी साइन अप केले आहे यावर याचा परिणाम होत नाही.

ॲड-ऑनच्या थंबनेलच्या खालील _Install it Now!_ बटणावर क्लिक करा. त्यानंतर ते Plugins पेजवर ॲक्टिव्हेट करा.

![Add-ons page with install button](/img/admin/addons-page.png)

_ॲड-ऑन इन्स्टॉल करण्यासाठी बटणावर क्लिक करा_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_तुमच्या Plugins पेजवर ते ॲक्टिव्हेट करा_

आता, Ultimate Multisite च्या Settings मेनूवर जा आणि _Network Settings_ टॅबमध्ये प्रवेश करा. तिथे, पेजच्या तळाशी, तुम्हाला _Languages_ नावाचा एक नवीन विभाग मिळेल.

![Settings page](/img/config/settings-general.png)

_एकदम नवीन विभाग दिसतो!_

तुम्ही ते रिकामे सोडू शकता जेणेकरून सर्व भाषा पर्याय म्हणून दिसतील, किंवा तुम्हाला कोणत्या भाषा उपलब्ध करायच्या आहेत, हे निवडू शकता. भाषा म्हणजे तुमच्या WordPress सेटअपमध्ये इन्स्टॉल केलेल्या भाषा.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_या WP इन्स्टॉलेशनवर उपलब्ध भाषांची यादी_

तुम्ही केलेले बदल सेव्ह केल्यानंतर, तुम्ही निवडलेल्या भाषांसह एक सिलेक्शन मेनू लॉगिन पेजवर आणि साइन-अप फ्लोवर देखील दिसेल.

![Login and Registration settings](/img/config/settings-login-registration.png)

_लॉगिन पेजवर भाषा निवडक_

![Registration page](/img/config/registration-page.png)

## भाषा URL द्वारे सेट करणे (Setting the Language via the URL) {#setting-the-language-via-the-url}

तुम्ही URL मधील locale पॅरामीटर बायपास करून देखील पेजची भाषा बदलू शकता. उदाहरणार्थ, जर तुम्हाला इटालियनमध्ये रजिस्ट्रेशन पेजवर रीडायरेक्ट करायचे असेल, तर URL मध्ये ?locale=it_IT जोडा.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

हे काम करण्यासाठी भाषा तुमच्या WordPress नेटवर्कवर इन्स्टॉल केलेली असणे आवश्यक आहे.
