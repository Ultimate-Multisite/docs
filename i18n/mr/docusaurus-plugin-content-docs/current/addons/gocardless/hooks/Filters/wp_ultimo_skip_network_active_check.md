# फिल्टर: wp_ultimo_skip_network_active_check

विकसकांना (developers) या तपासणीला (check) बायपास करण्याची (short-circuit) सुविधा देते.

जेव्हा तुम्ही composer-आधारित आणि इतर कस्टम सेटअप्स वापरत असता, जसे की Bedrock, जिथे mu-plugins म्हणून प्लगइन्स वापरणे सामान्य आहे, तेव्हा हे उपयुक्त ठरते.
