# GoCardless बदलांचा इतिहास {#gocardless-changelog}

Version: 1.0.3 - रिलीझ केल्याची तारीख 2026-05-05
* दुरुस्ती: जेव्हा GoCardless सदस्यता 'pending' (प्रतीक्षा) किंवा 'on-hold' (थांबलेल्या) स्थितीत रद्द केली जाते, तेव्हा विरहित (orphaned) प्रलंबित साइट्स आपोआप साफ केल्या जातात.
* सुधारित: जेव्हा Bedrock रूट ऑटोलॉडरने आधीच डिपेंडन्सी (dependencies) लोड केल्या असतात, तेव्हा प्लगइन ऑटोलॉडर वगळला जातो.

Version: 1.0.1 - रिलीझ केल्याची तारीख 2025-09-28

* प्रीफिक्स (prefix) 'ultimate-multisite' मध्ये बदलले; टेक्स्ट डोमेन (text domain) अपडेट केले; आणि आवृत्ती (version) वाढवली.


2021-09-24 - Version 1.0.0-beta.4

* जोडले: mu-plugins आधारित सेटअपसाठी `wp_ultimo_skip_network_active_check` फिल्टर जोडला;

2021-06-16 - Version 1.0.0-beta.3

* दुरुस्ती: सिग्नेचर (signatures) कन्फर्म करण्यात अयशस्वी होणारे वेबहुक हँडलर्स (Webhook handlers) दुरुस्त केले;

2021-06-14 - Version 1.0.0-beta.2

* सुधारित: GoCardless मॅन्डेट (mandate) तयार झाल्यानंतर साइट प्रकाशित (Publish) करण्याची सुविधा सुधारली;

Version 1.0.0-beta.1 - प्रारंभिक आवृत्ती
