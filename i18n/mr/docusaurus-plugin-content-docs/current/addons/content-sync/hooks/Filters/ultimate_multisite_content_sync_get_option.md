# Filter: ultimate_multisite_content_sync_get_option

हे नेटवर्क ऑप्शनचे 'slug' (URL-friendly) व्हॅल्यू परत करते.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $option_value | `mixed` | नेटवर्क ऑप्शनचे व्हॅल्यू. |
| $option_name | `string` | ऑप्शनचे नाव. |
| $default | `mixed` | डीफॉल्ट व्हॅल्यू. |

### Since {#since}

- 1.0.0
### Source {#source}

[`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-content-sync/blob/main/inc/class-helper.php#L126) मध्ये लाइन 126 वर परिभाषित केले आहे.


## Returns {#returns}
हे 'slugfield' नेटवर्क ऑप्शनचे व्हॅल्यू परत करते.
