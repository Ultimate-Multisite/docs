# सामग्री सिंक (Content Sync)

Multisite Ultimate: सामग्री सिंक

Multisite Ultimate साठी हा अ‍ॅडऑन (addon) नेटवर्क व्यवस्थापकांना (network administrators) एका साइटची सामग्री (content) नेटवर्कमधील इतर साइट्सवर सिंक (synchronize) करण्याची सुविधा देतो. हे पोस्ट्स (posts), पेजेस (pages), कस्टम पोस्ट प्रकार (custom post types), आणि Elementor टेम्पलेट्ससह सर्व पोस्ट प्रकारांना सपोर्ट करते. या अ‍ॅडऑनमध्ये सामग्री निवडण्यासाठी आणि लक्ष्यित साइट्स (target sites) निवडण्यासाठी एक मल्टी-स्टेप विझार्ड इंटरफेस (multi-step wizard interface) आहे, ज्यामध्ये बॅच प्रोसेसिंग (batch processing) आणि रिअल-टाइम प्रगती फीडबॅक (real-time progress feedback) मिळतो.

## वैशिष्ट्ये (Features) {#features}

* **सामग्री निवड (Content Selection):** थंबनेलसह व्हिज्युअल कार्ड ग्रिड आणि पोस्ट प्रकार फिल्टरिंग.
* **सर्व पोस्ट प्रकार (All Post Types):** पोस्ट्स, पेजेस, प्रॉडक्ट्स, Elementor टेम्पलेट्स आणि कोणतेही कस्टम पोस्ट प्रकार सिंक करा.
* **साइट निवड (Site Selection):** कोणत्या साइट्सवर सामग्री सिंक करायची आहे ते निवडा. यात सोर्स साइट वगळणे (source site exclusion) आणि चाइल्ड साइट पूर्व-निवड (child site pre-selection) करण्याची सुविधा आहे.
* **बॅच प्रोसेसिंग (Batch Processing):** एका वेळी १० साइट्सवर सिंक करणे आणि प्रगती अपडेट्स मिळणे.
* **मीडिया सिंक (Media Sync):** पोस्ट सामग्रीमधून (post content) मीडिया फाइल्स (इमेजेस, व्हिडिओ) आपोआप कॉपी करणे आणि त्यांना लक्ष्यित साइट्सवर पुन्हा मॅप (remap) करणे.
* **टॅक्सोनॉमी सिंक (Taxonomy Sync):** कॅटेगरी (categories), टॅग्स (tags) आणि कस्टम टॅक्सोनॉमी (custom taxonomies) जसेच्या तसे जतन करणे.
* **Elementor सपोर्ट:** CSS रीजेनरेशन (CSS regeneration) आणि टेम्पलेट ID रीमॅपिंग (template ID remapping) सह Elementor टेम्पलेट्ससाठी पूर्ण सपोर्ट.
* **संघर्ष निराकरण (Conflict Resolution):** विद्यमान सामग्री बदलायची की ती वगळायची (skip) याचा पर्याय.
* **कॅश व्यवस्थापन (Cache Management):** सिंक झाल्यानंतर लक्ष्यित साइट्सवर Elementor कॅश आपोआप साफ होतो.

## आवश्यकता (Requirements) {#requirements}

* WordPress 5.3 किंवा त्याहून अधिक
* PHP 7.4 किंवा त्याहून अधिक
* Multisite Ultimate plugin (सक्रिय)
* Elementor plugin (पर्यायी, Elementor टेम्पलेट सिंकसाठी आवश्यक)

## स्थापना (Installation) {#installation}

1. 'ultimate-multisite-content-sync' ही फाईल '/wp-content/plugins/' डिरेक्टरीमध्ये अपलोड करा.
2. तुमच्या WordPress Network Admin मधील 'Plugins' मेनूद्वारे प्लगइन सक्रिय (Activate) करा.
3. Ultimate Multisite → Settings → Content Sync मध्ये सेटिंग्ज कॉन्फिगर करा.
4. सिंक सुरू करण्यासाठी Ultimate Multisite → Content Sync वर जा.
