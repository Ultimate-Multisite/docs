# AI Site Builder

ultimate-multisite-ai-site-builder

## आवश्यक गोष्टी (Requirements) {#requirements}

- WordPress 5.3 किंवा त्याहून अधिक
- PHP 7.4 किंवा त्याहून अधिक
- Multisite Ultimate plugin (सक्रिय)

## इन्स्टॉलेशन (Installation) {#installation}

1. addon फाइल्स तुमच्या `/wp-content/plugins/` डिरेक्टरीमध्ये अपलोड करा
2. वर्डप्रेसमधील 'Plugins' मेनूद्वारे प्लगइन सक्रिय करा
3. नेटवर्क ॲडमिनमध्ये addon सेटिंग्ज कॉन्फिगर करा

## डेव्हलपमेंट (Development) {#development}

### सेटअप (Setup) {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### टेस्टिंग (Testing) {#testing}

या addon मध्ये एक सर्वसमावेशक टेस्टिंग फ्रेमवर्क समाविष्ट आहे:

- **युनिट टेस्ट्स (Unit Tests)**: वर्डप्रेस टेस्ट सूटसह PHPUnit-आधारित टेस्ट्स
- **बेस टेस्ट क्लास (Base Test Class)**: सामान्य टेस्ट परिस्थितींसाठी मदतनीस मेथड्स (helper methods) पुरवतो
- **टेस्ट कव्हरेज (Test Coverage)**: कोड कव्हरेज ट्रॅक करण्यासाठी कॉन्फिगर केले आहे
- **मल्टीसाइट टेस्टिंग (Multisite Testing)**: टेस्ट्स मल्टीसाइट वातावरणात चालतात

### कोड स्टँडर्ड्स (Code Standards) {#code-standards}

- **वर्डप्रेस कोडिंग स्टँडर्ड्स (WordPress Coding Standards)**: PHPCS द्वारे लागू केले जातात
- **PHP 7.4+ कंपॅटिबिलिटी (Compatibility)**: आधुनिक PHP फीचर्सला सपोर्ट
- **स्टॅटिक ॲनालिसिस (Static Analysis)**: टाइप चेक करण्यासाठी PHPStan इंटिग्रेशन
- **कोड मॉडर्नायझेशन (Code Modernization)**: PHP अपग्रेडसाठी Rector नियम

### बिल्ड सिस्टम (Build System) {#build-system}

- **ॲसेट प्रोसेसिंग (Asset Processing)**: CSS/JS फाइल्ससाठी मिनिफिकेशन (Minification)
- **ट्रान्सलेशन (Translation)**: POT फाइल जनरेशन
- **पॅकेजिंग (Packaging)**: वितरणासाठी आर्काइव्ह तयार करणे
- **डेव्हलपमेंट मोड (Development Mode)**: डीबगिंगसाठी अनमिनिफाइड ॲसेट्स

## कॉन्फिगरेशन (Configuration) {#configuration}

हे addon नेटवर्क ॲडमिन इंटरफेसद्वारे उपलब्ध असलेल्या विविध कॉन्फिगरेशन पर्याय (configuration options) सपोर्ट करते.

## हुक्स आणि फिल्टर्स (Hooks and Filters) {#hooks-and-filters}

### ॲक्शन्स (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - addon सुरू झाल्यानंतर (initialization) ट्रिगर होते
- `ultimate-multisite-ai-site-builder_loaded` - सर्व addon फाइल्स लोड झाल्यानंतर ट्रिगर होते

### फिल्टर्स (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - addon सेटिंग्जमध्ये बदल करण्यासाठी
- `ultimate-multisite-ai-site-builder_enabled` - addon सक्रिय/निष्क्रिय (enable/disable) स्थितीवर ओव्हरराईड करण्यासाठी
