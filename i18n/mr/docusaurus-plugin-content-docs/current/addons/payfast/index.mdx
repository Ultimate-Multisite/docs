# PayFast

Multisite Ultimate: Payfast Gateway

Ultimate Multisite मध्ये PayFast - दक्षिण आफ्रिकेतील एक जलद आणि सुरक्षित ऑनलाइन पेमेंट गेटवे - पेमेंट पर्याय म्हणून जोडतो.

## Installation

1. 'ultimate-multisite-payfast' हे फाइल '/wp-content/plugins/' डिरेक्टरीमध्ये अपलोड करा
2. WordPress मधील 'Plugins' मेनूद्वारे प्लगइन ॲक्टिव्हेट करा
3. प्लगइन सेट करण्यासाठी स्टेप-बाय-स्टेप विझार्ड फॉलो करा

## Usage Guide

PayFast हे दक्षिण आफ्रिकेतील व्यवसायांसाठी एक प्रमुख ऑनलाइन पेमेंट प्रोसेसर आहे. ऑनलाइन व्यापाऱ्यांना व्यवहार सुरळीत आणि सुरक्षित बनवून वाढण्यास मदत करणे हे त्याचे मुख्य उद्दिष्ट आहे. Ultimate Multisite वापरकर्त्यांच्या विनंतीनुसार, आम्ही एक ॲडऑन तयार करण्याचा निर्णय घेतला आहे, ज्यामुळे PayFast सोबत इंटिग्रेशन करणे शक्य होईल. या लेखात दाखवले आहे की तुम्ही तुमच्या Ultimate Multisite नेटवर्कमध्ये पेमेंट प्राप्त करण्यासाठी PayFast ला तुमचा पेमेंट गेटवे म्हणून कसे सेट करू शकता.

पहिली पायरी म्हणजे ॲडऑन इन्स्टॉल करणे.

ॲड-ऑन विंडो उघडेल. 'Install Now' वर क्लिक करा.

![Add-ons page with install option](/img/admin/addons-page.png)

इन्स्टॉलेशन पूर्ण झाल्यावर, तुम्हाला प्लगइन्स पेजवर रीडायरेक्ट केले जाईल. येथे, फक्त 'Network Activate' वर क्लिक करा आणि _Payfast_ ॲड-ऑन तुमच्या नेटवर्कवर ॲक्टिव्हेट होईल.

# PayFast सेट करणे

ॲड-ऑन इन्स्टॉल केल्यानंतर, तुम्ही आता Ultimate Multisite सेटिंग्जमध्ये जाऊन PayFast कॉन्फिगर (configure) करण्यास सुरुवात करू शकता.

**Ultimate Multisite settings** पेजवर, तुम्हाला "Payments" टॅब मिळेल. या टॅबखाली, तुम्हाला "**Payfast** " दिसेल, सोबत या पेमेंट गेटवेला सक्षम (enable) किंवा अक्षम (disable) करण्यासाठी एक टॉगल बटण असेल.

![Payment gateway settings](/img/config/settings-payment-gateways.png)पुढे जाऊन, त्याला सक्षम करण्यासाठी टॉगल बटणावर क्लिक करा. त्यानंतर ते बॉक्स विस्तारेल आणि तुम्हाला **Merchant ID, Merchant Key आणि PassPhrase** यांसारखी वेगवेगळी फील्ड्स दिसतील ज्यामध्ये तुम्हाला माहिती भरावी लागेल. ही माहिती तुम्हाला तुमच्या **Payfast** खात्यातून मिळू शकते.

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

पुढील पायरी म्हणजे तुम्हाला आवश्यक असलेली माहिती मिळवण्यासाठी तुमच्या PayFast खात्यात लॉग इन करणे.

लॉग इन केल्यावर, तुम्हाला तुमच्या **Payfast** खात्याच्या डॅशबोर्डवरून लगेच आवश्यक क्रेडेन्शियल्स (credentials) दिसतील.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->तुम्ही फक्त क्रेडेन्शियल कॉपी आणि पेस्ट करू शकता आणि तुमच्या Ultimate Multisite सेटिंग्जमध्ये तपशील भरू शकता. त्यानंतर बदल सेव्ह करण्यासाठी **"Save settings** " बटणावर क्लिक करा.
