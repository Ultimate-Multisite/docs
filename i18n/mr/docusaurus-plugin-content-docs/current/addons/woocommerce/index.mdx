# WooCommerce Integration

WooCommerce Integration हे आमच्या Manual Gateway प्रमाणेच काम करते. हे तुमच्या वापरकर्त्यांना खूप लवचिकता देते, ज्यामुळे त्यांना तुमच्या WooCommerce इन्स्टॉलेशनवर उपलब्ध असलेल्या कोणत्याही पेमेंट फॉर्मची निवड करता येते. आणि ते सोपे आहे: एकदा वापरकर्त्याचा बिलिंग कालावधी संपला की, वापरकर्त्याचे सबस्क्रिप्शन 'on hold' केले जाते. त्याच वेळी, नेटवर्कच्या मुख्य साइटवरील WooCommerce इन्स्टॉलेशनवर एक ऑर्डर तयार होते. त्यानंतर वापरकर्त्याला पेमेंट करण्यासाठी लिंकसह त्यांच्या ईमेलवर एक इनव्हॉइस (invoice) प्राप्त होते. पेमेंट मिळाल्यावर, सबस्क्रिप्शन नूतनीकरण (renew) होते.

## Features

- Ultimate Multisite आणि WooCommerce मध्ये सोपे एकत्रीकरण (Easy integration)
- अनेक पेमेंट पद्धती उपलब्ध
- WooCommerce Subscriptions सह पूर्णपणे स्वयंचलित प्रक्रिया

## Documentation

**WooCommerce Integration सेट करणे**

**Ultimate Multisite मध्ये तुमच्या पेमेंट पर्याय वाढवणे: WooCommerce Integration**

## Installation

1. 'ultimate-multisite-woocommerce' हे 'plugins' डिरेक्टरीमध्ये अपलोड करा.
2. WordPress मधील 'Plugins' मेनूद्वारे प्लगइन सक्रिय (Activate) करा.
3. प्लगइन सेट करण्यासाठी व्हिजार्डचे (Wizard) पायरी-पायरीने अनुसरण करा.

## Usage Guide

_**महत्त्वाची सूचना: हा लेख Ultimate Multisite आवृत्ती 2.x शी संबंधित आहे.**_

_**लक्ष द्या:** Ultimate Multisite: WooCommerce Integration साठी तुमच्या मुख्य साइटवर किमान WooCommerce सक्रिय असणे आवश्यक आहे._

आम्हाला माहिती आहे की काही देशांमध्ये _Stripe_ आणि _PayPal_ उपलब्ध नाहीत, ज्यामुळे Ultimate Multisite वापरकर्त्यांना आमचे प्लगइन प्रभावीपणे वापरण्यास मर्यादा येतात किंवा अडथळे येतात. म्हणून आम्ही _WooCommerce_ सोबत एकत्रीकरण करण्यासाठी एक **add-on** तयार केला आहे.

_WooCommerce_ हे एक अतिशय लोकप्रिय ई-कॉमर्स प्लगइन आहे. जगभरातील डेव्हलपर्सनी याला विविध पेमेंट गेटवे जोडण्यासाठी add-ons तयार केले आहेत. आम्ही याचा फायदा घेऊन Ultimate Multisite बिलिंग सिस्टीममध्ये तुम्ही वापरू शकता अशा पेमेंट गेटवेचा विस्तार केला आहे.

## Setting It Up

सर्वात आधी, तुम्हाला WooCommerce Integration add-on इन्स्टॉल करण्याची गरज आहे, जे **परवानाधारक वापरकर्त्यांसाठी विनामूल्य** आहे. ते इन्स्टॉल करण्यासाठी, तुमच्या सुपर ॲडमिन डॅशबोर्डवर Ultimate Multisite वर क्लिक करा आणि **Add-ons** निवडा.

![Add-ons page](/img/admin/addons-page.png)

येथे तुम्हाला सर्व Ultimate Multisite add-ons मिळतील. **Ultimate Multisite: WooCommerce Integration** add-on वर क्लिक करा.

![Add-ons page showing WooCommerce Integration](/img/admin/addons-page.png)

add-on तपशीलांसह एक विंडो उघडेल. फक्त **Install Now** वर क्लिक करा.

<!-- Screenshot unavailable: WooCommerce Integration addon install popup with Install Now button -->

इन्स्टॉलेशन पूर्ण झाल्यावर, तुम्हाला प्लगइन्स पेजवर रीडायरेक्ट केले जाईल. येथे, फक्त **Network Activate** वर क्लिक करा आणि _WooCommerce_ add-on तुमच्या नेटवर्कवर सक्रिय होईल.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for WooCommerce Integration -->

सक्रिय केल्यानंतर, जर तुमच्या वेबसाइटवर अजूनही WooCommerce प्लगइन इन्स्टॉल आणि सक्रिय नसेल, **तर तुम्हाला एक आठवण (reminder) मिळेल**.

<!-- Screenshot unavailable: Admin notice reminding to install and activate WooCommerce plugin -->

इन्स्टॉल केल्यावर, तुम्हाला Ultimate Multisite पेमेंट सेटिंग्जमध्ये एक अतिरिक्त पर्याय दिसेल. त्यानंतर तुम्ही तिथून _WooCommerce_ पर्याय **enable** करू शकता.

![Payment gateway settings](/img/config/settings-payment-gateways.png)

तुम्ही काही मूलभूत सेटिंग्ज कॉन्फिगर करू शकता, जसे की फ्रंट एंडवर दिसणारे डिस्प्ले नाव आणि तुमच्या वापरकर्त्यांना त्यांच्या चेकआउट प्रवाहामध्ये मार्गदर्शन करण्यासाठी सूचना. त्यानंतर तुम्ही तुमच्या **मुख्य साइटच्या _WooCommerce_ सेटिंग्ज** मध्ये जाऊन तुम्हाला वापरायचे असलेले पेमेंट गेटवे सक्षम (enable) आणि कॉन्फिगर करू शकता.

## How to manage the payment

WooCommerce Integration हे आमच्या **Manual Gateway** प्रमाणेच काम करते. तोटा म्हणजे प्रत्येक पेमेंट बिलिंग कालावधीच्या शेवटी मॅन्युअली करावे लागते. कार्यप्रवाह (flow) खालीलप्रमाणे काम करतो:

  * एकदा वापरकर्त्याचा बिलिंग सायकल संपला की, सदस्यत्व **'on hold'** केले जाते. त्याच वेळी, तुमच्या नेटवर्कच्या मुख्य साइटवरील _WooCommerce_ इन्स्टॉलेशनवर एक ऑर्डर तयार होते.

  * त्यानंतर वापरकर्त्याला पेमेंट करण्यासाठी लिंकसह त्यांच्या ईमेलवर एक इनव्हॉइस प्राप्त होते (ही लिंक त्यांच्या सबसाइट डॅशबोर्डवरील बिलिंग हिस्ट्रीमध्ये देखील दिसते). ही लिंक वापरकर्त्याला _**WooCommerce**_ **चेकआउट** वर रीडायरेक्ट करते, जिथे वापरकर्ते त्यांचे पेमेंट करण्यासाठी **उपलब्ध _WooCommerce_ गेटवे** पैकी एक निवडू शकतील.

  * पेमेंट मिळाल्यावर, सदस्यत्व **नूतनीकरण (renew)** होते, पेमेंट Ultimate Multisite वर लॉग केले जाते आणि ऑर्डर पूर्ण (complete) सेट केली जाते.

### Some Important Notes:

  * Ultimate Multisite ची डिफॉल्ट कार्यक्षमता या एकत्रीकरण फॉर्मसह अजूनही काम करते. उदाहरणार्थ, **Subscription Management** स्क्रीनवरून **रिफंड जारी करणे** (issuing refunds) हे निवडलेल्या पेमेंट फॉर्मचा वापर करून रिफंड जारी करण्यासाठी WooCommerce शी संवाद साधेल.

  * Ultimate Multisite: WooCommerce Integration ला काम करण्यासाठी WP-Cron वर अवलंबून राहावे लागते. बहुतेक वेळा हे खूप चांगले काम करते, परंतु जर साइटवर ट्रॅफिक कमी असेल, तर क्रॉनला (cron) शेड्यूल केलेल्या घटना तपासायला सांगण्यात चूक होते आणि ती घटना चुकते, ज्यामुळे अंमलबजावणीमध्ये विलंब होऊ शकतो.

## Woocommerce Subscriptions Plugin

प्रत्येक महिन्यात मॅन्युअली पेमेंट सुरू करणे अंतिम वापरकर्त्यांसाठी आदर्श नाही, कारण सबस्क्रिप्शन प्रत्येक बिलिंग सायकलमध्ये आपोआप पुनरावृत्त (automatically recur) होण्याची अपेक्षा असते. म्हणून आम्ही WooCommerce Integration add-on डिझाइन केला आहे जे **WooCommerce Subscriptions plugin ला समर्थन (support) करते**. यामुळे व्यवहार करणे तुमच्या अंतिम वापरकर्त्यांसाठी सोपे होईल.

तुम्हाला फक्त [Woocommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/) **इन्स्टॉल** करण्याची आणि ते तुमच्या **मुख्य साइटवर सक्रिय** करण्याची गरज आहे. त्यानंतर Ultimate Multisite पेमेंट सेटिंग्जवर परत जा आणि **integration enable** करा.

आणि एवढेच. तुमच्या अंतिम वापरकर्त्यांना त्यांच्या बिलिंग सायकलच्या शेवटी मॅन्युअली पेमेंट करण्याची गरज राहणार नाही.

तुमच्या मुख्य साइटवर _WooCommerce_ अंतर्गत ऑर्डर तयार होण्याव्यतिरिक्त, ते प्रत्येक खात्यासाठी एक सबस्क्रिप्शन देखील तयार करेल.
