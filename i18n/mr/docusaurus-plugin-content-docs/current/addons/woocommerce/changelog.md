# WooCommerce एकत्रीकरण बदलांचा इतिहास

Version 2.0.6 - रिलीझ झाले 2026-01-16
* सुधारणा: ॲडऑनमध्ये मुख्य सबस्क्रिप्शन (core subscriptions) समाविष्ट केले. आता WooCommerce Subscriptions extension ची गरज नाही.

Version 2.0.5 - रिलीझ झाले 2026-01-09
* सुधारणा: glotpress API कडून भाषांतरे (translations) लोड करणे.
* दुरुस्ती: काही पेज बिल्डर्समध्ये गंभीर त्रुटी (Fatal error) येत होती.
* दुरुस्ती: जेव्हा ग्राहक मुख्य साइटचा सदस्य असतो, तेव्हा अनंत रीडायरेक्ट (Infinite redirect) होत होता.

Version 2.0.4 - रिलीझ झाले 2025-11-14
* जोडले: अनेक भाषांसाठी भाषांतरे (Translations) जोडली.
* बदलले: नाव बदलून Ultimate Multisite: Woocommerce Integration केले.
* जोडले: Woocommerce 10.2.1 सोबत सुसंगतता (Compatibility).
* जोडले: Woocommerce Subscriptions 7.7.0 सोबत सुसंगतता.
* दुरुस्ती: PHP 8.4 सोबत सुसंगतता.
* दुरुस्ती: जेव्हा WC account page अस्तित्वात नसतो, तेव्हा रीडायरेक्ट होत होता.

Version 2.0.3 - रिलीझ झाले 2025-08-13
* बदलले: नवीन मार्केटप्लेससह स्वयंचलित अपडेट (automatic updates) सक्षम केले.

Version 2.0.2 - रिलीझ झाले 2025-07-05
* बदलले: नाव बदलून Multisite Ultimate: Woocommerce Integration केले.
* जोडले: Woocommerce 9.8.1 सोबत सुसंगतता.
* जोडले: Woocommerce Subscriptions 7.3.0 सोबत सुसंगतता.
* दुरुस्ती: ग्राहकाद्वारे सबस्क्रिप्शन रद्द करणे.
* दुरुस्ती: चेकआउट ब्लॉक (checkout block) वापरताना गंभीर त्रुटी (Fatal error) येत होती.
* सुधारणा: आता Woocommerce च्या उच्च कार्यक्षमतेच्या कस्टम ऑर्डर टेबलसह (custom order tables) सुसंगत.
* दुरुस्ती: WooCommerce चेकआउटवर रद्द करताना सदस्यत्व (membership) अपग्रेड होऊ शकत होते.

Version 2.0.1 - रिलीझ झाले 2023-08-09

* जोडले: Woocommerce 7.9.0 सोबत सुसंगतता.
* जोडले: Woocommerce Subscriptions 5.3.0 सोबत सुसंगतता.
* जोडले: सदस्यत्व अपडेटसाठी (membership updates) सपोर्ट.
* जोडले: Woocommerce मध्ये चाचणी (trials) आणि सेटअप फी (setup fees) बद्दल सूचना.
* जोडले: मेटा व्हॅल्यूसह (meta value) Ultimate Multisite Woocommerce उत्पादनांची ओळख.
* जोडले: सर्व Ultimate Multisite संबंधित Woocommerce उत्पादनांना मार्क करण्यासाठी एक तात्पुरती दुरुस्ती (one-time fix) समाविष्ट केली.
* जोडले: Woocommerce यादीतून Ultimate Multisite ने तयार केलेली उत्पादने काढली.
* सुधारणा: कार्टवर लागू करण्यासाठी एक गैर-पुनरावृत्ती (non-recurring) Woocommerce सूट (discount) तयार केली.
* सुधारणा: Woocommerce उत्पादनावर पुनरावृत्ती सूट (recurring discount) पुनर्संचयित केला.
* सुधारणा: Woocommerce उत्पादनावर पुनरावृत्ती सूट लेबल जोडले.
* सुधारणा: चेकआउटवर उत्पादनाचा प्रकार (product type) सुनिश्चित केला.
* दुरुस्ती: डाउनग्रेड प्रक्रियेदरम्यान सदस्यत्व स्थिती (membership status) कायम राखली.
* दुरुस्ती: रद्द करण्याच्या प्रक्रियेदरम्यान त्रुटी टाळण्यासाठी सबस्क्रिप्शन अस्तित्वात आहे की नाही हे तपासले.
* दुरुस्ती: Woocommerce सबस्क्रिप्शनमध्ये वापरण्यासाठी सुरुवात सबस्क्रिप्शन तारीख (start subscription date) जोडली.
* अंतर्गत: नवीन PHP 8.1 बिल्ड प्रक्रिया लागू केली.

Version 2.0.0 - संपूर्ण पुनर्लेखन (Complete rewrite).

* जोडले: गेटवे बदलताना किंवा सदस्यत्व रद्द करताना woo सबस्क्रिप्शन काढण्याची प्रक्रिया (Process cancellation method);
* जोडले: सदस्यत्व डाउनग्रेड आणि अपग्रेड करण्यासाठी हँडलर (Handler);
* सुधारणा: सबसाइट्सवरील ग्राहक अपडेट फॉर्मवर (customer update form) woocommerce dependencies लोड करणे, जेणेकरून खाते अपडेट करता येईल;
* सुधारणा: woocommerce कार्ट लोड करणे, जर अस्तित्वात नसेल तर;
* सुधारणा: चेकआउट प्रक्रिया करताना आपण मुख्य साइटच्या टेबलवर (main site tables) आहोत हे सुनिश्चित करणे;
* सुधारणा: अल्टिमो (Ultimo) रिन्यूअल ऑर्डर Woocommerce सबस्क्रिप्शन ऑर्डर व्हॅल्यूवर आधारित असेल, शेवटच्या पेमेंटवर नाही;
* दुरुस्ती: WU Membership बटण लिंकवर जाणे;
* दुरुस्ती: जेव्हा Woocommerce सबस्क्रिप्शन रिन्यूअल पेमेंट झाले असते, तेव्हा अल्टिमो ऑर्डरला 'पेड' सेट करणे;
* बिल्ड: बिल्डर म्हणून MPB जोडले;
