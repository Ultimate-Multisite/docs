# WooCommerce Integration बदलनोंद {#woocommerce-integration-changelog}

आवृत्ती 2.2.0 - 2026-07-01 रोजी प्रकाशित
* नवीन: Ultimate Multisite कर रकमा आता चेकआउटवेळी स्वतंत्र WooCommerce शुल्क ओळी म्हणून दिसतात, त्यामुळे पेमेंटपूर्वी करांची एकूण रक्कम अधिक स्पष्ट होते.
* नवीन: WooCommerce सदस्यतांच्या पुन्हा प्रयत्न कालावधीत अयशस्वी-नूतनीकरण निलंबन हवे असलेल्या साइट्ससाठी वैकल्पिक "अयशस्वी नूतनीकरणांवर सदस्यत्वे त्वरित निलंबित करा" सेटिंग आणि `wu_woo_suspend_on_payment_failure` फिल्टर जोडला.
* दुरुस्ती: अयशस्वी किंवा पुनर्प्राप्त नूतनीकरणांनंतर Ultimate Multisite सदस्यत्वांशी विसंगत राहू शकणाऱ्या WooCommerce सदस्यता स्थितींचे समायोजन केले.
* दुरुस्ती: Ultimate Multisite चलन यादीत WooCommerce स्टोअरचे चलन नसल्यास ते जोडले.
* दुरुस्ती: सदस्यांना WooCommerce चेकआउटकडे पुनर्निर्देशित करताना ग्राहकाचे बिलिंग तपशील जतन केले.
* सुधारित: Jetpack Autoloader 5 सह सुसंगतता जोडली.
* सुधारित: प्रकाशन पॅकेज निर्मिती स्वच्छ केली, जेणेकरून GitHub आणि मार्केटप्लेस झिप्समध्ये नेस्टेड स्टेजिंग निर्देशिका आणि विकास फाइल्स टाळल्या जातील.
