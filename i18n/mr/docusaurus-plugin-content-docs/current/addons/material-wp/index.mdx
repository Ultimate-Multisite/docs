# Material WP

WordPress चे admin interface खूप वर्षांपासून सारखेच आहे. पण तुम्हाला कधी विचार आला आहे का की Material Design व्हर्जनचा admin interface कसा दिसेल?

Material WP हे एक admin theme आहे, जे तुम्हाला तो लुक आणि फील देण्यासाठी तयार केले आहे. यामुळे तुम्ही तुमच्या clients ला अधिक customized Dashboard देऊ शकता.

## वैशिष्ट्ये

- Custom CSS सपोर्ट
- सोपे स्टायलिंग सेटिंग्ज
- मेनूची पुनर्रचना (Menu reordering)
- सोपे इम्पोर्ट/एक्सपोर्ट

## Documentation

**Material WP वापरून तुमचा Admin Panel कस्टमाइझ करा**

## इन्स्टॉलेशन

1. 'material-wp' हे '/wp-content/plugins/' डिरेक्टरीमध्ये अपलोड करा
2. WordPress मधील 'Plugins' मेनूद्वारे प्लगइन ॲक्टिव्हेट करा

## वापर मार्गदर्शिका

WordPress चे admin interface खूप वर्षांपासून सारखेच आहे. पण तुम्हाला कधी विचार आला आहे का की Material Design व्हर्जनचा admin interface कसा दिसेल?

Material WP हे एक admin theme आहे, जे तुम्हाला तो लुक आणि फील देण्यासाठी तयार केले आहे. यामुळे तुम्ही तुमच्या clients ला अधिक customized Dashboard देऊ शकता. Material WP हे Ultimate Multisite च्या premium plugins core चाही भाग आहे.

**इन्स्टॉल केल्यानंतर**, तुमच्या _Plugins_ पेजवर जा आणि तुमच्या Network वर ते ॲक्टिव्हेट करण्यासाठी क्लिक करा.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Material WP -->

_तुमच्या Plugins पेजवर नेटवर्क-व्यापी (network-wide) ॲक्टिव्हेट करा_

_**टीप:**_ _जर तुम्ही फक्त ते ॲक्टिव्हेट केले, तर ते तुमच्या clients कडे एका पर्यायासारखे दिसेल, ज्यावर फक्त त्या विशिष्ट वापरकर्त्यासाठी (user) थीम जोडण्यासाठी मार्क केले जाऊ शकते._

आता, तुमचा admin interface पूर्णपणे रीडिझाइन झाला आहे! तसेच, तुम्हाला डाव्या बाजूच्या कॉलममध्ये 'Material WP' नावाचे एक नवीन मेनू देखील मिळेल.
