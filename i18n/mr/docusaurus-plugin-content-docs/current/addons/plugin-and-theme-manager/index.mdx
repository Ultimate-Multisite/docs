# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Install केलेल्या plugins आणि themes ची मेटा माहिती (शीर्षक, लेखक, थंबनेल आणि वर्णन) संपादित करा किंवा लपवा. तुमच्या वापरकर्त्यांना फिल्टर करण्यासाठी आणि त्यांच्या वापरकर्ता पॅनेलमध्ये सुंदर कस्टम Plugins पेज जोडण्यासाठी plugins आणि themes साठी कॅटेगरी तयार करा.

या plugin चा वापर करण्यासाठी, तुम्ही तुमच्या नेटवर्क ॲडमिनवरील Plugins किंवा Themes पेजवर जाऊन "Edit Plugin/Theme info" वर क्लिक करू शकता.

## Installation {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' हे फाइल '/wp-content/plugins/' डिरेक्टरीमध्ये अपलोड करा
2. तुमच्या WordPress Network Admin मधील 'Plugins' मेनूद्वारे plugin ॲक्टिव्हेट करा

## Usage Guide {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager हे एक शक्तिशाली **premium** Ultimate Multisite add-on आहे, जे तुम्हाला तुमच्या नेटवर्कवरील plugin आणि theme पर्याय कसे दिसतील आणि वापरकर्ते त्यांच्याशी कसे संवाद साधतील यावर पूर्ण नियंत्रण देते.

हे add-on तीन उद्देशांसाठी उपयुक्त आहे:

  1. हे तुम्हाला plugins आणि themes या दोघांचीही मेटाडेटा पूर्णपणे बदलण्याची मुभा देते. यामध्ये नाव, वर्णन, लेखकाचे नाव आणि अगदी थंबनेल देखील समाविष्ट आहे!

  2. हे तुम्हाला plugins आणि themes साठी कॅटेगरी तयार करण्याची मुभा देते, ज्यामुळे तुमच्या वापरकर्त्यांना त्यांना सहजपणे सॉर्ट करता येते.

  3. हे तुम्हाला जुने, कंटाळवाणे plugins पेज काढून त्याऐवजी कॅटेगरीनुसार सॉर्ट करता येणारे नवीन themes-सारखे पेज देण्याचा पर्याय देते.

हे डाउनलोड आणि इन्स्टॉल केल्यानंतर, Ultimate Multisite Settings पेजवर एक नवीन टॅब - _Add-on Settings_ - दिसेल.
