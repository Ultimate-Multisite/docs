# Ultimate Multisite Documentation

Ultimate Multisite ドキュメントへようこそ。Ultimate Multisite は WordPress Multisite をフル Website as a Service (WaaS) プラットフォームに変換します。

## ドキュメントセクション

### [ユーザーガイド](/user-guide/getting-started/ultimate-multisite-101)

WaaS プラットフォームを設定、構成、管理するために必要なすべての情報です。インストール、決済ゲートウェイ、ドメインマッピング、クライアント管理などをカバーします。

### [開発者向けドキュメント](/developer/hooks)

Ultimate Multisite 上で開発する開発者向けの技術リファレンスです。フックリファレンス、REST API ドキュメント、アドオン開発ガイドを含みます。

### [アドオン](/addons)

有料アドオンプラグインのドキュメントで、WooCommerce 統合、分析、ドメイン販売などの追加機能で Ultimate Multisite を拡張します。
