# Mailchimp

Multisite Ultimate: Mailchimp Integration

Multisite Ultimate ネットワークを Mailchimp に統合します。特定のプランを Mailchimp リストにリンクし、サインアップ時に自動的に購読者をそれらのリストに追加できます。

## Installation

1. 「ultimate-multisite-mailchimp」を「/wp-content/plugins/」ディレクトリにアップロードします。
2. WordPress ネットワーク管理の「Plugins」メニューからプラグインを有効化します。

## Usage Guide

サブスクライバーのメールを Mailchimp リストと同期するのは、当社の Ultimate Multisite: MailChimp Integration Add-on を使えば非常に簡単です。このチュートリアルは、すべてを設定するのに役立ちます！

## Getting an API key

設定の詳細に入る前に、Mailchimp API キーが必要です。これにより、Ultimate Multisite は Mailchimp API と通信し、リストやグループを取得し、ユーザーがサインアップした際にメールアドレスをそれらのリストやグループに追加できます。

以下のチュートリアルに従って、独自の Mailchimp API キーを取得してください: [About API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-zimbKYFZo-B4D1E28C-1203-4E9A-AC7B-D8E9B5659B93.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-zimbKYFZo-B4D1E28C-1203-4E9A-AC7B-D8E9B5659B93.png)

_Mailchimp API キーを追加する場所_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## Integration Mode

Mailchimp Integration アドオンは、2つの「統合モード」を提供します。つまり、メールアドレスを同期するためにアドオンを使用する2つの異なる方法があります。

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-mc8dVzW5S-B7A9495E-05B8-4FBB-8954-80EBF4A9CB3B.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-mc8dVzW5S-B7A9495E-05B8-4FBB-8954-80EBF4A9CB3B.png)

_より柔軟性のある2つの統合モード！_

### Mode 1: Multiple Lists

デフォルトモードは **複数リスト** を使用します。このモードでは、各プランに対して複数の Mailchimp リストを選択できます。ユーザーを異なるリストに分割する場合に最適なオプションです。

例として、Mailchimp アカウントに3つの異なるリスト（**Newsletter**、**Plan A**、**Plan B**）があると仮定します。プランに関係なく、すべてのユーザーを Newsletter リストにも含めたいので、更新情報を見逃さないようにします。

これを実現するには、Multiple Lists モードを使用して、**Newsletter** と **Plan A** を Plan A Mailchimp 設定に追加し、**Newsletter** と **Plan B** を Plan B Mailchimp 設定に追加します。

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-kZS8-S_eh-4145C1B8-B903-4DFF-9D92-AF7E0A61445E.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/6017c85715d41b7c717cdcf9/images/6021265efb34b55df443e4ad/60212659ac2f834ec53865f3-kZS8-S%5Feh-4145C1B8-B903-4DFF-9D92-AF7E0A61445E.png)

_新規アカウントとキャンセル済みアカウントのデフォルトリストを選択する_

### Mode 2: Single List + Multiple Groups

Mailchimp では、リストを **グループ** と呼ばれる小さなユーザーグループに分割できます。これにより、同じキャンペーンを複数作成せずに、購読者の特定のサブセットにキャンペーンを送信できるため、後でそのキャンペーンの指標を分析しやすくなります。

例として、**My Network Subscribers** という単一リストを作成し、その中に **Plan A** と **Plan B** のグループを作ることができます。

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-tOsErbJy6-A468DB31-B539-493E-AF5F-F44A4F90A974.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-tOsErbJy6-A468DB31-B539-493E-AF5F-F44A4F90A974.png)

_「単一リスト + 複数グループ」統合モードを使用する場合は、デフォルトリストとグループを選択してください_

## Default Lists and Groups

統合モードを選択したら、デフォルトリスト（モードに応じてグループ）を選択します。特定のリストが選択されていないプランの場合、デフォルトリストが使用されます。

キャンセル済みアカウントのデフォルトリストも選択できます。ユーザーがサブスクリプションを終了し、ネットワークからアカウントを削除すると、そのメールアドレスはキャンセル済みアカウント用に設定されたリストに移動します。

## Enabling Double Subscription

同じ設定ページで **ダブルサブスクリプション** を有効にできます。このオプションを有効にすると、ユーザーは選択されたデフォルトリストと、そのユーザーのプランに選択されたリストの両方に追加されます。このオプションが無効の場合、ユーザーはプランに選択されたリストにのみ追加されます。特定のプランに設定がない場合は、デフォルトリストに追加されます。

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-OPUDRnO94-5D4C267A-14EB-43B4-B86F-0328D330B521.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-OPUDRnO94-5D4C267A-14EB-43B4-B86F-0328D330B521.png)

_サインアップ時にユーザーをデフォルトリストとプランリストの両方に登録できます_

## Selecting Lists and Groups for each of the Plans

**Plans** に移動し、編集したいプランを選択します。**Advanced Options** メタボックスに「MailChimp」という新しいタブが表示されます。

そのタブで、統合モードに応じて、複数のリストまたは単一リストと複数のグループを選択できます。

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-BMHjubpN5-C3183A30-687C-4837-8350-842CF959BB06.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-BMHjubpN5-C3183A30-687C-4837-8350-842CF959BB06.png)

_各プランで使用するリストとグループを選択してください_

そのプランに新規ユーザーがサインアップすると、メールアドレスが選択されたリストとグループに追加されます！
