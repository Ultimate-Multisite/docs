# Mailchimp {#mailchimp}

Multisite Ultimate: Mailchimp 連携

Multisite Ultimate ネットワークを Mailchimp と連携します。特定のプランを Mailchimp リストに関連付け、登録時に購読者をそれらのリストへ自動的に追加できます。

## インストール {#installation}

1. 'ultimate-multisite-mailchimp' を '/wp-content/plugins/' ディレクトリにアップロードします
2. WordPress ネットワーク管理の 'Plugins' メニューからプラグインを有効化します

## 利用ガイド {#usage-guide}

Ultimate Multisite: MailChimp 連携アドオンを使えば、購読者のメールアドレスを MailChimp リストと同期するのはとても簡単です。このチュートリアルでは、すべての設定を完了できるように案内します。

## API キーの取得 {#getting-an-api-key}

利用できる各種設定に入る前に、MailChimp API キーが必要です。これにより、Ultimate Multisite は MailChimp API と通信し、リストやグループを取得したり、ユーザー登録時にメールアドレスをそれらのリストやグループへ追加したりできます。

自分の MailChimp API キーを取得するには、このチュートリアルに従ってください: [API キーについて | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_MailChimp API キーを_ _**Ultimate Multisite 設定 - > アドオン設定 -> MailChimp**_ _に追加します_

## 連携モード {#integration-mode}

MailChimp 連携アドオンには 2 つの「連携モード」があります。つまり、アドオンを使ってメールアドレスを同期する方法が 2 種類あります。

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_柔軟性を高める 2 つの連携モードです！_

### モード 1: 複数リスト {#mode-1-multiple-lists}

既定のモードでは **複数リスト** を使用します。このモードでは、各プランに対して複数の MailChimp リストを選択できます。ユーザーを異なるリストに分けて管理している場合に最適です。

たとえば、MailChimp Account に **ニュースレター**、**プラン A**、**プラン B** という 3 つのリストがあるとします。プランに関係なく、すべてのユーザーをニュースレターのリストにも含めたい場合、更新情報を見逃さないようにできます。

これを実現するには、複数リストモードを使って、プラン A の MailChimp 設定に **ニュースレター** と **プラン A** の両方を追加し、プラン B の MailChimp 設定に **ニュースレター** と **プラン B** を追加します。

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_新しいアカウント用とキャンセル済みアカウント用の既定リストを選択します_

### モード 2: 単一リスト + 複数グループ {#mode-2-single-list--multiple-groups}

MailChimp では、リストを **グループ** と呼ばれる小さなユーザー区分に分けられます。これにより、同じキャンペーンの複数コピーを MailChimp 上に作成しなくても、購読者の特定の一部にキャンペーンを送信できるため、より細かく制御できます（同じキャンペーンを複数作ると、後でそのキャンペーンの指標を分析しにくくなります）。

たとえば、**マイネットワーク購読者** という単一のリストを用意し、その中に **プラン A** と **プラン B** のグループを作成できます。

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_「単一リスト + 複数グループ」連携モードを使用する場合は、既定のリストとグループを選択します_

## 既定のリストとグループ {#default-lists-and-groups}

特定の利用ケースに最も合う連携モードを選択したら、次に既定のリスト（モードによってはグループも）を選択します。特定のプランに個別のリストが選択されていない場合、既定のリストが使用されます。

キャンセル済みアカウント用の既定リストも選択できます。ユーザーがサブスクリプションを終了し、ネットワークからアカウントを削除すると、そのメールアドレスはキャンセル済みアカウント用に設定されたリストへ移動されます。

## 二重登録の有効化 {#enabling-double-subscription}

同じ設定ページで、**二重登録** を有効化できます。このオプションでは、選択された既定リストと、そのユーザーのプランに選択されたリストの両方にユーザーを追加します。このオプションが有効でない場合、ユーザーはそのユーザーのプランに選択されたリストにのみ追加されます。そのプランに何も設定されていない場合、ユーザーは選択された既定リストに追加されます。

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_登録時に、ユーザーを既定リストとプランのリストの両方へ登録できます_

## 各プランのリストとグループの選択 {#selecting-lists-and-groups-for-each-of-the-plans}

**プラン** に移動し、編集したいプランを選択します。**詳細オプション** メタボックスに、MailChimp という新しいタブが表示されることに注意してください。

そのタブでは、そのプランに対して、複数のリスト、または単一のリストと複数のグループ（連携モードによります）を選択できます。

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_各プランで使用したいリストとグループを選択します_

新しいユーザーがそのプランに登録すると、そのメールアドレスが選択したリストとグループに追加されます！
