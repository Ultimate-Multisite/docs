# Material WP

WordPress の管理画面は長い間ほとんど変わっていません。では、管理画面が Material Design 版になったらどのように見えるか、考えたことはありますか？ 

Material WP は、その見た目と操作感を実現するために開発された管理用テーマです。クライアントに、よりカスタマイズされた Dashboard を提供できます。

## 機能 {#features}

- カスタム CSS 対応
- 簡単なスタイル設定
- メニューの並べ替え
- 簡単なインポート／エクスポート

## ドキュメント {#documentation}

**Material WP で管理パネルをカスタマイズする**

## インストール {#installation}

1. 'material-wp' を '/wp-content/plugins/' ディレクトリにアップロードします
2. WordPress の「プラグイン」メニューからプラグインを有効化します

## 使用ガイド {#usage-guide}

WordPress の管理画面は長い間ほとんど変わっていません。では、管理画面が Material Design 版になったらどのように見えるか、考えたことはありますか？

Material WP は、その見た目と操作感を実現するために開発された管理用テーマです。クライアントに、よりカスタマイズされた Dashboard を提供できます。Material WP は Ultimate Multisite のプレミアムプラグインコアの一部でもあります。

**インストール**したら、_プラグイン_ ページに移動し、ネットワークで有効化します。

<!-- スクリーンショットは利用できません: WP プラグインページに Material WP のネットワークで有効化ボタンが表示されている -->

_プラグインページでネットワーク全体に有効化します_

_**注:**_ _単に有効化しただけの場合、クライアント側には、そのユーザーだけにテーマを追加するために選択できるオプションとして表示されます。_

これで、管理画面が完全に再設計されます。また、左側の列に Material WP という新しいメニューが追加されます
