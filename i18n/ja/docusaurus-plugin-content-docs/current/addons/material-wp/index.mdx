# Material WP

WordPress の管理画面は長い間同じままです。しかし、管理画面の Material Design バージョンがどのように見えるか、想像したことがありますか？  

Material WP は、あの見た目と感触を実現するために開発された管理テーマで、クライアントによりカスタマイズされたダッシュボードを提供するのに役立ちます。  

## Features

- Custom CSS サポート  
- 簡単なスタイリング設定  
- メニューの再配置  
- 簡単なインポート/エクスポート  

## Documentation

**Material WP で管理パネルをカスタマイズする**  

## Installation

1. 'material-wp' を '/wp-content/plugins/' ディレクトリにアップロードします。  
2. WordPress の 'Plugins' メニューからプラグインを有効化します。  

## Usage Guide

WordPress の管理画面は長い間同じままです。しかし、管理画面の Material Design バージョンがどのように見えるか、想像したことがありますか？  

Material WP は、あの見た目と感覚を実現するために開発された管理テーマで、クライアントによりカスタマイズされたダッシュボードを提供するのに役立ちます。Material WP は Ultimate Multisite のプレミアムプラグインコアにも含まれています。  

**インストール後**、_Plugins_ ページに移動し、ネットワークで有効化をクリックします。
