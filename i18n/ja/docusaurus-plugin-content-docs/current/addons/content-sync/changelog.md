# コンテンツ同期変更履歴

### バージョン 1.1.0 - 2025-12-31 発表
* 新機能: すべての投稿タイプ（投稿、ページ、カスタム投稿タイプ）の同期をサポート
* 新機能: コンテンツ選択ステップで投稿タイプフィルタリングを実装
* 新機能: ソースサイト選択ステップ - メインサイトだけでなく任意のサイトから同期可能
* 新機能: テンプレートサイト用に子サイトを事前選択
* 新機能: すべての投稿タイプで post_content からメディアを抽出
* 新機能: タクソノミー同期（カテゴリ、タグ、カスタムタクソノミー）
* 変更: 「Elementor Template Sync」から「コンテンツ同期」に名称変更
* 変更: 「テンプレート選択」ステップを「データ選択」に名称変更
* 改善: Elementor 専用コードは elementor_library 投稿タイプのみに実行されるように変更

### バージョン 1.0.0 - 2025-12-19 発表
* 初回リリース
* テンプレートとサイト選択のためのマルチステップウィザード
* リアルタイム進捗フィードバック付きバッチ処理
* 自動 ID 再割り当て付きフルメディア同期
* Ultimate Multisite との設定統合
* ネットワーク設定を介したサイト除外サポート
* 同期後に Elementor キャッシュをクリア
* テンプレートタクソノミーの保持
