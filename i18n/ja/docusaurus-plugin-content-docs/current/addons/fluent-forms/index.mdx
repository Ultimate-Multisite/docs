# Fluent Forms

このアドオンにより、Multisite Ultimate の製品、メンバーシップ、またはサイトで Fluent Forms に制限を追加できるようになります。Fluent Forms はカスタム投稿タイプの代わりにカスタムテーブルを使用するため、通常の制限オプションは Fluent Forms では機能しません。

## 機能

- Fluent Forms または Conversational Fluent Forms の制限を追加
- 顧客は制限に達したときにプランをアップグレードするよう促すフレンドリーなメッセージを受け取ります
