# アクション: wu_mt_providers_registered

デフォルトプロバイダーが登録された後に発火します。

このフックを使用してカスタムプロバイダーを登録します。

## パラメーター

| 名前 | タイプ | 説明 |
|------|------|-------------|
| $manager | `\Provider_Manager` | プロバイダーマネージャーのインスタンス。 |
