# ディスカバリーインタビュー

**ディスカバリーインタビュー**は、Theme Builderの設計プロセスにおける最初のステップです。このインタビューでは、お客様のサイト、ビジネス、デザインの好みに関する情報を収集し、パーソナライズされたテーマを作成します。

## 概要 {#overview}

ディスカバリーインタビューでは、一連の質問を通じて、お客様のブランド、ターゲットオーディエンス、デザインの目標を理解していただきます。バージョン1.16.0からは、デザインのインスピレーションを提供できる**写真アップロードのステップ**が追加されました。

## インタビューのステップ {#interview-steps}

### 1. ビジネス情報 {#1-business-information}

- サイト名とタグライン
- ビジネスの種類または業界
- 主なビジネス目標
- ターゲットオーディエンスの説明

### 2. デザインの好み {#2-design-preferences}

- 希望するカラースキーム
- デザインのスタイル（モダン、クラシック、ミニマル、遊び心があるなど）
- 参考となるウェブサイトやブランド
- 必須のデザイン要素

### 3. 写真アップロード（v1.16.0で追加） {#3-photo-upload-new-in-v1160}

デザインの方向性を決定するために、1枚以上の写真をアップロードしてください。

#### 対応フォーマット {#accepted-formats}

| フォーマット | 拡張子 | 詳細 |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | 写真におすすめ |
| **PNG** | `.png` | 透明度をサポート |
| **WebP** | `.webp` | 最新のフォーマットで、ファイルサイズが小さい |
| **GIF** | `.gif` | 静止画像のみ（アニメーション不可） |

#### ファイルの要件 {#file-requirements}

- **最大ファイルサイズ:** 10 MB
- **最小寸法:** 400 × 300 ピクセル
- **推奨寸法:** 1200 × 800 ピクセル以上
- **アスペクト比:** どの比率でも可

#### 写真がデザインの方向性に与える影響 {#how-photos-influence-design-direction}

アップロードされた写真は、Theme Builderが以下の点を理解するのに役立ちます。

- **カラーパレット** — 主要な色が抽出され、テーマに使用されます
- **ビジュアルスタイル** — モダン、素朴、ミニマル、鮮やかなど
- **ムードと雰囲気** — プロフェッショナル、クリエイティブ、フレンドリー、高級など
- **デザイン要素** — パターン、テクスチャ、構成スタイル
- **タイポグラフィのインスピレーション** — ビジュアルテーマを補完するフォントスタイル

### 4. コンテンツの構造 {#4-content-structure}

- 必要なメインページ（ホーム、会社概要、サービス、お問い合わせなど）
- ブログまたはニュースセクションの必要性
- Eコマース機能の必要性
- 特別な機能や連携の必要性

### 5. 確認と承認 {#5-review-and-confirmation}

- 提供されたすべての情報の要約
- デザイン生成を開始する前の最終確認

## 写真アップロードのベストプラクティス {#photo-upload-best-practices}

### アップロードすべきもの {#what-to-upload}

- **ブランドのインスピレーション** — ロゴ、カラーパレット、またはブランドガイドライン
- **競合サイト** — 参考にしたいサイトのスクリーンショット
- **ムードボード** — 希望する美学を表現する画像のコレクション
- **製品写真** — 展示したい製品がある場合
- **チームやオフィスの写真** — 個人的またはプロフェッショナルなブランディング用
- **ライフスタイル画像** — ターゲットオーディエンスを表現するもの

### 避けるべきもの {#what-to-avoid}

- **低品質な画像** — ぼやけている、またはピクセル化された写真
- **無関係な画像** — ブランドを代表していない写真
- **テキストが多すぎる画像** — テキストが大量に含まれるスクリーンショット
- **著作権のある素材** — 画像を使用する権利があることを確認してください
- **アニメーションGIF** — 静止画像のみが処理されます

### ベストな結果を得るためのヒント {#tips-for-best-results}

1. **2〜3枚の画像をアップロードする** — 複数の画像はより良いコンテキストを提供します
2. **高品質な写真を使用する** — 明確で、照明の行き届いた画像が最適です
3. **多様性を含める** — 色、スタイル、被写体の組み合わせを混ぜてみましょう
4. **コンテキストを提供する** — 各画像を選んだ理由を簡単に説明してください
5. **具体的にする** — 特定の要素が好きなら、それを指し示してください

## デザインの方向性出力 {#design-direction-output}

ディスカバリーインタビューの後、Theme Builderは以下のものを生成します。

- **カラーパレット** — プライマリ、セカンダリ、アクセントカラー
- **タイポグラフィ** — 見出しと本文用のフォントファミリー
- **レイアウトスタイル** — スペーシング、アライメント、コンポーネントの配置
- **ビジュアル要素** — アイコン、パターン、装飾要素
- **ムードボード** — デザインの方向性の視覚的な表現

## 次のステップ {#next-steps}

ディスカバリーインタビューを完了した後：

1. 生成されたデザインの方向性を確認する
2. フィードバックを提供するか、調整を依頼する
3. [Design Direction](./design-direction.md) ステップに進む
4. Theme Builderがカスタムテーマを作成します
