# ホスピタリティメニュー

**ホスピタリティメニュー**機能を使用すると、Theme Builderで構造化されたフード＆ビバレッジメニューページを生成し、WordPressサイトに直接埋め込むことができます。

## 概要 {#overview}

Theme Builderを使えば、レストラン、カフェ、バー、ケータリングサービスなど、ホスピタリティ関連のビジネス向けに、プロフェッショナルで構造化されたメニューページを作成できます。これらのメニューはサイトのデザインに完全に統合され、簡単に更新・管理が可能です。

## 対応ホスピタリティタイプ {#supported-hospitality-types}

- **レストラン** — フルサービスのダイニングメニュー
- **カフェ** — コーヒーや軽食のメニュー
- **バー＆ラウンジ** — 飲み物や前菜のメニュー
- **ベーカリー** — ペストリーやパンのメニュー
- **ケータリングサービス** — イベントメニューオプション
- **フードトラック** — 移動式のフードサービスメニュー
- **醸造所・ワイナリー** — 説明付きのビバレッジメニュー

## メニューの構造 {#menu-structure}

### メニューカテゴリー {#menu-categories}

メニューはカテゴリーに分類されます。

- **前菜 (Appetizers)** — スターターや小皿料理
- **メインディッシュ (Entrees)** — 主菜
- **サイドディッシュ (Sides)** — 付け合わせや野菜
- **デザート (Desserts)** — 甘いお菓子
- **ドリンク (Beverages)** — 飲み物（アルコールおよびノンアルコール）
- **スペシャル (Specials)** — 日替わりまたは季節限定の特別メニュー

### メニューアイテムの形式 {#menu-item-format}

各メニューアイテムには、以下の情報が含まれます。

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### アイテムフィールド {#item-fields}

| フィールド | タイプ | 説明 |
|-------|------|-------------|
| `name` | string | 料理または飲み物の名前 |
| `description` | string | アイテムの詳細な説明 |
| `price` | string | 価格（通貨形式で表示） |
| `dietary_info` | array | 食事制限情報（ヴィーガン、グルテンフリーなど） |
| `allergens` | array | 一般的なアレルゲン（ナッツ、甲殻類など） |
| `availability` | string | 利用可能期間（毎日、季節限定など） |

## ホスピタリティメニューの作成 {#creating-hospitality-menus}

### ステップ1：メニュー情報の提供 {#step-1-provide-menu-information}

Theme Builderにメニューについて伝えます。

```
アペタイザー、パスタ料理、メインディッシュ、デザートを含むイタリアンレストランのメニューを作成してください。価格と説明を含めてください。
```

### ステップ2：Theme Builderによるメニュー生成 {#step-2-theme-builder-generates-menu}

Theme Builderが実行すること：

1. 構造化されたメニューページを作成します。
2. サイトのテーマに合わせたデザインを適用します。
3. アイテムをカテゴリーに整理します。
4. 価格と説明を整形します。
5. 食事制限情報やアレルゲン情報を追加します。

### ステップ3：確認とカスタマイズ {#step-3-review-and-customize}

以下の操作が可能です。

1. メニューアイテムや価格の編集
2. カテゴリーの追加または削除
3. カテゴリー内のアイテムの並び替え
4. 説明や食事制限情報の更新
5. スタイリングやレイアウトの調整

## メニューの表示オプション {#menu-display-options}

### フルメニューページ {#full-menu-page}

完全なメニューを表示するための専用ページです。

- カテゴリーごとに整理されています
- 検索およびフィルタリングが可能です
- 印刷に適したレイアウトです
- モバイルに対応したデザインです

### メニューウィジェット {#menu-widget}

他のページにメニューセクションを埋め込むことができます。

- ホームページでのおすすめアイテム
- サイドバーでの日替わりスペシャル
- バーページでのドリンクメニュー
- フッターでのデザート特集

### メニューPDF {#menu-pdf}

ダウンロード可能なPDFメニューを生成します。

- プロフェッショナルなフォーマット
- 印刷可能な品質
- 画像や説明文が含まれます
- 共有やメールでの送信が簡単です

## 食事制限とアレルゲン情報 {#dietary-and-allergen-information}

### 食事制限属性 {#dietary-attributes}

アイテムに食事制限情報をマークします。

- **ヴィーガン (Vegan)** — 動物性製品不使用
- **ベジタリアン (Vegetarian)** — 肉類不使用
- **グルテンフリー (Gluten-Free)** — セリアック病の方に安全
- **乳製品不使用 (Dairy-Free)** — 牛乳製品不使用
- **ナッツフリー (Nut-Free)** — 木の実やピーナッツ不使用
- **低炭水化物 (Low-Carb)** — 炭水化物を抑えたもの
- **高タンパク (High-Protein)** — タンパク質が豊富なもの

### アレルゲン警告 {#allergen-warnings}

一般的なアレルゲンを含めます。

- **ナッツ (Nuts)** — 木の実とピーナッツ
- **甲殻類 (Shellfish)** — エビやカニなどの甲殻類、軟体動物
- **魚 (Fish)** — すべての魚種
- **乳製品 (Dairy)** — 牛乳および乳製品
- **卵 (Eggs)** — 鶏卵
- **大豆 (Soy)** — 大豆製品
- **グルテン (Gluten)** — 小麦やグルテンを含む穀物
- **ごま (Sesame)** — ごまの実と油

## メニュー管理 {#menu-management}

### 価格の更新 {#updating-prices}

メニュー価格を簡単に更新できます。

1. メニューページに移動します。
2. 「メニューを編集」をクリックします。
3. アイテムの価格を更新します。
4. 変更を保存します。
5. 変更はすぐにサイトに反映されます。

### 季節限定アイテムの追加 {#adding-seasonal-items}

季節ごとのメニューバリエーションを作成します。

1. 新しいメニューバージョンを作成します。
2. 季節限定アイテムを追加します。
3. アイテムを「季節限定」としてマークします。
4. 利用可能期間を設定します。
5. 期間中に自動的に表示されます。

### スペシャルメニューの管理 {#managing-specials}

日替わりまたは週替わりのスペシャルメニューを表示します。

1. 「スペシャル」カテゴリーを作成します。
2. 利用可能期間を設定したアイテムを追加します。
3. ホームページでスペシャルメニューを目立たせます。
4. 日次または週次で更新します。
5. 古いスペシャルメニューはアーカイブします。

## Theme Builderとの連携 {#integration-with-theme-builder}

ホスピタリティサイトでTheme Builderを使用する場合：

1. **自動メニュー検出** — フード/ビバレッジビジネスであるかを自動で識別します。
2. **メニューページ作成** — プロフェッショナルなメニューページを生成します。
3. **デザインの統一** — メニューがサイトのテーマにマッチします。
4. **モバイル最適化** — スマートフォンでも美しい表示になります。
5. **SEO最適化** — 検索エンジンに優しいメニューになります。

## ベストプラクティス {#best-practices}

### メニューデザイン {#menu-design}

- **明確な構成** — 論理的なカテゴリー構造を保つ
- **読みやすい説明** — 食欲をそそり、情報が豊富な説明文
- **一貫した価格設定** — 明確な通貨とフォーマットを使用する
- **プロの写真** — 高品質なフード画像を掲載する
- **余白** — ページを詰め込みすぎない

### コンテンツ {#content}

- **正確な説明** — アイテムを正確に描写する
- **スペシャルメニューの強調** — スペシャルメニューを目立たせる
- **アレルゲンの記載** — アレルゲンは必ずリストアップする
- **定期的な更新** — 価格やアイテムを最新の状態に保つ
- **食欲をそそる言葉遣い** — アイテムが美味しそうに聞こえるようにする

### アクセシビリティ {#accessibility}

- **読みやすいフォント** — 明確で読みやすい書体を使用する
- **十分なコントラスト** — テキストが読みやすいことを確認する
- **食事制限ラベル** — 食事制限オプションを明確にマークする
- **アレルゲン警告** — アレルゲンは目立つように表示する
- **モバイルフレンドリー** — すべてのデバイスでテストする
