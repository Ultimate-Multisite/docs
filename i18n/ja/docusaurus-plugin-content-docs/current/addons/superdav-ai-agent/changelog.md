# 変更履歴

## 1.16.0 — 2026年5月20日リリース

### 新機能

- **ロゴSVG生成機能**：テーマビルダーから、名前空間を考慮したサニタイズ処理を伴うカスタムロゴSVGを生成し、埋め込むことができるようになりました。
- **ディスカバリーインタビューでの写真アップロード**：テーマビルダーのディスカバリーインタビューに写真アップロードのステップが追加され、より豊かなデザインコンテキストを把握できるようになりました。
- **パレットコントラスト検証機能**：テーマに適用する前に、カラーペアがWCAG（ウェブコンテンツ・アクセシビリティ・ガイドライン）に準拠しているかを確認できます。
- **ホスピタリティメニュー**：テーマビルダーから、ホスピタリティビジネス向けの構造化されたフード＆ビバレッジメニューページを生成できるようになりました。
- **デスクトップおよびモバイルプレビューレンダリング**：デザインの方向性を選択する際に、デスクトップおよびモバイルデバイスでのデザインプレビューが可能になりました。
- **ナビゲーションラベルパラメータ**：メニュー作成機能が、ページタイトルとは別の`navigation_label`をサポートするようになりました。
- **Tier 1ツール利用可能性**：sd-ai-agent/site-scrapeが、テーマビルダーでデフォルト利用可能なTier 1ツールになりました。

### 修正点

- **AIクライアントキャッシュ**：クロスリクエストでの永続化のためにトランジェント（transients）を利用するようになり、長時間実行されるエージェントタスクでのデータ損失を防ぎます。
- **プラグイン行アクションリンク**：明確化のため、修正および名称変更を行いました。

## 1.10.0 — 2026年5月5日リリース

### 新機能

- **Tavilyインターネット検索**：Brave Searchと並行して、よりリッチなインターネット検索結果を得るためにTavilyを検索プロバイダーとして追加しました。
- **テーマ対応の組み込みスキル**：Block Themes、Classic Themes、Kadence Blocks、Kadence Themeのスキルガイドがプラグインに同梱されるようになりました。
- **サイトビルダーのコンタクトフォーム機能**：チャットインターフェースから任意のページにコンタクトフォームを直接追加できるようになりました。

### 改善点

- **WooCommerce統合のリファクタリング**：より高い信頼性と互換性を実現するため、ネイティブのWooCommerce APIを使用するようになりました。
- **プロバイダーリストの自動更新**：いずれかのプラグインが有効化または無効化された際に自動的に更新されます。

### 修正点

- **navigate-to機能**：一部の管理画面ページで無限リロードループが発生していた問題を修正しました。
- **list-posts機能**：カテゴリ名とタグ名をスラッグに正しく解決するようになりました。
- **WP-CLIコマンド**：以前のリファクタリングで失われていた名前空間エイリアスを復元しました。
- **イベント自動化**：自動化テーブルがまだ作成されていないサイトでも、適切に処理されるようになりました。
- **memory-save機能**：システムインストラクションビルダー内で正しい名前空間プレフィックスを使用するようになりました。
- **Scalarツール結果**：AIに返される前に、結果が正しくラップされるようになりました。
- **使用統計情報**：古いバージョンからアップグレードする際に、レガシーな機能キー形式を正しく処理するようになりました。
