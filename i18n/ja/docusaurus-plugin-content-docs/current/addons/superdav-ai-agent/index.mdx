# Superdav AI Agent

Superdav AI Agent は、Gratis AI Agent を拡張する高度な WordPress AI アシスタントです。テーマ対応スキル、強化されたインターネット検索機能、サイトビルダー機能、任意の自動化連携を追加します。バージョン 1.18.0 では、Google Calendar 読み取りツール、連絡先と参加者のマッピング、人による承認ゲート、リマインダーの重複排除、TextBee SMS 通知、管理型 Superdav サービスのプロビジョニング、信頼できる開発者ワークフロー向けに別配布される高度なコンパニオンパッケージが追加されています。

## 主な機能

- **テーマ対応スキル** — 有効な WordPress テーマとインストール済みプラグインに基づいて、案内と機能を自動的に適応させます
- **複数のインターネット検索プロバイダー** — より充実した検索結果と高度な調査機能のために、Brave Search と Tavily から選択できます
- **サイトビルダーの問い合わせフォーム** — インストール済みのフォームプラグインを使って、チャット画面から完全に機能する問い合わせフォームを直接作成できます
- **強化されたサイトビルダー** — プラグイン検出とエラー復旧が改善された、複数段階のサイト構築計画を利用できます
- **組み込みテーマスキル** — ブロックテーマ、クラシックテーマ、Kadence Blocks、Kadence Theme 向けの専門的な案内を提供します
- **改善された WooCommerce 連携** — 信頼性と互換性を高めるため、ネイティブの WooCommerce API を使用するように再構成されています
- **プロバイダーの自動検出** — プラグインやテーマが有効化または無効化されたときに、利用可能なものを自動的に検出して更新します
- **カスタムロゴ生成** — 名前空間に安全な自動サニタイズにより、カスタム SVG ロゴを生成して埋め込めます
- **パレットのコントラスト検証** — テーマに適用する前に、色の組み合わせが WCAG アクセシビリティに準拠しているか確認します
- **写真を使った発見プロセス** — Theme Builder の発見インタビュー中に写真をアップロードし、デザインの方向性に反映できます
- **ホスピタリティメニュー** — レストラン、カフェ、ホスピタリティ事業向けに、構造化された料理と飲み物のメニューページを生成します
- **デスクトップとモバイルのプレビュー** — デザイン方向を選ぶ際に、デスクトップ端末とモバイル端末でデザインをプレビューできます
- **Tier 1 サイトスクレイピング** — 競合サイトや参考サイトを分析するために、sd-ai-agent/site-scrape がデフォルトで利用できるようになりました
- **Google Calendar 読み取りツール** — 設定済みのカレンダーと予定を読み取り、自動化が今後のスケジュール文脈を判断できるようにします
- **連絡先と参加者のマッピング** — フォローアップ作業を生成する前に、カレンダー参加者を既知のサイトユーザーや連絡先と照合します
- **人による承認ゲート** — メッセージ送信、作業再開、変更実施の前に、慎重な扱いが必要な自動化を一時停止して確認できます
- **リマインダー記録と重複排除** — 送信済みのリマインダーを追跡し、自動化が繰り返し実行されても同じ受信者へ二重に通知しないようにします
- **TextBee SMS 通知** — SMS プロバイダーが有効な場合、TextBee を通じて設定済みのテキストメッセージ通知を送信します
- **管理型 Superdav サービスのプロビジョニング** — 手動でエンドポイントを設定する代わりに、自動セットアップで対応サイトをホスト型 Superdav エンドポイントに接続します
- **高度なコンパニオンパッケージ** — ファイルシステム、データベース、WP-CLI、REST ディスパッチャー、プラグインビルダー、git スナップショット、ユーザー管理、ベンチマークの各ワークフロー向けに別配布される、信頼できる開発者用ツールです

## 要件

- WordPress 6.0 以上
- PHP 8.1 以上
- WordPress Multisite（ネットワーク有効化）または単一サイトインストール
- 有効な AI プロバイダー接続プラグイン（Anthropic、OpenAI、または互換）
- Gratis AI Agent v1.9.0 以上（Superdav AI Agent は Gratis AI Agent を拡張します）

## インストール

1. プラグインファイルを `/wp-content/plugins/superdav-ai-agent` にアップロードします
2. WordPress の **プラグイン** 画面からプラグインを有効化します
3. Gratis AI Agent も有効化されていることを確認します
4. **Gratis AI Agent → 設定** に移動し、必要に応じてインターネット検索プロバイダー、管理型 Superdav サービス接続、Google Calendar、TextBee 認証情報を設定します

WordPress.org パッケージには、Superdav AI Agent の中核機能が含まれています。高度なコンパニオンパッケージは、信頼できる開発者ツールが明示的にインストールされ承認されている環境向けに、別途ビルドおよび配布されます。

## テーマ対応スキル

Superdav AI Agent は、有効なテーマとインストール済みプラグインを自動的に検出し、適切なテーマ対応スキルをアシスタントの文脈に読み込みます。利用可能なスキルは次のとおりです。

| スキル | 適用対象 |
|---|---|
| **ブロックテーマ** | WordPress のブロックベース（Full Site Editing）アーキテクチャを使用するテーマ |
| **クラシックテーマ** | PHP テンプレートとクラシックエディターを使用する従来の WordPress テーマ |
| **Kadence Blocks** | Kadence Blocks プラグインが有効なサイト |
| **Kadence Theme** | Kadence テーマが有効なテーマであるサイト |

各スキルの詳しい案内については、[テーマ対応スキル](../../user-guide/configuration/theme-aware-skills) を参照してください。

## インターネット検索プロバイダー

Superdav AI Agent v1.10.0 では、インターネット検索クエリ向けに Brave Search の代替として **Tavily** が追加されました。どちらのプロバイダーもリアルタイム検索結果を提供し、**Gratis AI Agent → 設定 → 高度** で設定できます。

| プロバイダー | 機能 | 設定 |
|---|---|---|
| **Brave Search** | 高速でプライバシー重視の検索結果 | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | より豊富な検索結果、高度な調査機能 | [Tavily API](https://tavily.com/) |

設定手順については、[インターネット検索](../../user-guide/configuration/internet-search)を参照してください。

## カレンダー、承認、通知ワークフロー

Superdav AI Agent v1.18.0 では、自動化ワークフローがコンテンツやサイト構築タスクの範囲を超えて拡張されます。

- **Google Calendar ツール**により、カレンダー認証情報が利用可能な場合、エージェントは設定済みカレンダーとイベント詳細を読み取れます。
- **連絡先と参加者のヘルパー**は、リマインダーやフォローアップタスクを準備する前に、イベント参加者を WordPress ユーザーまたは既知の連絡先に対応付けます。
- **人による承認ゲート**により、承認済みユーザーが提案された操作を確認するまで、自動化を一時停止できます。
- **リマインダー記録**により、自動化が再試行または繰り返しスケジュールされた場合でも、カレンダーや通知のリマインダーの重複を防ぎます。
- **TextBee SMS 対応**により、TextBee プロバイダーが設定されている場合にテキストメッセージ通知を利用できます。

これらのツールは、スケジュールを考慮した支援、参加者へのフォローアップ、機密性の高いメッセージ送信前に人による確認が必要なリマインダーワークフローに使用してください。

## Site Builder の問い合わせフォーム

Site Builder エージェントは、チャットインターフェースから直接問い合わせフォームを作成できるようになりました。Site Builder に問い合わせフォームの作成を依頼するだけで、インストール済みのフォーム plugin（Contact Form 7、WPForms、Fluent Forms、または Gravity Forms）を自動検出し、すぐに使えるフォームを生成します。

詳しい手順については、[Site Builder の問い合わせフォーム](../../user-guide/administration/site-builder-contact-forms)を参照してください。

## Theme Builder の機能

Superdav AI Agent v1.16.0 の Theme Builder には、強力な新機能が含まれています。

### 写真アップロード付きの発見インタビュー

Theme Builder の発見インタビューに、視覚的なインスピレーションを提供できる写真アップロード手順が追加されました。ブランド、競合サイト、デザインの参考写真をアップロードすると、Theme Builder がデザインの方向性を理解しやすくなります。

詳しい手順については、[発見インタビュー](./theme-builder/discovery-interview.md)を参照してください。

### デスクトップとモバイルのプレビューによるデザイン方向性

Theme Builder が完全な theme を生成する前に、デスクトップとモバイルの両方のデバイスでデザインを確認できます。複数のレスポンシブなブレークポイントでリアルタイムプレビューを確認できます。

詳しい手順については、[デザイン方向性](./theme-builder/design-direction.md)を参照してください。

### ホスピタリティメニュー

Theme Builder は、レストラン、カフェ、バー、その他のホスピタリティ事業向けに、構造化された食品・飲料メニューページを生成できるようになりました。メニューには価格、説明、食事制限情報、アレルゲン警告が含まれます。

詳しい手順については、[ホスピタリティメニュー](./theme-builder/hospitality-menus.md)を参照してください。

### 新しい機能

- **[Logo SVG を生成](./abilities/generate-logo-svg.md)** — 自動サニタイズ付きのカスタム SVG ロゴを作成
- **[Palette Contrast を検証](./abilities/validate-palette-contrast.md)** — 色の組み合わせが WCAG に準拠しているか確認
- **[Menu を作成](./abilities/create-menu.md)** — 個別のナビゲーションラベルを持つナビゲーションメニューを作成

利用可能なツールと Tier 1 サイトスクレイプツールについては、[ツールと設定](./theme-builder/tools.md)を参照してください。

## 設定

すべての設定は WordPress 管理画面の **Gratis AI Agent → 設定** にあります。Superdav AI Agent は次の設定オプションを追加します。

| 設定 | 説明 |
|---|---|
| **Internet Search Provider** | Brave Search と Tavily のどちらかを選択します（任意） |
| **Tavily API Key** | Tavily インターネット検索用の API key（Tavily を選択した場合） |
| **Managed Superdav Service** | 対応サイトをホスト型 Superdav サービスに接続し、endpoint を自動プロビジョニングします |
| **Google Calendar Credentials** | エージェントがカレンダーとイベントを読み取れるようにカレンダーアクセスを設定します |
| **TextBee SMS Provider** | 承認済み SMS 通知ワークフロー用の TextBee 認証情報を設定します |

## Advanced Companion Package

Superdav AI Agent v1.18.0 では、信頼できる開発者ワークフロー向けに、別途配布される Advanced companion package が追加されます。これらの機能は WordPress.org のコアパッケージには同梱されません。管理者がエージェントによる開発者ツールへのアクセスを明示的に信頼しているサイトにのみ、companion package をインストールしてください。

Advanced ツールには、ファイルシステム操作、データベースアクセス、WP-CLI、REST ディスパッチャーワークフロー、plugin ビルダーユーティリティ、git スナップショット、ユーザー管理ヘルパー、ベンチマークツールが含まれます。これらのツールは信頼できる管理者に限定し、本番サイトで有効化する前に承認ゲート設定を確認してください。

## v1.18.0 の改善点

- **Google Calendar 対応** — スケジュールを考慮した自動化のために、設定済みカレンダーとイベントを読み取ります
- **連絡先と参加者の対応付け** — カレンダー参加者をサイトユーザーや連絡先と照合します
- **人による承認ゲート** — 続行前に確認できるよう、機密性の高い自動化を一時停止します
- **リマインダーの重複排除** — 再試行や繰り返し実行時の重複通知を避けるため、リマインダーを記録します
- **TextBee SMS プロバイダー** — 必要な承認後に、設定済みのテキストメッセージ通知を送信します
- **Managed Superdav service の設定** — 対応サイト向けにホスト型サービス endpoint を自動プロビジョニングします
- **Advanced companion の配布** — 信頼できる開発者向けツールを WordPress.org コアビルドとは別に公開します
- **信頼性の修正** — モデル選択、リクエストタイムアウト、既定の設定モデル処理、推論テキスト処理、再試行ガイダンス、カレンダートークンの強化、リマインダーの強化、オンボーディング起動、確認済み機能の再開、WordPress.org パッケージングレビューの問題を改善します

## v1.16.0 の改善点

- **AI クライアントキャッシュ** — リクエストをまたいだ永続化のために transients で保持されるようになり、長時間実行される agent タスクでのデータ損失を防ぎます
- **Tier 1 サイトスクレイプツール** — 競合サイトや参考サイトの分析用に、sd-ai-agent/site-scrape が Theme Builder で標準利用できるようになりました
- **Plugin 行のアクションリンク** — わかりやすくなるよう修正し、名称を変更しました

## v1.10.0 の改善

- **WooCommerce 連携のリファクタリング** — 信頼性と互換性を高めるため、ネイティブの WooCommerce API を使用するようになりました
- **プロバイダー一覧の自動更新** — plugin の有効化または無効化を自動検出し、利用可能なプロバイダー一覧を更新します
- **強化されたサイトビルダー** — エラー復旧を改善し、複数フェーズのサイト構築計画を向上しました
- **テーマ対応スキル** — 有効なテーマに自動適応する新しい組み込みスキルを4つ追加しました

- 一部の管理ページで `navigate-to` ability が無限リロードループを引き起こす問題を修正しました

- 以前のリファクタリング後に WP-CLI コマンドで名前空間エイリアスが欠落していた問題を修正しました
- automation テーブルがまだ作成されていないサイトでも、イベント自動化が適切に処理されるよう修正しました
- システム指示ビルダーで `memory-save` ability が正しい名前空間プレフィックスを使用するよう修正しました
- スカラーのツール結果が AI に返される前に正しくラップされるよう修正しました
- 古いバージョンからのアップグレード時に、使用統計が従来の ability キー形式を正しく処理するよう修正しました
