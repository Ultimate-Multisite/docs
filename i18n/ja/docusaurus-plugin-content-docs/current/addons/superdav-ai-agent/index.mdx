# Superdav AI Agent

Superdav AI Agentは、Gratis AI Agentを拡張し、テーマに合わせた機能、強化されたインターネット検索機能、改善されたサイトビルダー機能を追加した、高度なWordPress AIアシスタントです。バージョン1.16.0では、カスタムロゴの生成、パレットのコントラスト検証、ディスカバリーインタビューでの写真アップロード、構造化されたホスピタリティメニューの生成、およびTheme Builderでのデスクトップ/モバイルプレビューレンダリングが追加されました。

## 主な機能

- **テーマ対応の機能** — アクティブなWordPressテーマやインストールされているプラグインに基づいて、ガイダンスと機能を自動的に調整します。
- **複数のインターネット検索プロバイダー** — よりリッチな検索結果と高度な調査機能のために、Brave SearchとTavilyから選択できます。
- **サイトビルダーのコンタクトフォーム** — インストールされているフォームプラグインを使用して、チャットインターフェースから完全に機能するコンタクトフォームを作成できます。
- **強化されたサイトビルダー** — より良いプラグインの発見とエラー回復機能を備えた、改善されたマルチフェーズのサイト構築プランを提供します。
- **組み込みのテーマ機能** — Block Themes、Classic Themes、Kadence Blocks、Kadence Themeに特化したガイダンスを提供します。
- **WooCommerce統合の改善** — より高い信頼性と互換性のために、ネイティブのWooCommerce APIを使用するように改修されました。
- **自動プロバイダー検出** — プラグインやテーマが有効化または無効化される際に、利用可能なプラグインとテーマを自動的に検出して更新します。
- **カスタムロゴ生成** — 自動的に名前空間を安全にサニタイズしながら、カスタムSVGロゴを生成し、埋め込むことができます。
- **パレットコントラスト検証** — テーマに適用する前に、色の組み合わせがWCAGのアクセシビリティ基準に準拠しているかを確認します。
- **写真によるディスカバリー強化** — Theme Builderのディスカバリーインタビュー中に写真をアップロードし、デザインの方向性に影響を与えることができます。
- **ホスピタリティメニュー** — レストラン、カフェ、ホスピタリティビジネス向けの構造化されたフード＆ビバレッジメニューページを生成します。
- **デスクトップおよびモバイルプレビュー** — デザインの方向性を選択する際に、デスクトップおよびモバイルデバイスでデザインをプレビューできます。
- **Tier 1サイトスクレイピング** — sd-ai-agent/site-scrapeがデフォルトで利用可能になり、競合他社やインスピレーションとなるウェブサイトの分析に使用できます。

## 要件

- WordPress 6.0以上
- PHP 8.1以上
- WordPress Multisite（ネットワーク有効化）またはシングルサイトのインストール
- アクティブなAIプロバイダーコネクタプラグイン（Anthropic、OpenAI、または互換性のあるもの）
- Gratis AI Agent v1.9.0以上（Superdav AI AgentはGratis AI Agentを拡張します）

## インストール方法

1. プラグインファイルを`/wp-content/plugins/superdav-ai-agent`にアップロードします。
2. WordPressの**プラグイン**画面からプラグインを有効化します。
3. Gratis AI Agentも有効化されていることを確認してください。
4. **Gratis AI Agent → 設定**に移動し、インターネット検索プロバイダーを設定します（オプション）。

## テーマ対応の機能

Superdav AI Agentは、アクティブなテーマとインストールされているプラグインを自動的に検出して、適切なテーマ対応の機能をアシスタントのコンテキストに読み込みます。利用可能な機能には以下が含まれます。

| 機能 | 適用対象 |
|---|---|
| **Block Themes** | WordPressのブロックベース（フルサイト編集）アーキテクチャを使用するテーマ |
| **Classic Themes** | PHPテンプレートとクラシックエディターを使用する従来のWordPressテーマ |
| **Kadence Blocks** | Kadence Blocksプラグインがアクティブなサイト |
| **Kadence Theme** | Kadenceテーマがアクティブテーマのサイト |

各機能の詳細なガイダンスについては、[Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills)を参照してください。

## インターネット検索プロバイダー

Superdav AI Agent v1.10.0では、インターネット検索クエリの代替としてTavilyがBrave Searchに追加されました。どちらのプロバイダーもリアルタイムの検索結果を提供し、**Gratis AI Agent → 設定 → 詳細**で設定できます。

| プロバイダー | 機能 | セットアップ |
|---|---|---|
| **Brave Search** | 高速でプライバシーに配慮した検索結果 | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | よりリッチな検索結果、高度な調査機能 | [Tavily API](https://tavily.com/) |

設定手順については、[Internet Search](../../user-guide/configuration/internet-search)を参照してください。

## サイトビルダーのコンタクトフォーム

サイトビルダーエージェントは、チャットインターフェースから直接コンタクトフォームを作成できるようになりました。サイトビルダーにコンタクトフォームの作成を依頼するだけで、インストールされているフォームプラグイン（Contact Form 7、WPForms、Fluent Forms、またはGravity Forms）を自動的に検出して、すぐに使えるフォームを生成します。

詳細なガイダンスについては、[Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms)を参照してください。

## Theme Builderの機能

Superdav AI Agent v1.16.0のTheme Builderには、強力な新機能が搭載されています。

### 写真アップロードによるディスカバリーインタビュー

Theme Builderのディスカバリーインタビューには、視覚的なインスピレーションを提供できる写真アップロードステップが追加されました。ブランド、競合サイト、またはデザインのインスピレーションとなる写真をアップロードすることで、Theme Builderがデザインの方向性を理解するのに役立ちます。

詳細なガイダンスについては、[Discovery Interview](./theme-builder/discovery-interview.md)を参照してください。

### デスクトップおよびモバイルプレビューによるデザインの方向性決定

Theme Builderが完全なテーマを生成する前に、デスクトップとモバイルの両方のデバイスでデザインを確認できます。複数のレスポンシブブレークポイントでリアルタイムプレビューを確認できます。

詳細なガイダンスについては、[Design Direction](./theme-builder/design-direction.md)を参照してください。

### ホスピタリティメニュー

Theme Builderは、レストラン、カフェ、バー、その他のホスピタリティビジネス向けの構造化されたフード＆ビバレッジメニューページを生成できるようになりました。メニューには、価格、説明、食事制限情報、アレルゲン警告が含まれます。

詳細なガイダンスについては、[Hospitality Menus](./theme-builder/hospitality-menus.md)を参照してください。

### 新しい機能

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — 自動サニタイズ機能付きでカスタムSVGロゴを作成します。
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — 色の組み合わせがWCAGに準拠しているかを確認します。
- **[Create Menu](./abilities/create-menu.md)** — 明確なナビゲーションラベルを持つナビゲーションメニューを作成します。

利用可能なツールとTier 1サイトスクレイピングツールに関する情報は、[Tools and Configuration](./theme-builder/tools.md)を参照してください。

## 設定

すべての設定は、WordPress管理画面の**Gratis AI Agent → 設定**にあります。Superdav AI Agentは、以下の設定オプションを追加します。

| 設定 | 説明 |
|---|---|
| **Internet Search Provider** | Brave SearchとTavilyから選択します（オプション） |
| **Tavily API Key** | Tavilyのインターネット検索用のAPIキー（Tavilyを選択した場合） |

## v1.16.0の改善点

- **AIクライアントキャッシュ** — 長時間実行されるエージェントタスクにおけるデータ損失を防ぐため、トランジェントによってバックアップされるようになりました。
- **Tier 1サイトスクレイピングツール** — 競合他社やインスピレーションとなるウェブサイトの分析のために、Theme Builderでsd-ai-agent/site-scrapeがデフォルトで利用可能になりました。
- **プラグイン行のアクションリンク** — 明確化のために修正および名前が変更されました。

## v1.10.0の改善点

- **WooCommerce統合の改修** — より高い信頼性と互換性のために、ネイティブのWooCommerce APIを使用するようになりました。
- **プロバイダーリストの自動更新** — プラグインが有効化または無効化された際に自動的に検出され、利用可能なプロバイダーリストが更新されます。
- **サイトビルダーの強化** — より良いエラー回復機能を備えた、改善されたマルチフェーズのサイト構築プランを提供します。
- **テーマ対応の機能** — アクティブなテーマに自動的に適応する4つの新しい組み込み機能が追加されました。

## v1.10.0のバグ修正

- 一部の管理ページで無限のリロードループを引き起こしていた`navigate-to`機能の修正。
- カテゴリ名とタグ名をスラッグに正しく解決できていなかった`list-posts`機能の修正。
- 前回のリファクタリング後、WP-CLIコマンドで名前空間エイリアスが欠落していた問題の修正。
- 自動化テーブルがまだ作成されていないサイトで、イベント自動化が正常に処理されるように修正。
- システムインストラクションビルダーで、`memory-save`機能が正しい名前空間プレフィックスを使用するように修正。
- スカラーツールの結果がAIに返される前に正しくラップされるように修正。
- 古いバージョンのエビリティキー形式からアップグレードする際に、使用統計が正しく処理されるように修正。
