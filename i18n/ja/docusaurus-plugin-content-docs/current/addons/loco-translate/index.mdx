# Loco Translate

Ultimate Multisite: Loco Translate Integration addon brings site‑specific translation capabilities to your WordPress multisite network. If you’ve ever needed different sites in your network to have different translations of the same plugin or theme, this addon is the perfect solution.

**このアドオンの機能**

このアドオンは、人気のある Loco Translate プラグインと Ultimate Multisite をシームレスに統合し、ネットワーク内の各サブサイトが独自のカスタム翻訳を維持できるようにします。マルチリージョンのプラットフォームを運営している場合、異なる言語コミュニティにサービスを提供している場合、または単にネットワーク全体で翻訳の柔軟性が必要な場合、このアドオンはそれを容易にします。

**対象**

* 異なるローカリゼーションニーズを持つマルチリージョン企業
* 多様なコミュニティにサービスを提供する教育機関
* 多言語サポートを提供する SaaS プラットフォーム
* 複数のクライアントサイトを管理するウェブエージェンシー
* 翻訳の柔軟性を必要とする任意のマルチサイトネットワーク

## インストール

1. Ultimate Multisite がインストールされ、ネットワークで有効化されていることを確認してください
2. Loco Translate プラグインをインストールして有効化してください
3. このアドオンをプラグインディレクトリにアップロードしてください
4. Loco Translate Integration アドオンをネットワークで有効化してください
5. 任意のサブサイトで Loco Translate を使用してサイト固有の翻訳を作成し始めてください

これで完了です！追加設定なしで統合が自動的に機能します。

## よくある質問

## インストール後に設定が必要ですか？

いいえ！統合は自動です。アドオンと Loco Translate が両方有効になっていると、サブサイトはすぐに独自のカスタム翻訳を作成できます。

## サブサイトがカスタム翻訳を作成しない場合はどうなりますか？

サイトは自動的にネットワーク全体の翻訳をフォールバックとして使用します。何も壊れず、すべてがシームレスに機能します。

## サブサイトはネットワーク全体の翻訳にアクセスできますか？

通常のサイト管理者は自分のサイトの翻訳のみを閲覧・編集できます。スーパー管理者は閲覧しているサイトに応じて、ネットワーク全体の翻訳とサイト固有の翻訳の両方にアクセスできます。

## これはプラグインとテーマの両方で機能しますか？

はい！サイト固有の翻訳は WordPress のプラグインとテーマの両方で機能します。

## これでマルチサイトネットワークが遅くなりますか？

いいえ。アドオンは効率的なファイル読み込みを使用し、必要に応じてのみサイト固有の翻訳をチェックします。目立つパフォーマンスへの影響はありません。

## ネットワーク翻訳を特定のサイトにコピーして開始点にできますか？

はい！Loco Translate で翻訳を編集する際に、ネットワーク翻訳を開始点として簡単に使用し、各サイトに合わせてカスタマイズできます。

## WooCommerce やその他のプラグインでも機能しますか？

もちろんです！標準の WordPress 翻訳関数を使用する任意のプラグインやテーマは、サイト固有の翻訳で機能します。
