# WooCommerce 連携変更履歴 {#woocommerce-integration-changelog}

バージョン 2.2.0 - 2026-07-01 リリース
* 新規: Ultimate Multisite の税額が、チェックアウト時に個別の WooCommerce 手数料行として表示されるようになり、支払い前に税合計が分かりやすくなりました。
* 新規: WooCommerce Subscriptions の再試行期間中に更新失敗による停止を行いたいサイト向けに、オプトインの「更新失敗時にメンバーシップを即時停止」設定と `wu_woo_suspend_on_payment_failure` フィルターを追加しました。
* 修正: 更新の失敗後または復旧後に Ultimate Multisite のメンバーシップと同期しないままになる可能性があった WooCommerce Subscription のステータスを調整しました。
* 修正: Ultimate Multisite の通貨一覧に WooCommerce ストア通貨がない場合、それを追加するようにしました。
* 修正: 購読者を WooCommerce チェックアウトへリダイレクトする際に、顧客の請求先情報を保持するようにしました。
* 改善: Jetpack Autoloader 5 との互換性を追加しました。
* 改善: リリースパッケージ生成を整理し、GitHub とマーケットプレイスの zip にネストされたステージングディレクトリや開発用ファイルが含まれないようにしました。

バージョン 2.0.6 - 2026-01-16 リリース
* 改善: アドオンにコアのサブスクリプションを含めました。Woocommerce Subscriptinos 拡張機能は不要になりました。

バージョン 2.0.5 - 2026-01-09 リリース
* 改善: glotpress API から翻訳を読み込むようにしました。
* 修正: 一部のページビルダーで致命的エラーが発生していました。
* 修正: 顧客がメインサイトのメンバーである場合の無限リダイレクト。

バージョン 2.0.4 - 2025-11-14 リリース
* 追加: さらに多くの言語の翻訳。
* 変更: Ultimate Multisite: Woocommerce Integration に名称変更しました。
* 追加: Woocommerce 10.2.1 との互換性。
* 追加: Woocommerce Subscriptions 7.7.0 との互換性。
* 修正: PHP 8.4 との互換性。
* 修正: WC Account ページが存在しない場合のリダイレクト。

バージョン 2.0.3 - 2025-08-13 リリース
* 変更: 新しいマーケットプレイスで自動更新を有効化しました。

バージョン 2.0.2 - 2025-07-05 リリース
* 変更: Multisite Ultimate: Woocommerce Integration に名称変更しました。
* 追加: Woocommerce 9.8.1 との互換性。
* 追加: Woocommerce Subscriptions 7.3.0 との互換性。
* 修正: 顧客によるサブスクリプションのキャンセル。
* 修正: チェックアウトブロック使用時の致命的エラー。
* 改善: Woocommerce の高性能カスタム注文テーブルに対応しました。
* 修正: WooCommerce チェックアウトでキャンセルしても、メンバーシップがアップグレードされる場合がありました。

バージョン 2.0.1 - 2023-08-09 リリース

* 追加: Woocommerce 7.9.0 との互換性。
* 追加: Woocommerce Subscriptions 5.3.0 との互換性。
* 追加: メンバーシップ更新のサポート。
* 追加: Woocommerce でのトライアルとセットアップ手数料に関する通知。
* 追加: メタ値による Ultimate Multisite Woocommerce 商品の識別。
* 追加: Ultimate Multisite 関連のすべての Woocommerce 商品をマークする一回限りの修正を挿入しました。
* 追加: Ultimate Multisite が作成した商品を Woocommerce 一覧から削除しました。
* 改善: カートに適用する非継続の Woocommerce 割引を作成しました。
* 改善: Woocommerce 商品に継続割引を復元しました。
* 改善: Woocommerce 商品に継続割引ラベルを追加しました。
* 改善: チェックアウト時の商品タイプを保証しました。
* 修正: ダウングレード処理中にメンバーシップステータスを維持しました。
* 修正: キャンセル処理中のエラーを避けるため、サブスクリプションの存在を確認しました。
* 修正: Woocommerce サブスクリプションで使用するサブスクリプション開始日を追加しました。
* 内部: 新しい PHP 8.1 ビルド処理を実装しました。

バージョン 2.0.0 - 完全な書き直し。

* 追加: ゲートウェイ変更時またはメンバーシップキャンセル時に woo サブスクリプションを削除するキャンセル処理メソッド。
* 追加: メンバーシップのダウングレードとアップグレードを行うハンドラー。
* 改善: Account 更新を可能にするため、サブサイトの顧客更新フォームで woocommerce 依存関係を読み込むようにしました。
* 改善: 存在しない場合に Woocommerce カートを正しく読み込むようにしました。
* 改善: チェックアウト処理時にメインサイトのテーブル上にいることを保証しました。
* 改善: Ultimo の更新注文を、直近の支払いではなく Woocommerce サブスクリプション注文額に基づくようにしました。
* 修正: WU メンバーシップボタンのリンク。
* 修正: Woocommerce サブスクリプションの更新が支払い済みになったとき、Ultimo 注文を支払い済みに設定しました。
* ビルド: ビルダーとして MPB を追加しました。

バージョン 2.0.0-beta-5 - 2022-01-21 リリース

* 内部: フックとフィルターのジェネレーターを追加しました。
* 内部: 開発者の作業性向上のため、Ultimate Multisite スタブを追加しました。
* 修正: 不要な場合に複数の商品が作成されるのを防止しました。
