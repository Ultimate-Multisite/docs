# WooCommerce Integration 変更履歴

Version 2.0.6 - リリース日 2026-01-16
* 改善: アドオンにコアサブスクリプションを含めます。Woocommerce Subscriptinos extensionは不要です。

Version 2.0.5 - リリース日 2026-01-09
* 改善: glotpress APIから翻訳を読み込みます。
* 修正: 一部ページビルダーで致命的なエラーが発生します。
* 修正: 顧客がメインサイトのメンバーの場合に無限リダイレクトが発生します。

Version 2.0.4 - リリース日 2025-11-14
* 追加: より多くの言語の翻訳を追加しました。
* 変更: Ultimate Multisite: Woocommerce Integrationにリネームしました。
* 追加: Woocommerce 10.2.1との互換性を追加しました。
* 追加: Woocommerce Subscriptions 7.7.0との互換性を追加しました。
* 修正: PHP 8.4との互換性を修正しました。
* 修正: WCアカウントページが存在しない場合にリダイレクトします。

Version 2.0.3 - リリース日 2025-08-13
* 変更: 新しいマーケットプレイスで自動更新を有効にしました。

Version 2.0.2 - リリース日 2025-07-05
* 変更: Multisite Ultimate: Woocommerce Integrationにリネームしました。
* 追加: Woocommerce 9.8.1との互換性を追加しました。
* 追加: Woocommerce Subscriptions 7.3.0との互換性を追加しました。
* 修正: 顧客によるサブスクリプションのキャンセルを修正しました。
* 修正: チェックアウトブロックを使用した際の致命的なエラーを修正しました。
* 改善: Woocommerceの高性能カスタムオーダーテーブルに対応しました。
* 修正: WooCommerceチェックアウトでのキャンセルがメンバーシップのアップグレードを引き起こす問題を修正しました。

Version 2.0.1 - リリース日 2023-08-09
* 追加: Woocommerce 7.9.0との互換性を追加しました。
* 追加: Woocommerce Subscriptions 5.3.0との互換性を追加しました。
* 追加: メンバーシップ更新のサポートを追加しました。
* 追加: Woocommerceでのトライアルと設定費用に関する通知を追加しました。
* 追加: Ultimate Multisite Woocommerce製品をメタ値で識別しました。
* 追加: Ultimate Multisite関連のWoocommerce製品をすべてマークするワンタイム修正を挿入しました。
* 追加: Ultimate Multisiteで作成された製品をWoocommerceリストから削除しました。
* 改善: カートに適用する非継続的Woocommerce割引を作成しました。
* 改善: Woocommerce製品への継続的割引を復元しました。
* 改善: Woocommerce製品に継続的割引ラベルを追加しました。
* 改善: チェックアウト時に製品タイプを確保しました。
* 修正: ダウングレードプロセス中にメンバーシップステータスを維持しました。
* 修正: キャンセルプロセス中にエラーを回避するためにサブスクリプションの存在を確認しました。
* 修正: Woocommerceサブスクリプションで使用する開始サブスクリプション日付を追加しました。
* 内部: 新しいPHP 8.1ビルドプロセスを実装しました。

Version 2.0.0 - 完全リライト
* 追加: ゲートウェイ変更またはメンバーシップキャンセル時にWooサブスクリプションを削除するキャンセル処理メソッドを追加しました。
* 追加: メンバーシップのダウングレードとアップグレードを処理するハンドラを追加しました。
* 改善: サブサイトの顧客更新フォームでWooCommerce依存関係を読み込み、アカウント更新を可能にしました。
* 改善: WooCommerceカートが存在しない場合に正しく読み込むようにしました。
* 改善: チェックアウト処理時にメインサイトのテーブルを使用していることを保証しました。
* 改善: Ultimoの更新注文をWooCommerceサブスクリプション注文価値に基づいて作成し、前回の支払いではなくします。
* 修正: WU Membershipボタンリンクへ遷移します。
* 修正: WooCommerceサブスクリプション更新が支払われた場合にUltimo注文を支払済みとして設定します。
* ビルド: MPBをビルダーとして追加しました。

Version 2.0.0-beta-5 - リリース日 2022-01-21
* 内部: フックとフィルタ生成器を追加しました。
* 内部: 開発者の利便性のためにUltimate Multisiteスタブを追加しました。
* 修正: 不要な複数製品の作成を防止しました。

Version 2.0.0-beta.4 - 2021-09-23
* 修正: WooCommerceをネットワークアクティブにする必要があるように修正しました（メインサイトのみではなく）。
* 改善: アドオンをmu-pluginとして使用できるようにフィルタを追加しました。

Version 2.0.0-beta.3 - 2021-05-28
* 修正: ダッシュボードアクセス制御が過剰だった問題を修正しました。
* 改善: Ultimate MultisiteトップメニューにWooCommerceヘルプリンクを追加しました。

Version 2.0.0-beta.2 - 2021-05-04
* 改善: WCS更新注文作成時にUltimoで保留中の支払いを作成します。
* 改善: Ultimate Multisite顧客データで請求情報を事前入力します。
* 改善: ゲートウェイ用の請求フィールドを再追加しました。

Version 2.0.0-beta.1 - 2021-05-04
* 初期ベータリリース

* 修正: WooCommerce Integrationは、WooCommerceがネットワークアクティブでなく、メインサイトでのみ有効化されている場合でも動作するようにしました。
