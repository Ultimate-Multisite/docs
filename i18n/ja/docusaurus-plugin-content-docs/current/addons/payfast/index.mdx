# PayFast

Multisite Ultimate: Payfast Gateway

Adds Payfast - a fast and secure online payments gateway in South Africa - as a Multisite Ultimate payment option.

## インストール

1. 「ultimate-multisite-payfast」を「/wp-content/plugins/」ディレクトリにアップロードします
2. WordPress の「プラグイン」メニューからプラグインを有効化します
3. ウィザードに従ってプラグインを設定します

## 使用ガイド

PayFast は南アフリカの企業向けの主要なオンライン決済プロセッサです。取引をシームレスかつ安全にすることでオンライン販売者の成長を支援することに専念しています。Ultimate Multisite のユーザーのリクエストに応じて、PayFast との統合を可能にするアドオンを作成することにしました。この記事では、Ultimate Multisite ネットワークで支払いを受け取るために PayFast を支払いゲートウェイとして設定する方法を示します。

最初のステップはアドオンをインストールすることです。

# Payfast アドオンのインストール方法:

アドオンページに移動し、Ultimate Multisite: Payfast Gateway アドオンを選択します。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-BmLWqj4yjt.png)

アドオンウィンドウが表示されます。『Install Now』をクリックします。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1jpCakOHNy.png)

インストールが完了すると、プラグインページにリダイレクトされます。ここで『Network Activate』をクリックすると、_Payfast_ アドオンがネットワーク上で有効化されます。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-49OQHBwPxk.png)

# PayFast の設定

アドオンをインストールしたら、Ultimate Multisite の設定に移動して PayFast の設定を開始できます。

**Ultimate Multisite settings** ページの「Payments」タブに「**Payfast**」が表示され、トグルボタンでこの決済ゲートウェイを有効/無効にできます。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-07bPkaGJka.png)トグルボタンをクリックして有効にします。すると、**Mechant ID、Merchant Key、PassPhrase** など入力が必要なフィールドが表示されます。これらの情報は **Payfast** アカウントから取得できます。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-pocxO2VPfM.png)

次に、PayFast アカウントにログインして必要な情報を取得します。

ログインすると、**Payfast** アカウントのダッシュボードに必要な認証情報が表示されます。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/kw0WnsxHIW.png)認証情報をコピーして Ultimate Multisite の設定に貼り付けます。次に **「設定を保存」** ボタンをクリックして変更を保存します。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-5MIrG21r3e.png)

これで完了です。PayFast の決済統合が完了しました！
