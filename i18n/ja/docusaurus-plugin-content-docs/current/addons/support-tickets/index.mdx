# サポートチケット

**Support Tickets** は、Ultimate Multisite 用の強力なアドオンで、包括的なサポートチケットシステムを WordPress ネットワークに追加します。ネットワークレベルと個別サイトレベルの両方で顧客サポートを提供する必要がある SaaS プラットフォーム、WaaS（Website as a Service）プロバイダー、およびマルチサイトネットワークに最適です。

## Support Tickets を選ぶ理由

* **Multi-Level Support**: ネットワーク管理者はネットワーク全体のチケットを処理し、サイトオーナーは自分の顧客サポートを管理します。
* **Professional Workflow**: 優先度、ステータス、割り当てを含む完全なチケットライフサイクル管理。
* **Customer Experience**: 美しくレスポンシブなインターフェースで、顧客が簡単にサポートを受けられます。
* **Secure & Reliable**: WordPress のセキュリティベストプラクティスに基づき構築され、データベースパフォーマンスを最適化しています。
* **Seamless Integration**: Ultimate Multisite の顧客およびサイト管理と完璧に連携します。

## 主な機能

**完全なチケット管理**

* ユニークなチケット番号システムで簡単に参照できます。
* 設定可能な優先度レベル（低・標準・高・重大）。
* 包括的なステータスワークフロー（オープン、進行中、待機中、解決済み、クローズ）。
* チケット分類（一般、技術、請求、機能リクエスト、バグ報告）。
* サポートエージェントにチケットを割り当て、オプションで自動割り当てを有効にします。

**スレッド化された会話**

* チケット返信の完全な会話スレッド化。
* スタッフと顧客の返信に対して異なるスタイリング。
* 顧客が閲覧できないプライベートスタッフノート。
* リアルタイム AJAX 対応インターフェース。
* フォーマットオプション付きのモダンテキストエディタ。

**ファイル添付**

* ドラッグ＆ドロップ対応の安全なファイルアップロード。
* 設定可能なファイルタイプとサイズ制限。
* 悪意のあるファイル検出機能付きの高度なセキュリティ。
* アクセス制御付きの保護されたダウンロード。

**スマート通知**

* 新規チケット、返信、ステータス変更時の自動メール通知。
* カスタマイズ可能なメールテンプレート。
* スタッフと顧客向けのロールベース通知。
* パーソナライズされたメッセージ用の動的テンプレート変数。

**強力な管理インターフェース**

* 並べ替えとフィルタリング機能付きの高度なチケットリスト。
* ステータス、優先度、割り当てのクイック編集。
* 効率的な管理のための一括操作。
* パフォーマンス指標付きの統計ダッシュボード。
* 包括的な設定パネル。

**美しい顧客ポータル**

* すべてのデバイスでクリーンでレスポンシブなインターフェース。
* モバイルファーストデザイン。
* 高度な検索とフィルタリング。
* フォーマットツールバー付きのモダンエディタ。
* 使いやすいチケット提出フォーム。

**柔軟な統合**

* ショートコードまたは Ultimate Multisite UI 要素を使用。
* サイトの任意の場所にチケットフォームとリストを表示。
* マルチサイトネットワーク構造とシームレスに連携。
* フックとフィルタで拡張可能。

## 適した用途

* 顧客サポートが必要な SaaS プラットフォーム
* 複数のクライアントサイトを管理する WaaS プロバイダー
* 階層化されたサポートを提供するメンバーシップサイト
* 複数ブランドを管理するネットワーク管理者
* 顧客サービスを提供するサイトオーナー

## インストール

1. Ultimate Multisite プラグインが WordPress ネットワークでインストールされ、アクティブになっていることを確認します。
2. アドオンファイルを `/wp-content/plugins/` ディレクトリにアップロードします。
3. WordPress の「プラグイン」メニューからネットワークアクティベートします。
4. ネットワーク管理 > サポートチケットに移動し、設定を構成します。

## よくある質問

## これは Ultimate Multisite が必要ですか？

はい、このアドオンは Ultimate Multisite がインストールされ、アクティブである必要があります。Ultimate Multisite の顧客およびサイト管理機能と深く統合されます。

## サイトオーナーは自分の顧客サポートを扱えますか？

はい！ サイトオーナーはチケットシステムを使用して自分の顧客にサポートを提供できます。ネットワーク管理者はネットワーク全体のサポートチケットを処理します。

## 顧客はどのようにチケットを提出しますか？

ショートコードまたは Ultimate Multisite UI 要素を使用して、サイトの任意の場所にチケット提出フォームを追加できます。顧客はフォームに入力し、リクエストを送信するだけです。

## チケットのステータスと優先度をカスタマイズできますか？

はい！ ネットワーク管理者設定で優先度レベル、ステータス、チケットタイプを完全にカスタマイズして、サポートワークフローに合わせることができます。

## ファイル添付に対応していますか？

はい、顧客とスタッフの両方がチケットと返信にファイルを添付できます。許可されたファイルタイプとサイズ制限を完全に制御できます。

## 通知はどのように処理されますか？

システムは新規チケット、返信、ステータス変更時に自動でメール通知を送信します。設定ですべてのメールテンプレートをカスタマイズできます。

## チケットを特定のサポートエージェントに割り当てられますか？

はい、手動でエージェントにチケットを割り当てるか、自動割り当てを有効にしてチケットを自動的に配布できます。

## スタッフ用のプライベートノートはありますか？

はい、スタッフメンバーは他のスタッフメンバーにのみ表示されるプライベート返信を追加できます。顧客には表示されません。

## モバイルフレンドリーですか？

はい！ インターフェースはモバイルファーストで構築され、すべてのデバイスで美しく機能します。

## ショートコードを使用してチケットリストとフォームを表示できますか？

はい！ 利用可能なショートコード：
* `[wu_support_tickets]` - 顧客チケットリストを表示
* `[wu_support_ticket_form]` - 新規チケット提出フォームを表示
* `[wu_support_ticket_view ticket_id="123"]` - 特定のチケットを表示

## アップロードできるファイルタイプは何ですか？

設定で許可されたファイルタイプを構成できます。システムは悪意のあるファイルから保護する高度なセキュリティスキャンを含みます。

## チケットデータをエクスポートできますか？

はい、アドオンにはチケットデータと統計情報のエクスポート機能が含まれています。
