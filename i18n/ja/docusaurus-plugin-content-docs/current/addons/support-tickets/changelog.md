# サポートチケット変更履歴 {#support-tickets-changelog}

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* 改善: vendor/ ディレクトリを Git 追跡から削除（すでに .gitignore で対象外）、リポジトリサイズを削減
* 改善: WordPress 7.0 までテスト済み

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* 修正: null 許容チケットモデルのセッターで null 値を受け入れるように修正
* 修正: 適切なマルチサイト対応のため、すべてのチケットをネットワーク全体のグローバルテーブルに保存
* 修正: 顧客の新規チケットフォームでスタッフ用項目を非表示にし、空白ページへのリダイレクトを修正
* 修正: メタデータ保存のため、未定義の add_meta() 呼び出しを正しい update_meta() に置き換え
* 修正: 未登録の権限チェックを正しい wu_view_all_support_tickets に置き換え
* 修正: チケットステータス、割り当て、クイック編集操作に不足していた AJAX ハンドラーを追加
* 修正: チケット表示でスタッフ返信を検出するメソッド名を修正
* 修正: 重複した返信ハンドラーを統合し、nonce アクション名を統一
* 修正: [wu_submit_ticket] shortcode 用に不足していたフロントエンド表示を追加
* 修正: 不足していた user_id カラムを追加し、Support_Ticket::get_user_id() メソッドを修正
* 修正: 優先度フィルターのオプションタグにある余分な二重引用符を削除
* 修正: スーパー管理者向けにネットワーク管理のチケット管理パネルを追加
* 改善: 管理画面の CSS を単一の外部スタイルシートに統合
* 改善: サブサイト管理メニューから設定サブメニューを削除
* 改善: サポートチケットページでのみフロントエンドアセットを条件付きで読み込み
* 改善: Bedrock ルートのオートローダーが依存関係をすでに読み込んでいる場合、plugin オートローダーをスキップ

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* 追加: 管理者とフロントエンド向けの完全なチケット表示機能
* 追加: チケット返信送信用の AJAX ハンドラー
* 追加: 適切なフォーム処理によるチケット返信機能の対応
* 追加: 管理画面でのチケット送信と返信に対する適切な通知表示
* 追加: 顧客の自己送信時に現在のユーザーへ自動的にチケットを関連付け
* 追加: 顧客がチケット所有者を上書きできないようにするセキュリティ強化
* 追加: 不足していたヘルパー関数（wu_format_date、wu_user_can_view_ticket など）
* 追加: 適切なファイル添付のダウンロードと処理
* 追加: チケット返信とステータス変更のメール通知システム
* 修正: チケット返信フォームに必要なチケット ID を含めるように修正
* 修正: 管理画面のチケット表示における通知の表示問題を修正
* 修正: サポートチケット関数の構文エラーを修正
* 改善: 返信と添付ファイル向けの適切なクエリおよびスキーマクラスによるデータベース構造
* 改善: チケットモデルで属性ではなく実プロパティへ移行

### 1.0.1 - 2025-09-28 {#101---2025-09-28}
* 更新: 一貫性のためプレフィックスを ultimate-multisite に変更
* 更新: テキストドメインを標準化
* 修正: 軽微なバグ修正と改善

### 1.0.0 - 2025-09-01 {#100---2025-09-01}
* 初回リリース
* 完全なチケット管理システム
* 多段階アクセス制御
* スレッド式会話システム
* ファイル添付対応
* メール通知システム
* 管理者および顧客向けインターフェース
* 統計とレポート
