# Support Tickets Changelog

### 1.0.2 - 2025-12-11
* Added: 管理者とフロントエンドの完全なチケット閲覧機能
* Added: チケット返信送信用のAJAX ハンドラ
* Added: 適切なフォーム処理を備えたチケット返信機能のサポート
* Added: 管理者でのチケット送信と返信に対する適切な通知表示
* Added: 顧客自身の送信に対して現在のユーザーと自動的にチケットを関連付け
* Added: 顧客がチケット所有権を上書きできないようにするセキュリティ強化
* Added: 欠落していたヘルパー関数（wu_format_date、wu_user_can_view_ticket など）
* Added: 適切なファイル添付ダウンロードと処理
* Added: チケット返信とステータス変更のためのメール通知システム
* Fixed: チケット返信フォームに必要なチケットIDが含まれるように
* Fixed: 管理者のチケットビューでの通知表示の問題
* Fixed: サポートチケット関数の構文エラー
* Improved: 応答と添付ファイル用の適切なクエリとスキーマクラスを備えたデータベース構造
* Improved: チケットモデルの属性ではなく実際のプロパティへの移行

### 1.0.1 - 2025-09-28
* Updated: 一貫性のためにプレフィックスを ultimate-multisite に変更
* Updated: テキストドメインの標準化
* Fixed: 小さなバグ修正と改善

### 1.0.0 - 2025-09-01
* Initial release
* 完全なチケット管理システム
* 多層アクセス制御
* スレッド化された会話システム
* ファイル添付サポート
* メール通知システム
* 管理者と顧客のインターフェース
* 統計とレポート
