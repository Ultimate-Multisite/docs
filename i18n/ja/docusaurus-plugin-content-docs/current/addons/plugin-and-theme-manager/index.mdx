# プラグイン & テーマ マネージャー

Multisite Ultimate: プラグインとテーママネージャー

インストール済みのプラグインとテーマのメタ情報（タイトル、作者、サムネイル、説明）を編集または非表示にします。プラグインとテーマのカテゴリを作成し、ユーザーがフィルタリングできるようにし、ユーザーのパネルに美しいカスタムプラグインページを追加します。

プラグインを使用するには、ネットワーク管理画面のプラグインまたはテーマページに移動し、「プラグイン/テーマ情報を編集」をクリックするだけです。

## インストール

1. 「ultimate-multisite-plugin-and-theme-manager」を '/wp-content/plugins/' ディレクトリにアップロードします。
2. WordPress ネットワーク管理画面の「Plugins」メニューからプラグインを有効化します。

## 使用ガイド

Ultimate Multisite: プラグインとテーママネージャー は、ネットワークユーザーがネットワーク上のプラグインとテーマのオプションをどのように表示し、操作するかを完全に制御できる強力な **プレミアム** Ultimate Multisite アドオンです。

このアドオンは3つの目的を果たします：

  1. プラグインとテーマのメタデータ（名前、説明、作者名、サムネイルを含む）を完全に変更できます！
  2. プラグインとテーマをカテゴリ分けでき、ユーザーが簡単に並べ替えられるようにします。
  3. 古くて退屈なプラグインページを、カテゴリ別に並べ替え可能なテーマ風ページに置き換えるオプションを提供します。

ダウンロードしてインストール後、Ultimate Multisite 設定ページに新しいタブ「_Add-on Settings_」が表示されます。
