# プラグインとテーマのマネージャー

Ultimate Multisite: プラグインとテーマのマネージャー

インストール済みのプラグインとテーマのメタ情報（タイトル、作者、サムネイル、説明）を編集または非表示にできます。プラグインとテーマにカテゴリを作成して、ユーザーが絞り込めるようにし、ユーザーのパネルに美しいカスタムのプラグインページを追加できます。

このプラグインを使うには、ネットワーク管理のプラグインまたはテーマページに移動し、「プラグイン/テーマ情報を編集」をクリックするだけです。

## インストール

1. 'ultimate-multisite-plugin-and-theme-manager' を '/wp-content/plugins/' ディレクトリにアップロードします
2. WordPress ネットワーク管理の 'Plugins' メニューからプラグインを有効化します

## 使用ガイド

Ultimate Multisite: プラグインとテーマのマネージャーは、ネットワーク上のユーザーがプラグインとテーマのオプションをどのように見て操作するかを完全に制御できる、強力な **premium** Ultimate Multisite アドオンです。

このアドオンには、3つの目的があります。

  1. プラグインとテーマの両方のメタデータを完全に変更できます。名前、説明、作者名、さらにはサムネイルも含まれます！

  2. プラグインとテーマをカテゴリ分けし、ユーザーが簡単に整理できるようにします。

  3. 古くて味気ないプラグインページを、カテゴリによる並べ替えに対応した新しいテーマ風ページに置き換えるオプションを提供します。

ダウンロードしてインストールすると、Ultimate Multisite 設定ページに新しいタブ - _アドオン設定_ \- が表示されます。
