# フィルタ: wp_ultimo_skip_network_active_check

開発者がこのチェックを短絡できるようにします。

これは、Composer ベースの構成や Bedrock などのカスタムセットアップを使用する際に有用です。例えば、mu-plugins としてプラグインを使用することが標準である場合です。

### 以降

- 2.0.0
