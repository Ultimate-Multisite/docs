# Plugin & Theme Manager 変更履歴

Version: 2.0.4 - 2026-XX-XX

* 修正: PHP 8.2+ の非推奨警告を修正し、WP_Ultimo_Plugin_And_Theme_Manager クラスで動的プロパティ作成を修正。

Version: 2.0.3 - 2025-09-28

* 修正: Ultimate Multisite v2.4 との互換性を修正。

Version: 2.0.1 - 2025-09-28

* プレフィックスを ultimate-multisite に変更し、テキストドメインを更新し、バージョンを上げる。

Version 2.0.0 - 2021-11-25 にリリース

* リライト: 2.0 の新しいコードを使用した完全リライト。

Version 1.2.5 - 11/03/2020

* 修正: 子テーマの場合に親テーマ名をフィルタリング。
* 修正: カテゴリフィルタが正確な一致のみプラグインを返すようにする。
* 修正: テーマページでフィルタが機能しない問題を修正。
* 修正: 必要に応じて削除されたテーマ/プラグイン情報をクリーンアップ。

Version 1.2.4 - 29/03/2019

* 修正: プラグイン詳細がモーダルウィンドウを開かない問題を修正。

Version 1.2.3 - 15/03/2019

* 修正: Material WP で使用時のレスポンシブ性の問題を修正。

Version 1.2.2 - 27/02/2019

* 修正: プラグインリストが20項目で切断される問題を修正。
* 改善: 対応するプラグインまたはテーマが削除されたときに、プラグインとテーマのカスタマイズデータが削除されるように改善。

Version 1.2.1 - 29/01/2019

* 修正: 前回リリースの小さな問題を修正。
* 改善: 大規模なプラグインとテーマテーブルのパフォーマンスを向上。

Version 1.2.0 - 21/01/2019

* 修正: たくさんの小さなバグを修正！

Version 1.1.2 - 07/12/2018

* 修正: オプションが無効になっているにもかかわらず、作者が表示される問題を修正。
* 改善: admin_notices フックから in_admin_header へインラインフォームを移動。
* 改善: サブサイトのエクスポート画面から Multisite Ultimate カスタム投稿タイプを削除。

Version 1.1.1 - 07/12/2018

* 改善: 更新用の新しい URL を追加。

Version 1.1.0 - 15/08/2018

* 修正: カテゴリ保存時の小さな問題を修正。
* 修正: JS のエラーを修正。
* 追加: プラグインページの新しい表示モデル: Plugin Cards!

1.0.0 - 初期リリース
