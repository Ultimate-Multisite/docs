# サポートエージェント変更履歴

## バージョン: 1.0.13 - 2025-09-28にリリース
* プレフィックスを ultimate-multisite に変更; テキストドメインを更新; バージョンを上げました。

## バージョン 1.0.12 - 2025-09-17にリリース
* 修正: Ultimate Multisite 2.4.4 との互換性。

## バージョン 1.0.11 - 2023-11-21にリリース
* 修正: Ultimate Multisite 2.3.0 との互換性。

## バージョン 1.0.10 - 2023-08-30にリリース
* 修正: Bulk Support Agents の削除。

## バージョン 1.0.9 - 2023-08-09にリリース
* 追加: チェックアウトフォームを読み取る機能を追加しました。
* 修正: サポートエージェントの権限フォームレイアウト;
* 修正: サポートエージェント削除フォームの警告メッセージ;
* 修正: 未承認ユーザーのためにネットワークダッシュボードウィジェットを非表示にしました。
* 修正: サポートエージェント編集と削除権限を修正しました。
* 修正: サポートエージェント作成プロセスを修正しました。
* 修正: 顧客をサポートエージェントとして追加できないようにしました。
* 修正: 未承認ユーザーをサポートエージェント編集管理ページにリダイレクトしました。
* 修正: サポートエージェント権限フォームのレイアウトを改善しました。
* 内部: 新しいビルドプロセスを実装しました。

## バージョン 1.0.8 - 2022-01-21にリリース
* 内部: フックとフィルタ生成器を追加しました;
* 内部: 開発者の利便性向上のために Ultimate Multisite スタブを追加しました;
* 追加: チェックアウトフォーム機能;
* 修正: Ultimate Multisite フォームとこのアドオンオプション間の権限名不一致;

## バージョン 1.0.7 - 2021-12-02にリリース
* 修正: サポートエージェント削除が機能しない;
* 修正: サポートエージェントの一括削除が機能しない;

## バージョン 1.0.6 - 2021-11-22にリリース
* 修正: コアのパフォーマンス最適化後に新しいモーダルが機能しない;
* 修正: コアにグリッドアイテムビューが存在し、アドオンには存在しない;

## バージョン 1.0.5 - 2021-10-31にリリース
* 修正: PHP 8 との小さな互換性問題;
* 改善: wp_ultimo_skip_network_active_check now uses wu_is_must_use();

## バージョン 1.0.4 - 2021-09-24にリリース
* 追加: mu-plugins ベースのセットアップ用に wp_ultimo_skip_network_active_check フィルタ;

## バージョン 1.0.3 - 2021-08-29にリリース
* 追加: REST API サポート - サポートエージェントを REST 経由で作成できるようにしました;

## バージョン 1.0.2 - 2021-07-26にリリース
* 修正: エージェント権限の更新が機能しない;

## バージョン 1.0.1 - 2021-05-28にリリース
* 追加: ユーザーが既にスーパー管理者の場合の通知;
* 追加: ユーザー切替を許可する権限;
* 改善: より良いフォームフィールド;
