# サポートエージェント

自分一人でネットワーク全体を管理するのは簡単なことではありません。ネットワークが拡大し始めると、サポートへの需要が増えていきます。作業負担を軽減するために人を雇うのは良いアイデアです。ただし、完全なアクセス権や権限を与えることはリスクが高く、ネットワークを危険にさらす可能性があります。そこで、このサポートエージェントアドオンが登場します。

このアドオンを使うと、WordPress ユーザーの新しいタイプ「サポートエージェント」を作成でき、ユーザーに付与する権限を完全に制御できます。権限の付与はトグルボタンをオンにするだけで、解除したい場合はオフにすれば簡単に取り消せます。

## 特徴

- WordPress と Ultimate Multisite の権限を項目別に一覧表示
- アクセスの有効化・無効化を簡単に行える
- エージェントアカウントを無制限に作成可能

## インストール

1. `ultimate-multisite-support-agents` を `/wp-content/plugins/` ディレクトリにアップロード
2. WordPress の「プラグイン」メニューからプラグインを有効化
3. ウィザードに従ってステップバイステップで設定を完了する
