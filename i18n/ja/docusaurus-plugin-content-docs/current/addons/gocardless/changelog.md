# GoCardless 変更履歴 {#gocardless-changelog}

Version: 1.0.3 - 2026-05-05 リリース
* 修正: GoCardless メンバーシップが保留中または一時停止状態でキャンセルされた場合、孤立した保留中のサイトがクリーンアップされるようになりました
* 改善: Bedrock ルートのオートローダーがすでに依存関係を読み込んでいる場合、Plugin のオートローダーをスキップするようにしました

Version: 1.0.1 - 2025-09-28 リリース

* 接頭辞を ultimate-multisite に変更。テキストドメインを更新。バージョンを更新。



2021-09-24 - Version 1.0.0-beta.4

* 追加: mu-plugins ベースのセットアップ向けに filter wp_ultimo_skip_network_active_check を追加。

2021-06-16 - Version 1.0.0-beta.3

* 修正: Webhook ハンドラーが署名の確認に失敗していた問題。

2021-06-14 - Version 1.0.0-beta.2

* 改善: GoCardless マンデートの作成後にサイトを公開。

Version 1.0.0-beta.1 - 初期バージョン
