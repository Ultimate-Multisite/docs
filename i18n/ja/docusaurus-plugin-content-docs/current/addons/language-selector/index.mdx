# 言語セレクター

Multisite Ultimate: サインアップ時の言語セレクター

サインアップフローに言語セレクターフィールドを追加します。

## Installation

1. 'ultimate-multisite-language-selector' を '/wp-content/plugins/' ディレクトリにアップロードします。
2. WordPress ネットワーク管理画面の 'Plugins' メニューからプラグインを有効化します。

## Usage Guide

この無料アドオンを使うと、ユーザーがサインアップフローとログインページで表示したい言語を選択できるようになります。

ライセンスを有効化した後（[ライセンスを有効化]../../user-guide/getting-started/installing-ultimate-multisite.md）ライセンスでアクセスできる Ultimate Multisite アドオンをインストールできるようになります。

Ultimate Multisite の設定ページに移動し、左側のメニューで _Add-ons_ ラベルをクリックします。

![Add-ons page](/img/admin/addons-page.png)

_Add-ons ページに移動して Language Selector のインストールを開始します_

そこで、提供しているすべてのアドオンが表示されます。

無料アドオンは、サブスクリプションプランに関係なく、すべてのお客様にご利用いただけます。

アドオンのサムネイルの下にある _Install it Now!_ ボタンをクリックします。その後、Plugins ページで有効化します。

![Add-ons page with install button](/img/admin/addons-page.png)

_アドオンをインストールするためにボタンをクリックしてください_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Plugins ページで有効化してください_

次に、Ultimate Multisite の設定メニューに移動し、_Network Settings_ タブを開きます。ページ下部に _Languages_ という新しいセクションが表示されます。

![Settings page](/img/config/settings-general.png)

_新しいセクションが表示されました！_

空白のままにするとすべての言語をオプションとして表示し、必要に応じて利用可能にしたい言語を選択できます。言語は、WordPress のセットアップにインストールされているものです。

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_この WordPress インストールで利用可能な言語の一覧_

変更を保存すると、選択した言語の選択メニューがログインページとサインアップフローの両方に表示されます。

![Login and Registration settings](/img/config/settings-login-registration.png)

_ログインページの言語セレクター_

![Registration page](/img/config/registration-page.png)

_サインアップフローの言語セレクター_

## Setting the Language via the URL

URL の locale パラメータをバイパスしてページの言語を変更することもできます。例えば、イタリア語の登録ページにリダイレクトしたい場合は、URL に ?locale=it_IT を追加します。

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

この機能を利用するには、言語が WordPress ネットワークにインストールされている必要があります。
