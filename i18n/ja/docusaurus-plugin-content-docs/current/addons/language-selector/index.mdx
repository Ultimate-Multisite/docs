# 言語セレクター

Ultimate Multisite: 登録時の言語セレクター

登録フローに言語セレクターのフィールドを追加します。

## インストール {#installation}

1. 'ultimate-multisite-language-selector' を '/wp-content/plugins/' ディレクトリにアップロードします
2. WordPress ネットワーク管理画面の「プラグイン」メニューから plugin を有効化します

## 使い方ガイド {#usage-guide}

この無料アドオンを使うと、ユーザーが登録フローとログインページで表示したい言語を選べるようになります。

[ライセンスを有効化](../../user-guide/getting-started/installing-ultimate-multisite.md)すると、ライセンスで利用できる Ultimate Multisite アドオンをインストールできるようになります。

Ultimate Multisite の設定ページに移動し、左側のメニューで _アドオン_ ラベルをクリックします。

![アドオンページ](/img/admin/addons-page.png)

_アドオンページに移動して、言語セレクターのインストールを開始します_

そこには、提供しているすべてのアドオンが表示されます。

無料アドオンは、申し込んだサブスクリプションプランに関係なく、すべてのお客様が利用できます。

アドオンのサムネイルの下にある _今すぐインストール！_ ボタンをクリックします。その後、プラグインページで有効化します。

![インストールボタン付きのアドオンページ](/img/admin/addons-page.png)

_ボタンをクリックしてアドオンをインストールします_

<!-- スクリーンショットなし: Language Selector アドオンの「ネットワークで有効化」ボタンを表示している WP プラグインページ -->

_プラグインページで有効化します_

次に、Ultimate Multisite の設定メニューに移動し、_ネットワーク設定_ タブを開きます。ページ下部に、_言語_ という新しいセクションが表示されます。

![設定ページ](/img/config/settings-general.png)

_まったく新しいセクションが表示されます！_

空欄のままにすると、すべての言語が選択肢として表示されます。または、利用可能にしたい言語だけを選択できます。言語は、WordPress の設定にインストールされているものです。

<!-- スクリーンショットなし: WP インストールで利用可能な言語を表示する言語選択ドロップダウン -->

_この WP インストールで利用可能な言語の一覧_

変更を保存すると、選択した言語の選択メニューがログインページと登録フローに表示されます。

![ログインと登録の設定](/img/config/settings-login-registration.png)

_ログインページの言語セレクター_

![登録ページ](/img/config/registration-page.png)

_登録フローの言語セレクター_

## URL で言語を設定する {#setting-the-language-via-the-url}

URL に locale パラメーターを渡すことで、ページの言語を変更することもできます。たとえば、イタリア語の登録ページにリダイレクトしたい場合は、URL に ?locale=it_IT を追加します。

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

これを機能させるには、その言語が WordPress ネットワークにインストールされている必要があります。
