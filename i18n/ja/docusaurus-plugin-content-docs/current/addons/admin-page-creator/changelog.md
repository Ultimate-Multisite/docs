# Admin Page Creator 変更履歴

### バージョン 1.9.0 - リリース日 2026-01-18

* 修正: 有効化時に致命的なエラーが発生する問題を修正しました。
* リファクタリング: コンテンツソースクラスを Singleton トレイトを使用し、親コンストラクタの初期化を正しく行うように変更しました。
* 修正: Vue.js テンプレートのコンパイルエラーによりコンテンツタイプセレクタが表示されない問題を修正しました。
* 修正: Vue データプロパティ（template_id、external_link_url_embedable、external_link_url_checking）が欠落している問題を修正しました。
* 修正: TinyMCE エディタがページロード時に正しく初期化されない問題を修正しました。
* 修正: Divi サポートファイルが Composer 経由で自動読み込みされない問題を修正しました。
* 修正: Divi サポートクラス内の破損した PHP 構文を修正しました。
* 改善: JavaScript アセットの整理とミニファイを改善しました。
* 改善: 最新の WordPress とページビルダーのバージョンとの互換性を改善しました。
