# アクション: wu_page_enqueue_scripts

ここにカスタムアクションを追加し、必要に応じてスクリプトとスタイルをキューに入れる場所を確保し、正しいページでのみ読み込まれるようにします

## パラメータ

| 名前 | タイプ | 説明 |
|------|------|-------------|
| $ | `\WU_Page` | 現在の管理ページ |

### バージョン

- 1.3.0
