# AffiliateWP

Multisite Ultimate: AffiliateWP 統合

強力な AffiliateWP を使って、Ultimo ネットワークのクライアント基盤を拡大しましょう！

## インストール

1. 'ultimate-multisite-git ' を '/wp-content/plugins/' ディレクトリにアップロードします
2. WordPress の「プラグイン」メニューから plugin を有効化します
3. 手順ごとのウィザードに従って plugin を設定します

## 使用ガイド

[AffiliateWP](https://affiliatewp.com/) は、オンラインビジネス向けのアフィリエイトプログラムを構築する際の定番ソリューションになっています。Ultimate Multisite: AffiliateWP は、プレミアムネットワークで同じ仕組みを構築できるようにする橋渡しです。

## はじめに

まず、[**AffiliateWP**](https://affiliatewp.com/) が **有効化されている必要があります**！ **連携対象の plugin** がない場合、この add-on は使用できません。クライアントにも各自のサイトで使わせる予定がない場合、AffiliateWP はメインサイトでのみ有効化してください。

また、**Ultimate Multisite** と **Ultimate Multisite: AffiliateWP** の両方を **ネットワーク全体で有効化** しておく必要があります。

## Ultimate Multisite を AffiliateWP 統合として追加する

メインサイトで Affiliates 設定に移動し、Integrations タブで Ultimate Multisite にチェックを入れます。これにより、[AffiliateWP](https://affiliatewp.com/) は Ultimate Multisite のサインアップフローからの訪問とコンバージョンの追跡を開始します。

<!-- スクリーンショット利用不可: Ultimate Multisite チェックボックスがある AffiliateWP Integrations 設定タブ -->

_下部の Ultimate Multisite ボックスにチェックを入れて保存します_

必要な作業はほぼこれだけです。紹介リンクからの訪問が追跡され、そのユーザーが後で初回支払いを行うと、コンバージョンが作成され、そのコンバージョンを担当したアフィリエイトに手数料が割り当てられます。

ただし、追跡されるのは新しく作成されたユーザーから受け取る初回支払いのみです。継続的な紹介支払いを有効にするには、[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) が必要です。

## 継続的な紹介を有効にする

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) をインストールしたら、Affiliates 設定の Recurring Referrals タブに移動し、継続的な紹介の使用を有効にします。

<!-- スクリーンショット利用不可: Enable Recurring Referrals トグルがある AffiliateWP Recurring Referrals 設定 -->

_とても簡単です。Enable Recurring Referrals を有効にすれば準備完了です！_

繰り返しになりますが、必要な作業はそれだけです！

これで、紹介された Account で新しい支払いが受け取られるたびに、担当アフィリエイトが自動的に自分の取り分を受け取ります！
