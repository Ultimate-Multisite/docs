# AffiliateWP

Multisite Ultimate: AffiliateWP 統合

強力な AffiliateWP を使って、あなたの Ultimo Network のクライアントベースを拡大しましょう！

## Installation

1. 「'ultimate-multisite-git'」を「'/wp-content/plugins/'」ディレクトリにアップロードしてください  
2. WordPress の「'Plugins'」メニューからプラグインを有効化してください  
3. ウィザードに従ってプラグインを設定してください  

## Usage Guide

[AffiliateWP](https://affiliatewp.com/) は、オンラインビジネスのアフィリエイトプログラムを構築する際の定番ソリューションとなっています。Ultimate Multisite: AffiliateWP は、プレミアムネットワークでも同じことを実現できる橋渡し役です。

## Getting Started

まず最初に、**AffiliateWP** を **有効化** しておく必要があります！統合プラグインが無いと、アドオンを利用できません。クライアントが自分のサイトでも使用する予定がない場合は、AffiliateWP はメインサイトのみで有効化してください。

また、**Ultimate Multisite** と **Ultimate Multisite: AffiliateWP** を **ネットワーク全体で有効化** する必要があります。

## Adding Ultimate Multisite as an AffiliateWP Integration

メインサイトで「Affiliates Settings」に移動し、Integrations タブで Ultimate Multisite にチェックを入れます。これにより、[AffiliateWP](https://affiliatewp.com/) が Ultimate Multisite のサインアップフローから訪問とコンバージョンを追跡し始めます。

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_下部の Ultimate Multisite ボックスにチェックを入れ、保存してください_

これだけで完了です。紹介リンクから来た訪問は追跡され、ユーザーが初回支払いを行った際にコンバージョンが作成され、コンバージョンに責任を持つアフィリエイトにコミッションが割り当てられます。

ただし、新規ユーザーから受け取る最初の支払いのみが追跡されます。継続的な紹介報酬を有効にするには、[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) が必要です。

## Enabling Recurring Referrals

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) をインストールしたら、Affiliates Settings の Recurring Referrals タブに移動し、継続的な紹介報酬を有効にします。

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_これだけで完了です：継続的な紹介報酬を有効にして、すべて設定完了！_

もう一度言いますが、これだけです！

これで、紹介されたアカウントで新しい支払いが行われるたびに、担当アフィリエイトが自動的に報酬を受け取ります！
