# アクション: wp_ultimo_affiliatewp_load

すべての依存関係が読み込まれたときに発火します

プラグイン開発者が新しい機能を追加できるようにします。例えば、新しいホスティングプロバイダーへのサポートなどです。

### Since

- 1.0.0
