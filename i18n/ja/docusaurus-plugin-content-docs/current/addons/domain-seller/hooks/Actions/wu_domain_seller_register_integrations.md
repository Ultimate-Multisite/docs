# アクション: wu_domain_seller_register_integrations

他のプラグインが追加のドメイン販売者統合を登録できるようにします。

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | 統合レジストリです。 |

### バージョン

- 2.1.0

### ソース

定義は [`inc/class-domain-seller.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-domain-seller/blob/main/inc/class-domain-seller.php#L275) の 275 行目にあります。
