# マルチネットワーク変更履歴 {#multi-network-changelog}

### 1.3.0 {#130}
* 新規: ネットワークテンプレートプレビュー機能 — 購入前にライブパネルでネットワークテンプレートを閲覧し、プレビューできます。
* 新規: チェックアウト時のネットワークテンプレート選択機能 — 顧客はサインアップのチェックアウト手続き中にネットワークテンプレートを選択できます。
* 新規: クロスネットワークのマジックリンク SSO — 顧客はマジックリンク認証でサブネットワークへシームレスにアクセスできます。
* 新規: ネットワークメディアの複製 — テンプレートからネットワークを複製するとき、サイトのメディアが正しく複製されます。
* 修正: マルチテナンシーアドオンなしでもネットワークの複製が動作するようになりました。複製後に特権管理者権限が再付与されます。
* 修正: ネットワーク作成が WooCommerce とすべての決済ゲートウェイに対応しました。
* 修正: フォールバックドメイン生成にポート番号が含まれるようになりました。
* 修正: すべてのテーマで正しく表示されるよう、テンプレート選択機能の CSS をキュー登録されたスタイルシートへ移動しました。
* 修正: サイトクエリスコープで未定義の `network_id` キーを防止しました。
* 修正: コンテキスト切り替え前にネットワーク権限を確認し、`wpdb` テーブルプロパティエラーを防止しました。
