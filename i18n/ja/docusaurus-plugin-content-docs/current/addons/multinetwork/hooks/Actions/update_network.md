# アクション: update_network

既存のネットワークが更新された後に発火します。

## パラメータ

| 名前 | タイプ | 説明 |
|------|------|-------------|
| $network_id | `int` | 追加されたネットワークのID。 |
| $args | `array` | ネットワーク引数の連想配列。 |

- 1.3.0
