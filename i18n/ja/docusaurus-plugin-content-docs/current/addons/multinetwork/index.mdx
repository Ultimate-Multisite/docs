# マルチネットワーク

The Ultimate Multisite: Multinetwork addon extends your Multisite Ultimate platform with powerful multinetwork functionality. This addon enables your customers to purchase and manage entire WordPress multisite networks, not just individual sites.

## 主な機能

* **ネットワークプロビジョニング**: お客様のために完全な WordPress マルチサイトネットワークを自動的に作成および構成します
* **ネットワーク管理**: マルチネットワーク環境に対する完全な管理機能
* **隔離されたネットワーク**: 各顧客ネットワークは独自のデータベーステーブルと設定を持ち、独立して動作します
* **スケーラブルなアーキテクチャ**: 最適化されたリソース割り当てで複数のネットワークを効率的に処理します
* **統合**: Multisite Ultimate のサブスクリプションおよび請求システムとシームレスに統合します

## 利用ケース

* **ホワイトラベルホスティング**: 顧客に独自のブランド化されたマルチサイトネットワークを提供します
* **代理店ソリューション**: 代理店が単一のプラットフォームから複数のクライアントネットワークを管理できるようにします
* **エンタープライズ展開**: 複雑なマルチネットワーク要件を持つ大規模組織をサポートします
* **教育プラットフォーム**: 異なる学校や部門のために別々のネットワークを作成します

## 必要条件

* Multisite Ultimate 2.0.0 以上
* WordPress マルチサイトインストール
* PHP 7.4 以上
* MySQL 5.6 以上

## インストール

1. プラグインファイルを `/wp-content/plugins/ultimate-multisite-multinetwork` ディレクトリにアップロードします
2. WordPress の 'プラグイン' スクリーンからプラグインを有効化します
3. Multisite Ultimate → 設定 からマルチネットワーク設定を構成します

## よくある質問

## マルチネットワークとは？

マルチネットワークは、単一の WordPress インストール上で複数の独立したマルチサイトネットワークを実行できる WordPress の機能です。

## 通常のマルチサイトとの違いは？

通常のマルチサイトは1つのネットワーク内で複数のサイトを許可します。マルチネットワークは、各ネットワークが独自のサイトを持つ複数のネットワークを、すべて単一の WordPress インストール内で実行できるようにします。

## 特別なサーバー構成が必要ですか？

はい、マルチネットワークには特定のサーバー設定と構成が必要です。詳細な設定手順については、ドキュメントをご参照ください。
