# Mailster

Ultimate Multisite Mailster Integration アドオンを使用して、マルチサイトネットワークを強力なメールリスト構築マシンに変換します。新規顧客がサインアップまたは購入した際に自動的に Mailster のメールリストに追加し、メンバーシッププラットフォームとメールマーケティング戦略をシームレスに結びつけます。

## 主な機能

**自動購読者同期**
* サインアップ時に顧客を Mailster リストに自動追加
* サインアップ時に即時同期するか、支払い確認後に同期するか選択
* 手動操作なしでメールリストを最新に保つ

**柔軟なリスト管理**
* すべての新規顧客に対してグローバルデフォルトリストを設定
* ターゲットキャンペーン用に製品別リストでデフォルトを上書き
* Mailster の馴染みのあるインターフェースでリスト割り当てを簡単に管理

**GDPR準拠のオプトインオプション**
* 自動購読かチェックボックスベースの同意のどちらかを選択
* 確認済み購読にダブルオプトインを有効化
* コンプライアンス要件に応じてオプトインチェックボックスを表示/非表示

**スマートデータマッピング**
* 顧客名と連絡先情報を自動同期
* 請求先住所を購読者フィールドにマッピング
* ターゲットキャンペーンのためにサインアップ日を追跡

**製品レベルの制御**
* 製品ごとに異なるリストを割り当て
* 顧客の購入に基づいてターゲットキャンペーンを作成
* 必要に応じて特定製品の Mailster を無効化

**プロフェッショナル体験**
* リスト選択のためのネイティブ Mailster UI 統合
* トラブルシューティングと検証のための詳細ログ
* 既存購読者のシームレスな更新

## こんな方におすすめ

* メールリストを構築するメンバーシップサイトのオーナー
* 新規顧客育成を目指す SaaS プラットフォーム
* 登録状況で学生をセグメント化するコース作成者
* Ultimate Multisite と Mailster を運用している方

## 仕組み

1. Ultimate Multisite ネットワークにアドオンをインストールして有効化
2. Ultimate Multisite の設定でグローバルデフォルトリストを構成
3. 必要に応じて製品別リストで上書き
4. オプトインモードと購読タイミングを選択
5. Mailster リストが自動で増えていくのを確認

## インストール

1. プラグインファイルを Ultimate Multisite のアドオンディレクトリにアップロード
2. メインサイトに Mailster がインストールされ有効化されていることを確認
3. Ultimate Multisite のアドオンページからアドオンを有効化
4. Settings > Mailster Integration に移動して設定
5. デフォルトリストを選択し、オプトイン設定を構成

## よくある質問

## このアドオンは Ultimate Multisite がなくても動作しますか？

いいえ、このアドオンは機能するために Ultimate Multisite が必要です。Mailster と Ultimate Multisite の顧客・製品システムを統合するように設計されています。

## すべてのサイトに Mailster をインストールする必要がありますか？

いいえ、Mailster はメインネットワークサイトにのみインストールして有効化すれば十分です。アドオンはサブサイトから自動的に接続します。

## 製品ごとに異なるリストを割り当てることはできますか？

はい！グローバルデフォルトリストを設定し、製品ごとに上書きできます。これにより、顧客の購入に基づいた高度にターゲット化されたメールキャンペーンを作成できます。

## 顧客が Mailster に既に存在する場合はどうなりますか？

デフォルトでは、アドオンは既存購読者を更新し、新しいリストに追加します。既存購読者をスキップしたい場合は、設定で無効にできます。

## これは GDPR に準拠していますか？

はい！アドオンには、顧客がメール受信に明示的に同意する必要があるチェックボックスベースのオプトインモードが含まれています。適切なプライバシーポリシーと組み合わせると、GDPR 要件を満たします。

## ダブルオプトインを使用できますか？

もちろんです！ダブルオプトイン設定を有効にすると、Mailster は購読を有効化する前に確認メールを送信します。
