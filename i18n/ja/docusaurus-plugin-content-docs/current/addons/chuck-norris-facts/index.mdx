# チャック・ノリスの事実

日々のWordPress管理タスクに少しユーモアを加え、**Ultimate Multisite - Chuck Norris Facts** アドオンを有効化すると、管理画面の右上隅にランダムなチャック・ノリスの事実が表示されます。

チームに笑顔をもたらし、長時間の管理セッション中に雰囲気を軽くしたり、チャック・ノリスの伝説的な素晴らしさを祝うのに最適です。

## チャック・ノリスの事実の理由

* **チームの士気を高める**: ネットワーク管理に楽しい要素を追加
* **アイスブレイク**: 新しいチームメンバーの会話のきっかけに最適
* **生産性向上**: 短い笑いが集中力と生産性を向上させる
* **時代を超えたユーモア**: チャック・ノリスの事実は決して古くならない

## 主な特徴

**110+ 伝説的な事実**  
クラシックなチャック・ノリスのジョークと事実を網羅したコレクションです。

**非侵襲的表示**  
事実は管理バーに非侵襲的に表示され、作業フローを妨げません。

**翻訳対応**  
すべての事実が翻訳関数でラップされており、言語ローカライズやカスタム事実の作成が簡単です。

**完全レスポンシブ**  
デスクトップ、タブレット、モバイル画面で正しく表示されます。チャック・ノリスはすべてのデバイスで素晴らしいです。

**RTLサポート**  
右から左への言語も完全にサポート。チャック・ノリスは言語の壁を越えます。

**ブロックエディタ対応**  
事実はブロックエディタでインテリジェントに隠され、インターフェースの衝突を回避します。

**アクセシビリティ重視**  
完全なアクセシビリティ準拠のためにスクリーンリーダーテキストが含まれています。

## インストール

1. Ultimate Multisite プラグインがインストールされ、WordPress ネットワークで有効化されていることを確認してください
2. アドオンファイルを `/wp-content/plugins/` ディレクトリにアップロードしてください
3. WordPress の 'Plugins' メニューからネットワークでプラグインを有効化してください
4. すべての管理画面でランダムなチャック・ノリスの事実を楽しんでください

## よくある質問

## Ultimate Multisite が必要ですか？

Ultimate Multisite アドオンとして設計されていますが、プラグインは独立して動作し、任意の WordPress インストールで実行できます。

## 自分の事実を追加できますか？

はい！事実は翻訳対応です。カスタム事実を含む `.po` ファイルを作成し、`/lang` ディレクトリに配置できます。また、事実配列をプログラムでフィルタリングすることも可能です。

## 管理画面の速度低下はありますか？

全くありません！プラグインは非常に軽量で、パフォーマンスに影響を与えません。テキストのみを表示し、外部 API 呼び出しや重い処理は行いません。

## 特定のページで無効にできますか？

事実は、ブロックエディタなどのページで自動的に非表示になります。WordPress のフックを使用して、事実が表示される場所をカスタマイズすることもできます。

## すべてのオーディエンスに適していますか？

はい！すべての事実はクリーンでファミリー向けのユーモアで、チャック・ノリスの伝説的なステータスを祝っています。

## 事実を別の言語に翻訳できますか？

もちろんです！すべての事実は WordPress の翻訳関数を使用しています。言語用の翻訳ファイルを作成し、`/lang` ディレクトリに配置してください。テキストドメイン: `ultimate-multisite-chuck-norris-facts`

## マルチサイトで動作しますか？

はい！マルチサイトネットワーク専用に設計されており、ネットワークで有効化できます。
