# VAT

Multisite Ultimate: ヨーロッパのVAT

Adds additional features and tools to Multisite Ultimate's default tax handling to better support European VAT.
Automatically adds EU VAT rates to the Tax Rates page from

## インストール

1. 'ultimate-multisite-vat' を '/wp-content/plugins/' ディレクトリにアップロードします。
2. WordPressの 'Plugins' メニューからプラグインを有効化します。
3. Ultimate Multisite > Settings > Taxes に移動し、EU Vat を有効にします。
4. Tax Rates をクリックし、下部の EU VAT Rates を更新して Tax Rates を保存します。
5. 完了！
