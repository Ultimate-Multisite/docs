# サイトエクスポーター

## コンテンツ

ネットワーク内のサブサイトを単一サイトインストールに移動したり、逆に移動したりする必要があることを理解しています。お客様がウェブサイトを移行することを決定した場合です。サイトエクスポーターとインポーターのアドオンは、このタスクを簡単にします。このツールは、zipファイルを使用してサイトをエクスポートおよびインポートし、バックアップとしても機能します。

## 機能

* サイトのエクスポートとインポート
* 使いやすいインターフェース
* サイトのバックアップを作成

## ドキュメント
**サイトのエクスポートとインポート方法**

## インストール

1. **ultimate-multisite-site-exporter** を **/wp-content/plugins/** ディレクトリにアップロードします
2. WordPress の **Plugins** メニューからプラグインを有効化します
3. ウィザードに従ってプラグインを設定します

## 使用ガイド

Ultimate Multisite を使用すると、ネットワークからサイトをエクスポートしたり、マルチサイトまたはシングルサイトにサイトをインポートしたりできます。

## サイトのエクスポート方法

ネットワークからサイトをエクスポートするには、Ultimate Multisite メニューの **Export & Import** に移動し、**Export Sites** タブで **Generate new Site Export** をクリックします。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-_bBztobh1xIFoj0Zo3zo7bx842u3XFMgpc2b8kiHmGmBU6RjuxlSOSY4ZNzaWTPOJrb0TIW_Q0Dv9WvT0yD-WwOkZ7bR36Ij5aJa_l19cDTfuj9DADQqI5dGy3eIeUmoQA=s0)

エクスポートしたいサイトを検索し、エクスポートファイルに含める項目を選択します。テーマ、プラグイン、アップロードを含めることができます。サイトが大きすぎる場合は、バックグラウンドでエクスポートプロセスを実行するように選択できます。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-fk-i4RP0uZFj8kgfcQeRfAqWtCBelOjpOQARFfeBisVkeRSc1Qywq6Dpy66MnB6oBbnvdfJM359LdRSgqCLU1tNozo1u1OFH5svxtld8TZAXHY-CPGh92gWz9HQ8nlUcjg=s0)

**Export Site** ボタンをクリックします。エクスポートされたサイトは **Existing Exports** リストに表示されます。**Download ZIP** ボタンをクリックしてファイルをダウンロードします。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-mouH16ZCUuNaUg0PwF2JmMhhsKPJOztau21QIeB7DXqAcX4VI5FL_itxvnKmFo439xuGUWsntM3doG6yBkjS2gi6bNTjg-kLHC19m19br5xsgBl0XH8hy5LvuPN1uNtx9g=s0)

## サイトのインポート方法

サイトをインポートするには、**Import a Site** タブに移動し、インポートしたいサイトの .zip ファイルをアップロードします。URL を新しいものに変更するか、空欄のままにして古い URL を保持できます。また、インポート後に .zip ファイルをインストールから削除するチェックボックスをオンにできます。次に **Begin Import** をクリックします。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-Uq6wNJ4TPTuAtSJqKunxSJEn5cli7vbwYGgL1G3fcHwdvU7gSwW_J4xlBAtZ1fyVbDU7Pupi8uNFhw1Lzm4S2fjVTkbQoz7fiZPIc1qiy8qj3iGt-MNVlyLWmIZXFxPLYA=s0)

処理には少し時間がかかる場合があります。インポーターの下にあるリストで保留中のインポートが表示されます。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-kMkwSY3bYrE5vjfJtLviYImhNBn_PwmuamaRDlQKS9rGwIy3a_jXV0glMNTU8fBmV6IKkNN7obXNQFqNaAJtAAbWDftzX76-qdCDKCrqdru_9taciFCayE9FNszZJTODvA=s0)

**Cancel Import** をクリックして処理を停止できます。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-oGmtCq7br6Y9GAfw_Ch53TP8nkJ0hw77sQJpSCQ7fEj5qk2lJo3WuMl7NYqB9c8SmORis53bWHILwGLxUTbpv4trWhxibu-yu5DNu6yNWXfkRo0bcnAZJ5F2K1hYIMH6PQ=s0)

処理が完了すると、サイトはすでにネットワーク上で利用可能になります。
