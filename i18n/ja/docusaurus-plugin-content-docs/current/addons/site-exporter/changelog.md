# Site Exporter Changelog

Version: 1.0.4 - リリース日 2025-11-25

* Ultimate Multisite 2.4との互換性を修正。

Version: 1.0.2 - リリース日 2025-09-28

* プレフィックスを ultimate-multisite に変更し、テキストドメインを更新し、バージョンを上げました。

### Version 1.0.1 - リリース日 2023-08-09

* 改善: インポーター ファイルの正しいファイルパスが取得されるようになりました。
* 修正: インポート中のエラーを防ぐためにオブジェクトキャッシュを無効化しました。
* 修正: インポーター プラグインのダウンロードリンクが機能しない問題を修正しました。

### Version 1.0.0 - リリース日 2022-12-23

* 修正: エクスポートフォームのポップアップが読み込まれない問題を修正しました;
* 修正: エクスポートページで JavaScript ファイルが読み込まれない問題を修正しました;
* 内部: 新しいビルドプロセスを追加しました;

### Version 1.0.0-beta.4 - リリース日 2022-01-13

* 内部: フックとフィルタ生成器を追加しました;
* 内部: 開発者の作業効率向上のために Ultimate Multisite スタブを追加しました;
