# 従量課金プラン

The Ultimate Multisite: Metered Plans addon は、柔軟な従量課金機能を導入することで、顧客への請求方法を変革します。超過時にユーザーをブロックする硬直した上限ではなく、追加使用量に対して自動で請求する超過料金を有効にします。

## 主な機能

* **従量課金**: 固定階層ではなく実際の使用量に応じて顧客に課金します
* **超過料金管理**: プラン上限を超える使用量に対して自動で請求します
* **使用量追跡**: 顧客のリソース消費をリアルタイムで監視します
* **柔軟な価格設定**: ストレージ、帯域幅、サイト数など、さまざまなリソースに対してカスタム料金を設定します
* **自動請求書発行**: 使用量しきい値を超えた際に自動で請求書を生成します
* **使用量レポート**: 顧客と管理者向けの詳細な使用量分析とレポートを提供します

## 適した用途

* **ホスティングプロバイダー**: 顧客のニーズに合わせて拡張可能なプランを提供します
* **SaaSプラットフォーム**: 実際のリソース消費に応じて請求します
* **エージェンシーネットワーク**: クライアントプロジェクト向けに柔軟な価格モデルを提供します
* **リセラーホスティング**: パートナーが使用量ベースの価格設定を提供できるようにします

## 動作概要

1. 従量課金リソース（ディスク容量、帯域幅、サイト数など）を設定します  
2. 基本プランの上限と超過料金を設定します  
3. システムはネットワーク全体で使用量を自動的に追跡します  
4. 顧客が上限を超えた場合、超過料金が計算されます  
5. 請求サイクルに基づいて請求書が自動的に生成されます  
6. 顧客はリアルタイムで使用量と今後の請求額を確認できます

## 対応リソース

* ディスクストレージ  
* 帯域幅/データ転送  
* サイト数  
* ユーザー数  
* カスタムリソースタイプ

## 要件

* Multisite Ultimate 2.0.0 以上  
* WordPress Multisite インストール  
* PHP 7.4 以上

## インストール

1. プラグインファイルを `/wp-content/plugins/ultimate-multisite-metered-plans` にアップロードします  
2. WordPress の「プラグイン」画面からプラグインを有効化します  
3. Multisite Ultimate → 設定 → 従量課金プラン に移動します  
4. 従量課金リソースと価格設定を構成します  
5. 商品編集画面で商品に従量課金を有効にします

## よくある質問

### 顧客が上限を超えた場合はどうなります？

アクセスをブロックする代わりに、システムは超過分を追跡し、設定した料金に基づいて次回の請求書に追加します。

### 異なる商品ごとに異なる超過料金を設定できますか？

はい、各商品は独自の従量課金リソースと超過料金を設定できます。

### 顧客は使用量についてどのように通知されますか？

顧客はダッシュボードで現在の使用量を確認でき、上限に近づいたり超過した際にオプションの通知を受け取ることができます。

### まだハードリミットを使用できますか？

はい、従量課金は商品ごとにオプションです。従来のハードリミット商品と従量課金商品を組み合わせることができます。
