# 従量制プラン変更履歴

バージョン 1.1.0 - 2026-05-05 リリース
- 新機能: マルチサイトのサブサイト向け AI トークン課金 — 顧客サイト全体の AI トークン使用量を追跡し、設定可能なトークン単価で課金
- 新機能: コネクターの適用処理を、動的な上限検出と書き込み反映で再構築し、すべてのコネクターでリアルタイムの正確性を確保
- 修正: データベーステーブルのアップグレードが BerlinDB スキーマ定義と正しく一致するようになり、新規インストール時のアップグレード失敗を防止
- 修正: データベースアップグレードのコールバックを正しい形式に変換し、無言で失敗するアップグレードを解決
- 修正: AI 使用量超過マークアップ入力フィールドで小数値を受け付けるように変更
- 修正: プラグイン読み込み時の致命的エラーと二重初期化の問題を解決
- 改善: 開発者環境が初回実行時に自己設定されるよう、check-env npm スクリプトを追加
