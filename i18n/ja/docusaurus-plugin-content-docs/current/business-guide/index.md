# Ultimate Multisiteでニッチなウェブサイトビジネスを構築する {#build-a-niche-website-business-with-ultimate-multisite}

ウェブサイト販売を試みるほとんどの人は、WixやSquarespace、何千ものフリーランサーと価格競争を強いられます。そして、その競争では負けてしまいます。勝つ人は、特定のターゲット層（オーディエンス）を選び、その層に合わせたソリューションを構築し、提供する価値に対して対価を請求します。

このコースでは、Ultimate Multisiteを使って、まさにその方法を学ぶことができます。

## これから構築するもの {#what-you-will-build}

このコースを通じて、あなたは**FitSite**という実在し、機能するネットワークを構築します。FitSiteは、フィットネススタジオ、パーソナルトレーナー、ジムオーナーのために特別に作られたウェブサイトプラットフォームです。

最終レッスンまでに、FitSiteは以下の要素を備えた完全なニッチプラットフォームになります。

- フィットネスビジネス向けに設計された目的特化型のサイトテンプレート
- フィットネスビジネスの実際の運営方法に合わせたプランの階層
- ブランド化されたチェックアウトおよびオンボーディングフロー
- 市場が支払う価格に合わせた価格設定
- フィットネス業界の顧客を見つけ、維持するための戦略

あなたはステップバイステップで進めていくため、コースの終わりには、実際に機能するプラットフォームと、選んだどんなニッチに対してもプロセスを再現するための知識の両方を得ることができます。

## なぜフィットネススタジオなのか？ {#why-fitness-studios}

誰もが真似できる具体的な例が必要でした。フィットネススタジオがビジネスとして成立している理由は以下の通りです。

- **誰もがビジネスを理解している** — 追従するために専門知識は必要ありません
- **ウェブサイトのニーズが明確である** — クラススケジュール、トレーナーのプロフィール、予約、所在地情報など
- **サービスが手薄な分野である** — 汎用的なウェブサイトビルダーでは、彼らの特定の課題は解決できません
- **自然なプランの階層が存在する** — 個人トレーナーと、複数の場所を持つジムチェーンでは、必要なものが異なります

FitSiteを構築する過程で学ぶことは、レストラン、不動産エージェント、法律事務所、教会、工務店、またはその他のあらゆる垂直分野（ニッチ）に直接応用できます。

## 事前準備 {#prerequisites}

- WordPress Multisiteのインストール（またはそれを行う意欲）
- 有効なUltimate Multisiteライセンス

コーディングは必要ありません。Ultimate Multisiteを初めて使う方でも大丈夫です。レッスン3はクリーンなインストールから始まります。
