# レッスン 4：ニッチなテンプレートの構築

テンプレートは、あなたのニッチな価値提案の核となる部分です。フィットネススタジオのオーナーがサインアップし、すでにクラススケジュール、トレーナーのプロフィール、適切な画像など、フィットネスサイトのようなデザインのサイトを見た場合、何も書かれていない「空のキャンバス」を見た場合よりも、滞在してくれる可能性がはるかに高くなります。

## ここまでの進捗 {#where-we-left-off}

Ultimate Multisiteをインストールし、設定した、動作するFitSiteネットワークが手元にあります。次に、FitSiteがフィットネスビジネス向けに特別に作られたように感じられるテンプレートを構築します。

## ニッチなテンプレートが重要な理由 {#why-niche-templates-matter}

汎用的なテンプレートは、お客様に難しい作業を強いてしまいます。つまり、必要なページが何か、どのようなコンテンツ構造が機能するか、そして業界に合わせてどのように見せるかを、お客様自身に考えてもらう必要があるのです。ニッチなテンプレートは、その手間（摩擦）を排除します。

FitSiteにサインアップしたフィットネススタジオのオーナーが目にするべきものは以下の通りです。

*   フィットネススタジオのウェブサイトのようなホームページ
*   クラス、スケジュール、トレーナー、料金設定のためのページがすでに作成されていること
*   業界に合った画像とプレースホルダーコンテンツ
*   プロフェッショナルで、フィットネスに特化したブランドイメージを持つデザイン

お客様は、そこに自分の詳細を埋めるだけで済みます。ゼロから作り始める必要はありません。

## テンプレートの計画 {#planning-your-templates}

構築を始める前に、どのようなテンプレートを提供するのかを決めましょう。FitSiteでは、以下の3種類を作成します。

### テンプレート 1：スタジオ必須（Studio Essential） {#template-1-studio-essential}

小規模なスタジオや個人トレーナー向け。

*   ヒーロー画像、クラスのハイライト、行動を促す要素（CTA）を備えた**ホームページ**
*   スタジオのストーリーとミッションを伝える**「会社概要」ページ**
*   スケジュールレイアウトを備えた**「クラス」ページ**
*   プロフィールカードを備えた**「トレーナー」ページ**
*   所在地マップとフォームを備えた**「お問い合わせ」ページ**
*   フィットネスに適した配色を用いた、クリーンでモダンなデザイン

### テンプレート 2：ジムプロ（Gym Pro） {#template-2-gym-pro}

より多くのサービスを提供する、確立されたジム向け。

*   スタジオ必須のすべての要素に加え、以下を含む：
*   料金表を備えた**「会員制度」ページ**
*   施設写真のための**「ギャラリー」ページ**
*   フィットネスのヒントやニュースのための**「ブログ」セクション**
*   ホームページの**「お客様の声」セクション**
*   **より目立つブランディング**オプション

### テンプレート 3：フィットネスチェーン（Fitness Chain） {#template-3-fitness-chain}

複数の場所（ロケーション）を持つ運営拠点向け。

*   ジムプロのすべての要素に加え、以下を含む：
*   複数の施設リストを備えた**「ロケーション」ページ**
*   **フランチャイズ/ロケーション**のサブページテンプレート
*   ロケーション固有の詳細を含む**一元化されたブランディング**
*   全ロケーションを網羅した**スタッフディレクトリ**

## テンプレートサイトの構築 {#building-a-template-site}

Ultimate Multisiteにおいて、テンプレートとは、新しい顧客サイトがどのような見た目になるように設定されたWordPressサイトのことです。作成手順は以下の通りです。

### ステップ 1：テンプレートサイトの作成 {#step-1-create-the-template-site}

1.  ネットワーク管理画面の **Sites > Add New** に移動します。
2.  `template-studio-essential` という名前のサイトを作成します。
3.  このサイトが、作業を行うためのキャンバスとなります。

### ステップ 2：テーマのインストールと設定 {#step-2-install-and-configure-the-theme}

テンプレートサイトのダッシュボードに切り替え、以下の作業を行います。

1.  フィットネスビジネスに適したテーマを有効化します。
2.  テーマの設定、色、タイポグラフィを設定します。
3.  フィットネスに適したナビゲーションを備えたヘッダーとフッターを設定します。

:::tip テーマの選択
フィットネス向けに見栄えがするが、お客様が管理できないほど複雑ではない、柔軟性の高いテーマを選びましょう。Astra、GeneratePress、またはKadenceのようなテーマは、軽量でカスタマイズ性が高く、サポートも充実しているためおすすめです。
:::

### ステップ 3：ページの作成 {#step-3-create-the-pages}

以下の要素を用いて、各ページを構築します。

*   フィットネスに関する自然な文章の**プレースホルダーコンテンツ**（「Lorem ipsum」ではなく「[スタジオ名]へようこそ」といった内容）
*   フィットネス活動を示す無料のストックフォトサイトからの**プレースホルダー画像**
*   ページビルダーまたはブロックエディターを使用した**機能的なレイアウト**

プレースホルダーコンテンツは、可能な限り指示的なものにしましょう。単なる汎用的なダミーテキストではなく、「ここに、スタジオの簡単な説明と、どのような点が特別かを記述してください。トレーニングの哲学、経験年数、またはクライアントが期待できることなどを記載しましょう。」といった具体的な指示を書きましょう。

### ステップ 4：プラグインの設定 {#step-4-configure-plugins}

フィットネススタジオが必要とするプラグインをインストールし、有効化します。

*   予約またはスケジューリングプラグイン（プランティアに適用される場合）
*   お問い合わせフォームプラグイン
*   SEOプラグイン（フィットネス関連のデフォルト設定を事前に行う）

### ステップ 5：テンプレートとしてのマーク付け {#step-5-mark-as-template}

1.  **Ultimate Multisite > Sites** に移動します。
2.  テンプレートサイトを編集します。
3.  **Site Template** トグルを有効にします。

提供したいテンプレートごとに、このプロセスを繰り返します。

## テンプレート品質チェックリスト {#template-quality-checklist}

テンプレートを公開する前に、以下の点を確認してください。

- [ ] すべてのページが正しく読み込まれ、プロフェッショナルに見える
- [ ] プレースホルダーコンテンツが役立ち、ニッチに特化している
- [ ] 画像が適切で、適切なライセンスを持っている
- [ ] すべてのページでモバイルレスポンシブが機能する
- [ ] ナビゲーションが論理的で完全である
- [ ] お問い合わせフォームが機能する
- [ ] 壊れたリンクや欠落したアセットがない
- [ ] ページ読み込み速度が許容範囲内である
