# サブスクリプションを解約するには？

Ultimate Multisite ライセンスをお持ちの場合、いつでも更新をキャンセルできます。以下の手順に従ってください。


<!-- Screenshot unavailable: Account page email with access link -->

購入時に入力したメールアドレスとパスワードを使用します。

<!-- Screenshot unavailable: Login page with email and password fields -->

これは Account ページの初期画面です。

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

サブスクリプションを解約するには、左側のメニューで _更新と請求_ に移動します。

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

右側の矢印をクリックしてサイドウィンドウを開きます。次に、**自動更新**をキャンセルするオプションを選択してください。

システムに確認メッセージが表示されます。

サブスクリプションを解約することに間違いがなければ、_**更新をキャンセル**_ ボタンをクリックしてください。この操作の後、簡単なアンケートへの回答を求められます。

完了です！サブスクリプションは自動更新されません。

**サブスクリプションの有効期限が切れるまで、有効なキーを利用できます**。サブスクリプションを再有効化したい場合は、手動で行う必要があります。
