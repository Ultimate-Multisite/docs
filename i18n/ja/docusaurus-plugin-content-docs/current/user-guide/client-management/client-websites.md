# クライアントのWebサイト（v2） {#your-clients-websites-v2}

ネットワークに登録した後、顧客は自分のDashboardにアクセスできます。

**Account**ページでは、支払い、メンバーシップ、ドメイン、プラン制限などに関する重要情報を確認できます。さらに、**マイサイト**ページにもアクセスでき、すべてのWebサイトを確認して管理できます。Dashboardでマイサイトをクリックするだけです。

![顧客Dashboard内のマイサイトリンク](/img/account-page/my-sites-link.png)

顧客のすべてのWebサイトが表示され、それぞれの直下に**管理**ボタンがあります。クリックすると、**その特定のWebサイトのDashboard**にリダイレクトされます。

複数のWebサイトを持てるプランでは、どのWebサイトが**主**サイトかも確認できます。

![管理ボタン付きで顧客のWebサイトを表示しているマイサイトページ](/img/account-page/my-sites-page.png)

サイト編集インターフェースの全体表示は次のとおりです。

![サイト編集の全体インターフェース](/img/admin/site-edit-full.png)

主Webサイトを変更したい場合は、**Account**メニューをクリックしてAccountページに移動する必要があります。

![顧客Dashboard内のAccountメニューリンク](/img/account-page/account-menu.png)

顧客の詳細を表示して編集することもできます。

![顧客編集インターフェース](/img/admin/customer-edit.png)

次に、**既定サイトを変更**をクリックします。

![Accountページの既定サイトを変更ボタン](/img/account-page/change-default-site.png)

ポップアップウィンドウが表示されます。ドロップダウンリストをクリックし、主WebサイトにしたいWebサイトを選択します。

![既定サイトを変更のドロップダウン選択](/img/account-page/change-default-site-dropdown.png)

最後に、**既定サイトを変更**をクリックすれば完了です。

![既定サイト変更の確認](/img/account-page/change-default-site-confirm.png)
