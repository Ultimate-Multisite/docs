# Why Ultimate Multisite

この記事にたどり着いたなら、私たちのプラグインがビジネスアイデアを実現する解決策になると聞いたことでしょう。この記事の終わりまでに必要な答えが得られることを願っています。

## What is Ultimate Multisite?

Ultimate Multisiteは、顧客にWaaS（Website as a Service）を提供できるWordPress Multisiteプラグインです。プラグインを使用すると、さまざまなサブスクリプション階層を設定し、顧客に定期的な料金を支払ってもらい、Multisiteネットワーク上にサイトをホストしてもらうことができます。

Ultimate Multisiteは、WooCommerceやStripeなどのさまざまな決済ゲートウェイで支払いを受け取り、ネットワークに新規ユーザーが登録した際に作業を大幅に簡素化するサイトテンプレートを作成し、ドメインをマッピングし、登録フォームをカスタマイズして顧客に合わせた体験を提供します。

そして、アドオンやWebhookで機能を拡張できるため、これが始まりにすぎません。

## Other features

Ultimate Multisite also offers:

  * Domain Mapping integration with Managed WordPress hosting providers: Ultimate Multisiteは、新しいドメインがマッピングされると、WP Engine、Cloudways、cPanel、RunCloud、Closteなどの管理パネルに自動的にマッピングドメインを追加します。手動で行う必要はありません。場合によっては、SSL証明書の取得も自動でトリガーします。

  * Webhooks: Ultimate Multisiteは強力なWebhookシステムを備えており、さまざまなサービスとネットワークを統合できます。さらに、Zapierとの直接統合を提供しており、Zapierのプラットフォーム上で利用可能な1000以上のアプリやサービスにネットワークを接続できます。

  * Powerful Admin Pages to help your customers using your favorite page builder: Ultimate MultisiteにはAdmin Pages Creatorという便利なアドオンがあります。これにより、Elementor、Beaver Builder、Brizyのテンプレートを使用して、顧客のダッシュボードに完全にカスタマイズされた管理ページを追加できます。

  * Two different Admin Themes to customize the dashboard experience of your customers: Material WPとPROをご確認ください。

And more advanced stuff:

  * Support to Template Overrides: Ultimate Multisiteを使用すると、コアプラグインファイルを変更することなく、サインアップフローのデフォルトHTMLテンプレートを独自のバージョンに置き換えることができます。

  * Search and Replace API: サイトテンプレートに{{site_name}}のようなプレースホルダーを注釈し、検索と置換 APIを使用して、サインアップ直後に顧客から提供された情報で自動的にプレースホルダーを埋めることもできます。

## Use cases

次に、Ultimate Multisiteの可能な利用ケースをいくつか見てみましょう。

### Case 1: An Agency

エージェンシーの主なスキルは、ホスティングやマーケティングなどの追加サービスとしてリストされることが多いウェブサイトのデザインにあります。  
エージェンシーにとって、Ultimate Multisiteは、単一のプラットフォーム上で複数のウェブサイトをホストおよび管理できるという点で、驚くべき価値提案を提供します。さらに、GeneratePress、Astra、OceanWPなどの特定のテーマでデザインを標準化しているエージェンシーは、Ultimate Multisiteの機能を活用して、新しいサイトごとにこれらのテーマを自動的に有効化できます。  
プラグインとテーマの厳密な管理は、Ultimate Multisiteの設定を通じて、プラグインとテーマを利用可能または非表示にし、新しいサイトでインスタンス化された際の有効化状態を制御することで、製品ごとに維持されます。  
エージェンシーは、Ultimate Multisiteが最も得意とすること、すなわち卓越したウェブサイトをデザインすることを可能にするため、安心感を得られます。

### Case 2: Niche Provider

「一つのことをやるなら、やり遂げる」という古い格言があります。多くの専門家にとって、これは単一のコアアイデアを中心に製品やサービスを作ることを意味します。  
ニッチなサイトネットワークの開発に注力している場合、共通のフレームワークとプラットフォームに基づくサービスを提供したいと考えるでしょう。そうしたときにサイトテンプレートが役立ちます！  
Ultimate Multisiteの革新的な機能の1つであるテンプレートサイトは、テーマがインストールされ有効化され、必要なプラグインがインストールされ有効化され、サンプル投稿やページが作成された状態です。顧客がテンプレートに基づいて新しいサイトを作成すると、テンプレートの内容と設定が新しく作成されたサイトにコピーされます。  
ニッチなサイトとサービスを提供するプロバイダーにとって、これはカスタムプラグインとデザインで即座に稼働可能なサイトを作成できるという比類のない優位性を提供します。顧客はサービスを完了するために最小限の入力だけを提供すればよいです。

### Case 3: WordPress Web Hosting

WordPressサイトをホストする方法は数多くありますが、事前にインストールされたWordPressバージョンを顧客に提供するだけで済むほど簡単なことはほとんどありません。これは、意味のあるサービスを提供するために多くの決定と考慮事項が統合される必要があるためです。  
Ultimate Multisiteは、この分野で優れた総合的なターンキーソリューションを提供し、WordPressサイトのホスティングを実現します。ソリューションには、サブスクリプションサービス、支払い収集、チェックアウトフォーム、割引クーポン、顧客コミュニケーションを提供するためのコアメカニズムが含まれます。  
WordPress Multisiteを正しくインストール、構成、維持するために必要な多くの重要な作業は、Ultimate Multisiteによって促進され、ネットワーク管理者はサービスやニッチに関連する製品階層、価格設定、サービスオファーなどの側面だけを考慮すれば済むようになります。  
Ultimate Multisiteとの統合を希望する開発者向けに、ソリューションはイベント通知のための包括的なRESTful APIとWebhookも提供します。  
多数の外部プラグインやライセンスに依存せず、Ultimate MultisiteはWix、Squarespace、[WordPress.com](https://WordPress.com)、その他に匹敵する機能豊富なソリューションを提供します。
