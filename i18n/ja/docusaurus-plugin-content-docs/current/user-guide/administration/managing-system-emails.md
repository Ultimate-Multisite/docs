# システムメールの管理

Ultimate Multisite は、サインアップ確認、支払い領収書、会員変更などのイベントに対してさまざまなシステムメールを送信します。これらのメールテンプレートをカスタマイズして、ブランドに合わせることができます。

## システムメールの表示

**Ultimate Multisite → Settings** に移動し、**Emails** タブをクリックしてメール関連の設定を確認します。

![設定 - メールタブ](/img/admin/settings-emails.png)

そこから、**System Emails** ページにアクセスして、すべてのメールテンプレートを確認できます。

![システムメール一覧](/img/admin/system-emails-list.png)

## 新しいメールテンプレートのインポート

新しい Ultimate Multisite バージョンやアドオンは、新しいメールテンプレートを登録する場合があります。競合を防ぐために、新しいテンプレートは自動的に追加されず、手動でインポートする必要があります。

新しいメールテンプレートをインポートするには:

1. **System Emails** ページに移動します
2. ページ上部にある **Reset & Import** ボタンをクリックします
3. **Import Emails** オプションを切り替えて、利用可能なテンプレートを表示します
4. インポートしたいテンプレートを選択し、確認します

## メールテンプレートのリセット

メールテンプレートをカスタマイズし、デフォルトの内容に戻したい場合、次の2つのオプションがあります：

1. **Delete and re-import** — システムメールを削除し、再度インポートします（送信メトリクスが消去されます）
2. **Reset via the tool** — **Reset & Import** ツールを使用し、**Reset** オプションを切り替えて、リセットしたいメールを選択します

Reset オプションは、送信メトリクスやその他のメタデータを保持しつつ、デフォルトの内容を復元するため、推奨されます。
