# システムメールの管理

Ultimate Multisite は、サインアップ確認、支払い領収書、メンバーシップ変更などのイベントに対して、さまざまなシステムメールを送信します。これらのメールテンプレートは、ブランドに合わせてカスタマイズできます。

## システムメールの表示 {#viewing-system-emails}

**Ultimate Multisite → Settings** に移動し、**Emails** タブをクリックすると、メール関連の設定を確認できます。

![設定 - Emails タブ](/img/admin/settings-emails.png)

そこから **System Emails** ページにアクセスして、すべてのメールテンプレートを確認できます。

![システムメール一覧](/img/admin/system-emails-list.png)

システムメールエディターを使用して、個別のメールテンプレートを編集することもできます。

![システムメールエディター](/img/admin/system-email-editor.png)

## 新しいメールテンプレートのインポート {#importing-new-email-templates}

新しい Ultimate Multisite のバージョンやアドオンで、新しいメールテンプレートが登録される場合があります。競合を防ぐため、新しいテンプレートは自動的には追加されません。手動でインポートする必要があります。

新しいメールテンプレートをインポートするには:

1. **System Emails** ページに移動します
2. ページ上部の **Reset & Import** ボタンをクリックします
3. **Import Emails** オプションを切り替えて、利用可能なテンプレートを表示します
4. インポートしたいテンプレートを選択して確認します

## メールテンプレートのリセット {#resetting-email-templates}

メールテンプレートをカスタマイズ済みで、既定の内容に戻したい場合は、2 つの方法があります。

1. **削除して再インポート** — システムメールを削除し、もう一度インポートします（これにより送信メトリクスは消去されます）
2. **ツールでリセット** — **Reset & Import** ツールを使用し、**Reset** オプションを切り替えて、リセットしたいメールを選択します

Reset オプションは、既定の内容を復元しながら送信メトリクスやその他のメタデータを保持するため、推奨されます。
