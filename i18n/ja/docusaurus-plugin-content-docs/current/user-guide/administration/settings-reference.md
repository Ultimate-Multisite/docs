# 設定リファレンス

このページでは、Ultimate Multisiteの日常的な管理や、最近の動作変更に影響を与える設定を追跡しています。

## その他のオプション

**その他のオプション**エリアは、**Ultimate Multisite > 設定 > ログインと登録**の下に表示されます。

| 設定 | 説明 |
|---|---|
| **Jumperを有効にする** | 管理エリアにJumperクイックナビゲーションツールを表示します。これを使用すると、Ultimate Multisiteの画面、ネットワークオブジェクト、およびサポートされている管理宛先に直接ジャンプできます。このショートカットを表示したくない場合は、無効にしてください。 |

## エラー報告とテレメトリ

以前のエラー報告のオプトイン設定は、設定ページから削除されました。匿名テレメトリは無効になっており、有効にするためのUIトグルはありません。

設定ページに関する内部の運用手順書やスクリーンショットを維持している場合は、管理者がもはや存在しない設定を探さないよう、古いエラー報告のオプトインフィールドへの言及を削除してください。

## インポート/エクスポート設定

**インポート/エクスポート**設定タブでは、どの設定を制御するかを説明し、サイトおよびネットワークのアーカイブについては**Ultimate Multisite > サイトのエクスポート**に直接リンクしています。インポート/エクスポートの構成には設定タブを使用し、単一サイトのエクスポート/インポートのワークフローには**ツール > エクスポートとインポート**を使用し、完全なネットワークエクスポートアーカイブが必要な場合はサイトエクスポートツールを使用してください。
