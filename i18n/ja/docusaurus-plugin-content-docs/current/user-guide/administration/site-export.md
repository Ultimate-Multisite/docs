# サイトのエクスポート

**Site Export** の管理画面では、ネットワーク管理者が単一のサイト、またはネットワーク全体を、移行、バックアップ、または引き渡し作業用のダウンロード可能なアーカイブとしてパッケージ化できます。

## 単一サイトのエクスポート {#exporting-one-site}

**Ultimate Multisite > Site Export** に移動し、**Generate new Site Export** を選択します。エクスポートしたいサブサイトを選択し、アーカイブにアップロード、プラグイン、テーマを含めるかどうかを選択します。

エクスポートが完了したら、**Existing Exports** リストから ZIP をダウンロードしてください。エクスポートされた ZIP は、自己起動型の `index.php` と `readme.txt` を含むようになったため、別途インポータープラグインをインストールすることなく、新しいホストにアップロードして起動できます。

## ネットワーク全体の**エクスポート** {#exporting-the-whole-network}

ネットワーク内のすべてのサブサイトを含む単一のアーカイブが必要な場合は、Site Export ページで **Network Export** を使用します。これは、ホストの移行、災害復旧の訓練、または各サブサイトが一緒に移動する必要があるステージング環境の再構築の前に特に役立ちます。

ネットワークエクスポートは単一サイトのエクスポートよりもはるかに大きくなる可能性があるため、トラフィックが少ない時間帯に実行し、アップロード、プラグイン、テーマ、生成された ZIP ファイルを格納するためのターゲットストレージに十分な空き容量があることを確認してください。

### ネットワークインポートバンドル {#network-import-bundles}

Ultimate Multisite 2.12.0 以降、Site Exporter は、ネットワーク全体のスムーズなサイト復元を目的とした特殊なアーカイブである**ネットワークインポートバンドル**を生成できます。ネットワークインポートバンドルには、複数のサイトを新しいネットワークインストールに復元するために必要なすべてのファイルとメタデータが含まれています。

#### ネットワークインポートバンドルの生成 {#generating-a-network-import-bundle}

1. **Ultimate Multisite > Site Export** に移動します。
2. **Generate new Network Export** をクリックします。
3. エクスポートタイプとして **Network Import Bundle** を選択します。
4. アップロード、プラグイン、テーマを含めるかどうかを選択します。
5. **Generate** をクリックします。
6. **Existing Exports** リストからバンドル ZIP をダウンロードします。

#### ネットワークインポートバンドルからの復元 {#restoring-from-a-network-import-bundle}

ネットワークインポートバンドルからサイトを復元するには、以下の手順を実行します。

1. ターゲットホストに Ultimate Multisite をインストールします。
2. マルチサイトセットアップウィザードを完了します。
3. 新しいネットワークで **Ultimate Multisite > Site Export** に移動します。
4. **Import Network Bundle** をクリックします。
5. ネットワークインポートバンドル ZIP ファイルをアップロードします。
6. 画面の指示に従ってインポートを完了します。
7. インポートプロセスにより、すべてのサイト、そのコンテンツ、および設定が復元されます。

ネットワークインポートバンドルが保持するもの：
- サイトコンテンツ（投稿、ページ、カスタム投稿タイプ）
- サイトの設定とオプション
- ユーザーのロールと権限
- プラグインとテーマ（バンドルに含まれている場合）
- メディアのアップロード（バンドルに含まれている場合）
- カスタムデータベーステーブルとデータ

## 自己起動型エクスポート ZIP のインストール {#installing-a-self-booting-export-zip}

自己起動型の ZIP を新しいホストに復元するには：

1. エクスポートされた ZIP の内容をターゲットのウェブルートにアップロードします。
2. ブラウザでアップロードされた `index.php` を開きます。
3. バンドルされたエクスポートパッケージから、画面のインストーラーの指示に従います。
4. 一時ファイルを削除する前に、バンドルされた `readme.txt` を確認し、エクスポート固有の注意事項を確認してください。

アドオン固有のインストールおよびインポートの詳細については、[Site Exporter addon documentation](/addons/site-exporter) を参照してください。

Ultimate Multisite 2.9.0 で追加された単一サイトツールについては、[Export & Import](/user-guide/administration/export-import) を参照してください。
