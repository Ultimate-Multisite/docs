# メンバーシップの管理 {#managing-memberships}

Ultimate Multisite では、ネットワーク上のメンバーシップを管理し、すべてのサブスクリプションにアクセスできます。プランや商品、サイト数、請求金額と頻度、支払い履歴、タイムスタンプなどのサブスクリプション詳細を確認・編集できます。

このページにアクセスするには、Ultimate Multisite メニューの **メンバーシップ** タブに移動します。

![メンバーシップ一覧ページ](/img/admin/memberships-list.png)

メンバーシップページでは、顧客と、その Account に関連付けられた商品の一覧を確認できます。メンバーシップはステータスで絞り込めます: 有効、保留中、一時停止中、期限切れ、キャンセル済み。

無料メンバーシップは **無期限** メンバーシップとして扱われます。有効期限日は設定されず、自動的に期限切れになることはありません。無料プランを試用期間後または固定期間後に終了させる必要がある場合は、試用期間付きの有料商品を使用するか、メンバーシップを手動でキャンセルするか、アクセスを変更すべきタイミングで顧客を別のプランに移動してください。

メンバーシップをクリックすると、請求情報、サイトまたは訪問数の上限など、メンバーシップ詳細を編集できるページにアクセスできます。

![メンバーシップ編集画面](/img/admin/membership-edit.png)

メンバーシップ編集ページの全体表示は次のとおりです:

![メンバーシップ編集画面の全体表示](/img/admin/membership-edit-full.png)

ネットワークに以前登録されたユーザーに関連付けた新しいメンバーシップを追加することもできます。**メンバーシップを追加** をクリックし、詳細を設定します。

![メンバーシップ追加モーダル](/img/admin/membership-add-modal.png)

次に **メンバーシップを作成** をクリックします。新しいメンバーシップがメンバーシップ一覧に表示されます。
