# 会員管理

Ultimate Multisite では、会員を管理し、ネットワーク上のすべてのサブスクリプションにアクセスできます。プランや商品、サイト数、請求金額と頻度、支払履歴、タイムスタンプなどのサブスクリプション詳細を確認・編集できます。

このページにアクセスするには、Ultimate Multisite メニューの **Memberships** タブへ移動してください。

![Memberships list page](/img/admin/memberships-list.png)

Memberships ページでは、顧客一覧とそれぞれのアカウントに関連付けられた商品を確認できます。会員は、ステータス（active、pending、on hold、expired、cancelled）でフィルタリングできます。

会員をクリックすると、請求情報、サイトまたは訪問制限などの会員詳細を編集できるページにアクセスします。

以前にネットワークに登録されたユーザーに関連付けられた新しい会員を追加することもできます。**Add Membership** をクリックし、詳細を設定してください。

![Add Membership modal](/img/admin/membership-add-modal.png)

その後 **Create Membership** をクリックします。新しい会員が会員一覧に表示されます。
