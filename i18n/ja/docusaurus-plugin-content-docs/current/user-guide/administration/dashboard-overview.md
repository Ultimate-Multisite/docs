# Dashboard Overview

Ultimate Multisite には Ultimate Multisite メニューの下に専用の **Dashboard** があり、さらにネットワークダッシュボードにウィジェットを追加します。本記事では、両方の場所で表示される内容を紹介します。

## The Network Dashboard

Ultimate Multisite は **My Sites → Network Admin → Dashboard** にあるネットワークレベルのダッシュボードに便利なウィジェットを挿入します。

![Ultimate Multisite ウィジェット付きネットワークダッシュボード](/img/admin/network-dashboard-full.png)

各ウィジェットが提供する内容は次のとおりです：

**First Steps:** 新規インストール時に表示され、支払い方法の設定やテストアカウントの作成など、セットアップタスクの完了を促します。

![First Steps ウィジェット](/img/admin/network-dashboard-first-steps.png)

**Summary:** 1日のサインアップ数と収益を報告します。MRR（Monthly Recurring Revenue）は、収益を生むメンバーシップを持つ顧客全体の予測総額を示します。

![Summary ウィジェット](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite 全体のイベント（サインアップ、キャンセル、サイト作成、削除、その他の活動）を報告します。

![Activity Stream ウィジェット](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** ネットワーク内のユーザーとサイトの簡易サマリーを表示し、新しいサイトやユーザーを作成するショートカットを提供します。

![Right Now ウィジェット](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** 最新の Ultimate Multisite ニュース（アップデート、重大バグ、セキュリティ修正）を取得して表示します。

![News ウィジェット](/img/admin/network-dashboard-news.png)

## The Ultimate Multisite Dashboard

ネットワークダッシュボードはネットワーク全体の包括的情報を表示しますが、Ultimate Multisite ダッシュボード（Ultimate Multisite メニューのトップレベルに位置）はサービスに関する情報を表示します。

![Ultimate Multisite ダッシュボード](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** 12か月のカレンダー期間にわたる月次収益の内訳を表示します。新規収益は青で、キャンセルは赤で表示されます。

![MRR 成長チャート](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** 時間範囲でダッシュボードデータをフィルタリングし、必要な情報に集中します。日付範囲は下部のウィジェットに表示される内容を制御します。

![日付範囲とダッシュボード統計](/img/admin/um-dashboard-date-range.png)

ダッシュボードウィジェットには次のものがあります：

- **Revenue** — 期間中の総収益（MRR と製品別内訳付き）
- **New Memberships** — 選択した期間に作成された新規メンバーシップ数
- **Signups by Countries** — サインアップの国別地理的内訳
- **Signups by Forms** — 各チェックアウトフォーム経由でサインアップした顧客数と、管理者によって追加された数
- **Most Visited Sites** — ネットワーク上で最も訪問されたサイト（Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting で訪問カウントを有効にする必要があります）

![ダッシュボード統計](/img/admin/um-dashboard-stats.png)
