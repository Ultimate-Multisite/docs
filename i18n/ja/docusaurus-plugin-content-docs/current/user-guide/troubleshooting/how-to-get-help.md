# ヘルプの取得方法

有効なライセンスキーを取得すると、Ultimate Multisite の設定中に遭遇する可能性のある問題を解決するために、当社のチームへのアクセスが付随します。

このレッスンでは、ライセンス保有者が利用できるサポートチャネルと、サポートチームに連絡する際に必要となる情報について説明します。

## ヘルプサイト

以下のリンクからアクセスできるヘルプサイトで、さまざまな記事やチュートリアルにアクセスできます。

[)

当社では、ユーザーがUltimate Multisiteの基本機能や機能に慣れることを推奨しています。ヘルプサイトには「はじめに」記事があり、アドオンのインストール方法や決済ゲートウェイの統合方法など、一般的な質問への回答も掲載されています。

## サポートへの連絡

ヘルプサイトで解決策が見つからない場合は、ウェブサイトの右下にあるスピーチバブルアイコンをクリックするか、support@ultimatemultisite.com に直接メールを送信して、サポートチームに連絡できます。

技術的な問題でサポートに連絡する際は、問題の詳細な説明とともに、スクリーンショット、エラーメッセージ、または問題を迅速に理解し評価するのに役立つ情報を提供してください。また、ライセンスキーの提示も必要です。サポートは有効なライセンス保有者のみが対象です。

## コミュニティサイト

当社の[コミュニティサイト](https://community.ultimatemultisite.com/)は、Ultimate Multisite ユーザーが互いに交流できる場です。メンバーは投稿・ディスカッションへのコメント、アドバイスの提供、協力、クラウドソーシングなどを行える情報システムとして機能します。プライベートスペース/ページを利用するには、Ultimate Multisite のライセンスキーを提示する必要があります。
