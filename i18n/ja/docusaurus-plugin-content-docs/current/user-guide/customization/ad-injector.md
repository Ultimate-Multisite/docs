# Ultimate Multisite: Ad Injector でネットワークに広告を配置する

Ultimate Multisite: Ad Injector は、プランの編集ページから直接プランに広告を配置するためのツールを提供する **無料 add-on** です。

Ultimate Multisite: Ad Injector をインストールしたら、_Plugins_ ページに移動して有効化します。

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate ボタンをクリックします_

add-on が有効になると、プラン編集画面の _Advanced Options_ メタボックスに、_Ad Settings_ という新しいオプションが追加されます。

![プラン編集画面の Ad Settings](/img/config/product-edit-full.png)

_これで、各プランの広告設定を編集できます_

そのタブでは、フロントエンドの投稿とページについて、コンテンツの前後に表示する広告コードを追加できます。

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_広告コードを挿入すると、その特定のプランの投稿とページに表示されます。_

そのプランのユーザー向けに、Admin Pages のヘッダーにも広告を含めることができます。

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_ユーザーのバックエンドに広告を表示したい場合は、Admin Panel Ad Code オプションに移動します_

### 広告を超えた活用

Ultimate Multisite: Ad Injector を使って、上位プラン向けにサポートを提供することもできます。

_Admin Panel Ads_ オプションに移動し、サポートツールの HTML コードを追加するだけです。

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_特定のプランにカスタマーサポートを追加します_

この方法なら、ユーザーは自分の Admin Panel から直接あなたに連絡できます！
