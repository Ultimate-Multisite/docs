# テーマ対応型スキル {#theme-aware-skills}

Superdav AI Agent v1.10.0 には、アクティブな WordPress テーマに自動的に適応する、4つの新しいテーマ対応型組み込みスキルが搭載されました。これらのスキルは、お使いのテーマの構造や機能に合わせて、専門的なガイダンスと機能を提供します。

## テーマ対応型スキルとは何ですか？ {#what-are-theme-aware-skills}

テーマ対応型スキルとは、AIアシスタントが、現在サイトでアクティブになっている WordPress テーマに基づいて自動的に選択する、事前に設定された知識ベースとツールセットのことです。テーマを切り替えるたびに、利用可能なスキルが自動的に更新されるため、手動での設定は一切必要ありません。

各スキルには以下の内容が含まれています。

- **テーマ固有のドキュメント** — テーマの利用方法やカスタマイズ方法に関するガイダンス
- **ブロックとパターンの参照** — 利用可能なブロック、パターン、デザインオプション
- **カスタマイズの例** — 一般的なタスクのためのコードスニペットや設定パターン
- **ベストプラクティス** — テーマのアーキテクチャやワークフローに関する推奨事項

## 利用可能なテーマ対応型スキル {#available-theme-aware-skills}

### ブロックテーマ (Block Themes) {#block-themes}

**適用対象:** WordPress のブロックベース（Full Site Editing）アーキテクチャを使用するテーマ。

このスキルは、以下の点に関するガイダンスを提供します。

- ブロックエディターを使用したテンプレートの作成と編集
- ブロックパターンや再利用可能なブロックの扱い方
- グローバルスタイルや theme.json の設定のカスタマイズ
- テーマブロックやバリエーションの使用
- サイト専用のカスタムブロックパターンの構築

**自動的に有効になる場合:** アクティブなテーマがブロックテーマである場合（`block-templates` 機能に対応している場合）。

### クラスシックテーマ (Classic Themes) {#classic-themes}

**適用対象:** PHPテンプレートとクラシックエディターを使用する従来の WordPress テーマ。

このスキルは、以下の点に関するガイダンスを提供します。

- PHPテンプレートファイルやフックの扱い方
- カスタマイザーを介したテーマの外観のカスタマイズ
- ウィジェットエリアやサイドバーの使用
- CSSの修正や子テーマでの開発
- テーマの階層構造やテンプレートタグの理解

**自動的に有効になる場合:** アクティブなテーマがクラシック（非ブロック）テーマである場合。

### Kadence Blocks {#kadence-blocks}

**適用対象:** 高度なブロックベースのデザインに Kadence Blocks プラグインを使用しているサイト。

このスキルは、以下の点に関するガイダンスを提供します。

- Kadence の高度なブロックライブラリ（ヒーロー、お客様の声、価格設定など）の使用方法
- Kadence ブロックの設定やレスポンシブオプションの構成
- Kadence のグリッドブロックやコンテナブロックを使ったカスタムレイアウトの構築
- Kadence ブロックをテーマに統合する方法
- Kadence のデザインシステムやプリセットの活用

**自動的に有効になる場合:** サイトに Kadence Blocks プラグインがアクティブな場合。

### Kadence Theme {#kadence-theme}

**適用対象:** ブロックベースのデザインとカスタマイズに Kadence テーマを使用しているサイト。

このスキルは、以下の点に関するガイダンスを提供します。

- グローバルスタイルや theme.json を介した Kadence テーマのカスタマイズ
- Kadence の組み込みブロックパターンやテンプレートの使用
- Kadence テーマの設定やオプションの構成
- Kadence のデザインシステムを使ったカスタムデザインの構築
- 人気のプラグインやツールとの Kadence の統合

**自動的に有効になる場合:** アクティブなテーマが Kadence テーマである場合。

## スキルの選択方法 {#how-skills-are-selected}

アシスタントは、メッセージごとにアクティブなテーマとインストールされているプラグインを自動的に検出します。一致するテーマ対応型スキルが利用可能な場合、それは自動的にアシスタントのコンテキストに読み込まれます。手動でスキルを有効にしたり切り替えたりする必要はありません。

### 複数のスキル {#multiple-skills}

サイトに複数のスキルが適用される場合（例：Kadence Blocks と Kadence Theme の両方がアクティブな場合）、アシスタントは適用可能なすべてのスキルにアクセスでき、それぞれからのガイダンスを参照できます。

### テーマの切り替え {#switching-themes}

アクティブなテーマを変更すると、アシスタントの利用可能なスキルは次のメッセージで自動的に更新されます。例えば：

1. **Block Themes** スキルがアクティブなブロックテーマを使用している。
2. クラシックテーマに切り替える。
3. 次のメッセージでは、**Classic Themes** スキルが自動的に読み込まれ、**Block Themes** スキルは利用できなくなります。

## テーマ対応型スキルの利用方法 {#using-theme-aware-skills}

テーマ対応型スキルを活用するには、チャットインターフェースで「何をしたいか」を説明するだけで十分です。アシスタントが自動的に適切なスキルのガイダンスを参照します。

### プロンプトの例 {#example-prompts}

**ブロックテーマの場合:**
> 「ブロックパターンを使って、背景画像と中央揃えのテキストを持つヒーローセクションを作成したい。」

**クラスシックテーマの場合:**
> 「子テーマを使って、サイドバーにカスタムウィジェットエリアを追加したい。」

**Kadence Blocks の場合:**
> 「Kadence Testimonials ブロックを使って、お客様の声のセクションを構築したい。」

**Kadence Theme の場合:**
> 「ヘッダーのレイアウトとナビゲーションメニューのスタイルをカスタマイズしたい。」

アシスタントは、アクティブなテーマとプラグインに合わせて、テーマ固有のガイダンスとコード例を提供します。

:::note
テーマ対応型スキルは、Superdav AI Agent v1.10.0 以降で自動的に利用可能です。追加の設定や構成は必要ありません。
:::
