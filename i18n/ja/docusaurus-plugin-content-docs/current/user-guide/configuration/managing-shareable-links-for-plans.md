# プランの共有リンク管理

Ultimate Multisite では、プランリンクをクライアントと直接共有するオプションを提供しています。これを行うには、任意のプラン編集ページに移動し、_Click to copy the Shareable Link_ ボタンをクリックしてください。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-2DJoMvapXh.png)

_プラン編集ページで、共有リンクを作成するボタンが見つかります_

この仕組みは、リンクを作成する際にデフォルトの請求頻度を使用します。つまり、デフォルトの請求が月額の場合、最終的に次のようなリンクが生成されます: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

共有リンクを四半期ベースにしたい場合は、末尾に 3 を追加する必要があります

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

プラン _premium_ を年額で選択するには、1 を追加し、その後に単語を付けます:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

異なる請求頻度ごとに1つのリンクだけを持つことはできません。各請求頻度ごとに特定のリンクを共有する必要があります。

リンクを顧客と共有すると、サインアップフローの最初のステップにリダイレクトされます。
