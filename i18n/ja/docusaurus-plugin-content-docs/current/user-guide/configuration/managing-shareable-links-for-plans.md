# プランの共有可能リンクの管理

Ultimate Multisite では、プランのリンクをクライアントに直接共有できます。これを行うには、任意のプランの編集ページに移動し、_クリックして共有可能リンクをコピー_ボタンをクリックします。

![共有可能リンクボタンがある製品編集ページ](/img/config/product-edit-full.png)

_プランの編集ページでは、共有可能リンクを作成するボタンを見つけられます_

この仕組みでは、リンクの作成にデフォルトの請求頻度が使われます。つまり、たとえばデフォルトの請求が月次の場合、最終的に次のようなリンクになります: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

共有可能リンクのデフォルトを四半期ごとにしたい場合は、末尾に 3 を追加する必要があります

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

年次スケジュールでプラン _premium_ を選択するには、1 を追加し、その後に次の単語を追加します。

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

異なる請求頻度に対してリンクを 1 つだけ用意することはできません。各請求頻度ごとに特定のリンクを 1 つ共有してください。

リンクを顧客と共有すると、そのリンクは登録フローの最初のステップへリダイレクトします。
