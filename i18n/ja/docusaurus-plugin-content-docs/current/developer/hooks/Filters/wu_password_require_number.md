# フィルタ: wu_password_require_number

フィルタはパスワードに数値を要求するかどうかを決定します。

## パラメータ

| 名前 | タイプ | 説明 |
|------|------|-------------|
| $require | `bool` | 数値を要求するかどうか。ルールが適用される場合はデフォルトで true です。 |
| $defender_active | `bool` | Defender Pro Strong Password が有効かどうか。 |

### 以降

- 2.4.0
