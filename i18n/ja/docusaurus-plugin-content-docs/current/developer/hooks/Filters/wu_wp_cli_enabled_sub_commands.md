# フィルタ: wu_wp_cli_enabled_sub_commands

このエンティティで有効な **sub_commands** をフィルタリングします。

## Parameters

| 名前 | タイプ | 説明 |
|------|------|------|
| $sub_commands | `array` | デフォルトの **sub_commands**。 |
| $command_base | `string` | ルート直後にコマンドで使用されるベース。 |
| $this | `\Base_Manager` | オブジェクトインスタンス。 |

### Since

- 2.0.0
