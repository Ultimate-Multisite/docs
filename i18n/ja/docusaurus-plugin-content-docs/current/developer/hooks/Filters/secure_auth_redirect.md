# フィルタ: secure_auth_redirect

セキュア認証リダイレクトを使用するかどうかをフィルタします。

## パラメータ

| Name | Type | Description |
|------|------|-------------|
| $secure | `bool` | セキュア認証リダイレクトを使用するかどうか。デフォルトは false。 |
