# Filter: wu_independent_billing_cycle_product_types

独立した請求サイクルを持つ製品タイプをフィルタリングします。

独立した請求サイクルを持つ製品は、カート内でプランの請求期間に合わせる必要はありません。これは、プランが月額か年額かに関係なく、常に年額で請求されるドメイン登録のような製品に便利です。

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | 独立した請求サイクルを持つ製品タイプの配列。 |
