# フィルタ: wu_password_require_lowercase

パスワードに小文字を必須にするかどうかをフィルタします。

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | 小文字を必須にするかどうか。ルールが適用される場合はデフォルトで true です。 |
| $defender_active | `bool` | Defender Pro Strong Password が有効かどうか。 |
