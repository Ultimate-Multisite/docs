# フィルタ: wu_view_override_replaceable_views

開発者が置換可能リストに追加フォルダを追加できるようにします。

追加フォルダを許可すると、Ultimate Multisite バージョンの代わりに古いコピーが読み込まれる可能性があるため、注意してください。

## パラメータ

| 名前 | 型 | 説明 |
|------|------|-------------|
| $replaceable_views | `array` | 許可されたフォルダのリスト。 |
