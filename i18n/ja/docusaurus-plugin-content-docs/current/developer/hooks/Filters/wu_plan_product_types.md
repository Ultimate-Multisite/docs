# フィルタ: wu_plan_product_types

プランとして扱うべき製品タイプをフィルタします。

このフィルタは、アドオンが検証と分離でプランとして認識されるべき追加の製品タイプを登録できるようにします。

## Parameters

| 名前 | タイプ | 説明 |
|------|------|------|
| $plan_types | `array` | プランとして扱うべき製品タイプの配列。 |

### バージョン

- 2.3.0

### ソース

定義は [`inc/functions/product.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/product.php#L196) の 196 行目にあります
