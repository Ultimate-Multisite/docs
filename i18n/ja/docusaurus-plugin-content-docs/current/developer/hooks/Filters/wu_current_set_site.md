# フィルタ: wu_current_set_site

開発者がデフォルトの動作を変更し、現在のサイトを別の方法で設定できるようにします。

## パラメータ

| 名前 | タイプ | 説明 |
|------|------|-------------|
| $site | `\WP_Ultimo\Models\Site` | 設定する現在のサイトです。 |
| $ | `self` | Currentクラスのインスタンスです。 |

- 2.0.9
