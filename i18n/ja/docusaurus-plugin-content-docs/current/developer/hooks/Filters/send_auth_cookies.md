# Filter: send_auth_cookies

認証クッキーが実際にクライアントに送信されるのを防止することを許可します。

## Parameters

| 名前 | タイプ | 説明 |
|------|------|-------------|
| $send | `bool` | クライアントに認証クッキーを送信するかどうか。 |

### バージョン

- 4.7.4
