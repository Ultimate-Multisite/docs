# アクション: wu_page_before_render

プラグイン開発者がページを表示する前に追加コンテンツを挿入できるようにします。

## パラメータ

| 名前 | タイプ | 説明 |
|------|------|-------------|
| $page_id | `string` | このページのIDです。 |
| $page | `object` | ページオブジェクトです。 |

### バージョン

- 1.8.2

### ソース

定義場所 [`inc/admin-pages/class-base-admin-page.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/admin-pages/class-base-admin-page.php#L388) は 388 行目にあります
