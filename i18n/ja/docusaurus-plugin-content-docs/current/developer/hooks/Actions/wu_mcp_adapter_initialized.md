# アクション: wu_mcp_adapter_initialized {#action-wumcpadapterinitialized}

MCP アダプタが初期化された後に発火します。

他のプラグインやテーマが独自の機能を登録できるようにします。

## パラメータ {#parameters}

| 名前 | 型 | 説明 |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP アダプタのインスタンス。 |

### バージョン {#since}

- 2.5.0
