# アクション: wu_mcp_adapter_initialized

MCP アダプタが初期化された後に発火します。

他のプラグインやテーマが独自の機能を登録できるようにします。

## パラメータ

| 名前 | 型 | 説明 |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP アダプタのインスタンス。 |

### バージョン

- 2.5.0
