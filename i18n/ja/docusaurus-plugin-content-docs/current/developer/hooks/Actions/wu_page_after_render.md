# アクション: wu_page_after_render

プラグイン開発者がページを表示した後に追加コンテンツを追加できるようにします

## パラメータ

| 名前 | タイプ | 説明 |
|------|------|-------------|
| $page_id | `string` | このページのID |
| $page | `object` | ページオブジェクト。 |

### バージョン

- 1.8.2
