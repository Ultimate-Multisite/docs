# アクション: wu_site_created

最初にサイトが作成された後に発火します。  
テンプレートから複製された場合は発火しません。

## Parameters

| 名前 | タイプ | 説明 |
|------|------|-------------|
| $data | `array` | 保存されるオブジェクトデータです。 |
| $site | `\Site` | オブジェクトインスタンスです。 |

### バージョン

- 2.0.0
