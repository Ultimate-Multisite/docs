# Toiminto: wu_site_created {#action-wusitecreated}

Käynnistyy, kun sivusto luodaan ensimmäistä kertaa.

Ei käynnisty, jos sivusto on monistettu mallipohjasta.

## Parametrit {#parameters}

| Nimi | Tyyppi | Kuvaus |
|------|------|-------------|
| $data | `array` | Tallennettavat objektin tiedot. |
| $site | `\Site` | Objektin instanssi. |
