# Action: wp_ultimo_load

Käynnistyy, kun kaikki riippuvuudet on ladattu

Mahdollistaa plugin-kehittäjille uuden toiminnallisuuden lisäämisen. Esimerkiksi tuen uusille Hosting-palveluntarjoajille jne.
