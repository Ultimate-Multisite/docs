Salli plugin-kehittäjien lisätä mukautettuja aliaksia lomakkeen validaattoriin.

## Parametrit

| Nimi | Tyyppi | Kuvaus |
|------|------|-------------|
| $validation_aliases | `array` | Taulukko, jossa id =&gt; alias. |
| $checkout | `\Checkout` | Checkout-luokka. |
