# Filter: wu_contains_element

Salli kehittäjien muuttaa alkuperäisen haun tuloksia.

Tämä on hyödyllistä kolmannen osapuolen rakentajille ja vastaaville.

## Parametrit {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | Jos elementti sisältyy sisältöön. |
| $content | `string` | Tarkasteltava sisältö. |
| $element | `self` | Nykyinen elementti. |
| $post | `null\|\WP_Post` | tarkistettava post. |
