# Suodatin: wu_maybe_redirect_to_admin_panel_exclusion_list

Luo parametrien poissulkulista, joka estää automaattisen uudelleenohjauksen.

Tätä tarvitaan, koska muuten sivunrakentaja ei pysty muokkaamaan kirjautumissivua sen jälkeen, kun se on määritetty.
