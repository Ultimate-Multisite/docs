# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

Suodata tuotetyypit, joita tulisi käsitellä sopimuksina.

Tämä filter sallii lisäosien rekisteröidä lisää tuotetyyppejä, jotka tulisi tunnistaa sopimuksiksi validoinnissa ja erottelussa.

## Parametrit {#parameters}

| Nimi | Tyyppi | Kuvaus |
|------|------|-------------|
| $plan_types | `array` | Tuotetyyppien taulukko, joita käsitellään sopimuksina. |

### Alkaen {#since}

- 2.3.0
