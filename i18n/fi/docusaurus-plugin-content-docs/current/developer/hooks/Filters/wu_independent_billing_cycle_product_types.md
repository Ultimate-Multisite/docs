# Filter: wu_independent_billing_cycle_product_types

Suodata tuotetyypit, joilla on itsenäiset laskutusjaksot.

Tuotteita, joilla on itsenäiset laskutusjaksot, ei pakoteta vastaamaan ostoskorissa suunnitelman laskutuskautta. Tämä on hyödyllistä tuotteille, kuten verkkotunnusten rekisteröinneille, jotka laskutetaan aina vuosittain riippumatta siitä, onko suunnitelma kuukausittainen vai vuosittainen.

## Parametrit {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Taulukko tuotetyypeistä, joilla on itsenäiset laskutusjaksot. |

- 2.5.0
