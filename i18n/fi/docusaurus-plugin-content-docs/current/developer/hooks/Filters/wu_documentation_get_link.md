# Suodatin: wu_documentation_get_link

Salli plugin-kehittäjien suodattaa linkkejä.

En ole varma, miten siitä voisi olla hyötyä, mutta siitä ei ole haittaakaan

## Parametrit

| Nimi | Tyyppi | Kuvaus |
|------|------|-------------|
| $link | `string` | Rekisteröity linkki |
| $slug | `string` | Slug, jota käytetään linkin hakemiseen |
| $default_link | `string` | Rekisteröity oletuslinkki |

### Alkaen

- 1.7.0
