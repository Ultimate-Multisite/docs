# Suodatin: wu_add_new_domain_modal_fields

Suodattaa uuden domainin lisäämisen modaalin kentät.

Mahdollistaa lisäosille (esim. Domain Seller) domainin rekisteröintikenttien muokkaamisen tai korvaamisen.

## Parametrit {#parameters}

| Nimi | Tyyppi | Kuvaus |
|------|------|-------------|
| $fields | `array` | Lomakkeen kentät. |
