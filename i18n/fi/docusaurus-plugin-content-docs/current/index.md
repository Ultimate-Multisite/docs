# Ultimate Multisite -dokumentaatio

Tervetuloa Ultimate Multisite -dokumentaatioon. Ultimate Multisite muuntaa WordPress Multisite -ympäristön täydelliseksi Website as a Service (WaaS) -alustaksi.

## Dokumentaation osiot {#documentation-sections}

### [Käyttöopas](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Kaikki, mitä tarvitset WaaS-alustasi käyttöönottoon, määrittämiseen ja hallintaan. Kattaa asennuksen, maksuyhdyskäytävät, domain-kartoituksen, asiakashallinnan ja paljon muuta.

### [Kehittäjädokumentaatio](/developer/hooks) {#developer-documentation}

Tekninen viite kehittäjille, jotka rakentavat Ultimate Multisite -alustan päälle. Sisältää hooks-viitteen, REST API -dokumentaation ja addon-kehitysoppaat.

### [Addonit](/addons) {#addons}

Dokumentaatio maksullisille addon plugin -laajennuksille, jotka laajentavat Ultimate Multisite -alustaa lisäominaisuuksilla, kuten WooCommerce-integraatiolla, analytiikalla, domainien myynnillä ja muulla.
