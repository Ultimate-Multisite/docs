# Suodatin: wp_ultimo_site_exporter_url

Antaa kehittäjille mahdollisuuden hallita suhteellista polkua pluginin juureen.

## Parametrit

| Nimi | Tyyppi | Kuvaus |
|------|------|-------------|
| $dir | `string` | Polku suhteessa pluginin juureen, johon haluat päästä. |

### Alkaen versiosta

- 0.0.1
### Lähde

Määritelty tiedostossa [`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-helper.php#L52) rivillä 52


## Palauttaa
Polun suhteessa pluginin juureen.
