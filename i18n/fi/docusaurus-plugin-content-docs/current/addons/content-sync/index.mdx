# Sisällön synkronointi

Multisite Ultimate: sisällön synkronointi

Tämä Multisite Ultimate -lisäosa antaa verkon ylläpitäjille mahdollisuuden synkronoida sisältöä miltä tahansa sivustolta muille verkon sivustoille. Se tukee kaikkia artikkelityyppejä, mukaan lukien artikkelit, sivut, mukautetut artikkelityypit ja Elementor-mallit. Lisäosa sisältää monivaiheisen ohjatun käyttöliittymän sisällön ja kohdesivustojen valintaan, eräajon sekä reaaliaikaisen edistymispalautteen.

## Ominaisuudet

* Sisällön valinta: Visuaalinen korttiruudukko pienoiskuvilla ja artikkelityypin suodatuksella
* Kaikki artikkelityypit: Synkronoi artikkelit, sivut, tuotteet, Elementor-mallit ja minkä tahansa mukautetun artikkelityypin
* Sivuston valinta: Valitse, mille sivustoille sisältö synkronoidaan, lähdesivuston poissulkemisella ja alasivustojen esivalinnalla
* Eräajo: Synkronoi 10 sivustolle kerrallaan edistymispäivitysten kanssa
* Median synkronointi: Kopioi ja uudelleenkohdistaa automaattisesti mediatiedostot (kuvat, videot) artikkelin sisällöstä kohdesivustoille
* Taksonomian synkronointi: Säilyttää kategoriat, avainsanat ja mukautetut taksonomiat
* Elementor-tuki: Täysi tuki Elementor-malleille CSS:n uudelleenluonnilla ja mallin ID-tunnusten uudelleenkohdistuksella
* Ristiriitojen ratkaisu: Mahdollisuus korvata olemassa oleva sisältö tai ohittaa se
* Välimuistin hallinta: Tyhjentää automaattisesti Elementor-välimuistin kohdesivustoilla synkronoinnin jälkeen

## Vaatimukset

* WordPress 5.3 tai uudempi
* PHP 7.4 tai uudempi
* Multisite Ultimate -lisäosa (aktiivinen)
* Elementor-lisäosa (valinnainen, vaaditaan Elementor-mallien synkronointiin)

## Asennus

1. Lataa 'ultimate-multisite-content-sync' hakemistoon '/wp-content/plugins/'
2. Aktivoi lisäosa WordPress-verkon hallinnan 'Lisäosat'-valikon kautta
3. Määritä asetukset kohdassa Ultimate Multisite → Asetukset → Sisällön synkronointi
4. Siirry kohtaan Ultimate Multisite → Sisällön synkronointi aloittaaksesi synkronoinnin
