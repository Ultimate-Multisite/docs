| Nimi | Tyyppi | Kuvaus |
|------|------|-------------|
| $ | `array\|bool` | Luettelo network ID:istä tai false, jos käyttäjällä ei ole networkeja. |
| $ | `int` | User ID, jolle networkit tulee palauttaa. |
