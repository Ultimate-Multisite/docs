# Fluent Forms

Tämä lisäosa mahdollistaa rajojen lisäämisen Fluent Formsille Multisite Ultimate -ympäristössä tuotteissa, jäsenyyksissä tai sivustoissa.
Fluent Forms käyttää mukautettuja tauluja mukautettujen artikkelityyppien sijaan, joten tavalliset rajoitusvaihtoehdot eivät toimi Fluent Formsin kanssa.

## Ominaisuudet {#features}

- Lisää rajoja Fluent Formsille tai Conversational Fluent Formsille
- Asiakkaat saavat ystävällisen viestin, joka kehottaa heitä päivittämään tilauksensa, kun he saavuttavat rajan
