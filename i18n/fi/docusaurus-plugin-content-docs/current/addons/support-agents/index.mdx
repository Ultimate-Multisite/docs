# Tukiedustajat

Kokonaisen verkoston hallinta yksin ei ole helppo tehtävä. Kun verkostosi alkaa kasvaa, alat huomata tuen kysynnän lisääntyvän. On hyvä ajatus ottaa mukaan ihmisiä, jotka auttavat keventämään työkuormaasi. Täyden pääsyn/käyttöoikeuden antaminen on kuitenkin riskialtista ja voi vaarantaa verkostosi. Tässä kohtaa tämä Tukiedustaja-lisäosa tulee avuksi.

Tämän lisäosan avulla voit luoda uuden WordPress-käyttäjätyypin nimeltä Tukiedustaja, ja se antaa sinulle täyden hallinnan siihen, mitä käyttöoikeuksia kyseisellä käyttäjällä on. Käyttöoikeuden myöntäminen on yhtä helppoa kuin vaihtopainikkeen kytkeminen päälle, ja sen poistaminen käy kytkemällä se pois päältä, jos päätät poistaa käyttöoikeuden.

## Ominaisuudet

- Eritelty luettelo WordPress- ja Ultimate Multisite -käyttöoikeuksista
- Ota pääsy käyttöön ja poista se käytöstä helposti
- Luo rajaton määrä edustajatilejä

## Asennus

1. Lataa 'ultimate-multisite-support-agents' hakemistoon '/wp-content/plugins/'
2. Aktivoi lisäosa WordPressin 'Lisäosat'-valikon kautta
3. Seuraa vaiheittaista ohjattua toimintoa lisäosan määrittämiseksi
