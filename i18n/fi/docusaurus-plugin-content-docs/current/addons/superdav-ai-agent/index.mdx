# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent on edistynyt WordPress AI -avustaja, joka laajentaa Gratis AI Agentia teematietoisilla taidoilla, parannetuilla internet-hakuominaisuuksilla, sivustonrakennusominaisuuksilla ja valinnaisilla automaatiointegraatioilla. Versio 1.18.0 lisää Google Calendar -lukutyökalut, yhteystietojen ja osallistujien yhdistämisen, ihmisen hyväksyntäportit, muistutusten deduplikoinnin, TextBee SMS -ilmoitukset, hallitun Superdav-palvelun käyttöönoton sekä erikseen jaeltavan Advanced-kumppanipaketin luotettujen kehittäjien työnkulkuihin.

## Tärkeimmät ominaisuudet {#key-features}

- **Teematietoiset taidot** — mukauttaa ohjeistuksen ja ominaisuudet automaattisesti aktiivisen WordPress-teemasi ja asennettujen lisäosien perusteella
- **Useita internet-hakupalveluntarjoajia** — valitse Brave Searchin ja Tavilyin väliltä saadaksesi monipuolisempia hakutuloksia ja edistyneitä tutkimusominaisuuksia
- **Sivustonrakentajan yhteydenottolomakkeet** — luo täysin toimivia yhteydenottolomakkeita suoraan chat-käyttöliittymästä käyttämällä asennettua lomakelisäosaasi
- **Parannettu sivustonrakentaja** — parannetut monivaiheiset sivuston rakennussuunnitelmat, joissa on parempi lisäosien tunnistus ja virheistä palautuminen
- **Sisäänrakennetut teemataidot** — erikoistunut ohjeistus lohkoteemoille, klassisille teemoille, Kadence Blocksille ja Kadence Theme -teemalle
- **Parannettu WooCommerce-integraatio** — uudistettu käyttämään natiiveja WooCommerce-rajapintoja paremman luotettavuuden ja yhteensopivuuden saavuttamiseksi
- **Automaattinen palveluntarjoajan tunnistus** — tunnistaa ja päivittää saatavilla olevat lisäosat ja teemat automaattisesti, kun ne aktivoidaan tai poistetaan käytöstä
- **Mukautettu logon luonti** — luo ja upota mukautettuja SVG-logoja automaattisella nimiavaruusturvallisella puhdistuksella
- **Paletin kontrastin validointi** — tarkista väriparien WCAG-saavutettavuusvaatimusten mukaisuus ennen niiden käyttöönottoa teemassasi
- **Valokuvilla tehostettu kartoitus** — lataa valokuvia Theme Builderin kartoitushaastattelun aikana vaikuttaaksesi suunnittelun suuntaan
- **Hospitality-valikot** — luo jäsenneltyjä ruoka- ja juomalistasivuja ravintoloille, kahviloille ja hospitality-yrityksille
- **Työpöytä- ja mobiiliesikatselut** — esikatsele suunnitteluasi työpöytä- ja mobiililaitteilla suunnittelusuunnan valinnan aikana
- **Tason 1 sivustojen kaavinta** — sd-ai-agent/site-scrape on nyt oletuksena saatavilla kilpailija- ja inspiraatiosivustojen analysointiin
- **Google Calendar -lukutyökalut** — lue määritettyjä kalentereita ja tapahtumia, jotta automaatiot voivat päätellä tulevan aikataulun kontekstia
- **Yhteystietojen ja osallistujien yhdistäminen** — yhdistä kalenterin osallistujat tunnettuihin sivuston käyttäjiin ja yhteystietoihin ennen jatkotyön luomista
- **Ihmisen hyväksyntäportit** — keskeytä arkaluonteiset automaatiot tarkistusta varten ennen viestien lähettämistä, työn jatkamista tai muutosten tekemistä
- **Muistutustietueet ja deduplikointi** — seuraa jo lähetettyjä muistutuksia, jotta toistuvat automaatioajot eivät ilmoita samalle vastaanottajalle kahdesti
- **TextBee SMS -ilmoitukset** — lähetä määritettyjä tekstiviesti-ilmoituksia TextBeen kautta, kun SMS-palveluntarjoaja on käytössä
- **Hallittu Superdav-palvelun käyttöönotto** — yhdistä tuetut sivustot hostattuihin Superdav-päätepisteisiin automaattisella käyttöönotolla manuaalisen päätepistemäärityksen sijaan
- **Advanced-kumppanipaketti** — erikseen jaeltavat luotetun kehittäjän työkalut tiedostojärjestelmän, tietokannan, WP-CLI:n, REST-välittäjän, lisäosarakentajan, git-tilannevedoksen, käyttäjähallinnan ja benchmark-työnkulkujen käyttöön

## Vaatimukset {#requirements}

- WordPress 6.0 tai uudempi
- PHP 8.1 tai uudempi
- WordPress Multisite (verkkoaktivoitu) tai yhden sivuston asennus
- Aktiivinen AI-palveluntarjoajan yhdistinlisäosa (Anthropic, OpenAI tai yhteensopiva)
- Gratis AI Agent v1.9.0 tai uudempi (Superdav AI Agent laajentaa Gratis AI Agentia)

## Asennus {#installation}

1. Lataa lisäosatiedostot kansioon `/wp-content/plugins/superdav-ai-agent`
2. Aktivoi lisäosa WordPressin **Lisäosat**-näkymässä
3. Varmista, että myös Gratis AI Agent on aktivoitu
4. Siirry kohtaan **Gratis AI Agent → Asetukset** ja määritä internet-hakupalveluntarjoajasi, hallitun Superdav-palvelun yhteys, Google Calendar tai TextBee-tunnistetiedot tarpeen mukaan

WordPress.org-paketti sisältää Superdav AI Agentin ydinominaisuudet. Advanced-kumppanipaketti rakennetaan ja jaellaan erikseen ympäristöihin, joissa luotettujen kehittäjien työkalut asennetaan ja valtuutetaan nimenomaisesti.


```
Tarkista huomisen määritetyt Google Calendar -tapahtumat ja luonnostele muistutukset osallistujille.
```

```
Lähetä SMS-muistutus, kun olen hyväksynyt viestin.
```

## Teematietoiset taidot {#theme-aware-skills}

Superdav AI Agent tunnistaa automaattisesti aktiivisen teemasi ja asennetut lisäosat ja lataa sopivan teematietoisen taidon avustajan kontekstiin. Saatavilla olevia taitoja ovat:

| Taito | Koskee |
|---|---|
| **Lohkoteemat** | Teemat, jotka käyttävät WordPressin lohkopohjaista (Full Site Editing) arkkitehtuuria |
| **Klassiset teemat** | Perinteiset WordPress-teemat, jotka käyttävät PHP-malleja ja klassista editoria |
| **Kadence Blocks** | Sivustot, joilla Kadence Blocks -lisäosa on aktiivinen |
| **Kadence Theme** | Sivustot, joilla Kadence-teema on aktiivisena teemana |

Katso [Teematietoiset taidot](../../user-guide/configuration/theme-aware-skills), jos haluat yksityiskohtaista ohjeistusta kustakin taidosta.

## Internet-hakupalveluntarjoajat {#internet-search-providers}

Superdav AI Agent v1.10.0 lisää **Tavilyin** vaihtoehdoksi Brave Searchille internet-hakukyselyihin. Molemmat palveluntarjoajat tarjoavat reaaliaikaisia hakutuloksia, ja ne voidaan määrittää kohdassa **Gratis AI Agent → Asetukset → Lisäasetukset**.

| Tarjoaja | Ominaisuudet | Määritys |
|---|---|---|
| **Brave Search** | Nopeat, yksityisyyteen keskittyvät hakutulokset | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Monipuolisemmat hakutulokset, edistyneet tutkimusominaisuudet | [Tavily API](https://tavily.com/) |

Katso määritysohjeet kohdasta [Internet-haku](../../user-guide/configuration/internet-search).

## Kalenteri-, hyväksyntä- ja ilmoitustyönkulut {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 laajentaa automaatiotyönkulkuja sisältö- ja sivustonrakennustehtävien ulkopuolelle:

- **Google Calendar -työkalut** antavat agentin lukea määritettyjä kalentereita ja tapahtumatietoja, kun kalenterin tunnistetiedot ovat saatavilla.
- **Yhteystieto- ja osallistuja-avustajat** yhdistävät tapahtuman osallistujat WordPress-käyttäjiin tai tunnettuihin yhteystietoihin ennen muistutusten tai jatkotehtävien valmistelua.
- **Ihmisen hyväksyntäportit** sallivat automaation keskeyttämisen, kunnes valtuutettu käyttäjä tarkistaa ehdotetun toiminnon.
- **Muistutustietueet** estävät päällekkäiset kalenteri- tai ilmoitusmuistutukset, kun automaatio yritetään uudelleen tai ajoitetaan toistuvasti.
- **TextBee SMS -tuki** mahdollistaa tekstiviesti-ilmoitukset, kun TextBee-tarjoaja on määritetty.

Käytä näitä työkaluja aikataulutietoiseen avustamiseen, osallistujien jälkiseurantaan ja muistutustyönkulkuihin, jotka edellyttävät edelleen ihmisen tarkistusta ennen arkaluonteisten viestien lähettämistä.

Katso yksityiskohtaiset ohjeet kohdasta [Sivustonrakentajan yhteydenottolomakkeet](../../user-guide/administration/site-builder-contact-forms).

## Määritykset {#configuration}

Kaikki asetukset ovat kohdassa **Gratis AI Agent → Settings** WordPress-ylläpidossa. Superdav AI Agent lisää seuraavat määritysvaihtoehdot:

| Asetus | Kuvaus |
|---|---|
| **Internet Search Provider** | Valitse Brave Searchin ja Tavilyn välillä (valinnainen) |
| **Tavily API Key** | API-avain Tavilyn internet-hakua varten (jos Tavily on valittu) |
| **Managed Superdav Service** | Yhdistä tuetut sivustot isännöityyn Superdav-palveluun automaattisella endpointien käyttöönotolla |
| **Google Calendar Credentials** | Määritä kalenterin käyttöoikeus, jotta agentti voi lukea kalentereita ja tapahtumia |
| **TextBee SMS Provider** | Määritä TextBee-tunnistetiedot hyväksyttyjä SMS-ilmoitustyönkulkuja varten |

## Advanced Companion Package {#advanced-companion-package}

Superdav AI Agent v1.18.0 lisää erikseen jaettavan Advanced companion package -paketin luotettuihin kehittäjätyönkulkuihin. Näitä ominaisuuksia ei sisällytetä WordPress.org-ydinpakettiin. Asenna companion package vain sivustoille, joilla ylläpitäjät nimenomaisesti luottavat agenttiin kehittäjätyökalujen käytössä.

Advanced-työkaluihin kuuluvat tiedostojärjestelmätoiminnot, tietokantakäyttö, WP-CLI, REST dispatcher -työnkulut, plugininrakennusapuohjelmat, git-tilannevedokset, käyttäjähallinnan avustajat ja vertailumittaustyökalut. Rajoita nämä työkalut luotetuille ylläpitäjille ja tarkista hyväksyntäporttien määritykset ennen niiden käyttöönottoa tuotantosivustoilla.
