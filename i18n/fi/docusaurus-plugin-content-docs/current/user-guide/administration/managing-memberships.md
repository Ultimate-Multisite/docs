# Jäsenyyksien hallinta {#managing-memberships}

Ultimate Multisite -järjestelmässä voit hallita jäsenyyksiäsi ja päästä käsiksi koko verkostosi tilausvaihtoehtoihin. Tarkista ja muokkaa tilaustietoja, kuten suunnitelmia ja tuotteita, sivujen määrää, laskutussummaa ja taajuutta, maksushistoria sekä aikaleimoja.

Pääset tälle sivulle siirtyä Ultimate Multisite -valikossa olevaan **Memberships**-välilehteen.

![Memberships list page](/img/admin/memberships-list.png)

Memberships-sivulla näet luettelon asiakkaistasi ja heihin liittyvistä tuotteista. Voit suodattaa jäsenyyksiä tilan mukaan: aktiivinen, odottava, pidettävänä, vanhentunut tai peruttu.

Ilmaiset jäsenyydet käsitellään **elämänhetken** jäsenyksinä. Niillä ei ole päättymispäivää eikä ne pääty automaattisesti vanhentumaan. Jos ilmainen suunnitelma pitäisi päättyä kokeilujakson tai kiinteän ajan jälkeen, käytä maksullista tuotetta, jolla on kokeilu, peruuta jäsenyys manuaalisesti tai siirrä asiakas toiselle suunnitelmalle, kun käyttöoikeudet muuttuvat.

Napsauta jotakin jäsenyyttä päästäksesi sivulle, josta voit muokata jäsenyyden yksityiskohtia – laskutus- ja maksutietoja, sivujen tai vierailujen rajoituksia ja paljon muuta.

![Membership edit interface](/img/admin/membership-edit.png)

Tässä on koko näkymä jäsenyyden muokkaussivusta:

![Membership edit full interface](/img/admin/membership-edit-full.png)

Voit myös lisätä uusia jäsenyyksiä, jotka liittyvät käyttäjiin, jotka ovat aiemmin rekisteröityneet verkostosi kautta. Napsauta **Add Membership** (Lisää jäsenyys) ja määritä tiedot.

![Add Membership modal](/img/admin/membership-add-modal.png)

Napsauta sitten **Create Membership** (Luo jäsenyys). Uusi jäsenyys ilmestyy jäsenyyksien luetteloon.
