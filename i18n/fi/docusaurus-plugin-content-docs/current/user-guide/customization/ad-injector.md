# Mainosten mainettasi Ultimate Multisite: Ad Injectorilla

Ultimate Multisite: Ad Injector on **ilmainen lisäosa**, joka antaa sinulle työkalut mainosten sijoittamiseen suunnitelmiisi suoraan niiden muokkaussivulta.

Asenna Ultimate Multisite: Ad Injector ja mene _Plugins_ -sivullesi aktivoitaksesi sen.

<!-- Kuvakaappauksen saatavuus puuttuu: WP Plugins -sivu, jossa on Network Activate -painike Ad Injector-lisäosalle -->

_Napsauta Network Activate -painiketta_

Kun lisäosa on aktiivinen, suunnitelman muokkaussivun _Advanced Options_ meta-laatikossa ilmestyy uusi vaihtoehto nimeltään _Ad Settings_.

![Ad Settings plan edit screen](/img/config/product-edit-full.png)

_Nyt voit muokata mainosasetuksia jokaiselle suunnitelmallesi._

Tällä välilehdellä voit lisätä mainoskoodeja sivuille ja postauksille etupäässä, sisällön ennen ja sen jälkeen.

<!-- Kuvakaappauksen saatavuus puuttuu: Ad code -kentät postauksille ja sivuille ennen sisältöä -->

<!-- Kuvakaappauksen saatavuus puuttuu: Esimerkki mainoksesta postaus-sivulla tietylle suunnitelmalta -->

_Lisää mainoskoodeja sisällön joukkoon, ja se näkyy kyseisen suunnitelman postauksissa ja sivuilla._

Voit myös lisätä mainoksia hallintapaneelien yläosaan kyseisen suunnitelman käyttäjille.

<!-- Kuvakaappauksen saatavuus puuttuu: Hallintapaneelin Ad Code -vaihtoehto suunnitelman asetuksissa -->

<!-- Kuvakaappauksen saatavuus puuttuu: Esimerkki mainoksesta käyttäjähallintapaneelissa -->

_Mene Admin Panel Ad Code -vaihtoehtoon, jos haluat näyttää mainoksia käyttäjän takana (back-end)_

### Mainonnan ulkopuolella

Voit myös käyttää Ultimate Multisite: Ad Injector -lisäosaa tukemaan korkeamman tason suunnitelmia.

Mene vain _Admin Panel Ads_ -vaihtoehtoon ja lisää HTML-koodin tukityökalullesi.

<!-- Kuvakaappauksen saatavuus puuttuu: Admin Panel Ads -vaihtoehto tukityökalun HTML-koodilla -->

<!-- Kuvakaappauksen saatavuus ei ole: Asiakaspalveluwidget näkyy hallintapaneelissa tietylle suunnitelmalle -->

_Lisää asiakaspalvelu tietyille suunnitelmille_

Tällä tavalla käyttäjät pääsevät yhteyteen sinulle suoraan hallintapaneelistaan!
