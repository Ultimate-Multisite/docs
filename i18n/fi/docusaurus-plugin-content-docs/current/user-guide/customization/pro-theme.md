# Muokkaa hallintapaneelia PRO-teemalla {#customizing-your-admin-panel-with-pro-theme}

Usein asiakkaan työtä tekemisen yhteydessä meillä on tarve tarjota asiakkaille räätälöity kokemus WordPress-hallintapaneelissa. Jos etsit ammattimaisempaa ulkoasua ja tuntua, PRO Theme voi auttaa sinua.

PRO Theme on hallintateeman, jonka kehittiin auttaaksesi räätälöidessasi asiakkaidesi Dashboardia. Se on myös osa Ultimate Multisite -lisäosien ydintoimintoja.

Asennuksen jälkeen siirry _Plugins_-sivulle ja klikkaa aktivoitaksesi sen verkostosi (Network).

<!-- Kuvakaavaa ei ole saatavilla: WP Plugins-sivu, jossa näkyy PRO Theme -verkon aktivointipainike -->

_Aktivoi se koko verkostolle Plugins-sivulla_

_**HUOMIO:**_ _Jos asennat sen vain kerran, se ilmestyy asiakkaidesi vaihtoehtona, jota voi merkitä lisätäkseen teeman vain kyseiselle käyttäjälle._

Nyt sinulla on hallintapaneeli täysin uudelleen suunniteltuna! Sinulla on myös uusi valikko vasemmassa sarakkeessa nimeltään PRO Theme.

<!-- Kuvakaavaa ei ole saatavilla: PRO Theme -suunnellun hallintapaneelin käyttöliittymä -->

_Uusi ulkoasu hallintapaneelillesi_

Tällä valikolla sinulla on pääsy viiteen välilehteen kautta:

<!-- Kuvakaavaa ei ole saatavilla: PRO Theme -tyyliasetukset-välilehti -->

_Tyyliasetukset (Styling Settings)_

<!-- Kuvakaavaa ei ole saatavilla: PRO Theme -toiminnallisuus-välilehti -->

_Toiminnallisuus (Functionality)_

<!-- Kuvakaavaa ei ole saatavilla: PRO Theme -mukautettu CSS-välilehti -->

_Mukautettu CSS (Custom CSS)_

<!-- Kuvakaavaa ei ole saatavilla: PRO Theme -tuonti/vienti-välilehti -->

_Tuonti/Vienti (Import/Export)_

<!-- Kuvakaavaa ei ole saatavilla: PRO Theme -aktivoi-välilehti -->

_Aktivoi PRO Theme (Activate PRO Theme)_

Jokaisessa niistä voit aktivoida ja poistaa käytöstä tyyliasetuksia, sallia tai estää toimintoja sekä tarjota asiakkaille räätälöintiä, jota he saattavat etsiä verkostosi sivustoista.

Tässä on esimerkki siitä, mitä voit saada muutosten jälkeen:

_Asiakkaan hallintapaneeli_
