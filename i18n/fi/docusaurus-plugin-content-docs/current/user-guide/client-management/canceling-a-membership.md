# Jäsenyyden peruuttaminen (v2)

_**TÄRKEÄ HUOMIO: Tämä artikkeli viittaa Ultimate Multisite -versioon 2.x.**_

Jäsenyyden peruuttaminen asiakkaalla on melko helppoa.

Jos käyttäjä haluaa perua jäsenyyden, hänen tulee ensin päästä **Tili**-sivulle.

Siellä hän voi klikata kohtaa **Vaara-alue (Danger Zone)** ja klikata **Poista tili**.

![Tilin sivun vaara-alue Poista tili -painike](/img/account-page/danger-zone.png)

Avautuu moduuli, jossa pyydetään vahvistusta. Heidän tulee kytkeä päälle **Vahvista tilin poisto** -vaihtoehto ja klikata **Poista tili**.

![Tilin poiston vahvistusikkuna](/img/account-page/delete-account-confirm.png)

Tämä kirjauttaa heidät pois sivustolta, peruuttaa jäsenyyden verkostossasi sekä poistaa sivuston verkostostasi.

![Kirjautumattuna tilin poiston jälkeen](/img/account-page/logged-out-after-delete.png)

![Jäsenyyden peruutuksen vahvistus](/img/account-page/membership-canceled.png)

**TÄRKEÄ:** Huomaa, että tämä toiminto ei poista asiakasta verkostostasi, sillä asiakkaalla voi olla useampia jäsenyyksiä tai yhtään jäsenyyttä linkitettynä.

![Asiakas on edelleen verkostossa peruuttamisen jälkeen](/img/admin/customer-after-cancel.png)

![Asiakas ilman aktiivisia jäsenyyksiä](/img/admin/memberships-customer-empty.png)
