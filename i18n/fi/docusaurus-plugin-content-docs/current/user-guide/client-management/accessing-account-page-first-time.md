# Miten pääsen ensimmäistä kertaa tiliin? {#how-can-i-access-my-account-page-for-the-first-time}

Tervetuloa Ultimate Multisiteen!

Pääset tiliisi linkin kautta, jonka saat sähköpostitse pluginimme ostamisen jälkeen:

<!-- Screenshot unavailable: Account page email with access link -->

Käytä sähköpostiosoitetta, jonka annoit oston yhteydessä, ja salasia, joka saat sähköpostiisi.

<!-- Screenshot unavailable: Login page with email and password fields -->

Koska tämä on ensimmäinen kerta, kun kirjautut tähän tiliin, sinulta pyydetään vaihtamaan salasana. Vaihda se ja klikkaa _Save_ (Tallenna) -painiketta.

<!-- Screenshot unavailable: Change password form with Save button -->

Tilinasi voit hallita kaikki tiliedatat, kuten laskut, uusiutukset, peruutus ja monia muita asioita.
