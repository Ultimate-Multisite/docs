# Ultimate Multisite दस्तावेज {#ultimate-multisite-documentation}

Ultimate Multisite दस्तावेजमा स्वागत छ। Ultimate Multisite ले WordPress Multisite लाई पूर्ण सेवाका रूपमा वेबसाइट (WaaS) प्लेटफर्ममा रूपान्तरण गर्छ।

## दस्तावेज खण्डहरू {#documentation-sections}

### [प्रयोगकर्ता मार्गदर्शिका](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

तपाईंको WaaS प्लेटफर्म सेटअप, कन्फिगर, र व्यवस्थापन गर्न चाहिने सबै कुरा। स्थापना, भुक्तानी गेटवेहरू, डोमेन म्यापिङ, ग्राहक व्यवस्थापन, र थप कुराहरू समेट्छ।

### [विकासकर्ता दस्तावेज](/developer/hooks) {#developer-documentation}

Ultimate Multisite मा निर्माण गर्ने विकासकर्ताहरूका लागि प्राविधिक सन्दर्भ। hooks सन्दर्भ, REST API दस्तावेज, र addon विकास मार्गदर्शिकाहरू समावेश छन्।

### [Addons](/addons) {#addons}

WooCommerce एकीकरण, विश्लेषण, डोमेन बिक्री, र थप जस्ता अतिरिक्त सुविधाहरूले Ultimate Multisite विस्तार गर्ने सशुल्क addon प्लगइनहरूको दस्तावेज।

- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- त्रुटिहरू रिपोर्ट गर्नुहोस्
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- मुख्य प्लगइन डाउनलोड गर्नुहोस्
