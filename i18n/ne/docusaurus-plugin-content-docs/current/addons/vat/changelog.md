# VAT परिवर्तन अभिलेख

संस्करण 1.0.7 - 2026-02-03 मा जारी

* सुधार: VIES SOAP सेवा MS_MAX_CONCURRENT_REQ त्रुटिका कारण German (DE) VAT नम्बरहरू प्रमाणीकरणमा असफल भइरहेका थिए। अझ भरपर्दो प्रमाणीकरणका लागि पुरानो SOAP API बाट आधिकारिक EU VIES REST API मा सारियो।
* सुधार: VIES सेवा अस्थायी रूपमा अनुपलब्ध हुँदा VAT नम्बरहरू अब गलत रूपमा अस्वीकार हुनुको सट्टा स्वीकार गरिन्छन्। पछि समीक्षा गर्नका लागि लग प्रविष्टि सिर्जना गरिन्छ।
* सुधार: VAT/Tax ID फिल्ड अब EU सदस्यहरूका लागि मात्र होइन, सबै देशहरूका लागि देखिन्छ। Non-EU tax ID हरू (जस्तै Swiss CHE नम्बरहरू) VIES प्रमाणीकरण बिना इनभ्वाइस प्रदर्शनका लागि भण्डारण गरिन्छन्।
* सुधार: Greece (GR), Monaco (MC), र Isle of Man (IM) का लागि VAT उपसर्ग खोज गलत एरे खोज तर्कका कारण बिग्रिएको थियो।
* सुधार: Site Exporter एडअनसँगै प्रयोग गर्दा घातक त्रुटिहरू ल्याउन सक्ने अनावश्यक wp-cli-bundle dev निर्भरता हटाइयो।
* सुधार: व्यापक इकाइ परीक्षण सुइट थपियो (५३ परीक्षणहरू)।

संस्करण 1.0.6 - 2026-01-25 मा जारी

* सुधार: घरेलु B2B कारोबारहरूले अब सही रूपमा VAT लगाउँछन्। EU VAT नियमहरूअनुसार, reverse charge केवल सीमापार B2B कारोबारहरूमा मात्र लागू हुन्छ, ग्राहकको देश कम्पनीको देशसँग मिल्दा होइन।
