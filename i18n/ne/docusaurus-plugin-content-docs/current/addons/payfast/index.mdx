# PayFast

Multisite Ultimate: Payfast Gateway

दक्षिण अफ्रिकामा छिटो र सुरक्षित अनलाइन भुक्तानी gateway - Payfast - लाई Multisite Ultimate भुक्तानी विकल्पका रूपमा थप्छ।

## स्थापना

1. 'ultimate-multisite-payfast' लाई '/wp-content/plugins/' directory मा upload गर्नुहोस्
2. WordPress मा 'Plugins' menu मार्फत plugin सक्रिय गर्नुहोस्
3. plugin set up गर्न चरण-दर-चरण Wizard पालना गर्नुहोस्

## प्रयोग मार्गदर्शिका

PayFast दक्षिण अफ्रिकाका व्यवसायहरूका लागि अग्रणी अनलाइन भुक्तानी processor हो। यो कारोबारहरूलाई सहज र सुरक्षित बनाएर अनलाइन व्यापारीहरूलाई बढ्न मद्दत गर्न समर्पित छ। Ultimate Multisite प्रयोगकर्ताको अनुरोधअनुसार, हामीले PayFast सँग integration सम्भव बनाउने addon सिर्जना गर्ने निर्णय गर्‍यौँ। यस लेखले तपाईंको Ultimate Multisite network मा भुक्तानी प्राप्त गर्न PayFast लाई तपाईंको भुक्तानी gateway का रूपमा कसरी set up गर्ने भनेर देखाउनेछ।

पहिलो चरण add on स्थापना गर्नु हो।

add-ons page मा जानुहोस् र Ultimate Multisite: Payfast Gateway add-on चयन गर्नुहोस्।

add-on window pop up हुनेछ। Install Now मा क्लिक गर्नुहोस्।

![Install विकल्प भएको add-ons page](/img/admin/addons-page.png)

स्थापना सकिएपछि, तपाईं plugins page मा redirect हुनुहुनेछ। यहाँ, केवल Network Activate मा क्लिक गर्नुहोस् र _Payfast_ add-on तपाईंको network मा सक्रिय हुनेछ।

# PayFast सेट अप गर्दै

add-on स्थापना गरेपछि, तपाईं अब Ultimate Multisite settings मा गएर PayFast configure गर्न सुरु गर्न सक्नुहुन्छ।

**Ultimate Multisite settings** page अन्तर्गत, तपाईं "Payments" tab भेट्टाउन सक्नुहुन्छ। यस tab अन्तर्गत, तपाईंले यो भुक्तानी gateway enable वा disable गर्न toggle button सहित "**Payfast** " देख्नुपर्छ।

![भुक्तानी gateway settings](/img/config/settings-payment-gateways.png)अगाडि बढ्नुहोस् र यसलाई enable गर्न toggle button मा क्लिक गर्नुहोस्। त्यसपछि यसले box expand गरेर तपाईंले भर्नुपर्ने विभिन्न fields देखाउनेछ, जस्तै **Mechant ID, Merchant Key and PassPhrase.** तपाईंले यो जानकारी आफ्नो **Payfast** account बाट प्राप्त गर्न सक्नुहुन्छ

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

अर्को चरण तपाईंलाई चाहिने जानकारी प्राप्त गर्न आफ्नो PayFast account मा login गर्नु हो।

login गरेपछि, तपाईंले आफ्नो **Payfast** account dashboard बाट तपाईंलाई चाहिने credentials तुरुन्तै देख्न सक्नुहुन्छ।

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->तपाईंले credential सजिलै copy र paste गरेर आफ्नो Ultimate Multisite settings मा विवरणहरू भर्न सक्नुहुन्छ। त्यसपछि परिवर्तनहरू save गर्न **"Save settings** " button मा क्लिक गर्नुहोस्।

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

यति हो, तपाईंको PayFast भुक्तानी integration पूरा भयो!
