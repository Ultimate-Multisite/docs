# Fluent Forms {#fluent-forms}

यो एड-अनले Multisite Ultimate मा उत्पादनहरू, सदस्यताहरू वा साइटहरूमा Fluent Forms का लागि सीमाहरू थप्न सम्भव बनाउँछ।
Fluent Forms ले अनुकूल पोस्ट प्रकारहरूको सट्टा अनुकूल तालिकाहरू प्रयोग गर्छ, त्यसैले सामान्य सीमित गर्ने विकल्पहरूले Fluent Forms सँग काम गर्दैनन्।

## सुविधाहरू {#features}

- Fluent Forms वा Conversational Fluent Forms का लागि सीमाहरू थप्नुहोस्
- ग्राहकहरूले सीमा पुगेपछि आफ्नो योजना अपग्रेड गर्न प्रेरित गर्ने मैत्रीपूर्ण सन्देश पाउँछन्
