# सहायता एजेन्टहरू {#support-agents}

पूरै नेटवर्क आफैँ व्यवस्थापन गर्नु सजिलो काम होइन। तपाईंको नेटवर्क बढ्न थालेपछि, तपाईंले सहायताको माग बढ्दै गएको देख्न थाल्नुहुनेछ। तपाईंको कामको बोझ हल्का बनाउन मद्दत गर्ने मानिसहरू ल्याउनु राम्रो विचार हुनेछ। तर, पूर्ण पहुँच/अनुमति दिनु जोखिमपूर्ण हुन्छ र तपाईंको नेटवर्कमा सम्झौता हुन सक्छ। यहीँ यो सहायता एजेन्ट add-on काम आउँछ।

यो add-on ले तपाईंलाई Support Agent भनिने नयाँ प्रकारको WordPress प्रयोगकर्ता सिर्जना गर्न दिन्छ, र त्यस प्रयोगकर्तासँग कुन अनुमतिहरू छन् भन्नेमा तपाईंलाई पूर्ण नियन्त्रण दिन्छ। अनुमति दिनु toggle button अन गर्नु जत्तिकै सजिलो छ, र हटाउने निर्णय गर्नुभयो भने यसलाई अफ गर्नु पनि उस्तै सजिलो छ।

## सुविधाहरू {#features}

- WordPress & Ultimate Multisite अनुमतिहरूको वस्तुगत सूची
- सजिलैसँग पहुँच सक्षम र असक्षम गर्नुहोस्
- असीमित संख्यामा Agent खाताहरू सिर्जना गर्नुहोस्

## स्थापना {#installation}

1. 'ultimate-multisite-support-agents' लाई '/wp-content/plugins/' directory मा अपलोड गर्नुहोस्
2. WordPress मा 'प्लगइनहरू' मेनु मार्फत plugin सक्रिय गर्नुहोस्
3. plugin सेटअप गर्न चरणबद्ध Wizard पछ्याउनुहोस्
