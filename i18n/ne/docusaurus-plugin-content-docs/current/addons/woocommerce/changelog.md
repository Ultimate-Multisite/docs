# WooCommerce एकीकरण परिवर्तन विवरण

संस्करण 2.2.0 - 2026-07-01 मा जारी गरियो
* नयाँ: Ultimate Multisite कर रकमहरू अब checkout मा अलग WooCommerce शुल्क पङ्क्तिहरूका रूपमा देखा पर्छन्, जसले भुक्तानी अघि कर कुलहरू अझ स्पष्ट बनाउँछ।
* नयाँ: WooCommerce सदस्यताहरूको पुनःप्रयास अवधिमा असफल-नवीकरण निलम्बन चाहने साइटहरूका लागि opt-in "असफल नवीकरणहरूमा सदस्यताहरू तुरुन्त निलम्बन गर्नुहोस्" सेटिङ र `wu_woo_suspend_on_payment_failure` filter थपियो।
* समाधान: असफल वा पुनःस्थापित नवीकरणपछि Ultimate Multisite सदस्यताहरूसँग समक्रमण बाहिर रहन सक्ने WooCommerce सदस्यता स्थितिहरू मिलाइयो।
* समाधान: हराइरहेको अवस्थामा Ultimate Multisite मुद्रा सूचीमा WooCommerce स्टोर मुद्रा थपियो।
* समाधान: ग्राहक बिलिङ विवरणहरू WooCommerce checkout मा सदस्यहरूलाई पुनर्निर्देश गर्दा सुरक्षित राखियो।
* सुधारिएको: Jetpack Autoloader 5 सँग अनुकूलता थपियो।
* सुधारिएको: रिलीज प्याकेज निर्माण सफा गरियो ताकि GitHub र बजार zipहरूले nested staging directories र विकास फाइलहरूबाट जोगिऊन्।
