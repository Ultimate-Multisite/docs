# Superdav AI Agent

Superdav AI Agent एउटा उन्नत WordPress AI सहायक हो जसले Gratis AI Agent लाई theme-सचेत सीपहरू, सुधारिएका इन्टरनेट खोज क्षमताहरू, साइट निर्माणकर्ता सुविधाहरू, र वैकल्पिक स्वचालन एकीकरणहरूसहित विस्तार गर्छ। संस्करण 1.18.0 ले Google Calendar पढ्ने उपकरणहरू, सम्पर्क र सहभागी म्यापिङ, मानव स्वीकृति गेटहरू, रिमाइन्डर डिडुप्लिकेसन, TextBee SMS सूचनाहरू, व्यवस्थापित Superdav सेवा प्रावधान, र विश्वासिला विकासकर्ता कार्यप्रवाहहरूका लागि छुट्टै वितरण गरिएको Advanced सहायक प्याकेज थप्छ।

## प्रमुख सुविधाहरू {#key-features}

- **Theme-सचेत सीपहरू** — तपाईंको सक्रिय WordPress theme र स्थापित प्लगइनहरूका आधारमा मार्गदर्शन र क्षमताहरू स्वतः अनुकूल गर्छ
- **धेरै इन्टरनेट खोज प्रदायकहरू** — अझ समृद्ध खोज नतिजाहरू र उन्नत अनुसन्धान क्षमताहरूका लागि Brave Search र Tavily मध्ये छनोट गर्नुहोस्
- **साइट निर्माणकर्ता सम्पर्क फारमहरू** — तपाईंको स्थापित फारम प्लगइन प्रयोग गरेर च्याट इन्टरफेसबाटै पूर्ण रूपमा कार्यात्मक सम्पर्क फारमहरू सिर्जना गर्नुहोस्
- **सुधारिएको साइट निर्माणकर्ता** — राम्रो प्लगइन खोज र त्रुटि पुनःप्राप्तिसहित सुधारिएका बहु-चरण साइट निर्माण योजनाहरू
- **अन्तर्निर्मित theme सीपहरू** — Block Themes, Classic Themes, Kadence Blocks, र Kadence Theme का लागि विशेष मार्गदर्शन
- **सुधारिएको WooCommerce एकीकरण** — राम्रो विश्वसनीयता र अनुकूलताका लागि मूल WooCommerce API हरू प्रयोग गर्न पुनर्संरचित
- **स्वचालित प्रदायक पहिचान** — प्लगइनहरू र theme हरू सक्रिय वा निष्क्रिय हुँदा उपलब्ध प्लगइनहरू र theme हरू स्वतः पत्ता लगाउँछ र ताजा गर्छ
- **अनुकूल लोगो उत्पादन** — स्वचालित namespace-सुरक्षित सफाइसँग अनुकूल SVG लोगोहरू उत्पन्न र एम्बेड गर्नुहोस्
- **प्यालेट कन्ट्रास्ट प्रमाणीकरण** — तपाईंको theme मा लागू गर्नुअघि WCAG पहुँचयोग्यता अनुपालनका लागि रङ जोडीहरू जाँच गर्नुहोस्
- **फोटो-सुधारिएको खोज** — डिजाइन दिशालाई प्रभाव पार्न Theme Builder खोज अन्तर्वार्ताका क्रममा फोटोहरू अपलोड गर्नुहोस्
- **आतिथ्य मेनुहरू** — रेस्टुरेन्ट, क्याफे, र आतिथ्य व्यवसायहरूका लागि संरचित खाना र पेय मेनु पृष्ठहरू उत्पन्न गर्नुहोस्
- **डेस्कटप र मोबाइल पूर्वावलोकनहरू** — डिजाइन-दिशा चयनका क्रममा डेस्कटप र मोबाइल उपकरणहरूमा आफ्नो डिजाइन पूर्वावलोकन गर्नुहोस्
- **टियर 1 साइट स्क्र्यापिङ** — प्रतिस्पर्धी र प्रेरणादायी वेबसाइटहरू विश्लेषण गर्न sd-ai-agent/site-scrape अब पूर्वनिर्धारित रूपमा उपलब्ध छ
- **Google Calendar पढ्ने उपकरणहरू** — कन्फिगर गरिएका क्यालेन्डर र कार्यक्रमहरू पढ्नुहोस् ताकि स्वचालनहरूले आगामी तालिका सन्दर्भबारे तर्क गर्न सकून्
- **सम्पर्क र सहभागी म्यापिङ** — फलो-अप काम उत्पन्न गर्नुअघि क्यालेन्डर सहभागीहरूलाई परिचित साइट प्रयोगकर्ताहरू र सम्पर्कहरूसँग मिलाउनुहोस्
- **मानव स्वीकृति गेटहरू** — सन्देश पठाउनु, काम पुनःसुरु गर्नु, वा परिवर्तनहरू गर्नु अघि समीक्षाका लागि संवेदनशील स्वचालनहरू रोक्नुहोस्
- **रिमाइन्डर रेकर्डहरू र डिडुप्लिकेसन** — पहिले नै पठाइएका रिमाइन्डरहरू ट्र्याक गर्नुहोस् ताकि दोहोरिने स्वचालन रनहरूले उही प्राप्तकर्तालाई दुई पटक सूचना नदिनून्
- **TextBee SMS सूचनाहरू** — SMS प्रदायक सक्षम हुँदा TextBee मार्फत कन्फिगर गरिएका पाठ-सन्देश सूचनाहरू पठाउनुहोस्
- **व्यवस्थित Superdav सेवा प्रावधान** — म्यानुअल endpoint कन्फिगरेसनको सट्टा स्वचालित सेटअपसहित समर्थित साइटहरूलाई hosted Superdav endpoints सँग जडान गर्नुहोस्
- **Advanced सहायक प्याकेज** — फाइलसिस्टम, डेटाबेस, WP-CLI, REST डिस्प्याचर, प्लगइन निर्माणकर्ता, git स्न्यापसट, प्रयोगकर्ता-व्यवस्थापन, र बेन्चमार्क कार्यप्रवाहहरूका लागि छुट्टै वितरण गरिएका विश्वासिला-विकासकर्ता उपकरणहरू

## आवश्यकताहरू {#requirements}

- WordPress 6.0 वा सोभन्दा माथि
- PHP 8.1 वा सोभन्दा माथि
- WordPress Multisite (नेटवर्क-सक्रिय) वा एकल-साइट स्थापना
- सक्रिय AI प्रदायक कनेक्टर प्लगइन (Anthropic, OpenAI, वा अनुकूल)
- Gratis AI Agent v1.9.0 वा सोभन्दा माथि (Superdav AI Agent ले Gratis AI Agent विस्तार गर्छ)

## स्थापना {#installation}

1. प्लगइन फाइलहरू `/wp-content/plugins/superdav-ai-agent` मा अपलोड गर्नुहोस्
2. WordPress मा **Plugins** स्क्रिनमार्फत प्लगइन सक्रिय गर्नुहोस्
3. Gratis AI Agent पनि सक्रिय गरिएको छ भनी सुनिश्चित गर्नुहोस्
4. **Gratis AI Agent → Settings** मा जानुहोस् र आवश्यकता अनुसार आफ्नो इन्टरनेट खोज प्रदायक, व्यवस्थित Superdav सेवा जडान, Google Calendar, वा TextBee प्रमाणपत्रहरू कन्फिगर गर्नुहोस्

WordPress.org प्याकेजमा मूल Superdav AI Agent सुविधाहरू समावेश छन्। Advanced सहायक प्याकेज त्यस्ता वातावरणहरूका लागि छुट्टै बनाइन्छ र वितरण गरिन्छ जहाँ विश्वासिला विकासकर्ता उपकरणहरू स्पष्ट रूपमा स्थापना र अधिकृत गरिएका हुन्छन्।

## द्रुत सुरुवात {#quick-start}

सक्रिय भएपछि, Superdav AI Agent ले theme-सचेत सीपहरू र अतिरिक्त क्षमताहरूसहित तपाईंको Gratis AI Agent स्थापना स्वतः सुधार गर्छ। WordPress प्रशासक साइडबारमा **Gratis AI Agent** प्यानल खोल्नुहोस् र नयाँ सुविधाहरू प्रयोग गर्न सुरु गर्नुहोस्।

**उदाहरण निर्देशनहरू:**

```
सम्पर्क पृष्ठमा सम्पर्क फारम थप्नुहोस्।
```

```
नवीनतम WordPress सुरक्षा उत्कृष्ट अभ्यासहरूका लागि वेब खोज्नुहोस्।
```

```
सक्रिय theme का ब्लक ढाँचाहरू प्रयोग गरेर hero खण्ड सिर्जना गर्नुहोस्।
```

```
भोलिका कन्फिगर गरिएका Google Calendar कार्यक्रमहरू समीक्षा गर्नुहोस् र सहभागीहरूका लागि रिमाइन्डर मस्यौदा गर्नुहोस्।
```

```
मैले सन्देश स्वीकृत गरेपछि SMS रिमाइन्डर पठाउनुहोस्।
```

## Theme-सचेत सीपहरू {#theme-aware-skills}

Superdav AI Agent ले तपाईंको सक्रिय theme र स्थापित प्लगइनहरू स्वतः पत्ता लगाउँछ, र सहायकको सन्दर्भमा उपयुक्त theme-सचेत सीप लोड गर्छ। उपलब्ध सीपहरूमा समावेश छन्:

| सीप | लागू हुने |
|---|---|
| **Block Themes** | WordPress ब्लक-आधारित (Full Site Editing) संरचना प्रयोग गर्ने theme हरू |
| **Classic Themes** | PHP टेम्प्लेटहरू र क्लासिक सम्पादक प्रयोग गर्ने परम्परागत WordPress theme हरू |
| **Kadence Blocks** | Kadence Blocks प्लगइन सक्रिय भएका साइटहरू |
| **Kadence Theme** | सक्रिय theme का रूपमा Kadence theme भएका साइटहरू |

प्रत्येक सीपबारे विस्तृत मार्गदर्शनका लागि [Theme-सचेत सीपहरू](../../user-guide/configuration/theme-aware-skills) हेर्नुहोस्।

## इन्टरनेट खोज प्रदायकहरू {#internet-search-providers}

Superdav AI Agent v1.10.0 ले इन्टरनेट खोज क्वेरीहरूका लागि Brave Search को विकल्पका रूपमा **Tavily** थप्छ। दुवै प्रदायकहरूले वास्तविक-समय खोज नतिजाहरू प्रदान गर्छन् र **Gratis AI Agent → Settings → Advanced** मा कन्फिगर गर्न सकिन्छ।

| प्रदायक | विशेषताहरू | सेटअप |
|---|---|---|
| **Brave Search** | छिटो, गोपनीयता-केन्द्रित खोज परिणामहरू | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | अझ समृद्ध खोज परिणामहरू, उन्नत अनुसन्धान क्षमताहरू | [Tavily API](https://tavily.com/) |

कन्फिगरेसन निर्देशनहरूको लागि [इन्टरनेट खोज](../../user-guide/configuration/internet-search) हेर्नुहोस्।

## क्यालेन्डर, स्वीकृति, र सूचना कार्यप्रवाहहरू {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 ले सामग्री र साइट-निर्माण कार्यहरूभन्दा बाहिर स्वचालन कार्यप्रवाहहरू विस्तार गर्छ:

- **Google Calendar उपकरणहरू** ले क्यालेन्डर प्रमाणपत्रहरू उपलब्ध हुँदा agent लाई कन्फिगर गरिएका क्यालेन्डरहरू र कार्यक्रम विवरणहरू पढ्न दिन्छन्।
- **सम्पर्क र सहभागी सहयोगीहरू** ले सम्झना वा फलो-अप कार्यहरू तयार गर्नु अघि कार्यक्रम सहभागीहरूलाई WordPress प्रयोगकर्ताहरू वा ज्ञात सम्पर्कहरूसँग मिलाउँछन्।
- **मानवीय स्वीकृति गेटहरू** ले प्रस्तावित कार्य अधिकृत प्रयोगकर्ताले समीक्षा नगरेसम्म स्वचालनलाई रोक्न अनुमति दिन्छन्।
- **सम्झना अभिलेखहरू** ले स्वचालन पुनः प्रयास गरिँदा वा बारम्बार तालिकाबद्ध गरिँदा डुप्लिकेट क्यालेन्डर वा सूचना सम्झनाहरू रोक्छन्।
- **TextBee SMS समर्थन** ले TextBee प्रदायक कन्फिगर गरिएको हुँदा पाठ-सन्देश सूचनाहरू सक्षम गर्छ।

संवेदनशील सन्देशहरू पठाइनुअघि अझै पनि मानवीय समीक्षा आवश्यक पर्ने तालिका-सचेत सहायता, सहभागी फलो-अप, र सम्झना कार्यप्रवाहहरूका लागि यी उपकरणहरू प्रयोग गर्नुहोस्।

## साइट निर्माणकर्ता सम्पर्क फारमहरू {#site-builder-contact-forms}

साइट निर्माणकर्ता agent ले अब च्याट इन्टरफेसबाट सिधै सम्पर्क फारमहरू सिर्जना गर्न सक्छ। साइट निर्माणकर्तालाई सम्पर्क फारम सिर्जना गर्न भन्नुहोस्, र यसले तपाईंको स्थापित फारम plugin (Contact Form 7, WPForms, Fluent Forms, वा Gravity Forms) स्वतः पत्ता लगाएर प्रयोगका लागि तयार फारम उत्पन्न गर्नेछ।

विस्तृत मार्गदर्शनका लागि [साइट निर्माणकर्ता सम्पर्क फारमहरू](../../user-guide/administration/site-builder-contact-forms) हेर्नुहोस्।

## Theme Builder सुविधाहरू {#theme-builder-features}

Superdav AI Agent v1.16.0 मा Theme Builder ले शक्तिशाली नयाँ क्षमताहरू समावेश गर्छ:

### फोटो अपलोडसहित खोज अन्तर्वार्ता {#discovery-interview-with-photo-upload}

Theme Builder खोज अन्तर्वार्तामा अब फोटो अपलोड चरण समावेश छ, जहाँ तपाईं दृश्य प्रेरणा उपलब्ध गराउन सक्नुहुन्छ। Theme Builder लाई तपाईंको डिजाइन दिशा बुझ्न मद्दत गर्न आफ्नो ब्रान्ड, प्रतिस्पर्धी साइटहरू, वा डिजाइन प्रेरणाका फोटोहरू अपलोड गर्नुहोस्।

विस्तृत मार्गदर्शनका लागि [खोज अन्तर्वार्ता](./theme-builder/discovery-interview.md) हेर्नुहोस्।

### डेस्कटप र मोबाइल पूर्वावलोकनसहित डिजाइन दिशा {#design-direction-with-desktop-and-mobile-previews}

Theme Builder ले तपाईंको पूर्ण theme उत्पन्न गर्नु अघि डेस्कटप र मोबाइल दुवै उपकरणहरूमा आफ्नो डिजाइन समीक्षा गर्नुहोस्। धेरै उत्तरदायी ब्रेकपोइन्टहरूमा वास्तविक-समय पूर्वावलोकनहरू हेर्नुहोस्।

विस्तृत मार्गदर्शनका लागि [डिजाइन दिशा](./theme-builder/design-direction.md) हेर्नुहोस्।

### आतिथ्य मेनुहरू {#hospitality-menus}

Theme Builder ले अब रेस्टुरेन्ट, क्याफे, बार, र अन्य आतिथ्य व्यवसायहरूका लागि संरचित खाना र पेय मेनु पृष्ठहरू उत्पन्न गर्न सक्छ। मेनुहरूमा मूल्य, वर्णन, आहारसम्बन्धी जानकारी, र एलर्जेन चेतावनीहरू समावेश हुन्छन्।

विस्तृत मार्गदर्शनका लागि [आतिथ्य मेनुहरू](./theme-builder/hospitality-menus.md) हेर्नुहोस्।

## कन्फिगरेसन {#configuration}

सबै सेटिङहरू WordPress admin मा **Gratis AI Agent → Settings** अन्तर्गत छन्। Superdav AI Agent ले निम्न कन्फिगरेसन विकल्पहरू थप्छ:

| सेटिङ | वर्णन |
|---|---|
| **Internet Search Provider** | Brave Search र Tavily बीच छनोट गर्नुहोस् (वैकल्पिक) |
| **Tavily API Key** | Tavily इन्टरनेट खोजका लागि API key (Tavily चयन गरिएको भए) |
| **Managed Superdav Service** | समर्थित साइटहरूलाई स्वचालित endpoint प्रावधानसहित hosted Superdav सेवासँग जडान गर्नुहोस् |
| **Google Calendar Credentials** | agent ले क्यालेन्डरहरू र कार्यक्रमहरू पढ्न सक्ने गरी क्यालेन्डर पहुँच कन्फिगर गर्नुहोस् |
| **TextBee SMS Provider** | स्वीकृत SMS सूचना कार्यप्रवाहहरूका लागि TextBee प्रमाणपत्रहरू कन्फिगर गर्नुहोस् |

## उन्नत सहायक प्याकेज {#advanced-companion-package}

Superdav AI Agent v1.18.0 ले विश्वसनीय developer कार्यप्रवाहहरूका लागि छुट्टै वितरण हुने उन्नत सहायक प्याकेज थप्छ। यी क्षमताहरू WordPress.org मूल प्याकेजमा समावेश गरिएका छैनन्। प्रशासकहरूले agent लाई developer उपकरणहरूमा पहुँच गर्न स्पष्ट रूपमा विश्वास गर्ने साइटहरूमा मात्र सहायक प्याकेज स्थापना गर्नुहोस्।

उन्नत उपकरणहरूमा फाइल प्रणाली सञ्चालनहरू, database पहुँच, WP-CLI, REST dispatcher कार्यप्रवाहहरू, plugin builder उपयोगिताहरू, git snapshots, प्रयोगकर्ता-व्यवस्थापन सहयोगीहरू, र benchmark tooling समावेश छन्। यी उपकरणहरूलाई विश्वसनीय प्रशासकहरूमा सीमित राख्नुहोस् र production साइटहरूमा सक्षम गर्नु अघि approval-gate कन्फिगरेसन समीक्षा गर्नुहोस्।
