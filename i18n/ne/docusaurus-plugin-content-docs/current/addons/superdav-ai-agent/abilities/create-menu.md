# मेनु सिर्जना गर्नुहोस्

**मेनु सिर्जना गर्नुहोस्** क्षमताले पृष्ठ शीर्षकहरूबाट अलग विशिष्ट नेभिगेसन लेबलहरूको समर्थनसहित WordPress नेभिगेसन मेनुहरू सिर्जना गर्छ।

## सिंहावलोकन {#overview}

यो क्षमताले `navigation_label` प्यारामिटर निर्दिष्ट गर्ने क्षमतासहित मानक मेनु सिर्जना कार्यक्षमता विस्तार गर्छ। यसले तपाईंलाई नेभिगेसनमा देखाइने लेबल पृष्ठ शीर्षकभन्दा फरक हुने मेनुहरू सिर्जना गर्न दिन्छ, जसले साइट संरचना र प्रयोगकर्ता अनुभवमा बढी लचकता प्रदान गर्छ।

## प्यारामिटरहरू {#parameters}

| प्यारामिटर | प्रकार | आवश्यक | विवरण |
|-----------|------|----------|-------------|
| `name` | string | हो | मेनुको नाम, जस्तै `Primary Navigation` |
| `location` | string | होइन | यो मेनु असाइन गर्ने theme स्थान, जस्तै `primary` |
| `navigation_label` | string | होइन | नेभिगेसनमा देखाइने लेबल (पृष्ठ शीर्षकबाट अलग) |

## फर्काइने मान {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## नेभिगेसन लेबल बनाम पृष्ठ शीर्षक {#navigation-label-vs-page-title}

`navigation_label` प्यारामिटरले तपाईंलाई आन्तरिक मेनु नामलाई प्रयोगकर्ताहरूलाई देखाइने लेबलबाट अलग गर्न दिन्छ:

- **`name`** — WordPress द्वारा प्रयोग गरिने आन्तरिक मेनु पहिचानकर्ता (जस्तै, "Primary Navigation")
- **`navigation_label`** — नेभिगेसनमा साइट आगन्तुकहरूलाई देखाइने लेबल (जस्तै, "Main Menu")

यो उपयोगी हुन्छ जब:
- तपाईंको आन्तरिक नामकरण परम्परा प्रयोगकर्तामुखी लेबलहरूभन्दा फरक हुन्छ
- तपाईं admin प्यानलमा भन्दा नेभिगेसनमा छोटा लेबलहरू चाहनुहुन्छ
- तपाईंलाई फरक लेबल लम्बाइ भएका धेरै भाषाहरू समर्थन गर्नुपर्ने हुन्छ
- तपाईं विशिष्ट क्षेत्रहरू वा प्रयोगकर्ता समूहहरूका लागि मेनुहरू बनाउँदै हुनुहुन्छ

## प्रयोग उदाहरणहरू {#usage-examples}

### उदाहरण १: नेभिगेसन लेबलसहित सरल मेनु {#example-1-simple-menu-with-navigation-label}

**प्रम्प्ट:**
```
"मेनु" नेभिगेसन लेबल भएको "मुख्य नेभिगेसन" नामको primary नेभिगेसन मेनु सिर्जना गर्नुहोस्।
```

**नतिजा:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### उदाहरण २: विशिष्ट Theme स्थानका लागि मेनु {#example-2-menu-for-specific-theme-location}

**प्रम्प्ट:**
```
"छिटो लिङ्कहरू" नेभिगेसन लेबल भएको "फुटर लिङ्कहरू" नामको footer मेनु सिर्जना गर्नुहोस् र यसलाई footer स्थानमा असाइन गर्नुहोस्।
```

**नतिजा:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Theme Builder सँग एकीकरण {#integration-with-theme-builder}

Theme Builder प्रयोग गर्दा, मेनु सिर्जना गर्ने क्षमताले:

1. उपलब्ध theme मेनु स्थानहरू स्वचालित रूपमा पत्ता लगाउँछ
2. तपाईंको डिजाइनका लागि उपयुक्त नेभिगेसन लेबलहरूसहित मेनुहरू सिर्जना गर्छ
3. मेनुहरूलाई सही theme स्थानहरूमा असाइन गर्छ
4. सिर्जनापछि मेनु वस्तुहरू थप्न समर्थन गर्छ

## सम्बन्धित क्षमताहरू {#related-abilities}

- **`add_menu_item`** — अवस्थित मेनुमा वस्तुहरू थप्नुहोस्
- **`update_menu`** — मेनुको नाम परिवर्तन गर्नुहोस् वा यसलाई theme स्थानमा फेरि असाइन गर्नुहोस्
- **`delete_menu`** — तपाईंको साइटबाट मेनु हटाउनुहोस्

## उत्तम अभ्यासहरू {#best-practices}

- **स्पष्ट नेभिगेसन लेबलहरू प्रयोग गर्नुहोस्** — प्रयोगकर्ताहरूका लागि लेबलहरू संक्षिप्त र वर्णनात्मक राख्नुहोस्
- **theme स्थानहरूसँग मिलाउनुहोस्** — सही प्रदर्शनका लागि मेनुहरूलाई सही theme स्थानमा असाइन गर्नुहोस्
- **मेनु संरचना योजना बनाउनुहोस्** — वस्तुहरू सिर्जना गर्नु अघि आफ्नो मेनु पदानुक्रम निर्णय गर्नुहोस्
- **प्रतिक्रियाशीलता परीक्षण गर्नुहोस्** — मोबाइल उपकरणहरूमा मेनुहरू सही रूपमा देखिन्छन् भनी प्रमाणित गर्नुहोस्
- **लेबलहरू स्थानीयकरण गर्नुहोस्** — फरक भाषा संस्करणहरूका लागि फरक नेभिगेसन लेबलहरू प्रयोग गर्नुहोस्

## सीमितताहरू {#limitations}

- नेभिगेसन लेबलहरू केवल प्रदर्शनका लागि हुन्; आन्तरिक `name` अझै पनि WordPress पहिचानका लागि प्रयोग हुन्छ
- Theme समर्थन फरक-फरक हुन्छ; सबै themes ले सबै मेनु स्थानहरू समर्थन गर्दैनन्
- मेनु सिर्जनापछि मेनु वस्तुहरू अलगै थप्नुपर्छ
- नेभिगेसन लेबल परिवर्तन गर्न मेनु अद्यावधिक गर्नुपर्छ
