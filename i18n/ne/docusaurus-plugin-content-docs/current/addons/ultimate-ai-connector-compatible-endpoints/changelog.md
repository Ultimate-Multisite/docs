# Ultimate AI Connector for Compatible Endpoints परिवर्तन विवरण {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## संस्करण २.०.० — २०२६-०४-२४ मा जारी {#version-200--released-on-2026-04-24}

* नयाँ: बहु-प्रदायक समर्थन — धेरै AI endpoints कन्फिगर गर्नुहोस् र प्रदायकहरूभरि स्वचालित fallback सहित अनुरोधहरू route गर्नुहोस्।
* सुधार: सही provider IDs, registration URLs, र प्रति प्रदायक HTTP filter scoping सहित बहु-प्रदायक SDK integration।
* सुधार: नयाँ प्रदायक card अब थप्दा स्वचालित रूपमा विस्तार हुन्छ; plugin update मा script cache busting।
* सुधार: eval() का लागि dynamic provider class namespace र भर्खरै थपिएका प्रदायकहरूका लागि auto-expand व्यवहार।
* सुधार: WordPress 6.9+ compatibility का लागि स्थिर Card/CardBody/CardHeader/CardDivider components (अब experimental छैनन्) प्रयोग गर्छ।
* सुधार: provider drag-to-reorder का लागि अपरिभाषित DragHandle लाई unicode grip icon ले प्रतिस्थापन गरियो।
* सुधार: प्रत्येक page load मा चल्ने अवरोधक HTTP request हटाइयो।
* सुधारिएको: GitHub Actions workflows लाई Node.js 24 मा upgrade गरियो।
