# भाषा चयनकर्ता

Multisite Ultimate: साइन-अपमा भाषा चयनकर्ता

साइन-अप प्रवाहमा भाषा चयनकर्ता फिल्ड थप्नुहोस्।

## स्थापना {#installation}

1. 'ultimate-multisite-language-selector' लाई '/wp-content/plugins/' निर्देशिकामा अपलोड गर्नुहोस्
2. आफ्नो WordPress Network Admin मा रहेको 'प्लगइनहरू' मेनुबाट प्लगइन सक्रिय गर्नुहोस्

## प्रयोग मार्गदर्शिका {#usage-guide}

यो निःशुल्क add-on मार्फत तपाईं आफ्ना प्रयोगकर्ताहरूलाई साइन-अप प्रवाह र लगइन पृष्ठमा कुन भाषा हेर्न चाहन्छन् भनेर छान्न दिन सक्नुहुन्छ।

[आफ्नो लाइसेन्स सक्रिय गरेपछि](../../user-guide/getting-started/installing-ultimate-multisite.md) तपाईंले आफ्नो लाइसेन्सले पहुँच दिने Ultimate Multisite add-ons स्थापना गर्न सक्नुहुनेछ।

आफ्नो Ultimate Multisite सेटिङ्स पृष्ठमा जानुहोस् र त्यसपछि, बायाँपट्टिको मेनुमा, _Add-ons_ लेबलमा क्लिक गर्नुहोस्।

![Add-ons पृष्ठ](/img/admin/addons-page.png)

_Language Selector स्थापना सुरु गर्न Add-ons पृष्ठमा जानुहोस्_

त्यहाँ, हामीले प्रदान गर्ने सबै add-ons भेट्नुहुनेछ।

निःशुल्क add-ons हाम्रा सबै ग्राहकहरूका लागि उपलब्ध छन्, उनीहरूले साइन अप गरेको सदस्यता योजनाको परवाह नगरी।

add-on को थम्बनेलको तल रहेको _Install it Now!_ बटनमा क्लिक गर्नुहोस्। त्यसपछि Plugins पृष्ठमा यसलाई सक्रिय गर्नुहोस्।

![स्थापना बटन भएको Add-ons पृष्ठ](/img/admin/addons-page.png)

_add-on स्थापना गर्न बटनमा क्लिक गर्नुहोस्_

<!-- स्क्रिनसट उपलब्ध छैन: WP Plugins पृष्ठले Language Selector addon का लागि Network Activate बटन देखाउँदै -->

_आफ्नो Plugins पृष्ठमा यसलाई सक्रिय गर्नुहोस्_

अब, Ultimate Multisiteको Settings मेनुमा जानुहोस् र _Network Settings_ ट्याबमा प्रवेश गर्नुहोस्। त्यहाँ, पृष्ठको तल, तपाईंले _Languages_ भनिने नयाँ खण्ड भेट्नुहुनेछ।

![Settings पृष्ठ](/img/config/settings-general.png)

_बिल्कुल नयाँ खण्ड देखा पर्छ!_

सबै भाषाहरूलाई विकल्पका रूपमा देखाउन यसलाई खाली छोड्न सक्नुहुन्छ वा कुन-कुन उपलब्ध गराउन चाहनुहुन्छ चयन गर्न सक्नुहुन्छ। भाषाहरू तपाईंले आफ्नो WordPress सेटअपमा स्थापना गरेका भाषाहरू हुन्।

<!-- स्क्रिनसट उपलब्ध छैन: WP स्थापनामा उपलब्ध भाषाहरू देखाउने भाषा चयन ड्रपडाउन -->

_यस WP स्थापनामा उपलब्ध भाषाहरूको सूची_

तपाईंले गरेका परिवर्तनहरू सुरक्षित गरेपछि, तपाईंले छानेका भाषाहरू भएको चयन मेनु Login पृष्ठका साथै Sign-up Flow मा पनि देखाइनेछ।

![Login र Registration सेटिङ्स](/img/config/settings-login-registration.png)

_Login पृष्ठमा Language Selector_

![Registration पृष्ठ](/img/config/registration-page.png)

_Sign-up Flow मा Language Selector_

## URL मार्फत भाषा सेट गर्ने {#setting-the-language-via-the-url}

तपाईं URL मा locale प्यारामिटर पास गरेर पृष्ठको भाषा पनि परिवर्तन गर्न सक्नुहुन्छ। उदाहरणका लागि, यदि तपाईं इटालियन भाषाको दर्ता पृष्ठमा रिडाइरेक्ट गर्न चाहनुहुन्छ भने, URL मा ?locale=it_IT थप्नुहोस्।

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

यसले काम गर्नका लागि भाषा तपाईंको WordPress नेटवर्कमा स्थापना भएको हुनुपर्छ।
