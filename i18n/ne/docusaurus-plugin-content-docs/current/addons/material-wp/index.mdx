# Material WP

WordPress admin इन्टरफेस लामो समयदेखि उस्तै छ। तर admin इन्टरफेसको Material Design संस्करण कस्तो देखिन्थ्यो होला भनेर तपाईंले कहिल्यै सोच्नुभएको छ?

Material WP तपाईंलाई त्यही रूप र अनुभव दिन विकास गरिएको admin थिम हो, जसले तपाईंलाई आफ्ना ग्राहकहरूलाई अझ बढी अनुकूलित Dashboard उपलब्ध गराउन मद्दत गर्छ।

## सुविधाहरू {#features}

- Custom CSS समर्थन
- सजिलो शैलीकरण सेटिङहरू
- मेनु पुनःक्रमबद्धता
- सजिलो आयात/निर्यात

## दस्तावेजीकरण {#documentation}

**Material WP सँग आफ्नो Admin Panel अनुकूलित गर्दै**

## स्थापना {#installation}

1. 'material-wp' लाई '/wp-content/plugins/' डाइरेक्टरीमा अपलोड गर्नुहोस्
2. WordPress मा 'प्लगइनहरू' मेनु मार्फत प्लगइन सक्रिय गर्नुहोस्

## प्रयोग मार्गदर्शिका {#usage-guide}

WordPress admin इन्टरफेस लामो समयदेखि उस्तै छ। तर admin इन्टरफेसको Material Design संस्करण कस्तो देखिन्थ्यो होला भनेर तपाईंले कहिल्यै सोच्नुभएको छ?

Material WP तपाईंलाई त्यही रूप र अनुभव दिन विकास गरिएको admin थिम हो, जसले तपाईंलाई आफ्ना ग्राहकहरूलाई अझ बढी अनुकूलित Dashboard उपलब्ध गराउन मद्दत गर्छ। Material WP Ultimate Multisite को premium प्लगइनहरू core को पनि भाग हो।

यसलाई **स्थापना गरेपछि**, आफ्नो _प्लगइनहरू_ पृष्ठमा जानुहोस् र आफ्नो Network मा यसलाई सक्रिय गर्न क्लिक गर्नुहोस्।

<!-- स्क्रिनसट उपलब्ध छैन: WP प्लगइनहरू पृष्ठले Material WP का लागि Network Activate बटन देखाउँदै -->

_आफ्नो प्लगइनहरू पृष्ठमा यसलाई network-wide सक्रिय गर्नुहोस्_

_**नोट:**_ _यदि तपाईंले यसलाई मात्र सक्रिय गर्नुभयो भने, यो तपाईंको ग्राहककोमा विकल्पका रूपमा देखिनेछ, जसलाई त्यो प्रयोगकर्ताका लागि मात्र थिम थप्न चिन्ह लगाउन सकिन्छ।_
