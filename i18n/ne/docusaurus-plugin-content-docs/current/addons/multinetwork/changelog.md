# Multi-Network परिवर्तन अभिलेख {#multi-network-changelog}

### 1.3.0 {#130}
* नयाँ: Network टेम्प्लेट पूर्वावलोकनकर्ता — खरिद गर्नुअघि प्रत्यक्ष प्यानलसहित Network टेम्प्लेटहरू ब्राउज र पूर्वावलोकन गर्नुहोस्।
* नयाँ: चेकआउटमा Network टेम्प्लेट चयनकर्ता — ग्राहकहरूले साइनअप चेकआउट प्रवाहका क्रममा Network टेम्प्लेट चयन गर्न सक्छन्।
* नयाँ: क्रस-Network magic link SSO — ग्राहकहरूले magic-link प्रमाणीकरणमार्फत उप-Networkहरूमा सहज रूपमा पहुँच गर्न सक्छन्।
* नयाँ: Network मिडिया क्लोनिङ — टेम्प्लेटबाट Networkहरू क्लोन गर्दा साइट मिडिया सही रूपमा नक्कल गरिन्छ।
* सुधार: Network क्लोनिङ अब multi-tenancy एडअन बिना पनि काम गर्छ; क्लोनपछि super admin विशेषाधिकारहरू पुनः प्रदान गरिन्छ।
* सुधार: Network सिर्जना अब WooCommerce र सबै भुक्तानी गेटवेहरूसँग मिल्दो छ।
* सुधार: fallback डोमेन सिर्जनामा पोर्ट नम्बर अब समावेश गरिएको छ।
* सुधार: सबै थिमहरूमा सही रेन्डरिङका लागि टेम्प्लेट चयनकर्ता CSS enqueued stylesheet मा सारियो।
* सुधार: साइट क्वेरी स्कोपमा अपरिभाषित `network_id` कुञ्जी विरुद्ध सुरक्षा।
* सुधार: context switching अघि Network अनुमति जाँचले `wpdb` तालिका गुण त्रुटिहरू रोक्छ।
