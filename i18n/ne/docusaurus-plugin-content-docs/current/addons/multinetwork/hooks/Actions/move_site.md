# कार्य: move_site

कुनै साइट नयाँ नेटवर्कमा सारिएपछि चल्छ।

## प्यारामिटरहरू {#parameters}

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $site_id | `int` | सारिएको साइटको ID। |
| $old_network_id | `int` | साइटका लागि मूल नेटवर्कको ID। |
| $new_network_id | `int` | साइट सारिएको नेटवर्कको ID। |
