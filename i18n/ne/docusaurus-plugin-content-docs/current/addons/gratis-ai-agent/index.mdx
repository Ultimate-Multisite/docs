# Gratis AI Agent

Gratis AI Agent एक स्वायत्त WordPress AI सहायक हो, जसले प्राकृतिक भाषाका निर्देशनहरूबाट पूर्ण WordPress साइटहरू योजना बनाउन, निर्माण गर्न र व्यवस्थापन गर्न सक्छ। यसले **क्षमताहरू** को संरचित सेट उपलब्ध गराउँछ — सहायकले बोलाउन सक्ने छुट्टाछुट्टै, परीक्षणयोग्य कार्यहरू — जसले सामग्री व्यवस्थापनदेखि डिजाइन प्रणाली अनुकूलनसम्म सबै कुरा समेट्छ। संस्करण 1.9.0 ले सामग्री सिर्जना गर्ने क्षमताहरू, क्लाइन्ट-साइड स्क्रिनसटमार्फत दृश्य पृष्ठ समीक्षा, पाँचवटा बिल्ट-इन सहायक प्रोफाइलहरू, र पहुँच नियन्त्रण तथा ब्रान्डिङका लागि सुविधा फ्ल्यागहरू थप्छ।

## मुख्य सुविधाहरू

- **क्षमता प्रणाली** — तपाईंको WordPress स्थापना मा सहायकले बोलाउन सक्ने कार्यहरूको मोड्युलर, विस्तारयोग्य सूची
- **सामग्री व्यवस्थापन** — पोस्ट र पृष्ठहरू सिर्जना गर्नुहोस् (पृष्ठ टेम्प्लेट तोक्नेसहित), एउटै कलमा धेरै पोस्टहरू ब्याचमा सिर्जना गर्नुहोस्, featured images सेट गर्नुहोस्, र सम्पर्क फारमहरू निर्माण गर्नुहोस्
- **दृश्य समीक्षा** — प्रत्यक्ष पृष्ठहरूको स्क्रिनसट खिच्नुहोस्, अघिल्लो/पछिल्लो अवस्थाबीच फरक तुलना गर्नुहोस्, र लेआउट, टाइपोग्राफी, रङ, तथा पहुँचयोग्यता समेट्ने AI-निर्मित डिजाइन समीक्षा प्राप्त गर्नुहोस्
- **पाँचवटा बिल्ट-इन सहायकहरू** — सामग्री लेखक, साइट निर्माता, डिजाइन स्टुडियो, प्लगइन प्रबन्धक, र सहायता सहायक; च्याट हेडरमा रहेको सहायक चयनकर्ताबाट स्विच गर्न मिल्ने
- **सुविधा फ्ल्यागहरू** — भूमिका प्रतिबन्ध, ह्वाइट-लेबल नामकरण, र विजेट अनुकूलनका लागि सेटिङहरू → सुविधा फ्ल्यागहरूमा पहुँच-नियन्त्रण र ब्रान्डिङ टगलहरू
- **कस्टम पोस्ट प्रकार व्यवस्थापन** — पुनः सुरु भएपछि पनि कायम रहने गरी कस्टम पोस्ट प्रकारहरू दर्ता, सूचीकरण, र मेटाउने
- **कस्टम ट्याक्सोनोमी व्यवस्थापन** — कायम रहने गरी कस्टम ट्याक्सोनोमीहरू दर्ता, सूचीकरण, र मेटाउने
- **डिजाइन प्रणाली क्षमताहरू** — कस्टम CSS इन्जेक्ट गर्नुहोस्, चयनित ब्लक प्याटर्नहरू व्यवस्थापन गर्नुहोस्, साइट लोगो सेट गर्नुहोस्, र theme.json प्रिसेटहरू लागू गर्नुहोस्
- **Global Styles व्यवस्थापन** — रङ प्यालेट, टाइपोग्राफी, स्पेसिङ, र थपका लागि theme.json मानहरू पढ्नुहोस् र लेख्नुहोस्
- **नेभिगेसन मेनु व्यवस्थापन** — नेभिगेसन मेनुहरू र तिनका आइटमहरू प्रोग्रामेटिक रूपमा सिर्जना र अपडेट गर्नुहोस्
- **विकल्प व्यवस्थापन** — महत्वपूर्ण सेटिङहरूको सुरक्षा गर्न बिल्ट-इन सुरक्षा ब्लकलिस्टसहित WordPress विकल्पहरू पढ्नुहोस् र लेख्नुहोस्
- **स्थापना गर्न मिल्ने क्षमताहरूको रजिस्ट्री** — मुख्य प्लगइन फाइलहरू परिमार्जन नगरी थप क्षमता प्याकहरू पत्ता लगाउनुहोस् र सक्रिय गर्नुहोस्
- **साइट निर्माता संयोजन v2** — प्लगइन खोज, प्रगति ट्र्याकिङ, र त्रुटि पुनःस्थापनासहित बहु-चरण साइट निर्माण योजनाहरू
- **WP-CLI बेन्चमार्क कमान्ड** — CI र मोडेल मूल्याङ्कनका लागि कमान्ड लाइनबाट सहायक क्षमता बेन्चमार्क सुइट चलाउनुहोस्

## आवश्यकताहरू

- WordPress 6.0 वा सोभन्दा माथि
- PHP 8.1 वा सोभन्दा माथि
- WordPress Multisite (नेटवर्क-सक्रिय) वा एकल-साइट स्थापना
- सक्रिय AI प्रदायक कनेक्टर प्लगइन (Anthropic, OpenAI, वा मिल्दो)

## स्थापना

1. प्लगइन फाइलहरू `/wp-content/plugins/gratis-ai-agent` मा अपलोड गर्नुहोस्
2. WordPress मा **प्लगइनहरू** स्क्रिनमार्फत प्लगइन सक्रिय गर्नुहोस्
3. **Gratis AI Agent → सेटिङहरू** मा जानुहोस् र आफ्नो AI प्रदायक जडान गर्नुहोस्
4. (वैकल्पिक) थप क्षमता प्याकहरू स्थापना गर्न **क्षमताहरू रजिस्ट्री** ब्राउज गर्नुहोस्

## छिटो सुरु

सक्रिय गरी AI प्रदायकसँग जडान भएपछि, WordPress व्यवस्थापक साइडबारमा **Gratis AI Agent** प्यानल खोल्नुहोस्। प्राकृतिक भाषामा निर्देशन टाइप गर्नुहोस् र सहायकले उपलब्ध क्षमताहरू प्रयोग गरेर आवश्यक चरणहरू योजना बनाएर कार्यान्वयन गर्नेछ।

**उदाहरण निर्देशनहरू:**

```
"Project Category" ट्याक्सोनोमीसहित "Portfolio" कस्टम पोस्ट प्रकार सिर्जना गर्नुहोस्।
```

```
थिममा डार्क-मोड रङ प्यालेट इन्जेक्ट गर्नुहोस् र प्राथमिक रङ #1a1a2e मा सेट गर्नुहोस्।
```

```
अनलाइन मेनु, बुकिङ फारम, र सम्पर्क पृष्ठसहित रेस्टुरेन्ट वेबसाइट निर्माण गर्नुहोस्।
```

## क्षमताहरूको अवलोकन

क्षमताहरू सहायकले गर्न सक्ने आधारभूत कार्यहरू हुन्। प्रत्येक क्षमता दर्ता गरिएको PHP class हो जसले रनटाइममा सहायकले पढ्ने schema (parameter र return type) उपलब्ध गराउँछ। क्षमताहरू कार्यात्मक क्षेत्रहरूमा समूहबद्ध गरिएका छन्:

| क्षेत्र | क्षमताहरू |
|---|---|
| **सामग्री व्यवस्थापन** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **दृश्य समीक्षा** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **कस्टम पोस्ट प्रकारहरू** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **कस्टम ट्याक्सोनोमीहरू** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **डिजाइन प्रणाली** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **नेभिगेसन मेनुहरू** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **विकल्प व्यवस्थापन** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **स्थापना गर्न मिल्ने क्षमताहरू** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **साइट निर्माता** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

पूर्ण parameter कागजात र प्रयोग उदाहरणहरूका लागि [क्षमताहरू सन्दर्भ](./abilities) हेर्नुहोस्।

## WP-CLI

Gratis AI Agent ले कमान्ड लाइनबाट बेन्चमार्क चलाउन र क्षमताहरू व्यवस्थापन गर्न `wp gratis-ai-agent` कमान्ड परिवारसँगै आउँछ। पूर्ण कमान्ड सूचीका लागि [WP-CLI सन्दर्भ](./wp-cli) हेर्नुहोस्।

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## कन्फिगरेसन

सबै सेटिङहरू WordPress व्यवस्थापकमा **Gratis AI Agent → सेटिङहरू** अन्तर्गत छन्:

| सेटिङ | विवरण |
|---|---|
| **AI प्रदायक** | सक्रिय AI प्रदायक कनेक्टर चयन र कन्फिगर गर्नुहोस् |
| **मोडेल** | सहायक सत्रहरूका लागि प्रयोग गर्ने भाषा मोडेल छनोट गर्नुहोस् |
| **अधिकतम टोकनहरू** | प्रत्येक अनुरोधको टोकन बजेट (पूर्वनिर्धारित 4096) |
| **क्षमता टाइमआउट** | एउटा व्यक्तिगत क्षमता चल्न सक्ने अधिकतम सेकेन्ड (पूर्वनिर्धारित 30) |
| **डिबग लगिङ** | क्षमता कलहरू र सहायकको तर्क WordPress debug log मा लग गर्नुहोस् |

## विकल्प व्यवस्थापन सुरक्षा ब्लकलिस्ट

विकल्प व्यवस्थापन क्षमतामा अन्तर्निहित ब्लकलिस्ट समावेश छ जसले एजेन्टलाई संवेदनशील WordPress विकल्पहरू पढ्न वा लेख्नबाट रोक्छ। पूर्वनिर्धारित ब्लकलिस्टले समेट्छ:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL विकल्पहरू — यीका लागि समर्पित स्थानान्तरण उपकरण प्रयोग गर्नुहोस्)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (गोप्य कीहरू)
- `active_plugins`, `template` (प्लगइन/थिम सक्रियता — समर्पित क्षमताहरू प्रयोग गर्नुहोस्)
- स्थापना गरिएका प्लगइनहरूले दर्ता गरेका भुक्तानी गेटवे प्रमाणहरू र API कीहरू

थप प्रविष्टिहरू `gratis_ai_agent_options_blocklist` फिल्टरमार्फत थप्न सकिन्छ।

## हुकहरू र फिल्टरहरू

### कार्यहरू

- `gratis_ai_agent_ability_registered` — क्षमता दर्ता भएपछि चलाइन्छ, क्षमता वर्गको नाम प्राप्त गर्छ
- `gratis_ai_agent_plan_started` — साइट निर्माणकर्ता योजना कार्यान्वयन सुरु हुँदा चलाइन्छ
- `gratis_ai_agent_plan_completed` — योजना सफलतापूर्वक समाप्त हुँदा चलाइन्छ
- `gratis_ai_agent_plan_failed` — योजनाले पुनःप्राप्त गर्न नसकिने त्रुटि सामना गर्दा चलाइन्छ

### फिल्टरहरू

- `gratis_ai_agent_abilities` — एजेन्टले लोड गर्नु अघि दर्ता गरिएको क्षमता सूची परिमार्जन गर्नुहोस्
- `gratis_ai_agent_options_blocklist` — विकल्प सुरक्षा ब्लकलिस्ट विस्तार वा घटाउनुहोस्
- `gratis_ai_agent_benchmark_questions` — मूल्याङ्कन सुइटमा अनुकूल बेन्चमार्क प्रश्नहरू थप्नुहोस्
- `gratis_ai_agent_plan_plugins` — साइट योजना उत्पादनको क्रममा प्लगइन खोज सूची परिमार्जन गर्नुहोस्
