# Filter: wu_apc_should_display_admin_menu

यदि अनुकूलन प्रक्रिया समाप्त गरेका छन् भने, प्रशासकहरूलाई admin menu पृष्ठहरू लुकाउन अनुमति दिनुहोस्।

## प्यारामिटरहरू

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $should_display_admin_menu | `bool` | हामीले पृष्ठहरू देखाउन आवश्यक छ कि छैन। |
| $network_active | `bool` | plugin network admin सक्रिय छ कि छैन जाँच गर्नुहोस्। |
