# फिल्टर: wu_cap_captcha_config {#filter-wucapcaptchaconfig}

Cap Captcha कन्फिगरेसन फिल्टर गर्नुहोस्।

विकासकर्ताहरूलाई Cap Captcha सेटिङहरू अनुकूलित गर्न अनुमति दिन्छ।

## प्यारामिटरहरू {#parameters}

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $config | `array` | Cap कन्फिगरेसन array। |
| $level | `string` | सुरक्षा स्तर (fast, medium, max)। |
