# थीम बिल्डर क्षमताहरू: स्कैफोल्ड र सक्रिय ब्लक темиहरू {#theme-builder-abilities-scaffold-and-activate-block-themes}

Superdav AI Agent v1.12.0 ले दुई शक्तिशाली क्षमताहरू प्रस्तुत गर्दछ जसले तपाईंलाई च्याट इन्टरफेसबाट नै कस्टम ब्लक темиहरू बनाउन र डिप्लॉय गर्न अनुमति दिन्छ।

## अवलोकन {#overview}

**scaffold-block-theme** र **activate-theme** क्षमताहरूले एजेन्टहरूलाई निम्न कार्यहरू गर्न सक्षम बनाउँछन्:
- तपाईंको विवरण अनुसार पूर्ण, उत्पादन-तयार ब्लक темиहरू उत्पन्न गर्ने
- कुनै पनि मैन्युअल हस्तक्षेप बिना साइटमा स्वचालित रूपमा теми सक्रिय गर्ने
- निर्देशित डिजाइन निर्णयहरू मार्फत सुसंगत दृश्य पहिचानहरू सिर्जना गर्ने

## स्कैफोल्ड ब्लक थीम (Scaffold Block Theme) {#scaffold-block-theme}

**scaffold-block-theme** क्षमताले पूर्ण विषय संरचना सहित नयाँ WordPress ब्लक थीम उत्पन्न गर्दछ, जसमा निम्न कुराहरू समावेश हुन्छन्:

- डिजाइन टोकनहरूसहित `theme.json` कन्फिगरेसन
- सामान्य लेआउटका लागि ब्लक टेम्प्लेट फाइलहरू
- कस्टम ब्लक शैली र भिन्नताहरू
- विषय मेटाडेटा र समर्थन घोषणाहरू

### कसरी आह्वान गर्ने (How to Invoke) {#how-to-invoke}

Superdav AI Agent सँगको तपाईंको च्याटमा, तपाईंले थीम निर्माणको अनुरोध गर्न सक्नुहुन्छ:

```
"एउटा 'Modern Agency' नामको ब्लक थीम बनाऊ, जसको रंग योजना नीलो र सेतो हो,
sans-serif टाइपोग्राफी प्रयोग गरियोस्, र एक व्यावसायिक लेआउट भएको हो।"
```

एजेन्टले निम्न कार्यहरू गर्नेछ:
1. कुराकानी मार्फत तपाईंको डिजाइन प्राथमिकताहरू सङ्कलन गर्ने
2. पूर्ण विषय संरचना उत्पन्न गर्ने
3. सबै आवश्यक विषय फाइलहरू बनाउने
4. सक्रियताका लागि विषय तयार गर्ने

### अपेक्षित आउटपुट (Expected Output) {#expected-output}

जब क्षमता सफलतापूर्वक निष्पादित हुन्छ, तपाईंले निम्न देख्नुहुनेछ:

- थीम स्कैफोल्ड गरियो भन्ने पुष्टि
- थीमको नाम र स्थान
- लागू गरिएका डिजाइन टोकनहरूको सारांश (रङहरू, टाइपोग्राफी, स्पेस)
- सक्रिय गर्न तयार भएको स्थिति

उदाहरण आउटपुट:
```
✓ Theme "Modern Agency" सफलतापूर्वक स्कैफोल्ड गरियो
  Location: /wp-content/themes/modern-agency/
  Colors: Primary #0066CC, Secondary #FFFFFF
  Typography: Inter (sans-serif)
  Status: सक्रिय गर्न तयार
```

## थीम सक्रिय गर्ने (Activate Theme) {#activate-theme}

**activate-theme** क्षमताले तपाईंको साइटलाई नयाँ बनाइएको वा पहिले नै बनेको ब्लक थीममा स्विच गर्न प्रयोग गरिन्छ।

### कसरी प्रयोग गर्ने {#how-to-invoke-1}

थीम बनाएपछि, तपाईं यसलाई तुरुन्तै सक्रिय (activate) गर्न सक्नुहुन्छ:

```
"Modern Agency theme लाई सक्रिय गर्नुहोस्"
```

वा कुनै पनि पहिले नै रहेको थीम सक्रिय गर्न:

```
"Twentytwentyfour theme मा स्विच गर्नुहोस्"
```

### अपेक्षित आउटपुट {#expected-output-1}

सक्रिय गर्दा सफल भएमा:

- सक्रिय भएको थीमको पुष्टि
- अघिल्लो थीमको नाम (सन्दर्भका लागि)
- थीम अहिले सञ्चालन भइरहेको साइट URL
- कुनै पनि थीम-विशिष्ट सेटअप नोटहरू

उदाहरण आउटपुट:
```
✓ Theme सफलतापूर्वक सक्रिय गरियो
  सक्रिय थीम: Modern Agency
  अघिल्लो थीम: Twentytwentyfour
  लाइभ छ: https://yoursite.com
  नोट: लेआउट पुष्टि गर्न आफ्नो होमपेज जाँच गर्नुहोस्
```

## कार्यप्रवाह: बनाउने र सक्रिय गर्ने (Scaffold and Activate) {#workflow-scaffold-and-activate}

सामान्य कार्यप्रवाहमा यी दुवै क्षमताहरू मिलाइन्छ:

१. **थीम निर्माणको अनुरोध**: "मेरो SaaS लन्डिङ पेजका लागि ब्लक थीम बनाउनुहोस्"
२. **एजेन्टले थीम बनाउँछ (scaffolds)**: फाइलहरू र डिजाइन टोकनहरू उत्पन्न गर्दछ।
३. **समीक्षा र परिमार्जन**: आवश्यक भएमा डिजाइन परिवर्तनहरूको बारेमा छलफल गर्नुहोस्।
४. **सक्रिय गर्ने**: "थीम अहिले सक्रिय गर्नुहोस्"
५. **परिचय जाँच गर्ने**: नयाँ डिजाइन सञ्चालन भयो कि भनी आफ्नो साइटमा जानुहोस्।

## डिजाइन टोकन र अनुकूलन (Design Tokens and Customization) {#design-tokens-and-customization}

बनाइएका थीमहरूले WordPress डिजाइन टोकनहरू (`theme.json` मार्फत) प्रयोग गर्छन् जसको लागि निम्न कुराहरू हुन्छन्:

- **रङहरू**: प्राथमिक, द्वितीयक, उच्चारण (accent), तटस्थ प्यालेट
- **टाइपोग्राफी**: फन्ट परिवार, आकार, वजन (weights), लाइन उच्चता (line heights)
- **स्पेसिंग**: प्याडिङ, मार्जिन, ग्याप स्केलहरू
- **बर्डरहरू**: त्रिज्या र चौडाइ टोकनहरू
- **छायाहरू (Shadows)**: उत्थानको स्तर

यी टोकनहरू `theme.json` मा केन्द्रित हुन्छन्, जसले गर्दा एउटै फाइलबाट आफ्नो सम्पूर्ण डिजाइन प्रणालीलाई सजिलै मिलाउन सकिन्छ।

## सीमाहरू र नोटहरू {#limitations-and-notes}

थीमहरू `/wp-content/themes/` मा ढाँचा (scaffold) गरिन्छ र तिनीले वर्डप्रेसको नामकरण नियमहरू पालना गर्नुपर्छ।
सक्रिय (Activation) गर्न तपाईंको वर्डप्रेस साइटमा उपयुक्त अनुमतिहरू हुनुपर्छ।
थीमहरूमा कस्टम PHP कोड न्यूनतम प्रयोग गर्नुहोस्; जटिल कार्यहरूको लागि प्लगइन प्रयोग गर्नुहोस्।
ब्लक थीमहरू वर्डप्रेस ५.९ र त्यसपछि राम्रोसँग काम गर्छन्।

## समस्या समाधान (Troubleshooting) {#troubleshooting}

**ढाँचा बनाएपछि थीम देखिँदैन**
- थीमको डाइरेक्टरी छ कि छैन र त्यसमा सही अनुमतिहरू छन् कि छैनन्, जाँच गर्नुहोस्।
- `theme.json` वैध JSON हो कि होइन, जाँच गर्नुहोस्।
- सुनिश्चित गर्नुहोस् कि थीमको नाम पहिले नै भएका अन्य थीमहरूसँग मेल खाएको छैन।

**सक्रिय (Activation) असफल हुन्छ**
- तपाईंसँग एडमिन अनुमतिहरू छन् भने पुष्टि गर्नुहोस्।
- वर्डप्रेसले थीम डाइरेक्टरी पढ्न सक्ने छ कि छैन, जाँच गर्नुहोस्।
- विवरणका लागि वर्डप्रेसको एरर लगहरू (error logs) हेर्नुहोस्।

**डिजाईन टोकन लागू हुँदैन**
- `theme.json` को सिन्ट्याक्स सही छ वा छैन, जाँच गर्नुहोस्।
- कुनै पनि caching प्लगइन सफा गर्नुहोस्।
- तपाईंले प्रयोग गरिरहनुभएको टोकनहरू वर्डप्रेसको संस्करणले समर्थन गर्छ कि गर्दैन, जाँच गर्नुहोस्।

## अर्को चरण (Next Steps) {#next-steps}

तपाईंको थीम सक्रिय गरेपछि, तपाईं निम्न कुराहरू गर्न सक्नुहुन्छ:
- टाइपोग्राफी, रंग र स्पेसलिङलाई परिष्कृत गर्न **Design System Aesthetics skill** प्रयोग गर्नुहोस्।
- वर्डप्रेस ब्लक एडिटर मार्फत व्यक्तिगत ब्लक शैलीहरू अनुकूलित (customize) गर्नुहोस्।
- थीमको `style.css` फाइलमा कस्टम CSS थप्नुहोस्।
- विशिष्ट पेज प्रकारहरूको लागि कस्टम ब्लक टेम्प्लेटहरू बनाउनुहोस्।
