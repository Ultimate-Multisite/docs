# प्रो थीम प्रयोगकर्ताको एडमिन प्यानेललाई कस्टमाइज़ गर्नुहोस्

ग्राहकहरूको काम गर्दा हामीलाई प्रायः वर्डप्रेसको एडमिन ड्यासबोर्डमा आफ्ना ग्राहकहरूलाई कस्टम अनुभव दिनुपर्ने आवश्यकता पर्छ। यदि तपाईं अझ व्यावसायिक लुक र फील खोज्दै हुनुहुन्छ भने, PRO Theme ले मद्दत गर्न सक्छ।

PRO Theme एक एडमिन थीम हो जुन तपाईंलाई आफ्ना ग्राहकहरूको ड्यासबोर्ड कस्टमाइज़ गर्दा सहयोग गर्न विकसित गरिएको हो। यो Ultimate Multisite को प्रीमियम प्लगइनको मुख्य भाग पनि हो।

यसलाई इन्स्टल गरेपछि, आफ्नो _Plugins_ पेजमा जानुहोस् र आफ्नो नेटवर्कमा सक्रिय (activate) गर्न क्लिक गर्नुहोस्।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_आफ्नो Plugins पेजमा नेटवर्क-व्यापी रूपमा यसलाई सक्रिय गर्नुहोस्_

_**नोट:**_ _यदि तपाईंले यो इन्स्टल गर्नुभयो भने, यसले तपाईंको ग्राहकहरूका लागि एक विकल्पको रूपमा देखिनेछ जसलाई केवल त्यो प्रयोगकर्ताको लागि थीम थप्न चिह्नित गर्न सकिन्छ._

अब, तपाईंको एडमिन इन्टरफेस पूर्ण रूपमा नयाँ डिजाइन गरिएको छ! बायाँ स्तम्भमा 'PRO Theme' नामक एउटा नयाँ मेनु पनि तपाईंसँग छ।
