# म मेरो सदस्यता कसरी रद्द गर्ने?

यदि तपाईंसँग Ultimate Multisite लाइसेन्स छ भने, तपाईं आफ्नो नवीकरण कुनै पनि समयमा रद्द गर्न सक्नुहुन्छ। बस तलका चरणहरू पालना गर्नुहोस्:

<!-- Screenshot unavailable: Account page email with access link -->

तपाईंले खरिद गर्दा दिनुभएको इमेल र पासवर्ड प्रयोग गर्नुहोस्।

<!-- Screenshot unavailable: Login page with email and password fields -->

यो तपाईंको Account Page को सुरुवाती स्क्रिन हो:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

आफ्नो सदस्यता रद्द गर्न, बायाँ मेनुमा गएर _Renewing & Billing._ मा जानुहोस्।

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

साइड विन्डो खोल्न दाहिने तीरमा क्लिक गर्नुहोस्। त्यसपछि, तपाईंले **auto-renew** रद्द गर्ने विकल्प चयन गर्नुपर्नेछ।

सिस्टमले तपाईंलाई एउटा पुष्टि सन्देश देखाउनेछ।

यदि तपाईं आफ्नो सदस्यता रद्द गर्न निश्चित हुनुहुन्छ भने, _**Cancel Renewals**_ बटनमा क्लिक गर्नुहोस्। यो कार्य गरेपछि, तपाईंलाई छिटो सर्वेक्षण भर्न अनुरोध गरिनेछ।

भयो! तपाईंको सदस्यता स्वचालित रूपमा नवीकरण हुने छैन।

**तपाईंसँग सदस्यता समाप्त नहुन्जेलसम्म एक वैध key रहनेछ।** यदि तपाईं आफ्नो सदस्यता पुनः सक्रिय गर्न चाहनुहुन्छ भने, तपाईंले त्यो म्यानुअल रूपमा गर्नुपर्नेछ।
