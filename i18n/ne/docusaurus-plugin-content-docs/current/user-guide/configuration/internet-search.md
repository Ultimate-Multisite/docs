# इन्टरनेट खोज (Internet Search) {#internet-search}

Gratis AI Agent v1.5.0 मा एउटा **इन्टरनेट सर्च** क्षमता थपिएको छ जसले च्याटको क्रममा AI सहायकलाई वेबबाट ताजा जानकारी निकाल्न मद्दत गर्छ। यो [Brave Search API](https://brave.com/search/api/) वा [Tavily API](https://tavily.com/) बाट संचालित हुन्छ।

## यो कसरी काम गर्छ {#how-it-works}

जब इन्टरनेट खोज सक्षम गरिन्छ, यदि प्रश्नको जवाफलाई वर्तमान वा बाह्य जानकारी चाहिन्छ भने (जस्तै: पछिल्लो समाचार, लाइभ मूल्य निर्धारण, वा मोडेलको प्रशिक्षण कटअफ पछि परिवर्तन हुन सक्ने कागजातहरू), सहायकले स्वचालित रूपमा तपाईंले कन्फिगर गर्नुभएको खोज प्रदायकलाई प्रश्न गर्छ।

परिणामहरू वास्तविक समयमा प्राप्त गरिन्छ र जवाफ उत्पन्न गर्नुअघि सहायकको सन्दर्भ (context) मा समावेश गरिन्छ। जब कुनै प्रश्नको जवाफ दिनका लागि यसले सर्च परिणाम प्रयोग गरेको संकेत दिन्छ।

## इन्टरनेट खोज सक्षम गर्ने {#enabling-internet-search}

इन्टरनेट खोजका लागि तपाईंले रोजेको खोज प्रदायकबाट एउटा API key चाहिन्छ। यसलाई कन्फिगर गर्न:

१. **Gratis AI Agent → Settings → Advanced** मा जानुहोस्।
२. **Internet Search Provider** ड्रपडाउन पत्ता लगाउनुहोस् र **Brave Search** वा **Tavily** मध्ये कुनै एक चयन गर्नुहोस्।
३. आफ्नो API key सम्बन्धित फिल्डमा प्रविष्ट गर्नुहोस्। प्रत्येक फिल्डको छेउमा साइन-अप लिङ्क देखाइएको हुन्छ।
४. **Save Settings** मा क्लिक गर्नुहोस्।

एक पटक की (key) सुरक्षित भएपछि, इन्टरनेट खोज क्षमता स्वचालित रूपमा सहायकका लागि उपलब्ध हुन्छ।

## Brave Search {#brave-search}

### Brave Search API Key कसरी प्राप्त गर्ने {#obtaining-a-brave-search-api-key}

१. [Brave Search API page](https://brave.com/search/api/) मा जानुहोस्।
२. एउटा प्लानको लागि साइन अप गर्नुहोस्। मासिक अनुरोध सीमास साथै निःशुल्क टियर उपलब्ध छ।
३. Brave Search डेवलपर ड्यासबोर्डबाट आफ्नो API key कपि गर्नुहोस्।
४. Gratis AI Agent सेटिङहरूमा **Brave Search API Key** फिल्डमा यसलाई पेस्ट गर्नुहोस्।

### प्रयोगको सीमा (Usage Limits) {#usage-limits}

उपयोगको बिल Brave Search ले गरिएका क्वेरीहरूको संख्यामा आधारित हुन्छ। प्रत्येक AI प्रतिक्रिया जसले सर्च ट्रिगर गर्छ, त्यसलाई एक क्वेरी मानिन्छ। अप्रत्याशित शुल्कबाट बच्न आफ्नो प्रयोग [Brave Search developer dashboard](https://brave.com/search/api/) मा निगरानी गर्नुहोस्।

## Tavily {#tavily}

Superdav AI Agent v1.10.0 ले **Tavily** लाई एउटा वैकल्पिक इन्टरनेट सर्च प्रदायकको रूपमा थप्छ, जसले अझ विस्तृत खोज परिणाम र उन्नत अनुसन्धान क्षमताहरू प्रदान गर्दछ।

### Tavily API Key कसरी प्राप्त गर्ने {#obtaining-a-tavily-api-key}

१. [Tavily API page](https://tavily.com/) मा जानुहोस्।
२. एउटा खाताको लागि साइन अप गर्नुहोस्। मासिक अनुरोध सीमा सहित निःशुल्क टियर उपलब्ध छ।
३. Tavily ड्यासबोर्डबाट आफ्नो API key कपी गर्नुहोस्।
४. Gratis AI Agent सेटिङहरूमा **Tavily API Key** फिल्डमा पेस्ट गर्नुहोस्।

### प्रयोगको सीमा {#usage-limits-1}

प्रयोग Tavily ले गरिएका API कलहरूको संख्यामा आधारित हुन्छ। प्रत्येक AI प्रतिक्रिया जसले सर्च ट्रिगर गर्छ, त्यसलाई एक कल मानिन्छ। अप्रत्याशित शुल्कबाट बच्न आफ्नो प्रयोग [Tavily dashboard](https://tavily.com/) मा निगरानी गर्नुहोस्।

## इन्टरनेट सर्च अक्षम गर्ने {#disabling-internet-search}

सक्रिय सर्च प्रदायक फिल्डबाट API key हटाउनुहोस् र सेभ गर्नुहोस्। यसले असिस्टेन्टलाई इन्टरनेट सर्चको क्षमता प्रदान गर्दैन।

:::note
इन्टरनेट सर्चले प्रतिक्रियाहरूमा ढिलाइ (latency) थप्छ किनभने असिस्टेन्टले जवाफ उत्पन्न गर्नुअघि सर्च परिणामको लागि पर्खनुपर्छ। समय संवेदनशील प्रयोगका लागि, वास्तविक-समय खोज आवश्यक छ कि छैन वा असिस्टेन्टको भित्री ज्ञान पर्याप्त छ कि छैन भन्ने कुरा विचार गर्नुहोस्।
:::
