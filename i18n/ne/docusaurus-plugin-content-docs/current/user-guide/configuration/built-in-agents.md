# Built-in Agents

Gratis AI Agent v1.9.0 मा पाँच बिल्ट-इन एजेन्टहरू उपलब्ध छन्, जसमध्ये प्रत्येकलाई त्यस क्षेत्रका सामान्य कामहरूको लागि तयार गरिएका उपकरणहरू (tools), एक विशेष सिस्टम प्रम्प्ट (system prompt) र सुरु गर्न मद्दत गर्ने सुझावहरू (suggestions) सहित पहिले नै कन्फिगर गरिएको हुन्छ। एजेन्टहरू बीच स्विच गर्दा सहायकले के गर्न सक्छ र कसरी प्रतिक्रिया दिन्छ भन्ने कुरा परिवर्तन हुन्छ — तपाईंको कुनै पनि सेटिङ नगरी।

## एजेन्ट भनेको के हो?

प्रत्येक एजेन्ट एक नाम दिइएको कन्फिगरेशन प्रोफाइल हो जसले निम्न कुराहरू मिलाउँछ:

- **Tools** — एजेन्टलाई प्रयोग गर्न अनुमति भएका क्षमताहरू (जस्तै, सामग्री लेखकमा पोस्ट बनाउने क्षमता; डिजाइन स्टुडियोमा CSS र theme.json को पहुँच)।
- **System prompt** — एजेन्टको टोन, प्राथमिकता र सीमाहरू सेट गर्ने निर्देशनहरू।
- **Suggestions** — तपाईंलाई छिटो सुरु गर्न मद्दत गर्न च्याट इन्टरफेसमा देखाइने पहिले नै लेखिएका प्रम्प्टहरू।

## एजेन्ट पिकर कसरी प्रयोग गर्ने?

१. वर्डप्रेस एडमिन साइडबारमा **Gratis AI Agent** प्यानल खोल्नुहोस्।
२. च्याट हेडरको माथि-बाँहिरी कुनामा रहेको **एजेन्ट आइकन** मा क्लिक गर्नुहोस् (आइकन सक्रिय एजेन्टलाई प्रतिबिम्बित गर्न परिवर्तन हुन्छ)।
३. **Agent Picker** एक form-table ओभरलेको रूपमा खुल्छ। प्रत्येक एजेन्टलाई उसको आइकन, नाम र एक लाइनको विवरणका साथ सूचीबद्ध गरिएको हुन्छ।
४. सक्रिय गर्न कुनै एजेन्टको पंक्तिमा क्लिक गर्नुहोस्। च्याट हेडर तुरुन्तै अपडेट हुन्छ।

तपाईं कुराकानीको बीचमै एजेन्टहरू बदल्न पनि सक्नुहुन्छ — नयाँ एजेन्टको सिस्टम प्रम्प्ट अर्को सन्देशदेखि लागू हुन्छ।

## पाँच बिल्ट-इन एजेन्टहरू

### Content Writer (सामग्री लेखक)

**फोकस:** पोस्ट, पेज र सम्पर्क फारमहरू बनाउने र सम्पादन गर्ने।

**उपलब्ध उपकरणहरू:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**यसले राम्रो काम गर्छ:**
- संक्षिप्त विवरण वा रूपरेखाबाट ब्लग पोस्टहरू ड्राफ्ट गर्ने र प्रकाशित गर्ने
- नयाँ साइटका लागि लान्जिङ पेजहरूको समूह बनाउने
- सम्पर्क र पूछताछ फारमहरू बनाउने

**प्रारम्भिक सुझावहरू:**
- *वर्डप्रेस मल्टिसाइटका फाइदाहरूबारे ५०० शब्दको ब्लग पोस्ट लेख्नुहोस्।*
- *एउटा 'About', 'Services', र 'Contact' पेज बनाउनुहोस् र प्रकाशित गर्नुहोस्।*
- *सम्पर्क पेजमा एक बुकिङ पूछताछ फारम थप्नुहोस्।*

---

### साइट बिल्डर (Site Builder)

**फोकस:** एउटै आदेशबाट सम्पूर्ण वेबसाइट निर्माण गर्ने।

**उपलब्ध उपकरणहरू:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**यसले राम्रो काम गर्छ:**
- वर्णन गरिएको व्यवसायको प्रकारका लागि धेरै चरणको साइट निर्माण योजना (site build plan) उत्पन्न गर्ने।
- प्रत्येक चरणलाई स्वचालित रूपमा चलाउने — संरचना, सामग्री, नेभिगेसन, डिजाइन।
- योजनाको बीचमा त्रुटिहरूबाट पनि रिकभर हुने, जसका लागि म्यानुअल हस्तक्षेप आवश्यक पर्दैन।
- निर्माणको एक भागको रूपमा सिफारिस गरिएका प्लगइनहरू इन्स्टल गर्ने।
- च्याट इन्टरफेसबाटै सम्पर्क फारमहरू बनाउने (Superdav AI Agent v1.10.0+ मा)।

**प्रारम्भिक सुझावहरू:**
- *एक ग्यालरी पोस्ट प्रकार, बुकिङ पेज र एक सम्पर्क फारम भएको फोटोग्राफी पोर्टफोलियो साइट बनाउनुहोस्।*
- *अनलाइन मेनु, खुला समय र टेबल बुकिंग पूछताछ फारम सहित एउटा रेस्टुरेन्ट वेबसाइट बनाउनुहोस्।*
- *सेवा पृष्ठहरू, पोर्टफोलियो खण्ड र ब्लग सहित फ्रीलांस परामर्श साइट सेटअप गर्नुहोस्।*
- *साइट बिल्डर प्रयोग गरेर सम्पर्क पेजमा एक सम्पर्क फारम थप्नुहोस्।*

### डिजाइन स्टुडियो (Design Studio)

**फोकस:** दृश्य अनुकूलन — रंग, टाइपोग्राफी, CSS, र ब्लक प्याटर्नहरू।

**उपलब्ध उपकरणहरू:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**यसले राम्रोसँग के गर्छ:**
- नाम दिइएका थीम प्रेसेट्सहरू (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog) लागू गर्ने।
- theme.json मार्फत ग्लोबल टाइपोग्राफी र कलर प्यालेटलाई मिलाउने।
- ब्रान्ड-विशिष्ट ओभरराइडहरूको लागि कस्टम CSS इन्जेक्ट गर्ने।
- पेजको स्क्रिनशट लिने र डिजाइन समस्याहरूका लागि त्यसको समीक्षा गर्ने।

**प्रारम्भिक सुझावहरू:**
- *warm-editorial प्रेसेट लागू गर्नुहोस् र त्यसपछि प्राथमिक रंग #2d6a4f सेट गर्नुहोस्।*
- *होमपेजको स्क्रिनशट लिनुहोस् र मलाई के सुधार गर्नुपर्छ भन्नुहोस्।*
- *पूर्ण-चौडाइको पृष्ठभूमि छवि र केन्द्रित शीर्षक भएको पुन: प्रयोग गर्न सकिने (reusable) हीरो ब्लक प्रेसर बनाउनुहोस्।*

### प्लगइन मेनेजर

**फोकस:** WordPress प्लगइनहरू पत्ता लगाउने, इन्स्टल गर्ने र व्यवस्थापन गर्ने।

**उपलब्ध उपकरणहरू:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**यसले राम्रोसँग के गर्छ:**
- वर्णन गरिएको प्रयोगको लागि सबैभन्दा उपयुक्त प्लगइन सिफारिस गर्ने।
- रजिस्ट्रीबाट एबिलिटी प्याकहरू इन्स्टल गर्ने।
- श्रेणी अनुसार उपलब्ध एबिलिटी क्याटलग ब्राउज गर्ने।

**प्रारम्भिक सुझावहरू:**
- *सदस्यता निर्देशिका (membership directory) को लागि सबैभन्दा राम्रो प्लगइन कुन हो?*
- *WooCommerce एबिलिटी प्याक इन्स्टल गर्नुहोस्।*
- *मलाई सबै उपलब्ध ई-कमर्स एबिलिटी प्याक देखाउनुहोस्।*

---

### सपोर्ट असिस्टेन्ट

**फोकस:** साइटको सामग्री, सेटिङहरू र WordPress कन्फिगरेसनका बारेमा प्रश्नहरूको उत्तर दिने।

**उपलब्ध उपकरणहरू:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**यो के राम्रो गर्छ:**
- साइटका अहिलेको सेटिङ र विकल्पहरू हेर्न मद्दत गर्छ।
- साइटमा कुन post types, taxonomies, र menus कस्तो कन्फिगर गरिएको छ भनेर बुझाउँछ।
- लाइभ मान (live values) पढ्दा "यो सेटिङले के गर्छ?" जस्ता प्रश्नहरूको जवाफ दिन्छ।
- परिवर्तन गर्नुअघि पढ्न मात्र प्रयोग गर्न सकिने डायग्नोस्टिक लेयरको रूपमा काम गर्छ।

**शुरुआती सुझावहरू:**
- *यस साइटमा अहिले कुन plugins र सेटिङहरू सक्रिय छन्?*
- *यस साइटमा दर्ता गरिएका सबै custom post types (कस्टम पोस्ट प्रकारहरू) सूचीबद्ध गर्नुहोस्।*
- *कुन नेभिगेसन मेनुहरू छन् र तिनीहरू कहाँ संलग्न छन्?*

---

## एजेन्टहरू अनुकूलन गर्ने (Customising Agents)

प्रत्येक built-in agent लाई `gratis_ai_agent_agents` फिल्टर मार्फत विस्तार गर्न वा प्रतिस्थापन गर्न सकिन्छ।

### विद्यमान एजेन्टमा कस्टम सिस्टम prompt थप्ने

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### नयाँ एजेन्ट दर्ता गर्ने

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'खोज इञ्जिनहरूको लागि पोस्ट र पेजहरूलाई अनुकूलित गर्दछ।',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'तपाईं एउटा SEO विशेषज्ञ हुनुहुन्छ। कीवर्ड अनुकूलन, मेटा विवरण (meta descriptions), र स्ट्रक्चर्ड डेटामा ध्यान केन्द्रित गर्नुहोस्।',
        'suggestions'   => [
            'होमपेजको शीर्षक र मेटा विवरणको समीक्षा गर्नुहोस्।',
            'अन्तिम पाँच पोस्टहरूको लागि शीर्षक ट्याग सुधारका सुझावहरू दिनुहोस्।',
        ],
    ];
    return $agents;
} );
```

नयाँ एजेन्टे तपाईंले फिल्टर चलाउनुभ तुरुन्तै एजेन्ट पिकरमा देखिने गर्छ।

### बिल्ट-इन एजेन्ट हटाउने

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
