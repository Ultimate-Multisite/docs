# WP Engine ইন্টিগ্রেশন {#wp-engine-integration}

## ওভারভিউ {#overview}
WP Engine হলো একটি প্রিমিয়াম ম্যানেজড ওয়ার্ডপ্রেস হোস্টিং প্ল্যাটফর্ম যা ওয়ার্ডপ্রেস সাইটগুলির জন্য অপ্টিমাইজ করা পারফরম্যান্স, নিরাপত্তা এবং স্কেলেবিলিটি প্রদান করে। এই ইন্টিগ্রেশনটি Ultimate Multisite এবং WP Engine-এর মধ্যে স্বয়ংক্রিয় ডোমেইন সিঙ্কিং সক্ষম করে।

## বৈশিষ্ট্যসমূহ {#features}
- স্বয়ংক্রিয় ডোমেইন সিঙ্কিং
- মাল্টিসাইট ইনস্টলেশনের জন্য সাবডোমেইন সাপোর্ট
- WP Engine-এর বিদ্যমান সিস্টেমগুলির সাথে নির্বিঘ্ন ইন্টিগ্রেশন

## প্রয়োজনীয়তা {#requirements}
ইন্টিগ্রেশনটি স্বয়ংক্রিয়ভাবে সনাক্ত করে যে আপনি WP Engine-এ হোস্টিং করছেন এবং বিল্ট-ইন WP Engine API ব্যবহার করে। যদি WP Engine প্লাগইন সক্রিয় থাকে এবং সঠিকভাবে কনফিগার করা থাকে তবে কোনো অতিরিক্ত কনফিগারেশনের প্রয়োজন হয় না।

তবে, যদি আপনার ইন্টিগ্রেশনটি ম্যানুয়ালি কনফিগার করার প্রয়োজন হয়, তাহলে আপনি আপনার `wp-config.php` ফাইলে এই ধ্রুবকগুলির মধ্যে যেকোনো একটি সংজ্ঞায়িত করতে পারেন:

```php
define('WPE_APIKEY', 'your_api_key'); // পছন্দের পদ্ধতি
// অথবা
define('WPE_API', 'your_api_key'); // বিকল্প পদ্ধতি
```

## সেটআপ নির্দেশাবলী {#setup-instructions}

### ১. WP Engine প্লাগইন যাচাই করুন {#1-verify-wp-engine-plugin}

যদি আপনি WP Engine-এ হোস্টিং করেন, তবে WP Engine প্লাগইনটি ইতিমধ্যেই ইনস্টল এবং সক্রিয় থাকা উচিত। নিম্নলিখিত বিষয়গুলি যাচাই করুন:

1. WP Engine প্লাগইনটি সক্রিয় আছে কিনা
2. ফাইলটি `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` বিদ্যমান কিনা

### ২. ইন্টিগ্রেশন সক্ষম করুন {#2-enable-the-integration}

1. আপনার ওয়ার্ডপ্রেস অ্যাডমিন থেকে, Ultimate Multisite > Settings এ যান
2. "Domain Mapping" ট্যাবে নেভিগেট করুন
3. "Host Integrations"-এ স্ক্রল করে নিচে নামুন
4. WP Engine ইন্টিগ্রেশন সক্ষম করুন
5. "Save Changes" पर क्लिक करें

## এটি কিভাবে কাজ করে {#how-it-works}

### ডোমেইন সিঙ্কিং {#domain-syncing}

যখন Ultimate Multisite-এ একটি ডোমেইন ম্যাপ করা হয়:

1. ইন্টিগ্রেশনটি আপনার WP Engine ইনস্টলেশনে ডোমেইনটি যোগ করার জন্য WP Engine API ব্যবহার করে
2. WP Engine ডোমেইন কনফিগারেশন এবং SSL সার্টিফিকেট ইস্যু পরিচালনা করে
3. যখন একটি ডোমেইন ম্যাপিং সরানো হয়, তখন ইন্টিগ্রেশনটি WP Engine থেকে সেই ডোমেইনটি সরিয়ে দেবে

### সাবডোমেইন সাপোর্ট {#subdomain-support}

सबडोमेन मल्टिसिट इन्स्टलेसनका लागि:

१. नयाँ साइट सिर्जना गर्दा यो इंटीग्रेशन प्रत्येक सबडोमेनलाई WP Engine मा जोड्छ।
२. WP Engine ले सबडोमेन कन्फिगरेसन ह्यान्डल गर्छ।
३. कुनै साइट डिलिट गर्दा, integration ले त्यो सबडोमेनलाई WP Engine बाट हटाउँछ।

## महत्त्वपूर्ण नोटहरू {#important-notes}

### वाइल्डकार्ड डोमेन (Wildcard Domains) {#wildcard-domains}

सबडोमेन मल्टिसिट इन्स्टलेसनका लागि, तपाईंले वाइल्डकार्ड डोमेन कन्फिगरेसनको लागि WP Engine सपोर्टलाई सम्पर्क गर्न सिफारिस गरिन्छ। यसले प्रत्येक सबडोमेनलाई छुट्टाछुट्टै थप्नुपर्ने आवश्यकता बिना सबै सबडोमेनहरू स्वचालित रूपमा काम गर्न अनुमति दिन्छ।

### SSL प्रमाणपत्रहरू (SSL Certificates) {#ssl-certificates}

यो integration मार्फत थपिएका सबै डोमेनहरूको लागि WP Engine ले SSL प्रमाणपत्र जारी गर्ने र नवीकरण गर्ने कार्य स्वचालित रूपमा गर्छ। कुनै अतिरिक्त कन्फिगरेसन आवश्यक छैन।

## समस्या निवारण (Troubleshooting) {#troubleshooting}

### API कनेक्टिभिटी समस्याहरू {#api-connection-issues}
- WP Engine plugin सक्रिय छ र सही तरिकाले कन्फिगर गरिएको छ कि हेर्नुहोस्।
- यदि तपाईंले मैन्युअल रूपमा API key परिभाषित गर्नुभएको छ भने, त्यो सही छ कि जाँच गर्नुहोस्।
- यदि तपाईंलाई API सँग कुनै समस्या भइरहेको छ भने, WP Engine सपोर्टलाई सम्पर्क गर्नुहोस्।

### डोमेन थपिएको छैन (Domain Not Added) {#domain-not-added}
- कुनै पनि त्रुटि सन्देशहरूका लागि Ultimate Multisite को logs जाँच गर्नुहोस्।
- डोमेन पहिले नै WP Engine मा थपिएको छैन कि हेर्नुहोस्।
- तपाईंले थप्दै गरेको डोमेनको संख्यालाई तपाईंको WP Engine प्लानले समर्थन गर्छ कि सुनिश्चित गर्नुहोस्।

### सबडोमेन समस्याहरू (Subdomain Issues) {#subdomain-issues}
- यदि सबडोमेन काम गरिरहेका छैनन् भने, वाइल्डकार्ड डोमेन कन्फिगरेसनको लागि WP Engine सपोर्टलाई सम्पर्क गर्नुहोस्।
- मुख्य डोमेन र सबडोमेनहरूको लागि तपाईंको DNS सेटिङहरू सही रूपमा कन्फिगर गरिएको छ कि हेर्नुहोस्।
