# फिल्टर: get_site_url_for_previewer

पूर्वावलोककमा प्रयोग हुने URL फिल्टर गर्न plugin विकासकर्ताहरूलाई अनुमति दिनुहोस्

## प्यारामिटरहरू

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $domain | `string` | अहिले प्रयोग भइरहेको पूर्वनिर्धारित domain, हेरफेरका लागि उपयोगी |
| $domain_options | `array` | Ultimate Multisite सेटिङ्स -&gt; नेटवर्क सेटिङ्स -&gt; Domain विकल्पहरूमा प्रविष्ट गरिएका सबै domain विकल्पहरूको सूची |
