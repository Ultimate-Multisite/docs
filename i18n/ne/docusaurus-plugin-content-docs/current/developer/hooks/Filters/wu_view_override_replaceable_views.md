# फिल्टर: wu_view_override_replaceable_views

विकासकर्ताहरूलाई बदल्न मिल्ने सूचीमा थप फोल्डरहरू थप्न अनुमति दिन्छ।

सावधान हुनुहोस्, किनकि थप फोल्डरहरूलाई अनुमति दिँदा Ultimate Multisite संस्करणहरूको सट्टा पुराना प्रतिलिपिहरू लोड हुन सक्छन्।

## प्यारामिटरहरू

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $replaceable_views | `array` | अनुमति दिइएका फोल्डरहरूको सूची। |

- 2.0.0
