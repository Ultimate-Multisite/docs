# फिल्टर: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

स्वतन्त्र billing cycle भएका product types फिल्टर गर्नुहोस्।

स्वतन्त्र billing cycle भएका Products cart मा plan को billing period सँग मिल्न बाध्य हुँदैनन्। यो domain registrations जस्ता products का लागि उपयोगी हुन्छ, जसले plan मासिक वा वार्षिक भए पनि सधैँ वार्षिक रूपमा bill गर्छन्।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | स्वतन्त्र billing cycle भएका product types को Array। |
