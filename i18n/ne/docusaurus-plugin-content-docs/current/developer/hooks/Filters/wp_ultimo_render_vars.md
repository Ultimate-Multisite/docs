# फिल्टर: wp_ultimo_render_vars {#filter-wpultimorendervars}

प्लगइन विकासकर्ताहरूलाई विश्वव्यापी रूपमा render context मा अतिरिक्त variable थप्न अनुमति दिन्छ।

## प्यारामिटरहरू {#parameters}

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $args | `array` | render call द्वारा पास गरिएका variables समावेश गर्ने array। |
| $view | `string` | render गरिने view को नाम। |
| $default_view | `string` | fallback_view को नाम |

- 2.0.0
