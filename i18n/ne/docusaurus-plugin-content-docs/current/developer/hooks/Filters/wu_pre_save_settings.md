# फिल्टर: wu_pre_save_settings

Ultimate Multisite द्वारा सुरक्षित गर्नुअघि विकासकर्ताहरूलाई सेटिङहरू फिल्टर गर्न अनुमति दिनुहोस्।

## प्यारामिटरहरू {#parameters}

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $settings | `array` | सुरक्षित गरिने सेटिङहरू। |
| $settings_to_save | `array` | थप्नुपर्ने नयाँ सेटिङहरू। |
| $saved_settings | `array` | हाल सुरक्षित गरिएका सेटिङहरू। |

### संस्करणदेखि {#since}

- 2.0.18
### स्रोत {#source}

[`inc/class-settings.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-settings.php#L296) मा लाइन 296 मा परिभाषित गरिएको छ।
