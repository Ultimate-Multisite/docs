# फिल्टर: wu_documentation_get_link {#filter-wudocumentationgetlink}

प्लगइन विकासकर्ताहरूलाई लिङ्कहरू फिल्टर गर्न अनुमति दिन्छ।

यो कसरी उपयोगी हुन सक्छ भन्ने निश्चित छैन, तर यसलाई राख्दा हानि हुँदैन

## प्यारामिटरहरू {#parameters}

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $link | `string` | दर्ता गरिएको लिङ्क |
| $slug | `string` | लिङ्क प्राप्त गर्न प्रयोग गरिएको स्लग |
| $default_link | `string` | दर्ता गरिएको पूर्वनिर्धारित लिङ्क |
