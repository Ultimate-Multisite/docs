# फिल्टर: wu_password_require_special

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $require | `bool` | विशेष वर्णहरू आवश्यक पर्ने कि नपर्ने। नियमहरू लागू हुँदा पूर्वनिर्धारित true। |
| $defender_active | `bool` | Defender Pro Strong Password सक्रिय छ कि छैन। |
