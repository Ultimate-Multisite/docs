# Action: wu_site_created {#action-wusitecreated}

Löst aus, nachdem eine Website zum ersten Mal erstellt wurde.  
Löst nicht aus, wenn von einer Vorlage dupliziert wird.

| $data | `array` | Die Objekt-Daten, die gespeichert werden. |
| $site | `\Site` | Die Objekt-Instanz. |
