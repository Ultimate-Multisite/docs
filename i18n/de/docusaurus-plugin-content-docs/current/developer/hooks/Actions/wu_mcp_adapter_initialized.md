# Aktion: wu_mcp_adapter_initialized

Löst aus, nachdem der MCP adapter initialisiert wurde.

Erlaubt anderen Plugins und Themes, ihre eigenen Fähigkeiten zu registrieren.

## Parameter

| Name | Typ | Beschreibung |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Die MCP adapter Instanz. |

- 2.5.0
