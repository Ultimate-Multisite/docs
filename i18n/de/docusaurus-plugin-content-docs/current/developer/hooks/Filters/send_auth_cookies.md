# Filter: send_auth_cookies {#filter-sendauthcookies}

Erlaubt, dass Authentifizierungs-Cookies nicht tatsächlich an den Client gesendet werden.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $send | `bool` | Ob Authentifizierungs-Cookies an den Client gesendet werden sollen. |
