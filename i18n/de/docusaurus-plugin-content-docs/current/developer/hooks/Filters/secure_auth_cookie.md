# Filter: secure_auth_cookie

Filtert, ob das Authentifizierungs-Cookie nur über HTTPS gesendet werden soll.

## Parameters

| Name | Typ | Beschreibung |
|------|------|-------------|
| $secure | `bool` | Ob das Cookie nur über HTTPS gesendet werden soll. |
| $user_id | `int` | Benutzer-ID. |

### Seit

- 3.1.0
