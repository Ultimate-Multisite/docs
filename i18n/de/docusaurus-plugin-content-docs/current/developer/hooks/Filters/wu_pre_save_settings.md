# Filter: wu_pre_save_settings

Ermöglicht Entwicklern, Einstellungen vor dem Speichern durch Ultimate Multisite zu filtern.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | Die Einstellungen, die gespeichert werden sollen. |
| $settings_to_save | `array` | Die neuen Einstellungen, die hinzugefügt werden sollen. |
| $saved_settings | `array` | Die aktuell gespeicherten Einstellungen. |

### Since

- 2.0.18
