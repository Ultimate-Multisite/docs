# Filter: wu_independent_billing_cycle_product_types

Filter die Produkttypen, die unabhängige Abrechnungszyklen haben.

Produkte mit unabhängigen Abrechnungszyklen müssen nicht an die Abrechnungsperiode des Plans im Warenkorb angepasst werden. Das ist nützlich für Produkte wie Domainregistrierungen, die immer jährlich abgerechnet werden, unabhängig davon, ob der Plan monatlich oder jährlich ist.

## Parameters

| Name | Typ | Beschreibung |
|------|------|-------------|
| $types | `array` | Array von Produkttypen mit unabhängigen Abrechnungszyklen. |
