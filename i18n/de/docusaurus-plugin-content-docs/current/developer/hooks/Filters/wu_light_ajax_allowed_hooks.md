# Filter: wu_light_ajax_allowed_hooks

Aus Sicherheitsgründen begrenzen wir die Anzahl der Aktionen, die zum Hooken verfügbar sind. Dieser Filter ermöglicht es Entwicklern, diese Liste bei Bedarf zu erweitern.

- 2.0.0

Definiert in [`inc/class-light-ajax.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-light-ajax.php#L96) bei Zeile 96
