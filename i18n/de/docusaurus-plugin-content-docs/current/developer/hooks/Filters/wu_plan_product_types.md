# Filter: wu_plan_product_types

Filter die Produkttypen, die als Pläne behandelt werden sollen.

Dieser Filter ermöglicht es Addons, zusätzliche Produkttypen zu registrieren, die in der Validierung und Trennung als Pläne erkannt werden sollen.

## Parameter

| Name | Typ | Beschreibung |
|------|------|-------------|
| $plan_types | `array` | Array von Produkttypen, die als Pläne behandelt werden sollen. |
