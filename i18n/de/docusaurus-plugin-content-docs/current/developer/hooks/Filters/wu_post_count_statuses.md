# Filter: wu_post_count_statuses

Erlaube Plugin-Entwicklern, zu ändern, welcher Beitragstatus gezählt werden soll. Standardmäßig werden veröffentlichte und private Beiträge gezählt.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $post_status | `array` | Die Liste der Beitragstatus |
| $post_type | `string` | Der Beitragstyp-Slug |
