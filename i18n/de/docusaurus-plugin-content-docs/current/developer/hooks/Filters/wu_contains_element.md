# Filter: wu_contains_element

Ermöglicht Entwicklern, die Ergebnisse der ersten Suche zu ändern.

Dies ist nützlich für Drittanbieter-Builder und Ähnliches.

## Parameter

| Name | Typ | Beschreibung |
|------|------|-------------|
| $contains_elements | `bool` | Wenn das Element im Inhalt enthalten ist. |
| $content | `string` | Der zu untersuchende Inhalt. |
| $element | `self` | Das aktuelle Element. |
| $post | `null\|\WP_Post` | Post, der überprüft werden soll. |
