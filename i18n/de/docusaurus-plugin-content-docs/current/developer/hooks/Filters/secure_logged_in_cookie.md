# Filter: secure_logged_in_cookie

Filtert, ob das angemeldete Cookie nur über HTTPS gesendet werden soll.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $secure_logged_in_cookie | `bool` | Ob das angemeldete Cookie nur über HTTPS gesendet werden soll. |
| $user_id | `int` | Benutzer-ID. |
| $secure | `bool` | Ob das Auth-Cookie nur über HTTPS gesendet werden soll. |

### Since

- 3.1.0
