# Filter: wu_view_override_replaceable_views

Erlaubt Entwicklern, zusätzliche Ordner zur ersetzbaren Liste hinzuzufügen.  
Seien Sie vorsichtig, da das Hinzufügen zusätzlicher Ordner dazu führen kann, dass veraltete Kopien anstelle der Ultimate Multisite-Versionen geladen werden.

## Parameter

| Name | Typ | Beschreibung |
|------|------|-------------|
| $replaceable_views | `array` | Liste der erlaubten Ordner. |
