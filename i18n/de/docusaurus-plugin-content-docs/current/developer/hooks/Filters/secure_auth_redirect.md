# Filter: secure_auth_redirect

Filtert, ob ein sicherer Authentifizierungs-Redirect verwendet werden soll.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $secure | `bool` | Ob ein sicherer Authentifizierungs-Redirect verwendet werden soll. Standardwert: false. |

- 3.1.0
