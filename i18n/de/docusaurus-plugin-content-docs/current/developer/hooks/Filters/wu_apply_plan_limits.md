# Filter: wu_apply_plan_limits

Erlaube Plugin-Entwicklern, die Beschränkungen zu umgehen.

Du kannst diesen Filter verwenden, um beliebigen Code auszuführen, bevor irgendwelche Beschränkungen initiiert werden. Wenn dein Filter einen wahren Wert zurückgibt, wird der Prozess fortgesetzt; wenn er einen falschen Wert zurückgibt, kehrt der Code zurück und keiner der unten aufgeführten Hooks wird ausgeführt.

- 1.7.0
