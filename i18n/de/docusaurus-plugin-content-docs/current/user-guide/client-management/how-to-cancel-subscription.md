# Wie kann ich mein Abonnement kündigen? {#how-can-i-cancel-my-subscription}

Wenn du eine Ultimate Multisite-Lizenz hast, kannst du deren Verlängerung jederzeit kündigen. Folge einfach den untenstehenden Schritten:


<!-- Screenshot unavailable: Account page email with access link -->

Verwende die E-Mail-Adresse, die du beim Kauf angegeben hast, und dein Passwort.

<!-- Screenshot unavailable: Login page with email and password fields -->

Dies ist der Startbildschirm deiner Account-Seite:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Um dein Abonnement zu kündigen, gehe im Menü auf der linken Seite zu _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Klicke auf den Pfeil rechts, um ein Seitenfenster zu öffnen. Dann solltest du die Option auswählen, mit der die **automatische Verlängerung** gekündigt wird.

Das System zeigt dir eine Bestätigungsmeldung an.

Wenn du sicher bist, dass du dein Abonnement kündigen möchtest, klicke einfach auf die Schaltfläche _**Cancel Renewals**_. Nach dieser Aktion wirst du gebeten, eine kurze Umfrage zu beantworten.

Fertig! Dein Abonnement wird nicht automatisch verlängert.

**Du hast einen gültigen Schlüssel, bis dein Abonnement abläuft**. Falls du dein Abonnement reaktivieren möchtest, musst du dies manuell tun.
