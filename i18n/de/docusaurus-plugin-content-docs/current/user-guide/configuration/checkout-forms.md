# Checkout-Formulare

Checkout-Formulare sind eine einfache und flexible Möglichkeit, verschiedene Ansätze auszuprobieren, wenn man neue Kunden gewinnen möchte.

Ultimate Multisite 2.0 bietet einen Checkout-Formular-Editor, mit dem Sie so viele Formulare erstellen können, wie Sie möchten, mit unterschiedlichen Feldern, angebotenen Produkten usw.

Um auf diese Funktion zuzugreifen, gehen Sie zum Menü Checkout-Formulare in der linken Seitenleiste.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-TusdCVS7N7.png)

Auf dieser Seite sehen Sie alle Checkout-Formulare, die Sie haben.

Wenn Sie ein neues erstellen möchten, klicken Sie einfach oben auf der Seite auf Checkout-Formular hinzufügen.

Sie können eine dieser drei Optionen als Ausgangspunkt wählen: einzelner Schritt, mehrstufig oder leer. Dann klicken Sie auf Editor öffnen.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-OVx7AlxoX5.png)

Alternativ können Sie die bereits vorhandenen Formulare bearbeiten oder duplizieren, indem Sie auf die Optionen unter deren Namen klicken. Dort finden Sie außerdem die Optionen, um den Shortcode des Formulars zu kopieren oder das Formular zu löschen.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-LdsdAu67WF.png)

### Bearbeiten eines Checkout-Formulars

Sie können Checkout-Formulare für verschiedene Zwecke erstellen. In diesem Beispiel arbeiten wir an einem Registrierungsformular.

Nachdem Sie zum Checkout-Formular-Editor navigiert haben, geben Sie Ihrem Formular einen Namen (der nur intern verwendet wird) und einen Slug (zum Beispiel zur Erstellung von Shortcodes).

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-RvbtUn4r3w.png)

Formulare bestehen aus Schritten und Feldern. Sie können einen neuen Schritt hinzufügen, indem Sie auf Neuen Checkout-Schritt hinzufügen klicken.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-O5GsWoPE35.png)

Im ersten Tab des Modalfensters füllen Sie den Inhalt Ihres Formularschritts aus. Geben Sie ihm eine ID, einen Namen und eine Beschreibung. Diese Angaben werden hauptsächlich intern verwendet.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-juXHmB0fGH.png)

Als nächstes legen Sie die Sichtbarkeit des Schritts fest. Sie können zwischen „Immer anzeigen“, „Nur für angemeldete Benutzer anzeigen“ oder „Nur für Gäste anzeigen“ wählen.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-2gZRyHCxed.png)

Schließlich konfigurieren Sie den Schrittstil. Diese Felder sind optional.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-lPHJcayky5.png)

Jetzt ist es Zeit, Felder zu unserem ersten Schritt hinzuzufügen. Klicken Sie einfach auf Neues Feld hinzufügen und wählen Sie den gewünschten Abschnittstyp aus.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-z5FpDeUVmD.png)![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-JxtVsd5aGu.png)

Jedes Feld hat unterschiedliche Parameter, die ausgefüllt werden müssen. Für diesen ersten Einstieg wählen wir das Feld „Benutzername“.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-go026QZtjh.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-RroHNTL0TJ.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-OG4O4XkEGu.png)

Sie können so viele Schritte und Felder hinzufügen, wie Sie benötigen. Um Ihre Produkte für Ihre Kunden anzuzeigen, damit sie eines auswählen können, verwenden Sie das Feld Pricing Table. Wenn Sie möchten, dass Ihre Kunden ein Template auswählen, fügen Sie das Feld Template Selection hinzu. Und so weiter.

_**Hinweis:** username, email, password, site title, site URL, order summary, payment, and submit button are mandatory fields to create a checkout form._

Während Sie an Ihrem Checkout-Formular arbeiten, können Sie jederzeit die Vorschau-Schaltfläche verwenden, um zu sehen, wie Ihre Kunden das Formular sehen. Sie können auch zwischen der Ansicht als bestehender Benutzer oder Besucher wechseln.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-kjDtBjBdSH.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-0SKLl73PG5.png)

Schließlich können Sie in den erweiterten Optionen die Nachricht für die Seite „Thank You“ konfigurieren, Snippets zum Tracking von Conversions hinzufügen, benutzerdefiniertes CSS zu Ihrem Checkout-Formular hinzufügen oder es auf bestimmte Länder beschränken.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-QC2Q5kIZ9b.png)

Sie können Ihr Checkout-Formular auch manuell aktivieren oder deaktivieren, indem Sie diese Option in der rechten Spalte umschalten, oder das Formular dauerhaft löschen.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1iRUaR7Gh1.png)

Vergessen Sie nicht, Ihr Checkout-Formular zu speichern!

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-OKgUbhf41x.png)

Um den Shortcode Ihres Formulars zu erhalten, klicken Sie auf Shortcode generieren und kopieren Sie das Ergebnis, das im Modalfenster angezeigt wird.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-7d68oNc7yI.png)
