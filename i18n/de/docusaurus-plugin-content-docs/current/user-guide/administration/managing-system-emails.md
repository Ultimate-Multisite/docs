# Verwaltung von System-E-Mails

Ultimate Multisite sendet verschiedene System-E-Mails für Ereignisse wie Anmeldebestätigungen, Zahlungsbelege und Mitgliedschaftsänderungen. Sie können diese E-Mail-Vorlagen an Ihre Marke anpassen.

## System-E-Mails anzeigen

Navigieren Sie zu **Ultimate Multisite → Einstellungen** und klicken Sie auf die Registerkarte **E-Mails**, um die e-mail-bezogenen Einstellungen anzuzeigen.

![Settings - Emails tab](/img/admin/settings-emails.png)

Dort können Sie die Seite **System-E-Mails** öffnen, um alle E-Mail-Vorlagen anzuzeigen.

![System Emails list](/img/admin/system-emails-list.png)

## Neue E-Mail-Vorlagen importieren

Neue Ultimate Multisite-Versionen und Add‑Ons können neue E-Mail-Vorlagen registrieren. Um Konflikte zu vermeiden, werden neue Vorlagen nicht automatisch hinzugefügt – Sie müssen sie manuell importieren.

Um neue E-Mail-Vorlagen zu importieren:

1. Gehen Sie zur Seite **System-E-Mails**
2. Klicken Sie auf die Schaltfläche **Reset & Import** oben auf der Seite
3. Schalten Sie die Option **Import Emails** ein, um verfügbare Vorlagen anzuzeigen
4. Wählen Sie die Vorlagen aus, die Sie importieren möchten, und bestätigen Sie

## E-Mail-Vorlagen zurücksetzen

Wenn Sie eine E-Mail-Vorlage angepasst haben und zum Standardinhalt zurückkehren möchten, haben Sie zwei Optionen:

1. **Löschen und erneut importieren** – Löschen Sie die System-E-Mail und importieren Sie sie erneut (dies löscht die Versandmetriken)
2. **Zurücksetzen über das Tool** – Verwenden Sie das Tool **Reset & Import**, schalten Sie die Option **Reset** ein und wählen Sie die E-Mails aus, die Sie zurücksetzen möchten

Die Reset-Option ist vorzuziehen, da sie Versandmetriken und andere Metadaten beibehält und gleichzeitig den Standardinhalt wiederherstellt.
