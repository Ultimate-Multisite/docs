# Dashboard Overview

Ultimate Multisite verfügt über ein dediziertes **Dashboard** im Ultimate Multisite-Menü und fügt außerdem Widgets zum Network Dashboard hinzu. Dieser Artikel beschreibt, was Sie an beiden Stellen sehen werden.

## The Network Dashboard

Ultimate Multisite fügt nützliche Widgets in das netzwerkweite Dashboard ein, das unter **My Sites → Network Admin → Dashboard** zu finden ist.

![Netzwerk-Dashboard mit Ultimate Multisite-Widgets](/img/admin/network-dashboard-full.png)

Hier ist, was jedes Widget bietet:

**First Steps:** Erscheint bei einer frischen Installation und erinnert Sie daran, Setup-Aufgaben wie die Konfiguration einer Zahlungsmethode und die Erstellung eines Testkontos abzuschließen.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** Gibt die Anzahl der Anmeldungen und den Umsatz für den Tag an. Der MRR (Monthly Recurring Revenue) gibt einen prognostizierten Gesamtwert über Kunden mit umsatzgenerierenden Mitgliedschaften an.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Berichtet über Ereignisse in Ultimate Multisite — Anmeldungen, Kündigungen, Seitenerstellungen, Löschungen und andere Aktivitäten.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Zeigt eine kurze Zusammenfassung von Benutzern und Seiten im Netzwerk mit Shortcuts zum Erstellen neuer Seiten oder Benutzer.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Holt und zeigt die neuesten Ultimate Multisite-Nachrichten einschließlich Updates, kritischer Fehler und Sicherheitsupdates.

![News widget](/img/admin/network-dashboard-news.png)

## The Ultimate Multisite Dashboard

Während das Netzwerk-Dashboard ganzheitliche Informationen über das Netzwerk liefert, präsentiert das Ultimate Multisite-Dashboard (auf der obersten Ebene des Ultimate Multisite-Menüs) Informationen zum Service.

![Ultimate Multisite-Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** Zeigt eine monatliche Aufschlüsselung des Umsatzes über einen 12‑monatigen Kalenderzeitraum. Neuer Umsatz wird in Blau angezeigt, während Kündigungen in Rot erscheinen.

![MRR-Wachstumsdiagramm](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** Filtert Dashboard-Daten nach Zeitraum, um sich auf die benötigten Informationen zu konzentrieren. Der Datumsbereich steuert, was in den untenstehenden Widgets angezeigt wird.

![Datumsbereich und Dashboard-Statistiken](/img/admin/um-dashboard-date-range.png)

Die Dashboard-Widgets umfassen:

- **Revenue** — Gesamtumsatz für den Zeitraum mit MRR und pro Produkt Aufschlüsselungen
- **New Memberships** — Wie viele neue Mitgliedschaften im ausgewählten Zeitraum erstellt wurden
- **Signups by Countries** — Geografische Aufschlüsselung der Anmeldungen
- **Signups by Forms** — Wie viele Kunden sich über jedes Checkout-Formular angemeldet haben bzw. von Admins hinzugefügt wurden
- **Most Visited Sites** — Die am häufigsten besuchten Seiten in Ihrem Netzwerk (erfordert das Aktivieren der Besuchszählung unter **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)

![Dashboard-Statistiken](/img/admin/um-dashboard-stats.png)
