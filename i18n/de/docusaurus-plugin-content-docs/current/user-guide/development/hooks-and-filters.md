# Ultimate Multisite Hooks und Filter

Ultimate Multisite besteht aus einer ganzen Menge an Funktionen und Datenbankabfragen. Es verfügt über Hooks und Filter, die es Ihnen ermöglichen, bestehenden Funktionen neuen Code hinzuzufügen oder sie zu modifizieren.

Wir haben eine Liste dieser Hooks und Filter als Referenz erstellt, um fortgeschrittenen Benutzern zu helfen, ihr Netzwerk weiter anzupassen. Sie können die Liste über Github zugreifen, indem Sie auf den untenstehenden Link klicken.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
