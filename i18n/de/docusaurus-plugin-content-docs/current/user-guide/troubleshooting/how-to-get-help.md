# Wie man Hilfe bekommt

Sie haben Ihren gültigen Lizenzschlüssel erhalten, und damit haben Sie Zugang zu unserem Team, das Ihnen bei Problemen hilft, die Sie während der Einrichtung von Ultimate Multisite möglicherweise erleben.

Diese Lektion behandelt die verfügbaren Support‑Kanäle für Lizenzinhaber und welche Art von Informationen Sie bereithalten sollten, um unser Support‑Team zu kontaktieren.

## Help Site

Sie können verschiedene Artikel und Tutorials von unserer Hilfeseite abrufen, die Sie im untenstehenden Link finden.  

[)

Wir ermutigen unsere Benutzer, sich mit den grundlegenden Funktionen und Merkmalen von Ultimate Multisite vertraut zu machen. Auf unserer Hilfeseite gibt es „Getting Started“-Artikel, und dort finden Sie auch Antworten auf häufig gestellte Fragen, wie z. B. wie man Add‑Ons installiert, Zahlungs­gateways integriert usw.

## Contact Support

Falls Sie nicht weiterkommen und die Antwort nicht auf unserer Hilfeseite finden, können Sie unser Support‑Team entweder erreichen, indem Sie auf das Sprechblasen‑Icon in der unteren rechten Ecke unserer Website klicken, oder indem Sie eine E‑Mail direkt an support@ultimatemultisite.com senden.

Wenn Sie unseren Support wegen technischer Probleme kontaktieren, stellen Sie bitte sicher, dass Sie eine detaillierte Beschreibung Ihres Problems zusammen mit einem Screenshot, Fehlermeldungen oder anderen Informationen bereitstellen, die unseren Support‑Mitarbeitern helfen, Ihr Problem leicht zu verstehen und zu bewerten. Sie müssen außerdem Ihren Lizenzschlüssel angeben. Nur aktive Lizenzinhaber haben Anspruch auf Support.

## Community Site

Unsere [community site](https://community.ultimatemultisite.com/) ist ein Ort, an dem Ultimate Multisite‑Nutzer miteinander interagieren können. Es dient als Informationssystem, in dem Mitglieder Beiträge veröffentlichen, Diskussionen kommentieren, Ratschläge geben, zusammenarbeiten oder Crowdsourcing betreiben können. Sie müssen Ihren Ultimate Multisite‑Lizenzschlüssel angeben, um private Bereiche/Seiten freizuschalten.
