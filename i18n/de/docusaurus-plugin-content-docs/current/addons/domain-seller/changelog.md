# Domain Seller Änderungsprotokoll

Version: 2.0.1 - Freigegeben am 2025-09-28

* Präfix in ultimate-multisite umbenennen; Textdomain aktualisieren; Versionssprung.



### Version 2.0.0 - 2024-XX-XX

**Große Neugestaltung für Multisite Ultimate v2**

* **New:** Vollständige Neugestaltung mit moderner PHP 7.4+ Architektur
* **New:** Nahtlose Integration mit dem Multisite Ultimate v2 Checkout-System
* **New:** Domain-Produktverwaltung mit flexiblen Preisoptionen
* **New:** Architektur zur Unterstützung mehrerer Domain-Provider
* **New:** Automatische Verlängerung und Abonnement-Integration
* **New:** Kunden-Domain-Verwaltungsoberfläche
* **New:** Admin-Domainüberwachung und Protokolle
* **New:** Gutscheinsupport für Domain-Produkte
* **New:** Umfassendes Einstellungsmanagement
* **New:** Entwicklerfreundliche erweiterbare Codebasis
* **Improved:** Aktualisierter OpenSRS-Provider mit voller Funktionsunterstützung
* **Improved:** Modernes UI konsistent mit Multisite Ultimate v2
* **Fixed:** Alle veralteten v1-Code auf v2-Standards aktualisiert
* **Removed:** Legacy v1-Kompatibilität (breaking change)
