# Metered Plans

Der Ultimate Multisite: Metered Plans Addon verändert die Art und Weise, wie Sie Ihre Kunden abrechnen, indem er flexible verbrauchsbasierte Abrechnungsfunktionen einführt. Anstatt starre Hard-Limits, die Benutzer blockieren, wenn sie überschritten werden, ermöglichen Sie Überziehungsgebühren, die Kunden automatisch für zusätzlichen Verbrauch abrechnen.

## Hauptfunktionen

* **Metered Billing**: Kunden basierend auf tatsächlicher Nutzung statt fester Stufen abrechnen
* **Overage Management**: Automatisch für Nutzung, die die Planlimits überschreitet, abrechnen
* **Usage Tracking**: Verbrauch in Echtzeit überwachen
* **Flexible Pricing**: Flexible Preisgestaltung: Festlegen von individuellen Tarifen für verschiedene Ressourcen (Speicher, Bandbreite, Sites usw.)
* **Automatic Invoicing**: Automatisches Rechnungswesen: Erstellen von Rechnungen automatisch, wenn Verbrauchsgrenzen überschritten werden
* **Usage Reports**: Verbrauchsberichte: Detaillierte Verbrauchsanalyse und Berichte für Kunden und Administratoren

## Ideal für

* **Hosting Providers**: Anbieten skalierbarer Pläne, die mit den Bedürfnissen der Kunden wachsen
* **SaaS Platforms**: Abrechnung basierend auf tatsächlicher Ressourcennutzung
* **Agency Networks**: Flexible Preismodelle für Kundenprojekte
* **Reseller Hosting**: Partnern ermöglichen, nutzungsbasierte Preisgestaltung anzubieten

## Funktionsweise

1. Konfigurieren Sie verbrauchsbasierte Ressourcen (Festplattenspeicher, Bandbreite, Anzahl der Sites usw.)
2. Setzen Sie Basisplanlimits und Überziehungsraten
3. Das System verfolgt automatisch den Verbrauch im gesamten Netzwerk
4. Wenn Kunden die Limits überschreiten, werden Überziehungsgebühren berechnet
5. Rechnungen werden automatisch basierend auf dem Abrechnungszyklus erstellt
6. Kunden können ihren Verbrauch und kommende Gebühren in Echtzeit einsehen

## Unterstützte Ressourcen

* Festplattenspeicher
* Bandbreite/Datenübertragung
* Anzahl der Sites
* Anzahl der Benutzer
* Benutzerdefinierte Ressourcentypen

## Anforderungen

* Multisite Ultimate 2.0.0 oder höher
* WordPress Multisite-Installation
* PHP 7.4 oder höher

## Installation

1. Upload the plugin files to `/wp-content/plugins/ultimate-multisite-metered-plans`
2. Aktivieren Sie das Plugin über den Bildschirm 'Plugins' in WordPress
3. Navigieren Sie zu Multisite Ultimate → Einstellungen → Metered Plans
4. Konfigurieren Sie Ihre verbrauchsbasierte Ressourcen und Preisgestaltung
5. Aktivieren Sie verbrauchsbasierte Abrechnung für Produkte im Produkteditor

## Häufig gestellte Fragen

### Was passiert, wenn ein Kunde sein Limit überschreitet?

Anstatt den Zugriff zu blockieren, verfolgt das System die Überziehung und fügt sie ihrer nächsten Rechnung basierend auf den von Ihnen konfigurierten Tarifen hinzu.

### Kann ich unterschiedliche Überziehungsraten für verschiedene Produkte festlegen?

Ja, jedes Produkt kann einzigartige verbrauchsbasierte Ressourcen und Überziehungspreise haben.

### Wie werden Kunden über den Verbrauch informiert?

Kunden können ihren aktuellen Verbrauch in ihrem Dashboard einsehen und optionale Benachrichtigungen erhalten, wenn sie sich den Limits nähern oder diese überschreiten.

### Kann ich weiterhin Hard-Limits verwenden?

Ja, verbrauchsbasierte Abrechnung ist pro Produkt optional. Sie können traditionelle Hard-Limit-Produkte mit verbrauchsbasierter Abrechnung mischen.
