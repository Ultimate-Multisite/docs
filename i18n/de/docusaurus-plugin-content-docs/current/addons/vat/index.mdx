# VAT

Multisite Ultimate: Europäische VAT

Fügt zusätzliche Funktionen und Werkzeuge zum Standard-Steuerhandling von Multisite Ultimate hinzu, um die europäische VAT besser zu unterstützen.  
Automatisch fügt EU VAT-Sätze zur Seite 'Tax Rates' hinzu von

## Installation

1. Laden Sie 'ultimate-multisite-vat' in das Verzeichnis '/wp-content/plugins/' hoch  
2. Aktivieren Sie das Plugin über das Menü 'Plugins' in WordPress  
3. Gehen Sie zu Ultimate Multisite > Einstellungen > Steuern und aktivieren Sie EU Vat.  
4. Klicken Sie auf Tax Rates und aktualisieren Sie EU VAT Rates unten und speichern Sie die Tax Rates.  
5. Fertig!
