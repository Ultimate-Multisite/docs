# Filter: networks_user_is_network_admin {#filter-networksuserisnetworkadmin}

Filtert die Netzwerke, für die ein Benutzer Administrator ist.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $ | `array\|bool` | Liste von Netzwerk-IDs oder false, wenn keine Netzwerke für den Benutzer vorhanden sind. |
| $ | `int` | Benutzer-ID, für die die Netzwerke zurückgegeben werden sollen. |
