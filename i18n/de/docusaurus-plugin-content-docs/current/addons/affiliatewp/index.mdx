# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

Nutzen Sie das leistungsstarke AffiliateWP, um die Kundenbasis Ihres Ultimo Network zu erweitern!

## Installation

1. Laden Sie 'ultimate-multisite-git ' in das Verzeichnis '/wp-content/plugins/' hoch
2. Aktivieren Sie das Plugin über das Menü 'Plugins' in WordPress
3. Folgen Sie dem Schritt-für-Schritt-Assistenten, um das Plugin einzurichten

## Usage Guide

[AffiliateWP](https://affiliatewp.com/) ist die bevorzugte Lösung, wenn Sie ein Affiliate-Programm für Ihr Online-Geschäft erstellen. Ultimate Multisite: AffiliateWP ist die Brücke, die es Ihnen ermöglicht, dasselbe für Ihr Premium-Netzwerk zu erstellen.

## Getting Started

Zunächst müssen Sie **AffiliateWP** **aktiv** haben! Sie können unser Add‑On nicht nutzen, wenn Sie **das Plugin, das sich damit integriert**, nicht haben. Falls Sie nicht planen, Ihre Kunden ebenfalls auf ihren Websites nutzen zu lassen, sollte AffiliateWP nur auf Ihrer Hauptseite aktiv sein.

Außerdem benötigen Sie sowohl **Ultimate Multisite** als auch **Ultimate Multisite: AffiliateWP** **netzweit aktiviert**.

## Adding Ultimate Multisite as an AffiliateWP Integration

Gehen Sie auf Ihrer Hauptseite zu Affiliates Settings und aktivieren Sie auf der Registerkarte Integrations die Option Ultimate Multisite. Dadurch wird [AffiliateWP](https://affiliatewp.com/) angewiesen, Besuche und Conversions aus dem Anmeldefluss von Ultimate Multisite zu verfolgen.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_Check the Ultimate Multisite box at the bottom and save_

Das ist im Wesentlichen alles, was nötig ist. Besuche, die über Empfehlungslinks kommen, werden verfolgt, und wenn der Benutzer später seine erste Zahlung tätigt, wird die Conversion erstellt und eine Provision für den verantwortlichen Affiliate zurückgestellt.

Nur die erste Zahlung des neu erstellten Benutzers wird jedoch verfolgt. Um wiederkehrende Empfehlungszahlungen zu ermöglichen, benötigen Sie das [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Enabling Recurring Referrals

Nach der Installation des [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), gehen Sie zum Reiter Affiliates Settings Recurring Referrals und aktivieren die Verwendung wiederkehrender Empfehlungen.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_As simple as it gets: Enable Recurring Referrals and you’re all set!_

Wieder einmal, das ist alles, was nötig ist!

Jetzt erhält der verantwortliche Affiliate jedes Mal, wenn eine neue Zahlung auf einem empfohlenen Konto eingeht, automatisch seinen Anteil!
