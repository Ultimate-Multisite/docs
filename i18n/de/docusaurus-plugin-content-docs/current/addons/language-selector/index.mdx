# Sprachauswahl

Multisite Ultimate: Sprachauswahl bei der Registrierung

Füge dem Registrierungsablauf ein Sprachauswahlfeld hinzu.

## Installation

1. Lade 'ultimate-multisite-language-selector' in das Verzeichnis '/wp-content/plugins/' hoch
2. Aktiviere das Plugin über das Menü 'Plugins' in deinem WordPress Netzwerk-Admin

## Nutzungsanleitung

Mit diesem kostenlosen Add-on kannst du deine Benutzer auswählen lassen, welche Sprache sie im Registrierungsablauf und auf der Anmeldeseite sehen möchten.

Nachdem du [deine Lizenz aktiviert hast](../../user-guide/getting-started/installing-ultimate-multisite.md), kannst du Ultimate Multisite Add-ons installieren, auf die dir deine Lizenz Zugriff gewährt.

Gehe zu deiner Ultimate Multisite Einstellungsseite und klicke dann im Menü links auf die Beschriftung _Add-ons_.

![Add-ons-Seite](/img/admin/addons-page.png)

_Gehe zur Add-ons-Seite, um mit der Installation von Sprachauswahl zu beginnen_

Dort findest du alle Add-ons, die wir anbieten.

Kostenlose Add-ons stehen all unseren Kunden zur Verfügung, unabhängig vom Abonnementplan, für den sie sich angemeldet haben.

Klicke auf die Schaltfläche _Jetzt installieren!_ unter dem Vorschaubild des Add-ons. Aktiviere es dann auf der Plugins-Seite.

![Add-ons-Seite mit Installationsschaltfläche](/img/admin/addons-page.png)

_Klicke auf die Schaltfläche, um das Add-on zu installieren_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Aktiviere es auf deiner Plugins-Seite_

Gehe nun zum Einstellungsmenü von Ultimate Multisite und öffne den Tab _Netzwerkeinstellungen_. Dort findest du unten auf der Seite einen neuen Abschnitt namens _Sprachen_.

![Einstellungsseite](/img/config/settings-general.png)

_Ein brandneuer Abschnitt erscheint!_

Du kannst ihn leer lassen, um alle Sprachen als Optionen anzuzeigen, oder auswählen, welche du verfügbar machen möchtest. Die Sprachen sind diejenigen, die du in deiner WordPress Einrichtung installiert hast.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Liste der auf dieser WP-Installation verfügbaren Sprachen_

Nachdem du die vorgenommenen Änderungen gespeichert hast, wird auf der Anmeldeseite sowie im Registrierungsablauf ein Auswahlmenü mit den von dir ausgewählten Sprachen angezeigt.

![Anmelde- und Registrierungseinstellungen](/img/config/settings-login-registration.png)

_Sprachauswahl auf der Anmeldeseite_

![Registrierungsseite](/img/config/registration-page.png)

_Sprachauswahl im Registrierungsablauf_

## Sprache über die URL festlegen

Du kannst die Sprache einer Seite auch ändern, indem du den Locale-Parameter an die URL übergibst. Wenn du zum Beispiel auf eine Registrierungsseite auf Italienisch weiterleiten möchtest, füge ?locale=it_IT zur URL hinzu.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Die Sprache muss in deinem WordPress Netzwerk installiert sein, damit dies funktioniert.
