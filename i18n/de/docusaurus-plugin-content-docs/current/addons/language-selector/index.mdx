# Sprachwahl

Multisite Ultimate: Sprachwahl bei der Registrierung

Fügen Sie ein Sprachwahlfeld zum Registrierungsablauf hinzu.

## Installation

1. Laden Sie 'ultimate-multisite-language-selector' in das Verzeichnis '/wp-content/plugins/' hoch
2. Aktivieren Sie das Plugin über das Menü 'Plugins' in Ihrem WordPress Network Admin

## Nutzungsanleitung

Mit diesem kostenlosen Add‑on können Sie Ihren Benutzern erlauben, die Sprache auszuwählen, die sie im Registrierungsablauf und auf der Anmeldeseite sehen möchten.

Nach dem [Aktivieren Ihrer Lizenz] können Sie Ultimate Multisite Add‑ons installieren, auf die Ihre Lizenz Zugriff gewährt.

Gehen Sie zu Ihrer Ultimate Multisite Einstellungsseite und klicken Sie anschließend im linken Menü auf die Bezeichnung _Add-ons_.

![Add-ons page](/img/admin/addons-page.png)

_Gehen Sie zur Add-ons-Seite, um die Sprachwahl zu installieren_

Es gibt dort alle Add‑ons, die wir anbieten.

Kostenlose Add‑ons stehen allen unseren Kunden zur Verfügung, unabhängig vom abonnierten Plan.

Klicken Sie auf die Schaltfläche _Jetzt installieren!_ unter dem Add‑on‑Vorschaubild. Aktivieren Sie es anschließend auf der Plugins‑Seite.

![Add-ons page with install button](/img/admin/addons-page.png)

_Klicken Sie auf die Schaltfläche, um das Add‑on zu installieren_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Aktivieren Sie es auf Ihrer Plugins‑Seite_

Gehen Sie nun zum Menü Einstellungen von Ultimate Multisite und öffnen Sie den Tab _Network Settings_. Dort finden Sie unten auf der Seite einen neuen Abschnitt namens _Languages_.

![Settings page](/img/config/settings-general.png)

_Ein brandneuer Abschnitt erscheint!_

Sie können ihn leer lassen, um alle Sprachen als Optionen anzuzeigen, oder auswählen, welche Sie verfügbar machen möchten. Die Sprachen sind diejenigen, die Sie in Ihrer WordPress‑Installation installiert haben.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Liste der auf dieser WP‑Installation verfügbaren Sprachen_

Nach dem Speichern der vorgenommenen Änderungen wird ein Auswahlmenü mit den von Ihnen gewählten Sprachen sowohl auf der Anmeldeseite als auch im Registrierungsablauf angezeigt.

![Login and Registration settings](/img/config/settings-login-registration.png)

_Sprachwahl auf der Anmeldeseite_

![Registration page](/img/config/registration-page.png)

_Sprachwahl im Registrierungsablauf_

## Sprachwahl über die URL

Sie können die Sprache einer Seite auch ändern, indem Sie den Locale‑Parameter in der URL umgehen. Wenn Sie beispielsweise zu einer Registrierungsseite auf Italienisch weiterleiten möchten, fügen Sie ?locale=it_IT zur URL hinzu.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Die Sprache muss in Ihrem WordPress‑Netzwerk installiert sein, damit dies funktioniert.
