# Fluent Forms

Dieses Add-on ermöglicht es, Limits für Fluent Forms auf Multisite Ultimate in Produkten, Mitgliedschaften oder Websites hinzuzufügen.

## Features

- Limits für Fluent Forms oder Conversational Fluent Forms hinzufügen
- Kunden erhalten eine freundliche Nachricht, die sie auffordert, ihr Paket zu upgraden, wenn sie das Limit erreichen
