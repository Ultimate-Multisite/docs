# Loco Translate

Das Ultimate Multisite: Loco Translate Integration Addon bietet site-spezifische Übersetzungsfunktionen für Ihr WordPress-Multisite-Netzwerk. Wenn Sie jemals unterschiedliche Sites in Ihrem Netzwerk benötigen, die unterschiedliche Übersetzungen desselben Plugins oder Themes haben, ist dieses Addon die perfekte Lösung.

**Was dieses Addon macht**

Dieses Addon integriert Ultimate Multisite nahtlos mit dem beliebten Loco Translate Plugin und ermöglicht es jeder Subsite in Ihrem Netzwerk, ihre eigenen benutzerdefinierten Übersetzungen zu pflegen. Egal, ob Sie eine mehrregionale Plattform betreiben, verschiedene Sprachgemeinschaften bedienen oder einfach nur Übersetzungsflexibilität im gesamten Netzwerk benötigen, dieses Addon macht es mühelos.

**Ideal für:**

* Unternehmen mit mehrregionaler Ausrichtung und unterschiedlichen Lokalisierungsbedürfnissen
* Bildungseinrichtungen, die vielfältige Gemeinschaften bedienen
* SaaS-Plattformen, die mehrsprachige Unterstützung anbieten
* Webagenturen, die mehrere Kundenseiten verwalten
* Jedes Multisite-Netzwerk, das Übersetzungsflexibilität benötigt

## Installation

1. Stellen Sie sicher, dass Ultimate Multisite installiert und im Netzwerk aktiviert ist
2. Installieren und aktivieren Sie das Loco Translate Plugin
3. Laden Sie dieses Addon in Ihr Plugins-Verzeichnis hoch
4. Aktivieren Sie das Loco Translate Integration Addon im Netzwerk
5. Beginnen Sie mit der Erstellung von site-spezifischen Übersetzungen über Loco Translate auf jeder Subsite

Das war's! Die Integration funktioniert automatisch ohne zusätzliche Konfiguration.

## Frequently Asked Questions

## Do I need to configure anything after installation?

Nein! Die Integration ist automatisch. Sobald sowohl das Addon als auch Loco Translate aktiv sind, können Subsites sofort ihre eigenen benutzerdefinierten Übersetzungen erstellen.

## What happens if a subsite doesn't create custom translations?

Die Site verwendet automatisch die netzwerkweiten Übersetzungen als Fallback. Nichts bricht ab, und alles funktioniert nahtlos.

## Can subsites access network-wide translations?

Reguläre Site-Admins sehen und bearbeiten nur die Übersetzungen ihrer eigenen Site. Super-Admins können sowohl netzwerkweite als auch site-spezifische Übersetzungen je nach betrachteter Site einsehen.

## Does this work with both plugins and themes?

Ja! Site-spezifische Übersetzungen funktionieren sowohl für WordPress-Plugins als auch für Themes.

## Will this slow down my multisite network?

Nein. Das Addon verwendet effizientes Dateiladen, das nur bei Bedarf nach site-spezifischen Übersetzungen prüft. Es gibt keine merkliche Leistungsbeeinträchtigung.

## Can I copy network translations to a specific site as a starting point?

Ja! Beim Bearbeiten von Übersetzungen in Loco Translate können Sie einfach mit den netzwerkweiten Übersetzungen beginnen und diese für jede Site anpassen.

## Does this work with WooCommerce and other plugins?

Absolut! Jeder Plugin oder Theme, der Standard-WordPress-Übersetzungsfunktionen verwendet, funktioniert mit site-spezifischen Übersetzungen.
