# PayFast

Multisite Ultimate: Payfast Gateway

Fügt Payfast - ein schnelles und sicheres Online-Zahlungsgateway in Südafrika - als Multisite Ultimate Zahlungsoption hinzu.

## Installation

1. Laden Sie 'ultimate-multisite-payfast' in das Verzeichnis '/wp-content/plugins/' hoch
2. Aktivieren Sie das Plugin über das Menü 'Plugins' in WordPress
3. Folgen Sie dem Schritt-für-Schritt-Assistenten, um das Plugin einzurichten

## Usage Guide

PayFast ist ein führender Online-Zahlungsprozessor für Unternehmen in Südafrika. Er ist darauf ausgerichtet, Online-Händlern beim Wachstum zu helfen, indem er Transaktionen nahtlos und sicher gestaltet. Auf Wunsch eines Ultimate Multisite-Benutzers haben wir beschlossen, ein Addon zu erstellen, das die Integration mit PayFast ermöglicht. Dieser Artikel zeigt, wie Sie PayFast als Ihr Zahlungsgateway einrichten, um Zahlungen in Ihrem Ultimate Multisite-Netzwerk zu empfangen.

Der erste Schritt besteht darin, das Addon zu installieren.

# Wie man das Payfast-Addon installiert:

Gehen Sie zur Addons-Seite und wählen Sie das Ultimate Multisite: Payfast Gateway Addon aus.

![Add-ons page](/img/admin/addons-page.png)

Das Addon-Fenster wird angezeigt. Klicken Sie auf Installieren.

![Add-ons page with install option](/img/admin/addons-page.png)

Nach Abschluss der Installation werden Sie zur Plugin-Seite weitergeleitet. Klicken Sie hier einfach auf Netzwerk aktivieren, und das _Payfast_-Addon wird in Ihrem Netzwerk aktiviert.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# PayFast einrichten

Nach der Installation des Addons können Sie nun zu den Ultimate Multisite-Einstellungen gehen und PayFast konfigurieren.

Unter der Seite **Ultimate Multisite settings** finden Sie die Registerkarte „Payments“. Unter dieser Registerkarte sollten Sie **Payfast** zusammen mit einem Schalter finden, um dieses Zahlungsgateway ein- oder auszuschalten.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Go ahead and click the toggle button to enable it. It will then expand the box to show different fields that you need to populate such as **Mechant ID, Merchant Key and PassPhrase.** You can get this information from your **Payfast** account

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Der nächste Schritt besteht darin, sich bei Ihrem PayFast-Konto anzumelden, um die benötigten Informationen zu erhalten.

Nach dem Einloggen sehen Sie sofort die Anmeldedaten, die Sie aus Ihrem **Payfast**-Kontodashboard benötigen.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->You can simply copy and paste the credential and populate the details on your Ultimate Multisite settings. Then click the **"Save settings** " button to save the changes.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Das war's, Ihre PayFast-Zahlungsintegration ist abgeschlossen!
