# Support Agents Änderungsprotokoll

Version: 1.0.13 - Veröffentlicht am 2025-09-28

* Prefix umbenennen zu ultimate-multisite; Textdomain aktualisieren; Versionssprung.

Version 1.0.12 - Veröffentlicht am 2025-09-17

* Fix: Kompatibilität mit Ultimate Multisite 2.4.4.

Version 1.0.11 - Veröffentlicht am 2023-11-21

* Fix: Kompatibilität mit Ultimate Multisite 2.3.0.

Version 1.0.10 - Veröffentlicht am 2023-08-30

* Fix: Entfernen von Bulk Support Agents.

Version 1.0.9 - Veröffentlicht am 2023-08-09

* Hinzugefügt: Die Möglichkeit, Checkout-Formulare zu lesen.
* Fix: Layout des Berechtigungsformulars für Support Agents;
* Fix: Warnmeldung im Löschen-Formular für Support Agents;
* Fix: Netzwerk-Dashboard-Widgets für nicht autorisierte Benutzer ausgeblendet.
* Fix: Korrigierte Bearbeiten- und Löschrechte für Support Agents.
* Fix: Prozess zum Erstellen von Support Agents korrigiert.
* Fix: Verhindert, dass ein Kunde als Support Agent hinzugefügt wird.
* Fix: Nicht autorisierte Benutzer zur Admin-Seite zum Bearbeiten von Support Agents umgeleitet.
* Fix: Layout des Berechtigungsformulars für Support Agents verbessert.
* Intern: Neuer Build-Prozess implementiert.

Version 1.0.8 - Veröffentlicht am 2022-01-21

* Intern: Hooks- und Filtergenerator hinzugefügt;
* Intern: Ultimate Multisite-Stubs für Entwickler-Qualität des Lebens hinzugefügt;
* Hinzugefügt: Checkout-Formular-Funktionen;
* Fix: Namensinkonsistenz der Berechtigungen zwischen Ultimate Multisite-Formularen und den Optionen dieses Add-ons.

Version 1.0.7 - Veröffentlicht am 2021-12-02

* Fix: Löschen von Support Agents funktioniert nicht;
* Fix: Bulk-Löschen von Support Agents funktioniert nicht;

Version 1.0.6 - Veröffentlicht am 2021-11-22

* Fix: Neuer Modal-Dialog funktioniert nicht nach Performance-Optimierungen im Kern;
* Fix: Grid-Item-Ansicht vorhanden im Kern, aber nicht im Add-on;

Version 1.0.5 - Veröffentlicht am 2021-10-31

* Fix: Kleine Inkompatibilitäten mit PHP 8;
* Verbesserungen: wp_ultimo_skip_network_active_check verwendet jetzt wu_is_must_use();

Version 1.0.4 - 2021-09-24

* Hinzugefügt: Filter wp_ultimo_skip_network_active_check für mu-plugins-basierte Setups;

Version 1.0.3 - Veröffentlicht am 2021-08-29

* Hinzugefügt: Unterstützung der REST API – Support Agents können über REST erstellt werden;

Version 1.0.2 - Veröffentlicht am 2021-07-26

* Fix: Aktualisieren der Agenten-Berechtigungen funktioniert nicht;

Version 1.0.1 - Veröffentlicht am 2021-05-28

* Hinzugefügt: Hinweis, wenn der Benutzer bereits ein Super-Admin ist;
* Hinzugefügt: Berechtigung zum Wechseln von Benutzern;
* Verbesserungen: Bessere Formularfelder;

Version 1.0.0 - Initiale Veröffentlichung
