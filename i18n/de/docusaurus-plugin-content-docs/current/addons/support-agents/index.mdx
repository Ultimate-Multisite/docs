# Support Agents

Ein ganzes Netzwerk selbst zu verwalten ist keine leichte Aufgabe. Sobald Ihr Netzwerk wächst, werden Sie einen steigenden Bedarf an Support bemerken. Es ist eine gute Idee, Personen einzubinden, die Ihnen helfen, Ihre Arbeitsbelastung zu reduzieren. Allerdings birgt die Gewährung von vollem Zugriff/Erlaubnis Risiken und kann Ihr Netzwerk gefährden. Genau hier kommt dieses Support Agent Add-on ins Spiel.

## Features

- Aufgelistete Liste der WordPress- & Ultimate Multisite-Berechtigungen
- Zugriff einfach aktivieren & deaktivieren
- Unbegrenzte Anzahl von Agent-Konten erstellen

## Installation

1. Laden Sie 'ultimate-multisite-support-agents' in das Verzeichnis '/wp-content/plugins/' hoch
2. Aktivieren Sie das Plugin über das Menü 'Plugins' in WordPress
3. Folgen Sie dem Schritt-für-Schritt-Assistenten, um das Plugin einzurichten
