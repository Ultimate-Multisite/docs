# Admin Page Creator

## Inhalt
Admin Page Creator ermöglicht es Ihnen, neue Admin-Seiten und Unterseiten innerhalb des WordPress-Admin-Dashboards einfach hinzuzufügen. Sie kann verwendet werden, um eine Willkommensseite oder eine Tutorial-Seite zu erstellen, die als Landing-Page für neue Benutzer dienen kann. Sie können Page Builder wie Beaver Builder, Elementor, Brizy und Oxygen verwenden, um die Seiten ohne Aufwand zu gestalten.

Es bietet außerdem Unterstützung für benutzerdefiniertes HTML, CSS und JavaScript sowie Zugriffskontrolle basierend auf Plan und Benutzerrolle!

## Funktionen
* Unterstützung für benutzerdefiniertes HTML und CSS
* JavaScript-Unterstützung
* PHP Content Mode
* Berechtigung basierend auf Benutzerrolle
* Unterstützung für Page Builder

## Installation

1. Upload **ultimate-multisite-admin-page-creator** to the **/wp-content/plugins/** directory
2. Activate the plugin through the **Plugins** menu in your WordPress Network Admin

## Nutzungsleitfaden

Während WordPress es wirklich einfach macht, Inhalte auf der Front-End-Seite von Websites hinzuzufügen, erfordert das Erstellen von Seiten und Unterseiten auf der Admin-Seite einige grundlegende Programmierkenntnisse. Ultimate Multisite: Admin Page Creator hat zum Ziel, diese gleiche Benutzerfreundlichkeit des Front-Ends auf das Backend Ihrer Netzwerkseiten zu bringen.

## Warum Admin Page Creator verwenden?

Vielleicht möchten Sie eine benutzerdefinierte Seite im Admin-Panel Ihrer Kunden mit Tutorials anbieten, vielleicht möchten Sie eine benutzerdefinierte Seite mit FAQs und einem Kontakt-Widget hinzufügen. Die Möglichkeiten sind mit Admin Page Creator endlos, und das ist es, was wir im heutigen Tutorial erkunden werden.

## Erstellen Ihrer ersten benutzerdefinierten Admin-Seite

Nach der Installation und Aktivierung von Ultimate Multisite: Admin Page Creator gehen Sie zum Netzwerk-Admin-Menü und wählen Sie Admin Pages.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-Aml1wdfXt-69C790DA-7E3F-4DDD-A381-7F6633D91B2A.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-Aml1wdfXt-69C790DA-7E3F-4DDD-A381-7F6633D91B2A.png)

_Liste der benutzerdefinierten Admin-Seiten im Netzwerk-Admin_

Sie sehen eine Liste aller bisher erstellten Admin-Seiten (da dies Ihr erster Besuch ist, wird die Liste leer sein). Klicken Sie auf **Neue Admin-Seite hinzufügen**. Sie werden zum Admin Page Editor weitergeleitet.

## Die Elemente einer benutzerdefinierten Admin-Seite

Im Admin Page Editor haben Sie verschiedene Optionen, um Ihre Seite anzupassen.

### Seitentitel

Dies ist der Titel, der oben auf der Seite angezeigt wird, nicht das Label im Admin-Menü.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-ujx2jIkyR-29CF9DC4-9B94-493B-99C1-5D7F4FD13C4C.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-ujx2jIkyR-29CF9DC4-9B94-493B-99C1-5D7F4FD13C4C.png)

_Geben Sie einen Seitentitel ein_

### Inhaltstyp-Auswahl

Ultimate Multisite: Admin Page Creator bietet Ihnen zwei Optionen für den Inhalt der Seite: Sie können entweder den Standard-WYSIWYG-Editor von WordPress verwenden oder zu einem HTML-Editor wechseln.

Wenn Sie mit HTML nicht vertraut sind, bleiben Sie beim Standard-WordPress-Editor. Sie können den Inhalt der Seite genauso bearbeiten, wie Sie es bei einem normalen Beitrag oder einer normalen Seite tun würden.

## Menüoptionen

Auf der Seite, im Meta-Box „Allgemeine Optionen“, können Sie den Menütyp, das Label und vieles mehr konfigurieren.

### Menütypen

Ihre neue benutzerdefinierte Admin-Seite kann zwei verschiedene Menütypen haben: Top-Level-Menü und Untermenü.

Eine Top-Level-Menüseite, wie der Name schon sagt, fügt Ihre neu erstellte Seite dem Admin-Menü neben den Standard-WordPress-Seiten hinzu.

Wenn Sie die Top-Level-Option wählen, können Sie auch die Menüreihenfolge und ein Menüicon auswählen.

Wenn Sie diese Seite als Untermenüseite erstellen möchten, können Sie eine übergeordnete Seite auswählen, an die diese Seite angehängt wird. Sie können Ihre benutzerdefinierte Seite zu den Standard-WordPress-Seiten oder zu Ihren eigenen Top-Level-Admin-Seiten hinzufügen.

## Erweiterte Optionen

Sie sollten in der Lage sein, allerlei Inhalte zu Ihren benutzerdefinierten Admin-Seiten hinzuzufügen. Dazu gehören CSS-Regeln und -Dateien, JavaScript und externe Bibliotheken. Sie können dies mit dem Tab „Erweiterte Optionen“ tun.

### Custom CSS

Im CSS-Tab können Sie Ihren eigenen CSS-Code hinzufügen sowie CSS-Dateien von entfernten Websites importieren (wie CDNs, Google Fonts usw.).

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-NwRQA4Lgk-AFD43C6C-1200-48C9-BDD9-64420A03EEB7.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-NwRQA4Lgk-AFD43C6C-1200-48C9-BDD9-64420A03EEB7.png)

_Benutzerdefiniertes CSS wird ebenfalls unterstützt. Sie können auch externe Style-Dateien einbinden!_

### Custom JavaScript

Das Gleiche gilt für JavaScript. Sie können externe Bibliotheken hinzufügen, die Sie in Ihrem benutzerdefinierten JavaScript-Code verwenden.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-Uspm22WJI-E1D47212-D5C5-4B19-8E41-2BA85EADC4FE.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-Uspm22WJI-E1D47212-D5C5-4B19-8E41-2BA85EADC4FE.png)

_Fügen Sie Ihren eigenen JavaScript-Code hinzu!_

## Berechtigungen

Auch in den erweiterten Optionen können Sie auswählen, welche Subsites Zugriff auf diese Seite haben. Das ist nützlich, wenn Sie beispielsweise unterschiedlichen Planstufen exklusiven Inhalt bereitstellen möchten.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-AJ3iDzsqH-E1389350-44A7-4D61-BE8A-6D5330B02333.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-AJ3iDzsqH-E1389350-44A7-4D61-BE8A-6D5330B02333.png)

_Sie haben die volle Kontrolle darüber, wo diese Seite angezeigt wird_

**Wichtig** : Beide Bedingungen müssen erfüllt sein, damit die Seite angezeigt wird. Das bedeutet, dass die Seite nicht im Panel des Benutzers erscheint, wenn der Benutzer ein Kunde eines bestimmten Plans ist, aber nicht die ausgewählte Rolle hat (oder umgekehrt).

## Aktiv

Wenn Sie eine benutzerdefinierte Admin-Seite deaktivieren möchten, ohne sie zu löschen oder die Berechtigungen zu ändern, deaktivieren Sie einfach die Option Aktiv.

Das Deaktivieren einer Top-Level-Seite deaktiviert automatisch alle Unterseiten, die an diese übergeordnete Seite angehängt sind.
