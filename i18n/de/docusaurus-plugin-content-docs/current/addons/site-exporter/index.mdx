# Site Exporter

## Content

Wir verstehen die Notwendigkeit, Unterseiten innerhalb Ihres Netzwerks zu verschieben, um zu einer Einzel-Website-Installation zu gelangen und umgekehrt, falls Ihr Kunde entscheidet, seine Website zu migrieren. Das Site Exporter und Importer Addon erleichtert diese Aufgabe. Dieses Tool exportiert und importiert Websites mithilfe von ZIP-Dateien, die auch als Backup dienen können.

## Features

* Exportieren und Importieren von Websites
* Benutzerfreundliche Oberfläche
* Erstellen Sie ein Backup der Website

## Installation

1. Laden Sie **ultimate-multisite-site-exporter** in das Verzeichnis **/wp-content/plugins/** hoch
2. Aktivieren Sie das Plugin über das Menü **Plugins** in WordPress
3. Folgen Sie dem Schritt-für-Schritt-Assistenten, um das Plugin einzurichten

## Usage Guide

Mit Ultimate Multisite können Sie eine Website aus Ihrem Netzwerk exportieren oder eine Website in Ihre Multisite- oder Einzel-Website-Installation importieren.

To start the process, go to our add-ons page and install the Ultimate Multisite: Site Exporter & Importer add-on.![Add-ons page](/img/admin/addons-page.png)

![Add-ons page with install option](/img/admin/addons-page.png)

Go to your Plugins page and click to **Network Activate** the add-on you have just installed.<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Site Exporter addon -->

Die Option **Export & Import** finden Sie im Ultimate Multisite-Menü auf der linken Seite.

<!-- Screenshot unavailable: Ultimate Multisite menu showing Export and Import option -->

## How to export a site

Um eine Website aus Ihrem Netzwerk zu exportieren, gehen Sie zu **Export & Import** im Ultimate Multisite-Menü und klicken Sie auf der Registerkarte **Export Sites** auf **Generate new Site Export**.

<!-- Screenshot unavailable: Export Sites tab with Generate new Site Export button -->

Search for the site you want to export and select the items you want to include in the exported file. You can include themes, plugins, and uploads. If your site is too big, you can choose to run the export process in the background.<!-- Screenshot unavailable: Site export options with site search, include themes/plugins/uploads checkboxes -->

Click the **Export Site** button. Your exported site will be available on the **Existing Exports** list. Click the **Download ZIP** button to download the file.<!-- Screenshot unavailable: Existing Exports list with Download ZIP button -->

## How to import a site

Um eine Website zu importieren, gehen Sie zur Registerkarte **Import a Site** und laden Sie die .zip-Datei der Website hoch, die Sie importieren möchten. Sie können die URL auf eine neue ändern oder die alte URL beibehalten, indem Sie das Feld leer lassen. Außerdem können Sie das Kontrollkästchen aktivieren, um die .zip-Datei nach dem Import aus Ihrer Installation zu löschen. Klicken Sie dann auf **Begin Import**.

<!-- Screenshot unavailable: Import a Site tab with zip upload field and Begin Import button -->

Der Prozess kann etwas dauern. Sie sehen einen ausstehenden Import in der Liste unter dem Importer.

<!-- Screenshot unavailable: Pending import in the list below the importer -->

Sie können den Prozess stoppen, indem Sie auf **Cancel Import** klicken.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Nach Abschluss des Prozesses steht Ihre Website bereits in Ihrem Netzwerk zur Verfügung.

## How to import a site on a single install

Um eine Website in Ihre Einzel-Installation zu importieren, installieren und aktivieren Sie zunächst den Importer auf Ihrer Website. Sie finden den Download-Link im Exporter-Tab.

<!-- Screenshot unavailable: Exporter tab with link to download single-site importer -->

Sie finden den Importer im Menü **Tools** auf der linken Seite, unter dem Unterpunkt **Export & Import**.

<!-- Screenshot unavailable: Single-site Tools menu showing Export and Import submenu -->

Laden Sie die .zip-Datei der Website hoch, die Sie importieren möchten. Sie können die URL auf eine neue ändern oder die alte URL beibehalten, indem Sie das Feld leer lassen. Außerdem können Sie das Kontrollkästchen aktivieren, um die .zip-Datei nach dem Import aus Ihrer Installation zu löschen. Klicken Sie dann auf **Begin Import**.

<!-- Screenshot unavailable: Single-site import form with zip upload and Begin Import button -->

Der Prozess kann etwas dauern. Sie sehen einen ausstehenden Import in der Liste unter dem Importer.

<!-- Screenshot unavailable: Single-site pending import in the list -->

Sie können den Prozess stoppen, indem Sie auf **Cancel Import** klicken.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Nach Abschluss des Prozesses wird Ihre Website durch die gerade importierte Website ersetzt.
