# Site Exporter

## Content

Wir verstehen die Notwendigkeit, Unterseiten innerhalb Ihres Netzwerks zu verschieben, um zu einer Einzel-Website-Installation zu gelangen und umgekehrt, falls Ihr Kunde entscheidet, seine Website zu migrieren. Das Site Exporter und Importer Addon erleichtert diese Aufgabe. Dieses Tool exportiert und importiert Websites mithilfe von ZIP-Dateien, die auch als Backup dienen können.

## Features

* Exportieren und Importieren von Websites
* Benutzerfreundliche Oberfläche
* Erstellen Sie ein Backup der Website

## Documentation

## Installation

1. Laden Sie **ultimate-multisite-site-exporter** in das Verzeichnis **/wp-content/plugins/** hoch
2. Aktivieren Sie das Plugin über das Menü **Plugins** in WordPress
3. Folgen Sie dem Schritt-für-Schritt-Assistenten, um das Plugin einzurichten

## How to export a site

Um eine Website aus Ihrem Netzwerk zu exportieren, gehen Sie zu **Export & Import** im Ultimate Multisite-Menü und klicken Sie auf der Registerkarte **Export Sites** auf **Generate new Site Export**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-_bBztobh1xIFoj0Zo3zo7bx842u3XFMgpc2b8kiHmGmBU6RjuxlSOSY4ZNzaWTPOJrb0TIW_Q0Dv9WvT0yD-WwOkZ7bR36Ij5aJa_l19cDTfuj9DADQqI5dGy3eIeUmoQA=s0)

Suchen Sie die Website, die Sie exportieren möchten, und wählen Sie die Elemente aus, die Sie in die exportierte Datei aufnehmen möchten. Sie können Themes, Plugins und Uploads einbeziehen. Wenn Ihre Website zu groß ist, können Sie den Exportprozess im Hintergrund ausführen.![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-fk-i4RP0uZFj8kgfcQeRfAqWtCBelOjpOQARFfeBisVkeRSc1Qywq6Dpy66MnB6oBbnvdfJM359LdRSgqCLU1tNozo1u1OFH5svxtld8TZAXHY-CPGh92gWz9HQ8nlUcjg=s0)

Klicken Sie auf die Schaltfläche **Export Site**. Ihre exportierte Website wird in der Liste **Existing Exports** angezeigt. Klicken Sie auf die Schaltfläche **Download ZIP**, um die Datei herunterzuladen.![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-mouH16ZCUuNaUg0PwF2JmMhhsKPJOztau21QIeB7DXqAcX4VI5FL_itxvnKmFo439xuGUWsntM3doG6yBkjS2gi6bNTjg-kLHC19m19br5xsgBl0XH8hy5LvuPN1uNtx9g=s0)

## How to import a site

Um eine Website zu importieren, gehen Sie zur Registerkarte **Import a Site** und laden Sie die .zip-Datei der Website hoch, die Sie importieren möchten. Sie können die URL auf eine neue ändern oder die alte URL beibehalten, indem Sie das Feld leer lassen. Außerdem können Sie das Kontrollkästchen aktivieren, um die .zip-Datei nach dem Import aus Ihrer Installation zu löschen. Klicken Sie dann auf **Begin Import**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-Uq6wNJ4TPTuAtSJqKunxSJEn5cli7vbwYGgL1G3fcHwdvU7gSwW_J4xlBAtZ1fyVbDU7Pupi8uNFhw1Lzm4S2fjVTkbQoz7fiZPIc1qiy8qj3iGt-MNVlyLWmIZXFxPLYA=s0)

Der Prozess kann etwas dauern. Sie sehen einen ausstehenden Import in der Liste unter dem Importer.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-kMkwSY3bYrE5vjfJtLviYImhNBn_PwmuamaRDlQKS9rGwIy3a_jXV0glMNTU8fBmV6IKkNN7obXNQFqNaAJtAAbWDftzX76-qdCDKCrqdru_9taciFCayE9FNszZJTODvA=s0)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-oGmtCq7br6Y9GAfw_Ch53TP8nkJ0hw77sQJpSCQ7fEj5qk2lJo3WuMl7NYqB9c8SmORis53bWHILwGLxUTbpv4trWhxibu-yu5DNu6yNWXfkRo0bcnAZJ5F2K1hYIMH6PQ=s0)
