# Site Exporter {#site-exporter}

## Inhalt {#content}

Wir verstehen, wie wichtig es ist, Unter-Websites innerhalb Ihres Netzwerks in eine Einzel-Website-Installation verschieben zu können und umgekehrt, falls Ihr Kunde sich entscheidet, seine Website zu migrieren. Das Site-Exporter- und -Importer-Addon erleichtert diese Aufgabe. Dieses Tool exportiert und importiert Websites mithilfe von ZIP-Dateien, die auch als Backup dienen können.

## Funktionen {#features}

* Websites exportieren und importieren

* Benutzerfreundliche Oberfläche

* Website-Backup erstellen

## Dokumentation {#documentation}
**So exportieren und importieren Sie eine Website**

## Installation {#installation}

1. Laden Sie **ultimate-multisite-site-exporter** in das Verzeichnis **/wp-content/plugins/** hoch
2. Aktivieren Sie das Plugin über das Menü **Plugins** in WordPress
3. Folgen Sie dem Schritt-für-Schritt-Assistenten, um das Plugin einzurichten

## Nutzungsanleitung {#usage-guide}

Mit Ultimate Multisite können Sie eine Website aus Ihrem Netzwerk exportieren oder eine Website in Ihre Multisite- oder Einzel-Website-Installation importieren.

Um den Prozess zu starten, gehen Sie zu unserer Add-ons-Seite und installieren Sie das Add-on Ultimate Multisite: Site Exporter & Importer.![Add-ons-Seite](/img/admin/addons-page.png)

![Add-ons-Seite mit Installationsoption](/img/admin/addons-page.png)

Gehen Sie zu Ihrer Plugins-Seite und klicken Sie auf **Netzwerkweit aktivieren**, um das Add-on zu aktivieren, das Sie gerade installiert haben.<!-- Screenshot nicht verfügbar: WP Plugins-Seite mit Schaltfläche „Netzwerkweit aktivieren“ für das Site Exporter-Addon -->

Die Option **Export & Import** finden Sie im Ultimate Multisite-Menü auf der linken Seite.

## So exportieren Sie eine Website {#how-to-export-a-site}

Um eine Website aus Ihrem Netzwerk zu exportieren, gehen Sie im Ultimate Multisite-Menü zu **Export & Import** und klicken Sie auf dem Tab **Websites exportieren** auf **Neuen Website-Export erstellen**.

<!-- Screenshot nicht verfügbar: Tab „Websites exportieren“ mit Schaltfläche „Neuen Website-Export erstellen“ -->

Suchen Sie nach der Website, die Sie exportieren möchten, und wählen Sie die Elemente aus, die Sie in die exportierte Datei aufnehmen möchten. Sie können Themes, Plugins und Uploads einschließen. Wenn Ihre Website zu groß ist, können Sie den Exportprozess im Hintergrund ausführen lassen.<!-- Screenshot nicht verfügbar: Website-Exportoptionen mit Website-Suche und Kontrollkästchen zum Einschließen von Themes/Plugins/Uploads -->

Klicken Sie auf die Schaltfläche **Website exportieren**. Ihre exportierte Website ist in der Liste **Vorhandene Exporte** verfügbar. Klicken Sie auf die Schaltfläche **ZIP herunterladen**, um die Datei herunterzuladen. Export-ZIPs enthalten eine selbststartende `index.php` und `readme.txt`, sodass sie auf einem neuen Host installiert werden können, ohne zuerst ein separates Importer-Plugin zu installieren.<!-- Screenshot nicht verfügbar: Liste „Vorhandene Exporte“ mit Schaltfläche „ZIP herunterladen“ -->

## So exportieren Sie das gesamte Netzwerk {#how-to-export-the-whole-network}

Um alle Unter-Websites gemeinsam zu exportieren, gehen Sie im Ultimate Multisite-Menü zu **Export & Import** und verwenden Sie die Option **Netzwerk-Export** auf dem Exportbildschirm. Das generierte Archiv enthält alle Unter-Websites in einer einzigen ZIP-Datei für Host-Migrationen, Staging-Neuaufbauten oder Backups auf Netzwerkebene.

Netzwerk-Exporte können groß sein. Führen Sie sie aus, wenn der Traffic gering ist, und bestätigen Sie, dass der Server über ausreichend Speicherplatz für das Archiv und alle eingeschlossenen Uploads, Plugins und Themes verfügt.

## So importieren Sie eine Website {#how-to-import-a-site}

Um eine Website zu importieren, gehen Sie zum Tab **Website importieren** und laden Sie die .zip-Datei der Website hoch, die Sie importieren möchten. Sie können die URL in eine neue ändern oder die alte URL beibehalten, indem Sie das Feld leer lassen. Außerdem können Sie das Kästchen aktivieren, um die .zip-Datei nach dem Import aus Ihrer Installation zu löschen. Klicken Sie dann auf **Import starten**.

<!-- Screenshot nicht verfügbar: Tab „Website importieren“ mit ZIP-Upload-Feld und Schaltfläche „Import starten“ -->

Der Prozess kann eine Weile dauern. Sie sehen einen ausstehenden Import in der Liste unter dem Importer.

<!-- Screenshot nicht verfügbar: Ausstehender Import in der Liste unter dem Importer -->

Sie können den Prozess stoppen, indem Sie auf **Import abbrechen** klicken.

<!-- Screenshot nicht verfügbar: Schaltfläche „Import abbrechen“, um den Importprozess zu stoppen -->

Nachdem der Prozess abgeschlossen ist, ist Ihre Website bereits in Ihrem Netzwerk verfügbar.

## So importieren Sie eine Website in eine Einzelinstallation {#how-to-import-a-site-on-a-single-install}

Um eine Website in Ihre Einzelinstallation zu importieren, installieren und aktivieren Sie zuerst den Importer auf Ihrer Website. Den Link zum Herunterladen finden Sie auf dem Exporter-Tab.

<!-- Screenshot nicht verfügbar: Exporter-Tab mit Link zum Herunterladen des Einzel-Website-Importers -->

Sie finden den Importer im Menü **Tools** auf der linken Seite im Unterpunkt **Export & Import**.

<!-- Screenshot nicht verfügbar: Einzel-Website-Menü „Tools“ mit Untermenü „Export und Import“ -->

Laden Sie die .zip-Datei der Website hoch, die Sie importieren möchten. Sie können die URL in eine neue ändern oder die alte URL beibehalten, indem Sie das Feld leer lassen. Außerdem können Sie das Kästchen aktivieren, um die .zip-Datei nach dem Import aus Ihrer Installation zu löschen. Klicken Sie dann auf **Import starten**.

<!-- Screenshot nicht verfügbar: Einzel-Website-Importformular mit ZIP-Upload und Schaltfläche „Import starten“ -->

Der Prozess kann eine Weile dauern. Sie sehen einen ausstehenden Import in der Liste unter dem Importer.

<!-- Screenshot nicht verfügbar: Ausstehender Einzel-Website-Import in der Liste -->

Sie können den Prozess stoppen, indem Sie auf **Import abbrechen** klicken.

<!-- Screenshot nicht verfügbar: Schaltfläche „Import abbrechen“, um den Importprozess zu stoppen -->

Nachdem der Prozess abgeschlossen ist, wird Ihre Website durch die Website ersetzt, die Sie gerade importiert haben.
