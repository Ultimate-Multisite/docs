# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Bearbeiten oder verbergen Sie die Metadaten (Titel, Autor, Miniaturansicht und Beschreibung) installierter Plugins und Themes. Erstellen Sie Kategorien für Plugins und Themes, damit Ihre Benutzer sie filtern können, und fügen Sie eine schöne benutzerdefinierte Plugins-Seite zu Ihrem Benutzerpanel hinzu.

Um das Plugin zu verwenden, navigieren Sie einfach zu Ihren Plugins- oder Themes-Seiten im Netzwerk-Admin und klicken Sie auf „Edit Plugin/Theme info“.

## Installation

1. Laden Sie 'ultimate-multisite-plugin-and-theme-manager' in das Verzeichnis '/wp-content/plugins/' hoch
2. Aktivieren Sie das Plugin über das Menü 'Plugins' in Ihrem WordPress Network Admin

## Usage Guide

Ultimate Multisite: Plugin and Theme Manager ist ein leistungsstarkes **premium** Ultimate Multisite Add-on, das Ihnen die volle Kontrolle darüber gibt, wie Ihre Netzwerkbenutzer die Plugin- und Theme-Optionen auf Ihrem Netzwerk sehen und mit ihnen interagieren.

Dieses Add-on hat drei Zwecke:

  1. Es ermöglicht Ihnen, die Metadaten beider Plugins und Themes vollständig zu ändern. Dazu gehören Name, Beschreibung, Autorname und sogar Miniaturansichten!

  2. Es ermöglicht Ihnen, Plugins und Themes zu kategorisieren, damit Ihre Benutzer sie leicht sortieren können.

  3. Es bietet Ihnen die Möglichkeit, die alte, langweilige Plugins-Seite durch eine neue, themenähnliche Seite mit Sortierung nach Kategorien zu ersetzen.

Nach dem Herunterladen und Installieren erscheint ein neuer Tab - _Add-on Settings_ \- auf der Ultimate Multisite Settings-Seite.
