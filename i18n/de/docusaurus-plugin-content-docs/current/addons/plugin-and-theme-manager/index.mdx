# Plugin- & Theme-Manager

Multisite Ultimate: Plugin- und Theme-Manager

Bearbeite oder verberge die Metainformationen (Titel, Autor, Vorschaubild und Beschreibung) installierter Plugins und Themes. Erstelle Kategorien für Plugins und Themes, damit deine Benutzer sie filtern können, und füge dem Panel deiner Benutzer eine schöne benutzerdefinierte Plugins-Seite hinzu.

Um das Plugin zu verwenden, kannst du einfach in deinem Network Admin zu deinen Plugins- oder Themes-Seiten navigieren und auf „Plugin-/Theme-Info bearbeiten“ klicken.

## Installation {#installation}

1. Lade 'ultimate-multisite-plugin-and-theme-manager' in das Verzeichnis '/wp-content/plugins/' hoch
2. Aktiviere das Plugin über das Menü 'Plugins' in deinem WordPress Network Admin

## Nutzungsanleitung {#usage-guide}

Ultimate Multisite: Plugin- und Theme-Manager ist ein leistungsstarkes **Premium**-Add-on für Ultimate Multisite, das dir die volle Kontrolle darüber gibt, wie deine Netzwerkbenutzer die Plugin- und Theme-Optionen in deinem Netzwerk sehen und damit interagieren.

Dieses Add-on erfüllt drei Zwecke:

  1. Es ermöglicht dir, die Metadaten von Plugins und Themes vollständig zu ändern. Dazu gehören Name, Beschreibung, Autorenname und sogar Vorschaubilder!

  2. Es ermöglicht dir, Plugins und Themes zu kategorisieren, damit deine Benutzer sie einfach sortieren können.

  3. Es gibt dir die Möglichkeit, die alte, langweilige Plugins-Seite durch eine neue, Themes-ähnliche Seite mit Sortierung nach Kategorien zu ersetzen.

Nach dem Herunterladen und Installieren wird auf der Ultimate Multisite-Einstellungsseite ein neuer Tab - _Add-on-Einstellungen_ \- angezeigt.
