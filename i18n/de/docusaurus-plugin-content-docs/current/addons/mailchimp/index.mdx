# Mailchimp

Multisite Ultimate: Mailchimp Integration

Integrieren Sie Ihr Multisite Ultimate-Netzwerk mit Ihrem Mailchimp. Sie können bestimmte Pläne mit Mailchimp-Listen verknüpfen und Abonnenten automatisch bei der Anmeldung zu diesen Listen hinzufügen.

## Installation

1. Laden Sie 'ultimate-multisite-mailchimp' in das Verzeichnis '/wp-content/plugins/' hoch
2. Aktivieren Sie das Plugin über das Menü 'Plugins' in Ihrem WordPress Network Admin

## Usage Guide

Das Synchronisieren der E-Mails von Abonnenten mit Ihren Mailchimp-Listen ist mit unserem Ultimate Multisite: MailChimp Integration Add-on wirklich einfach. Dieses Tutorial soll Ihnen dabei helfen, alles einzurichten!

## Getting an API key

Bevor wir in die verschiedenen verfügbaren Einstellungen einsteigen, benötigen wir einen Mailchimp-API-Schlüssel. Damit kann Ultimate Multisite mit der Mailchimp-API kommunizieren, um Listen, Gruppen abzurufen und E-Mail-Adressen zu diesen Listen und Gruppen hinzuzufügen, wenn sich Benutzer anmelden.

Folgen Sie diesem Tutorial, um Ihren eigenen Mailchimp-API-Schlüssel zu erhalten: [Über API-Schlüssel | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-zimbKYFZo-B4D1E28C-1203-4E9A-AC7B-D8E9B5659B93.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-zimbKYFZo-B4D1E28C-1203-4E9A-AC7B-D8E9B5659B93.png)

_Fügen Sie Ihren Mailchimp-API-Schlüssel in_ _**Ultimate Multisite Settings -> Add-on Settings -> Mailchimp**_

## Integration Mode

Das Mailchimp Integration Add-on bietet zwei „Integrationsmodi“, was bedeutet, dass es zwei verschiedene Möglichkeiten gibt, das Add-on zum Synchronisieren von E-Mail-Adressen zu verwenden.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-mc8dVzW5S-B7A9495E-05B8-4FBB-8954-80EBF4A9CB3B.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-mc8dVzW5S-B7A9495E-05B8-4FBB-8954-80EBF4A9CB3B.png)

_Zwei Integrationsmodi für mehr Flexibilität!_

### Mode 1: Multiple Lists

Der Standardmodus nutzt **Multiple Lists**. Dieser Modus ermöglicht es Ihnen, für jeden Plan mehrere Mailchimp-Listen auszuwählen. Das ist die beste Option, wenn Sie Ihre Benutzer in verschiedene Listen segmentieren.

Beispielsweise nehmen wir an, Sie haben drei verschiedene Listen in Ihrem Mailchimp-Konto: **Newsletter**, **Plan A** und **Plan B**. Sie möchten, dass alle Benutzer, unabhängig von ihrem Plan, ebenfalls in der Newsletter-Liste enthalten sind, damit sie Ihre Updates nicht verpassen.

Um dies zu erreichen, können Sie den Multiple Lists-Modus verwenden, um sowohl die Listen **Newsletter** als auch **Plan A** zu den Plan A Mailchimp-Einstellungen hinzuzufügen, und **Newsletter** sowie **Plan B** zu den Plan B Mailchimp-Einstellungen.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-kZS8-S_eh-4145C1B8-B903-4DFF-9D92-AF7E0A61445E.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/6017c85715d41b7c717cdcf9/images/6021265efb34b55df443e4ad/60212659ac2f834ec53865f3-kZS8-S%5Feh-4145C1B8-B903-4DFF-9D92-AF7E0A61445E.png)

_Wählen Sie die Standardlisten für neue Konten und für gekündigte Konten aus_

### Mode 2: Single List + Multiple Groups

Mailchimp ermöglicht es Ihnen, Ihre Liste in kleinere Gruppen von Benutzern namens **Groups** zu segmentieren. Dadurch haben Sie mehr Kontrolle, da Sie eine Kampagne an einen bestimmten Teil Ihrer Abonnenten senden können, ohne mehrere Kopien derselben Kampagne auf Mailchimp zu erstellen (was die Analyse der Kennzahlen dieser Kampagne erschwert).

Beispielsweise können Sie eine einzige Liste namens **My Network Subscribers** haben und darin Gruppen für **Plan A** und **Plan B** einrichten.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-tOsErbJy6-A468DB31-B539-493E-AF5F-F44A4F90A974.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-tOsErbJy6-A468DB31-B539-493E-AF5F-F44A4F90A974.png)

_Wählen Sie die Standardlisten und Gruppen aus, wenn Sie den „single list + multiple groups“-Integrationsmodus verwenden_

## Default Lists and Groups

Nachdem Sie ausgewählt haben, welcher Integrationsmodus für Ihren Anwendungsfall am sinnvollsten ist, ist es an der Zeit, die Standardlisten (und Gruppen, je nach Modus) auszuwählen. Die Standardliste wird verwendet, wenn ein bestimmter Plan keine spezifische Liste ausgewählt hat.

Sie können auch die Standardliste für gekündigte Konten auswählen. Wann immer ein Benutzer sein Abonnement kündigt und sein Konto aus Ihrem Netzwerk entfernt, wird seine E-Mail-Adresse in die für gekündigte Konten festgelegte Liste verschoben.

## Enabling Double Subscription

Auf derselben Einstellungsseite können Sie **Double Subscription** aktivieren. Diese Option fügt den Benutzer sowohl der ausgewählten Standardliste als auch der Liste hinzu, die dem Plan des Benutzers zugeordnet ist. Wenn diese Option nicht aktiviert ist, wird der Benutzer nur der für den Plan des Benutzers ausgewählten Liste hinzugefügt. Wenn für diesen speziellen Plan keine Liste festgelegt ist, wird der Benutzer der ausgewählten Standardliste hinzugefügt.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-OPUDRnO94-5D4C267A-14EB-43B4-B86F-0328D330B521.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-OPUDRnO94-5D4C267A-14EB-43B4-B86F-0328D330B521.png)

_Sie können Ihre Benutzer so abonnieren, dass sie sowohl die Standardliste als auch die Planliste bei der Anmeldung erhalten_

## Selecting Lists and Groups for each of the Plans

Gehen Sie zu Ihren **Plans** und wählen Sie den Plan aus, den Sie bearbeiten möchten. Beachten Sie, dass im **Advanced Options** Meta-Box ein neuer Tab namens Mailchimp verfügbar ist.

Auf diesem Tab können Sie mehrere Listen oder eine einzelne Liste und mehrere Gruppen (je nach Integrationsmodus) für diesen Plan auswählen.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-BMHjubpN5-C3183A30-687C-4837-8350-842CF959BB06.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-BMHjubpN5-C3183A30-687C-4837-8350-842CF959BB06.png)

_Wählen Sie die Listen und Gruppen aus, die Sie für jeden Ihrer Pläne verwenden möchten_

Wenn sich ein neuer Benutzer für diesen Plan anmeldet, wird seine E-Mail-Adresse zu den ausgewählten Listen und Gruppen hinzugefügt!
