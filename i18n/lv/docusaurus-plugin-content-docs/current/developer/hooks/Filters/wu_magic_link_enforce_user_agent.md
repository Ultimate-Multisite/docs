# Filtrs: wu_magic_link_enforce_user_agent {#filter-wumagiclinkenforceuseragent}

Filtrē, vai piespiedu kārtā pārbaudīt lietotāja aģentu.

Iestatiet uz false, lai tokeni darbotos dažādās pārlūkprogrammās/ierīcēs. Tas samazina drošību, bet palielina lietojamību.

## Parametri {#parameters}

| Nosaukums | Tips | Apraksts |
|------|------|-------------|
| $enforce | `bool` | Vai piespiedu kārtā nodrošināt lietotāja aģenta atbilstību. |
