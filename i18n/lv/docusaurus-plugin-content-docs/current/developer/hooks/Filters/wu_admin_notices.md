# Filtrs: wu_admin_notices

Ļauj izstrādātājiem filtrēt administratora paziņojumus, ko pievieno Ultimate Multisite.

## Parametri

| Nosaukums | Tips | Apraksts |
|------|------|-------------|
| $notices | `array` | Paziņojumu saraksts šim konkrētajam panelim. |
| $all_notices | `array` | Pievienoto paziņojumu saraksts, sadalīts pēc paneļa. |
| $panel | `string` | Panelis, kuram izgūt paziņojumus. |
| $filter | `string` | Vai noraidāmie paziņojumi ir izfiltrēti. |
| $dismissed_messages | `array` | Noraidīto paziņojumu atslēgu saraksts. |
