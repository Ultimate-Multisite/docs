# Kā kārtot savam konta lapai pirmo reizi? {#how-can-i-access-my-account-page-for-the-first-time}

Laiks labienīja Ultimate Multisite!

Pievienojies savam konta lapai ar saiti, ko saņēmsi e-pastā pēc mūsu plāksmes iepirkuma:

<!-- Screenshot unavailable: Account page email with access link -->

Izmantojiet e-pastu, ko sniegli jautājumā iepircimas laikā, un par Latvian katru vārdu (parolī), ko saņēmsi e-pastā.

<!-- Screenshot unavailable: Login page with email and password fields -->

Tā kā tas ir jūsu pirmais laiks iekļūt šim kontam, jums tiks jautāts mainīt paroli. Mainiet to un nospiediet pogu _Save_ (Saglabāt).

<!-- Screenshot unavailable: Change password form with Save button -->

Jums savā konta lapā varat pārvaldīt visus savus konta datus, piemēram, fakturējumus, atjauninājumus, atcelto apmeklējumus un daudzas citās lietas.
