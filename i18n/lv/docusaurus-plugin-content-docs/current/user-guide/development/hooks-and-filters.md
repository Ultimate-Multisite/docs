# Ultimate Multisite hooks unkļūves (filters)

Ultimate Multisite ir veidots no daudz funkciju un databāzes laukumu. Tas ir ar "hooks" un "filters", kas ļauj jums izmaiņot vai pievienot jaunu kodu jau eksistējošām funkcijām.

Mēs sagādājām šo hooks un filters sarakstu, lai palīdzētu tehniski iezīmētā lietotā tur vairāk personalizēt savu tīklu. Šo sarakstu varat piekļūt GitHub no attiecīgās saites:

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
