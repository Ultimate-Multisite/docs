# Kā kādu palīdzību saņemt no Codeable?

Reiz tiem, kad mums patiešām nepieciešams kaut kas fantastisks pielāgojums, lai mūsu tīkla strūmes tieši tā, kā mēs vēlamos. Ja tas ir tas, ko jūs meklējat, mēs jums ieteikam [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), vietu, kurā varat atrast pieredzējusies attīstītājus ar Ultimate Multisite uzmanību.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ir platforma, kas savona klientus kvalificēti WordPress freela attīstītājiem, kuri var strādāt projektos jebkuras izmēra. Viņiem ir ļoti skaidrs pieteikuma proces, lai nodrošinātu, ka jūs esat atbilstoši ekspertiem, kas kvalificēti tieši tam uzdevumam, ko jūs domājat.

### Kā izveidot projektu Codeable?

Vispirms, dodieties uz [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) un noklikšķiniet uz pogas **Start a Project** (Sākt projektu).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Tagad, lai iesniegtu projektu, noklikšķiniet uz kreisās puses **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

"I need help with" (Man vajadzī palīdzību ar) izvēlieties **Customizing** (Pielāgojums):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

"My" (Mans) izvēlieties **Plugin** (Plugin):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Pēc tam noklikšķiniet uz pogas **Continue** (Turpināt).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Tagad ir laiks aprakstīt savu projektu un vajadzības vislabākā veidā. Mēģiniet sniegt pēc iespējas vairāk informāciju!

<!-- Screenshot unavailable: Codeable project description form -->

Atbildiet uz šiem jautājumiem, lai palīdzētu ar cenām.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Pēc tam, ja jums vēl nav [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) konta, izveido viens un nospiediet pogu **Publish My Project** (Publicēt man ko projektu).

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Kad būs publicējis savu [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projektu, platforma jums nodrošinās saistīties ar izvilcinātājiem, kuri atbilst jūsu projekta prasībām un paredzētajai prasmei.

Šis proces palīdz atrast tieši cilvēku, kuram ir pieredze WordPress un Ultimate Multisite!

Codeable minimālais darba skaits ir $70–$120 USD stundu laikā. Izskatu nosacījumi mainās atkarībā no darba apjoma, sarežģītības un projekta svarīguma.

Kad saņemsiet izskatu, jūs varat uzņemt darbinieku, kuram labāk pielāgojas jūsu vajadzībām un kopā sākāt savu projektu!
