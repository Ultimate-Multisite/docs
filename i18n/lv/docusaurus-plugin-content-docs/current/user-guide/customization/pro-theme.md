# Pārpersonalizēšana administrācijas panelim ar PRO tēmām

Darbā, kad mēs veidām klientiem darbus, bieži ir nepieciešamība sniegt viņiem personalizētu pieredzi WordPress administrācijas paneli. Ja jūs meklējat profesionālu skatu un jomtu, PRO Theme var jums palīdzēt.

PRO Theme ir administrācijas tēma, kas izstrādāta, lai palīdzētu jums personalizēt klientu paneli. Tas ir arī daļa no Ultimate Multisite premium pluginu pamata.

Instrucijām pēc instalēšanas dodieties uz savu _Plugins_ lapu un noklikšķiniet uz to, lai aktivizētu to visā jūsu tīklā.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktivizējiet to visā tīklā savā Plugins lapā_

_**PĀRMANĪBA:**_ _Ja jūs to tikai instalējat, tas parādīsies klientam kā opcija, ko vari atzīmēt, lai pievienotu tēmām tikai šajam lietotājam._

Tagad jums ir pilnībā redizainēts administrācijas interfeiss! Jums ir arī jauns menu left kolumnā ar nosaukumu PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Jaunais skatsamība administrācijas paneli_

Šajā menī jums būs piekļuve pie piecijiem tabulām:

<!-- Screenshot unavailable: PRO Theme Styling Settings tab -->

_Styling Settings_ (Stilisma iestatījumi)

<!-- Screenshot unavailable: PRO Theme Functionality tab -->

_Functionality_ (Funkcionalitāte)

<!-- Screenshot unavailable: PRO Theme Custom CSS tab -->

_Custom CSS_ (Personalizēta CSS)

<!-- Screenshot unavailable: PRO Theme Import/Export tab -->

_Import/Export_ (Ievads/Izveide)

_Activate PRO Theme_ (Aktivizēt PRO Theme)
