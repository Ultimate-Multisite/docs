# Ultimate Multisite Dashboard uncerēšana

Pirkties iepazīstamies ar vairāk uzlabotajiem konfigurēšanas opcijām un Ultimate Multisite operacionālajiem aspektiem, atspējiet skatīt dashboardu.

Līdzīgi kā daudziem WordPress pluginiem, Ultimate Multisite izveido tīkla līmeņa menu iekšmeklētāju ar nosaukumu **Ultimate Multisite**, kuram ir vairāki apakšmenu iekšmeklētāji.

## Tīkla dashboardu widgeti {#network-dashboard-widgets}

Ultimate Multisite ieplūst dažus noderīgus widgetus tīkla līmeņa dashboarda. Šo dashboardu var atrast, navigējot uz **My Sites → Network Admin → Dashboard**. Widgeti ir izveidoti nodrošinot vieglu piekļuvi informācijai un biežām operācijām tīkla administratoram.

![Tīkla dashboard ar Ultimate Multisite widgetiem](/img/admin/network-dashboard-full.png)

### Pirmais soļums {#first-steps}

![Pirmais soļuma widget](/img/admin/network-dashboard-first-steps.png)

Šis widget parādās Ultimate Multisite jauna instalācijā un atgādina tīkla administratoram pabeigt papildu darbus, piemēram, maksājuma metodes konfigurēšanu un testbrauktu izveidi.

### Kopsavilkums {#summary}

![Kopsavilkuma widget](/img/admin/network-dashboard-summary.png)

Kopsavilkuma widget atzīmē reģistrācijas skaitu un viņu ienamniecības ienamniecību dienas laikā. MRR (Mēnesīgs atkārtots ienamniecība) norāda projicētu kopējo summu visiem klientiem, kuri ir ienamniecības ienamniecības ienamniecības.

### Aktivitātes strāvas {#activity-stream}

![Aktivitātes strāvas widget](/img/admin/network-dashboard-activity-stream.png)

Šis widget atzīmē notikumus Ultimate Multisite apmeklējumos. Notikumi ietver reģistrācijas, atbalstu, vietņu izveidi, izslēgšanu un citus aktivitātes.

Par vairāk detaļu aprakstu lūdzu apskatīt dokumentācijas Events sadaļu.

### Tagad {#right-now}

![Tagad widget](/img/admin/network-dashboard-right-now.png)

Šis lietderīgs widget parāda īsu kopsavilkumu par lietotājiem un vietnēm tīklā. Augšējās vienkāršas skatuves ļauj vienreizspēju izveidot jaunas vietnes vai lietotājus.

### Ziņas un diskusijas {#news-and-discussions}

![Ziņu un diskusiju widget](/img/admin/network-dashboard-news.png)

Šis widget savokās un parāda jaunākās Ultimate Multisite ziņas. Lūdzu, sekojiet šim widgetam informācijai par atjauninājumiem, kritiskajām kļūdiem un drošības labojumiem.

## Ultimate Multisite Dashboard {#ultimate-multisite-dashboard}

Tur, kur tīkla līmeņa dashboard prezentē vispārīgu informāciju par tīklu, Ultimate Multisite dashboard (meklējiet to Ultimate Multisite menija augšpusē) prezentē informāciju par pakalpojumu.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### Mēnesīgs atkārtots ienākumu (MRR) augums {#monthly-recurring-revenue-growth}

MRR (Mēnesīgs atkārtots ienākums) grafiks parāda mēnesīgu pārskatu ienākumiem 12 mēnešu kalendā periodā.

![MRR Auguma grafiks](/img/admin/um-dashboard-mrr.png)

Jauni ienākumi tiek uzraudzīti zilā krāsā, bet atbalsts tiek atspoguļots sarkanā krāsā.

### Datuma diapazīja atbalsts {#date-range-support}

Atkarībā no klientu un aktivitātes skaits statistikas var kļūt pārāk daudz. Dashboarda iekļauts datuma diapazīja atbalsts ļauj tīkla administratoram koncentrēties tikai uz nepieciešamajām informācijām un laika periodu.

![Datuma diapazījas izvēlne un statistikas](/img/admin/um-dashboard-date-range.png)

Datuma diapazīja norāda redzamo widgetu, tostarp ienākumus, jaunas pievienošanas, reģistrācijas pēc valstī, reģistrācijas pēc formām un visvisaprakstīgākajiem vietnēm periodu.

![Dashboard statistikas widgeti](/img/admin/um-dashboard-stats.png)

:::tip
Lai **Visaprakstīgākajiem vietnēm** (Most Visited Sites) widgetu darbinātu, atrodiet un ieslēciet apmeklējumu skaits **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**.
:::
