# Uzlabojot dalījam saites planiem

Ultimate Multisite piedāvā jums iespēju tieši ar klientiem dalīt planu saites. Lai to darītu, dodieties uz jebkurām jūsu planu rediģēšanas lapām un noklikšķiniet uz pogas _Click to copy the Shareable Link_ (Noklikšķiniet kopēt saiti).

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Planu rediģēšanas lapā varat atrast pogu, kas ļauj izveidot dalījamās saites._

Pieminājums: šis mehānisms izmanto default billing frekvenci saites veidošanai. Tas nozīmē, ka ja jūsu default billing ir mēnesīgs, piemēram, beigās jums būs saite tāds: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Ja vēlaties, lai dalījamā saite par default būtu trimēnesīga, jums jāpievieno 3 beigās:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Lai izvēlotu plani _premium_ gadu plānam, jums jāpievieno 1 un vārds:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Atcerieties, ka nevarat turēt vienu saiti dažādiem billing frekvencēm. Jums jādalās viens konkrēts saits katras jūsu billing frekvences ar klientiem.

Kad dalīsiet saiti saviem klientiem, tas tos novirzīs uz jūsu reģistrācijas procesu pirmo soļu.
