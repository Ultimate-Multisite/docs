# VAT

Ultimate Multisite: Eiropas PVN

Pievieno papildu funkcijas un rīkus Ultimate Multisite noklusējuma nodokļu apstrādei, lai labāk atbalstītu Eiropas PVN.
Automātiski pievieno ES PVN likmes Nodokļu likmju lapai no

## Instalēšana {#installation}

1. Augšupielādējiet 'ultimate-multisite-vat' direktorijā '/wp-content/plugins/'
2. Aktivizējiet spraudni WordPress izvēlnē 'Spraudņi'
3. Dodieties uz Ultimate Multisite > Iestatījumi > Nodokļi un iespējojiet ES PVN.
4. Noklikšķiniet uz Nodokļu likmes un apakšā atjauniniet ES PVN likmes, pēc tam saglabājiet nodokļu likmes.
5. Gatavs!
