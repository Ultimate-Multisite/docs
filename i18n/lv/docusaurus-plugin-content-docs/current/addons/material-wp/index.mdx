# Material WP

WordPress administratora saskarne jau gadiem ir bijusi tāda pati. Bet vai esat kādreiz aizdomājies, kā izskatītos administratora saskarnes Material Design versija? 

Material WP ir administratora theme, kas izstrādāta, lai sniegtu jums šo izskatu un sajūtu, palīdzot nodrošināt jūsu klientiem vairāk pielāgotu Dashboard.

## Funkcijas

- Pielāgota CSS atbalsts
- Vienkārši stila iestatījumi
- Izvēlnes pārkārtošana
- Vienkāršs imports/eksports

## Dokumentācija

**Administratora paneļa pielāgošana ar Material WP**

## Instalēšana

1. Augšupielādējiet 'material-wp' direktorijā '/wp-content/plugins/'
2. Aktivizējiet plugin, izmantojot WordPress izvēlni 'Spraudņi'

## Lietošanas ceļvedis
