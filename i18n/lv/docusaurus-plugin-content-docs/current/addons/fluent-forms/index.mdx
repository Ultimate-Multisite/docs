# Fluent Forms

Šis papildinājums ļauj pievienot ierobežojumus Fluent Forms vietnē Multisite Ultimate produktos, dalībās vai vietnēs.
Fluent Forms izmanto pielāgotas tabulas, nevis pielāgotus ierakstu tipus, tāpēc parastās ierobežošanas opcijas nedarbojas ar Fluent Forms.

## Funkcijas {#features}

- Pievienojiet ierobežojumus Fluent Forms vai Conversational Fluent Forms
- Klienti saņem draudzīgu ziņojumu ar aicinājumu jaunināt savu plānu, kad viņi sasniedz ierobežojumu
