# Filtrs: signup_get_available_languages

Filtrē pieejamo valodu sarakstu priekšgala vietņu reģistrācijām.

Tukša masīva nodošana šim hook atspējos šī iestatījuma izvadi reģistrācijas veidlapā, un, veidojot vietni, tiks izmantota noklusējuma valoda. Valodas, kas vēl nav instalētas, tiks noņemtas.

## Parametri

| Name | Type | Description |
|------|------|-------------|
| $available_languages | `array` | Pieejamās valodas. |

### Kopš

- 4.4.0
