# GoCardless izmaiņu žurnāls

* Labots: pamestas gaidošās vietnes tagad tiek notīrītas, kad GoCardless dalība tiek atcelta, kamēr tā ir gaidošā vai aizturētā stāvoklī
* Uzlabots: plugin automātiskais ielādētājs tiek izlaists, ja Bedrock saknes automātiskais ielādētājs jau ir ielādējis atkarības

* Labots: Webhook apstrādātājiem neizdevās apstiprināt parakstus;

2021-06-14 - Versija 1.0.0-beta.2
