# Satura sinhronizācija

Multisite Ultimate: Satura sinhronizācija

Šis Multisite Ultimate papildinājums ļauj tīkla administratoriem sinhronizēt saturu no jebkuras vietnes uz citām vietnēm tīklā. Tas atbalsta visus ierakstu tipus, tostarp ierakstus, lapas, pielāgotus ierakstu tipus un Elementor veidnes. Papildinājumā ir iekļauta vairāku soļu vedņa saskarne satura un mērķa vietņu atlasei, ar pakešapstrādi un reāllaika progresa atgriezenisko saiti.

## Funkcijas

* Satura atlase: Vizuāls kartīšu režģis ar sīktēliem un ierakstu tipu filtrēšanu
* Visi ierakstu tipi: Sinhronizējiet ierakstus, lapas, produktus, Elementor veidnes un jebkuru pielāgotu ierakstu tipu
* Vietņu atlase: Izvēlieties, uz kurām vietnēm sinhronizēt saturu, ar avota vietnes izslēgšanu un bērna vietņu iepriekšēju atlasi
* Pakešapstrāde: Sinhronizē uz 10 vietnēm vienlaikus ar progresa atjauninājumiem
* Multivides sinhronizācija: Automātiski kopē un pārkartē multivides failus (attēlus, video) no ieraksta satura uz mērķa vietnēm
* Taksonomiju sinhronizācija: Saglabā kategorijas, birkas un pielāgotas taksonomijas
* Elementor atbalsts: Pilns Elementor veidņu atbalsts ar CSS atkārtotu ģenerēšanu un veidņu ID pārkartēšanu
* Konfliktu risināšana: Iespēja aizstāt esošo saturu vai to izlaist
* Kešatmiņas pārvaldība: Automātiski notīra Elementor kešatmiņu mērķa vietnēs pēc sinhronizācijas

## Prasības

* WordPress 5.3 vai jaunāka versija
* PHP 7.4 vai jaunāka versija
* Multisite Ultimate plugin (aktīvs)
* Elementor plugin (neobligāts, nepieciešams Elementor veidņu sinhronizācijai)

## Instalēšana

1. Augšupielādējiet 'ultimate-multisite-content-sync' direktorijā '/wp-content/plugins/'
2. Aktivizējiet plugin, izmantojot izvēlni 'Spraudņi' savā WordPress Network Admin
3. Konfigurējiet iestatījumus sadaļā Ultimate Multisite → Iestatījumi → Satura sinhronizācija
4. Dodieties uz Ultimate Multisite → Satura sinhronizācija, lai sāktu sinhronizāciju
