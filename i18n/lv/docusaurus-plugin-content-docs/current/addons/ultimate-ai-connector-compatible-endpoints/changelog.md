# Ultimate AI Connector for saderīgu galapunktu izmaiņu žurnāls

## Versija 2.0.0 — Izlaista 2026-04-24 {#version-200--released-on-2026-04-24}

* Jauns: vairāku pakalpojumu sniedzēju atbalsts — konfigurējiet vairākus AI galapunktus un maršrutējiet pieprasījumus ar automātisku atkāpšanos starp pakalpojumu sniedzējiem.
* Labojums: vairāku pakalpojumu sniedzēju SDK integrācija ar pareiziem pakalpojumu sniedzēju ID, reģistrācijas URL un HTTP filtra tvērumu katram pakalpojumu sniedzējam.
* Labojums: jauna pakalpojumu sniedzēja kartīte tagad automātiski izvēršas pievienošanas brīdī; skripta kešatmiņas apiešana plugin atjaunināšanas laikā.
* Labojums: dinamiska pakalpojumu sniedzēja klases nosaukumvieta eval() un automātiskas izvēršanas darbībai tikko pievienotiem pakalpojumu sniedzējiem.
* Labojums: izmanto stabilus Card/CardBody/CardHeader/CardDivider komponentus (vairs ne eksperimentālus) WordPress 6.9+ saderībai.
* Labojums: aizstāts nedefinētais DragHandle ar Unicode satvēriena ikonu pakalpojumu sniedzēju vilkšanai, lai pārkārtotu.
* Labojums: novērsts bloķējošs HTTP pieprasījums, kas tika palaists katrā lapas ielādē.
* Uzlabots: GitHub Actions darbplūsmas jauninātas uz Node.js 24.
