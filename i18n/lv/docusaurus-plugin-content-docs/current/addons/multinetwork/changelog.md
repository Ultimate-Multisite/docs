# Multi-Network izmaiņu žurnāls

### 1.3.0
* Jauns: tīkla veidnes priekšskatītājs — pārlūkojiet un priekšskatiet tīkla veidnes ar tiešsaistes paneli pirms iegādes.
* Jauns: tīkla veidnes izvēlne norēķināšanās laikā — klienti var atlasīt tīkla veidni reģistrācijas norēķināšanās plūsmas laikā.
* Jauns: starptīklu magic link SSO — klienti var nevainojami piekļūt apakštīkliem, izmantojot magic-link autentifikāciju.
* Jauns: tīkla multivides klonēšana — vietnes multivide tiek pareizi dublēta, klonējot tīklus no veidnes.
* Labots: tīkla klonēšana tagad darbojas bez multi-tenancy addon; super admin privilēģijas tiek atkārtoti piešķirtas pēc klonēšanas.
* Labots: tīkla izveide tagad ir saderīga ar WooCommerce un visām maksājumu vārtejām.
* Labots: porta numurs tagad ir iekļauts rezerves domēna ģenerēšanā.
* Labots: veidnes izvēlnes CSS pārvietots uz rindā iekļautu stila lapu pareizai attēlošanai visās tēmās.
* Labots: aizsardzība pret nedefinētu `network_id` atslēgu vietnes vaicājuma tvērumā.
* Labots: tīkla atļauju pārbaude pirms konteksta pārslēgšanas novērš `wpdb` tabulas īpašību kļūdas.
