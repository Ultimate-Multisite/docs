# Darbība: move_site

Tiek palaista pēc tam, kad vietne ir pārvietota uz jaunu tīklu.

## Parametri {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | Pārvietotās vietnes ID. |
| $old_network_id | `int` | Vietnes sākotnējā tīkla ID. |
| $new_network_id | `int` | Tā tīkla ID, uz kuru vietne ir pārvietota. |
