# Atbalsta aģenti

Pārvaldīt veselu tīklu vienatnē nav viegls uzdevums. Kad jūsu tīkls sāk augt, jūs sāksiet pamanīt pieaugošu pieprasījumu pēc atbalsta. Būs laba doma piesaistīt cilvēkus, kas palīdzēs jums samazināt darba slodzi. Tomēr pilnas piekļuves/atļauju piešķiršana ir riskanta un var apdraudēt jūsu tīklu. Tieši šeit noder šis Support Agent papildinājums.

Šis papildinājums ļauj izveidot jaunu WordPress lietotāja tipu, ko sauc par Support Agent, un sniedz jums pilnīgu kontroli pār to, kādas atļaujas šim lietotājam ir. Atļaujas piešķiršana ir tikpat vienkārša kā pārslēgšanas pogas ieslēgšana, un tās izslēgšana, ja nolemjat to noņemt.

## Funkcijas {#features}

- Detalizēts WordPress un Ultimate Multisite atļauju saraksts
- Viegli iespējot un atspējot piekļuvi
- Izveidojiet neierobežotu skaitu aģentu kontu

## Instalēšana {#installation}

1. Augšupielādējiet 'ultimate-multisite-support-agents' direktorijā '/wp-content/plugins/'
2. Aktivizējiet spraudni, izmantojot WordPress izvēlni 'Spraudņi'
3. Izpildiet vedni soli pa solim, lai iestatītu spraudni
