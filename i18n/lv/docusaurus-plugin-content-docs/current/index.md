# Ultimate Multisite Dokumentācija {#ultimate-multisite-documentation}

Laipni lūdzam Ultimate Multisite dokumentācijā. Ultimate Multisite pārveido WordPress Multisite par pilnvērtīgu Tīmekļa vietne kā pakalpojums (WaaS) platformu.

## Dokumentācijas sadaļas {#documentation-sections}

### [Lietotāja rokasgrāmata](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Viss nepieciešamais, lai iestatītu, konfigurētu un pārvaldītu savu WaaS platformu. Aptver instalēšanu, maksājumu vārtejas, domēnu kartēšanu, klientu pārvaldību un daudz ko citu.

### [Izstrādātāju dokumentācija](/developer/hooks) {#developer-documentation}

Tehniskā uzziņa izstrādātājiem, kas veido uz Ultimate Multisite pamata. Ietver hooks uzziņu, REST API dokumentāciju un papildinājumu izstrādes rokasgrāmatas.

### [Papildinājumi](/addons) {#addons}

Dokumentācija maksas papildinājumu plugins, kas paplašina Ultimate Multisite ar papildu funkcijām, piemēram, WooCommerce integrāciju, analītiku, domēnu pārdošanu un daudz ko citu.

- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- Ziņojiet par kļūdām
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- Lejupielādējiet pamata plugin
