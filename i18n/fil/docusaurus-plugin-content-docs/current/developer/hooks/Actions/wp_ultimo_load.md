# Action: wp_ultimo_load

Ito ay tumutugon (triggers) kapag lahat ng mga dependencies ay na-load na.

Pinapayagan nito ang mga developer ng plugin na magdagdag ng bagong functionality. Halimbawa, pagdaragdag ng suporta para sa mga bagong Hosting provider, at iba pa.
