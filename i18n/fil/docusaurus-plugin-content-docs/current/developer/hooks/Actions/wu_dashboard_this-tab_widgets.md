| Name | Type | Description |
|------|------|-------------|

| $screen | `\WP_Screen` | Ang object ng screen. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instance ng admin page ng Ultimate Multisite. |
