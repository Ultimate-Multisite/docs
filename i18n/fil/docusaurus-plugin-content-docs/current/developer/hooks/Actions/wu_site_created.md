# Action: wu_site_created {#action-wusitecreated}

Tumatakbo ito pagkatapos na makagawa ng site sa unang pagkakataon.

Hindi ito tumatakbo kung kinopya lang ito mula sa isang template.

## Mga Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $data | `array` | Ang data ng object na ise-save. |
| $site | `\Site` | Ang instance ng object. |
