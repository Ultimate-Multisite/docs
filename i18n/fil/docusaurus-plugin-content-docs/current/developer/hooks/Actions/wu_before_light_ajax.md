# Action: wu_before_light_ajax {#action-wubeforelightajax}

Sa ilang pagkakataon, kailangan nating magdagdag ng karagdagang functionality para mapahandle ang mga aksyon.

Ang action na ito ay magagamit para mag-load ng mga karagdagang dependencies kapag kinakailangan.

### Simula {#since}

- 2.0.0
