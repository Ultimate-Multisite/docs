# Action: wu_mcp_abilities_registered {#action-wumcpabilitiesregistered}

Ito ay tinatawag pagkatapos na mairehistro ang mga kakayahan (abilities) ng MCP para sa isang entity.

## Mga Parameter {#parameters}

| Name | Type | Deskripsyon |
|------|------|-------------|
| $ability_prefix | `string` | Ang prefix ng kakayahan. |
| $model_name | `string` | Ang pangalan ng model. |
| $this | `object` | Ang instance ng manager. |
