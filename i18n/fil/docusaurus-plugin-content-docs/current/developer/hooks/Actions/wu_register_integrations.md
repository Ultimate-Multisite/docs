# Action: wu_register_integrations

Ito ay tinatawag para payagan ang mga integration na magrehistro ng sarili nila.

Dito ire-rehistro ng Core ang mga integration nito.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | Ang instance ng registry. |
