# Filter: wu_view_override_replaceable_views

Nagpapahintulot ito sa mga developer na magdagdag ng karagdagang folder sa listahan ng mga pwedeng palitan (replaceable list).

Mag-ingat po, dahil ang pagpapahintulot ng karagdagang folder ay maaaring maging sanhi na ang mga lumang kopya (out-of-date copies) ang ma-load sa halip na ang mga bersyon mula sa Ultimate Multisite.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Listahan ng mga pinapayagang folder. |

### Since

- 2.0.0
### Source

Tinutukoy sa [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) sa linya 50
