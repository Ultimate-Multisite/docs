# Filter: wu_password_require_uppercase

Sinusuri nito kung kailangan ba ang malalaking titik (uppercase letters) sa mga password.

## Mga Parameter

| Name | Type | Paglalarawan |
|------|------|-------------|
| $require | `bool` | Kung kailangan ang malalaking titik. Default na `true` kapag ipinatutupad ang mga patakaran. |
| $defender_active | `bool` | Kung aktibo ba ang Defender Pro Strong Password. |

### Mula pa noong

- 2.4.0
### Pinagmulan

Tinukoy sa [`inc/class-scripts.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-scripts.php#L553) sa linya 553
