# Filter: wu_stripe_existing_product_id

Sinusala (filters) nito ang ID ng produktong titingnan. Kung umiiral ito, ang bagong subscription ay gagamit ng produktong ito.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $product_id | `string` | ID ng Stripe product na titingnan. |
| $name | `string` | Pangalan ng produkto. |
