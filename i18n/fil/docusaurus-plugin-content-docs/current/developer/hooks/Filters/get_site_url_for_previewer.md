# Filter: get_site_url_for_previewer

Nagpapahintulot sa mga developer ng plugin na i-filter ang URL na ginagamit sa previewer.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $domain | `string` | Ang default na domain na ginagamit ngayon, kapaki-pakinabang para sa mga pagbabago (manipulations). |
| $domain_options | `array` | Listahan ng lahat ng domain options na ipinasok sa Ultimate Multisite Settings -> Network Settings -> Domain Options |
