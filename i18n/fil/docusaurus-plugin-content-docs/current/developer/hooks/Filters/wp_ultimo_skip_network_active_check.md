# Filter: wp_ultimo_skip_network_active_check

Nagpapahintulot ito sa mga developer na i-bypass (short-circuit) ang pag-check na ito.

Kapaki-pakinabang ito kapag gumagamit ka ng mga setup na base sa composer at iba pang custom na pag-aayos, tulad ng Bedrock, halimbawa, kung saan ang paggamit ng mga plugin bilang mu-plugins ang karaniwan.
