# Filter: wu_\{$id\}_form_fields

Sinusala (Filters) nito ang mga field sa isang form. Ang form ay tinutukoy gamit ang ID na nasa pangalan ng filter.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | Listahan ng mga field na kasama sa form. |

- 2.0.0
