# Filter: wu_post_count

Nagpapahintulot sa mga developer ng plugin na baguhin ang kabuuang bilang ng post.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $count | `int` | Ang kabuuang bilang ng post |
| $post_counts | `object` | Object ng WordPress na ibinabalik ng `wp_count_posts` function |
| $post_type | `string` | Ang slug ng post type |
