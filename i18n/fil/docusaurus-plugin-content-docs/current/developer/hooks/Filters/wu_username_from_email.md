# Filter: wu_username_from_email {#filter-wuusernamefromemail}

Ito ay ginagamit para i-filter ang username ng bagong kustomer.

## Mga Parameter {#parameters}

| Name | Type | Deskripsyon |
|------|------|-------------|
| $username | `string` | Ang username ng kustomer. |
| $email | `string` | Ang email address ng bagong kustomer. |
| $new_user_args | `array` | Array ng mga argumento para sa bagong user, na maaaring kasama ang first at last name. |
| $suffix | `string` | String na idinadagdag sa username para maging unique ito. |
