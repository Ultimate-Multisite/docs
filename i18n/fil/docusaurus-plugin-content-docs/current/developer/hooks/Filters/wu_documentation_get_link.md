# Filter: wu_documentation_get_link

Pinapayagan nito ang mga developer ng plugin na salain (filter) ang mga link.

Hindi sigurado kung paano ito magiging kapaki-pakinabang, pero walang masama sa pagkakaroon nito.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $link | `string` | Ang link na narehistro |
| $slug | `string` | Ang slug na ginamit para makuha ang link |
| $default_link | `string` | Ang default na link na narehistro |

- 1.7.0
