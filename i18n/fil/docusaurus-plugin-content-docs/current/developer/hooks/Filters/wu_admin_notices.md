# Filter: wu_admin_notices

Papayagan nito ang mga developer na i-filter ang mga admin notice na idinadagdag ng Ultimate Multisite.

## Mga Parameter

| Pangalan | Uri | Deskripsyon |
|------|------|-------------|
| $notices | `array` | Listahan ng mga notice para sa partikular na panel na iyon. |
| $all_notices | `array` | Listahan ng mga notice na idinagdag, na pinaghiwalay (segregated) ayon sa panel. |
| $panel | `string` | Panel na kukunin ang mga notice. |
| $filter | `string` | Kung ang mga notice na maaaring i-dismiss ay na-filter na. |
| $dismissed_messages | `array` | Listahan ng mga susi (keys) ng mga na-dismiss na notice. |
