# Filter: wu_sso_enabled {#filter-wussoenabled}

Pag-e-enable o pag-disable ng kakayahan ng single-sign-on (SSO) sa iba't ibang domain.

Maaari mong i-filter ang halagang ito para tuluyang patayin ang single-sign-on, o para kondisyonal itong i-enable.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | Dapat bang naka-enable ang SSO? True para naka-on, false para naka-off. |

### Since {#since}

- 2.0.11
### Source {#source}

Tinutukoy sa [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) sa linya 110


## Returns {#returns}
Kung naka-enable o hindi ang SSO.
