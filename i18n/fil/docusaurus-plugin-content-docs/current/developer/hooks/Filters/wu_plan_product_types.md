# Filter: wu_plan_product_types

Sinusuri (filter) nito ang mga uri ng produkto na dapat ituring bilang mga plans.

Pinapayagan ng filter na ito ang mga addons na magrehistro ng karagdagang uri ng produkto na dapat kilalanin bilang mga plans sa proseso ng pagpapatunay (validation) at pagkakategorya (segregation).

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Array ng mga uri ng produkto na ituturing bilang plans. |

### Since

- 2.3.0
