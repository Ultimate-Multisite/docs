# Filter: wu_contains_element

Pinapayagan nito ang mga developer na baguhin ang resulta ng paunang paghahanap.

Kapaki-pakinabang ito para sa mga third-party builders at katulad nito.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | Kung ang elemento ay nakapaloob sa nilalaman (content). |
| $content | `string` | Ang nilalaman na sinusuri. |
| $element | `self` | Ang kasalukuyang elemento. |
| $post | `null\|\WP_Post` | post na susuriin. |
