# Filter: wu_independent_billing_cycle_product_types

Sinusala (Filter) nito ang mga uri ng produkto na may sariling siklo ng pagbabayad (independent billing cycles).

Ang mga produkto na may independent billing cycles ay hindi kailangang tumugma sa billing period ng plan sa cart. Kapaki-pakinabang ito para sa mga produkto tulad ng domain registrations na palaging nagbabayad taun-taon, kahit na ang plan ay buwanan o taunan.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Array ng mga uri ng produkto na may independent billing cycles. |

### Since {#since}

- 2.5.0
