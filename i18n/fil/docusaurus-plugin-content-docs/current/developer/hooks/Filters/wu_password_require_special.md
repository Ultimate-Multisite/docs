# Filter: wu_password_require_special

Sinusuri nito kung kailangan bang magkaroon ng special characters ang mga password.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Kung kailangan ba ang special characters. Default na *true* ito kapag ipinatutupad ang mga patakaran. |
| $defender_active | `bool` | Kung aktibo ba ang Defender Pro Strong Password. |
