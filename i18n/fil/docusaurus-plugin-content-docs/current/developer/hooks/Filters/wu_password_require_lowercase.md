# Filter: wu_password_require_lowercase

Sinusuri nito kung kailangan ba ang maliliit na titik (lowercase) sa mga password.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Kung kailangan ba ang maliliit na titik. Default na *true* kapag ipinatutupad ang mga patakaran. |
| $defender_active | `bool` | Kung aktibo ba ang Defender Pro Strong Password. |

- 2.4.0
