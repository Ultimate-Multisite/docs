# Action: wp_ultimo_affiliatewp_load

Tumatakbo ito kapag kumpleto nang ma-load ang lahat ng dependencies.

Nagbibigay-daan ito sa mga developer ng plugin na magdagdag ng bagong functionality. Halimbawa, pagdaragdag ng suporta para sa mga bagong Hosting provider, at iba pa.
