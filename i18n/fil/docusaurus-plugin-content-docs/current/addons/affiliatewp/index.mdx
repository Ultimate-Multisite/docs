# AffiliateWP

Multisite Ultimate: Integrasyon sa AffiliateWP

Gamitin ang makapangyarihang AffiliateWP para palakihin ang base ng kliyente ng iyong Ultimo Network!

## Pag-install

1. I-upload ang 'ultimate-multisite-git ' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng 'Plugins' menu sa WordPress
3. Sundin ang step by step Wizard para i-set up ang plugin

## Usage Guide

Ang [AffiliateWP](https://affiliatewp.com/) ay naging pangunahing solusyon kapag nagtatayo ka ng affiliate program para sa iyong online business. Ang Ultimate Multisite: AffiliateWP ay tulay na magpapahintulot sa iyo na gawin ang parehong bagay para sa iyong premium network.

## Getting Started

Una sa lahat, kailangan mong **i-activate** ang [**AffiliateWP**](https://affiliatewp.com/)! Hindi mo magagamit ang aming add-on kung wala kang **plugin na nakakonekta dito**. Kung hindi mo planong hayaan ang iyong mga kliyente na gamitin ito sa kanilang mga site, dapat i-activate lang ang AffiliateWP sa iyong pangunahing site.

Kailangan mo ring i-activate ang parehong **Ultimate Multisite** at **Ultimate Multisite: AffiliateWP** **sa buong network**.

## Pagdaragdag ng Ultimate Multisite bilang AffiliateWP Integration

Sa iyong main site, pumunta sa Affiliates Settings, at sa Integrations tab, i-check ang Ultimate Multisite. Ito ay magsasabi sa [AffiliateWP](https://affiliatewp.com/) na simulan nang subaybayan ang mga pagbisita at conversion mula sa Ultimate Multisite sign-up flow.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_I-check ang Ultimate Multisite box sa ibaba at i-save_

Iyan lang halos ang kailangan. Ang mga pagbisita na galing sa referral links ay susubaybayan at kapag nagbayad na ang user, gagawin ang conversion at maglalaan ng komisyon sa affiliate na responsable sa conversion.

Tandaan, ang unang bayad na matatanggap mula sa bagong nilikhang user lamang ang susubaybayan. Para ma-enable ang recurring referral payments, kailangan mo ang [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Enabling Recurring Referrals

Pagkatapos i-install ang [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), pumunta sa Affiliates Settings Recurring Referrals tab, at i-enable ang paggamit ng recurring referrals.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_Sobrang dali lang: I-enable ang Recurring Referrals at handa ka na!_

Muli, iyan lang ang kailangan!

Ngayon, sa tuwing may matatanggap na bagong bayad sa isang referred account, ang responsable na affiliate ay awtomatikong makakatanggap ng sarili nitong bahagi!
