# Filter: wu_cap_captcha_config

Ito ay nagfi-filter (nagbabago) sa configuration ng Cap Captcha.

Pinapayagan nito ang mga developer na i-customize ang mga setting ng Cap Captcha.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Ang array ng configuration ng Cap. |
| $level | `string` | Ang antas ng seguridad (fast, medium, max). |

- 1.2.0
