# Filter: wu_captcha_whitelisted

Sinusuri nito kung ang kasalukuyang gumagamit o request ay dapat i-whitelist (pinapayagan).

## Mga Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $whitelisted | `bool` | Kung kailangan bang laktawan ang pag-validate ng captcha. |
