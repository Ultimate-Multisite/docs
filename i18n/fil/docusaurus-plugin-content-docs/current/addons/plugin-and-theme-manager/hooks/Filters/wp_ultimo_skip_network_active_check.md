# Filter: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Nagpapahintulot ito sa mga developer na i-bypass (short-circuit) ang pag-check na ito.

Kapaki-pakinabang ito kapag gumagamit ka ng mga custom setup na base sa Composer at iba pa, tulad ng Bedrock, halimbawa, kung saan ang paggamit ng mga plugin bilang mu-plugins ang karaniwan.

- 2.0.0
