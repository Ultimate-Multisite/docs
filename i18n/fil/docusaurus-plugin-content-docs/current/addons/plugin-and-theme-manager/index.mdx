# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

I-edit o itago ang mga *meta information* (pamagat, may-akda, *thumbnail*, at paglalarawan) ng mga naka-install na plugin at theme. Gumawa ng mga kategorya para sa mga plugin at theme upang payagan ang iyong mga user na i-filter ang mga ito at magdagdag ng magandang *custom Plugins page* sa *user's panel* nila.

Upang gamitin ang plugin, maaari ka lang pumunta sa iyong *Plugins* o *Themes* page sa *network admin* mo at i-click ang "Edit Plugin/Theme info".

## Installation {#installation}

1. I-upload ang 'ultimate-multisite-plugin-and-theme-manager' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng 'Plugins' menu sa iyong WordPress Network Admin

## Usage Guide {#usage-guide}

Ang Ultimate Multisite: Plugin and Theme Manager ay isang makapangyarihang **premium** Ultimate Multisite add-on na nagbibigay sa iyo ng ganap na kontrol sa kung paano makikita at makikipag-ugnayan ang mga user ng iyong network sa mga opsyon ng plugin at theme sa iyong network.

Ang add-on na ito ay may tatlong layunin:

  1. Pinapayagan ka nitong ganap na baguhin ang *metadata* ng parehong mga plugin at theme. Kasama rito ang pangalan, paglalarawan, pangalan ng may-akda, at maging ang mga *thumbnail*!

  2. Pinapayagan ka nitong i-categorize ang mga plugin at theme, upang madali silang ma-sort ng iyong mga user.

  3. Binibigyan ka nito ng opsyon na palitan ang luma at nakakaantok na *plugins page* ng isang bagong *themes-like page* na may pag-aayos ayon sa mga kategorya.

Pagkatapos itong i-download at i-install, makikita ang isang bagong tab — _Add-on Settings_ — sa Ultimate Multisite Settings page.
