# Multi-Tenancy

Ihiwalay ang mga subsites sa iba't ibang server at database para sa mas magandang performance at seguridad.

## Mga Kinakailangan

- WordPress 5.3 o mas mataas
- PHP 7.4 o mas mataas
- Multisite Ultimate plugin (aktibo)

## Pag-install

1. I-upload ang mga addon files sa iyong `/wp-content/plugins/` directory
2. I-activate ang plugin sa pamamagitan ng 'Plugins' menu sa WordPress
3. I-configure ang mga setting ng addon sa network admin

## Pagpapaunlad

### Pag-set up

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Pagsubok (Testing)

Kasama sa addon ang isang kumpletong testing framework:

- **Unit Tests**: Mga pagsubok na gumagamit ng PHPUnit at WordPress test suite
- **Base Test Class**: Nagbibigay ng mga helper method para sa mga karaniwang sitwasyon ng pagsubok
- **Test Coverage**: In-configure para subaybayan ang code coverage
- **Multisite Testing**: Ang mga pagsubok ay tumatakbo sa multisite environment

### Pamantayan ng Code (Code Standards)

- **WordPress Coding Standards**: Ipinapatupad gamit ang PHPCS
- **PHP 7.4+ Compatibility**: Sinusuportahan ang modernong PHP features
- **Static Analysis**: Integrasyon ng PHPStan para sa type checking
- **Code Modernization**: Mga patakaran ng Rector para sa pag-upgrade ng PHP

### Build System

- **Asset Processing**: Minification para sa mga CSS/JS files
- **Translation**: Pagbuo ng POT file
- **Packaging**: Paglikha ng archive para sa pamamahagi
- **Development Mode**: Hindi minified assets para sa debugging

## Pag-configure (Configuration)

Sinusuportahan ng addon ang iba't ibang opsyon sa pag-configure na maa-access sa pamamagitan ng network admin interface.

## Hooks and Filters

### Actions

- `ultimate-multisite-addon-template_init` - Tumutugtog pagkatapos i-initialize ang addon
- `ultimate-multisite-addon-template_loaded` - Tumutugtog pagkatapos ma-load ang lahat ng addon files

### Filters

- `ultimate-multisite-addon-template_settings` - Baguhin ang mga setting ng addon
- `ultimate-multisite-addon-template_enabled` - I-override ang status ng pag-enable/disable ng addon
