# Superdav AI Agent

Ang Superdav AI Agent ay isang advanced na WordPress AI assistant na nagpapalawak sa Gratis AI Agent gamit ang mga kasanayang nakakaangkop sa theme, pinahusay na kakayahan sa paghahanap sa internet, mga feature ng tagabuo ng site, at opsyonal na mga integrasyon sa automation. Nagdaragdag ang bersyon 1.18.0 ng mga tool sa pagbasa ng Google Calendar, pagmamapa ng contact at attendee, mga human approval gate, pag-aalis ng dobleng reminder, mga notification sa TextBee SMS, pinamamahalaang paglalaan ng serbisyong Superdav, at isang hiwalay na ipinamamahaging Advanced na kasamang package para sa mga pinagkakatiwalaang developer workflow.

## Mahahalagang Feature

- **Mga kasanayang nakakaangkop sa theme** — awtomatikong inaangkop ang gabay at mga kakayahan batay sa iyong aktibong WordPress theme at mga naka-install na plugin
- **Maramihang provider ng paghahanap sa internet** — pumili sa pagitan ng Brave Search at Tavily para sa mas mayamang resulta ng paghahanap at advanced na kakayahan sa pananaliksik
- **Mga contact form ng tagabuo ng site** — gumawa ng ganap na gumaganang mga contact form direkta mula sa chat interface gamit ang iyong naka-install na form plugin
- **Pinahusay na tagabuo ng site** — pinahusay na multi-phase na mga plano sa pagbuo ng site na may mas mahusay na pagtuklas ng plugin at pagbawi mula sa error
- **Mga built-in na kasanayan sa theme** — espesyal na gabay para sa Block Themes, Classic Themes, Kadence Blocks, at Kadence Theme
- **Pinahusay na integrasyon sa WooCommerce** — nirefactor upang gumamit ng mga native na WooCommerce API para sa mas mahusay na pagiging maaasahan at compatibility
- **Awtomatikong pagtukoy ng provider** — awtomatikong tinutukoy at nire-refresh ang mga available na plugin at theme kapag na-activate o na-deactivate ang mga ito
- **Pagbuo ng custom na logo** — bumuo at mag-embed ng mga custom na SVG logo na may awtomatikong namespace-safe na sanitisation
- **Pag-validate ng contrast ng palette** — suriin ang mga pares ng kulay para sa pagsunod sa WCAG accessibility bago ilapat sa iyong theme
- **Pagtuklas na pinahusay ng larawan** — mag-upload ng mga larawan sa panahon ng discovery interview ng Theme Builder upang maimpluwensiyahan ang direksiyon ng disenyo
- **Mga menu para sa hospitality** — bumuo ng mga nakaayos na pahina ng menu ng pagkain at inumin para sa mga restaurant, cafe, at negosyong hospitality
- **Mga preview sa desktop at mobile** — i-preview ang iyong disenyo sa desktop at mobile device habang pumipili ng direksiyon ng disenyo
- **Tier 1 na pag-scrape ng site** — available na ngayon bilang default ang sd-ai-agent/site-scrape para sa pagsusuri ng mga website ng kakumpitensya at inspirasyon
- **Mga tool sa pagbasa ng Google Calendar** — basahin ang mga naka-configure na calendar at event upang makapag-isip ang mga automation tungkol sa konteksto ng paparating na schedule
- **Pagmamapa ng contact at attendee** — itugma ang mga kalahok sa calendar sa mga kilalang user at contact ng site bago bumuo ng follow-up na gawain
- **Mga human approval gate** — i-pause ang sensitibong mga automation para sa review bago magpadala ng mga mensahe, ipagpatuloy ang gawain, o gumawa ng mga pagbabago
- **Mga tala ng reminder at pag-aalis ng doble** — subaybayan ang mga reminder na naipadala na upang hindi abisuhan ng mga paulit-ulit na takbo ng automation ang parehong tatanggap nang dalawang beses
- **Mga notification sa TextBee SMS** — magpadala ng mga naka-configure na text-message notification sa pamamagitan ng TextBee kapag naka-enable ang isang SMS provider
- **Pinamamahalaang paglalaan ng serbisyong Superdav** — ikonekta ang mga sinusuportahang site sa naka-host na mga Superdav endpoint na may awtomatikong setup sa halip na manu-manong configuration ng endpoint
- **Advanced na kasamang package** — hiwalay na ipinamamahaging mga tool para sa pinagkakatiwalaang developer para sa filesystem, database, WP-CLI, REST dispatcher, tagabuo ng plugin, git snapshot, pamamahala ng user, at mga benchmark workflow

## Mga Kinakailangan

- WordPress 6.0 o mas mataas
- PHP 8.1 o mas mataas
- WordPress Multisite (network-activated) o single-site na installation
- Isang aktibong plugin ng AI provider connector (Anthropic, OpenAI, o compatible)
- Gratis AI Agent v1.9.0 o mas mataas (pinapalawak ng Superdav AI Agent ang Gratis AI Agent)

## Installation

1. I-upload ang mga file ng plugin sa `/wp-content/plugins/superdav-ai-agent`
2. I-activate ang plugin sa pamamagitan ng screen na **Plugins** sa WordPress
3. Tiyaking naka-activate din ang Gratis AI Agent
4. Pumunta sa **Gratis AI Agent → Settings** at i-configure ang iyong provider ng paghahanap sa internet, koneksiyon sa pinamamahalaang serbisyong Superdav, Google Calendar, o mga credential ng TextBee ayon sa kailangan

Naglalaman ang package ng WordPress.org ng mga pangunahing feature ng Superdav AI Agent. Ang Advanced na kasamang package ay binubuo at ipinamamahagi nang hiwalay para sa mga environment kung saan ang mga pinagkakatiwalaang developer tool ay hayagang naka-install at awtorisado.

## Mga Provider ng Paghahanap sa Internet

Nagdaragdag ang Superdav AI Agent v1.10.0 ng **Tavily** bilang alternatibo sa Brave Search para sa mga query sa paghahanap sa internet. Parehong nag-aalok ang dalawang provider ng real-time na mga resulta ng paghahanap at maaaring i-configure sa **Gratis AI Agent → Settings → Advanced**.

| Tagapagbigay | Mga Tampok | Pagsasaayos |
|---|---|---|
| **Brave Search** | Mabilis, nakatuon sa pribasidad na mga resulta ng paghahanap | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Mas mayamang mga resulta ng paghahanap, mas advanced na kakayahan sa pananaliksik | [Tavily API](https://tavily.com/) |

Tingnan ang [Paghahanap sa Internet](../../user-guide/configuration/internet-search) para sa mga tagubilin sa pagsasaayos.

## Mga Workflow ng Kalendaryo, Pag-apruba, at Abiso

Pinalalawak ng Superdav AI Agent v1.18.0 ang mga workflow ng awtomasyon lampas sa mga gawain sa nilalaman at paggawa ng site:

- **Mga tool ng Google Calendar** ay nagpapahintulot sa agent na basahin ang mga naka-configure na kalendaryo at mga detalye ng kaganapan kapag available ang mga credential ng kalendaryo.
- **Mga helper para sa contact at attendee** ay nagma-map ng mga kalahok sa kaganapan sa mga user ng WordPress o kilalang contact bago maghanda ng mga paalala o follow-up na gawain.
- **Mga gate ng pag-apruba ng tao** ay nagpapahintulot sa isang awtomasyon na huminto muna hanggang masuri ng isang awtorisadong user ang iminungkahing aksyon.
- **Mga talaan ng paalala** ay pumipigil sa dobleng mga paalala sa kalendaryo o abiso kapag muling sinubukan o paulit-ulit na naka-iskedyul ang isang awtomasyon.
- **Suporta sa TextBee SMS** ay nagpapagana ng mga abiso sa text message kapag naka-configure ang TextBee provider.

Gamitin ang mga tool na ito para sa tulong na may kamalayan sa iskedyul, follow-up sa attendee, at mga workflow ng paalala na nangangailangan pa rin ng pagsusuri ng tao bago ipadala ang mga sensitibong mensahe.

## Pagsasaayos

Ang lahat ng setting ay nasa ilalim ng **Gratis AI Agent → Settings** sa WordPress admin. Nagdaragdag ang Superdav AI Agent ng mga sumusunod na opsyon sa pagsasaayos:

| Setting | Paglalarawan |
|---|---|
| **Internet Search Provider** | Pumili sa pagitan ng Brave Search at Tavily (opsyonal) |
| **Tavily API Key** | API key para sa paghahanap sa internet ng Tavily (kung napili ang Tavily) |
| **Managed Superdav Service** | Ikonekta ang mga suportadong site sa hosted na Superdav service na may awtomatikong paglalaan ng mga dulong-punto |
| **Google Calendar Credentials** | I-configure ang access sa kalendaryo upang mabasa ng agent ang mga kalendaryo at kaganapan |
| **TextBee SMS Provider** | I-configure ang mga credential ng TextBee para sa mga naaprubahang workflow ng abiso sa SMS |

## Advanced Companion Package

Nagdaragdag ang Superdav AI Agent v1.18.0 ng hiwalay na ipinapamahaging Advanced companion package para sa mga pinagkakatiwalaang workflow ng developer. Hindi kasama ang mga kakayahang ito sa WordPress.org core package. I-install lamang ang companion package sa mga site kung saan tahasang pinagkakatiwalaan ng mga administrator ang agent na mag-access ng developer tooling.

Kasama sa Advanced tools ang mga operasyon sa filesystem, access sa database, WP-CLI, mga workflow ng REST dispatcher, mga utility ng plugin builder, mga git snapshot, mga helper sa pamamahala ng user, at benchmark tooling. Panatilihing limitado ang mga tool na ito sa mga pinagkakatiwalaang administrator at suriin ang pagsasaayos ng approval-gate bago paganahin ang mga ito sa mga production site.
