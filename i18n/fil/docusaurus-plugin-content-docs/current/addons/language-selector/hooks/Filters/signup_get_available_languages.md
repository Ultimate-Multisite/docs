# Filter: signup_get_available_languages

Sinusala (filters) nito ang listahan ng mga magagamit na wika para sa pagpaparehistro ng site sa front-end.

Kung magpasa ka ng walang laman na array sa hook na ito, hindi na lalabas ang setting sa signup form, at gagamitin ang default na wika kapag gumagawa ng site. Aalisin din nito ang mga wikang hindi pa naka-install.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $available_languages | `array` | Mga magagamit na wika. |
