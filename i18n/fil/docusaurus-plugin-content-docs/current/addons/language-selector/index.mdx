# Language Selector

Multisite Ultimate: Language Selector sa Sign-up

Magdagdag ng language selector field sa sign-up flow.

## Pag-install

1. I-upload ang 'ultimate-multisite-language-selector' sa direktoryo na '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng 'Plugins' menu sa iyong WordPress Network Admin

## Gabay sa Paggamit

Sa libreng add-on na ito, mapapayagan mo ang iyong mga user na pumili kung anong wika ang gusto nilang makita sa sign-up flow at sa login page.

Pagkatapos mong [i-activate ang iyong lisensya](../../user-guide/getting-started/installing-ultimate-multisite.md) makakapag-install ka na ng Ultimate Multisite add-ons na kasama sa iyong lisensya.

Pumunta sa Ultimate Multisite settings page at pagkatapos, sa menu sa kaliwa, i-click ang label na _Add-ons_.

![Add-ons page](/img/admin/addons-page.png)

_Pumunta sa Add-ons page para magsimula sa pag-install ng Language Selector_

Doon, makikita mo ang lahat ng add-ons na inaalok namin.

Ang mga free add-ons ay available sa lahat ng aming customer, anuman ang subscription plan na kanilang sinubukan.

I-click ang _Install it Now!_ button sa ilalim ng thumbnail ng add-on. Pagkatapos ay i-activate ito sa Plugins page.

![Add-ons page with install button](/img/admin/addons-page.png)

_I-click ang button para i-install ang add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_I-activate ito sa iyong Plugins page_

Ngayon, pumunta sa Ultimate Multisite's Settings menu at i-enter ang _Network Settings_ tab. Doon, sa ilalim ng page, makakakita ka ng bagong section na tinatawag na _Languages_.

![Settings page](/img/config/settings-general.png)

_May bagong section na lumabas!_

Maaari mo itong iwanang blangko para ipakita ang lahat ng wika bilang pagpipilian, o maaari kang pumili kung alin ang gusto mong gawing available. Ang mga wika ay ang mga naka-install sa iyong WordPress setup.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Listahan ng mga wika na available sa WP install na ito_

Pagkatapos mong i-save ang mga pagbabago, isang selection menu na may mga wikang pinili mo ang ipapakita sa Login page pati na rin sa Sign-up Flow.

![Login and Registration settings](/img/config/settings-login-registration.png)

_Language Selector sa Login Page_

![Registration page](/img/config/registration-page.png)

_Language Selector sa Sign-up Flow_

## Pag-set ng Wika Gamit ang URL

Maaari mo ring baguhin ang wika ng isang page sa pamamagitan ng pag-bypass sa locale parameter sa URL. Kung gusto mong mag-redirect sa isang register page sa Italian, halimbawa, magdagdag ng ?locale=it_IT sa URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Kailangang naka-install ang wika sa iyong WordPress network para gumana ito.
