# Material WP

Matagal nang pareho ang admin interface ng WordPress. Pero nagtaka ka na ba kung ano magiging itsura ng admin interface kung ito ay Material Design version?

Ang Material WP ay isang admin theme na ginawa para mabigyan ka ng itsura at pakiramdam na iyon, na tumutulong sa iyo na magbigay sa iyong mga kliyente ng mas customized na Dashboard.

## Mga Tampok

- Suporta sa Custom CSS
- Madaling i-style na mga setting
- Pag-aayos (reordering) ng Menu
- Madaling i-import/export

## Documentation

**Pag-customize ng iyong Admin Panel gamit ang Material WP**

## Installation

1. I-upload ang 'material-wp' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng 'Plugins' menu sa WordPress

## Usage Guide
