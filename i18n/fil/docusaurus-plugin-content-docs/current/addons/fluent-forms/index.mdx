# Fluent Forms {#fluent-forms}

Ginagawang posible ng add-on na ito na maglagay ng limitasyon sa Fluent Forms sa Multisite Ultimate, para sa mga produkto, membership, o site.
Gumagamit ang Fluent Forms ng custom tables sa halip na custom post types, kaya hindi gumagana ang normal na mga pagpipilian sa paglimita (limiting options) kapag ginamit ito kasama ang Fluent Forms.

## Mga Tampok {#features}

- Magdagdag ng limitasyon para sa Fluent Forms o Conversational Fluent Forms
- Makakatanggap ang mga customer ng isang magiliw na mensahe na nagpapaalala sa kanila na i-upgrade ang kanilang plan kapag naabot na nila ang limitasyon
