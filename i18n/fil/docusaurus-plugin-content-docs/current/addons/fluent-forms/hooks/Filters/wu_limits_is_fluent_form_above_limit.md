# Filter: wu_limits_is_fluent_form_above_limit {#filter-wulimitsisfluentformabovelimit}

Sinusuri kung ang isang ibinigay na uri ng form ay pinapayagan sa planong ito. Pinapayagan nito ang mga developer ng plugin na i-filter ang return value.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $value | `bool` | Kung ang uri ng form ay lumampas sa limitasyon o hindi |
| $form_type | `string` | Ang uri ng form na sinusuri |
| $form_count | `int` | Kasalukuyang bilang ng form |
| $quota | `int` | Ang pinapayagang quota |
