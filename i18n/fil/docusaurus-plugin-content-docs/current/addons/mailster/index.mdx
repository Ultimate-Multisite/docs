# Mailster {#mailster}

Gawing makapangyarihang makina sa pagbuo ng email list ang iyong multisite network gamit ang Ultimate Multisite Mailster Integration addon. Awtomatikong idadagdag ang mga bagong customer sa iyong Mailster email lists kapag sila ay nag-sign up o bumili, na lumilikha ng maayos na koneksyon sa pagitan ng iyong membership platform at email marketing strategy.

## Key Features {#key-features}

**Automatic Subscriber Sync**
* Awtomatikong idadagdag ang mga customer sa Mailster lists kapag sila ay nag-sign up
* Pumili kung gusto mong mag-sync agad pagka-sign up o maghintay ng kumpirmasyon ng bayad
* Panatilihing sariwa at updated ang iyong email lists nang walang manual na pag-interbensyon

**Flexible List Management**
* Magtakda ng global default lists para sa lahat ng bagong customer
* Maaari mong palitan ang defaults gamit ang product-specific lists para sa targeted campaigns
* Madaling pamahalaan ang list assignment gamit ang pamilyar na interface ng Mailster

**GDPR-Compliant Opt-In Options**
* Pumili sa pagitan ng automatic subscription o checkbox-based consent
* I-enable ang double opt-in para sa kinumpirmang subscriptions
* Ipakita/itago ang opt-in checkbox batay sa iyong compliance requirements

**Smart Data Mapping**
* Awtomatikong i-sync ang pangalan ng customer at contact information
* I-map ang billing addresses sa subscriber fields
* Subaybayan ang signup dates para sa targeted campaigns

**Product-Level Control**
* Mag-assign ng iba't ibang lists sa iba't ibang produkto
* Gumawa ng targeted campaigns batay sa kung ano ang binibili ng mga customer
* I-disable ang Mailster para sa mga tiyak na produkto kapag kinakailangan

**Professional Experience**
* Native Mailster UI integration para sa pagpili ng listahan
* Detalyadong logging para sa troubleshooting at verification
* Maayos na pag-update ng mga existing subscribers

## Perfect For {#perfect-for}

* Mga may-ari ng membership site na nagtatayo ng email list
* Mga SaaS platform na gustong alagaan ang mga bagong customer
* Mga gumagawa ng kurso na nagse-segment ng estudyante batay sa pag-enroll
* Sinumang gumagamit ng Ultimate Multisite kasama ang Mailster

## How It Works {#how-it-works}

1. I-install at i-activate ang addon sa iyong Ultimate Multisite network
2. I-configure ang global default lists sa Ultimate Multisite settings
3. Opsyonal na palitan gamit ang product-specific lists
4. Pumili ng opt-in mode at subscription timing
5. Panoorin ang paglaki ng iyong Mailster lists nang awtomatiko!

## Installation {#installation}

1. I-upload ang plugin files sa iyong Ultimate Multisite addons directory
2. Siguraduhin na naka-install at active ang Mailster sa iyong main site
3. I-activate ang addon sa pamamagitan ng Ultimate Multisite addons page
4. Pumunta sa Settings > Mailster Integration para i-configure
5. Piliin ang iyong default lists at i-configure ang opt-in preferences

## Frequently Asked Questions {#frequently-asked-questions}

## Gumagana ba ang addon na ito nang walang Ultimate Multisite? {#does-this-addon-work-without-ultimate-multisite}

Hindi, kailangan ng addon na ito ang Ultimate Multisite para gumana. Ito ay sadyang dinisenyo upang i-integrate ang Mailster sa customer at product system ng Ultimate Multisite.

## Kailangan ko ba na naka-install ang Mailster sa lahat ng site? {#do-i-need-mailster-installed-on-all-sites}

Hindi, kailangan lang i-install at i-activate ang Mailster sa iyong main network site. Awtomatikong kakonekta ang addon dito mula sa lahat ng subsites.

## Maaari ba akong mag-assign ng iba't ibang listahan sa iba't ibang produkto? {#can-i-assign-different-lists-to-different-products}

Oo! Maaari kang magtakda ng global default lists, pagkatapos ay palitan ang mga ito sa bawat produkto. Pinapayagan ka nitong gumawa ng highly targeted email campaigns batay sa kung ano ang binibili ng mga customer.

## Ano ang mangyayari kung ang isang customer ay umiiral na sa Mailster? {#what-happens-if-a-customer-already-exists-in-mailster}

By default, iuuupdate ng addon ang mga existing subscribers at idadagdag sila sa mga bagong listahan. Maaari mo itong i-disable sa settings kung mas gusto mong laktawan ang mga existing subscribers.

## Sumusunod ba ito sa GDPR? {#is-this-gdpr-compliant}

Oo! Kasama sa addon ang checkbox-based opt-in mode kung saan kailangang magbigay ng malinaw na pahintulot ang mga customer na tumanggap ng email. Ito ay tumutugon sa mga kinakailangan ng GDPR kapag sinamahan ng tamang privacy policies.

## Maaari ba akong gumamit ng double opt-in? {#can-i-use-double-opt-in}

Absolutely! I-enable ang double opt-in setting at magpapadala ang Mailster ng confirmation emails bago i-activate ang mga subscriptions.
