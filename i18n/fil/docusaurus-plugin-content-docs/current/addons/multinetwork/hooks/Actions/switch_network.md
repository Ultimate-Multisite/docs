# Action: switch_network

Tumutakbo ito kapag nagpapalit ng kasalukuyang network context.

## Mga Parameter

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ID ng network na papuntahan. |
| $old_network_id | `int` | ID ng dating kasalukuyang network. |

### Mula pa noong

- 1.3.0
