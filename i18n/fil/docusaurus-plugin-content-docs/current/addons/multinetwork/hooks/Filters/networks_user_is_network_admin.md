# Filter: networks_user_is_network_admin {#filter-networksuserisnetworkadmin}

Sinusala (filters) nito ang mga network na pinamamahalaan (administrator) ng isang user.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $ | `array\|bool` | Listahan ng mga network ID, o `false` kung walang network para sa user. |
| $ | `int` | User ID na gusto mong ibalik ang mga network. |
