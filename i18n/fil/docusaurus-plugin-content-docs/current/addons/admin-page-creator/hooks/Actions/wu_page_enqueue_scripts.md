# Action: wu_page_enqueue_scripts

Magdadagdag tayo ng custom action dito para magkaroon tayo ng lugar kung saan maaari nating i-enqueue ang mga scripts at styles na kailangan, at para masigurado na ito ay maglo-load lang sa tamang mga pahina.

## Mga Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $ | `\WU_Page` | Ang kasalukuyang admin page |

### Simula {#since}

- 1.3.0
