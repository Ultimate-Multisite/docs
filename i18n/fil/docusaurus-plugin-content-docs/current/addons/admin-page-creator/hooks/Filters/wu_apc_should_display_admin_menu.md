## Parameters

| Name | Type | Description |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Kung kailangan nating ipakita ang mga pahina. |
| $network_active | `bool` | Titingnan kung ang plugin ay aktibo sa network admin. |
