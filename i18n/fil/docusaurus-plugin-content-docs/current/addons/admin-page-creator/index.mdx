# Admin Page Creator

## Content
Ang Admin Page Creator ay magpapahintulot sa iyo na madaling magdagdag ng mga bagong admin page at sub-pages sa loob ng WordPress admin dashboard. Puwede itong gamitin para gumawa ng welcome page o tutorial page na nagsisilbing landing page para sa mga bagong user. Maaari kang gumamit ng page builder tulad ng Beaver Builder, Elementor, Brizy, at Oxygen para mag-design ng mga page nang walang hirap.

Nag-aalok din ito ng suporta para sa custom HTML, CSS, at JavaScript, pati na rin ng access control batay sa plan at user role!

## Features
* Suporta sa Custom HTML at CSS
* Suporta sa JavaScript
* PHP Content Mode
* Pahintulot batay sa User role
* Suporta sa page builders

## Documentation
[Creating Custom Admin Pages and Sub-pages using Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Installation

1. I-upload ang **ultimate-multisite-admin-page-creator** sa direktoryong **/wp-content/plugins/**
2. I-activate ang plugin sa pamamagitan ng **Plugins** menu sa iyong WordPress Network Admin

## Usage Guide

Habang napakadali para sa WordPress na magdagdag ng content sa front end ng mga site, ang paggawa ng mga page at sub-pages sa admin side ay nangangailangan ng kaunting basic coding skills. Ang Ultimate Multisite: Admin Page Creator ay naglalayong dalhin ang parehong kadalian ng paggamit ng front end sa back end ng mga site sa iyong network.

## Why use Admin Page Creator?

Baka gusto mong mag-alok ng custom page sa admin panel ng iyong mga kliyente na may mga tutorial, o baka gusto mong magdagdag ng custom page na may FAQs at contact widget. Walang katapusan ang mga posibilidad gamit ang Admin Page Creator, at iyon ang ating pag-eexplore sa tutorial na ito.

## Creating your first Custom Admin Page

Pagkatapos i-install at i-activate ang Ultimate Multisite: Admin Page Creator, pumunta sa network admin menu at i-click ang Admin Pages.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Custom Admin Pages list sa Network Admin_

Makakakita ka ng listahan ng lahat ng admin pages na ginawa mo hanggang ngayon (dahil ito ang iyong unang pagbisita, walang laman ang listahan). I-click lang ang **Add new Admin Page**. Ire-redirect ka sa Admin Page editor.

## The elements of a Custom Admin Page

Sa Admin Page editor, makakakita ka ng maraming iba't ibang opsyon para i-customize ang iyong page.

### Page Title

Ito ang pamagat na ipapakita sa itaas ng page, hindi ang label sa admin menu.

Nagbibigay sa iyo ang Ultimate Multisite: Admin Page Creator ng dalawang opsyon pagdating sa content ng page: maaari kang gumamit ng default WordPress WYSIWYG editor o lumipat sa isang HTML editor.

Kung hindi ka pamilyar sa HTML, manatili ka sa default WordPress editor. Magagawa mo pa ring i-edit ang content ng page na parang nag-eedit ka ng normal na Post o Page.

## Menu Options

Sa gilid, sa General Options meta-box, makakapag-configure ka ng menu type, label, at marami pang iba.

### Menu Types

Ang iyong bagong custom admin page ay maaaring magkaroon ng dalawang magkaibang menu type: top-level menu at submenu.

Ang isang top-level menu page, tulad ng pangalan, ay magdadagdag ng iyong bagong ginawang page sa admin menu kasama ng default WordPress pages.

Kung pipiliin mo ang top-level option, makakapili ka rin ng menu order at menu icon.

Kung gusto mong gawing sub-menu page ang page na ito, makakapili ka ng parent page na pagkakakabit ng page na ito. Maaari mong idagdag ang iyong custom page sa default WordPress pages o sa iyong top-level custom admin pages.

<!-- Screenshot unavailable: Admin Page Creator menu type options with label, order, and icon settings -->

_Ganap na i-customize ang menu item para sa Custom Page na ito_

<!-- Screenshot unavailable: Admin Page Creator sub-menu parent page selection -->

_Maaari mo ring idagdag ang page na ito bilang sub-menu ng ibang menu pages, kasama na ang sarili mong top-level ones._

## Advanced Options

Dapat ay makakapagdagdag ka ng lahat ng uri ng content sa iyong custom admin pages. Kasama rito ang CSS rules at files, JavaScript, at external libraries. Magagawa mo iyon gamit ang Advanced Options Tab.

### Custom CSS

Sa CSS tab, makakapagdagdag ka ng sarili mong custom CSS code, pati na rin ng pag-import ng CSS files mula sa mga remote sites (tulad ng CDNs, Google Fonts, atbp.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Sinusuportahan din ang Custom CSS. Maaari ka ring mag-include ng external style files!_

### Custom JavaScript

Pareho rin ang totoo para sa JavaScript. Maaari kang magdagdag ng external libraries para gamitin sa iyong custom JavaScript code.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Magdagdag ng sarili mong JavaScript code!_

## Permissions

Sa Advanced Options din, makakapili ka kung aling sub-sites ang magkakaroon ng access sa page na ito. Ito ay kapaki-pakinabang kung nagpaplano kang magbigay ng iba't ibang exclusive content sa iba't ibang plan tiers, halimbawa.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Mayroon kang ganap na kontrol kung saan ipapakita ang page na ito_

**Mahalaga** : Kailangang matugunan ang parehong kondisyon para maipakita ang page. Ibig sabihin, kung ang user ay kliyente ng isang plan, ngunit wala siyang isa sa mga napiling roles (o vice-versa) hindi lalabas ang page sa kanilang panel.

## Active

Kung gusto mong i-disable ang isang custom admin page nang hindi ito binubura o nilalaro ang permissions, i-uncheck lang ang Active option.

Ang pag-deactivate ng isang top-level page ay awtomatikong magde-deactivate sa lahat ng sub-pages na nakakabit sa parent page na iyon.
