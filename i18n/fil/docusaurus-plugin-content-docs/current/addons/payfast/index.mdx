# PayFast

Multisite Ultimate: Payfast Gateway

Nagdaragdag ito ng Payfast – isang mabilis at ligtas na online payment gateway sa South Africa – bilang isang opsyon sa pagbabayad sa Multisite Ultimate.

## Installation {#installation}

1. I-upload ang 'ultimate-multisite-payfast' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng 'Plugins' menu sa WordPress
3. Sundin ang step-by-step Wizard para i-set up ang plugin

## Usage Guide {#usage-guide}

Ang PayFast ay isang nangungunang online payment processor para sa mga negosyo sa South Africa. Nakatuon ito sa pagtulong sa mga online merchant na lumago sa pamamagitan ng paggawa ng mga transaksyon na maayos at ligtas. Ayon sa hiling ng mga user ng Ultimate Multisite, nagpasya kaming gumawa ng isang add-on na magpapahintulot ng integrasyon sa PayFast. Ipinapakita ng artikulong ito kung paano i-set up ang PayFast bilang iyong payment gateway para makakatanggap ka ng bayad sa iyong Ultimate Multisite network.

Ang unang hakbang ay i-install ang add-on.

# Paano i-install ang Payfast add-on:

Pumunta sa add-ons page at piliin ang Ultimate Multisite: Payfast Gateway add-on.

![Add-ons page](/img/admin/addons-page.png)

Lalabas ang add-on window. I-click ang Install Now.

![Add-ons page with install option](/img/admin/addons-page.png)

Pagkatapos ma-install, ire-redirect ka sa plugins page. Dito, i-click mo lang ang Network Activate at ma-a-activate ang _Payfast_ add-on sa iyong network.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Pag-set up ng PayFast

Pagkatapos i-install ang add-on, maaari ka nang pumunta sa Ultimate Multisite settings at simulan ang pag-configure ng PayFast.

Sa ilalim ng **Ultimate Multisite settings** page, makikita mo ang tab na "Payments." Sa tab na ito, dapat mong makita ang "**Payfast** " kasama ang isang toggle button para i-enable o i-disable ang payment gateway na ito.

![Payment gateway settings](/img/config/settings-payment-gateways.png)I-click mo lang ang toggle button para i-enable ito. Pagkatapos, lalawak ang box para ipakita ang iba't ibang fields na kailangan mong punan tulad ng **Merchant ID, Merchant Key, at PassPhrase.** Makukuha mo ang impormasyong ito mula sa iyong **Payfast** account

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Ang susunod na hakbang ay mag-login sa iyong PayFast account para makuha ang impormasyong kailangan mo.

Pag-login, agad mong makikita ang mga kredensyal na kailangan mo mula sa **Payfast** account dashboard.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Maaari mo lang i-copy at i-paste ang kredensyal at punan ang mga detalye sa iyong Ultimate Multisite settings. Pagkatapos, i-click ang **"Save settings** " button para ma-save ang mga pagbabago.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Ayan na, kumpleto na ang iyong PayFast payment integration!
