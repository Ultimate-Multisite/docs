# Content Sync

Multisite Ultimate: Content Sync

Ang addon na ito para sa Multisite Ultimate ay nagpapahintulot sa mga network administrator na i-synchronize ang nilalaman mula sa anumang site patungo sa ibang mga site sa network. Sinusuportahan nito ang lahat ng post types, kasama ang posts, pages, custom post types, at Elementor templates. Kasama sa addon ay isang multi-step wizard interface para sa pagpili ng nilalaman at mga target na site, na may batch processing at real-time na feedback sa progreso.

## Features

* Content Selection: Visual card grid na may thumbnails at pag-filter ng post type
* All Post Types: Sinynchronous ang posts, pages, products, Elementor templates, at anumang custom post type
* Site Selection: Pumili kung saang mga site isici-sync ang nilalaman, kasama ang pag-exclude ng source site at pre-selection ng child site
* Batch Processing: Sinynchronous sa 10 sites nang sabay-sabay kasama ang mga update sa progreso
* Media Sync: Awtomatikong kinokopya at inaayos ang mga media file (larawan, video) mula sa post content patungo sa target sites
* Taxonomy Sync: Pinapanatili ang mga kategorya, tag, at custom taxonomies
* Elementor Support: Buong suporta para sa Elementor templates kasama ang CSS regeneration at template ID remapping
* Conflict Resolution: Pagpipilian na palitan ang umiiral na nilalaman o laktawan ito
* Cache Management: Awtomatikong naglilinis ng Elementor cache sa target sites pagkatapos ng sync

## Requirements

* WordPress 5.3 or higher
* PHP 7.4 or higher
* Multisite Ultimate plugin (active)
* Elementor plugin (optional, required for Elementor template sync)

## Installation

1. I-upload ang 'ultimate-multisite-content-sync' sa direktoryo na '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng 'Plugins' menu sa iyong WordPress Network Admin
3. I-configure ang mga setting sa Ultimate Multisite → Settings → Content Sync
4. Pumunta sa Ultimate Multisite → Content Sync para simulan ang pag-sync
