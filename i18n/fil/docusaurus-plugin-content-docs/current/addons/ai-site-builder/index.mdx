# AI Site Builder {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Mga Kinakailangan (Requirements) {#requirements}

- WordPress 5.3 o mas mataas
- PHP 7.4 o mas mataas
- Multisite Ultimate plugin (aktibo)

## Pag-install (Installation) {#installation}

1. I-upload ang mga addon files sa iyong `/wp-content/plugins/` directory
2. I-activate ang plugin sa pamamagitan ng 'Plugins' menu sa WordPress
3. I-configure ang mga setting ng addon sa network admin

## Pagpapaunlad (Development) {#development}

### Pag-set Up (Setup) {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Pag-test (Testing) {#testing}

Kasama sa addon ang isang kumpletong testing framework:

- **Unit Tests**: Mga test na gumagamit ng PHPUnit at WordPress test suite
- **Base Test Class**: Nagbibigay ng mga helper method para sa mga karaniwang sitwasyon ng pag-test
- **Test Coverage**: In-configure para subaybayan ang code coverage
- **Multisite Testing**: Ang mga test ay tumatakbo sa multisite environment

### Code Standards {#code-standards}

- **WordPress Coding Standards**: Ipinapatupad gamit ang PHPCS
- **PHP 7.4+ Compatibility**: Sinusuportahan ang mga modernong feature ng PHP
- **Static Analysis**: Integrasyon ng PHPStan para sa type checking
- **Code Modernization**: Mga patakaran ng Rector para sa pag-upgrade ng PHP

### Build System {#build-system}

- **Asset Processing**: Minification para sa mga CSS/JS files
- **Translation**: Pagbuo ng POT file
- **Packaging**: Paglikha ng archive para sa pamamahagi
- **Development Mode**: Hindi minified assets para sa debugging

## Pag-configure (Configuration) {#configuration}

Sinusuportahan ng addon ang iba't ibang opsyon sa pag-configure na maa-access sa pamamagitan ng network admin interface.

## Hooks at Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - Tumutunog pagkatapos i-initialize ang addon
- `ultimate-multisite-ai-site-builder_loaded` - Tumutunog pagkatapos ma-load ang lahat ng addon files

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Baguhin ang mga setting ng addon
- `ultimate-multisite-ai-site-builder_enabled` - Palitan ang status ng pag-enable/disable ng addon
