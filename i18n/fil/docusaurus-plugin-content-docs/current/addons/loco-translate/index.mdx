# Loco Translate

Ang Ultimate Multisite: Loco Translate Integration addon ay nagdadala ng kakayahan sa pagsasalin na specific sa bawat site sa iyong WordPress multisite network. Kung kailangan mo dati na ang iba't ibang site sa iyong network ay magkaroon ng iba't ibang pagsasalin para sa iisang plugin o theme, ito ang perpektong solusyon.

**Ano ang Ginagawa ng Addon na Ito**

Ang addon na ito ay awtomatikong pinagsasama ang Ultimate Multisite sa sikat na Loco Translate plugin, na nagpapahintulot sa bawat subsite sa iyong network na magpanatili ng sarili nitong custom na pagsasalin. Kahit na nagpapatakbo ka ng isang multi-regional platform, naglilingkod sa iba't ibang komunidad ng wika, o kailangan mo lang ng flexibility sa pagsasalin sa buong network, ginagawa nitong madali ang proseso.

**Perpekto Para sa:**

* Mga negosyo na maraming rehiyon at may iba't ibang pangangailangan sa lokalisasyon
* Mga institusyong pang-edukasyon na naglilingkod sa iba't ibang komunidad
* Mga SaaS platform na nag-aalok ng multilingual support
* Mga web agency na namamahala ng maraming site ng kliyente
* Anumang multisite network na nangangailangan ng flexibility sa pagsasalin

## Pag-install

1. Siguraduhin na naka-install at naka-activate ang Ultimate Multisite
2. I-install at i-activate ang Loco Translate plugin
3. I-upload ang addon na ito sa iyong plugins directory
4. I-activate ang Loco Translate Integration addon sa buong network
5. Magsimulang gumawa ng site-specific translations sa pamamagitan ng Loco Translate sa anumang subsite

Ayan na! Awtomatikong gumagana ang integration nang walang anumang karagdagang configuration.

## Madalas Itanong (FAQ)

## Kailangan ko bang mag-configure pa pagkatapos i-install?

Hindi! Awtomatiko ang integration. Kapag pareho nang active ang addon at ang Loco Translate, agad na makakapagsimula ang mga subsite na gumawa ng sarili nilang custom na pagsasalin.

## Ano ang mangyayari kung hindi gumawa ng custom translations ang isang subsite?

Gagamitin ng site ang mga network-wide translations bilang backup. Walang masisira, at lahat ay magpapatuloy nang maayos.

## Maaari bang ma-access ng mga subsite ang network-wide translations?

Ang mga regular site admin ay makikita at makakapag-edit lamang ng pagsasalin ng sarili nilang site. Ang mga super admin naman ay makaka-access ng parehong network-wide at site-specific translations depende sa site na kanilang tinitingnan.

## Gumagana ba ito sa plugins at themes?

Oo! Gumagana ang site-specific translations para sa mga WordPress plugin at theme.

## Mapapabagal ba nito ang aking multisite network?

Hindi. Gumagamit ang addon ng efficient file loading na tanging sa oras na kailangan lang ang site-specific translations ay titingin. Walang mapapansing epekto sa performance.

## Maaari ko bang kopyahin ang network translations sa isang specific site bilang panimula?

Oo! Habang nag-e-edit ng translations sa Loco Translate, madali kang makakapagsimula gamit ang network translations at maaari mo itong i-customize para sa bawat site.

## Gumagana ba ito sa WooCommerce at iba pang plugins?

Talagang! Anumang plugin o theme na gumagamit ng standard WordPress translation functions ay gagana sa site-specific translations.
