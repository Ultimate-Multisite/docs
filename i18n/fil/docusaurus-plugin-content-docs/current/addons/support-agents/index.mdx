# Support Agents

Hindi madali ang pamamahala ng isang buong network nang mag-isa. Habang lumalaki ang iyong network, makakaranas ka ng lumalaking pangangailangan sa suporta. Magandang ideya na magdagdag ng mga tao na makakatulong sa pagpapagaan ng iyong trabaho. Gayunpaman, ang pagbibigay ng buong access/pahintulot ay mapanganib at maaaring makompromiso ang seguridad ng iyong network. Dito pumapasok ang Support Agent add-on na ito.

Pinapayagan ka ng add-on na ito na gumawa ng bagong uri ng user sa WordPress na tinatawag na Support Agent, at binibigyan ka ng kumpletong kontrol sa mga pahintulot na mayroon ang user na iyon. Ang pagbibigay ng pahintulot ay kasingdali lang ng pag-on ng isang toggle button, at maaari mo itong i-off kung magdesisyon kang tanggalin ito.

## Features {#features}

- Detalyadong listahan ng mga pahintulot sa WordPress at Ultimate Multisite
- Madaling i-enable at i-disable ang access
- Makakagawa ng walang limitasyong bilang ng Agent accounts

## Installation {#installation}

1. I-upload ang 'ultimate-multisite-support-agents' sa direktoryo na '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng 'Plugins' menu sa WordPress
3. Sundin ang step-by-step Wizard para i-set up ang plugin
