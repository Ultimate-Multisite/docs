# Pamamahala ng Shareable Links para sa mga Plan

Binibigyan ka ng Ultimate Multisite ng opsyon na ibahagi ang mga plan link nang direkta sa iyong mga kliyente. Para magawa ito, pumunta sa edit page ng alinman sa iyong mga plan at i-click ang _Click to copy the Shareable Link_ button.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Sa Edit Page ng Plan, makikita mo ang button para gumawa ng shareable links_

Tandaan na gumagamit ang mekanismong ito ng default na billing frequency para buuin ang link. Ibig sabihin, kung ang default mong billing ay monthly halimbawa, sa huli ay magkakaroon ka ng link na ganito: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Kung gusto mong ang shareable link ay naka-default sa quarterly, kailangan mong magdagdag ng 3 sa dulo

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Para piliin ang plan na _premium_ sa yearly na iskedyul, magdagdag ka ng 1 pagkatapos ang salita:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Tandaan na hindi ka maaaring magkaroon ng isang link lamang para sa iba't ibang billing frequency. Kailangan mong magbahagi ng isang partikular na link para sa bawat isa sa iyong mga billing frequency.

Kapag naibahagi mo na ang link sa iyong mga customer, ire-redirect sila nito sa unang hakbang ng iyong signup flow.
