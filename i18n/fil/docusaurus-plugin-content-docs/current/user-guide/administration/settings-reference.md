# Sanggunian ng mga Setting

Sinusubaybayan ng pahinang ito ang mga setting na nakakaapekto sa pang-araw-araw na pangangasiwa at mga kamakailang pagbabago sa paggana (behaviour) sa Ultimate Multisite.

## Other Options

Ang lugar na **Other Options** ay makikita sa ilalim ng **Ultimate Multisite > Settings > Login & Registration**.

| Setting | Paglalarawan |
|---|---|
| **Enable Jumper** | Nagpapakita ito ng Jumper quick navigation tool sa admin area. Gamitin ito para direktang makarating sa mga screen ng Ultimate Multisite, mga network object, at mga sinusuportahang destinasyon sa admin. I-disable ito kung ayaw mong makita ang shortcut na ito. |

## Error reporting and telemetry

Inalis na ang dating setting na kailangan i-opt-in para sa pag-uulat ng error mula sa settings page. Ang anonymous telemetry ay naka-disable at walang UI toggle para i-enable ito.

Kung nagpapanatili ka ng internal runbooks o screenshots para sa settings page, alisin ang mga pagtukoy sa lumang error-reporting opt-in field para hindi maghanap ng setting ang mga administrator na wala na.

## Import/Export settings

Ang tab na **Import/Export** settings ay naglalarawan kung anong mga setting ang kinokontrol nito at direktang nagli-link sa **Ultimate Multisite > Site Export** para sa mga archive ng site at network. Gamitin ang settings tab para sa configuration ng import/export, gamitin ang **Tools > Export & Import** para sa workflow ng pag-export/pag-import ng isang site, at gamitin ang Site Export tool kapag kailangan mo ng kumpletong Network Export archive.
