# Pamamahala ng mga Membership

Sa Ultimate Multisite, maaari mong pamahalaan ang mga membership at ma-access ang lahat ng subscription sa iyong network. Tingnan at i-edit ang mga detalye ng subscription tulad ng mga plan at produkto, bilang ng mga site, halaga at dalas ng pagsingil, kasaysayan ng pagbabayad, at mga timestamp.

Para ma-access ang pahinang ito, pumunta sa **Memberships** tab sa Ultimate Multisite menu.

![Listahan ng mga membership](/img/admin/memberships-list.png)

Sa Memberships page, makikita mo ang listahan ng iyong mga customer at ang mga produktong naka-link sa kanilang mga account. Maaari mong i-filter ang mga membership ayon sa status: active, pending, on hold, expired, o cancelled.

I-click ang isang membership para ma-access ang pahina kung saan mo mae-edit ang mga detalye ng membership â€” impormasyon sa pagsingil, mga limit sa site o pagbisita, at iba pa.

Maaari ka ring magdagdag ng bagong membership na naka-link sa mga user na dating nakarehistro sa iyong network. I-click ang **Add Membership** at i-configure ang mga detalye.

![Add Membership modal](/img/admin/membership-add-modal.png)

Pagkatapos, i-click ang **Create Membership**. Lalabas ang bagong membership sa listahan ng mga membership.
