# Pangkalahatang-tanaw sa Dashboard

Ang Ultimate Multisite ay may sariling **Dashboard** sa ilalim ng Ultimate Multisite menu, at nagdadagdag din ng mga widget sa Network Dashboard. Saklaw ng artikulong ito kung ano ang makikita mo sa dalawang lugar na ito.

## Ang Network Dashboard

Naglalagay ang Ultimate Multisite ng mga kapaki-pakinabang na widget sa network-level dashboard na matatagpuan sa **My Sites → Network Admin → Dashboard**.

![Network Dashboard na may Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Narito kung ano ang ibinibigay ng bawat widget:

**First Steps:** Lumalabas ito sa bagong installation at nagpapaalala sa iyo na kumpletuhin ang mga setup task tulad ng pag-configure ng payment method at paggawa ng test account.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** Ipinapakita ang bilang ng mga signup at kita para sa araw. Ang MRR (Monthly Recurring Revenue) ay nagpapakita ng projected total mula sa mga customer na may revenue-generating membership.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Nag-uulat ng mga event sa buong Ultimate Multisite — mga signup, cancellation, paggawa ng site, pagbura, at iba pang aktibidad.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Nagpapakita ng maikling buod ng mga user at site sa network kasama ang mga shortcut para gumawa ng bagong site o user.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Kumukuha at nagpapakita ng pinakabagong balita tungkol sa Ultimate Multisite kabilang ang mga update, mahahalagang bug, at security fix.

![News widget](/img/admin/network-dashboard-news.png)

## Ang Ultimate Multisite Dashboard

Habang ang network dashboard ay nagpapakita ng pangkalahatang impormasyon tungkol sa network, ang Ultimate Multisite dashboard (sa pinakataas na bahagi ng Ultimate Multisite menu) ay nagpapakita ng impormasyon tungkol sa serbisyo.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** Nagpapakita ng buwanang breakdown ng kita sa loob ng 12-buwang panahon. Ang bagong kita ay ipinapakita sa asul habang ang mga cancellation ay nasa pula.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** I-filter ang dashboard data ayon sa panahon para tumutok sa impormasyong kailangan mo. Kinokontrol ng date range kung ano ang ipinapakita sa mga widget sa ibaba.

![Date range at dashboard stats](/img/admin/um-dashboard-date-range.png)

Kabilang sa mga dashboard widget ang:

- **Revenue** — Kabuuang kita para sa panahon kasama ang MRR at breakdown sa bawat produkto
- **New Memberships** — Ilang bagong membership ang nagawa sa napiling panahon
- **Signups by Countries** — Heograpikong breakdown ng mga signup
- **Signups by Forms** — Ilang customer ang nag-sign up sa bawat checkout form kumpara sa mga idinagdag ng admin
- **Most Visited Sites** — Ang mga pinakabinibisitang site sa iyong network (kailangang i-enable ang visit counting sa **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
