# Mga Madalas Itanong

Narito ang ilang **Madalas Itanong** na kadalasang lumalabas sa mga kahilingan ng suporta.

  * **Ano ang Ultimate Multisite?**

Ang Ultimate Multisite ay isang WordPress plugin, ginawa para sa mga WordPress multisite install, na nagpapalit ng iyong WordPress install sa isang premium na network ng mga site – katulad ng [WordPress.com](https://WordPress.com) – na nagpapahintulot sa mga kliyente na gumawa ng mga site sa pamamagitan ng buwanan, quarterly, o taunang bayarin (maaari ka ring gumawa ng mga Libreng plano).

  * **Gagana ba ang Ultimate Multisite nang walang Multisite?**

Hindi. Ang Ultimate Multisite ay nangangailangan ng WordPress multisite, subdirectory, o subdomain.

  * **Ano ang mga kinakailangan para sa Ultimate Multisite?**

Ang Ultimate Multisite ay nangangailangan ng:

**Bersyon ng WordPress** : v5.3+ (Inirerekomenda: Pinakabagong Stable na Bersyon)  
**Bersyon ng PHP** : 7.4.x (Darating pa lamang ang suporta sa > 8.0)  
**Bersyon ng MySQL** : v5+ (Inirerekomenda: Bersyon 5.6, kung sakaling wala kang 8.0 sa iyong hosting provider)

_Kailangang ma-activate din ang Multisite (subdomain o subdirectory)._ [Paano Mag-install ng WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Gagana ba ang Ultimate Multisite sa Multisite na may mga Subdirectory?**

Oo. Sinusuportahan ng Ultimate Multisite ang parehong subdirectory install at subdirectory install.

  * **Gagana ba ang Ultimate Multisite sa Multisite na may mga Subdomain?**

Oo. Sinusuportahan ng Ultimate Multisite ang parehong subdirectory install at subdomain install.

  * **Maaari ko bang gamitin ang Ultimate Multisite para limitahan kung aling mga Plugin at Theme ang magagamit ng mga user?**

Oo. Sa Ultimate Multisite maaari mong piliin kung aling mga plugin at theme ang available sa bawat plano sa iyong network.

  * **Maaari ko bang gamitin ang Ultimate Multisite para limitahan ang bilang ng mga post na maaaring gawin ng isang user sa kanyang site?**

Oo. Sa Ultimate Multisite maaari kang magtakda ng mga quota sa post para sa **bawat available na post type** sa iyong network. Maaari mo ring limitahan ang bilang ng mga user na maaaring imbitahin ng admin, ang **dami ng disk space,** at marami pang iba.

  * **Sinusuportahan ba ng Ultimate Multisite ang mga Site Template?**

Oo. Sa Ultimate Multisite maaari kang lumikha ng maraming Site Template hangga't gusto mo. Ang mga Site Template ay hindi mga theme, sila ay mga normal na site sa iyong network. Maaari kang mag-activate ng mga plugin, theme, at i-customize ang Site Template na iyon sa anumang paraan na gusto mo. Kung pipili ang isang user ng template na iyon, ang kanyang site ay gagawin bilang kopya ng template na iyon, kasama ang lahat ng mga customization.

Maaari mo ring itakda ang isang partikular na template para sa isang plano. Sa paggawa nito, bawat user na mag-sign up sa planong iyon ay magkakaroon ng site na ginawa bilang duplicate ng Site Template na iyon.

  * **Nag-iintegrate ba ang Ultimate Multisite sa WooCommerce?**

Oo!

Kung gusto mong magkaroon ng WooCommerce ang mga user ng iyong network para sa kanila, na nagpapahintulot sa kanila na mag-set up ng mga tindahan, posible iyon gamit ang Ultimate Multisite. Maaari mo ring limitahan ang bilang ng mga produkto na maaaring gawin ng iyong mga kliyente sa bawat tier ng plano.

Sa mga tuntunin ng mga pagbabayad ng subscription para sa Ultimate Multisite, maaari mong gamitin ang [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) para payagan ang iyong mga kliyente ng Ultimate Multisite na magbayad gamit ang mga WooCommerce gateway sa iyong pangunahing site. Sa ganitong paraan, maaari mong samantalahin ang malaking library ng mga payment gateway na ginawa para sa WooCommerce at gamitin ang mga ito para singilin ang iyong mga kliyente ng Ultimate Multisite.

  * **May paraan ba para i-migrate ang mga subscription mula sa WPMUDEV Pro Sites?**

Oo, nag-aalok na ngayon ang Ultimate Multisite ng [isang kapaki-pakinabang na tool para tulungan kang mag-migrate](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) ng iyong Pro Sites network.

  * **Bakit walang laman ang tab na "Allowed Plugins" sa aking Plan Edit Page?**

Ang Ultimate Multisite ay nagpapahintulot lamang sa iyo na payagan o tanggihan ang access sa mga plugin na hindi network-activated sa iyong network. Kapag ang isang plugin ay network-activated, sa kalikasan nito, ito ay ilo-load bilang default sa bawat site ng iyong network.

Kung wala kang nakikitang mga plugin na nakalista sa tab na "Allowed Plugins", tingnan kung ang nais na plugin ay network-active o hindi.

  * **Maaari ko bang gamitin ang sarili kong disenyo ng Pricing Table?**

Oo. Simula sa bersyon 1.4.0, sinusuportahan ng Ultimate Multisite ang mga template override.

  * **Gumagana ba ang Ultimate Multisite sa WP Engine?**

Oo. Ang Ultimate Multisite ay palaging sinusuportahan ng WP Engine, ngunit ang pagpapaandar ng domain mapping ay nangangailangan ng mga manu-manong karagdagang hakbang mula sa admin (pagdaragdag ng domain sa WP Engine panel). **Hindi na ito ang kaso simula sa bersyon 1.6.0.** Ang Ultimate Multisite ay direktang nag-iintegrate sa WP Engine API at ang mga domain ay awtomatikong naidadagdag sa panel pagkatapos ng mapping.

  * **Aling hosting provider ang inirerekomenda ninyo?**

Dapat gumana ang Ultimate Multisite sa anumang hosting provider. Ang pagkakaiba lamang ay ang ilang hosting company tulad ng Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane at [Runcloud.io](http://Runcloud.io) ay may mas mahigpit na integration para sa mga kakayahan ng domain mapping kaya gumagana sila agad. Para sa ibang mga hosting, maaaring kailanganin mong i-set up ang domain mapping nang manu-mano.
