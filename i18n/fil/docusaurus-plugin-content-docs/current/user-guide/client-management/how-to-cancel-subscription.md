# Paano ko makakansela ang aking subscription?

Kung may Ultimate Multisite license ka, puwede mong kanselahin ang renewal nito kahit kailan mo gusto. Sundin lang ang mga hakbang sa ibaba:


<!-- Screenshot unavailable: Account page email with access link -->

Gamitin ang email na ibinigay mo noong bumili ka at ang iyong password.

<!-- Screenshot unavailable: Login page with email and password fields -->

Ito ang unang screen ng iyong Account Page:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Para kanselahin ang iyong subscription, sa menu sa kaliwa, pumunta sa _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

I-click ang arrow sa kanan para magbukas ng side window. Pagkatapos, piliin ang option na magkakansela ng **auto-renew**.

Magpapakita ang system ng confirmation message.

Kung sigurado kang gusto mong kanselahin ang iyong subscription, i-click lang ang _**Cancel Renewals**_ button. Pagkatapos nito, hihilingin sa iyo na sagutin ang isang maikling survey.

Tapos na! Hindi na awtomatikong mare-renew ang iyong subscription.

**Magkakaroon ka pa rin ng valid na key hanggang mag-expire ang iyong subscription**. Kung gusto mong i-reactivate ang iyong subscription, kailangan mo itong gawin nang manu-mano.
