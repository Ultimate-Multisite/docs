# Mga Website ng Iyong Kliyente (v2) {#your-clients-websites-v2}

Pagkatapos magparehistro sa iyong network, maa-access ng iyong mga customer ang kanilang Dashboard.

Bukod sa pahina ng **Account**, kung saan makikita nila ang mahalagang impormasyon tungkol sa kanilang mga pagbabayad, membership, domain, mga limitasyon ng plano, at iba pa, mayroon din silang access sa pahina ng **Aking Mga Site**, kung saan makikita at mapapamahalaan nila ang lahat ng kanilang mga website. Ang kailangan lang nilang gawin ay mag-click sa Aking Mga Site sa kanilang Dashboard.

![Link ng Aking Mga Site sa Dashboard ng customer](/img/account-page/my-sites-link.png)

Ipapakita sa kanila ang lahat ng kanilang mga website, na may button na **Pamahalaan** sa ilalim mismo ng bawat isa. Kung iki-click nila ito, ire-redirect sila sa **Dashboard ng partikular na website na ito**.

Makikita rin nila kung alin sa kanilang mga website ang **pangunahing** website para sa mga planong nagpapahintulot sa kanila na magkaroon ng higit sa isang website.

![Pahina ng Aking Mga Site na nagpapakita ng mga website ng customer na may mga button na Pamahalaan](/img/account-page/my-sites-page.png)

Narito ang buong tanaw ng interface sa pag-edit ng website:

![Buong interface sa pag-edit ng website](/img/admin/site-edit-full.png)

Kung gusto nilang palitan ang kanilang pangunahing website, kailangan nilang pumunta sa pahina ng Account sa pamamagitan ng pag-click sa menu na **Account**.

![Link ng menu na Account sa Dashboard ng customer](/img/account-page/account-menu.png)

Maaari mo ring tingnan at i-edit ang mga detalye ng customer:

![Interface sa pag-edit ng customer](/img/admin/customer-edit.png)

Pagkatapos, dapat nilang i-click ang **Palitan ang Default na Website**.

![Button na Palitan ang Default na Website sa pahina ng Account](/img/account-page/change-default-site.png)

May lalabas na pop-up window. I-click ang drop-down list at piliin kung aling website ang gusto mong gawing pangunahing website.

![Pagpili sa dropdown ng Palitan ang Default na Website](/img/account-page/change-default-site-dropdown.png)

Pagkatapos, i-click lang ang **Palitan ang Default na Website** at tapos ka na!

![Pagkumpirma sa pagbabago ng default na website](/img/account-page/change-default-site-confirm.png)
