# Pag-customize ng iyong Admin Panel gamit ang PRO Theme

Madalas kapag gumagawa tayo ng trabaho para sa mga kliyente, kailangan nating bigyan sila ng custom na karanasan sa admin dashboard ng WordPress. Kung naghahanap ka ng mas propesyonal na hitsura at dating, baka makatulong sa iyo ang PRO Theme.

Ang PRO Theme ay isang admin theme na ginawa para tulungan ka sa pag-customize ng Dashboard ng iyong mga kliyente. Ito rin ay bahagi ng mga premium plugin ng Ultimate Multisite.

Pagkatapos i-install, pumunta sa iyong _Plugins_ page at i-click para i-activate ito sa iyong Network.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_I-activate ito sa buong network sa iyong Plugins page_

_**PAALALA:**_ _Kung i-install mo lang ito, makikita ito ng iyong mga kliyente bilang isang opsyon na puwedeng piliin para idagdag ang theme sa kanilang user account lang._

Ngayon, ganap nang na-redesign ang iyong admin interface! Mayroon ka ring bagong menu sa kaliwang column na tinatawag na PRO Theme.
