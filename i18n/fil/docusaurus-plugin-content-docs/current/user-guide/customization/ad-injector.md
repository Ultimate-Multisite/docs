# Paglalagay ng Ads sa Iyong Network gamit ang Ultimate Multisite: Ad Injector

Ang Ultimate Multisite: Ad Injector ay isang **libreng add-on** na nagbibigay sa iyo ng mga tool para maglagay ng ads sa iyong mga plan direkta mula sa kanilang edit page.

Pagkatapos i-install ang Ultimate Multisite: Ad Injector, pumunta sa iyong _Plugins_ page para i-activate ito.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_I-click ang Network Activate button_

Kapag aktibo na ang add-on, ang _Advanced Options_ meta box sa plan edit screen ay magkakaroon ng bagong option na tinatawag na _Ad Settings_.

![Ad Settings sa plan edit screen](/img/config/product-edit-full.png)

_Ngayon ay puwede mo nang i-edit ang ad settings para sa bawat isa sa iyong mga plan_

Sa tab na iyon, maaari kang magdagdag ng ad codes para sa mga post at page sa front-end, bago at pagkatapos ng content.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Pagkatapos ilagay ang code ng iyong ad, lalabas ito sa mga post at page ng partikular na plan na iyon._

Maaari ka ring maglagay ng ads sa header ng Admin Pages para sa mga user ng plan na iyon.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Pumunta sa Admin Panel Ad Code option kung gusto mong magpakita ng ads sa back-end ng iyong mga user_

### Higit Pa sa Advertising

Maaari mo ring gamitin ang Ultimate Multisite: Ad Injector para mag-alok ng support sa mga high-tier plan.

Pumunta lang sa _Admin Panel Ads_ option at idagdag ang HTML code para sa iyong support tool.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Magdagdag ng customer support sa mga partikular na plan_

Sa ganitong paraan, maaari kang makausap ng mga tao direkta mula sa kanilang Admin Panel!
