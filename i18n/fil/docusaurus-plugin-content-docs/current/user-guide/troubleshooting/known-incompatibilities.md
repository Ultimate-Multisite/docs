# Mga Kilalang Hindi Tugma sa Ibang Plugin

Pagkatapos i-install at i-activate ang Ultimate Multisite, maaaring makaranas ka ng mga problema dahil sa mga hindi pa naaayos na incompatibility sa ibang mga plugin.

Aktibo kaming nagtatrabaho para mabawasan ang listahang ito sa bawat bagong release, pero kung nakalista dito ang isang plugin, maaaring kailanganin mong i-deactivate ito para gumana nang maayos ang iyong Ultimate Multisite Network.

Narito ang listahan ng mga ito at kung ano ang dapat mong gawin sa mga ganitong sitwasyon:

**Ilang plugin ng WPMU DEV** Ang ilan sa mga plugin na _hindi na_ minamaintain ng WPMU DEV ay kilalang nagdudulot ng pagtigil ng Ultimate Multisite registration flow sa huling hakbang (hindi makagawa ng account ang user). Narito ang listahan ng mga plugin na nagdudulot ng mga problemang ito:

  * Signup Language: _May add-on ang Ultimate Multisite na nag-aalok ng parehong functionality, na tinatawag na Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Kasama na ito sa Ultimate Multisite, kaya hindi na kailangan ng dagdag na plugin._

  * New Blog Templates: _Kasama na ito sa Ultimate Multisite, kaya hindi na kailangan ng dagdag na plugin._

  * Set Password on Multisite Blog Creation: _Kasama na ito sa Ultimate Multisite, kaya hindi na kailangan ng dagdag na plugin._

  * Signup TOS: _Kasama na ito sa Ultimate Multisite, kaya hindi na kailangan ng dagdag na plugin._

  * Anti-Splog: Kahit hindi kasing-powerful, _kasama na rin ito sa Ultimate Multisite._

**Tandaan**: Lahat ng ibang WPMU Dev plugin ay magagamit nang normal kasama ang Ultimate Multisite. Kabilang dito ang _Smush_, _Forminator_, _Defender,_ atbp.
