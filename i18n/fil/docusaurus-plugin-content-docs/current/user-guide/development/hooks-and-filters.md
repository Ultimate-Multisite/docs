# Mga Hooks at Filters ng Ultimate Multisite

Ang Ultimate Multisite ay binubuo ng maraming functions at database queries. Mayroon itong mga hooks at filters na magbibigay-daan sa iyo na baguhin o magdagdag ng bagong code sa mga kasalukuyang functions.

Gumawa kami ng listahan ng mga hooks at filters na ito bilang sanggunian para tulungan ang mga advanced na users na mas i-customize pa ang kanilang network. Maaari mong i-access ang listahan mula sa Github sa pamamagitan ng pag-click sa link sa ibaba.
