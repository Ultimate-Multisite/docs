# Paano Kumuha ng Custom Development Help mula sa Codeable?

Minsan, talagang kailangan natin ng magagandang customization para magawa natin ang ating network flows nang eksakto sa gusto natin. Kung ito ang hinahanap mo, inirerekomenda namin ang [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), isang lugar kung saan makakahanap ka ng mga bihasang developer na may kaalaman sa Ultimate Multisite.

Ang [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ay isang platform na nagkokonekta sa mga customer sa mga kwalipikadong WordPress freelance developer, na kayang magtrabaho sa mga proyekto ng anumang laki. May masusing application process sila para matiyak na maka-match ka sa mga eksperto na partikular na kwalipikado para sa task na nasa isip mo.

### Paano gumawa ng proyekto sa Codeable?

Una, pumunta sa [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) at i-click ang **Start a Project** button.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Ngayon, para mag-submit ng proyekto, i-click sa kaliwa ang **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Sa "I need help with", piliin ang **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Sa "My", piliin ang **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Pagkatapos, i-click ang **Continue** button.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Ngayon, oras na para ilarawan ang iyong proyekto at mga pangangailangan sa pinakamahusay na paraan na kaya mo. Subukang magbigay ng pinakamaraming impormasyon hangga't maaari!

<!-- Screenshot unavailable: Codeable project description form -->

Sagutin ang mga sumusunod na tanong para makatulong sa pricing.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Pagkatapos niyan, kung wala ka pang [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) account, gumawa ng isa at i-click ang **Publish My Project** button.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Kapag na-publish mo na ang iyong [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) na proyekto, ikokonekta ka ng platform sa mga developer na tumutugma sa skill set na kinakailangan para sa iyong proyekto.

Ang prosesong ito ay makakatulong na mahanap ang tamang tao para sa iyong trabaho, na may karanasan sa WordPress at Ultimate Multisite!
