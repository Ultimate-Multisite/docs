# Pag-install ng Ultimate Multisite

:::note
Ipinapalagay ng tutorial na ito na naka-install at naka-configure na ang WordPress Multisite sa iyong system. Para malaman kung paano, tingnan ang [tutorial na ito](https://www.wpbeginner.com/glossary/multisite/) mula sa WP Beginner.
:::

## Pag-install ng Plugin

Available ang Ultimate Multisite nang libre sa [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/).

Mula sa iyong **Network Admin Dashboard**, pumunta sa **Plugins â†’ Add New Plugin**.

![Pahina ng Network Admin Add New Plugin](/img/installation/add-new-plugin.png)

Hanapin ang **"Ultimate Multisite"** (may panipi para sa eksaktong tugma) at lalabas ito bilang unang resulta. I-click ang **Install Now**.

![Mga resulta ng paghahanap na nagpapakita ng Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

Kapag naka-install na, i-click ang **Network Activate** para i-activate ang plugin sa buong network mo.

![Naka-install na plugin na may Network Activate button](/img/installation/plugin-installed.png)

Pagkatapos ng activation, awtomatiko kang ire-redirect sa Setup Wizard.

![Na-activate na plugin at na-redirect sa wizard](/img/installation/plugin-activated.png)

## Setup Wizard

Gagabayan ka ng Setup Wizard sa pag-configure ng Ultimate Multisite sa loob ng humigit-kumulang 10 minuto.

### Welcome

I-click ang **Get Started** para magsimula.

![Welcome screen ng Setup Wizard](/img/installation/wizard-welcome.png)

### Pre-install Checks

Tinitingnan ng hakbang na ito ang impormasyon ng iyong system at WordPress installation para matiyak na natutugunan nito ang mga kinakailangan ng Ultimate Multisite. Kung maayos ang lahat, i-click ang **Go to the Next Step**.

![Pre-install checks na nagpapakita ng system requirements](/img/installation/wizard-pre-install-checks.png)

### Installation

Gagawa ang installer ng mga kinakailangang database table at i-install ang `sunrise.php` file na kailangan ng Ultimate Multisite para gumana. I-click ang **Install** para magpatuloy.

![Hakbang sa installation na nagpapakita ng database tables at sunrise.php](/img/installation/wizard-installation.png)

### Your Company

Punan ang impormasyon ng iyong kompanya at itakda ang iyong default na currency. Gagamitin ang impormasyong ito sa buong WaaS platform mo. I-click ang **Continue** kapag tapos na.

![Hakbang sa configuration ng Your Company](/img/installation/wizard-your-company.png)

### Default Content

Pinapahintulutan ka ng hakbang na ito na mag-install ng mga predefined na template, produkto, at iba pang starter content. Magandang paraan ito para maging pamilyar sa mga feature ng Ultimate Multisite. I-click ang **Install** para idagdag ang default content, o laktawan ang hakbang na ito kung mas gusto mong magsimula mula sa simula.

![Hakbang sa pag-install ng default content](/img/installation/wizard-default-content.png)

### Recommended Plugins

Opsyonal na mag-install ng mga inirerekomendang companion plugin. I-click ang **Install** para idagdag sila o laktawan para magpatuloy.

![Hakbang sa recommended plugins](/img/installation/wizard-recommended-plugins.png)

### Ready!

Tapos na! Kumpleto na ang iyong Ultimate Multisite installation. Maaari ka nang magsimulang bumuo ng iyong Website as a Service platform mula sa **Network Admin Dashboard**.

![Setup complete - Ready screen](/img/installation/wizard-ready.png)

![Network Admin Dashboard na may aktibong Ultimate Multisite](/img/installation/network-dashboard.png)

Sige, mag-enjoy ka!
