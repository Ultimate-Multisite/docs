# Mga Kinakailangan ng Ultimate Multisite

Ang unang tanong na maaaring pumasok sa isip mo bago bumili ng Ultimate Multisite ay kung ano ang mga kinakailangan para patakbuhin ang aming plugin. Sa kabutihang-palad, iilan lang ang mga pangunahing kinakailangan para magamit ang Ultimate Multisite: isang **domain name**, isang mahusay na **hosting provider** at isang WordPress **Multisite** installation. Iyon lang ang kailangan mo!

## Domain Name

Una, pag-usapan natin ang domain name.

Sa pagpili ng domain name, isaisip mo kung ano ang iyong partikular na niche at kung ano ang gagawin mo gamit ang Ultimate Multisite: Gusto mo bang gumawa ng ahensya? Magbenta ng mga website na tulad ng online shop? O baka isang WordPress web hosting? Lahat ng ito ay posible sa Ultimate Multisite.

Nagbibigay ang aming plugin ng mga subscription service, pagkolekta ng bayad, checkout form, discount voucher at komunikasyon sa mga customer. Malaking bahagi ng trabahong kailangan para mag-install, mag-configure at mag-maintain ng WordPress Multisite ay pinapadali ng Ultimate Multisite.

Anumang niche na piliin mo ay maaaring pamahalaan gamit ang aming plugin basta't nagbebenta ka ng mga website bilang serbisyo! Siguraduhing pumili ng domain name na akma sa iyong business plan.

## Hosting Provider

Ngayong mayroon ka nang nakakaakit na domain name na kumakatawan sa iyong niche at negosyo, kailangan mong pumili ng magandang web hosting company para i-host ang iyong network gamit ang Ultimate Multisite. May ilang magagandang opsyon sa merkado. May mga hosting provider na mahusay na nag-i-integrate sa Ultimate Multisite, na maaaring maging mahalagang salik depende sa kung ano ang plano mong gawin.

Dapat gumana ang Ultimate Multisite sa kahit anong hosting provider. Ang tanging pagkakaiba ay may ilang hosting company tulad ng _WPMU DEV Hosting_, _WP Engine_, _Closte_, _Cloudways_, _Gridpane_ at _Runcloud_ na may mas mahigpit na integration para sa domain mapping. Para sa ibang hosting, maaaring kailanganin mong i-map nang manu-mano ang mga custom domain. Magandang ideya rin na kumuha ng hosting provider na sumusuporta sa AutoSSL. Sinusuportahan ito ng _Closte_, _Runcloud_ at _cPanel_.

Pagdating sa mga server specification, walang tiyak na pormula dahil depende ito sa mga feature at function na iyong ibibigay at sa laki ng iyong network. Maaaring magandang simula ang shared hosting, pero kapag nagsimulang lumaki ang iyong network, malamang na kakailanganin mo ng dedicated VPS.

## WordPress Multisite

Ngayon, pag-usapan naman natin ang WordPress Multisite installation.

Pinapayagan ka ng WordPress Multisite na magkaroon ng network ng mga site sa isang installation lamang. Ito ay built-in na feature, pero hindi ito aktibo bilang default. Kailangan mong i-enable ang feature na ito sa loob ng iyong WordPress installation at web hosting control panel.

Ang pagkakaroon ng WordPress na naka-enable ang Multisite **ang pundasyon ng Ultimate Multisite**. Dahil ang pangunahing gamit ng Ultimate Multisite ay tulungan ang mga user na pamahalaan at magbenta ng mga website bilang serbisyo, kailangan itong gawin sa WordPress na naka-enable ang Multisite.

Mayroon kaming **isang detalyadong artikulo** na nagtuturo nang hakbang-hakbang kung ano ang kailangang gawin para ma-enable ang Multisite feature sa iyong WordPress installation.

Pagkatapos mong ma-enable ang Multisite sa WordPress, oras na para i-install ang Ultimate Multisite. Para mai-install nang tama ang Ultimate Multisite, kailangan mong magkaroon ng:

  * WordPress: v5.3+ (Inirerekomenda: Pinakabagong stable version)

  * PHP: 7.4.x (suporta sa > 8.0 paparating na)

  * MySQL: v5+ (Inirerekomenda: 5.6, kung sakaling wala kang 8.0 na available sa iyong hosting provider)

Ito lamang ang mga software requirement para sa Ultimate Multisite.

Tandaan na maaaring gumana ang Ultimate Multisite sa PHP 8.0 pero inirerekomenda naming patakbuhin ito gamit ang PHP 7.4.x.

Gayundin, inirerekomenda naming i-install ito sa **mga main domain**, hindi sa mga subdomain o subdirectory. Maaari itong gumana sa mga subdomain/subdirectory, pero maaaring magkaroon ng ilang error.

Para sa detalyadong gabay kung paano i-install ang Ultimate Multisite, **mag-click dito**.
