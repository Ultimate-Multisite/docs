# Ultimate Multisite Documentation

Maligayang pagdating sa Ultimate Multisite documentation. Ginagawang Website as a Service (WaaS) platform ang Ultimate Multisite mula sa WordPress Multisite.

## Documentation Sections

### [User Guide](/user-guide/getting-started/ultimate-multisite-101)

Lahat ng kailangan mo para i-set up, i-configure, at pamahalaan ang iyong WaaS platform. Sakop nito ang pag-install, payment gateways, domain mapping, pamamahala ng kliyente, at marami pa.

### [Developer Documentation](/developer/hooks)

Teknikal na sanggunian para sa mga developer na gumagawa gamit ang Ultimate Multisite. Kasama rito ang hooks reference, dokumentasyon ng REST API, at mga gabay sa pagbuo ng addon.

### [Addons](/addons)

Dokumentasyon para sa mga bayad na addon plugin na nagpapalawak sa Ultimate Multisite ng mga karagdagang features tulad ng WooCommerce integration, analytics, pagbebenta ng domain, at marami pa.
