# Gwahanu'r Panel Administracion gyda PRO Theme

Yn aml pan fyddwch yn gwneud gwaith i cyddeistrywyr, mae angen i chi rhoi ddefnyddio gyflwyniad cyfathrebuol yn y dashboard WordPress o'ch cyddeistrywyr. Os ydych chi'n chwilio am ffurf a chwarae mwy proffesiynol, gall PRO Theme helpu chi.

Mae PRO Theme yn theme admin a ddefnyddiatwyd i helpu chi pan fyddwch yn gwneud cyfathrebu'r Dashboard o'ch cyddeistrywyr. Ac hefyd mae'n rhan o core y pluginau priwmaidd Ultimate Multisite.

Ar ôl ei chwarae, myndwch i'r peidiad _Plugins_ eich a chwilio i chwarae ar gyfer eich Network.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Chwarae'i network-wide ar peidiad Plugins_

_**NOTIA:**_ _Os ydych chi'n gwneud ei chwarae yn unig, bydd hi'n dangos ar eich cyddeistrywyr fel opsiwn a gallwch ddynnu i dros y cyfathrebueth yn unig i'r bennydd hwn._

Er hyn, mae gennych chi'r gyfathrebueth admin eich cael ei redig allan! Mae gennych hefyd menu newydd ar yr olenni lleol gyda PRO Theme.
