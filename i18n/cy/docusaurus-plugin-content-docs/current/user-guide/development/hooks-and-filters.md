# Hookau a filtru Ultimate Multisite

Mae Ultimate Multisite wedi'i gynhyrchu o gyfathrebu o ffrifau a cwestiynau databwriaeth. Mae ganddo hookau a filtru sydd yn sylweddol i'ch gallwch newid neu drosglwyddo code newydd i ffrifau sydd y bydd.

Rydyn ni'n gynnig llyfr cyflwyniad o'r hookau a filtru hyn fel cyfeiriad i helpu bennwrion advanc yn gallu gwneud ei rheoli'r cyd-llwybr (network) gyda nhw. Gallwch cyffredinol y llyfr hon o Github gan clisi'r linc isodol:

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
