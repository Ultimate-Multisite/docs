# Pethau'r ychwanegiadau gyda pluginau eraill

Ar ôl i'w chwarae a chynllunio Ultimate Multisite, gallwch dod â'r angen o problemau oherwydd ychwanegiadau eraill sydd wedi'u gwneud yn ddefnyddiol.

Rydyn ni yn gweithio ar gyfer i ddifyn y rheol hwn gyda chwarae newydd, ond os mae plugin wedi'i listri yma, gallwch angen ei deaktiveio i gael yr Ultimate Multisite Network yn ffurfiol.

Dyma llyfr ychwanegiadau hyn a beth byddwch yn ei wneud mewn perthyn:

**Rhagwn WPMU DEV pluginau** Mae rhai o'r pluginau sydd wedi'u gwneud yn ddefnyddiol gan WPMU DEV yn cael eu gael yn angen i sefydlu'r rheol registration Ultimate Multisite ar y cyflwyniad olaf (mae'r benyn nid yw'n gallu creu cyfrifol). Dyma llyfr y pluginau sydd wedi'u gwneud yn gallu golygu'r pethau hyn:

  * Signup Language: _Mae Ultimate Multisite gyda chwarae add-on sydd â'r fynnal yn cynnwys, gan enw ei 'Ultimate Multisite: Language Switcher'._

  * Set Blog Description on Blog Creation: _Mae hyn wedi'i gweithio mewn i Ultimate Multisite, felly nid oes angen plugin addol._

  * New Blog Templates: _Mae hyn wedi'i gweithio mewn i Ultimate Multisite, felly nid oes angen plugin addol._

  * Set Password on Multisite Blog Creation: _Mae hyn wedi'i gweithio mewn i Ultimate Multisite, felly nid oes angen plugin addol._

  * Signup TOS: _Mae hyn wedi'i gweithio mewn i Ultimate Multisite, felly nid oes angen plugin addol._

  * Anti-Splog: Er fod yn ddefnyddiol nad yw'n gryf oherwydd mae hefyd wedi'i gweithio mewn i Ultimate Multisite._

**Noti** : Gallai'r holl pluginau WPMU Dev eraill cael eu defnyddio'n norm gyda Ultimate Multisite. Mae gan ychwanegiadau fel _Smush_ , _Forminator_ , _Defender,_ osodol.
