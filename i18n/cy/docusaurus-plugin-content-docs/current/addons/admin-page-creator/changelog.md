# Changelog y Gweithredig Adrefi Admin Page Creator

### Versiwn 1.9.0 - Rheoliwyd ar 2026-01-18

* Arwain: Erru fatalau pan mae cyflwyniad yn cael ei gyflwyno.
* Refactor: Clasusau ffynonig (Content source classes) i ddefnyddio Singleton trait gyda chwarae parent (parent constructor) sydd yn iawn.
* Arwain: Erru compiliad template Vue.js yn rhoi'n rhyfedd i ddallt y selector o ddethau (content type selector).
* Arwain: Prosiedau data Vue (template_id, external_link_url_embedable, external_link_url_checking) yn ddefnyddio.
* Arwain: Editor TinyMCE nid oedd yn dechrau'n iawn ar gyfer cyflwyniad y peidiad.
* Arwain: Fael y dillad cefnogaeth Divi nid yn cael ei ddarganfod drwy Composer.
* Arwain: Syntax PHP wedi'i ddwfn mewn clasus cefnogaeth Divi.
* Gyfeiriad: Cyflawniad a minification (minify) o ddefnyddio JavaScript.
* Gyfeiriad: Cyflawniad gyda'r ffurf WordPress a chyddlydd peidiad (page builder) yn gyflymestriol.
