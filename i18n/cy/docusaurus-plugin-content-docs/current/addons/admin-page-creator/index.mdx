# Gwrthwrdd Admin Page Creator

## Cynnwysiad (Content)
Bydd y gwrthwrdd Admin Page Creator yn rhoi'n gallu i chi iddo sicr yn ddefnyddio newydd admin pages a sub-pages yn y dashboard WordPress. Gallai ei ddefnyddio i creu un gyfrifol neu gyfrifol tutorial sydd wedi'i gweithredu fel gyfrifol cyflwyniad ar gyfer bennewidiol newydd. Gallwch ddefnyddio page builder fel Beaver Builder, Elementor, Brizy, a Oxygen i ddefnyddio'r pages heb anghofio.

Mae'n cynnig hefyd cefnogaeth HTML, CSS a JavaScript cyflogedig, fel hefyd mae rheoli cyffredin ar adeiladu plan a rôl benyddol!

## Gymelau (Features)
* Cefnogaeth HTML a CSS cyflogedig
* Cefnogaeth JavaScript
* Mod Cynllun PHP (PHP Content Mode)
* Rheoli ar adeiladu Rôl Benyddol (Permission based on User role)
* Cefnogaeth i page builders

## Ddaccymuniad (Documentation)
[Creu Admin Pages a Sub-pages Cyflogedig Trwy Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Gymparu (Installation)

1. Lleolwch **ultimate-multisite-admin-page-creator** i'r dyfodol **/wp-content/plugins/**
2. Actiwnwch y plugin drwy'r menywod **Plugins** yn Adeiladu Network WordPress eich.

## Cyfarwyddiad Defnyddio (Usage Guide)

Er mwyn WordPress mae'n rhyfeddu iawn i sicr ychwanegu cynnwys i'r ffordd o flaen y safonau, ond mae creu pages ac sub-pages ar y llyfrgell admin yn rhoi angen rhywfaint o sgiliau codi. Ultimate Multisite: Admin Page Creator yn ceisio dod â'r unigol yfedd o ddefnyddio'r ffordd o flaen i'r llyfrgell ochr-ganedd (back end) o'ch safonau.

## Pam i ddefnyddio Admin Page Creator?

Efallai eich bod eisiau cynnig pages cyflogol ar y panel admin o'ch golybus gyda tutorialau, felly ebyd eisiau ychwanegu pages cyflogol gyda FAQ a widget cyfathogh. Mae'r ddefnyddiolwch yn llwyr gyda Admin Page Creator, ac mae hyn sydd yn cael ei chwarae mewn tutorial hwnnw.

![Admin Pages list](/img/admin/admin-pages-list.png)

_Mae creu Pages Tutorial gyda playlist YouTube wedi'i cyflwyno nawr!_

_Dyma'r resgyniad o flaen ar y Panel Dashboard ybysgol_

## Creu eich Pages Admin Cyflogol gyntaf

Ar ôl i chwarae a chwarae Ultimate Multisite: Admin Page Creator, mynd i'r menu admin o'ch safonau ac mynd i Admin Pages.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Llyfrgell Custom Admin Pages ar y Network Admin_

Byddwch yn gweld llyfrgell o'r holl pages admin sydd eich creu wedi'i chwilio hi (gan fod hyn yn y cyfarfod gyntaf, bydd y llyfrgell yn ôl). Goedwch chi ddechrau clisi **Add new Admin Page**. Byddwch yn cael eich reiddirol i'r editor Pages Admin.

## Y elementau o Pages Admin Cyflogol

Ar gyfer y redytwr ar y peidiad Admin, bydd gennych chiweliadau i chysbwosi eich peidiad.

### Titell Peidiad (Page Title)

Mae hyn yn teitl sydd yn cael ei ddangos ar ddefnyddi'r peidiad, nid label menu admin.

<!-- Screenshot unavailable: Admin Page editor showing Page Title field -->

_Cyflwynwch Titell Peidiad_

### Seictwr Tyd (Content Type Selector)

Mae Ultimate Multisite: Admin Page Creator yn rhoi ddau opsiyn i chi gyda'r cynnwys y peidiad: gallwch defnyddio'r redytwr WYSIWYG (What You See Is What You Get) o WordPress fel defnyddiwch, neu gwahanu i redytwr HTML.

Os ydych chi nid yn ymddangos â HTML, parhau gyda'r redytwr WordPress defnyddiol. Gallwch redig y cynnwys y peidiad fel yr wynebu byddai'n cael eich redig gyda Post neu Peidiad arferol.

<!-- Screenshot unavailable: Admin Page Creator using the default WordPress WYSIWYG editor -->

_Defnyddio'r redytwr WYSIWYG o WordPress defnyddiol_

<!-- Screenshot unavailable: Admin Page Creator using the HTML editor with syntax highlighting -->

_Defnyddio'r opsiwn redytwr HTML, gyda syntax highlight a gairau eros_

## Opsiynau Menu (Menu Options)

Ar y talyb, ar y meta-box General Options, gallwch ddefnyddio'r rheoliad yr adran menu, label, a chwarae llawer o bethau.

### Tydau Menu (Menu Types)

Gall eich peidiad admin newydd gael ddau tyndlyntau menu: menu o ddefnydd top-level a submenu.

Mae peidiad menu o ddefnydd top-level, fel ystod yr enw, yn ychwanegu'r peidiad newydd sydd wedi'i creu i'r menu admin gyda'r peidiadau WordPress arferol.

Os ydych chi'n debydoli'r opsiwn o ddefnydd top-level, gallwch hefyd wybilydd y cyfeiriad menu a icon menu.

Os byddwch yn gallwch gwneud y'adennau hwn yn lleoliad menyc (sub-menu page), byddwch yn gallu dewis path parent i ddatblygu'r adennau hyn. Gallwch drosglai eich adennau cyfathrebuol i adennau WordPress cyffredinol neu i'ch adennau admin top-level.

<!-- Screenshot unavailable: Admin Page Creator menu type options with label, order, and icon settings -->

_Gallwch hefyd ddatblygu'r adennau hyn fel sub-menu o gyfathrebuadau menyc byw, gan gynnwys eich adennau top-level cyfathrebuol ein. _

## Opsiadau Cynhwysion (Advanced Options)

Byddwch yn gallu ddatblygu pob math o cynnwys i'ch adennau admin cyfathrebuol. Mae hyn yn cynnwys rheolau CSS a ffilyfrifau, JavaScript, a llyfrgellau allanol hefyd. Gallwch wneud hyn drwy ddefnyddio y Tab Opsiadau Cynhwysion (Advanced Options).

### CSS Cyfathrebuol (Custom CSS)

Ar y tab CSS, gallwch ddatblygu eich cod CSS cyfathrebuol, fel hefyd ei chwilio ffilyfrifau CSS o safleoedd allanol (fel CDNs, Google Fonts, etc.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Mae CSS Cyfathrebuol yn cael ei chymryd hefyd. Gallwch hefyd gynnwys ffilyfrifau styl allanol!_

### JavaScript Cyfathrebuol (Custom JavaScript)

Mae hyn yn gywir hefyd ar gyfer JavaScript. Gallwch ddatblygu llyfrgellau allanol i ddefnyddio ar eich cod JavaScript cyfathrebuol.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Cynllun eich cod JavaScript ein!_

## Pethau (Permissions)

Yn Adran Opsiadau Advanced, byddwch yn gallu dewis y cyflwyniad ychwanegol y byddai'n cael ei effeithio ar y sub-sites. Mae hyn yn ddefnyddiol os ydych chi’n gofyn i gynnwys cynnwys unigol yn rhywfaint i ddeiliadau plan gwahanol, felly.

<!-- Screenshot unavailable: Cyddeisio am gyflwyniad ychwanegol ar gyfer rheoli cyffredin a cyffredin rôl -->

_Mae ganrhaedd cyflwyniad hwn yn cael ei reoli gan fy nghaliad_

**Betyg** : Mae'r ddau amodau'n angen i fod y peidiad yn cael ei dangos. Mae hynny yn golygu bod mae'r benyw yn ddefnyddirol os oes ybain yn cleient o plan genny, ond nad yw'n cael un o gyflwyniadau wedi'i dewis (neu yn gyflog), bydd y peidiad yn cael ei dangos ar eu panel.
