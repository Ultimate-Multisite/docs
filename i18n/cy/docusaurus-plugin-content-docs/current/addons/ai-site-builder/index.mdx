# AI Site Builder {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Ymddiantaueddau {#requirements}

- WordPress 5.3 neu ddiweddarach
- PHP 7.4 neu ddiweddarach
- Plugin Multisite Ultimate (defnyddio)

## Rheoliad {#installation}

1. Rhowch y cyfathrebu'r addon i'r dyfodol `/wp-content/plugins/`
2. Actiwwch y plugin drwy'r menyn 'Plugins' yn WordPress
3. Rheoliwch y setegau'r addon yn y admin cyd-gynllun (network admin)

## Datblygiad {#development}

### Rheoliad {#setup}

```bash
# Rhowch y cyfathrebu'r dependants i'r gweithrediad
composer install
npm install

# Runo'r gwylioedd
npm run test

# Runo'r gwylioedd standard cod
vendor/bin/phpcs
vendor/bin/phpstan

# Gyfyngu'r pwoblau styl cod
vendor/bin/phpcbf

# Cynllunio ar gyfer y proddection
npm run build
```

### Cynodiadau (Testing) {#testing}

Mae'r addon yn cynnwys ffurfiolyn cynaliad cyfan:

- **Cynodiadau Unigol**: Cynodiadau wedi'i gweithredu ar adalaeth PHPUnit gyda set testau WordPress.
- **Class Cynodiad Cyfan**: Mae'n rhoi fforddion cynorthwyo am gyfathrebu testau cyffredinol.
- **Cynodiadau Cod**: Mae'n cael eu cyflwyno i ddefnyddio ar gyfer cerdded cod (code coverage).
- **Cynodiadau Multisite**: Cynodiadau yn cael eu gweithredu mewn anghytuntrio multisite.

### Standardau Cod {#code-standards}

**Standardau Codig WordPress**: wedi cael ei gyfyngol trwy PHPCS
**Cymrydolod PHP 7.4+**: Mae ffychiadau PHP modern yn cael eu chynllunio
**Analysau Statach (Static Analysis)**: Ychwanegiad PHPStan i gwylio tybedd (type checking)
**Modernau Codig (Code Modernization)**: Rheoli Rector ar gyfer cyfathriadau PHP

### System Gweithredol (Build System) {#build-system}

- **Prosesau Asset**: Minification ar gyfer ffili CSS/JS
- **Cyfieithiad**: Cynllunio fyliau POT
- **Pencadlu (Packaging)**: Creu archiwiaid ar gyfer disgyblhau
- **Moddolion Mae'n Gweithredol (Development Mode)**: Assetau nid wedi'u minify i'r ddefnyddio am ddargano

## Cyfluniadau (Configuration) {#configuration}

Mae'r addon yn chymryd ychwanegiadau cyfluniad yn cael eu cyfathri ar gyfer drwy ffurf admin cyd-llwybr.

## Hookau a Fyrddau (Hooks and Filters) {#hooks-and-filters}

### Gweithredoedd (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - Mae'n cael ei gweithredu ar ôl ychwanegiad addon
- `ultimate-multisite-ai-site-builder_loaded` - Mae'n cael ei gweithredu ar ôl i gyd y ffili addon wedi'u chynllunio

### Fyrddau (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Gyfeiriad ychwanegiadau addon
- `ultimate-multisite-ai-site-builder_enabled` - Mae'n gwneud ystodol i'r cyfathri addon yn cael ei gael/ddysbellu
