# Cofnod Newidiadau Aml-Rwydwaith {#multi-network-changelog}

### 1.3.0 {#130}
* Newydd: Rhagolygwr templed rhwydwaith — porwch a rhagolygu templedi rhwydwaith gyda phanel byw cyn prynu.
* Newydd: Dewisydd templed rhwydwaith wrth dalu — gall cwsmeriaid ddewis templed rhwydwaith yn ystod llif cofrestru wrth dalu.
* Newydd: SSO dolen hud traws-rwydwaith — gall cwsmeriaid gyrchu is-rwydweithiau yn ddi-dor drwy ddilysu dolen hud.
* Newydd: Clonio cyfryngau rhwydwaith — caiff cyfryngau safle eu dyblygu’n gywir wrth glonio rhwydweithiau o dempled.
* Trwsiad: Mae clonio rhwydwaith bellach yn gweithio heb yr ategyn aml-denantiaeth; caiff breintiau super admin eu hail-roi ar ôl clonio.
* Trwsiad: Mae creu rhwydwaith bellach yn gydnaws â WooCommerce a phob porth talu.
* Trwsiad: Mae rhif y porth bellach wedi’i gynnwys wrth gynhyrchu parth wrth gefn.
* Trwsiad: Symudwyd CSS y dewisydd templed i daflen arddull wedi’i rhoi mewn ciw i’w rendro’n gywir ar bob thema.
* Trwsiad: Diogelu rhag allwedd `network_id` heb ei diffinio yng nghwmpas ymholiad safle.
* Trwsiad: Mae gwirio caniatâd rhwydwaith cyn newid cyd-destun yn atal gwallau priodwedd tabl `wpdb`.
