# Fluent Forms {#fluent-forms}

Mae'r ychwanegyn hwn yn ei gwneud yn bosibl ychwanegu terfynau at Fluent Forms ar Multisite Ultimate mewn cynhyrchion, aelodaethau neu safleoedd.
Mae Fluent Forms yn defnyddio tablau personol yn lle mathau post personol, felly nid yw'r opsiynau cyfyngu arferol yn gweithio gyda Fluent Forms.
