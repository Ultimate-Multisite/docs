# Ultimate Multisite Документација {#ultimate-multisite-documentation}

Добре дојдовте во документацијата за Ultimate Multisite. Ultimate Multisite го трансформира WordPress Multisite во целосна платформа „веб-локација како услуга“ (WaaS).

## Секции на документацијата {#documentation-sections}

### [Кориснички водич](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Сè што ви е потребно за да ја поставите, конфигурирате и управувате со вашата WaaS платформа. Опфаќа инсталација, платежни портали, мапирање домени, управување со клиенти и повеќе.

### [Документација за програмери](/developer/hooks) {#developer-documentation}

Техничка референца за програмери што градат врз Ultimate Multisite. Вклучува референца за hooks, REST API документација и водичи за развој на додатоци.

### [Додатоци](/addons) {#addons}

Документација за платени додаток-приклучоци што го прошируваат Ultimate Multisite со дополнителни функции како WooCommerce интеграција, аналитика, продажба на домени и повеќе.

- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- Преземете го основниот plugin
