# Како можам да добијам пристап до својата страница за акаунт по првиот пат?

Добредојдовте во Ultimate Multisite!

Достапите до вашата страница за акаунт преку линкот што го добивте на е-поштата по купувањето на нашиот плагин:

<!-- Скриншот не е достапен: Е-пошта со линк за пристап до страницата за акаунт -->

Користите е-поштата што ја внесувате при купувањето и лозинката што ја добивте во е-поштата.

<!-- Скриншот не е достапен: Страница за најава со текстови за е-пошта и лозинка -->

Бидејќи ова е првиот пат кога ќе се најавите во овој акаунт, ќе ви биде побарано да го промените лозинката. Променете ја и кликнете на копчето _Save_ (Зачувај).

<!-- Скриншот не е достапен: Форма за промена на лозинка со копче Save -->

На вашата страница за акаунт, можете да управувате со сите детали на вашиот акаунт, како што се фактурирањата, реновирањето, отстранувањето и многу други работи.
