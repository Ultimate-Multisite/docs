# Како да добиете кајдење за развивање од Codeable? {#how-to-get-custom-development-help-from-codeable}

Понема пак ви е потребно некоја супер персонализација за да го направиме нашите работни процеси точно како што сакаме. Ако тоа е на што барате, препорачуваме [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), место каде можете да најдете искусени развивачи со експертиза во Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) е платформа која ги поврзува клиентите со квалифични фрилансерски развивачи на WordPress, способни да работат на проекти од било какав обем. Тие имаат детален процес на апликација за да се осигурате дека ќе бидете парот со експерти кои уште специфично квалификувани за задачата што ја имате предвид.

### Како да креирате проект во Codeable? {#how-to-create-a-project-in-codeable}

Прво, отидете на [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) и кликнете на копчето **Start a Project** (Започни проект).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Сега, за да поднесете проект, кликнете налево на **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Во „I need help with“ (Треба ми помош со), изберете **Customizing** (Персонализација):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Во „My“ (Мој), изберете **Plugin** (Плагинот):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Потоа кликнете на копчето **Continue** (Продолжи).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Сега е време да опишете го вашиот проект и потребата со најдобар начин што можете. Обидете се да дадете што повеќе информации!

<!-- Screenshot unavailable: Codeable project description form -->

Одговорете на следните прашања за помош при ценирање.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Потоа, ако још немате [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) акаунт, креирај го и кликнете на копчето **Publish My Project** (Објави мој проект).

<!-- Снимка недоступна: креирање акаунт во Codeable и копче Publish My Project -->

Како само што ја објавите вашиот [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) проект, платформата ќе ве поврзе со развивачи кои одговараат на вештините што ви ги бараат за вашиот проект.

Овој процес ќе ви помогне да најдете соодветниот човек за вашата работа, кој има искуство со WordPress и Ultimate Multisite!

Минималната цена на Codeable е $70–$120 USD по час. Проценетите ќе се менуваат во зависност од обемот на работата, сложноста и брзината на проектот.

Кога ќе добиете вашата проценка, ќе можете да го нацртате развивачот кој најдобро одговара на вашите потреби и заедно да започнете со вашиот проект!
