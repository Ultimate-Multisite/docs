# Ultimate Multisite Хукови и филтри {#ultimate-multisite-hooks-and-filters}

Ultimate Multisite е составен од многу функции и пребарувања во базата на податоци. Тој има хукови (hooks) и филтри кои ќе ви овозможат да ги промените или додадете нови кодови во постоечките функции.

Ние направивме листа на овие хукови и филтри како референтна точка за да помогнете на напреднатите корисници да ги персонализираат својата мрежа уште повеќе. Можете да пристапите до листата од Github преку кликање на линкот подолу.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
