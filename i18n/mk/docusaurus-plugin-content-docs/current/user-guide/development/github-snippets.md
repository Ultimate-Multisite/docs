# Како да користите филтри (snippets) на Ultimate Multisite во нашиот GitHub репозиториум {#how-to-use-ultimate-multisite-snippets-on-our-github-repository}

Во GitHub репозиториумот има готови код филтри (code snippets) што често се бараат од корисниците на Ultimate Multisite кои сакаат да додадат мали функционалности, како на пример да вметнат скрипт за Google Analytics на страници за регистрација или да го скријат мета боксот од административната табла.

Оваа станица ќе ви покаже како да ги користите овие кодови или поспецифично каде да ги поставите.

Можете да ги најдете филтрите на линкот подолу:

https://github.com/next-press/wp-ultimo-snippets/

Има два начини за да додадете кодот:

  1. На вашиот `functions.php` файл на вашата тема.

  2. Must-Use Plugins (mu-plugins).

# Како да додадете филтри (snippet) во функцијата `functions.php` на вашата тема. {#how-to-add-the-snippet-on-your-themes-functionsphp-file}

  1. Влезте во административната табла на вашата WordPress Network и отидете на Themes > Theme Editor (Вие можете да го видите сликата подолу).

  2. На страницата "Edit Themes", осигурајте се дека сте избрале активна темата од менито за падање (dropdown) што се на направата на вашиот екран во горниот десен агол (#3 на сликата подолу).

  3. Кликнете на `functions.php` 파일을 под секцијата "Theme Files" за да го вчите файлот. Пролијте надолу до дна и залепете го филтриот Ultimate Multisite што го добивте од GitHub репозиториумот.

# Како да создадете Must-Use Plugins (mu-plugins) {#how-to-create-must-use-plugins-mu-plugins}

WordPress има функција која ви овозможува да вчите кастом функционалност наречена "Must-Use Plugins", или кратко "mu-plugins".

Овие специјални mu-plugins се вчувуваат пред сите други редовни плагини и не можат да бидат деактивирани. Во multisite мрежа, кодот во овие mu-plugins ќе се вчитува на сите страници во вашата инсталација.

1. Користете FTP или SSH за пристап до файловата система на вашата WordPress инсталација.

2. Во директоријумот `wp-content` на вашата инсталација на WordPress, создадете нова директорија со име: `mu-plugins`.

<!-- Скриншот не е достапен: Файл менажувач што покажува ја директоријумот wp-content со папката mu-plugins -->

3. Создадете нов PHP файл на вашиот компјутер со име `wu-snippet.php` користејќи Notepad или било каков редагор за код.

4. Ставете го кодот од Ultimate Multisite што го добивте од GitHub репозиториумот во овој файл и зачувајте го. Можете да додадете и овој код над кодот наsnippet-от за да го означите вашиот mu plugin.
