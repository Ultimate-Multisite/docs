# Интеграција со GridPane

## Преглед
GridPane е специјализирана контролна панел за хостирање WordPress, изградена за сериозни професионалци на WordPress. Ова интегрира автоматско синхронизирање на домените и управување со сертификатите SSL помеѓу Ultimate Multisite и GridPane.

## Функции
- Автоматско синхронизирање на домените
- Управување со сертификати SSL
- Автоматска конфигурација на константната SUNRISE

## Задолжинија
Следниот променлив мора да биде дефиниран во вашата `wp-config.php` дафаил:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'ваша_api_ключ');
define('WU_GRIDPANE_SERVER_ID', 'ваши_server_id');
define('WU_GRIDPANE_APP_ID', 'ваши_site_id');
```

## Инструкции за поставување

### 1. Доставете своите GridPane API податоци

1. Најавете се во вашата GridPane административна панел
2. Одете на "Settings" > "API" (Поставки > API)
3. Генерарној API ключу ако не го имате
4. Копирајте го вашиот API ключ

### 2. Доставете своите ID-и за сервер и сайт

1. Во вашата GridPane административна панел, отидете на "Servers" (Сервери)
2. Изберете серверот каде е хостиран вашиот WordPress multisite
3. Забележете Server ID (виден во URL или на страницата со детали за сервер)
4. Одете на "Sites" (Сайтови) и изберете го вашиот WordPress сайт
5. Забележете Site ID (виден во URL или на страницата со детали за сайт)

### 3. Додадете константи во wp-config.php

Додадете ги следните константи во вашата `wp-config.php` дафаил:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'ваша_api_ключ');
define('WU_GRIDPANE_SERVER_ID', 'ваши_server_id');
define('WU_GRIDPANE_APP_ID', 'ваши_site_id');
```

### 4. Вклучете интеграцијата

1. Во вашата WordPress административна панел, отидете на Ultimate Multisite > Settings (Поставки)
2. Направите се на табот "Domain Mapping" (Мапирање на домените)
3. Пролискувајте надолу до "Host Integrations" (Интеграции за хостови)
4. Вклучете интеграцијата со GridPane
5. Кликнете на "Save Changes" (Зачувај промени)

## Како работи

Кога некој домен се мапира во Ultimate Multisite:

1. Интеграцијата испраќа барање на API на GridPane за да го додаде доменот во вашата локација.
2. GridPane автоматски управува со провизијање на SSL сертификати.
3. Кога ќе се откаже мапирање на домен, интеграцијата ќе го откаже доминот од GridPane.

Интеграцијата исто така автоматски управува со константната SUNRISE во вашата wp-config.php датотека, која е потребна за правилно функционирање на мапирањето на домените.

## Управување со константната SUNRISE

Една уникатна карактеристика на интеграцијата со GridPane е што она автоматски ја враќа константната SUNRISE во wp-config.php за да спре конфликти со сопствениот систем за мапирање домени на GridPane. Ова обезбедува дека двете системи можат да работат заедно без проблеми.

## Решавање проблеми (Troubleshooting)

### Проблеми со врската на API
- Проверете дали вашата API ключ е точен
- Проверете дали вашите серверски и локациски ID се правилни
- Осигурајте се дека вашата сметка во GridPane има потребните дозволи

### Проблеми со SSL сертификатите
- GridPane може да им позема неколку време за издавање на SSL сертификати
- Проверете дали вашите домени правилно насочуваат кон IP адресата на вашиот сервер
- Проверете ги поставките за SSL во GridPane за вашата локација

### Домен не е додаден
- Проверете ги логатите на Ultimate Multisite за некои пораки за грешка
- Проверете дали доминот веќе не е додаден во GridPane
- Осигурајте се дека вашите DNS записи за доменота се правилно конфигурирани
