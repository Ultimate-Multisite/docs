# Способности на Theme Builder: Scaffold и Активирање Block Themes

Superdav AI Agent v1.12.0 уводи две моќни способности кои ви овозможуваат да генерирате и доставите кастомни block themes директно од чат интерфејсот.

## Преглед {#overview}

Способите **scaffold-block-theme** и **activate-theme** овозможуваат агентите да:
- Генерираат целосни, готови за продукција блок теми врз основа на вашите спецификации
- Автоматски активираат теми на вашата локација без ручен вмешаток
- Создаваат конзистентна визуелна идентитет преку насочени одлуки за дизајн

## Scaffold Block Theme (Создавање основите на блок темата) {#scaffold-block-theme}

Способноста **scaffold-block-theme** генерира нова WordPress block theme со целосна структура на тема, вклучувајќи:

- `theme.json` конфигурација со design tokens
- Блок шаблони за чести типови распоред (layouts)
- Кастом стилови и варијации на блоковите
- Метадански податоци на темата и декларации за поддршка

### Како да ја повикате {#how-to-invoke}

Во вашиот чат со Superdav AI Agent, можете да побарате генерирање на тема:

```
"Создај блок тема наречена 'Modern Agency' со сина и бела палитра,
sans-serif типографија и професионален распоред"
```

Агентот ќе:
1. Скупчи вашите дизајн преференции преку разговор
2. Генерира целосна структура на темата
3. Создаде сите неопходни датотеки за темата
4. Подготви ја темата за активирање

### Очекувано излезо {#expected-output}

Кога способноста ќе се изврши успешно, ќе видите:

- Потврда дека темата е скафолдирана (scaffolded)
- Име и локација на темата
- Резиме на применетните design tokens (бои, типографија, размери)
- Статус за активирање

Пример за излез:
```
✓ Тема "Modern Agency" успешно скафолдирана
  Локација: /wp-content/themes/modern-agency/
  Бои: Примарна #0066CC, Вторична #FFFFFF
  Типографија: Inter (sans-serif)
  Статус: Готов за активирање
```

## Activate Theme (Активирај Тема) {#activate-theme}

Способностата **activate-theme** префрла вашиот сайт на новоскафолден или постоечен блочен тема.

### Како да ја активирате {#how-to-invoke-1}

По скафолдирање на тема, можете веднаш да ја активирате:

```
"Активирај ја Modern Agency темата"
```

Или активирајте која било постоечка тема:

```
"Префрли се на Twentytwentyfour темата"
```

### Очекуван излез {#expected-output-1}

Кога активацијата е успешна:

- Потврда за активната тема
- Претходно име на темата (за референција)
- URL на сайтот каде темата сега функционира
- Секоја специфична забелешка за поставување на темата

Пример за излез:
```
✓ Темата е успешно активирана
  Активна тема: Modern Agency
  Претходна тема: Twentytwentyfour
  Функционира на: https://yoursite.com
  Забелешка: Проверете ја вашата главна страница за да потврдите изгледот
```

## Работен процес: Скафолдирање и Активирање {#workflow-scaffold-and-activate}

Типичен работен процес комбинира двете способности:

1. **Забавување на генерација на тема**: "Создај блочна тема за моја SaaS приземјувачка страница"
2. **Агентот скафолдира темата**: Генерализира датотеки и дизајнерски токени (design tokens)
3. **Преглед и фино настройка**: Дискутирајте промени во дизајнот ако е потребно
4. **Активирање**: "Активирај ја темата сега"
5. **Проверка**: Посетете го вашиот сайт за да потврдите дека новиот дизајн функционира

## Дизајнерски токени и Персонализација {#design-tokens-and-customization}

Скафолдените теми користат WordPress дизајнерски токени (преку `theme.json`) за:

- **Бои**: Примарна, вторична, акцентна, неутрална палета
- **Типографија**: Серији на фонтови, големини, тежини, високи налици
- **Простор**: Податоци (padding), маргини (margin), скала за размери меѓу елементите (gap scales)
- **Бројкови/Радиуси**: Токени за радиус и ширина
- **Сјајни ефекти (Shadows)**: Нивови на надворешно подигнување

Овие токени се централизирани во `theme.json`, што го прави лесен прилагодување на целата вашата дизајнерска система од една единствена датотека.

## Ограничувања и Забелешки {#limitations-and-notes}

Темите се структурираат во директоријумот `/wp-content/themes/` и мора да следат правилата за именовање на WordPress.
Активирањето бара соодветни дозволи на вашата WordPress локација.
Прилагодени PHP кодови во теми се минимални; користете плагини за сложна функционалност.
Темите базирани на блокови најдобро работат со WordPress 5.9 и подоцна.

## Решавање проблеми (Troubleshooting) {#troubleshooting}

**Темата не се појавува по структурирањето (scaffolding)**
- Проверете дали директоријумот на темата постои и дали има правилни дозволи за пристап.
- Проверете дали `theme.json` е валиден JSON.
- Сигурно проверете дали името на темата не конфликтира со постоечките теми.

**Активирањето не успева**
- Потврдете дека имате администраторски дозволи.
- Проверете дали директоријумот на темата е читав за WordPress.
- Прегледајте WordPress грешки лог файслови за детали.

**Дизајн токените не се применуваат**
- Проверете дали синтаксијата во `theme.json` е правилна.
- Очистите било кои плагини за кеширање (caching plugins).
- Проверете дали вашата верзија на WordPress поддржува токените што ги користите.

## Следен чекори (Next Steps) {#next-steps}

По активирањето на вашата тема, можете да:
- Користите вештината **Design System Aesthetics** за фино настротување на типографијата, боите и размажувањето (spacing).
- Прилагодете стилот на по单个 блокови преку WordPress блок редарот.
- Додадете соодветен CSS во датотеката `style.css` на темата.
- Создадете кастом шаблони за блокови за специфични типови страници.
