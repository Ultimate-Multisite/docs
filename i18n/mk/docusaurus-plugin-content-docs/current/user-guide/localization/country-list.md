# Преупотреба наредбување на списокот земји во Ultimate Multisite

Дропдаун за избор на земја е нареден азбукално врз основа на англиските варијанти на имињата на земјите. Ова може значи дека понекогаш земјата што е домашна за вашата компанија и/или најмногу од вашите клиенти е погрешена во средина на огромниот список на имиња на земјите.

За да ви помогнеме да го исправите ова и да додадете ја вашата земја по изборот на врвот на списокот за избор, можете да користите следниот snippet (можете да го додадете во mu-plugin):

add_filter('wu_get_countries', function($countries) {

// Променете DE со двострениот код на вашата земја

// Пример подолу за Германија.

unset($countries['DE']);

// Променете DE со двострениот код на вашата земја

// и името на името на вашата земја.

// Пример подолу, за Германија.

$countries = array_merge(array( 'DE' => __('Germany', 'ultimate-multisite'), ), $countries);

return $countries;

});
