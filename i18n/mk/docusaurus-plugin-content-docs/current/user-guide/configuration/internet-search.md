# Пребарување на интернет пребарување {#internet-search}

Gratis AI Agent v1.5.0 додава способност за **Интернет пребарување**, која овозможува асистентот да добие актуални информации од веб страницата во текот на разговорот. Ова функционира со помош на [Brave Search API](https://brave.com/search/api/) или [Tavily API](https://tavily.com/).

## Како работи {#how-it-works}

Кога е овозможено интернет пребарување, асистентот може автоматски да прашава вашиот конфигуриран провизиониран извор за пребарување кога ќе определи дека прашањето бара актуални или надворешни информации — на пример, последни вести, живо цена или документација која може да се промени од датумот на обука на модел.

Резултатите се добиваат во реално време и вметнуваат во контекстот на асистентот пред тоа што генерира одговор. Асистентот покажува кога го користе пребарување резултатите за да одговори прашањето.

## Овозможување на интернет пребарување {#enabling-internet-search}

Интернет пребарување бара API клуч од вашиот избрани извор за пребарување. За конфигурација:

1. Одете на **Gratis AI Agent → Settings → Advanced**.
2. Најдете менито (dropdown) **Internet Search Provider** и изберете било **Brave Search** или **Tavily**.
3. Внесете го вашиот API клуч во соодветното поле. Линкови за регистрација се прикажуваат до секое поле.
4. Кликнете на **Save Settings** (Зачувај поставки).

Откако клучот е зачуван, способноста за интернет пребарување автоматски станува достапна за асистентот.

## Brave Search {#brave-search}

### Добивање API клуч за Brave Search {#obtaining-a-brave-search-api-key}

1. Одете на [Brave Search API страницата](https://brave.com/search/api/).
2. Регистрирај се за план. Достапен е беспла ниво со лимит на барања месечно.
3. Копирај го вашиот API клуч од административната табла (developer dashboard) на Brave Search.
4. Внесете го во полето **Brave Search API Key** во поставките на Gratis AI Agent.

### Ограничувања за користење {#usage-limits}

Искок се фактурира од Brave Search врз број на изпратени пребарувања. Секо AI одговор кој предизвикува пребарување се броју како еден пребарувачки. Следете го вашиот користење во [Tavily developer dashboard](https://tavily.com/) за да избегнете неочекивани начини на фактурирање.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 додава **Tavily** како алтернативен провизија за интернет пребарување, нудејќи побогати резултати од пребарувања и напредни способности за истражување.

### Придобивка на Tavily API Клуч {#obtaining-a-tavily-api-key}

1. Обидете се да го посетите [Tavily API страницата](https://tavily.com/).
2. Регистрирајте се за акаунт. Достапен е безбеден план со месечен лимит на барања.
3. Копирајте ја вашата API клуче од Tavily dashboard-от.
4. Вметнете го во полето **Tavily API Key** во поставките на Gratis AI Agent.

### Ограничување на користење {#usage-limits-1}

Користење се фактурира од Tavily врз основа на бројот на извики на API. Секо AI одговор кој предизвикува пребарување се броју како еден извик. Следете го вашето користење во [Tavily dashboard-от](https://tavily.com/) за да избегнете неочекивани начини на фактурирање.

## Деактивирање на интернет пребарување {#disabling-internet-search}

Променете ја API клучето од активното поле за провизија за пребарување и зачувајте го. Способоноста за интернет пребарување повеќе нема да биде нудена на асистентот.

:::note
Интернет пребарување додава забавување на одговорите бидејќи асистентот мора да почека за резултатите од пребарувања пред да генерира одговор. За случаи кои бараат брзо реагирање, размислете дали реалното време пребарување е потребно или дали вграденото знаење на асистентот е доволно.
:::
