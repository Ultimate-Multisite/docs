# Вградени агенти

Gratis AI Agent v1.9.0 се испорачува со пет вградени агенти, секој однапред конфигуриран со фокусиран сет алатки, приспособен системски промпт и почетни предлози усогласени со вообичаени задачи во таа област. Префрлањето меѓу агенти менува што може да прави асистентот и како одговара — без никаква конфигурација од ваша страна. Superdav AI Agent v1.18.0 може да додаде алатки свесни за распоред, записи за потсетници, порти за одобрување и SMS известувања во овие работни текови кога поврзаните интеграции се конфигурирани.

## Што е агент?

Секој агент е именуван конфигурациски профил што комбинира:

- **Алатки** — способностите што агентот смее да ги повика (на пр. Пишувач на содржина има пристап до способности за создавање објави; Студио за дизајн има пристап до способности за CSS и theme.json)
- **Системски промпт** — инструкции што го поставуваат тонот, приоритетите и ограничувањата на агентот
- **Предлози** — однапред напишани промптови прикажани во интерфејсот за разговор за да ви помогнат брзо да започнете

## Пристапување до избирачот на агенти

1. Отворете го панелот **Gratis AI Agent** во администраторската странична лента на WordPress.
2. Кликнете на **иконата за агент** во горниот лев агол на заглавјето на разговорот (иконата се менува за да го одрази активниот агент).
3. **Избирачот на агенти** се отвора како преклоп со формулар-табела. Секој агент е наведен со својата икона, име и опис во една линија.
4. Кликнете на ред со агент за да го активирате. Заглавјето на разговорот веднаш се ажурира.

Можете да префрлате агенти и среде разговор — системскиот промпт на новиот агент стапува во сила од следната порака.

## Петте вградени агенти

### Пишувач на содржина

**Фокус:** Создавање и уредување објави, страници и контакт-формулари.

**Достапни алатки:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Со овозможени интеграции на Superdav AI Agent v1.18.0, конфигуриран календарски контекст, порти за одобрување, потсетници и алатки за SMS известувања може да бидат достапни и за одобрени работни текови.

**Во што е добар:**
- Подготвување на нацрти и објавување блог-објави од бриф или скица
- Создавање групи од целни страници за нов сајт
- Градење формулари за контакт и прашања
- Поставување истакнати слики на објави од URL или пребарување
- Подготвување пораки за следење по настан од конфигуриран Google Calendar контекст, потоа паузирање за одобрување пред испраќање известувања

**Почетни предлози:**
- *Напиши блог-објава од 500 зборови за придобивките од WordPress multisite.*
- *Создај страници За нас, Услуги и Контакт и објави ги.*
- *Додај формулар за прашање за резервација на страницата Контакт.*
- *Подготви потсетник за присутните на утрешниот конфигуриран календарски настан и почекај одобрување пред да го испратиш.*

---

### Градител на сајтови

**Фокус:** Создавање веб-сајт од почеток до крај од еден промпт.

**Достапни алатки:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Со Superdav AI Agent v1.18.0, конфигурирани алатки за управувана услуга, одобрување, потсетници, календар и SMS може да бидат достапни таму каде што администраторите ги овозможуваат.

**Во што е добар:**
- Генерирање повеќефазен план за изградба на сајт за опишан тип бизнис
- Автономно извршување на секоја фаза — структура, содржина, навигација, дизајн
- Закрепнување од грешки среде план без потреба од рачна интервенција
- Инсталирање препорачани plugins како дел од изградбата
- Создавање контакт-формулари директно од интерфејсот за разговор (Superdav AI Agent v1.10.0+)
- Координирање потсетници за лансирање или следење на присутни без дуплирани известувања кога се овозможени порти за одобрување и записи за потсетници

**Почетни предлози:**
- *Изгради портфолио-сајт за фотографија со тип на објава за галерија, страница за резервации и контакт-формулар.*
- *Создај веб-сајт за ресторан со онлајн мени, работно време и формулар за прашање за резервација на маса.*
- *Постави сајт за фриленс консултации со страници за услуги, секција за портфолио и блог.*
- *Додај контакт-формулар на страницата Контакт користејќи го градителот на сајтови.*
- *Откако списокот за проверка за лансирање на сајтот ќе биде одобрен, испрати SMS потсетник до конфигурираниот контакт на засегнатата страна.*

### Студио за дизајн

**Фокус:** Визуелно приспособување — бои, типографија, CSS и блок-обрасци.

**Достапни алатки:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Во што е добар:**
- Применување именувани theme preset-и (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Фино приспособување глобална типографија и палети на бои преку theme.json
- Вметнување приспособен CSS за надминувања специфични за бренд
- Правење снимка од екран на страница и нејзино прегледување за проблеми со дизајнот

**Почетни предлози:**
- *Примени го warm-editorial preset-от и потоа постави ја примарната боја на #2d6a4f.*
- *Направи снимка од екранот на почетната страница и кажи ми што би подобрил.*
- *Создај повторно употреблив hero блок-образец со слика во позадина преку целата ширина и центриран наслов.*

### Менаџер на plugins

**Фокус:** Откривање, инсталирање и управување со WordPress plugins.

**Достапни алатки:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Во што е добар:**
- Препорачување најдобар plugin за опишан случај на употреба
- Инсталирање пакети со способности од регистарот
- Прелистување на достапниот каталог на способности по категорија

**Почетни предлози:**
- *Кој е најдобриот plugin за директориум за членство?*
- *Инсталирај го пакетот со WooCommerce способности.*
- *Прикажи ми ги сите достапни пакети со способности за е-трговија.*

---

### Асистент за поддршка

**Фокус:** Одговарање прашања за содржината на сајтот, поставките и WordPress конфигурацијата.

**Достапни алатки:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Што прави добро:**
- Пребарување на тековните поставки и опции на сајтот
- Објаснување кои типови објави, таксономии и менија се конфигурирани на сајтот
- Одговарање на прашања од типот „што прави оваа поставка?“ преку читање вредности во живо
- Служење како дијагностички слој само за читање пред да се направат промени

**Почетни предлози:**
- *Кои додатоци и поставки се моментално активни на овој сајт?*
- *Наведи ги сите сопствени типови објави регистрирани на овој сајт.*
- *Кои навигациски менија постојат и каде се доделени?*

## Интеграции за автоматизација на Superdav

Кога интеграциите на Superdav AI Agent v1.18.0 се конфигурирани, вградените агенти можат да учествуваат во побезбедни автоматизирани работни текови свесни за распоред:

- **Алатките за читање на Google Calendar** им овозможуваат на агентите да ги прегледаат конфигурираните календари и настани пред да подготват последователна работа.
- **Мапирањето контакти и присутни** помага да се усогласат учесниците во настани со WordPress корисници или познати контакти.
- **Портите за човечко одобрување** ги паузираат чувствителните дејства додека овластен корисник не ги прегледа и потврди.
- **Записите за потсетници** спречуваат дупликат известувања кога закажаните задачи повторно се обидуваат или се повторуваат.
- **TextBee SMS известувањата** испраќаат конфигурирани текстуални пораки само кога SMS акредитивите и дозволите за работниот тек се овозможени.

Препорачан работен тек: побарајте од агентот да ја подготви пораката или дејството, прегледајте го барањето за одобрување, потоа дозволете одобреното дејство да продолжи. За повторливи потсетници, држете го отстранувањето дупликати на потсетници овозможено за истиот настан или контакт да не биде известуван повеќепати.

## Приспособување агенти

Секој вграден агент може да се прошири или замени преку филтерот `gratis_ai_agent_agents`.

### Додавање сопствен системски prompt на постоечки агент

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Регистрирање нов агент

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Новиот агент се појавува во избирачот на агенти веднаш откако филтерот ќе се изврши.

### Отстранување вграден агент

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
