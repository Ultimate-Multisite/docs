# Вградени Агенти

Gratis AI Agent v1.9.0 испраќа пет вградени агенти, секој од нив е предварително конфигуриран со фокусирана сметка на алатки, специфичен систем промпт и почетни предлози подредени на чести задачи во таа област. Префрлување помеѓу агентите менува што асистент може да прави и како реагира — без никаква конфигурација од ваша страна.

## Што е Агент?

Секој агент е именован профил за конфигурација кој комбинира:

- **Инструменти (Tools)** — способностите кои агентот има дозволено да ги повика (на пример, пишувач на содржина има пристап до способности за креирање пост; Дизајн Студио има пристап до CSS и theme.json способности).
- **Систем промпт (System prompt)** — упатства кои ја поставува тонот, приоритетите и ограничувања на агентот.
- **Предлози (Suggestions)** — предварително напишани промптови што се прикажуваат во чат интерфејсот за да ви помогнат брзо да започнете работа.

## Достапување на Изборот на Агенти (Agent Picker)

1. Отворете панелот **Gratis AI Agent** во страничниот бар на администрацијата на WordPress.
2. Кликнете на **иконата на агентот** во горниот лев агол на главниот заглавје (иконата се менува за да ја прикаже активната агента).
3. Отворени ќе биде **Agent Picker** како преласка табела (form-table overlay). Секој агент е листаден со своја икона, име и едноречно описание.
4. Кликнете на ред на агентот за да го активирате. Главниот заглавје се ажурира веднаш.

Можете исто така да префрлите агенти во текот на разговорот — системниот промпт на новиот агент влијае од следното порака.

## Пет Вградени Агенти

### Пишувач на Содржина (Content Writer)

**Фокус:** Создавање и уредување пост, страници и контакт форми.

**Достапни инструменти:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**Што му прави добро:**
- Пишење и објавување блог постови од кратко опишан текст или план
- Создавање групи лендинг страници за нов сайт
- Градење форми за контакт и прашања

**Почетни предлози:**
- *Напишете 500 зборови блог пост за парите на WordPress multisite.*
- *Создадете страници „За нас“, „Услуги“ и „Контакт“ и објавете ги.*
- *Додадете форма за резервација/прашања на Контакт страницата.*

---

### Site Builder (Изгравач на сайт)

**Фокус:** Целосна креација на веб-страница од еден промпт.

**Достапни алатки:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**Што му прави добро:**
- Генерација на план за изградба на сайт во повеќе фази за опишан тип бизнис.
- Извршување на секоја фаза автоматски — структура, содржина, навигација, дизајн.
- Поправка од грешки во текот на планот без потреба од рачно вмешателство.
- Инсталирање препорачани plugin-ови како дел од изградбата.
- Создавање форми за контакт директно од чат интерфејсот (Superdav AI Agent v1.10.0+).

**Почетни предлози:**
- *Изградете сайт за фотографија со тип пост „Галерија“, страница за резервации и форма за контакт.*
- *Создадете веб-страница за ресторан со онлайн мени, работно време и форма за прашања за резервација на маса.*
- *Поставете сайт за фриланс консултанти со страници за услуги, секција за портфолио и блог.*
- *Додадете форма за контакт на Контакт страницата користејќи ја алатката Site Builder.*

---

### Design Studio (Студио за дизајн)

**Фокус:** Визуелна персонализација — бои, типографија, CSS и блокови со обрас.

**Доступни алатки:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Што е добро:**
- Прилагање имената претходни теми (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Фино настройка глобалната типографија и палитрите на бои преку `theme.json`
- Инжектирање кастом CSS за надгради за брендот
- Земање скраиншот на страница и преглед за дистрибутивни проблеми

**Почетни предлози:**
- *Примени претходниот претходник warm-editorial и потоа постави ја основната боја на #2d6a4f.*
- *Земи скраиншот на главната страница и кажи ми што би подобриле.*
- *Создај повторно користен блок шаблон за херо (hero block pattern) со целосен фонов со слика и централно насловен текст.*

---

### Plugin Manager (Управувач на плагини)

**Фокус:** Искушување, инсталирање и управување со WordPress плагините.

**Доступни алатки:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Што е добро:**
- Препорачување на најдобриот плагин за опишан случај употреба
- Инсталирање пакети со способности од регистрирањето
- Преглед на достапниот каталог на способности по категории

**Почетни предлози:**
- *Кој е најдобар плагин за директориум за членство?*
- *Инсталирај го пакетот со способности WooCommerce.*
- *Покажи ми сите достапни пакети со способности за е-комерција.*

---

### Support Assistant (Асистент за поддршка)

**Фокус:** Одговорување на прашања за содржината на сайтот, поставките и конфигурацијата на WordPress.

**Доступни алатки:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Што е добро:**
- Пребарување на моменталните поставки и опции на сайтот.
- Објаснување што се конфигурирани типови постови (post types), таксономии (taxonomies) и мениу.
- Одговорување на прашања како „што прави ова поставка?“ преку прочитање на живи вредности.
- Служување како читачка диагностична слој пред да се направат промени.

**Почетни предлози:**
- *Кои плагини и поставки се активни на овој сайт?*
- *Налистете сите кастом постови типови (custom post types) регистрирани на овој сайт.*
- *Кои навигациски мениу постојат и каде им е назначено?*

## Прилагодување агентите

Секој вграден агент може да се прошири или замени преку филтрите `gratis_ai_agent_agents`.

### Додадење кастом систем промпта на постоечки агент

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Регистрирање на нов агент

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Оптимизира постови и страници за пребарувачки мостови.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'Ти си SEO специјалист. Фокусирај се на оптимизација на клучни зборови, мета описи и структурирани податоци.',
        'suggestions'   => [
            'Прегледај наслов и мета опис на главната страница.',
            'Предложи подобрувања за таг-тагла (title tag) за петте најнови постови.',
        ],
    ];
    return $agents;
} );
```

Новиот агент се појавува веднаш подоцна како избор на агенти, веднаш по тоа што се изврши филтрирањето.

### Се отстранување на вграден агент

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
