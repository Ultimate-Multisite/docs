# Събитие: Регистрација на акаунт преку Zapier {#event-register-an-account-via-zapier}

Во ставата [Интегрирање на Ultimate Multisite со Zapier](zapier.md), разговаравме за тоа како да користиме Zapier за извршување различни дела во Ultimate Multisite врз основа на тригери и настани. Во оваа стаатка, ќе ви покажеме како можете да интегрирате трети страни апликации. Ќе користиме Google Sheets како извор на податоци и ќе испратиме информации во Ultimate Multisite за регистрација на акаунт.

Пред да го поврзете Zapier, отидете на **Ultimate Multisite > Settings > API & Webhooks** и потврдете дека API е активиран. Копирајте ја API Клучот (API Key) и API Тајникот (API Secret) од оваа екранска страница кога Zapier ќе побара податоци за акаунтот на Ultimate Multisite.

![Поставки на API и Webhooks со API Клуч, API Тајна и опции за активирање на API](/img/admin/settings-api-webhooks.png)

Прво, треба да создадете **Google Sheet** под својот Google Drive. Убедете се дека правилно дефинирате секоја колона така што ќе можете лесно да ги мапите податоците подоцна.

По создавањето на Google sheet, можете да се најавите во својот Zapier акаунт и да започнете со креирање на zap.

Под полето за пребарување **"App event"** изберете **"Google Sheets"**.

Потоа за полето "**Event** " изберете **"New spreadsheet row** (Нов ред во таблицата) и притиснете **"Continue"** (Продолжи).

Следната чекор ќе ви побара да изберете **Google Account** каде е зачуван Google Sheet. Значи само се осигурајте дека е наведени правилниот Google акаунт.

Под **"Set up trigger** ", ќе треба да изберете и дефинирате го Google Sheet-от и линијата (worksheet) што ќе ја користите каде ќе доаѓаат податоците. Само продолжи со пополнување на овие информации и притиснете **"Continue"**.

Следно е да го **тестирате вашиот тригер** за да се осигурате дека вашиот Google Sheet е правилно поврзан.

Ако тестот ви успее, ќе видите резултат со некои вредности од вашите таблици. Притиснете **"Continue"** за да продолжиме.

Следећи корак је подесити другу акцију која ће створити или регистровати наслов у Ultimate Multisite-у. У поље за претрагу изаберите "**Ultimate Multisite(2.0.2)** ".

Под полем "**Event** ", изaberте "**Register an Account in Ultimate Multisite** " а затим кликните на дугме "**Continue** ".

Под "**Set up an action** ", ћете видети различите поља доступна за податке о клијенти, чланице, производе и тако нешто. Можете да мапирате вредности у своју Google Sheet-у и придесите их одговарањем пољу где треба да буду попуњени, као што је показано на скрану испод.

Након што мапирате вредности, можете тестирати акцију.
