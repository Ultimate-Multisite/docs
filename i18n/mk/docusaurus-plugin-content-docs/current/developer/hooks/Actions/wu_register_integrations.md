# Акција: wu_register_integrations

Се активира за да им овозможи на интеграциите да се регистрираат.

Јадрото ги регистрира своите интеграции тука.

## Параметри

| Име | Тип | Опис |
|------|------|-------------|
| $registry | `\Integration_Registry` | Инстанцата на регистарот. |

### Од верзија

- 2.5.0
