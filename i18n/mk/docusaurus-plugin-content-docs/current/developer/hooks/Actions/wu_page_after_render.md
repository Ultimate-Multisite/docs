# Action: wu_page_after_render

Овозможува програмерите на plugin да додадат дополнителна содржина откако ќе ја испечатиме страницата

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $page_id | `string` | id на оваа страница |
| $page | `object` | Објектот на страницата. |
