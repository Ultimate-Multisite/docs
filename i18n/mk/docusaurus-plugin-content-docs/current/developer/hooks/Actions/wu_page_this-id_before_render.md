# Action: wu_page_\{$this->id\}_before_render {#action-wupagethis-idbeforerender}

Дозволува развивачите на приклучоци да додадат дополнителна содржина пред да ја отпечатиме страницата.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $page_id | `string` | ID-то на оваа страница. |
| $page | `object` | Објектот на страницата. |
