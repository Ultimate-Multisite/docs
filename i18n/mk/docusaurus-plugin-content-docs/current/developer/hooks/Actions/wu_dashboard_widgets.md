# Action: wu_dashboard_widgets

Овозможува програмерите на приклучоци да додаваат виџети во мрежниот Dashboard панел.

## Параметри

| Име | Тип | Опис |
|------|------|-------------|
| $tab | `string` | Тековниот таб. |
| $screen | `\WP_Screen` | Објектот на екранот. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Инстанца на администраторската страница на Ultimate Multisite. |

### Од верзија

- 2.0.0
