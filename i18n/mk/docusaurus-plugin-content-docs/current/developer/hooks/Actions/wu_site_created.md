# Action: wu_site_created

Се активира откако сајт ќе се создаде за првпат.

Не се активира ако е дуплиран од шаблон.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $data | `array` | Податоците на објектот што ќе се зачуваат. |
| $site | `\Site` | Инстанцата на објектот. |
