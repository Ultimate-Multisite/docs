# Action: wu_dashboard_\{$this->tab\}_widgets {#action-wudashboardthis-tabwidgets}

Дозволува развивачите на приклучоци да додаваат виџети во панелот на мрежниот Dashboard.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $tab | `string` | Тековниот таб. |
| $screen | `\WP_Screen` | Објектот на екранот. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Инстанца на администраторската страница на Ultimate Multisite. |
