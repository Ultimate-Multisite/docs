# Акција: wu_mcp_adapter_initialized

Се активира откако MCP adapter ќе се иницијализира.

Им овозможува на други plugin-и и themes да регистрираат свои способности.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Инстанцата на MCP adapter. |

### Од верзија {#since}

- 2.5.0
