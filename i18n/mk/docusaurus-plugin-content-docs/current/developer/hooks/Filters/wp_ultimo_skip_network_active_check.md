# Филтер: wp_ultimo_skip_network_active_check

Овозможува програмерите да ја прекинат оваа проверка предвреме.

Ова е корисно кога се користат поставки базирани на composer и други прилагодени поставки, како Bedrock, на пример, каде што користењето додатоци како mu-plugins е вообичаено.

- 2.0.0
