# Филтер: wu_add_new_domain_modal_fields

Ги филтрира полињата за модалниот прозорец за додавање нов домен.

Им овозможува на додатоците (на пр. Domain Seller) да ги изменат или заменат полињата за регистрација на домен.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $fields | `array` | Полињата на формуларот. |
