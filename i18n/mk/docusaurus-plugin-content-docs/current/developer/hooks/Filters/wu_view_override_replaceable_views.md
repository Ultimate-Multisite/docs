# Филтер: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Им дозволува на програмерите да додадат дополнителни папки во заменливата листа.

Бидете внимателни, бидејќи дозволувањето дополнителни папки може да предизвика да се вчитаат застарени копии наместо верзиите на Ultimate Multisite.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Листа на дозволени папки. |

### Од верзија {#since}

- 2.0.0
### Извор {#source}

Дефинирано во [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) на линија 50
