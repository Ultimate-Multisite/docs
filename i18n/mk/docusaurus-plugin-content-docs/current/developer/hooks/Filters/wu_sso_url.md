# Филтер: wu_sso_url

Ги филтрира генерираните SSO URL-адреси пред да бидат вратени за меѓудоменски кориснички дејства.

Користете го овој филтер кога интеграција треба да додаде доверлив контекст на SSO врска за суверен тенант или да ја замени URL-адресата на посредникот, притоа зачувувајќи ја проверката на token од Ultimate Multisite.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $sso_url | `string` | Генерирана SSO URL-адреса. |
| $user | `WP_User` | Корисник кој ќе биде автентициран преку SSO посетата. |
| $site_id | `int` | ID на целната локација за посетата. |
| $redirect_to | `string` | Одредишна URL-адреса по успешна SSO проверка. |

### Од {#since}

- 2.13.0

### Извор {#source}

Дефинирано во `inc/sso/class-sso.php`.


## Враќа {#returns}

Филтрирана SSO URL-адреса.
