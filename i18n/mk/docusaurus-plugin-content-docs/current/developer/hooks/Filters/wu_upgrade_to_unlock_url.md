# Филтер: wu_upgrade_to_unlock_url {#filter-wuupgradetounlockurl}

Им овозможува на програмерите да ја променат URL-адресата за надградба за отклучување

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $url | `string` | URL-адресата за надградба. |
| $args | `array` | Модулот и типот на ограничување што треба да се надгради. |
