# Филтер: wu_current_set_site

Им овозможува на програмерите да го изменат стандардното однесување и поинаку да ја постават тековната страница.

## Параметри

| Име | Тип | Опис |
|------|------|-------------|
| $site | `\WP_Ultimo\Models\Site` | Тековната страница што треба да се постави. |
| $ | `self` | Инстанцата на класата Current. |

### Од

- 2.0.9
