# Филтер: wu_sso_enabled

Овозможете/оневозможете ја можноста за единствена најава меѓу домени.

Филтрирајте ја оваа вредност за целосно да ја исклучите единствената најава, или наместо тоа условно да ја овозможите.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | Дали SSO треба да биде овозможено? True за вклучено, вредност налик на false за исклучено. |

### Од верзија

- 2.0.11
### Извор

Дефинирано во [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) на линија 110


## Враќа
Дали SSO е овозможено или не.
