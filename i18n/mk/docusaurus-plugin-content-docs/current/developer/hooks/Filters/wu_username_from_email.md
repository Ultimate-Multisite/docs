# Филтер: wu_username_from_email

Филтрирај корисничко име на нов клиент.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $username | `string` | Корисничко име на клиент. |
| $email | `string` | Адреса на е-пошта на нов клиент. |
| $new_user_args | `array` | Низа од аргументи за нов корисник, можеби вклучувајќи име и презиме. |
| $suffix | `string` | Додај низа на корисничкото име за да биде уникатно. |
