# Филтер: wu_pre_save_settings {#filter-wupresavesettings}

Им овозможува на програмерите да ги филтрираат поставките пред зачувување од Ultimate Multisite.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $settings | `array` | Поставките што треба да се зачуваат. |
| $settings_to_save | `array` | Новите поставки што треба да се додадат. |
| $saved_settings | `array` | Тековните зачувани поставки. |

### Од верзија {#since}

- 2.0.18
