# Филтер: wu_checkout_validation_rules

Овозможува на програмерите на додатоци да ги филтрираат правилата за валидација.

## Параметри

| Име | Тип | Опис |
|------|------|-------------|
| $validation_rules | `array` | Правилата за валидација што ќе се користат. |
| $checkout | `\Checkout` | Класата за наплата. |

### Од верзија

- 2.0.20
