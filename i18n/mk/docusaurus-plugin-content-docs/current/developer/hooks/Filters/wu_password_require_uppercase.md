# Filter: wu_password_require_uppercase

Филтрира дали да се бараат големи букви во лозинките.

## Параметри

| Име | Тип | Опис |
|------|------|-------------|
| $require | `bool` | Дали да се бараат големи букви. Стандардно е true кога правилата се применуваат. |
| $defender_active | `bool` | Дали Defender Pro Strong Password е активен. |
