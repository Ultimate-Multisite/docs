# Филтер: wu_contains_element

Им овозможува на програмерите да ги променат резултатите од почетното пребарување.

Ова е корисно за надворешни градители и слично.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $contains_elements | `bool` | Дали елементот е содржан во содржината. |
| $content | `string` | Содржината што се испитува. |
| $element | `self` | Тековниот елемент. |
| $post | `null\|\WP_Post` | објава за проверка. |
