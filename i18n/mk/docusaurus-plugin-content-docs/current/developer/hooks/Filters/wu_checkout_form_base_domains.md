# Филтер: wu_checkout_form_base_domains

Ги филтрира споделените основни домени од формуларот за наплата што не треба да станат записи за мапирани домени.

Користете го овој филтер кога интеграција обезбедува дополнителни основни домени за полињата **URL на сајтот** во формуларот за наплата. Домените вратени од овој филтер се третираат како споделени хостови за регистрација наместо како приспособени домени по сајт.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $domains | `array` | Споделени основни домени собрани од конфигурацијата на формуларот за наплата. |

### Од верзија {#since}

- 2.13.0

### Извор {#source}

Дефинирано во `inc/functions/domain.php`.


## Враќа {#returns}

Низа од нормализирани основни домени од формуларот за наплата.
