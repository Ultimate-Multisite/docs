# Fluent Forms

Овој додаток овозможува додавање ограничувања за Fluent Forms на Multisite Ultimate во производи, членства или сајтови.
Fluent Forms користи сопствени табели наместо сопствени типови на објави, па затоа вообичаените опции за ограничување не функционираат со Fluent Forms.

## Функции

- Додајте ограничувања за Fluent Forms или Conversational Fluent Forms
- Клиентите добиваат пријателска порака што ги поттикнува да го надградат својот план кога ќе го достигнат ограничувањето
