# Повеќе валути

Ultimate Multisite: Повеќе валути ѝ овозможува на вашата мрежа да прифаќа плаќања во повеќе валути. Автоматски ги конвертира цените користејќи живи или рачни девизни курсеви, ви овозможува да поставите точни цени по производ за која било валута и ја открива претпочитаната валута на секој посетител преку геолокација — сè управувано од еден администраторски Dashboard.

## Клучни функции

- **Автоматска конверзија на девизни курсеви** — цените се конвертираат од вашата основна валута користејќи курсеви преземени од избраниот давател
- **Презапишувања на цени по производ** — поставете точни цени специфични за валута на кој било производ за да ја заобиколите автоматската конверзија
- **Поле за избор на валута при регистрација** — им овозможува на клиентите да ја изберат својата претпочитана валута за време на наплата/регистрација
- **Повеќе даватели на девизни курсеви** — ECB, Frankfurter, Stripe или целосно рачни курсеви
- **Откривање валута врз основа на геолокација** — автоматски однапред ја избира валутата што одговара на земјата на посетителот
- **Администраторски Dashboard** — управувајте со активни валути, девизни курсеви и поставки за давателот од едно место

## Барања

- Ultimate Multisite 2.0.0 или понова верзија
- WordPress Multisite инсталација
- PHP 7.4 или понова верзија

## Инсталација

1. Прикачете ги датотеките на plugin-от во `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Активирајте го plugin-от преку екранот **Plugin-и** во WordPress (мрежно активирање)
3. Одете до **Ultimate Multisite → Повеќе валути** за да го отворите администраторскиот Dashboard
4. Додајте ги валутите што сакате да ги прифаќате и конфигурирајте го вашиот давател на девизни курсеви

## Конфигурација

### Администраторски Dashboard

Администраторскиот Dashboard за повеќе валути (**Ultimate Multisite → Повеќе валути**) е централното место за:

- Додавање, овозможување или оневозможување валути
- Избор на давател на девизни курсеви и освежување на курсевите по барање
- Преглед на тековните курсеви за сите активни валути
- Поставување рачни курсеви кога автоматските даватели не се соодветни

### Даватели на девизни курсеви

| Давател | Опис |
|---|---|
| **ECB** | Европска централна банка — бесплатни дневни курсеви за мрежи базирани на EUR |
| **Frankfurter** | Open-source огледален API на ECB — бесплатен, не е потребен API клуч |
| **Stripe** | Ги користи живите FX курсеви на Stripe — бара активна Stripe интеграција |
| **Рачно** | Директно внесувате курсеви; нема повици кон надворешен API |

Изберете го вашиот давател под **Ultimate Multisite → Повеќе валути → Поставки → Давател на девизни курсеви**. Курсевите може рачно да се освежат во секое време или да се закажат за автоматско ажурирање.

### Презапишувања на цени по производ

Кога автоматската конверзија не е доволно прецизна, можете да поставите точна цена за одредена валута на кој било производ:

1. Отворете го уредувачот на производи (**Ultimate Multisite → Производи → Уреди**)
2. Скролувајте до секцијата **Цени по валута**
3. Внесете ја точната цена за секоја валута што сакате да ја презапишете
4. Зачувајте го производот

Производите без презапишување ја користат конвертираната цена од вашата основна валута.

### Поле за избор на валута при регистрација

Полето за избор на валута може да се додаде во вашиот тек за регистрација/наплата за клиентите да ја изберат својата претпочитана валута при регистрација:

1. Одете до **Ultimate Multisite → Повеќе валути → Поставки**
2. Овозможете **Прикажи изборник за валута при регистрација**
3. По избор, поставете стандардна валута или дозволете геолокацијата да ја одреди однапред избраната вредност

Избраната валута се зачувува со претплатата и се користи за сите последователни наплати.

### Откривање валута врз основа на геолокација

Кога откривањето преку геолокација е овозможено, додатокот ја мапира земјата на посетителот со валута и однапред ја избира во полето за избор на валута. За да овозможите:

1. Одете до **Ultimate Multisite → Повеќе валути → Поставки**
2. Овозможете **Откриј валута од локацијата на посетителот**
3. Мапирајте земји со валути во табелата **Мапирање на геолокација**

Ако не постои мапирање за земјата на посетителот, како резервна опција се користи основната валута на мрежата.

## Често поставувани прашања

### Што се случува ако давателот на девизни курсеви е недостапен?

Додатокот се враќа на последните успешно преземени курсеви. Ако никогаш не биле преземени курсеви, се враќа на рачните курсеви што сте ги конфигурирале. Се препорачува да конфигурирате рачни курсеви како безбедносна мрежа.

### Може ли да користам различни даватели за различни валути?

Не — еден давател е активен во даден момент за сите валути. Користете **Рачно** ако ви е потребна целосна контрола врз поединечните курсеви.

### Дали геолокацијата бара посебен plugin?

Не е потребен надворешен plugin за геолокација. Додатокот го користи вграденото откривање IP-до-земја на WordPress. За поголема точност, посветена услуга за геолокација може да се интегрира преку обезбедените filter hooks.

### Дали презапишувањата по производ се применуваат пред или по данок?

Презапишувањата по производ ја поставуваат цената пред данок во целната валута. Данокот се пресметува врз цената од презапишувањето користејќи ја вашата постојна даночна конфигурација.
