# Дневник на промени за извозникот на сајтови

Верзија: 1.0.6 - Објавено на 2026-05-11
* Ново: Извозите на сајтови сега вклучуваат самоподигнувачки index.php, така што ZIP може да се инсталира на нов хост без посебна инсталација на plugin
* Ново: Мрежниот извоз им овозможува на администраторите да ги извезат сите подсајтови во една архива од администраторската страница за извоз на сајтови

Верзија: 1.0.5 - Објавено на 2026-05-05
* Поправка: Додаден е __callStatic во WP_CLI polyfill stub за да се ракува со немокирани методи како get_config, спречувајќи неуспеси при bootstrap на тестовите

Верзија: 1.0.4 - Објавено на 2025-11-25

* Поправка на компатибилноста со Ultimate Multisite 2.4.

Верзија: 1.0.2 - Објавено на 2025-09-28

* Преименување на префиксот во ultimate-multisite; ажурирање на text domain; зголемување на верзијата.

### Верзија 1.0.1 - Објавено на 2023-08-09 {#version-101---released-on-2023-08-09}

* Подобрено: Сега осигурува дека се добива точната патека на датотеката на датотеката за увоз.
* Поправено: Оневозможен object cache за да се спречат грешки за време на увозот.
* Поправено: Поправен е проблемот со тоа што врската за преземање на importer plugin не работеше.

### Верзија 1.0.0 - Објавено на 2022-12-23 {#version-100---released-on-2022-12-23}

* Поправено: Скокачкиот прозорец на формуларот за извоз не се вчитуваше;
* Поправено: Javascript датотеките не се вчитуваа на страницата за извоз;
* Внатрешно: Нов процес на изработка;

### Верзија 1.0.0-beta.4 - Објавено на 2022-01-13 {#version-100-beta4---released-on-2022-01-13}

* Внатрешно: Додаден генератор на hooks и filters;
* Внатрешно: Додадени WP Ultimo stubs за подобар квалитет на живот за програмери;
