# Филтер: wu_site_exporter_plugin_exclusion_list {#filter-wusiteexporterpluginexclusionlist}

| Име | Тип | Опис |
|------|------|-------------|
| $plugin_list | `array` | Приклучоците што ќе бидат исклучени. |
