# Филтер: wu_apc_get_meta {#filter-wuapcgetmeta}

Им овозможува на програмерите да ги филтрираат конечните вредности и да додаваат нови случаи за обработка.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $default_value | `string` | Стандардната вредност. |
| $meta_type | `string` | Тип на метаподатоците што треба да се преземат. |
| $meta_key | `string` | Име на метаполето. |

### Од {#since}

- 1.4.0
### Извор {#source}

Дефинирано во [`inc/wu-apc-functions.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-admin-page-creator/blob/main/inc/wu-apc-functions.php#L93) на линија 93


## Враќа {#returns}
Нова стандардна вредност.
