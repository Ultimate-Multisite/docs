# Дневник на промени за VAT

Верзија 1.0.7 - Објавена на 2026-02-03

* Поправка: Германските (DE) VAT броеви не поминуваа валидација поради грешки MS_MAX_CONCURRENT_REQ од VIES SOAP услугата. Преминато е од застарениот SOAP API на официјалниот EU VIES REST API за посигурна валидација.
* Поправка: VAT броевите сега се прифаќаат кога VIES услугата е привремено недостапна, наместо погрешно да се одбиваат. Се создава запис во дневник за подоцнежен преглед.
* Поправка: Полето VAT/Даночен ID сега е видливо за сите земји, не само за членките на ЕУ. Даночните ID од земји надвор од ЕУ (на пр. швајцарски CHE броеви) се зачувуваат за приказ на фактура без VIES валидација.
* Поправка: Пребарувањето VAT префикс за Грција (GR), Монако (MC) и Островот Ман (IM) беше расипано поради неточна логика за пребарување во низа.
* Подобрување: Отстранета е непотребната dev зависност wp-cli-bundle што можеше да предизвика фатални грешки кога се користи заедно со Site Exporter додатокот.
* Подобрување: Додаден е сеопфатен пакет unit тестови (53 тестови).

Верзија 1.0.6 - Објавена на 2026-01-25

* Поправка: Домашните B2B трансакции сега правилно наплатуваат VAT. Според правилата за VAT на ЕУ, reverse charge се применува само на прекугранични B2B трансакции, а не кога земјата на клиентот се совпаѓа со земјата на компанијата.

Верзија 1.0.5 - Објавена на 2026-01-22

* Поправка: Додатокот не се вчитуваше или не работеше правилно.
* Промена: Изворот на податоци за VAT стапки е префрлен од euvatrates.com на репозиториумот ibericode/vat-rates за посигурни и активно одржувани податоци.
* Поправка: Исправена е печатна грешка во вредноста на опцијата super_reduced_rates.
* Подобрување: Додадени се проверки за null при ракување со код на земја за да се спречат грешки.
* Промена: Отстранети се вклучените датотеки за превод во корист на автоматски ажурирања преку Traduttore.
