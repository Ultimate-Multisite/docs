# Агенти за поддршка

Управувањето со цела мрежа самостојно не е лесна задача. Како што вашата мрежа почнува да расте, ќе почнете да гледате зголемена побарувачка за поддршка. Добра идеја е да вклучите луѓе што ќе ви помогнат да го намалите обемот на работа. Сепак, давањето целосен пристап/дозвола е ризично и може да ја загрози вашата мрежа. Тука доаѓа овој додаток за агенти за поддршка.

Овој додаток ви овозможува да создадете нов тип WordPress корисник наречен агент за поддршка и ви дава целосна контрола врз тоа какви дозволи има тој корисник. Доделувањето дозвола е едноставно како вклучување прекинувач, а исклучувањето е исто толку лесно ако одлучите да ја отстраните.

## Функции

- Детализиран список на дозволи за WordPress и Ultimate Multisite
- Лесно овозможување и оневозможување пристап
- Создавање неограничен број сметки за агенти

## Инсталација

1. Поставете 'ultimate-multisite-support-agents' во директориумот '/wp-content/plugins/'
2. Активирајте го додатокот преку менито 'Plugins' во WordPress
3. Следете го асистентот чекор по чекор за да го поставите додатокот
