# Креирај мени {#create-menu}

Способноста **Креирај мени** создава WordPress навигациски менија со поддршка за посебни навигациски ознаки одделни од насловите на страниците.

## Преглед {#overview}

Оваа способност ја проширува стандардната функционалност за создавање менија со можност за задавање параметар `navigation_label`. Ова ви овозможува да создавате менија каде што ознаката прикажана во навигацијата се разликува од насловот на страницата, обезбедувајќи поголема флексибилност во структурата на сајтот и корисничкото искуство.

## Параметри {#parameters}

| Параметар | Тип | Задолжително | Опис |
|-----------|------|----------|-------------|
| `name` | string | Да | Име на мени, на пр. `Primary Navigation` |
| `location` | string | Не | Локација на темата на која ќе се додели ова мени, на пр. `primary` |
| `navigation_label` | string | Не | Ознака што ќе се прикаже во навигацијата (различна од насловот на страницата) |

## Повратна вредност {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## Навигациска ознака наспроти наслов на страница {#navigation-label-vs-page-title}

Параметарот `navigation_label` ви овозможува да го одвоите внатрешното име на менито од ознаката што им се прикажува на корисниците:

- **`name`** — Внатрешен идентификатор на мени што го користи WordPress (на пр., "Primary Navigation")
- **`navigation_label`** — Ознаката што им се прикажува на посетителите на сајтот во навигацијата (на пр., "Main Menu")

Ова е корисно кога:
- Вашата внатрешна конвенција за именување се разликува од ознаките што ги гледаат корисниците
- Сакате пократки ознаки во навигацијата отколку во административниот панел
- Треба да поддржите повеќе јазици со различни должини на ознаки
- Градите менија за конкретни региони или кориснички групи

## Примери за употреба {#usage-examples}

### Пример 1: Едноставно мени со навигациска ознака {#example-1-simple-menu-with-navigation-label}

**Промпт:**
```
Создај примарно навигациско мени наречено "Main Navigation" со навигациската ознака "Menu".
```

**Резултат:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### Пример 2: Мени за конкретна локација на тема {#example-2-menu-for-specific-theme-location}

**Промпт:**
```
Создај мени во подножјето наречено "Footer Links" со навигациска ознака "Quick Links" и додели го на локацијата во подножјето.
```

**Резултат:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Интеграција со Theme Builder {#integration-with-theme-builder}

Кога користите Theme Builder, способноста Креирај мени:

1. Автоматски ги открива достапните локации за менија во темата
2. Создава менија со соодветни навигациски ознаки за вашиот дизајн
3. Ги доделува менијата на точните локации во темата
4. Поддржува додавање ставки во менито по создавањето

## Поврзани способности {#related-abilities}

- **`add_menu_item`** — Додајте ставки во постоечко мени
- **`update_menu`** — Преименувајте мени или преназначете го на локација во тема
- **`delete_menu`** — Отстранете мени од вашиот сајт

## Најдобри практики {#best-practices}

- **Користете јасни навигациски ознаки** — одржувајте ги ознаките концизни и описни за корисниците
- **Усогласете ги локациите на темата** — доделете ги менијата на точната локација во темата за правилен приказ
- **Планирајте ја структурата на менито** — одлучете за хиерархијата на вашето мени пред да создадете ставки
- **Тестирајте ја респонзивноста** — проверете дали менијата се прикажуваат правилно на мобилни уреди
- **Локализирајте ги ознаките** — користете различни навигациски ознаки за различни јазични верзии

## Ограничувања {#limitations}

- Навигациските ознаки служат само за приказ; внатрешното `name` сè уште се користи за WordPress идентификација
- Поддршката од темата варира; не сите теми ги поддржуваат сите локации за менија
- Ставките во менито мора да се додадат одделно по создавањето на менито
- Промената на навигациска ознака бара ажурирање на менито
