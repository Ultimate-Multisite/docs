# Mailchimp

Multisite Ultimate: Интеграција со Mailchimp

Интегрирајте ја вашата Multisite Ultimate мрежа со вашиот Mailchimp. Можете да поврзете одредени планови со Mailchimp листи и автоматски да додавате претплатници на тие листи при регистрација.

## Инсталација {#installation}

1. Прикачете 'ultimate-multisite-mailchimp' во директориумот '/wp-content/plugins/'
2. Активирајте го додатокот преку менито 'Додатоци' во вашиот WordPress Network Admin

## Водич за користење {#usage-guide}

Синхронизирањето на е-поштата на претплатниците со вашите MailChimp листи е навистина лесно со нашиот Ultimate Multisite: MailChimp Integration додаток. Овој водич има за цел да ви помогне да поставите сè!

## Добивање API клуч {#getting-an-api-key}

Пред да почнеме да ги разгледуваме различните достапни поставки, ни треба MailChimp API клуч. Ова ќе му овозможи на Ultimate Multisite да комуницира со MailChimp API за да презема листи, групи и да додава е-адреси на тие листи и групи кога корисниците се регистрираат.

Следете го овој водич за да добиете сопствен MailChimp API клуч: [За API клучевите | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_Додајте го вашиот MailChimp API клуч во_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## Режим на интеграција {#integration-mode}

MailChimp Integration додатокот нуди два „режима на интеграција“, што значи дека постојат два различни начини на кои можете да го користите додатокот за синхронизирање е-адреси.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Два режима на интеграција за поголема флексибилност!_

### Режим 1: Повеќе листи {#mode-1-multiple-lists}

Стандардниот режим користи **Повеќе листи**. Овој режим ќе ви овозможи да изберете повеќе MailChimp листи за секој од плановите. Ова е најдобрата опција ако ги сегментирате вашите корисници во различни листи.

На пример, да претпоставиме дека имате 3 различни листи на вашата MailChimp сметка: **Билтен** , **План A** , и **План B**. Сакате сите корисници, без оглед на нивниот план, да бидат вклучени и во листата Билтен, за да не ги пропуштат вашите ажурирања.

За да го постигнете тоа, можете да го користите режимот Повеќе листи за да ги додадете двете листи **Билтен** и **План A** во MailChimp поставките за План A, и **Билтен** и **План B** во MailChimp поставките за План B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Изберете ги стандардните листи за нови сметки и за откажани сметки_

### Режим 2: Една листа + повеќе групи {#mode-2-single-list--multiple-groups}

MailChimp ви овозможува да ја сегментирате вашата листа во помали групи корисници наречени **Групи**. Ова ви овозможува поголема контрола бидејќи можете да испратите кампања до одреден подзбир од вашите претплатници без да мора да креирате повеќе копии од истата кампања на MailChimp (што го отежнува анализирањето на метриките на таа кампања потоа).

На пример, можете да имате една листа наречена **Претплатници на мојата мрежа,** и во неа да имате групи за **План A** и **План B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Изберете ги стандардните листи и групи, ако го користите режимот на интеграција „една листа + повеќе групи“_

## Стандардни листи и групи {#default-lists-and-groups}

Откако ќе изберете кој режим на интеграција има најмногу смисла за вашиот конкретен случај на употреба, време е да ги изберете стандардните листи (и групи, во зависност од режимот). Стандардната листа ќе се користи кога даден план нема избрана конкретна листа.

Можете исто така да ја изберете стандардната листа за откажани сметки. Секогаш кога корисник ќе ја прекине својата претплата и ќе ја отстрани својата сметка од вашата мрежа, неговата или нејзината е-адреса ќе биде преместена во листата поставена за откажани сметки.

## Овозможување двојна претплата {#enabling-double-subscription}

На истата страница со поставки, ќе можете да овозможите **Двојна претплата**. Оваа опција ќе го додаде корисникот и во избраната стандардна листа И во листата избрана за планот на тој корисник. Ако оваа опција не е овозможена, корисникот ќе биде додаден само во листата избрана за планот на корисникот. Ако не е поставена ниту една за тој конкретен план, тогаш корисникот ќе биде додаден во избраната стандардна листа.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_Можете да ги претплатите вашите корисници и на стандардната листа и на листата на планот при регистрација_

## Избирање листи и групи за секој од плановите {#selecting-lists-and-groups-for-each-of-the-plans}

Одете во вашите **Планови** и изберете го планот што сакате да го уредите. Имајте предвид дека во meta-box-от **Напредни опции**, ќе биде достапна нова картичка наречена MailChimp.

На таа картичка, ќе можете да изберете повеќе листи или една листа и повеќе групи (во зависност од режимот на интеграција) за тој план.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Изберете ги листите и групите што сакате да ги користите за секој од вашите планови_

Кога нов корисник ќе се регистрира за тој план, неговата е-адреса ќе биде додадена во избраните листи и групи!
