# Филтер: wu_cap_captcha_config {#filter-wucapcaptchaconfig}

Филтрирајте ја конфигурацијата на Cap Captcha.

Им овозможува на програмерите да ги приспособат поставките за Cap Captcha.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $config | `array` | Низата за конфигурација на Cap. |
| $level | `string` | Нивото на безбедност (fast, medium, max). |
