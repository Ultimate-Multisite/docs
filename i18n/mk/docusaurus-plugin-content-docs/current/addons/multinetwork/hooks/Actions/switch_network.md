# Дејство: switch_network

Се активира кога тековниот мрежен контекст се префрла.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $new_network_id | `int` | ID на мрежата на која се префрла. |
| $old_network_id | `int` | ID на претходно тековната мрежа. |
