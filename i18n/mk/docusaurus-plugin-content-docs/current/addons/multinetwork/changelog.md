# Дневник на измени за Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* Ново: Прегледувач на мрежни шаблони — прелистувајте и прегледувајте мрежни шаблони со панел во живо пред купување.
* Ново: Избирач на мрежни шаблони при checkout — клиентите можат да изберат мрежен шаблон за време на текот за signup checkout.
* Ново: Cross-network magic link SSO — клиентите можат беспрекорно да пристапуваат до подмрежи преку magic-link автентикација.
* Ново: Клонирање на мрежни медиуми — медиумите на веб-локацијата правилно се дуплираат при клонирање мрежи од шаблон.
* Поправка: Клонирањето мрежи сега работи без multi-tenancy додатокот; привилегиите на super admin повторно се доделуваат по клонирањето.
* Поправка: Создавањето мрежи сега е компатибилно со WooCommerce и сите payment gateways.
* Поправка: Бројот на портата сега е вклучен во генерирањето fallback домен.
* Поправка: CSS за избирачот на шаблони е преместен во enqueued stylesheet за правилно прикажување на сите теми.
* Поправка: Заштита од недефиниран `network_id` клуч во опсегот на барањето за веб-локација.
* Поправка: Проверката на мрежни дозволи пред менување контекст спречува грешки во својствата на `wpdb` табелите.
