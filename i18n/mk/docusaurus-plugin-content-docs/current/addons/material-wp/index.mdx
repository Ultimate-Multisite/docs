# Material WP {#material-wp}

Администраторскиот интерфејс на WordPress е ист со години. Но, дали некогаш сте се запрашале како би изгледала Material Design верзија на администраторскиот интерфејс? 

Material WP е администраторска тема развиена за да ви ги даде тој изглед и чувство, помагајќи ви да им обезбедите на вашите клиенти поприлагоден Dashboard.

## Функции {#features}

- Поддршка за прилагоден CSS
- Лесни поставки за стилизирање
- Преуредување на менијата
- Лесен увоз/извоз

## Документација {#documentation}

**Прилагодување на вашиот администраторски панел со Material WP**

## Инсталација {#installation}

1. Поставете 'material-wp' во директориумот '/wp-content/plugins/'
2. Активирајте го приклучокот преку менито 'Приклучоци' во WordPress
