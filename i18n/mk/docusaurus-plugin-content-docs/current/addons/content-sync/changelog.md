# Дневник на промени за синхронизација на содржина

### Верзија 1.1.0 - Објавено на 2025-12-31 {#version-110---released-on-2025-12-31}
* Ново: Поддршка за синхронизација на сите типови објави (објави, страници, приспособени типови објави)
* Ново: Филтрирање по тип на објава во чекорот за избор на содржина
* Ново: Чекор за избор на изворен сајт - синхронизирајте од кој било сајт, не само од главниот сајт
* Ново: Предизбор на подсајт за шаблонски сајтови
* Ново: Извлекување медиуми од post_content за сите типови објави
* Ново: Синхронизација на таксономии (категории, ознаки, приспособени таксономии)
* Променето: Преименувано од "Elementor Template Sync" во "Content Sync"
* Променето: Чекорот "Select Templates" е преименуван во "Select Data"
* Подобрено: Кодот специфичен за Elementor сега се извршува само за типот објава elementor_library

### Верзија 1.0.0 - Објавено на 2025-12-19 {#version-100---released-on-2025-12-19}
* Почетно издание
* Волшебник во повеќе чекори за избор на шаблон и сајт
* Сериска обработка со повратни информации за напредок во реално време
* Целосна синхронизација на медиуми со автоматско премапирање на ID
* Интеграција на поставки со Ultimate Multisite
* Поддршка за исклучување сајтови преку мрежни поставки
* Чистење на кешот на Elementor по синхронизација
* Зачувување на таксономијата на шаблонот
