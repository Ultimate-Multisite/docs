# Çalakî: move_site {#action-movesite}

Piştî ku malperek ji bo torêke nû hatibe veguhastin tê xebitandin.

## Parametre {#parameters}

| Nav | Cure | Danasîn |
|------|------|-------------|
| $site_id | `int` | ID ya malpera ku hatiye veguhastin. |
| $old_network_id | `int` | ID ya tora resen ji bo malperê. |
| $new_network_id | `int` | ID ya tora ku malper hatiye veguhastin bo wê. |

- 1.3.0
