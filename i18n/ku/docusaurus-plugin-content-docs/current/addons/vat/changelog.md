# Guhertinên VAT {#vat-changelog}

Guherto 1.0.7 - Di 2026-02-03 de hate berdan

* Çareserî: Hejmarên VAT yên Almanî (DE) ji ber çewtiyên VIES SOAP service MS_MAX_CONCURRENT_REQ di pejirandinê de têk diçûn. Ji SOAP API ya kevn derbasî EU VIES REST API ya fermî bû ji bo pejirandina bawertir.
* Çareserî: Dema VIES service demkî ne berdest be, hejmarên VAT êdî tên pejirandin, li şûna ku bi xeletî bên redkirin. Têketinek log ji bo vekolîna paşê tê çêkirin.
* Çareserî: Qada VAT/Tax ID êdî ji bo hemû welatan xuya ye, ne tenê ji bo endamên EU. Nasnameyên bacê yên derveyî-EU (mînak hejmarên Swiss CHE) ji bo nîşandana fatûreyê bê pejirandina VIES tên tomarkirin.
* Çareserî: Lêgerîna pêşgira VAT ji bo Yewnanistan (GR), Monaco (MC), û Isle of Man (IM) ji ber lojîka çewt a lêgerîna array şikestî bû.
* Başkirin: Girêdana pêşvebirinê ya nehewce wp-cli-bundle hate rakirin ku dikaribû dema bi addon-a Site Exporter re tê bikaranîn çewtiyên fatal çêbike.
* Başkirin: Komika testên yekeyî ya berfireh hate zêdekirin (53 test).

Guherto 1.0.6 - Di 2026-01-25 de hate berdan

* Çareserî: Danûstandinên navxweyî yên B2B êdî VAT bi rastî lê zêde dikin. Li gor rêzikên EU VAT, reverse charge tenê ji bo danûstandinên B2B yên derbasî-sînor derbasdar e, ne dema welatê kiryarî bi welatê şirketê re yek be.

* Çareserî: Tîpaşa nirxa vebijarka super_reduced_rates hate rastkirin.
