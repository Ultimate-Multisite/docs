# Guherînên Metered Plans

Guherto 1.1.0 - Di 2026-05-05 de hat berdan
- Nû: Hesabkirina AI token ji bo binkarên multisite — şopandin û hesabkirina bikaranîna AI token li ser siteyên mişteriyan bi rêjeyên ji bo her token ku dikarin werin sazkirin
- Nû: Sepandina Connector ji nû ve bi dîtina sînorên dînamîk û nivîsîna rasterast hat avakirin, da ku rastiya demê ya rast li ser hemû connectoran were garantîkirin
- Çakirin: Nûvekirinên tabloya database êdî bi awayekî rast bi pênaseyên BerlinDB schema re hatin hevgirtin, ku têkçûnên nûvekirinê di sazikinên nû de asteng dike
- Çakirin: Paşvegerên nûvekirina database hatin veguherandin bo forma rast, ku têkçûnên bêdeng ên nûvekirinê çareser dike
- Çakirin: Nirxên parçe êdî di qada têketina AI Usage Overage Markup de têne pejirandin
- Çakirin: Çewtiyên mirinî û pirsgirêkên destpêkirina ducarî di barkirina plugin de hatin çareserkirin
- Başkirin: Skrîpta check-env npm hat zêdekirin da ku hawîrdorên pêşdebir bi xwe di xebitandina yekem de xwe saz bikin
