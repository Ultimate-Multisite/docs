# Pir-kirêdari

Bin-malperan li ser server û danegehan belav bike ji bo performans û ewlehiyê

## Nîşanên derketina guhertoya 1.2.0 {#version-120-release-highlights}

Multi-Tenancy 1.2.0 bingeha kirêdara serbixwe zêde dike ji bo herikên xebata veqetandina hilberînê:

- Têketina xweser a kirêdarê ya bê-dewlet bi serdanên SSO yên demkurt û parastî li dijî dubare-lîstinê.
- Veqetandina danegehê û pergala pelan a ji bo her bin-malperê ji bo kirêdarên serbixwe.
- Kapasîteya Hostinger pir-kirêdariyê ji bo amadekirina kirêdarê ya host-hişyar.
- Dergehên piştrastkirina koçberiya serbixwe ji bo kontrolên daneyên kevn û pêvajoya şandina async.
- Girêdana hosta danegehê ya li heman makîneyê ku dikare were mîhengkirin ji bo sazkirinên Bedrock, FrankenPHP, û localhost.
- Başkirinên destpêkirina kirêdarê, nivîskara danegehê, destpêkirina schema, û amadekirina bikarhêneran.
- Rakirina serbixwe ya ewletir ku bawernameyên kirêdarê paqij dike.

Ji bo [herika xebata koçberiya serbixwe](/user-guide/getting-started/sovereign-migration-workflow), [veqetandina pir-kirêdariyê](/user-guide/configuration/multi-tenancy-isolation), û [karên kirêdara serbixwe](/user-guide/administration/sovereign-tenant-operations) rêberên bikarhênerê bibîne.

## Pêdivî {#requirements}

- WordPress 5.3 an bilindtir
- PHP 7.4 an bilindtir
- Multisite Ultimate plugin (çalak)

## Sazkirin {#installation}

1. Pelên pêvekê bar bike nav peldanka xwe ya `/wp-content/plugins/`
2. Pêvekê bi menuya 'Plugins' di WordPress de çalak bike
3. Mîhengên pêvekê di rêveberiya torê de saz bike

## Pêşvebirin {#development}

### Sazkirin {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Testkirin {#testing}

Pêvek çarçoveyeke testkirinê ya berfireh dihewîne:

- **Testên Yekeyî**: Testên bingeha-PHPUnit bi koma testa WordPress
- **Pola Bingehîn a Testê**: Rêbazên alîkar ji bo senaryoyên testa hevpar pêşkêş dike
- **Bergirtina Testê**: Ji bo şopandina bergirtina kodê hatî mîhengkirin
- **Testkirina Multisite**: Test di jîngeha multisite de dimeşin

### Standardên Kodê {#code-standards}

- **Standardên Kodnivîsîna WordPress**: Bi PHPCS têne ferzkirin
- **Lihevhatina PHP 7.4+**: Taybetiyên nûjen ên PHP têne piştgirîkirin
- **Analîza Statîk**: Yekgirtina PHPStan ji bo kontrolkirina cureyan
- **Nûjenkirina Kodê**: Rêgezên Rector ji bo bilindkirinên PHP

### Pergala Avakirinê {#build-system}

- **Pêvajokirina Heyberan**: Biçûkkirin ji bo pelên CSS/JS
- **Werger**: Çêkirina pela POT
- **Pakêt kirin**: Çêkirina arşîvê ji bo belavkirinê
- **Moda Pêşvebirinê**: Heyberên nebiçûkkirî ji bo debugkirinê

## Veavakirin {#configuration}

Pêvek gelek vebijarkên veavakirinê piştgirî dike ku bi rûyê rêveberiya torê dikarin werin gihîştin.

### Veavakirina kirêdara serbixwe {#sovereign-tenant-configuration}

Ji bo kirêdarên serbixwe, berî rêkirina trafîkê ber bi kirêdarê ve, danegeha kirêdarê, kokê pergala pelan, girêdana hosta danegehê, û kapasîteya hostê saz bike. Piştrastkirina koçberiyê piştî her guhertinekê bimeşîne ku bandorê li bawernameyên danegehê, girêdana hostê, rêkirin, an amadekirina bikarhêneran dike.

`localhost` tenê dema ku dem-çalakiya kirêdarê û destûrên danegehê gihîştina socket-a herêmî an heman makîneyê hêvî dikin bi kar bîne. `127.0.0.1` an navê hosta xizmetê bi kar bîne dema ku ew hosta destûrê ye ku ji aliyê peydakera te ve hatî veavakirin.

### Kapasîteya Hostinger {#hostinger-capability}

Kapasîteya Hostinger pir-kirêdariyê bi token-a hevpar a yekgirtina Hostinger dixebite. Berî piştrastkirinê piştrast bike ku domain di hPanel de heye, DNS ber bi Account-a Hostinger ve nîşan dide, û mîhengên danegeha kirêdarê bi hPanel re li hev dikin.

## Hooks û Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-addon-template_init` - Piştî destpêkirina pêvekê tê şewitandin
- `ultimate-multisite-addon-template_loaded` - Piştî barkirina hemû pelên pêvekê tê şewitandin

### Filters {#filters}

- `ultimate-multisite-addon-template_settings` - Mîhengên pêvekê biguherîne
- `ultimate-multisite-addon-template_enabled` - Rewşa çalak/neçalak a pêvekê derbas bike
