# Çalakî: wu_page_added

Destûrê bide pêşvebirên pêvekê ku dema rûpel tên tomarkirin tiştên din jî bixebitînin.

Berevajî wu_page_load, ku tenê dema rûpelek taybet tê dîtin dixebite, ev hook di dema tomarkirinê de ji bo her rûpela rêveberiyê ya ku bi koda WP Ultimo tê zêdekirin dixebite.

## Parametre {#parameters}

| Nav | Cure | Danasîn |
|------|------|-------------|
| $page_id | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | ID ya vê rûpelê. |
| $page_hook | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | Hook-a ku ji bo vê rûpelê hatiye çêkirin. |

- 2.0.0
