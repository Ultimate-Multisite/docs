# Gravity Forms {#gravity-forms}

**Bi hêza Gravity Forms ezmûna qeydkirina bikarhênerên xwe veguherîne**

Avakerê forman ê herî pêşketî yê cîhanê bi Multisite Ultimate re bêkêmasî yek bike da ku formên qeydkirinê yên balkêş û bi veguherîna bilind çêbike, qeydkirinên endamtiyê zêde bike û dahatê bilind bike.
