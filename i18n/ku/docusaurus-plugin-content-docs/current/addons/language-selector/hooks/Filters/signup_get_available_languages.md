# Parzûn: signup_get_available_languages

Lîsteya zimanên berdest ji bo qeydkirinên site yên front-end parzûn dike.

Ger arrayeke vala bidin vê hookê, derketina vê mîhengê li ser forma qeydkirinê dê neçalak bibe, û dema site tê çêkirin zimanê standard dê were bikaranîn. Zimanên ku berê nehatine sazkirin dê bên rakirin.

## Parametre

|------|------|-------------|
| $available_languages | `array` | Zimanên berdest. |
