# Fluent Forms

Ev pêvek dihêle ku hûn sînorên Fluent Forms li ser Multisite Ultimate di hilber, endamtiyan an malperan de zêde bikin.
Fluent Forms li şûna cureyên nivîsên taybet tabloyên taybet bi kar tîne, ji ber vê yekê vebijarkên sînorkirina asayî bi Fluent Forms re naxebitin.

## Taybetmendî {#features}

- Sînorên ji bo Fluent Forms an Conversational Fluent Forms zêde bikin
- Dema mişterî digihîjin sînorî, peyameke dostane dibînin ku wan teşwîq dike planê xwe bilind bikin
