# Parzûn: wu_admin_notices

Destûr bide pêşdebirin ku agahdariyên rêveberiyê yên ji aliyê Ultimate Multisite ve hatine zêdekirin parzûn bikin.

## Parametre {#parameters}

| Nav | Cure | Danasîn |
|------|------|-------------|
| $notices | `array` | Lîsteya agahdariyan ji bo wê panelê ya taybet. |
| $all_notices | `array` | Lîsteya agahdariyên hatine zêdekirin, li gor panelan veqetandî. |
| $panel | `string` | Panela ku agahdarî jê werin wergirtin. |
| $filter | `string` | Heke agahdariyên ku dikarin bên girtin hatibin parzûnkirin. |
| $dismissed_messages | `array` | Lîsteya mifteyên agahdariyên hatine girtin. |
