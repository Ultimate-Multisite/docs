# Filter: wu_get_site_domain_and_path

Destûr dide pêşdeberan ku cotên domain/path biguherînin.

Ev dikare ji bo gelek tiştan kêrhatî be, wek pêkanîna cureyek çareseriya staging, serverên cuda, hwd.

## Parametre

| Nav | Cure | Rave |
|------|------|-------------|
| $d | `object` | Objekta niha ku mifteyên domain û path dihewîne. |
| $path_or_subdomain | `string` | path/subdomain-a resen ku ji fonksiyonê re hatiye şandin. |
