# Parzûn: wu_bypass_unset_current_user {#filter-wubypassunsetcurrentuser}

Destûrê bide pêşdebirvanan ku codeya rakirina bikarhênerê heyî derbas bikin.

Ger tiştek ji null cuda were vegerandin, ew ê rakirina bikarhênerê heyî yê têketî derbas bike. Ev dikare di hin rewşan de bikêr be, wek mînak dema ku bi malperên jêr re tê xebitandin ku wek panelên rêveberiyê tên bikaranîn.

## Parametre {#parameters}

| Nav | Cure | Danasîn |
|------|------|-------------|
| $null_or_bypass | `mixed` | Null ji bo berdewamkirinê, her tiştekî din ji bo derbaskirina wê. |
| $current_user | `false\|\WP_User` | Objekta bikarhênerê heyî. |
