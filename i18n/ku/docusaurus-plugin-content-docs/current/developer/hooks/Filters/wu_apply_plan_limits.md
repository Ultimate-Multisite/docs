# Filter: wu_apply_plan_limits {#filter-wuapplyplanlimits}

Rê bide pêşdebirên plugin ku sînor bi kurtî derbas bikin.

Tu dikarî vê filter bikar bînî da ku berî ku her sînorek dest pê bike, code ya dilxwaz bixebitînî. Ger filter ya te her nirxeke rastî vegerîne, pêvajo dê berdewam bike; ger her nirxeke nerastî vegerîne, code dê vegere û tu hookên jêrîn nayên xebitandin.
