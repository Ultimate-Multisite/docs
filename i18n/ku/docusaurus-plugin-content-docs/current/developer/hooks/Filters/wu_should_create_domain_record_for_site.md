# Parzûn: wu_should_create_domain_record_for_site

Parzûn dike ka Ultimate Multisite divê ji bo malperek nû hatî çêkirin tomarek domainê biafirîne an na.

Vî parzûnê bikar bîne da ku çêkirina xweber a tomara domainê ji bo malperên ku domaîna bingehîn a forma checkoutê ya hevpar, hostek hundirîn, an domainê ku entegrasyonek din wê cuda bi rê ve bibe, asteng bikî an paş bixe.

## Parametre {#parameters}

| Nav | Cure | Şirove |
|------|------|-------------|
| $create | `bool` | Ka divê tomara domainê were çêkirin an na. |
| $site | `WP_Site` | Objekta malpera nû hatî çêkirin. |

### Ji guhertoya {#since}

- 2.13.0

### Çavkanî {#source}

Di `inc/functions/domain.php` de hatiye pênasekirin.


## Vedigerîne {#returns}

Boolean ku nîşan dide ka tomara domainê were çêkirin an na.
