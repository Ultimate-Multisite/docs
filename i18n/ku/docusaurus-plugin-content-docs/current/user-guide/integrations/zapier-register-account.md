# رویداد: از طریق Zapier حساب ایجاد کردن

در مقاله [ادغام Ultimate Multisite با Zapier](zapier.md)، درباره نحوه استفاده از Zapier برای انجام کارهای مختلف در Ultimate Multisite بر اساس محرک‌ها (triggers) و رویدادها صحبت کردیم. در این مقاله، به شما نشان خواهیم داد که چگونه می‌توانید اپلیکیشن‌های شخص ثالث را ادغام کنید. ما از Google Sheets به عنوان منبع داده استفاده می‌کنیم و اطلاعات را برای ثبت حساب در Ultimate Multisite ارسال می‌کنیم.

قبل از اتصال Zapier، به **Ultimate Multisite > Settings > API & Webhooks** بروید و مطمئن شوید که API فعال است. وقتی Zapier از شما درخواست اعتبارنامه حساب Ultimate Multisite کرد، کلید API (API Key) و راز API (API Secret) را از این صفحه کپی کنید.

![تنظیمات API و Webhooks با کلید API، راز API و گزینه فعال کردن API](/img/admin/settings-api-webhooks.png)

ابتدا، باید یک **Google Sheet** در Google Drive خود ایجاد کنید. مطمئن شوید که هر ستون را به درستی تعریف می‌کنید تا بتوانید بعداً داده‌ها را به راحتی با هم مطابقت دهید (map).

بعد از ساخت گوگل شیت، می‌توانید وارد حساب Zapier خود شوید و شروع به ساخت یک zap (کار خودکار) کنید.

در قسمت جستجو برای **"App event"** گزینه **"Google Sheets"** را انتخاب کنید.

سپس در قسمت "**Event** " گزینه **"New spreadsheet row"** (ردیف جدید شیت) را انتخاب کرده و روی **"Continue"** کلیک کنید.

مرحله بعدی از شما می‌خواهد یک **Google Account** (حساب گوگل) که **Google Sheet** در آن ذخیره شده است را انتخاب کنید. پس فقط مطمئن شوید که حساب گوگل صحیح مشخص شده باشد.

در قسمت **"Set up trigger"**، باید گوگل شیت و ورکشیت (Worksheet) مورد نظر خود را که داده‌ها از آن می‌آید، انتخاب و مشخص کنید. کافی است این موارد را پر کرده و روی **"Continue"** کلیک کنید.

مرحله بعدی برای **"test your trigger"** (تست محرک خود) است تا مطمئن شوید گوگل شیت شما به درستی متصل شده است.

اگر تست شما موفقیت‌آمیز بود، باید نتیجه‌ای ببینید که مقادیری از شیت‌های شما را نشان می‌دهد. برای ادامه روی **"Continue"** کلیک کنید.

پێشین هەنگاوە دوایەکە کە دەبێت ئەو کارە دیاری بکەیت کە بەکارهێنەرێک یان ئەکاونتێکی نوێ دروست دەکات لە Ultimate Multisite دا. لە فیلدی گەڕان (search field) ، "**Ultimate Multisite(2.0.2)** " هەڵبژێرە.

لە فیلدی "**Event** "دا، هەڵبژێرە "**Register an Account in Ultimate Multisite** " پاشان دابەزێنە دایەنێ "**Continue** ".

لە خاڵی "**Set up an action** "دا، دەتوانیت فیلدەکانی جیاواز بۆ داتاکانی کڕیار، ئەندازییەکان (memberships)، بەرهەمەکان و هتد لەخۆ بگرێت ببینیت. دەتوانیت ئەو بەهایانەی لە Google Sheet دا هەڵبژێریت و پێشان بدەیت لەو فیلدەی گونجاوەی کە دەبێت داتاکان لەوێ پڕ بکرێنەوە، وەک لە وێنەی خوارەوە نیشان دەدرێت.

پاش ئەوەی بەهاکان هەڵبژێریت، دەتوانیت ئەو کارە (action) یەک تاقیبکەیت (test).
