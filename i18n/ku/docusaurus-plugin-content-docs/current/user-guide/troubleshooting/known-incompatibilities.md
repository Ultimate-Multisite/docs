# ناسازکارییە دیارشترا لەگەڵ پلاتینەکان

پاش جێهێشتنی و چالاککردنەوەی Ultimate Multisite، دەتوانیت ڕووبوونەکان بەهۆی ناسازکارییەکان لەگەڵ پلەکان (plugins) دیکە ڕووبدات.

ئێمە بە هەموو وردبینی نوێیەکە کەم دەکەینەوە، بەڵام ئەگەر چەند پلەیەکی لێرە لیست بکرێت، پێویستە دەبێت ئەو پلە یانەیەک لەناوچوون (deactivate) بکەیت بۆ ئەوەی تۆڕی Ultimate Multisite بە تەواوی کار بکات.

لێرەدا لیستێکیان و ئەوە چی دەبێت کە دەتوانیت لەم حاڵەتەدا بکەیت:

**چەند پلەیەک لە WPMU DEV** هەندێک لە پلەکان کە دوووە لەلایەن WPMU DEV بەردەوام نیشاندراون، ناسراون کە ڕێڕەوی تۆمارکردنی Ultimate Multisite لە هەنگاو کۆتاییدا دەستپێدەکات (کە بەکارهێنەر بتوانێت ئەکاونتی دروست بکات). لێرە لیستێکی ئەو پلەکان هەیە کە ئەم کێشە دروست دەکەن:

  * Signup Language: _Ultimate Multisite وەک پلاتینێکی زیادە دەبێت کە هەمان کارکردن بە ناوی Ultimate Multisite: Language Switcher_ پێشکەش دەکات.

  * Set Blog Description on Blog Creation: _ئەم کارە لە خودی Ultimate Multisite دا دروستکراوە، بۆیە پێویستی بە پلەیەکی دیکە نییە._

  * New Blog Templates: _ئەم کارە لە خودی Ultimate Multisite دا دروستکراوە، بۆیە پێویستی بە پلەیەکی دیکە نییە._

  * Set Password on Multisite Blog Creation: _ئەم کارە لە خودی Ultimate Multisite دا دروستکراوە، بۆیە پێویستی بە پلەیەکی دیکە نییە._

  * Signup TOS: _ئەم کارە لە خودی Ultimate Multisite دا دروستکراوە، بۆیە پێویستی بە پلەیەکی دیکە نییە._

  * Anti-Splog: سەرەڕای ئەوەی کاریگەرتر نییە، ئەمەش _لەناو Ultimate Multisite دا دروستکراوە._

**تێبینی**: هەموو پلەکانی تر لە WPMU Dev دەتوانیان بە شێوەیەکی ئاسایی لەگەڵ Ultimate Multisite بەکاربهێنرێت. نموونەکان: _Smush_، _Forminator_، _Defender_ و هتد.
