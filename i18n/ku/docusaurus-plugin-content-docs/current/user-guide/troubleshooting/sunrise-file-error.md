# هەڵەی دامەزراندنی فایلەکە {#error-installing-the-sunrise-file}

فایلەکەی `sunrise.php` فایلی تایبەتە کە وۆرسپریډ (WordPress) لە کاتی دەستکردەوەی خۆی لێی دەگەڕێت. بۆ ئەوەی وۆرسپریډ بتوانێت ئەم فایلە بەدەست بهێنێت، دەبێت لە ناو دۆلەی **wp-content** بێت.

کاتێک Ultimate Multisite چالاک دەکەیت و پڕۆسەی ڕێنمایی (setup wizard) وەک ئەو یەکەی کە لە سکرین شاتەکەدا هەیە، Ultimate Multisite هەوڵ دەدات فایلەکەی `sunrise.php` بۆ دۆلەی wp-content کۆپی بکات.

<!-- سکرین شاتی بەدوایە: لاپەڕەی ڕێنمایی کارکردن کە هەنگاو دامەزراندنی sunrise.php دەردەخات -->

زۆربەی کات، ئێمە دەتوانین سەرکەوتوو دەکەین فایلەکەی کۆپی بکەین و هەموو شتێک بە باشی کار دەکات. بەڵام ئەگەر شتێک بە دروست نەبێت (وەک ڕێزماندنی دۆلەی فایل، بۆ نموونە)، دەتوانیت لەو دۆخێکدا ببیت کە Ultimate Multisite ناتوانێت فایلەکە کۆپی بکات.

ئەگەر پەیامی هەڵەی Ultimo وەربگریت، دەبینیت کە بەم شێوەیە ڕوودەدات: **Sunrise copy failed** (کۆپکردنەوەی Sunrise شکستخۆدا).

<!-- سکرین شاتی بەدوایە: پەیامی هەڵەی نیشاندەرەوە کە کۆپی کردنەکە شکستخۆدا -->

بۆ چارەسەرکردنی ئەمە، دەتوانیت ببیتە ناو دۆلەی فایلی wp-ultimo و فایلەکەی `sunrise.php` کۆپی بکەیت و لەناو دۆلەی wp-content یەکەت چێبسەری (paste) بکەیت. دوای ئەوە، لاپەڕەی ڕێنماییەکە دەگەڕێنەوە و تاقیکردنەوەکان دەبێت سەرکەوتوو بن.

<!-- سکرین شاتی بەدوایە: مۆدەر فایلی کە sunrise.php لەناو دۆلەی wp-ultimo plugin کۆدا نیشان دەدات --> لە هەر حاڵەتێکدا، ئەمە پێویستی بە چاککردنێکی گشتی بۆ ڕێزماندنی دۆلەی فایلكردن هەیە بۆ ئەوەی لە داهاتوودا کێشەیەک نییە هەبێت (تەنها لەگەڵ Ultimate Multisite نەک بە پلاگین و قالبەکانی دیکەش).

**Health Check tool** کە بەشێکە لە وۆرسپریډ (دەتوانیت بە ڕێگەی **admin panel > Tools > Health Check** دەربکەیت)، توانای ئەوەی هەیە بڵێیت ئایا ڕێزماندنی دۆلەی فایلكردن بە ئەو بەهایانە کراوەتەوە کە دەتوانن کێشەیان بۆ وۆرسپریډ دروست بکەن یان نا.
