# تێگەیشتن لە داش보اردی Ultimate Multisite {#understanding-the-ultimate-multisite-dashboard}

پێش ئەوەی ئێمە زیاتر لە ڕێکخستنە پێشکەوتوو و لایەنە کارکردنی Ultimate Multisite ببینین، بیستوورەکانمان بە داشبۆردارەکە.

وەک چەندین پلاتینی WordPress، Ultimate Multisite یەکێکە کە دەستوورێکی مێنیوی نێژمێتی (network-level menu entry) بە ناوی **Ultimate Multisite** دروست دەکات و ژمارەیەکی زیرەکی مێنیوی خوارەوە لەسەر.

## ویجێتاڵەکانی داش보اردی تۆڕ (Network Dashboard Widgets) {#network-dashboard-widgets}

Ultimate Multisite چەندین ویجێتاڵی کارآمدی دەداتە ناو داشبۆرداری نێژمێتی. ئەم داشبۆردارە بە گەڕانەوە بۆ **My Sites → Network Admin → Dashboard** دەبینراوە. ئەم ویجێتاڵانە دیزاین کراون بۆ ئەوەی دەستگەیشتنی ئاسایی بۆ زانیاری و کارە باوەکان بۆ بەڕێوەبەری تۆڕ (network administrator) بێت.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### هەنگاوە یەکەمەکان (First Steps) {#first-steps}

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

ئەم ویجێتاڵە لەسەر جێگیرکردنی نوێی Ultimate Multisite دەردەکەوێت و بە بەڕێوەبەری تۆڕ یادەوەری دەکات بۆ تەواوکردنی ئەرکەکانی دیکە وەک ڕێکخستنی شێوازی پامێرکردن (payment method) و دروستکردنی خاڵی تاقیکردنەوە.

### پوختەکان (Summary) {#summary}

![Summary widget](/img/admin/network-dashboard-summary.png)

ویجێتاڵی Summary دەربارە لەسەر ژمارەی ئەندامانی تۆمارکردن و دراوەیان بۆ ئەو ڕۆژانە. MRR (Monthly Recurring Revenue) ئاماژەیە بۆ کۆی گشتی پێشبینیکراوی لەسەر کڕیاران بە یەکێک لەو ئەوانەی ماوەی کارپێکردنی دراوە هەیە.

### ڕێڕەوی چالاکی (Activity Stream) {#activity-stream}

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

ئەم ویجێتاڵە دەربارە لەسەر ڕوودۆخەکان لە Ultimate Multisite دا. ئەو ڕوودۆخە پێکهاتوون: تۆمارکردن، لێبوردن (cancellations)، دروستکردنی سایت، دیاریکردنەوە و چالاکیەکانی دیکە.

تکایە بۆ وەسفی وردتر بەشی Events لە dokumantacija بخوێنەرەوە.

### ئێستا چی دەبێت (Right Now) {#right-now}

![Right Now widget](/img/admin/network-dashboard-right-now.png)

ئەم ویجێتی یارمەتیدەر پوختەیەک لە بەکارهێنەران و شوێنەکان لە تۆڕەکەدا نیشاندێت. پێشتر، ئاسایشییەکان لە سەرەوە ڕێگە دەدەن کە بە چاوەڕوانییەک دەتوانیت شوێن یان بەکارهێنەرێکی نوێ دروست بکەیت.

### هەواڵ و گفتوگۆکان (News and Discussions) {#news-and-discussions}

![News and Discussions widget](/img/admin/network-dashboard-news.png)

ئەم ویجێتیە نوێترین هەواڵەکانی Ultimate Multisite دەگەڕێتەوە و نیشان دەدات. بۆ زانیارییەکانی پێداچوونەوەکان، ئەو کێشەی گرنگان (bugs)، و چارەسەرە ئاسایشییە ئامانج بکە بە چاوەڕوانی ئەم ویجێتییە.

## داشبۆرد Ultimate Multisite (Ultimate Multisite Dashboard) {#ultimate-multisite-dashboard}

لەدا کە داشبۆردی نۆڕمەڵەی تۆڕەکە زانیارییە گشتی دەدات، داشبۆردی Ultimate Multisite (کە لە سەرەوەی مێنیو Ultimate Multisite دایە) زانیارییەکانی خزمەتگوزارییەکە دەدات.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### گەشەکردنی دراوەی بازنەی ماوەیە (Monthly Recurring Revenue Growth) {#monthly-recurring-revenue-growth}

چارتەکە بۆ دراوەی بازنەی ماوەیەکان (MRR) پاشکەوتنی مانگانەی دراوە لەسەر ١٢ مانگی زمانی نیشان دەدات.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

دراوەی نوێ بە ڕەنگی دوورنیش (blue) دیاری دەکرێت و لێدانەوەکان بە ڕەنگی سێدار (red) نیشان دەدرێن.

### پشتگیریکردنی خاڵەکانی کاتژمێریی زمانی (Date-range Support) {#date-range-support}

لە ڕێگەی ژمارەی کڕیاران و چالاکی، ئامارەکان دەکرێت زۆر بێز بن. پشتگیریکردنی ناوەڕۆکەکە کە لە داشبۆردەکەدا دروستکراوە، ڕێگە بە بەڕێوەبەری تۆڕەکە دەدات تەنها سەرنجی لەسەر ئەو زانیارییە و ئەو پۆلەی کاتی پێویست بێت، وەربگرێت.

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

پێوەرە کاتژمێریەکە ماوەی ئەو زانیارییانە دەبێت کە لە ویجێتاوەکانی خوارەوە نیشان دەدرێن، لەوانەش دراوەکان، ئەندامنووسی نوێ، تۆمارکردن بەپێی وڵات، تۆمارکردن بەپێی فۆرم، و زیاترین شوێنەکانی سەردان.

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
بۆ ئەوەی ویجێتی **Most Visited Sites** کار بکات، لە ڕێگەی Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting دا، کۆنترۆڵی باوونی سەردان (visit counting) چالاک بکە.
:::
