# فرمەکان بۆ دروستکردنی سایت

Superdav AI Agent v1.10.0 دەتوانێت بە شێوەیەکی ڕاستەوخۆ لە ناو چاتەکەدا، بە بەکارهێنانی Site Builder agent، فرمەڵەکانی پەیوەندیکردن (contact forms) دروست بکات. ئەمەش وەک ئەوە دەبێت کە بتوانید فرمەڵەکان بە تەواوی کارکردن بۆ هەر پەیامێک زیاد بکەن، بدون ئەوەی لە چاتەکە بڕۆیتەوە.

## گشتیاری

قابلی دروستکردنی فرمێل پەیوەندیکردن ی Site Builder (`create_contact_form`) بە شێوەیەکی ئۆتۆماتیکی پلە پڵۆگ (plugin) کە دامەزراوەکەت دەدات، ناسینێت و لەسەر بنەمای تواناکانی ئەو پڵۆگە، فرمێل پەیوەندیکردن دروست دەکات. پڵۆگەکانی فرمێل پشتیوانیی ئەمانە:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

ئاسستنت بە شێوەیەکی ئۆتۆماتیکی باشترین پڵۆگی فرمێل لەسەر سایتەکەت دەبینێت و فرمێل پەیوەندیکردنێک دروست دەکات کە تایبەتە بەو پڵۆگەی دیاریکراو.

## دروستکردنی فرمێل پەیوەندیکردن

بۆ دروستکردنی فرمێل پەیوەندیکردن بە بەکارهێنانی Site Builder:

1. **Gratis AI Agent** چات پینێلەکە لە ئادمن پۆرت ووردپرسدا بکەرەوە.
2. دەربکەووە بۆ agent ی **Site Builder** بە ڕاکێشانەوەی ئای کۆنتر (agent icon) لە سەری چاتەکە.
3. ئەو فرمێل پەیوەندیکردنەی دەتەوێت دروست بکەیت، وەسفی بکەیت. بۆ نموونە:

   > "فرمێل پەیوەندیکردنی زیاد بکە بەشی ناوی، ئیمیل، پەیام و ژمارەی تەلەفۆن."

   یان سادە بڵێ:

   > "فرمێل پەیوەندیکردنی بۆ پەیجی Contact دروست بکە."

4. Site Builder فرمێل پەیوەندیکردنەکە دروست دەکات و کۆدێکی کورت (shortcode) دەدات کە ئامادەیە بۆ زیادکردن (embed).

## بەکارهێنانی کۆدێکی کورت دروستکراو

پاش ئەوەی Site Builder فرمێل پەیوەندیکردنەکە دروست کرد، ئەو کۆدێکی کورت دەداتەوە (بۆ نموونە، `[contact-form-7 id="123"]`). دەتوانیت:

1. **لە پەیامێک یان پۆستێکدا زیاد بکەیت** — کۆدەکە بکۆ بە وەرگرتن و لە هەر پەیامێک یان پۆستێکدا بەکاربهێنە، بە بەکارهێنانی block editor یان classic editor.
2. **بە بەکارهێنانی Site Builder زیاد بکەیت** — داوای لای Site Builder بکە کە فرمێلەکە ئۆتۆماتیکی لەسەر پەیجی دیاریکراو زیاد بکات:

   > "فرمێل پەیوەندیکردنی بۆ پەیجی Contact زیاد بکە."

٣. **لە قالبێکدا بەکاری بهێنە** — ئەگەر لە PHP ئارامی هەبوویت، دەتوانیت ئەو shortcode یە بۆ فایلێکی template ی تێم (theme) زیاد بکەیت.

## گۆڕینی فۆرمەکە (Contact Form Customizing)

پاش ئەوەی Site Builder فۆرمەکە دروست کرد، دەتوانیت زیاتر گۆڕانکاری لەسەر بکەیت:

### بە ڕێگەی ئۆتۆماتیک چات (Via the Chat Interface)

بپرسە لە Site Builder بۆ گۆڕینی فۆرمەکە:

> "فۆرم کنتاکتەکە به‌روزرسیک بکە بۆ زیادکردنی فیلدی بابەت (subject field) و دروستکردنی فیلدی پەیام (message field) بە پێویستی."

Site Builder فۆرمەکە دەبۆزێنێتەوە و shortcode ی گۆڕاوەکە دەداتەوە.

### بە ڕێگەی ئۆتۆماتیک ئەدمن ئۆتۆماتیکی پلەگینی فۆرم (Via the Form Plugin's Admin Interface)

دەتوانیت فۆرمەکە ڕاستەوخۆ لە تنظیماتەکانی پلەگینی فۆرمەکەت گۆڕیت:

١. بچۆ بۆ **Contact Form 7** (یان پلەگینی فۆرمەکەی کە دامەزراوە) لە ئەدمن پۆرت وۆردپرسدا.
٢. فۆرمەکە بدۆزەوە کە لەلایەن Site Builder دروستکراوە.
٣. فیلدەکانی فۆرم، ئەو ڕێزموونانەی پشتڕاستکردن (validation rules)، و ئیمەیڵ نوێکردن (email notifications) بە پێویستی دەستکاری بکەیت.

## پلەگینی فۆرم و ناساندنی (Form Plugins and Compatibility)

Site Builder بە شێوەیەکی خۆکار دەدۆزێتەوە کە کام پلەگینی فۆرم لەسەر سایتەکەت دامەزراوە و بۆ دروستکردنی فۆرم کنتاکتەکە لەو پلەگینە بەکاری دێت. ئەگەر چەندین پلەگینی فۆرم دامەزراون، Site Builder پێش ئەم ڕێژەیە پێشنیار دەکات:

١. Contact Form 7
٢. WPForms
٣. Fluent Forms
٤. Gravity Forms

ئەگەر هیچ یەک لەم پلەگینان نەدابن، Site Builder پێش دروستکردنی فۆرم کنتاکتەکە، پێشنیار دەکات کە یەکێکیان دامەزراو بکەیت.

## ئیمەیڵ نوێکردن (Email Notifications)

فۆرمەکانی کنتاکت کە لەلایەن Site Builder دروست دەکرێن بە شێوەیەکی پیش‌تەوا، بۆ ئەوەی ئیمەیڵی نوێکردن بۆ ئیدارەی سایتەکە دابنێت. دەتوانیت ئیمەیڵی گیروانەکەی و پەیامی نوێکردنی گۆڕانکاری لەویش بگۆڕیت:

١. بچۆ بۆ ئەدمن ئۆتۆماتیکی پلەگینی فۆرمەکەت.
٢. فۆرمەکە بدۆزەوە کە لەلایەن Site Builder دروستکراوە.
٣. تنظیماتەکانی ئیمەیڵ نوێکردن دەستکاری بکەیت.

بۆ ڕێنمایی زیاتر، سەردانی dokumantasi پلەگینی فۆرمەکەت بکە:

- [ئەمرۆنی ئیمەی Contact Form 7](https://contactform7.com/docs/)
- [ئەمرۆنی ئیمەی WPForms](https://wpforms.com/docs/)
- [ئەمرۆنی ئیمەی Fluent Forms](https://fluentforms.com/docs/)
- [ئەمرۆنی ئیمەی Gravity Forms](https://docs.gravityforms.com/)

## نموونەکانی بەکارهێنان

### فۆرمێکی سادەی پەیوەندیکردن (Simple Contact Form)

> "فۆرمێکی سادەی پەیوەندیکردن دروست بکە کە فیلدا بۆ ناوی کارلێ، ئیمێل و پەیامەکە هەبێت."

### فۆرمی چەند هەنگاوێک (Multi-Step Form)

> "فۆرمێکی پەیوەندیکردن دروست بکە کە هەنگاوی یەکەم بۆ زانیاریی پەیوەندیکردن و هەنگاوی دووەم بۆ پەیام و ڕێگەی پێداویستی پەیوەندیکردن هەبێت."

### فۆرمێک لەگەڵ لۆجیکی بەراوردکراو (Form with Conditional Logic)

> "فۆرمێکی پەیوەندیکردن دروست بکە کە بەپێی هەڵبژاردنی بەکارهێنەر لە دابەزاندنێک، فیلداکان جیاواز دەبن."

### فۆرمێک لەگەڵ بارکردنی فایل (Form with File Upload)

> "فۆرمێکی پەیوەندیکردن دروست بکە کە ڕێگە بدات بە بەکارهێنەران فایل یان پێڵاوێک بار بکەن."

:::note
دروستکردنی فۆرمی پەیوەندیکردن لە Superdav AI Agent v1.10.0 و بەرزتردا دەکرێت. بۆ بەکارهێنانی ئەم توانایە، ئەوەی Site Builder فعال بێت پێویستە.
:::
