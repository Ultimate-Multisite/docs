# بەڕێوەبردنی ئیمیلە سیستمییەکان

Ultimate Multisite بۆ ڕوودۆکە جیاواز وەک پەیامەکانی پشتڕاستکردنەوەی تۆمار، ڕاپۆرتەکانی پارەدان، و گۆڕانکاریی ئەندازیاری ئەلۆمبێنت (membership) ئیمیل دەدات. دەتوانیت ئەم قالبە ئیمیلانە بگۆڕیت بۆ ئەوەی لەگەڵ براندەکەت یەک بێت.

## بینینی ئیمیلە سیستمییەکان {#viewing-system-emails}

بڕۆ بۆ **Ultimate Multisite → Settings** و سەرەتا لەو تابلۆیەی **Emails** (ئیمیلەکان) کلیک بکە بۆ دۆزینەوەی هە settings-ی پەیوەندیدار بە ئیمیل.

![Settings - Emails tab](/img/admin/settings-emails.png)

لەそこاندا دەتوانیت بگاتە لاپەڕەی **System Emails** (ئیمیلە سیستمییەکان) بۆ بینینی هەموو قالبەکانی ئیمیل.

![System Emails list](/img/admin/system-emails-list.png)

شۆڕی ئەو ئیمیلە قالبانە بەکاری دەهێنیت لە ڕێگەی **System email editor** (سەرچاوەی संचای ئیمیل سیستمی) بۆ هەر قالبێکی تایبەت:

![System email editor](/img/admin/system-email-editor.png)

## گەیاندنی قالبە ئیمیلە نوێیەکان {#importing-new-email-templates}

وەرگرتنی وەرگرتنی وەرگرتنی نواندنی Ultimate Multisite و add-ons دەتوانرێت قالبەکانی ئیمیلێکی نوێ بەکاربهێنن. بۆ ئەوەی لەکاتی تێکچوونەکە ڕووبەڕووبیکات، قالبە نوێیەکان بە شێوەیەکی خۆکار زیاد ناکرێنەوە — پێویستە بە دەستی هێز یان manually (بە دەستی) وەربگرین.

بۆ گەیاندنی قالبە ئیمیلە نوێیەکان:

1. بچۆ بۆ لاپەڕەی **System Emails**
2. دابەزێن کلیکی **Reset & Import** لە سەرەوەی ئەو لاپەڕەیە
3. هەڵبژێرە پستی **Import Emails** (گەیاندنی ئیمیلەکان) بۆ بینینی قالبە بەکارخستراوەکان
4. ئەو قالبانەی کە دەتەوێت بگەیەنیت هەڵبژێرە و پشتڕاست بکە

## گۆڕینی قالبەکانی ئیمیل {#resetting-email-templates}

ئەگەر یەکێک لە قالبە ئیمیلەکان شخصی دەگرتووە و دەتەوێت بگەڕێ بۆ ناوەڕۆکی سەرەکی (default content)، دوو ڕێگەی هەیە:

1. **Delete and re-import** — ئیمێلی سیستمیەکە دیار بکە و دووبارە بگەیەنەوە (ئەم کارە پێوانەکانی فرستکردنی پەیامکان دەپاکرێت).
2. **Reset via the tool** — بەکاریان بهێنە ئامرازەکەی **Reset & Import**، هەڵبژێرە ڕێگەی **Reset** (گەڕانەوە)، و ئەو ئیمیلانەی کە دەتەوێت بگۆڕیت.

ئامرازی Reset زیاتر پێشنیار دەکرێت چونکە پێوانەکانی فرستکردن و زانیارییە دیکەکان پارێزراوەت، لەکاتێکدا ناوەڕۆکی سەرەکییەکە دەگەڕێنەوە بۆ شوێنی خۆی.
