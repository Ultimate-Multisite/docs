# بەڕێوەبردنی پارە و فاکچurare

## ڕێکخستنەکانی پارە {#payment-settings}

پێش ئەوەی دەست بکەیت بەتکایەتی پارە وەربگرتن، پێویستە ڕێکخستنەکانی پەیوەست بە پارەکان دابنێیت. بگواشبە **Ultimate Multisite → Settings** و لەسەر تابلۆی **Payment** (پارە) clique بکە.

![Payment settings tab](/img/admin/settings-payments-top.png)

لێرەدا بینینی تەواوی لاپەڕەی ڕێکخستنەکانی پارە هەیە:

![Payment settings full page](/img/admin/settings-payments-full.png)

### هەڵبژاردنەکانی پارەی گشتی {#general-payment-options}

لە ڕێکخستنە گشتییەکاندا دەتوانیت ئەم خاڵانە دیاری بکەیت:

- **Currency** (پێناسەی یەکەم) — ئەو یەکەم نیشانەی پارەیە کە بۆ هەموو مامەڵە بەکاردێت
- **Currency Position** (مۆکەمی پارە) — شوێنێک کە نیشانەکە لەسەر پارەکە دەبێت (پێش/پاش ڕەقمەکە)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### گۆڕینی ڕێگاکانی پارە (Payment Gateways) {#payment-gateways}

Ultimate Multisite پشتگیری دەکات چەندین ڕێگای پارە. دەتوانیت هەر یەکێک لەوانەیە بەکاربهێنیت و ڕێکبخەیت لە تابلۆی Payment settings.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

ڕێگاکان کە دەتوانیت بەکاری بهێنیت:

- **Stripe** — پارەی کارت بە شێوەزاری Stripe
- **PayPal** — پارەی PayPal
- **Manual** (دەستکردن) — بۆ پڕۆسێسینگێکی نەک ئۆفلاین یان تایبەت

هەر ڕێگای پارە وەک ناوچەیەکی تایبەتی خۆی هەیە کە لەوێ API keys و دیکە ڕێکخستنییەکان دەنووسیت.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### Mode ی Sandbox (تەستی) {#sandbox-mode}

دەتوانیت **Sandbox Mode** (مۆدی تاقیکردنەوە) چالاک بکەیت بۆ ئەوەی پێش ئەوەی کار بە ڕاستیی دەبێت، پارەیەک لەسەر تاقیکردن دابنێیت. کاتێک Sandbox mode چالاکە، هیچ پارەیەکی ڕاستیی نیشان ناکرێت.

## بینینی پارەکان {#viewing-payments}

گواشبە لاپەڕەی **Payments** لە خوار Ultimate Multisite بۆ بینینی هەموو مامەڵەکان لەسەر تۆڕەکەت.

![Payments list](/img/admin/payments-list.png)

دەتوانیت پارەکان بەپێی ئەو دۆخەیان (کەوا تەواو بووە، چاوەڕوان دەکرێت، شکستخواز بووە، پارە وەرگیراوەتەوە) فیلتر بکەیت و بۆ مامەڵەکانی تایبەت گەڕان بکەیت.

ئەو پایتەرییەکە دەربکەیت بۆ دۆخێکی تەواوی وردەکارییەکانی وەک بەشەکان، ئەندامیی پەیوەندیدار، زانیاریی کڕیار و داتای گۆڕینی پارە.

## فاکچورەکان (Invoices) {#invoices}

Ultimate Multisite دەتوانێت بە شێوەیەکی ئۆتۆماتیکی فاکچور بۆ پارەکان دروست بکات. دەتوانیت قالب و شێوازی نمرەکردنی فاکچور لە ڕێگەی تنظیمات پایتەرییەکە (Payment settings) بگۆڕیت.

کۆبوونەوەی وردەکارییەکانی فاکچورمان لەوانەیە:

- **ناوی کۆمپانیا و ئادرس** کە لەسەر فاکچورە دەردەکەوێت
- **شێوازی نمرەکردنی فاکچور** و ڕیزبەندییەکەی
- **لوگۆ** کە لە سەری فاکچورەکەدا دەردەکەوێت
- **دەقێکی پاشەوەی تایبەت (Custom footer text)** بۆ بەهاکان، ئاماژەکان یان زانیارییە یاساییەکان

بۆ گۆڕینی قالب فاکچورەکە، بگە بە **Ultimate Multisite → Settings → Payment** و لێی بگەڕێ بۆ ئەو تنظیموانەی پەیوەستن بە فاکچورەوە.
