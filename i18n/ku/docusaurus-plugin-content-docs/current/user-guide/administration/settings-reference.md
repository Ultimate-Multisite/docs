# Reference Settings

ئەم داهاتو پەیجەکە تنظیمەکان دەبینێت کە کاری ڕۆژانەی بەڕێوەبردنی کار و گۆڕانکارییە نوێتر لە ڕەفتارەکانی Ultimate Multisite کاریگەرییان لەسەر ئەوان هەیە.

## بڕوانامەکان (Other Options)

ناوچەکەی **بڕوانامەکان** لە خوارەوە دەبینرێت لە ژێر **Ultimate Multisite > Settings > Login & Registration**.

| Setting | Description |
|---|---|
| **Enable Jumper** | ئامرازەکە وێنەی ڕێگای خێرا (Jumper) لە ناوچەکەی بەڕێوەبردنی کاردا دەردەخات. بیکە بەکاری بۆ ئەوەی ڕاستەوخۆ بگەیتە سەر شاشەکان، ئۆبجێکتییە تۆڕیەکانی Ultimate Multisite و شوێنەکانی بەڕێوەبردنی پشتگیریکراو. ئەگەر نەتەوێت ئەم پێشوەختەیە دەربکەوێت، بیکە بە مۆدەکە (Disable). |

## ڕاپۆرتی هەڵەی کارکردن و تێبینییەکان (Error reporting and telemetry)

پێشتر ئەو تنظیمەی کە بۆ ئەوەی لە ڕاپۆرتکردنی هەڵەکان یەک دەستکاری بکەیت، لە پەیجی تنظیماندا لەناوچەکەدا لەناوچووە. تێبینییە نامۆیی (Anonymous telemetry) بەکار نایەت و هیچ دۆپۆڕێکی ئایکۆنێک نییە بۆ چالاککردنی ئەو کارە.

ئەگەر تۆ ڕاپۆرتی ناوخۆیی یان وێنەی شاشەکانی پەیجی تنظیمانەکە فێری کردووە، پێشنیار دەکرێت ئاماژە بە ئەو فیلدەی کۆنەی ڕاپۆرتکردنی هەڵەکان بکەیتەوە بۆ ئەوەی بەڕێوەبردنی کاران نەگەڕێن بۆ دۆزینێکی تنظیمێک کە دواتوو نییە.

## تنظیمەکانی گواستن/بەرگەگرتن (Import/Export settings)

تەبوی **Import/Export** دەربارەی ئەو تنظیمانە دەگێڕێتەوە کە چۆن کۆنتڕۆڵ دەکەن و ڕاستەوخۆ پەیوەندی بە **Ultimate Multisite > Site Export** بۆ آرشای سایت و تۆڕی کاردا دەدات. لەناوچەکەدا بۆ پێکهێنانی گواستن/بەرگەگرتن، لە **Tools > Export & Import** بەکار بهێنە بۆ کاری گواستن/بەرگەگرتنی یەک-سایتی (single-site)، و کاتێک پێویستیت بە آرشای گواستن/بەرگەگرتنی تەواوی تۆڕی کاردا، ئامراز Site Export بەکار بهێنە.

## تنظیمەکانی کۆنکتۆری پڕۆڤایدر AI (AI provider connector settings)

تنظیمەکانی کۆنکتۆری پڕۆڤایدر AI ئێستا تەنها ناوچە پۆڵەکانی هەژمارە OAuth کە پشتگیریکراون دەردەخات:

| پроваیدر | ڕێگەی سیستابوون (Setup flow) |
|---|---|
| **Anthropic Max** | یەک یان زیاتر لە هەژمارە Anthropic Max یان بە ئۆتۆماتیسپی OAuthیەوە دەبین. ئەگەر بڕوانامەیەکی sandboxed browser ناتوانێت ڕێگەپێدانی (redirect) بە شێوەیەکی خۆکار تەواو بکات، لە کاتی ئەوەدا بیکەوتە پشتیوانی manually OAuth. |
| **OpenAI ChatGPT/Codex** | هەژمارەکانی ChatGPT لە ڕێگەی هەمان Workflowی پۆڵی OAuthەوە دەبین. کارکردنەکان کە پشتیوانییان هەیە، دەتوانن دوای ئەوەی هەژمارهات پەیوەست بوون، بەکاربهێنن ئامرازەکانی ChatGPT Codex. |
| **Google AI Pro** | هەژمارەکانی Google AI Pro لە ڕێگەی OAuthیەوە دەبین، پاشان ئەگەر لیستەکەی ناگۆڕێت، connectorەکە دووبارە refresh بکە. |

Cursor Pro زیاتر پشتگیریکراوە نیشانەی کارکردن نییە. وێنەکان، runbooks (دۆسیەکانی کار)، یان هەنگاوەکانی سەرەتایی کە ئاماژەیە بۆ سیستابوونی Cursor Pro یان ڕێگاکانی connectorەکە، دەبێت لەسەردا نەبن.

کاتێک دایەیت یان دەهێنیت هەژمارەکانی پроваیدر، ئیمەیەک کە بە شێوەیەکی دروست بێت بۆ ئەو هەژمارهاتەی کە دووبارە refresh دەکرێت یان دەدرێتەوە، بنووسە و پێش ئەوەی کارکردنەکان پشت بە connector بدەیت تاقیکردنەوە بکە.
