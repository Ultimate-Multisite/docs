# بەڕێوەبردنی ئەندامان (Managing Memberships)

لە Ultimate Multisite دا دەتوانیت ئەندامەکان و هەموو چوارچێوەکانی سابسکرایبکردن لە تۆڕەکەت بەڕێوەبەرت. داتاکانی سابسکرایبکردن و پڕۆژەکان، ژمارەی سایتەکان، ببڕینی پارە و ماوەی بارکردنەوە، مۆبایلەکانی پارە، و تایمەستامپەکان دەتوانیت داراو بکەیت و بگۆڕیت.

 بۆ گەڕانەوە سەر ئەم لاپەڕەیە، بچۆ بۆ تابلۆی **Memberships** لە منووی Ultimate Multisite.

![Memberships list page](/img/admin/memberships-list.png)

لە لاپەڕەی Memberships دەتوانیت لیستێکی کڕیارەکان و پڕۆژە پەیوەندیدارەکانیان ببینیت. دەتوانیت ئەندامەکان بەپێی ئەو دۆخەیان فیلتر بکەیت: چالاک (active)، چاوەڕوان (pending)، لە دوای وەستان (on hold)، کۆتایی هاتوو (expired)، یان لێکراوە (cancelled).

ئەندامانی بێ بەرامبەر وەک ئەندامانی **ماوەی ژیان** دادەنرێن. ئەوان ماوەیەکی کۆتایی نییە و بە شێوەیەکی ئۆتۆماتیک کۆتایی دەهێنن. ئەگەر پلانی بێ بەرامبەر دوای تروایل یان ماوەیەکی دیاریکراو کۆتایی پێبێت، لە پڕۆژەیەکی فەرمی هەبوو کە تروایلی هەیە بەکاربهێنیت، یان دەستبەجێ ئەندامەکە بگۆڕیت بۆ پلانی دیکە کاتێک دەبێت دەستگەیشتنی ئەو کارە بگۆڕێت.

سەر یەک لە ئەندامەکان clique بکە بۆ گەڕانەوە سەر لاپەڕەیەک کە دەتوانیت داتاکانی ئەندامەکە بگۆڕیت — زانیارییەکانی بارکردن، سنوورەکانی سایت یان سەردان، و زیاتر.

![Membership edit interface](/img/admin/membership-edit.png)

لێرەدا بینینی تەواوی لاپەڕەی گۆڕینی ئەندامەکە دەبینیت:

![Membership edit full interface](/img/admin/membership-edit-full.png)

دەتوانیت ئەندامە نوێکان زیاد بکەیت کە لە پێشتر لە تۆڕەکەت تۆمارکراون. **Add Membership** clique بکە و داتاکان ڕێکبخە.

![Add Membership modal](/img/admin/membership-add-modal.png)

پاشان **Create Membership** clique بکە. ئەندامە نوێەکە لەسەر لیستەکەی Memberships دەردەکەوێت.
