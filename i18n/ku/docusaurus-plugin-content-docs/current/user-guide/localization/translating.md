# وەرگێڕانی Ultimate Multisite

Ultimate Multisite بە زمانی ئینگلیزی دروست کراوە، بەڵام ئەم پڵۆگەکە بە تەواوی دەکرێت وەربگێڕدرێت و ئێمە بەخێی گۆڕانکارییەکانی وەرگێڕان دەکەین.

## وەرگێڕانی پڵۆگەکەی سەرەکی (Core Plugin Translations) {#core-plugin-translations}

پڵۆگەکەی Ultimate Multisite کە بە زمانی ئینگلیزی دروستکراوە، دەکرێت لە ڕێگەی پلاتفۆرمی وەرگێڕانێکی فەرمی **WordPress.org** وەربگێڕدرێت:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

ئەمە سیستەمی وەرگێڕانی estàndard ی کۆمەڵگای WordPressە. کاتێک وەرگێڕانەکان ئاستی پەسەندکردنی پێویستیان گرتن، بە شێوەیەکی خۆدۆز لە ڕێگەی نوێکردنەوەکانی پڵۆگەکانی WordPress بۆ هەموو بەکارهێنەرانی Ultimate Multisite دابەش دەکرێت.

**ئێمە پێویستە یارمەتیت بدەین!** ئەگەر زمانی تر لە ئینگلیزی زانیاری دەکەیت، تکایە بیر بکەرەوە کە وەرگێڕانێک بەخێربکەیت. وەرگێڕانی چەند ڕستەیەکی کەمیش یارمەتیدا دەدات. بۆ دەستپێکردن:

1. سەردانی [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) بکە
2. زمانی خۆت هەڵبژێرە
3. لەسەر دAssuming "Contribute Translation" (وەرگێڕان پێشکەش بکە) بێت
4. پێویستە یەککارتنی وردی WordPress.org هەبێت

ئەگەر دەتەوێت دەبن وەرگێڕەرێکی وەرگێڕان بۆ زمانی خۆت بیت (واتە بتوانیت وەرگێڕانەکان پەسەند بکەیت)، لە ڕێگەی [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) پەیوەندی بە ئێمە بکە.

## وەرگێڕانی ئەپڵۆگەکان (Addon Translations) {#addon-translations}

وەرگێڕانی ئەپڵۆگەکان بە شێوەیەکی جیاواز لە شوێنەکانی ئەم پلاتفۆکە دەکرێت:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

وەرگێڕانی ئەپڵۆگەکان ئێستا بە شێوەیەکی AI دروستکراون وەک سەرەتای کار. ئەوان هەموو زمانە پشتگیریکراوەکان دەگرن، بەڵام لەوانەیە ڕستەکاندا چەند ڕستەی نادەروست یان ناڕاستیی هەبێت. ئێمە گۆڕانکارییەکانی مرۆڤ دەکەین و پێشنیار دەبین بۆ ئەوەی زیاتر سروشتی و دروست بن.

## وەرگێڕانی ناوخۆیی (Local Translations) {#local-translations}

ئەگەر زیاتر دەتەوێت وەرگێڕان لەسەر پڕۆژەی خۆت بە شێوەیەکی ناوخۆ ئەنجام بدەیت، نەک بەشداریکردن لە سەرەوە (upstream)، دوو ڕێگە هەیە:

### بەکارهێنانی Poedit {#using-poedit}

دەتوانی لەم بەرنامەی دیسکتاپ **[Poedit](https://poedit.net)** بەکاربهێنیت بۆ وەرگێڕانی فایلەکە `.pot`:

1. Poedit-ی لە [poedit.net](https://poedit.net) داونلود بکە.
2. فایلەکە `.pot` یەکەی لە `wp-content/plugins/ultimate-multisite/lang/` دابنێ.
3. دەقەکان (strings) لە ئۆتۆپەری Poedit وەرگێڕە.
4. هەڵبژێرە -- Poedit فایلەکانی `.po` و `.mo` دروست دەکات.
5. فایلە ئەوتۆپەریەکە لە ناو فۆلเดری `lang` دابنێ.

### بەکارهێنانی Loco Translate {#using-loco-translate}

پلاگینی **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** ی WordPress ڕێگە دەدات کە ڕاستەوخۆ لە پۆینت ئەدمن (admin panel) وەرگێڕان بکەبێت، بەبێ پێویستیی ئامراز دیسکتاپ. ئەو بلاگەکە بۆ دروستکردن و کۆکردنەوەی فایلەکانی وەرگێڕان بۆ تۆ دەکرێت.
