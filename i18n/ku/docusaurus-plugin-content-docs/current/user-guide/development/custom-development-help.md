# چۆنی یارمەتی پراکتیکی تایبەت لە Codeable وەربگرت؟

هەندێک جار، ئێمە بە ڕاستی پێویست دەکەین هەندێک تایبەتمەندی دیاریکراو بۆ ئەوەی کارکردن و پڕکردنەوەی تۆڕەکەیان بەو شێوازەی کە دەتانەوێت بێت. ئەگەر ئەمە ئەو شتەیە کە دەجووڵیت، ئێمە [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) پێشنیار دەکەین، شوێنێک کە لەوێدا دەتوانیت دیزاینەر و پەرەپێدەرانی باخشوو هەبێت بە شارەزایی لە Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) پلاتفۆرمێکە کە کڕیارەکان دەبەستێتەوە بە دیزاینەرانی فری لانس و وردی WordPress، کە توانای کار لەسەر پراکتسێکی هەر قەبارەیەک هەیە. ئەوان پرۆسەی داواکارییەکی وردیان هەیە بۆ دڵنیابوون لەوەی کە تۆ لەگەڵ خاوەن شارەزاییەکان یەک دەبێت کە تایبەت بە ئەو کارەی کە مەبەستتە.

### چۆن پراکتسێک دروست بکەیت لە Codeable؟ {#how-to-create-a-project-in-codeable}

پێشتر، بچۆ بۆ [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) و پستی بەختەوەی **Start a Project** (دابەستکردنی پراکتس).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

ئێستا، بۆ ئەوەی پراکتسێک پێشکەش بکەیت، لە لایەنی چەپ کلیک بکە بۆ **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

لە بەشی "I need help with" (من پێویستم بە یارمەتی هەیە)، هەڵبژێرە **Customizing** (تایبەتمەندی تایبەت):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

لە بەشی "My" (من)، هەڵبژێرە **Plugin** (پلاگین):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

ئەو کات کلیک بکە لەسەر دuguورەوە بۆ **Continue** (دایەکی).

<!-- Screenshot unavailable: Codeable project form Continue button -->

ئێستا، کاتێکە دەبێت پراکتسەکەت و پێداویستییەکانت بە باشترین شێواز کە دەتوانیت باس بکەیت. هەوڵ بدە زۆرترین زانیاری دەربارەی خۆت بدەیت!

<!-- Screenshot unavailable: Codeable project description form -->

پاسکردن بە پرسیارەکانی خوارەوە یارمەتی دروستکردنی نرخ دەدات.

<!-- Screenshot unavailable: Codeable project pricing questions -->

پاش ئەوەی ئەگەر یەکەم بار نیشانەی [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) نییە، یەکێکیان دروست بکە و د assuming **Publish My Project** बटنی هەڵبژێرە.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

کاتێک پڕۆژەی [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) یەکەم جار بڵاو کردەوە، پلاتفۆرم دەبێت تۆ پەیوەندی بکات بە پەرەپێکراوەکان کە تواناکانی پێویست بۆ پڕۆژەکەت هەیە.

ئەم کارە یارمەتیت دەدات تا ئەو کەسە دروست بکەیت کە بۆ کارەکەت گونجاوە و ئەزموونی وردی لەگەڵ WordPress و Ultimate Multisite هەیە!

کەمترین نرخەکانی Codeable $70–$120 دۆلار بە کاتژمێرە. پێداچوونەوەکان دەگۆڕێن بەپێی قۆناغی کار، ئاستی وردی کار و خێرایی پڕۆژەکە.

کاتێک پێداچوونەوەکەت گرتەوە، دەتوانیت ئەو دزڤەرە بگرین کە باشترین یەک بۆ پێویستییەکانتە و هاوکار بە پڕۆژەکەت دەست بکەین!
