# جستجوی اینترنت (Internet Search)

Gratis AI Agent v1.5.0 یه قابلیت جدید به اسم **جستجوی اینترنت** اضافه کرده که به دستیار هوش مصنوعی اجازه می‌ده در طول گفتگو اطلاعات به‌روز از وب رو پیدا کنه. این قابلیت با استفاده از [Brave Search API](https://brave.com/search/api/) یا [Tavily API](https://tavily.com/) کار می‌کنه.

## چطور کار می‌کنه؟ {#how-it-works}

وقتی جستجوی اینترنت فعال باشه، دستیار به صورت خودکار وقتی تشخیص بده که یه سوال نیاز به اطلاعات جدید یا بیرونی داره (مثلاً اخبار اخیر، قیمت‌های زنده، یا مستنداتی که ممکنه از زمان آموزش مدل تغییر کرده باشن)، اون رو به سرویس جستجوی تنظیم شده شما می‌فرسته.

نتایج در لحظه بازیابی می‌شن و قبل از اینکه دستیار پاسخی بده، توی زمینه (context) خودش قرار می‌گیرن. دستیار هم مشخص می‌کنه که وقتی از نتایج جستجو برای پاسخ دادن به سوالی استفاده کرده.

## فعال کردن جستجوی اینترنت {#enabling-internet-search}

برای استفاده از جستجوی اینترنت، نیاز به یک کلید API از سرویس جستجوی انتخابی خود دارید. برای تنظیمش:

۱. به **Gratis AI Agent → Settings → Advanced** بروید.
۲. منوی کشویی **Internet Search Provider** رو پیدا کنید و یکی از گزینه‌های **Brave Search** یا **Tavily** رو انتخاب کنید.
۳. کلید API خودتون رو توی فیلد مربوطه وارد کنید. لینک‌های ثبت‌نام کنار هر فیلد نمایش داده می‌شن.
۴. روی **Save Settings** کلیک کنید.

وقتی کلید ذخیره شد، قابلیت جستجوی اینترنت به صورت خودکار برای دستیار در دسترس خواهد بود.

## Brave Search {#brave-search}

### دریافت کلید API Brave Search {#obtaining-a-brave-search-api-key}

۱. به [Brave Search API page](https://brave.com/search/api/) بروید.
۲. برای یک طرح (plan) ثبت‌نام کنید. یک سطح رایگان با محدودیت درخواست ماهانه در دسترس است.
۳. کلید API خودتون رو از داشبورد توسعه‌دهنده Brave Search کپی کنید.
۴. اون رو توی فیلد **Brave Search API Key** در تنظیمات Gratis AI Agent پیست (Paste) کنید.

### محدودیت‌های استفاده {#usage-limits}

استفاده لەم کارە بەپێی ژمارەی پرسیارەکان کە دەکەیت، لەلایەن Brave Searchەوە وەک نرخ دەکرێت. هەر واژینێکی AI کە لێدەکات گەڕان بکرێت، وەک یەک پرسیار دادەنرێت. بۆ ئەوەی پارە بە شێوەیەکی ناخۆش نەبێت، بەکاریگەنان لە داش보ار [Brave Search developer dashboard](https://brave.com/search/api/) بەدوای بەکارهێنانی خۆت دەبینیت.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 **Tavily** وەک سەرچاوەیەکی گەڕان لە ئینتەرنێت وەک جێگرەوە زیاد دەکات، کە ئەنجامی گەڕانێکی بەرفراوانتر و تواناکانی تاقیکردنەوەی پێشکەوتوو دەبەخشێت.

### بەدەست هێنانی API Key ی Tavily {#obtaining-a-tavily-api-key}

1. سەردانی [Tavily API page](https://tavily.com/) بکە.
2. بۆ هەژماری تۆ تۆمار بکە. پلەیەکی بێ بەرامبەر (free tier) لەگەڵ سنووری داواکاری مانگانە هەیە.
3. API Key ی خۆت لە داش보ار Tavily kopîی بکە.
4. لە فیلدی **Tavily API Key** لە ڕێکخراوەکانی Gratis AI Agentدا چێگرە.

### سنوورەکانی بەکارهێنان {#usage-limits-1}

استفاده بەپێی ژمارەی کۆکۆڵەکانی API کە دەکەیت، لەلایەن Tavilyەوە نرخ دەکرێت. هەر واژینێکی AI کە لێدەکات گەڕان بکرێت، وەک یەک کۆکۆڵ دادەنرێت. بۆ ئەوەی پارە بە شێوەیەکی ناخۆش نەبێت، بەکاریگەنان لە داش보ار [Tavily dashboard](https://tavily.com/) بەدوای بەکارهێنانی خۆت دەبینیت.

## ناتواندنەوەی گەڕان لە ئینتەرنێت {#disabling-internet-search}

API Key یەکە لە فیلدی سەرچاوەی گەڕان فعالدا هەڵبژێرە و سیوەر بکە. توانای گەڕان لە ئینتەرنێت دواتر بۆ ئەنجامدهێنانی وەڵامدانەوەی کارمەندەکە پێشکەش ناکرێت.

:::note
گەڕان لە ئینتەرنێتی کەمێک کێشەی کات (latency) بە وەڵامەکان دەدات چونکە کارمەندەکە دەبێت چاوەڕێی ئەنجامی گەڕان بکات پێش ئەوەی وەڵامێکی دروست بدات. بۆ ئەو حاڵەتانەی کە کاتیان گرنگن، بیر لەوە بکەرەوە ئایا گەڕانی ڕاستەوخۆ (real-time search) پێویستە یان ئایا زانیارییە ناوخۆیی کارمەندەکە بەشێوەیەکی باش دەبێت.
:::
