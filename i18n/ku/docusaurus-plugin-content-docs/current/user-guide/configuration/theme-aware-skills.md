# مهارات آگاه بە قالب (Theme-Aware Skills) {#theme-aware-skills}

Superdav AI Agent v1.10.0 چوار مهارەتی نوێی ئامادە دابین کردووە کە بە شێوەیەکی خۆکار دەگونجێن و بە شێوەیەکی ئۆتۆماتیک لەگەڵ قالب وردپرێس (WordPress) کە چالاکە بۆ سایتەکەت. ئەم مهاراتان ڕێنمایی و تواناکانی تایبەت پێشکەش دەکەن کە بە پێی پێکهاتە و تایبەتمەندیەکانی ئەو قالبە دیاریکراوەکان دەگونجێن.

## مهارات آگاه بە قالب چیین؟ {#what-are-theme-aware-skills}

مهاراتە ئامادە (Theme-aware skills) بنکەی زانیاری و کۆمەڵێک ئامراز کە خودی ئەسپێری AI بە شێوەیەکی ئۆتۆماتیک دەبنە هەڵبژاردن لەسەر بنەمای ئەو قالبەی وردپرێس کە ئێستا لەسەر سایتەکەت چالاکە. کاتێک تۆ قالب دەگۆڕیت، مهاراتەکانی بەکارهێنانی ئەسپێری ئۆتۆماتیک دەگونجێن — پێویستی بە ڕێکخستنی دەستکرد نییە.

هەر مهاراتێک لەخۆدەگرێت:

- **دۆکیومێنتە تایبەت بە قالبەکە** — ڕێنمایی بۆ بەکارهێنانی و گۆڕینی (customizing) ئەو قالبە
- **ئەندامەکانی Block و Pattern** — ئەوانەی کە دەتوانیت بەکاریان بهێنیت، هەروەها پترنەکان و ڕێگاکانی دیزاین
- **نموونەکانی گۆڕین (Customization examples)** — کۆدە بچووکەکان و شێوازەکانی ڕێکخستنی بۆ ئەرکە باوەکان
- **سەرچاوە باشترین شێوازی کارکردن (Best practices)** — پێشنیارەکان بۆ پێکهاتەی قالبەکە و ڕێوشوێنی کاری.

## مهاراتە ئامادە بەکارھێنانی (Available Theme-Aware Skills) {#available-theme-aware-skills}

### قالبە پۆلێمبەکان (Block Themes) {#block-themes}

**ئەمە بۆ:** ئەو قالبانە کە لەسەر بنەمای پێکهاتەی Block وردپرێس (Full Site Editing) کاردەکەن.

مهارەتی Block Themes ڕێنمایی دەدات لەسەر:

- دروستکردن و دەستکاری Template (پۆلێمب) بە بەکارهێنانی block editor
- کار کردن لەگەڵ block patterns و block یانە کە دەکرێت دووبارە بەکاربهێنرن
- گۆڕینی ستاایلەکانی گشتی و تنظیمەکانی theme.json
- بەکارهێنانی theme blocks و variatyon (تغییرات)
- دروستکردنی block pattern تایبەت بۆ سایتەکەت

**ئەمە ئۆتۆماتیک دەستکار دەکرێت کاتێک:** قالب چالاکە तुमच्याیەکی Block Theme بێت (پشتیوانی پێکهاتەی `block-templates` دەکات).

### قالبە کلاسیکییەکان (Classic Themes) {#classic-themes}

**ئەمە بۆ:** ئەو قالبانە پزیشکی وردپرێس کە لەسەر PHP templates و classic editor کاردەکەن.

مهارەتی Classic Themes ڕێنمایی دەدات لەسەر:

کارکردن لەگەڵ فایلە template (template) و hookەکانی PHP
پێناساندنی ڕووبینی تێمەکە بە شێوەی تایبەت (Customizer)
فراوانکردنی dartboard-a (widget areas) و sidebarکان
تغییردەدانی CSS و پەرەپێدانی child theme
فهمی هێزی تێم (theme hierarchy) و template tags

**ئەمە بە شێوەیەکی ئۆتۆماتیک دەست پێ دەکات کاتێک:** تێمەکە کە چالاکەیت، تێمێکی کلاسیکی (ناوەکانی block نین).

### Kadence Blocks {#kadence-blocks}
**بۆ কারা دەگاتە کار:** ئەو سایتانەی لەکار دێن Kadence Blocks plugin بۆ دیزاینی پێشکەوتوو بە بنەمای block.

ئامرازەکانی Kadence Blocks ڕێنمایی دەدەن لەسەر:

- بەکارهێنانی کۆمەڵەی پیشەنگاوەی پێشکەوتووی Kadence (وەک Hero، Testimonials، Pricing و هتد).
- ڕێکخستنی تنظیمات و پيشنیاسەتەکانی blockی Kadence.
- دروستکردنی دیزاینە تایبەت بە بەکارهێنانی grid و container blocks ی Kadence.
- یەکگرتووکردنەوەی Kadence blocks لەگەڵ تێمەکەت.
- بەکارهێنانی سیستەمی دیزاینی (design system) و پێشوەختەکانی Kadence.

**ئەمە بە شێوەیەکی ئۆتۆماتیک دەست پێ دەکات کاتێک:** Kadence Blocks plugin لەسەر سایتەکەت چالاکە.

### Kadence Theme {#kadence-theme}
**بۆ কারা دەگاتە کار:** ئەو سایتانەی تێمەکە بەکاردێن بۆ دیزاین و تایبەتمەندکردن بە بنەمای block.

ئامرازەکانی Kadence Theme ڕێنمایی دەدەن لەسەر:

- تغییردەدانی تێمەکە بە شێوەی global styles و theme.json.
- بەکارهێنانی پترۆنەکان (patterns) و template (template) ی ناوخۆیی Kadence.
- ڕێکخستنی تنظیمات و پيشنیاسەتەکانی تێمەکە لە Kadence.
- دروستکردنی دیزاینە تایبەت بە سیستەمی دیزاینی Kadence.
- یەکگرتووکردنەوەی Kadence لەگەڵ pluginە و ئامرازە بەناوبانکراوەکان.

**ئەمە بە شێوەیەکی ئۆتۆماتیک دەست پێ دەکات کاتێک:** تێمەکە کە چالاکەیت، تێمێکی Kadence-ی کلاسیکییە.

## چۆن Skills هەڵدەبژێردرێن؟ {#how-skills-are-selected}

Assistant (یارمەتیدەر) بە شێوەیەکی ئۆتۆماتیک تێمەکەی چالاک و pluginەکانی جێگیرکراوەکان لە هەر پەیامێکدا دەدۆزێتەوە. ئەگەر یەک Skill-ێکی ناسراو کە پەیوەست بێت بە تێمەکە، بە شێوەیەکی ئۆتۆماتیک بۆ context (پێکهاتەی) assistantەکە دابنێریت. پێویست نییە بە دەستی خۆت Skills چالاک بکەیت یان بگۆڕیت.

### Multiple Skills {#multiple-skills}

ئەگەر چەند مهارتییەک بۆ سایتت دەبێت (بۆ نموونە، ئەگەر هەردوو Kadence Blocks و Kadence Theme چالاک بێت)، ئاسستێنت دەتوانێت سەرچاوەی ڕێنماییەکانی هەر یەکێکیان بەکاربهێنێت.

### گۆڕینی Theme (سایتی) {#switching-themes}

کاتێک تۆ theme-ی چالاکت دەگۆڕیت، مهارەتە دروستکراوەکانی ئاسستێنت لە پەیامەکەی دوایەکەدا بە شێوەیەکی خۆدۆزەوە دەگۆڕێن. بۆ نموونە:

1. تۆ لە ئێستادا لە کارکردندا یان theme-ی بلاک (block theme) بەکار دێت و مهارەتی **Block Themes** چالاکە.
2. تۆ گۆڕان دەدەیت بۆ theme-ێکی کلاسیکی (classic theme).
3. لە پەیامەکەت دوای ئەوە، مهارەتی **Classic Themes** بە شێوەیەکی خۆدۆزەوە بار دەکەوێت، و مهارەتی **Block Themes** دووڵەمەند ناهاتوو دەبێت.

## بەکارهێنانی مهارەتە ئاوەزدار Theme (Theme-Aware Skills) {#using-theme-aware-skills}

بۆ ئەوەی لە مهارەتی ئاوەزدار theme یەکدا بەکاربهێنیت، تەنها ئەو شتەی دەتەوێت لە چاتەکەدا باس بکە. ئاسستێنت بە شێوەیەکی خۆدۆزەوە ڕێنمایی مهارەتی گونجاو دەدات.

### نموونەکانی پرسیارکردن (Example Prompts) {#example-prompts}

**بۆ Block Themes:**
> "سەرچاوەیەکی سەرەکی (hero section) دروست بکە بە وێنەی پسراوە و دەقێکی ناوەندکراو بە شێوەی پترنەکانی بلاک."

**بۆ Classic Themes:**
> "بەکارھێنانی child theme، یاری ناوچەیەکی ویستجەک (widget area) بۆ لایەنی کاتژمێرەکە (sidebar) زیاد بکە."

**بۆ Kadence Blocks:**
> "سەرچاوەیەکی بەڵگەنامەی تێبینی (testimonials section) دروست بکە بە بەکارهێنانی بلاک Kadence Testimonials."

**بۆ Kadence Theme:**
> "دابەشکردنی دیزاینی سەردانەکە و ڕێکخستنی شێوازی مێنیو (navigation menu styling) دەستپێبدە."

ئاسستێنت ڕێنمایی-ی تایبەت بە theme و نموونەکانی کۆد پێشکەش دەکات کە بۆ theme و plugin-ی چالاکت تایبەتمەند بێت.

:::note
مهارەتی ئاوەزدار theme لە Superdav AI Agent v1.10.0 و دوای ئەوە بە شێوەیەکی خۆدۆزەوە دەبێت. پێویستی بە سیستراوەکانی زیاتر یان ڕێکخستنی تایبەت نییە.
:::
