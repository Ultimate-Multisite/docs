# چۆن دەتوانم بۆ یەکەم جار سەرچاوەی تایبەتی خۆم ببینم؟ {#how-can-i-access-my-account-page-for-the-first-time}

بەخێربێن بۆ Ultimate Multisite!

تەواوی کارکردنی تایبەت بە ئەکاونتەکەت لە ڕێگەی لینکێک بکەرەوە کە لە ئیمێلەکەدا دوای کڕینی پلاتین (plugin) ئێمە دەستپێدات:

<!-- Screenshot unavailable: Account page email with access link -->

لە ئیمێلەکەدا، بەکارهێنانی ئیمێلی کە لە کاتی کڕیندا پێشکەوتوویت و ئەو وشەی پاسوورەی کە لە ئیمێلەکەت دابنێتی بۆ دەتوانیت.

<!-- Screenshot unavailable: Login page with email and password fields -->

لەبەر ئەوەی ئەمە یەکەم جارە کە دەداتەوە بەم ئەکاونتە، پێشت دەپرسێت کە پاسوورەی خۆت بگۆڕیت. پاسوورەکە بگۆڕە و دابەزێن (Save) بۆ ئەوەی گۆڕانکارییەکە ڕەنگە هەڵبگرێت.

<!-- Screenshot unavailable: Change password form with Save button -->

لە سەر داش보ارەکەت، دەتوانیت هەموو وردەکارییەکانی ئەکاونتەکەت بەڕێوەبەر بکەیت، وەک بارکردنەوە (billings)، نوێکردنەوە (renewals)، لغارکردن (cancellations) و زۆر شتێکی دیکە.
