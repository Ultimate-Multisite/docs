# چۆن دەتوانم سێبسکرایبەکەم بڕم؟ {#how-can-i-cancel-my-subscription}

ئەگەر بەڵگەنامەی Ultimate Multisite هەبوو، دەتوانیت هەر کاتێک کە دەتەوێت نوێکردنەوەکەی بڕیت. تەنها ئەم هەنگوعە بەدوای بکە:

<!-- Screenshot unavailable: Account page email with access link -->

فایدیەکەی کە لە کاتی کڕیندا پێشکەش کردووە و پۆسڵەکەت، و پۆسڵەکەت بەکاربهێنە.

<!-- Screenshot unavailable: Login page with email and password fields -->

ئەمە سەرەتاییی لایەنی هەژمارەکەتە:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

بۆ بڕینی سێبسکرایبەکەت، لە مێنیی لایەنی چەپدا، بگواشبە بە _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

سێهەڵگرەکە بۆ ڕاست بکە بۆ دۆڵێکی لایەنەوە بکەرەوە. پاشان دەتوانیت ئەو هەڵبژاردەیە هەڵبژێریت کە نوێکردنەوەی **auto-renew** (نوێکردنەوەی خۆکار) بڕێت.

سیستەم یەکێک لەو پەیامە ڕاگەیاندنی دەتەکان دەداتت.

ئەگەر دڵنیایت کە دەتەوێت سێبسکرایبەکەت بڕیت، تەنها پەنجەی _**Cancel Renewals**_ (بڕینی نوێکردنەوەکان) بکە. دوای ئەم کارە، پێشت دەداتت بۆ وەڵامدانەوەی پرسیارێکی کورت.

کەتام! سێبسکرایبەکەت بە شێوەیەکی خۆکار نوێ نەکراوەت.

**تو تا کاتی کۆتایی هاتنی ماوەی سێبسکرایبەکەت، کلێی بەکارھێنانی ڕاست دەبێت**. ئەگەر دەتەوێت سێبسکرایبەکەت دووبارە چالاک بکەیتەوە، پێویستە بە شێوەیەکی دەستکرد بیکەیت.
