# لغای ئەندازیاری یەکەم (v2)

_**تێبینی گرنگ: ئەم بابەتە ئاماژەیە بۆ Ultimate Multisite وەرگرتنی 2.x.**_

لغای کارمەند لەو لایەدا کە دەتوانیت لغاکەی لغێنیت، زۆر ئاسانە.

ئەگەر بەکارهێنەرەکەت دەتەوێت ئەندازیاری خۆی لغێنێت، پێویستە سەرەتا بگاتە لاپەڕەی **Account** (کارمەند).

لەوێ، دەبێت سەری **Danger Zone** (ناو حەموو) بکەن و هەرسنە بۆ **Delete Account** (لغۆکردنی کارمەند).

![Danger Zone لە لاپەڕەی کارمەند لەگەڵ دۆمبیلی لغۆکردنی کارمەند](/img/account-page/danger-zone.png)

یەکModuleName دەکرێتەوە کە داوا دەکات ڕەزامەندی بکەن. دەبێت **Confirm Account Deletion** (پەیوەندی بە لغۆکردنی کارمەند) لەسەر togglen و هەرسنە بۆ **Delete Account**.

![دۆلۆگەی پەسەندکردنی لغۆکردنی کارمەند](/img/account-page/delete-account-confirm.png)

ئەم کارە بەو شێوەیە دەبێت کە لە سایتەکەدا لە سیستەمەکە دابنێیت و ئەندازیارییەکەت لە تۆڕەکەت لغێنیت، هەروەها سایتەکەت لە تۆڕەکەت پاک بکەیت.

![لە دوای لغۆکردنی کارمەند لە سیستەمەکە دابنێیە](/img/account-page/logged-out-after-delete.png)

![پەیوەندیی لغۆکردنی ئەندازیارییەکە لەسەر پەڕەکان](/img/account-page/membership-canceled.png)

**گرنگە:** ئاگادار بیت کە ئەم کارە بەکارهێنەرەکەت لە تۆڕەکەت دەردەبێت، چونکە یەک کارمەند دەتوانرێت زیاتر لە یەک ئەندازیاریی هەبێت یان هیچ ئەندازیارییەکی پەیوەست نییە.

![کارمەند لە تۆڕەکەدا دوای لغۆکردن لەسەر پەڕەکان](/img/admin/customer-after-cancel.png)

![کارمەندێک کە هیچ ئەندازیاری چالاک نییە](/img/admin/memberships-customer-empty.png)
