# وێبەکانی کلاینتەکەت (v2)

پاش ئەوەی لە تۆڕەکەت تۆمار بکەیت، کڕیارەکان دەتوانن بچنە داشبۆردی خۆیان.

لەگەڵ **Account** (حساب) پەیجەکەدا، کە تێیدا دەتوانن زانیاری گرنگ سەبارەت بە پارەدان، ئەندامبوون، دۆمینەکان، سنوورەکانی پلاندانان و زیاتر لەوەش ببینن، دەتوانن بچنە **My Sites** (وێبەکانی من) پەیجەکەدا، کە تێیدا دەتوانن هەموو وێبەکانی خۆیان ببینن و بەڕێوەبەر بکەن. هەموو کارێک کە پێویستیانە ئەوەیە لە داشبۆردەکەت سەرنجی **My Sites** بکەن.

![Link بۆ My Sites لە داشبۆردی کڕیار](/img/account-page/my-sites-link.png)

هەموو وێبەکانیان دەبینن، بە چافتی **Manage** (بەڕێوەبردنی) لە خوار یەکێک لەوانەیە. ئەگەر لەسەر ئەو دۆکەمە بێت، دەچێتە داشبۆردی ئەو وێبەی تایبەتە.

دەتوانن هەروەها ببینن چیا وێبەکانیان ناونیشانی **primary** (پێشوەختە)ن بۆ پلاندانان کە ڕێگە بە ئەوە دەدات کە زیاتر لە یەک وێب هەبێت.

![My Sites پەیجەکە کە وێبەکانی کڕیار و دۆکەمەکانی Manage دەردەخات](/img/account-page/my-sites-page.png)

لێرەدا بینینی تەواوی پێکهاتەی دەستکاری وێبەکە هەیە:

![پێکهاتەی تەواوی دەستکاری وێبەکە](/img/admin/site-edit-full.png)

ئەگەر دەتان quieran وێبەکەی سەرەکی خۆیان بگۆڕن، پێویستە بچنە پەیج **Account** بە چافتی **Account**.

![Linkی مینیوی Account لە داشبۆردی کڕیار](/img/account-page/account-menu.png)

دەتوانن زانیاری کڕیارەکان ببینن و دەستکاری بکەن:

![پێکهاتەی دەستکاری کڕیار](/img/admin/customer-edit.png)

ئەو کاتدا، پێویستە لەسەر **Change Default Site** (گۆڕینی وێب سەرەکی) بێت.

![چافتی Change Default Site لە پەیج Account](/img/account-page/change-default-site.png)

پۆپ-آپێک دەردەکەوێت. لە لیستەکەی دروستکردنی خێرایی (drop-down list) هەڵبژاردن بکەن کە ئەو وێبەیە کە دەتانە وێب سەرەکی بێت.

![راوەکە بۆ هەڵبژاردنی Change Default Site](/img/account-page/change-default-site-dropdown.png)

پاش ئەوە، تەنها لەسەر **Change Default Site** بێت و کارەکەت تەواو بوو!

![تاییدی لە گۆڕینی شوێنی سەرەکی](https://example.com/img/account-page/change-default-site-confirm.png)
