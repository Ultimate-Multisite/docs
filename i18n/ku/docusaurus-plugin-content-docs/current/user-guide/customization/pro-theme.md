# تنظیم پنل مدیریت خود با قالب PRO {#customizing-your-admin-panel-with-pro-theme}

اغلب وقتی کار مشتری انجام می‌دیم، نیاز داریم به مشتریانمان یک تجربه سفارشی در داشبورد وردپرس‌شون ارائه دهیم. اگر دنبال ظاهری حرفه‌ای‌تر هستید، قالب PRO می‌تواند به شما کمک کند.

قالب PRO یک قالب مدیریت است که برای کمک به شما در شخصی‌سازی داشبورد مشتریانت توسعه یافته است. و همچنین بخشی از هسته افزونه‌های پریمیوم Ultimate Multisite است.

بعد از نصب آن، به صفحه _Plugins_ خود بروید و روی فعال کردن آن در شبکه (Network) کلیک کنید.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_فعال‌سازی آن در کل شبکه شما در صفحه افزونه‌ها_

_**توجه:**_ _اگر فقط نصب کنید، برای مشتریان شما به عنوان یک گزینه نمایش داده می‌شود که می‌توانید با علامت زدن آن، قالب را فقط برای آن کاربر اضافه کنید._

حالا، رابط کاربری مدیریت خود کاملاً بازطراحی شده است! همچنین یک منوی جدید در ستون سمت چپ به نام PRO Theme خواهید داشت.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_ظاهر کاملاً جدید برای پنل مدیریتی شما_
