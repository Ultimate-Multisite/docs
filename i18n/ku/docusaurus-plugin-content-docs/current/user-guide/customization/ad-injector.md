# ڕاکێشاندنی ئۆفڵایەکان لە تۆڕەکەت بە Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector یەکێکە لە **پلانەکانی زیادکردن (add-on) کە بەبەرچاوە** و پێداویستییەکە بۆ ئەوەی دەتوانیت ئۆفڵایەکان لە ڕێگەی دیمەنتی (edit page) تیایدا دانان.

پاش جێبەجێکردنی Ultimate Multisite: Ad Injector، بچۆ بۆ لایەنی _Plugins_ یەکەت بۆ چالاککردنەوەی وایە.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_کلیک بکە لە دغڕاندنی تۆڕ (Network Activate)_

کاتێک ئەو add-onە چالاک بوو، ناوچەی _Advanced Options_ لە شاشەی دەستکاری پلانەکەدا کارکردنێکی نوێی بە ناوی _Ad Settings_ هەبووە.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_ئێستا دەتوانیت تنظیمەکانی ئۆفڵا بۆ هەر یەکێک لە پلانەکانت بگۆڕیت._

لەو تابلۆیە، دەتوانیت کۆدی ئۆفڵاکان بۆ پۆستەکان و پەڕەکان لە پێش و دوای ناوەڕۆکەکەدا لە فرۆن end (front-end) زیاد بکەیت.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_پاش داواکردنی کۆدی ئۆفڵاکەت، لەسەر پۆستەکان و پەڕەکان بەو پلانەی دیاریکراوە دەردەکەوێت._

شيمكن هەبێت ئۆفڵایەکان لە سەری (header)ی لایەنە ئەدمنییەکانی (Admin Pages) بۆ بەکارهێنەرانی ئەو پلانەیە.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_ئەگەر دەتەوێت ئۆفڵایەکان لە پاشبەردەی (back-end) بەکارهێنەرەکەت نیشان بدات، بچۆ بۆ هەڵبژاردنی Ad Code لە Admin Panel._

### زیاتر لە ڕاکێشانەوەی ئۆفڵا {#beyond-advertising}

دەتوانیت Ultimate Multisite: Ad Injector بەکاری بهێنیت بۆ پێشکەشکردنی پشتگیریکردن بۆ پلانە بەرزەکانی (high-tier plans).

تەنها بچۆ بۆ هەڵبژاردنی _Admin Panel Ads_ و کۆدی HTML بۆ ئامراز پشتیوانییەکەت زیاد بکە.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

_پشتیوانی کڕیار بۆ پلاندانەکانی دیاریکراو زیاد بکە._

بە شێوەیە، خەڵک دەتوانن ڕاستەوخۆ لە پانێ ئەدمنیان بە تۆ بگەن!
