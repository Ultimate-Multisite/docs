# یادداشتەکانی ڕێخاندن (Release Notes) {#release-notes}

## وەرگرتنی ٢.١٣.٠ — لە ٢٠٢٦-٠٦-٠٥ {#version-2130--released-on-2026-06-05}

- نوێ: پشتگیریکردنی sovereign-tenant بۆ هەژمار کڕیار، چاپی (checkout)، بڕین، سایت، فاکچۆرا، گۆڕینی قالب و مۆبایلینگ دابنێتی بۆ ئەوەی تۆڕەکانی تەنەنت دەتوانن کڕیاران بگونجێنەوە بۆ سایت سەرەکی بۆ کارلێککردنی کارە بەڕێوەبرێت.
- نوێ: چاکنۆڵەکانی نوێکردنەوەی تایبەتمەندییەکان بۆ ئەندامبوونی دۆر پێویستە، بۆ ئەوەی گیتوێرییەکان دەتوانن ئۆتۆماتیک ڕێکخستنی دووبارەیان بەهێز بکەن کاتێک ڕێککەوتنی بڕین، سابسکرایب یان توکنەکە لە ناوچەکەی پێشوەختەکە نەبوو.
- نوێ: کارلێککردنەوەی loopback بە HMAC بۆ دروستکردنی سایتەکان کە لە چاوەڕوانیدا دەبن، بۆ ئەوەی پڕۆسەی دابینکردنی سایت لە کاتی تاخیربوونی کارەکانی پاشگر (background jobs) لە میزبانەکاندا زیاتر پێویست بێت.
- نوێ: خشتەی پەرەپێدەران (developer extension points) بۆ SSO URLs، ناوچەکانی سەرەکی فۆرم چاپی، و دروستکردنی تۆمارە ناوچە تۆمارە بە شێوەیەکی ئۆتۆماتیک.
- چاک: SSO زیاتر پێویست بوو لەسەر ناوچە مابووبووەکان، سەردانەکانی Brokerی ناڕێکخراو (anonymous broker visits)، خەرجکردنەوەی کارەکان (logout)، و تێکچوونی پەیوەندی نێوان پلۆگینەکان.
- چاک: دروستکردنی سایت لە چاوەڕوانیدا ئێستا لە فلیگەکانی بڵاوکردنەوەی کۆن بە شێوەیەکی باشتر دەست دەکات و کڕیاران لەسەر شاشەی دروستکردنی سایتەکە ناتوانن لەبێت.
- چاک: تۆمارە ناوچەکان بۆ ناوچە سەرەکی فۆرم چاپی، زیاتر دروست ناکرێنەوە، و کارەکانی پاشگرانی میزبان (host-provider background jobs) کە بەکار نایەنان، کاتێک هیچ یەکگرتووییەکە چالاک نییە، دەبینن.
- چاک: حاڵەتە سنووردارەکانی فۆرم چاپی، بڕین، ڕێکخستنی وشەکان (password reset)، پشکنینی ئیمیل، گۆڕینی قالب، تورەکان و داشبۆردی کڕیار زیاتر ناتوانن ڕێگری بکەن لە ڕێڕەوی کارکردنی ئاسایی کڕیار.
- چاک: ئیمیلە بڵاوکراوەکان ئێستا بە شێوەیەکی تایبەت دەبن بۆ گونجاوکردنی وەرگرتراوان، لە هەمان کاتدا لەو کاتی تێکچوونی SMTP/پلۆگینە فیتال (fatal errors) کە لیستەکانی وەرگرتراوان یان گواستنەوەی ئیمیل شکست دەدەن.
- چاک: نوێکردنەوەی ئەندامبوونی، نیشاندانی کاتی کۆتایی، و حاڵەتە سنووردارەکانی کۆکردنەوەی پارە ئێستا ڕێگری دەکەن لە کۆتایی هاتنی خەرجکردنەوە، کراش (crash) یان لەدەستدانی پارەی پێویست.
- باشتر: ناساندنی WordPress تا 7.0 تاقیکراوە، داراییەکانی Vue بۆ بەرهەمەکان لە سەرچاوەکانی npm دەدرێنەوە، و گۆڕینی Cypress end-to-end زیاتر پشکنین دەکات بۆ ڕێڕەووی چاپی، سیستەمسازی (setup)، SSO و گیتوێر.

## Version 2.12.0 — ڕێکەوترا لە ٢٠٢٦-٠٥-١٥ {#version-2120--released-on-2026-05-15}

- نوێ: Hostinger (hPanel) وەک پڕۆڤایدرێکی هاوڵگرتوو بە پشتیوانی نیشانراوە، لەگەڵ یەکخستنی مۆدالەکانی دۆمێین مۆدالەکان.
- نوێ: Site Exporter ئێستا کۆنترۆڵی گواستنەوەی تۆڕ (network import bundles) بۆ گەڕاندەوەی ورد و بەردەوام لەسەر قۆناغێکی تۆڕدا دەگرێت.
- چارەسەر: ئیمیلەکانی BCC broadcast ئێستا لە هەڵگرتنی ناوی گیرفانانی نیشانراوە بەکاردێن بۆ ڕێگرتن لە دیاریکردنی پیتەکانی گیرفانان.
- چارەسەر: بەڵامینی ماوەی کۆتایی ئەبێت لە کاتی شۆردن بە بەهایەکی نیتی، دووڵە (corrupted) نایەتەوە.
- چارەسەر: نوێکردنەوەی ماوەی ئەندازیاری Stripe ئێستا بە شێوەیەکی دروست دەکات کە پێداووکارییەکان پاک بکات و لە پڕۆسەی deleteDiscount API کە کۆن بوو بەکار دەهێنێت.
- چارەسەر: ڕێگریکردنی SSO لەسەر سایتەکان کە مۆدالەکانی دۆمێینان، ئێستا بۆ ڕێگرتن لە چۆڵکردنی نامۆیی نامێن (infinite redirect loops) سنووردار کراوە.
- چارەسەر: هەڵبژاردنی وێنەی سیستەمەکە لە ئامێرە پێکهاتەکانی Set up wizard ئێستا بە شێوەیەکی دروست دەکات کە مۆدال داتای بنەڕەتی دەگۆڕێت.
- چارەسەر: CLI ی Site Exporter ئێستا هەڵبژاردنی وردی سیستەمەکە (network site selection) بە شێوەیەکی دروست پارێزراوەتەوە.
- باشترکردن: wp-cli کە لە پڕۆژەی plugin دابەش بوو، لە پڕۆژەکەدا نەهات، ئەمەش گەورەبوونی قۆناغی plugin کەم کردەوە.

## Version 2.11.0 — ڕێکەوترا لە ٢٠٢٦-٠٥-١١ {#version-2110--released-on-2026-05-11}

- نوێ: Site exports ئێستا `index.php` خۆبەڕێوەبەر (self-booting) دەدات بۆ ئەوەی ZIPەکە بتوانێت لەسەر هاوڵگرێکی نوێ دامەزرێنرێت بەبێ پێویستی جێگیرکردنی pluginێکی جیاواز.
- نوێ: Network export ڕێگە دەدات بە بەڕێوەبەری کاران بۆ دابەشکردنی هەموو سایتەکانی توند لە یەک آرشایفێک لە دارالی Site Export admin page.
- نوێ: گۆڕینی پلانی Site Templates ئێستا بە زنجیرەیەکی پشتیوانی (fallback chain) جێگیرکراوە، کە بە شێوەیەکی دروست دەبێت بۆ سنووردارکردنی بەردەوامبوونی مۆدالەکانی template لەسەر سنووری پلانەکە.
- نوێ: ویرایتی فۆرم編輯ی Checkout کاتێک پڕۆژەیەکی پێویست بەبێ دیاریکردنی فیلدی دا زیاد بکرێت، ڕێدەوێت.
- نوێ: تابلۆی تنظیمەکانی Import/Export ئێستا ناو و جیاوازی وردی کاری خۆی دەگەیەنێت و ڕاستەوخۆ لینک دەکات بۆ ئامیرای Site Export.

- نوێ: لودای فایلەکانی sunrise بۆ MU-pluginەکانیsunrise کە تایبەتمەندیی نوێیان هەیە، زیاد کرا.
- باشترکراو: ڕێگاکانی ڕاپۆرتی هەڵە لە ڕێکخراوی Settings دابوو کەم بکرێنەوە.
- چارەسەر: کارتۆکی سایتەکە بۆ لاپەڕەی سوپاس (Thank-you page) — وێنەکە ئێستا بە باشی کراوە و لینکەکان بە شێوەیەکی دروست دیزاین کراون.
- چارەسەر: پێشکەشکار وێنە لە thum.io بگۆڕایەوە بۆ WordPress.com mShots.
- چارەسەر: ڕێکخستنی تۆمارکردن (Registration) و ڕۆڵی پیش‌داتوو (Default Role) ئێستا بە باشی دیار دەبن لە کاتی دامەزراندنی نوێ.
- چارەسەر: `get_site_url()` دوو هەڵەی پێشتر ناردووە کاتێک ناو دۆمەینەکە پورتێکی هەیە.
- چارەسەر: فایلەکانی مێدیا (media files) ئێستا بە دروستی کۆپی دەکرێنەوە کاتێک تنظیماتی `copy_media` خاوەنی نییە.
- چارەسەر: Object cache دوای نووسینی سایتەکە لەسەر تۆڕی کارکردن (network-activate sitemeta write) بە باشی دیار کراوە.
- چارەسەر: دۆمەینەکانی تایبەت (Custom domain) بە شێوەیەکی ئۆتۆماتیک بۆ سەرەکی (primary) گۆڕانەوە لە کاتی پشکنینی DNS بۆ دۆمەینە سێ بەشاییەکان.
- چارەسەر: ئەندامبوونی باکووری (Pending membership) کاتێک پارەی منداڵییەکە کۆتایی هات لە بەکارهێنان پاک بکرێتەوە.
- چارەسەر: چاککردنی توانای وشە (Password strength checker) دووبارە چالاک بوو دوای لێکردنەوەی پشکنینی لاگۆینەکە لە ناو فۆرمەکەدا.
- چارەسەر: گەڕانەوەی دۆخی لاپەڕەی نامەوەر (Infinite page reload) لەسەر لاپەڕەی سوپاس ڕابگرت کاتێک سایتەکە پێشتر دروستکراوە.
- چارەسەر: کارکردنی تۆمارکردنی WP core لە کاتی چالاککردنی plugin و هەڵبژاردنی Settings بە یەکەوە سینک دەکرێت.
- چارەسەر: گۆڕینی کۆتایی (Null expiration guard) زیاد کرا بۆ `calculate_expiration` بۆ ناساندنی PHP 8.4.
- چارەسەر: دووبارە تۆمارکردن لەکار دێت کاتێک کڕیارەکە بەشێوەیەکی چالاک ئەندامبوونی هەیە.
- چارەسەر: چاککردنی خاڵ (Null check) بۆ `date_expiration` لە فۆرمەکانی Checkout زیاد کرا.
- چارەسەر: پڕۆڤایشی سایتەکە بەهێزتر کراوە — سنووردارکردن، پێشنیاری ئەندامبوونی، و گۆڕینی دۆمەین.
- چارەسەر: لیبلی دۆخی پیشکەوتنی پێش دامەزراندن (Pre-install check status label) لە کاتی شکست دەبێت بە NOT Activated دروست کراوە.
- چارەسەر: دۆمەینی فۆرمەکە بۆ لینکەکانی پشکنینی ئیمەیڵ (email verification URLs) بەکاردراوە.
- چارەسەر: لاگۆین ئۆتۆماتیک دوای Checkout کاتێک هیچ فیلدێکی وشە نییە.
- چارەسەر: ئەندامبوونی بێ بەرامبەر (Free memberships) زیاتر کۆتایی هات لە بەکارهێنان — وەک ژیانێکی پڕ دەبینرێت.

- ڕێکخستن: گۆڕینی پەیامی تاییدکردنی ئیمایل (Email verification gate) بۆ ئەوەی سایتەکە تا کاتێک بەکارهێنەر ئیمایەک لەسەر تایید بکات، نیشان بدات.
- ڕێکخستن: ڕێگاکانی سەرەکی پەیجەکانی SES v2 API و ناساندنی تایبەتی (identity route) دروستکراونەوە.
- ڕێکخستن: هۆکەکە `wu_inline_login_error` لە بلوکێکی گەردوون (catch block) بۆ پێش نیشاندانی فرۆشتن (pre-submit) دابین کراوە.
