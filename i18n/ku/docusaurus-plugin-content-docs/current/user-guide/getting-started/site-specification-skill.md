# مهارتی پێویستیی سایت (Site Specification Skill)

**مهارت مشخص کردن سایت (Site Specification skill)** ڕێگایێکی ڕێکخراوە بۆ تۆمارکردنی ئامانجەکان، خەڵکی نیشانكردن و ناسنامەی براندەکەت لەسەر سایتەکەت. ئەم زانیارییە لە یادگاریی **site_brief** یەکدا هەڵدەگرێت، کە ئەژمارەکان لە هەموو جلسەکاندا بیردەکەونەوە بۆ دابینکردنی یارمەتییەکی جێگیر و پێکەوە بە وردبینی ناوەڕۆک.

## مهارتی پێشاری سایت چییە؟ (What is Site Specification?) {#what-is-site-specification}

مهارت مشخص کردن سایت پرۆسەی تۆمارکردنی:

- **پێناسەی سایتەکە**: سایتەکەت چی دەکات و بۆچی هەیە
- **خەڵکی ئامانجدار**: کی سەردانی سایتەکەت دەکات و چیت پێویستیانە
- **ناسنامەی براند (Brand identity)**: ڕەنگەکانت، دەنگیی تۆ و شێوازی بینراوەکەت
- **ئامانجەکانی کارگێڕیی**: سەرکەوتن بۆ سایتەکەت چۆن دەربکێت
- **پێکهاتەی ناوەڕۆک (Content structure)**: سایتەکەت چۆن ڕێکخراوە

ئەم پێناسەیە دەبێتە **site_brief** ی تۆ، یادگارییەکی جێگیر کە ئەژمارەکان بۆ تێگەیشتن لە پۆستەکە بەکاری دەهێنن.

## بۆچی مهارتی پێشاری سایت بەکاربهێنیت؟ (Why Use Site Specification?) {#why-use-site-specification}

### جێگیری لە هەموو جلساتدا (Consistency Across Sessions) {#consistency-across-sessions}

بێ ئەوەی site_brief هەبێت، هەر جارێک کە دەستپێدەکەیتەوە جلسەیەکی نوێ، دەبێت دووبارە ڕوون بکەیتەوە بۆچی سایتەکەت هەیە. بە وردی لەگەڵیدا، ئەژمارەکان بە خێرایی دەزانن:

- ئامانج و خەڵکی نیشانكردن و سایتەکەت
- ڕەنگ و دەنگی براندەکەت
- پێکهاتەی ناوەڕۆکەکەت
- ئامانجەکانی کارگێڕییەکەت

### پێشنیاری باشتر (Better Recommendations) {#better-recommendations}

ئەژمارەکان site_brief ی تۆ بەکاری دەهێنن بۆ:

- پێشنیارکردنی تایبەتمەندییەکان کە لەگەڵ مەبەستی سایتەکەت ڕێکدەخەن
- پێشنیارکردنی پێکهاتەی ناوەڕۆکێک کە لەگەڵ ئامانجەکانت یەک دەبێت
- پێشنیارکردنی دیزاینێک کە لەگەڵ ناسنامەی براندەکەت یەک دەبێت
- ڕێگرتن لە پێشنیاری تایبەتمەندییە نادەروستەکان

### سەرنجڕاکێشانێکی خێراتر (Faster Onboarding) {#faster-onboarding}

ئەژمارەکان نوێنەرایەتی جاران (یان ئەژمارەکان لە جلسەکانی نوێدا) دەتوانن بە خێرایی ئامادە بن بۆ کارکردن بە بخوێنانی site_brief ی تۆ، لەبری پرسیارکردنی ڕوونكردنەوە.

## دەستپێکردنی مهارتی پێشاری سایت (Initiating Site Specification) {#initiating-site-specification}

### لە ماوەی سەرەتای Theme Builderدا (During Theme Builder Onboarding) {#during-theme-builder-onboarding}

مهارت مشخص کردن سایت بە شێوەیەکی ئۆتۆماتیک لە ماوەی **Theme Builder onboarding flow** دا دەستپێدەکەیت. ئەژمارەکان پرسیار دەکەن و site_brief ی تۆ دروست دەکەنەوە.

### دەستپێکردنی دەستی (Manual Initiation) {#manual-initiation}

تۆ دەتوانی هەر کاتێک دەستپێبکەیت بە دیاریکردنی تایبەتمەندی سایتەکەت:

```
"بۆچی منو پێداویستی بە دیاریکردنی تایبەتمەندی سایتەکەم هەیە؟"
```

یان

```
"یارمەتیم بدە دروستکردنی پوختەی سایت بکەم."
```

## پڕۆسەی دیاریکردنی تایبەتمەندی سایت (Site Specification Process) {#the-site-specification-process}

### هەنگاوی یەکەم: مەبەستی سایتەکە (Site Purpose) {#step-1-site-purpose}

ئەژمارەکە پرسیار دەکات:

```
مەبەستی سەرەکی سایتەکەت چییە؟
- دوکان بۆ فرۆشتنی شتەکان (E-commerce store)
- ڕۆژنامەی کارێشان یان شوێنی ناوەڕۆک (Blog or content site)
- پۆرتفۆلیۆ یان نیشاندانی کارەکان (Portfolio or showcase)
- بەرنامەی SaaS (SaaS application)
- کۆمەڵگە یان فۆ럼 (Community or forum)
- دیکە: [بشاردە]
```

تۆ دەتوانی یەکێک لەو پۆلێسە هەڵبگریت یان مەبەستت بە وشە بۆت بنووسیت.

### هەنگاوی دووەم: خەڵکی ئامانجدار (Target Audience) {#step-2-target-audience}

```
خەڵکی سەرەکی تۆ چییە؟
- بەکارهێنەران / خەڵکی گشتی
- پیشەوانانی کار و بازرگانی
- پەرەپێکراوەکان / بەکارهێنەرانی تەکنیکی
- خوێندکاران / مامۆستایان
- دیکە: [بشاردە]

ئامانجە سەرەکییەکانی ئەوان چیین؟
```

### هەنگاوی سێیەم: ناسنامەی براند (Brand Identity) {#step-3-brand-identity}

```
ڕەنگەکانت لە براندەکەت چیین؟
- ڕەنگی سەرەکی: [دیاریکردنی ڕەنگ یان کۆدێکی هێز]
- ڕەنگی دووەم: [دیاریکردنی ڕەنگ یان کۆدێکی هێز]
- ڕەنگی پاشەڕۆش (Accent color): [ئەویش ئەگەر هەبێت]

چۆن دەتوانیت تोनی براندەکەت وەسف بکەیت؟
- پیشەیی / کارگێڕیی
- ئایینی / هونەری
- وشکە / ئاسایی
- سادە / مۆدێرن
- گەرم / دوستانه

```

### هەنگاوی چوارەم: ئامانجەکانی بازرگانی (Business Goals) {#step-4-business-goals}

```
سەرکەوتن بۆ سایتەکەت چۆن دەردەکەوێت؟
- بەدەستهێنانی سەرچاوەکان (Leads generate)
- فرۆشتنی بەرهەمەکان
- دروستکردنی کۆمەڵگە
- بەشی زانین و ناساندنی خۆت
- دامەزراندنی ڕەوت (Authority establish)
- دیکە: [بشاردە]

مەرجی سەرەکی تۆ چییە؟
- دراوە (Revenue)
- کارکردنەوەی بەکارهێنەران (User engagement)
- گەیاندنی ناوەڕۆک (Content reach)
- گواستنەوە بۆ خەرجکردن (Conversions)
- دیکە
```

### هەنگاوی پێنجەم: پێکهاتەی ناوەڕۆک (Content Structure) {#step-5-content-structure}

```
ناوەرۆکت چۆن ڕێکخراوە؟
- ڕاستەوخۆ (Flat - هەموو ناوەڕۆکان لە یەک ئاستن)
- پیاسەت (Hierarchical - پۆل و زیرەکی پۆل)
- کاتوار (Chronological - بە شێوازی ڕۆژنامەی کارێشان)
- پێکەوە بە دروستکردنی بەرهەم (Product-based - کاتالوگ)
- دیکە: [بشاردە]

تۆ چ جۆرە ناوەڕۆکەکان بەکاری دەهێنیت؟
- بابەتەکانی ڕۆژنامەی کارێشان (Blog posts)
- لاپەڕەکانی بەرهەم (Product pages)
- لێکۆڵینەوەی حاڵەتەکان (Case studies)
- ڕێنمایی و پڕۆسەکان (Documentation)
- ڤیدیۆکان
- دیکە
```

## بیرگە (Memory) ی `site_brief` ی تۆ {#your-sitebrief-memory}

پاش تەواوکردنی دیاریکردنی تایبەتمەندی سایت، زانیارییەکانی تۆ بە شێوەیەکی **site_brief** لە یادەوەری ئەژمارەکەدا هەڵدەگیرێت. ئەمە تۆماری ڕێکخراوەییە کە پێکهاتەی زیرەکی پۆل و سەرەکیان دەگرێت:

## مشاهده و به‌روزرسانی خلاصه سایت (site_brief) شما {#viewing-and-updating-your-sitebrief}

### مشاهده خلاصه سایت خود {#view-your-sitebrief}

از دستیار بپرسید:

```
"نمایش خلاصه سایت من را بده"
```

یا

```
"درباره سایت من چه می‌دانی؟"
```

دستیار مشخصات ذخیره شده شما را نمایش می‌دهد.

### به‌روزرسانی خلاصه سایت خود {#update-your-sitebrief}

اگر سایت شما تغییر کرد، می‌توانید آن را به‌روز کنید:

```
"خلاصه سایت من را به‌روز کن: ما اکنون به مشتریان B2B هدف داریم"
```

یا

```
"مشخصات سایت من را تازه کن"
```

این کار مهارت Site Specification را با اطلاعات موجود شما به عنوان نقطه شروع دوباره اجرا می‌کند.

## نحوه استفاده عوامل (Agents) از خلاصه سایت (site_brief) {#how-agents-use-sitebrief}

### توصیه‌های طراحی {#design-recommendations}

وقتی درخواست تغییر طراحی می‌کنید، عوامل به خلاصه سایت شما مراجعه می‌کنند:

```
شما: "طراحی مجدد صفحه اصلی من را انجام بده"
عامل: "بر اساس خلاصه سایت شما، شما در حال هدف قرار دادن زنانی با سن ۲۵ تا ۴۵ سال با لحنی گرم و هنری هستید. من طرحی ایجاد خواهم کرد که بر جواهرات دست‌ساز و ارزش‌های پایدار شما تأکید کند."
```

### پیشنهادها برای ویژگی‌ها (Feature Suggestions) {#feature-suggestions}

عوامل ویژگی‌هایی را پیشنهاد می‌دهند که با اهداف شما همخوانی دارد:

```
شما: "چه ویژگی‌هایی باید اضافه کنم؟"
عامل: "برای یک فروشگاه تجارت الکترونیک متمرکز بر وفاداری مشتری، من برنامه وفاداری، نظرات مشتریان، خبرنامه ایمیلی و توصیه‌های محصول را پیشنهاد می‌کنم."
```

### ساختار محتوا (Content Structure) {#content-structure}

عوامل سازماندهی محتوا را بر اساس ساختار شما پیشنهاد می‌دهند:

شێوەی چۆن بلاگەکەم ڕێک بخەم؟
ئەژمار: "لەبەر ئەوەی تۆ بەکارهێنانی پۆلۆگێکی بەرهەم (product catalog) لەگەڵ بلاگت دەکەیت، پێم دەڵێم پۆستەکان بەپێی پۆلۆگەکانی بەرهەم ڕێک بخەیت و 'ڕێنمایییە ستاایلکردنی' (styling guides) دروست بکەیت کە چەندین بەرهەم یەک لەسەر نیشان بدەن."

## باشترین شێوازی کارکردن (Best Practices) {#best-practices}

### ورد و تایبەت ببن (Be Specific) {#be-specific}

لەبری "هەڵمەزاری گشتی" (general audience)، خەڵکی ڕاستیەکەت وەسف بکە:

- ✓ "زوانە ٢٥-٤٥ ساڵان، گرنگی بە فاشنێکی پاشەکەوت (sustainable fashion) دەدەن."
- ✗ "هەر کەسێک"

### بەردەوام به‌روزرسیک بکە (Update Regularly) {#update-regularly}

کاتێک سایتەکەت گەشەی کرد، `site_brief` ی خۆت بەردەوام ببنە نوێ:

- کاتێک بۆ خەڵکی نوێیەک دەگۆڕیت
- کاتێک لایەنی بەرهەمێکی نوێ زیاد دەکەیت
- کاتێک ناسنامەی براندەکەت دەگۆڕێت
- کاتێک ئامانجەکانی کارتان دەگۆڕێن

### شێواز و دەستەواژەیەکی یەک ببن (Use Consistent Terminology) {#use-consistent-terminology}

لە هەموو چاوکاندا یەک وشە بەکاربهێنە:

- ✓ هەمیشە بڵێ "سەرچاوەی ڕەنگەوتوو" (sustainable jewelry) (نەخێر "سەرچاوەی دووەم" و "پێشوەختەکانی سەوز")
- ✓ هەمان شێوە بەکاربهێنە بۆ ئاماژەدان بە خەڵکی خۆت

### پۆلێسی دەنگ و ڕوونكردنەوەی پێویست ببن (Include Context) {#include-context}

زانیارییەکی پاشەکەوت بدات کە یارمەتی ئەژمارەکان دەدات تێبگەن چۆن بڕیار دەدەیت:

- "ئەو کەسانەی کوالێتی زیاتر لەราجەدا بەڵگە دەدەن."
- "خەڵکی ئێمە زۆر ئامادە بۆ تەکنەلۆژیا و دیزاینی نوێ چاوەڕوان دەکەین."
- "ئێمە کۆمپانیای بچووکێکی سەرەتایی (bootstrapped startup) یین، بۆیە پێویستە چارەسەرەکانی ئێمە کارپێکراون بن."

## پەیوەندی بە ڕێکخستنی Theme Builderەوە {#relationship-to-theme-builder-onboarding}

فێری `Site Specification` لە ناو **Theme Builder onboarding flow** داتووتی. کاتێک فێربوونی تەواوت کرد، `site_brief` ی تۆ بە شێوەیەکی ئۆتۆماتیک دروست دەبێت بە زانیارییەکانی ئەو شوێنەی پێداویت.

دەتوانیش `Site Specification` بە شێوەیەکی سەربەخۆ ئەنجام بدەیت ئەگەر دەتەوێت:

- دوای ڕێکخستنی سەرەتایی، وردکاری بکەیت لە تایبەتمەندیی خۆت
- وەک سایتەکە گەشە دەکات، `site_brief` ی خۆت بەردەوام ببنە نوێ
- پێش دەستپێکردنی Theme Builder، وردکارییەکی ورد دروست بکەیت

## چارەسەرکردنی کێشەکان (Troubleshooting) {#troubleshooting}

**سایت_بریفەکەم بەکارنەهێنێت**
- دڵنیا بە لەوەی ئەژمارەکە دەستگەیشت بە بیرەوەری (memory)
- داوای لێبکەیت کە "site brief ی مندا بیرەوە" بکات (recall)
- چاک بکە کە بیرەوەری لە ڕێکخستنەکەت چالاکە (enabled)

```php
// Custom logic when checkout completes
```

**دۆزیت نوێ دەستپێکردن لەگەڵ یەک `site_brief` نوێ**
- بە ئەژمارەکە بپرسە: "Site brief-م پاک بکە و نوێی دەست پێبکە"
- پاشان Site Specification (تایبەتمەندیی سایت) دووبارە چاودێری بکە

**ئەژمارەکە پێشنیاری ئەوە دەکات کە لەگەڵ `site_brief`-م یەک ناکات**
- بە ئەژمارەکە بپرسە: "Site brief-م بیرەوە و داوای دروستکردنێکی نوێ بکە"
- ئەگەر `site_brief`-ت پێشەنگ بوو، دەستکاری بکە بۆ نوی کردنەوەی شتەکانت
- لە داواکارییەکانت زانیاری زیاتری بدە

## هەنگاوەکانی دوای ئەمە (Next Steps) {#next-steps}

پاش ئەوەی تایبەتمەندی سایتەکەت دیار کردووە:

1. **بۆ Theme Builder بەکاربهێنە**: یەکەم قالبێکی تایبەت بۆ سایتت دروست بکە لەسەر بنەمای `site_brief`-ت
2. **دیزاینی پاشکەوتە**: توانای Design System Aesthetics (ڕێنمایی دیزاین سیستمی) بەکاربهێنە بۆ کارکردن لەسەر دیزاینی ورد و جزییەتانە
3. **گرنگی ناوەڕۆکەکە ڕێکبخە**: داوای پێشنیاری ساختەری ناوەڕۆک لە ئەژمارەکانت بکە
4. **تایبەتمەندییەکان دروست بکە**: داواکارییەک بۆ تایبەتمەندییەکان بکە کە لەگەڵ ئامانجەکانی کارگێڕیت یەک بێت
