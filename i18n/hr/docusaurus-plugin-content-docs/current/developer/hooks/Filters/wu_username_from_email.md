# Filtar: wu_username_from_email

Filtrirajte korisničko ime novog kupca.

## Parametri {#parameters}

| Naziv | Vrsta | Opis |
|------|------|-------------|
| $username | `string` | Korisničko ime kupca. |
| $email | `string` | Adresa e-pošte novog kupca. |
| $new_user_args | `array` | Niz argumenata novog korisnika, možda uključujući ime i prezime. |
| $suffix | `string` | Dodajte niz korisničkom imenu kako bi bilo jedinstveno. |
