# Filtar: wp_ultimo_render_vars

Omogućuje developerima plugina globalno dodavanje dodatne varijable u kontekst renderiranja.

## Parametri {#parameters}

| Naziv | Tip | Opis |
|------|------|-------------|
| $args | `array` | Niz koji sadrži varijable proslijeđene pozivom za renderiranje. |
| $view | `string` | Naziv prikaza koji će se renderirati. |
| $default_view | `string` | Naziv fallback_view |

### Od verzije {#since}

- 2.0.0
