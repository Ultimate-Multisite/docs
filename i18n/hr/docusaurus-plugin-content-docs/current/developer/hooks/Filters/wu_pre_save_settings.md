# Filtar: wu_pre_save_settings {#filter-wupresavesettings}

Omogućuje developerima filtriranje postavki prije spremanja od strane Ultimate Multisite.

## Parametri {#parameters}

| Naziv | Tip | Opis |
|------|------|-------------|
| $settings | `array` | Postavke koje će biti spremljene. |
| $settings_to_save | `array` | Nove postavke za dodavanje. |
| $saved_settings | `array` | Trenutačno spremljene postavke. |

- 2.0.18
