# Filtar: wu_magic_link_enforce_user_agent {#filter-wumagiclinkenforceuseragent}

Filtrira treba li nametnuti provjeru korisničkog agenta.

Postavite na false kako biste omogućili da tokeni rade u različitim preglednicima/uređajima. To smanjuje sigurnost, ali povećava upotrebljivost.

## Parametri {#parameters}

| Naziv | Vrsta | Opis |
|------|------|-------------|
| $enforce | `bool` | Treba li nametnuti podudaranje korisničkog agenta. |
