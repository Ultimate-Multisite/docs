# Agenti za podršku {#support-agents}

Upravljanje cijelom mrežom samostalno nije lak zadatak. Kako vaša mreža počne rasti, počet ćete primjećivati sve veću potražnju za podrškom. Bit će dobra ideja uključiti ljude koji će vam pomoći smanjiti radno opterećenje. Međutim, davanje punog pristupa/dopuštenja rizično je i može ugroziti vašu mrežu. Tu dolazi ovaj dodatak za agente za podršku.

Ovaj dodatak omogućuje vam stvaranje nove vrste WordPress korisnika pod nazivom agent za podršku i daje vam potpunu kontrolu nad time koja dopuštenja taj korisnik ima. Dodjeljivanje dopuštenja jednostavno je poput uključivanja prekidača, a isključivanje je jednako jednostavno ako ga odlučite ukloniti.

## Značajke {#features}

- Detaljan popis WordPress i Ultimate Multisite dopuštenja
- Jednostavno omogućavanje i onemogućavanje pristupa
- Stvorite neograničen broj računa agenata

## Instalacija {#installation}

1. Prenesite 'ultimate-multisite-support-agents' u direktorij '/wp-content/plugins/'
2. Aktivirajte plugin putem izbornika 'Dodaci' u WordPressu
3. Slijedite čarobnjak korak po korak kako biste postavili plugin
