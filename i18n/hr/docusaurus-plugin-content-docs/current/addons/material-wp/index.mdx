# Material WP {#material-wp}

Administratorsko sučelje WordPressa isto je već godinama. No jeste li se ikada zapitali kako bi izgledala Material Design verzija administratorskog sučelja? 

Material WP je administratorska tema razvijena kako bi vam pružila taj izgled i dojam, pomažući vam da svojim klijentima omogućite prilagođeniji Dashboard.

## Značajke {#features}

- Podrška za prilagođeni CSS
- Jednostavne postavke stiliziranja
- Promjena redoslijeda izbornika
- Jednostavan uvoz/izvoz

## Dokumentacija {#documentation}

**Prilagodba vaše administratorske ploče uz Material WP**

## Instalacija {#installation}

1. Prenesite 'material-wp' u direktorij '/wp-content/plugins/'
2. Aktivirajte dodatak putem izbornika 'Dodaci' u WordPressu

## Vodič za upotrebu {#usage-guide}
