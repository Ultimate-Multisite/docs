# Dnevnik promjena za Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* Novo: Preglednik mrežnih predložaka — pregledavajte i pretpregledajte mrežne predloške s panelom uživo prije kupnje.
* Novo: Odabir mrežnog predloška pri checkoutu — klijenti mogu odabrati mrežni predložak tijekom tijeka signup checkouta.
* Novo: Cross-network magic link SSO — klijenti mogu neprimjetno pristupati podmrežama putem magic-link autentikacije.
* Novo: Kloniranje mrežnih medija — mediji stranice ispravno se dupliciraju pri kloniranju mreža iz predloška.
* Popravak: Kloniranje mreže sada radi bez dodatka za multi-tenancy; super admin ovlasti ponovno se dodjeljuju nakon kloniranja.
* Popravak: Stvaranje mreže sada je kompatibilno s WooCommerce i svim payment gatewayjima.
* Popravak: Broj porta sada je uključen u generiranje rezervne domene.
* Popravak: CSS odabira predloška premješten je u učitani stylesheet radi ispravnog prikaza na svim temama.
* Popravak: Zaštita od nedefiniranog ključa `network_id` u opsegu upita stranice.
* Popravak: Provjera mrežnih dopuštenja prije promjene konteksta sprječava pogreške svojstva tablice `wpdb`.
