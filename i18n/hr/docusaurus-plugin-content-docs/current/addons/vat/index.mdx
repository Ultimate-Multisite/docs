# VAT

Multisite Ultimate: europski PDV

Dodaje dodatne značajke i alate zadanom rukovanju porezima u Multisite Ultimate kako bi bolje podržao europski PDV.
Automatski dodaje stope EU PDV-a na stranicu Porezne stope iz

## Instalacija

1. Prenesite 'ultimate-multisite-vat' u direktorij '/wp-content/plugins/'
2. Aktivirajte plugin kroz izbornik 'Dodaci' u WordPress
3. Idite na Ultimate Multisite > Postavke > Porezi i omogućite EU PDV.
4. Kliknite na Porezne stope i Ažuriraj stope EU PDV-a pri dnu te spremite porezne stope.
5. Gotovo!
