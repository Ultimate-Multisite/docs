# Fluent Forms

Ovaj dodatak omogućuje dodavanje ograničenja za Fluent Forms na Multisite Ultimate u proizvodima, članstvima ili web-lokacijama.
Fluent Forms koristi prilagođene tablice umjesto prilagođenih vrsta objava, pa uobičajene opcije ograničavanja ne rade s Fluent Forms.

## Značajke {#features}

- Dodajte ograničenja za Fluent Forms ili Conversational Fluent Forms
- Kupci dobivaju prijateljsku poruku koja ih potiče da nadograde svoj plan kada dosegnu ograničenje
