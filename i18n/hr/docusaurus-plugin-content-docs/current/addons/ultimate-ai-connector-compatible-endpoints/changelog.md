# Dnevnik promjena za Ultimate AI Connector za kompatibilne krajnje točke {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## Verzija 2.0.0 — Objavljeno 2026-04-24 {#version-200--released-on-2026-04-24}

* Novo: Podrška za više pružatelja — konfigurirajte više AI krajnjih točaka i usmjeravajte zahtjeve uz automatsko prebacivanje na rezervnog pružatelja.
* Ispravak: Integracija SDK-a za više pružatelja s ispravnim ID-jevima pružatelja, URL-ovima za registraciju i opsegom HTTP filtara po pružatelju.
* Ispravak: Nova kartica pružatelja sada se automatski proširuje pri dodavanju; zaobilaženje predmemorije skripti pri ažuriranju dodatka.
* Ispravak: Dinamički prostor naziva klase pružatelja za eval() i ponašanje automatskog proširivanja za novododane pružatelje.
* Ispravak: Koristi stabilne komponente Card/CardBody/CardHeader/CardDivider (više nisu eksperimentalne) za kompatibilnost s WordPress 6.9+.
* Ispravak: Zamijenjen nedefinirani DragHandle Unicode ikonom hvataljke za povlačenje pružatelja radi promjene redoslijeda.
* Ispravak: Uklonjen blokirajući HTTP zahtjev koji se pokretao pri svakom učitavanju stranice.
* Poboljšano: GitHub Actions tijekovi rada nadograđeni na Node.js 24.
