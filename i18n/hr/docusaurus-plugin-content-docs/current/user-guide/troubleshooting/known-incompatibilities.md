# Poznate nekompatibilnosti s drugim pluginovima {#known-incompatibilities-with-other-plugins}

Nakon instalacije i aktiviranja Ultimate Multisite, možete naići na probleme zbog neispravno riješenih nekompatibilnosti s drugim pluginovima.

Aktivno radimo na smanjenju ove liste s svakom novom verzijom, ali ako je neki plugin naveden ovdje, možda ćete morati ga deaktivirati kako biste imali potpuno funkcionalnu Ultimate Multisite Network.

Evo liste tih problema i što trebate učiniti u takvim slučajevima:

**Neki WPMU DEV pluginovi** Neki pluginovi koje više održava WPMU DEV poznati su da uzrokuju zaustavljanje toka registracije Ultimate Multisite na posljednjem koraku (korisnik ne može kreirati račun). Evo liste pluginova koji uzrokuju te probleme:

  * Signup Language: _Ultimate Multisite ima dodatni modul koji nudi istu funkcionalnost, zvan Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Ovo je ugrađeno u Ultimate Multisite, tako da nema potrebe za dodatnim pluginom._

  * New Blog Templates: _Ovo je ugrađeno u Ultimate Multisite, tako da nema potrebe za dodatnim pluginom._

  * Set Password on Multisite Blog Creation: _Ovo je ugrađeno u Ultimate Multisite, tako da nema potrebe za dodatnim pluginom._

  * Signup TOS: _Ovo je ugrađeno u Ultimate Multisite, tako da nema potrebe za dodatnim pluginom._

  * Anti-Splog: Iako je manje moćan, ovaj plugin je također _ugrađen u Ultimate Multisite._

**Napomena**: Svi drugi WPMU Dev pluginovi se mogu normalno koristiti uz Ultimate Multisite. Primjeri uključuju _Smush_, _Forminator_, _Defender_ itd.
