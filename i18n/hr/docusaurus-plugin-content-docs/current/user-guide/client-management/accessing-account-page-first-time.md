# Kako mogu prvi put pristupiti svom korisničkom računu? {#how-can-i-access-my-account-page-for-the-first-time}

Dobrodošli u Ultimate Multisite!

Pristupite svom korisničkom računu putem linka koji ste dobili u e-mailu nakon kupnje našeg plugin-a:

<!-- Screenshot unavailable: Account page email with access link -->

Koristite e-mail adresu koju ste naveli prilikom kupnje i lozinku koji ste dobili u e-mailu.

<!-- Screenshot unavailable: Login page with email and password fields -->

Budući da se radi o vašem prvom prijavljivanju na ovaj račun, bit će vam traženo promijeniti lozinku. Promijenite je i kliknite na dugme _Save_ (Spremi).

<!-- Screenshot unavailable: Change password form with Save button -->

Na svom korisničkom računu možete upravljati svim podacima o računu, kao što su fakturiranje, obnove, otkazivanja i mnogo drugih stvari.
