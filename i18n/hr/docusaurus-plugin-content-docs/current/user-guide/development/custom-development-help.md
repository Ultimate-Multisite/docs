# Kako dobiti prilagođenu razvojnu pomoć od Codeable-a?

Ponekad nam stvarno treba neka sjajna prilagodba kako bismo našim radnim tokovima na mreži bili točno onakvi kakvi želimo. Ako tražite upravo to, preporučujemo [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), mjesto gdje možete pronaći iskusne developere s iskustvom u Ultimate Multisite-u.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) je platforma koja povezuje kupce s kvalificiranim freelanceriom za WordPress, sposobnim raditi na projektima bilo koje veličine. Imaju pažljiv proces prijave kako biste bili upareni s stručnjacima koji su specifično kvalificirani za zadatak koji imate na umu.

### Kako kreirati projekt u Codeable-u?

Prvo, idite na [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) i kliknite na dugme **Start a Project** (Počni projekt).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Sada, da biste podnijeli projekt, kliknite na lijevo za **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

U polju "I need help with" (Potrebno mi je pomoć s) odaberite **Customizing** (Prilagođavanje):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

U polju "My" (Moj), odaberite **Plugin** (Dodatak):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Zatim kliknite na dugme **Continue** (Nastavi).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Sada je vrijeme da opišete svoj projekt i potrebe najboljim putem koji možete. Pokušajte dati što više informacija!

<!-- Screenshot unavailable: Codeable project description form -->

Odgovorite na sljedeća pitanja kako biste pomogli s cijenom.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Nakon toga, ako još nemate [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) račun, stvorite jedan i kliknite na gumb **Publish My Project** (Objavi moj projekt).

<!-- Zaslon nije dostupan: kreiranje računa Codeable i gumb Publish My Project -->

Čim objavite svoj [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projekt, platforma će vas povezati s programerima koji odgovaraju potrebnom skupinu vještina za vaš projekt.

Ovaj proces će vam pomoći da pronađete pravi čovjek za posao, onaj s iskustvom u WordPressu i Ultimate Multisite!

Minimalna cijena Codeable-a je 70–120 USD po satu. Procjene će se mijenjati ovisno o obimu posla, složenosti i željama projekta.

Čim dobijete procjenu, moći ćete angažirati programera koji najbolje odgovara vašim potrebama i zajedno započeti svoj projekt!
