# Prilago vaš Admin Panel s PRO temom {#customizing-your-admin-panel-with-pro-theme}

Često, kad radimo za klijente, moramo im ponuditi prilagođeno iskustvo u admin paneli WordPressa. Ako tražite profesionalniji izgled i dojam, PRO Theme vam može pomoći.

PRO Theme je admin tema razvijena da vam olakša prilagođavanje dashboarda vaših klijenata. Također je dio jezgre premium pluginova Ultimate Multisite.

Nakon instalacije, idite na stranicu _Plugins_ i kliknite za aktivaciju na svojoj mreži (Network).

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktivirajte ga na cijeloj mreži na stranici Plugins._

_**NAPOMENA:**_ _Ako ga samo instalirate, on će se pojaviti kao opcija za vašeg klijenta koju možete označiti da dodajete temu samo tom korisniku._

Sada imate potpuno redizajnirani admin interfejs! Također imate novo meni na lijevoj koloni pod nazivom PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Potpuno novi izgled za vaš Admin Panel._

Na ovom meniju ćete imati pristup pet kartica (tabova):

<!-- Screenshot unavailable: PRO Theme Styling Settings tab -->

_Styling Settings_ (Podešavanja izgleda)

<!-- Screenshot unavailable: PRO Theme Functionality tab -->

_Functionality_ (Funkcionalnost)

<!-- Screenshot unavailable: PRO Theme Custom CSS tab -->

_Custom CSS_ (Prilagođeni CSS)

<!-- Screenshot unavailable: PRO Theme Import/Export tab -->

_Import/Export_ (Uvoz/Izvoz)

<!-- Screenshot unavailable: PRO Theme Activate tab -->

_Activate PRO Theme_ (Aktiviraj PRO Theme)

U svakoj od njih možete omogućiti i onemogućiti elemente izgleda, dozvoliti ili blokirati funkcionalnosti, te pružiti svojim klijentima prilagodbu koju možda traže u vašoj mreži stranica.
