# Postavljanje oglasa na vašu mrežu s Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector je **besplatan dodatak** koji vam daje alate za postavljanje oglasa na vaše planove direktno s stranice za uređivanje.

Nakon instaliranja Ultimate Multisite: Ad Injector-a, idite na vašu _Plugins_ stranicu kako biste ga aktivirali.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Kliknite na gumb Network Activate_

Kada je dodatak aktivan, u bloku _Advanced Options_ na ekranu uređivanja plana pojavi se nova opcija pod nazivom _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Sada možete uređivati postavke oglasa za svaki od svojih planova_

Na toj kartici, moći ćete dodati kodove oglasa za objave i stranice u prednjem dijelu (front-end), prije i poslije sadržaja.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Nakon što umetnete kod oglasa, on će se prikazati na objavama i stranicama tog specifičnog plana._

Možete uključiti oglase i u zaglavlje (header) administrativnih stranica za korisnike tog plana.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Idite na opciju Admin Panel Ad Code ako želite prikazati oglase na pozadini (back-end) vašeg korisnika._

### Više od oglašavanja {#beyond-advertising}

Možete koristiti Ultimate Multisite: Ad Injector i za pružanje podrške visokim planovima.

Samo idite na opciju _Admin Panel Ads_ i dodajte HTML kod za vaš alat za podršku.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Snimak ekrana nedostupan: Widget za korisnu podršku prikazan na admin paneli za određeni plan -->

_Dodajte korisničku podršku određenim planovima_

Na ovaj način ljudi mogu doći do vas direktno iz svoje Admin Paneli!
