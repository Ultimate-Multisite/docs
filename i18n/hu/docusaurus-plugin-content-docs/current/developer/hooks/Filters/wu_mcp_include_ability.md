# Szűrő: wu_mcp_include_ability

Ez a szűrő lehetővé teszi, hogy átírja az MCP képesség hozzáadásának döntését.

## Paraméterek

| Neve | Típus | Leírás |
|------|------|-------------|
| $include | `bool` | Megtekinti, hogy a képességet be kell-e hagyni. |
| $ability_id | `string` | A képesség azonosítója. |
