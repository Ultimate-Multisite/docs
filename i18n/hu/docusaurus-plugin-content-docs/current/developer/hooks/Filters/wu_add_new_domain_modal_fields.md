# Szűrő: wu_add_new_domain_modal_fields

Szűrőzi az új tartomány hozzáadási modális ablak mezőit.

Megengedi az addonoknak (például a Domain Sellernek) módosítani vagy cserélni a tartomány regisztrációs mezőit.

## Paraméterek {#parameters}

| Neve | Típus | Leírás |
|------|------|-------------|
| $fields | `array` | Az űrlid mezői. |

### Desde {#since}

- 2.1.0
