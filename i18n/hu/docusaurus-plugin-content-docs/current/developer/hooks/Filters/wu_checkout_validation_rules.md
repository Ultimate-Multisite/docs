# Szűrő: wu_checkout_validation_rules {#filter-wucheckoutvalidationrules}

Ez a szűrő lehetővé teszi a plugin fejlesztők számára, hogy a validációs szabályokat szűrhessék.

## Paraméterek {#parameters}

| Neve | Típus | Leírás |
|------|------|-------------|
| $validation_rules | `array` | Az használni kívánt validációs szabályok. |
| $checkout | `\Checkout` | A checkout osztály. |

### Since {#since}

- 2.0.20
### Source {#source}

Definíciója található [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L2092) fájlban, 2092-sorban.
