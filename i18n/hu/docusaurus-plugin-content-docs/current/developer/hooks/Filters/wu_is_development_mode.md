# Szűrő: wu_is_development_mode

Ez a szűrő lehetővé teszi a plugin-fejlesztők számára, hogy további teszteket adjanak hozzá a fejlesztői módhoz.

## Paraméterek {#parameters}

| Neve | Típus | Leírás |
|------|------|-------------|
| $is_development_mode | `bool` | Az aktuális fejlesztési állapot. |
| $site_url | `string` | Az oldal URL-e. |

### Since {#since}

- 2.0.0
### Source {#source}

Definálva a [`inc/domain-mapping/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/domain-mapping/class-helper.php#L63) fájlban, 63-án soron.
