# Szűrő: wu_membership_update_forms {#filter-wumembershipupdateforms}

Lehetővé teszi a fejlesztőknek új frissítő űrlisi slugok hozzáadását.

## Paraméterek {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $slugs | `array` | Egy lista az elkerülendő űrlisi slugokról. |

### Forrás {#source}

Definálva a [`inc/ui/class-checkout-element.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/ui/class-checkout-element.php#L709) fájlban, 709-sorban.
