# Filter: wu_current_set_customer

Ez a filter lehetővé teszi a fejlesztők számára, hogy módosítsák az alapértelmezett viselkedést, és másképp állítsák be az aktuális vásárlót.

## Paraméterek

| Neve | Típus | Leírás |
|------|------|-------------|
| $customer | `\WP_Ultimo\Models\Customer` | Az aktuális vásárló, amelyet beállítani szeretne. |
| $ | `self` | Az aktuális osztály példányát. |
