# Szűrő: wu_core_installer_install_sunrise {#filter-wucoreinstallerinstallsunrise}

Megengedi a hosting szolgáltatóknak, hogy másképp telepítsék a konstansot.

Ha `true`-t döndet, megakadályozza az Ultimate Multisite-ot abban, hogy a `wp-config` fájlba írjen.
