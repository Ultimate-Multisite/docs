# Szűrő: wu_current_set_membership

Megengedi a fejlesztőknek, hogy módosítsák az alapértelmezett viselkedést, és másképp állítsák be az aktuális tagsági szintet.

## Paraméterek {#parameters}

| Neve | Típus | Leírás |
|------|------|-------------|
| $membership | `\WP_Ultimo\Models\Membership` | Az aktuális tagsági szint, amelyet beállítani kell. |
| $ | `self` | Az aktuális osztály példányát. |

### Desde {#since}

- 2.0.18
