# Szűrő: wu_current_set_site

Ez a szűrő lehetővé teszi a fejlesztők számára, hogy módosítsák az alapértelmezett viselkedést, és eltérő helyre állítsák az aktuális helyszínt.

## Paraméterek

| Neve | Típus | Leírás |
|------|------|-------------|
| $site | `\WP_Ultimo\Models\Site` | Az aktuális helyszín, amelyet beállítani kell. |
| $ | `self` | Az aktuális osztály példányát. |

### Since

- 2.0.9
