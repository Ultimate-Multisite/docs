# Szűrő: wu_documentation_get_link

Engedélyezi a plugin fejlesztőknek a linkek szűrését.

Nem vagyok biztos abban, hogy milyen hasznos lehet, de nem árt semmi.

## Paraméterek

| Neve | Típus | Leírás |
|------|------|-------------|
| $link | `string` | A regisztrált link |
| $slug | `string` | A link lekéréséhez használt slug |
| $default_link | `string` | A regisztrált alapértelmezett link |

### Desde

- 1.7.0
### Forrás

Definíciója található [`inc/class-documentation.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-documentation.php#L125) fájlban, 125-sorban.
