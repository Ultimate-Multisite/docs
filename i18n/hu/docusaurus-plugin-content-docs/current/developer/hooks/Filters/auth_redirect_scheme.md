# Szűrő: auth_redirect_scheme {#filter-authredirectscheme}

Szűrőzi az hitelesítő átirányítási rendszert.

## Paraméterek {#parameters}

| Neve | Típus | Leírás |
|------|------|-------------|
| $scheme | `string` | Az hitelesítő átirányítási rendszert. Alapértelmezett értéke üres. |

### Desde {#since}

- 2.9.0
