# Szűrő: wu_sites_column_limit

Ez a szűrő megváltoztatja az értékét a `get pluginssites` funkcióban, amikor az egész hálózatot (network) vizsgáljuk.

### Forrás

Meghatározva a [`inc/admin/class-network-usage-columns.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/admin/class-network-usage-columns.php#L77) fájlban, 77-edik soron.
