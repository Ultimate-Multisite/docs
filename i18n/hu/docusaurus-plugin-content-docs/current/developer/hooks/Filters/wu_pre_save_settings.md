# Szűrő: wu_pre_save_settings

Ultimate Multisite segítségével engedi a fejlesztőknek a beállítások szűrését a mentés előtt.

## Paraméterek {#parameters}

| Neve | Típus | Leírás |
|------|------|-------------|
| $settings | `array` | A mentendő beállítások. |
| $settings_to_save | `array` | A hozzáadandó új beállítások. |
| $saved_settings | `array` | A jelenleg mentett beállítások. |

### Desde {#since}

- 2.0.18
### Forrás {#source}

Definíciója található [`inc/class-settings.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-settings.php#L296) fájlban a 296-sorban.
