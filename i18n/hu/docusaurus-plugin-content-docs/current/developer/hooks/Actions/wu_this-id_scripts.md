# Action: wu_{$this->id}_scripts

Ezzel a hívással (hook-kal) kerül sor a szkriptek bevitele (enqueue scripts).

Az elem használja ezt a hívást, hogy regisztrálja a `register_scripts` metódusát.

### Since

- 2.0.0
### Source

Definíciója található a [`inc/ui/class-base-element.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/ui/class-base-element.php#L613) fájlban, 613-sorban.
