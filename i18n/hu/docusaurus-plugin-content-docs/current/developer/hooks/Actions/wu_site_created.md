# Akció: wu_site_created {#action-wusitecreated}

El fog futni, amikor egy weboldalt (site) először létrehoznak.

Nem fut, ha a weboldal egy sablonból másolásra kerül.

## Paraméterek {#parameters}

| Név | Típus | Leírás |
|------|------|-------------|
| $data | `array` | Az objektum adatai, amelyeket tárolni kell. |
| $site | `\Site` | Az objektum példányát. |

### Desde {#since}

- 2.0.0
