# Action: wu_mcp_adapter_initialized

Kiviteleződik, miután az MCP adapter inicializálódott.

Lehetővé teszi más bővítményeknek és témáknak, hogy regisztrálják saját képességeiket.

## Paraméterek

| Neve | Típus | Leírás |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Az MCP adapter példányát. |

### Desde

- 2.5.0
