# Action: wp_ultimo_load

Ekkor aktiválódik, amikor az összes függőség betöltődött.

Lehetővé teszi a plugin fejlesztők számára, hogy új funkciókat adjanak hozzá. Például új hosting szolgáltatók támogatását, stb.

### Since

- 2.0.0
### Source

Definálva van a [`inc/class-wp-ultimo.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-wp-ultimo.php#L212) fájlban, 212-sorban.
