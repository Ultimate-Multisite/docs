# Action: wu_register_integrations {#action-wuregisterintegrations}

Az akció elindításához szükséges, hogy az integrációk regisztrálhassák magukat.

A Core (fő rendszer) itt regisztrálja az integrációit.

## Paraméterek {#parameters}

| Neve | Típusa | Leírás |
|------|------|-------------|
| $registry | `\Integration_Registry` | A regisztráció példányát. |

###óta

- 2.5.0
