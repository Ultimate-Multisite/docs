# Action: wp_ultimo_host_providers_load {#action-wpultimohostprovidersload}

Ez az action lehetővé teszi a fejlesztők számára, hogy saját hosting szolgáltató integrációkat págszolgáltatások (plugins) segítségével adjanak hozzá.

### Since {#since}

- 2.0.0
### Source {#source}

Definíciója található a [`inc/managers/class-domain-manager.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/managers/class-domain-manager.php#L1107) fájlban, 1107-sorban.
