# Action: wu_register_rest_routes

Megengedi, hogy további útvonalakat (routes) regisztráljanak.

Ezt használjuk az `/register` végpontunkhoz (endpoint).

## Paraméterek

| Neve | Típus | Leírás |
|------|------|-------------|
| $api_instance | `self` | Az aktuális API példány (instance). |
