# Hogyan kérj egyedi fejlesztési segítséget a Codeable-től?

Néha igazán szükségünk van egy remek testreszabásra, hogy a hálózati folyamataink pontosan úgy működjenek, ahogy szeretnénk. Ha erre van szükséged, a [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) szolgáltatást ajánljuk – ez egy olyan hely, ahol tapasztalt, Ultimate Multisite szakértelemmel rendelkező fejlesztőket találhatsz.

A [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) egy olyan platform, amely megrendelőket köt össze képzett WordPress szabadúszó fejlesztőkkel, akik bármilyen méretű projekten képesek dolgozni. Szigorú jelentkezési folyamatuk biztosítja, hogy olyan szakértőkkel kerülj kapcsolatba, akik kifejezetten a te feladatodhoz értnek.

### Hogyan hozz létre egy projektet a Codeable-ben?

Először látogass el a [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) oldalára, és kattints a **Start a Project** gombra.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Most a projekt beküldéséhez kattints bal oldalon a **Start a Project** lehetőségre.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Az "I need help with" mezőnél válaszd a **Customizing** opciót:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

A "My" mezőnél válaszd a **Plugin** opciót:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Ezután kattints a **Continue** gombra.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Most itt az ideje, hogy a lehető legrészletesebben leírd a projektedet és az igényeidet. Próbálj meg minél több információt megadni!

<!-- Screenshot unavailable: Codeable project description form -->

Válaszolj a következő kérdésekre, amelyek segítenek az árazásban.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Ezután, ha még nincs [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) fiókod, hozz létre egyet, és kattints a **Publish My Project** gombra.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Miután közzétetted a [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projektedet, a platform összekapcsol olyan fejlesztőkkel, akik megfelelnek a projektedhez szükséges készségeknek.

Ez a folyamat segít megtalálni a megfelelő embert a munkádhoz, akinek van tapasztalata a WordPress és az Ultimate Multisite terén!

A Codeable minimum díja 70–120 USD óránként. A becslések a munka terjedelmétől, összetettségétől és sürgősségétől függően változnak.

Miután megkaptad az árajánlatot, felbérelheted az igényeidnek leginkább megfelelő fejlesztőt, és közösen elindíthatjátok a projektet!
