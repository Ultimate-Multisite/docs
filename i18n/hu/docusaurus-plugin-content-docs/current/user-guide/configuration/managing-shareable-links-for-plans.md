# Megosztható linkek kezelése a csomagokhoz

Az Ultimate Multisite lehetőséget kínál arra, hogy közvetlenül megoszd a csomag linkjeit az ügyfelekkel. Ehhez navigálj bármelyik csomag szerkesztési oldalára, és kattints a _Kattints a megosztható link másolásához_ gombra.

![Termék szerkesztési oldala a megosztható link gombbal](/img/config/product-edit-full.png)

_A csomag szerkesztési oldalán találod a megosztható linkek létrehozására szolgáló gombot_

Fontos tudni, hogy ez a mechanizmus az alapértelmezett számlázási gyakoriságot használja a link összeállításához. Ez azt jelenti, hogy ha az alapértelmezett számlázás például havi, akkor a végén egy ilyen linket kapsz: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Ha azt szeretnéd, hogy a megosztható link alapértelmezetten negyedéves legyen, a végére hozzá kell adnod a 3-as számot:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Ha a _premium_ csomagot éves ütemezéssel szeretnéd kiválasztani, add hozzá az 1-est, majd a szót:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Ne feledd, hogy nem lehet egyetlen link a különböző számlázási gyakoriságokhoz. Minden egyes számlázási gyakorisághoz külön linket kell megosztanod.

Miután megosztottad a linket az ügyfeleiddel, az átirányítja őket a regisztrációs folyamat első lépéséhez.
