# Hogyan érem el először a fiókomat?

Üdvözlünk az Ultimate Multisite-ban!

A fiókodat a plugin megvásárlása után kapott e-mailben található linken keresztül érheted el:

<!-- Screenshot unavailable: Account page email with access link -->

Használd a vásárláskor megadott e-mail címedet és az e-mailben kapott jelszavadat.

<!-- Screenshot unavailable: Login page with email and password fields -->

Mivel ez az első bejelentkezésed erre a fiókra, meg kell változtatnod a jelszavadat. Add meg az új jelszót, majd kattints a _Mentés_ gombra.

<!-- Screenshot unavailable: Change password form with Save button -->

A fiókoldaladon kezelheted az összes fiókbeállításodat, például a számlázást, megújításokat, lemondásokat és sok minden mást.
