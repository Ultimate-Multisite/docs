# Ismert összeférhetetlenségek más bővítményekkel

Az Ultimate Multisite telepítése és aktiválása után előfordulhat, hogy problémákba ütközöl más bővítményekkel való összeférhetetlenség miatt.

Minden új kiadással aktívan dolgozunk azon, hogy csökkentsük ezt a listát, de ha egy bővítmény szerepel itt, lehet, hogy ki kell kapcsolnod ahhoz, hogy az Ultimate Multisite Network teljesen működőképes legyen.

Íme a lista, és hogy mit érdemes tenned ezekben az esetekben:

**Néhány WPMU DEV bővítmény** A WPMU DEV által _már nem_ karbantartott bővítmények közül néhány ismerten megakasztja az Ultimate Multisite regisztrációs folyamatát az utolsó lépésnél (a felhasználó nem tudja létrehozni a fiókját). Íme a problémát okozó bővítmények listája:

  * Signup Language: _Az Ultimate Multisite rendelkezik egy kiegészítővel, amely ugyanezt a funkciót nyújtja: Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Ez az Ultimate Multisite beépített funkciója, így nincs szükség külön bővítményre._

  * New Blog Templates: _Ez az Ultimate Multisite beépített funkciója, így nincs szükség külön bővítményre._

  * Set Password on Multisite Blog Creation: _Ez az Ultimate Multisite beépített funkciója, így nincs szükség külön bővítményre._

  * Signup TOS: _Ez az Ultimate Multisite beépített funkciója, így nincs szükség külön bővítményre._

  * Anti-Splog: Bár kevésbé erőteljes, de ez is _az Ultimate Multisite beépített funkciója._

**Megjegyzés**: Az összes többi WPMU Dev bővítmény gond nélkül használható az Ultimate Multisite mellett. Ilyenek például a _Smush_, a _Forminator_, a _Defender_ stb.
