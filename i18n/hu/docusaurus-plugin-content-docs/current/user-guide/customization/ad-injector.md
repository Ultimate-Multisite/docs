# Hirdetések elhelyezése a hálózatodon az Ultimate Multisite: Ad Injector segítségével

Az Ultimate Multisite: Ad Injector egy **ingyenes bővítmény**, amellyel közvetlenül a csomagok szerkesztési oldaláról helyezhetsz el hirdetéseket.

Az Ultimate Multisite: Ad Injector telepítése után menj a _Bővítmények_ oldalra az aktiváláshoz.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Kattints a Hálózati aktiválás gombra_

Miután a bővítmény aktív, a csomag szerkesztési képernyőjén található _Speciális beállítások_ meta dobozban megjelenik egy új opció _Hirdetési beállítások_ néven.

![Hirdetési beállítások a csomag szerkesztési képernyőjén](/img/config/product-edit-full.png)

_Mostantól minden csomagodhoz külön szerkesztheted a hirdetési beállításokat_

Ezen a fülön adhatod meg a hirdetési kódokat a bejegyzésekhez és oldalakhoz a front-enden, a tartalom előtt és után.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_A hirdetési kód beillesztése után a hirdetés megjelenik az adott csomag bejegyzésein és oldalain._

Hirdetéseket jeleníthetsz meg az adott csomagot használók Admin oldalainak fejlécében is.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Az Admin panel hirdetési kód opcióval hirdetéseket jeleníthetsz meg a felhasználók admin felületén_

### Nem csak hirdetésekre jó

Az Ultimate Multisite: Ad Injector segítségével támogatást is nyújthatsz a magasabb szintű csomagok felhasználóinak.

Egyszerűen menj az _Admin panel hirdetések_ opcióhoz, és illeszd be az ügyfélszolgálati eszközöd HTML kódját.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Adj ügyfélszolgálatot meghatározott csomagokhoz_

Így az emberek közvetlenül az Admin paneljükről érhetnek el téged!
