# Az adminisztrációs panel testreszabása a PRO Theme segítségével

Ügyfélmunkák során gyakran felmerül az igény, hogy egyedi élményt nyújtsunk ügyfeleinknek a WordPress admin felületén. Ha professzionálisabb megjelenést és használati élményt keresel, a PRO Theme segíthet ebben.

A PRO Theme egy adminisztrációs téma, amelyet kifejezetten az ügyfelek Dashboard-jának testreszabásához fejlesztettek. Emellett az Ultimate Multisite prémium bővítményeinek alapcsomagjához is tartozik.

A telepítés után menj a _Bővítmények_ oldalra, és aktiváld a hálózaton.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktiváld hálózati szinten a Bővítmények oldalon_

_**MEGJEGYZÉS:**_ _Ha csak telepíted, de nem aktiválod hálózati szinten, az ügyfeleidnél opcióként fog megjelenni, amit bejelölve csak az adott felhasználónál lesz aktív a téma._

Mostanra teljesen megújult az admin felületed! A bal oldali menüben egy új menüpont is megjelent PRO Theme néven.
