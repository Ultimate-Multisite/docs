# Ultimate Multisite Dokumentáció

Üdvözöljük az Ultimate Multisite dokumentációjában. Az Ultimate Multisite egy teljes Weboldal mint Szolgáltatás (WaaS) platformként alakítja át a WordPress Multisite-ot.

## Dokumentáció Szakciók {#documentation-sections}

### [Felhasználói útmutató](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Minden, amit tudnia kell, hogy beállítsa, konfigurálja és kezelje WaaS platformját. Leírja az instalációt, a fizetési kapuk használatát, a domain mapping-et, az ügyfélkezelést és még sok más témát.

### [Fejlesztői dokumentáció](/developer/hooks) {#developer-documentation}

Technikai hivatkozási anyag a fejlesztők számára, akik az Ultimate Multisite alapjain dolgoznak. Tartalmazza a hook-ok hivatkozását, a REST API dokumentációját és az addon fejlesztési útmutatóit.

### [Bővítmények](/addons) {#addons}

Dokumentáció fizetős addon plugin-ekről, amelyek további funkciókkal bővítik az Ultimate Multisite-ot, például WooCommerce integrációval, analitikával, domain eladásával és még sok más dolgot.
