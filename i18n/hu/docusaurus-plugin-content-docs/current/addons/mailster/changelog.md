# Mailster Változatok naplója

### 1.0.0
* Első kiadás
* Automatikus feliratkozó szinkronizáció regisztráció vagy fizetés esetén
* Termékhez kötött lista hozzárendelések
* Tikusdobozalapú hozzájárulás (opt-in) GDPR megfelelőség érdekében
* Kettős hozzájárulás (double opt-in) támogatás
* Teljes körű mezőmapping (mező hozzárendelés)
* Részletes naplózás
