# Szűrő: network_options_list {#filter-networkoptionslist}

Szűrőzi a hálózatopciók listáját, amelyek függnek a hálózat tartományától és elérési útjától.

## Paraméterek {#parameters}

| Neve | Típus | Leírás |
|------|------|-------------|
| $network_options | `array` | Lista hálózatopció neveiről. |

### Desde {#since}

- 1.3.0
### Forrás {#source}

Megértve a [`inc/functions/network.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-multinetwork/blob/main/inc/functions/network.php#L1615) fájlban, 1615-sorban.
