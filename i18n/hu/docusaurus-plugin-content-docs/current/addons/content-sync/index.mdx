# Tartalom Szinkronizálása (Content Sync) {#content-sync}

Multisite Ultimate: Tartalom Szinkronizálása

Az Ultimate Multisitehez készült ez a kiegészítő lehetővé teszi a hálózat adminok számára, hogy szinkronizálják a tartalmat az egyik oldalt más oldalakba a hálózatban. Az addon minden poszt típust támogat: posztokat, oldaltípusokat, testre szabott poszt típust és Elementor sablonokat. A kiegészítő egy több lépcsős „wizard” felületet tartalmaz a tartalom és a céloldalak kiválasztásához, batch feldolgozást és valós idejű haladás visszajelzést biztosít.

## Funkciók (Features) {#features}

* Tartalom Kiválasztása: Vizsgáló kártyás rács miniatűrökkel és poszt típuskutatással
* Minden Poszt Típusa: Szinkronizálja a posztokat, oldaltípusokat, termékeket, Elementor sablonokat és bármilyen testre szabott poszt típust
* Oldaltípus Kiválasztása: Kiválasztja, mely oldalakra szinkronizálja a tartalmat, forrásoldal kizárással és gyermekoldal előzetes kiválasztással
* Batch Feldolgozás: 10 oldalt szinkronizál egyszerre, haladás frissítésével
* Média Szinkronizálás: Automatikusan másolja és újrarendeli a média fájlokat (képek, videók) a poszt tartalmából a céloldalakra
* Taxonómia Szinkronizálása: Megőri a kategóriákat, címkéket és testre szabott taxonómiákat
* Elementor Támogatás: Teljes támogatást biztosít az Elementor sablonoknak CSS regenerációval és sablon ID újrarendelésével
* Konfliktus Kezelése: Opció, hogy lecserélje a meglévő tartalmat, vagy kihagyja
* Cache Kezelés: Automatikusan törli az Elementor cache-t a céloldalakon a szinkronizáció után

## Feltételek (Requirements) {#requirements}

* WordPress 5.3 vagy annál újabb
* PHP 7.4 vagy annál újabb
* Ultimate Multisite plugin (aktív)
* Elementor plugin (opcionális, szükséges az Elementor sablon szinkronizálásához)

## Telepítés (Installation) {#installation}

1. Töltse fel az 'ultimate-multisite-content-sync' fájlt a '/wp-content/plugins/' könyvtárba
2. Aktiválja a kiegészítőt a WordPress Network Admin „Plugins” menüpontján
3. Konfigurálja a beállításokat az Ultimate Multisite → Settings → Content Sync menüpont alatt
4. Navigáljon az Ultimate Multisite → Content Sync menüpontra, hogy elindítsa a szinkronizációt
