# Szűrő: ultimate_multisite_content_sync_url

Megengedi a fejlesztőknek kezelni a plugin gyökerehez viszonyított relatív útvonalat.

## Paraméterek

| Neve | Típus | Leírás |
|------|------|-------------|
| $dir | `string` | Az elérni kívánt út a plugin gyökerétől számított. |

### Elkövetkezés

- 1.0.0
### Forrás

Definálva a [`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-content-sync/blob/main/inc/class-helper.php#L52) fájlban, 52-sorban


## Visszaadja
Az út a plugin gyökerétől számított.
