# Action: wp_ultimo_woocommerce_load {#action-wpultimowoocommerceload}

Ekkor fog ki, miután az összes add-on függőség betöltődött.

Lehetővé teszi a plugin fejlesztők számára, hogy új funkciókat adjanak hozzá.

### Desde {#since}

- 2.0.0
### Forrás {#source}

Definíciója található a [`inc/class-wp-ultimo-woocommerce.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/class-wp-ultimo-woocommerce.php#L90) fájlban, 90-edik soron.
