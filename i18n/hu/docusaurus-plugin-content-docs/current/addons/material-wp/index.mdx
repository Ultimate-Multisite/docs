# Material WP {#material-wp}

Az admin felületének megjelenése évek óta nem változott. De gondoltál már bele, hogy egy Material Design stílusú admin felület milyen lenne?

A Material WP egy admin témátípus, amelyet azért fejlesztettek ki, hogy ezt a megjelenést és felhasználói élményt nyújtsák neked, ezzel segítenek abban, hogy a klientusaidnak egy személyre szabottabb Dashboard-ot biztosíts.

## Funkciók {#features}

- Saját CSS támogatás
- Egyszerű stílusbeállítások
- Menürendezés
- Egyszerű import/export

## Dokumentáció {#documentation}

**A Material WP-vel személyre szabható Admin Panel**

## Telepítés {#installation}

1. Töltse fel a 'material-wp'-t a '/wp-content/plugins/' könyvtárba
2. Aktiválja a plugin-t a WordPress 'Plugins' menüpontján
