# Támogatási Ügynökök

Egy teljes hálózat menedzselése magadad sem könnyű feladat. Ahogy a hálózatod növekedni kezd, egyre nagyobb igényt fogsz látni a támogatásra. Jó ötlet lehet, hogy olyan embereket hozol be, akik segítenek enyhíteni a munkaterhelésedet. Azonban teljes hozzáférés vagy engedélyek adása kockázatos, és kompromisszumot okozhat a hálózatod biztonságát. Itt jön szóba ez a Support Agent add-on.

Ez az add-on lehetővé teszi, hogy létrehozz egy új típusú WordPress felhasználót, nevetve Support Agent. Teljes kontrollt biztosít abban, hogy milyen engedélyekkel rendelkezik az adott felhasználó. Az engedélyek megadása olyan egyszerű, mint egy kapcsoló bekapcsolása, és kikapcsolása is könnyű, ha el akarod távolítani.

## Jellemzők {#features}

- Részletes lista a WordPress és Ultimate Multisite engedélyeiről
- Könnyen engedélyezhető és kikapcsolható hozzáférés
- Korlátlan számú Ügynök (Agent) fiók létrehozása

## Telepítés {#installation}

1. Töltse fel az 'ultimate-multisite-support-agents'-t a '/wp-content/plugins/' könyvtárba
2. A WordPress 'Plugins' menüpontja alatt aktiválja a plugin-t
3. Kövesse az ütemadi (step-by-step) Wizard utasításait a plugin beállításához
