# Filter: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Ez a szűrő lehetővé teszi a fejlesztők számára, hogy rövidre kapcsoltassák (short-circuit) ezt az ellenőrzést.

Ez hasznos lehet olyan esetekben, amikor composer-al alapuló vagy más egyedi beállításokat, például a Bedrock-ot használják, ahol a mu-pluginként való plugin használata az általános gyakorlat.

### Since {#since}

- 2.0.0
### Source {#source}

Definíciója található a [`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-support-agents/blob/main/inc/class-requirements.php#L229) fájlban, 229-sorban.
