# Action: wp_ultimo_support_agents_load

Ekkor aktiválódik, amikor az összes függőség betöltődött.

Megengedi a plugin fejlesztőknek, hogy új funkciókat adjanak hozzá. Például új hosting szolgáltatók támogatását, stb.
