# Gratis AI Agent {#gratis-ai-agent}

A Gratis AI Agent egy öngyönetülő WordPress AI asszisztens, amely természetes nyelvi utasítások alapján képes teljes WordPress oldalak tervezésére, építésére és kezelésére. Strukturált **képességeket** (abilities) biztosít, amelyek diskrét, tesztelhető műveleteket jelentenek, amelyeket az asszisztens hívhat létre. Ezek a képességek a tartalomkezeléstől a design rendszer személyre szabásáig takarítanak. Az 1.9.0 verzió hozzáad egy tartalomkészítő képességet, vizuális oldaláttekintést kliensoldali képernyőfotókkal, öt beépített asszisztens profilját, valamint funkciók kapcsolóválasztóit (feature flags) az hozzáférés- és márkaeléréshez.

## Főbb funkciók {#key-features}

- **Abilities rendszer** — egy moduláris, bővíthető műveletkatalógus, amit az asszisztens hívhat létre az Ön WordPress telepítésén.
- **Tartalomkezelés** — posztok és oldalak létrehozása (beleértve a lap sablonok hozzárendelését), több poszt tömör létrehozása egy hívásban, kiemelt kép beállítása és kapcsolatfelvételi űrlapok építése.
- **Vizsgálat (Visual Review)** — élő oldalak képernyőfotóinak rögzítése, összehasonlító (before/after) képek létrehozása, és AI által generált design áttekintés, amely a felépítéstől, tipográfiától, színeitől és hozzáférhetőségről szól.
- **Öt beépített asszisztens** — Tartalomíró, Oldalépítő, Design Stúdió, Plugin Kezelő és Támogatási Asszisztens; ezeket az asszisztenseket a chat fejeindeki Asszisztens Kiválasztó (Agent Picker) segítségével lehet váltani.
- **Funkciók kapcsolóválasztói (Feature flags)** — hozzáférés- és márkaelérés kapcsolóválasztói a Beállítások → Funkciók kapcsolóválasztói menüpont alatt, amelyek szerepkorlátozásra, fehér címkézésre (white-label) és widget személyre szabására szolgálnak.
- **Custom Post Type kezelése** — hozzárendel, listáz, és töröl személyre szabott poszt típusokat, amelyek a újraindítások során is megőrződnek.
- **Custom Taxonomy kezelése** — hozzárendel, listáz, és töröl személyre szabott taxónómiákat, amelyek a újraindítások során is megőrződnek.
- **Design System képességei** — hozzáad személyre szabott CSS-t, kezel személyre szabott blokkmustereket, beállítja az oldalt logóját, és alkalmazza a theme.json előállításait.
- **Globális stílusok kezelése** — olvas és ír theme.json értékeket a színszémák, tipográfiák, távolságok és még sok más számára.
- **Navigációs menü kezelése** — programatikusan létrehoz és frissíteni navigációs menüeket és azok elemeit.
- **Opciók kezelése** — olvas és ír WordPress opciókat egy beépített biztonsági blokklistával, amely védi a kritikus beállításokat.
- **Teleírható Abilities Regisztrátor** — felfedez és aktivál további képessék-csomagokat anélkül, hogy módosítanák a magasan plugin fájljait.
- **Site Builder Orchestration v2** — több lépésből álló oldalépítési terveket, plugin felfedezéssel, haladás nyomon követésével és hiba-felügyelettel.
- **WP-CLI benchmark parancs** — futtatja az asszisztens képesség benchmark csomagját a parancssorral, CI-hez és modellértékeléshez.

## Feltételek {#requirements}

- WordPress 6.0 vagy magasabb
- PHP 8.1 vagy magasabb
- WordPress Multisite (hálózat szinten aktivált) vagy single-site telepítés
- Egy aktív AI szolgáltató csatlakoztató plugin (Anthropic, OpenAI vagy kompatibilis)

## Telepítés {#installation}

1. Töltse fel a plugin fájljait a `/wp-content/plugins/gratis-ai-agent` útvonalba.
2. Aktiválja a pluginet a WordPress **Plugins** képernyőjéről.
3. Navigáljon a **Gratis AI Agent → Settings** menüpontra és csatlakoztassa AI szolgáltatóját.
4. (Opcionális) Böngészzen az **Abilities Registry**-ben, hogy további képessék-csomagokat telepítson.

## Gyors elindítás {#quick-start}

Aktiválás után és az AI szolgáltatóhoz csatlakozás után nyissa meg a **Gratis AI Agent** paneljét a WordPress admin oldali sávban. Írjon be egy természetes nyelvi utasítást, és az asszisztens tervezni és végrehajtani fogja a szükséges lépéseket a rendelkezésre álló képességeket felhasználva.

**Példa utasítások:**

```
Create a "Portfolio" custom post type with a "Project Category" taxonomy.
```

```
Inject a dark-mode colour palette into the theme and set the primary colour to #1a1a2e.
```

```
Build a restaurant website with an online menu, booking form, and contact page.
```

## Abilities áttekintése {#abilities-overview}

Az Abilities az atomikus műveletek, amelyeket az asszisztens elvégezhet. Minden ability egy regisztrált PHP osztály, amely egy szémát (paramétereket és visszaadási típusokat) biztosít, amit az asszisztens futási időben olvas. Az abilities-eket funkcionális területekre csoportosították:

| Terület | Abilities |
|---|---|
| **Tartalomkezelés** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Vizsgálat (Visual Review)** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Design System** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Globális stílusok** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Navigációs menük** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Opciók kezelése** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Teleírható Abilities** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Látogasson el az [Abilities Reference](./abilities) oldalra a teljes paraméterdokumentáció és használati példákért.

## WP-CLI {#wp-cli}

A Gratis AI Agent egy `wp gratis-ai-agent` parancssorcsaládját biztosítja a benchmarkok futtatásához és a képességek kezeléséhez a parancssor felől. Látogasson el a [WP-CLI Reference](./wp-cli) oldalra a teljes parancslistához.

```bash
# Futtatja a teljes benchmark csomagot
wp gratis-ai-agent benchmark

# Listázza a telepített abilities-eket
wp gratis-ai-agent abilities list

# Telepíti egy abilities-csomagot
wp gratis-ai-agent abilities install <slug>
```

## Konfiguráció {#configuration}

Az összes beállítás a WordPress admin oldali **Gratis AI Agent → Settings** menüpont alatt található:

| Beállítás | Leírás |
|---|---|
| **AI Provider** | Válassza ki és konfigurálja az aktív AI szolgáltató csatlakoztatóját. |
| **Model** | Válassza ki a nyelvmodellét az asszisztens munkamenetekhez. |
| **Max Tokens** | Per-kérés token költségvetés (alapértelmezett 4096). |
| **Ability Timeout** | Maximális másodperce, amennyi ideig egy egyes ability futhat (alapértelmezett 30). |
| **Debug Logging** | Naplózza az ability hívásait és az asszisztens élettelmenetét a WordPress debug naplóba. |

## Opciók Kezelés Biztonsági Blokklistája {#options-management-safety-blocklist}

Az Opciók Kezelés ability-je egy beépített blokklistát tartalmaz, amely megakadályozza az asszisztens számára, hogy olvaszon vagy írjon érzékeny WordPress opciókat. Az alapértelmezett blokklista a következőket foglalja magában:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL opciók — ezekhez használjon dedikált migrációs eszközt)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (titkos kulcsok)
- `active_plugins`, `template` (plugin/téma aktivációja — használjon dedikált abilities-eket)
- Fizetési kapu hitelesítő adatai és API kulcsai, amelyeket telepített pluginek regisztrálnak.

További elemeket a `gratis_ai_agent_options_blocklist` szűrő segítségével lehet hozzáadni.

## Hooks és Szűrők {#hooks-and-filters}

### Akciók (Actions) {#actions}

- `gratis_ai_agent_ability_registered` — kiüti egy ability regisztrálása után, átadva az ability osztály nevét.
- `gratis_ai_agent_plan_started` — kiüti, amikor egy oldalépítési terv elkezdi futtatását.
- `gratis_ai_agent_plan_completed` — kiüti, amikor egy terv sikeresen fejeződik be.
- `gratis_ai_agent_plan_failed` — kiüti, amikor egy terv visszafordíthatatlan hibával találkozik.

### Szűrők (Filters) {#filters}

- `gratis_ai_agent_abilities` — módosítja a regisztrált ability listáját, mielőtt az asszisztens betölti.
- `gratis_ai_agent_options_blocklist` — bővíti vagy csökkenti az opciók biztonsági blokklistáját.
- `gratis_ai_agent_benchmark_questions` — hozzáad személyre szabott benchmark kérdéseket az értékelési csomaghoz.
- `gratis_ai_agent_plan_plugins` — módosítja a plugin felfedezési listát az oldalterv generálás során.
