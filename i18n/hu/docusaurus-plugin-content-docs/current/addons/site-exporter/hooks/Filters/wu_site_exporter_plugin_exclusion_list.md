# Szűrő: wu_site_exporter_plugin_exclusion_list

Ez a szűrő lehetővé teszi a fejlesztők számára, hogy kezeljenek egy plugin listát, amit kizárólagosan a generált zip fájlban nem szeretnének látni.

## Paraméterek

| Neve | Típus | Leírás |
|------|------|-------------|
| $plugin_list | `array` | Azok a pluginek, amelyeket kizárni kell. |

### Since

- 2.0.0
### Source

Definíciója található [`inc/class-wp-ultimo-site-exporter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-wp-ultimo-site-exporter.php#L210) fájlban, 210-sorban.
