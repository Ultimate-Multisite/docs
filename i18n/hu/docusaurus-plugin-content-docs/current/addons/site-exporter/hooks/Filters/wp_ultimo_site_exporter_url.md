# Szűrő: wp_ultimo_site_exporter_url

Megengedi a fejlesztőknek, hogy kezeljék a plugin alapkönyvtárához viszonyított relatív elérési útvonalat.

## Paraméterek

| Neve | Típus | Leírás |
|------|------|-------------|
| $dir | `string` | Az elérhető útvonal, amely relatív a plugin alapkönyvtárához. |

### Since

- 0.0.1
### Source

Definálva van a [`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-helper.php#L52) fájlban, 52-sorban


## Várható érték
A plugin alapkönyvtárához viszonyított relatív útvonal.
