# Action: wu_captcha_integrations_init

Eles kiüli, amikor az összes integráció inicializálódott.

A külső (third-party) integrációk itt kapcsolódhatnak hozzá, hogy saját funkciókat adjanak hozzá.

### Since {#since}

- 1.2.0
### Source {#source}

Definíciója található a [`inc/captcha-providers/class-captcha-manager.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-captcha/blob/main/inc/captcha-providers/class-captcha-manager.php#L106) fájlban, 106-sorban.
