# Szűrő: wp_ultimo_skip_network_active_check

Ez a szűrő lehetővé teszi a fejlesztők számára, hogy kihagyják (short-circuit) ezt az ellenőrzést.

Ez hasznos lehet olyan esetekben, amikor composer alapú vagy más egyedi beállításokat használunk, például a Bedrock-ot, ahol a mu-pluginok használata az általános gyakorlat.

### Since

- 2.0.0
