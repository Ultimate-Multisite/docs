# Action: wp_ultimo_plugin_and_theme_manager_load {#action-wpultimopluginandthememanagerload}

Elküldődik, amikor az összes függőség betöltődött.

Ez lehetővé teszi a plugin fejlesztők számára, hogy új funkciókat adjanak hozzá. Például új hosting szolgáltatók támogatása, stb.
