# Szűrő: wu_mt_syncable_user_meta_keys

Szűrőzi a felhasználói meta kulcsok listáját, amelyeket távoli csomókra kell szinkronizálni.

## Paraméterek {#parameters}

| Neve | Típus | Leírás |
|------|------|-------------|
| $syncable_keys | `array` | Az alapértelmezett meta kulcsok, amelyeket szinkronizálni kell. |

- 1.0.0
