# Multi-Tenancy

Elosztja a subsite-okat (altoldalakat) több szerveren és adatbázison a teljesítmény és a biztonság érdekében.

## Feltételek

- WordPress 5.3 vagy annál újabb
- PHP 7.4 vagy annál újabb
- Multisite Ultimate plugin (aktív)

## Telepítés

1. Töltse fel az addon fájljait a `/wp-content/plugins/` könyvtárba
2. Aktiválja az addont a WordPress 'Plugins' menüpontja keresztül
3. Konfigurálja az addon beállításait a hálózat admin felületen

## Fejlesztés

### Beállítások

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Tesztelés

Az addon egy átfogó tesztelő keretrendszert tartalmaz:

- **Egységtesztek (Unit Tests)**: PHPUnit alapú tesztek a WordPress teszt suite-el
- **Bázisteszteszkategória (Base Test Class)**: Segédmetódusokat biztosít gyakori tesztforgatóköregekhez
- **Tesztfedettség (Test Coverage)**: Beállítva a kódfedettség nyomon követésére
- **Multisite tesztelés**: A tesztek egy multisite környezetben futnak

### Kódstandardok

- **WordPress Kódstandardok**: PHPCS segítségével biztosított
- **PHP 7.4+ Kompatibilitás**: Támogatja a modern PHP funkciók
- **Statikus elemzés (Static Analysis)**: PHPStan integráció típusellenőrzéshez
- **Kódmodernizáció**: Rector szabályok PHP frissítéshez

### Build Rendszer

- **Asset feldolgozás**: Minifikáció a CSS/JS fájlokhoz
- **Fordítás**: POT fájl generálás
- **Csomagolás (Packaging)**: Archívum létrehozása a kiosztáshoz
- **Fejlesztési mód (Development Mode)**: Minifikáció nélküli assetek hibakerőzéshez

## Konfiguráció

Az addon számos konfigurációs opciót támogat, amelyek a hálózat admin felületen érhetők el.

## Hooks és Szűrők

### Akciók (Actions)

- `ultimate-multisite-addon-template_init` - Elküldődik az addon inicializációja után
- `ultimate-multisite-addon-template_loaded` - Elküldődik az addon összes fájljának betöltése után

### Szűrők (Filters)

- `ultimate-multisite-addon-template_settings` - Megváltoztatja az addon beállításait
- `ultimate-multisite-addon-template_enabled` - Áthullja az addon aktiválás/deaktiválás állapotát
