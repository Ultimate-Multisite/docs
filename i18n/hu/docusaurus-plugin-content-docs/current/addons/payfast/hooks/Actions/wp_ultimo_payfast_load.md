# Action: wp_ultimo_payfast_load {#action-wpultimopayfastload}

Futtatódik, amikor az összes függőség betöltődött.

Lehetővé teszi a plugin fejlesztőknek új funkciók hozzáadását. Például új hosting szolgáltatók támogatása, stb.
