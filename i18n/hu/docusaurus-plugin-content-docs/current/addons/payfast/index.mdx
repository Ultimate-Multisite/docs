# PayFast

Multisite Ultimate: PayFast Gateway

Hozzáadja a PayFast-et – egy gyors és biztonságos online fizetési gateway-t Dél-Afrikában – mint fizetési opciót a Multisite Ultimate-hez.

## Telepítés {#installation}

1. Töltse fel az 'ultimate-multisite-payfast'-et a '/wp-content/plugins/' könyvtárba
2. Aktiválja az eklót a WordPress 'Plugins' menüpontján keresztül
3. Kövesse a lépésről lépésre haladó Wizard utasításait az eklót beállításához

## Használati Útoguide {#usage-guide}

A PayFast Dél-Afrikában vezető online fizetési szolgáltató. Célja, hogy a tranzakciókat zökkenőmentes és biztonságosá tegye az online kereskedők számára, segítve ezzel a növekedésüket. Az Ultimate Multisite felhasználói kérésére úgy döntöttünk, hogy létrehozunk egy addon-t, amely lehetővé teszi a PayFast-el integrációt. Ez az cikk bemutatja, hogyan állíthatja be a PayFast-et fizetési gateway-ként az Ultimate Multisite hálózatában fizetések fogadására.

Az első lépés az addon telepítése.

# Hogyan telepítemem az Addon-t a PayFast-hez:

Gyon az add-ons oldalra, és válassza ki az Ultimate Multisite: Payfast Gateway add-ont.

![Add-ons page](/img/admin/addons-page.png)

Megugrik az addon ablak. Kattintson az Install Now gombra.

![Add-ons page with install option](/img/admin/addons-page.png)

A telepítés után továbbításra kerül az eklők oldalára. Itt egyszerűen kattintson a Network Activate gombra, és a _Payfast_ addon aktiválódik az egész hálózatán.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# A PayFast Beállítása

Az addon telepítése után most Ultimate Multisite beállításaihoz navigálhat, és elkezdi a PayFast konfigurálását.

Az **Ultimate Multisite settings** oldalon megtalálja a "Payments" (Fizetések) menüpontot. Ezen a menüpont alatt láthatja a "**Payfast**"-et egy kapcsoló gombbal, amivel aktiválhatja vagy inaktívá teheti ezt a fizetési gateway-t.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Kattintson a kapcsoló gombra, hogy aktiválja. Ezután kibontja a mezőket, ahol meg kell töltenie azokat a különböző adatokat, mint például a **Merchant ID, Merchant Key és PassPhrase.** Ezeket az információkat a **Payfast** fiókjából szerezheti be.

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

A következő lépés az, hogy bejelentkezik a PayFast fiókjába, hogy megszerezze az ahhoz szükséges információkat.

Bejelentkezése után azonnal láthatja az ahhoz szükséges hitelesítő adatokkal a **Payfast** fiók irányítói felületén (dashboard).

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Egyszerűen másolhatja és beillesztheti az hitelesítő adatokat, és kitöltheti a részleteket az Ultimate Multisite beállításain. Ezután kattintson a **"Save settings** " gombra a változtatások mentéséhez.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Ez az egész! A PayFast fizetési integrációja kész!
