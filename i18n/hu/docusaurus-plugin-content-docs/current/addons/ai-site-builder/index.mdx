# AI Site Builder

ultimate-multisite-ai-site-builder

## Előfeltételek {#requirements}

- WordPress 5.3 vagy magasabb
- PHP 7.4 vagy magasabb
- Multisite Ultimate plugin (aktiválva)

## Telepítés {#installation}

1. Töltse fel az addon fájljait a `/wp-content/plugins/` könyvtárába
2. Aktiválja a plugin-t a WordPress 'Plugins' menüpontján keresztül
3. Konfigurálja az addon beállításait a hálózat adminisztrációjában

## Fejlesztés {#development}

### Beállítás {#setup}

```bash
# Telepíti a függőségeket
composer install
npm install

# Futatja a teszteket
npm run test

# Futatja a kódstílus ellenőrzéseket
vendor/bin/phpcs
vendor/bin/phpstan

# Javítja a kódstílus problémáit
vendor/bin/phpcbf

# Összeszerelése a produkcióhoz
npm run build
```

### Tesztelés {#testing}

Az addon egy átfogó tesztelő keretet tartalmaz:

- **Egységtesztelés (Unit Tests)**: PHPUnit alapú tesztek a WordPress teszt suite-el
- **Alap Teszt Osztály (Base Test Class)**: Segítő metódusokat biztosít gyakori teszt forgatókönyemekhez
- **Tesztfedettség (Test Coverage)**: Konfigurálva a kódfedettség nyomon követésére
- **Multisite Tesztelés**: A tesztek multisite környezetben futnak

### Kódstandardok {#code-standards}

- **WordPress Kódstandardok**: PHPCS segítségével biztosított
- **PHP 7.4+ Kompatibilitás**: Támogatott modern PHP funkciók
- **Statisztikus Elemzés (Static Analysis)**: PHPStan integráció típusellenőrzéshez
- **Kódmodernizáció**: Rector szabályok PHP frissítéshez

### Összerelési Rendszer (Build System) {#build-system}

- **Asset Kezelés**: Minifikáció a CSS/JS fájlokhoz
- **Fordítás**: POT fájl generálás
- **Csomagolás**: Archív létrehozása a kiosztáshoz
- **Fejlesztési Mód**: Nem minifikált assetek hibakerőzéshez

## Konfiguráció {#configuration}

Az addon számos konfigurációs opciót támogat, amelyek a hálózat adminisztrációs felületen érhetők el.

## Hooks és Szűrők {#hooks-and-filters}

### Akciók (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - Elküldődik az addon inicializációja után
- `ultimate-multisite-ai-site-builder_loaded` - Elküldődik az addon összes fájljának betöltése után

### Szűrők (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Megváltoztatja az addon beállításait
- `ultimate-multisite-ai-site-builder_enabled` - Áthullja az addon aktiválás/deaktiválás állapotát
