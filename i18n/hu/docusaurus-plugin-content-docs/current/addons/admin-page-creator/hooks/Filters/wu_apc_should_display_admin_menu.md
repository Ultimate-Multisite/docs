# Filter: wu_apc_should_display_admin_menu

Engedélyezi az adminoknak az admin menü oldalai elrejtését, ha befejezték a testreszabási folyamatot.

## Paraméterek

| Name | Type | Leírás |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Ha megjeleníteni kell az oldalakat. |
| $network_active | `bool` | Ellenőrzi, hogy a plugin a hálózat adminisztrátor számára aktív-e. |

### Desde

- 1.4.0
