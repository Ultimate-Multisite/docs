# Fluent Forms

Ez az add-on lehetővé teszi, hogy korlátozásokokat állítsanak be a Fluent Forms-ben a Multisite Ultimate-en, legyen szó termékekről, tagságokról vagy weboldalakról.
Mivel a Fluent Forms nem a szokásos custom post type-okat, hanem saját táblákat használ, a standard korlátozási opciók nem működnek vele.

## Jellemzők {#features}

- Korlátozások hozzáadása a Fluent Forms-hez vagy a Conversational Fluent Forms-hez.
- A felhasználók barátságos üzenetet kapnak, amely arra ösztönzi őket, hogy emelítsék a tervük szintjét, amikor elérnek a korlátot.
