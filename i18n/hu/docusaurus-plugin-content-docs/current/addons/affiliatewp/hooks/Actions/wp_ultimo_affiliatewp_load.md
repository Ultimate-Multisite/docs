# Action: wp_ultimo_affiliatewp_load {#action-wpultimoaffiliatewpload}

Ekkor aktiválódik, amikor az összes függőség betöltődött.

Lehetővé teszi a plugin fejlesztők számára, hogy új funkciókat adjanak hozzá. Például új hosting szolgáltatók támogatását, stb.

### Desde {#since}

- 1.0.0
