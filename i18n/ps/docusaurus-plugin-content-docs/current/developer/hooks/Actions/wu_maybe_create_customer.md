# عمل: wu_maybe_create_customer

plugin جوړوونکو ته اجازه ورکوي چې کله پېرودونکی اضافه شي، نور اضافي کارونه ترسره کړي.

دلته هغه hooks ورزیاتوو چې د پېرودونکي-&gt;کارن اصلي سایټ ته هم د اضافه کولو لپاره دي، د بېلګې په توګه.

## پارامترونه

| نوم | ډول | تشریح |
|------|------|-------------|
| $customer | `\Customer` | هغه پېرودونکی چې ښايي جوړ شوی وي. |
| $checkout | `\Checkout` | اوسنی checkout کلاس. |
