# فلټر: auth_cookie_expiration

د تصدیق کوکي د پای ته رسېدو مودې موده فلټر کوي.

## پارامېټرونه

| نوم | ډول | تشریح |
|------|------|-------------|
| $length | `int` | د پای ته رسېدو مودې موده په ثانیو کې. |
| $user_id | `int` | د کاروونکي ID. |
| $remember | `bool` | آیا د کاروونکي ننوتل په یاد وساتل شي. اصلي ارزښت false دی. |

### له نسخې راهیسې

- 2.8.0

په [`inc/sso/auth-functions.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/auth-functions.php#L52) کې په ۵۲مه کرښه تعریف شوی.
