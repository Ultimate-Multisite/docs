# فلټر: wu_get_site_domain_and_path

پراختیاکوونکو ته اجازه ورکوي چې د ډومېن/لار جوړې بدل کړي.

دا د ګڼو شیانو لپاره ګټور کېدای شي، لکه د کوم ډول staging حل پلي کول، بېلابېل سرورونه، او داسې نور.

## پارامېټرونه {#parameters}

| نوم | ډول | تشریح |
|------|------|-------------|
| $d | `object` | اوسنی څیز چې د ډومېن او لار کلیدونه لري. |
| $path_or_subdomain | `string` | اصلي لار/فرعي ډومېن چې فنکشن ته لېږدول شوی. |

### له {#since}

- 2.0.0
### سرچینه {#source}
