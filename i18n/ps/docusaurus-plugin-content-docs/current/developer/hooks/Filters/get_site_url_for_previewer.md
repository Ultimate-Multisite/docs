# فلټر: get_site_url_for_previewer

plugin جوړوونکو ته اجازه ورکوي چې هغه URL فلټر کړي چې په previewer کې کارېږي

## پارامېټرونه

| نوم | ډول | تشریح |
|------|------|-------------|
| $domain | `string` | اوس‌مهال کارېدونکی اصلي domain، د بدلونونو لپاره ګټور |
| $domain_options | `array` | د ټولو domain غوراویو لېست چې په Ultimate Multisite Settings -&gt; Network Settings -&gt; Domain Options کې داخل شوي |
