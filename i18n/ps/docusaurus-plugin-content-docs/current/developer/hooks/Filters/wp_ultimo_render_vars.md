# فلټر: wp_ultimo_render_vars

plugin پراختیاکوونکو ته اجازه ورکوي چې په نړیوال ډول د render شرایطو ته اضافي متغیر ورزیات کړي.

## پارامیټرونه {#parameters}

| نوم | ډول | تشریح |
|------|------|-------------|
| $args | `array` | هغه Array چې د render غوښتنې له خوا لېږدول شوي متغیرونه لري. |
| $view | `string` | د هغه view نوم چې render کېږي. |
| $default_view | `string` | د fallback_view نوم |

### له نسخې راهیسې {#since}

- 2.0.0
