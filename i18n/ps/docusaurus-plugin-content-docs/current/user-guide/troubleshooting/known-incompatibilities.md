# داسې پېژندل شوي نهمیتونه له अरولو Plugin-ҳо

هنگامی چې شما Ultimate Multisite نصب او فعال мекунед, метавон баъзе масъалаҳоро аз сабаби неин ҳалли кардани нетавони муносибат бо дигар плагинҳо доред.

Мом ҳамаи ин рӯйхатро бо ҳар як нокии навтарин версия кам кардан барои кор мекунем, аммо агар ягон плагин дар ин ҷо фароҳамъдад шавад, шумо метавонед онро беактивӣ кунед то Ultimate Multisite Network-и пурра коркарда ба даст оред.

Ин рӯйхати онҳо ва он аст, ки шумо бояд дар ин ҳолат чӣ кор кунед:

**Баъзе плагинҳои WPMU DEV** Баъзе плагинҳое, ки аз ҷониби WPMU DEV нигоҳ накарда мешаванд, маълум аст, ки онҳо равиши қайд кардани Ultimate Multisite-ро дар қадами охир (ки корбар наметавонад ҳисобот кунад) қатъ мекунанд. Ин рӯйхати плагинҳоест, ки ин масъалаҳоро ба вуҷуд меоранд:

  * Signup Language: _Ultimate Multisite онро ҳамчун як add-on пешниҳод мекунад, ки ҳамин корро дорад ва номи он Ultimate Multisite: Language Switcher аст._

  * Set Blog Description on Blog Creation: _Ин дар Ultimate Multisite воқеъ мебошад, аз ин рӯ ба плагинҳои иловагӣ ниёз надоред._

  * New Blog Templates: _Ин дар Ultimate Multisite воқеъ мебошад, аз ин рӯ ба плагинҳои иловагӣ ниёз надоред._

  * Set Password on Multisite Blog Creation: _Ин дар Ultimate Multisite воқеъ мебошад, аз ин рӯ ба плагинҳои иловагӣ ниёз надоред._

  * Signup TOS: _Ин дар Ultimate Multisite воқеъ мебошад, аз ин рӯ ба плагинҳои иловагӣ ниёз надоред._

  * Anti-Splog: Ҳарчанд он камтар қувваи дорад, он низ _дар Ultimate Multisite воқеъ аст._

**Нотавони** : Ҳамаи дигар плагинҳои WPMU Dev метавонанд одатан дар паҳлӯи Ultimate Multisite истифода шаванд. Мисолҳо аз онҳо мегиранд: _Smush_ , _Forminator_ , _Defender_, ва ғайра.
