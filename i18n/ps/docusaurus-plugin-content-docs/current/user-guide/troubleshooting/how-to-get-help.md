# څنګه مرسته ترلاسه کړئ {#how-to-get-help}

تاسو د خپل اعتبارلي License key ترلاسه کړی دی، او دا تاسو ته زموږ ټیم سره مرستې ورکولو حق ورکوي چې په دې وخت کې کومه مسائلی حل کړئ چې تاسو یې د Ultimate Multisite تنظیم کولو په وخت کې ممکن ورسره راغلی شي.

دا درس د License holders لپاره موجود درپره مرستės کانالونه او دا ډول معلومات دي چې تاسو باید د زموږ مرستės ټیم سره اړیکه ونیسئ، په لاس کې لرئ.

## مرستės سایت (Help Site) {#help-site}

تاسو کولی شئ له زموږ مرستės سایت څخه مختلف artikلازو او Tutorials ته ورسئ چې تاسو یې زیر لینک کې وموندئ.

[)

موږ د हामرو کارکوونکو ته تشویله کوو چې خپل ځانګړي Ultimate Multisite خoganې او کارکرد augلۍ په اړه پوهی ترلاسه کړئ. زموږ مرستės سایتमा "Getting Started" artikلازو هم موجود دي او ډېر عام پوښتنې، لکه څنګه چې add-ons څنګه نصب کړئ، یا payment gateways څنګه راټول کړئ، داسې معلومات به تل ورته وموندئ.

## مرستės سره اړیکه ونیسئ (Contact Support) {#contact-support}

که تاسو په مرستės سایت کې ځان پاتې کړی او ځواب نه ولای شئ، نو تاسو کولی شئ خپل مرستės ټیم ته دوه لارې له کارولو څخه: یا د زموږ ویب پاڼې لاندې ښي اړیکه ونیسلوله speech bubble icon (اسپيچ ببل آئیکن) 클릭 کړئ، یا مستقیم په support@ultimatemultisite.com کې ایمیل ولیکئ.

د تکنالوژیک مسائلو لپاره که تاسو مرستės ټیم ته اړیکه ونیسئ، نو مهرباني وکړئ خپل مسأله په تفصیل سره بیان کړئ او ورته screenshot (اسکرین شاټ)، error messages (غلط پیغامونه)، یا کوم بل معلومات ورکړئ چې زموږ مرستės کارکوونکو ته ستاسو مسأله په آساني پیژندل او ارزولې کېدل به کمک وکړي. تاسو باید خپل license key هم وړاندې کړئ. یوازې فعال License holders- ته مرسته ورکول اجازه ورکول کیږي.

## ټولنیز سایت (Community Site) {#community-site}

د زموږ [community site](https://community.ultimatemultisite.com/) د Ultimate Multisite کارکوونکو لپاره یو ځای دی چې دوی له یو بل سره تعامل وکړي. دا د معلومات سیستم په توګه کار کوي چیرې چې راکبونکي کولی شي پوسټونه ولیکي، په بحثونو کې نظر ورکول وکړي، مشورې ورکول وکړي، همکارۍ وکړي، یا ټولنیز रूपमा معلومات جمع کړي (crowdsource). تاسو باید خپل Ultimate Multisite license key هم وړاندې کړئ ترڅو ځانګړتیا شوي ځایونه/پېژندل کېدونکي صفحات فعال کړئ.
