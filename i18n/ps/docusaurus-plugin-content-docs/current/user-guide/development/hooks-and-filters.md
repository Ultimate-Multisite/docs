# Ultimate Multisite Hooks او فیلترونه

Ultimate Multisite په حقیقت کې ډېرې فنکشنونه او د ډیټابیس پوښتنه (database queries) جوړوي. دا هم فلټرونه (filters) او هوکونه (hooks) لري چې تاسو ته اجازه ورکوي اوسني فنکشنونه بدل کړئ یا نوی کوډ اضافه کړئ.

موږ یو لیست جوړ کړی دی د دې هوکونو او فیلترونو لپاره چې متخصصان (advanced users) په خپل شبکه کې لوړ سطح پرتو کولو لپاره کار وکړي. تاسو کولی شئ دا لیست له Github څخه لینک کلیک کولو وسیله هم ترلاسه کړئ:

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
