# Codeable څخه اپنی اندازه کولو لپاره همکاري ترلاسه कसरी गर्ने؟

कहिलेकाहीँ, हामीलाई हाम्रो नेटवर्कहरू एकदमै हाम्रो मनपर्ने तरिकाले बनाउन केही उत्कृष्ट अनुकूलन (customization) चाहिन्छ। यदि तपाईं यस्तै खोज्दै हुनुहुन्छ भने, हामी [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) लाई सिफारिस गर्छौं, जहाँ तपाईं Ultimate Multisite विशेषज्ञता भएका अनुभवी डेभलपरहरू फेला पार्न सक्नुहुन्छ।

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) एउटा यस्तो प्लेटफर्म हो जसले ग्राहकहरूलाई योग्य WordPress फ्रीलान्स डेभलपरहरूसँग जोड्छ, जसले कुनै पनि आकारको परियोजनामा काम गर्न सक्छन्। उनीहरूको आवेदन गर्ने प्रक्रिया एकदमै ध्यानपूर्वक गरिएको हुन्छ ताकि तपाईंलाई तपाईंले सोचिरहनुभएको कार्यका लागि विशेष रूपमा योग्य विशेषज्ञहरूसँग मिलाउन सकियोस्।

### Codeable मा परियोजना कसरी बनाउने? {#how-to-create-a-project-in-codeable}

सबैभन्दा पहिले, [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) मा जानुहोस् र **Start a Project** बटनमा क्लिक गर्नुहोस्।

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

अब, परियोजना पेश गर्नको लागि, बायाँतिर गएर **Start a Project** मा क्लिक गर्नुहोस्।

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

"I need help with" (मलाई सहयोग चाहिन्छ) मा, **Customizing** छान्नुहोस् :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

"My" (मेरो) मा, **Plugin** छान्नुहोस् :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

त्यसपछि **Continue** बटनमा क्लिक गर्नुहोस्।

<!-- Screenshot unavailable: Codeable project form Continue button -->

अब, तपाईंले आफ्नो परियोजना र आवश्यकताहरू वर्णन गर्नको लागि सबैभन्दा राम्रो तरिका अपनाउनुहोस्। सकेसम्म धेरै जानकारी दिन प्रयास गर्नुहोस्!

<!-- Screenshot unavailable: Codeable project description form -->

मूल्य निर्धारणमा मद्दत गर्न निम्न प्रश्नहरूको जवाफ दिनुहोस्।

<!-- Screenshot unavailable: Codeable project pricing questions -->

د دې وروسته، که تاسو پر [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) اکاونټ نه لرئ، یو جوړ کړئ او د **Publish My Project** د ګوډۍ (button) باندې کلیک وکړئ.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

که تاسو خپل [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) پروژه veröffentیٰ کړی وي، نو پلتفۆرم تاسو د هغه د جوړوونکو سره تړلیږي چې د ستاسو پروژه لپاره اړین مهارتونه لري.

دا پروسه به ستاسو کار لپاره مناسب شخص پیدا کولو کې مرسته وکړي، چې تجربه یې په WordPress او Ultimate Multisite کې وي!

Codeable د کمینه نرخ $70–$120 USD هر ساعت دی. تخمینات د کارको Umfang، پیچیدنې او پروژه ته چټکۍ سره ورته بدلې کیږي.

که تاسو خپل تخمینہ ترلاسه کړئ، نو تاسو کولی شئ هغه جوړوونکی چې ستاسو اړتیاوو لپاره غوره مناسب وي، له پوښتنه (project) په ګډه شروع کړئ!
