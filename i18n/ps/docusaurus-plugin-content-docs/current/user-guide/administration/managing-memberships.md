# مدیریت اشتراک‌ها (Managing Memberships) {#managing-memberships}

Ultimate Multisite تهتري به تاسو اجازه ورکوي چې د خپل شبکه ټولې سبسکرپشنونه وکاروئ او ورته پام وکړئ. تاسو کولی شئ د پلانونو او محصولاتو تفصیلات، د سایتونو تعداد، د بیا کولو مقدار او دوري، د پیسې اخیستنې تاریخ او وخت ټاپو (timestamps) په څیر بدل کړئ او ویره کړئ.

د دې صفحې ته رسولو لپاره، माسي Ultimate Multisite کې **Memberships** 탭 ته لاړ شئ.

![Memberships list page](/img/admin/memberships-list.png)

په Memberships صفحه کې تاسو کولی شئ د خپل کليونټو او د دوی حسابونو سره تړلو محصولاتو لیست وګورئ. تاسو کولی شئ د وضعیت (status) له مخې اشتراکونه فیلتر کړئ: فعال (active)، انتظار کېدونکي (pending)، په توګه منځته (on hold)، منقضی شوي (expired)، یا باطل شوي (cancelled).

ننوتنې اشتراکونه **پایاخره** (lifetime) اشتراکونه تر نظر ګڼل کیږي. دوی ته د منقضیولې تاریخ نه ورکول کیږي او به په خودی ډول منقضی نه شي. که یو وړیا پلان باید وروسته له آزمویني دوره یا ثابت دورې وروسته پایله کړي، نو تاسو کولی شئ یو پیسې ترلاسه کوونکی محصول چې آزمویني دوره لري کارولیږئ، اشتراک manuallly باطل کړئ، یا کله چې دوی ورته رامنځته کیږي، هغه کليونټ نورو پلان ته منتقل کړئ.

د یوه اشتراک باندې کلیک وکړئ ترڅو د هغه صفحې ته رسولو کېدل ترلاسه کړئ چیرې چې تاسو کولی شئ د اشتراک په تفصیلات بدل کړئ – د پیسې اخیستنې معلومات، د سایت یا لیدو محدودیتونه او نور.

![Membership edit interface](/img/admin/membership-edit.png)

دا د اشتراک په بدلولو صفحې یو پوره ښودنه ده:

![Membership edit full interface](/img/admin/membership-edit-full.png)

تاسو کولی شئ هم نوي اشتراکونه اضافه کړئ چې پخوانیو کليونټانو ته ورته ثبت شوي وي. **Add Membership** باندې کلیک وکړئ او تفصیلات تنظیم کړئ.

![Add Membership modal](/img/admin/membership-add-modal.png)

بیا **Create Membership** باندې کلیک وکړئ. نوي اشتراک د اشتراکونو لیست کې ښکاره کیږي.
