# مدیریت سایت‌ها {#managing-sites}

سایت‌ها (یا زیرسایت‌ها) کسب و کار WaaS شماست. Ultimate Multisite سه نوع سایت دارد:

- **مالک مشتری (Customer-owned)** — سایت‌هایی که متعلق به مشتریان خاص هستند
- **قالب‌های سایت (Site Templates)** — سایت‌های از پیش تعریف شده‌ای که مشتریان می‌توانند به عنوان نقطه شروع انتخاب کنند
- **سایت اصلی (Main Site)** — شبکه اصلی شما

## مشاهده سایت‌ها {#viewing-sites}

برای دیدن تمام زیرسایت‌های شبکه خود، به **Ultimate Multisite → Sites** بروید. هر سایت با نوعش مشخص می‌شود و می‌توانید آن را بر اساس همه سایت‌ها، مالک مشتری، قالب‌ها یا در انتظار فیلتر کنید.

![صفحه لیست سایت‌ها](/img/admin/sites-list.png)

## اضافه کردن یک سایت جدید {#adding-a-new-site}

برای ایجاد یک سایت جدید، روی دکمه **Add Site** کلیک کنید. شما باید موارد زیر را ارائه دهید:

- **عنوان سایت (Site Title)** — نام سایت جدید
- **آدرس سایت (Site URL)** — دامنه/مسیر برای سایت
- **نوع سایت (Site Type)** — اینکه آیا این یک سایت مشتری، قالب یا سایت معمولی است

گزینه **Copy Site** به شما اجازه می‌دهد بر اساس یک قالب سایت موجود، یک سایت جدید بسازید. وقتی فعال شود، می‌توانید انتخاب کنید که از کدام قالب به عنوان نقطه شروع استفاده کنید. مطمئن شوید که **Copy Media on Duplication** برای شامل فایل‌های رسانه‌ای باشد فعال است.

## مدیریت یک سایت موجود {#managing-an-existing-site}

برای باز کردن صفحه **Edit Site** روی هر سایتی روی **Manage** کلیک کنید. در اینجا شما موارد زیر را خواهید یافت:

### اطلاعات پایه (Basic Information) {#basic-information}

نام سایت، نوع آن، شناسه سایت و توضیحاتش. همچنین دامنه نگاشت شده، عضویت مرتبط و حساب کاربری مشتری که مالک سایت است را مشاهده خواهید کرد.

### گزینه‌های سایت (Site Options) {#site-options}

قابلیت‌ها و محدودیت‌های سایت را تنظیم کنید:

- **محدودیت بازدیدها (Visit limits)** — حداکثر تعداد بازدیدهای سایت
- **محدودیت حساب کاربری (User account limits)** — محدودیت‌های هر نقش کاربر
- **فضای دیسک (Disk space)** — سهم ذخیره‌سازی برای سایت
- **دامنه سفارشی (Custom domain)** — فعال کردن نگاشت دامنه برای این سایت
- **دید سایت‌ها و قالب‌ها (Plugin and theme visibility)** — کنترل اینکه کدام افزونه‌ها و قالب‌ها قابل مشاهده، پنهان یا از پیش فعال هستند

د پیشټاکه، ویبسایټونه هغه محدودیتونه تعقیبوي چې د ملګرتیا (membership) سطح کې تنظیم شوي دي، ورته پاتې کیږي. که تاسو په ویبسایټको سطح کې محدودیتونه تنظیم کړئ، نو دا تنظیمات ملګرتیا ayarlاتو ته precedence ورکړي او دوی یې بدلولی دي.

### اړیکې (Associations) {#associations}

د ویبسایټ د خپله معلومات لاندې پیدا کیږي:

- **موجوده دومیني (Mapped domains)** چې له ویبسایټ سره تړلي دي
- **ملګرتیا (Membership)** چې ویبسایټ د کوم ملګرتیا अन्तर्गत دی
- **د کارونکي حساب (Customer account)** چې له ویبسایټ سره تړلی دی

### د ښي اړخ (Right Sidebar) {#right-sidebar}

په ښي اړخ کې تاسو کولی شئ:

- **د ویبسایټ فعال یا غیر فعال کړئ** په یوه toggled button سره
- **د ویبسایټ ډول بدل کړئ** او مالکیت بیا تخصیص کړئ
- **د ویبسایټ عکس/Thumbnail تنظیم کړئ** (که د ویبسایټ قالبونو لپاره پر فرونت end کې ښودل شي)
- **د ویبسایټ په دائمی ډول حذف کړئ**

:::warning
د ویبسایټ حذف کول بدلون نه کیږي. ویبسایټ او د هغه ټول محتوا د永远 حذف کیږي.
:::
