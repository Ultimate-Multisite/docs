# ډش بورد اوورفيو

Ultimate Multisite د Ultimate Multisite منو کې یو ځانګړی **Dashboard** لري، او دا هم към Network Dashboard ته ویجټیکونه اضافه کوي. دا مقاله په دواړو ځایونو کې څه وګورئ، دا توضیحوي.

## Network Dashboard

Ultimate Multisite ګټور ويجټیکونه د network-level dashboard کې ورسره کوي چې تاسو یې **My Sites → Network Admin → Dashboard** کې وموندئ.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

هر ويجټیک څه وړاندې کوي:

**First Steps:** په تازه نصب شوي نظام کې ښکاري او تاسو ته یادونه کوي چې Setup کارونه پوره کړئ لکه د طالایي طریقې تنظیمول او یو آزمویني حساب جوړول.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** د ورځني ثبتو او ترلاسه شوي پیسو تعداد راپور ورکوي. MRR (Monthly Recurring Revenue) د کلونو لپاره تخمینی ټول تګلارې ښیي چې پیسې ګټونکي सदस्यता لرونکو کلاینټانو ته تعلق دي.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** د Ultimate Multisite په ګډون پیښو راپور ورکوي — ثبتونه، منسوخی، سایت جوړول، حذف کول او نور فعالیتونه.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** د network کې د کارونکو او سائټونو لنډ خلاصې ښیي چې نوی سائټونه یا کارونکي جوړولو لپاره اختصار لري.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** وروستۍ Ultimate Multisite خبرې، په ګاتړل کې تازه معلومات، حساس غلطۍ او امنیتي اصلاحات ترلاسه او ښیي.

![News widget](/img/admin/network-dashboard-news.png)

## The Ultimate Multisite Dashboard

پداسې حال کې چې network dashboard د network په اړه ټول شامل معلومات وړاندې کوي، Ultimately Multisite dashboard (په Ultimate Multisite منوको سرتاس کې) د خدمت په اړه معلومات وړاندې کوي.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**ماهي ماهرانه درآمد شهري (Monthly Recurring Revenue Growth):** دا د ګوره په ۱۲ میاشتنۍ تقویمه پورې د درآمد د تفکیک په صورت کې ښیي. نوی درآمد په نیولوي (blue) ښودل کیږي، پاتې کېدل (cancellations) په سرخ ښودل کیږي.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**د تاریخ د محدودیتونه:** تاسو کولی شئ د وخت د محدودیتونو له لارې د ګوره معلومات فیلتر کړئ ترڅو върху هغه معلومات تمرکز وکړئ چې تاسو یې اړتیا لرئ. د تاریخ محدودیتونه هغه څه رقم کوي چې په زیرلý ویجټونو کې ښودل کیږي.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

د ګوره ویجټونه شامل دي:

- **درآمد (Revenue)** — د دې دور لپاره ټول درآمد تر含لو شهري tekrوارونکی درآمد (MRR) او په هر محصول پورې تفکیک
- **نوی ملګرتیاوې (New Memberships)** — په انتخاب شوي دوره کې څومره نوي ملګرتیاوې جوړې شوې دي
- **د هولې برخو له لارې ثبتونه (Signups by Countries)** — د ثبتونو جغرافیایی تفکیک
- **د formuونو له لارې ثبتونه (Signups by Forms)** — په هر چاکഔچه فرم له لارې څومره کليان ثبت شوي دي مقابل هغه چې مدیران اضافه کړی دي
- **زیاده تر ډېر ورته لیدل شوي ویبسایټونه (Most Visited Sites)** — د ستاسو شبکه کې تر ډېره لیدل شوي ویبسایټونه (د دې لپاره چې په **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** کې لیدل محدودیت او شمارول فعال کړئ)
