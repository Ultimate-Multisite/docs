# انٹرنیٹ سرچ

Gratis AI Agent v1.5.0-এ একটি **Internet Search** ক্ষমতা যোগ করা হয়েছে, যা কথোপকথনের সময় এআই সহকারীকে ওয়েব থেকে একদম হালনাগাদ তথ্য পুনরুদ্ধার করতে দেয়। এটি [Brave Search API](https://brave.com/search/api/) বা [Tavily API](https://tavily.com/) দ্বারা চালিত হয়।

## এটি কিভাবে কাজ করে

যখন ইন্টারনেট সার্চ চালু করা থাকে, তখন এআই সহকারী যখন বুঝতে পারে যে কোনো প্রশ্নের জন্য বর্তমান বা বাইরের তথ্যের প্রয়োজন — যেমন সাম্প্রতিক খবর, লাইভ মূল্য, বা মডেল প্রশিক্ষণের কাটঅফ তারিখের পর পরিবর্তিত হতে পারে এমন ডকুমেন্টেশন — তখন সে স্বয়ংক্রিয়ভাবে আপনার কনফিগার করা সার্চ প্রোভাইডারের কাছে প্রশ্ন করতে পারে।

ফলাফলগুলি রিয়েল টাইমে পুনরুদ্ধার করা হয় এবং উত্তর তৈরি করার আগে এআই-এর কনটেক্সটে প্রবেশ করানো হয়। যখন কোনো প্রশ্নের উত্তর দেওয়ার জন্য এটি সার্চ রেজাল্ট ব্যবহার করে, তখন সহকারী তা নির্দেশ করে।

## ইন্টারনেট সার্চ চালু করা

ইন্টারনেট সার্চের জন্য আপনার নির্বাচিত সার্চ প্রোভাইডারের একটি API কী প্রয়োজন। এটি সেটআপ করতে:

১. **Gratis AI Agent → Settings → Advanced**-এ যান।
২. **Internet Search Provider** ড্রপডাউনটি খুঁজে বের করুন এবং এর মধ্যে **Brave Search** বা **Tavily** নির্বাচন করুন।
৩. সংশ্লিষ্ট ফিল্ডে আপনার API কী প্রবেশ করান। প্রতিটি ফিল্ডের পাশে সাইন-আপ লিঙ্ক দেখানো হবে।
৪. **Save Settings**-এ ক্লিক করুন।

কী সেভ করার পরে, ইন্টারনেট সার্চ ক্ষমতাটি স্বয়ংক্রিয়ভাবে সহকারীকে ব্যবহারের জন্য উপলব্ধ হয়ে যাবে।

## Brave Search

### একটি Brave Search API কী পাওয়া

১. [Brave Search API পেজ](https://brave.com/search/api/) ভিজিট করুন।
২. একটি প্ল্যানের জন্য সাইন আপ করুন। মাসিক অনুরোধের সীমা সহ একটি ফ্রি টিয়ার উপলব্ধ আছে।
৩. Brave Search ডেভেলপার ড্যাশবোর্ড থেকে আপনার API কী কপি করুন।
৪. Gratis AI Agent সেটিংসের **Brave Search API Key** ফিল্ডে এটি পেস্ট করুন।

### ব্যবহারের সীমা

استعمال د دې برخه په Brave Search باندې د پرېښو پوښتنو او درخواستونو شمېر پورې کرایي کیږي. هر هغه AI ځواب چې سرچش (search) ولري، یو پوښتنه ګڼل کېږي. ترڅو له ناڅاپي کرایي کولو څخه ډډه وکړئ، خپل استعمال په [Brave Search developer dashboard](https://brave.com/search/api/) کې تعقیب کړئ.

## Tavily

Superdav AI Agent v1.10.0 د **Tavily** یو بدلون (alternative) د اینټرنیټ سرچش وړاندې کوي، چې ډېر لوړ او پیشینې څیړنې (research capabilities) وړاندې کوي.

### Tavily API Key ترلاسه کول

۱. [Tavily API page](https://tavily.com/) ته ورشي.
۲. د اکاونټ لپاره ثبت او راغلي وکړئ. یو وړیا سطح (free tier) هم موجود دی چې په میاشتې کې محدود شمېر درخواستونه ورکوي.
۳. خپل API key له Tavily dashboard څخه کاپی کړئ.
۴. هغه یې په Gratis AI Agent تنظیمات کې د **Tavily API Key** خان کې ورسره کړئ.

### استعمال محدودیتونه (Usage Limits)

استعمال د Tavily پر اساس د API کالونو شمېر پورې کرایي کیږي. هر هغه AI ځواب چې سرچش ولري، یو کال ګڼل کېږي. ترڅو له ناڅاپي کرایي کولو څخه ډډه وکړئ، خپل استعمال په [Tavily dashboard](https://tavily.com/) کې تعقیب کړئ.

## اینټرنیټ سرچش غیر فعال کول (Disabling Internet Search)

API key یتې فعال سرچش پليټ فارم څخه منوځته کړئ او سېوولئ (save). د助手 لپاره اینټرنیټ سرچش قابلیت له دې وروسته وړاندې نه کیږي.

:::note
اینټرنیټ سرچش ځوابونه تولیدولو خوندیت (latency) اضافه کوي ځکه چې助手 باید ځواب تولید کولو څخه مخکې سرچشको نتایج انتظار کړي. د وخت حساس استعمال لپاره، فکر وکړئ چې آیا ریل-ټاین سرچش اړین دی یا آیا د助手内置 معلومات کافی دي.
:::
