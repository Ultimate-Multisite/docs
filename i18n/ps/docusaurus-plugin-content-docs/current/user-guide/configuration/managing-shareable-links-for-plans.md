# دایرکټل لینکونه مدیریت کول د پلانونو لپاره {#managing-shareable-links-for-plans}

Ultimate Multisite تاسو ته اجازه ورکوي چې پلانونو د لینکونه مستقیم ورسره کلوونکو سره شریک کړئ. دا کار کولو لپاره، بلکې د کوم پلانونو ویرایش (edit) صفحات ته لاړ شئ او "_Click to copy the Shareable Link_" د ګډون وړ لینک د ګډون لپاره کلیک وکړئ.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_د پلان ویرایشي صفحه کې، تاسو کولی شئ د ګډون وړ لینکونه جوړولو د دکټن (button) پیدا کړئ._

دلته وګورئ چې دا میکانیزم د پیشټ ډیټل (default billing frequency) کارولای لپاره کار کوي ترڅو لینک جوړ کړي. دا دا معنی لري چې که ستاسو پیشټ ډیټل میاشتې وي، زموږ په پایله کې تاسو به یو لینک لرئ لکه دا: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

که تاسو غواړئ چې ګډون وړ لینک د ربعي (quarterly) لپاره پیشټ وي، نو تاسو باید په پایله کې 3 اضافه کړئ:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

که تاسو پلان "_premium_" د ساليانه چاپیریال (yearly schedule) لپاره غوره کوئ، نو تاسو باید 1 او ورته کلمه اضافه کړئ:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

په یاد ولرئ چې تاسو کولی شئ یوازې یو لینک د مختلف پیشټ ډیټل لپاره ونه لرئ. تاسو باید د هر ډول پیشټ ډیټل لپاره یو ځانګړی لینک شریک کړئ.

کله چې تاسو لینک ورسره خپل کلوونکي شریک کوئ، نو دا دوی ته ستاسو ثبت کولو پروسېको په пърو مرحله کې ريډی렉ټ (redirect) ورکوي.
