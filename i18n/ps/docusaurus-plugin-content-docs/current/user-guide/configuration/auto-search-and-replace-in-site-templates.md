# سیت ٹیمپليټونو کې خودکار търسنه او بدلول (v2)

_**دا تعلیمی راهندل د Ultimate Multisite نسخه ۲.x ته اړتیا لري.**_

Ultimate Multisite د ډیر قوي خوند څخه دی چې تاسو ته اجازه ورکوي په رეგისტრاشن فرم کې هر ډول متن، रنګونه، او انتخاب شوي felds اضافه کړئ. کله چې موږ دا معلومات ترلاسه کړو، موږ کولی شو له هغه کار څخه کار ونیو چې په سیت ٹیمپليټونو کې منتخب شوي ځانګړتو برخو ته محتوا له مخکې پرې پُر کړو. بیا، کله چې نوی سایت راغلاست، Ultimate Multisite د Placeholder (محلول ځایونه) بدلوي په هغه واقعی معلومات چې رეგისტრاشن کې ورته ورکړل شوی دی.

د مثال په توګه، تاسو کولی شئ خپل ٹیمپليټ سیتونو ته placeholder (محلول ځایونه) اضافه کړئ. Placeholder یوه دوه curly braces سره راؤندل باید اضافه شي - {{placeholder_name}}.

بیا، تاسو ببس یو متناظر رეგისტრاشن فیلد اضافه کولی شئ ترڅو دا معلومات ترلاسه کړئ.

خپل کارمند (customer) بیا د رეგისტრاشن په وخت کې هغه فیلد پوره کولی شي.

Ultimate Multisite بیا Placeholder (محلول ځایونه) له ورته کارمند طرفه ورکړل شوي معلومات سره خودکار بدلوي.

## **"سیت چې placeholder لرو" problemet حل کول**

دا ټول ښه دی، خو موږ یو بد مشکله ته ورشي: اوس زموږ سیت ٹیمپليټونه - چې کارمænd (customers) یې راغلاست، Placeholder (محلول ځایونه) له بد شکل دي چې ډیر خبرې نه کوي.

د دې حل لپاره، موږ د placeholder (محلول ځایونو) لپاره مې جعلی ارزښتونه (fake values) تنظیم کولو انتخاب وړاندې کړی دی، او موږ دا ارزښتونه په سیت ٹیمپليټونو کې د کارمænd راغلاست له مخکې търسنه او بدلولو لپاره کار کوو.

تاسو کولی شئ د placeholder (محلول ځایونو) ویډیو ایډتور ته ورسره کړئ ترڅو **Ultimate Multisite > Settings > Sites** ته ځئ، په Site Template Options همड्डा وکړئ، او بیا **Edit Placeholders** لینک باندې کلیک وکړئ.

![Site Template Options area in the Sites settings page](/img/config/settings-sites-templates-section.png)

دا تاسو ته د placeholder (محلول ځایونو) محتوا ایډتور ته ورسره کوي، چیرې چې تاسو کولی شئ placeholder (محلول ځایونه) او د دوی اړوند محتوا اضافه کړئ.
