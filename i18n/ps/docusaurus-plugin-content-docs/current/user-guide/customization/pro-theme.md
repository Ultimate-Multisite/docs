# خپل ایڈمن پینل د PRO Theme سره شخصی کول {#customizing-your-admin-panel-with-pro-theme}

کله چې موږ د کلاینټ کار کوو، ډیری وختونه موږ باید خپل کلاینټانو ته ونه یوه ځانګړت تجربه په WordPress د ایڈمن ډشټ‌بورد کې ورکړو. که تاسو یو ډېر حرفي او ښکلی شکل غواړئ، PRO Theme به ستاسو مرسته وکړي.

PRO Theme یو ایڈمن theme دی چې ستاسو کلاینټانو د ډشټ‌بورد شخصی کولو کې ستونزه وکړئ. او دا هم Ultimate Multisite د پرمیوم plugin core څخه یوه برخه ده.

د دې نصب کولو وروسته، ته خپل _Plugins_ صفحه ته لاړ شئ او په خپل Network کې یې فعالولو لپاره کلیک وکړئ.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_په ستاسو د Plugins صفحه کې په ټول Network کې فعال کړئ_

_**نوټ:**_ _که تاسو یوازې یې نصب کړی، نو دا د کلاینټانو ته به د یوې অপشن په توګه ښکاره شي چې کولی شي فقط د هغه کارولو لپاره theme اضافه شي._

خله چې ستاسو ایڈمن interface پوره بیا ډیزاین شوی! ستاسو لنډه ستونزه کې هم PRO Theme یوه نوې منو (menu) لري چې "PRO Theme" وی نوم لري.
