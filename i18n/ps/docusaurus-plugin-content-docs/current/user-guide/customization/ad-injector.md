# خپل شبکهमा اشتہارونه درج کولو لپاره Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector دا یو **رایگان اضافي** دی چې تاسو ته اجازه ورکوي په راتلونکي ویرایش صفحه له لارې خپل پلانونوमा اشتہارونه درج کولوका लागि وسایل وي.

Ultimate Multisite: Ad Injector نصب गरेपछि، فعالဖို့को लागि आफ्नो _Plugins_ पेजमा जानुहोस्।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate बटनमा क्लिक गर्नुहोस्_

एक पटक यो اضافي सक्रिय भएपछि, प्लान सम्पादन स्क्रिनमा रहेको _Advanced Options_ मेटा बक्समा _Ad Settings_ नामक नयाँ विकल्प देखिनेछ।

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_अब तपाईंले आफ्ना प्रत्येक प्लानका लागि विज्ञापन सेटिङहरू सम्पादन गर्न सक्नुहुन्छ_

त्यो ट्याबमा, तपाईंले फ्रन्ट-एन्डमा पोस्ट र पेजहरूको लागि विज्ञापन कोड थप्न सक्नुहुन्छ, सामग्रीको अगाडि र पछाडि।
