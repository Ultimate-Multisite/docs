# داسېสมาชิก ختمول (v2)

_**مهم یادداشت: دا مقاله Ultimate Multisite نسخه 2.x ته اشاره کوي.**_

د کارندونکو طرف د سټورینګ ختمول ډېر ساده دی.

که ستاسو کاربری غواړي خپل सदस्यता ختم کړي، باید په پخوانیو **Account** صفحه رامنځته کړي.

هنو، دوی باید تر دې **Danger Zone** پورې کلیک کړي او د **Delete Account** لپاره کلیک کړي.

![د اکاونټ صفحې خطر زون چې Delete Account د ګوډۍ د ګوډۍ په توګه ښودل کیږي](/img/account-page/danger-zone.png)

یو ماډل खुسار کیږي چې له دوی څخه تایید غواړي. دوی باید **Confirm Account Deletion** خپله گزینه فعال کړي او د **Delete Account** لپاره کلیک کړي.

![تایید اکاونټ ختمول ډیالوګ (/img/account-page/delete-account-confirm.png)

دا دوی له سټورینګ څخه وېرېږي او ستاسو شبکه کې د دوی सदस्यता ختموي، او د دوی سټورینګ هم ستاسو شبکه څخه منځ ته اخلي.

![د اکاونټ ختمولو وروسته وېرېدو (/img/account-page/logged-out-after-delete.png)

![مستندوي ختمول تایید](/img/account-page/membership-canceled.png)

**مهم:** په دې توجه وکړئ چې دا عمل کارندونکي له ستاسو شبکه څخه منځ ته نه کوي، ځکه که یو کارندونکی می‌تواند له هغه باندې یوه یا ډېر सदस्यता ولري او هیڅ هم सदस्यता لینک شوی نه وي.

![ختمولو وروسته کارندونکی په شبکه کې ورسنده دی (/img/admin/customer-after-cancel.png)

![کارندونکی چې فعال सदस्यता نلرو](/img/admin/memberships-customer-empty.png)
