# څنګه زه به سبسکرپشني ختم کړم؟ {#how-can-i-cancel-my-subscription}

که تاسو Ultimate Multisite license لرئ، تاسو کولی شئ د بیا تازه کولو (renewal) وخت چې غواړئ، یې ختم کړئ. ستا دا لارښوونې ورسره کیږئ:

<!-- Screenshot unavailable: Account page email with access link -->

د هغه ایمیل او خپل پاسور प्रयोग کړئ چې په کلوونه یې کرښه کړی دی او د پیرودولو وخت کې ورکړل شوی و.

<!-- Screenshot unavailable: Login page with email and password fields -->

دا ستاسو د اکاونټ صفحې پیل والی سکرین دی:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

ستاسو سبسکرپشني ختم کولو لپاره، په چپ اړخ کې مینو ته لاړ شئ او "Renewing & Billing" (بیا تازه کول او بیا کولو معلومات) ته ورسئ.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

د ښي اړخ څخه تیرلونکي نشان باندې کلیک وکړئ ترڅو یو اړخ پټه وي. بیا، تاسو باید هغه انتخاب غوره کړئ چې **auto-renew** (د خودکار بیا تازه کولو) ختموي.

سیستم ستاسو د تایید پیغام ښیېږي.

که تاسو ډاډه یم چې غواړئ سبسکرپشني ختم کړئ، که څه هم، فقط "**Cancel Renewals**" (بیا تازه کول ختم کړئ) د ګوټۍ باندې کلیک وکړئ. د دې عمل وروسته، ستاسو څخه یو چټک سروی ختمول کیږي.

تام! ستاسو سبسکرپشني خودکار به بیا تازه نه شي.

**ستاسو کې یوه لاغری (key) تا هغه وخت ورکړل کیږي چې ستاسو سبسکرپشني ختم وي**. که تاسو غواړئ ستاسو سبسکرپشني بیا فعال کړئ، تاسو باید دا کار په manuall (د ځانګړي لارښو سره) وکړئ.
