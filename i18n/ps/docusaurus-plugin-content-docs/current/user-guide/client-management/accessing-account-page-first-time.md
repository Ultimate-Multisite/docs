# څنګه زه په پہلونو ته ورسره؟ {#how-can-i-access-my-account-page-for-the-first-time}

خوشالیو ته ښه راغلاست!

د خپل اکاونټ صفحې ته د هغه لینک له لارې ورسره کړئ چې تاسو د ماینځي د پليګاین (plugin) کلاه ونیو څخه ترلاسه کړی دی:

<!-- Screenshot unavailable: Account page email with access link -->

د هغه ایمیل کارول چې تاسو یې د کلاه کولو په وخت کې ورکړی و، او هغه پاسور چې تاسو ایمیل کې ترلاسه کړی و.

<!-- Screenshot unavailable: Login page with email and password fields -->

چون دا ستاسو لغو (logging in) د دې اکاونټ لپاره تللته ده، تاسو ته غوښتنه وشي چې خپل پاسور بدلون وکړئ. هغه بدل کړئ او _Save_ د ګډ按钮 باندې کلیک وکړئ.

<!-- Screenshot unavailable: Change password form with Save button -->

د ستاسو اکاونټ صفحې کې، تاسو کولی شئ ټول ستاسو اکاونټي معلومات مدیریت کړئ لکه د فادو (billings)، بیا کولو (renewals)، لغو (cancellations)، او نور ډیر شیان.
