# داسې کارانانو د ویبسایټونه (v2) {#your-clients-websites-v2}

د خپل شبکه کې ثبتي کولو وروسته، کلاینټونه خپل ډشډبورد ته ورسره کولی شي.

د **Account** صفحې له سربیره، چې چیرې دوی می‌تواند د دوی د بیا کولو معلومات، ملګرتیاو، د ډومینونه، پلان محدودیتونه او نور مهم معلومات وګوري، دوی به **My Sites** صفحې ته هم ورسره وي، چې په دې کې دوی کولی شي ټول ویبسایټونه وګوري او اداره کړي. دوی باید یوازې د خپل ډشډبوردमा My Sites باندې کلیک وکړي.

![د کلاینټ د ډشډبورد کې د My Sites لینک](/img/account-page/my-sites-link.png)

له هوښیارۍ، دوی ټول ویبسایټونه ورته ښودل کیږي، او د هر یو داسې په دوه یې **Manage** (مدارو) د ګډ按钮 سره به وي. که دوی کلیک وکړي، دوی به د دې ځانګړي ویبسایټको **dashboard** ته رول شي.

دوی دا هم وګوري کولی شي چې کوم یو له دوی د ویبسایټونه د پلانونو لپاره **primary** (اساسي) دی چې دوی اجازه ورکوي چې ډېر یوه ویبسایت ولري.

![My Sites صفحه چې کلاینټانو د ویبسایټونو سره Manage د ګډي برخې ښیي](/img/account-page/my-sites-page.png)

دا په ویبسایت ویرایش (edit) چارچوب کې ټول توضیحات دي:

![د ویبسایت ویرایش پوره چارچوب](/img/admin/site-edit-full.png)

که دوی غواړي خپل اصلي ویبسایت بدلو، دوی باید د **Account** صفحې ته ورشي چې **Account** منو (menu) باندې کلیک وکړي.

![د کلاینټ د ډشډبورد کې د Account منو لینک](/img/account-page/account-menu.png)

دوی کولی شي د کلاینټ د معلومات هم وګوري او ویرایش کړي:

![کلاینټ ویرایش چارچوب](/img/admin/customer-edit.png)

هغه بیا باید **Change Default Site** (د پرانیستې ویبسایت بدلول) باندې کلیک وکړي.

![د Account صفحه کې د Change Default Site د ګډي برخې върху تصویر](/img/account-page/change-default-site.png)

یو پاپ اپ ویندو به ښودل شي. په دراپ ڈاؤن لیست باندې کلیک وکړئ او غوره کړئ چې کوم ویبسایت غواړئ چې اصلي ویبسایت جوړ کړئ.

![Change Default Site د دراپ ڈاؤن انتخاب](/img/account-page/change-default-site-dropdown.png)

بیا، یوازې **Change Default Site** باندې کلیک وکړئ او کار بشپړ!

![Confirmation of the default site change](/img/account-page/change-default-site-confirm.png)
