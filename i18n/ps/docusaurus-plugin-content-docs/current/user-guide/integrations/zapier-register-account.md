# رویداد: Zapier از طریق Google Sheets کے ذریعے اکاؤنٹ رجسٹر کرنا

اس مضمون [Integrating Ultimate Multisite with Zapier](zapier.md) میں، ہم نے بات کی تھی کہ کیسے آپ Zapier کا استعمال ٹرگرز اور ایونٹس کی بنیاد پر Ultimate Multisite کے اندر مختلف کام کرنے کے لیے کر سکتے ہیں۔ اس مضمون میں، ہم دکھائیں گے کہ آپ تھرڈ پارٹی ایپلی کیشنز کو کیسے جوڑ سکتے ہیں۔ ہم ڈیٹا کے ذریعہ کے طور پر Google Sheets کا استعمال کریں گے اور معلومات کو Ultimate Multisite کو بھیجیں تاکہ اکاؤنٹ رجسٹر ہو سکے۔

Zapier سے پہلے کنیکٹ کرنے کے لیے، **Ultimate Multisite > Settings > API & Webhooks** پر جائیں اور اس بات کی تصدیق کریں کہ API فعال ہے۔ جب Zapier آپ سے Ultimate Multisite اکاؤنٹ کی تفصیلات مانگے تو اسکرین سے API Key اور API Secret کاپی کر لیں۔

![API and Webhooks settings with API Key, API Secret, and Enable API options](/img/admin/settings-api-webhooks.png)

سب سے پہلے، آپ کو اپنی Google Drive کے تحت ایک **Google Sheet** بنانی ہوگی۔ اس بات کو یقینی بنائیں کہ آپ ہر کالم کو ٹھیک سے بیان کریں تاکہ بعد میں ڈیٹا کو آسانی سے مپ (map) کر سکیں۔

ایک Google sheet بنانے کے بعد، آپ اپنے Zapier اکاؤنٹ میں لاگ ان کر سکتے ہیں اور ایک zap بنانا شروع کر سکتے ہیں۔

**"App event"** کے سرچ فیلڈ کے نیچے **"Google Sheets"** منتخب کریں۔

پھر "**Event** " فیلڈ کے لیے **"New spreadsheet row"** منتخب کریں اور **"Continue"** پر کلک کریں۔

اگلا قدم آپ سے ایک **Google Account** منتخب کرنے کے لیے پوچھے گا جہاں **Google Sheet** محفوظ ہے۔ لہذا بس اس بات کو یقینی بنائیں کہ صحیح گوگل اکاؤنٹ درج کیا گیا ہے۔

**"Set up trigger"** کے نیچے، آپ کو وہ گوگل اسپریڈ شیٹ اور ورکشیت منتخب اور بتانی ہوگی جس کا استعمال آپ ڈیٹا حاصل کرنے کے لیے کریں گے۔ بس اسے پُر کر دیں اور **"Continue"** پر کلک کریں۔

اگلا قدم یہ ہے کہ اپنے گوگل شیت کو ٹھیک سے جوڑنے کے لیے **"test your trigger"** کریں۔

اگر آپ کی جانچ کامیاب ہو جاتی ہے، تو آپ کو نتائج میں کچھ ویلیوز نظر آنی چاہئیں جو آپ کی اسپریڈ شیٹس سے آتی ہیں۔ آگے بڑھنے کے لیے **"Continue"** پر کلک کریں۔

د تلل مرحله چې دوه عمل جوړوي چې د Ultimate Multisite کې حساب جوړ کړي یا ثبت کړي، هغه ده. په سرچyny خانې کې "**Ultimate Multisite(2.0.2)** " غوره کړئ.

د "**Event** " خانې تحت، غوره کړئ "**Register an Account in Ultimate Multisite** " او بیا "**Continue** " د ګډه کولو لپاره کلیک وکړئ.

د "**Set up an action** " تحت، تاسو به وګورئ چې د کارندوی معلوماتو، MEMBERSHIPS (عضویتونه)، PRODUCTS (مجموعې)، او نورو لپاره مختلف خانې رامنځته شوي دي. تاسو کولی شئ هغه ارزښتونه په خپل Google Sheet کې نقش کړئ او دوی له هغه ځای ته ورکړئ چیرې چې باید ورسره پر ځای شي، لکه څنګه چې په زیرلý تصویر کې ښودل شو.

د ارزښتونو د نقش کولو وروسته، تاسو کولی شئ عمله (action) آزموینئ.
