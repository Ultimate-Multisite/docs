# د Chuck Norris حقایق

د **Ultimate Multisite - Chuck Norris Facts** اضافې سره خپلو ورځنیو WordPress اداري دندو ته لږه ټوکه ورزیاته کړئ. کله چې فعاله شي، د هرې اداري پاڼې په پاسنۍ ښي کونج کې به د Chuck Norris یو تصادفي حقیقت ښکاره شي.

ستاسو ډلې ته د موسکا راوستلو، د اوږدو اداري ناستو پر مهال د فضا سپکولو، یا یوازې د هغه افسانوي کمال لمانځلو لپاره بشپړ دی چې Chuck Norris دی.

## ولې د Chuck Norris حقایق؟

* **د ډلې روحیه لوړه کړئ**: خپلې شبکې ادارې ته یو ساتېری عنصر ورزیات کړئ
* **یخ مات کړئ**: د نوو ډلې غړو لپاره د خبرو ښه پیل
* **د تولید وړتیا زیاتوالی**: یوه چټکه خندا تمرکز او تولید وړتیا ښه کولی شي
* **بې‌وخته ټوکه**: د Chuck Norris حقایق هېڅکله زړېږي نه

## مهمې ځانګړتیاوې

**۱۱۰+ افسانوي حقایق**
د کلاسیکو Chuck Norris ټوکو او حقایقو یوه پراخه ټولګه چې خپله افسانه لمانځي.

**بې‌مزاحمته ښودنه**
حقایق په اداري پټه کې په داسې نامزاحمه بڼه ښکاري چې ستاسو د کار بهیر سره مداخله نه کوي.

**ژباړې ته چمتو**
ټول حقایق د ژباړې په دندو کې نغښتل شوي، چې ستاسو ژبې ته یې ځایي کول یا ستاسو د ډلې لپاره ځانګړي حقایق جوړول اسانه کوي.

**بشپړ ځواب ویونکی**
حقایق په ډیسکټاپ، ټابلیټ، او موبایل پردو کې سم ښکاري - Chuck Norris په ټولو وسیلو کې عالي دی.

**د RTL ملاتړ**
د ښي-نه-کیڼ ژبو لپاره بشپړ ملاتړ، ځکه Chuck Norris د ژبې خنډونه تېروي.

**له بلاک سمونګر سره سازګار**
حقایق په هوښیارۍ سره په بلاک سمونګر کې پټول کېږي څو د انترفېس له ټکرونو مخنیوی وشي.

**پر لاسرسي متمرکز**
د بشپړ لاسرسي د مطابقت لپاره مناسب د سکرین لوستونکي متن پکې شامل دی.

## د چا لپاره بشپړ دی

* د څو-سایټ شبکې مدیران چې غواړي شخصیت ورزیات کړي
* پراختیایي ډلې چې ښه ټوکه ستایي
* هر څوک چې د Chuck Norris د کمال ورځني خوراک ته اړتیا لري
* هغه سایټونه چې غواړي یو ساتېری، مثبت کاري چاپېریال جوړ کړي

## لګول

1. ډاډ ترلاسه کړئ چې Ultimate Multisite plugin ستاسو په WordPress شبکه کې لګول شوی او فعال دی
2. د اضافې فایلونه خپل `/wp-content/plugins/` ډایرکټرۍ ته پورته کړئ
3. plugin په WordPress کې د 'Plugins' مېنو له لارې د شبکې په کچه فعال کړئ
4. په هره اداري پاڼه کې د Chuck Norris له تصادفي حقایقو خوند واخلئ!

هېڅ تنظیم ته اړتیا نشته - یوازې یې فعال کړئ او موسک شئ!

## ډېرې پوښتل شوې پوښتنې

## ایا دا Ultimate Multisite ته اړتیا لري؟

که څه هم د Ultimate Multisite اضافې په توګه ډیزاین شوی، plugin په خپلواک ډول کار کوي او د WordPress په هر نصب کې چلولی شي.

## ایا زه خپل حقایق ورزیاتولی شم؟

هو! حقایق ژباړې ته چمتو دي. تاسو کولی شئ له ځانګړو حقایقو سره خپل `.po` فایل جوړ کړئ او هغه په `/lang` ډایرکټرۍ کې کېږدئ. یا تاسو کولی شئ د حقایقو array په پروګرامي ډول filter کړئ.

## ایا دا زما اداره ورو کوي؟

هېڅکله نه! plugin ډېر سپک دی او پر کړنو هېڅ اغېز نه لري. دا یوازې متن ښکاره کوي - نه بهرنۍ API غوښتنې او نه درنه پروسسونه.

## ایا زه یې په ځینو پاڼو کې غیرفعالولی شم؟

حقایق په هغو پاڼو کې په اتومات ډول پټېږي چې ښايي مداخله وکړي (لکه بلاک سمونګر). تاسو همدارنګه کولی شئ WordPress hooks وکاروئ څو دا ځانګړی کړئ چې حقایق چېرته ښکاره شي.

## ایا حقایق د ټولو لیدونکو لپاره مناسب دي؟

هو! ټول حقایق پاک، د کورنۍ لپاره مناسب طنز دی چې د Chuck Norris افسانوي مقام لمانځي.

## ایا زه حقایق بلې ژبې ته ژباړلی شم؟

بېشکه! ټول حقایق د WordPress ژباړې دندې کاروي. د خپلې ژبې لپاره د ژباړې فایل جوړ کړئ او هغه په `/lang` ډایرکټرۍ کې کېږدئ. د متن ډومېن: `ultimate-multisite-chuck-norris-facts`

## ایا دا له څو-سایټ سره کار کوي؟

هو! دا په ځانګړي ډول د څو-سایټ شبکو لپاره ډیزاین شوی او د شبکې په کچه فعالېدلی شي.
