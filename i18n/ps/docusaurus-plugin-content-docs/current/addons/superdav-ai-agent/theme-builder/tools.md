# وسیلې او تنظیمات {#tools-and-configuration}

Theme Builder ستاسو WordPress وېبپاڼې د شننې، ډیزاین او جوړولو لپاره د وسیلو یوه ټولګه کاروي. په نسخه 1.16.0 کې، **sd-ai-agent/site-scrape اوس د ۱ کچې وسیله ده**، چې په Theme Builder کې یې په تلواله توګه د لاسرسي وړ کوي. Superdav AI Agent v1.18.0 هم د باور وړ پرمختیايي کاري بهیرونو لپاره په جلا ډول وېشل شوی پرمختللی ملګری کڅوړه معرفي کوي؛ دغه وسیلې د WordPress.org اصلي کڅوړې برخه نه دي او باید جلا نصب او مجازې شي.

## شته وسیلې {#available-tools}

### د ۱ کچې وسیلې (تل شته) {#tier-1-tools-always-available}

د ۱ کچې وسیلې په Theme Builder کې له هر ډول اضافي تنظیم پرته په تلواله توګه شته.

#### sd-ai-agent/site-scrape {#sd-ai-agentsite-scrape}

**موخه:** شته وېبپاڼې وڅېړئ څو د ډیزاین الهام، د منځپانګې جوړښت، او فعالیت راوباسي.

**وړتیاوې:**

- **د وېبپاڼې شننه** — د سیالانو یا الهام وېبپاڼې راټولول او شنل
- **د ډیزاین راایستل** — رنګونه، فونټونه، او د ترتیب نمونې پېژندل
- **د منځپانګې جوړښت** — د پاڼې تنظیم او مرتبه درک کول
- **د ځانګړنو موندنه** — pluginونه او فعالیت پېژندل
- **د کړنې شننه** — د پاڼې چټکتیا او ښه‌والي کتل
- **د SEO شننه** — meta tagونه او جوړښت‌لرونکي معلومات بیاکتل

**کارونه:**

```
د example.com ډیزاین وڅېړه او زما د وېبپاڼې لپاره یې د الهام په توګه وکاروه.
```

**څه شی راوباسي:**

- د رنګونو مجموعه او د رنګ کارونه
- ټایپوګرافي (فونټونه او اندازې)
- د ترتیب جوړښت او واټنونه
- د تګ راتګ نمونې
- د منځپانګې تنظیم
- د انځورونو او رسنیو کارونه
- متقابل عناصر
- د موبایل ځواب‌ویونکې بڼه

**محدودیتونه:**

- د پټنوم-خوندي وېبپاڼې نشي راټولولی
- د robots.txt او وېبپاڼې تګلارو درناوی کوي
- ښايي خوځنده منځپانګه ونه نیسي
- په JavaScript درنو وېبپاڼو کې ښايي شننه محدوده وي
- سترې وېبپاڼې ښايي د شننې لپاره ډېر وخت واخلي

### د ۲ کچې وسیلې (اختیاري) {#tier-2-tools-optional}

د ۲ کچې وسیلې هغه وخت شته وي چې په څرګند ډول د Theme Builder په تنظیماتو کې فعالې شي.

#### پرمختللې شننه {#advanced-analytics}

د کړنې تفصیلي کچې برابروي:

- د پاڼې د بارېدو وختونه
- Core Web Vitals
- د SEO نمره
- د لاسرسي نمره
- د غوره کړنو نمره

#### د منځپانګې ښه‌کوونکی {#content-optimizer}

د دې لپاره ښه‌والي شنني او وړاندیزونه کوي:

- لوستلو اسانتیا
- د SEO ښه‌والی
- د کلیدي کلمو کارونه
- د منځپانګې جوړښت
- د عمل-غوښتنې ځای پر ځای کول

### د ۳ کچې وسیلې (Premium) {#tier-3-tools-premium}

د ۳ کچې وسیلې اضافي تنظیم یا premium ځانګړنو ته اړتیا لري.

#### د AI منځپانګې جوړوونکی {#ai-content-generator}

د دې لپاره منځپانګه جوړوي:

- د محصول تشریحات
- د خدمت پاڼې
- د بلاګ لیکنې
- meta تشریحات
- د عمل-غوښتنې متن

#### پرمختللې ډیزاین وسیلې {#advanced-design-tools}

پرمختللې ډیزاین وړتیاوې برابروي:

- د دودیز CSS جوړول
- د انیمېشن جوړول
- د متقابل عنصر ډیزاین
- پرمختللې رنګ تیوري
- د ټایپوګرافي ښه‌والی

### پرمختللې ملګرې وسیلې (جلا کڅوړه) {#advanced-companion-tools-separate-package}

پرمختللې ملګرې کڅوړه د Superdav AI Agent له اصلي خپرونې څخه جلا وېشل کېږي. دا د باور وړ پرمختیاکوونکو او د وېبپاڼې-مالک چاپېریالونو لپاره ده، چېرې چې مدیران په څرګند ډول agent ته اجازه ورکوي چې لوړ-اغېز لرونکې وسیلې وکاروي.

د پرمختللې ملګرې وړتیاوې کېدای شي دا شاملې کړي:

- **د فایل‌سیسټم وسیلې** — په باور وړ چاپېریالونو کې منل شوي فایلونه کتل او بدلول
- **د ډیټابېس وسیلې** — د وېبپاڼې معلومات پوښتل یا تازه کول کله چې په څرګند ډول اجازه ورکړل شوې وي
- **د WP-CLI وسیلې** — د WordPress ساتنې او کتنې امرونه چلول
- **د REST لېږدونکي وسیلې** — ثبت شوي REST endpointونه د کنټرول شوو لېږد کاري بهیرونو له لارې رابلل
- **د plugin جوړوونکي وسیلې** — د plugin کوډ جوړښت جوړول او پرې تکراري کار کول
- **د Git snapshot وسیلې** — د خطرناکو پرمختیايي عملیاتو مخکې snapshotونه جوړول
- **د کارن-مدیریت وسیلې** — هلته چې اجازه وي د کارن ریکارډونو بیاکتنې یا سمون کې مرسته کول
- **د Benchmark وسیلې** — د پرمختیايي بیاکتنې لپاره د کړنې یا وړتیا benchmarkونه چلول

دا د هر نصب لپاره د عموماً شته Theme Builder وسیلو په توګه مه مستندوئ. دا یوازې هغه وخت شته وي چې پرمختللې ملګرې کڅوړه نصب، فعاله، او باور وړ مدیرانو ته محدوده شوې وي.

## د وسیلو تنظیمات {#tool-configuration}

### د وسیلو فعالول {#enabling-tools}

په Theme Builder کې د اضافي وسیلو د فعالولو لپاره:

1. **Gratis AI Agent → Settings** ته لاړ شئ
2. **Theme Builder → Tools** ته لاړ شئ
3. د اړتیا سره سم وسیلې فعالې/غیرفعالې کړئ
4. تنظیمات خوندي کړئ

### د وسیلو اجازې {#tool-permissions}

ځینې وسیلې اجازو ته اړتیا لري:

- **د وېبپاڼې راټولول** — انټرنېټ لاسرسي ته اړتیا لري
- **شننه** — د Google Analytics یوځای‌کولو ته اړتیا لري
- **د منځپانګې جوړول** — API کیلي‌ګانو ته اړتیا لري
- **پرمختللې ځانګړنې** — ښايي premium ګډون ته اړتیا ولري
- **پرمختللې ملګرې وسیلې** — جلا وېشل شوې پرمختللې کڅوړې او د مدیر څرګند باور ته اړتیا لري

### API کیلي‌ګانې او اعتبارلیکونه {#api-keys-and-credentials}

د هغو وسیلو لپاره API کیلي‌ګانې تنظیم کړئ چې ورته اړتیا لري:

1. **Gratis AI Agent → Settings → API Keys** ته لاړ شئ
2. د هرې وسیلې لپاره اعتبارلیکونه دننه کړئ
3. اړیکه وازمویئ
4. په خوندي ډول یې خوندي کړئ

### د پرمختللې ملګرې کڅوړې نصب {#installing-the-advanced-companion-package}

پرمختللې ملګرې کڅوړه د WordPress.org اصلي ZIP څخه جلا خپرېږي. دا یوازې د خپرونې لپاره د رسمي پروژې وېش چینل څخه نصب کړئ، بیا لاسرسی باور وړ مدیرانو ته محدود کړئ. په production وېبپاڼه کې د فایل‌سیسټم، ډیټابېس، WP-CLI، REST لېږدونکي، plugin جوړوونکي، کارن-مدیریت، یا benchmark وسیلو د فعالولو مخکې د انساني تایید دروازې او د git snapshot چلند بیاکتنه وکړئ.

## د sd-ai-agent/site-scrape کارول {#using-sd-ai-agentsite-scrape}

### بنسټیز کارونه {#basic-usage}

له Theme Builder څخه وغواړئ چې یوه وېبپاڼه وڅېړي:

```
په competitor.com کې زما د سیال وېبپاڼې ډیزاین وڅېړه
او زما د وېبپاڼې لپاره د ډیزاین ښه‌والي وړاندیز کړه.
```

### ځانګړې شننه {#specific-analysis}

د شننې ځانګړي ډولونه وغواړئ:

```
له example.com څخه د رنګونو مجموعه راوباسه او د الهام په توګه یې وکاروه.
```

```
د example.com د تګ راتګ جوړښت وڅېړه او زما وېبپاڼې ته ورته
تنظیم پلي کړه.
```

```
د example.com د موبایل ځواب‌ویونکې بڼه وګوره او ډاډ ترلاسه کړه چې زما وېبپاڼه
هم همدومره ځواب‌ویونکې ده.
```

### د پرتلنې شننه {#comparison-analysis}

څو وېبپاڼې سره پرتله کړئ:

```
د site1.com او site2.com ډیزاینونه سره پرتله کړه او داسې ډیزاین جوړ کړه
چې د دواړو غوره عناصر سره یوځای کړي.
```

## د وسیلو محدودیتونه او پام‌وړ ټکي {#tool-limitations-and-considerations}

### د کچې محدودول {#rate-limiting}

- Scraping د سرور د زیات بار د مخنیوي لپاره د کچې له مخې محدود شوی
- ورته site ته ګڼې غوښتنې ښايي ورو کړل شي
- لوی siteونه ښايي د شننې لپاره ډېر وخت ونیسي

### د منځپانګې دقت {#content-accuracy}

- متحرکه منځپانګه ښايي بشپړه ونه نیول شي
- د JavaScript له لارې ښودل شوې منځپانګه ښايي نیمګړې وي
- د حقیقي وخت معلومات ښايي منعکس نه شي
- ځینې منځپانګې ښايي د تادیې تر شا وي

### قانوني او اخلاقي ملاحظات {#legal-and-ethical-considerations}

- robots.txt او د site تګلارې درناوی وکړئ
- د بیا خپرولو لپاره د کاپي‌رایټ لرونکې منځپانګه مه Scrape کوئ
- شننه د الهام لپاره وکاروئ، نه د کاپي کولو لپاره
- تایید کړئ چې د استخراج شوې منځپانګې د کارولو حق لرئ
- د شنل شوو siteونو د خدمت شرطونه تعقیب کړئ

### پر کړنو اغېز {#performance-impact}

- د لویو siteونو Scraping ښايي وخت ونیسي
- څو هممهاله Scrapeونه ښايي ورو وي
- د شبکې اتصال پر سرعت اغېز کوي
- لوی datasets ښايي لا ډېر پروسس ته اړتیا ولري

## غوره کړنلارې {#best-practices}

### د Site شننې کارول {#using-site-analysis}

1. **ګڼ siteونه وشنئ** — له څو سرچینو الهام راټول کړئ
2. **پر جوړښت تمرکز وکړئ** — د ترتیب او سازمان بڼې زده کړئ
3. **رنګونه استخراج کړئ** — د پیل ټکو په توګه د رنګونو ټولګې وکاروئ
4. **ټایپوګرافي وڅېړئ** — هغه د فونټ ترکیبونه وپېژنئ چې خوښېږي مو
5. **ناوېګېشن بیاکتنه وکړئ** — د مېنو سازمان درک کړئ

### اخلاقي Scraping {#ethical-scraping}

1. **robots.txt ته درناوی وکړئ** — د site لارښوونې تعقیب کړئ
2. **منځپانګه مه کاپي کوئ** — شننه یوازې د الهام لپاره وکاروئ
3. **حقونه تایید کړئ** — ډاډ ترلاسه کړئ چې استخراج شوې منځپانګه کارولی شئ
4. **سرچینې منسوبې کړئ** — د الهام سرچینو ته کریډیټ ورکړئ
5. **شرطونه تعقیب کړئ** — د site د خدمت شرطونو سره سم عمل وکړئ

### د وسیلې اغېزمنتیا زیاتول {#maximizing-tool-effectiveness}

1. **مشخص اوسئ** — د شننې مشخص ډولونه وغواړئ
2. **شرایط برابر کړئ** — د خپل site موخه تشریح کړئ
3. **تمې وټاکئ** — خپلې د ډیزاین موخې بیان کړئ
4. **تکرار وکړئ** — د پایلو پر بنسټ یې ښه کړئ
5. **وسیلې یوځای کړئ** — د جامعې شننې لپاره ګڼې وسیلې وکاروئ

## د ستونزو حل {#troubleshooting}

### Site نه Scrape کېږي {#site-wont-scrape}

- وګورئ چې site عامه لاسرسی لري که نه
- تایید کړئ چې robots.txt د Scraping اجازه ورکوي
- یو بل site وازمایئ
- د انټرنېټ اتصال وګورئ
- که ستونزه دوام وکړي، له ملاتړ سره اړیکه ونیسئ

### نیمګړې شننه {#incomplete-analysis}

- site ښايي متحرکه منځپانګه ولري
- JavaScript ښايي بشپړ نه وي ښودل شوی
- لوی siteونه ښايي timeout شي
- پر ځای یې د مشخصو پاڼو شننه وازمایئ
- د شننې مشخص ډولونه وغواړئ

### ورو کړنې {#slow-performance}

- لوی siteونه د شننې لپاره ډېر وخت نیسي
- څو هممهاله Scrapeونه ورو وي
- د شبکې اتصال پر سرعت اغېز کوي
- د کمې بوختیا په ساعتونو کې هڅه وکړئ
- لومړی کوچنۍ برخې وشنئ
