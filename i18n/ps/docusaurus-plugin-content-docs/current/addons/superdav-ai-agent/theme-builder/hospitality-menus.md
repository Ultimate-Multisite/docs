# د مېلمه پالنې مېنوګانې

د **مېلمه پالنې مېنوګانې** ځانګړنه Theme Builder ته اجازه ورکوي چې جوړښتي خواړه او څښاک مېنو پاڼې مستقیماً ستاسو په WordPress سایټ کې جوړې او ځای پر ځای کړي.

## عمومي کتنه

Theme Builder اوس کولی شي د مېلمه پالنې کاروبارونو لپاره مسلکي، جوړښتي مېنو پاڼې جوړې کړي، چې پکې رستورانتونه، کافې، بارونه، او د کېټرنګ خدمتونه شامل دي. دا مېنوګانې ستاسو د سایټ له ډیزاین سره بشپړې مدغمې دي او په اسانۍ سره تازه او مدیریت کېدای شي.

## ملاتړ شوې د مېلمه پالنې ډولونه

- **رستورانتونه** — د بشپړ خدمت خوړو مېنوګانې
- **کافې** — د کافي او سپکو خوړو مېنوګانې
- **بارونه او لاونجونه** — د څښاک او سپکو اشتهايي خوړو مېنوګانې
- **بیکرۍ** — د پیسټري او ډوډۍ مېنوګانې
- **د کېټرنګ خدمتونه** — د پېښو د مېنو انتخابونه
- **د خوړو ټرکونه** — د ګرځنده خوړو خدمت مېنوګانې
- **بریورۍ او واینرۍ** — د توضیحاتو سره د څښاک مېنوګانې

## د مېنو جوړښت

### د مېنو کټګورۍ

مېنوګانې په کټګوریو کې تنظیمېږي:

- **اشتهايي خواړه** — پیل کوونکي او کوچني پلیټونه
- **اصلي خوړونه** — اصلي کورسونه
- **اړخیز خواړه** — ملګري خواړه او سبزیجات
- **خواږه** — خوږې خوراکي خوښۍ
- **څښاک** — څښاکونه (الکولي او غیر الکولي)
- **ځانګړي وړاندیزونه** — ورځني یا موسمي ځانګړي وړاندیزونه

### د مېنو توکي بڼه

د مېنو هر توکی پکې شامل وي:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### د توکي فیلډونه

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | د خوړو یا څښاک نوم |
| `description` | string | د توکي تفصیلي توضیح |
| `price` | string | بیه (د اسعارو سره بڼه شوې) |
| `dietary_info` | array | د خوړو اړوند ځانګړنې (Vegan, Gluten-Free, او نور) |
| `allergens` | array | عام الرجنونه (مغز لرونکي، صدف لرونکي سمندري خواړه، او نور) |
| `availability` | string | کله شتون لري (ورځنی، موسمي، او نور) |

## د مېلمه پالنې مېنوګانو جوړول

### لومړی ګام: د مېنو معلومات برابر کړئ

Theme Builder ته د خپلې مېنو په اړه ووایاست:

```
زما د ایټالوي رستورانت لپاره یوه رستورانت مېنو جوړه کړه چې اشتهايي خواړه،
د پاستا خواړه، اصلي کورسونه، او خواږه ولري. بیې او توضیحات پکې شامل کړه.
```

### دویم ګام: Theme Builder مېنو جوړوي

Theme Builder:

1. یوه جوړښتي مېنو پاڼه جوړوي
2. دا ستاسو د سایټ له theme سره سم ډیزاینوي
3. توکي په کټګوریو کې تنظیموي
4. بیې او توضیحات بڼه کوي
5. د خوړو اړوند او الرجن معلومات زیاتوي

### درېیم ګام: بیاکتنه او دودیزول

تاسو کولی شئ:

1. د مېنو توکي او بیې سمې کړئ
2. کټګورۍ زیاتې یا لرې کړئ
3. په کټګوریو کې توکي بیا ترتیب کړئ
4. توضیحات او د خوړو اړوند معلومات تازه کړئ
5. سټایل او جوړښت عیار کړئ

## د مېنو ښودنې انتخابونه

### بشپړه مېنو پاڼه

یوه ځانګړې پاڼه چې ستاسو بشپړه مېنو ښيي:

- د کټګورۍ له مخې تنظیم شوې
- د لټون او فلټر وړ
- د چاپ لپاره مناسب جوړښت
- د موبایل لپاره ځواب ویونکی ډیزاین

### د مېنو وېجټ

د مېنو برخې په نورو پاڼو کې ځای پر ځای کړئ:

- په کورپاڼه کې غوره شوي توکي
- په څنګ پټه کې ورځني ځانګړي وړاندیزونه
- په بار پاڼه کې د څښاک مېنو
- په فوټر کې د خوږو ننداره

### د مېنو PDF

د ښکته کولو وړ PDF مېنو جوړه کړئ:

- مسلکي بڼه ورکول
- د چاپ لپاره چمتو کیفیت
- انځورونه او توضیحات پکې شامل دي
- د شریکولو او برېښنالیک کولو لپاره اسانه

## د خوړو اړوند او الرجن معلومات

### د خوړو اړوند ځانګړنې

توکي د خوړو اړوند معلوماتو سره نښه کړئ:

- **Vegan** — هېڅ حیواني محصولات نه لري
- **Vegetarian** — غوښه نه لري
- **Gluten-Free** — د سیلیاک ناروغۍ لپاره خوندي
- **Dairy-Free** — د شیدو محصولات نه لري
- **Nut-Free** — د ونو مغز یا ممپلي نه لري
- **Low-Carb** — کم شوي کاربوهایډرېټونه
- **High-Protein** — له پروټین څخه بډای

### د الرجن خبرداري

عام الرجنونه شامل کړئ:

- **مغز لرونکي** — د ونو مغز او ممپلي
- **صدف لرونکي سمندري خواړه** — کرستاسيان او مولسکونه
- **کبان** — د کبانو ټول ډولونه
- **لبنیات** — شیدې او د شیدو محصولات
- **هګۍ** — د چرګ هګۍ
- **سویا** — د سویا محصولات
- **ګلوټن** — غنم او ګلوټن لرونکي غلې
- **کنجد** — د کنجد دانې او تېل

## د مېنو مدیریت

### د بیو تازه کول

د مېنو بیې په اسانۍ سره تازه کړئ:

1. د مېنو پاڼې ته لاړ شئ
2. "مېنو سمول" کلیک کړئ
3. د توکو بیې تازه کړئ
4. بدلونونه خوندي کړئ
5. بدلونونه سمدستي ستاسو په سایټ کې ښکاري

### د موسمي توکو زیاتول

د موسمي مېنو بدلونونه جوړ کړئ:

1. د مېنو نوې نسخه جوړه کړئ
2. موسمي توکي زیات کړئ
3. توکي د "موسمي" په توګه نښه کړئ
4. د شتون نېټې مهالوېش کړئ
5. د موسم پر مهال یې په اوتومات ډول وښایئ

### د ځانګړو وړاندیزونو مدیریت

ورځني یا اوونیز ځانګړي وړاندیزونه وښایئ:

1. د "ځانګړي وړاندیزونه" کټګوري جوړه کړئ
2. د شتون نېټو سره توکي زیات کړئ
3. په کورپاڼه کې ځانګړي وړاندیزونه روښانه کړئ
4. هره ورځ یا هره اوونۍ یې تازه کړئ
5. زاړه ځانګړي وړاندیزونه ارشیف کړئ

## له Theme Builder سره ادغام

کله چې د مېلمه پالنې سایټونو لپاره Theme Builder کاروئ:

1. **اوتومات مېنو پېژندنه** — پېژني چې ایا تاسو د خوړو/څښاک کاروبار یاست
2. **د مېنو پاڼې جوړول** — مسلکي مېنو پاڼې جوړوي
3. **د ډیزاین سمون** — مېنوګانې ستاسو د سایټ له theme سره سمون خوري
4. **د موبایل ښه کول** — مېنوګانې په تلیفونونو کې ښکلي ښودل کېږي
5. **SEO ښه کول** — مېنوګانې د لټون-ماشین لپاره مناسبې دي

## غوره کړنلارې

### د مېنو ډیزاین

- **روښانه تنظیم** — منطقي کټګوري جوړښت
- **لوستلو وړ توضیحات** — اشتها راوړونکي او معلوماتي
- **یو شان بیه ورکول** — روښانه اسعار او بڼه ورکول
- **مسلکي عکسونه** — د خوړو لوړ کیفیت انځورونه
- **سپینه تشه** — پاڼه مه ډېره ګڼه ګوڼه کوئ

### منځپانګه

- **دقیق توضیحات** — توکي په دقت سره تشریح کړئ
- **ځانګړي وړاندیزونه روښانه کړئ** — ځانګړي وړاندیزونه څرګند کړئ
- **الرجنونه شامل کړئ** — تل الرجنونه لیست کړئ
- **په منظم ډول تازه کړئ** — بیې او توکي تازه وساتئ
- **اشتها راوړونکې ژبه وکاروئ** — توکي خوندور وښایئ

### لاسرسی

- **لوستلو وړ فونټونه** — روښانه، د لوستلو وړ لیکبڼې وکاروئ
- **کافي تضاد** — ډاډ ترلاسه کړئ چې متن لوستل کېدای شي
- **د خوړو اړوند نښې** — د خوړو اړوند انتخابونه په روښانه ډول نښه کړئ
- **د الرجن خبرداري** — الرجنونه په څرګند ډول وښایئ
- **د موبایل لپاره مناسب** — په ټولو وسیلو کې یې وازمویئ
