# د ډیزاین لوری {#design-direction}

د **ډیزاین لوری** ګام تاسو ته اجازه درکوي چې مخکې له دې چې Theme Builder بشپړ theme جوړ کړي، د خپل theme لیدیز ډیزاین وڅېړئ، سم یې کړئ، او وروستی یې کړئ.

## لنډیز {#overview}

د Discovery Interview له بشپړېدو وروسته، Theme Builder ستاسو د ډیزاین لوری د **desktop او mobile preview rendering** سره وړاندې کوي. دا تاسو ته اجازه درکوي چې د theme ته له ژمن کېدو مخکې په بېلابېلو وسیلو کې ووینئ چې ستاسو ډیزاین به دقیقاً څنګه ښکاري.

## د ډیزاین لوري برخې {#design-direction-components}

### 1. د رنګونو پالېټه {#1-color-palette}

ستاسو د theme د رنګونو طرحه، په شمول:

- **اصلي رنګ** — د نښې اصلي رنګ چې د تڼیو، لینکونو، او ټینګارونو لپاره کارېږي
- **دویم رنګ** — د تنوع لپاره بشپړوونکی رنګ
- **ټینګاري رنګ** — د مهمو عناصرو لپاره روښانوونکی رنګ
- **ناحیوي رنګونه** — د شالیدونو او متن لپاره خړ او سپین رنګونه
- **د متن رنګ** — د لوستلو وړتیا لپاره اصلي متن رنګ

### 2. ټایپوګرافي {#2-typography}

د فونټ ټاکنې د دې لپاره:

- **د سرلیک فونټ** — د پاڼې سرلیکونو او د برخو سرلیکونو لپاره کارېږي
- **د بدن فونټ** — د پاراګراف متن او منځپانګې لپاره کارېږي
- **د فونټ اندازې** — د بېلابېلو پردې اندازو لپاره ځواب ویونکې اندازه ټاکنه
- **د کرښې لوړوالی** — د لوستلو وړتیا لپاره د کرښو ترمنځ واټن
- **د فونټ وزنونه** — غټ، عادي، او سپک ډولونه

### 3. جوړښت او واټنونه {#3-layout-and-spacing}

- **د کانتېنر پلنوالی** — د منځپانګې لپاره اعظمي پلنوالی
- **دنننی او بهرنی واټنونه** — د عناصرو شاوخوا واټن
- **د ګرېډ سیستم** — د ځواب ویونکي ډیزاین لپاره د ستنو جوړښت
- **د برخو واټن** — د UI عناصرو ترمنځ فاصله

### 4. لیدیز عناصر {#4-visual-elements}

- **د تڼیو سټایلونه** — د اصلي، دویم، او درېیمې کچې تڼیو ډیزاینونه
- **د کارت ډیزاینونه** — د منځپانګې بلاکونو لپاره جوړښتونه
- **آیکونونه** — د آیکون سټایل او اندازه
- **چوکاټونه او سیوري** — نازک لیدیز ژوروالی
- **د hover اغېزې** — د متقابل حالت سټایل

## د preview rendering {#preview-rendering}

### Desktop Preview {#desktop-preview}

desktop preview ستاسو ډیزاین په بشپړ پلنوالی کې ښيي (معمولاً 1200px یا پراخ):

- بشپړ د تګ راتګ مینو
- بشپړ جوړښت له ټولو ستنو سره
- لویه ټایپوګرافي او واټن
- ټول لیدیز عناصر په بشپړه اندازه

### Mobile Preview {#mobile-preview}

mobile preview ستاسو ډیزاین د mobile په پلنوالی کې ښيي (معمولاً 375px):

- ځواب ویونکی تګ راتګ (hamburger menu)
- د یوې ستنې جوړښت
- سمون شوې ټایپوګرافي او واټن
- د لمس لپاره مناسبې د تڼیو اندازې

### ځواب ویونکي ماتېدونکي ټکي {#responsive-breakpoints}

Theme Builder په دې ماتېدونکو ټکو کې preview ګانې render کوي:

| وسیله | پلنوالی | Preview |
|--------|-------|---------|
| **Mobile** | 375px | عمودي تلیفون |
| **Tablet** | 768px | افقي ټابلیټ |
| **Desktop** | 1200px | بشپړ پلنوالی desktop |
| **Large Desktop** | 1920px | ډېر پراخ نندارتون |

## د خپل ډیزاین سمول {#refining-your-design}

### سمونونه کول {#making-adjustments}

تاسو کولی شئ خپل د ډیزاین لوری داسې سم کړئ:

1. **رنګونه سمول** — په پالېټه کې هر رنګ بدل کړئ
2. **فونټونه بدلول** — بېلابېل لیک ډولونه وټاکئ
3. **واټن بدلول** — دنننی او بهرنی واټن سم کړئ
4. **جوړښت تازه کول** — د کانتېنر پلنوالی او د ګرېډ ستنې بدل کړئ
5. **عناصر دودیزول** — د تڼیو سټایلونه، د کارت ډیزاینونه، او نور بدل کړئ.

### د Preview تازه کېدل {#preview-updates}

بدلونونه په رښتیني وخت کې څرګندېږي:

- Desktop preview سمدستي تازه کېږي
- Mobile preview سمدستي تازه کېږي
- ټول ځواب ویونکي ماتېدونکي ټکي تازه کېږي
- تاسو کولی شئ د بدلونونو د تایید لپاره د preview ګانو ترمنځ واوړئ

### د پرتله کولو لید {#comparison-view}

خپل د ډیزاین لوری له دې سره پرتله کړئ:

- **اصلي ډیزاین** — وګورئ څه بدل شوي
- **د سیالانو ډیزاینونه** — له الهامي site ګانو سره پرتله کړئ
- **پخوانۍ نسخې** — پخوانیو ډیزاین لوریو ته بېرته وګرځئ

## د ډیزاین لوري تایید {#design-direction-approval}

کله چې له خپل ډیزاین څخه راضي شئ:

1. **دواړه preview ګانې وڅېړئ** — ډاډ ترلاسه کړئ چې desktop او mobile ښه ښکاري
2. **ټول رنګونه وګورئ** — تضاد او لاسرسی ډاډمن کړئ
3. **ټایپوګرافي وازمویئ** — په ټولو اندازو کې د لوستلو وړتیا تایید کړئ
4. **جوړښت تایید کړئ** — واټن او سمون وګورئ
5. **ډیزاین تایید کړئ** — د theme جوړولو ته لاړ شئ

## راتلونکي ګامونه {#next-steps}

د خپل ډیزاین لوري له تایید وروسته:

1. Theme Builder ستاسو بشپړ theme جوړوي
2. theme ستاسو په WordPress site کې نصبېږي
3. تاسو کولی شئ د WordPress customizer په کارولو سره نور دودیز بدلونونه وکړئ
4. [Hospitality Menus](./hospitality-menus.md) یا نورو ځانګړنو ته لاړ شئ

## غوره کړنې {#best-practices}

- **په رښتینو وسیلو وازمویئ** — که ممکن وي، واقعي تلیفونونه او ټابلیټونه وکاروئ
- **د لوستلو وړتیا وګورئ** — ډاډ ترلاسه کړئ چې متن په ټولو اندازو کې لوستل کېدای شي
- **تضاد تایید کړئ** — د لاسرسي لپاره [Validate Palette Contrast](../abilities/validate-palette-contrast.md) وکاروئ
- **کارکردګي په پام کې ونیسئ** — انځورونه او فونټونه د چټکتیا لپاره اصلاح کړئ
- **د منځپانګې لپاره پلان جوړ کړئ** — ډاډ ترلاسه کړئ چې جوړښت ستاسو له واقعي منځپانګې سره کار کوي

## د ستونزو حل {#troubleshooting}

### Preview نه تازه کېږي {#preview-not-updating}

- پاڼه تازه کړئ
- د براوزر cache پاک کړئ
- بل براوزر وازمویئ
- د انټرنېټ نښلون وګورئ

### رنګونه بېلابېل ښکاري {#colors-look-different}

- د مانیټر د رنګونو امستنې وګورئ
- په بېلابېلو وسیلو کې یې وازمویئ
- د لاسرسي وسیلو سره د رنګ تضاد تایید کړئ
- د رنګ ړوندتوب شبیه کوونکي په پام کې ونیسئ

### د ټایپوګرافي ستونزې {#typography-issues}

- تایید کړئ چې د فونټ فایلونه load کېږي
- د بېلابېلو ماتېدونکو ټکو پر مهال د فونټ اندازه وګورئ
- له واقعي منځپانګې سره یې وازمویئ
- د لوستلو وړتیا لپاره د کرښې اوږدوالی په پام کې ونیسئ
