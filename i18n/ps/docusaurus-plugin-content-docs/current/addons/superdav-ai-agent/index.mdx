# Superdav AI Agent

Superdav AI Agent یو پرمختللی WordPress AI مرستیال دی چې Gratis AI Agent د theme-خبره مهارتونو، پیاوړو انټرنېټي لټون وړتیاوو، د سایټ جوړوونکي ځانګړتیاوو، او اختیاري اتومات‌سازۍ ادغامونو سره پراخوي. نسخه 1.18.0 د Google Calendar لوستلو وسیلې، د اړیکو او ګډونوالو نښلول، د انسان د تایید دروازې، د یادونې د تکرار مخنیوی، TextBee SMS خبرتیاوې، د مدیریت شوي Superdav خدمت برابرول، او د باوري پرمخ‌وړونکو کاري بهیرونو لپاره جلا وېشل شوې Advanced ملګرې کڅوړه زیاتوي.

## مهمې ځانګړتیاوې {#key-features}

- **Theme-خبره مهارتونه** — په اوتومات ډول ستاسو د فعال WordPress theme او نصب شوو plugins پر بنسټ لارښوونه او وړتیاوې عیاروي
- **څو انټرنېټي لټون چمتوکوونکي** — د لا بډایه لټون پایلو او پرمختللو څېړنیزو وړتیاوو لپاره د Brave Search او Tavily ترمنځ انتخاب وکړئ
- **د سایټ جوړوونکي د اړیکې فورمې** — د خپل نصب شوي form plugin په کارولو سره نېغ په نېغه د chat له انٹرفیس څخه بشپړې فعالې د اړیکې فورمې جوړې کړئ
- **پیاوړی سایټ جوړوونکی** — د plugins د ښه موندنې او د تېروتنې د بېرته راګرځېدو سره ښه شوي څو پړاوي سایټ جوړولو پلانونه
- **دننه جوړ theme مهارتونه** — د Block Themes، Classic Themes، Kadence Blocks، او Kadence Theme لپاره ځانګړې لارښوونه
- **ښه شوی WooCommerce ادغام** — د ښه باور وړتیا او سازګارۍ لپاره د اصلي WooCommerce APIs کارولو ته بیا جوړ شوی
- **اوتومات چمتوکوونکی پېژندنه** — شته plugins او themes په اوتومات ډول پېژني او تازه کوي کله چې فعال یا غیرفعال شي
- **دودیز لوګو تولید** — د اوتومات namespace-خوندي پاکولو سره دودیز SVG لوګوګان تولید او ځای‌پرځای کړئ
- **د رنګ‌لړ د تضاد تایید** — خپل theme ته تر پلي کولو مخکې د WCAG لاسرسي د مطابقت لپاره رنګ-جوړې وګورئ
- **د عکس-پیاوړې موندنه** — د Theme Builder د موندنې مرکې پر مهال عکسونه پورته کړئ څو د ډیزاین لوری اغېزمن کړي
- **د مېلمه‌پالنې مېنوګانې** — د رستورانتونو، کافې‌ګانو، او مېلمه‌پالنې کاروبارونو لپاره جوړښت لرونکې د خوړو او څښاک مېنو پاڼې تولید کړئ
- **د ډیسکتاپ او موبایل مخکتنې** — د ډیزاین-لوري د انتخاب پر مهال خپل ډیزاین په ډیسکتاپ او موبایل وسیلو کې وګورئ
- **د Tier 1 سایټ scraping** — sd-ai-agent/site-scrape اوس د سیالانو او الهام ویب‌سایټونو د شننې لپاره په تلواله توګه شته
- **Google Calendar د لوستلو وسیلې** — تنظیم شوي Calendarونه او پېښې ولولئ څو اتومات‌سازۍ د راتلونکي مهالویش د زمینه په اړه استدلال وکړي
- **د اړیکو او ګډونوالو نښلول** — د تعقیبي کار له جوړولو مخکې د calendar ګډونوال له پېژندل شوو سایټ کاروونکو او اړیکو سره برابر کړئ
- **د انسان د تایید دروازې** — حساسې اتومات‌سازۍ د پیغامونو تر لېږلو، د کار تر بیا پیلولو، یا بدلونونو کولو مخکې د بیاکتنې لپاره وځنډوئ
- **د یادونې ثبتونه او د تکرار مخنیوی** — هغه یادونې تعقیب کړئ چې مخکې لېږل شوې دي څو تکراري اتومات‌سازۍ هماغه ترلاسه کوونکي ته دوه ځله خبرتیا ورنه کړي
- **TextBee SMS خبرتیاوې** — کله چې د SMS چمتوکوونکی فعال وي، د TextBee له لارې تنظیم شوې د متن-پیغام خبرتیاوې ولېږئ
- **د مدیریت شوي Superdav خدمت برابرول** — ملاتړ شوي سایټونه د لاسي endpoint تنظیم پر ځای د اوتومات setup سره hosted Superdav endpoints ته ونښلوئ
- **Advanced ملګرې کڅوړه** — د فایل‌سیسټم، ډیټابېس، WP-CLI، REST dispatcher، plugin جوړوونکي، git snapshot، کارن-مدیریت، او benchmark کاري بهیرونو لپاره جلا وېشل شوي د باوري-پرمخ‌وړونکو وسیلې

## اړتیاوې {#requirements}

- WordPress 6.0 یا لوړ
- PHP 8.1 یا لوړ
- WordPress Multisite (network-activated) یا single-site نصب
- یو فعال AI provider connector plugin (Anthropic، OpenAI، یا سازګار)
- Gratis AI Agent v1.9.0 یا لوړ (Superdav AI Agent د Gratis AI Agent پراخوي)

## نصب {#installation}

1. د plugin فایلونه `/wp-content/plugins/superdav-ai-agent` ته پورته کړئ
2. plugin په WordPress کې د **Plugins** پردې له لارې فعال کړئ
3. ډاډ ترلاسه کړئ چې Gratis AI Agent هم فعال دی
4. **Gratis AI Agent → Settings** ته لاړ شئ او خپل انټرنېټي لټون چمتوکوونکی، د مدیریت شوي Superdav خدمت تړاو، Google Calendar، یا TextBee اسناد د اړتیا سره سم تنظیم کړئ

د WordPress.org کڅوړه د Superdav AI Agent اصلي ځانګړتیاوې لري. Advanced ملګرې کڅوړه د هغو چاپېریالونو لپاره جلا جوړیږي او وېشل کېږي چې پکې د باوري پرمخ‌وړونکو وسیلې په ښکاره ډول نصب او مجازې شوې وي.

## Theme-خبره مهارتونه {#theme-aware-skills}

Superdav AI Agent په اوتومات ډول ستاسو فعال theme او نصب شوي plugins پېژني، او مناسب theme-خبره مهارت د مرستیال context ته باروي. شته مهارتونه پکې شامل دي:

| مهارت | پر څه پلي کېږي |
|---|---|
| **Block Themes** | هغه themes چې د WordPress د block-based (Full Site Editing) جوړښت کاروي |
| **Classic Themes** | دودیز WordPress themes چې PHP templates او classic editor کاروي |
| **Kadence Blocks** | هغه سایټونه چې Kadence Blocks plugin پکې فعال وي |
| **Kadence Theme** | هغه سایټونه چې Kadence theme د فعال theme په توګه لري |

د هر مهارت په اړه تفصیلي لارښوونې لپاره [Theme-خبره مهارتونه](../../user-guide/configuration/theme-aware-skills) وګورئ.

## انټرنېټي لټون چمتوکوونکي {#internet-search-providers}

Superdav AI Agent v1.10.0 د انټرنېټي لټون پوښتنو لپاره د Brave Search د بدیل په توګه **Tavily** زیاتوي. دواړه چمتوکوونکي real-time لټون پایلې وړاندې کوي او په **Gratis AI Agent → Settings → Advanced** کې تنظیمېدای شي.

| برابرونکی | ځانګړتیاوې | تنظیم |
|---|---|---|
| **Brave Search** | چټکې، د محرمیت پر بنسټ د لټون پایلې | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | بډایه لټون پایلې، پرمختللې څېړنیزې وړتیاوې | [Tavily API](https://tavily.com/) |

د تنظیم لارښوونو لپاره [انټرنېټ لټون](../../user-guide/configuration/internet-search) وګورئ.

## د تقویم، تصویب، او خبرتیا کاري بهیرونه {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 د اتومات کاري بهیرونه د منځپانګې او سایټ جوړونې دندو هاخوا پراخوي:

- **Google Calendar وسایل** agent ته اجازه ورکوي چې تنظیم شوي تقویمونه او د پېښو جزئیات ولولي کله چې د تقویم اسناد موجود وي.
- **د اړیکو او ګډونوالو مرستندویان** د پېښو ګډونوال د WordPress کاروونکو یا پېژندل شویو اړیکو سره نښلوي مخکې له دې چې یادونې یا تعقیبي دندې چمتو کړي.
- **د انسان تصویب دروازې** اتومات ته اجازه ورکوي چې تر هغه ودرېږي څو یو مجاز کاروونکی وړاندیز شوی عمل بیاکتنه کړي.
- **د یادونې ثبتونه** د تقویم یا خبرتیا تکراري یادونې مخنیوی کوي کله چې اتومات بیا هڅه کېږي یا په تکراري ډول مهالویش کېږي.
- **TextBee SMS ملاتړ** د متن-پیغام خبرتیاوې فعالوي کله چې TextBee برابرونکی تنظیم شوی وي.

دا وسایل د مهالویش-خبرې مرستې، د ګډونوالو تعقیب، او د یادونې کاري بهیرونو لپاره وکاروئ چې لا هم د حساسو پیغامونو له لېږلو مخکې د انسان بیاکتنې ته اړتیا لري.

## تنظیم {#configuration}

ټول تنظیمات د WordPress اډمین کې د **Gratis AI Agent → Settings** لاندې دي. Superdav AI Agent لاندې د تنظیم اختیارونه زیاتوي:

| تنظیم | تشریح |
|---|---|

| **مدیریت شوی Superdav خدمت** | ملاتړ شوي سایټونه د اتومات endpoint برابرولو سره کوربه شوي Superdav خدمت ته ونښلوئ |

| **TextBee SMS برابرونکی** | د تصویب شویو SMS خبرتیا کاري بهیرونو لپاره د TextBee اسناد تنظیم کړئ |

## پرمختللی ملګری بسته {#advanced-companion-package}

Superdav AI Agent v1.18.0 د باوري پراختیاکوونکو کاري بهیرونو لپاره جلا وېشل کېدونکې پرمختللې ملګرې بسته زیاتوي. دا وړتیاوې د WordPress.org اصلي بسته کې نه دي یوځای شوې. ملګرې بسته یوازې پر هغو سایټونو نصب کړئ چې اډمینان په ښکاره ډول پر agent باور لري چې د پراختیاکوونکي وسایلو ته لاسرسی ولري.

پرمختللي وسایل د فایل‌سیسټم عملیات، ډیټابېس لاسرسی، WP-CLI، REST وېشونکي کاري بهیرونه، د plugin جوړونکي اسانتیاوې، git snapshots، د کاروونکي-مدیریت مرستندویان، او د بنچمارک وسایل پکې شامل دي. دا وسایل تر باوري اډمینانو محدود وساتئ او پر تولیدي سایټونو د فعالولو مخکې د تصویب-دروازې تنظیم بیاکتنه وکړئ.
