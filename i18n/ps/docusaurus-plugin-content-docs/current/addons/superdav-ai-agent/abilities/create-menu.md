# مېنو جوړول

د **مېنو جوړول** وړتیا د WordPress د تګ‌راتګ مېنوګانې جوړوي، د هغو جلا تګ‌راتګ لیبلونو په ملاتړ چې د پاڼې له سرلیکونو څخه بېل وي.

## لنډه کتنه

دا وړتیا د معیاري مېنو جوړولو کړنه د `navigation_label` پارامېټر د ټاکلو په وړتیا غځوي. دا تاسو ته اجازه درکوي داسې مېنوګانې جوړې کړئ چې په تګ‌راتګ کې ښودل کېدونکی لیبل د پاڼې له سرلیک څخه توپیر ولري، او د سایټ په جوړښت او د کارونکي په تجربه کې ډېر انعطاف برابروي.

## پارامېټرونه

| پارامېټر | ډول | اړین | تشریح |
|-----------|------|----------|-------------|
| `name` | string | هو | د مېنو نوم، لکه `Primary Navigation` |
| `location` | string | نه | د Theme ځای چې دا مېنو ورته وټاکل شي، لکه `primary` |
| `navigation_label` | string | نه | هغه لیبل چې په تګ‌راتګ کې ښودل کېږي (د پاڼې له سرلیک څخه جلا) |

## راستنېدونکی ارزښت

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## د تګ‌راتګ لیبل د پاڼې سرلیک پر وړاندې

د `navigation_label` پارامېټر تاسو ته اجازه درکوي چې د داخلي مېنو نوم له هغه لیبل څخه جلا کړئ چې کاروونکو ته ښودل کېږي:

- **`name`** — د داخلي مېنو پېژندونکی چې WordPress یې کاروي (لکه، "Primary Navigation")
- **`navigation_label`** — هغه لیبل چې د سایټ لیدونکو ته په تګ‌راتګ کې ښودل کېږي (لکه، "Main Menu")

دا هغه وخت ګټور دی کله چې:
- ستاسو د داخلي نومونې دود د کاروونکو پر مخ له ښودل کېدونکو لیبلونو سره توپیر ولري
- تاسو په تګ‌راتګ کې د اډمین له تختې څخه لنډ لیبلونه غواړئ
- تاسو اړتیا لرئ څو ژبې د بېلابېلو لیبل اوږدوالیو سره ملاتړ کړئ
- تاسو د ځانګړو سیمو یا کاروونکو ډلو لپاره مېنوګانې جوړوئ

## د کارونې بېلګې

### بېلګه ۱: ساده مېنو د تګ‌راتګ لیبل سره

**غوښتنه:**
```
د "Main Navigation" په نوم یوه اصلي تګ‌راتګ مېنو د "Menu" د تګ‌راتګ لیبل سره جوړه کړئ.
```

**پایله:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### بېلګه ۲: د ځانګړي Theme ځای لپاره مېنو

**غوښتنه:**
```
د "Footer Links" په نوم یوه فوټر مېنو د "Quick Links" تګ‌راتګ لیبل سره جوړه کړئ او د فوټر ځای ته یې وټاکئ.
```

**پایله:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## له Theme Builder سره یوځای کول

کله چې Theme Builder کاروئ، د مېنو جوړولو وړتیا:

1. په اتومات ډول د Theme شته مېنو ځایونه پېژني
2. ستاسو د ډیزاین لپاره له مناسبو تګ‌راتګ لیبلونو سره مېنوګانې جوړوي
3. مېنوګانې سمو Theme ځایونو ته ټاکي
4. له جوړېدو وروسته د مېنو توکو زیاتولو ملاتړ کوي

## اړوندې وړتیاوې

- **`add_menu_item`** — موجودې مېنو ته توکي زیات کړئ
- **`update_menu`** — مېنو ته نوی نوم ورکړئ یا یې د Theme ځای ته بیا وټاکئ
- **`delete_menu`** — له خپل سایټ څخه مېنو لرې کړئ

## غوره کړنې

- **روښانه تګ‌راتګ لیبلونه وکاروئ** — لیبلونه د کاروونکو لپاره لنډ او تشریحي وساتئ
- **د Theme ځایونو سره سمون ورکړئ** — د سم ښودلو لپاره مېنوګانې سم Theme ځای ته وټاکئ
- **د مېنو جوړښت پلان کړئ** — د توکو له جوړولو مخکې د خپلې مېنو سلسله وټاکئ
- **ځواب‌ویونکې بڼه وازمویئ** — ډاډ ترلاسه کړئ چې مېنوګانې په ګرځنده وسیلو کې سمې ښودل کېږي
- **لیبلونه ځایي کړئ** — د بېلابېلو ژبنیو نسخو لپاره بېلابېل تګ‌راتګ لیبلونه وکاروئ

## محدودیتونه

- د تګ‌راتګ لیبلونه یوازې د ښودلو لپاره دي؛ داخلي `name` لا هم د WordPress د پېژندنې لپاره کارول کېږي
- د Theme ملاتړ توپیر لري؛ ټول Themeګانې د مېنو ټول ځایونه نه ملاتړ کوي
- د مېنو توکي باید د مېنو له جوړېدو وروسته جلا زیات شي
- د تګ‌راتګ لیبل بدلول د مېنو تازه کولو ته اړتیا لري
