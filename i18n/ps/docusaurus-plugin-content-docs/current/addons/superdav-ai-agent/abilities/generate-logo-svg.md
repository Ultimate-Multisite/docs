# د لوګو SVG جوړول {#generate-logo-svg}

د **لوګو SVG جوړول** وړتیا Theme Builder ته اجازه ورکوي چې دودیز لوګو SVGs په مستقیم ډول ستاسو WordPress وېبپاڼه کې جوړ او ځای پر ځای کړي، د اتومات نوم‌فضا-خوندي پاکونې سره.

## ټولیزه کتنه {#overview}

دا وړتیا ستاسو د وېبپاڼې د نښې‌جوړونې لوري او ډیزاین غوره‌توبونو پر بنسټ د پراخېدو وړ وېکټوري ګرافیک (SVG) لوګوګانې جوړوي. جوړ شوي SVGs په اتومات ډول پاکېږي څو ډاډ ترلاسه شي چې په WordPress کې د کارولو لپاره خوندي دي، په داسې حال کې چې لیدیز بشپړتیا ساتي.

## پارامترونه {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | Yes | د هغې وېبپاڼې نوم چې لوګو ورته جوړېږي |
| `style` | string | Yes | د ډیزاین سبک (لکه، "عصري"، "کلاسیک"، "لږینه"، "لوبیز") |
| `colors` | array | No | د hex رنګ کوډونو لړۍ چې په لوګو کې به وکارول شي (لکه، `["#678233", "#ffffff"]`) |
| `width` | number | No | د SVG پلنوالی په پیکسلونو کې (اصلي: 200) |
| `height` | number | No | د SVG لوړوالی په پیکسلونو کې (اصلي: 200) |
| `include_text` | boolean | No | آیا د وېبپاڼې نوم د متن په توګه په لوګو کې شامل شي که نه (اصلي: true) |

## د وتۍ بڼه {#output-format}

دا وړتیا د لاندې جوړښت لرونکی SVG تار بېرته ورکوي:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## د SVG پاکونې چلند {#svg-sanitisation-behaviour}

جوړ شوي SVGs د اتومات نوم‌فضا-خوندي پاکونې له بهیر څخه تېرېږي تر څو:

- **ناخوندي ځانګړتیاوې لرې کړي** — د پېښو سمبالوونکي، سکرېپټونه، او احتمالي خطرناکې ځانګړتیاوې لرې کوي
- **نوم‌فضاوې وساتي** — د سم ښودلو لپاره د SVG نوم‌فضاوې (xmlns, xlink) ساتي
- **جوړښت تایید کړي** — ډاډ ورکوي چې SVG د W3C معیارونو سره سمون لري
- **موجودیتونه کوډ کړي** — په متني منځپانګه کې ځانګړي کرکټرونه په سمه توګه خوندي کوي
- **بهرنۍ حوالې لرې کړي** — بهرنۍ stylesheets او د انځور حوالې لرې کوي

دا ډاډ ورکوي چې SVG د اضافي پاکونې اړتیا پرته په مستقیم ډول په WordPress کې د ځای پر ځای کولو لپاره خوندي دی.

## د کارولو بېلګه {#usage-example}

**غوښتنه:**
```
زما د "CloudSync" په نوم ټېکنالوژۍ نوې کمپنۍ لپاره د آبي او سپین رنګونو په کارولو یو عصري لوګو جوړ کړه.
```

**پایله:**
دا وړتیا داسې SVG لوګو جوړوي چې:
- د "CloudSync" وېبپاڼې نوم پکې شامل وي
- ټاکل شوی آبي او سپین رنګیز سکیم کاروي
- د عصري ډیزاین اصول تعقیبوي
- په اتومات ډول پاک شوی وي او کارولو ته چمتو وي

## له Theme Builder سره یوځای‌کول {#integration-with-theme-builder}

کله چې د Theme Builder د ډیزاین-لوري ټاکنه کاروئ، د لوګو SVG جوړولو وړتیا:

1. ستاسو د ډیزاین لوری او رنګیزه پالېټه شننه کوي
2. ستاسو له غوره‌توبونو سره سم دودیز SVG لوګو جوړوي
3. لوګو په اتومات ډول ستاسو د وېبپاڼې د سرلیک/نښې‌جوړونې برخه کې ځای پر ځای کوي
4. SVG په WordPress رسنیو کې د دودیز لوګو په توګه زېرمه کوي

## غوره کړنې {#best-practices}

- **روښانه د سبک غوره‌توبونه ورکړئ** — هغه ډیزاین سبک تشریح کړئ چې غواړئ (عصري، کلاسیک، لوبیز، او نور)
- **رنګونه مشخص کړئ** — د یووالي لپاره د خپلې نښې رنګونه شامل کړئ
- **ښودنه وازمویئ** — تایید کړئ چې لوګو په بېلابېلو پردې اندازو کې سم ښکاري
- **نور یې دودیز کړئ** — د اندازې او ځای پر ځای کولو د سمون لپاره د WordPress د لوګو دودیزولو اوزار وکاروئ

## محدودیتونه {#limitations}

- SVG لوګوګانې د ثابتو ګرافیکونو په توګه جوړېږي (خوځنده نه دي)
- پېچلې لوګوګانې چې ډېر عناصر لري ښايي لاسي سمون ته اړتیا ولري
- دودیز فونټونه نه ملاتړ کېږي؛ متن د سیسټم فونټونه کاروي
- ډېرې لویې یا ډېرې کوچنۍ اندازې ښايي پر کیفیت اغېز وکړي
