# بدلون‌لیک

## 1.18.0 — په 2026-06-29 خپور شو {#1180--released-on-2026-06-29}

### نوي {#new}

- **Google Calendar وسیلې** — د مهال‌وېش-خبره اتومات‌سازیانو لپاره تنظیم شوي کيلېنډرونه او پېښې ولولئ
- **د اړیکو نقشه‌کول او د ګډونوالو مرستندویان** — د کيلېنډر ګډونوال د وېبپاڼې کاروونکو او اړیکو سره سمونوي
- **د انساني تایید دروازې او د یادونې ثبتونه** — اتومات‌سازیانې د بیاکتنې لپاره دروي او له تکراري خبرتیاوو مخنیوی کوي
- **TextBee SMS برابرونکی** — د TextBee له لارې تنظیم شوې متني-پیغام خبرتیاوې لېږي
- **پرمختللې ملګرې بسته** — د باور وړ-پراختیاکوونکي د فایل‌سیسټم، ډیټابېس، WP-CLI، REST وېشونکي، plugin جوړوونکي، git snapshot، د کارن-مدیریت، او بنچمارک وسیلې زیاتوي چې د WordPress.org جوړښت څخه جلا وېشل کېږي

### ښه شوي {#improved}

- **د مدیریت شوي Superdav خدمت جوړونه** — د ملاتړ شوو وېبپاڼو لپاره کوربه شوي خدمت endpointونه او اتومات نښلون چمتو کول زیاتوي
- **د خپرونې بسته‌بندي** — جلا اصلي او Advanced ZIPونه جوړوي، دواړه پر GitHub خپروي، او یوازې اصلي بسته WordPress.org ته لېږي

### سم شوي {#fixed}

- **د AI غوښتنې باورمنتیا** — د ماډل ټاکنه، د غوښتنې وخت‌پایونه، د جوړونې تلواله ماډل، د استدلال-متن اداره، او د ناسمو وسیلې غږونو لپاره د بیا هڅې لارښوونه ښه کوي
- **د کيلېنډر او یادونې ټینګښت** — د Google Calendar tokenونه او د یادونې د تکرار مخنیوی پیاوړی کوي
- **پېلونه او د تایید بیاپیلونه** — د مخکني اړخ پېلونه او تایید شوي د وړتیا بیاپیلونه سموي
- **د WordPress.org بسته‌بندۍ بیاکتنې ستونزې** — د اصلي خپرونې لپاره د بسته‌بندۍ د بیاکتنې نظرونه حل کوي

## 1.16.0 — په 2026-05-20 خپور شو {#1160--released-on-2026-05-20}

### نوي {#new-1}

- **د Logo SVG جوړولو وړتیا** — Theme Builder اوس کولی شي د نوم‌فضا-خوندي پاکونې سره دودیز logo SVGونه جوړ او ځای‌پرځای کړي
- **د موندنې مرکه کې د انځور پورته کول** — د Theme Builder د موندنې مرکه اوس د لا بډای ډیزاین سیاق لپاره د انځور پورته کولو ګام لري
- **د رنګ‌لړ تضاد تاییدولو وړتیا** — پر theme له پلي کولو مخکې د WCAG د مطابقت لپاره د رنګ جوړې ګوري
- **د مېلمه‌پالنې مینوګانې** — Theme Builder اوس د مېلمه‌پالنې کاروبارونو لپاره جوړښتي د خوړو او څښاکو د مینو پاڼې جوړولی شي
- **د ډېسکټاپ او موبایل مخکتنې وړاندې کول** — د ډیزاین-لوري ټاکنې پر مهال خپل ډیزاین پر ډېسکټاپ او موبایل وسیلو وګورئ
- **د ګرځښت لېبل پارامیټر** — د Menu جوړولو وړتیا اوس د پاڼې له سرلیک څخه جلا ځانګړی `navigation_label` ملاتړ کوي
- **د ټیر ۱ وسیلې شتون** — sd-ai-agent/site-scrape اوس د ټیر ۱ وسیله ده چې په Theme Builder کې په تلواله ډول شته

### سم شوي {#fixed-1}

- **د AI Client زېرمه** — اوس د transients له خوا د غوښتنو ترمنځ دوام لپاره ملاتړ کېږي، چې د اوږدمهالو agent دندو کې د معلوماتو له لاسه ورکولو مخه نیسي
- **د plugin قطار کړنې لینکونه** — د روښانتیا لپاره سم او بیا نومول شوي

## 1.10.0 — په 2026-05-05 خپور شو {#1100--released-on-2026-05-05}

### نوي {#new-2}

- **Tavily انټرنېټ لټون** — د Brave Search ترڅنګ د لا بډایو انټرنېټ لټون پایلو لپاره Tavily د لټون برابرونکي په توګه زیاتوي
- **د theme-خبره جوړې دننه مهارتونه** — د Block Themes، Classic Themes، Kadence Blocks، او Kadence Theme د مهارت لارښودونه اوس له plugin سره راځي
- **د وېبپاڼې جوړوونکي د اړیکې فورم وړتیا** — له چټ انترفېس څخه نېغ هرې پاڼې ته د اړیکې فورم زیاتوي

### ښه شوي {#improved-1}

- **WooCommerce ادغام بیا جوړ شو** — اوس د غوره باورمنتیا او سازګارۍ لپاره اصلي WooCommerce APIګانې کاروي
- **د برابرونکو لېست په اتومات ډول تازه کېږي** — کله چې هر plugin فعال یا غیرفعال شي

### سم شوي {#fixed-2}

- **navigate-to وړتیا** — په ځینو admin پاڼو کې د بې‌پایه بیا پورته کولو کړۍ سمه شوه
- **list-posts وړتیا** — اوس د کټګورۍ او ټګ نومونه په سمه توګه slugs ته حل کوي
- **WP-CLI قوماندې** — له پخوانۍ بیا جوړونې وروسته ورکې namespace aliasونه بېرته راوستل شول
- **د پېښې اتومات‌سازي** — هغه وېبپاڼې په نرمۍ اداره کوي چې د اتومات‌سازۍ جدولونه یې لا نه دي جوړ شوي
- **memory-save وړتیا** — اوس د سیسټم د لارښوونې جوړوونکي کې سم namespace prefix کاروي
- **Scalar وسیلې پایلې** — اوس AI ته له ستنېدو مخکې په سمه توګه پوښل کېږي
- **د کارونې شمېرې** — اوس له زړو نسخو څخه د لوړونې پر مهال د پخواني وړتیا key بڼه په سمه توګه اداره کوي
