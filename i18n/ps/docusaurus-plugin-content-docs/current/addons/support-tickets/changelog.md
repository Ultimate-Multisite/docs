* سمون: ناتعریف شوي add_meta() غږونې د metadata زېرمه کولو لپاره په سم update_meta() بدلې شوې
* سمون: د ناثبت شوي وړتیا چک د سم wu_view_all_support_tickets سره بدل شو
* سمون: د ټکټ حالت، سپارنې، او چټک-سمون کړنو لپاره ورک AJAX handlers ورزیات شول
* سمون: د ټکټ لیدونو کې د کارکوونکي ځواب پېژندنې لپاره د مېتود نوم سم شو
* سمون: تکراري ځواب handlers یوځای شول او د nonce کړنې نومونه همغږي شول
* سمون: د [wu_submit_ticket] shortcode لپاره ورک مخکینۍ لید ورزیات شو
* سمون: ورکه user_id ستن ورزیاته شوه او د Support_Ticket::get_user_id() مېتود سم شو
* سمون: د لومړیتوب filter option tag کې اضافي دوه‌ګونی کوتیشن لرې شو
* سمون: د super admins لپاره د شبکې د مدیر ټکټ مدیریت panel ورزیات شو
* ښه شوی: د مدیر CSS په یوه واحده بهرنۍ stylesheet کې یوځای شو
* ښه شوی: د فرعي سایټ د مدیر مینو څخه Settings فرعي مینو لرې شوه
* ښه شوی: د مخکینۍ برخې شتمنۍ یوازې د ملاتړ ټکټ پاڼو کې په مشروط ډول پورته کېږي
* ښه شوی: د plugin autoloader پرېښودل کېږي کله چې د Bedrock root autoloader لا له مخکې dependencies پورته کړي وي
