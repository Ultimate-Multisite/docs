# Material WP

د WordPress اداري انترفېس له ډېرې مودې راهیسې هماغسې پاتې شوی. خو ایا کله مو فکر کړی چې د اداري انترفېس د Material Design نسخه به څنګه ښکاري؟ 

Material WP یو اداري قالب دی چې تاسو ته هماغه بڼه او احساس درکړي، او مرسته کوي چې خپلو پېرېدونکو ته یو لا زیات دودیز شوی Dashboard وړاندې کړئ.

## ځانګړتیاوې

- د دودیز CSS ملاتړ
- اسانه د سټایل تنظیمات
- د مېنو بیا تنظیمول
- اسانه واردول/صادرول

## اسناد

**د Material WP په وسیله خپل اداري پینل دودیزول**

## نصب

1. 'material-wp' د '/wp-content/plugins/' ډایرېکټرۍ ته پورته کړئ
2. لګون په WordPress کې د 'لګونونو' له مېنو څخه فعال کړئ
