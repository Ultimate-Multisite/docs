# د ملاتړ استازي

په خپله د یوې بشپړې شبکې اداره کول اسانه کار نه دی. کله چې ستاسو شبکه وده پیل کړي، تاسو به د ملاتړ لپاره زیاتېدونکې غوښتنه ووینئ. دا به ښه نظر وي چې داسې کسان راولئ چې ستاسو د کار بار کمولو کې مرسته وکړي. خو بشپړ لاسرسی/اجازه ورکول خطرناک دي او ښايي ستاسو شبکه له ګواښ سره مخ کړي. همدلته د ملاتړ استازي دا add-on مرسته کوي.

دا add-on تاسو ته اجازه درکوي چې د WordPress کارن یو نوی ډول جوړ کړئ چې د ملاتړ استازی بلل کېږي، او تاسو ته پر دې بشپړ کنټرول درکوي چې دغه کارن کومې اجازې لري. اجازه ورکول د یوه toggle تڼۍ د فعالولو په څېر اسانه دي، او که یې لرې کول وغواړئ، بېرته یې بندول هم همدومره اسانه دي.

## ځانګړتیاوې {#features}

- د WordPress او Ultimate Multisite اجازو تفصیلي لړلیک
- لاسرسی په اسانۍ سره فعال او غیرفعال کړئ
- د استازو نامحدود شمېر Accountونه جوړ کړئ

## نصب {#installation}

1. 'ultimate-multisite-support-agents' د '/wp-content/plugins/' ډایرکټرۍ ته پورته کړئ
2. plugin په WordPress کې د 'پلگینونو' له مېنو څخه فعال کړئ
3. د plugin د تنظیم لپاره ګام په ګام Wizard تعقیب کړئ
