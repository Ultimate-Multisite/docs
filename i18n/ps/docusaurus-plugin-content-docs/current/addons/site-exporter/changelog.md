# د Site Exporter د بدلونونو لړلیک {#site-exporter-changelog}

Version: 1.0.6 - په 2026-05-11 خپور شو
* نوی: د site صادرات اوس یو ځان-پیلېدونکی index.php ورسره تړي، نو ZIP پر نوي کوربه د جلا plugin نصب پرته نصبېدای شي
* نوی: د Network صادرول مدیرانو ته اجازه ورکوي چې ټول فرعي siteونه د Site Export د admin پاڼې څخه په یوه واحد ارشیف کې صادر کړي

Version: 1.0.5 - په 2026-05-05 خپور شو
* سمونه: د WP_CLI polyfill stub ته __callStatic ورزیات شو څو نه-mock شوې مېتودونه لکه get_config اداره کړي، او د ازموینې د bootstrap ناکامۍ مخه ونیسي

Version: 1.0.4 - په 2025-11-25 خپور شو

* له Ultimate Multisite 2.4 سره د سازګارۍ سمونه.

Version: 1.0.2 - په 2025-09-28 خپور شو

* prefix ultimate-multisite ته ونوموئ؛ text domain تازه کړئ؛ version لوړونه.

### Version 1.0.1 - په 2023-08-09 خپور شو {#version-101---released-on-2023-08-09}

* ښه شوی: اوس ډاډمنوي چې د importer فایل سم فایل مسیر ترلاسه شي.
* سم شوی: object cache غیر فعال شو څو د واردولو پر مهال د تېروتنو مخه ونیول شي.
* سم شوی: د importer plugin د ښکته کولو لینک د نه کار کولو ستونزه حل شوه.

### Version 1.0.0 - په 2022-12-23 خپور شو {#version-100---released-on-2022-12-23}

* سم شوی: د صادرولو فورم popup نه بارېده؛
* سم شوی: Javascript فایلونه د صادرولو په پاڼه کې نه بارېدل؛
* داخلي: نوی جوړولو بهیر؛

### Version 1.0.0-beta.4 - په 2022-01-13 خپور شو {#version-100-beta4---released-on-2022-01-13}

* داخلي: د hooks او filters جوړوونکی ورزیات شو؛
* داخلي: د پراختیاکوونکي د ژوند د کیفیت لپاره WP Ultimo stubs ورزیات شول؛
