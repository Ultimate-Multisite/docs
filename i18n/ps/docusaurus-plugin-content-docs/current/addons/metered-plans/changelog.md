# د Metered Plans بدلون‌لیک

نسخه 1.1.0 - په 2026-05-05 خپره شوه
- نوی: د څو سایټي فرعي سایټونو لپاره د AI ټوکن بیل کول — د پیرودونکو په سایټونو کې د AI ټوکن کارونه د تنظیمېدونکو هر-ټوکن نرخونو سره وڅارئ او بیل یې کړئ
- نوی: د نښلونکي پلي کول د خوځنده حد موندنې او لیکنې-له-لارې سره بیا جوړ شول، چې په ټولو نښلونکو کې د ریښتیني وخت کره‌والی ډاډمنوي
- سمونه: د Database جدول لوړونې اوس د BerlinDB سکیمې تعریفونو سره سمې برابرې شوې، چې په نویو نصبونو کې د لوړونې ناکامۍ مخنیوی کوي
- سمونه: د Database لوړونې callbackونه سمې بڼې ته واړول شول، چې پټې لوړونې ناکامۍ حل کوي
- سمونه: جزئي ارزښتونه اوس د AI Usage Overage Markup دننه کولو ډګر کې منل کېږي
- سمونه: د plugin د پورته کېدو پر مهال وژونکې تېروتنې او د دوه ځله پیل کېدو ستونزې حل شوې
- ښه شوی: د check-env npm script زیات شو، څو د پراختیاکوونکي چاپېریالونه په لومړي ځلې چلولو کې ځان تنظیم کړي
