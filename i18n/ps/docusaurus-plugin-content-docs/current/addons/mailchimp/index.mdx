# Mailchimp {#mailchimp}

Multisite Ultimate: د Mailchimp یوځای کول

خپله Multisite Ultimate شبکه له خپل Mailchimp سره یوځای کړئ. تاسو کولی شئ ځانګړي پلانونه د Mailchimp لړلیکونو سره ونښلوئ او د نوم‌لیکنې پر مهال ګډونوال په اوتومات ډول هغو لړلیکونو ته ورزیات کړئ.

## لګول {#installation}

1. 'ultimate-multisite-mailchimp' د '/wp-content/plugins/' ډایرکټرۍ ته پورته کړئ
2. پلگین د خپل WordPress Network Admin کې د 'پلگینونه' مینو له لارې فعال کړئ

## د کارولو لارښود {#usage-guide}

ستاسو د MailChimp لړلیکونو سره د ګډونوالو د برېښنالیکونو همغږي کول زموږ د Ultimate Multisite: MailChimp Integration اضافه له لارې ډېر اسانه دي. دا لارښود موخه لري چې له تاسو سره د هر څه په برابرولو کې مرسته وکړي!

## د API کیلي ترلاسه کول {#getting-an-api-key}

مخکې له دې چې موجودو بېلابېلو امستنو ته لاړ شو، موږ د MailChimp API کیلي ته اړتیا لرو. دا به Ultimate Multisite ته اجازه ورکړي چې د MailChimp API سره خبرې وکړي، څو لړلیکونه او ګروپونه ترلاسه کړي، او کله چې کاروونکي نوم‌لیکنه کوي، د هغوی برېښنالیک پتې هغو لړلیکونو او ګروپونو ته ورزیاتې کړي.

د خپلې MailChimp API کیلي ترلاسه کولو لپاره دا لارښود تعقیب کړئ: [د API کیليو په اړه | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

_خپله MailChimp API کیلي دلته زیاته کړئ_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## د یوځای کولو حالت {#integration-mode}

د MailChimp یوځای کولو اضافه دوه “د یوځای کولو حالتونه” وړاندې کوي، یعنې دوه بېلابېلې لارې شته چې تاسو یې د برېښنالیک پتې همغږي کولو لپاره له دې اضافې سره کارولی شئ.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_د زیات انعطاف لپاره د یوځای کولو دوه حالتونه!_

### حالت ۱: ګڼ لړلیکونه {#mode-1-multiple-lists}

تلواله حالت له **ګڼو لړلیکونو** څخه کار اخلي. دا حالت به تاسو ته اجازه درکړي چې د هر پلان لپاره څو MailChimp لړلیکونه وټاکئ. دا تر ټولو ښه انتخاب دی که تاسو خپل کاروونکي په بېلابېلو لړلیکونو کې وېشئ.

د بېلګې په توګه، راځئ فرض کړو چې تاسو په خپل MailChimp Account کې ۳ بېلابېل لړلیکونه لرئ: **خبرپاڼه** ، **پلان الف** ، او **پلان ب**. تاسو غواړئ ټول کاروونکي، د هغوی له پلان پرته، د خبرپاڼې په لړلیک کې هم شامل وي، څو ستاسو تازه معلومات له لاسه ورنه کړي.

د دې د ترلاسه کولو لپاره، تاسو کولی شئ د ګڼو لړلیکونو حالت وکاروئ، څو دواړه لړلیکونه **خبرپاڼه** او **پلان الف** د پلان الف MailChimp امستنو ته ورزیات کړئ، او **خبرپاڼه** او **پلان ب** د پلان ب MailChimp امستنو ته ورزیات کړئ.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_د نویو Accountونو او لغوه شوو Accountونو لپاره تلواله لړلیکونه وټاکئ_

### حالت ۲: یو لړلیک + ګڼ ګروپونه {#mode-2-single-list--multiple-groups}

MailChimp تاسو ته اجازه درکوي چې خپل لړلیک د کاروونکو په کوچنیو ګروپونو ووېشئ چې **ګروپونه** بلل کېږي. دا تاسو ته زیات کنټرول درکوي، ځکه تاسو کولی شئ د خپلو ګډونوالو یوې ځانګړې فرعي ډلې ته کمپاین واستوئ، بې له دې چې په MailChimp کې د هماغه کمپاین ګڼې کاپۍ جوړې کړئ (چې وروسته د هغه کمپاین د مېټریکونو شننه ستونزمنوي).

د بېلګې په توګه، تاسو کولی شئ یو واحد لړلیک ولرئ چې **زما د شبکې ګډونوال،** نومېږي، او دننه پکې د **پلان الف** او **پلان ب** لپاره ګروپونه ولرئ.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_تلواله لړلیکونه او ګروپونه وټاکئ، که د “یو لړلیک + ګڼ ګروپونه” د یوځای کولو حالت کاروئ_

## تلواله لړلیکونه او ګروپونه {#default-lists-and-groups}

وروسته له دې چې تاسو وټاکئ کوم د یوځای کولو حالت ستاسو د ځانګړي کارونې مورد ته ډېر مناسب دی، وخت دی چې تلواله لړلیکونه (او ګروپونه، د حالت له مخې) وټاکئ. تلواله لړلیک به هغه وخت وکارول شي چې ورکړل شوی پلان کوم ځانګړی ټاکل شوی لړلیک ونه لري.

تاسو کولی شئ د لغوه شوو Accountونو لپاره هم تلواله لړلیک وټاکئ. هر کله چې یو کاروونکی خپل ګډون پای ته رسوي او خپل Account ستاسو له شبکې څخه لرې کوي، د هغه یا هغې برېښنالیک پته به د لغوه شوو Accountونو لپاره ټاکل شوي لړلیک ته ولېږدول شي.

## د دوه‌ګوني ګډون فعالول {#enabling-double-subscription}

په هماغه امستنو پاڼه کې، تاسو به وکولی شئ **دوه‌ګونی ګډون** فعال کړئ. دا غوراوی به کاروونکی هم ټاکل شوي تلواله لړلیک ته او هم د هغه کاروونکي د پلان لپاره ټاکل شوي لړلیک ته ورزیات کړي. که دا غوراوی فعال نه وي، کاروونکی به یوازې د کاروونکي د پلان لپاره ټاکل شوي لړلیک ته ورزیات شي. که د هغه ځانګړي پلان لپاره هېڅ نه وي ټاکل شوي، نو کاروونکی به ټاکل شوي تلواله لړلیک ته ورزیات شي.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_تاسو کولی شئ خپل کاروونکي د نوم‌لیکنې پر مهال هم تلواله لړلیک او هم د پلان لړلیک ته ګډون کړي_

## د هر پلان لپاره د لړلیکونو او ګروپونو ټاکل {#selecting-lists-and-groups-for-each-of-the-plans}

خپلو **پلانونو** ته لاړ شئ او هغه پلان وټاکئ چې سمول یې غواړئ. یادونه وکړئ چې د **پرمختللي غوراوي** مېټا-بکس کې به د MailChimp په نوم نوی ټب موجود وي.

په هغه ټب کې، تاسو به وکولی شئ د هغه پلان لپاره ګڼ لړلیکونه یا یو واحد لړلیک او ګڼ ګروپونه (د یوځای کولو حالت له مخې) وټاکئ.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_هغه لړلیکونه او ګروپونه وټاکئ چې غواړئ د خپلو هر یوه پلانونو لپاره یې وکاروئ_

کله چې یو نوی کاروونکی د هغه پلان لپاره نوم‌لیکنه وکړي، د هغه برېښنالیک پته به ټاکل شوو لړلیکونو او ګروپونو ته ورزیاته شي!
