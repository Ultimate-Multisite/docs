# د منځپانګې همغږي {#content-sync}

Multisite Ultimate: د منځپانګې همغږي

د Multisite Ultimate لپاره دا اضافه د شبکې مدیرانو ته اجازه ورکوي چې له هرې وېبپاڼې څخه نورو وېبپاڼو ته په شبکه کې منځپانګه همغږې کړي. دا د ټولو ليکنو ډولونو ملاتړ کوي، چې پکې ليکنې، پاڼې، دودیز د ليکنې ډولونه، او Elementor کينډۍ شاملې دي. په دې اضافه کې د منځپانګې او هدف وېبپاڼو د ټاکلو لپاره څو-پړاوی وزرډ انٹرفېس شامل دی، له ډله‌ييز پروسس او د پرمختګ ريښتيني-وخت غبرګون سره.

## ځانګړنې {#features}

* د منځپانګې ټاکنه: د تمبنېلونو او د ليکنې ډول چاڼګرۍ سره بصري کارت جال
* د ليکنې ټول ډولونه: ليکنې، پاڼې، محصولات، Elementor کينډۍ، او هر دودیز د ليکنې ډول همغږی کوي
* د وېبپاڼې ټاکنه: وټاکئ چې منځپانګه کومو وېبپاڼو ته همغږې شي، د سرچينې وېبپاڼې له اېستلو او د فرعي وېبپاڼې له مخکې ټاکنې سره
* ډله‌ييز پروسس: په يو وخت کې تر ۱۰ وېبپاڼو پورې د پرمختګ له تازه‌معلوماتو سره همغږی کوي
* د رسنيو همغږي: په اوتومات ډول د ليکنې له منځپانګې څخه هدف وېبپاڼو ته د رسنيو دوتنې (انځورونه، ويډيوګانې) کاپي او بيا نښلوي
* د ټکسونومي همغږي: کټګورۍ، ټګونه، او دودیزې ټکسونومۍ ساتي
* د Elementor ملاتړ: د CSS بيا جوړولو او د کينډۍ ID بيا نښلولو سره د Elementor کينډيو بشپړ ملاتړ
* د شخړې حل: د شته منځپانګې د بدلولو يا له هغې د تېرېدو اختيار
* د زېرمتون مديريت: له همغږۍ وروسته په هدف وېبپاڼو کې د Elementor زېرمتون په اوتومات ډول پاکوي

## اړتیاوې {#requirements}

* WordPress 5.3 يا لوړ
* PHP 7.4 يا لوړ
* Ultimate Multisite پلګین (فعال)
* Elementor پلګین (اختياري، د Elementor کينډۍ همغږۍ لپاره اړين)

## نصب {#installation}

1. 'ultimate-multisite-content-sync' د '/wp-content/plugins/' لارښود ته پورته کړئ
2. پلګین په خپل WordPress شبکې اداره کې د 'Plugins' مېنو له لارې فعال کړئ
3. امستنې په Ultimate Multisite → Settings → Content Sync کې تنظيم کړئ
4. د همغږۍ پيلولو لپاره Ultimate Multisite → Content Sync ته لاړ شئ
