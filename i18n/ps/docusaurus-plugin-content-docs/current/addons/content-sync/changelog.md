# د منځپانګې هممهاله کولو بدلون‌لیک {#content-sync-changelog}

### نسخه 1.1.0 - په 2025-12-31 خپره شوه {#version-110---released-on-2025-12-31}
* نوی: د ټولو لیکنو ډولونو د هممهاله کولو ملاتړ (لیکنې، پاڼې، دودیز د لیکنې ډولونه)
* نوی: د منځپانګې د ټاکنې په ګام کې د لیکنې ډول چاڼول
* نوی: د سرچینې سایټ د ټاکنې ګام - له هر سایټ څخه هممهاله کول، نه یوازې له اصلي سایټ څخه
* نوی: د کينډۍ سایټونو لپاره د ماشوم سایټ له مخکې ټاکنه
* نوی: د ټولو لیکنو ډولونو لپاره له post_content څخه د رسنیو راایستل
* نوی: د ډلبندۍ هممهاله کول (وېشنيزې، نښانونه، دودیزې ډلبندۍ)
* بدل شو: له "Elementor Template Sync" څخه "Content Sync" ته نوم بدل شو
* بدل شو: د "Select Templates" ګام نوم "Select Data" ته بدل شو
* ښه شو: د Elementor ځانګړی کوډ اوس یوازې د elementor_library لیکنې ډول لپاره چلېږي

### نسخه 1.0.0 - په 2025-12-19 خپره شوه {#version-100---released-on-2025-12-19}
* لومړنۍ خپرونه
* د کينډۍ او سایټ د ټاکنې لپاره څو-ګامه لارښود
* د ریښتیني وخت پرمختګ غبرګون سره ډله‌ییزه پروسس
* د اتومات ID بیا-نقشې کولو سره بشپړ د رسنیو هممهاله کول
* له Ultimate Multisite سره د امستنو ادغام
* د شبکې امستنو له لارې د سایټ اېستلو ملاتړ
* له هممهاله کولو وروسته د Elementor کَش پاکول
* د کينډۍ ډلبندۍ ساتنه
