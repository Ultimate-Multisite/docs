# د VAT بدلونلیک

نسخه 1.0.7 - په 2026-02-03 خپره شوې

* سمونه: د جرمني (DE) VAT شمېرې د VIES SOAP خدمت د MS_MAX_CONCURRENT_REQ تېروتنو له امله په اعتبارسنجۍ کې ناکامېدې. د لا باوري اعتبارسنجۍ لپاره له پخواني SOAP API څخه رسمي EU VIES REST API ته واوښت.
* سمونه: VAT شمېرې اوس منل کېږي کله چې د VIES خدمت لنډمهاله نه وي موجود، پر ځای د دې چې په ناسم ډول رد شي. د وروسته بیاکتنې لپاره د لاګ یوه ننوتنه جوړېږي.
* سمونه: د VAT/Tax ID ډګر اوس د ټولو هېوادونو لپاره ښکاري، یوازې د EU غړو لپاره نه. د غیر-EU مالیاتي پېژندپاڼې (لکه د سویس CHE شمېرې) د فاکتور ښودنې لپاره له VIES اعتبارسنجۍ پرته ساتل کېږي.
* سمونه: د یونان (GR)، موناکو (MC)، او Isle of Man (IM) لپاره د VAT مخکښ لټون د ناسم ارې لټون منطق له امله خراب و.
* ښه‌والی: غیرضروري wp-cli-bundle د پراختیا تړاو لرې شو چې د Site Exporter اضافې ترڅنګ د کارولو پر مهال یې جدي تېروتنې رامنځته کولای شوې.
* ښه‌والی: بشپړ unit test ټولګه (53 ازموینې) زیاته شوه.

نسخه 1.0.6 - په 2026-01-25 خپره شوې

* سمونه: کورنۍ B2B راکړې ورکړې اوس په سمه توګه VAT اخلي. د EU VAT د اصولو له مخې، reverse charge یوازې د پولې هاخوا B2B راکړو ورکړو لپاره پلي کېږي، نه هغه وخت چې د پېرودونکي هېواد د شرکت له هېواد سره برابر وي.
