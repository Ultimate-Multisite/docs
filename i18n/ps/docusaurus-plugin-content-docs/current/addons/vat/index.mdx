# VAT {#vat}

Multisite Ultimate: اروپايي VAT

د Multisite Ultimate د اصلي مالیې سمبالښت ته اضافي ځانګړتیاوې او وسیلې زیاتوي، څو د اروپايي VAT ښه ملاتړ وکړي.
په اوتومات ډول د EU VAT نرخونه د مالیې نرخونو پاڼې ته له

## نصب {#installation}

1. 'ultimate-multisite-vat' د '/wp-content/plugins/' لارښود ته پورته کړئ
2. پلګین په WordPress کې د 'پلګینونه' مینو له لارې فعال کړئ
3. Ultimate Multisite > تنظیمات > مالیې ته لاړ شئ او EU Vat فعال کړئ.
4. د مالیې نرخونو باندې کلیک وکړئ، په ښکته برخه کې د EU VAT نرخونه تازه کړئ او د مالیې نرخونه خوندي کړئ.
5. بشپړ شو!
