# Gravity Forms {#gravity-forms}

**د Gravity Forms په ځواک سره د خپل کارن د نوم‌لیکنې تجربه بدله کړئ**

د نړۍ تر ټولو پرمختللی فورم جوړوونکی له Multisite Ultimate سره بې‌درزه یوځای کړئ، څو ښکلي، د لوړ بدلون لرونکي د نوم‌لیکنې فورمونه جوړ کړئ چې ستاسو د غړیتوب نوم‌لیکنې زیاتوي او عاید لوړوي.

---

## ✨ ولې دا یوځای‌والی وټاکئ؟ {#-why-choose-this-integration}

### 🎯 **بدلونونه تر ۴۰٪ پورې زیات کړئ** {#-boost-conversions-by-up-to-40}
ستړي کوونکي تلوالیز د checkout فورمونه په ښکلو، دودیزو Gravity Forms بدل کړئ چې لیدونکي په پیسې ورکوونکو پېرودونکو بدلوي.

### 🛠️ **Ultimate انعطاف** {#️-ultimate-flexibility}
هر فورم ډګر د Multisite Ultimate هرې ډاټا نقطې ته ونښلوئ. هماغه معلومات راټول کړئ چې اړتیا ورته لرئ، بالکل په هغه ډول چې تاسو یې غواړئ.

### 💳 **د تادیې ګڼ انتخابونه** {#-multiple-payment-options}
د اعظمي payment gateway سازګارۍ لپاره هم مستقیم checkout redirect او هم WooCommerce یوځای‌والی ملاتړ کړئ.

### ⚡ **هیڅ کوډ لیکلو ته اړتیا نشته** {#-zero-coding-required}
د یوه روان admin انترفېس سره د اشاره او کلېک تنظیمات. تخنیکي تخصص ته اړتیا نشته.

## 🚦 دا څنګه کار کوي {#-how-it-works}

### 1. **خپل فورم ډیزاین کړئ** {#1-design-your-form}
د بشپړې نوم‌لیکنې تجربې جوړولو لپاره د Gravity Forms drag-and-drop جوړوونکی وکاروئ.

### 2. **خپل ډګرونه نقشه کړئ** {#2-map-your-fields}
د فورم ډګرونه زموږ د لیدیز نقشه کولو انترفېس له لارې د Multisite Ultimate ډاټا نقطو سره ونښلوئ.

### 3. **تادیه تنظیم کړئ** {#3-configure-payment}
د تادیې پروسس لپاره د checkout redirect یا WooCommerce یوځای‌والي ترمنځ انتخاب وکړئ.

### 4. **ژوندی یې کړئ** {#4-go-live}
خپل فورم خپور کړئ او د لیدونکو په پیسې ورکوونکو پېرودونکو بدلول پیل کړئ!
