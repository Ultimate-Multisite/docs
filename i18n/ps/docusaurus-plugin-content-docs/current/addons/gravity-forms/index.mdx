# Gravity Forms

**د Gravity Forms په ځواک سره د خپل کارن د نوم‌لیکنې تجربه بدله کړئ**

د نړۍ تر ټولو پرمختللی فورم جوړوونکی له Multisite Ultimate سره بې‌درزه یوځای کړئ، څو ښکلي، د لوړ بدلون لرونکي د نوم‌لیکنې فورمونه جوړ کړئ چې ستاسو د غړیتوب نوم‌لیکنې زیاتوي او عاید لوړوي.

---

## ✨ ولې دا یوځای‌والی وټاکئ؟

### 🎯 **بدلونونه تر ۴۰٪ پورې زیات کړئ**
ستړي کوونکي تلوالیز د checkout فورمونه په ښکلو، دودیزو Gravity Forms بدل کړئ چې لیدونکي په پیسې ورکوونکو پېرودونکو بدلوي.

### 🛠️ **Ultimate انعطاف**
هر فورم ډګر د Multisite Ultimate هرې ډاټا نقطې ته ونښلوئ. هماغه معلومات راټول کړئ چې اړتیا ورته لرئ، بالکل په هغه ډول چې تاسو یې غواړئ.

### 💳 **د تادیې ګڼ انتخابونه**
د اعظمي payment gateway سازګارۍ لپاره هم مستقیم checkout redirect او هم WooCommerce یوځای‌والی ملاتړ کړئ.

### ⚡ **هیڅ کوډ لیکلو ته اړتیا نشته**
د یوه روان admin انترفېس سره د اشاره او کلېک تنظیمات. تخنیکي تخصص ته اړتیا نشته.

## 🌟 مهمې ځانګړتیاوې

### 📋 **پرمختللی فورم جوړول**
- **څو پړاوی فورمونه** د پېچلو نوم‌لیکنې بهیرونو لپاره
- **شرطي منطق** د ډګرونو د خوځنده ښودلو/پټولو لپاره
- **دوتنې پورته کول** د سندونو او انځورونو لپاره
- **بډایه ډګر ډولونه**: لاسلیکونه، محاسبې، د محصول ټاکنه
- **موبایل ته ځواب ویونکي** فورمونه چې په هر وسیله کې ښه ښکاري

### 🔗 **بې‌درزه ډاټا نقشه کول**
- **لیدیز ډګر نقشه کول** انترفېس
- **د پېرودونکي معلومات**: نومونه، برېښنالیک، د بل ورکولو جزئیات
- **د غړیتوب ډاټا**: پلانونه، موده، دودیز ډګرونه
- **د سایټ جوړول**: سرلیکونه، URLs، د کينډۍ ټاکنه
- **دودیزه ډاټا راټولونه** ستاسو د ځانګړو اړتیاوو لپاره

### 💰 **انعطاف منونکې د تادیې پروسس**
- **Checkout redirect**: د Multisite Ultimate تادیې پاڼو ته بې‌درزه لېږد
- **WooCommerce یوځای‌والی**: له سلګونو payment gateways څخه ګټه واخلئ
- **وړیا آزموینې او پروموشنونه** ملاتړ
- **اتومات انوایس جوړول** او د رسید تولید

### 🎨 **مسلکي د کارن تجربه**
- **دودیزې تایید پاڼې** له merge tags سره
- **اتومات برېښنالیک خبرتیاوې** د پېرودونکو او admins لپاره
- **د پرمختګ نښې** د څو پړاوی فورمونو لپاره
- **د تېروتنې اعتبارسنجونه** له ګټورو پیغامونو سره
- **د برانډ یووالی** ستاسو د سایټ له ډیزاین سره

### 🛡️ **د تصدۍ کچې امنیت**
- **د سپام ساتنه** له دننه CAPTCHA انتخابونو سره
- **د ډاټا اعتبارسنجونه** او پاکول
- **WordPress امنیت** غوره کړنې
- **GDPR مطابقت** ځانګړتیاوې
- **خوندي دوتنې پورته کول** او زېرمه

## 🚦 دا څنګه کار کوي

### 1. **خپل فورم ډیزاین کړئ**
د بشپړې نوم‌لیکنې تجربې جوړولو لپاره د Gravity Forms drag-and-drop جوړوونکی وکاروئ.

### 2. **خپل ډګرونه نقشه کړئ**
د فورم ډګرونه زموږ د لیدیز نقشه کولو انترفېس له لارې د Multisite Ultimate ډاټا نقطو سره ونښلوئ.

### 3. **تادیه تنظیم کړئ**
د تادیې پروسس لپاره د checkout redirect یا WooCommerce یوځای‌والي ترمنځ انتخاب وکړئ.

### 4. **ژوندی یې کړئ**
خپل فورم خپور کړئ او د لیدونکو په پیسې ورکوونکو پېرودونکو بدلول پیل کړئ!
