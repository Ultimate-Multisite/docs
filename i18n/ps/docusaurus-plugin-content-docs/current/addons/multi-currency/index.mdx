# Multi-Currency {#multi-currency}

Ultimate Multisite: Multi-Currency ستاسو شبکې ته دا وړتیا ورکوي چې په څو اسعارو کې پیسې ومني. دا په اوتومات ډول بیې د ژوندیو یا لاسي تبادلې نرخونو په کارولو بدلوي، تاسو ته اجازه درکوي چې د هر اسعار لپاره د هر محصول دقیقې بیې وټاکئ، او د هر لیدونکي غوره اسعار د جغرافیایي موقعیت له لارې پېژني — دا ټول له یوه واحد اداري Dashboard څخه اداره کېږي.

## کلیدي ځانګړتیاوې {#key-features}

- **د تبادلې نرخ اوتومات بدلون** — بیې ستاسو له بنسټیز اسعار څخه د هغه نرخونو په کارولو بدلېږي چې ستاسو له غوره شوي برابرونکي څخه ترلاسه کېږي
- **د هر محصول د بیې ځانګړي بدلونونه** — په هر محصول کې د ځانګړو اسعارو دقیقې بیې وټاکئ څو اوتومات بدلون پرېښودل شي
- **د نوم‌لیکنې د اسعار ټاکونکي ډګر** — پېرودونکو ته اجازه ورکوي چې د checkout/نوم‌لیکنې پر مهال خپل غوره اسعار وټاکي
- **د تبادلې نرخ څو برابرونکي** — ECB، Frankfurter، Stripe، یا بشپړ لاسي نرخونه
- **د جغرافیایي موقعیت پر بنسټ د اسعار پېژندنه** — په اوتومات ډول هغه اسعار مخکې ټاکي چې د لیدونکي له هېواد سره سمون لري
- **اداري Dashboard** — فعال اسعار، د تبادلې نرخونه، او د برابرونکي امستنې له یوه ځایه اداره کړئ

## اړتیاوې {#requirements}

- Ultimate Multisite 2.0.0 یا لوړ
- د WordPress Multisite نصب
- PHP 7.4 یا لوړ

## نصب {#installation}

1. د plugin فایلونه `/wp-content/plugins/ultimate-multisite-multi-currency` ته پورته کړئ
2. plugin د WordPress د **Plugins** پردې له لارې فعال کړئ (Network Activate)
3. **Ultimate Multisite → Multi-Currency** ته لاړ شئ څو اداري Dashboard پرانیزئ
4. هغه اسعار ورزیات کړئ چې منل یې غواړئ او خپل د تبادلې نرخ برابرونکی تنظیم کړئ

## تنظیمات {#configuration}

### اداري Dashboard {#admin-dashboard}

د Multi-Currency اداري Dashboard (**Ultimate Multisite → Multi-Currency**) مرکزي ځای دی د دې لپاره چې:

- اسعار ورزیات، فعال، یا غیرفعال کړئ
- د تبادلې نرخ برابرونکی وټاکئ او نرخونه د غوښتنې پر مهال تازه کړئ
- د ټولو فعالو اسعارو اوسني نرخونه وګورئ
- کله چې اوتومات برابرونکي مناسب نه وي، لاسي نرخونه وټاکئ

### د تبادلې نرخ برابرونکي {#exchange-rate-providers}

| برابرونکی | تشریح |
|---|---|
| **ECB** | European Central Bank — د EUR پر بنسټ شبکو لپاره وړیا ورځني نرخونه |
| **Frankfurter** | د خلاصې سرچینې ECB هنداره API — وړیا، API کیلي ته اړتیا نشته |
| **Stripe** | د Stripe ژوندۍ FX نرخونه کاروي — فعال Stripe ادغام ته اړتیا لري |
| **Manual** | تاسو نرخونه مستقیم دننه کوئ؛ هېڅ بهرني API غوښتنې نشته |

خپل برابرونکی د **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider** لاندې وټاکئ. نرخونه هر وخت په لاسي ډول تازه کېدای شي یا د اوتومات تازه کېدو لپاره مهالویش کېدای شي.

### د هر محصول د بیې ځانګړي بدلونونه {#per-product-pricing-overrides}

کله چې اوتومات بدلون کافي دقیق نه وي، تاسو کولی شئ په هر محصول کې د ځانګړو اسعارو لپاره دقیقه بیه وټاکئ:

1. د محصول سمونګر پرانیزئ (**Ultimate Multisite → Products → Edit**)
2. د **Currency Pricing** برخې ته ښکته لاړ شئ
3. د هر هغه اسعار لپاره دقیقه بیه دننه کړئ چې بدلول یې غواړئ
4. محصول خوندي کړئ

هغه محصولات چې ځانګړی بدلون نه لري، ستاسو له بنسټیز اسعار څخه بدله شوې بیه کاروي.

### د نوم‌لیکنې د اسعار ټاکونکي ډګر {#currency-selector-signup-field}

د اسعار ټاکونکي ډګر ستاسو د نوم‌لیکنې/checkout بهیر ته ورزیاتېدای شي څو پېرودونکي د نوم‌لیکنې پر مهال خپل غوره اسعار وټاکي:

1. **Ultimate Multisite → Multi-Currency → Settings** ته لاړ شئ
2. **Show currency selector on signup** فعال کړئ
3. په اختیاري ډول یو تلواله اسعار وټاکئ یا جغرافیایي موقعیت ته اجازه ورکړئ چې مخکې ټاکل شوی ارزښت وټاکي

ټاکل شوی اسعار له ګډون سره زېرمه کېږي او د ټولو راتلونکو بېلونو لپاره کارول کېږي.

### د جغرافیایي موقعیت پر بنسټ د اسعار پېژندنه {#geolocation-based-currency-detection}

کله چې د جغرافیایي موقعیت پېژندنه فعاله وي، addon د لیدونکي هېواد له یوه اسعار سره نښلوي او هغه د اسعار ټاکونکي ډګر کې مخکې ټاکي. د فعالولو لپاره:

1. **Ultimate Multisite → Multi-Currency → Settings** ته لاړ شئ
2. **Detect currency from visitor location** فعال کړئ
3. هېوادونه د **Geolocation Mapping** جدول کې اسعارو ته ونښلوئ

که د لیدونکي هېواد لپاره هېڅ نښلون شتون ونه لري، د شبکې بنسټیز اسعار د بدیل په توګه کارول کېږي.

## ډېرې پوښتل شوې پوښتنې {#frequently-asked-questions}

### که د تبادلې نرخ برابرونکی شتون ونه لري څه پېښېږي؟ {#what-happens-if-an-exchange-rate-provider-is-unavailable}

addon وروستیو په بریالیتوب ترلاسه شویو نرخونو ته بېرته ګرځي. که هېڅ نرخونه تر اوسه نه وي ترلاسه شوي، دا هغو لاسي نرخونو ته بېرته ګرځي چې تاسو تنظیم کړي دي. سپارښتنه کېږي چې لاسي نرخونه د خوندي بدیل په توګه تنظیم کړئ.

### ایا زه د بېلابېلو اسعارو لپاره بېلابېل برابرونکي کارولی شم؟ {#can-i-use-different-providers-for-different-currencies}

نه — په یوه وخت کې د ټولو اسعارو لپاره یو برابرونکی فعال وي. که د انفرادي نرخونو بشپړ کنټرول ته اړتیا لرئ، **Manual** وکاروئ.

### ایا جغرافیایي موقعیت جلا plugin ته اړتیا لري؟ {#does-geolocation-require-a-separate-plugin}

هېڅ بهرني جغرافیایي موقعیت plugin ته اړتیا نشته. addon د WordPress جوړ دننه IP-تر-هېواد پېژندنه کاروي. د لوړې کره‌والي لپاره، ځانګړی جغرافیایي موقعیت خدمت د ورکړل شوو filter hooks له لارې مدغم کېدای شي.

### ایا د هر محصول ځانګړي بدلونونه له مالیې مخکې پلي کېږي که وروسته؟ {#are-per-product-overrides-applied-before-or-after-tax}

د هر محصول ځانګړي بدلونونه په هدف اسعار کې له مالیې مخکې بیه ټاکي. مالیه ستاسو د شته مالیې تنظیماتو په کارولو د ځانګړې بدلون بیې پر سر محاسبه کېږي.
