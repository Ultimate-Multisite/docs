# د Multi-Network بدلون‌لیک

### 1.3.0 {#130}
* نوی: د شبکې کينډۍ مخکتونکی — د پېرودلو مخکې د ژوندۍ تختې له لارې د شبکې کينډۍ وپلټئ او مخکتنه یې وکړئ.
* نوی: په checkout کې د شبکې کينډۍ ټاکونکی — پېرېدونکي کولای شي د نوم‌لیکنې checkout بهیر پر مهال د شبکې کينډۍ وټاکي.
* نوی: د شبکو ترمنځ magic link SSO — پېرېدونکي کولای شي د magic-link تصدیق له لارې فرعي شبکو ته په بې‌وقفې ډول لاسرسی ومومي.
* نوی: د شبکې رسنیو کاپي کول — د کينډۍ څخه د شبکو د کاپي کولو پر مهال د سایټ رسنۍ په سمه توګه نقل کېږي.
* سمون: د شبکې کاپي کول اوس د multi-tenancy addon پرته کار کوي؛ د کاپي کولو وروسته super admin امتیازونه بیا ورکول کېږي.
* سمون: د شبکې جوړول اوس له WooCommerce او ټولو payment gateways سره سازګار دي.
* سمون: د پورټ شمېره اوس د احتیاطي ډومېن په جوړولو کې شامله ده.
* سمون: د کينډۍ ټاکونکي CSS د ټولو themes لپاره د سمې ښودنې په موخه enqueued stylesheet ته ولېږدول شو.
* سمون: د سایټ پوښتنې په محدوده کې د نامعلوم `network_id` کیلي پر ضد ساتنه.
* سمون: د context switching مخکې د شبکې اجازې کتنه د `wpdb` د جدول property تېروتنې مخه نیسي.
