# د WooCommerce Integration د بدلونونو تاریخچه {#woocommerce-integration-changelog}

Version 2.2.0 - په 2026-07-01 خپور شو
* نوی: د Ultimate Multisite د مالیې مقدارونه اوس په checkout کې د جلا WooCommerce فیس کرښو په توګه ښکاري، چې د تادیې تر مخه د مالیې ټولیز مقدارونه روښانه کوي.
* نوی: د هغو سایټونو لپاره چې د WooCommerce Subscriptions د بیا هڅې کړکۍ په جریان کې د ناکامې بیا نوي کېدو ځنډول غواړي، د ګډون په خوښه "Suspend Memberships Immediately on Failed Renewals" ترتیب او `wu_woo_suspend_on_payment_failure` filter اضافه شو.
* سمون: د WooCommerce Subscription حالتونه همغږي شول چې د ناکامو یا بېرته رغېدلو بیا نوي کېدو وروسته له Ultimate Multisite غړیتوبونو سره له همغږۍ پاتې کېدای شول.
* سمون: د WooCommerce پلورنځي اسعار د Ultimate Multisite د اسعارو لېست ته اضافه شول کله چې ورک وو.
* سمون: WooCommerce checkout ته د subscribers د لېږلو پر مهال د پېرېدونکي د بلینګ جزییات وساتل شول.
* ښه شوی: له Jetpack Autoloader 5 سره سازګاري اضافه شوه.
* ښه شوی: د خپرونې د بسته جوړونه پاکه شوه څو د GitHub او بازار زېپونه له ځاله‌ییزو staging پوښیو او د پراختیا له فایلونو ډډه وکړي.
