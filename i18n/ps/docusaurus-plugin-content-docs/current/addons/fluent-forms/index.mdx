# Fluent Forms

دا اضافه په Multisite Ultimate کې په محصولاتو، غړیتوبونو یا سایټونو کې Fluent Forms ته د محدودیتونو زیاتول ممکن کوي.
Fluent Forms د دودیزو پوسټ ډولونو پر ځای دودیز جدولونه کاروي، نو عادي محدودوونکي اختیارونه له Fluent Forms سره کار نه کوي.

## ځانګړتیاوې

- د Fluent Forms یا د خبرو اترو Fluent Forms لپاره محدودیتونه زیات کړئ
