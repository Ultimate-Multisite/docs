# د ادارې پاڼې جوړوونکی

## منځپانګه
د ادارې پاڼې جوړوونکی به تاسو ته اجازه درکړي چې د WordPress د ادارې Dashboard دننه نوې ادارې پاڼې او فرعي پاڼې په اسانۍ ورزیاتې کړئ. دا د ښه راغلاست پاڼې یا ښوونیزې پاڼې جوړولو لپاره کارېدلی شي چې د نوو کارنانو لپاره د ننوتلو پاڼې په توګه خدمت وکړي. تاسو کولی شئ د Beaver Builder، Elementor، Brizy، او Oxygen په څېر د پاڼې جوړوونکي وکاروئ ترڅو پاڼې بې له ستونزې ډیزاین کړئ.

دا د دودیز HTML، CSS او JavaScript ملاتړ هم وړاندې کوي، همدارنګه د پلان او د کارن رول پر بنسټ د لاسرسي کنټرول!

## د کارولو لارښود

په داسې حال کې چې WordPress د وېبپاڼو مخې برخې ته د منځپانګې ورزیاتول رښتیا هم اسانه کوي، د ادارې په اړخ کې د پاڼو او فرعي پاڼو جوړول ځینې بنسټیزې کوډ لیکنې وړتیاوې غواړي. Ultimate Multisite: د ادارې پاڼې جوړوونکی موخه لري چې د مخې برخې هماغه د کارولو اسانتیا ستاسو د شبکې وېبپاڼو شاتنۍ برخې ته راوړي.

## د مېنو انتخابونه

### د مېنو ډولونه

ستاسو نوې دودیزه ادارې پاڼه دوه بېلابېل مېنو ډولونه لرلی شي: د لوړې کچې مېنو او فرعي مېنو.

د لوړې کچې مېنو پاڼه، لکه څنګه چې نوم یې ښيي، ستاسو نوې جوړه شوې پاڼه به د WordPress اصلي پاڼو ترڅنګ د ادارې مېنو ته ورزیاته کړي.

که تاسو د لوړې کچې انتخاب وټاکئ، تاسو به وکولی شئ د مېنو ترتیب او د مېنو نښه هم وټاکئ.

که تاسو غواړئ دا پاڼه د فرعي مېنو پاڼه کړئ، تاسو به وکولی شئ یوه مور پاڼه وټاکئ چې دا پاڼه ورسره ونښلول شي. تاسو کولی شئ خپله دودیزه پاڼه د WordPress اصلي پاڼو یا خپلو لوړې کچې دودیزو ادارې پاڼو ته ورزیاته کړئ.
