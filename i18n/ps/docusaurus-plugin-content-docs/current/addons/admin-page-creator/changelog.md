### نسخه 1.9.0 - په 2026-01-18 خپره شوه {#version-190---released-on-2026-01-18}

* سم شو: د فعالولو پر مهال Fatal تېروتنې.

* سم شو: د Divi ملاتړ فایل د Composer له لارې په اتومات ډول نه پورته کېده
* سم شو: د Divi ملاتړ کلاس کې خراب PHP syntax
* ښه شو: د JavaScript شتمنیو تنظیم او کوچني کول
* ښه شو: له وروستي WordPress او د پاڼې جوړوونکي نسخو سره همغږي
