# PayFast

Multisite Ultimate: Payfast Gateway

Payfast — په سویلي افریقا کې د چټکو او خوندي انلاین تادیاتو دروازه — د Multisite Ultimate د تادیې د اختیار په توګه زیاتوي.

## لګول {#installation}

1. 'ultimate-multisite-payfast' د '/wp-content/plugins/' ډایرکټرۍ ته پورته کړئ
2. د WordPress د 'Plugins' مینو له لارې plugin فعال کړئ
3. د plugin د تنظیمولو لپاره ګام په ګام Wizard تعقیب کړئ

## د کارونې لارښود {#usage-guide}

PayFast په سویلي افریقا کې د کاروبارونو لپاره د انلاین تادیاتو مخکښ پروسس کوونکی دی. دا د انلاین پلورونکو د ودې له ملاتړ سره ژمن دی، د راکړې ورکړې بهیر اسانه او خوندي کوي. د Ultimate Multisite کارونکي د غوښتنې له مخې، موږ پرېکړه وکړه چې داسې اضافه جوړه کړو چې له PayFast سره ادغام ممکن کړي. دا لیکنه به وښيي چې څنګه PayFast د خپلې Ultimate Multisite شبکې کې د تادیې ترلاسه کولو لپاره د تادیې دروازې په توګه تنظیم کړئ.

لومړی ګام د اضافې لګول دي.

# د Payfast اضافې د لګولو څرنګوالی:

د اضافو پاڼې ته لاړ شئ او د Ultimate Multisite: Payfast Gateway اضافه وټاکئ.

![د اضافو پاڼه](/img/admin/addons-page.png)

د اضافې کړکۍ به راښکاره شي. په Install Now کلیک وکړئ.

![د لګولو اختیار سره د اضافو پاڼه](/img/admin/addons-page.png)

کله چې لګول بشپړ شي، تاسو به د plugins پاڼې ته ولېږدول شئ. دلته، یوازې په Network Activate کلیک وکړئ او د _Payfast_ اضافه به ستاسو په شبکه کې فعاله شي.

# د PayFast تنظیمول

د اضافې له لګولو وروسته، تاسو اوس د Ultimate Multisite تنظیماتو ته تللی شئ او د PayFast تنظیمول پیلولی شئ.

د **Ultimate Multisite settings** پاڼې لاندې، تاسو د "Payments" ټب موندلی شئ. د دې ټب لاندې، تاسو باید "**Payfast** " د یوې بدلون تڼۍ سره ووینئ چې دا د تادیې دروازه فعاله یا غیرفعاله کړي.

![د تادیې دروازې تنظیمات](/img/config/settings-payment-gateways.png)مخته لاړ شئ او د فعالولو لپاره د بدلون تڼۍ کلیک کړئ. بیا به دا بکس پراخ شي څو بېلابېل ډګرونه وښيي چې تاسو یې باید ډک کړئ، لکه **Mechant ID، Merchant Key او PassPhrase.** تاسو دا معلومات له خپل **Payfast** Account څخه ترلاسه کولی شئ

<!-- سکرین‌شاټ نشته: د PayFast پراخ شوي تنظیمات چې د Merchant ID، Merchant Key، او PassPhrase ډګرونه ښيي -->

بل ګام دا دی چې خپل PayFast Account ته ننوزئ څو هغه معلومات ترلاسه کړئ چې اړتیا ورته لرئ.

له ننوتلو وروسته، تاسو کولی شئ سملاسي له خپل **Payfast** Account Dashboard څخه هغه اسناد ووینئ چې اړتیا ورته لرئ.

<!-- سکرین‌شاټ نشته: د PayFast Account Dashboard چې د Merchant ID او Merchant Key اسناد ښيي -->تاسو کولی شئ سندونه په ساده ډول کاپي او پیسټ کړئ او په خپلو Ultimate Multisite تنظیماتو کې جزئیات ډک کړئ. بیا د بدلونونو د خوندي کولو لپاره د **"Save settings** " تڼۍ کلیک کړئ.

<!-- سکرین‌شاټ نشته: د Ultimate Multisite PayFast تنظیمات چې اسناد پکې ډک شوي او د Save تڼۍ ښيي -->

همدومره، ستاسو د PayFast تادیې ادغام بشپړ شو!
