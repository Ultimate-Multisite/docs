# Ultimate Multisite డాక్యుమెంటేషన్ {#ultimate-multisite-documentation}

Ultimate Multisite డాక్యుమెంటేషన్‌కి స్వాగతం. Ultimate Multisite అనేది WordPress Multisiteని పూర్తి వెబ్‌సైట్ యాజ్ ఎ సర్వీస్ (WaaS) ప్లాట్‌ఫామ్‌గా మారుస్తుంది.

## డాక్యుమెంటేషన్ విభాగాలు {#documentation-sections}

### [User Guide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

మీ WaaS ప్లాట్‌ఫామ్‌ను సెటప్ చేయడానికి, కాన్ఫిగర్ చేయడానికి మరియు నిర్వహించడానికి మీకు అవసరమైన ప్రతిదీ ఇక్కడ ఉంది. ఇందులో ఇన్‌స్టాలేషన్, పేమెంట్ గేట్‌వేలు, డొమైన్ మ్యాపింగ్, క్లయింట్ మేనేజ్‌మెంట్ మరియు మరిన్నింటి వివరాలు ఉన్నాయి.

### [Developer Documentation](/developer/hooks) {#developer-documentation}

Ultimate Multisite పై పనిచేసే డెవలపర్‌ల కోసం ఇది ఒక సాంకేతిక రిఫరెన్స్. ఇందులో హుక్స్ రిఫరెన్స్, REST API డాక్యుమెంటేషన్ మరియు అడాన్ డెవలప్‌మెంట్ గైడ్‌లు ఉన్నాయి.

### [Addons](/addons) {#addons}

WooCommerce ఇంటిగ్రేషన్, అనలిటిక్స్, డొమైన్ సెల్లింగ్ వంటి అదనపు ఫీచర్‌లతో Ultimate Multisiteని విస్తరించే చెల్లింపు అడాన్ ప్లగిన్‌ల డాక్యుమెంటేషన్ ఇది.

- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- బగ్స్ రిపోర్ట్ చేయడానికి
