# Plugin & Theme Manager

Multisite Ultimate: Plugin మరియు Theme Manager

ఇన్‌స్టాల్ చేయబడిన plugins మరియు themes యొక్క meta సమాచారాన్ని (శీర్షిక, రచయిత, థంబ్‌నెయిల్ మరియు వివరణ) ఎడిట్ చేయవచ్చు లేదా దాచవచ్చు. మీ వినియోగదారులు వాటిని ఫిల్టర్ చేయడానికి మరియు మీ వినియోగదారుల ప్యానెల్‌లో అందమైన కస్టమ్ Plugins పేజీని జోడించడానికి, plugins మరియు themes కోసం కేటగిరీలను సృష్టించండి.

ఈ pluginని ఉపయోగించడానికి, మీరు మీ network admin లోని Plugins లేదా Themes పేజీకి వెళ్లి, "Edit Plugin/Theme info" పై క్లిక్ చేయవచ్చు.

## Installation

1. 'ultimate-multisite-plugin-and-theme-manager' ను '/wp-content/plugins/' డైరెక్టరీలోకి అప్‌లోడ్ చేయండి
2. మీ WordPress Network Admin లోని 'Plugins' మెనూ ద్వారా pluginని యాక్టివేట్ చేయండి

## Usage Guide

Ultimate Multisite: Plugin మరియు Theme Manager అనేది ఒక శక్తివంతమైన **premium** Ultimate Multisite add-on. ఇది మీ network లోని plugin మరియు theme ఆప్షన్లను మీ network వినియోగదారులు ఎలా చూస్తారు మరియు వాటితో ఎలా ఇంటరాక్ట్ అవుతారో అనే దానిపై మీకు పూర్తి నియంత్రణను ఇస్తుంది.

ఈ add-on మూడు ప్రయోజనాలను అందిస్తుంది:

  1. ఇది plugins మరియు themes రెండింటి యొక్క metadataని పూర్తిగా మార్చడానికి మిమ్మల్ని అనుమతిస్తుంది. ఇందులో పేరు, వివరణ, రచయిత పేరు, మరియు థంబ్‌నెయిల్ కూడా ఉంటాయి!

  2. మీ వినియోగదారులు వాటిని సులభంగా క్రమబద్ధీకరించడానికి (sort out) plugins మరియు themes కు కేటగిరీలను జోడించడానికి ఇది మిమ్మల్ని అనుమతిస్తుంది.

  3. పాత, విసుగు పుట్టించే plugins పేజీని, కేటగిరీల ద్వారా క్రమబద్ధీకరించబడిన (sorted) కొత్త themes లాంటి పేజీతో భర్తీ చేసే ఎంపికను ఇది మీకు ఇస్తుంది.

దీనిని డౌన్‌లోడ్ చేసి, ఇన్‌స్టాల్ చేసిన తర్వాత, Ultimate Multisite Settings పేజీలో _Add-on Settings_ అనే కొత్త ట్యాబ్ కనిపిస్తుంది.
