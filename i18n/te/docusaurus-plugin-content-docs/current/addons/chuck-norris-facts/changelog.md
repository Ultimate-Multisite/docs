# చక్ నార్సిస్ ఫ్యాక్ట్స్ మార్పుల వివరాలు

### 1.0.1 - 2025-11-03
* జోడించబడింది: Ultimate Multisite అప్‌డేట్ సర్వర్ ద్వారా ఆటోమేటిక్ అప్‌డేట్ ఫంక్షనాలిటీ.
* నవీకరించబడింది: WordPress 6.6 వరకు పరీక్షించబడింది.
* నవీకరించబడింది: composer autoloader తో ప్లగిన్ స్ట్రక్చర్‌ను మెరుగుపరిచాము.
* నవీకరించబడింది: ఆధునిక WordPress ఇన్‌స్టాలేషన్‌లతో అనుకూలతను పెంచాము.

### 1.0.0 - 2024-10-21
* తొలి విడుదల
* 110+ చక్ నార్సిస్ ఫ్యాక్ట్స్
* పూర్తి అనువాద మద్దతు
* అన్ని స్క్రీన్ సైజులకు అనుగుణంగా ఉండే డిజైన్ (Responsive design)
* RTL భాషా మద్దతు
* బ్లాక్ ఎడిటర్ అనుకూలత
* స్క్రీన్ రీడర్ మద్దతుతో యాక్సెసిబిలిటీ ఫీచర్లు
