# Loco Translate మార్పుల వివరాలు {#loco-translate-changelog}

### 1.0.1 {#101}
* లేటెస్ట్ Loco Translate వెర్షన్‌తో అనుకూలత మెరుగుపడింది
* మెరుగైన భద్రత కోసం యాక్సెస్ నియంత్రణ (access control) మెరుగుపరచబడింది
* పనితీరు మెరుగుదలలు (Performance optimizations)

### 1.0.0 {#100}
* తొలి విడుదల (Initial release)
* సైట్-నిర్దిష్ట అనువాద డైరెక్టరీలు (Site-specific translation directories)
* అనువాద ప్రాధాన్యత వ్యవస్థ (Translation priority system)
* యాక్సెస్ నియంత్రణ అమలు (Access control implementation)
* Loco Translate తో అంతరాయం లేని UI ఏకీకరణ (Seamless UI integration)
