# Filter: ultimate_multisite_content_sync_url

ప్లగిన్ మూలం (plugin root) నుండి కావాల్సిన రిలేటివ్ పాత్‌ను నిర్వహించడానికి ఇది డెవలపర్‌లకు సహాయపడుతుంది.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | ప్లగిన్ మూలం నుండి మీరు యాక్సెస్ చేయాలనుకుంటున్న రిలేటివ్ పాత్. |

### Since {#since}

- 1.0.0
### Source {#source}

[`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-content-sync/blob/main/inc/class-helper.php#L52) లోని 52వ లైన్‌లో నిర్వచించబడింది.

## Returns {#returns}
ప్లగిన్ మూలం నుండి రిలేటివ్ పాత్.
