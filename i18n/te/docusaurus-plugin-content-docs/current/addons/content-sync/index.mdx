# కంటెంట్ సింక్ (Content Sync)

మల్టీసైట్ అల్టిమేట్: కంటెంట్ సింక్ (Multisite Ultimate: Content Sync)

Multisite Ultimate కోసం ఈ అడాన్ (addon) నెట్‌వర్క్ అడ్మినిస్ట్రేటర్స్‌కు ఒక సైట్ నుండి నెట్‌వర్క్‌లోని ఇతర సైట్‌లకు కంటెంట్‌ను సింక్ (synchronize) చేయడానికి అనుమతిస్తుంది. ఇది పోస్టులు (posts), పేజీలు (pages), కస్టమ్ పోస్ట్ టైప్‌లు (custom post types), మరియు Elementor టెంప్లేట్‌లతో సహా అన్ని పోస్ట్ రకాలను సపోర్ట్ చేస్తుంది. కంటెంట్‌ను ఎంచుకోవడానికి మరియు లక్ష్య సైట్‌లను (target sites) ఎంచుకోవడానికి ఈ అడాన్ మల్టీ-స్టెప్ విజార్డ్ ఇంటర్‌ఫేస్‌ను అందిస్తుంది. ఇందులో బ్యాచ్ ప్రాసెసింగ్ మరియు రియల్-టైమ్ ప్రగతి ఫీడ్‌బ్యాక్ కూడా ఉంటుంది.

## ఫీచర్లు (Features) {#features}

* **కంటెంట్ ఎంపిక (Content Selection):** థంబ్‌నెయిల్స్ మరియు పోస్ట్ టైప్ ఫిల్టరింగ్‌తో విజువల్ కార్డ్ గ్రిడ్.
* **అన్ని పోస్ట్ రకాలు (All Post Types):** పోస్టులు, పేజీలు, ఉత్పత్తులు (products), Elementor టెంప్లేట్‌లు మరియు ఏ కస్టమ్ పోస్ట్ టైప్‌నైనా సింక్ చేయవచ్చు.
* **సైట్ ఎంపిక (Site Selection):** కంటెంట్‌ను ఏ సైట్‌లకు సింక్ చేయాలో ఎంచుకోవచ్చు. ఇందులో సోర్స్ సైట్‌ను మినహాయించడం మరియు చైల్డ్ సైట్‌లను ముందే ఎంచుకునే అవకాశం ఉంటుంది.
* **బ్యాచ్ ప్రాసెసింగ్ (Batch Processing):** ఒకేసారి 10 సైట్‌లకు సింక్ చేస్తుంది మరియు ప్రగతి అప్‌డేట్‌లను అందిస్తుంది.
* **మీడియా సింక్ (Media Sync):** పోస్ట్ కంటెంట్ నుండి లక్ష్య సైట్‌లకు మీడియా ఫైళ్లను (చిత్రాలు, వీడియోలు) ఆటోమేటిక్‌గా కాపీ చేసి, మ్యాప్ చేస్తుంది.
* **టాక్సానమీ సింక్ (Taxonomy Sync):** కేటగిరీలు (categories), ట్యాగ్‌లు (tags), మరియు కస్టమ్ టాక్సానమీలను (custom taxonomies) అలాగే ఉంచుతుంది.
* **Elementor సపోర్ట్ (Elementor Support):** CSS రీజెనరేషన్ మరియు టెంప్లేట్ ID రీమాపింగ్‌తో Elementor టెంప్లేట్‌లకు పూర్తి సపోర్ట్.
* **సంఘర్షణ పరిష్కారం (Conflict Resolution):** ఇప్పటికే ఉన్న కంటెంట్‌ను భర్తీ చేసే లేదా దానిని దాటవేసే (skip) ఎంపిక.
* **కాష్ నిర్వహణ (Cache Management):** సింక్ అయిన తర్వాత లక్ష్య సైట్‌లలో Elementor కాష్‌ను ఆటోమేటిక్‌గా క్లియర్ చేస్తుంది.

## అవసరాలు (Requirements) {#requirements}

* WordPress 5.3 లేదా అంతకంటే ఎక్కువ
* PHP 7.4 లేదా అంతకంటే ఎక్కువ
* Multisite Ultimate plugin (యాక్టివ్‌గా ఉండాలి)
* Elementor plugin (ఐచ్ఛికం, Elementor టెంప్లేట్ సింక్ కోసం అవసరం)

## ఇన్‌స్టాలేషన్ (Installation) {#installation}

1. 'ultimate-multisite-content-sync' ను '/wp-content/plugins/' డైరెక్టరీలోకి అప్‌లోడ్ చేయండి.
2. మీ WordPress Network Admin లోని 'Plugins' మెనూ ద్వారా ప్లగిన్‌ను యాక్టివేట్ చేయండి.
3. Ultimate Multisite → Settings → Content Sync లో సెట్టింగ్‌లను కాన్ఫిగర్ చేయండి.
4. సింకింగ్ ప్రారంభించడానికి Ultimate Multisite → Content Sync కి వెళ్లండి.
