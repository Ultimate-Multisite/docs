# కంటెంట్ సింక్ మార్పుల జాబితా

### వెర్షన్ 1.1.0 - విడుదల తేదీ 2025-12-31 {#version-110---released-on-2025-12-31}
* కొత్తగా: అన్ని పోస్ట్ రకాలను (posts, pages, custom post types) సింక్ చేయడానికి మద్దతు.
* కొత్తగా: కంటెంట్ ఎంపిక దశలో పోస్ట్ రకం ఫిల్టరింగ్.
* కొత్తగా: సోర్స్ సైట్ ఎంపిక దశ - ప్రధాన సైట్‌కే కాకుండా ఏ సైట్ నుండి అయినా సింక్ చేయవచ్చు.
* కొత్తగా: టెంప్లేట్ సైట్‌ల కోసం చైల్డ్ సైట్ ప్రీ-ఎంపిక.
* కొత్తగా: అన్ని పోస్ట్ రకాల కోసం post_content నుండి మీడియా ఎక్స్‌ట్రాక్షన్.
* కొత్తగా: టాక్సానమీ సింకింగ్ (categories, tags, custom taxonomies).
* మార్పు: "Elementor Template Sync" నుండి "Content Sync"గా పేరు మార్చబడింది.
* మార్పు: "Select Templates" దశ పేరు "Select Data"గా మార్చబడింది.
* మెరుగుపడింది: Elementor-నిర్దిష్ట కోడ్ ఇప్పుడు elementor_library పోస్ట్ రకానికి మాత్రమే నడుస్తుంది.

### వెర్షన్ 1.0.0 - విడుదల తేదీ 2025-12-19 {#version-100---released-on-2025-12-19}
* మొదటి విడుదల
* టెంప్లేట్ మరియు సైట్ ఎంపిక కోసం బహుళ-దశల విజర్డ్.
* రియల్-టైమ్ పురోగతి ఫీడ్‌బ్యాక్‌తో బ్యాచ్ ప్రాసెసింగ్.
* ఆటోమేటిక్ ID రీమ్యాపింగ్‌తో పూర్తి మీడియా సింక్.
* Ultimate Multisite తో సెట్టింగ్‌ల అనుసంధానం.
* నెట్‌వర్క్ సెట్టింగ్‌ల ద్వారా సైట్ మినహాయింపు మద్దతు.
* సింక్ తర్వాత Elementor కాష్ క్లియరింగ్.
* టెంప్లేట్ టాక్సానమీ సంరక్షణ.
