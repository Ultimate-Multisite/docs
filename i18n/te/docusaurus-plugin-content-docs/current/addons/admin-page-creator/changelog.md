# Admin Page Creator మార్పుల జాబితా

### Version 1.9.0 - Released on 2026-01-18

* Fixed: యాక్టివేట్ చేసేటప్పుడు వచ్చే తీవ్రమైన లోపాలు (Fatal errors) సరిచేయబడ్డాయి.
* Refactored: కంటెంట్ సోర్స్ క్లాస్‌లను సరైన పేరెంట్ కన్‌స్ట్రక్టర్ ఇనిషియలైజేషన్‌తో సింగిల్టన్ ట్రెయిట్‌ను ఉపయోగించేలా పునర్నిర్మించారు.
* Fixed: కంటెంట్ రకం సెలెక్టర్ రెండర్ కాకుండా ఆపు చేసే Vue.js టెంప్లేట్ కంపైలేషన్ లోపాలు సరిచేయబడ్డాయి.
* Fixed: తప్పిపోయిన Vue డేటా ప్రాపర్టీలు (template_id, external_link_url_embedable, external_link_url_checking) సరిచేయబడ్డాయి.
* Fixed: పేజీ లోడ్ అయినప్పుడు TinyMCE ఎడిటర్ సరిగ్గా ఇనిషియలైజ్ కాకపోవడం సరిచేయబడింది.
* Fixed: Composer ద్వారా Divi సపోర్ట్ ఫైల్ ఆటోలోడ్ కాకపోవడం సరిచేయబడింది.
* Fixed: Divi సపోర్ట్ క్లాస్‌లో పాడైన PHP సింటాక్స్ సరిచేయబడింది.
* Improved: JavaScript ఆస్సెట్ ఆర్గనైజేషన్ మరియు మినిఫికేషన్ మెరుగుపరచబడ్డాయి.
* Improved: తాజా WordPress మరియు పేజ్ బిల్డర్ వెర్షన్‌లతో అనుకూలత మెరుగుపరచబడింది.
