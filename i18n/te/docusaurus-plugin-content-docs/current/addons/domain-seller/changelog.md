# Domain Seller మార్పుల వివరాలు

Version 1.1.0 - 2026-05-08 న విడుదల చేయబడింది
- కొత్తగా: ResellerClub, Enom, మరియు OpenSRS రిజిస్ట్రార్ల కోసం DNS రికార్డ్ సృష్టి (add_dns_record) అమలు చేయబడింది
- సరిదిద్దబడింది: డిఫాల్ట్ DNS రికార్డ్స్ పార్సర్ ఇప్పుడు {DOMAIN} మరియు {SITE_URL} టోకెన్‌లను సహిస్తుంది
- సరిదిద్దబడింది: site_url తో ఘర్షణ (collision) రాకుండా డొమైన్ ఎంపిక చెకౌట్ ఫీల్డ్ స్లగ్స్ పేస్నేమ్ చేయబడ్డాయి

* **మెరుగుపరచబడింది:** డొమైన్ ఉత్పత్తి సెట్టింగ్‌లు కోర్ విడ్జెట్ సిస్టమ్ ద్వారా ఉత్పత్తి ఎడిట్ పేజీలో ఇన్లైన్‌లో రెండర్ అవుతాయి

* **మెరుగుపరచబడింది:** TLD ఇన్‌పోర్ట్ విజర్డ్‌ను అన్ని ప్రొవైడర్ల నుండి వన్-క్లిక్ sync కి సరళీకృతం చేసాము

* **మెరుగుపరచబడింది:** TLD ని ఒకే నెట్‌వర్క్ ఆప్షన్‌లో కేంద్రీకృతం చేయడం జరిగింది
