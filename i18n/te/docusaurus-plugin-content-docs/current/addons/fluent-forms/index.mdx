# Fluent Forms {#fluent-forms}

ఈ అడాన్ (add-on) ద్వారా, మీరు Multisite Ultimate లోని ఉత్పత్తులు (products), సభ్యత్వాలు (memberships) లేదా సైట్‌లలోని Fluent Forms పై పరిమితులను (limits) సెట్ చేయవచ్చు.
Fluent Forms అనేది కస్టమ్ పోస్ట్ రకాలు (custom post types) కాకుండా కస్టమ్ టేబుల్స్ (custom tables) ఉపయోగిస్తుంది. అందుకే, సాధారణ పరిమితి ఎంపికలు (normal limiting options) Fluent Forms తో పనిచేయవు.

## ఫీచర్లు {#features}

- Fluent Forms లేదా Conversational Fluent Forms కోసం పరిమితులను జోడించడం.
- వినియోగదారుడు పరిమితిని చేరుకున్నప్పుడు, తమ ప్లాన్‌ను అప్‌గ్రేడ్ చేసుకోవాలని గుర్తుచేసే ఒక స్నేహపూర్వక సందేశం వారికి కనిపిస్తుంది.
