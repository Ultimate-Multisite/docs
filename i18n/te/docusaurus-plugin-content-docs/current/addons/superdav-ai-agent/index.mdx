# Superdav AI Agent

Superdav AI Agent అనేది అభివృద్ధి చెందిన WordPress AI సహాయకుడు. ఇది Gratis AI Agent‌ను థీమ్-అవగాహన నైపుణ్యాలు, మెరుగైన ఇంటర్నెట్ శోధన సామర్థ్యాలు, సైట్ నిర్మాణ లక్షణాలు, ఐచ్చిక ఆటోమేషన్ అనుసంధానాలతో విస్తరిస్తుంది. వెర్షన్ 1.18.0 Google Calendar చదివే సాధనాలు, సంప్రదింపు మరియు హాజరుదారు మ్యాపింగ్, మానవ ఆమోద గేట్లు, రిమైండర్ పునరావృత నివారణ, TextBee SMS నోటిఫికేషన్‌లు, నిర్వహిత Superdav సేవ ప్రొవిజనింగ్, మరియు విశ్వసనీయ డెవలపర్ పనిప్రవాహాల కోసం విడిగా పంపిణీ చేసే Advanced సహచర ప్యాకేజీని జోడిస్తుంది.

## ముఖ్య లక్షణాలు {#key-features}

- **థీమ్-అవగాహన నైపుణ్యాలు** — మీ క్రియాశీల WordPress థీమ్ మరియు ఇన్‌స్టాల్ చేసిన ప్లగిన్‌ల ఆధారంగా మార్గదర్శకతను, సామర్థ్యాలను స్వయంచాలకంగా అనుసరిస్తుంది
- **బహుళ ఇంటర్నెట్ శోధన ప్రదాతలు** — మరింత సమృద్ధిగల శోధన ఫలితాలు మరియు అధునాతన పరిశోధన సామర్థ్యాల కోసం Brave Search మరియు Tavily మధ్య ఎంచుకోండి
- **సైట్ నిర్మాణ సంప్రదింపు ఫారమ్‌లు** — మీరు ఇన్‌స్టాల్ చేసిన ఫారమ్ ప్లగిన్‌ను ఉపయోగించి చాట్ ఇంటర్‌ఫేస్ నుంచే పూర్తిగా పనిచేసే సంప్రదింపు ఫారమ్‌లను సృష్టించండి
- **మెరుగైన సైట్ నిర్మాణం** — మెరుగైన ప్లగిన్ గుర్తింపు మరియు లోప పునరుద్ధరణతో మెరుగుపరచిన బహుళ-దశల సైట్ నిర్మాణ ప్రణాళికలు
- **అంతర్నిర్మిత థీమ్ నైపుణ్యాలు** — బ్లాక్ థీమ్‌లు, క్లాసిక్ థీమ్‌లు, Kadence Blocks, మరియు Kadence Theme కోసం ప్రత్యేక మార్గదర్శకత
- **మెరుగైన WooCommerce అనుసంధానం** — మరింత విశ్వసనీయత మరియు అనుకూలత కోసం స్థానిక WooCommerce APIలను ఉపయోగించేలా పునర్నిర్మించబడింది
- **స్వయంచాలక ప్రదాత గుర్తింపు** — అందుబాటులో ఉన్న ప్లగిన్‌లు మరియు థీమ్‌లు క్రియాశీలం చేసినప్పుడు లేదా నిలిపివేసినప్పుడు వాటిని స్వయంచాలకంగా గుర్తించి తాజాకరిస్తుంది
- **అనుకూల లోగో సృష్టి** — స్వయంచాలక నేమ్‌స్పేస్-సురక్షిత శుద్ధితో అనుకూల SVG లోగోలను సృష్టించి ఎంబెడ్ చేయండి
- **ప్యాలెట్ కాంట్రాస్ట్ ధృవీకరణ** — మీ థీమ్‌కు వర్తింపజేయడానికి ముందు WCAG ప్రాప్యత అనుసరణ కోసం రంగు జంటలను తనిఖీ చేయండి
- **ఫోటో-మెరుగుపరచిన ఆవిష్కరణ** — రూపకల్పన దిశపై ప్రభావం చూపడానికి Theme Builder ఆవిష్కరణ ఇంటర్వ్యూలో ఫోటోలను అప్‌లోడ్ చేయండి
- **ఆతిథ్య మెనూలు** — రెస్టారెంట్‌లు, కేఫేలు, మరియు ఆతిథ్య వ్యాపారాల కోసం నిర్మిత ఆహార మరియు పానీయాల మెను పేజీలను సృష్టించండి
- **డెస్క్‌టాప్ మరియు మొబైల్ ప్రివ్యూలు** — రూపకల్పన-దిశ ఎంపిక సమయంలో మీ రూపకల్పనను డెస్క్‌టాప్ మరియు మొబైల్ పరికరాల్లో ప్రివ్యూ చేయండి
- **టియర్ 1 సైట్ స్క్రేపింగ్** — పోటీదారుల మరియు ప్రేరణ వెబ్‌సైట్‌లను విశ్లేషించడానికి sd-ai-agent/site-scrape ఇప్పుడు డిఫాల్ట్‌గా అందుబాటులో ఉంది
- **Google Calendar చదివే సాధనాలు** — రాబోయే షెడ్యూల్ సందర్భం గురించి ఆటోమేషన్‌లు తర్కించేందుకు కాన్ఫిగర్ చేసిన క్యాలెండర్‌లు మరియు ఈవెంట్‌లను చదవండి
- **సంప్రదింపు మరియు హాజరుదారు మ్యాపింగ్** — అనుసరణ పనిని సృష్టించే ముందు క్యాలెండర్ పాల్గొనేవారిని తెలిసిన సైట్ వినియోగదారులు మరియు సంప్రదింపులతో సరిపోల్చండి
- **మానవ ఆమోద గేట్లు** — సందేశాలు పంపడం, పనిని కొనసాగించడం, లేదా మార్పులు చేయడం ముందు సమీక్ష కోసం సున్నితమైన ఆటోమేషన్‌లను విరామం చేయండి
- **రిమైండర్ రికార్డులు మరియు పునరావృత నివారణ** — ఇప్పటికే పంపిన రిమైండర్‌లను ట్రాక్ చేయండి, తద్వారా పునరావృత ఆటోమేషన్ రన్‌లు అదే గ్రహీతకు రెండుసార్లు తెలియజేయవు
- **TextBee SMS నోటిఫికేషన్‌లు** — SMS ప్రదాత ప్రారంభించబడినప్పుడు TextBee ద్వారా కాన్ఫిగర్ చేసిన టెక్స్ట్-సందేశ నోటిఫికేషన్‌లను పంపండి
- **నిర్వహిత Superdav సేవ ప్రొవిజనింగ్** — మాన్యువల్ ఎండ్‌పాయింట్ కాన్ఫిగరేషన్‌కు బదులుగా స్వయంచాలక సెటప్‌తో మద్దతు ఉన్న సైట్‌లను హోస్ట్ చేసిన Superdav ఎండ్‌పాయింట్‌లకు కనెక్ట్ చేయండి
- **Advanced సహచర ప్యాకేజీ** — ఫైల్‌సిస్టమ్, డేటాబేస్, WP-CLI, REST డిస్పాచర్, ప్లగిన్ బిల్డర్, git స్నాప్‌షాట్, వినియోగదారు-నిర్వహణ, మరియు బెంచ్‌మార్క్ పనిప్రవాహాల కోసం విడిగా పంపిణీ చేసే విశ్వసనీయ-డెవలపర్ సాధనాలు

## అవసరాలు {#requirements}

- WordPress 6.0 లేదా అంతకంటే ఎక్కువ
- PHP 8.1 లేదా అంతకంటే ఎక్కువ
- WordPress Multisite (నెట్‌వర్క్-క్రియాశీలం చేసినది) లేదా ఏక-సైట్ ఇన్‌స్టాలేషన్
- క్రియాశీల AI ప్రదాత కనెక్టర్ ప్లగిన్ (Anthropic, OpenAI, లేదా అనుకూలమైనది)
- Gratis AI Agent v1.9.0 లేదా అంతకంటే ఎక్కువ (Superdav AI Agent, Gratis AI Agent‌ను విస్తరిస్తుంది)

## ఇన్‌స్టాలేషన్ {#installation}

1. ప్లగిన్ ఫైల్‌లను `/wp-content/plugins/superdav-ai-agent`కు అప్‌లోడ్ చేయండి
2. WordPressలో **ప్లగిన్‌లు** స్క్రీన్ ద్వారా ప్లగిన్‌ను క్రియాశీలం చేయండి
3. Gratis AI Agent కూడా క్రియాశీలం చేయబడిందని నిర్ధారించండి
4. **Gratis AI Agent → సెట్టింగ్‌లు**కు వెళ్లి, అవసరమైన విధంగా మీ ఇంటర్నెట్ శోధన ప్రదాత, నిర్వహిత Superdav సేవ కనెక్షన్, Google Calendar, లేదా TextBee ఆధారాలను కాన్ఫిగర్ చేయండి

WordPress.org ప్యాకేజీలో ప్రధాన Superdav AI Agent లక్షణాలు ఉంటాయి. విశ్వసనీయ డెవలపర్ సాధనాలు స్పష్టంగా ఇన్‌స్టాల్ చేసి అధీకరించిన వాతావరణాల కోసం Advanced సహచర ప్యాకేజీని విడిగా నిర్మించి పంపిణీ చేస్తారు.

## త్వరిత ప్రారంభం {#quick-start}

క్రియాశీలం చేసిన తర్వాత, Superdav AI Agent మీ Gratis AI Agent ఇన్‌స్టాలేషన్‌ను థీమ్-అవగాహన నైపుణ్యాలు మరియు అదనపు సామర్థ్యాలతో స్వయంచాలకంగా మెరుగుపరుస్తుంది. WordPress నిర్వాహక సైడ్‌బార్‌లో **Gratis AI Agent** ప్యానెల్‌ను తెరిచి కొత్త లక్షణాలను ఉపయోగించడం ప్రారంభించండి.

**ఉదాహరణ సూచనలు:**

```
సంప్రదింపు పేజీకి ఒక సంప్రదింపు ఫారమ్‌ను జోడించండి.
```

```
తాజా WordPress భద్రత ఉత్తమ పద్ధతుల కోసం వెబ్‌లో శోధించండి.
```

```
క్రియాశీల థీమ్ యొక్క బ్లాక్ నమూనాలను ఉపయోగించి హీరో విభాగాన్ని సృష్టించండి.
```

```
రేపటి కాన్ఫిగర్ చేసిన Google Calendar ఈవెంట్‌లను సమీక్షించి, హాజరుదారుల కోసం రిమైండర్ ముసాయిదాలను తయారు చేయండి.
```

```
నేను సందేశాన్ని ఆమోదించిన తర్వాత SMS రిమైండర్ పంపండి.
```

## థీమ్-అవగాహన నైపుణ్యాలు {#theme-aware-skills}

Superdav AI Agent మీ క్రియాశీల థీమ్ మరియు ఇన్‌స్టాల్ చేసిన ప్లగిన్‌లను స్వయంచాలకంగా గుర్తించి, తగిన థీమ్-అవగాహన నైపుణ్యాన్ని సహాయకుడి సందర్భంలో లోడ్ చేస్తుంది. అందుబాటులో ఉన్న నైపుణ్యాలు ఇవి:

| నైపుణ్యం | వర్తించేది |
|---|---|
| **బ్లాక్ థీమ్‌లు** | WordPress బ్లాక్-ఆధారిత (పూర్తి సైట్ ఎడిటింగ్) నిర్మాణాన్ని ఉపయోగించే థీమ్‌లు |
| **క్లాసిక్ థీమ్‌లు** | PHP టెంప్లేట్‌లు మరియు క్లాసిక్ ఎడిటర్‌ను ఉపయోగించే సంప్రదాయ WordPress థీమ్‌లు |
| **Kadence Blocks** | Kadence Blocks ప్లగిన్ క్రియాశీలంగా ఉన్న సైట్‌లు |
| **Kadence Theme** | క్రియాశీల థీమ్‌గా Kadence థీమ్ ఉన్న సైట్‌లు |

ప్రతి నైపుణ్యంపై విపులమైన మార్గదర్శకత కోసం [థీమ్-అవగాహన నైపుణ్యాలు](../../user-guide/configuration/theme-aware-skills) చూడండి.

## ఇంటర్నెట్ శోధన ప్రదాతలు {#internet-search-providers}

Superdav AI Agent v1.10.0 ఇంటర్నెట్ శోధన ప్రశ్నలకు Brave Searchకు ప్రత్యామ్నాయంగా **Tavily**ని జోడిస్తుంది. రెండు ప్రదాతలు నిజ-సమయ శోధన ఫలితాలను అందిస్తాయి మరియు **Gratis AI Agent → సెట్టింగ్‌లు → అధునాతన**లో కాన్ఫిగర్ చేయవచ్చు.

| ప్రొవైడర్ | ఫీచర్లు | సెటప్ |
|---|---|---|
| **Brave Search** | వేగవంతమైన, గోప్యతపై దృష్టి పెట్టిన శోధన ఫలితాలు | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | మరింత సమృద్ధిగా ఉన్న శోధన ఫలితాలు, అధునాతన పరిశోధన సామర్థ్యాలు | [Tavily API](https://tavily.com/) |

కాన్ఫిగరేషన్ సూచనల కోసం [ఇంటర్నెట్ శోధన](../../user-guide/configuration/internet-search) చూడండి.

## క్యాలెండర్, ఆమోదం, మరియు నోటిఫికేషన్ వర్క్‌ఫ్లోలు {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 కంటెంట్ మరియు సైట్ నిర్మాణ పనులకు మించి ఆటోమేషన్ వర్క్‌ఫ్లోలను విస్తరిస్తుంది:

- **Google Calendar టూల్స్** క్యాలెండర్ క్రెడెన్షియల్స్ అందుబాటులో ఉన్నప్పుడు కాన్ఫిగర్ చేసిన క్యాలెండర్లు మరియు ఈవెంట్ వివరాలను ఏజెంట్ చదవడానికి అనుమతిస్తాయి.
- **కాంటాక్ట్ మరియు హాజరుదారుల సహాయకాలు** రిమైండర్లు లేదా ఫాలో-అప్ పనులు సిద్ధం చేయడానికి ముందు ఈవెంట్ పాల్గొనేవారిని WordPress వినియోగదారులు లేదా తెలిసిన కాంటాక్ట్‌లకు మ్యాప్ చేస్తాయి.
- **మానవ ఆమోద గేట్లు** ప్రతిపాదిత చర్యను అధీకృత వినియోగదారు సమీక్షించే వరకు ఆటోమేషన్‌ను నిలిపివేయడానికి అనుమతిస్తాయి.
- **రిమైండర్ రికార్డులు** ఆటోమేషన్ మళ్లీ ప్రయత్నించినప్పుడు లేదా పదేపదే షెడ్యూల్ చేసినప్పుడు నకిలీ క్యాలెండర్ లేదా నోటిఫికేషన్ రిమైండర్లను నివారిస్తాయి.
- **TextBee SMS మద్దతు** TextBee ప్రొవైడర్ కాన్ఫిగర్ చేసినప్పుడు టెక్స్ట్-మెసేజ్ నోటిఫికేషన్‌లను ప్రారంభిస్తుంది.

సున్నితమైన సందేశాలు పంపే ముందు ఇంకా మానవ సమీక్ష అవసరమైన షెడ్యూల్-అవగాహన సహాయం, హాజరుదారుల ఫాలో-అప్, మరియు రిమైండర్ వర్క్‌ఫ్లోల కోసం ఈ టూల్స్‌ను ఉపయోగించండి.

## సైట్ బిల్డర్ కాంటాక్ట్ ఫారమ్‌లు {#site-builder-contact-forms}

సైట్ బిల్డర్ ఏజెంట్ ఇప్పుడు చాట్ ఇంటర్‌ఫేస్ నుంచే నేరుగా కాంటాక్ట్ ఫారమ్‌లను సృష్టించగలదు. కాంటాక్ట్ ఫారమ్ సృష్టించమని సైట్ బిల్డర్‌ను అడగండి. అది మీ ఇన్‌స్టాల్ చేసిన ఫారమ్ pluginను (Contact Form 7, WPForms, Fluent Forms, లేదా Gravity Forms) స్వయంచాలకంగా గుర్తించి, ఉపయోగించడానికి సిద్ధంగా ఉన్న ఫారమ్‌ను రూపొందిస్తుంది.

వివరమైన మార్గదర్శకత్వం కోసం [సైట్ బిల్డర్ కాంటాక్ట్ ఫారమ్‌లు](../../user-guide/administration/site-builder-contact-forms) చూడండి.

## Theme Builder ఫీచర్లు {#theme-builder-features}

Superdav AI Agent v1.16.0లోని Theme Builder శక్తివంతమైన కొత్త సామర్థ్యాలను కలిగి ఉంది:

### ఫోటో అప్‌లోడ్‌తో డిస్కవరీ ఇంటర్వ్యూ {#discovery-interview-with-photo-upload}

Theme Builder డిస్కవరీ ఇంటర్వ్యూలో ఇప్పుడు మీరు దృశ్య ప్రేరణను అందించగల ఫోటో అప్‌లోడ్ దశ ఉంది. మీ డిజైన్ దిశను Theme Builder అర్థం చేసుకోవడానికి మీ బ్రాండ్, పోటీదారుల సైట్‌లు, లేదా డిజైన్ ప్రేరణకు సంబంధించిన ఫోటోలను అప్‌లోడ్ చేయండి.

వివరమైన మార్గదర్శకత్వం కోసం [డిస్కవరీ ఇంటర్వ్యూ](./theme-builder/discovery-interview.md) చూడండి.

### డెస్క్‌టాప్ మరియు మొబైల్ ప్రివ్యూలతో డిజైన్ దిశ {#design-direction-with-desktop-and-mobile-previews}

Theme Builder మీ పూర్తి themeను రూపొందించే ముందు మీ డిజైన్‌ను డెస్క్‌టాప్ మరియు మొబైల్ పరికరాల్లో సమీక్షించండి. అనేక రెస్పాన్సివ్ బ్రేక్‌పాయింట్లలో రియల్-టైమ్ ప్రివ్యూలను చూడండి.

వివరమైన మార్గదర్శకత్వం కోసం [డిజైన్ దిశ](./theme-builder/design-direction.md) చూడండి.

### హాస్పిటాలిటీ మెనూలు {#hospitality-menus}

Theme Builder ఇప్పుడు రెస్టారెంట్లు, కేఫేలు, బార్‌లు, మరియు ఇతర హాస్పిటాలిటీ వ్యాపారాల కోసం నిర్మిత ఆహార మరియు పానీయాల మెనూ పేజీలను రూపొందించగలదు. మెనూలలో ధరలు, వివరణలు, ఆహార సంబంధిత సమాచారం, మరియు అలర్జెన్ హెచ్చరికలు ఉంటాయి.

వివరమైన మార్గదర్శకత్వం కోసం [హాస్పిటాలిటీ మెనూలు](./theme-builder/hospitality-menus.md) చూడండి.

## కాన్ఫిగరేషన్ {#configuration}

అన్ని సెట్టింగ్‌లు WordPress అడ్మిన్‌లో **Gratis AI Agent → Settings** కింద ఉంటాయి. Superdav AI Agent ఈ క్రింది కాన్ఫిగరేషన్ ఎంపికలను జోడిస్తుంది:

| సెట్టింగ్ | వివరణ |
|---|---|
| **ఇంటర్నెట్ శోధన ప్రొవైడర్** | Brave Search మరియు Tavily మధ్య ఎంచుకోండి (ఐచ్చికం) |
| **Tavily API Key** | Tavily ఇంటర్నెట్ శోధన కోసం API key (Tavily ఎంచుకుంటే) |
| **Managed Superdav Service** | ఆటోమేటిక్ endpoint ప్రొవిజనింగ్‌తో మద్దతు ఉన్న సైట్‌లను హోస్టెడ్ Superdav సేవకు కనెక్ట్ చేయండి |
| **Google Calendar క్రెడెన్షియల్స్** | ఏజెంట్ క్యాలెండర్లు మరియు ఈవెంట్‌లను చదవగలిగేలా క్యాలెండర్ యాక్సెస్‌ను కాన్ఫిగర్ చేయండి |
| **TextBee SMS ప్రొవైడర్** | ఆమోదించిన SMS నోటిఫికేషన్ వర్క్‌ఫ్లోల కోసం TextBee క్రెడెన్షియల్స్‌ను కాన్ఫిగర్ చేయండి |

## అధునాతన కంపానియన్ ప్యాకేజీ {#advanced-companion-package}

Superdav AI Agent v1.18.0 విశ్వసనీయ డెవలపర్ వర్క్‌ఫ్లోల కోసం వేరుగా పంపిణీ చేసే అధునాతన కంపానియన్ ప్యాకేజీని జోడిస్తుంది. ఈ సామర్థ్యాలు WordPress.org కోర్ ప్యాకేజీలో కలిపి ఉండవు. డెవలపర్ టూలింగ్‌ను యాక్సెస్ చేయడానికి నిర్వాహకులు ఏజెంట్‌ను స్పష్టంగా విశ్వసించే సైట్‌లలో మాత్రమే కంపానియన్ ప్యాకేజీని ఇన్‌స్టాల్ చేయండి.

అధునాతన టూల్స్‌లో ఫైల్‌సిస్టమ్ ఆపరేషన్‌లు, డేటాబేస్ యాక్సెస్, WP-CLI, REST డిస్పాచర్ వర్క్‌ఫ్లోలు, plugin బిల్డర్ యుటిలిటీలు, git స్నాప్‌షాట్‌లు, వినియోగదారు-నిర్వహణ సహాయకాలు, మరియు బెంచ్‌మార్క్ టూలింగ్ ఉంటాయి. ఈ టూల్స్‌ను విశ్వసనీయ నిర్వాహకులకు మాత్రమే పరిమితం చేయండి. ప్రొడక్షన్ సైట్‌లలో వీటిని ప్రారంభించే ముందు ఆమోద-గేట్ కాన్ఫిగరేషన్‌ను సమీక్షించండి.
