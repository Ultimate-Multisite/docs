# టూల్స్ మరియు కాన్ఫిగరేషన్

Theme Builder మీ WordPress సైట్‌ను విశ్లేషించడానికి, డిజైన్ చేయడానికి మరియు నిర్మించడానికి అనేక టూల్స్‌ను ఉపయోగిస్తుంది. వెర్షన్ 1.16.0 లో, **sd-ai-agent/site-scrape అనేది ఇప్పుడు Tier 1 టూల్**గా మారింది, కాబట్టి ఇది Theme Builderలో డిఫాల్ట్‌గా అందుబాటులో ఉంటుంది.

## అందుబాటులో ఉన్న టూల్స్

### Tier 1 టూల్స్ (ఎల్లప్పుడూ అందుబాటులో ఉంటాయి)

Tier 1 టూల్స్ అదనపు కాన్ఫిగరేషన్ అవసరం లేకుండా Theme Builderలో డిఫాల్ట్‌గా అందుబాటులో ఉంటాయి.

#### sd-ai-agent/site-scrape

**ఉద్దేశ్యం:** డిజైన్ ప్రేరణ, కంటెంట్ నిర్మాణం మరియు ఫంక్షనాలిటీని సంగ్రహించడానికి (extract) ఇప్పటికే ఉన్న వెబ్‌సైట్‌లను విశ్లేషించడం.

**సామర్థ్యాలు:**

- **వెబ్‌సైట్ విశ్లేషణ** — పోటీదారుల లేదా ప్రేరణ పొందిన వెబ్‌సైట్‌లను స్క్రాప్ చేసి విశ్లేషించడం
- **డిజైన్ సంగ్రహణ** — రంగులు, ఫాంట్‌లు మరియు లేఅవుట్ నమూనాలను గుర్తించడం
- **కంటెంట్ నిర్మాణం** — పేజీ ఆర్గనైజేషన్ మరియు హైరార్కీని అర్థం చేసుకోవడం
- **ఫీచర్ డిటెక్షన్** — ప్లగిన్‌లు మరియు ఫంక్షనాలిటీని గుర్తించడం
- **పనితీరు విశ్లేషణ** — పేజీ వేగం మరియు ఆప్టిమైజేషన్‌ను తనిఖీ చేయడం
- **SEO విశ్లేషణ** — మెటా ట్యాగ్‌లు మరియు స్ట్రక్చర్డ్ డేటాను సమీక్షించడం

**ఉపయోగం:**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**ఇది ఏమి సంగ్రహిస్తుంది:**

- రంగు పాలెట్ మరియు రంగు వినియోగం
- టైపోగ్రఫీ (ఫాంట్‌లు మరియు సైజులు)
- లేఅవుట్ నిర్మాణం మరియు స్పేసింగ్
- నావిగేషన్ నమూనాలు
- కంటెంట్ ఆర్గనైజేషన్
- చిత్రాలు మరియు మీడియా వినియోగం
- ఇంటరాక్టివ్ ఎలిమెంట్స్
- మొబైల్ రెస్పాన్సివ్‌నెస్

**పరిమితులు:**

- పాస్‌వర్డ్‌తో రక్షించబడిన సైట్‌లను స్క్రాప్ చేయలేదు
- robots.txt మరియు సైట్ పాలసీలను గౌరవిస్తుంది
- డైనమిక్ కంటెంట్‌ను సంగ్రహించకపోవచ్చు
- JavaScript-heavy సైట్‌లకు విశ్లేషణ పరిమితంగా ఉండవచ్చు
- పెద్ద సైట్‌లను విశ్లేషించడానికి ఎక్కువ సమయం పట్టవచ్చు

### Tier 2 టూల్స్ (ఐచ్ఛికం)

Tier 2 టూల్స్ Theme Builder సెట్టింగ్‌లలో స్పష్టంగా ఎనేబుల్ చేసినప్పుడు అందుబాటులో ఉంటాయి.

#### Advanced Analytics

వివరణాత్మక పనితీరు మెట్రిక్‌లను అందిస్తుంది:

- పేజీ లోడ్ సమయాలు
- Core Web Vitals
- SEO స్కోర్
- యాక్సెసిబిలిటీ స్కోర్
- ఉత్తమ పద్ధతుల స్కోర్

#### Content Optimizer

వీటి కోసం విశ్లేషించి మెరుగుదలలను సూచిస్తుంది:

- చదవడానికి సులభతనం (Readability)
- SEO ఆప్టిమైజేషన్
- కీవర్డ్ వినియోగం
- కంటెంట్ నిర్మాణం
- కాల్-టు-యాక్షన్ ప్లేస్‌మెంట్

### Tier 3 టూల్స్ (ప్రీమియం)

Tier 3 టూల్స్ అదనపు కాన్ఫిగరేషన్ లేదా ప్రీమియం ఫీచర్‌లు అవసరం.

#### AI Content Generator

వీటి కోసం కంటెంట్‌ను రూపొందిస్తుంది:

- ఉత్పత్తి వివరణలు
- సర్వీస్ పేజీలు
- బ్లాగ్ పోస్ట్‌లు
- మెటా వివరణలు
- కాల్-టు-యాక్షన్ కాపీ

#### Advanced Design Tools

అధునాతన డిజైన్ సామర్థ్యాలను అందిస్తుంది:

- కస్టమ్ CSS జనరేషన్
- యానిమేషన్ సృష్టి
- ఇంటరాక్టివ్ ఎలిమెంట్ డిజైన్
- అడ్వాన్స్‌డ్ కలర్ థియరీ
- టైపోగ్రఫీ ఆప్టిమైజేషన్

## టూల్ కాన్ఫిగరేషన్

### టూల్స్‌ను ఎనేబుల్ చేయడం

Theme Builderలో అదనపు టూల్స్‌ను ఎనేబుల్ చేయడానికి:

1. **Gratis AI Agent → Settings**కి నావిగేట్ చేయండి
2. **Theme Builder → Tools**కి వెళ్లండి
3. అవసరాన్ని బట్టి టూల్స్‌ను ఆన్/ఆఫ్ చేయండి
4. సెట్టింగ్‌లను సేవ్ చేయండి

### టూల్ పర్మిషన్స్

కొన్ని టూల్స్‌కు పర్మిషన్స్ అవసరం:

- **Site scraping** — ఇంటర్నెట్ యాక్సెస్ అవసరం
- **Analytics** — Google Analytics ఇంటిగ్రేషన్ అవసరం
- **Content generation** — API కీలు అవసరం
- **Advanced features** — ప్రీమియం సబ్‌స్క్రిప్షన్ అవసరం కావచ్చు

### API కీలు మరియు క్రెడెన్షియల్స్

అవి అవసరమైన టూల్స్‌ కోసం API కీలను కాన్ఫిగర్ చేయండి:

1. **Gratis AI Agent → Settings → API Keys**కి వెళ్లండి
2. ప్రతి టూల్ కోసం క్రెడెన్షియల్స్‌ను నమోదు చేయండి
3. కనెక్షన్‌ను పరీక్షించండి
4. సురక్షితంగా సేవ్ చేయండి

## sd-ai-agent/site-scrape ఉపయోగించడం

### ప్రాథమిక ఉపయోగం

ఒక వెబ్‌సైట్‌ను విశ్లేషించమని Theme Builderని అడగండి:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### నిర్దిష్ట విశ్లేషణ

నిర్దిష్ట రకాల విశ్లేషణను అభ్యర్థించండి:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### పోలిక విశ్లేషణ

అనేక సైట్‌లను పోల్చండి:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## టూల్ పరిమితులు మరియు పరిశీలనలు

### రేట్ లిమిటింగ్ (Rate Limiting)

- సర్వర్ ఓవర్‌లోడ్‌ను నివారించడానికి స్క్రాపింగ్‌కు రేట్ లిమిట్ ఉంటుంది
- ఒకే సైట్‌కు బహుళ అభ్యర్థనలు పరిమితం చేయబడవచ్చు (throttled)
- పెద్ద సైట్‌లను విశ్లేషించడానికి ఎక్కువ సమయం పట్టవచ్చు

### కంటెంట్ ఖచ్చితత్వం (Content Accuracy)

- డైనమిక్ కంటెంట్‌ను పూర్తిగా సంగ్రహించకపోవచ్చు
- JavaScript ద్వారా రెండర్ చేయబడిన కంటెంట్ అసంపూర్ణంగా ఉండవచ్చు
- రియల్-టైమ్ డేటా ప్రతిబింబించకపోవచ్చు
- కొంత కంటెంట్ పేవాల్స్ వెనుక ఉండవచ్చు

### చట్టపరమైన మరియు నైతిక పరిశీలనలు

- robots.txt మరియు సైట్ పాలసీలను గౌరవించండి
- పునరుత్పత్తి కోసం కాపీరైట్ చేయబడిన కంటెంట్‌ను స్క్రాప్ చేయవద్దు
- విశ్లేషణను ప్రేరణ కోసం ఉపయోగించండి, కాపీ చేయడానికి కాదు
- సంగ్రహించిన కంటెంట్‌ను ఉపయోగించడానికి మీకు హక్కులు ఉన్నాయని నిర్ధారించుకోండి
- విశ్లేషించిన సైట్‌ల సేవా నిబంధనలను పాటించండి

### పనితీరు ప్రభావం (Performance Impact)

- పెద్ద సైట్‌లను స్క్రాప్ చేయడానికి సమయం పట్టవచ్చు
- బహుళ ఏకకాల స్క్రాప్‌లు నెమ్మదిగా ఉండవచ్చు
- నెట్‌వర్క్ కనెక్టివిటీ వేగాన్ని ప్రభావితం చేస్తుంది
- పెద్ద డేటాసెట్‌లకు ఎక్కువ ప్రాసెసింగ్ అవసరం కావచ్చు

## ఉత్తమ పద్ధతులు (Best Practices)

### సైట్ విశ్లేషణ ఉపయోగించడం

1. **అనేక సైట్‌లను విశ్లేషించండి** — అనేక మూలాల నుండి ప్రేరణ పొందండి
2. **నిర్మాణంపై దృష్టి పెట్టండి** — లేఅవుట్ మరియు ఆర్గనైజేషన్ నమూనాలను నేర్చుకోండి
3. **రంగులను సంగ్రహించండి** — రంగు పాలెట్‌లను ప్రారంభ బిందువులుగా ఉపయోగించండి
4. **టైపోగ్రఫీని అధ్యయనం చేయండి** — మీకు నచ్చిన ఫాంట్ కలయికలను గుర్తించండి
5. **నావిగేషన్‌ను సమీక్షించండి** — మెనూ ఆర్గనైజేషన్‌ను అర్థం చేసుకోండి

### నైతిక స్క్రాపింగ్

1. **robots.txt ని గౌరవించండి** — సైట్ మార్గదర్శకాలను పాటించండి
2. **కంటెంట్‌ను కాపీ చేయవద్దు** — విశ్లేషణను ప్రేరణ కోసం మాత్రమే ఉపయోగించండి
3. **హక్కులను ధృవీకరించండి** — సంగ్రహించిన కంటెంట్‌ను ఉపయోగించవచ్చో లేదో నిర్ధారించుకోండి
4. **మూలాలను పేర్కొనండి** — ప్రేరణ పొందిన మూలాలకు క్రెడిట్ ఇవ్వండి
5. **నిబంధనలను పాటించండి** — సైట్ సేవా నిబంధనలకు కట్టుబడి ఉండండి

### టూల్ సామర్థ్యాన్ని పెంచడం

1. **నిర్దిష్టంగా ఉండండి** — నిర్దిష్ట రకాల విశ్లేషణను అభ్యర్థించండి
2. **సందర్భాన్ని అందించండి** — మీ సైట్ యొక్క ఉద్దేశ్యాన్ని వివరించండి
3. **అంచనాలను సెట్ చేయండి** — మీ డిజైన్ లక్ష్యాలను వివరించండి
4. **పునరావృతం చేయండి** — ఫలితాల ఆధారంగా మెరుగుపరచండి
5. **టూల్స్‌ను కలపండి** — సమగ్ర విశ్లేషణ కోసం బహుళ టూల్స్‌ను ఉపయోగించండి

### సైట్ స్క్రాప్ కాకపోతే

- సైట్ పబ్లిక్‌గా అందుబాటులో ఉందో లేదో తనిఖీ చేయండి
- స్క్రాపింగ్‌కు robots.txt అనుమతిస్తుందో లేదో ధృవీకరించండి
- వేరే సైట్‌ను ప్రయత్నించండి
- ఇంటర్నెట్ కనెక్షన్‌ను తనిఖీ చేయండి
- సమస్య కొనసాగితే సపోర్ట్‌ను సంప్రదించండి

### విశ్లేషణ అసంపూర్ణంగా ఉంటే

- సైట్‌లో డైనమిక్ కంటెంట్ ఉండవచ్చు
- JavaScript పూర్తిగా రెండర్ చేయకపోవచ్చు
- పెద్ద సైట్‌లు టైమౌట్ కావచ్చు
- బదులుగా నిర్దిష్ట పేజీలను విశ్లేషించడానికి ప్రయత్నించండి
- నిర్దిష్ట విశ్లేషణ రకాలను అభ్యర్థించండి

### నెమ్మదిగా పనితీరు

- పెద్ద సైట్‌లను విశ్లేషించడానికి ఎక్కువ సమయం పడుతుంది
- బహుళ ఏకకాల స్క్రాప్‌లు నెమ్మదిగా ఉంటాయి
- నెట్‌వర్క్ కనెక్టివిటీ వేగాన్ని ప్రభావితం చేస్తుంది
- రద్దీ లేని సమయాల్లో ప్రయత్నించండి
- ముందుగా చిన్న విభాగాలను విశ్లేషించండి
