# సాధనాలు మరియు కాన్ఫిగరేషన్

Theme Builder మీ WordPress సైట్‌ను విశ్లేషించడానికి, డిజైన్ చేయడానికి, నిర్మించడానికి కొన్ని సాధనాలను ఉపయోగిస్తుంది. వెర్షన్ 1.16.0లో, **sd-ai-agent/site-scrape ఇప్పుడు Tier 1 సాధనం**, కాబట్టి ఇది Theme Builderలో డిఫాల్ట్‌గా అందుబాటులో ఉంటుంది. Superdav AI Agent v1.18.0 నమ్మకమైన డెవలపర్ పనిప్రవాహాల కోసం విడిగా పంపిణీ చేసే Advanced సహచర ప్యాకేజ్‌ను కూడా పరిచయం చేస్తుంది; ఆ సాధనాలు WordPress.org కోర్ ప్యాకేజ్‌లో భాగం కావు, వాటిని విడిగా ఇన్‌స్టాల్ చేసి అనుమతించాలి.

## అందుబాటులో ఉన్న సాధనాలు

### Tier 1 సాధనాలు (ఎల్లప్పుడూ అందుబాటులో)

Tier 1 సాధనాలు అదనపు కాన్ఫిగరేషన్ లేకుండా Theme Builderలో డిఫాల్ట్‌గా అందుబాటులో ఉంటాయి.

#### sd-ai-agent/site-scrape

**ఉద్దేశ్యం:** డిజైన్ ప్రేరణ, కంటెంట్ నిర్మాణం, కార్యాచరణను పొందడానికి ప్రస్తుత వెబ్‌సైట్‌లను విశ్లేషించడం.

**సామర్థ్యాలు:**

- **వెబ్‌సైట్ విశ్లేషణ** — పోటీదారుల లేదా ప్రేరణాత్మక వెబ్‌సైట్‌లను స్క్రేప్ చేసి విశ్లేషించడం
- **డిజైన్ వెలికితీత** — రంగులు, ఫాంట్లు, లేఅవుట్ నమూనాలను గుర్తించడం
- **కంటెంట్ నిర్మాణం** — పేజీ అమరిక మరియు శ్రేణిని అర్థం చేసుకోవడం
- **ఫీచర్ గుర్తింపు** — ప్లగిన్‌లు మరియు కార్యాచరణను గుర్తించడం
- **పనితీరు విశ్లేషణ** — పేజీ వేగం మరియు ఆప్టిమైజేషన్‌ను తనిఖీ చేయడం
- **SEO విశ్లేషణ** — మెటా ట్యాగ్‌లు మరియు నిర్మిత డేటాను సమీక్షించడం

**వినియోగం:**

```
example.com డిజైన్‌ను విశ్లేషించి, నా సైట్‌కు ప్రేరణగా ఉపయోగించండి.
```

**ఇది వెలికితీసేవి:**

- రంగుల పాలెట్ మరియు రంగుల వినియోగం
- టైపోగ్రఫీ (ఫాంట్లు మరియు పరిమాణాలు)
- లేఅవుట్ నిర్మాణం మరియు ఖాళీల అమరిక
- నావిగేషన్ నమూనాలు
- కంటెంట్ అమరిక
- చిత్రాలు మరియు మీడియా వినియోగం
- పరస్పర చర్య అంశాలు
- మొబైల్ స్పందనశీలత

**పరిమితులు:**

- పాస్‌వర్డ్‌తో రక్షిత సైట్‌లను స్క్రేప్ చేయలేరు
- robots.txt మరియు సైట్ విధానాలను గౌరవిస్తుంది
- డైనమిక్ కంటెంట్‌ను పూర్తిగా పట్టుకోకపోవచ్చు
- JavaScript ఎక్కువగా ఉన్న సైట్‌లలో విశ్లేషణ పరిమితంగా ఉండవచ్చు
- పెద్ద సైట్‌లను విశ్లేషించడానికి ఎక్కువ సమయం పట్టవచ్చు

### Tier 2 సాధనాలు (ఐచ్చికం)

Theme Builder సెట్టింగ్‌లలో స్పష్టంగా ప్రారంభించినప్పుడు Tier 2 సాధనాలు అందుబాటులో ఉంటాయి.

#### అభివృద్ధి చెందిన విశ్లేషణలు

వివరమైన పనితీరు కొలమానాలను అందిస్తుంది:

- పేజీ లోడ్ సమయాలు
- Core Web Vitals
- SEO స్కోర్
- యాక్సెసిబిలిటీ స్కోర్
- ఉత్తమ పద్ధతుల స్కోర్

#### కంటెంట్ ఆప్టిమైజర్

వీటికి మెరుగుదలలను విశ్లేషించి సూచిస్తుంది:

- చదవడానికి సౌలభ్యం
- SEO ఆప్టిమైజేషన్
- కీవర్డ్ వినియోగం
- కంటెంట్ నిర్మాణం
- చర్యకు పిలుపు స్థానం

### Tier 3 సాధనాలు (ప్రీమియం)

Tier 3 సాధనాలకు అదనపు కాన్ఫిగరేషన్ లేదా ప్రీమియం ఫీచర్లు అవసరం.

#### AI కంటెంట్ జనరేటర్

వీటికి కంటెంట్ సృష్టిస్తుంది:

- ఉత్పత్తి వివరణలు
- సేవా పేజీలు
- బ్లాగ్ పోస్ట్‌లు
- మెటా వివరణలు
- చర్యకు పిలుపు కాపీ

#### అభివృద్ధి చెందిన డిజైన్ సాధనాలు

అభివృద్ధి చెందిన డిజైన్ సామర్థ్యాలను అందిస్తుంది:

- అనుకూల CSS సృష్టి
- యానిమేషన్ సృష్టి
- పరస్పర చర్య అంశాల డిజైన్
- అభివృద్ధి చెందిన రంగు సిద్ధాంతం
- టైపోగ్రఫీ ఆప్టిమైజేషన్

### Advanced సహచర సాధనాలు (విడివిడిగా ఉండే ప్యాకేజ్)

Advanced సహచర ప్యాకేజ్‌ను కోర్ Superdav AI Agent విడుదల నుంచి విడిగా పంపిణీ చేస్తారు. అధిక ప్రభావం కలిగించే సాధనాలను ఏజెంట్ ఉపయోగించడానికి నిర్వాహకులు స్పష్టంగా అనుమతించే నమ్మకమైన డెవలపర్ మరియు సైట్ యజమాని పరిసరాల కోసం ఇది ఉద్దేశించబడింది.

Advanced సహచర సామర్థ్యాల్లో ఇవి ఉండవచ్చు:

- **ఫైల్‌సిస్టమ్ సాధనాలు** — నమ్మకమైన పరిసరాల్లో ఆమోదించిన ఫైళ్లను పరిశీలించి మార్చడం
- **డేటాబేస్ సాధనాలు** — స్పష్టంగా అనుమతించినప్పుడు సైట్ డేటాను ప్రశ్నించడం లేదా నవీకరించడం
- **WP-CLI సాధనాలు** — WordPress నిర్వహణ మరియు పరిశీలన కమాండ్‌లను అమలు చేయడం
- **REST డిస్ప్యాచర్ సాధనాలు** — నియంత్రిత డిస్ప్యాచ్ పనిప్రవాహాల ద్వారా నమోదు చేసిన REST endpoints‌ను కాల్ చేయడం
- **ప్లగిన్ బిల్డర్ సాధనాలు** — ప్లగిన్ కోడ్‌ను స్కాఫోల్డ్ చేసి పునరావృతంగా మెరుగుపరచడం
- **Git స్నాప్‌షాట్ సాధనాలు** — ప్రమాదకర డెవలపర్ చర్యల ముందు స్నాప్‌షాట్‌లు సృష్టించడం
- **వినియోగదారు నిర్వహణ సాధనాలు** — అనుమతి ఉన్న చోట వినియోగదారు రికార్డులను సమీక్షించడంలో లేదా సవరించడంలో సహాయం చేయడం
- **బెంచ్‌మార్క్ సాధనాలు** — అభివృద్ధి సమీక్ష కోసం పనితీరు లేదా సామర్థ్య బెంచ్‌మార్క్‌లను అమలు చేయడం

ప్రతి ఇన్‌స్టాలేషన్‌కు సాధారణంగా అందుబాటులో ఉండే Theme Builder సాధనాలుగా వీటిని డాక్యుమెంట్ చేయవద్దు. Advanced సహచర ప్యాకేజ్ ఇన్‌స్టాల్ చేసి, ప్రారంభించి, నమ్మకమైన నిర్వాహకులకు పరిమితం చేసినప్పుడు మాత్రమే ఇవి అందుబాటులో ఉంటాయి.

## సాధన కాన్ఫిగరేషన్

### సాధనాలను ప్రారంభించడం

Theme Builderలో అదనపు సాధనాలను ప్రారంభించడానికి:

1. **Gratis AI Agent → Settings**కు వెళ్లండి
2. **Theme Builder → Tools**కు వెళ్లండి
3. అవసరమైనట్లు సాధనాలను ఆన్/ఆఫ్ చేయండి
4. సెట్టింగ్‌లను సేవ్ చేయండి

### సాధన అనుమతులు

కొన్ని సాధనాలకు అనుమతులు అవసరం:

- **సైట్ స్క్రేపింగ్** — ఇంటర్నెట్ యాక్సెస్ అవసరం
- **విశ్లేషణలు** — Google Analytics ఇంటిగ్రేషన్ అవసరం
- **కంటెంట్ సృష్టి** — API keys అవసరం
- **అభివృద్ధి చెందిన ఫీచర్లు** — ప్రీమియం సబ్‌స్క్రిప్షన్ అవసరం కావచ్చు
- **Advanced సహచర సాధనాలు** — విడిగా పంపిణీ చేసిన Advanced ప్యాకేజ్ మరియు స్పష్టమైన నిర్వాహక నమ్మకం అవసరం

### API Keys మరియు ఆధారాలు

అవి అవసరమయ్యే సాధనాల కోసం API keys కాన్ఫిగర్ చేయండి:

1. **Gratis AI Agent → Settings → API Keys**కు వెళ్లండి
2. ప్రతి సాధనం కోసం ఆధారాలను నమోదు చేయండి
3. కనెక్షన్‌ను పరీక్షించండి
4. సురక్షితంగా సేవ్ చేయండి

### Advanced సహచర ప్యాకేజ్‌ను ఇన్‌స్టాల్ చేయడం

Advanced సహచర ప్యాకేజ్ WordPress.org కోర్ ZIP నుంచి విడిగా విడుదల అవుతుంది. దాన్ని విడుదలకు సంబంధించిన అధికారిక ప్రాజెక్ట్ పంపిణీ ఛానెల్ నుంచే ఇన్‌స్టాల్ చేయండి, తర్వాత యాక్సెస్‌ను నమ్మకమైన నిర్వాహకులకు మాత్రమే పరిమితం చేయండి. ప్రొడక్షన్ సైట్‌లో ఫైల్‌సిస్టమ్, డేటాబేస్, WP-CLI, REST డిస్ప్యాచర్, ప్లగిన్ బిల్డర్, వినియోగదారు నిర్వహణ, లేదా బెంచ్‌మార్క్ సాధనాలను ప్రారంభించే ముందు మానవ ఆమోద గేట్లు మరియు git స్నాప్‌షాట్ ప్రవర్తనను సమీక్షించండి.

## sd-ai-agent/site-scrape వినియోగం

### ప్రాథమిక వినియోగం

వెబ్‌సైట్‌ను విశ్లేషించమని Theme Builderను అడగండి:

```
competitor.comలో ఉన్న నా పోటీదారుడి సైట్ డిజైన్‌ను విశ్లేషించి
నా సైట్‌కు డిజైన్ మెరుగుదలలను సూచించండి.
```

### నిర్దిష్ట విశ్లేషణ

నిర్దిష్ట రకాల విశ్లేషణను అభ్యర్థించండి:

```
example.com నుంచి రంగుల పాలెట్‌ను వెలికితీసి, దాన్ని ప్రేరణగా ఉపయోగించండి.
```

```
example.com నావిగేషన్ నిర్మాణాన్ని విశ్లేషించి, ఇలాంటి
అమరికను నా సైట్‌కు వర్తింపజేయండి.
```

```
example.com మొబైల్ స్పందనశీలతను తనిఖీ చేసి, నా సైట్
అదే స్థాయిలో స్పందనశీలంగా ఉందని నిర్ధారించండి.
```

### పోలిక విశ్లేషణ

అనేక సైట్‌లను పోల్చండి:

```
site1.com మరియు site2.com డిజైన్‌లను పోల్చి, రెండింటిలోని
ఉత్తమ అంశాలను కలిపిన డిజైన్‌ను సృష్టించండి.
```

## సాధన పరిమితులు మరియు పరిగణనలు

### రేట్ పరిమితి

- సర్వర్‌పై అధిక భారం పడకుండా నిరోధించడానికి స్క్రేపింగ్‌కు రేటు పరిమితి ఉంటుంది
- అదే సైట్‌కు అనేక అభ్యర్థనలు చేస్తే వేగం తగ్గించబడవచ్చు
- పెద్ద సైట్‌లను విశ్లేషించడానికి ఎక్కువ సమయం పట్టవచ్చు

### కంటెంట్ ఖచ్చితత్వం

- డైనమిక్ కంటెంట్ పూర్తిగా సంగ్రహించబడకపోవచ్చు
- JavaScript ద్వారా రెండర్ చేసిన కంటెంట్ అసంపూర్ణంగా ఉండవచ్చు
- తక్షణ డేటా ప్రతిబింబించకపోవచ్చు
- కొంత కంటెంట్ చెల్లింపు గోడల వెనుక ఉండవచ్చు

### చట్టపరమైన మరియు నైతిక అంశాలు

- robots.txt మరియు సైట్ విధానాలను గౌరవించండి
- కాపీరైట్ ఉన్న కంటెంట్‌ను పునరుత్పత్తి కోసం స్క్రేప్ చేయవద్దు
- విశ్లేషణను స్ఫూర్తి కోసం ఉపయోగించండి, నకలు చేయడానికి కాదు
- సంగ్రహించిన కంటెంట్‌ను ఉపయోగించే హక్కులు మీకు ఉన్నాయో లేదో ధృవీకరించండి
- విశ్లేషించిన సైట్‌ల సేవా నిబంధనలను పాటించండి

### పనితీరుపై ప్రభావం

- పెద్ద సైట్‌లను స్క్రేప్ చేయడానికి సమయం పట్టవచ్చు
- ఒకేసారి అనేక స్క్రేప్‌లు చేస్తే అవి నెమ్మదిగా ఉండవచ్చు
- నెట్‌వర్క్ కనెక్టివిటీ వేగాన్ని ప్రభావితం చేస్తుంది
- పెద్ద డేటాసెట్‌లకు ఎక్కువ ప్రాసెసింగ్ అవసరం కావచ్చు

## ఉత్తమ పద్ధతులు

### సైట్ విశ్లేషణను ఉపయోగించడం

1. **అనేక సైట్‌లను విశ్లేషించండి** — పలు మూలాల నుండి స్ఫూర్తిని సేకరించండి
2. **నిర్మాణంపై దృష్టి పెట్టండి** — లేఅవుట్ మరియు నిర్వహణ నమూనాలను తెలుసుకోండి
3. **రంగులను సంగ్రహించండి** — రంగుల ప్యాలెట్‌లను ప్రారంభ బిందువులుగా ఉపయోగించండి
4. **టైపోగ్రఫీని అధ్యయనం చేయండి** — మీకు నచ్చిన ఫాంట్ కలయికలను గుర్తించండి
5. **నావిగేషన్‌ను సమీక్షించండి** — మెనూ నిర్వహణను అర్థం చేసుకోండి

### నైతిక స్క్రేపింగ్

1. **robots.txt‌ను గౌరవించండి** — సైట్ మార్గదర్శకాలను అనుసరించండి
2. **కంటెంట్‌ను నకలు చేయవద్దు** — విశ్లేషణను స్ఫూర్తి కోసం మాత్రమే ఉపయోగించండి
3. **హక్కులను ధృవీకరించండి** — సంగ్రహించిన కంటెంట్‌ను ఉపయోగించవచ్చని నిర్ధారించుకోండి
4. **మూలాలను పేర్కొనండి** — స్ఫూర్తి మూలాలకు క్రెడిట్ ఇవ్వండి
5. **నిబంధనలను పాటించండి** — సైట్ సేవా నిబంధనలను అనుసరించండి

### సాధన ప్రభావాన్ని గరిష్ఠం చేయడం

1. **స్పష్టంగా ఉండండి** — నిర్దిష్ట రకాల విశ్లేషణను అభ్యర్థించండి
2. **సందర్భాన్ని ఇవ్వండి** — మీ సైట్ ఉద్దేశ్యాన్ని వివరించండి
3. **అంచనాలను నిర్దేశించండి** — మీ డిజైన్ లక్ష్యాలను వివరించండి
4. **మళ్లీ మెరుగుపరచండి** — ఫలితాల ఆధారంగా సవరించండి
5. **సాధనాలను కలపండి** — సమగ్ర విశ్లేషణ కోసం అనేక సాధనాలను ఉపయోగించండి

## సమస్యల పరిష్కారం

### సైట్ స్క్రేప్ కావడం లేదు

- సైట్ ప్రజలకు అందుబాటులో ఉందో లేదో తనిఖీ చేయండి
- robots.txt స్క్రేపింగ్‌ను అనుమతిస్తుందో లేదో ధృవీకరించండి
- వేరే సైట్‌ను ప్రయత్నించండి
- ఇంటర్నెట్ కనెక్షన్‌ను తనిఖీ చేయండి
- సమస్య కొనసాగితే మద్దతును సంప్రదించండి

### అసంపూర్ణ విశ్లేషణ

- సైట్‌లో డైనమిక్ కంటెంట్ ఉండవచ్చు
- JavaScript పూర్తిగా రెండర్ కాకపోవచ్చు
- పెద్ద సైట్‌లకు సమయం ముగియవచ్చు
- బదులుగా నిర్దిష్ట పేజీలను విశ్లేషించడానికి ప్రయత్నించండి
- నిర్దిష్ట విశ్లేషణ రకాలను అభ్యర్థించండి

### నెమ్మదైన పనితీరు

- పెద్ద సైట్‌లను విశ్లేషించడానికి ఎక్కువ సమయం పడుతుంది
- ఒకేసారి అనేక స్క్రేప్‌లు చేస్తే అవి నెమ్మదిగా ఉంటాయి
- నెట్‌వర్క్ కనెక్టివిటీ వేగాన్ని ప్రభావితం చేస్తుంది
- రద్దీ తక్కువగా ఉన్న సమయాల్లో ప్రయత్నించండి
- ముందుగా చిన్న విభాగాలను విశ్లేషించండి
