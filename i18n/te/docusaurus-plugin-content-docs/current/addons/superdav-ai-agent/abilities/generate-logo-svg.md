# లోగో SVGని రూపొందించడం

**Generate Logo SVG** సామర్థ్యం (ability) ద్వారా, Theme Builder మీ WordPress సైట్‌లోకి కస్టమ్ లోగో SVGలను ఆటోమేటిక్ నేమ్‌స్పేస్-సేఫ్ శుభ్రపరచడంతో (sanitisation) నేరుగా సృష్టించి, పొందుపరచగలదు.

## అవలోకనం (Overview)

ఈ సామర్థ్యం మీ సైట్ బ్రాండింగ్ దిశ మరియు డిజైన్ ప్రాధాన్యతల ఆధారంగా స్కేలబుల్ వెక్టర్ గ్రాఫిక్స్ (SVG) లోగోలను రూపొందిస్తుంది. రూపొందించబడిన SVGలు దృశ్య సమగ్రతను (visual integrity) కాపాడుకుంటూనే, WordPress లో ఉపయోగించడానికి సురక్షితంగా ఉండేలా ఆటోమేటిక్‌గా శుభ్రపరచబడతాయి.

## పారామీటర్లు (Parameters)

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | Yes | లోగోను రూపొందించడానికి అవసరమైన సైట్ పేరు |
| `style` | string | Yes | డిజైన్ శైలి (ఉదాహరణకు, "modern", "classic", "minimalist", "playful") |
| `colors` | array | No | లోగోలో ఉపయోగించాల్సిన హెక్స్ రంగు కోడ్‌ల శ్రేణి (ఉదాహరణకు, `["#678233", "#ffffff"]`) |
| `width` | number | No | పిక్సెల్స్‌లో SVG వెడల్పు (డిఫాల్ట్: 200) |
| `height` | number | No | పిక్సెల్స్‌లో SVG ఎత్తు (డిఫాల్ట్: 200) |
| `include_text` | boolean | No | లోగోలో సైట్ పేరును టెక్స్ట్‌గా చేర్చాలా వద్దా (డిఫాల్ట్: true) |

## అవుట్‌పుట్ ఫార్మాట్ (Output Format)

ఈ సామర్థ్యం కింది నిర్మాణంతో కూడిన SVG స్ట్రింగ్‌ను తిరిగి ఇస్తుంది:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG శుభ్రపరచడం (Sanitisation) ప్రవర్తన

రూపొందించబడిన SVGలు కింది వాటి కోసం ఆటోమేటిక్ నేమ్‌స్పేస్-సేఫ్ శుభ్రపరచ ప్రక్రియకు లోనవుతాయి:

- **అసురక్షిత లక్షణాలను తొలగిస్తుంది** — ఈవెంట్ హ్యాండ్లర్‌లు, స్క్రిప్ట్‌లు మరియు సంభావ్యంగా ప్రమాదకరమైన లక్షణాలను తొలగిస్తుంది.
- **నేమ్‌స్పేస్‌లను కాపాడుతుంది** — సరైన రెండరింగ్‌ కోసం SVG నేమ్‌స్పేస్‌లను (xmlns, xlink) నిర్వహిస్తుంది.
- **నిర్మాణాన్ని ధృవీకరిస్తుంది** — SVG అనేది W3C ప్రమాణాలకు అనుగుణంగా ఉందని నిర్ధారిస్తుంది.
- **ఎంటిటీలను ఎన్‌కోడ్ చేస్తుంది** — టెక్స్ట్ కంటెంట్‌లో ప్రత్యేక అక్షరాలను సరిగ్గా తప్పించుకుంటుంది (escape).
- **బాహ్య సూచనలను తొలగిస్తుంది** — బాహ్య స్టైల్షీట్‌లు మరియు ఇమేజ్ రిఫరెన్స్‌లను తొలగిస్తుంది.

దీని ద్వారా, అదనపు శుభ్రపరచడం అవసరం లేకుండా SVGని నేరుగా WordPress లో పొందుపరచడానికి సురక్షితంగా ఉంటుంది.

## వినియోగ ఉదాహరణ (Usage Example)

**Prompt:**
```
Generate a modern logo for my tech startup called "CloudSync" using blue and white colors.
```

**Result:**
ఈ సామర్థ్యం ఒక SVG లోగోను సృష్టిస్తుంది, అది:
- "CloudSync" అనే సైట్ పేరును పొందుపరుస్తుంది
- పేర్కొన్న నీలం మరియు తెలుపు రంగు పథకాన్ని ఉపయోగిస్తుంది
- ఆధునిక డిజైన్ సూత్రాలను పాటిస్తుంది
- ఆటోమేటిక్‌గా శుభ్రపరచబడి, ఉపయోగించడానికి సిద్ధంగా ఉంటుంది

## Theme Builder తో అనుసంధానం (Integration with Theme Builder)

Theme Builder యొక్క డిజైన్-దిశ ఎంపికను ఉపయోగించినప్పుడు, Generate Logo SVG సామర్థ్యం:

1. మీ డిజైన్ దిశ మరియు రంగు పాలెట్‌ను విశ్లేషిస్తుంది.
2. మీ ప్రాధాన్యతలకు సరిపోయే కస్టమ్ SVG లోగోను రూపొందిస్తుంది.
3. లోగోను మీ సైట్ హెడర్/బ్రాండింగ్ ప్రాంతంలో ఆటోమేటిక్‌గా పొందుపరుస్తుంది.
4. SVGని WordPress మీడియాలో కస్టమ్ లోగోగా నిల్వ చేస్తుంది.

## ఉత్తమ పద్ధతులు (Best Practices)

- **స్పష్టమైన శైలి ప్రాధాన్యతలను అందించండి** — మీరు కోరుకునే డిజైన్ శైలిని వివరించండి (ఆధునిక, క్లాసిక్, ప్లేఫుల్, మొదలైనవి).
- **రంగులను పేర్కొనండి** — స్థిరత్వం కోసం మీ బ్రాండ్ రంగులను చేర్చండి.
- **రెండరింగ్‌ను పరీక్షించండి** — వివిధ స్క్రీన్ సైజులలో లోగో సరిగ్గా ప్రదర్శించబడుతుందో లేదో ధృవీకరించండి.
- **మరింత కస్టమైజ్ చేయండి** — పరిమాణం మరియు ప్లేస్‌మెంట్‌ను సర్దుబాటు చేయడానికి WordPress లోని లోగో కస్టమైజేషన్ టూల్స్‌ను ఉపయోగించండి.

## పరిమితులు (Limitations)

- SVG లోగోలు స్టాటిక్ గ్రాఫిక్స్‌గా (అనిమవేషన్ చేయబడినవి కావు) రూపొందించబడతాయి.
- అనేక అంశాలు ఉన్న సంక్లిష్ట లోగోలకు మాన్యువల్ మెరుగుదల అవసరం కావచ్చు.
- కస్టమ్ ఫాంట్‌లు మద్దతు ఇవ్వబడవు; టెక్స్ట్ సిస్టమ్ ఫాంట్‌లను ఉపయోగిస్తుంది.
- చాలా పెద్ద లేదా చాలా చిన్న కొలతలు నాణ్యతను ప్రభావితం చేయవచ్చు.
