# మార్పుల చిట్టా

## 1.18.0 — 2026-06-29న విడుదలైంది {#1180--released-on-2026-06-29}

### కొత్తవి {#new}

- **Google Calendar సాధనాలు** — షెడ్యూల్‌ను పరిగణనలోకి తీసుకునే ఆటోమేషన్‌ల కోసం కాన్ఫిగర్ చేసిన క్యాలెండర్లు మరియు ఈవెంట్‌లను చదవండి
- **సంప్రదింపు మ్యాపింగ్ మరియు హాజరుదారుల సహాయకాలు** — క్యాలెండర్ పాల్గొనేవారిని సైట్ వినియోగదారులు మరియు సంప్రదింపులతో సరిపోల్చండి
- **మానవ ఆమోద గేట్లు మరియు రిమైండర్ రికార్డులు** — సమీక్ష కోసం ఆటోమేషన్‌లను ఆపి, నకిలీ నోటిఫికేషన్‌లను నివారించండి
- **TextBee SMS ప్రొవైడర్** — TextBee ద్వారా కాన్ఫిగర్ చేసిన టెక్స్ట్ సందేశ నోటిఫికేషన్‌లను పంపండి
- **అధునాతన సహచర ప్యాకేజీ** — WordPress.org బిల్డ్‌కు విడిగా పంపిణీ చేసే విశ్వసనీయ-డెవలపర్ ఫైల్‌సిస్టమ్, డేటాబేస్, WP-CLI, REST డిస్పాచర్, plugin బిల్డర్, git స్నాప్‌షాట్, వినియోగదారు-నిర్వహణ, బెంచ్‌మార్క్ సాధనాలను జోడించండి

### మెరుగుపరచినవి {#improved}

- **నిర్వహిత Superdav సేవ సెటప్** — మద్దతు ఉన్న సైట్‌ల కోసం హోస్ట్ చేసిన సేవ endpointలను మరియు ఆటోమేటిక్ కనెక్షన్ ప్రొవిజనింగ్‌ను జోడించండి
- **విడుదల ప్యాకేజింగ్** — వేర్వేరు కోర్ మరియు Advanced ZIPలను బిల్డ్ చేసి, రెండింటినీ GitHubలో ప్రచురించి, WordPress.orgకు కోర్ ప్యాకేజీని మాత్రమే పంపండి

### సరిచేసినవి {#fixed}

- **AI అభ్యర్థన నమ్మకత** — మోడల్ ఎంపిక, అభ్యర్థన టైమ్‌అవుట్‌లు, డిఫాల్ట్ సెటప్ మోడల్, రీజనింగ్-టెక్స్ట్ నిర్వహణ, చెల్లని సాధన కాల్‌లకు మళ్లీ ప్రయత్నించే మార్గదర్శకతను మెరుగుపరచండి
- **క్యాలెండర్ మరియు రిమైండర్ బలోపేతం** — Google Calendar టోకెన్‌లను మరియు రిమైండర్ డీడూప్లికేషన్‌ను బలపరచండి
- **ఆన్‌బోర్డింగ్ మరియు ఆమోద పునఃప్రారంభాలు** — ఫ్రంట్‌ఎండ్ ఆన్‌బోర్డింగ్ ప్రారంభాలను మరియు ధృవీకరించిన సామర్థ్య పునఃప్రారంభాలను సరిచేయండి
- **WordPress.org ప్యాకేజింగ్ సమీక్ష సమస్యలు** — కోర్ విడుదలకు సంబంధించిన ప్యాకేజింగ్ సమీక్ష అభిప్రాయాన్ని పరిష్కరించండి

## 1.16.0 — 2026-05-20న విడుదలైంది {#1160--released-on-2026-05-20}

### కొత్తవి {#new-1}

- **Logo SVG రూపొందించే సామర్థ్యం** — Theme Builder ఇప్పుడు నేమ్‌స్పేస్-సురక్షిత శుద్ధీకరణతో అనుకూల logo SVGలను రూపొందించి చేర్చగలదు
- **డిస్కవరీ ఇంటర్వ్యూలో ఫోటో అప్‌లోడ్** — మెరుగైన డిజైన్ సందర్భం కోసం Theme Builder డిస్కవరీ ఇంటర్వ్యూలో ఇప్పుడు ఫోటో అప్‌లోడ్ దశ ఉంది
- **పాలెట్ కాంట్రాస్ట్ ధృవీకరించే సామర్థ్యం** — themeకు వర్తించే ముందు రంగు జంటలు WCAGకు అనుగుణంగా ఉన్నాయో లేదో తనిఖీ చేయండి
- **ఆతిథ్య మెనూలు** — ఆతిథ్య వ్యాపారాల కోసం Theme Builder ఇప్పుడు నిర్మిత ఆహార మరియు పానీయాల మెను పేజీలను రూపొందించగలదు
- **డెస్క్‌టాప్ మరియు మొబైల్ ప్రీవ్యూ రెండరింగ్** — డిజైన్-దిశ ఎంపిక సమయంలో మీ డిజైన్‌ను డెస్క్‌టాప్ మరియు మొబైల్ పరికరాల్లో ప్రీవ్యూ చేయండి
- **నావిగేషన్ లేబుల్ పరామితి** — Create Menu సామర్థ్యం ఇప్పుడు పేజీ శీర్షికకు వేరుగా ఉండే ప్రత్యేక `navigation_label`కు మద్దతు ఇస్తుంది
- **Tier 1 సాధన లభ్యత** — sd-ai-agent/site-scrape ఇప్పుడు Theme Builderలో డిఫాల్ట్‌గా అందుబాటులో ఉండే Tier 1 సాధనం

### సరిచేసినవి {#fixed-1}

- **AI క్లయింట్ క్యాష్** — ఇప్పుడు క్రాస్-రిక్వెస్ట్ నిలకడ కోసం ట్రాన్సియెంట్స్‌తో మద్దతు ఇస్తుంది, దీర్ఘకాలం నడిచే agent పనుల్లో డేటా నష్టాన్ని నివారిస్తుంది
- **Plugin వరుస చర్య లింకులు** — స్పష్టత కోసం సరిచేసి పేరు మార్చారు

## 1.10.0 — 2026-05-05న విడుదలైంది {#1100--released-on-2026-05-05}

### కొత్తవి {#new-2}

- **Tavily ఇంటర్నెట్ శోధన** — Brave Searchతో పాటు మరింత సమృద్ధిగా ఇంటర్నెట్ శోధన ఫలితాల కోసం Tavilyని శోధన ప్రొవైడర్‌గా జోడించండి
- **Theme-అవగాహన కలిగిన అంతర్నిర్మిత నైపుణ్యాలు** — Block Themes, Classic Themes, Kadence Blocks, Kadence Theme నైపుణ్య మార్గదర్శకాలు ఇప్పుడు pluginతో వస్తాయి
- **సైట్ బిల్డర్ సంప్రదింపు ఫారమ్ సామర్థ్యం** — చాట్ ఇంటర్‌ఫేస్‌ నుంచే ఏ పేజీకైనా సంప్రదింపు ఫారమ్‌ను జోడించండి

### మెరుగుపరచినవి {#improved-1}

- **WooCommerce సమీకరణ పునర్నిర్మించబడింది** — మెరుగైన నమ్మకత మరియు అనుకూలత కోసం ఇప్పుడు స్థానిక WooCommerce APIలను ఉపయోగిస్తుంది
- **ప్రొవైడర్ జాబితా ఆటోమేటిక్‌గా రిఫ్రెష్ అవుతుంది** — ఏ plugin అయినా సక్రియం లేదా నిష్క్రియం చేసినప్పుడు

### సరిచేసినవి {#fixed-2}

- **navigate-to సామర్థ్యం** — కొన్ని admin పేజీల్లో అనంత రీలోడ్ లూప్‌ను సరిచేశారు
- **list-posts సామర్థ్యం** — ఇప్పుడు వర్గం మరియు ట్యాగ్ పేర్లను slugsగా సరిగ్గా పరిష్కరిస్తుంది
- **WP-CLI కమాండ్‌లు** — మునుపటి రీఫ్యాక్టర్ తర్వాత కనిపించని నేమ్‌స్పేస్ అలియాస్‌లను పునరుద్ధరించారు
- **ఈవెంట్ ఆటోమేషన్** — ఆటోమేషన్ పట్టికలు ఇంకా సృష్టించని సైట్‌లను సునాయాసంగా నిర్వహిస్తుంది
- **memory-save సామర్థ్యం** — ఇప్పుడు సిస్టమ్ ఇన్‌స్ట్రక్షన్ బిల్డర్‌లో సరైన నేమ్‌స్పేస్ ప్రిఫిక్స్‌ను ఉపయోగిస్తుంది
- **స్కేలర్ సాధన ఫలితాలు** — ఇప్పుడు AIకి తిరిగి ఇవ్వడానికి ముందు సరిగ్గా చుట్టబడ్డాయి
- **వినియోగ గణాంకాలు** — పాత వెర్షన్‌ల నుంచి అప్‌గ్రేడ్ చేసినప్పుడు లెగసీ సామర్థ్య కీ ఫార్మాట్‌ను ఇప్పుడు సరిగ్గా నిర్వహిస్తాయి
