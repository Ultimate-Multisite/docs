# Filter: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

ఈ చెక్‌ను బైపాస్ చేయడానికి (short-circuit) డెవలపర్‌లకు ఇది అనుమతిస్తుంది.

మీరు composer-ఆధారిత (composer-based) మరియు ఇతర కస్టమ్ సెటప్‌లను, ఉదాహరణకు Bedrock వంటి వాటిని ఉపయోగిస్తున్నప్పుడు ఇది ఉపయోగపడుతుంది. అలాంటి సెటప్‌లలో mu-pluginsని ప్లగిన్‌లుగా ఉపయోగించడం అనేది సాధారణ పద్ధతి (norm).
