# సపోర్ట్ ఏజెంట్లు (Support Agents) {#support-agents}

ఒక మొత్తం నెట్‌వర్క్‌ను మీరే నిర్వహించడం అంత సులభమైన పని కాదు. మీ నెట్‌వర్క్ పెరగడం మొదలయ్యాక, మీకు సపోర్ట్‌కు డిమాండ్ పెరుగుతుందని మీరు గమనిస్తారు. మీ పనిభారాన్ని తగ్గించడానికి సహాయపడే వ్యక్తులను తీసుకురావడం మంచి ఆలోచన. అయితే, పూర్తి యాక్సెస్/పర్మిషన్ ఇవ్వడం ప్రమాదకరం మరియు మీ నెట్‌వర్క్‌కు నష్టం కలిగించవచ్చు. ఇక్కడే ఈ Support Agent add-on ఉపయోగపడుతుంది.

ఈ add-on ద్వారా మీరు 'Support Agent' అనే కొత్త రకమైన WordPress యూజర్‌ను సృష్టించవచ్చు, మరియు ఆ యూజర్ ఏయే పర్మిషన్స్ కలిగి ఉండాలో మీకు పూర్తి నియంత్రణ ఇస్తుంది. పర్మిషన్ ఇవ్వడం అనేది ఒక టోగుల్ బటన్‌ను ఆన్ చేయడం అంత సులభం, మరియు తొలగించాలని నిర్ణయించుకుంటే దాన్ని ఆఫ్ చేయడం అంతే సులభం.

## ఫీచర్లు (Features) {#features}

- WordPress & Ultimate Multisite పర్మిషన్ల యొక్క వివరణాత్మక జాబితా
- యాక్సెస్‌ను సులభంగా ఎనేబుల్ (Enable) మరియు డిసేబుల్ (Disable) చేయవచ్చు
- అపరిమిత సంఖ్యలో ఏజెంట్ ఖాతాలను సృష్టించవచ్చు

## ఇన్‌స్టాలేషన్ (Installation) {#installation}

1. 'ultimate-multisite-support-agents' ను '/wp-content/plugins/' డైరెక్టరీలోకి అప్‌లోడ్ చేయండి
2. WordPress లోని 'Plugins' మెనూ ద్వారా ప్లగిన్‌ను యాక్టివేట్ చేయండి
3. ప్లగిన్‌ను సెటప్ చేయడానికి స్టెప్ బై స్టెప్ విజర్డ్‌ను అనుసరించండి
