# Ultimate AI Connector for Compatible Endpoints మార్పుల వివరాలు

## వెర్షన్ 2.0.0 — విడుదల తేదీ 2026-04-24 {#version-200--released-on-2026-04-24}

* కొత్తది: బహుళ-ప్రвайడర్ మద్దతు — మీరు అనేక AI ఎండ్‌పాయింట్‌లను కాన్ఫిగర్ చేయవచ్చు మరియు ప్రвайడర్ల మధ్య ఆటోమేటిక్ ఫాల్‌బ్యాక్‌తో రిక్వెస్ట్‌లను రూట్ చేయవచ్చు.
* పరిష్కారం: ప్రతి ప్రвайడర్ కోసం సరైన ప్రвайడర్ IDలు, రిజిస్ట్రేషన్ URLలు మరియు HTTP ఫిల్టర్ స్కోపింగ్‌తో బహుళ-ప్రвайడర్ SDK ఇంటిగ్రేషన్.
* పరిష్కారం: కొత్త ప్రвайడర్ కార్డ్ ఇప్పుడు జోడించిన వెంటనే ఆటోమేటిక్‌గా విస్తరిస్తుంది; ప్లగిన్ అప్‌డేట్ అయినప్పుడు స్క్రిప్ట్ కాష్ బస్టింగ్.
* పరిష్కారం: eval() కోసం డైనమిక్ ప్రвайడర్ క్లాస్ నేమ్‌స్పేస్ మరియు కొత్తగా జోడించిన ప్రвайడర్ల కోసం ఆటో-ఎక్స్‌పాండ్ ప్రవర్తన.
* పరిష్కారం: WordPress 6.9+ అనుకూలత కోసం స్థిరమైన Card/CardBody/CardHeader/CardDivider కాంపోనెంట్‌లను ఉపయోగించడం (ఇక ప్రయోగాత్మకం కాదు).
* పరిష్కారం: ప్రвайడర్‌ను డ్రాగ్ చేసి మళ్లీ ఆర్డర్ చేయడానికి అస్పష్టమైన DragHandle స్థానంలో యూనికోడ్ గ్రిప్ ఐకాన్‌ను ఉపయోగించడం.
* పరిష్కారం: ప్రతి పేజీ లోడ్ అయినప్పుడు ట్రిగ్గర్ అయ్యే బ్లాకింగ్ HTTP రిక్వెస్ట్‌ను తొలగించడం.
* మెరుగుపడింది: GitHub Actions వర్క్‌ఫ్లోలు Node.js 24 కి అప్‌గ్రేడ్ చేయబడ్డాయి.
