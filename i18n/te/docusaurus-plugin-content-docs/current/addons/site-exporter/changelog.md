# సైట్ ఎక్స్‌పోర్టర్ మార్పుల వివరాలు {#site-exporter-changelog}

Version: 1.0.6 - విడుదల చేయబడింది 2026-05-11
* కొత్తగా: సైట్ ఎగుమతులు ఇప్పుడు ఒక స్వయంగా ప్రారంభమయ్యే index.php ని కలిపి ఇస్తాయి. కాబట్టి, ప్రత్యేకమైన ప్లగిన్ ఇన్‌స్టాల్ చేయకుండానే, ఈ ZIP ని కొత్త హోస్ట్‌లో కూడా ఇన్‌స్టాల్ చేయవచ్చు.
* కొత్తగా: నెట్‌వర్క్ ఎగుమతి ఫీచర్ ద్వారా, అడ్మినిస్ట్రేటర్లు సైట్ ఎక్స్‌పోర్ట్ అడ్మిన్ పేజీ నుండి అన్ని సబ్‌సైట్‌లను ఒకే ఆర్కైవ్‌లో ఎగుమతి చేయవచ్చు.

Version: 1.0.5 - విడుదల చేయబడింది 2026-05-05
* పరిష్కారం: టెస్ట్ బూట్‌స్ట్రాప్ వైఫల్యాలను నివారించడానికి, `get_config` వంటి మ็อก చేయని పద్ధతులను నిర్వహించడానికి WP_CLI polyfill stub కు `__callStatic` జోడించబడింది.

Version: 1.0.4 - విడుదల చేయబడింది 2025-11-25

* పరిష్కారం: Ultimate Multisite 2.4 తో అనుకూలత (compatibility) మెరుగుపరచబడింది.

Version: 1.0.2 - విడుదల చేయబడింది 2025-09-28

* ప్రిఫిక్స్‌ను ultimate-multisite గా మార్చబడింది; టెక్స్ట్ డొమైన్ అప్‌డేట్ చేయబడింది; వెర్షన్ నంబర్ పెంచబడింది.

### Version 1.0.1 - విడుదల చేయబడింది 2023-08-09 {#version-101---released-on-2023-08-09}

* మెరుగుపడింది: ఇప్పుడు ఇంపార్టర్ ఫైల్ యొక్క సరైన ఫైల్ పాత్‌ను పొందుతుందని నిర్ధారిస్తుంది.
* పరిష్కారం: ఇన్‌పోర్ట్ సమయంలో వచ్చే లోపాలను నివారించడానికి ఆబ్జెక్ట్ కాష్ (object cache) ని డిసేబుల్ చేయడం జరిగింది.
* పరిష్కారం: ఇంపార్టర్ ప్లగిన్ డౌన్‌లోడ్ లింక్ పనిచేయకపోవడం అనే సమస్యను పరిష్కరించబడింది.
