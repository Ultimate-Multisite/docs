# Action: update_network {#action-updatenetwork}

## పారామీటర్లు (Parameters) {#parameters}

| పేరు (Name) | రకం (Type) | వివరణ (Description) |
|------|------|-------------|
| $network_id | `int` | జోడించబడిన నెట్‌వర్క్ యొక్క ID. |
| $args | `array` | నెట్‌వర్క్ ఆర్గ్యుమెంట్ల యొక్క అసోసియేటివ్ శ్రేణి (Associative array). |
