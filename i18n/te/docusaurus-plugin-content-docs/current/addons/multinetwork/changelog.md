# మల్టీ-నెట్‌వర్క్ మార్పుల జాబితా

### 1.3.0
* కొత్తగా: నెట్‌వర్క్ టెంప్లేట్ ప్రివ్యూయర్ — కొనుగోలు చేయడానికి ముందు లైవ్ ప్యానెల్‌తో నెట్‌వర్క్ టెంప్లేట్‌లను బ్రౌజ్ చేసి ప్రివ్యూ చేయవచ్చు.
* కొత్తగా: చెక్అవుట్‌లో నెట్‌వర్క్ టెంప్లేట్ పికర్ — సైన్అప్ చెక్అవుట్ ఫ్లో సమయంలో కస్టమర్‌లు నెట్‌వర్క్ టెంప్లేట్‌ను ఎంచుకోగలరు.
* కొత్తగా: క్రాస్-నెట్‌వర్క్ మ్యాజిక్ లింక్ SSO — కస్టమర్‌లు మ్యాజిక్-లింక్ అథెంటికేషన్ ద్వారా సబ్-నెట్‌వర్క్‌లను సులభంగా యాక్సెస్ చేయవచ్చు.
* కొత్తగా: నెట్‌వర్క్ మీడియా క్లోనింగ్ — టెంప్లేట్ నుండి నెట్‌వర్క్‌లను క్లోనింగ్ చేసేటప్పుడు సైట్ మీడియా సరిగ్గా డూప్లికేట్ అవుతుంది.
* పరిష్కారం: మల్టీ-టెనన్సీ యాడ్‌ఆన్ లేకుండా నెట్‌వర్క్ క్లోనింగ్ ఇప్పుడు పనిచేస్తుంది; క్లోనింగ్ తర్వాత సూపర్ అడ్మిన్ ప్రివిలేజ్‌లు తిరిగి ఇవ్వబడతాయి.
* పరిష్కారం: నెట్‌వర్క్ క్రియేషన్ ఇప్పుడు WooCommerce మరియు అన్ని పేమెంట్ గేట్‌వేలతో అనుకూలంగా ఉంది.
* పరిష్కారం: ఫాల్‌బ్యాక్ డొమైన్ జనరేషన్‌లో పోర్ట్ నంబర్ ఇప్పుడు చేర్చబడింది.
* పరిష్కారం: అన్ని థీమ్‌లలో సరైన రెండరింగ్‌ కోసం టెంప్లేట్ పికర్ CSS ను ఎన్‌క్యూ చేయబడిన స్టైల్‌షీట్‌కు తరలించబడింది.
* పరిష్కారం: సైట్ క్వరీ స్కోప్‌లో నిర్వచించబడని `network_id` కీకి రక్షణ కల్పించబడింది.
* పరిష్కారం: కాంటెక్స్ట్ స్విచ్చింగ్ కంటే ముందు నెట్‌వర్క్ పర్మిషన్ చెక్ `wpdb` టేబుల్ ప్రాపర్టీ ఎర్రర్‌లను నివారిస్తుంది.
