# భాషా సెలెక్టర్ (Language Selector) {#language-selector}

మల్టీసైట్ అల్టిమేట్: సైన్-అప్‌లో భాషా సెలెక్టర్

సైన్-అప్ ఫ్లోలో ఒక భాషా సెలెక్టర్ ఫీల్డ్‌ను జోడించండి.

## ఇన్‌స్టాలేషన్ (Installation) {#installation}

1. 'ultimate-multisite-language-selector' ను '/wp-content/plugins/' డైరెక్టరీలోకి అప్‌లోడ్ చేయండి.
2. మీ WordPress Network Admin లోని 'Plugins' మెనూ ద్వారా ప్లగిన్‌ను యాక్టివేట్ చేయండి.

## ఉపయోగించే విధానం (Usage Guide) {#usage-guide}

ఈ ఉచిత అడ్-ఆన్‌తో, సైన్-అప్ ఫ్లోలో మరియు లాగిన్ పేజీలో తమకు ఏ భాష కావాలి అని మీ వినియోగదారులు ఎంచుకునేలా చేయవచ్చు.

[మీ లైసెన్స్‌ను యాక్టివేట్ చేసిన తర్వాత](../../user-guide/getting-started/installing-ultimate-multisite.md), మీ లైసెన్స్ ద్వారా యాక్సెస్ లభించే Ultimate Multisite అడ్-ఆన్‌లను మీరు ఇన్‌స్టాల్ చేయగలుగుతారు.

మీ Ultimate Multisite సెట్టింగ్‌ల పేజీకి వెళ్లి, ఆపై ఎడమ వైపు ఉన్న మెనూలో _Add-ons_ అనే లేబుల్‌పై క్లిక్ చేయండి.

![Add-ons page](/img/admin/addons-page.png)

_Language Selector ను ఇన్‌స్టాల్ చేయడం ప్రారంభించడానికి Add-ons పేజీకి వెళ్లండి_

అక్కడ, మేము అందించే అన్ని అడ్-ఆన్‌లను మీరు కనుగొంటారు.

సబ్‌స్క్రిప్షన్ ప్లాన్ ఏమైనా, మా అందరు కస్టమర్లకు ఉచిత అడ్-ఆన్‌లు అందుబాటులో ఉంటాయి.

అడ్-ఆన్ థంబ్‌నెయిల్ కింద ఉన్న _Install it Now!_ బటన్‌పై క్లిక్ చేయండి. ఆపై Plugins పేజీలో దాన్ని యాక్టివేట్ చేయండి.

![Add-ons page with install button](/img/admin/addons-page.png)

_అడ్-ఆన్‌ను ఇన్‌స్టాల్ చేయడానికి బటన్‌పై క్లిక్ చేయండి_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_మీ Plugins పేజీలో దాన్ని యాక్టివేట్ చేయండి_

ఇప్పుడు, Ultimate Multisite సెట్టింగ్‌ల మెనూకి వెళ్లి, _Network Settings_ ట్యాబ్‌లోకి వెళ్లండి. అక్కడ, పేజీ దిగువన, _Languages_ అనే కొత్త సెక్షన్ కనిపిస్తుంది.

![Settings page](/img/config/settings-general.png)

_ఒక కొత్త సెక్షన్ కనిపించింది!_

అన్ని భాషలను ఆప్షన్‌లుగా చూపించడానికి మీరు దీన్ని ఖాళీగా వదిలేయవచ్చు లేదా అందుబాటులో ఉంచాలనుకునే భాషలను ఎంచుకోవచ్చు. ఈ భాషలు మీ WordPress సెటప్‌లో ఇన్‌స్టాల్ చేయబడిన భాషలే.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

![List of languages available on this WP install](/img/config/settings-general.png)

_ఈ WP ఇన్‌స్టాల్‌లో అందుబాటులో ఉన్న భాషల జాబితా_

మీరు చేసిన మార్పులను సేవ్ చేసిన తర్వాత, మీరు ఎంచుకున్న భాషలతో కూడిన సెలక్షన్ మెనూ లాగిన్ పేజీలో మరియు సైన్-అప్ ఫ్లోలో కూడా కనిపిస్తుంది.

![Login and Registration settings](/img/config/settings-login-registration.png)

_లాగిన్ పేజీలో భాషా సెలెక్టర్_

![Registration page](/img/config/registration-page.png)

_సైన్-అప్ ఫ్లోలో భాషా సెలెక్టర్_

## URL ద్వారా భాషను సెట్ చేయడం (Setting the Language via the URL) {#setting-the-language-via-the-url}

మీరు URL లోని లోకల్ (locale) పారామీటర్‌ను దాటవేసి కూడా పేజీ భాషను మార్చవచ్చు. ఉదాహరణకు, మీరు ఇటాలియన్‌లో రిజిస్టర్ పేజీకి రీడైరెక్ట్ చేయాలనుకుంటే, URL కి ?locale=it_IT అని జోడించండి.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ఇది పని చేయడానికి, ఆ భాష మీ WordPress నెట్‌వర్క్‌లో ఇన్‌స్టాల్ చేయబడి ఉండాలి.
