# AI Site Builder {#ai-site-builder}

ultimate-multisite-ai-site-builder

## అవసరాలు (Requirements) {#requirements}

- WordPress 5.3 లేదా అంతకంటే ఎక్కువ
- PHP 7.4 లేదా అంతకంటే ఎక్కువ
- Multisite Ultimate ప్లగిన్ (యాక్టివేట్ చేయబడి ఉండాలి)

## ఇన్‌స్టాలేషన్ (Installation) {#installation}

1. అడ్డాన్ ఫైళ్లను మీ `/wp-content/plugins/` డైరెక్టరీలోకి అప్‌లోడ్ చేయండి.
2. WordPress లోని 'Plugins' మెనూ ద్వారా ప్లగిన్‌ను యాక్టివేట్ చేయండి.
3. నెట్‌వర్క్ అడ్మిన్ లో అడ్డాన్ సెట్టింగ్‌లను కాన్ఫిగర్ చేయండి.

## డెవలప్‌మెంట్ (Development) {#development}

### సెటప్ (Setup) {#setup}

```bash
# డిపెండెన్సీలను ఇన్‌స్టాల్ చేయండి
composer install
npm install

# టెస్ట్‌లను రన్ చేయండి
npm run test

# కోడ్ స్టాండర్డ్స్ చెక్‌లను రన్ చేయండి
vendor/bin/phpcs
vendor/bin/phpstan

# కోడ్ స్టైల్ సమస్యలను సరిచేయండి
vendor/bin/phpcbf

# ప్రొడక్షన్ కోసం బిల్డ్ చేయండి
npm run build
```

### టెస్టింగ్ (Testing) {#testing}

ఈ అడ్డాన్‌లో సమగ్రమైన టెస్టింగ్ ఫ్రేమ్‌వర్క్ ఉంది:

- **యూనిట్ టెస్ట్‌లు (Unit Tests)**: WordPress టెస్ట్ సూట్‌తో PHPUnit ఆధారిత పరీక్షలు.
- **బేస్ టెస్ట్ క్లాస్ (Base Test Class)**: సాధారణ టెస్ట్ దృశ్యాల కోసం సహాయక పద్ధతులను అందిస్తుంది.
- **టెస్ట్ కవరేజ్ (Test Coverage)**: కోడ్ కవరేజీని ట్రాక్ చేయడానికి కాన్ఫిగర్ చేయబడింది.
- **మల్టీసైట్ టెస్టింగ్ (Multisite Testing)**: టెస్ట్‌లు మల్టీసైట్ ఎన్విరాన్‌మెంట్‌లో రన్ అవుతాయి.

### కోడ్ స్టాండర్డ్స్ (Code Standards) {#code-standards}

- **WordPress కోడింగ్ స్టాండర్డ్స్**: PHPCS ద్వారా అమలు చేయబడుతుంది.
- **PHP 7.4+ అనుకూలత**: ఆధునిక PHP ఫీచర్లు మద్దతు ఇస్తాయి.
- **స్టాటిక్ అనాలిసిస్ (Static Analysis)**: టైప్ చెకింగ్ కోసం PHPStan ఇంటిగ్రేషన్.
- **కోడ్ ఆధునికీకరణ (Code Modernization)**: PHP అప్‌గ్రేడ్‌ల కోసం Rector నియమాలు.

### బిల్డ్ సిస్టమ్ (Build System) {#build-system}

- **ఆస్తి ప్రాసెసింగ్ (Asset Processing)**: CSS/JS ఫైళ్ల కోసం మినిఫికేషన్.
- **అనువాదం (Translation)**: POT ఫైల్ జనరేషన్.
- **ప్యాకేజింగ్ (Packaging)**: పంపిణీ కోసం ఆర్కైవ్ సృష్టి.
- **డెవలప్‌మెంట్ మోడ్ (Development Mode)**: డీబగ్గింగ్ కోసం మినిఫై చేయని ఆస్తులు.

## కాన్ఫిగరేషన్ (Configuration) {#configuration}

ఈ అడ్డాన్ నెట్‌వర్క్ అడ్మిన్ ఇంటర్‌ఫేస్ ద్వారా యాక్సెస్ చేయగల వివిధ కాన్ఫిగరేషన్ ఎంపికలను మద్దతు ఇస్తుంది.

## హుక్స్ మరియు ఫిల్టర్లు (Hooks and Filters) {#hooks-and-filters}

### యాక్షన్స్ (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - అడ్డాన్ ప్రారంభమైన తర్వాత ట్రిగ్గర్ అవుతుంది.
- `ultimate-multisite-ai-site-builder_loaded` - అన్ని అడ్డాన్ ఫైళ్లు లోడ్ అయిన తర్వాత ట్రిగ్గర్ అవుతుంది.

### ఫిల్టర్లు (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - అడ్డాన్ సెట్టింగ్‌లను మార్చడానికి.
- `ultimate-multisite-ai-site-builder_enabled` - అడ్డాన్ ఎనేబుల్/డిసేబుల్ స్థితిని ఓవర్‌రైడ్ చేయడానికి.
