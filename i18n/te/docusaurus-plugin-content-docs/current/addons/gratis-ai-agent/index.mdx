# Gratis AI Agent

Gratis AI Agent అనేది ఒక స్వయంప్రతిపత్తి గల (autonomous) WordPress AI assistant. ఇది సహజ భాషా సూచనల ద్వారా పూర్తి WordPress సైట్‌లను ప్లాన్ చేయగలదు, నిర్మించగలదు మరియు నిర్వహించగలదు. ఇది **సామర్థ్యాలు (abilities)** అనే నిర్మాణాత్మకమైన సెట్‌ను అందిస్తుంది — ఇవి ఏజెంట్ పిలవగలిగే వివిక్త, పరీక్షించదగిన చర్యలు. కంటెంట్ నిర్వహణ నుండి డిజైన్ సిస్టమ్ కస్టమైజేషన్ వరకు ప్రతిదీ ఇది కవర్ చేస్తుంది. వెర్షన్ 1.9.0 లో కంటెంట్ సృష్టి సామర్థ్యాలు, క్లయింట్-సైడ్ స్క్రీన్‌షాట్‌ల ద్వారా విజువల్ పేజీ రివ్యూ, ఐదు అంతర్నిర్మిత ఏజెంట్ ప్రొఫైల్స్, మరియు యాక్సెస్ కంట్రోల్ మరియు బ్రాండింగ్ కోసం ఫీచర్ ఫ్లాగ్‌లను జోడించారు.

## ముఖ్య లక్షణాలు (Key Features)

- **సామర్థ్య వ్యవస్థ (Abilities system)** — మీ WordPress ఇన్‌స్టాలేషన్‌లో ఏజెంట్ పిలవగలిగే చర్యల యొక్క మాడ్యులర్, విస్తరించదగిన కేటలాగ్.
- **కంటెంట్ నిర్వహణ (Content Management)** — పోస్ట్‌లు మరియు పేజీలను సృష్టించడం (పేజీ టెంప్లేట్‌లను కేటాయించడం కూడా), ఒకే కాల్‌లో బహుళ పోస్ట్‌లను బ్యాచ్-క్రియేట్ చేయడం, ఫీచర్డ్ ఇమేజ్‌లను సెట్ చేయడం మరియు కాంటాక్ట్ ఫారమ్‌లను నిర్మించడం.
- **విజువల్ రివ్యూ (Visual Review)** — లైవ్ పేజీల స్క్రీన్‌షాట్‌లను క్యాప్చర్ చేయడం, ముందు/తర్వాత (before/after) స్టేట్‌లను పోల్చడం, మరియు లేఅవుట్, టైపోగ్రఫీ, రంగు మరియు యాక్సెసిబిలిటీని కవర్ చేస్తూ AI రూపొందించిన డిజైన్ రివ్యూ పొందడం.
- **ఐదు అంతర్నిర్మిత ఏజెంట్లు (Five built-in agents)** — కంటెంట్ రైటర్, సైట్ బిల్డర్, డిజైన్ స్టూడియో, ప్లగిన్ మేనేజర్ మరియు సపోర్ట్ అసిస్టెంట్; ఇవి చాట్ హెడర్‌లోని ఏజెంట్ పికర్‌ ద్వారా మార్చవచ్చు.
- **ఫీచర్ ఫ్లాగ్‌లు (Feature flags)** — రోల్ పరిమితులు, వైట్‌లేబుల్ పేరు మరియు విడ్జెట్ కస్టమైజేషన్ కోసం Settings → Feature Flags లో యాక్సెస్ కంట్రోల్ మరియు బ్రాండింగ్ టోగుల్స్.
- **కస్టమ్ పోస్ట్ రకం నిర్వహణ (Custom Post Type management)** — కస్టమ్ పోస్ట్ రకాలను రిజిస్టర్ చేయడం, జాబితా చేయడం మరియు డిలీట్ చేయడం, రీస్టార్ట్‌ల మధ్య నిలకడ (persistence) ఉంటుంది.
- **కస్టమ్ టాక్సానమీ నిర్వహణ (Custom Taxonomy management)** — కస్టమ్ టాక్సానమీలను రిజిస్టర్ చేయడం, జాబితా చేయడం మరియు డిలీట్ చేయడం, నిలకడ ఉంటుంది.
- **డిజైన్ సిస్టమ్ సామర్థ్యాలు (Design System abilities)** — కస్టమ్ CSS ఇంజెక్ట్ చేయడం, క్యూరేటెడ్ బ్లాక్ ప్యాటర్న్‌లను నిర్వహించడం, సైట్ లోగోను సెట్ చేయడం మరియు theme.json ప్రీసెట్‌లను వర్తింపజేయడం.
- **గ్లోబల్ స్టైల్స్ నిర్వహణ (Global Styles management)** — కలర్ పాలెట్‌లు, టైపోగ్రఫీ, స్పేసింగ్ మరియు మరిన్నింటి కోసం theme.json విలువలను చదవడం మరియు రాయడం.
- **నావిగేషన్ మెనూ నిర్వహణ (Navigation Menu management)** — నావిగేషన్ మెనూలను మరియు వాటి అంశాలను ప్రోగ్రామాటిక్‌గా సృష్టించడం మరియు అప్‌డేట్ చేయడం.
- **ఆప్షన్స్ నిర్వహణ (Options Management)** — కీలక సెట్టింగ్‌లను రక్షించడానికి అంతర్నిర్మిత సేఫ్టీ బ్లాక్‌లిస్ట్ తో WordPress ఆప్షన్స్‌ను చదవడం మరియు రాయడం.
- **ఇన్‌స్టాల్ చేయదగిన సామర్థ్యాల రిజిస్ట్రీ (Installable Abilities Registry)** — కోర్ ప్లగిన్ ఫైళ్లను మార్చకుండా అదనపు సామర్థ్య ప్యాక్‌లను కనుగొని యాక్టివేట్ చేయడం.
- **సైట్ బిల్డర్ ఆర్కెస్ట్రేషన్ v2 (Site Builder Orchestration v2)** — ప్లగిన్ డిస్కవరీ, ప్రగతి ట్రాకింగ్ మరియు ఎర్రర్ రికవరీతో కూడిన బహుళ-దశల సైట్ బిల్డ్ ప్లాన్‌లు.
- **WP-CLI బెంచ్‌మార్క్ కమాండ్ (WP-CLI benchmark command)** — CI మరియు మోడల్ మూల్యాంకనం కోసం కమాండ్ లైన్ నుండి ఏజెంట్ సామర్థ్య బెంచ్‌మార్క్ సూట్‌ను అమలు చేయడం.

## అవసరాలు (Requirements)

- WordPress 6.0 లేదా అంతకంటే ఎక్కువ
- PHP 8.1 లేదా అంతకంటే ఎక్కువ
- WordPress Multisite (network-activated) లేదా single-site ఇన్‌స్టాలేషన్
- ఒక క్రియాశీల AI ప్రొవైడర్ కనెక్టర్ ప్లగిన్ (Anthropic, OpenAI, లేదా అనుకూలమైనది)

## ఇన్‌స్టాలేషన్ (Installation)

1. ప్లగిన్ ఫైళ్లను `/wp-content/plugins/gratis-ai-agent` కు అప్‌లోడ్ చేయండి.
2. WordPress లోని **Plugins** స్క్రీన్ ద్వారా ప్లగిన్‌ను యాక్టివేట్ చేయండి.
3. **Gratis AI Agent → Settings** కు నావిగేట్ చేసి మీ AI ప్రొవైడర్‌ను కనెక్ట్ చేయండి.
4. (ఐచ్ఛికం) అదనపు సామర్థ్య ప్యాక్‌లను ఇన్‌స్టాల్ చేయడానికి **Abilities Registry** ను బ్రౌజ్ చేయండి.

## త్వరిత ప్రారంభం (Quick Start)

యాక్టివేట్ చేసి, AI ప్రొవైడర్‌కు కనెక్ట్ చేసిన తర్వాత, WordPress అడ్మిన్ సైడ్‌బార్‌లో **Gratis AI Agent** ప్యానెల్‌ను తెరవండి. సహజ భాషా సూచనను టైప్ చేయండి, ఏజెంట్ అందుబాటులో ఉన్న సామర్థ్యాలను ఉపయోగించి అవసరమైన దశలను ప్లాన్ చేసి అమలు చేస్తుంది.

**ఉదాహరణ సూచనలు:**

```
Create a "Portfolio" custom post type with a "Project Category" taxonomy.
```

```
Inject a dark-mode colour palette into the theme and set the primary colour to #1a1a2e.
```

```
Build a restaurant website with an online menu, booking form, and contact page.
```

## సామర్థ్యాల అవలోకనం (Abilities Overview)

సామర్థ్యాలు (Abilities) అంటే ఏజెంట్ చేయగలిగే అణు చర్యలు (atomic actions). ప్రతి సామర్థ్యం అనేది ఒక రిజిస్టర్ చేయబడిన PHP class. ఇది ఒక స్కీమాను (parameters మరియు return types) అందిస్తుంది, దీనిని ఏజెంట్ రన్‌టైమ్‌లో చదువుతుంది. సామర్థ్యాలను క్రింది క్రియాత్మక ప్రాంతాలలో సమూహపరుస్తారు:

| Area | Abilities |
|---|---|
| **Content Management** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Visual Review** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Design System** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Navigation Menus** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Options Management** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Installable Abilities** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

పూర్తి పారామీటర్ డాక్యుమెంటేషన్ మరియు వినియోగ ఉదాహరణల కోసం [Abilities Reference](./abilities) చూడండి.

## WP-CLI

Gratis AI Agent కమాండ్ లైన్ నుండి బెంచ్‌మార్క్‌లను అమలు చేయడానికి మరియు సామర్థ్యాలను నిర్వహించడానికి `wp gratis-ai-agent` కమాండ్ ఫ్యామిలీని అందిస్తుంది. పూర్తి కమాండ్ జాబితా కోసం [WP-CLI Reference](./wp-cli) చూడండి.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## కాన్ఫిగరేషన్ (Configuration)

అన్ని సెట్టింగ్‌లు WordPress అడ్మిన్‌లో **Gratis AI Agent → Settings** లో ఉన్నాయి:

| Setting | Description |
|---|---|
| **AI Provider** | క్రియాశీల AI ప్రొవైడర్ కనెక్టర్ ఎంచుకుని కాన్ఫిగర్ చేయండి |
| **Model** | ఏజెంట్ సెషన్‌ల కోసం ఉపయోగించాల్సిన భాషా మోడల్‌ను ఎంచుకోండి |
| **Max Tokens** | ప్రతి-అభ్యర్థన టోకెన్ బడ్జెట్ (డిఫాల్ట్ 4096) |
| **Ability Timeout** | ఒక వ్యక్తిగత సామర్థ్యం గరిష్టంగా ఎంత సెకన్ల వరకు రన్ అవుతుంది (డిఫాల్ట్ 30) |
| **Debug Logging** | సామర్థ్య కాల్‌లు మరియు ఏజెంట్ తర్కాన్ని (reasoning) WordPress డీబగ్ లాగ్‌లో లాగ్ చేయండి |

## Options Management Safety Blocklist

Options Management సామర్థ్యం అనేది ఏజెంట్ సున్నితమైన WordPress ఆప్షన్స్‌ను చదవడం లేదా రాయకుండా నిరోధించే అంతర్నిర్మిత బ్లాక్‌లిస్ట్‌ను కలిగి ఉంటుంది. డిఫాల్ట్ బ్లాక్‌లిస్ట్ కవర్ చేస్తుంది:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL ఆప్షన్స్ — వీటి కోసం ప్రత్యేక మైగ్రేషన్ టూల్‌ను ఉపయోగించండి)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (సీక్రెట్ కీలు)
- `active_plugins`, `template` (ప్లగిన్/థీమ్ యాక్టివేషన్ — ప్రత్యేక సామర్థ్యాలను ఉపయోగించండి)
- ఇన్‌స్టాల్ చేయబడిన ప్లగిన్‌ల ద్వారా రిజిస్టర్ చేయబడిన పేమెంట్ గేట్‌వే క్రెడెన్షియల్స్ మరియు API కీలు

అదనపు ఎంట్రీలను `gratis_ai_agent_options_blocklist` ఫిల్టర్ ద్వారా జోడించవచ్చు.

## Hooks and Filters

### Actions

- `gratis_ai_agent_ability_registered` — ఒక సామర్థ్యం రిజిస్టర్ అయిన తర్వాత ట్రిగ్గర్ అవుతుంది, సామర్థ్య క్లాస్ పేరును అందుకుంటుంది.
- `gratis_ai_agent_plan_started` — సైట్ బిల్డర్ ప్లాన్ అమలు కావడానికి ప్రారంభమైనప్పుడు ట్రిగ్గర్ అవుతుంది.
- `gratis_ai_agent_plan_completed` — ఒక ప్లాన్ విజయవంతంగా పూర్తయినప్పుడు ట్రిగ్గర్ అవుతుంది.
- `gratis_ai_agent_plan_failed` — ఒక ప్లాన్ రికవరీ చేయలేని ఎర్రర్‌ను ఎదుర్కొన్నప్పుడు ట్రిగ్గర్ అవుతుంది.

### Filters

- `gratis_ai_agent_abilities` — ఏజెంట్ లోడ్ చేయడానికి ముందు రిజిస్టర్ చేయబడిన సామర్థ్య జాబితాను మార్చండి.
- `gratis_ai_agent_options_blocklist` — ఆప్షన్స్ సేఫ్టీ బ్లాక్‌లిస్ట్‌ను విస్తరించండి లేదా తగ్గించండి.
- `gratis_ai_agent_benchmark_questions` — మూల్యాంకన సూట్‌లో కస్టమ్ బెంచ్‌మార్క్ ప్రశ్నలను జోడించండి.
- `gratis_ai_agent_plan_plugins` — సైట్ ప్లాన్ జనరేషన్ సమయంలో ప్లగిన్ డిస్కవరీ జాబితాను మార్చండి.
