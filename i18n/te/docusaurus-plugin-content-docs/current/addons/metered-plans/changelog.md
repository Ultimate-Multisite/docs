# వినియోగ ఆధారిత ప్లాన్‌ల మార్పుల వివరాలు

Version 1.1.0 - విడుదల చేయబడింది 2026-05-05
- కొత్తగా: మల్టీసైట్ సబ్‌సైట్‌ల కోసం AI టోకెన్ బిల్లింగ్ — కాన్ఫిగర్ చేయదగిన ప్రతి-టోకెన్ రేట్ల ద్వారా కస్టమర్ సైట్‌ల అంతటా AI టోకెన్ వినియోగాన్ని ట్రాక్ చేసి బిల్ చేయవచ్చు.
- కొత్తగా: కనెక్టర్ అమలును డైనమిక్ లిమిట్ డిస్కవరీ మరియు రైట్-త్రూతో పునర్నిర్మించారు, అన్ని కనెక్టర్లలో రియల్-టైమ్ ఖచ్చితత్వాన్ని నిర్ధారిస్తుంది.
- పరిష్కారం: డేటాబేస్ టేబుల్ అప్‌గ్రేడ్‌లు ఇప్పుడు బర్లిన్‌డిబి స్కీమా నిర్వచనాలతో సరిగ్గా అమర్చబడ్డాయి, కొత్త ఇన్‌స్టాలేషన్‌లలో అప్‌గ్రేడ్ వైఫల్యాలను నిరోధిస్తుంది.
- పరిష్కారం: డేటాబేస్ అప్‌గ్రేడ్ కాల్‌బ్యాక్‌లను సరైన ఫార్మాట్‌లోకి మార్చడం జరిగింది, ఇది సైలెంట్ అప్‌గ్రేడ్ వైఫల్యాలను పరిష్కరిస్తుంది.
- పరిష్కారం: AI వినియోగ అదనపు మార్కుప్ (Overage Markup) ఇన్‌పుట్ ఫీల్డ్‌లో ఇప్పుడు భిన్నమైన విలువలను (Fractional values) అంగీకరిస్తుంది.
- పరిష్కారం: ప్లగిన్ లోడ్ అయినప్పుడు జరిగిన తీవ్రమైన లోపాలు (fatal errors) మరియు డబుల్-ఇనిషియలైజేషన్ సమస్యలను పరిష్కరించబడింది.
- మెరుగుపరచబడింది: డెవలపర్ ఎన్విరాన్‌మెంట్‌లు మొదటిసారి రన్ అయినప్పుడు సెల్ఫ్-కాన్ఫిగర్ అయ్యేలా check-env npm స్క్రిప్ట్‌ను జోడించబడింది.
