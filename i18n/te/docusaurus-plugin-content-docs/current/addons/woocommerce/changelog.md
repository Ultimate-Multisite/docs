# WooCommerce Integration మార్పుల చిట్టా {#woocommerce-integration-changelog}

సంచిక 2.2.0 - 2026-07-01న విడుదలైంది
* కొత్తది: Ultimate Multisite పన్ను మొత్తాలు ఇప్పుడు checkout వద్ద ప్రత్యేక WooCommerce రుసుము వరుసలుగా కనిపిస్తాయి. చెల్లింపుకు ముందు పన్ను మొత్తాలు మరింత స్పష్టంగా ఉంటాయి.
* కొత్తది: WooCommerce Subscriptions మళ్లీ ప్రయత్నించే వ్యవధిలో, విఫలమైన పునరుద్ధరణలప్పుడు నిలిపివేత కావాలనుకునే సైట్ల కోసం "విఫలమైన పునరుద్ధరణలపై సభ్యత్వాలను వెంటనే నిలిపివేయి" అనే ఐచ్ఛిక సెట్టింగ్ మరియు `wu_woo_suspend_on_payment_failure` ఫిల్టర్ జోడించబడింది.
* పరిష్కారం: విఫలమైన లేదా తిరిగి సరిచేసిన పునరుద్ధరణల తర్వాత Ultimate Multisite సభ్యత్వాలతో సమకాలీకరణ తప్పిపోయే WooCommerce Subscription స్థితులను సరిపోల్చాం.
* పరిష్కారం: WooCommerce స్టోర్ కరెన్సీ Ultimate Multisite కరెన్సీ జాబితాలో లేకపోతే దానిని జోడించాం.
* పరిష్కారం: సబ్‌స్క్రైబర్లను WooCommerce checkoutకు మళ్లించేటప్పుడు కస్టమర్ బిల్లింగ్ వివరాలను అలాగే ఉంచాం.
* మెరుగుదల: Jetpack Autoloader 5తో అనుకూలత జోడించబడింది.
* మెరుగుదల: GitHub మరియు మార్కెట్‌ప్లేస్ జిప్‌లలో పొరలుగా ఉన్న స్టేజింగ్ డైరెక్టరీలు, డెవలప్‌మెంట్ ఫైళ్లు లేకుండా విడుదల ప్యాకేజీ తయారీని శుభ్రపరిచాం.
