# Multi-Currency

Ultimate Multisite: Multi-Currency మీ నెట్‌వర్క్‌కు బహుళ కరెన్సీలలో చెల్లింపులు స్వీకరించడానికి సహాయపడుతుంది. ఇది లైవ్ లేదా మాన్యువల్ ఎక్స్ఛేంజ్ రేట్లని ఉపయోగించి ధరలను ఆటోమేటిక్‌గా మారుస్తుంది, ఏ కరెన్సీకైనా ఉత్పత్తికి ఖచ్చితమైన ధరలను సెట్ చేయడానికి అనుమతిస్తుంది, మరియు జియోలోకేషన్ ద్వారా ప్రతి సందర్శకుడికి ఇష్టమైన కరెన్సీని గుర్తించగలదు — ఇవన్నీ ఒకే అడ్మిన్ డాష్‌బోర్డ్ నుండి నిర్వహించవచ్చు.

## ముఖ్య లక్షణాలు (Key Features) {#key-features}

- **ఆటోమేటిక్ ఎక్స్ఛేంజ్ రేట్ కన్వర్షన్** — మీరు ఎంచుకున్న ప్రొవైడర్ నుండి తీసుకున్న రేట్లను ఉపయోగించి మీ బేస్ కరెన్సీ నుండి ధరలను మారుస్తుంది.
- **ఉత్పత్తి వారీగా ధరల ఓవర్‌రైడ్స్** — ఆటోమేటిక్ కన్వర్షన్‌ను దాటవేయడానికి ఏ ఉత్పత్తిపై అయినా కరెన్సీ-నిర్దిష్ట ఖచ్చితమైన ధరలను సెట్ చేయవచ్చు.
- **కరెన్సీ సెలెక్టర్ సైన్అప్ ఫీల్డ్** — చెక్అవుట్/సైన్అప్‌ సమయంలో కస్టమర్‌లు తమకు ఇష్టమైన కరెన్సీని ఎంచుకోవడానికి ఇది వీలు కల్పిస్తుంది.
- **బహుళ ఎక్స్ఛేంజ్ రేట్ ప్రొవైడర్లు** — ECB, Frankfurter, Stripe, లేదా పూర్తిగా మాన్యువల్ రేట్లు.
- **జియోలోకేషన్ ఆధారిత కరెన్సీ గుర్తింపు** — సందర్శకుడి దేశంతో సరిపోయే కరెన్సీని ఆటోమేటిక్‌గా ప్రీ-సెలెక్ట్ చేస్తుంది.
- **అడ్మిన్ డాష్‌బోర్డ్** — యాక్టివ్ కరెన్సీలు, ఎక్స్ఛేంజ్ రేట్లు మరియు ప్రొవైడర్ సెట్టింగ్‌లను ఒకే చోట నిర్వహించండి.

## అవసరాలు (Requirements) {#requirements}

- Ultimate Multisite 2.0.0 లేదా అంతకంటే ఎక్కువ
- WordPress Multisite ఇన్‌స్టాలేషన్
- PHP 7.4 లేదా అంతకంటే ఎక్కువ

## ఇన్‌స్టాలేషన్ (Installation) {#installation}

1. ప్లగిన్ ఫైళ్లను `/wp-content/plugins/ultimate-multisite-multi-currency` కు అప్‌లోడ్ చేయండి.
2. WordPress లోని **Plugins** స్క్రీన్ ద్వారా ప్లగిన్‌ను యాక్టివేట్ చేయండి (Network Activate).
3. అడ్మిన్ డాష్‌బోర్డ్‌ను తెరవడానికి **Ultimate Multisite → Multi-Currency** కు నావిగేట్ చేయండి.
4. మీరు స్వీకరించాలనుకునే కరెన్సీలను జోడించి, మీ ఎక్స్ఛేంజ్ రేట్ ప్రొవైడర్‌ను కాన్ఫిగర్ చేయండి.

## కాన్ఫిగరేషన్ (Configuration) {#configuration}

### అడ్మిన్ డాష్‌బోర్డ్ (Admin Dashboard) {#admin-dashboard}

Multi-Currency అడ్మిన్ డాష్‌బోర్డ్ (**Ultimate Multisite → Multi-Currency**) అనేది ఈ క్రింది వాటికి కేంద్ర స్థానం:

- కరెన్సీలను జోడించడం, ఎనేబుల్ చేయడం లేదా డిసేబుల్ చేయడం.
- ఎక్స్ఛేంజ్ రేట్ ప్రొవైడర్‌ను ఎంచుకోవడం మరియు అవసరమైనప్పుడు రేట్లను రిఫ్రెష్ చేయడం.
- అన్ని యాక్టివ్ కరెన్సీల ప్రస్తుత రేట్లను సమీక్షించడం.
- ఆటోమేటిక్ ప్రొవైడర్లు సరిపోనప్పుడు మాన్యువల్ రేట్లను సెట్ చేయడం.

### ఎక్స్ఛేంజ్ రేట్ ప్రొవైడర్లు (Exchange Rate Providers) {#exchange-rate-providers}

| ప్రొవైడర్ | వివరణ |
|---|---|
| **ECB** | యూరోపియన్ సెంట్రల్ బ్యాంక్ — EUR ఆధారిత నెట్‌వర్క్‌ల కోసం ఉచిత రోజువారీ రేట్లు. |
| **Frankfurter** | ఓపెన్-సోర్స్ ECB మిర్రర్ API — ఉచితం, API కీ అవసరం లేదు. |
| **Stripe** | Stripe యొక్క లైవ్ FX రేట్లను ఉపయోగిస్తుంది — యాక్టివ్ Stripe ఇంటిగ్రేషన్ అవసరం. |
| **Manual** | మీరు నేరుగా రేట్లను నమోదు చేస్తారు; ఎటువంటి బాహ్య API కాల్స్ అవసరం లేదు. |

**Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider** కింద మీ ప్రొవైడర్‌ను ఎంచుకోండి. రేట్లను ఎప్పుడైనా మాన్యువల్‌గా రిఫ్రెష్ చేయవచ్చు లేదా ఆటోమేటిక్‌గా అప్‌డేట్ అయ్యేలా షెడ్యూల్ చేయవచ్చు.

### ఉత్పత్తి వారీగా ధరల ఓవర్‌రైడ్స్ (Per-Product Pricing Overrides) {#per-product-pricing-overrides}

ఆటోమేటిక్ కన్వర్షన్ సరిపోనంత ఖచ్చితంగా లేనప్పుడు, మీరు ఏ ఉత్పత్తిపై అయినా నిర్దిష్ట కరెన్సీ కోసం ఖచ్చితమైన ధరను సెట్ చేయవచ్చు:

1. ఉత్పత్తి ఎడిటర్‌ను తెరవండి (**Ultimate Multisite → Products → Edit**).
2. **Currency Pricing** సెక్షన్‌కు స్క్రోల్ చేయండి.
3. మీరు ఓవర్‌రైడ్ చేయాలనుకునే ప్రతి కరెన్సీ కోసం ఖచ్చితమైన ధరను నమోదు చేయండి.
4. ఉత్పత్తిని సేవ్ చేయండి.

ఓవర్‌రైడ్ లేని ఉత్పత్తులు మీ బేస్ కరెన్సీ నుండి కన్వర్ట్ చేయబడిన ధరను ఉపయోగిస్తాయి.

### కరెన్సీ సెలెక్టర్ సైన్అప్ ఫీల్డ్ (Currency Selector Signup Field) {#currency-selector-signup-field}

కస్టమర్‌లు రిజిస్ట్రేషన్ సమయంలో తమకు ఇష్టమైన కరెన్సీని ఎంచుకునేలా కరెన్సీ సెలెక్టర్ ఫీల్డ్‌ను మీ సైన్అప్/చెక్అవుట్ ఫ్లోలో జోడించవచ్చు:

1. **Ultimate Multisite → Multi-Currency → Settings** కు వెళ్లండి.
2. **Show currency selector on signup** ను ఎనేబుల్ చేయండి.
3. ఐచ్ఛికంగా డిఫాల్ట్ కరెన్సీని సెట్ చేయవచ్చు లేదా జియోలోకేషన్ ద్వారా ప్రీ-సెలెక్ట్ చేయబడిన విలువను నిర్ణయించేలా చేయవచ్చు.

ఎంచుకున్న కరెన్సీని సబ్‌స్క్రిప్షన్‌తో పాటు నిల్వ చేస్తారు మరియు తదుపరి అన్ని బిల్లింగ్‌ల కోసం ఉపయోగిస్తారు.

### జియోలోకేషన్ ఆధారిత కరెన్సీ గుర్తింపు (Geolocation-Based Currency Detection) {#geolocation-based-currency-detection}

జియోలోకేషన్ డిటెక్షన్ ఎనేబుల్ అయినప్పుడు, ఈ యాడ్-ఆన్ సందర్శకుడి దేశాన్ని కరెన్సీకి మ్యాప్ చేసి, కరెన్సీ సెలెక్టర్ ఫీల్డ్‌లో దానిని ప్రీ-సెలెక్ట్ చేస్తుంది. ఎనేబుల్ చేయడానికి:

1. **Ultimate Multisite → Multi-Currency → Settings** కు వెళ్లండి.
2. **Detect currency from visitor location** ను ఎనేబుల్ చేయండి.
3. **Geolocation Mapping** టేబుల్‌లో దేశాలను కరెన్సీలకు మ్యాప్ చేయండి.

సందర్శకుడి దేశానికి మ్యాపింగ్ లేకపోతే, నెట్‌వర్క్ యొక్క బేస్ కరెన్సీని ఫాల్‌బ్యాక్‌గా ఉపయోగిస్తుంది.

## తరచుగా అడిగే ప్రశ్నలు (Frequently Asked Questions) {#frequently-asked-questions}

### ఎక్స్ఛేంజ్ రేట్ ప్రొవైడర్ అందుబాటులో లేకపోతే ఏమి జరుగుతుంది? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

యాడ్-ఆన్ చివరిగా విజయవంతంగా తీసుకున్న రేట్లకు ఫాల్‌బ్యాక్ అవుతుంది. ఎప్పుడూ రేట్లు తీసుకోకపోతే, మీరు కాన్ఫిగర్ చేసిన మాన్యువల్ రేట్లకు ఫాల్‌బ్యాక్ అవుతుంది. భద్రతా వలయంలా మాన్యువల్ రేట్లను కాన్ఫిగర్ చేయాలని సిఫార్సు చేయబడింది.

### నేను వేర్వేరు కరెన్సీల కోసం వేర్వేరు ప్రొవైడర్లను ఉపయోగించగలనా? {#can-i-use-different-providers-for-different-currencies}

లేదు — అన్ని కరెన్సీల కోసం ఒక ప్రొవైడర్ మాత్రమే యాక్టివ్‌గా ఉంటుంది. వ్యక్తిగత రేట్లపై పూర్తి నియంత్రణ కావాలంటే **Manual** ను ఉపయోగించండి.

### జియోలోకేషన్ కోసం ప్రత్యేక ప్లగిన్ అవసరమా? {#does-geolocation-require-a-separate-plugin}

బాహ్య జియోలోకేషన్ ప్లగిన్ అవసరం లేదు. యాడ్-ఆన్ వర్డ్‌ప్రెస్ యొక్క అంతర్నిర్మిత IP-టు-కంట్రీ డిటెక్షన్ ను ఉపయోగిస్తుంది. ఎక్కువ ఖచ్చితత్వం కోసం, అందించిన ఫిల్టర్ హుక్స్ ద్వారా ప్రత్యేక జియోలోకేషన్ సేవను ఇంటిగ్రేట్ చేయవచ్చు.

### ఉత్పత్తి ఓవర్‌రైడ్స్ పన్నుకు ముందునా లేక తర్వాతనా వర్తిస్తాయి? {#are-per-product-overrides-applied-before-or-after-tax}

ఉత్పత్తి ఓవర్‌రైడ్స్ టార్గెట్ కరెన్సీలో ప్రీ-ట్యాక్స్ ధరను సెట్ చేస్తాయి. మీ ప్రస్తుత పన్ను కాన్ఫిగరేషన్ ఉపయోగించి ఓవర్‌రైడ్ ధరపై పన్ను లెక్కించబడుతుంది.
