# ఈమెయిల్స్ మరియు బ్రాడ్‌కాస్ట్‌లు పంపడం (v2)

_**ముఖ్యమైన గమనిక: ఈ వ్యాసం Ultimate Multisite వెర్షన్ 2.x కి సంబంధించినది.**_

Ultimate Multisite లో మీ కస్టమర్లతో కమ్యూనికేట్ చేయడానికి ఒక ఫీచర్ ఉంది. దీని ద్వారా మీరు ఒక నిర్దిష్ట యూజర్‌కు లేదా యూజర్ల సమూహానికి ఈమెయిల్ పంపవచ్చు, అలాగే వారి admin dashboard లో ప్రకటనలను broadcast చేయడానికి నోటీసులు పంపవచ్చు.

## Broadcasts తో మీ కస్టమర్ల dashboard లో admin notices జోడించడం

Ultimate Multisite broadcast ఫీచర్ ఉపయోగించి, మీరు మీ యూజర్ల subsite admin dashboard లో **admin notices** జోడించవచ్చు.

సిస్టమ్ మెయింటెనెన్స్ గురించి ప్రకటన చేయాల్సినప్పుడు లేదా మీ ప్రస్తుత యూజర్లకు కొత్త ప్రొడక్ట్‌లు లేదా సర్వీసులు ఆఫర్ చేయాల్సినప్పుడు ఇది చాలా ఉపయోగకరంగా ఉంటుంది. మీ యూజర్ dashboard లో admin notice ఇలా కనిపిస్తుంది.

![కస్టమర్ dashboard లో చూపబడిన Admin notice broadcast](/img/admin/broadcasts-list.png)

admin notice ప్రారంభించడానికి, మీ network admin dashboard కి వెళ్ళి, **Ultimate Multisite** మెనూ కింద **Broadcasts** ఆప్షన్ కనుగొనండి.

![Ultimate Multisite admin లో Broadcasts మెనూ](/img/admin/broadcasts-list.png)

ఈ పేజీ నుండి, పై భాగంలో ఉన్న **Add Broadcast** బటన్ క్లిక్ చేయండి.

ఇది Add broadcast modal window తెరుస్తుంది, ఇక్కడ మీరు ఏ రకమైన broadcast పంపాలనుకుంటున్నారో ఎంచుకోవచ్చు.

**Message** ఎంచుకుని, **Next Step** బటన్ క్లిక్ చేయండి.

![Message type ఎంచుకున్న Add broadcast modal](/img/admin/broadcasts-list.png)

తదుపరి విండో మిమ్మల్ని **Target customer** లేదా **Target product** అడుగుతుంది. మీరు ఒకటి కంటే ఎక్కువ యూజర్లను లేదా ఒకటి కంటే ఎక్కువ ప్రొడక్ట్‌లను ఎంచుకోవచ్చని గమనించండి.

యూజర్ అకౌంట్ లేదా ప్రొడక్ట్ కోసం వెతకడానికి, ఫీల్డ్ లోపల keyword టైప్ చేయడం ప్రారంభించాలి.

**Message type** ఫీల్డ్ కింద, మీరు notice యొక్క రంగు ఎంచుకోవచ్చు. ఇది మీ మెసేజ్ యొక్క ఆవశ్యకతను నొక్కి చెబుతుంది.

ఆ తర్వాత **Next Step** క్లిక్ చేయవచ్చు.

![Broadcast కోసం Target customer మరియు product ఎంపిక](/img/admin/broadcasts-list.png)

తదుపరి విండో లో మీరు subject మరియు యూజర్లకు broadcast చేయాలనుకునే content/message ఎంటర్ చేసి మీ మెసేజ్ compose చేయడం ప్రారంభించవచ్చు.

![Broadcast message subject మరియు content editor](/img/admin/broadcasts-list.png)

మీ మెసేజ్ రాసిన తర్వాత, **Send** బటన్ నొక్కవచ్చు.

అంతే. admin notice వెంటనే మీ యూజర్ dashboard లో చూపబడుతుంది.

## మీ కస్టమర్లకు ఈమెయిల్స్ పంపడం

Ultimate Multisite broadcast ఫీచర్ ఉపయోగించి, మీరు మీ యూజర్లకు ఈమెయిల్ పంపవచ్చు. నిర్దిష్ట యూజర్లకు మాత్రమే ఈమెయిల్ పంపడం లేదా వారు subscribe చేసిన product లేదా plan ఆధారంగా నిర్దిష్ట యూజర్ గ్రూప్‌ను target చేయడం వంటి ఆప్షన్లు మీకు ఉన్నాయి.

ఈమెయిల్ broadcast ప్రారంభించడానికి, మీ network admin dashboard కి వెళ్ళి, Ultimate Multisite మెనూ కింద Broadcast ఆప్షన్ కనుగొనండి.

![Ultimate Multisite admin లో Broadcasts పేజీ](/img/admin/broadcasts-list.png)

ఈ పేజీ నుండి, పై భాగంలో ఉన్న **Add broadcast** బటన్ క్లిక్ చేయండి.

ఇది Add broadcast modal window తెరుస్తుంది, ఇక్కడ మీరు ఏ రకమైన broadcast పంపాలనుకుంటున్నారో ఎంచుకోవచ్చు. **Email** ఎంచుకుని, **Next Step** బటన్ క్లిక్ చేయండి.

![Email type ఎంచుకున్న Add broadcast modal](/img/admin/broadcasts-list.png)

తదుపరి విండో మిమ్మల్ని **Target customer** లేదా **Target product** అడుగుతుంది. మీరు ఒకటి కంటే ఎక్కువ యూజర్లను లేదా ఒకటి కంటే ఎక్కువ ప్రొడక్ట్‌లను ఎంచుకోవచ్చని గమనించండి.

యూజర్ అకౌంట్ లేదా ప్రొడక్ట్ కోసం వెతకడానికి, ఫీల్డ్ లోపల keyword టైప్ చేయడం ప్రారంభించాలి.

మీ target audience ఎంచుకున్న తర్వాత, **Next Step** క్లిక్ చేయవచ్చు.

![ఈమెయిల్ broadcast కోసం Target customer మరియు product ఎంపిక](/img/admin/broadcasts-list.png)

తదుపరి విండో లో మీరు subject మరియు యూజర్లకు పంపాలనుకునే content/message ఎంటర్ చేసి మీ ఈమెయిల్ compose చేయడం ప్రారంభించవచ్చు.

![ఈమెయిల్ broadcast subject మరియు content editor](/img/admin/broadcasts-list.png)

మీ మెసేజ్ రాసిన తర్వాత, **Send** బటన్ నొక్కవచ్చు.

broadcast ఫీచర్ ఉపయోగించి మీ end-users కి ఈమెయిల్ పంపడం ఇంత సులభం.

## System emails

Ultimate Multisite లో System emails అంటే registration, payment, domain mapping వంటి కొన్ని actions తర్వాత సిస్టమ్ పంపే **ఆటోమేటిక్ నోటిఫికేషన్లు**. ఈ ఈమెయిల్స్‌ను Ultimate Multisite settings నుండి edit లేదా modify చేయవచ్చు. మరొక Ultimate Multisite installation నుండి ఇప్పటికే ఉన్న settings ను reset మరియు import చేయడానికి కూడా ఒక ఫీచర్ ఉంది.

### Resetting & Importing

కొత్త Ultimate Multisite వెర్షన్లు, అలాగే add-ons, ఎప్పటికప్పుడు కొత్త emails register చేయవచ్చు మరియు చేస్తాయి.

conflicts మరియు ఇతర issues నివారించడానికి, **మీ install లో కొత్త email templates ను System Emails గా ఆటోమేటిక్‌గా జోడించము**, అవి ఒక నిర్దిష్ట ఫీచర్ సరిగ్గా పనిచేయడానికి అవసరమైతే తప్ప.

అయితే, super admins మరియు agents ఈ కొత్తగా register అయిన emails ను importer tool ద్వారా import చేయవచ్చు. ఈ ప్రక్రియ కొత్త email template యొక్క content మరియు configuration తో కొత్త system email సృష్టిస్తుంది, super admin కి వారు కోరుకున్న modifications చేయడానికి లేదా అవి ఉన్నట్లే ఉంచడానికి అనుమతిస్తుంది.

#### System emails ఎలా import చేయాలి

మీ Ultimate Multisite Settings పేజీకి వెళ్ళి, **Emails** tab కి వెళ్ళండి.

![Ultimate Multisite settings లో Emails tab](/img/config/settings-emails.png)

ఆ తర్వాత, sidebar లో **Customize System Emails** బటన్ క్లిక్ చేయండి.

![Sidebar లో Customize System Emails బటన్](/img/config/settings-emails.png)

System Emails పేజీ లో, పై భాగంలో **Reset & Import** action బటన్ కనిపిస్తుంది. ఆ బటన్ క్లిక్ చేస్తే import మరియు reset modal window తెరుచుకుంటుంది.

![System Emails పేజీ లో Reset and Import action బటన్](/img/config/settings-emails.png)

ఆ తర్వాత, ఏ system emails import చేయడానికి అందుబాటులో ఉన్నాయో చూడటానికి Import Emails options toggle చేయవచ్చు.

![అందుబాటులో ఉన్న system emails చూపే Import Emails options](/img/config/settings-emails.png)

#### System Emails Reset చేయడం

కొన్నిసార్లు, మీరు ఒక email template కి చేసిన మార్పులు మీకు ఇక పనికిరావడం లేదని గ్రహించి, దానిని వాటి **default state** కి reset చేయాలనుకుంటారు.

అలాంటి సందర్భాల్లో, మీకు రెండు ఆప్షన్లు ఉన్నాయి: మీరు system email ను delete చేసి మళ్ళీ import చేయవచ్చు (పై instructions ఉపయోగించి) - ఇది send metrics మరియు ఇతర విషయాలను తొలగిస్తుంది, కాబట్టి ఈ method తక్కువ preferred.

లేదా మీరు ఆ email template reset చేయడానికి **Reset & Import tool** ఉపయోగించవచ్చు.

email template reset చేయడానికి, మీరు Reset & Import tool చేరుకునే వరకు పై steps అనుసరించవచ్చు, ఆ తర్వాత, **Reset** option toggle చేసి వాటి default content కి reset చేయాలనుకునే emails ఎంచుకోండి.

![Email templates ను defaults కి restore చేయడానికి Reset option](/img/config/settings-emails.png)
