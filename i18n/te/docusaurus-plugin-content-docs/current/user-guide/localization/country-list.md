# Ultimate Multisite దేశాల జాబితాను పునఃక్రమీకరించడం

దేశాన్ని ఎంచుకునే dropdown ఇంగ్లీష్ దేశ పేర్ల ఆధారంగా అక్షర క్రమంలో అమర్చబడి ఉంటుంది. దీని వల్ల, కొన్నిసార్లు మీ కంపెనీ స్వదేశం మరియు/లేదా మీ చాలా మంది కస్టమర్ల దేశం పెద్ద దేశాల జాబితా మధ్యలో కనుగొనడం కష్టంగా ఉంటుంది.

ఈ సమస్యను పరిష్కరించి, మీకు కావలసిన దేశాన్ని ఎంపిక జాబితా పైభాగంలో చేర్చడానికి, క్రింది కోడ్ స్నిప్పెట్‌ను ఉపయోగించవచ్చు (దీన్ని mu-plugin లో చేర్చవచ్చు):

add_filter('wu_get_countries', function($countries) {

// DE ను మీ దేశం యొక్క రెండు-అక్షరాల కోడ్‌కు మార్చండి

// క్రింద Germany కోసం ఉదాహరణ ఇవ్వబడింది.

unset($countries['DE']);

// DE ను మీ దేశం యొక్క రెండు-అక్షరాల కోడ్‌కు

// మరియు మీ దేశం పేరుకు మార్చండి.

// క్రింద Germany కోసం ఉదాహరణ ఇవ్వబడింది.

$countries = array_merge(array( 'DE' => __('Germany', 'ultimate-multisite'), ), $countries);

return $countries;

});
