# Ultimate Multisite ను అనువదించడం

Ultimate Multisite ఆంగ్లంలో అభివృద్ధి చేయబడింది, కానీ ఈ plugin పూర్తిగా అనువదించదగినది మరియు మేము అనువాద సహకారాలను చురుకుగా స్వాగతిస్తున్నాము.

## Core Plugin అనువాదాలు

Core Ultimate Multisite plugin అధికారిక **WordPress.org అనువాద వేదిక** ద్వారా అనువదించబడుతుంది:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

ఇది ప్రామాణిక WordPress సమాజ అనువాద వ్యవస్థ. అనువాదాలు తగినంత ఆమోద పరిమితికి చేరుకున్నప్పుడు, అవి WordPress plugin అప్‌డేట్‌ల ద్వారా అన్ని Ultimate Multisite వినియోగదారులకు స్వయంచాలకంగా పంపిణీ చేయబడతాయి.

**మీ సహాయం అవసరం!** మీరు ఆంగ్లం కాకుండా వేరే భాష మాట్లాడితే, దయచేసి అనువాదాలు అందించడాన్ని పరిగణించండి. కొన్ని strings ను అనువదించినా కూడా సహాయకరంగా ఉంటుంది. ప్రారంభించడానికి:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) ను సందర్శించండి
2. మీ భాషను ఎంచుకోండి
3. "Contribute Translation" పై క్లిక్ చేయండి
4. మీకు ఉచిత WordPress.org ఖాతా అవసరం

మీ భాష కోసం అనువాద సంపాదకులు (అనువాదాలను ఆమోదించగల సామర్థ్యం ఉన్నవారు) కావాలనుకుంటే, [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) ద్వారా సంప్రదించండి.

## Addon అనువాదాలు

Addon plugins ఇక్కడ వేరుగా అనువదించబడతాయి:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon అనువాదాలు ప్రస్తుతం ప్రారంభ బిందువుగా AI-ద్వారా రూపొందించబడ్డాయి. అవి అన్ని మద్దతు ఉన్న భాషలను కవర్ చేస్తాయి, కానీ అసౌకర్యమైన పదబంధాలు లేదా తప్పులు ఉండవచ్చు. వాటిని మరింత సహజంగా మరియు ఖచ్చితంగా చేయడానికి మానవ అనువాదకులు సమీక్షించి మెరుగుదలలు సూచించడాన్ని మేము స్వాగతిస్తున్నాము.

## స్థానిక అనువాదాలు

మీరు upstream కు సహకరించడం కాకుండా మీ స్వంత ప్రాజెక్ట్ కోసం స్థానికంగా అనువాదాలు సృష్టించాలనుకుంటే, రెండు ఎంపికలు ఉన్నాయి:

### Poedit ఉపయోగించడం

`.pot` ఫైల్‌ను అనువదించడానికి మీరు **[Poedit](https://poedit.net)** డెస్క్‌టాప్ అప్లికేషన్‌ను ఉపయోగించవచ్చు:

1. [poedit.net](https://poedit.net) నుండి Poedit ను డౌన్‌లోడ్ చేయండి
2. `wp-content/plugins/ultimate-multisite/lang/` నుండి `.pot` ఫైల్‌ను తెరవండి
3. Poedit ఇంటర్‌ఫేస్‌లో strings ను అనువదించండి
4. సేవ్ చేయండి -- Poedit `.po` మరియు `.mo` ఫైల్‌లను రూపొందిస్తుంది
5. అవుట్‌పుట్ ఫైల్‌లను `lang` ఫోల్డర్‌లో ఉంచండి

### Loco Translate ఉపయోగించడం

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin ఏ డెస్క్‌టాప్ సాధనాలు లేకుండా నేరుగా admin panel నుండి అనువదించడానికి మిమ్మల్ని అనుమతిస్తుంది. ఇది మీ కోసం అనువాద ఫైల్‌లను సృష్టించడం మరియు కంపైల్ చేయడం నిర్వహిస్తుంది.
