# నా ఖాతా పేజీని మొదటిసారి ఎలా యాక్సెస్ చేయాలి?

Ultimate Multisite కి స్వాగతం!

మా plugin కొనుగోలు చేసిన తర్వాత మీ ఇ-మెయిల్‌లో అందుకున్న లింక్ ద్వారా మీ ఖాతా పేజీని యాక్సెస్ చేయండి:

<!-- Screenshot unavailable: Account page email with access link -->

కొనుగోలు సమయంలో మీరు అందించిన ఇమెయిల్ మరియు మీ ఇమెయిల్‌లో అందుకున్న పాస్‌వర్డ్ ఉపయోగించండి.

<!-- Screenshot unavailable: Login page with email and password fields -->

ఈ ఖాతాలో మీరు మొదటిసారి లాగిన్ అవుతున్నందున, మీ పాస్‌వర్డ్ మార్చమని అడగబడతారు. దాన్ని మార్చి _Save_ బటన్ క్లిక్ చేయండి.

<!-- Screenshot unavailable: Change password form with Save button -->

మీ ఖాతా పేజీలో, బిల్లింగ్‌లు, రెన్యూవల్‌లు, క్యాన్సిలేషన్‌లు మరియు ఇతర అనేక విషయాలు వంటి మీ ఖాతా వివరాలన్నింటినీ నిర్వహించవచ్చు.
