# సైట్ ఎగుమతి (Site Export) {#site-export}

**సైట్ ఎగుమతి** అడ్మిన్ పేజీ నెట్‌వర్క్ అడ్మినిస్ట్రేటర్లకు ఒక సైట్‌ను, లేదా మొత్తం నెట్‌వర్క్‌ను, మైగ్రేషన్, బ్యాకప్ లేదా హ్యాండోవర్ పని కోసం డౌన్‌లోడ్ చేయదగిన ఆర్కైవ్‌గా ప్యాకేజీ చేయడానికి వీలు కల్పిస్తుంది.

## ఒక సైట్‌ను ఎగుమతి చేయడం (Exporting one site) {#exporting-one-site}

**Ultimate Multisite > Site Export** కి వెళ్లి, **Generate new Site Export** ఎంచుకోండి. మీరు ఎగుమతి చేయాలనుకుంటున్న సబ్‌సైట్‌ను ఎంచుకుని, ఆర్కైవ్‌లో అప్‌లోడ్‌లు (uploads), ప్లగిన్‌లు (plugins), మరియు థీమ్‌లు (themes) చేర్చాలో లేదో ఎంచుకోండి.

ఎగుమతి పూర్తయిన తర్వాత, **Existing Exports** జాబితా నుండి ZIP ఫైల్‌ను డౌన్‌లోడ్ చేసుకోండి. ఎగుమతి ZIP ఫైల్‌లలో ఇప్పుడు సెల్ఫ్-బూటింగ్ `index.php` మరియు `readme.txt` ఉంటాయి, కాబట్టి ఈ ఆర్కైవ్‌ను కొత్త హోస్ట్‌కు అప్‌లోడ్ చేసి, ప్రత్యేకమైన ఇంపార్టర్ ప్లగిన్‌ను ఇన్‌స్టాల్ చేయకుండానే ప్రారంభించవచ్చు.

## మొత్తం నెట్‌వర్క్‌ను ఎగుమతి చేయడం (Exporting the whole network) {#exporting-the-whole-network}

నెట్‌వర్క్‌లోని అన్ని సబ్‌సైట్‌లను కలిగి ఉన్న ఒకే ఆర్కైవ్ కావాలంటే, సైట్ ఎగుమతి పేజీలో **Network Export** ఉపయోగించండి. ప్రతి సబ్‌సైట్ కలిసి ప్రయాణించాల్సిన సందర్భాలలో, ముఖ్యంగా హోస్ట్ మైగ్రేషన్స్, డిజాస్టర్-రికవరీ డ్రిల్స్ లేదా స్టేజింగ్ రీబిల్డ్‌ల ముందు ఇది చాలా ఉపయోగకరంగా ఉంటుంది.

నెట్‌వర్క్ ఎగుమతి అనేది ఒకే సైట్ ఎగుమతి కంటే చాలా పెద్దదిగా ఉండవచ్చు, కాబట్టి దీనిని తక్కువ ట్రాఫిక్ ఉన్న సమయంలో రన్ చేయండి మరియు అప్‌లోడ్‌లు, ప్లగిన్‌లు, థీమ్‌లు మరియు రూపొందించబడిన ZIP ఫైల్‌ల కోసం లక్ష్య నిల్వలో (target storage) తగినంత ఖాళీ స్థలం ఉందో లేదో నిర్ధారించుకోండి.

### నెట్‌వర్క్ ఇంపార్ట్ బండిల్స్ (Network Import Bundles) {#network-import-bundles}

Ultimate Multisite 2.12.0 తో ప్రారంభం నుండి, సైట్ ఎగుమతికర్త (Site Exporter) **నెట్‌వర్క్ ఇంపార్ట్ బండిల్స్**ను రూపొందించగలదు — ఇవి నెట్‌వర్క్-వ్యాప్త సైట్ పునరుద్ధరణ కోసం ప్రత్యేకంగా రూపొందించబడిన ఆర్కైవ్‌లు. నెట్‌వర్క్ ఇంపార్ట్ బండిల్‌లో బహుళ సైట్‌లను కొత్త నెట్‌వర్క్ ఇన్‌స్టాలేషన్‌కు పునరుద్ధరించడానికి అవసరమైన అన్ని ఫైళ్లు మరియు మెటాడేటా ఉంటాయి.

#### నెట్‌వర్క్ ఇంపార్ట్ బండిల్‌ను రూపొందించడం (Generating a Network Import Bundle) {#generating-a-network-import-bundle}

1. **Ultimate Multisite > Site Export** కి వెళ్లండి
2. **Generate new Network Export** పై క్లిక్ చేయండి
3. ఎగుమతి రకంగా **Network Import Bundle** ను ఎంచుకోండి
4. అప్‌లోడ్‌లు, ప్లగిన్‌లు, మరియు థీమ్‌లు చేర్చాలా వద్దా అనేది ఎంచుకోండి
5. **Generate** పై క్లిక్ చేయండి
6. **Existing Exports** జాబితా నుండి బండిల్ ZIP ను డౌన్‌లోడ్ చేసుకోండి

#### నెట్‌వర్క్ ఇంపార్ట్ బండిల్ నుండి పునరుద్ధరించడం (Restoring from a Network Import Bundle) {#restoring-from-a-network-import-bundle}

నెట్‌వర్క్ ఇంపార్ట్ బండిల్ నుండి సైట్‌లను పునరుద్ధరించడానికి:

1. మీ లక్ష్య హోస్ట్‌పై Ultimate Multisite ను ఇన్‌స్టాల్ చేయండి
2. మల్టీసైట్ సెటప్ విజర్డ్‌ను పూర్తి చేయండి
3. కొత్త నెట్‌వర్క్‌లో **Ultimate Multisite > Site Export** కి వెళ్లండి
4. **Import Network Bundle** పై క్లిక్ చేయండి
5. నెట్‌వర్క్ ఇంపార్ట్ బండిల్ ZIP ఫైల్‌ను అప్‌లోడ్ చేయండి
6. స్క్రీన్‌పై కనిపించే ఇంపార్ట్ సూచనలను అనుసరించండి
7. ఇంపార్ట్ ప్రక్రియ అన్ని సైట్‌లు, వాటి కంటెంట్ మరియు కాన్ఫిగరేషన్‌లను పునరుద్ధరిస్తుంది

నెట్‌వర్క్ ఇంపార్ట్ బండిల్స్ వీటిని భద్రపరుస్తాయి:
- సైట్ కంటెంట్ (పోస్ట్‌లు, పేజీలు, కస్టమ్ పోస్ట్ రకాలు)
- సైట్ సెట్టింగ్‌లు మరియు ఆప్షన్‌లు
- యూజర్ రోల్స్ మరియు పర్మిషన్లు
- ప్లగిన్‌లు మరియు థీమ్‌లు (బండిల్‌లో చేర్చబడితే)
- మీడియా అప్‌లోడ్‌లు (బండిల్‌లో చేర్చబడితే)
- కస్టమ్ డేటాబేస్ టేబుల్స్ మరియు డేటా

## సెల్ఫ్-బూటింగ్ ఎగుమతి ZIP ను ఇన్‌స్టాల్ చేయడం (Installing a self-booting export ZIP) {#installing-a-self-booting-export-zip}

కొత్త హోస్ట్‌పై సెల్ఫ్-బూటింగ్ ZIP ను పునరుద్ధరించడానికి:

1. ఎగుమతి చేసిన ZIP కంటెంట్‌లను లక్ష్య వెబ్ రూట్‌లో (target web root) అప్‌లోడ్ చేయండి.
2. బ్రౌజర్‌లో అప్‌లోడ్ చేయబడిన `index.php` ను తెరవండి.
3. బండిల్ చేయబడిన ఎగుమతి ప్యాకేజీ నుండి స్క్రీన్‌పై కనిపించే ఇన్‌స్టాలర్ సూచనలను అనుసరించండి.
4. తాత్కాలిక ఫైళ్లను తొలగించే ముందు, ఎగుమతికి సంబంధించిన గమనికల కోసం బండిల్ చేయబడిన `readme.txt` ను సమీక్షించండి.

అదనపు (addon) నిర్దిష్ట ఇన్‌స్టాలేషన్ మరియు ఇంపార్ట్ వివరాల కోసం, [Site Exporter addon documentation](/addons/site-exporter) చూడండి.

Ultimate Multisite 2.9.0 లో జోడించబడిన సింగిల్-సైట్ టూల్ కోసం, [Export & Import](/user-guide/administration/export-import) చూడండి.
