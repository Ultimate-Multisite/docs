# డాష్‌బోర్డ్ అవలోకనం

Ultimate Multisite మెనూ కింద ఒక ప్రత్యేక **Dashboard** ఉంటుంది, అలాగే Network Dashboard కు విడ్జెట్లను జోడిస్తుంది. ఈ ఆర్టికల్‌లో రెండు చోట్లా మీరు ఏమి చూస్తారో వివరిస్తాము.

## Network Dashboard

Ultimate Multisite నెట్‌వర్క్-స్థాయి డాష్‌బోర్డ్‌లో ఉపయోగకరమైన విడ్జెట్లను జోడిస్తుంది, ఇది **My Sites → Network Admin → Dashboard** లో కనుగొనవచ్చు.

![Ultimate Multisite విడ్జెట్లతో Network Dashboard](/img/admin/network-dashboard-full.png)

ప్రతి విడ్జెట్ ఏమి అందిస్తుందో ఇక్కడ చూడండి:

**First Steps:** కొత్త ఇన్‌స్టాలేషన్‌లో కనిపిస్తుంది మరియు చెల్లింపు పద్ధతిని కాన్ఫిగర్ చేయడం, టెస్ట్ ఖాతా సృష్టించడం వంటి సెటప్ టాస్క్‌లను పూర్తి చేయమని గుర్తు చేస్తుంది.

![First Steps విడ్జెట్](/img/admin/network-dashboard-first-steps.png)

**Summary:** ఆ రోజు సైన్‌అప్‌ల సంఖ్య మరియు ఆదాయాన్ని రిపోర్ట్ చేస్తుంది. MRR (Monthly Recurring Revenue) అనేది ఆదాయం-ఉత్పత్తి చేసే మెంబర్‌షిప్‌లు ఉన్న కస్టమర్ల నుండి అంచనా వేసిన మొత్తాన్ని సూచిస్తుంది.

![Summary విడ్జెట్](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite అంతటా జరిగే ఈవెంట్లను రిపోర్ట్ చేస్తుంది — సైన్‌అప్‌లు, రద్దులు, సైట్ సృష్టులు, తొలగింపులు మరియు ఇతర కార్యకలాపాలు.

![Activity Stream విడ్జెట్](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** నెట్‌వర్క్‌లోని యూజర్లు మరియు సైట్ల యొక్క సంక్షిప్త సారాంశాన్ని కొత్త సైట్లు లేదా యూజర్లను సృష్టించడానికి షార్ట్‌కట్లతో ప్రదర్శిస్తుంది.

![Right Now విడ్జెట్](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** అప్‌డేట్లు, క్లిష్టమైన బగ్‌లు మరియు సెక్యూరిటీ ఫిక్సులతో సహా తాజా Ultimate Multisite వార్తలను పొంది ప్రదర్శిస్తుంది.

![News విడ్జెట్](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

Network dashboard నెట్‌వర్క్ గురించి సమగ్ర సమాచారాన్ని అందిస్తే, Ultimate Multisite dashboard (Ultimate Multisite మెనూ యొక్క టాప్-లెవల్‌లో) సర్వీస్ గురించి సమాచారాన్ని అందిస్తుంది.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** 12-నెలల క్యాలెండర్ కాలంలో ఆదాయం యొక్క నెలవారీ విభజనను ప్రదర్శిస్తుంది. కొత్త ఆదాయం నీలం రంగులో చూపబడుతుంది, రద్దులు ఎరుపు రంగులో కనిపిస్తాయి.

![MRR Growth చార్ట్](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** మీకు అవసరమైన సమాచారంపై దృష్టి పెట్టడానికి కాల వ్యవధి ప్రకారం డాష్‌బోర్డ్ డేటాను ఫిల్టర్ చేయండి. ఈ తేదీ పరిధి క్రింద ఉన్న విడ్జెట్లలో ఏమి ప్రదర్శించబడుతుందో నియంత్రిస్తుంది.

![తేదీ పరిధి మరియు డాష్‌బోర్డ్ స్టాట్స్](/img/admin/um-dashboard-date-range.png)

డాష్‌బోర్డ్ విడ్జెట్లలో ఇవి ఉన్నాయి:

- **Revenue** — MRR మరియు ప్రతి-ప్రొడక్ట్ విభజనలతో ఆ కాలానికి మొత్తం ఆదాయం
- **New Memberships** — ఎంచుకున్న కాలంలో ఎన్ని కొత్త మెంబర్‌షిప్‌లు సృష్టించబడ్డాయి
- **Signups by Countries** — సైన్‌అప్‌ల భౌగోళిక విభజన
- **Signups by Forms** — ప్రతి చెకౌట్ ఫారమ్ ద్వారా ఎంత మంది కస్టమర్లు సైన్ అప్ చేశారు vs అడ్మిన్‌లు జోడించినవి
- **Most Visited Sites** — మీ నెట్‌వర్క్‌లో అత్యధికంగా సందర్శించబడిన సైట్లు (**Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** వద్ద విజిట్ కౌంటింగ్‌ను ఎనేబుల్ చేయడం అవసరం)

![డాష్‌బోర్డ్ స్టాటిస్టిక్స్](/img/admin/um-dashboard-stats.png)
