# ఇతర ప్లగిన్‌లతో తెలిసిన అననుకూలతలు

Ultimate Multisite ను ఇన్‌స్టాల్ చేసి యాక్టివేట్ చేసిన తర్వాత, ఇతర ప్లగిన్‌లతో పరిష్కరించని అననుకూలతల వల్ల మీకు సమస్యలు ఎదురు కావచ్చు.

ప్రతి కొత్త రిలీజ్‌తో ఈ జాబితాను తగ్గించడానికి మేము చురుకుగా పని చేస్తున్నాము, కానీ ఒక ప్లగిన్ ఇక్కడ జాబితా చేయబడి ఉంటే, పూర్తిగా పనిచేసే Ultimate Multisite Network కోసం మీరు దాన్ని డీయాక్టివేట్ చేయాల్సి రావచ్చు.

ఇక్కడ వాటి జాబితా మరియు ఈ సందర్భాలలో మీరు ఏమి చేయాలో ఉంది:

**కొన్ని WPMU DEV ప్లగిన్‌లు** WPMU DEV ద్వారా _ఇకపై_ నిర్వహించబడని కొన్ని ప్లగిన్‌లు Ultimate Multisite రిజిస్ట్రేషన్ ప్రక్రియను చివరి దశలో ఆపేస్తాయని తెలిసింది (వినియోగదారు ఖాతాను సృష్టించలేరు). ఆ సమస్యలకు కారణమయ్యే ప్లగిన్‌ల జాబితా ఇక్కడ ఉంది:

  * Signup Language: _Ultimate Multisite లో అదే ఫంక్షనాలిటీని అందించే యాడ్-ఆన్ ఉంది, దాని పేరు Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _ఇది Ultimate Multisite లో అంతర్నిర్మితంగా ఉంది, కాబట్టి అదనపు ప్లగిన్ అవసరం లేదు._

  * New Blog Templates: _ఇది Ultimate Multisite లో అంతర్నిర్మితంగా ఉంది, కాబట్టి అదనపు ప్లగిన్ అవసరం లేదు._

  * Set Password on Multisite Blog Creation: _ఇది Ultimate Multisite లో అంతర్నిర్మితంగా ఉంది, కాబట్టి అదనపు ప్లగిన్ అవసరం లేదు._

  * Signup TOS: _ఇది Ultimate Multisite లో అంతర్నిర్మితంగా ఉంది, కాబట్టి అదనపు ప్లగిన్ అవసరం లేదు._

  * Anti-Splog: తక్కువ శక్తివంతమైనప్పటికీ, ఇది కూడా Ultimate Multisite లో _అంతర్నిర్మితంగా ఉంది._

**గమనిక** : అన్ని ఇతర WPMU Dev ప్లగిన్‌లను Ultimate Multisite తో సాధారణంగా ఉపయోగించవచ్చు. ఉదాహరణలుగా _Smush_ , _Forminator_ , _Defender,_ మొదలైనవి ఉన్నాయి.
