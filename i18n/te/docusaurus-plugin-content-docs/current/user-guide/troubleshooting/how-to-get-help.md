# సహాయం ఎలా పొందాలి

మీకు చెల్లుబాటు అయ్యే License key వచ్చింది, దానితో పాటు Ultimate Multisite సెటప్ చేసేటప్పుడు మీకు ఎదురయ్యే సమస్యలను పరిష్కరించడానికి మా టీమ్ సహాయం కూడా అందుబాటులో ఉంటుంది.

ఈ పాఠంలో license హోల్డర్లకు అందుబాటులో ఉన్న support channels మరియు మా support టీమ్‌ను సంప్రదించడానికి మీ వద్ద ఉండవలసిన సమాచారం గురించి తెలుసుకుందాం.

## Help Site

మా help site నుండి వివిధ ఆర్టికల్స్ మరియు ట్యుటోరియల్స్‌ను యాక్సెస్ చేయవచ్చు, దీని లింక్ క్రింద ఉంది.

[)

Ultimate Multisite యొక్క ప్రాథమిక features మరియు functions గురించి మా వినియోగదారులు తెలుసుకోవాలని మేము ప్రోత్సహిస్తున్నాము. మా help site లో "Getting Started" ఆర్టికల్స్ అందుబాటులో ఉన్నాయి మరియు add-ons ఎలా install చేయాలి, payment gateways ఎలా integrate చేయాలి వంటి సాధారణ ప్రశ్నలకు సమాధానాలు కూడా అక్కడ దొరుకుతాయి.

## Support ను సంప్రదించండి

మీరు ఎక్కడైనా ఇరుక్కుపోయి, మా help site లో సమాధానం కనుగొనలేకపోతే, మా website యొక్క కుడి దిగువ భాగంలో ఉన్న speech bubble icon పై క్లిక్ చేయడం ద్వారా లేదా నేరుగా support@ultimatemultisite.com కు email పంపడం ద్వారా మా support టీమ్‌ను సంప్రదించవచ్చు.

టెక్నికల్ సమస్యల కోసం support ను సంప్రదించేటప్పుడు, మీ సమస్య యొక్క వివరణాత్మక description తో పాటు screenshot, error messages, లేదా మా support వారికి మీ సమస్యను సులభంగా అర్థం చేసుకోవడానికి మరియు మూల్యాంకనం చేయడానికి సహాయపడే ఏదైనా సమాచారాన్ని అందించండి. మీరు మీ license key కూడా అందించాలి. యాక్టివ్ license హోల్డర్లకు మాత్రమే support అందించబడుతుంది.

## Community Site

మా [community site](https://community.ultimatemultisite.com/) Ultimate Multisite వినియోగదారులు ఒకరితో ఒకరు సంభాషించడానికి ఒక వేదిక. ఇది సభ్యులు పోస్ట్ చేయగల, చర్చలపై కామెంట్ చేయగల, సలహాలు ఇవ్వగల, సహకరించగల లేదా crowdsource చేయగల ఒక సమాచార వ్యవస్థగా పనిచేస్తుంది. ప్రైవేట్ spaces/pages ను అన్‌లాక్ చేయడానికి మీ Ultimate Multisite license key అందించాల్సి ఉంటుంది.
