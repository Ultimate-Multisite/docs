# అంతర్నిర్మిత ఏజెంట్లు (Built-in Agents)

Gratis AI Agent v1.9.0 ఐదు అంతర్నిర్మిత ఏజెంట్స్‌తో వస్తుంది. ప్రతి ఏజెంట్ ఒక నిర్దిష్టమైన టూల్స్ సెట్, ఒక ప్రత్యేకమైన సిస్టమ్ ప్రాంప్ట్ మరియు ఆ రంగంలో సాధారణ పనులకు సరిపోయే స్టార్టర్ సూచనలతో ముందుగా సెటప్ చేయబడి ఉంటుంది. ఏజెంట్ల మధ్య మారడం వలన అసిస్టెంట్ ఏమి చేయగలదో మరియు ఎలా స్పందిస్తుందో మారుతుంది — దీనికి మీ వైపు నుండి ఎలాంటి కాన్ఫిగరేషన్ అవసరం లేదు.

## ఏజెంట్ అంటే ఏమిటి? (What Is an Agent?)

ప్రతి ఏజెంట్ అనేది ఒక పేరు గల కాన్ఫిగరేషన్ ప్రొఫైల్. ఇది ఈ క్రింది వాటిని మిళితం చేస్తుంది:

- **టూల్స్ (Tools)** — ఏజెంట్ ఉపయోగించడానికి అనుమతించబడిన సామర్థ్యాలు (ఉదాహరణకు, ఒక కంటెంట్ రైటర్‌కు పోస్ట్ సృష్టించే సామర్థ్యాలు అందుబాటులో ఉంటాయి; ఒక డిజైన్ స్టూడియోకు CSS మరియు theme.json సామర్థ్యాలు అందుబాటులో ఉంటాయి).
- **సిస్టమ్ ప్రాంప్ట్ (System prompt)** — ఏజెంట్ యొక్క టోన్, ప్రాధాన్యతలు మరియు పరిమితులను సెట్ చేసే సూచనలు.
- **సూచనలు (Suggestions)** — త్వరగా ప్రారంభించడానికి సహాయపడే, చాట్ ఇంటర్‌ఫేస్‌లో ముందుగా రాసిపెట్టిన ప్రాంప్ట్‌లు.

## ఏజెంట్ పికర్‌ను యాక్సెస్ చేయడం (Accessing the Agent Picker)

1. వర్డ్‌ప్రెస్ అడ్మిన్ సైడ్‌బార్‌లో **Gratis AI Agent** ప్యానెల్‌ను తెరవండి.
2. చాట్ హెడర్‌లో ఎగువ-ఎడమ వైపున ఉన్న **ఏజెంట్ ఐకాన్‌పై** క్లిక్ చేయండి (యాక్టివ్ ఏజెంట్‌ను ప్రతిబింబించడానికి ఐకాన్ మారుతుంది).
3. **ఏజెంట్ పికర్ (Agent Picker)** అనేది ఒక ఫారం-టేబుల్ ఓవర్‌లేగా 열립니다. ప్రతి ఏజెంట్ దాని ఐకాన్, పేరు మరియు ఒక లైన్ వివరణతో జాబితా చేయబడుతుంది.
4. దాన్ని యాక్టివేట్ చేయడానికి ఏజెంట్ వరుసపై క్లిక్ చేయండి. చాట్ హెడర్ వెంటనే అప్‌డేట్ అవుతుంది.

మీరు సంభాషణ మధ్యలో కూడా ఏజెంట్లను మార్చవచ్చు — కొత్త ఏజెంట్ యొక్క సిస్టమ్ ప్రాంప్ట్ తదుపరి సందేశం నుండి అమలులోకి వస్తుంది.

## ఐదు అంతర్నిర్మిత ఏజెంట్లు (The Five Built-in Agents)

### కంటెంట్ రైటర్ (Content Writer)

**ఫోకస్:** పోస్ట్‌లు, పేజీలు మరియు కాంటాక్ట్ ఫారమ్‌లను సృష్టించడం మరియు ఎడిట్ చేయడం.

**అందుబాటులో ఉన్న టూల్స్:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**ఏం బాగా చేస్తుంది:**
- ఒక బ్రీఫ్ లేదా అవుట్‌లైన్ నుండి బ్లాగ్ పోస్ట్‌లను డ్రాఫ్ట్ చేయడం మరియు పబ్లిష్ చేయడం.
- కొత్త సైట్ కోసం ల్యాండింగ్ పేజీల బ్యాచ్‌లను సృష్టించడం.
- కాంటాక్ట్ మరియు ఎంక్వైరీ ఫారమ్‌లను నిర్మించడం.
- URL లేదా సెర్చ్ ద్వారా పోస్ట్‌లపై ఫీచర్డ్ ఇమేజ్‌లను సెట్ చేయడం.

**స్టార్టర్ సూచనలు:**
- *WordPress multisite ప్రయోజనాల గురించి 500 పదాల బ్లాగ్ పోస్ట్ రాయండి.*
- *ఒక About, Services, మరియు Contact పేజీని సృష్టించి పబ్లిష్ చేయండి.*
- *Contact పేజీకి బుకింగ్ ఎంక్వైరీ ఫారమ్‌ను జోడించండి.*

---

### సైట్ బిల్డర్ (Site Builder)

**ఫోకస్:** ఒకే ప్రాంప్ట్ నుండి ఎండ్-టు-ఎండ్ వెబ్‌సైట్ సృష్టి.

**అందుబాటులో ఉన్న టూల్స్:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**ఏం బాగా చేస్తుంది:**
- వివరించబడిన వ్యాపార రకానికి బహుళ-దశల సైట్ బిల్డ్ ప్లాన్‌ను రూపొందించడం.
- ప్రతి దశను స్వయంచాలకంగా అమలు చేయడం — నిర్మాణం, కంటెంట్, నావిగేషన్, డిజైన్.
- మాన్యువల్ ఇంటర్‌వెన్షన్ అవసరం లేకుండా ప్లాన్ మధ్యలో ఎర్రర్ల నుండి కోలుకోవడం.
- బిల్డ్ వ భాగంగా సిఫార్సు చేయబడిన ప్లగిన్‌లను ఇన్‌స్టాల్ చేయడం.
- చాట్ ఇంటర్‌ఫేస్ నుండి నేరుగా కాంటాక్ట్ ఫారమ్‌లను సృష్టించడం (Superdav AI Agent v1.10.0+).

**స్టార్టర్ సూచనలు:**
- *గ్యాలరీ పోస్ట్ టైప్‌తో, బుకింగ్ పేజీతో మరియు కాంటాక్ట్ ఫారమ్‌తో ఫోటోగ్రఫీ పోర్ట్‌ఫోలియో సైట్‌ను నిర్మించండి.*
- *ఆన్‌లైన్ మెనూ, ఆపరేటింగ్ అవర్స్ మరియు టేబుల్-బుకింగ్ ఎంక్వైరీ ఫారమ్‌తో రెస్టారెంట్ వెబ్‌సైట్‌ను సృష్టించండి.*
- *సర్వీస్ పేజీలు, పోర్ట్‌ఫోలియో సెక్షన్ మరియు బ్లాగ్‌తో ఫ్రీలాన్స్ కన్సల్టింగ్ సైట్‌ను సెటప్ చేయండి.*
- *సైట్ బిల్డర్‌ను ఉపయోగించి Contact పేజీకి కాంటాక్ట్ ఫారమ్‌ను జోడించండి.*

### డిజైన్ స్టూడియో (Design Studio)

**ఫోకస్:** విజువల్ కస్టమైజేషన్ — రంగులు, టైపోగ్రఫీ, CSS మరియు బ్లాక్ ప్యాటర్న్‌లు.

**అందుబాటులో ఉన్న టూల్స్:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**ఏం బాగా చేస్తుంది:**
- పేరున్న థీమ్ ప్రీసెట్‌లను వర్తింపజేయడం (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog).
- theme.json ద్వారా గ్లోబల్ టైపోగ్రఫీ మరియు కలర్ పాలెట్‌లను ఫైన్-ట్యూన్ చేయడం.
- బ్రాండ్-నిర్దిష్ట ఓవర్‌రైడ్‌ల కోసం కస్టమ్ CSSను ఇంజెక్ట్ చేయడం.
- ఒక పేజీ యొక్క స్క్రీన్‌షాట్‌ను తీసి, డిజైన్ సమస్యల కోసం సమీక్షించడం.

**స్టార్టర్ సూచనలు:**
- *warm-editorial ప్రీసెట్‌ను వర్తింపజేసి, ఆపై ప్రాథమిక రంగును #2d6a4f గా సెట్ చేయండి.*
- *హోమ్‌పేజీ స్క్రీన్‌షాట్‌ను తీసి, మీరు ఏమి మెరుగుపరుస్తారో నాకు చెప్పండి.*
- *పూర్తి-వెడల్పు నేపథ్య చిత్రం మరియు సెంటర్‌డ్ హెడ్డింగ్‌తో పునర్వినియోగపరచదగిన హీరో బ్లాక్ ప్యాటర్న్‌ను సృష్టించండి.*

### ప్లగిన్ మేనేజర్ (Plugin Manager)

**ఫోకస్:** వర్డ్‌ప్రెస్ ప్లగిన్‌లను కనుగొనడం, ఇన్‌స్టాల్ చేయడం మరియు నిర్వహించడం.

**అందుబాటులో ఉన్న టూల్స్:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**ఏం బాగా చేస్తుంది:**
- వివరించబడిన ఉపయోగ సందర్భానికి ఉత్తమమైన ప్లగిన్‌ను సిఫార్సు చేయడం.
- రిజిస్ట్రీ నుండి సామర్థ్య ప్యాక్‌లను ఇన్‌స్టాల్ చేయడం.
- కేటగిరీ ద్వారా అందుబాటులో ఉన్న సామర్థ్య కేటలాగ్‌ను బ్రౌజ్ చేయడం.

**స్టార్టర్ సూచనలు:**
- *Membership directory కోసం ఉత్తమమైన ప్లగిన్ ఏది?*
- *WooCommerce abilities pack ను ఇన్‌స్టాల్ చేయండి.*
- *అందుబాటులో ఉన్న అన్ని ఈ-కామర్స్ ability packs నాకు చూపించండి.*

### సపోర్ట్ అసిస్టెంట్ (Support Assistant)

**ఫోకస్:** సైట్ కంటెంట్, సెట్టింగ్‌లు మరియు వర్డ్‌ప్రెస్ కాన్ఫిగరేషన్ గురించి ప్రశ్నలకు సమాధానం ఇవ్వడం.

**అందుబాటులో ఉన్న టూల్స్:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**ఏం బాగా చేస్తుంది:**
- ప్రస్తుత సైట్ సెట్టింగ్‌లు మరియు ఆప్షన్‌లను చూడటం.
- సైట్‌లో ఏ పోస్ట్ టైప్‌లు, టాక్సోనమీలు మరియు మెనూలు కాన్ఫిగర్ చేయబడ్డాయో వివరించడం.
- లైవ్ విలువలను చదవడం ద్వారా "ఈ సెట్టింగ్ ఏమి చేస్తుంది?" అనే ప్రశ్నలకు సమాధానం ఇవ్వడం.
- మార్పులు చేసే ముందు చదవడానికి మాత్రమే పరిమితమైన డయాగ్నస్టిక్ లేయర్‌గా పనిచేయడం.

**స్టార్టర్ సూచనలు:**
- *ఈ సైట్‌లో ప్రస్తుతం ఏ ప్లగిన్‌లు మరియు సెట్టింగ్‌లు యాక్టివ్‌గా ఉన్నాయి?*
- *ఈ సైట్‌లో రిజిస్టర్ చేయబడిన అన్ని కస్టమ్ పోస్ట్ టైప్‌లను జాబితా చేయండి.*
- *ఏ నావిగేషన్ మెనూలు ఉన్నాయి మరియు అవి ఎక్కడ కేటాయించబడ్డాయి?*

## ఏజెంట్లను కస్టమైజ్ చేయడం (Customising Agents)

ప్రతి అంతర్నిర్మిత ఏజెంట్‌ను `gratis_ai_agent_agents` ఫిల్టర్ ద్వారా విస్తరించవచ్చు లేదా భర్తీ చేయవచ్చు.

### ఇప్పటికే ఉన్న ఏజెంట్‌కు కస్టమ్ సిస్టమ్ ప్రాంప్ట్‌ను జోడించడం

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### కొత్త ఏజెంట్‌ను రిజిస్టర్ చేయడం

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
```
