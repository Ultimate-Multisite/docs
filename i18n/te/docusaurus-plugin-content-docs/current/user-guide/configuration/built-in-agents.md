# అంతర్నిర్మిత ఏజెంట్లు

Gratis AI Agent v1.9.0 ఐదు అంతర్నిర్మిత ఏజెంట్లతో వస్తుంది. ప్రతి ఏజెంట్‌లో నిర్దిష్ట పనుల కోసం ముందుగానే అమర్చిన సాధనాల సమితి, అనుకూలమైన సిస్టమ్ ప్రాంప్ట్, ఆ ప్రాంతంలోని సాధారణ పనులకు సరిపోయే ప్రారంభ సూచనలు ఉంటాయి. ఏజెంట్ల మధ్య మారితే సహాయకుడు చేయగలిగే పని, స్పందించే విధానం మారుతుంది — మీవైపు నుండి ఎలాంటి కాన్ఫిగరేషన్ అవసరం లేదు. సంబంధిత ఇంటిగ్రేషన్లు కాన్ఫిగర్ చేసినప్పుడు Superdav AI Agent v1.18.0 ఈ పనిప్రవాహాలకు షెడ్యూల్-అవగాహన ఉన్న సాధనాలు, రిమైండర్ రికార్డులు, ఆమోద గేట్లు, SMS నోటిఫికేషన్లు జోడించగలదు.

## ఏజెంట్ అంటే ఏమిటి?

ప్రతి ఏజెంట్ ఒక పేరుగల కాన్ఫిగరేషన్ ప్రొఫైల్. ఇది వీటిని కలిపి ఉంచుతుంది:

- **సాధనాలు** — ఏజెంట్ ఉపయోగించడానికి అనుమతించబడిన సామర్థ్యాలు (ఉదా. Content Writerకి పోస్ట్-సృష్టి సామర్థ్యాలకు ప్రాప్యత ఉంటుంది; Design Studioకి CSS మరియు theme.json సామర్థ్యాలకు ప్రాప్యత ఉంటుంది)
- **సిస్టమ్ ప్రాంప్ట్** — ఏజెంట్ స్వరం, ప్రాధాన్యతలు, పరిమితులను నిర్ణయించే సూచనలు
- **సూచనలు** — మీరు త్వరగా ప్రారంభించేందుకు చాట్ ఇంటర్‌ఫేస్‌లో చూపించే ముందే రాసిన ప్రాంప్ట్‌లు

## ఏజెంట్ ఎంపికకర్తను తెరవడం

1. WordPress అడ్మిన్ సైడ్‌బార్‌లో **Gratis AI Agent** ప్యానెల్‌ను తెరవండి.
2. చాట్ హెడర్ ఎగువ-ఎడమలోని **ఏజెంట్ ఐకాన్**‌ను క్లిక్ చేయండి (సక్రియ ఏజెంట్‌ను ప్రతిబింబించేలా ఐకాన్ మారుతుంది).
3. **ఏజెంట్ ఎంపికకర్త** ఫారమ్-టేబుల్ ఓవర్‌లేగా తెరుచుకుంటుంది. ప్రతి ఏజెంట్ దాని ఐకాన్, పేరు, ఒక-లైన్ వివరణతో జాబితాలో కనిపిస్తుంది.
4. దాన్ని సక్రియం చేయడానికి ఒక ఏజెంట్ వరుసను క్లిక్ చేయండి. చాట్ హెడర్ వెంటనే నవీకరించబడుతుంది.

మీరు సంభాషణ మధ్యలో కూడా ఏజెంట్లను మార్చవచ్చు — కొత్త ఏజెంట్ యొక్క సిస్టమ్ ప్రాంప్ట్ తదుపరి సందేశం నుండి అమలులోకి వస్తుంది.

## ఐదు అంతర్నిర్మిత ఏజెంట్లు

### కంటెంట్ రచయిత

**దృష్టి:** పోస్ట్‌లు, పేజీలు, సంప్రదింపు ఫారమ్‌లను సృష్టించడం మరియు సవరించడం.

**అందుబాటులో ఉన్న సాధనాలు:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Superdav AI Agent v1.18.0 ఇంటిగ్రేషన్లు ప్రారంభించి ఉంటే, కాన్ఫిగర్ చేసిన క్యాలెండర్ సందర్భం, ఆమోద గేట్లు, రిమైండర్లు, SMS నోటిఫికేషన్ సాధనాలు కూడా ఆమోదించబడిన పనిప్రవాహాలకు అందుబాటులో ఉండవచ్చు.

**ఇది బాగా చేసే పనులు:**
- సంక్షిప్త వివరణ లేదా అవుట్‌లైన్ నుండి బ్లాగ్ పోస్ట్‌లను ముసాయిదా చేయడం మరియు ప్రచురించడం
- కొత్త సైట్ కోసం ల్యాండింగ్ పేజీల సమూహాలను సృష్టించడం
- సంప్రదింపు మరియు విచారణ ఫారమ్‌లను నిర్మించడం
- URL లేదా శోధన నుండి పోస్ట్‌లపై ఫీచర్డ్ చిత్రాలను సెట్ చేయడం
- కాన్ఫిగర్ చేసిన Google Calendar సందర్భం నుండి ఈవెంట్ ఫాలో-అప్ సందేశాలను ముసాయిదా చేసి, నోటిఫికేషన్లు పంపే ముందు ఆమోదం కోసం ఆగడం

**ప్రారంభ సూచనలు:**
- *WordPress మల్టీసైట్ ప్రయోజనాల గురించి 500 పదాల బ్లాగ్ పోస్ట్ రాయండి.*
- *About, Services, Contact పేజీలను సృష్టించి ప్రచురించండి.*
- *Contact పేజీకి బుకింగ్ విచారణ ఫారమ్‌ను జోడించండి.*
- *రేపటి కాన్ఫిగర్ చేసిన క్యాలెండర్ ఈవెంట్‌కు హాజరయ్యే వారి కోసం రిమైండర్ ముసాయిదా చేసి, పంపే ముందు ఆమోదం కోసం వేచి ఉండండి.*

### సైట్ నిర్మాణకర్త

**దృష్టి:** ఒకే ప్రాంప్ట్ నుండి పూర్తి వెబ్‌సైట్ సృష్టి.

**అందుబాటులో ఉన్న సాధనాలు:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Superdav AI Agent v1.18.0తో, అడ్మినిస్ట్రేటర్లు ప్రారంభించిన చోట కాన్ఫిగర్ చేసిన మేనేజ్డ్-సర్వీస్, ఆమోదం, రిమైండర్, క్యాలెండర్, SMS సాధనాలు అందుబాటులో ఉండవచ్చు.

**ఇది బాగా చేసే పనులు:**
- వివరించిన వ్యాపార రకానికి బహుళ-దశల సైట్ నిర్మాణ ప్రణాళికను రూపొందించడం
- ప్రతి దశను స్వయంచాలకంగా అమలు చేయడం — నిర్మాణం, కంటెంట్, నావిగేషన్, డిజైన్
- చేతితో జోక్యం అవసరం లేకుండా ప్రణాళిక మధ్యలో వచ్చిన లోపాల నుండి కోలుకోవడం
- నిర్మాణంలో భాగంగా సిఫార్సు చేసిన pluginలను ఇన్‌స్టాల్ చేయడం
- చాట్ ఇంటర్‌ఫేస్ నుంచే నేరుగా సంప్రదింపు ఫారమ్‌లను సృష్టించడం (Superdav AI Agent v1.10.0+)
- ఆమోద గేట్లు మరియు రిమైండర్ రికార్డులు ప్రారంభించబడినప్పుడు డూప్లికేట్ నోటిఫికేషన్లు లేకుండా ప్రారంభ రిమైండర్లు లేదా హాజరైన వారి ఫాలో-అప్‌ను సమన్వయం చేయడం

**ప్రారంభ సూచనలు:**
- *గ్యాలరీ పోస్ట్ రకం, బుకింగ్ పేజీ, సంప్రదింపు ఫారమ్‌తో ఫోటోగ్రఫీ పోర్ట్‌ఫోలియో సైట్‌ను నిర్మించండి.*
- *ఆన్‌లైన్ మెనూ, తెరవబడే సమయాలు, టేబుల్-బుకింగ్ విచారణ ఫారమ్‌తో రెస్టారెంట్ వెబ్‌సైట్‌ను సృష్టించండి.*
- *సేవా పేజీలు, పోర్ట్‌ఫోలియో విభాగం, బ్లాగ్‌తో ఫ్రీలాన్స్ కన్సల్టింగ్ సైట్‌ను సెటప్ చేయండి.*
- *సైట్ నిర్మాణకర్తను ఉపయోగించి Contact పేజీకి సంప్రదింపు ఫారమ్‌ను జోడించండి.*
- *సైట్ ప్రారంభ చెక్‌లిస్ట్ ఆమోదించబడిన తర్వాత, కాన్ఫిగర్ చేసిన స్టేక్‌హోల్డర్ సంప్రదింపుకు SMS రిమైండర్ పంపండి.*

### డిజైన్ స్టూడియో

**దృష్టి:** దృశ్య అనుకూలీకరణ — రంగులు, టైపోగ్రఫీ, CSS, బ్లాక్ నమూనాలు.

**అందుబాటులో ఉన్న సాధనాలు:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**ఇది బాగా చేసే పనులు:**
- పేరుగల theme ప్రీసెట్‌లను వర్తింపజేయడం (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- theme.json ద్వారా గ్లోబల్ టైపోగ్రఫీ మరియు రంగు ప్యాలెట్‌లను మెరుగుపరచడం
- బ్రాండ్-నిర్దిష్ట ఓవర్‌రైడ్‌ల కోసం కస్టమ్ CSSను ఇంజెక్ట్ చేయడం
- పేజీ స్క్రీన్‌షాట్ తీసుకుని డిజైన్ సమస్యల కోసం సమీక్షించడం

**ప్రారంభ సూచనలు:**
- *warm-editorial ప్రీసెట్‌ను వర్తింపజేసి, ఆపై ప్రాథమిక రంగును #2d6a4fగా సెట్ చేయండి.*
- *హోమ్‌పేజీ స్క్రీన్‌షాట్ తీసుకుని, మీరు ఏమి మెరుగుపరుస్తారో చెప్పండి.*
- *పూర్తి-వెడల్పు నేపథ్య చిత్రం మరియు మధ్యలో ఉన్న శీర్షికతో మళ్లీ ఉపయోగించగల హీరో బ్లాక్ నమూనాను సృష్టించండి.*

### Plugin మేనేజర్

**దృష్టి:** WordPress pluginలను కనుగొనడం, ఇన్‌స్టాల్ చేయడం, నిర్వహించడం.

**అందుబాటులో ఉన్న సాధనాలు:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**ఇది బాగా చేసే పనులు:**
- వివరించిన వినియోగ సందర్భానికి ఉత్తమ pluginను సిఫార్సు చేయడం
- రిజిస్ట్రీ నుండి సామర్థ్య ప్యాక్‌లను ఇన్‌స్టాల్ చేయడం
- వర్గం ప్రకారం అందుబాటులో ఉన్న సామర్థ్య క్యాటలాగ్‌ను బ్రౌజ్ చేయడం

**ప్రారంభ సూచనలు:**
- *సభ్యత్వ డైరెక్టరీకి ఉత్తమ plugin ఏది?*
- *WooCommerce సామర్థ్యాల ప్యాక్‌ను ఇన్‌స్టాల్ చేయండి.*
- *అందుబాటులో ఉన్న అన్ని ఇకామర్స్ సామర్థ్య ప్యాక్‌లను చూపించండి.*

### సహాయ సహాయకుడు

**దృష్టి:** సైట్ కంటెంట్, సెట్టింగ్‌లు, WordPress కాన్ఫిగరేషన్ గురించి ప్రశ్నలకు సమాధానం ఇవ్వడం.

**అందుబాటులో ఉన్న సాధనాలు:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**ఇది బాగా చేసే పనులు:**

## ఏజెంట్లను అనుకూలీకరించడం

ప్రతి అంతర్నిర్మిత ఏజెంట్‌ను `gratis_ai_agent_agents` ఫిల్టర్ ద్వారా విస్తరించవచ్చు లేదా భర్తీ చేయవచ్చు.

### ఉన్న ఏజెంట్‌కు అనుకూల సిస్టమ్ ప్రాంప్ట్‌ను జోడించడం

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### కొత్త ఏజెంట్‌ను నమోదు చేయడం

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

ఫిల్టర్ నడిచిన వెంటనే కొత్త ఏజెంట్ ఏజెంట్ ఎంపికలో కనిపిస్తుంది.

### అంతర్నిర్మిత ఏజెంట్‌ను తొలగించడం

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
