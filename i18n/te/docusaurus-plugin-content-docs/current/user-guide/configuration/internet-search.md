# ఇంటర్నెట్ సెర్చ్ {#internet-search}

Gratis AI Agent v1.5.0 లో **ఇంటర్నెట్ సెర్చ్** సామర్థ్యం జోడించబడింది. ఇది AI అసిస్టెంట్‌కు సంభాషణ సమయంలో వెబ్ నుండి తాజా సమాచారాన్ని తిరిగి పొందే (retrieve) సామర్థ్యాన్ని ఇస్తుంది. ఇది [Brave Search API](https://brave.com/search/api/) లేదా [Tavily API](https://tavily.com/) ద్వారా నడుస్తుంది.

## ఇది ఎలా పనిచేస్తుంది {#how-it-works}

ఇంటర్నెట్ సెర్చ్ ఎనేబుల్ చేయబడినప్పుడు, అసిస్టెంట్ ఒక ప్రశ్న ప్రస్తుత లేదా బాహ్య సమాచారం అవసరమని నిర్ణయించుకున్నప్పుడు, అది ఆటోమేటిక్‌గా మీరు కాన్ఫిగర్ చేసిన సెర్చ్ ప్రొవైడర్‌ను ప్రశ్నిస్తుంది (query). ఉదాహరణకు, ఇటీవలి వార్తలు, లైవ్ ధరలు, లేదా మోడల్ శిక్షణ కటాఫ్ తేదీ తర్వాత మారిన డాక్యుమెంటేషన్ వంటివి.

ఫలితాలు రియల్ టైమ్‌లో తిరిగి పొంది, ప్రతిస్పందనను రూపొందించడానికి ముందు అసిస్టెంట్ సందర్భంలో (context) చేర్చబడతాయి. ప్రశ్నలకు సమాధానం ఇవ్వడానికి సెర్చ్ ఫలితాలను ఉపయోగించినప్పుడు అసిస్టెంట్ దాన్ని సూచిస్తుంది.

## ఇంటర్నెట్ సెర్చ్ ఎనేబుల్ చేయడం {#enabling-internet-search}

ఇంటర్నెట్ సెర్చ్ కోసం మీ ఎంచుకున్న సెర్చ్ ప్రొవైడర్ నుండి API కీ అవసరం. దీనిని కాన్ఫిగర్ చేయడానికి:

1. **Gratis AI Agent → Settings → Advanced** కు వెళ్లండి.
2. **Internet Search Provider** డ్రాప్‌డౌన్ ను గుర్తించి, **Brave Search** లేదా **Tavily** లో ఏదో ఒకటి ఎంచుకోండి.
3. సంబంధిత ఫీల్డ్‌లో మీ API కీని నమోదు చేయండి. ప్రతి ఫీల్డ్ పక్కన సైన్-అప్ లింక్‌లు కనిపిస్తాయి.
4. **Save Settings** పై క్లిక్ చేయండి.

కీ సేవ్ చేయబడిన తర్వాత, ఇంటర్నెట్ సెర్చ్ సామర్థ్యం అసిస్టెంట్‌కు ఆటోమేటిక్‌గా అందుబాటులో ఉంటుంది.

## Brave Search {#brave-search}

### Brave Search API కీని పొందడం {#obtaining-a-brave-search-api-key}

1. [Brave Search API పేజీ](https://brave.com/search/api/) ని సందర్శించండి.
2. ఒక ప్లాన్‌కు సైన్ అప్ చేయండి. నెలవారీ రిక్వెస్ట్ పరిమితితో ఉచిత టయర్ అందుబాటులో ఉంది.
3. Brave Search డెవలపర్ డాష్‌బోర్డ్ నుండి మీ API కీని కాపీ చేయండి.
4. దానిని Gratis AI Agent సెట్టింగ్‌లలోని **Brave Search API Key** ఫీల్డ్‌లో పేస్ట్ చేయండి.

### వినియోగ పరిమితులు (Usage Limits) {#usage-limits}

వినియోగం అనేది చేసిన ప్రశ్నల సంఖ్య ఆధారంగా Brave Search ద్వారా బిల్ చేయబడుతుంది. సెర్చ్‌ను ట్రిగ్గర్ చేసే ప్రతి AI ప్రతిస్పందన ఒక ప్రశ్నగా లెక్కించబడుతుంది. ఊహించని ఛార్జీలను నివారించడానికి [Brave Search developer dashboard](https://brave.com/search/api/) లో మీ వినియోగాన్ని పర్యవేక్షించండి.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 లో **Tavily** ను ప్రత్యామ్నాయ ఇంటర్నెట్ సెర్చ్ ప్రొవైడర్‌గా జోడించింది. ఇది మరింత సమృద్ధిగా సెర్చ్ ఫలితాలు మరియు అధునాతన పరిశోధన సామర్థ్యాలను అందిస్తుంది.

### Tavily API కీని పొందడం {#obtaining-a-tavily-api-key}

1. [Tavily API పేజీ](https://tavily.com/) ని సందర్శించండి.
2. ఒక ఖాతా కోసం సైన్ అప్ చేయండి. నెలవారీ రిక్వెస్ట్ పరిమితితో ఉచిత టయర్ అందుబాటులో ఉంది.
3. Tavily డాష్‌బోర్డ్ నుండి మీ API కీని కాపీ చేయండి.
4. దానిని Gratis AI Agent సెట్టింగ్‌లలోని **Tavily API Key** ఫీల్డ్‌లో పేస్ట్ చేయండి.

### వినియోగ పరిమితులు (Usage Limits) {#usage-limits-1}

వినియోగం అనేది చేసిన API కాల్‌ల సంఖ్య ఆధారంగా Tavily ద్వారా బిల్ చేయబడుతుంది. సెర్చ్‌ను ట్రిగ్గర్ చేసే ప్రతి AI ప్రతిస్పందన ఒక కాల్‌గా లెక్కించబడుతుంది. ఊహించని ఛార్జీలను నివారించడానికి [Tavily dashboard](https://tavily.com/) లో మీ వినియోగాన్ని పర్యవేక్షించండి.

## ఇంటర్నెట్ సెర్చ్ డిసేబుల్ చేయడం {#disabling-internet-search}

యాక్టివ్ సెర్చ్ ప్రొవైడర్ ఫీల్డ్ నుండి API కీని తొలగించి, సేవ్ చేయండి. ఇంటర్నెట్ సెర్చ్ సామర్థ్యం అసిస్టెంట్‌కు ఇకపై అందించబడదు.

:::note
అసిస్టెంట్ సమాధానం ఇవ్వడానికి సెర్చ్ ఫలితాల కోసం వేచి ఉండాల్సి రావడం వలన ఇంటర్నెట్ సెర్చ్ ప్రతిస్పందనలకు ఆలస్యాన్ని (latency) పెంచుతుంది. సమయానికి సంబంధించిన ఉపయోగ సందర్భాల కోసం, రియల్-టైమ్ సెర్చ్ అవసరమా లేదా అసిస్టెంట్ యొక్క అంతర్నిర్మిత జ్ఞానం సరిపోతుందా అని పరిశీలించండి.
:::
