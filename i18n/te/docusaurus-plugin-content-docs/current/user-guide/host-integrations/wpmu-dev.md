# WPMU DEV ఇంటిగ్రేషన్

## అవలోకనం
WPMU DEV అనేది WordPress సైట్‌ల కోసం హోస్టింగ్, ప్లగిన్‌లు మరియు సేవలను అందించే సమగ్ర WordPress ప్లాట్‌ఫారమ్. ఈ ఇంటిగ్రేషన్ Ultimate Multisite మరియు WPMU DEV హోస్టింగ్ మధ్య ఆటోమేటిక్ డొమైన్ సింకింగ్ మరియు SSL సర్టిఫికేట్ నిర్వహణను అనుమతిస్తుంది.

## ఫీచర్లు
- ఆటోమేటిక్ డొమైన్ సింకింగ్
- SSL సర్టిఫికేట్ నిర్వహణ
- విస్తరించిన SSL సర్టిఫికేట్ వెరిఫికేషన్ ప్రయత్నాలు

## అవసరాలు
మీరు WPMU DEV లో హోస్ట్ చేస్తున్నారా అనేది ఇంటిగ్రేషన్ ఆటోమేటిక్‌గా గుర్తించి, అంతర్నిర్మిత API ని ఉపయోగిస్తుంది. మీరు WPMU DEV లో హోస్ట్ చేస్తుంటే అదనపు కాన్ఫిగరేషన్ అవసరం లేదు.

WPMU DEV లో హోస్ట్ చేసేటప్పుడు ఆటోమేటిక్‌గా నిర్వచించబడే `WPMUDEV_HOSTING_SITE_ID` constant ఉనికిని ఇంటిగ్రేషన్ తనిఖీ చేస్తుంది.

## సెటప్ సూచనలు

### 1. WPMU DEV హోస్టింగ్‌ని వెరిఫై చేయండి

మీరు WPMU DEV లో హోస్ట్ చేస్తుంటే, అవసరమైన constants ఇప్పటికే నిర్వచించబడి ఉండాలి. కింది వాటిని వెరిఫై చేయండి:

1. మీ ఎన్విరాన్‌మెంట్‌లో `WPMUDEV_HOSTING_SITE_ID` constant నిర్వచించబడి ఉంది
2. మీకు API యాక్సెస్‌తో యాక్టివ్ WPMU DEV మెంబర్‌షిప్ ఉంది

### 2. ఇంటిగ్రేషన్‌ని ఎనేబుల్ చేయండి

1. మీ WordPress అడ్మిన్‌లో, Ultimate Multisite > Settings కి వెళ్ళండి
2. "Domain Mapping" ట్యాబ్‌కి నావిగేట్ చేయండి
3. "Host Integrations" కి స్క్రోల్ చేయండి
4. WPMU DEV ఇంటిగ్రేషన్‌ని ఎనేబుల్ చేయండి
5. "Save Changes" క్లిక్ చేయండి

## ఇది ఎలా పని చేస్తుంది

### డొమైన్ సింకింగ్

Ultimate Multisite లో డొమైన్ మ్యాప్ చేసినప్పుడు:

1. మీ హోస్టింగ్ అకౌంట్‌కి డొమైన్‌ని జోడించడానికి ఇంటిగ్రేషన్ WPMU DEV API ని ఉపయోగిస్తుంది
2. ఇది డొమైన్ యొక్క www వెర్షన్‌ను కూడా ఆటోమేటిక్‌గా జోడిస్తుంది
3. WPMU DEV డొమైన్ కాన్ఫిగరేషన్ మరియు SSL సర్టిఫికేట్ జారీని నిర్వహిస్తుంది

### SSL సర్టిఫికేట్ నిర్వహణ

SSL సర్టిఫికేట్‌లు జారీ చేయడానికి మరియు ఇన్‌స్టాల్ చేయడానికి కొంత సమయం పట్టవచ్చు కాబట్టి, WPMU DEV హోస్టింగ్ కోసం SSL సర్టిఫికేట్ వెరిఫికేషన్ ప్రయత్నాల సంఖ్యను పెంచేలా ఇంటిగ్రేషన్ కాన్ఫిగర్ చేయబడింది. డిఫాల్ట్‌గా, ఇది SSL సర్టిఫికేట్ వెరిఫికేషన్ కోసం 10 సార్లు వరకు ప్రయత్నిస్తుంది, సాధారణ 5 ప్రయత్నాలతో పోలిస్తే.

## ముఖ్యమైన గమనికలు

### డొమైన్ తొలగింపు

ప్రస్తుతం, WPMU DEV API డొమైన్‌లను తొలగించే మార్గాన్ని అందించదు. Ultimate Multisite లో డొమైన్ మ్యాపింగ్ తొలగించబడినప్పుడు, డొమైన్ మీ WPMU DEV హోస్టింగ్ అకౌంట్‌లో ఉంటుంది. అవసరమైతే, మీరు దీన్ని WPMU DEV హోస్టింగ్ డ్యాష్‌బోర్డ్ నుండి మాన్యువల్‌గా తొలగించాలి.

### API ఆథెంటికేషన్

మీ WordPress డేటాబేస్‌లో `wpmudev_apikey` ఆప్షన్‌గా నిల్వ చేయబడిన WPMU DEV API కీని ఇంటిగ్రేషన్ ఉపయోగిస్తుంది. మీ సైట్‌ను WPMU DEV కి కనెక్ట్ చేసినప్పుడు ఇది ఆటోమేటిక్‌గా సెటప్ అవుతుంది.

## ట్రబుల్‌షూటింగ్

### API కనెక్షన్ సమస్యలు
- మీ సైట్ WPMU DEV కి సరిగ్గా కనెక్ట్ అయిందో వెరిఫై చేయండి
- మీ WordPress డేటాబేస్‌లో `wpmudev_apikey` ఆప్షన్ సెట్ చేయబడిందో తనిఖీ చేయండి
- మీ WPMU DEV మెంబర్‌షిప్ యాక్టివ్‌గా ఉందో నిర్ధారించుకోండి

### SSL సర్టిఫికేట్ సమస్యలు
- WPMU DEV SSL సర్టిఫికేట్‌లను జారీ చేయడానికి కొంత సమయం పట్టవచ్చు (సాధారణంగా 5-15 నిమిషాలు)
- SSL సర్టిఫికేట్‌ల కోసం 10 సార్లు వరకు తనిఖీ చేసేలా ఇంటిగ్రేషన్ కాన్ఫిగర్ చేయబడింది
- అనేక ప్రయత్నాల తర్వాత కూడా SSL సర్టిఫికేట్‌లు జారీ కావడం లేదంటే, WPMU DEV సపోర్ట్‌ని సంప్రదించండి

### డొమైన్ జోడించబడలేదు
- ఏవైనా ఎర్రర్ మెసేజ్‌ల కోసం Ultimate Multisite లాగ్‌లను తనిఖీ చేయండి
- డొమైన్ ఇప్పటికే WPMU DEV కి జోడించబడలేదని వెరిఫై చేయండి
- మీరు జోడిస్తున్న డొమైన్‌ల సంఖ్యకు మీ WPMU DEV హోస్టింగ్ ప్లాన్ మద్దతు ఇస్తుందని నిర్ధారించుకోండి
