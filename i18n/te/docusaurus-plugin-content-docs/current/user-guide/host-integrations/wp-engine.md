# WP Engine ఇంటిగ్రేషన్

## అవలోకనం
WP Engine అనేది WordPress సైట్‌లకు ఆప్టిమైజ్డ్ పనితీరు, భద్రత మరియు స్కేలబిలిటీని అందించే ప్రీమియం మేనేజ్డ్ WordPress హోస్టింగ్ ప్లాట్‌ఫారమ్. ఈ ఇంటిగ్రేషన్ Ultimate Multisite మరియు WP Engine మధ్య ఆటోమేటిక్ డొమైన్ సింకింగ్‌ను ప్రారంభిస్తుంది.

## ఫీచర్లు
- ఆటోమేటిక్ డొమైన్ సింకింగ్
- మల్టీసైట్ ఇన్‌స్టాలేషన్‌లకు సబ్‌డొమైన్ సపోర్ట్
- WP Engine యొక్క ఇప్పటికే ఉన్న సిస్టమ్‌లతో అతుకులేని ఇంటిగ్రేషన్

## అవసరాలు
మీరు WP Engine లో హోస్ట్ చేస్తున్నట్లయితే ఈ ఇంటిగ్రేషన్ స్వయంచాలకంగా గుర్తించి, బిల్ట్-ఇన్ WP Engine API ని ఉపయోగిస్తుంది. WP Engine plugin యాక్టివ్‌గా ఉండి సరిగ్గా కాన్ఫిగర్ చేయబడి ఉంటే అదనపు కాన్ఫిగరేషన్ అవసరం లేదు.

అయితే, మీరు ఇంటిగ్రేషన్‌ను మాన్యువల్‌గా కాన్ఫిగర్ చేయాలనుకుంటే, మీ `wp-config.php` ఫైల్‌లో ఈ constants లో ఒకదానిని డిఫైన్ చేయవచ్చు:

```php
define('WPE_APIKEY', 'your_api_key'); // ప్రాధాన్య పద్ధతి
// OR
define('WPE_API', 'your_api_key'); // ప్రత్యామ్నాయ పద్ధతి
```

## సెటప్ సూచనలు

### 1. WP Engine Plugin ని వెరిఫై చేయండి

మీరు WP Engine లో హోస్ట్ చేస్తుంటే, WP Engine plugin ఇప్పటికే ఇన్‌స్టాల్ చేయబడి యాక్టివేట్ చేయబడి ఉండాలి. వీటిని వెరిఫై చేయండి:

1. WP Engine plugin యాక్టివ్‌గా ఉంది
2. `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` ఫైల్ ఉనికిలో ఉంది

### 2. ఇంటిగ్రేషన్‌ను ఎనేబుల్ చేయండి

1. మీ WordPress admin లో, Ultimate Multisite > Settings కు వెళ్ళండి
2. "Domain Mapping" ట్యాబ్‌కు నావిగేట్ చేయండి
3. "Host Integrations" కు స్క్రోల్ చేయండి
4. WP Engine ఇంటిగ్రేషన్‌ను ఎనేబుల్ చేయండి
5. "Save Changes" క్లిక్ చేయండి

## ఇది ఎలా పని చేస్తుంది

### డొమైన్ సింకింగ్

Ultimate Multisite లో ఒక డొమైన్ మ్యాప్ చేయబడినప్పుడు:

1. ఇంటిగ్రేషన్ మీ WP Engine ఇన్‌స్టాలేషన్‌కు డొమైన్‌ను జోడించడానికి WP Engine API ని ఉపయోగిస్తుంది
2. WP Engine డొమైన్ కాన్ఫిగరేషన్ మరియు SSL సర్టిఫికేట్ జారీని నిర్వహిస్తుంది
3. డొమైన్ మ్యాపింగ్ తీసివేయబడినప్పుడు, ఇంటిగ్రేషన్ WP Engine నుండి డొమైన్‌ను తీసివేస్తుంది

### సబ్‌డొమైన్ సపోర్ట్

సబ్‌డొమైన్ మల్టీసైట్ ఇన్‌స్టాలేషన్‌ల కోసం:

1. కొత్త సైట్ సృష్టించబడినప్పుడు ఇంటిగ్రేషన్ ప్రతి సబ్‌డొమైన్‌ను WP Engine కు జోడిస్తుంది
2. WP Engine సబ్‌డొమైన్ కాన్ఫిగరేషన్‌ను నిర్వహిస్తుంది
3. సైట్ డిలీట్ చేయబడినప్పుడు, ఇంటిగ్రేషన్ WP Engine నుండి సబ్‌డొమైన్‌ను తీసివేస్తుంది

## ముఖ్యమైన గమనికలు

### Wildcard డొమైన్‌లు

సబ్‌డొమైన్ మల్టీసైట్ ఇన్‌స్టాలేషన్‌ల కోసం, wildcard డొమైన్ కాన్ఫిగరేషన్ కోసం WP Engine సపోర్ట్‌ను సంప్రదించమని సిఫార్సు చేయబడింది. ఇది ప్రతి ఒక్కటిని వ్యక్తిగతంగా జోడించాల్సిన అవసరం లేకుండా అన్ని సబ్‌డొమైన్‌లు స్వయంచాలకంగా పని చేయడానికి అనుమతిస్తుంది.

### SSL సర్టిఫికేట్‌లు

ఈ ఇంటిగ్రేషన్ ద్వారా జోడించబడిన అన్ని డొమైన్‌లకు WP Engine స్వయంచాలకంగా SSL సర్టిఫికేట్ జారీ మరియు రెన్యూవల్‌ను నిర్వహిస్తుంది. అదనపు కాన్ఫిగరేషన్ అవసరం లేదు.

## సమస్య పరిష్కారం

### API కనెక్షన్ సమస్యలు
- WP Engine plugin యాక్టివ్‌గా ఉండి సరిగ్గా కాన్ఫిగర్ చేయబడిందో వెరిఫై చేయండి
- మీరు API key ని మాన్యువల్‌గా డిఫైన్ చేసి ఉంటే, అది సరైనదో తనిఖీ చేయండి
- API తో సమస్యలు ఉంటే WP Engine సపోర్ట్‌ను సంప్రదించండి

### డొమైన్ జోడించబడలేదు
- ఏవైనా ఎర్రర్ మెసేజ్‌ల కోసం Ultimate Multisite లాగ్‌లను తనిఖీ చేయండి
- డొమైన్ ఇప్పటికే WP Engine కు జోడించబడలేదని వెరిఫై చేయండి
- మీ WP Engine ప్లాన్ మీరు జోడిస్తున్న డొమైన్‌ల సంఖ్యకు సపోర్ట్ చేస్తుందని నిర్ధారించుకోండి

### సబ్‌డొమైన్ సమస్యలు
- సబ్‌డొమైన్‌లు పని చేయకపోతే, wildcard డొమైన్ కాన్ఫిగరేషన్ కోసం WP Engine సపోర్ట్‌ను సంప్రదించండి
- మీ DNS సెట్టింగ్‌లు మెయిన్ డొమైన్ మరియు సబ్‌డొమైన్‌ల కోసం సరిగ్గా కాన్ఫిగర్ చేయబడి ఉన్నాయని వెరిఫై చేయండి
