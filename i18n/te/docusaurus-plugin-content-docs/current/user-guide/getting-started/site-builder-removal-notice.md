# సైట్ బిల్డర్ మోడ్ తొలగింపు నోటీసు

**Superdav AI Agent v1.12.0 లో Site Builder mode తొలగించబడింది.** మీరు Site Builder mode వాడుతున్నట్లయితే, థీమ్ సృష్టి మరియు సైట్ సెటప్ కోసం మీరు **Setup Assistant agent** కి మారాలి.

## ఏమి జరిగింది?

### Site Builder Mode (పాత విధానం)

Site Builder mode అనేది ఈ క్రింది వాటి కోసం విజర్డ్ ఆధారిత ఇంటర్‌ఫేస్:

- టెంప్లేట్‌ల నుండి సైట్‌లను సృష్టించడం
- ప్రాథమిక సెట్టింగ్‌లను కాన్ఫిగర్ చేయడం
- ఒక థీమ్‌ను ఎంచుకోవడం
- ప్రారంభ కంటెంట్‌ను సెటప్ చేయడం

### దేనితో భర్తీ చేశారు?

**Setup Assistant agent** ఇప్పుడు ఈ క్రింది వాటితో అన్ని Site Builder ఫంక్షనాలిటీని నిర్వహిస్తుంది:

- మరింత ఫ్లెక్సిబుల్, ఏజెంట్-గైడెడ్ సెటప్
- మెరుగైన థీమ్ కస్టమైజేషన్ ఆప్షన్లు
- Theme Builder ఆన్‌బోర్డింగ్‌తో అనుసంధానం
- భవిష్యత్తు సెషన్‌ల కోసం నిరంతర site_brief మెమరీ

## మీరు Site Builder Mode వాడుతున్నట్లయితే

### మీ సైట్‌లు సురక్షితంగా ఉన్నాయి

- Site Builder mode తో సృష్టించిన ప్రస్తుత సైట్‌లు పనిచేస్తూనే ఉంటాయి
- డేటా నష్టం లేదా సైట్ అంతరాయం ఉండదు
- మీరు మీ సైట్‌లను మామూలుగానే నిర్వహించవచ్చు

### Setup Assistant Agent కి మారండి

కొత్త సైట్ సెటప్ లేదా థీమ్ మార్పుల కోసం, Setup Assistant agent ని ఉపయోగించండి:

```
"Help me set up a new site"
```

లేదా

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent అనేది మరింత ఫ్లెక్సిబిలిటీతో అదే ఫంక్షనాలిటీని అందిస్తుంది.

## పోలిక: Site Builder vs. Setup Assistant

| ఫీచర్ | Site Builder (తొలగించబడింది) | Setup Assistant (కొత్తది) |
|---------|----------------------|----------------------|
| సెటప్ పద్ధతి | విజర్డ్ ఫారం | ఏజెంట్ సంభాషణ |
| థీమ్ ఎంపిక | ముందుగా నిర్వచించిన టెంప్లేట్‌లు | కస్టమ్ జనరేషన్ |
| కస్టమైజేషన్ | పరిమిత ఆప్షన్లు | పూర్తి డిజైన్ సిస్టమ్ |
| సైట్ బ్రీఫ్ | నిల్వ చేయబడదు | నిరంతర మెమరీ |
| భవిష్యత్తు సెషన్‌లు | సెటప్‌ను పునరావృతం చేయడం | నిల్వ చేయబడిన site_brief వాడకం |
| ఫ్లెక్సిబిలిటీ | స్థిరమైన వర్క్‌ఫ్లో | అనుకూల సంభాషణ |

## Setup Assistant Agent కి మదింపు చేయడం

### కొత్త సైట్‌ల కోసం

Site Builder mode ని ఉపయోగించడానికి బదులుగా:

1. అభ్యర్థించండి: "Help me set up a new site"
2. Setup Assistant agent మీకు ఈ క్రింది వాటి ద్వారా గైడ్ చేస్తుంది:
   - సైట్ ఉద్దేశ్యం మరియు లక్ష్యాలు
   - లక్ష్య ప్రేక్షకులు (Target audience)
   - బ్రాండ్ గుర్తింపు
   - థీమ్ జనరేషన్
   - ప్రారంభ కాన్ఫిగరేషన్

### ఇప్పటికే ఉన్న సైట్‌ల కోసం

మీకు Site Builder mode నుండి ఒక ఇప్పటికే ఉన్న సైట్ ఉంటే:

1. మీరు దానిని యథావిధిగా ఉపయోగించడం కొనసాగించవచ్చు
2. థీమ్‌ను అప్‌డేట్ చేయడానికి, అభ్యర్థించండి: "Redesign my site"
3. Setup Assistant agent మీకు కొత్త థీమ్‌ను సృష్టించడంలో సహాయపడుతుంది
4. మీ సైట్ డేటా మారదు

### థీమ్ మార్పుల కోసం

Site Builder mode యొక్క థీమ్ ఎంపికకు బదులుగా:

1. అభ్యర్థించండి: "Change my theme"
2. Setup Assistant agent ఈ క్రిందివి చేస్తుంది:
   - మీ డిజైన్ ప్రాధాన్యతలను అడుగుతుంది
   - కస్టమ్ థీమ్‌ను జనరేట్ చేస్తుంది
   - దానిని మీ సైట్‌లో యాక్టివేట్ చేస్తుంది

## ముఖ్య తేడాలు

### Site Builder Mode

```
1. ఒక టెంప్లేట్‌ను ఎంచుకోండి
2. ఒక థీమ్‌ను ఎంచుకోండి
3. ప్రాథమిక సెట్టింగ్‌లను కాన్ఫిగర్ చేయండి
4. పూర్తయింది
```

### Setup Assistant Agent

```
1. మీ సైట్ ఉద్దేశ్యాన్ని వివరించండి
2. మీ లక్ష్య ప్రేక్షకులను నిర్వచించండి
3. డిజైన్ ప్రాధాన్యతలను ఎంచుకోండి
4. ఏజెంట్ కస్టమ్ థీమ్‌ను జనరేట్ చేస్తుంది
5. ఏజెంట్ థీమ్‌ను యాక్టివేట్ చేస్తుంది
6. భవిష్యత్తు సెషన్‌ల కోసం సైట్ బ్రీఫ్ నిల్వ చేయబడుతుంది
```

## Setup Assistant Agent యొక్క ప్రయోజనాలు

### మరింత ఫ్లెక్సిబుల్

- మీ సైట్‌ను సహజ భాషలో వివరించండి
- కస్టమ్ సిఫార్సులను పొందండి
- మీ నిర్దిష్ట అవసరాలకు అనుగుణంగా మారుతుంది

### మెరుగైన కస్టమైజేషన్

- కస్టమ్ థీమ్ జనరేషన్
- డిజైన్ సిస్టమ్ నిర్ణయాలు
- నిరంతర డిజైన్ టోకెన్స్

### నిరంతర మెమరీ

- మీ site_brief నిల్వ చేయబడుతుంది
- భవిష్యత్తు ఏజెంట్లు మీ సైట్‌ను అర్థం చేసుకుంటారు
- సెటప్ సమాచారాన్ని మళ్లీ చెప్పాల్సిన అవసరం లేదు

### అనుసంధాన వర్క్‌ఫ్లో

- Theme Builder ఆన్‌బోర్డింగ్
- Design System Aesthetics skill
- Ability Visibility controls
- అన్నీ కలిసి సజావుగా పనిచేస్తాయి

## ట్రబుల్‌షూటింగ్

### నాకు Site Builder mode కనిపించడం లేదు

Site Builder mode తొలగించబడింది. దయచేసి Setup Assistant agent ని ఉపయోగించండి:

```
"Help me set up a new site"
```

### Site Builder నుండి సైట్‌ను పునఃసృష్టించాలనుకుంటున్నాను

మీరు Setup Assistant agent తో దానిని పునఃసృష్టించవచ్చు:

1. అభ్యర్థించండి: "Help me set up a new site"
2. మీ అసలు సైట్ ఉద్దేశ్యం మరియు డిజైన్‌ను వివరించండి
3. ఏజెంట్ ఒకేలాంటి థీమ్‌ను జనరేట్ చేస్తుంది
4. మీ site_brief భవిష్యత్తు రిఫరెన్స్ కోసం నిల్వ చేయబడుతుంది

### నా పాత Site Builder సైట్ పనిచేయడం లేదు

Site Builder mode తో సృష్టించిన ప్రస్తుత సైట్‌లు పనిచేస్తూనే ఉంటాయి. మీకు సమస్యలు ఎదురైతే:

1. మీ థీమ్ ఇంకా యాక్టివ్‌గా ఉందో లేదో తనిఖీ చేయండి
2. మీ ప్లగిన్‌లు ఎనేబుల్ అయ్యాయో లేదో ధృవీకరించండి
3. WordPress error logs ను తనిఖీ చేయండి
4. సమస్యలు కొనసాగితే సపోర్ట్‌ను సంప్రదించండి

### నేను నా పాత Site Builder టెంప్లేట్‌లను ఇంకా ఉపయోగించవచ్చా?

Site Builder టెంప్లేట్‌లు అందుబాటులో లేవు. అయితే:

- మీ ప్రస్తుత సైట్‌లు పనిచేస్తూనే ఉంటాయి
- మీరు Setup Assistant agent తో ఇలాంటి సైట్‌లను పునఃసృష్టించవచ్చు
- Setup Assistant agent మరింత కస్టమైజేషన్ ఆప్షన్లను అందిస్తుంది

## తదుపరి చర్యలు

1. **కొత్త సైట్‌ల కోసం**: Setup Assistant agent ని ఉపయోగించండి
2. **ప్రస్తుత సైట్‌ల కోసం**: యథావిధిగా ఉపయోగించడం కొనసాగించండి
3. **థీమ్ మార్పుల కోసం**: Setup Assistant agent నుండి సహాయం అభ్యర్థించండి
4. **డిజైన్ మెరుగుదల కోసం**: Design System Aesthetics skill ని ఉపయోగించండి

## సంబంధిత అంశాలు

- **Theme Builder Onboarding**: కస్టమ్ థీమ్‌ల కోసం గైడెడ్ సెటప్
- **Setup Assistant Agent**: ఏజెంట్-గైడెడ్ సైట్ సెటప్
- **Site Specification Skill**: మీ సైట్ లక్ష్యాలు మరియు ప్రేక్షకులను నిర్వచించడం
- **Design System Aesthetics Skill**: మీ సైట్ యొక్క విజువల్ గుర్తింపును మెరుగుపరచడం
