# విడుదల నోట్స్

## Version 2.12.0 — Released on 2026-05-15

- కొత్తగా: డొమైన్ మ్యాపింగ్ ఇంటిగ్రేషన్‌తో మద్దతు ఉన్న హోస్ట్ ప్రొవైడర్‌గా Hostinger (hPanel) జోడించబడింది.
- కొత్తగా: నెట్‌వర్క్ అంతటా సైట్‌ను సులభంగా పునరుద్ధరించడానికి Site Exporter ఇప్పుడు నెట్‌వర్క్ ఇంపోర్ట్ బండిళ్లను నిర్వహిస్తుంది.
- పరిష్కారం: BCC బ్రాడ్‌కాస్ట్ ఈమెయిల్‌లు ఇప్పుడు రిసీవర్ల చిరునామాలను బహిర్గతం చేయకుండా undisclosed-recipients హెడర్‌ను ఉపయోగిస్తాయి.
- పరిష్కారం: నాన్-డేట్ విలువతో సేవ్ చేసినప్పుడు Membership గడువు తేదీ ఇకపై పాడైపోదు.
- పరిష్కారం: Stripe Membership అప్‌డేట్‌లు ఇప్పుడు వాడుకలో లేని deleteDiscount APIని కాల్ చేయకుండా డిస్కౌంట్‌లను సరిగ్గా క్లియర్ చేస్తాయి.
- పరిష్కారం: డొమైన్-మ్యాప్ చేయబడిన సైట్‌లపై SSO రీడైరెక్ట్‌లు ఇప్పుడు అనంత రీడైరెక్ట్ లూప్‌లను నివారించడానికి పరిమితం చేయబడ్డాయి.
- పరిష్కారం: సెటప్ విజర్డ్ ఇమేజ్ పికర్ ఎంపిక ఇప్పుడు అంతర్లీన డేటా మోడల్‌ను సరిగ్గా అప్‌డేట్ చేస్తుంది.
- పరిష్కారం: Site Exporter CLI ఇప్పుడు సరైన డిఫాల్ట్ నెట్‌వర్క్ సైట్ ఎంపికను భద్రపరుస్తుంది.
- మెరుగుపడింది: ప్లాగీన్ సైజును తగ్గించడానికి బండిల్ చేయబడిన wp-cliని ప్యాకేజీ నుండి తొలగించబడింది.

## Version 2.11.0 — Released on 2026-05-11

- కొత్తగా: సైట్ ఎక్స్‌పోర్ట్‌లు ఇప్పుడు ఒక self-booting `index.php`ని బండిల్ చేస్తాయి, తద్వారా ZIP ఫైల్‌ను వేరే ప్లాగీన్ ఇన్‌స్టాల్ చేయకుండా కొత్త హోస్ట్‌లో ఇన్‌స్టాల్ చేయవచ్చు.
- కొత్తగా: నెట్‌వర్క్ ఎక్స్‌పోర్ట్ ద్వారా అడ్మినిస్ట్రేటర్లు Site Export అడ్మిన్ పేజీ నుండి అన్ని సబ్‌సైట్‌లను ఒకే ఆర్కైవ్‌గా ఎక్స్‌పోర్ట్ చేయవచ్చు.
- కొత్తగా: Site Templates ప్లాన్ టోగుల్ ఇప్పుడు ఫాల్‌బ్యాక్ 체యిన్ ద్వారా అమలు చేయబడుతుంది, తద్వారా ప్లాన్ పరిమితుల కోసం టెంప్లేట్ లభ్యతను సరిగ్గా పరిమితం చేస్తుంది.
- కొత్తగా: చెక్అవుట్ ఫారం ఎడిటర్, తప్పనిసరి ఫీల్డ్ కాన్ఫిగర్ చేయకుండా ఉత్పత్తిని జోడించినప్పుడు హెచ్చరిస్తుంది.
- కొత్తగా: Import/Export సెట్టింగ్‌ల ట్యాబ్ దాని పరిధిని స్పష్టంగా వివరిస్తుంది మరియు Site Export టూల్‌కు నేరుగా లింక్ చేస్తుంది.

## Version 2.10.0 — Released on 2026-05-05

- కొత్తగా: OAuth ఫ్లాగ్ గేట్‌తో మాన్యువల్ క్రెడెన్షియల్ ఎంట్రీ కోసం PayPal గైడెడ్ సెటప్ విజర్డ్, సులభమైన గేట్‌వే కాన్ఫిగరేషన్ కోసం.
- కొత్తగా: Template switch కస్టమర్ ప్యానెల్ ఇప్పుడు current-template కార్డ్, పర్సిస్టెంట్ గ్రిడ్ మరియు **Reset current template** బటన్‌తో తిరిగి రూపొందించబడింది.
- పరిష్కారం: టెంప్లేట్ స్విచ్చింగ్ ఇప్పుడు AJAX విఫలమైనప్పుడు UI ఆగిపోదు (hang అవ్వదు).
- పరిష్కారం: టెంప్లేట్ స్విచ్చింగ్ అనుమతి స్టేట్‌లు అనధికారిక యాక్సెస్‌కు వ్యతిరేకంగా సురక్షితం చేయబడ్డాయి.
- పరిష్కారం: సైట్ ఓవర్‌రైడ్ ఇన్‌పుట్‌లు సేవ్ చేయడానికి ముందు ధృవీకరించబడతాయి (validated).
- పరిష్కారం: చిరునామా ఖాళీగా ఉన్నప్పుడు బిల్లింగ్ చిరునామా ప్రాంప్ట్ ఇప్పుడు చూపబడుతుంది.
- పరిష్కారం: PHP 8.1 null-to-string deprecation నోటీసులు పరిష్కరించబడ్డాయి.
- పరిష్కారం: Currents lazy-loaded before init hook, సమయ సమస్యలను నివారించడానికి.
- పరిష్కారం: అన్ని లాగిన్ ఫ్లోలలో ఫిల్టర్ చేయబడిన SSO మార్గాన్ని (path) గౌరవిస్తుంది.
- పరిష్కారం: ఖాళీ సైట్ ఐడెంటిటీ ఆప్షన్‌లు సేవ్ చేసినప్పుడు భద్రపరచబడతాయి.
