# Ultimate Multisite అవసరాలు

Ultimate Multisite కొనుగోలు చేయడానికి ముందు మీరు మొదట అడిగే ప్రశ్న ఏమిటంటే - మా plugin నడపడానికి ఏమేమి అవసరం? అదృష్టవశాత్తు, Ultimate Multisite నడపడానికి కొన్ని ప్రాథమిక అవసరాలు మాత్రమే ఉన్నాయి: ఒక **domain name**, మంచి **hosting provider** మరియు WordPress **Multisite** installation. ఇంతే మీకు కావలసింది!

## Domain Name

ముందుగా, domain name గురించి మాట్లాడుదాం.

Domain name ఎంచుకునేటప్పుడు, మీ నిర్దిష్ట niche ఏమిటో మరియు Ultimate Multisite తో మీరు ఏమి సృష్టించబోతున్నారో గుర్తుంచుకోండి: మీరు ఒక agency సృష్టించాలనుకుంటున్నారా? Shop-like websites అమ్మాలనుకుంటున్నారా? లేదా WordPress web hosting ఏర్పాటు చేయాలనుకుంటున్నారా? Ultimate Multisite తో అన్నీ సాధ్యమే.

మా plugin subscription services, payment collection, checkout forms, discount vouchers మరియు customer communications అందిస్తుంది. WordPress Multisite install చేయడానికి, configure చేయడానికి మరియు maintain చేయడానికి అవసరమైన చాలా పనిని Ultimate Multisite సులభతరం చేస్తుంది.

మీరు websites as a service అమ్మినంత కాలం, మీరు ఎంచుకున్న ఏ niche అయినా మా plugin ఉపయోగించి నిర్వహించవచ్చు! మీ business plan కు సరిపోయే domain name ఎంచుకోండి.

## Hosting Provider

మీ niche మరియు business కు సరిపోయే ఆకర్షణీయమైన domain name ఉన్న తర్వాత, Ultimate Multisite తో మీ network host చేయడానికి మంచి web hosting company ఎంచుకోవాలి. మార్కెట్లో కొన్ని మంచి options ఉన్నాయి. కొన్ని hosting providers Ultimate Multisite తో బాగా integrate అవుతాయి, మీరు ఏమి build చేయాలనుకుంటున్నారో బట్టి ఇది నిర్ణయాత్మక అంశం కావచ్చు.

Ultimate Multisite ఏ hosting provider లోనైనా పని చేయాలి. ఒక్క తేడా ఏమిటంటే _WPMU DEV Hosting_, _WP Engine_, _Closte_, _Cloudways_, _Gridpane_ మరియు _Runcloud_ వంటి కొన్ని hosting companies domain mapping కోసం మెరుగైన integration కలిగి ఉంటాయి. ఇతర hosting లో, మీరు custom domains ను manually map చేయాల్సి రావచ్చు. AutoSSL support ఉన్న hosting provider తీసుకోవడం కూడా మంచి ఆలోచన. _Closte_, _Runcloud_ మరియు _cPanel_ దీనికి support ఇస్తాయి.

Server specifications విషయంలో, మీరు అందించే features మరియు functions పై మరియు మీ network పరిమాణంపై ఆధారపడి ఉంటుంది కాబట్టి నిర్దిష్ట formula ఏదీ లేదు. Shared hosting మంచి ప్రారంభం కావచ్చు, కానీ మీ network పెరగడం మొదలయ్యాక, మీకు బహుశా dedicated VPS అవసరమవుతుంది.

## WordPress Multisite

ఇప్పుడు, WordPress Multisite installation గురించి కొంచెం మాట్లాడుదాం.

WordPress Multisite ఒకే installation లో sites network కలిగి ఉండటానికి అనుమతిస్తుంది. ఇది built-in feature, కానీ default గా active గా ఉండదు. మీ WordPress installation మరియు web hosting control panel లో ఈ feature ను enable చేయాలి.

Multisite enable చేసిన WordPress **Ultimate Multisite కు base**. Ultimate Multisite యొక్క ప్రధాన ఉపయోగం users కు websites as a service manage చేయడానికి మరియు అమ్మడానికి సహాయం చేయడం కాబట్టి, ఇది Multisite enable చేసిన WordPress లో చేయాలి.

మీ WordPress installation లో Multisite feature enable చేయడానికి ఏమి చేయాలో step by step నేర్పించే **సమగ్ర వ్యాసం** మా దగ్గర ఉంది.

WordPress లో Multisite enable చేసిన తర్వాత, Ultimate Multisite install చేయడానికి సమయం వచ్చింది. Ultimate Multisite సరిగ్గా install చేయడానికి, మీ దగ్గర ఇవి ఉండాలి:

  * WordPress: v5.3+ (సిఫార్సు: చివరి stable version)

  * PHP: 7.4.x (> 8.0 support త్వరలో రాబోతోంది)

  * MySQL: v5+ (సిఫార్సు: 5.6, మీ hosting provider లో 8.0 అందుబాటులో లేకపోతే)

Ultimate Multisite కు ఇవి మాత్రమే software requirements.

Ultimate Multisite PHP 8.0 తో పని చేయవచ్చు కానీ PHP 7.4.x తో run చేయమని మేము సిఫార్సు చేస్తున్నాము.

అలాగే, **main domains** లో install చేయమని మేము సిఫార్సు చేస్తున్నాము, subdomains లేదా subdirectories లో కాదు. ఇది subdomains/subdirectories లో పని చేయవచ్చు, కానీ కొన్ని errors రావచ్చు.
