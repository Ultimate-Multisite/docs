# Action: wu_mcp_abilities_registered

MCP సామర్థ్యాలు (abilities) ఒక ఎంటిటీ కోసం రిజిస్టర్ అయిన తర్వాత ఇది ట్రిగ్గర్ అవుతుంది.

## పారామీటర్లు (Parameters)

| పేరు (Name) | రకం (Type) | వివరణ (Description) |
|------|------|-------------|
| $ability_prefix | `string` | సామర్థ్యం యొక్క ప్రిఫిక్స్ (ability prefix). |
| $model_name | `string` | మోడల్ పేరు (model name). |
| $this | `object` | మేనేజర్ ఇన్‌స్టాన్స్ (manager instance). |

### ఎప్పటి నుండి (Since)

- 2.5.0
