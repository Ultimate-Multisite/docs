# Action: wu_sso_loaded

అవసరమైతే, ప్లగిన్ డెవలపర్‌లు అదనపు హుక్స్ (hooks) జోడించడానికి ఇది అనుమతిస్తుంది.

SSO అనేది `sunrise` సమయంలో నడిచే కాబట్టి, దీన్ని `init` వరకు ఆలస్యం చేయవలసి ఉంటుంది.
