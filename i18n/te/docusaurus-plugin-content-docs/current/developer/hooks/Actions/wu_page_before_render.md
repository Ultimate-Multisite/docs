# Action: wu_page_before_render

పేజీని ప్రదర్శించడానికి (print) ముందు, ప్లగిన్ డెవలపర్‌లు అదనపు కంటెంట్‌ను జోడించడానికి ఇది అనుమతిస్తుంది.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | ఈ పేజీ యొక్క ID. |
| $page | `object` | పేజీ ఆబ్జెక్ట్. |
