# Action: wu_mcp_adapter_initialized

MCP అడాప్టర్ ప్రారంభించబడిన తర్వాత (initialized) ఇది ట్రిగ్గర్ అవుతుంది.

ఇతర ప్లగిన్‌లు మరియు థీమ్‌లు తమ సొంత సామర్థ్యాలను (abilities) నమోదు చేసుకోవడానికి ఇది అనుమతిస్తుంది.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP అడాప్టర్ యొక్క ఉదాహరణ (instance). |
