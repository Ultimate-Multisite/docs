# Action: wu_{$this->id}_scripts {#action-wuthis-idscripts}

స్క్రిప్ట్‌లను ఎన్‌క్యూ చేసే హుక్‌ను (hook) ఇది ట్రిగ్గర్ చేస్తుంది.

ఈ ఎలిమెంట్ దాని `register_scripts` పద్ధతిని హుక్ చేయడానికి దీనిని ఉపయోగిస్తుంది.
