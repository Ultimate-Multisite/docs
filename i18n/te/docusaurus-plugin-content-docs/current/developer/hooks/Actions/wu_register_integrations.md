# Action: wu_register_integrations

ఇంటిగ్రేషన్స్ తమను తాము రిజిస్టర్ చేసుకోవడానికి ఇది ట్రిగ్గర్ అవుతుంది.

కోర్ (Core) ఇక్కడ దాని ఇంటిగ్రేషన్స్‌ను రిజిస్టర్ చేస్తుంది.

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | రిజిస్ట్రీ ఇన్‌స్టాన్స్. |
