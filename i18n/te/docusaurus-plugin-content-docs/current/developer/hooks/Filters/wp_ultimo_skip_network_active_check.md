# Filter: wp_ultimo_skip_network_active_check

ఈ తనిఖరాన్ని (check) దాటవేయడానికి (short-circuit) డెవలపర్‌లకు ఇది అనుమతిస్తుంది.

మీరు composer ఆధారిత మరియు ఇతర కస్టమ్ సెటప్‌లను ఉపయోగిస్తున్నప్పుడు, ఉదాహరణకు Bedrock వంటి వాటిలో, mu-pluginsని ప్లగిన్‌లుగా ఉపయోగించడం సాధారణంగా ఉన్నప్పుడు ఇది ఉపయోగపడుతుంది.
