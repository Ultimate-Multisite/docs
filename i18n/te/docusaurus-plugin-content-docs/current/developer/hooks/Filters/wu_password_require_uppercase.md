# Filter: wu_password_require_uppercase {#filter-wupasswordrequireuppercase}

పాస్‌వర్డ్‌లలో అప్పర్కేస్ (పెద్ద అక్షరాలు) అవసరమా కాదా అని నిర్ణయించడానికి ఉపయోగించే ఫిల్టర్ ఇది.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | అప్పర్కేస్ అవసరమా కాదా. నియమాలు అమలు చేయబడినప్పుడు డిఫాల్ట్‌గా ఇది true ఉంటుంది. |
| $defender_active | `bool` | Defender Pro Strong Password యాక్టివ్‌గా ఉందో లేదో. |
