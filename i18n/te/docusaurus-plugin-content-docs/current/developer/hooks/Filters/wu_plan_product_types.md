# Filter: wu_plan_product_types

ప్లాన్‌లుగా పరిగణించాల్సిన ఉత్పత్తి రకాలను ఫిల్టర్ చేస్తుంది.

ఈ ఫిల్టర్ అడాన్‌లు (addons) ధ్రువీకరణ (validation) మరియు విభజన (segregation) సమయంలో ప్లాన్‌లుగా గుర్తించబడవలసిన అదనపు ఉత్పత్తి రకాలను నమోదు చేయడానికి అనుమతిస్తుంది.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | ప్లాన్‌లుగా పరిగణించాల్సిన ఉత్పత్తి రకాల శ్రేణి (Array). |
