# Filter: wu_maybe_redirect_to_admin_panel_exclusion_list

ఆటో-రీడైరెక్ట్‌ను నిరోధించే పారామీటర్ల మినహాయింపు జాబితాను (exclusion list) సృష్టించండి.

దీనిని ఉపయోగించకపోతే, పేజ్ బిల్డర్ (page builder) లాగిన్ పేజీని ఎడిట్ చేయలేకపోవచ్చు, ఒకసారి అది నిర్వచించబడిన తర్వాత.
