# Filter: wu_pre_save_settings

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | సేవ్ చేయవలసిన సెట్టింగ్‌లు. |
| $settings_to_save | `array` | జోడించాల్సిన కొత్త సెట్టింగ్‌లు. |
| $saved_settings | `array` | ప్రస్తుతం సేవ్ చేయబడిన సెట్టింగ్‌లు. |
