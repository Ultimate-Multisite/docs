# Filter: wu_checkout_validation_rules

ప్లగిన్ డెవలపర్‌లు ధ్రువీకరణ నియమాలను (validation rules) ఫిల్టర్ చేయడానికి ఇది వీలు కల్పిస్తుంది.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $validation_rules | `array` | ఉపయోగించాల్సిన ధ్రువీకరణ నియమాలు. |
| $checkout | `\Checkout` | చెక్అవుట్ క్లాస్. |
