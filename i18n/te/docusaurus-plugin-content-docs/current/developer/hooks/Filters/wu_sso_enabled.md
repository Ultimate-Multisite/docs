# Filter: wu_sso_enabled

క్రాస్-డోమైన్ సింగిల్-సైన్-ఆన్ (single-sign-on) సామర్థ్యాన్ని ఎనేబుల్/డిసేబుల్ చేయడానికి ఉపయోగిస్తారు.

సింగిల్-సైన్-ఆన్‌ను పూర్తిగా ఆపివేయడానికి లేదా షరతులతో కూడిన విధంగా ఎనేబుల్ చేయడానికి ఈ విలువను ఫిల్టర్ చేయవచ్చు.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | SSO ఎనేబుల్ చేయాలా? ఆన్ చేయడానికి True, ఆఫ్ చేయడానికి False. |

### Since {#since}

- 2.0.11
### Source {#source}

[`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) లోని 110వ లైన్‌లో నిర్వచించబడింది.

## Returns {#returns}
SSO ఎనేబుల్ చేయబడిందా లేదా అనే సమాచారాన్ని తిరిగి ఇస్తుంది.
