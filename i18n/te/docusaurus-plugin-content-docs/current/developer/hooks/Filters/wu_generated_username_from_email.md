# Filter: wu_generated_username_from_email

రూపొందించబడిన కస్టమర్ యూజర్‌నేమ్‌ను ఫిల్టర్ చేస్తుంది.

## పారామీటర్లు {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $username | `string` | రూపొందించబడిన యూజర్‌నేమ్. |
| $email | `string` | కొత్త కస్టమర్ ఇమెయిల్ చిరునామా. |
| $new_user_args | `array` | కొత్త యూజర్ ఆర్గ్యుమెంట్ల శ్రేణి, ఇందులో మొదటి మరియు చివరి పేర్లు కూడా ఉండవచ్చు. |
| $suffix | `string` | యూజర్‌నేమ్‌ను ప్రత్యేకంగా (unique) చేయడానికి జోడించాల్సిన స్ట్రింగ్. |
