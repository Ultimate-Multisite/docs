# Filter: wu_bypass_checkout_form

## పారామీటర్లు {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $bypass | `string\|bool` | చెక్అవుట్ ఫారమ్‌ను బైపాస్ చేయాలా, లేదా ఫారమ్ స్థానంలో తిరిగి ఇవ్వాల్సిన స్ట్రింగ్ (string) ఏమిటి అనే వివరాలు. |
| $atts | `array` | చెక్అవుట్ బ్లాక్/షార్ట్‌కోడ్ యొక్క పారామీటర్లు. |
