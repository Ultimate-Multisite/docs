# Filter: wp_ultimo_redirect_url_after_signup {#filter-wpultimoredirecturlaftersignup}

రీడైరెక్ట్ URLను సెట్ చేయడానికి ఇది ఉపయోగపడుతుంది.

ఇది పాత ఫిల్టర్ (legacy filter). ఇక్కడ పంపబడే కొన్ని పారామీటర్లు, ఉదాహరణకు $site_id, అందుబాటులో ఉండకపోవచ్చు.

### ఎప్పటి నుండి {#since}

- 1.1.3: డెవలపర్‌లు రీడైరెక్ట్ URLను ఫిల్టర్ చేయడానికి ఇది వీలు కల్పిస్తుంది.
