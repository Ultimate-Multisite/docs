# Filter: wu_password_require_number {#filter-wupasswordrequirenumber}

పాస్‌వర్డ్‌లలో నంబర్‌లు అవసరమా కాదా అని నిర్ణయించడానికి ఉపయోగించే ఫిల్టర్.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | నంబర్‌లు అవసరమా కాదా అని నిర్ణయిస్తుంది. నియమాలు అమలు చేయబడినప్పుడు డిఫాల్ట్‌గా ఇది true ఉంటుంది. |
| $defender_active | `bool` | Defender Pro Strong Password యాక్టివ్‌గా ఉందో లేదో తెలుపుతుంది. |
