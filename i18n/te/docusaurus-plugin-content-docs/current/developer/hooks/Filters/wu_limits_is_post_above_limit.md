# Filter: wu_limits_is_post_above_limit

ఇచ్చిన పోస్ట్ రకం (post type) ఈ ప్లాన్‌లో అనుమతించబడిందో లేదో ఇది చెక్ చేస్తుంది. ప్లగిన్ డెవలపర్‌లు ఈ రిటర్న్ విలువను ఫిల్టర్ చేయడానికి దీనిని ఉపయోగించవచ్చు.
