# Filter: wu_documentation_get_link

ప్లగిన్ డెవలపర్‌లు లింక్‌లను ఫిల్టర్ చేయడానికి ఇది అనుమతిస్తుంది.

ఇది ఎంత ఉపయోగకరంగా ఉంటుందో నాకు ఖచ్చితంగా తెలియదు, కానీ దీన్ని ఉంచడంలో ఎలాంటి నష్టం లేదు.

## పారామీటర్లు

| Name | Type | వివరణ |
|------|------|-------------|
| $link | `string` | రిజిస్టర్ చేయబడిన లింక్ |
| $slug | `string` | లింక్‌ను తిరిగి పొందడానికి ఉపయోగించే స్లగ్ |
| $default_link | `string` | రిజిస్టర్ చేయబడిన డిఫాల్ట్ లింక్ |
