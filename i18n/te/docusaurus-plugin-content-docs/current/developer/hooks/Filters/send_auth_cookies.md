# Filter: send_auth_cookies

క్లయింట్‌కు ఆథెంటికేషన్ కుకీలు (auth cookies) వాస్తవంగా పంపబడకుండా నిరోధించడానికి ఇది సహాయపడుతుంది.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $send | `bool` | క్లయింట్‌కు ఆథెంటికేషన్ కుకీలను పంపాలా వద్దా అని నిర్ణయిస్తుంది. |
