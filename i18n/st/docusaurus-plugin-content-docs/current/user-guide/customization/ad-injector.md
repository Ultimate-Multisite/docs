# Go Pelele Ads ka Netong ea hau le Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector ke **add-on e tshepa** e ntse u le litaba ho ba le litaba tsa hau ka sebope sa ho edit.

Ha o ntshepa Ultimate Multisite: Ad Injector, a go tla ka _Plugins_ page ea hau ho ba lehle.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Thaba le "Network Activate"_

Ha e le add-on e tshepa, _Advanced Options_ meta box ea sebope sa ho edit o tla ba le bonolo o ntlha o tla bolela "Ad Settings".

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Lona u ka edit litaba tsa ad ho tsoalo ea hau tse fapaneng tsa sebope sa hau._

Ka tab eona, u tla ka ntshepa codes ea ad ho posts le pages ea front-end, haholo le fa le fa content.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Ha o tsena codes ea ad ea hau, e tla bona ka posts le pages ea sebope seo sehlopeng sa hau._

U ka ba le ho fana ka ads tse fapaneng ka header ea Admin Pages ho batho ba le sebope seo.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Ha u batla ho bona ads ka back-end ea hau ea bophelo, a go tla ka _Admin Panel Ads_ option._

Ha o tsena _Admin Panel Ads_ option le ntshepa HTML ea hau ea tool ea tšepo.
