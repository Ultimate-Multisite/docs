# Tšhobo ea Dashboard {#dashboard-overview}

**Tšhobo ya Mofuta:** Ho fana ka mofuta o fapaneng ka mofuta le mofuta o fapaneng ka mofuta ho tsamaisa tšhobo ya dashboard ka mofuta o oona jo o nang le se o na le se o nang le se. Tšhobo ya mofuta e fana ka mofuta o oona jo o nang le se o na le se o nang le se.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Tšhobo tsa dashboard di fana:

- **Mofuta** — Mofuta o tšhobo ho mofuta jo o tsamaisa ka mofuta le mofuta o fapaneng ka mofuta.
- **Mofuta O Ntlha (New Memberships)** — Ke mofuta o ntlha o ne a fetoga ka mofuta o oona jo o tsamaisa ka mofuta.
- **Tšhobo Ho Tsamaisa ka Lenta** — Mofuta o fapaneng ka lenta la tšhobo ya tsamaiso.
- **Mofuta O Ntlha Ka Lenta** — Ke mofuta o ne a fetoga ka mofuta jo o tsamaisa ka mofuta le mofuta o fapaneng ka mofuta ho ntlha.
- **Sites Le Ho Tsamaiswa Hloa (Most Visited Sites)** — Sites tse di tsamaiswang hlobo sa lona (di hloka ho feta ho fana ka tšhobo ya tsamaisa ka Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting).
