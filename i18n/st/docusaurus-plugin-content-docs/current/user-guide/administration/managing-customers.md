# Ho Fana a Moreroa (Managing Customers)

Peje ya **Customers** (Moreroa) le bontsha lishomelelo la ba bangwe ba ba ntswetseng ba ba tsamaisang ka network ya hau. Peje e bontsha litlhahloho tsa boikotso jwa mofuta o o tshwanang, ka metsi ya email, tsela ya go tsamaya (last login), le sebope **switch to** se se bontsha hore u ka tsamaya ka user ya subsite ka ho ba site administrator.

![Customers list page](/img/admin/customers-list.png)

Thaba **Manage** se se o tla bona pele ga mofuta o o tshwanang ho bona litlhahloho tsa boikotso jwa mofuta o o tshwanang.

Peje ya tshedi ya mofuta o o tshwanang, sebope **At a Glance** se bontsha ID ya mofuta o o tshwanang, tsela ya go tsamaya (last login time), le sebope sa metsi o o tsamaisang ke mofuta o o tshwanang.

Sebope sa **Customer Options** se se bontsha hore u ka dira mofuta o o tshwanang o o ba **VIP** (sebope se se fetang ho e tshepa ho e bona haholo). Sebope sa **Billing Info** se bontsha metsi ya boikotso jo boikotso jwa email, tsela, le leka.

Sebope sa **Fields** se bontsha username ya mofuta o o tshwanang, email, le se se se fetang status ya **Email Verification**.

Mme ka botshelong ba sebope sa Customer Options, u tla bona sebope se se fapaneng se se fapaneng:

- **Memberships** — Sebope sa boikotso jo boikotso jwa mofuta o o tshwanang o o tsamaisang
- **Payments** — Litlhahloho tsa metsi ya boikotso jwa mofuta o o tshwanang
- **Sites** — Sites tse mofuta o o tshwanang o o se na le
- **Events** — Litlhahloho tse di tsamaisang le mofuta o o tshwanang (verification ya email, metsi ya boikotso jo boikotso jwa metsi, le ho ba ka dira refund)

Thaba arrow icon (→) se se fapaneng se se fetang ho e bona peje ya tshedi. Ka ntlha, go tsamaya ka arrow se se fetang ho peje ya ho edit site se se o tla u tloha peje ya ho edit site se se na le sebope sa boikotso jo boikotso jwa metsi, metsi ya mofuta, le status ya ho baetso.

Se se fetse ka memberships le payments — se se fapaneng se se fetang ho peje ya tshedi se se o tla u tloha ho edit status ya membership, mofuta o o tshwanang, metsi ya boikotso jo boikotso jwa metsi, metsi ya gateway ya metsi, le ho dira refund.

Ka ntlha ya peje ya mofuta o o tshwanang, u tla bona:

- **Ho Lihlobo Lehle & IPs** — Khoenjalo la login, lapa la IP, le leka (country)
- **Tsoatsa (Timestamps)** — Ka mofuta o ne a fetileng le ka mofuta o ne a fetileng ho tsoatsa la ntshwalo la morao
- **Ho Tsoatsa** — Opion ho tsoatsa morao ka ho fana ka ho feta ka ho fana

:::warning
Ho tsoatsa morao ho fana ka ho fetoga ka ho fana le ho nna le ho tsamaea.
:::
