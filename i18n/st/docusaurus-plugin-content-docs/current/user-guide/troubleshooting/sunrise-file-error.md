# Thaoa Hohope ka Sunrise File

Le file sunrise.php ke le file e le file se thata ka WordPress e le a go tlhola ho itse. Ho ka ntlha ka WordPress ho ka tsebe file sunrise.php, e hloka e le lehlile ka folder ya **wp-content**.

E re Ultimate Multisite e le ka fana ka file our sunrise.php ka folder ya wp-content.

<!-- Screenshot unavailable: Setup wizard page showing sunrise.php installation step -->

Ka nako a ba leboha, re ka tseba ho kopora file se ka hlakileng. Le fa go na le ntho e ntse e le teng (ka lehlile ka tlhahlo ya folder, re lebisa), o ka le ka leba ka ntho e ntse e le teng ka Ultimate Multisite e ka fana ka file.

Ha o bua le litlhahlo la ho dirwa ke Ultimo, o tla bona se se tseba se se dirileng: **Sunrise copy failed**.

<!-- Screenshot unavailable: Error message showing Sunrise copy failed -->

Ho a fetola ntho e tsebahala, o ka kopora file sunrise.php mo folder ya wp-ultimo plugin le ho fana ka e mo folder ya wp-content. Ha o etsa ntho eona, o tla bua le litlhahlo la ho tsamaya ka wizard page le tlhahlo e tla baetsoeng.

<!-- Screenshot unavailable: File manager showing sunrise.php inside wp-ultimo plugin folder --> Le fa go na le ntho e ntse e le teng, o ka le thata ho tseba ka tlhahlo ya folder ea hau ho fana ka ho tsamaya ka ho hlakileng (le fa lehlile ka Ultimate Multisite le li plugin le themes tse ling).

**Health Check tool** e ntse e le seka sa WordPress (o ka e tsamaya ka **admin panel > Tools > Health Check** ea site ea hau) e ka go bona hore o na le tlhahlo ya folder e ntse e le ho ba le ntho e ka etsa mangelo le a ka etsa problems ka WordPress.
