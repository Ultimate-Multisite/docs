# Ho Lapa Le Litlhokwa Lo Litlhokwa (Shareable Links)

Ultimate Multisite o na le tsela e ntle ho u ka rona litlhokwa la lapa (plan links) ho fana ka baaka ba hau. Ho bala, fana ka page ea ho edit oa lapa hau le khutlisa bohlokoa ba "Click to copy the Shareable Link".

![Peti ea ho hlophisa le liphakiso](https://example.com/img/config/product-edit-full.png)

_Ha o tla ba ka le Page e fanaang le Plan, u tla bona boithakiso ba ho fetola liphakiso tse ka hlalosang._

A re, ke tseba se fa tsela e fanaang le ho fetola link. Re batla ho tseba hore sebo se se fetola link se se sebelisa se se tsamaisang (default billing frequency). Ho na le sefapso sa hau se se tsamaisang se bo-monthly, mohlodi, ho a tloha, o tla le link se se ba le se se fetileng se se se fetola: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Ha o batla hore sehlokomelo sa shareable link se fanahe ho ba le quarterly, o tla hloka ho feta 3 ho a tsoalo.

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Ho ba le plan _premium_ ka tsela ya boano, o tla tsamaetsa ho fana ka 1 le litlhoko:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Hloka le hore ho tseba hore haholo ho ka na le lii link tse fapaneng tsa tšepo ea khutso (billing frequencies). Ho hloka ho ba le le link le le fetang le le leng le le fetang le le leng le fetang.

Ha o fana ka sebope le ba hau, ho tla ba fana ka khothatso ho laola (signup) haholo.
