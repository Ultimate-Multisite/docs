# Ka dirisi ka ubatla Ultimate Multisite snippets ka GitHub repository ea ronao

Ho na leetsoets'a (code snippets) tse di fali ka GitHub repository e leng ba ba tlameha ho ba le functionality e le e le lehle, jo bo ba ba ba bang ba Ultimate Multisite ba batlang ho ba le Google Analytics script ka pages tsa sign-up kapa ho ba le meta box e le e lehle mo admin dashboard.

Lia le tsela ea hore o bone hore na leba ho ubatla tse.

O ka bona snippets tse di fali ka link seletse:

https://github.com/next-press/wp-ultimo-snippets/

Ho na le litse tse 2 tsa hore o ubatle code

  1. Ka file ea functions.php ea theme ea hau.

  2. Must-Use Plugins (mu-plugins)

# Ka dirisi ka snippet ea hau ka file ea functions.php ea theme ea hau.

  1. Ho fihla mo WordPress Network admin dashboard le go tla ka Themes >Theme Editor (Bone screenshot e lebelo).

  2. Ka page ea "Edit Themes", re le hore o bone seo sehlopetso sa theme ea hau e le e lehle ka dropdown field eo e teng fa mofuta oa ho lehle ha lehle mo lefelong la ho lehle la hau (e le #3 ea screenshot e lebelo).

  3. Thaba file ea functions.php ka botle ea section ea "Theme Files" ho u batla file. Ho tsamaela ka ho lehle ho tla ka ho fana ka Ultimate Multisite snippet o ntse o bone oa GitHub repository.

# Ka dirisi ka Must-Use Plugins (mu-plugins)

WordPress e na le feature ea hore o ka u batla functionality e le e lehle ea custom e bolelang "Must-Use Plugins", kapa "mu-plugins" ho fana.

E mu-plugins tse nthese tse di batla ho ba lehle fa litse tse ling tsa plugins, haholo, ha ho ka ba lehle (deactivated). Mo multisite network, code ea mu-plugins e tla ubatla fa litse tsohle tsa hau.

1\. U batla FTP kapa SSH ho ba lele mo filesystem ea WordPress ea hau.

2. Moroka mo directory wa WordPress o tla dirago, tsabela direka le ntlha ya: mu-plugins.

<!-- Screenshot e ne e ntse a fana: File manager e bontsha wp-content directory le folder mu-plugins -->

3. Tsabela file PHP se se ntse ka computer ya hae se se ntse ka wu-snippet.php ka go rata Notepad kgotsa code editor.

4. Setse tsabela code snippet ya Ultimate Multisite e e ntse ka GitHub repository mo file, kgotsa setse e le ntlha ya code snippet ho dira gore o bolelle mu plugin wa hau.
