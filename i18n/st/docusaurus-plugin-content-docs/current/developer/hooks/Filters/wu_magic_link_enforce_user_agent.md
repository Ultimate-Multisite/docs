# Filter: wu_magic_link_enforce_user_agent

Sefa hore na netefatso ya user agent e qobellwe.

E behe ho false ho dumella ditoken ho sebetsa ho dibrowser/disebediswa tse fapaneng. Sena se fokotsa tshireletso empa se eketsa bonolo ba tshebediso.

## Dipharamitha {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | Hore na ho qobellwe ho bapisa user agent. |
