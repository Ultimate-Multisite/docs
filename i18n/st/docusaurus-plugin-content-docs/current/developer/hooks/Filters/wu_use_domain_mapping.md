# Sefe: wu_use_domain_mapping {#filter-wuusedomainmapping}

Etsa qeto hore na mapping e lokela ho sebelisoa

Hangata, o tla batla ho lumella feela mappings tse sebetsang hore li sebelisoe. Leha ho le joalo, haeba o batla ho sebelisa logic e tsoetseng pele haholoanyane, kapa ho lumella domains tse sa sebetseng hore le tsona li etsoe mapping, sefa mona feela.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $is_active | `bool` | Na mapping e lokela ho nkoa e le e sebetsang? |
| $mapping | `\Domain` | Mapping eo re e hlahlobang |
| $domain | `string` |  |
