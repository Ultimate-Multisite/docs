# Action: wu_page_\{$this->id\}_register_widgets {#action-wupagethis-idregisterwidgets}

E qala ka mor'a hore widgets di ngodiswe bakeng sa leqephe lena.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Dipharamitha {#parameters}

| Lebitso | Mofuta | Tlhaloso |
|------|------|-------------|
| $id | `string` | id ya leqephe. |
| $page_hook | `string` | hook ya leqephe. |
| $page | `object` | object ya leqephe. |
