# Ketso: switch_network {#action-switchnetwork}

E thunya ha moelelo oa network ea hajoale o fetoloa.

## Dipharamitha {#parameters}

| Lebitso | Mofuta | Tlhaloso |
|------|------|-------------|
| $new_network_id | `int` | ID ea network eo ho fetoleloang ho eona. |
| $old_network_id | `int` | ID ea network e neng e le ea hajoale pele. |
