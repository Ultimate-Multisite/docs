# Sefe: networks_pre_user_is_network_admin {#filter-networkspreuserisnetworkadmin}

E sefa dinetwork tseo mosebedisi e leng motsamaisi wa tsona, ho kgutsufatsa tshebetso.

## Dipharamitha {#parameters}

| Lebitso | Mofuta | Tlhaloso |
|------|------|-------------|
| $ | `array\|bool\|null` | Lenane la di-ID tsa network kapa false. Ntho efe kapa efe ntle le null e tla kgutsufatsa tshebetso. |
| $ | `int` | ID ya mosebedisi eo dinetwork di lokelang ho kgutliswa bakeng sa yona. |
