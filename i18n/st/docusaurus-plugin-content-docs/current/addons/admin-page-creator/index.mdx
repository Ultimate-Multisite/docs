# Admin Page Creator

## Dikahare {#content}
Admin Page Creator e tla o dumella ho eketsa maqephe a matjha a tsamaiso le maqephe a manyane habonolo ka hara WordPress admin dashboard. E ka sebediswa ho etsa leqephe la kamohelo kapa leqephe la thuto le ka sebetsang e le leqephe la ho fihlela bakeng sa basebedisi ba batjha. O ka sebedisa page builder jwalo ka Beaver Builder, Elementor, Brizy, le Oxygen ho rala maqephe ntle le bothata.

E boetse e fana ka tshehetso ya HTML, CSS le JavaScript ya moetlo, hammoho le taolo ya phihlello e itshetlehileng hodima plan le karolo ya mosebedisi!

## Dikarolo {#features}
* Tshehetso ya HTML le CSS ya moetlo
* Tshehetso ya JavaScript
* Mokgwa wa Dikahare tsa PHP
* Tumello e itshetlehileng hodima karolo ya Mosebedisi
* Tshehetso ho page builders

## Ditokomane {#documentation}
[Ho etsa Maqephe a Tsamaiso a Moetlo le Maqephe a Manyane ka Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Ho kenya {#installation}

1. Kenya **ultimate-multisite-admin-page-creator** bukeng ya **/wp-content/plugins/**
2. Kenya plugin tshebetsong ka menu ya **Plugins** ho WordPress Network Admin ya hao

## Tataiso ya Tshebediso {#usage-guide}

Le hoja WordPress e etsa hore ho be bonolo haholo ho eketsa dikahare karolong e ka pele ya disete, ho etsa maqephe le maqephe a manyane ka lehlakoreng la tsamaiso ho hloka bokgoni bo itseng ba motheo ba ho ngola khoutu. Ultimate Multisite: Admin Page Creator e ikemiseditse ho tlisa bonolo boo bo tshwanang ba tshebediso ba karolo e ka pele karolong e ka morao ya disete tsa netweke ya hao.

## Hobaneng o sebedisa Admin Page Creator? {#why-use-admin-page-creator}

Mohlomong o batla ho fana ka leqephe la moetlo phaneleng ya tsamaiso ya bareki ba hao le nang le dithuto, mohlomong o batla ho eketsa leqephe la moetlo le nang le FAQs le widget ya kgokahano. Menyetla ha e fele ka Admin Page Creator, mme ke seo re tla se hlahloba thutong ya kajeno.

## Ho etsa Leqephe la hao la pele la Tsamaiso la Moetlo {#creating-your-first-custom-admin-page}

Kamora ho kenya le ho kenya tshebetsong Ultimate Multisite: Admin Page Creator, leba ho menu ya tsamaiso ya netweke ya hao mme o ye ho Admin Pages.

![Lenane la Maqephe a Tsamaiso a Moetlo ho Network Admin](/img/admin/admin-pages-list.png)

_Lenane la Maqephe a Tsamaiso a Moetlo ho Network Admin_

O tla bona lenane la maqephe ohle a tsamaiso ao o a entseng ho fihlela jwale (kaha lena ke ketelo ya hao ya pele, lenane le tla be le se na letho). Tswela pele mme o tobetse **Eketsa Leqephe le letjha la Tsamaiso**. O tla fetisetswa ho mohlophisi wa Admin Page.

## Dikgetho tsa Menu {#menu-options}

Ka lehlakoreng, ho meta-box ya General Options, o tla kgona ho hlophisa mofuta wa menu, letshwao, le tse ding tse ngata.

### Mefuta ya Menu {#menu-types}

Leqephe la hao le letjha la tsamaiso la moetlo le ka ba le mefuta e mmedi e fapaneng ya menu: menu ya boemo bo hodimo le submenu.

Leqephe la menu ya boemo bo hodimo, jwalo ka ha lebitso le bontsha, le tla eketsa leqephe leo o sa tswa le etsa ho menu ya tsamaiso mmoho le maqephe a kamehla a WordPress.

Haeba o kgetha kgetho ya boemo bo hodimo, o tla kgona ho kgetha tatellano ya menu le letshwao la menu hape.

Haeba o batla ho etsa leqephe lena leqephe la sub-menu, o tla kgona ho kgetha leqephe la motswadi leo o ka hokelang leqephe lena ho lona. O ka eketsa leqephe la hao la moetlo maqephe a kamehla a WordPress kapa maqephe a hao a tsamaiso a moetlo a boemo bo hodimo.

<!-- Setshwantsho ha se fumanehe: Dikgetho tsa mofuta wa menu tsa Admin Page Creator tse nang le disetting tsa letshwao, tatellano, le icon -->

_Fetola ntho ya menu ka botlalo bakeng sa Leqephe lena la Moetlo_

_O ka boela wa eketsa leqephe lena e le sub-menu ya maqephe a mang a menu, ho kenyeletswa a hao a moetlo a boemo bo hodimo._

## Dikgetho tse Tsoetseng Pele {#advanced-options}

O lokela ho kgona ho eketsa mefuta yohle ya dikahare maqephe a hao a tsamaiso a moetlo. Seo se kenyeletsa melao le difaele tsa CSS, JavaScript, le dilaeborari tsa kantle hape. O ka etsa seo ka ho sebedisa Advanced Options Tab.

### CSS ya Moetlo {#custom-css}

Ho tab ya CSS, o ka eketsa khoutu ya hao ya CSS ya moetlo, hammoho le ho kenya difaele tsa CSS ho tswa diseteng tse hole (jwalo ka CDNs, Google Fonts, jj).

<!-- Setshwantsho ha se fumanehe: Mohlophisi wa CSS ya moetlo wa Admin Page Creator ka ho kenya stylesheet ya kantle -->

_CSS ya moetlo le yona e tshehetswa. O ka boela wa kenyelletsa difaele tsa setaele tsa kantle!_

### JavaScript ya Moetlo {#custom-javascript}

Ho tshwanang ho sebetsa le ho JavaScript. O ka eketsa dilaeborari tsa kantle ho di sebedisa khoutung ya hao ya JavaScript ya moetlo.

<!-- Setshwantsho ha se fumanehe: Mohlophisi wa JavaScript ya moetlo wa Admin Page Creator ka ho kenya laeborari ya kantle -->

_Eketsa khoutu ya hao ya JavaScript!_

## Ditumello {#permissions}

Hape ho Advanced Options, o tla kgona ho kgetha hore na ke disete dife tse manyane tse tla ba le phihlello ho leqephe lena. Sena se molemo haeba o rera ho fana ka dikahare tse ikgethileng tse fapaneng ho maemo a fapaneng a plan, mohlala.

<!-- Setshwantsho ha se fumanehe: Disetting tsa ditumello tsa Admin Page Creator bakeng sa taolo ya phihlello e itshetlehileng hodima plan le karolo -->

_O na le taolo e felletseng hodima moo leqephe lena le tla hlaha teng_

**Bohlokwa** : Maemo ka bobedi a tlameha ho phethahala hore leqephe le bontshwe. Seo se bolela hore haeba mosebedisi e le moreki wa moralo o itseng, empa a se na e nngwe ya mesebetsi e kgethilweng (kapa ka ho fapana) leqephe le ke ke la hlaha phaneleng ya hae.

## E sebetsang {#active}

Ho tima leqephe la boemo bo hodimo ho tla tima ka boiketsetso maqephe ohle a manyane a hoketsweng leqepheng leo la motswadi.
