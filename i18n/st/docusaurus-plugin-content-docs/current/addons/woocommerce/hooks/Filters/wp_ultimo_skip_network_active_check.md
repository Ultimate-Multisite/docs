# Filter: wp_ultimo_skip_network_active_check

E lumella bahlahisi ho short-circuit tlhahlobo ya ho kenya tshebetsong network.

Sena se molemo ha ho sebediswa ditlhophiso tse thehilweng ho composer le tse ding tse ikgethileng, jwalo ka Bedrock, mohlala, moo ho sebedisa plugins e le mu-plugins e leng tlwaelo.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $skip_network_activation_check | `bool` | Hore na re lokela ho tlola tlhahlobo kapa tjhe, ka kamehla ke false. |

### Since

- 2.0.0
### Source

E hlalositswe ho [`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/class-requirements.php#L272) moleng wa 272


## Returns
true haeba o batla ho tlola tlhahlobo, ho seng jwalo false.
