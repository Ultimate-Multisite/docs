# ფილტრი: wu_sso_url

ფილტრავს გენერირებულ SSO URL-ებს, სანამ ისინი დაბრუნდება დომენებს შორის მომხმარებლის ქმედებებისთვის.

გამოიყენეთ ეს ფილტრი, როცა ინტეგრაციას სჭირდება სანდო კონტექსტის დამატება სუვერენული tenant-ის SSO ბმულზე ან broker URL-ის ჩანაცვლება Ultimate Multisite-ის token-ის ვალიდაციის შენარჩუნებით.

## პარამეტრები {#parameters}

| სახელი | ტიპი | აღწერა |
|------|------|-------------|
| $sso_url | `string` | გენერირებული SSO URL. |
| $user | `WP_User` | მომხმარებელი, რომელიც ავთენტიფიცირდება SSO ვიზიტით. |
| $site_id | `int` | სამიზნე საიტის ID ვიზიტისთვის. |
| $redirect_to | `string` | დანიშნულების URL წარმატებული SSO ვალიდაციის შემდეგ. |

### ვერსიიდან {#since}

- 2.13.0

### წყარო {#source}

განსაზღვრულია `inc/sso/class-sso.php`-ში.


## აბრუნებს {#returns}

გაფილტრულ SSO URL-ს.
