# ფილტრი: wu_contains_element {#filter-wucontainselement}

საშუალებას აძლევს დეველოპერებს შეცვალონ საწყისი ძიების შედეგები.

ეს სასარგებლოა მესამე მხარის builder-ებისთვის და მსგავსი შემთხვევებისთვის.

## პარამეტრები {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | თუ ელემენტი კონტენტშია მოთავსებული. |
| $content | `string` | კონტენტი, რომელიც მოწმდება. |
| $element | `self` | მიმდინარე ელემენტი. |
| $post | `null\|\WP_Post` | შესამოწმებელი post. |
