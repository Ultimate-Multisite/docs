# მომხმარებლების მართვა {#managing-customers}

**Customers** გვერდი გვიჩვენებს თქვენს ქსელში დარეგისტრინული წევრების სია. ამ გვერდზე გამოტანილია ძირითადი მომხმარებლის ინფორმაცია, მათ შორის სახელი, ელფოსტის მისამართი, ბოლო შესვლის დრო და **გადართვა**-ის ბმული, რომელიც საშუალებას გაძლევთ შეხვიდეთ მომხმარებლის ქვესაიტზე ადმინისტრატორის სახით.

![Customers list page](/img/admin/customers-list.png)

წარმოადგინეთ **Manage** (მართვა) ღილაკი თითოეული მომხმარებლის ბარათის ქვეშ, რომ ნახოთ ამ მომხმარებლის შესახებ დეტალური ინფორმაცია.

მომხმარებლის დეტალურ გვერდზე, **At a Glance** (მყისიერი მიმოხილვა) ბლოკში ჩანს მომხმარებლის ID, ბოლო შესვლის დრო და ამ მომხმარებლის მიერ გაწეული საერთო თანხა.

**Customer Options** (მომხმარებლის ვარიანტები) ბლოკი საშუალებას გაძლევთ დაასახელოთ ეს მომხმარებელი **VIP**-ად (თეგი, რომელიც მათ უფრო ადვილად პოვნად აქცევს). **Billing Info** (გადახდის ინფორმაცია) ტაბულში ჩანს გადახდის დეტალები, როგორიცაა ელფოსტა, მისამართი და ქვეყანა.

**Fields** (ველები) ბლოკში ჩანს მომხმარებლის მომხმარებლის სახელი, ელფოსტა და საშუალებას გაძლევთ შეცვალოთ **Email Verification** (ელფოსტის ვერიფიკაციის) სტატუსი.

Customer Options ბლოკის ქვემოთ იპოვით კიდევ ოთხ ბლოკს:

- **Memberships** (წევრი თაობა) — მიმდინარე გამოწერები, რომლებსაც მომხმარებელი ატარებს
- **Payments** (გადახდები) — მომხმარებლის გადახდის ისტორია
- **Sites** (საიტები) — საიტები, რომლებიც ამ მომხმარებელს ეკუთვნის
- **Events** (ღონისძიებები) — ღონისძიებები, რომლებიც დაკავშირებულია ამ მომხმარებელთან (ელფოსტის ვერიფიკაცია, გადახდის ცვლილებები და ა.შ.)

ნებისმიერ ელემენტთან (ტაബിനോ ბლოკს) არსებული შუასული (→) ხატულაზე დაწექით, რომ გადახვიდეთ მისი დეტალურ გვერდზე. მაგალითად, საიტის გვერდის შუასულის დაჭერამ გადაგაგიყვანთ საიტის რედაქტირების გვერდზე, სადაც შეგიძლიათ შეცვალოთ წევრი თაობის ასოციაცია, საიტის ტიპი და აქტიური სტატუსი.

ოდესმე იგივე ვრცელდება წევრი თაობებსა და გადახდებზე — თითოეული მათგანი მიუთითებს დეტალურ გვერდზე, სადაც შეგიძლიათ შეცვალოთ წევრი თაობის სტატუსი, დაკავშირებული მომხმარებელი, პროდუქტები, გადახდის გეიტਵੇ (payment gateway) პარამეტრები და 심ilmente, შეგიძლიათ გააკეთოთ თანხის დაბრუნება (refund).

მომხმარებლის გვერდის მარჯვენა მხარეს იპოვით:

* **ბოლო შესვლა და IP მისამართები** — ბოლო შესვლის დრო, IP მისამართი და ქვეყანა
* **დროის შტამპები (Timestamps)** — როდის შეიქმნა მომხმარებლის ანგარიში და ბოლოს როდის შეიცვალა
* **წაშლა** — ვარიანტი მომხმარებლის მუდმივი წაშლისთვის

:::warning
მომხმარებლის წაშლა მუდმივია და აღარ შეიძლება მისი დაბრუნება.
:::
