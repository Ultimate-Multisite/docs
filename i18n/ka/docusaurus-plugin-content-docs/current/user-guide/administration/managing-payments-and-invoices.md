# გადახდების და ინვოისების მართვა

## გადახდის პარამეტრები

გადახდების მიღებას დაწყებამდე საჭიროა გადახდებთან დაკავშირებული პარამეტრების კონფიგურირება. გადადით **Ultimate Multisite → Settings**-ზე და დააწექი **Payment** ტაბულზე.

![Payment settings tab](/img/admin/settings-payments-top.png)

აი, გადახდის პარამეტრების გვერდის სრული ხედვა:

![Payment settings full page](/img/admin/settings-payments-full.png)

### ზოგადი გადახდის ვარიანტები

ზოგად პარამეტრებში შეგიძლია დააკონფიგურირო:

- **ავტორის (Currency)** — ტრანზაქციებისთვის გამოყენებული ნაგულისხმევი ვალუტა
- **ავტორის პოზიცია (Currency Position)** — სადაც ვალუტის სიმბოლო გამოჩნდება (სכრილის წინ/შემდეგ)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### გადახდის გეითვეები (Payment Gateways)

Ultimate Multisite მხარს უჭერს მრავალ გადახდის გეითვს. თითოეული მათგანის ჩართვა და კონფიგურაცია შეგიძლია Payment settings ტაბიდან.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

ხელმისაწვდომი გეითვები მოიცავს:

- **Stripe** — Stripe-ის საშუალებით საკრედიტო ბარათების გადახდები
- **PayPal** — PayPal-ის გადახდები
- **Manual (მექანიკური)** — ოფლაინ ან მორგებული გადახდის დამუშავებისთვის

თითოეულ გეითვს აქვს საკუთარი კონფიგურაციის განყოფილება, სადაც API გასაღებები და სხვა პარამეტრები შედიან.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### Sandbox რეჟიმი (Sandbox Mode)

შეგიძლია ჩართო **Sandbox Mode**, რათა გადახდის ინტეგრაცია რეალურ ცხოვრებაში გაშვებამდე შეამოწმო. როდესაც sandbox mode აქტიურია, არანაირი რეალური დანახარჯები არ მოხდება.

## გადახდების ნახვა

გადადით Ultimate Multisite-ის ქვეშ არსებულ **Payments** გვერდზე, რომ ნახო ყველა ტრანზაქცია შენს ქსელში.

![Payments list](/img/admin/payments-list.png)

შეგიძლია გადახდები ფილტრაცია გააკეთო სტატუსის მიხედვით (დასრულებული, დაუგეგმავი, წარუმატებელი, დაბრუნებული) და კონკრეტული ტრანზაქციების ძებნა.

გადადით გადახდაზე, რომ ნახოთ მისი სრული დეტალები, მათ შორის ხარჯი დანამატები, დაკავშირებული წევრობა, მომხმარებლის ინფორმაცია და გადახდის გეითვების მონაცემები.

## ინვოისები (Invoices)

Ultimate Multisite-ს შეუძლია ავტომატურად შექმნას ინვოისები გადახდებისთვის. თქვენ შეგიძლიათ ჩამოტვირთოთ ინვოისის შაბლონი და ნომრის ფორმატი პარამეტრები Payment-ში.

ინვოისის მორგების ვარიანტები მოიცავს:

- **კომპანიის სახელი და მისამართი**, რომელიც გამოჩნდება ინვოისებზე
- **ინვოისის ნომრის** ფორმატი და თანმიმდევრობა
- **ლოგოს**, რომელიც გამოჩნდება ინვოისის სათაურზე
- **კონტაქტის ტექსტის (ფუტერის)** მორგება პირობებისთვის, შენიშვნებისთვის ან სამართლებრივი ინფორმაციისთვის

ინვოისის შაბლონის დასამორგებლად გადადით **Ultimate Multisite → Settings → Payment**-ზე და იპოვეთ ინვოისთან დაკავშირებული პარამეტრები.
