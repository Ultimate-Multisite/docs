# კონტაქტის ფორმების შექმნა საიტის აწყობის საშუალებით

Superdav AI Agent v1.10.0-ს აქვს შესაძლებლობა, რომ პირდაპირ ჩატის ინტერფეისიდან Site Builder აგენტს გამოყენებით შექმნათ კონტაქტის ფორმები. ეს საშუალებას გაძლევთ დაამატოთ სრულად ფუნქციონალური კონტაქტის ფორმები ნებისმიერ გვერდზე, senza გამოსვლის ჩატისგან.

## მიმოხილვა {#overview}

Site Builder-ის კონტაქტის ფორმის შექმნის შესაძლებლობა (`create_contact_form`) ავტომატურად აღმოაჩენს თქვენს დაინსტალირებულ ფორმ პლაგინს და იყენებს ამ პლაგინის ნატურალურ შესაძლებლობებს კონტაქტის ფორმის შესაქმნელად. მხარდაჭერილ ფორმ პლაგინებში შედის:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

ასისტენტი ავტომატურად აირჩევს თქვენს საიტზე არსებულ საუკეთესო ხელმისაწვდომ ფორმ პლაგინს და გენერირებს კონტაქტის ფორმას, რომელიც ამ პლაგინს შეესაბამება.

## კონტაქტის ფორმის შექმნა {#creating-a-contact-form}

Site Builder-ის გამოყენებით კონტაქტის ფორმის შესაქმნელად:

1. გახსენით **Gratis AI Agent** ჩატის პანელი WordPress ადმინისტრაციაში.
2. გადადით **Site Builder** აგენტზე, დააჭირეთ ჩატის ჰედერის მენიუს ეკრანზე არსებულ აგენტის ხატულას.
3. აღწერეთ კონტაქტის ფორმა, რომლის შექმნას გსურთ. მაგალითად:

   > "დაამატე კონტაქტის ფორმა Contact გვერდზე სახელი, ელფოსტის, შეტყობინების და ტელეფონის ნომრის ველით."

   ან უბრალოდ:

   > "შექმენი კონტაქტის ფორმა Contact გვერდისთვის."

4. Site Builder გენერირებს კონტაქტის ფორმას და აბრუნებს შორტკოდს (shortcode), რომელიც მზად არის ჩასატვირთად.

## გენერირებული შორტკოდის გამოყენება {#using-the-generated-shortcode}

Site Builder-მა კონტაქტის ფორმის შექმნის შემდეგ, ის აბრუნებს შორტკოდს (მაგალითად: `[contact-form-7 id="123"]`). თქვენ შეგიძლიათ:

1. **ჩასვათ გვერდზე ან პოსტში** — დააკოპირეთ შორტკოდი და ჩასვით ნებისმიერ გვერდზე ან პოსტში ბლოკ-რედაქტორის (block editor) ან კლასიკური რედაქტორის გამოყენებით.
2. **დაამატოთ ის Site Builder-ის მეშვეობით** — სთხოვეთ Site Builder-ს, რომ ფორმა ავტომატურად დაამატოს კონკრეტულ გვერდზე:

   > "დაამატე კონტაქტის ფორმა Contact გვერდზე."

3. **გამოიყენეთ 템პლეითში** — თუ PHP-ს დახვეწილი ხართ, შეგიძლიათ შორტკოდი თემის 템플릿 ფაილში დაამატოთ.

## კონტაქტის ფორმის მორგება {#customizing-the-contact-form}

საიტის აწყობის შემდეგ, როდესაც Site Builder-მა შექმნა კონტაქტის ფორმა, შეგიძლიათ მას further (უფრო) დაამორექტირდეთ:

### ჩატის ინტერფეისიდან {#via-the-chat-interface}

בקשו Site Builder-ს ფორმის შეცვლას:

> "განახლეთ კონტაქტის ფორმა, რომ დაემატოს საგნის ველი და გახადოს შეტყობინების ველი აუცილებელი."

Site Builder განაახლებს ფორმას და დაუბრუნდება განახლებულ შორტკოდს.

### ფორმის პლაგინის ადმინისტრაციული ინტერფეისიდან {#via-the-form-plugins-admin-interface}

შეგიძლიათ ფორმა პირდაპირ თქვენი ფორმის პლაგინის პარამეტრებშიც შეცვალოთ:

1. გადადით WordPress-ის ადმინში **Contact Form 7**-ზე (ან თქვენს დანერგილ ფორმის პლაგინზე).
2. იპოვეთ Site Builder-მა შექმნილი ფორმა.
3. დაამატეთ/შეცვალეთ ფორმის ველი, ვალიდაციის წესები და ელფოსტის შეტყობინებები საჭიროებისამებრ.

## ფორმის პლაგინები და თავსებადობა {#form-plugins-and-compatibility}

Site Builder ავტომატურად აღpoznებს, თუ რომელი ფორმის პლაგინი არის დანერგილი თქვენს საიტზე და იყენებს მას კონტაქტის ფორმის შესაქმნელად. თუ რამდენიმე ფორმის პლაგინი დანერგილია, Site Builder მათ ამ თანმიმდევრობით უპირატესობას ანიჭებს:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

თუ ეს პლაგინები არ არის დანერგილი, Site Builder გირჩევთ, რომ ერთი მათგანი დანერგოთ კონტაქტის ფორმის შექმნის წინ.

## ელფოსტის შეტყობინებები {#email-notifications}

Site Builder-ის მიერ შექმნილი კონტაქტის ფორმები ნაგულისხმევად არის დაყენებული იმისთვის, რომ ელფოსტის შეტყობინებები გაგზავნოს საიტის ადმინისტრატორს. შეგიძლიათ შეცვალოთ მიმღების ელფოსტის მისამართი და შეტყობინების ტექსტი:

1. გადადით თქვენი ფორმის პლაგინის ადმინისტრაციულ ინტერფეისში.
2. იპოვეთ Site Builder-მა შექმნილი ფორმა.
3. დაამატეთ/შეცვალეთ ელფოსტის შეტყობინების პარამეტრები.

დეტალური ინსტრუქციებისთვის, გადადით თქვენი ფორმის პლაგინის დოკუმენტაციაში:

- [Email Notifications for Contact Form 7](https://contactform7.com/docs/)
- [Email Notifications for WPForms](https://wpforms.com/docs/)
- [Email Notifications for Fluent Forms](https://fluentforms.com/docs/)
- [Email Notifications for Gravity Forms](https://docs.gravityforms.com/)

## მაგალითები გამოყენებისთვის (Example Use Cases) {#example-use-cases}

### მარტივი საკონტაქტო ფორმა (Simple Contact Form) {#simple-contact-form}

> "შექმენით მარტივი საკონტაქტო ფორმა სახელის, ელფოსტის და შეტყობინების ველით."

### მრავალეტაპიანი ფორმა (Multi-Step Form) {#multi-step-form}

> "შექმენით საკონტაქტო ფორმა, რომელსაც აქვს პირველი ნაბიჯი საკონტაქტო ინფორმაციისთვის და მეორე ნაბიჯი შეტყობინებისთვის და სასურველი საკონტაქტო მეთოდისთვის."

### ფორმა პირობითი ლოგიკით (Form with Conditional Logic) {#form-with-conditional-logic}

> "შექმენით საკონტაქტო ფორმა, რომელიც აჩვენებს სხვადასხვა ველი მომხმარებლის არჩევანის მიხედვით ჩამDesc-დან (dropdown)."

### ფაილის ატვირთვა მქონე ფორმა (Form with File Upload) {#form-with-file-upload}

> "შექმენით საკონტაქტო ფორმა, რომელიც საშუალებას მისცემს მომხმარებლებს ატვირთონ ფაილი ან დანართი."

:::note
საკონტაქტო ფორმის შექმნა შესაძლებელია Superdav AI Agent v1.10.0-ისა და უფრო ახალი ვერსიების გამოყენებით. ამ შესაძლებლობისთვის საჭიროა Site Builder აგენტი (agent) აქტიური იყოს.
:::
