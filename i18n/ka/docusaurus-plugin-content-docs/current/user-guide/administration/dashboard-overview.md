# დეშბორდის მიმოხილვა {#dashboard-overview}

Ultimate Multisite-ს აქვს სპეციალური **Dashboard** (დეშბორდი) Ultimate Multisite მენიუს ქვეშ, ასევე ამატებს ვიჯეტებს Network Dashboard-ზე. ეს სტატია განიხილავს, თუ რა დაინახავთ ორივე ადგილას.

## ქსელური დეშბორდი (The Network Dashboard) {#the-network-dashboard}

Ultimate Multisite ინექცირებს სასარგებლო ვიჯეტებს ქსელის დონის დეშბორდში, რომელიც შეგიძლიათ იპოვოთ **My Sites → Network Admin → Dashboard**-ზე.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

აი, რა გთავაზობთ თითოეული ვიჯეტის:

**პირველი ნაბიჯები (First Steps):** ჩნდება ახალი ინსტალაციისას და გიხსნით დავალებებს, როგორიცაა გადახდის მეთოდის კონფიგურირება და ტესტის ანგარიშის შექმნა.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**შეჯამება (Summary):** აცხადებს დღის სალარმო მომხმარებლებისა და შემოსავლების რაოდენობას. MRR (Monthly Recurring Revenue - ყოველთვიური განმეორებადი შემოსავალი) მიუთითებს პროექტირებულ საერთო თანხაზე მომხმარებლებზე, რომლებსაც შემოსავლიანი წევრობა აქვთ.

![Summary widget](/img/admin/network-dashboard-summary.png)

**აქტივობის ნაკადი (Activity Stream):** აცხადებს მოვლენებს Ultimate Multisite-ში — ჩანაწერებს, გაუქმებებს, საიტების შექმნას, წაშლას და სხვა აქტივობებს.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**ამჟამინდელი (Right Now):** აჩვენებს ქსელში მომხმარებლებისა და საიტების მოკლე მიმოხილვას, ახალი საიტებისა თუ მომხმარებლების შესაქმნელად აქვს შტუტ-შორებები.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**ახალი ამბები და დისკუსიები (News & Discussions):** იღებს და აჩვენებს Ultimate Multisite-ის უახლეს სიახლეებს, მათ შორის განახლებებს, კრიტიკულ ბაგებსა და უსაფრთხოების გამოსწორებებს.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite დეშბორდი (The Ultimate Multisite Dashboard) {#the-ultimate-multisite-dashboard}

მიუხედავად იმისა, რომ ქსელური დეშბორდი ქსელის შესახებ ყოვლისმომცველ ინფორმაციას წარმოადგენს, Ultimate Multisite დეშბორდი (Ultimate Multisite მენიუს ზედა დონეზე) წარმოადგენს სერვისის შესახებ ინფორმაციას.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**ყოველთვიური განმეორებადი შემოსავლის ზრდა (Monthly Recurring Revenue Growth):** ეს გვიჩვენებს შემოსავლის ყოველთვიურ მიმოხილვას 12 თვის კალენდარული პერიოდის განმავლობაში. ახალი შემოსავალი ნBlues-შია და გაუქმებების შემთხვევაში წითლად ჩანს.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**თარიღის დიაპაზონის მხარდაჭერა (Date-range Support):** შეგიძლიათ ფილტრაცია გააკეთოთ დაფის მონაცემებზე დროის პერიოდით, რათა ყურადღება მიექცეს იმ ინფორმაციას, რომელიც თქვენ გჭირდებათ. თარიღის დიაპაზონი განსაზღვრავს, რა იქნება ნაჩვენები ქვემოთ მოცემული ვიჯეტებში.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

დაფის ვიჯეტებში შედის:

- **შემოსავალი (Revenue)** — პერიოდის საერთო შემოსავალი MRR-ით და პროდუქტის მიხედვით დეტალური გადანაწილება
- **ახალი წევრობები (New Memberships)** — რამდენად ბევრი ახალი წევრი შეიქმნა არჩეულ პერიოდში
- **წევრი isc. ქვეყნების მიხედვით (Signups by Countries)** — რეგისტრაციების გეოგრაფიული მიმოხილვა
- **რეგისტრაცია ფორმების მიხედვით (Signups by Forms)** — რამდენად ბევრი მომხმარებელი დარეგისტრირდა თითოეულ გადახდის ფორმაზე და რამდენს დაამატეს ადმინისტრატორებმა
- **ყველაზე მეტად მო방 visit-ის საიტები (Most Visited Sites)** — თქვენი ქსელის ყველაზე ხშირად訪ლით საიტები (საჭიროა ჩართოთ მონახულება ვიზიტების დათვლის ფუნქცია: **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)
