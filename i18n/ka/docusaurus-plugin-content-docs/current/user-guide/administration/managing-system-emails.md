# სისტემური ელფოსტლების მართვა

Ultimate Multisite სხვადასხვა სისტემური ელფოსტლებს იგზავნის ღონისძიებებისთვის, როგორიცაა დარეგისტრირების დადასტურება, გადახდის ქვითარი და წევრობის ცვლილებები. თქვენ შეგიძლიათ ეს ელფოსტის შაბლონები შეცვალოთ საკუთარი ბრენდის შესაბამისად.

## სისტემური ელფოსტლების ნახვა

გადადით **Ultimate Multisite → Settings**-ზე და დააწკაპუნეთ **Emails** ტაბზე, რომ ნახოთ ელფოსტით დაკავშირებული პარამეტრები.

![Settings - Emails tab](/img/admin/settings-emails.png)

იქდან შეგიძლიათ მიწვდეთ **System Emails** გვერდს და ნახოთ ყველა არსებული შაბლონი.

![System Emails list](/img/admin/system-emails-list.png)

თქვენ ასევე შეგიძლიათ შეცვალოთ ინდივიდუალური ელფოსტის შაბლონები სისტემური ელფოსტის რედაქტორის გამოყენებით:

![System email editor](/img/admin/system-email-editor.png)

## ახალი ელფოსტის შაბლონების იმპორტი

Ultimate Multisite-ის ახალი ვერსიები და add-ons შეიძლება ახალ ელფოსტის შაბლონებსაც შექმნან. კონფლიქტების თავიდან ასაცილებლად, ახალი შაბლონები ავტომატურად არ იმატებს — თქვენ მათ ხელით უნდა იმპორტიროთ.

ახალი ელფოსტის შაბლონების იმპორტისთვის:

1. გადადით **System Emails** გვერდზე
2. დააწკაპუნეთ გვერდის ზედა ნაწილში არსებულ **Reset & Import** ღილაკზე
3. ჩართეთ **Import Emails** ოფცია, რომ ნახოთ ხელმისაწვდომი შაბლონები
4. აირჩიეთ ის შაბლონები, რომელთა იმპორტი გსურთ და დაადასტურეთ

## ელფოსტის შაბლონების გადატვირთვა (Reset)

თუ თქვენ შეცვალეთ ელფოსტის შაბლონი და გსურთ დაბრუნდეთ დეფოლტულ კონტენტზე, გაქვთ ორი ვარიანტი:

1. **წაშლა და ხელახალი იმპორტი** — წაშალეთ სისტემური ელფოსტა და ისევ იმპორტიруეთ (ეს წაშლის გაგზავნის მეტრიკებს)
2. **გადატვირთვა ინსტრუმენტის საშუალებით** — გამოიყენეთ **Reset & Import** ინსტრument, ჩართეთ **Reset** ოფცია და აირჩიეთ ის ელფოსტები, რომელთა გადატვირთვა გსურთ

Reset-ის ვარიანტი სასურველია, რადგან ის ინარჩუნებს გაგზავნის მეტრიკებს და სხვა მეტამონაცემებს, ხოლო დეფოლტული კონტენტის აღდგენას უწყობს ხელს.
