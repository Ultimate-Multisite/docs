# წევრობის მართვა

Ultimate Multisite-ზე თქვენ შეგიძლიათ მართოთ წევრობები და მიიღოთ доступ თქვენი ქსელის ყველა გამოწერა. შეამოწმეთ და გადახედეთ გამოწერის დეტალებს, როგორიცაა გეგმები და პროდუქტები, საიტების რაოდენობა, ბილინგის თანხა და სიხშირე, გადახდის ისტორია და დროის შტამპები.

ამ გვერდზე მისასვლელად, გადადით Ultimate Multisite-ის მენიუს **Memberships** (წევრობები) ჩანართზე.

![Memberships list page](/img/admin/memberships-list.png)

Memberships გვერდზე თქვენ დაინახავთ თქვენი მომხმარებლების სია და მათ ანგარიშებთან დაკავშირებული პროდუქტები. შეგიძლიათ ფილტრაცია გააკეთოთ წევრობების სტატუსის მიხედვით: აქტიური, მოსალოდნელი (pending), შეჩერებული (on hold), ვადის გასული (expired) ან გაუქმებული (cancelled).

უფასო წევრობები განიხილება როგორც **ხელმოწერის ხანგრძლივობის შეუზღუდავი** (lifetime) წევრობები. მათ არ აქვთ ვადის გასვლის თარიღი და ავტომატურად არ იწურებიან. თუ უფასო გეგმა უნდა დასრულდეს ტესტის ან ფიქსირებული პერიოდის შემდეგ, გამოიყენეთ გადახდილი პროდუქტი ტესტთან ერთად, გააუქმეთ წევრობა ხელით, ან მომხმარებელი სხვა გეგმაზე გადაიტანეთ, როდესაც მისი доступ შეიცვლება.

წევრობაზე დაწკაპუნებით მიიღებთ გვერდს, სადაც შეგიძლიათ შეცვალოთ წევრობის დეტალები — ბილინგის ინფორმაცია, საიტების ან ვიზიტების ლიმიტები და სხვა.

![Membership edit interface](/img/admin/membership-edit.png)

აი, წევრობის რედაქტირების სრული ხედვა:

![Membership edit full interface](/img/admin/membership-edit-full.png)

თქვენ ასევე შეგიძლიათ დაამატოთ ახალი წევრობები მომხმარებლებთან, რომლებიც ადრე თქვენს ქსელში დარეგისტრირდნენ. დააჭირეთ **Add Membership** (წევრობის დამატება) და დააკონფიგურირეთ დეტალები.

![Add Membership modal](/img/admin/membership-add-modal.png)

შემდეგ დააჭირეთ **Create Membership** (წევრობის შექმნა). ახალი წევრობა გამოჩნდება წევრობების სიაში.
