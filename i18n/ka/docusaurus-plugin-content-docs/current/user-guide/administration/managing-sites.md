# საიტების მართვა {#managing-sites}

საიტები (ან სუბსაიტები) თქვენი WaaS ბიზნესის ძირითადი ნაწილია. Ultimate Multisite-ს აქვს 3 ტიპის საიტი:

- **კლიენტის საკუთრებაში** — კონკრეტული მომხმარებლებისთვის განკუთვნილი საიტები
- **საიტის შაბლონები (Site Templates)** — წინასწარ განსაზღვრული საიტები, რომელთა გამოყენებაც მომხმარებელს შეუძლია როგორც დასაწყისი
- **მთავარი საიტი (Main Site)** — თქვენი პრიმიდარული ქსელი საიტი

## საიტების ნახვა {#viewing-sites}

ნახეთ **Ultimate Multisite → Sites** და ნახავთ თქვენი ქსელის ყველა სუბსაიტს. თითოეული საიტი აღინიშნება ტიპით და შეგიძლიათ ფილტრაცია: ყველა საიტი, კლიენტის საკუთრებაში, შაბლონები ან დაუყოვნებლივ დამუშავებელი (Pending).

![Sites list page](/img/admin/sites-list.png)

## ახალი საიტის დამატება {#adding-a-new-site}

დააჭირეთ ღილაკს **Add Site** ახალი საიტის შესაქმნელად. თქვენ დაგჭირდებათ მიაწოდოთ:

- **საიტის სახელი (Site Title)** — ახალი საიტის დასახელება
- **საიტის URL (Site URL)** — დომენი/პათი საიტისთვის
- **საიტის ტიპი (Site Type)** — არის თუ არა ეს კლიენტის საიტი, შაბლონი თუ ჩვეულებრივი საიტი

**Copy Site** ოფცია საშუალებას გაძლევთ შექმნათ ახალი საიტი არსებული საიტის შაბლონის საფუძველზე. ჩართვისას შეგიძლიათ აირჩიოთ, რომელ შაბლონს გამოიყენებთ დასაწყებად. დარწმუნდით, რომ **Copy Media on Duplication** არის ჩართული, რათა მედია ფაილებიც შევიდეს.

## არსებული საიტის მართვა {#managing-an-existing-site}

დააჭირეთ **Manage** ნებისმიერ საიტზე, რათა გაიხსნას **Edit Site** გვერდი. აქ თქვენ იპოვით:

### ძირითადი ინფორმაცია (Basic Information) {#basic-information}

საიტის სახელი, ტიპი, ID და აღწერა. ასევე ნახავთ მასთან დაკავშირებულ დომენს, წევრობას და იმ მომხმარებლის ანგარიშს, რომელიც საიტს ეკუთვნის.

### საიტის პარამეტრები (Site Options) {#site-options}

დააკონფიგურირეთ საიტის შესაძლებლობები და ლიმიტები:

- **ვიზიტების ლიმიტი (Visit limits)** — მაქსიმალური რაოდენობა საიტზე ვიზიტებისთვის
- **მომხმარებლის ანგარიშის ლიმიტები (User account limits)** — ლიმიტები მომხმარებლის როლზე
- **დისკის სივრცე (Disk space)** — მეხსიერების割り当て ( quota) საიტისთვის
- **კონკრეტული დომენი (Custom domain)** — დომენის მაცნობა ამ საიტისთვის
- **პლაგინებისა და თემების ხილვადობა (Plugin and theme visibility)** — კონტროლი იმ პლაგინებისა და თემების მიმართ, რომლებიც ჩანს, დამალულია ან წინასწარ აქტიურია

בברירת מחדל, אתרים פועלים לפי המגבלות שהוגדרו ברמת החברות (membership level). הגדרת מגבלות לרמת האתר תעדיף את הגדרות החברות.

### קישורים (Associations) {#associations}

מתחת לאפשרויות האתר תמצא מידע על:

- **דומיינים ממופים** המשויכים לאתר
- **חברות** שאליה שייך האתר
- **חשבון לקוח** המקושר לאתר

### סרגל צד ימין (Right Sidebar) {#right-sidebar}

בצד ימין תוכל:

- **להפעיל/לכבות את האתר** באמצעות כפתור מתג
- **לשנות את סוג האתר** או להקצות מחדש בעלות (ownership)
- **להגדיר תמונה/תמונת קישור לאתר** (המוצגת בצד הקדמי עבור תבניות אתרים)
- **למחוק את האתר** לצמיתות

:::warning
מחיקת אתר אינה ניתנת להחלפה. האתר וכל התוכן שלו יימחקו לצמיתות.
:::
