# თქვენი ადმინ პანელის მორგება PRO თემით {#customizing-your-admin-panel-with-pro-theme}

ხშირად კლიენტებთან მუშაობისას გვჭირდება, რომ მათ WordPress-ის ადმინ პანელში საკუთარი გამოცდილება დავუწე. თუ უფრო პროფესიონალურ სახეს და განწყობას ეძებთ, PRO Theme შეიძლება დაგეხმაროთ.

PRO Theme არის ადმინ თემა, რომელიც შექმნილია თქვენს კლიენტების პანელის მორგებისთვის. ის ასევე Ultimate Multisite-ის პრემიუმ პლაგინების ძირითადი ნაწილია.

და cài đặtების შემდეგ, გადადით თქვენი _Plugins_ გვერდზე და აქ დააჭირეთ ჩართვის ღილაკს თქვენს ქსელში.

<!-- სურათი ხელმისაწვდომია: WP Plugins-ის გვერდი PRO Theme-ისთვის ქსელის ჩართვის ღილაკს აჩვენებს -->

_გაუშვით ის მთელი ქსელის დონეზე Plugin-ის გვერდზე_

_**შენიშვნა:**_ _თუ უბრალოდ დაინსტალირებთ, ის თქვენი კლიენტებისთვის გამოჩნდება როგორც ვარიანტი, რომლის დამატება მხოლოდ ამ მომხმარებლისთვის შეიძლება იყოს მონიშნული._

ახლა თქვენ გაქვთ ადმინ ინტერფეისი სრულად გადაკეთებული! მარცხენა სვეტში ახალი მენიუც გექნებათ: PRO Theme.
