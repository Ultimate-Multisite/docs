# როგორ მივიღოთ დახმარება

თქვენ გაქვთ თქვენი ვალიდური ლიცენზიის გასაღები, და ამას თან ახლავს წვდომა ჩვენს გუნდთან, რომელიც დაგეხმარება პრობლემების მოგვარებაში, რომლებსაც შეიძლება შეგხვდეთ Ultimate Multisite-ის ინსტალაციის დროს.

ეს გაკვეთილი მიმოიხილავს ლიცენზიის მფლობელებისთვის ხელმისაწვდომ დახმარების არხებს და იმ ინფორმაციას, რაც უნდა გქონდეთ ხელთ, რომ დაუკავშირდეთ ჩვენს მხარდაჭერის გუნდს.

## დახმარების საიტი (Help Site) {#help-site}

თქვენ შეგიძლიათ შეხვიდეთ სხვადასხვა სტატიებში და ტუტორიალებში ჩვენს დახმარების საიტზე, რომელიც შეგიძლიათ იპოვოთ ქვემოთ მოცემულ ბმულში.

[)

ჩვენ ვუწოდებთ მომხმარებლებს, რომ გაეცნონ Ultimate Multisite-ის ძირითად ფუნქციებსა და შესაძლებლობებს. ჩვენს დახმარების საიტზე არსებობს "Getting Started" (დაწყებისთვის) სტატიები და ხშირად დასმული კითხვებზე პასუხებიც იქ შეგიძლიათ იპოვოთ, მაგალითად, როგორ დავინსტალიროთ add-ons, როგორ გავაერთიანოთ გადახდის გეიტვეი (payment gateways) და ა.შ.

## მხარდაჭერის დახმარება (Contact Support) {#contact-support}

თუ გაგიჭირდება და ვერ პოულობთ პასუხს ჩვენს დახმარების საიტზე, შეგიძლიათ მიმართოთ ჩვენს მხარდაჭერის გუნდს ან დაწეროთ ელფოსტა პირდაპირ support@ultimatemultisite.com-ზე.

როდესაც ტექნიკური პრობლემების შესახებ იხმართ მხარდაჭერას, დარწმუნდით, რომ დეტალური აღწერა გაქვთ თქვენი პრობლემის შესახებ და თან გყავთ სკრინშოტი, შეცდომის Meldings (error messages) ან ნებისმიერი ინფორმაცია, რომელიც ჩვენს ადამიანებს დაგეხმარება პრობლემის მარტივად გაგებაში და შეფასებაში. ასევე, თქვენ უნდა წარმოადგინოთ თქვენი ლიცენზიის გასაღები. მხოლოდ აქტიური ლიცენზიის მფლობელებს ექნებათ მხარდაჭერა.

## საზოგადოების საიტი (Community Site) {#community-site}

ჩვენს [community site](https://community.ultimatemultisite.com/) არის ადგილი Ultimate Multisite მომხმარებლებისთვის ერთმანეთთან ურთიერთობისთვის. ეს არის ინფორმაციის სისტემა, სადაც წევრებს შეუძლიათ პოსტები დაწერონ, დისკუსიებზე კომენტარები დაწერონ, რჩევები მისცენ, თანამშრომლობდნენ ან კოლექტიური ძალის გამოყენებით (crowdsource) დახმარებას. თქვენ მოგიწევთ Ultimate Multisite ლიცენზიის გასაღების წარდგენას პერსონალური სივრცეებისა და გვერდების გასახსნელად.
