# ავტომატური ძებნა და შეცვლა საიტის შაბლონებში (v2)

_**ეს ტუტორიალი მოითხოვს Ultimate Multisite-ის ვერსიას 2.x.**_

Ultimate Multisite-ის ერთ-ერთი ყველაზე ძლიერი ფუნქცია არის უნარი, რომ დაამატოთ ნებისმიერი ტექსტი, ფერი და შეტანილ ველები რეგისტრაციის ფორმას. ამ მონაცემების შეგროვების შემდეგ, ჩვენ შეგვიძლია გამოვიყენოთ ისინი იმ საიტის შაბლონის გარკვეული ნაწილების წინასწარი შევსებისთვის. შემდეგ კი, როდესაც ახალი საიტი გამოვაქვეყნებთ, Ultimate Multisite-მა ადგილები (placeholders) ჩანაცვლებს რეალური ინფორმაციით, რომელიც რეგისტრაციის დროს შეიყვანეს.

მაგალითად, თქვენ შეგიძლიათ შექმნათ შაბლონური საიტები ადგილების გამოყენებით. ადგილები უნდა დაემატოს ორმაგი ფრჩხილებით - {{placeholder_name}}.

შემდეგ შეგიძლიათ უბრალოდ დაამატოთ შესაბამისი რეგისტრაციის ველი ამ მონაცემების აღებისთვის.

თქვენი მომხმარებელი შემდეგ შეძლებს ამ ველის შევსებას რეგისტრაციის დროს.

Ultimate Multisite-მა შემდეგ ავტომატურად ჩანაცვლებს ადგილებს მომხმარებლის მიერ მოწოდებული მონაცემებით.

## **"შაბლონი ადგილებით სავსე" პრობლემის გადაჭრა** {#solving-the-template-full-of-placeholders-problem}

ეს ყველაფერი კარგია, მაგრამ ჩვენ ვხვდებით უხერხულ პრობლემას: ახლა ჩვენს საიტის შაბლონებში - რომელზეც მომხმარებლებმა შეიძლება მოvisit- ਕਰਨ - სავსეა უხერხული ადგილები, რომლებიც ბევრ რამეს არ გვიტყვის.

ამის გადასაჭრელად, ჩვენ გთავაზობთ ვარიანტს, რომ დავამატოთ ადგილებისთვის ყალბი მნიშვნელობები (fake values), და ამ მნიშვნელობებს ვიყენებთ შაბლონების საიტებზე ძებნისა და შეცვლისთვის იმ დროსაც, როდესაც თქვენი მომხმარებლები მათ ეწვიExtras.

თქვენ შეგიძლიათ მიწვდეთ შაბლონის ადგილების რედაქტორს გადასვლით **Ultimate Multisite > Settings > Sites**-ზე, სლაიდით Site Template Options பகுதிக்கு და შემდეგ დაჭერით **Edit Placeholders** ბმულზე.

![Site Template Options area in the Sites settings page](/img/config/settings-sites-templates-section.png)

ეს გადაგაგიყვანთ ადგილების კონტენტის რედაქტორში, სადაც შეგიძლიათ დაამატოთ ადგილები და მათ შესაბამისი შინაარსი.
