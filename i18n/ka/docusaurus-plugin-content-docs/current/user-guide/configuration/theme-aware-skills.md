# თემაზე ორიენტირებული უნარები (Theme-Aware Skills)

Superdav AI Agent v1.10.0-ს ახლავს ოთხი ახალი, თემაზე ორიენტირებული ჩაშენებული უნარი, რომლებიც ავტომატურად ერწყმის თქვენს აქტიურ WordPress თემას. ეს უნარები სპეციალიზებულ მითითებებსა და შესაძლებლობებს გთავაზობთ, რომლებიც შეესაბამება თქვენი თემის არქიტექტურასა და ფუნქციონალს.

## რა არის თემაზე ორიენტირებული უნარები?

თემაზე ორიენტირებული უნარები არის წინასწარ კონფიგურირებული ცოდნის ბაზები და ინსტრუმენტების ნაკრებები, რომლებსაც AI ასისტენტი ავტომატურად ირჩევს თქვენს საიტზე ამჟამინდელ აქტიურ WordPress თემაზე დაყრდნობით. როდესაც თემას შეცვლით, ასისტენტის ხელმისაწვდომი უნარები ავტომატურად განახლდება — არ არის საჭირო ხელით კონფიგურირება.

თითოეულ უნარს აქვს:

- **თემა-სპეციფიკური დოკუმენტაცია** — მითითებები თემის გამოყენებისა და მორგებისთვის
- **ბლოკებისა და პატერნების მითითებები** — ხელმისაწვდომი ბლოკები, პატერნები და დიზაინის ვარიანტები
- **მორგების მაგალითები** — კოდის ნაჭრები და კონფიგურაციის შაბლონები საერთო ამოცანებისთვის
- **საუკეთესო პრაქტიკა (Best practices)** — რეკომენდაციები თემის არქიტექტურისა და სამუშაო პროცესისთვის

## ხელმისაწვდომი თემაზე ორიენტირებული უნარები

### ბლოკ თემები (Block Themes)

**გამოიყენება:** თემებისთვის, რომლებიც იყენებენ WordPress-ის ბლოკებზე დაფუძნებულ არქიტექტურას (Full Site Editing).

Block Themes უნარი გთავაზობთ მითითებებს:

- ტემპლატების შექმნა და რედაქტირება ბლოკ-რედაქტორის გამოყენებით
- ბლოკ პატერნებთან და ხელახლა გამოყენებად ბლოკებთან მუშაობა
- გლობალური სტილებისა და theme.json პარამეტრების მორგება
- თემა-ბლოკებისა და ვარიაციების გამოყენება
- თქვენი საიტისთვის საკუთარი ბლოკ პატერნების შექმნა

**ავტომატურად ირთვება, როდესაც:** თქვენი აქტიური თემა არის ბლოკ თემა (მხარს უჭერს `block-templates` ფუნქციას).

### კლასიკური თემები (Classic Themes)

**გამოიყენება:** ტრადიციული WordPress თემებისთვის, რომლებიც იყენებენ PHP ტემპლატებს და კლასიკურ რედაქტორს.

Classic Themes უნარი გთავაზობთ მითითებებს:

PHP შაბლონებთან და ჰუკებთან მუშაობა
თემის გარეგნობის მორგება Customizer-ის საშუალებით
ვი젯ების ზონებისა და საპირისპირო ზოლების გამოყენება
CSS-ის შეცვლა და child theme-ის განვითარება
თემის რიგის (hierarchy) და შაბლონის ტეგების გაგება

**ავტომატურად ირთვება, როდესაც:** თქვენს აქტიურ თემას კლასიკური (non-block) თემა აქვს.

### Kadence Blocks

**適用დება:** იმ საიტებისთვის, რომლებიც იყენებენ Kadence Blocks პლაგინს მოწინავე ბლოკებზე დაფუძნებულ დიზაინისთვის.

Kadence Blocks უნარი გთავაზობთ მითითებებს შემდეგ საკითხებზე:

- Kadence-ის მოწინავე ბლოკების ბიბლიოთეკის გამოყენება (Hero, Testimonials, Pricing და ა.შ.)
- Kadence ბლოკების პარამეტრების და Responsive ვარიანტების კონფიგურირება
- Kadence-ის Grid და Container ბლოკებით მორგებული განლაგებების შექმნა
- Kadence ბლოკების თქვენი თემისთან ინტეგრირება
- Kadence-ის დიზაინის სისტემისა და პრესეტების გამოყენება

**ავტომატურად ირთვება, როდესაც:** Kadence Blocks პლაგინი აქტიურია თქვენს საიტზე.

### Kadence Theme

**適用დება:** იმ საიტებისთვის, რომლებიც იყენებენ Kadence თემას ბლოკებზე დაფუძნებულ დიზაინსა და მორგებას.

Kadence Theme უნარი გთავაზობთ მითითებებს შემდეგ საკითხებზე:

- Kadence თემის მორგება გლობალური სტილებით და theme.json-ით
- Kadence-ის ჩაშენებული ბლოკების პატერნებისა და შაბლონების გამოყენება
- Kadence თემის პარამეტრებისა და ვარიანტების კონფიგურირება
- Kadence-ის დიზაინ სისტემით მორგებული განლაგებების შექმნა
- Kadence-ის ინტეგრირება პოპულარული პლაგინებთან და ხელსაწყოებთან

**ავტომატურად ირთვება, როდესაც:** Kadence თემა არის თქვენი აქტიური თემა.

## როგორ ხდება უნარების შერჩევა

ასისტენტი ავტომატურად აღpoznებს თქვენს აქტიურ თემას და დაინსტალირებულ პლაგინებს ყოველი შეტყობინებისას. თუ შესაბამისი თემის ცნობიერება (theme-aware) უნარი ხელმისაწვდომია, ის ავტომატურად ჩაიტვირთება ასისტენტის კონტექსტში. თქვენ არ გჭირდებათ უნარების ხელით გააქტიურება ან გადართვა.

### მრავალუნარები

თუ თქვენს საიტზე რამდენიმე უნარია საჭირო (მაგალითად, თუ აქტიური გაქვთ როგორც Kadence Blocks, ასევე Kadence Theme), ასისტენტს ხელმისაწვდომია ყველა შესაბამისი უნარი და შეუძლია დაისახოს მიმართავდეს თითოეული მათგანის მითითებებს.

### თემის შეცვლა (Switching Themes)

როდესაც თქვენ ცვლით აქტიურ თემას, ასისტენტის ხელმისაწვდომი უნარები ავტომატურად განახლდება შემდეგ შეტყობინებაში. მაგალითად:

1. თქვენ იყენებთ ბლოკების თემას და **Block Themes** (ბლოკების თემები) უნარია აქტიური.
2. თქვენ გადადიხართ კლასიკურ თემაზე.
3. შემდეგ შეტყობინებაში ავტომატურად ჩატვირთება **Classic Themes** (კლასიკური თემები) უნარი და **Block Themes** უნარი აღარ იქნება ხელმისაწვდომი.

## თემის ცნობად უნარების გამოყენება (Using Theme-Aware Skills)

თემის ცნობად უნარის გამოყენებისთვის, უბრალოდ აღწერეთ, რისი გაკეთება გსურთ ჩატის ინტერფეისში. ასისტენტი ავტომატურად დაისახავს შესაბამისი უნარის მითითებებს.

### მაგალითები პრომპტებისთვის (Example Prompts)

**Block Themes-ისთვის:**
> "შექმენით ჰერო სექცია ფონით სურათით და ცენტრში განლაგებული ტექსტი ბლოკების პატერნებით."

**Classic Themes-ისთვის:**
> "დაამატეთ საკუთარი ვიჯეტის ზონა საSidebarn-ს child theme-ით."

**Kadence Blocks-ისთვის:**
> "გაშენეთ ტესტიმონIAL-ები Kadence Testimonials ბლოკის გამოყენებით."

**Kadence Theme-ისთვის:**
> "შეასწორეთ ჰედერის განლაგება და ნავიგაციის მენიუს სტილი."

ასისტენტი მოგაწვდით თემა-სპეციფიკურ მითითებებსა და კოდის მაგალითებს, რომლებიც შეესაბამება თქვენს აქტიურ თემას და პლაგინებს.

:::note
თემის ცნობად უნარები ავტომატურად ხელმისაწვდომია Superdav AI Agent v1.10.0-ისა და შემდეგ ვერსიებში. დამატებითი დაყენება ან კონფიგურაცია საჭირო არ არის.
:::
