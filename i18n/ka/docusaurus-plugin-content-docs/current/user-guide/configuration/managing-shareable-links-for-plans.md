# Shareable Link-ების მართვა გეგმებისთვის {#managing-shareable-links-for-plans}

Ultimate Multisite საშუალებას გაძლევთ პირდაპირ თქვენს კლიენტებთან დაမျှოთ გეგმის ლინკები. ამისათვის, გადადით ნებისმიერ გეგმის რედაქტირების გვერდზე და დააჭირეთ ღილაკს „_Click to copy the Shareable Link_“ (დაჭერით Shareable Link-ის ასლი).

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_გეგმის რედაქტირების გვერდზე შეგიძლიათ იპოვოთ ღილაკი shareable link-ების შესაქმნელად_

დააკვირდით, რომ ეს მექანიზმი ლინკის შესაქმნელად იყენებს დეფოლტტ ბილინგის სიხშირეს. ეს ნიშნავს, რომ თუ თქვენი დეფოლტтный ბილინგი ყოველთვიურია, მაგალითად, საბოლოოდ თქვენ გექნებათ ასეთი ლინკი: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

თუ გსურთ shareable link-მა დეფოლტურად იყოს კვარტალური, დაგჭირდებათ 3-ის დამატება ბოლოს:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

იმისათვის, რომ გეგმა _premium_ ყოველწლიურად აირჩიოთ, დაამატებთ 1-ს შემდეგ სიტყვას:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

დახურულად გაითვალისწინეთ, რომ თქვენ მხოლოდ ერთი ლინკის არ შეიძლება ჰქონდეთ სხვადასხვა ბილინგის სიხშირისთვის. თქვენ უნდა გააზიაროთ ერთი კონკრეტული ლინკი თითოეული თქვენი ბილინგის სიხშირისთვის.

მას შემდეგ, რაც ლინკს მომხმარებლებთან გააზიარებთ, ის მათ რეგისტრაციის პროცესის პირველ ნაბიჯზე გადაიტანს.
