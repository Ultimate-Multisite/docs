# ინტერნეტით ძებნა (Internet Search)

Gratis AI Agent v1.5.0-ს ახლავს **ინტერნეტის ძებნის** შესაძლებლობა, რომელიც ხელოვნურ ინტელექტს საშუალებას აძლევს საუბრის განმავლობაში ვებგვერდებიდან უახლეს ინფორმაციას მოიპოვოს. ეს ფუნქცია მუშაობს [Brave Search API]-ზე (https://brave.com/search/api/) ან [Tavily API]-ზე (https://tavily.com/).

## როგორ მუშაობს

როდესაც ინტერნეტ ძებნა ჩართულია, ასისტენტი ავტომატურად შეძლებს თქვენ მიერ კონფიგურირებული საძიებო პროვაიდერის მოთხოვნას იმ შემთხვევაში, თუ დაადგენს, რომ კითხვა საჭიროებს აქტუალურ ან გარე ინფორმაციას — მაგალითად, ბოლო ამბებს, ცოცხალ ფასებს ან დოკუმენტაციას, რომელიც შეიძლება შეიცვალოს მოდელის ტრენინგის შეწყვეტის შემდეგ.

შედეგები მიიღება რეალურ დროში და ინექტირდება ასისტენტის კონტექსტში მანამდე, სანამ ის პასუხს გენერირებს. ასისტენტი აჩვენებს, თუ როდის გამოიყენა ძებნის შედეგები კითხვის დასახმარებლად.

## ინტერნეტის ძებნის ჩართვა

ინტერნეტის ძებნისთვის საჭიროა API გასაღები თქვენი არჩეული საძიებო პროვაიდერისგან. მისი კონფიგურირებისთვის:

1. გადადით **Gratis AI Agent → Settings → Advanced**-ზე.
2. იპოვეთ ჩამონათვალი **Internet Search Provider** და აირჩიეთ **Brave Search** ან **Tavily**.
3. შეიყვანეთ თქვენი API გასაღები შესაბამის ველში. თითოეული ველი გვერდითა განთავსებულია რეგისტრაციის ბმულებით.
4. დააჭირეთ **Save Settings**-ს.

გასაღების შენახვის შემდეგ, ინტერნეტის ძებნის შესაძლებლობა ავტომატურად ხელმისაწვდომია ასისტენტისთვის.

## Brave Search

### Brave Search API გასაღების მოპოვება

1. ეწვიეთ [Brave Search API გვერდს](https://brave.com/search/api/).
2. დარეგისტრირდით გეგმაზე. უფასო დონე ხელმისაწვდომია ყოველთვიური მოთხოვნების ლიმიტით.
3. დააკოპირეთ თქვენი API გასაღები Brave Search-ის დეველოპერის პანელიდან.
4. ჩასვით ის **Gratis AI Agent**-ის პარამეტრებში არსებულ **Brave Search API Key** ველში.

### გამოყენების ლიმიტები

გამოყენება ფასიანია Brave Search-ის მიერ, რაც დამოკიდებულია განხორციელებული მოთხოვნების რაოდენობაზე. თითოეული AI პასუხი, რომელიც ძებნას იწვევს, ერთი მოთხოვნა (query) დავით. გამოიყენეთ [Brave Search-ის დეველოპერული დაფა](https://brave.com/search/api/) თქვენი გამოყენების მონიტორინგისთვის და არ გამოგივიდეთ მოულოდნელი დანახარჯები.

## Tavily

Superdav AI Agent v1.10.0-ის მეშვეობით დამატებულია **Tavily** როგორც ალტერნატიული ინტერნეტ ძებნის პროვაიდერი, რომელიც გთავაზობთ უფრო მდიდარ შედეგებსა და მოწინავე კვლევის შესაძლებლობებს.

### Tavily API გასაღების მიღება

1. გადადით [Tavily-ის API გვერდზე](https://tavily.com/).
2. გაიარეთ რეგისტრაცია ან შექმენით ანგარიში. უფასო დონე ხელმისაწვდომია ყოველთვიური მოთხოვნების ლიმიტით.
3. დააკოპირეთ თქვენი API გასაღები Tavily-ის დ^{*}.
4. ჩასვით ის **Tavily API Key** ველით Gratis AI Agent-ის პარამეტრებში.

### გამოყენების ლიმიტები

გამოყენება ფასიანია Tavily-ის მიერ განხორციელებული API მოთხოვნების რაოდენობის მიხედვით. თითოეული AI პასუხი, რომელიც ძებნას იწვევს, ერთი მოთხოვნა დავით. მონიტორინგისთვის გამოიყენეთ [Tavily-ის დ^{*}.](https://tavily.com/) და არ გამოგივიდეთ მოულოდნელი დანახარჯები.

## ინტერნეტ ძებნის გამორთვა

API გასაღებს მოაშორეთ აქტიური ძებნის პროვაიდერის ველიდან და შეინახეთ ცვლილება. ასე, ასისტენტს აღარ შესთავაზდება ინტერნეტ ძებნის შესაძლებლობა.

:::note
ინტერნეტ ძებნა პასუხებზე დამატებით ლატენტობას (გვიანებას) იწვევს, რადგან ასისტენტმა ჯერ ძიების შედეგების მოლოდინი უნდა დაელოდოს პასუხის გენერირებისთვის. დროზე ორიენტირებული შემთხვევებისთვის გაითვალისწინეთ, საჭიროა თუ არა რეალურ დროში ძებნა ან საკმარისი იქნება თუ არა ასისტენტის ჩაშენებული ცოდნა.
:::
