# ფასდაკლების კოდების შექმნა (v2)

_**მნიშვნელოვანი შენიშვნა: ეს სტატია ეხება Ultimate Multisite-ის ვერსიას 2.x.**_

Ultimate Multisite-ით შეგიძლიათ შექმნათ ფასდაკლების კოდები, რათა თქვენს კლიენტებს შეთავაზოთ ფასდაკლება მათ გამოწერებზე. და მათი შექმნა ძალიან მარტივია!

## ფასდაკლების კოდების შექმნა და რედაქტირება {#creating-and-editing-discount-codes}

ფასდაკლების კოდის შესაქმნელად ან მის რედაქტირებისთვის გადადით **Ultimate Multisite > Discount Codes**-ში.

![ფასდაკლების კოდების სია — ცარიელი მდგომარეობა, სანამ კოდები არსებობს](/img/config/discount-codes-empty.png)

იქ თქვენ დაინახავთ იმ ფასდაკლების კოდების სიას, რომლებიც უკვე შექმენით.

ახალი კუპონის შესაქმნელად შეგიძლიათ დააწკაპუნოთ **Add Discount Code**-ზე, ან არსებულის რედაქტირებისთვის გადადით მათზე მაუსით და დააჭირეთ **Edit**.

![ფასდაკლების კოდების სია, სადაც მაუსის გადატანა აჩვენებს Edit და Delete ბმულებს](/img/config/discount-codes-list-hover.png)

![Add Discount Code ღილაკი გვერდის სათაურში](/img/config/discount-codes-add-button.png)

გადახვიდებით იმ გვერდზე, სადაც შექმნით ან რედაქტირებთ თქვენს კუპონის კოდს. ამ მაგალითში ახალს შევქმნით.

![ფასდაკლების კოდის რედაქტვის გვერდი, ყველა განყოფილება ჩანს](/img/config/discount-code-edit.png)

მოდით, დავხედოთ აქ ხელმისაწვდომ aux (პარამეტრებს):

**Discount Code-ის შეყვანა:** ეს არის თქვენი ფასდაკლების კოდის სახელი. ეს არ არის ის კოდი, რომელსაც მომხმარებლებმა checkout ფორმაში გამოიყენებენ.

**Description (აღწერა):** აქ შეგიძლიათ მოკლედ აღწეროთ, რისთვისაა ეს კუპონი.

![ფასდაკლების კოდის სახელისა და აღწერილობის ველი რედაქტვის გვერდის ზედა ნაწილში](/img/config/discount-code-description.png)

შეგიძლიათ ასევე ნახოთ ფასდაკლების კოდი აქტიური თუ არა:

![ფასდაკლების კოდის აქტიური სტატუსი](/img/config/discount-code-active.png)

**Coupon code (კუპონის კოდი):** სწორედ აქ განსაზღვრავთ იმ კოდს, რომელსაც მომხმარებლებმა checkout-ის დროს უნდა შეიყვანონ.

![Coupon code field where customers enter the code at checkout](/img/config/discount-code-coupon-field.png)

**კომუნალური შეღავათი (Discount):** აქ შეგიძლიათ დააყენოთ ან **პროცენტული** ან **ფიქსირებული თანხის** შეღავათი თქვენი კოდისთვის.

![შღავათის თანხის დაყენება პროცენტული ან ფიქსირებული თანხით 드롭დაუნი](/img/config/discount-code-amount.png)

**გაახლოს განახლებებზე (Apply to renewals):** თუ ეს ვარიანტი გამორთულია, ეს შეღავათის კოდი გამოყენებული იქნება მხოლოდ **პირველ გადახდას**. ყველა სხვა გადახდაზე შეღავათი არ გაქვთ. თუ ეს ვარიანტი ჩართულია, შეღავათის კოდი მოქმედებს ყველა მომავალი გადახდისთვის.

**დამზადების საფასურის შეღავათი (Setup fee discount):** თუ ეს ვარიანტი გამორთულია, კუპონის კოდს **არანაირი შეღავათი არ მისცემს შეკვეთის დამზადების საფასურისთვის**. თუ ეს ვარიანტი ჩართულია, შეგიძლიათ დააყენოთ შეღავათი (პროცენტული ან ფიქსირებული თანხის), რომელიც ამ კუპონის კოდს გამოიყენებს თქვენი პლანების დამზადების საფასურზე.

![გაახლოს განახლებებზე და დამზადების საფასურის შეღავათის ჩართვის ვარიანტები](/img/config/discount-code-renewals.png)

**აქტიური (Active):** ხელით გაააქტიურეთ ან გამორთეთ ეს კუპონის კოდი.

![აქტიური გადამრთველი შეღავათის კოდის ხელით ჩართვის ან გამორთვისთვის](/img/config/discount-code-active.png)

**მოწინავე ვარიანტებში (Advanced Options)** გვაქვს შემდეგი პარამეტრები:

![შღავათის კოდის მოწინავე ვარიანტები](/img/config/discount-code-advanced.png)

**გამოყენების ლიმიტი (Limit uses):**

  * **გამოყენებები (Uses):** აქ შეგიძლიათ ნახოთ, რამდენჯერ არის გამოყენებული შეღავათის კოდი.

  * **მაქსიმალური გამოყენება (Max uses):** ეს შეზღუდავს იმას, თუ რამდენჯერ შეიძლება გამოიყენოს მომხმარებელმა ეს შეღავათის კოდი. მაგალითად, თუ აქ ჩაწერید 10-ს, კუპონი მხოლოდ 10-ჯერ შეიძლება გამოყენებულ იქნეს. ამ ლიმიტის შემდეგ, კუპონის კოდი აღარ იქნება გამოყენებული.

![გამოყენების ლიმიტის დაყენება მიმდინარე გამოყენებებისა და მაქსიმალური გამოყენების ველით](/img/config/discount-code-limit-uses.png)

**დაწყების და ვადის თარიღები:** აქ შეძლებთ თქვენს კუპონზე დამატების დაწყების თარიღისა და/ან ვადის თარიღის არჩევას.

![Start and expiration date fields for scheduling the discount code](/img/config/discount-code-dates.png)

**პროდუქტების შეზღუდვა:** თუ ჩართავთ **Select products**-ს, თქვენ დაინახავთ ყველა თქვენს პროდუქტს. გექნება საშუალება ხელით აირჩიოთ (ჩართვის ან გამორთვის მეშვეობით), რომელი პროდუქტი შეიძლება მიიღოს ეს კუპონის კოდი. აქ გამორთული პროდუქტები არ შეიცვლება, თუ თქვენი მომხმარებლები ამ კუპონის კოდის გამოყენებას მათზე შეეცდებიან.

![Limit products section with per-product toggle switches](/img/config/discount-code-limit-products.png)

ყველა ამ ვარიანტის დაყენების შემდეგ, დააჭირეთ **Save Discount Code**-ს, რომ თქვენი კუპონი შეინახოთ და მესმის!

![Save Discount Code button at the bottom of the edit page](/img/config/discount-code-save.png)

კუპონი ახლა თქვენს სიაშია და იქიდან შეგიძლიათ დააწკაპუნოთ, რომ **გამოხვიდეთ ან წაშალოთ** ის.

![Discount code row in the list with Edit and Delete hover actions](/img/config/discount-codes-list-hover.png)

### URL პარამეტრების გამოყენება: {#using-url-parameters}

თუ გსურთ თქვენი ფასების ცხრილების დამატებით კონსტიტუირება ან ლამაზი კუპონის კოდის გვერდის შექმნა თქვენს ვებსაიტზე და ავტომატურად გამოიყენოთ დისკავერსის ფორმაში დისკავონ კოდი, შეგიძლიათ ეს URL პარამეტრების საშუალებით.

პირველ რიგში, თქვენ გჭირდებათ თქვენი გეგმისთვის გაზიარებადი ბმულის (shareable link) მიღება. ამისათვის გადადით **Ultimate Multisite > Products**-ში და აირჩიეთ გეგმა.

დააჭირეთ ღილაკს **Click to Copy Shareable Link**. ეს მოგცემთ გაზიარებად ბმულს ამ კონკრეტული გეგმისთვის. ჩვენს შემთხვევაში, გაზიარებადი ბმული იყო [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_.

![Product page with shareable link button](/img/config/products-list.png)

ამ კონკრეტულ პაკეტზე თქვენი ფასდაკლების კოდის გამოყენებისთვის, უბრალოდ URL-ში დაამატეთ პარამეტრი **?discount_code=XXX**. სადაც **XXX** არის კუპონის კოდი.

ჩვენს მაგალითში, ჩვენ ამ კონკრეტულ პროდუქტზე კუპონის კოდის **50OFF** გამოყენებას განვახორციელებთ.

ამ კონკრეტული პაკეტისა და 50OFF ფასდაკლების კოდის გამოყენების URL ასე გამოიყურება: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_.
