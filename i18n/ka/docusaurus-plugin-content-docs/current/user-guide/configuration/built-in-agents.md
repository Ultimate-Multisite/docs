# ჩაშენებული აგენტები

Gratis AI Agent v1.9.0-ს მოჰყვება ხუთი ჩაშენებული აგენტი, თითოეული წინასწარ კონფიგურირებულია ფოკუსირებული ინსტრუმენტების ნაკრებით, მორგებული სისტემური მითითებით და საწყისი შეთავაზებებით, რომლებიც შეესაბამება ამ სფეროში გავრცელებულ ამოცანებს. აგენტებს შორის გადართვა ცვლის იმას, რისი გაკეთება შეუძლია ასისტენტს და როგორ პასუხობს ის — თქვენი მხრიდან რაიმე კონფიგურაციის გარეშე. Superdav AI Agent v1.18.0-ს შეუძლია დაამატოს განრიგის მცოდნე ინსტრუმენტები, შეხსენების ჩანაწერები, დამტკიცების კარიბჭეები და SMS შეტყობინებები ამ სამუშაო პროცესებს, როდესაც დაკავშირებული ინტეგრაციები კონფიგურირებულია.

## რა არის აგენტი? {#what-is-an-agent}

თითოეული აგენტი არის დასახელებული კონფიგურაციის პროფილი, რომელიც აერთიანებს:

- **ინსტრუმენტები** — შესაძლებლობები, რომელთა გამოძახებაც აგენტს შეუძლია (მაგ. Content Writer-ს აქვს წვდომა ჩანაწერების შექმნის შესაძლებლობებზე; Design Studio-ს აქვს წვდომა CSS და theme.json შესაძლებლობებზე)
- **სისტემური მითითება** — ინსტრუქციები, რომლებიც განსაზღვრავს აგენტის ტონს, პრიორიტეტებსა და შეზღუდვებს
- **შეთავაზებები** — წინასწარ დაწერილი მითითებები, რომლებიც ჩატის ინტერფეისში ჩანს, რათა სწრაფად დაიწყოთ

## აგენტის ამრჩევზე წვდომა {#accessing-the-agent-picker}

1. გახსენით **Gratis AI Agent** პანელი WordPress-ის ადმინისტრატორის გვერდით ზოლში.
2. დააწკაპუნეთ **აგენტის ხატულაზე** ჩატის სათაურის ზედა მარცხენა კუთხეში (ხატულა იცვლება აქტიური აგენტის ასახვისთვის).
3. **აგენტის ამრჩევი** იხსნება ფორმა-ცხრილის გადაფარვის სახით. თითოეული აგენტი ჩამოთვლილია თავისი ხატულით, სახელით და ერთსტრიქონიანი აღწერით.
4. დააწკაპუნეთ აგენტის მწკრივზე მის გასააქტიურებლად. ჩატის სათაური დაუყოვნებლივ განახლდება.

თქვენ ასევე შეგიძლიათ აგენტების გადართვა საუბრის შუაში — ახალი აგენტის სისტემური მითითება ძალაში შევა შემდეგი შეტყობინებიდან.

## ხუთი ჩაშენებული აგენტი {#the-five-built-in-agents}

### კონტენტის ავტორი {#content-writer}

**ფოკუსი:** ჩანაწერების, გვერდებისა და საკონტაქტო ფორმების შექმნა და რედაქტირება.

**ხელმისაწვდომი ინსტრუმენტები:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Superdav AI Agent v1.18.0-ის ინტეგრაციების ჩართვის შემთხვევაში, კონფიგურირებული კალენდრის კონტექსტი, დამტკიცების კარიბჭეები, შეხსენებები და SMS შეტყობინებების ინსტრუმენტები ასევე შეიძლება ხელმისაწვდომი იყოს დამტკიცებული სამუშაო პროცესებისთვის.

**რას აკეთებს კარგად:**
- ბლოგის ჩანაწერების მონახაზის შექმნა და გამოქვეყნება მოკლე აღწერიდან ან სქემიდან
- ახალი საიტისთვის ლენდინგ გვერდების პაკეტების შექმნა
- საკონტაქტო და მოთხოვნის ფორმების შექმნა
- URL-იდან ან ძიებიდან ჩანაწერებზე გამორჩეული სურათების დაყენება
- კონფიგურირებული Google Calendar-ის კონტექსტიდან ღონისძიების შემდგომი შეტყობინებების მონახაზის შექმნა, შემდეგ კი შეტყობინებების გაგზავნამდე დამტკიცებისთვის შეჩერება

**საწყისი შეთავაზებები:**
- *დაწერე 500-სიტყვიანი ბლოგის ჩანაწერი WordPress multisite-ის უპირატესობების შესახებ.*
- *შექმენი About, Services და Contact გვერდები და გამოაქვეყნე ისინი.*
- *დაამატე ჯავშნის მოთხოვნის ფორმა Contact გვერდზე.*
- *შეადგინე შეხსენება ხვალინდელი კონფიგურირებული კალენდრის ღონისძიების მონაწილეებისთვის და დაელოდე დამტკიცებას მის გაგზავნამდე.*

### საიტის ამწყობი {#site-builder}

**ფოკუსი:** ვებსაიტის სრულად შექმნა ერთი მითითებიდან.

**ხელმისაწვდომი ინსტრუმენტები:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Superdav AI Agent v1.18.0-ით, კონფიგურირებული მართვადი სერვისის, დამტკიცების, შეხსენების, კალენდრისა და SMS ინსტრუმენტები შეიძლება ხელმისაწვდომი იყოს იქ, სადაც ადმინისტრატორები ჩართავენ მათ.

**რას აკეთებს კარგად:**
- აღწერილი ბიზნესის ტიპისთვის მრავალფაზიანი საიტის აგების გეგმის გენერირება
- თითოეული ფაზის ავტონომიურად შესრულება — სტრუქტურა, კონტენტი, ნავიგაცია, დიზაინი
- გეგმის შუაში შეცდომებიდან აღდგენა ხელით ჩარევის საჭიროების გარეშე
- რეკომენდებული plugin-ების დაყენება აგების ნაწილად
- საკონტაქტო ფორმების შექმნა პირდაპირ ჩატის ინტერფეისიდან (Superdav AI Agent v1.10.0+)
- გაშვების შეხსენებების ან მონაწილეთა შემდგომი კომუნიკაციის კოორდინირება დუბლირებული შეტყობინებების გარეშე, როდესაც დამტკიცების კარიბჭეები და შეხსენების ჩანაწერები ჩართულია

**საწყისი შეთავაზებები:**
- *ააგე ფოტოგრაფიის პორტფოლიოს საიტი გალერეის ჩანაწერის ტიპით, დაჯავშნის გვერდით და საკონტაქტო ფორმით.*
- *შექმენი რესტორნის ვებსაიტი ონლაინ მენიუთი, გახსნის საათებით და მაგიდის დაჯავშნის მოთხოვნის ფორმით.*
- *მოაწყე ფრილანს კონსულტანტის საიტი სერვისების გვერდებით, პორტფოლიოს სექციით და ბლოგით.*
- *დაამატე საკონტაქტო ფორმა Contact გვერდზე საიტის ამწყობის გამოყენებით.*
- *საიტის გაშვების საკონტროლო სიის დამტკიცების შემდეგ, გაუგზავნე SMS შეხსენება კონფიგურირებულ დაინტერესებულ საკონტაქტო პირს.*

### დიზაინის სტუდია {#design-studio}

**ფოკუსი:** ვიზუალური პერსონალიზაცია — ფერები, ტიპოგრაფია, CSS და ბლოკის შაბლონები.

**ხელმისაწვდომი ინსტრუმენტები:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**რას აკეთებს კარგად:**
- დასახელებული თემის პრესეტების გამოყენება (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- გლობალური ტიპოგრაფიისა და ფერთა პალიტრების დახვეწა theme.json-ის მეშვეობით
- მორგებული CSS-ის ჩასმა ბრენდის სპეციფიკური გადაფარვებისთვის
- გვერდის ეკრანის ანაბეჭდის გადაღება და მისი დიზაინის პრობლემების შესაფასებლად განხილვა

**საწყისი შეთავაზებები:**
- *გამოიყენე warm-editorial პრესეტი და შემდეგ დააყენე ძირითადი ფერი #2d6a4f-ზე.*
- *გადაიღე მთავარი გვერდის ეკრანის ანაბეჭდი და მითხარი, რას გააუმჯობესებდი.*
- *შექმენი ხელახლა გამოსაყენებელი hero ბლოკის შაბლონი სრული სიგანის ფონის სურათით და ცენტრში გასწორებული სათაურით.*

### plugin-ების მმართველი {#plugin-manager}

**ფოკუსი:** WordPress plugin-ების აღმოჩენა, დაყენება და მართვა.

**ხელმისაწვდომი ინსტრუმენტები:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**რას აკეთებს კარგად:**
- აღწერილი გამოყენების შემთხვევისთვის საუკეთესო plugin-ის რეკომენდირება
- შესაძლებლობების პაკეტების დაყენება რეესტრიდან
- ხელმისაწვდომი შესაძლებლობების კატალოგის დათვალიერება კატეგორიის მიხედვით

**საწყისი შეთავაზებები:**
- *რომელია საუკეთესო plugin membership დირექტორიისთვის?*
- *დააყენე WooCommerce შესაძლებლობების პაკეტი.*
- *მაჩვენე ყველა ხელმისაწვდომი ecommerce შესაძლებლობების პაკეტი.*

### მხარდაჭერის ასისტენტი {#support-assistant}

**ფოკუსი:** კითხვებზე პასუხის გაცემა საიტის კონტენტის, პარამეტრებისა და WordPress კონფიგურაციის შესახებ.

**ხელმისაწვდომი ინსტრუმენტები:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**რას აკეთებს კარგად:**
- მიმდინარე საიტის პარამეტრებისა და ვარიანტების მოძიება
- განმარტება, რომელი ჩანაწერის ტიპები, ტაქსონომიები და მენიუებია კონფიგურირებული საიტზე
- კითხვებზე „რას აკეთებს ეს პარამეტრი?“ პასუხის გაცემა ცოცხალი მნიშვნელობების წაკითხვით
- ცვლილებების განხორციელებამდე მხოლოდ წაკითხვადი დიაგნოსტიკური ფენის როლში მოქმედება

**საწყისი შეთავაზებები:**
- *რომელი plugins და პარამეტრებია ამჟამად აქტიური ამ საიტზე?*
- *ჩამოთვალე ყველა მორგებული ჩანაწერის ტიპი, რომელიც რეგისტრირებულია ამ საიტზე.*
- *რომელი ნავიგაციის მენიუები არსებობს და სად არის ისინი მინიჭებული?*

## Superdav Automation ინტეგრაციები {#superdav-automation-integrations}

## აგენტების მორგება {#customising-agents}

თითოეული ჩაშენებული აგენტი შეიძლება გაფართოვდეს ან ჩანაცვლდეს `gratis_ai_agent_agents` ფილტრის მეშვეობით.

### მორგებული სისტემური prompt-ის დამატება არსებულ აგენტში {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### ახალი აგენტის რეგისტრაცია {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

ახალი აგენტი გამოჩნდება Agent Picker-ში ფილტრის გაშვებისთანავე.

### ჩაშენებული აგენტის წაშლა {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
