# Ultimate Multisite-ის თარგმნა

Ultimate Multisite ინგლისურად არის შექმნილი, მაგრამ პლაგინი სრულად ითარგმნება და ჩვენ აქტიურად ვიღებთ თარგმანების წვლილის შეტანას.

## ძირითადი პლაგინის თარგმანები {#core-plugin-translations}

ძირითადი Ultimate Multisite პლაგინი ითარგმნება ოფიციალურ **WordPress.org-ის თარგმანების პლატფორმაზე**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

ეს არის სტანდარტული WordPress-ის საზოგადოების თარგმნის სისტემა. როდესაც თარგმანები მიაღწევს საკმარის დამტკიცების ზღვარს, ისინი ავტომატურად გადანაწილდება Ultimate Multisite-ის ყველა მომხმარებლისთვის WordPress პლაგინის განახლებების საშუალებით.

**გვჭირდება თქვენი დახმარება!** თუ ინგლისურის გარდა სხვა ენაზე საუბრობთ, გთხოვთ, დახმარებისთვის თარგმანების შეტანა გაითვალისწინოთ. რამდენიმე სტრიქონის თარგმნაც კი დაგვეხმარება. დასაწყებად:

1. ეწვიეთ [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. აირჩიეთ თქვენი ენა
3. დააწკაპუნეთ "Contribute Translation" (თარგმანის შეტანა) ღილაკზე
4. დაგჭირდებათ უფასო WordPress.org ანგარიში

თუ გსურთ გახდეთ თარგმნის რედაქტორი თქვენი ენისთვის (რომელიც შეგიძლიათ თარგმანების დამტკიცება), შეახსენეთ ეს [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) მეშვეობით.

## Addon-ის თარგმანები {#addon-translations}

Addon პლაგინები ითარგმნება ცალკე:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon-ის თარგმნები ამ ეტაპზე ხელოვნური ინტელექტის მიერ არის გენერირებული საწყისი წერტილისთვის. ისინი მოიცავს ყველა მხარდაჭერილ ენას, მაგრამ შეიძლება შეიცავდეს უხეშ ფორმულირებას ან არასწორ ინფორმაციას. ჩვენ歓迎ებთ ადამიანებს, რომლებიც შეამოწმებენ და შემოგთავაზებენ გაუმჯობესებებს მათ უფრო ბუნებრივად და ზუსტად გასაგებად.

## ადგილობრივი თარგმანები {#local-translations}

თუ გსურთ თარგმანების ადგილობრივად შექმნა საკუთარი პროექტისთვის და არა უშუალოდ კონტრიბუციის გაკეთება, არსებობს ორი ვარიანტი:

### Poedit-ის გამოყენება {#using-poedit}

თქვენ შეგიძლიათ გამოიყენოთ დესკტოპ აპლიკაცია **[Poedit](https://poedit.net)** `.pot` ფაილის თარგმნისთვის:

1. ჩამოტვირთეთ Poedit [poedit.net]-დან
2. გახსენით `.pot` ფაილი `wp-content/plugins/ultimate-multisite/lang/`-დან
3. თარგმნეთ სტრიქონები Poedit-ის ინტერფეისში
4. შეინახეთ -- Poedit აგენერირებს `.po` და `.mo` ფაილებს
5. განათავსეთ შედეგანი ფაილები `lang` საქაღალდეში

### Loco Translate-ის გამოყენება {#using-loco-translate}

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress პლაგინი საშუალებას გაძლევთ თარგმნოთ პირდაპირ ადმინისტრაციული პანელიდან, გარე დესკტოპ ინსტრუმენტების გარეშე. ის თქვენთვის თარგმანების ფაილების შექმნასა და კომპილაციას უზრუნველყოფს.
