# ელფოსტის და ბროკასტების გაგზავნა (v2)

_**მნიშვნელოვანი შენიშვნა: ეს სტატია Ultimate Multisite-ის ვერსია 2.x-ისთვის.**_

Ultimate Multisite-ს აქვს ფუნქცია, რომელიც საშუალებას გაძლევთ დაუკავშირდეთ თქვენს მომხმარებლებს, გამოგზავნოთ ელფოსტა კონკრეტულ მომხმარებელს ან მომხმარებელთა ჯგუფს, ასევე გამოქვეყნოთ შეტყობინებები მათ ადმინისტრაციულ პანელზე ბროკასტის სახით.

## დაამატეთ ადმინისტრაციული შეტყობინებები თქვენი მომხმარებლების პანელზე ბროკასტების საშუალებით

Ultimate Multisite-ის ბროკასტის ფუნქციის გამოყენებით, შეგიძლიათ დაამატოთ **ადმინისტრაციული შეტყობინებები** თქვენი მომხმარებლის ქვესაიტის ადმინისტრაციულ პანელზე.

ეს ძალიან სასარგებლო იქნება, თუ გჭირდებათ რაიმე განცხადების გაკეთება, მაგალითად, სისტემის შენახვის შესახებ შეტყობინების გაგზავნა ან ახალი პროდუქტებისა და სერვისების შეთავაზება თქვენს არსებულ მომხმარებლებს. ასე გამოიყურნება ადმინისტრაციული შეტყობინება თქვენი მომხმარებლის პანელზე.

<!-- Screenshot unavailable: ადმინისტრაციული შეტყობინების ბროკასტი მომხმარებლის ქვესაიტის პანელზე -->

ადმინისტრაციული შეტყობინების დასაწყებად, გადადით თქვენი ქსელის ადმინისტრაციულ პანელზე და **Ultimate Multisite** მენიუს ქვეშ ნახავთ **Broadcasts** (ბროკასტები) ვარიანტს.

![Ultimate Multisite-ის ადმინისტრაციის ბროკასტების სია](/img/admin/broadcasts-list.png)

არსებული ბროკასტების რედაქტირებაც შეგიძლიათ:

![ბროკასტის რედაქტირების ინტერფეისი](/img/admin/broadcast-edit.png)

ამ გვერდიდან ზემოთ დააჭირეთ **Add Broadcast** (დამატება ბროკასტი).

ეს გამოიტანს Add broadcast მოდალურ ფანჯარას, სადაც შეძლებთ აირჩიოთ, რა ტიპის ბროკასტს გსურთ გაგზავნოთ.

გაგრძელეთ და აირჩიეთ **Message** (შეტყობინება), შემდეგ დააჭირეთ ღილაკს **Next Step** (შემდეგი ნაბიჯი).

![Add broadcast მოდალი, Message-ის ვარიანტის შერჩევით](/img/admin/broadcast-add-message.png)

შემდეგი ფანჯარა გეკითხება ან **Target customer** (მიზნობრივი მომხმარებელი), ან **Target product** (მიზნობრივი პროდუქტი). გაითვალისწინეთ, რომ შეგიძლიათ აირჩიოთ ერთზე მეტი მომხმარებელი ან ერთზე მეტი პროდუქტი.

კონტაქტის ან პროდუქტის ძებნისთვის, თქვენ უნდა დაიწყოთ სიტყვის ჩ입ვა ველიში.

**Message type** (საკითხის ტიპი) ველიდან შეგიძლიათ აირჩიოთ შეტყობინების ფერი. ეს ხაზს უსვამს თქვენი შეტყობინების სასწავლო მნიშვნელობას.

შემდეგ შეგიძლიათ დააჭიროთ **Next Step** (ნაბიჯის შემდეგ).

![Target customers, target product and message type fields for a Message broadcast](/img/admin/broadcast-message-targets.png)

შემდეგი ფანჯარა არის ადგილი, სადაც შეძლებთ თქვენი შეტყობინების შედგენას, სათაურის და შინაარსის/შეტყობინების ჩაწერით, რომლის გაგზავნაც მომხმარებლებისთვის გსურთ.

![Broadcast message subject and content editor on the compose step](/img/admin/broadcast-edit.png)

შემდეგ შექმნის შემდეგ შეგიძლიათ დააჭიროთ **Send** (გაგზავნა) ღილაკს.

და ამით ყველაფერი დასრულებულია. ადმინისტრატორის შეტყობინება მაშინვე გამოჩნდება თქვენი მომხმარებლის დეშბორზე.

## ელექტრონული ფოსტის გაგზავნა თქვენს კლიენტებისთვის

Ultimate Multisite-ის ბროკასტის ფუნქციის გამოყენებით, შეგიძლიათ გაუგზავნოთ ელფოსტა თქვენს მომხმარებლებს. გექნება არჩევანი, რომ ელფოსტა გამოგზავნოთ მხოლოდ კონკრეტულ მომხმარებლებისთვის ან მიმართოთ კონკრეტულ მომხმარებელ ჯგუფს პროდუქტის ან პლანის საფუძველზე, რომლის ქვეშაც არის დარეგისტრირებული.

ელფოსტის ბროკასტის დასაწყებად, გადადით თქვენი ქსელის ადმინისტრაციის დეშბორტზე და Ultimate Multisite მენიუს ქვეშ იპოვით **Broadcast** (ბროკასიონის) ოფციას.

![Broadcasts list page used as the starting point for an email broadcast](/img/admin/broadcasts-list.png)

ამ გვერდიდან დააჭირეთ ზედა ნაწილში არსებულ **Add broadcast** (ბროკასტის დამატება) ღილაკს.

ეს გამოიტანს Add broadcast მოდალურ ფანჯარას, სადაც შეძლებთ აირჩიოთ, რა ტიპის ბროკასიონს გსურთ გაგზავნოთ. გადადით და აირჩიეთ **Email** (ელფოსტა), შემდეგ დააჭირეთ **Next Step** ღილაკს.

![Add broadcast modal with the Email option selected](/img/admin/broadcast-add-email.png)

შემდეგი ფანჯარა მოგთხოვთ ან **Target customer** (მიზნობრივი მომხმარებელი), ან **Target product** (მიზნობრივი პროდუქტი). გაითვალისწინეთ, რომ შეგიძლიათ აირჩიოთ ერთზე მეტი მომხმარებელი ან ერთზე მეტი პროდუქტი.

კონტაქტის ან პროდუქტის ძებნისთვის საჭიროა საკვანძო სიტყვის ჩ입ლით შევიدأ დაწეროთ ველი.

მას შემდეგ, რაც მიზნობრივი აუდიტორიას აირჩევთ, შეგიძლიათ დააწკაპუნოთ **Next Step**-ზე.

![მიზნობრივი მომხმარებლებისა და პროდუქტის შერჩევა ელექტრონული ბროკასტისთვის](/img/admin/broadcast-email-targets.png)

შემდეგი ფანჯარა არის ადგილი, სადაც შეგიძლიათ დაიწყოთ თქვენი ელფოსტის შედგენა, შევიტანოთ თემა და ის შინაარსი/მესიჯი, რომელსაც მომხმარებლებისთვის გსურთ გაგზავნა.

<!-- Screenshot unavailable: ელფოსტის ბროკასტის თემის და შინაარსის რედაქტორი კომპოზიციის ნაბიჯზე -->

მესიჯის შექმნის შემდეგ, შეგიძლიათ დააჭიროთ **Send** ღილაკს.

და ასე მარტივია ელფოსტის გაგზავნა თქვენს საბოლოო მომხმარებლებთან ბროკასტის ფუნქციონალით.

## სისტემური ელფოსტები (System emails)

Ultimate Multisite-ში სისტემური ელფოსტები არის **ავტომატური შეტყობინებები**, რომლებიც სისტემის მიერ იგზავნება გარკვეული ქმედებების შემდეგ, როგორიცაა რეგისტრაცია, გადახდა, დომენის მაცნობა და ა.შ. ეს ელფოსტები შეიძლება შეიცვალოს ან შეცვალოს Ultimate Multisite-ის პარამეტრებიდან. ასევე, მას აქვს ფუნქცია, რომელიც საშუალებას გაძლევთ გადაიტანოთ და იმპორტირდეთ არსებული პარამეტრები სხვა Ultimate Multisite ინს stall-იდან.

### გადატანა და იმპორტი (Resetting & Importing)

ახალი ვერსიების Ultimate Multisite-ისა და add-ons-ებისთვის დროდადრო შეიძლება ახალი ელფოსტები დარეგისტრირდეს.

კონფლიქტებისა და სხვა პრობლემების თავიდან ასაცილებლად, **ჩვენ ავტომატურად არ დავამატებთ ახალ ელფოსტის შაბლონებს სისტემურ ელფოსტებად თქვენს ინს stall-ზე**, თუ ისინი მნიშვნელოვანი არ არის კონკრეტული ფუნქციის სწორ მუშაობისთვის.

თუმცა, სუპერ ადმინისტრატორებსა და აგენტებს შეუძლიათ ამ ახლად დარეგისტრირებული ელფოსტების იმპორტირება იმპორტის ინსტრუმენტის საშუალებით. ეს პროცესი შექმნის ახალ სისტემურ ელფოსტას, ახალი შაბლონის შინაარსითა და კონფიგურაციით, რაც სუპერ ადმინისტრატორს საშუალებას მისცემს ნებისმიერი ცვლილება შეიტანოს ან მათ როგორც არის დატოვოს.

#### სისტემური ელფოსტების იმპორტირება როგორ ხდება

გადადით თქვენი Ultimate Multisite Settings გვერდზე და გადადით **Emails** ტაბულზე.

![Ultimate Multisite settings-ის Emails ტაბის სურათი System Emails ნაწილს აჩვენებს](/img/config/settings-emails-tab.png)

შემდეგ, સાйдბარში დააჭირეთ ღილაკს **Customize System Emails**.

<!-- Screenshot unavailable: System Emails სა이드ბარის პანელზე Customize System Emails ღილაკი -->

System Emails გვერდზე ზემოთ ნახავთ მოქმედების ღილაკს **Reset & Import**. ამ ღილაკზე დაჭერით გაიხსნება იმპორტის და გადატვირთვის (reset) მოდალური ფანჯარა.

![System Emails ადმინისტრაციული გვერდის Reset ან Import ღილაკი](/img/admin/system-emails-reset-import.png)

შემდეგ, შეგიძლიათ ჩართოთ Import Emails ოფციები და ნახოთ, თუ რომელი სისტემის ელექტრონული წერილები შეიძლება იმპორტირდეს.

#### System Emails-ის გადატვირთვა (Reseting System Emails)

სხვა დროს შეიძლება აღმოგარჩეთ, რომ კონკრეტულ ელექტრონული შაბლონზე (template) გაუკეთಿಸಿದ ცვლილებები თქვენთვის აღარ მუშაობს და გსურთ მისი დაბრუნება **დეფოლტულ მდგომარეობაში**.

ამ შემთხვევაში, თქვენ გაქვთ ორი ვარიანტი: შეგიძლიათ უბრალოდ წაშალოთ სისტემის ელ-ფოსტა და ის ხელახლა იმპორტირდეს (ზემოთ მოცემული ინსტრუქციების გამოყენებით) - რაც წაშლის გაგზავნის მეტრიკებს და სხვა რამეს, ამიტომ ეს მეთოდი ყველაზე ნაკლებად სასურველია.

ან შეგიძლიათ გამოიყენოთ **Reset & Import tool** იმ ელ-ფოსტის შაბლონის გადატვირთვისთვის.

ელ-ფოსტის შაბლონის გადატვირთვისთვის, შეგიძლიათ გააკეთოთ ზემოთ მოცემული ნაბიჯები და მიაღწიოთ Reset & Import ინსტრუმენტს, შემდეგ ჩართეთ **Reset** ოფცია და აირჩიეთ ის ელ-ფოსტები, რომელთა დაბრუნებაც გსურთ დეფოლტულ შინაარსზე.
