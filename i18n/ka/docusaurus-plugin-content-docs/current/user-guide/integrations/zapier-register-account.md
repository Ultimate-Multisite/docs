# მოვლენა:Zapier-ის საშუალებით ანგარიშის შექმნა {#event-register-an-account-via-zapier}

სტატიაში [Ultimate Multisite-ის ინტეგრირება Zapier-თან](zapier.md) განვიხილეთ, როგორ გამოვიყენოთ Zapier სხვადასხვა მოქმედებების შესასრულებლად Ultimate Multisite-ში ტრიგერებისა და მოვლენების საფუძველზე. ამ სტატიაში покажем, თუ როგორ შეგიძლიათ ინტეგრირება მესამე მხარის აპლიკაციებთან. ჩვენ გამოვიყენებთ Google Sheets-ს მონაცემთა წყაროდ და ინფორმაციას გადავურთავ Ultimate Multisite-ში ანგარიშის შესაქმნელად.

Zapier-თან დაკავშირებამდე, გადადით **Ultimate Multisite > Settings > API & Webhooks**-ზე და დაადასტურეთ, რომ API-ს აქვს ჩართული (enabled). როდესაც Zapier მოითხოვს Ultimate Multisite-ის ანგარიშის მონაცემებს, დააკოპირეთ API Key და API Secret ამ ეკრანიდან.

![API და Webhooks პარამეტრები API Key-ით, API Secret-ით და API-ის ჩართვის ვარიანტებით](/img/admin/settings-api-webhooks.png)

პირველ რიგში, თქვენ გჭირდებათ **Google Sheet**-ის შექმნა თქვენს Google Drive-ში. დარწმუნდით, რომ თითოეული სვეტი სწორად განსაზღვრულია, რათა მოგვიანებით მონაცემების მარტივად დაუკავშირდეს (map).

Google Sheet-ის შექმნის შემდეგ, შეგიძლიათ შეხვიდეთ თქვენს Zapier ანგარიშში და დაიწყოთ zap-ის შექმნა.

**"App event"-ის** საძიებელ ველიდან აირჩიეთ **"Google Sheets"**.

შემდეგ, **"Event"** ველიდან აირჩიეთ **"New spreadsheet row"** (ახალი ცხრილის სტრიქონი) და დააჭირეთ **"Continue"**-ს.

შემდეგი ნაბიჯი მოგთხოვთ Google Account-ის შერჩევას, სადაც შეინახულია **Google Sheet**. ასე რომ, უბრალოდ დარწმუნდით, რომ სწორი Google ანგარიშია მითითებული.

**"Set up trigger"-ის** ქვეშ თქვენ დაგჭირდებათ Google Spreadsheet-ის შერჩევა და განსაზღვრა იმ ფურცლის (worksheet), საიდან მოვა მონაცემები. უბრალოდ შეავსეთ ეს ველი და დააჭირეთ **"Continue"**.

შემდეგ არის **"test your trigger"**, რათა დარწმუნდეთ, რომ თქვენი Google Sheet სწორად არის დაკავშირებული.

თუ ტესტი წარმატებულია, თქვენ უნდა ნახოთ შედეგი, რომელიც აჩვენებს თქვენი ცხრილების ზოგიერთ მნიშვნელობას. გააგრძელებლად დააჭირეთ **"Continue"**.

შემდეგი ნაბიჯი არის მეორე მოქმედების დაყენება, რომელიც შექმნის ან ჩაመყარებს ანგარიშს Ultimate Multisite-ში. ძებნის ველიდან აირჩიეთ "**Ultimate Multisite(2.0.2)** ".

"Event"-ის ველში აირჩიეთ "**Register an Account in Ultimate Multisite**", შემდეგ კი დააჭირეთ "**Continue**" ღილაკს.

"Set up an action"-ქვეშ თქვენ დაინახავთ სხვადასხვა ველი, რომლებიც ხელმისაწვდომია მომხმარებლის მონაცემებისთვის, წევრობისთვის, პროდუქტებისთვის და ა.შ. შეგიძლიათ მნიშვნელობები მიბმა (map) გააკეთოთ თქვენს Google Sheet-ზე და დაურთოთ ისინი შესაბამის ველებში, სადაც უნდა შეივსონ, როგორც ქვემოთ მოცემულ სკრინშუტშია ნაჩვენები.

მნიშვნელობების მიბმასთან დაკავშირებით, შეგიძლიათ შეამოწმოთ მოქმედება.
