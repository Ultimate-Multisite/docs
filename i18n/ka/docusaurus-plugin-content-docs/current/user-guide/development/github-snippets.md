# როგორ გამოვიყენოთ Ultimate Multisite სნიპეტები ჩვენს GitHub რეპოზიტორიუმზე {#how-to-use-ultimate-multisite-snippets-on-our-github-repository}

GitHub რეპოზიტორიუმში არსებობს კოდის ნაჭრები, რომლებიც ხშირად მოითხოვება Ultimate Multisite მომხმარებლებისგან, რომლებსაც სურთ მცირე ფუნქციონალურობების დამატება, მაგალითად, Google Analytics-ის სკრიპტის დამატება რეგისტრაციის გვერდებზე ან მეტა ბოქსის დამალვა ადმინ პანელზე.

ამ სტატიაში გიჩვენებთ, როგორ გამოვიყენოთ ეს კოდები და კონკრეტულად, სად უნდა განათავსოთ ისინი.

სნიპეტებს შეგიძლიათ იპოვოთ ქვემოთ მოცემული ბმულზე:

https://github.com/next-press/wp-ultimo-snippets/

კოდის დამატების ორი გზა გაქვთ:

  1. თქვენი თემის functions.php ფაილში.

  2. Must-Use Plugins (mu-plugins)

# როგორ დავამატოთ სნიპეტი თქვენი თემის functions.php ფაილში. {#how-to-add-the-snippet-on-your-themes-functionsphp-file}

  1. შეხვიდით WordPress Network ადმინ პანელში და გადადით Themes > Theme Editor-ზე (გაიხედეთ ქვემოთ მოცემულ სურათზე).

  2. "Edit Themes" გვერდზე დარწმუნდით, რომ თქვენს აქტიურ თემას აირჩევთ ჩამონათვალიდან, რომელიც მდებარეობს ეკრანის ზედა მარჯვენა მხარეს (#3-ე სურათზე).

  3. დააწექით functions.php ფაილს "Theme Files" განყოფილების ქვეშ ფაილის ჩატვირთვისთვის. გადით ქვემოთ ბოლომდე და ჩასვით Ultimate Multisite სნიპეტი, რომელიც GitHub რეპოზიტორიიდან მიიღეთ.

# როგორ შევქმნათ Must-Use Plugins (mu-plugins) {#how-to-create-must-use-plugins-mu-plugins}

WordPress-ს აქვს ფუნქცია, რომელიც საშუალებას გაძლევთ დატვირთოთ მორგებული ფუნქციონალი, რომელსაც "Must-Use Plugins" ან მოკლედ "mu-plugins" ეწოდება.

ეს სპეციალური mu-plugins იტვირთება ყველა სხვა სტანდარტული პლაგინის წინ და მათ არ შეუძლიათ დეაქტივაცია. მულტისೈტ ქსელში, ამ mu-plugins-ის კოდი ჩატვირთება თქვენი ინსტალაციის ყველა საიტზე.

1. გამოიყენეთ FTP ან SSH თქვენი WordPress ინს stall-ის ფაილების სისტემის მისასვლელად.

2. თქვენი WordPress ინსტალაციის `wp-content` დირექტორიაში შექმენით ახალი დირექტორი სახელწოდებით: `mu-plugins`.

<!-- სურათი ხელმისაწვდომია არა: ფაილის მენეჯერი, რომელიც აჩვენებს wp-content დირექტორიას mu-plugins საქაღალდეს -->

3. თქვენს კომპიუტერზე შექმენით ახალი PHP ფაილი სახელწოდებით `wu-snippet.php` და გამოიყენეთ Notepad ან ნებისმიერი კოდის რედაქტორი.

4. ჩასვით Ultimate Multisite-ის კოდის ნაწყვეტი, რომელიც მიიღეთ GitHub-დან, ამ ფაილში და შეინახეთ. ასევე შეგიძლიათ ამ კოდის დამატება კოდის ნაწყვეტის ზემოთ, რომ თქვენს mu პლაგინსთვის დაარქვათ შენი სახელი (label).
