# როგორ შევიდე პირველად თქვენს ანგარიშზე?

გამარჯობათ Ultimate Multisite-ში!

თქვენი ანგარიშის გვერდზე წვდომა შეგიძლიათ იმ ელფოსტის ბმრით, რომელიც მიიღეთ ჩვენი პლაგინის ყიდვის შემდეგ:

<!-- სურათი ხელმისაწვდომია არა: ანგარიშის გვერის ელფოსტה ბმულით -->

გამოიყენეთ ის ელფოსტა, რომელიც შეძენისას მოგვაწოდეთ და პაროლი, რომელიც თქვენ ელფოსტაში მიიღეთ.

<!-- სურათი ხელმისაწვდომია არა: შესვლის გვერდი ელფოსტის და პაროლის ველით -->

რადგან ეს პირველი შეყვანაა ამ ანგარიშზე, მოგთხოვთ პაროლის შეცვლა. შეცვალეთ ის და დააჭირეთ _Save_ ღილაკს.

<!-- სურათი ხელმისაწვდომია არა: პაროლის შეცვის ფორმა Save ღილაკით -->

თქვენს ანგარიშის გვერდზე შეგიძლიათ მართოთ ყველა თქვენი ანგარიშის დეტალი, როგორიცაა გადასახადები, განახლებები, გაუქმებები და ბევრი სხვა რამ.
