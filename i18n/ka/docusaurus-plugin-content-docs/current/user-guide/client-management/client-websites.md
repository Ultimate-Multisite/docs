# თქვენი კლიენტების ვებგვერდები (v2)

საკონტაქტო ქსელში დარეგისტრირების შემდეგ, თქვენი მომხმარებლები შეძლებენ მივიდნენ საკუთარ დაფაზე.

**Account** გვერდის გარდა, სადაც მათ შეუძლიათ ნახონ მნიშვნელოვანი ინფორმაცია გადახდების, წევრობის, დომეინების, გეგმის შეზღუდვებისა და სხვა საკითხებთან დაკავშირებით, მათ ასევე აქვს доступ **My Sites** გვერდზე, სადაც შეძლებენ ყველა თავიანთი ვებგვერდის ნახვასა და მართვას. ყველაფერი რაც უნდა გააკეთონ, არის My Sites-ის დაჭერა მათ დაფაზე.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

მათ აჩვენებენ ყველა თავიანთ ვებგვერდს, თითოეული მათგანის ქვემოთ **Manage** ღილაკი. თუ darauf დააჭერენ, გადავიდებიან ამ კონკრეტული ვებგვერდის **dashboard-ზე**.

ასევე შეძლებენ ნახონ, რომელი ვებგვერდი არის **პირველადი**, იმ გეგმებისთვის, რომლებიც მათ საშუალებას აძლევს ერთზე მეტი ვებგვერდის მართვა.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

აი გვერდის რედაქტირების სრული ხედი:

![Site edit full interface](/img/admin/site-edit-full.png)

თუ მათ wollen зміან თავიანთ პირველადი ვებგვერდს, მათ უნდა გადავიდნენ Account გვერდზე და დააჭირონ **Account** მენიუს.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

მათ ასევე შეუძლიათ ნახონ და შეცვალონ მომხმარებლის დეტალები:

![Customer edit interface](/img/admin/customer-edit.png)

შემდეგ, მათ უნდა დააჭირონ **Change Default Site**.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

გამო გამოჩნდება პოპ-აპ ფანჯარა. დააწექით ჩამოსაშლელ სიაზე და აირჩიეთ, რომელი ვებგვერდი გსურთ გახდეს პირველადი ვებგვერდი.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

შემდეგ, უბრალოდ დააჭირეთ **Change Default Site** და დაგეხმარებით!

![სൈლის შეცვლის დადასტურება](/img/account-page/change-default-site-confirm.png)
