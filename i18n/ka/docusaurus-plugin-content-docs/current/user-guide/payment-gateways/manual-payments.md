# ხელით გადახდების დაყენება (v2) {#setting-up-manual-payments-v2}

_**მნიშვნელოვანი შენიშვნა: ეს სტატია ეხება Ultimate Multisite ვერსიას 2.x.**_

ხელით გადახდები არის გზა, რომ მომხმარებლებს შესთავაზოთ სხვა გადახდის მეთოდები, თუ **Stripe** ან **PayPal** მათთვის ხელმისაწვდომი არ არის. ეს შეიძლება იყოს საბანკო გადარიცხვა ან ნებისმიერი სხვა გადახდის მეთოდი, რომელიც თქვენი მომხმარებლებისთვის ადგილობრივად არის ხელმისაწვდომი.

## როგორ ჩართოთ ხელით გადახდები {#how-to-enable-manual-payments}

ხელით გადახდის დაყენება ძალიან მარტივია. საჭიროა მხოლოდ მისი ჩართვა გადახდის კარიბჭეებში და დეტალური ინსტრუქციების მითითება, თუ როგორ უნდა გამოგიგზავნოს მომხმარებელმა გადახდა.

პირველ რიგში, გადადით **Ultimate Multisite > Settings > Payments**. **Payment Gateways**-ის ქვემოთ ჩართეთ **Manual**. დაინახავთ, რომ თქვენთვის გამოჩნდება **Payment Instructions** ველი.

ამ ველში დაამატეთ ინფორმაცია, რომელიც თქვენს მომხმარებელს გადახდის შესასრულებლად დასჭირდება. მაგალითად, ეს შეიძლება იყოს თქვენი საბანკო ანგარიშის დეტალები და თქვენი ელფოსტა, რათა მომხმარებელმა გადახდის დადასტურება გამოგიგზავნოთ.

![ხელით გადახდის კარიბჭის გადამრთველი Payment Instructions ტექსტური ველით](/img/config/manual-gateway-expanded.png)

აი ხელით გადახდის კარიბჭის პარამეტრების ინტერფეისი:

![ხელით გადახდის კარიბჭის პარამეტრები](/img/config/manual-gateway-settings.png)

დაყენების შემდეგ, უბრალოდ დააჭირეთ **Save Settings**-ს და ყველაფერი მზადაა. როდესაც მომხმარებლები თქვენს ქსელში დარეგისტრირდებიან, ისინი ნახავენ შეტყობინებას, რომ შესყიდვის დასასრულებლად მიიღებენ თქვენს ინსტრუქციებს.

![რეგისტრაციის დადასტურების შეტყობინება, რომელიც მომხმარებელს აცნობებს, რომ ის გადახდის ინსტრუქციებს მიიღებს](/img/frontend/registration-manual-notice.png)

ისინი ასევე მიიღებენ შეტყობინებას თქვენს **Thank You** გვერდზე თქვენი გადახდის ინსტრუქციებით.

## ხელით გადახდების დადასტურება {#confirming-manual-payments}

ხელით გადახდის დასადასტურებლად, მარცხენა ზოლში გადადით **Payments** მენიუში. იქ შეგიძლიათ ნახოთ თქვენი ქსელის ყველა გადახდა და მათი დეტალები, მათ შორის მათი **status**. ხელით გადახდას ყოველთვის ექნება **Pending** სტატუსი, სანამ მას ხელით არ შეცვლით.

![გადახდების სია, სადაც ნაჩვენებია მომლოდინე ხელით გადახდა](/img/admin/payments-list.png)

გადახდის გვერდზე შესასვლელად დააჭირეთ **reference code**-ს. ამ გვერდზე გაქვთ მომლოდინე გადახდის ყველა დეტალი, როგორიცაა reference ID, პროდუქტები, დროის ნიშნულები და სხვა.

![გადახდის დეტალების გვერდი, სადაც ნაჩვენებია reference code, პროდუქტები და ჯამები](/img/admin/payment-edit.png)

მარჯვენა სვეტში შეგიძლიათ შეცვალოთ გადახდის სტატუსი. მისი **Completed**-ზე შეცვლა და **Activate Membership** პარამეტრის ჩართვა გაააქტიურებს თქვენი მომხმარებლის საიტს და მისი membership აქტიური იქნება.

![გადახდის რედაქტირების გვერდი, სადაც Status დაყენებულია Completed-ზე და Activate Membership ჩართულია](/img/admin/payment-activate-membership.png)
