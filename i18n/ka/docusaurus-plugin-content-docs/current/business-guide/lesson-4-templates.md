# გაკვეთილი 4: ნიშური შაბლონების შექმნა

შაბლონები თქვენი ნიშური ღირებულების შეთავაზების ბირთვია. ფიტნეს-სტუდიის მფლობელი, რომელიც რეგისტრირდება და ხედავს საიტს, რომელიც უკვე ფიტნეს-ვებსაიტს ჰგავს -- გაკვეთილების განრიგით, ტრენერების პროფილებით და შესაბამისი ვიზუალებით -- ბევრად უფრო დიდი ალბათობით დარჩება, ვიდრე ის, ვინც ცარიელ ტილოს ხედავს.

## სად გავჩერდით {#where-we-left-off}

გვაქვს მოქმედი FitSite ქსელი, სადაც Ultimate Multisite დაყენებული და კონფიგურირებულია. ახლა ვქმნით შაბლონებს, რომლებიც FitSite-ს ფიტნეს-ბიზნესებისთვის სპეციალურად შექმნილად აგრძნობინებს.

## რატომ არის ნიშური შაბლონები მნიშვნელოვანი {#why-niche-templates-matter}

ზოგადი შაბლონები თქვენს მომხმარებლებს აიძულებს რთული სამუშაო თავად გააკეთონ: გაარკვიონ, რა გვერდები სჭირდებათ, როგორი კონტენტის სტრუქტურა მუშაობს და როგორ მისცენ მას თავიანთი ინდუსტრიისთვის შესაფერისი იერი. ნიშური შაბლონები ამ ხახუნს აქრობს.

ფიტნეს-სტუდიის მფლობელმა, რომელიც FitSite-ზე რეგისტრირდება, უნდა ნახოს:

- მთავარი გვერდი, რომელიც ფიტნეს-სტუდიის ვებსაიტს ჰგავს
- უკვე შექმნილი გვერდები გაკვეთილებისთვის, განრიგებისთვის, ტრენერებისთვის და ფასებისთვის
- ვიზუალები და ჩანაცვლებითი კონტენტი, რომელიც მათ ინდუსტრიას შეესაბამება
- დიზაინი, რომელიც ფიტნესისთვის პროფესიონალურად და ბრენდთან შესაბამისობაში გამოიყურება

ისინი ავსებენ თავიანთ დეტალებს. ნულიდან არ იწყებენ.

## თქვენი შაბლონების დაგეგმვა {#planning-your-templates}

შექმნამდე გადაწყვიტეთ, რომელი შაბლონები შესთავაზოთ. FitSite-სთვის სამს შევქმნით:

### შაბლონი 1: Studio Essential {#template-1-studio-essential}

პატარა სტუდიებისა და პერსონალური ტრენერებისთვის.

- **მთავარი გვერდი** გმირი გამოსახულებით, გაკვეთილების გამოკვეთით და მოქმედებისკენ მოწოდებით
- **ჩვენ შესახებ** გვერდი სტუდიის ისტორიითა და მისიით
- **გაკვეთილები** გვერდი განრიგის განლაგებით
- **ტრენერები** გვერდი პროფილის ბარათებით
- **კონტაქტი** გვერდი მდებარეობის რუკითა და ფორმით
- **სუფთა, თანამედროვე დიზაინი** ფიტნესისთვის შესაფერისი ფერთა სქემით

### შაბლონი 2: Gym Pro {#template-2-gym-pro}

დამკვიდრებული სპორტდარბაზებისთვის, რომლებსაც მეტი სერვისი აქვთ.

- ყველაფერი Studio Essential-ში, დამატებით:
- **წევრობა** გვერდი ფასების ცხრილით
- **გალერეა** გვერდი ობიექტის ფოტოებისათვის
- **ბლოგი** სექცია ფიტნეს-რჩევებისა და სიახლეებისთვის
- **რეკომენდაციები** სექცია მთავარ გვერდზე
- **უფრო გამოკვეთილი ბრენდინგის** ვარიანტები

### შაბლონი 3: Fitness Chain {#template-3-fitness-chain}

მრავალლოკაციური ოპერაციებისთვის.

- ყველაფერი Gym Pro-ში, დამატებით:
- **ლოკაციები** გვერდი რამდენიმე ობიექტის ჩამონათვლით
- **ფრენჩაიზის/ლოკაციის** ქვეგვერდის შაბლონი
- **ცენტრალიზებული ბრენდინგი** ლოკაციისთვის სპეციფიკური დეტალებით
- **პერსონალის კატალოგი** ლოკაციებს შორის

## შაბლონური საიტის შექმნა {#building-a-template-site}

Ultimate Multisite-ში შაბლონი უბრალოდ WordPress საიტია, რომელიც ისეა კონფიგურირებული, როგორც გსურთ, რომ ახალი მომხმარებლების საიტები გამოიყურებოდეს. აი, როგორ შექმნათ ის:

### ნაბიჯი 1: შექმენით შაბლონური საიტი {#step-1-create-the-template-site}

1. გადადით **საიტები > ახლის დამატება** თქვენი ქსელის ადმინისტრაციაში
2. შექმენით საიტი სახელად `template-studio-essential`
3. ეს საიტი ხდება თქვენი სამუშაო ტილო

### ნაბიჯი 2: დააყენეთ და დააკონფიგურირეთ თემა {#step-2-install-and-configure-the-theme}

გადაერთეთ შაბლონური საიტის Dashboard-ზე და:

1. გაააქტიურეთ ფიტნეს-ბიზნესებისთვის შესაფერისი თემა
2. დააკონფიგურირეთ თემის პარამეტრები, ფერები და ტიპოგრაფია
3. დააყენეთ ჰედერი და ფუტერი ფიტნესისთვის შესაფერისი ნავიგაციით

:::tip თემის შერჩევა
აირჩიეთ თემა, რომელიც საკმარისად მოქნილია, რომ ფიტნესისთვის კარგად გამოიყურებოდეს, მაგრამ იმდენად რთული არ იყოს, რომ თქვენს მომხმარებლებს მისი მართვა გაუჭირდეთ. ისეთი თემები, როგორიცაა Astra, GeneratePress ან Kadence, კარგად მუშაობს, რადგან მსუბუქია, მორგებადია და კარგი მხარდაჭერა აქვს.
:::

### ნაბიჯი 3: შექმენით გვერდები {#step-3-create-the-pages}

ააწყეთ თითოეული გვერდი შემდეგით:

- **ჩანაცვლებითი კონტენტი**, რომელიც ფიტნესისთვის ბუნებრივად იკითხება ("კეთილი იყოს თქვენი მობრძანება [Studio Name]-ში" და არა "Lorem ipsum")
- **ჩანაცვლებითი სურათები** უფასო სტოკ-ფოტოების საიტებიდან, რომლებიც ფიტნეს-აქტივობებს აჩვენებს
- **ფუნქციური განლაგებები** გვერდების კონსტრუქტორის ან ბლოკური რედაქტორის გამოყენებით

სადაც შესაძლებელია, ჩანაცვლებითი კონტენტი ინსტრუქციული გახადეთ. ზოგადი შემავსებელი ტექსტის ნაცვლად დაწერეთ ასეთი კონტენტი: "ჩაანაცვლეთ ეს თქვენი სტუდიის მოკლე აღწერით და იმით, რაც მას განსაკუთრებულს ხდის. ახსენეთ თქვენი სავარჯიშო ფილოსოფია, გამოცდილების წლები ან ის, რას შეიძლება ელოდნენ კლიენტები."

### ნაბიჯი 4: დააკონფიგურირეთ პლაგინები {#step-4-configure-plugins}

დააყენეთ და გაააქტიურეთ პლაგინები, რომლებიც ფიტნეს-სტუდიებს სჭირდებათ:

- დაჯავშნის ან განრიგის პლაგინი (თუ თქვენს გეგმის დონეს შეესაბამება)
- საკონტაქტო ფორმის პლაგინი
- SEO პლაგინი (წინასწარ კონფიგურირებული ფიტნესთან დაკავშირებული ნაგულისხმევებით)

### ნაბიჯი 5: მონიშნეთ შაბლონად {#step-5-mark-as-template}

1. გადადით **Ultimate Multisite > საიტები**
2. დაარედაქტირეთ შაბლონური საიტი
3. ჩართეთ **საიტის შაბლონი** გადამრთველი

გაიმეორეთ ეს პროცესი თითოეული შაბლონისთვის, რომლის შეთავაზებაც გსურთ.

## შაბლონის ხარისხის საკონტროლო სია {#template-quality-checklist}

სანამ შაბლონს ხელმისაწვდომს გახდით, გადაამოწმეთ:

- [ ] ყველა გვერდი სწორად იტვირთება და პროფესიონალურად გამოიყურება
- [ ] ჩანაცვლებითი კონტენტი სასარგებლოა და ნიშას შეესაბამება
- [ ] სურათები შესაფერისია და სათანადოდ ლიცენზირებული
- [ ] მობილურ მოწყობილობებზე ადაპტიურობა ყველა გვერდზე მუშაობს
- [ ] ნავიგაცია ლოგიკური და სრულყოფილია
- [ ] საკონტაქტო ფორმები მუშაობს
- [ ] არ არის გაფუჭებული ბმულები ან დაკარგული რესურსები
- [ ] გვერდის ჩატვირთვის სიჩქარე მისაღებია

## FitSite ქსელი ამ ეტაპზე {#the-fitsite-network-so-far}

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured)
├── Hosting with wildcard SSL
├── Domain mapping configured
├── Site Templates
│   ├── Studio Essential (small studios, trainers)
│   ├── Gym Pro (established gyms)
│   └── Fitness Chain (multi-location)
└── Ready for plan configuration (next lesson)
```

## რა ავაშენეთ ამ გაკვეთილში {#what-we-built-this-lesson}

- **სამი ნიშაზე მორგებული შაბლონი**, შექმნილი სხვადასხვა ზომის ფიტნეს-ბიზნესებისთვის
- **ფიტნესისთვის შესაფერისი კონტენტი და ვიზუალები**, რომლებიც პლატფორმას სპეციალურად შექმნილად აგრძნობინებს
- **ინსტრუქციული ჩანაცვლებითი კონტენტი**, რომელიც მომხმარებლებს მორგების პროცესში გზას უჩვენებს
- **ხარისხის საკონტროლო სია**, რათა დარწმუნდეთ, რომ შაბლონები წარმოებისთვის მზადაა
