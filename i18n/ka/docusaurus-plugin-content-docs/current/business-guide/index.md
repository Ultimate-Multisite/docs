# შექმენით ნიშური ვებსაიტების ბიზნესი Ultimate Multisite-ით

ადამიანების უმეტესობა, ვინც ვებსაიტების გაყიდვას ცდილობს, ფასში ეჯიბრება Wix-ს, Squarespace-ს და ათასობით ფრილანსერს. ისინი აგებენ. ისინი, ვინც იგებენ, ირჩევენ კონკრეტულ აუდიტორიას, ქმნიან ამ აუდიტორიაზე მორგებულ გადაწყვეტას და თანხას იღებენ იმ ღირებულებისთვის, რომელსაც აწვდიან.

ეს კურსი გასწავლით, როგორ გააკეთოთ ზუსტად ეს Ultimate Multisite-ის გამოყენებით.

## რას შექმნით {#what-you-will-build}

ამ კურსის განმავლობაში შექმნით რეალურ, მოქმედ ქსელს სახელად **FitSite** -- ვებსაიტების პლატფორმას, რომელიც სპეციალურად ფიტნეს სტუდიებისთვის, პერსონალური მწვრთნელებისთვის და სავარჯიშო დარბაზების მფლობელებისთვის არის შექმნილი.

ბოლო გაკვეთილისთვის FitSite იქნება სრულყოფილი ნიშური პლატფორმა, რომელსაც ექნება:

- მიზნობრივად შექმნილი საიტის შაბლონები, რომლებიც ფიტნეს ბიზნესებისთვის არის შემუშავებული
- გეგმის დონეები, რომლებიც შეესაბამება იმას, თუ როგორ მუშაობენ ფიტნეს ბიზნესები რეალურად
- ბრენდირებული გადახდის გაფორმებისა და საწყისი დაყენების პროცესი
- ფასები, რომლებიც მორგებულია იმას, რისი გადახდაც ბაზარს შეუძლია
- სტრატეგია ფიტნეს ვერტიკალში მომხმარებლების პოვნისა და შენარჩუნებისთვის

თქვენ ნაბიჯ-ნაბიჯ მიჰყვებით პროცესს და ბოლოს გექნებათ როგორც მოქმედი პლატფორმა, ისე ცოდნა, რომ იგივე პროცესი გაიმეოროთ ნებისმიერი ნიშისთვის, რომელსაც აირჩევთ.

## რატომ ფიტნეს სტუდიები? {#why-fitness-studios}

გვჭირდებოდა კონკრეტული მაგალითი, რომელსაც ნებისმიერი მიჰყვებოდა. ფიტნეს სტუდიები კარგი მაგალითია, რადგან:

- **ყველას ესმის ეს ბიზნესი** -- პროცესის მიყოლისთვის სპეციალური ცოდნა არ არის საჭირო
- **მათ აქვთ მკაფიო ვებსაიტის საჭიროებები** -- გაკვეთილების განრიგები, მწვრთნელების პროფილები, დაჯავშნა, მდებარეობის ინფორმაცია
- **ისინი არასაკმარისად არიან მომსახურებული** -- ზოგადი ვებსაიტის კონსტრუქტორები მათ კონკრეტულ პრობლემებს არ წყვეტენ
- **ბუნებრივი გეგმის დონეები არსებობს** -- ერთ პერსონალურ მწვრთნელს სხვა საჭიროებები აქვს, ვიდრე რამდენიმე ლოკაციის მქონე სავარჯიშო დარბაზების ქსელს

ყველაფერი, რასაც FitSite-ის შექმნისას ისწავლით, პირდაპირ გამოიყენება ნებისმიერ ნიშაზე, რომელსაც აირჩევთ: რესტორნები, უძრავი ქონების აგენტები, იურიდიული ფირმები, ეკლესიები, სამშენებლო და მომსახურების კონტრაქტორები ან ნებისმიერი სხვა ვერტიკალი.

## წინაპირობები {#prerequisites}

- WordPress Multisite ინსტალაცია (ან მისი დაყენების სურვილი)
- აქტიური Ultimate Multisite ლიცენზია
- WordPress-ის ადმინისტრაციულ Dashboard-თან საბაზისო გაცნობადობა

კოდირება საჭირო არ არის. თუ Ultimate Multisite აქამდე არასოდეს გამოგიყენებიათ, ეს პრობლემა არ არის -- მე-3 გაკვეთილი სუფთა ინსტალაციიდან იწყება.
