# Multi-Network ცვლილებების ჟურნალი

### 1.3.0
* ახალი: ქსელის შაბლონის წინასწარი მნახველი — დაათვალიერეთ და წინასწარ ნახეთ ქსელის შაბლონები ცოცხალი პანელით შეძენამდე.
* ახალი: ქსელის შაბლონის ამრჩევი გადახდისას — მომხმარებლებს შეუძლიათ აირჩიონ ქსელის შაბლონი რეგისტრაციის გადახდის პროცესში.
* ახალი: ქსელებს შორის magic link SSO — მომხმარებლებს შეუძლიათ შეუფერხებლად წვდომა ქვე-ქსელებზე magic-link ავთენტიფიკაციის მეშვეობით.
* ახალი: ქსელის მედიის კლონირება — საიტის მედია სწორად დუბლირდება ქსელების შაბლონიდან კლონირებისას.
* გასწორება: ქსელის კლონირება ახლა მუშაობს multi-tenancy addon-ის გარეშე; super admin პრივილეგიები ხელახლა ენიჭება კლონირების შემდეგ.
* გასწორება: ქსელის შექმნა ახლა თავსებადია WooCommerce-თან და ყველა გადახდის gateway-სთან.
* გასწორება: პორტის ნომერი ახლა შედის fallback დომენის გენერაციაში.
* გასწორება: შაბლონის ამრჩევის CSS გადატანილია enqueued stylesheet-ში ყველა თემაზე სწორი რენდერისთვის.
* გასწორება: დაცვა განუსაზღვრელი `network_id` გასაღებისგან საიტის მოთხოვნის scope-ში.
* გასწორება: ქსელის ნებართვის შემოწმება კონტექსტის გადართვამდე თავიდან იცილებს `wpdb` ცხრილის თვისების შეცდომებს.
