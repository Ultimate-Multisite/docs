# Mailchimp

Multisite Ultimate: Mailchimp ინტეგრაცია

დააკავშირეთ თქვენი Multisite Ultimate ქსელი თქვენს Mailchimp-თან. შეგიძლიათ გარკვეული გეგმები დაუკავშიროთ Mailchimp სიებს და რეგისტრაციისას გამომწერები ავტომატურად დაამატოთ ამ სიებში.

## ინსტალაცია

1. ატვირთეთ 'ultimate-multisite-mailchimp' '/wp-content/plugins/' დირექტორიაში
2. გაააქტიურეთ პლაგინი 'პლაგინების' მენიუდან თქვენს WordPress Network Admin-ში

## გამოყენების სახელმძღვანელო

გამომწერების ელფოსტის მისამართების სინქრონიზაცია თქვენს MailChimp სიებთან ძალიან მარტივია ჩვენი Ultimate Multisite: MailChimp ინტეგრაციის დამატებით. ეს სახელმძღვანელო დაგეხმარებათ ყველაფრის გამართვაში!

## API გასაღების მიღება

სანამ ხელმისაწვდომ სხვადასხვა პარამეტრებზე გადავალთ, გვჭირდება MailChimp API გასაღები. ეს Ultimate Multisite-ს საშუალებას მისცემს, დაუკავშირდეს MailChimp API-ს სიების, ჯგუფების მისაღებად და მომხმარებლების რეგისტრაციისას ელფოსტის მისამართების ამ სიებსა და ჯგუფებში დასამატებლად.

მიჰყევით ამ სახელმძღვანელოს, რომ მიიღოთ თქვენი MailChimp API გასაღები: [API გასაღებების შესახებ | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

## ინტეგრაციის რეჟიმი

MailChimp ინტეგრაციის დამატება გთავაზობთ ორ „ინტეგრაციის რეჟიმს“, რაც ნიშნავს, რომ ელფოსტის მისამართების სინქრონიზაციისთვის დამატების გამოყენების ორი სხვადასხვა გზა არსებობს.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_ორი ინტეგრაციის რეჟიმი მეტი მოქნილობისთვის!_

### რეჟიმი 1: მრავალი სია

ნაგულისხმევი რეჟიმი იყენებს **მრავალ სიას**. ეს რეჟიმი საშუალებას მოგცემთ, თითოეული გეგმისთვის აირჩიოთ რამდენიმე MailChimp სია. ეს საუკეთესო ვარიანტია, თუ თქვენს მომხმარებლებს სხვადასხვა სიებად ყოფთ.

მაგალითად, დავუშვათ, თქვენს MailChimp Account-ზე გაქვთ 3 სხვადასხვა სია: **სიახლეების ბიულეტენი** , **გეგმა A** , და **გეგმა B**. გსურთ, ყველა მომხმარებელი, მათი გეგმის მიუხედავად, ასევე იყოს სიახლეების ბიულეტენის სიაში, რათა არ გამოტოვონ თქვენი განახლებები.

ამის მისაღწევად, შეგიძლიათ გამოიყენოთ მრავალი სიის რეჟიმი, რათა ორივე სია **სიახლეების ბიულეტენი** და **გეგმა A** დაამატოთ გეგმა A-ის MailChimp პარამეტრებში, ხოლო **სიახლეების ბიულეტენი** და **გეგმა B** — გეგმა B-ის MailChimp პარამეტრებში.

_აირჩიეთ ნაგულისხმევი სიები ახალი ანგარიშებისთვის და გაუქმებული ანგარიშებისთვის_

### რეჟიმი 2: ერთი სია + მრავალი ჯგუფი

MailChimp საშუალებას გაძლევთ, თქვენი სია დაყოთ მომხმარებლების უფრო მცირე ჯგუფებად, რომლებსაც **ჯგუფები** ეწოდება. ეს გაძლევთ მეტ კონტროლს, რადგან შეგიძლიათ კამპანია გაუგზავნოთ თქვენი გამომწერების კონკრეტულ ქვეჯგუფს, MailChimp-ზე იმავე კამპანიის მრავალი ასლის შექმნის გარეშე (რაც შემდეგ ამ კამპანიის მეტრიკების ანალიზს ართულებს).

მაგალითად, შეგიძლიათ გქონდეთ ერთი სია სახელად **ჩემი ქსელის გამომწერები,** და მის შიგნით გქონდეთ ჯგუფები **გეგმა A**-სა და **გეგმა B**-სთვის.

_აირჩიეთ ნაგულისხმევი სიები და ჯგუფები, თუ იყენებთ „ერთი სია + მრავალი ჯგუფი“ ინტეგრაციის რეჟიმს_

## ნაგულისხმევი სიები და ჯგუფები

მას შემდეგ, რაც აირჩევთ, რომელი ინტეგრაციის რეჟიმი შეესაბამება უკეთ თქვენს კონკრეტულ გამოყენების შემთხვევას, დროა აირჩიოთ ნაგულისხმევი სიები (და ჯგუფები, რეჟიმის მიხედვით). ნაგულისხმევი სია გამოყენებული იქნება მაშინ, როდესაც მოცემულ გეგმას კონკრეტული სია არჩეული არ აქვს.

ასევე შეგიძლიათ აირჩიოთ ნაგულისხმევი სია გაუქმებული ანგარიშებისთვის. როდესაც მომხმარებელი წყვეტს თავის გამოწერას და შლის თავის ანგარიშს თქვენი ქსელიდან, მისი ელფოსტის მისამართი გადატანილი იქნება გაუქმებული ანგარიშებისთვის დაყენებულ სიაში.

## ორმაგი გამოწერის ჩართვა

იმავე პარამეტრების გვერდზე შეძლებთ ჩართოთ **ორმაგი გამოწერა**. ეს პარამეტრი მომხმარებელს დაამატებს როგორც არჩეულ ნაგულისხმევ სიაში, ისე ამ მომხმარებლის გეგმისთვის არჩეულ სიაში. თუ ეს პარამეტრი ჩართული არ არის, მომხმარებელი დაემატება მხოლოდ მომხმარებლის გეგმისთვის არჩეულ სიაში. თუ ამ კონკრეტული გეგმისთვის არაფერი არის დაყენებული, მაშინ მომხმარებელი დაემატება არჩეულ ნაგულისხმევ სიაში.

## თითოეული გეგმისთვის სიებისა და ჯგუფების არჩევა

გადადით თქვენს **გეგმებში** და აირჩიეთ გეგმა, რომლის რედაქტირებაც გსურთ. გაითვალისწინეთ, რომ **გაფართოებული პარამეტრების** მეტა-ბლოკში ხელმისაწვდომი იქნება ახალი ჩანართი სახელად MailChimp.

ამ ჩანართზე შეძლებთ ამ გეგმისთვის აირჩიოთ მრავალი სია ან ერთი სია და მრავალი ჯგუფი (ინტეგრაციის რეჟიმის მიხედვით).
