# Logo SVG-ის გენერირება

**Generate Logo SVG** შესაძლებლობა Theme Builder-ს საშუალებას აძლევს, შექმნას და ჩასვას მორგებული logo SVG-ები პირდაპირ თქვენს WordPress საიტში, namespace-უსაფრთხო ავტომატური გასუფთავებით.

## მიმოხილვა

ეს შესაძლებლობა აგენერირებს მასშტაბირებად ვექტორულ გრაფიკას (SVG) ლოგოებს თქვენი საიტის ბრენდინგის მიმართულებისა და დიზაინის პრეფერენციების საფუძველზე. გენერირებული SVG-ები ავტომატურად სუფთავდება, რათა WordPress-ში გამოყენებისთვის უსაფრთხო იყოს და ამავდროულად შეინარჩუნოს ვიზუალური მთლიანობა.

## პარამეტრები

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | Yes | საიტის სახელი, რომლისთვისაც უნდა გენერირდეს ლოგო |
| `style` | string | Yes | დიზაინის სტილი (მაგ., "modern", "classic", "minimalist", "playful") |
| `colors` | array | No | hex ფერის კოდების მასივი, რომლებიც ლოგოში უნდა გამოიყენოთ (მაგ., `["#678233", "#ffffff"]`) |
| `width` | number | No | SVG-ის სიგანე პიქსელებში (ნაგულისხმევი: 200) |
| `height` | number | No | SVG-ის სიმაღლე პიქსელებში (ნაგულისხმევი: 200) |
| `include_text` | boolean | No | ჩაერთოს თუ არა საიტის სახელი ტექსტად ლოგოში (ნაგულისხმევი: true) |

## გამოტანის ფორმატი

ეს შესაძლებლობა აბრუნებს SVG სტრიქონს შემდეგი სტრუქტურით:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG-ის გასუფთავების ქცევა

გენერირებული SVG-ები გადის namespace-უსაფრთხო ავტომატურ გასუფთავებას, რათა:

- **ამოიშალოს არასაფრთხო ატრიბუტები** — შლის event handler-ებს, სკრიპტებს და პოტენციურად საშიშ ატრიბუტებს
- **შენარჩუნდეს namespace-ები** — ინარჩუნებს SVG namespace-ებს (xmlns, xlink) სწორად რენდერინგისთვის
- **დადასტურდეს სტრუქტურა** — უზრუნველყოფს, რომ SVG შეესაბამებოდეს W3C სტანდარტებს
- **დაიკოდოს entities** — სწორად უკეთებს escape-ს სპეციალურ სიმბოლოებს ტექსტურ შიგთავსში
- **ამოიშალოს გარე მითითებები** — შლის გარე stylesheet-ებსა და სურათების მითითებებს

ეს უზრუნველყოფს, რომ SVG უსაფრთხო იყოს პირდაპირ WordPress-ში ჩასასმელად დამატებითი გასუფთავების საჭიროების გარეშე.

## გამოყენების მაგალითი

**Prompt:**
```
დააგენერირე თანამედროვე ლოგო ჩემი ტექნოლოგიური სტარტაპისთვის, სახელად "CloudSync", ლურჯი და თეთრი ფერების გამოყენებით.
```

**შედეგი:**
ეს შესაძლებლობა ქმნის SVG ლოგოს, რომელიც:
- შეიცავს საიტის სახელს "CloudSync"
- იყენებს მითითებულ ლურჯ და თეთრ ფერთა სქემას
- მიჰყვება თანამედროვე დიზაინის პრინციპებს
- ავტომატურად გასუფთავებულია და მზადაა გამოსაყენებლად

## ინტეგრაცია Theme Builder-თან

Theme Builder-ის დიზაინის მიმართულების არჩევისას Generate Logo SVG შესაძლებლობა:

1. აანალიზებს თქვენს დიზაინის მიმართულებასა და ფერთა პალიტრას
2. აგენერირებს მორგებულ SVG ლოგოს, რომელიც შეესაბამება თქვენს პრეფერენციებს
3. ავტომატურად სვამს ლოგოს თქვენი საიტის header/ბრენდინგის არეში
4. ინახავს SVG-ს როგორც მორგებულ ლოგოს WordPress მედიაში

## საუკეთესო პრაქტიკები

- **მიუთითეთ მკაფიო სტილის პრეფერენციები** — აღწერეთ სასურველი დიზაინის სტილი (თანამედროვე, კლასიკური, playful და ა.შ.)
- **მიუთითეთ ფერები** — თანმიმდევრულობისთვის ჩართეთ თქვენი ბრენდის ფერები
- **შეამოწმეთ რენდერინგი** — გადაამოწმეთ, რომ ლოგო სწორად ჩანს ეკრანის სხვადასხვა ზომაზე
- **შემდგომ მოარგეთ** — გამოიყენეთ WordPress-ის ლოგოს მორგების ინსტრუმენტები ზომისა და განთავსების დასარეგულირებლად

## შეზღუდვები

- SVG ლოგოები გენერირდება როგორც სტატიკური გრაფიკა (არა ანიმირებული)
- მრავალი ელემენტის მქონე რთულ ლოგოებს შეიძლება დასჭირდეს ხელით დახვეწა
- მორგებული შრიფტები მხარდაჭერილი არ არის; ტექსტი იყენებს სისტემურ შრიფტებს
- ძალიან დიდმა ან ძალიან მცირე ზომებმა შეიძლება ხარისხზე იმოქმედოს
