# Superdav AI Agent

Superdav AI Agent არის გაფართოებული WordPress AI ასისტენტი, რომელიც აფართოებს Gratis AI Agent-ს თემაზე მორგებული უნარებით, გაძლიერებული ინტერნეტში ძიების შესაძლებლობებით, საიტის კონსტრუქტორის ფუნქციებით და სურვილისამებრ ავტომატიზაციის ინტეგრაციებით. ვერსია 1.18.0 ამატებს Google Calendar-ის წაკითხვის ხელსაწყოებს, კონტაქტებისა და დამსწრეების შესაბამისობის დადგენას, ადამიანის მიერ დამტკიცების ეტაპებს, შეხსენებების დუბლირების თავიდან აცილებას, TextBee SMS შეტყობინებებს, მართულ Superdav სერვისის უზრუნველყოფას და ცალკე გავრცელებულ Advanced კომპანიონ პაკეტს სანდო დეველოპერული სამუშაო პროცესებისთვის.

## ძირითადი ფუნქციები

- **თემაზე მორგებული უნარები** — ავტომატურად არგებს მითითებებსა და შესაძლებლობებს თქვენს აქტიურ WordPress თემასა და დაყენებულ პლაგინებს
- **ინტერნეტში ძიების მრავალი პროვაიდერი** — აირჩიეთ Brave Search-სა და Tavily-ს შორის უფრო მდიდარი ძიების შედეგებისა და გაფართოებული კვლევის შესაძლებლობებისთვის
- **საიტის კონსტრუქტორის საკონტაქტო ფორმები** — შექმენით სრულად ფუნქციური საკონტაქტო ფორმები პირდაპირ ჩატის ინტერფეისიდან, თქვენი დაყენებული ფორმების პლაგინის გამოყენებით
- **გაუმჯობესებული საიტის კონსტრუქტორი** — გაუმჯობესებული მრავალფაზიანი საიტის აგების გეგმები უკეთესი პლაგინების აღმოჩენითა და შეცდომების აღდგენით
- **ჩაშენებული თემის უნარები** — სპეციალიზებული მითითებები Block Themes-ისთვის, Classic Themes-ისთვის, Kadence Blocks-ისთვის და Kadence Theme-ისთვის
- **გაუმჯობესებული WooCommerce ინტეგრაცია** — გადაკეთებულია მშობლიური WooCommerce API-ების გამოსაყენებლად უკეთესი საიმედოობისა და თავსებადობისთვის
- **პროვაიდერის ავტომატური აღმოჩენა** — ავტომატურად პოულობს და აახლებს ხელმისაწვდომ პლაგინებსა და თემებს, როცა ისინი აქტიურდება ან ითიშება
- **მორგებული ლოგოს გენერირება** — შექმენით და ჩასვით მორგებული SVG ლოგოები namespace-ისთვის უსაფრთხო ავტომატური გასუფთავებით
- **პალიტრის კონტრასტის ვალიდაცია** — შეამოწმეთ ფერების წყვილები WCAG ხელმისაწვდომობის შესაბამისობაზე, სანამ მათ თქვენს თემაზე გამოიყენებთ
- **ფოტოებით გაუმჯობესებული აღმოჩენა** — ატვირთეთ ფოტოები Theme Builder აღმოჩენის ინტერვიუს დროს, დიზაინის მიმართულებაზე გავლენის მოსახდენად
- **სტუმარმასპინძლობის მენიუები** — შექმენით სტრუქტურირებული საკვებისა და სასმელების მენიუს გვერდები რესტორნებისთვის, კაფეებისთვის და სტუმარმასპინძლობის ბიზნესებისთვის
- **დესკტოპისა და მობილურის გადახედვები** — გადახედეთ თქვენს დიზაინს დესკტოპსა და მობილურ მოწყობილობებზე დიზაინის მიმართულების არჩევისას
- **Tier 1 საიტის scraping** — sd-ai-agent/site-scrape ახლა ნაგულისხმევად ხელმისაწვდომია კონკურენტებისა და შთაგონების ვებსაიტების ანალიზისთვის
- **Google Calendar-ის წაკითხვის ხელსაწყოები** — წაიკითხეთ კონფიგურირებული კალენდრები და მოვლენები, რათა ავტომატიზაციებმა შეძლონ მომავალი განრიგის კონტექსტზე მსჯელობა
- **კონტაქტებისა და დამსწრეების შესაბამისობის დადგენა** — დააკავშირეთ კალენდრის მონაწილეები ცნობილ საიტის მომხმარებლებსა და კონტაქტებს შემდგომი სამუშაოს გენერირებამდე
- **ადამიანის მიერ დამტკიცების ეტაპები** — შეაჩერეთ მგრძნობიარე ავტომატიზაციები გადახედვისთვის შეტყობინებების გაგზავნამდე, სამუშაოს გაგრძელებამდე ან ცვლილებების შეტანამდე
- **შეხსენებების ჩანაწერები და დუბლირების თავიდან აცილება** — აკონტროლეთ უკვე გაგზავნილი შეხსენებები, რათა განმეორებით გაშვებულმა ავტომატიზაციამ ერთსა და იმავე მიმღებს ორჯერ არ გაუგზავნოს შეტყობინება
- **TextBee SMS შეტყობინებები** — გააგზავნეთ კონფიგურირებული ტექსტური შეტყობინებები TextBee-ის მეშვეობით, როცა SMS პროვაიდერი ჩართულია
- **მართული Superdav სერვისის უზრუნველყოფა** — დააკავშირეთ მხარდაჭერილი საიტები ჰოსტირებულ Superdav ბოლო წერტილებთან ავტომატური დაყენებით, ხელით ბოლო წერტილების კონფიგურაციის ნაცვლად
- **Advanced კომპანიონ პაკეტი** — ცალკე გავრცელებული სანდო დეველოპერული ხელსაწყოები ფაილური სისტემის, მონაცემთა ბაზის, WP-CLI-ის, REST დისპეტჩერის, პლაგინის კონსტრუქტორის, git snapshot-ის, მომხმარებლების მართვისა და benchmark სამუშაო პროცესებისთვის

## მოთხოვნები

- WordPress 6.0 ან უფრო მაღალი
- PHP 8.1 ან უფრო მაღალი
- WordPress Multisite (ქსელურად აქტივირებული) ან ერთსაიტიანი ინსტალაცია
- აქტიური AI პროვაიდერის დამაკავშირებელი პლაგინი (Anthropic, OpenAI ან თავსებადი)
- Gratis AI Agent v1.9.0 ან უფრო მაღალი (Superdav AI Agent აფართოებს Gratis AI Agent-ს)

## ინსტალაცია

1. ატვირთეთ პლაგინის ფაილები `/wp-content/plugins/superdav-ai-agent`-ში
2. გაააქტიურეთ პლაგინი **პლაგინების** ეკრანიდან WordPress-ში
3. დარწმუნდით, რომ Gratis AI Agent ასევე გააქტიურებულია
4. გადადით **Gratis AI Agent → Settings**-ში და საჭიროებისამებრ დააკონფიგურირეთ თქვენი ინტერნეტში ძიების პროვაიდერი, მართული Superdav სერვისის კავშირი, Google Calendar ან TextBee-ის ავტორიზაციის მონაცემები

WordPress.org პაკეტი შეიცავს Superdav AI Agent-ის ძირითად ფუნქციებს. Advanced კომპანიონ პაკეტი ცალკე იქმნება და ვრცელდება გარემოებისთვის, სადაც სანდო დეველოპერული ხელსაწყოები აშკარად დაყენებული და ავტორიზებულია.

## სწრაფი დაწყება

გააქტიურების შემდეგ, Superdav AI Agent ავტომატურად აძლიერებს თქვენს Gratis AI Agent ინსტალაციას თემაზე მორგებული უნარებითა და დამატებითი შესაძლებლობებით. გახსენით **Gratis AI Agent** პანელი WordPress-ის ადმინისტრატორის გვერდითა ზოლში და დაიწყეთ ახალი ფუნქციების გამოყენება.

**მაგალითი ინსტრუქციები:**

```
დაამატე საკონტაქტო ფორმა Contact გვერდზე.
```

```
მოძებნე ვებში WordPress-ის უსაფრთხოების უახლესი საუკეთესო პრაქტიკები.
```

```
შექმენი hero სექცია აქტიური თემის ბლოკის შაბლონების გამოყენებით.
```

```
გადახედე ხვალინდელ კონფიგურირებულ Google Calendar მოვლენებს და მოამზადე შეხსენებების მონახაზები დამსწრეებისთვის.
```

```
გააგზავნე SMS შეხსენება მას შემდეგ, რაც შეტყობინებას დავამტკიცებ.
```

## თემაზე მორგებული უნარები

Superdav AI Agent ავტომატურად პოულობს თქვენს აქტიურ თემასა და დაყენებულ პლაგინებს და ტვირთავს შესაბამის თემაზე მორგებულ უნარს ასისტენტის კონტექსტში. ხელმისაწვდომი უნარებია:

| უნარი | ვრცელდება |
|---|---|
| **Block Themes** | თემები, რომლებიც იყენებს WordPress-ის ბლოკებზე დაფუძნებულ (Full Site Editing) არქიტექტურას |
| **Classic Themes** | ტრადიციული WordPress თემები, რომლებიც იყენებს PHP შაბლონებსა და კლასიკურ რედაქტორს |
| **Kadence Blocks** | საიტები, რომლებზეც Kadence Blocks პლაგინი აქტიურია |
| **Kadence Theme** | საიტები, რომლებზეც Kadence თემა აქტიური თემაა |

იხილეთ [თემაზე მორგებული უნარები](../../user-guide/configuration/theme-aware-skills) თითოეული უნარის დეტალური მითითებებისთვის.

## ინტერნეტში ძიების პროვაიდერები

Superdav AI Agent v1.10.0 ამატებს **Tavily**-ს Brave Search-ის ალტერნატივად ინტერნეტში ძიების მოთხოვნებისთვის. ორივე პროვაიდერი გთავაზობთ რეალურ დროში ძიების შედეგებს და მათი კონფიგურაცია შესაძლებელია **Gratis AI Agent → Settings → Advanced**-ში.

| პროვაიდერი | ფუნქციები | დაყენება |
|---|---|---|
| **Brave Search** | სწრაფი, კონფიდენციალურობაზე ორიენტირებული ძიების შედეგები | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | უფრო მდიდარი ძიების შედეგები, გაფართოებული კვლევის შესაძლებლობები | [Tavily API](https://tavily.com/) |

კონფიგურაციის ინსტრუქციებისთვის იხილეთ [ინტერნეტ-ძიება](../../user-guide/configuration/internet-search).

## კალენდრის, დამტკიცებისა და შეტყობინებების სამუშაო პროცესები

Superdav AI Agent v1.18.0 აფართოებს ავტომატიზაციის სამუშაო პროცესებს კონტენტისა და საიტის შექმნის ამოცანების მიღმა:

- **Google Calendar-ის ხელსაწყოები** აგენტს საშუალებას აძლევს წაიკითხოს კონფიგურირებული კალენდრები და ღონისძიებების დეტალები, როცა კალენდრის სერტიფიკატები ხელმისაწვდომია.
- **კონტაქტებისა და დამსწრეთა დამხმარე ხელსაწყოები** ღონისძიების მონაწილეებს უკავშირებს WordPress-ის მომხმარებლებს ან ცნობილ კონტაქტებს, სანამ შეხსენებებს ან შემდგომ ამოცანებს მოამზადებს.
- **ადამიანის დამტკიცების კარიბჭეები** ავტომატიზაციას აძლევს საშუალებას შეჩერდეს, სანამ ავტორიზებული მომხმარებელი შემოთავაზებულ მოქმედებას განიხილავს.
