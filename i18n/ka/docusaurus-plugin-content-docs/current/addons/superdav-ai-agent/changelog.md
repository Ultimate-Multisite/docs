# ცვლილებების ჟურნალი {#changelog}

## 1.18.0 — გამოშვებულია 2026-06-29 {#1180--released-on-2026-06-29}

### ახალი {#new}

- **Google Calendar-ის ხელსაწყოები** — წაიკითხეთ კონფიგურირებული კალენდრები და მოვლენები განრიგზე მორგებული ავტომატიზაციებისთვის
- **კონტაქტების შესაბამისობა და მონაწილეთა დამხმარეები** — შეუსაბამეთ კალენდრის მონაწილეები საიტის მომხმარებლებსა და კონტაქტებს
- **ადამიანის დამტკიცების კარიბჭეები და შეხსენების ჩანაწერები** — შეაჩერეთ ავტომატიზაციები განხილვისთვის და თავიდან აიცილეთ დუბლირებული შეტყობინებები
- **TextBee SMS პროვაიდერი** — გაგზავნეთ კონფიგურირებული ტექსტური შეტყობინებები TextBee-ის მეშვეობით
- **გაფართოებული თანმხლები პაკეტი** — დაამატეთ სანდო დეველოპერის ფაილური სისტემის, მონაცემთა ბაზის, WP-CLI-ის, REST დისპეტჩერის, plugin-ის შემქმნელის, git snapshot-ის, მომხმარებელთა მართვისა და benchmark-ის ხელსაწყოები, რომლებიც WordPress.org-ის build-ისგან განცალკევებით ვრცელდება

### გაუმჯობესდა {#improved}

- **Managed Superdav სერვისის დაყენება** — დაამატეთ hosted სერვისის endpoint-ები და ავტომატური კავშირის provision-ინგი მხარდაჭერილი საიტებისთვის
- **გამოშვების შეფუთვა** — ააგეთ ცალკე core და გაფართოებული ZIP-ები, გამოაქვეყნეთ ორივე GitHub-ზე და WordPress.org-ზე გაგზავნეთ მხოლოდ core პაკეტი

### გასწორდა {#fixed}

- **AI მოთხოვნების საიმედოობა** — გაუმჯობესდა მოდელის არჩევა, მოთხოვნის timeout-ები, ნაგულისხმევი დაყენების მოდელი, reasoning-text-ის დამუშავება და ხელახალი ცდის მითითებები არასწორი ხელსაწყოს გამოძახებებისთვის
- **კალენდრისა და შეხსენებების გამყარება** — გაძლიერდა Google Calendar-ის token-ები და შეხსენებების დუბლირების თავიდან აცილება
- **Onboarding და დამტკიცების გაგრძელებები** — გასწორდა frontend onboarding-ის გაშვებები და დადასტურებული შესაძლებლობების გაგრძელებები
- **WordPress.org-ის შეფუთვის განხილვის საკითხები** — გათვალისწინებულია შეფუთვის განხილვის უკუკავშირი core გამოშვებისთვის

## 1.16.0 — გამოშვებულია 2026-05-20 {#1160--released-on-2026-05-20}

### ახალი {#new-1}

- **Generate Logo SVG შესაძლებლობა** — Theme Builder-ს ახლა შეუძლია შექმნას და ჩასვას მორგებული logo SVG-ები namespace-თან უსაფრთხო სანიტიზაციით
- **ფოტოს ატვირთვა discovery interview-ში** — Theme Builder-ის discovery interview ახლა მოიცავს ფოტოს ატვირთვის ნაბიჯს უფრო მდიდარი დიზაინის კონტექსტისთვის
- **Validate Palette Contrast შესაძლებლობა** — შეამოწმეთ ფერების წყვილები WCAG შესაბამისობაზე theme-ზე გამოყენებამდე
- **სტუმარმასპინძლობის მენიუები** — Theme Builder-ს ახლა შეუძლია შექმნას სტრუქტურირებული საკვებისა და სასმელების მენიუს გვერდები სტუმარმასპინძლობის ბიზნესებისთვის
- **desktop და mobile preview rendering** — დაათვალიერეთ თქვენი დიზაინი desktop და mobile მოწყობილობებზე დიზაინის მიმართულების არჩევისას
- **ნავიგაციის label პარამეტრი** — Create Menu შესაძლებლობა ახლა მხარს უჭერს ცალკე `navigation_label`-ს, რომელიც გვერდის სათაურისგან განსხვავებულია
- **Tier 1 ხელსაწყოს ხელმისაწვდომობა** — sd-ai-agent/site-scrape ახლა Tier 1 ხელსაწყოა, რომელიც ნაგულისხმევად ხელმისაწვდომია Theme Builder-ში

### გასწორდა {#fixed-1}

- **AI Client-ის cache** — ახლა ეყრდნობა transients-ს მოთხოვნებს შორის მდგრადობისთვის, რაც თავიდან აცილებს მონაცემთა დაკარგვას ხანგრძლივად გაშვებულ agent ამოცანებზე
- **Plugin-ის მწკრივის მოქმედების ბმულები** — გასწორდა და გადაერქვა მეტი სიცხადისთვის

## 1.10.0 — გამოშვებულია 2026-05-05 {#1100--released-on-2026-05-05}

### ახალი {#new-2}

- **Tavily ინტერნეტ-ძიება** — დაამატეთ Tavily როგორც ძიების პროვაიდერი უფრო მდიდარი ინტერნეტ-ძიების შედეგებისთვის Brave Search-თან ერთად
- **Theme-ზე მორგებული ჩაშენებული უნარები** — Block Themes, Classic Themes, Kadence Blocks და Kadence Theme უნარების გზამკვლევები ახლა plugin-თან ერთად მოდის
- **საიტის შემქმნელის საკონტაქტო ფორმის შესაძლებლობა** — დაამატეთ საკონტაქტო ფორმა ნებისმიერ გვერდზე პირდაპირ chat ინტერფეისიდან

### გაუმჯობესდა {#improved-1}

- **WooCommerce ინტეგრაცია გადაკეთდა** — ახლა იყენებს native WooCommerce API-ებს უკეთესი საიმედოობისა და თავსებადობისთვის
- **პროვაიდერების სია ავტომატურად განახლდება** — როცა ნებისმიერი plugin აქტიურდება ან დეაქტიურდება

### გასწორდა {#fixed-2}

- **navigate-to შესაძლებლობა** — გასწორდა უსასრულო reload ციკლი ზოგიერთ admin გვერდზე
- **list-posts შესაძლებლობა** — ახლა სწორად უკავშირებს category და tag სახელებს slug-ებს
- **WP-CLI ბრძანებები** — აღდგა დაკარგული namespace alias-ები წინა refactor-ის შემდეგ
- **მოვლენების ავტომატიზაცია** — გამართულად ამუშავებს საიტებს, სადაც ავტომატიზაციის ცხრილები ჯერ არ არის შექმნილი
- **memory-save შესაძლებლობა** — ახლა იყენებს სწორ namespace prefix-ს სისტემური ინსტრუქციის შემქმნელში
- **Scalar ხელსაწყოს შედეგები** — ახლა სწორად იფუთება AI-სთვის დაბრუნებამდე
- **გამოყენების სტატისტიკა** — ახლა სწორად ამუშავებს ძველი შესაძლებლობის key ფორმატს ძველი ვერსიებიდან upgrade-ისას
