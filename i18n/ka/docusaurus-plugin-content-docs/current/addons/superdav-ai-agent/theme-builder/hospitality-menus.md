# Hospitality მენიუები {#hospitality-menus}

**Hospitality მენიუები** ფუნქცია Theme Builder-ს საშუალებას აძლევს, შექმნას და ჩააშენოს სტრუქტურირებული საკვებისა და სასმელების მენიუს გვერდები პირდაპირ თქვენს WordPress საიტში.

## მიმოხილვა {#overview}

Theme Builder-ს ახლა შეუძლია შექმნას პროფესიონალური, სტრუქტურირებული მენიუს გვერდები სტუმარ-მასპინძლობის ბიზნესებისთვის, მათ შორის რესტორნებისთვის, კაფეებისთვის, ბარებისთვის და ქეითერინგის სერვისებისთვის. ეს მენიუები სრულად ინტეგრირებულია თქვენი საიტის დიზაინში და მათი განახლება და მართვა მარტივად შეიძლება.

## მხარდაჭერილი Hospitality ტიპები {#supported-hospitality-types}

- **რესტორნები** — სრული მომსახურების კვების მენიუები
- **კაფეები** — ყავისა და მსუბუქი კერძების მენიუები
- **ბარები და ლაუნჯები** — სასმელებისა და აპეტაიზერების მენიუები
- **საცხობები** — ნამცხვრებისა და პურის მენიუები
- **ქეითერინგის სერვისები** — ღონისძიების მენიუს ვარიანტები
- **ფუდ ტრაკები** — მობილური საკვების სერვისის მენიუები
- **ლუდსახარშები და ღვინის საწარმოები** — სასმელების მენიუები აღწერებით

## მენიუს სტრუქტურა {#menu-structure}

### მენიუს კატეგორიები {#menu-categories}

მენიუები ორგანიზებულია კატეგორიებად:

- **აპეტაიზერები** — დასაწყისი კერძები და მცირე პორციები
- **ძირითადი კერძები** — მთავარი კერძები
- **გარნირები** — დანამატები და ბოსტნეული
- **დესერტები** — ტკბილი ნუგბარი
- **სასმელები** — სასმელები (ალკოჰოლური და უალკოჰოლო)
- **სპეციალური შეთავაზებები** — დღიური ან სეზონური სპეციალური შეთავაზებები

### მენიუს ელემენტის ფორმატი {#menu-item-format}

მენიუს თითოეული ელემენტი შეიცავს:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### ელემენტის ველები {#item-fields}

| ველი | ტიპი | აღწერა |
|-------|------|-------------|
| `name` | string | კერძის ან სასმელის სახელი |
| `description` | string | ელემენტის დეტალური აღწერა |
| `price` | string | ფასი (ფორმატირებული ვალუტით) |
| `dietary_info` | array | კვებითი ატრიბუტები (ვეგანური, გლუტენის გარეშე და ა.შ.) |
| `allergens` | array | გავრცელებული ალერგენები (თხილეული, მოლუსკები და ა.შ.) |
| `availability` | string | როდის არის ხელმისაწვდომი (ყოველდღიურად, სეზონურად და ა.შ.) |

## Hospitality მენიუების შექმნა {#creating-hospitality-menus}

### ნაბიჯი 1: მიაწოდეთ მენიუს ინფორმაცია {#step-1-provide-menu-information}

მოუყევით Theme Builder-ს თქვენი მენიუს შესახებ:

```
შექმენი რესტორნის მენიუ ჩემი იტალიური რესტორნისთვის აპეტაიზერებით,
პასტის კერძებით, მთავარი კერძებით და დესერტებით. ჩართე ფასები და აღწერები.
```

### ნაბიჯი 2: Theme Builder ქმნის მენიუს {#step-2-theme-builder-generates-menu}

Theme Builder:

1. ქმნის სტრუქტურირებულ მენიუს გვერდს
2. აფორმებს მას თქვენი საიტის თემასთან შესატყვისად
3. აწყობს ელემენტებს კატეგორიებად
4. აფორმატებს ფასებსა და აღწერებს
5. ამატებს კვებით და ალერგენების ინფორმაციას

### ნაბიჯი 3: გადახედვა და მორგება {#step-3-review-and-customize}

თქვენ შეგიძლიათ:

1. მენიუს ელემენტებისა და ფასების რედაქტირება
2. კატეგორიების დამატება ან წაშლა
3. ელემენტების გადალაგება კატეგორიების შიგნით
4. აღწერებისა და კვებითი ინფორმაციის განახლება
5. სტილისა და განლაგების მორგება

## მენიუს ჩვენების ვარიანტები {#menu-display-options}

### სრული მენიუს გვერდი {#full-menu-page}

გამოყოფილი გვერდი, რომელიც აჩვენებს თქვენს სრულ მენიუს:

- ორგანიზებულია კატეგორიების მიხედვით
- ძიებისა და ფილტრაციის შესაძლებლობით
- ბეჭდვისთვის მოსახერხებელი განლაგება
- მობილურზე ადაპტირებული დიზაინი

### მენიუს ვიჯეტი {#menu-widget}

ჩააშენეთ მენიუს სექციები სხვა გვერდებზე:

- გამორჩეული ელემენტები მთავარ გვერდზე
- დღიური სპეციალური შეთავაზებები გვერდით ზოლში
- სასმელების მენიუ ბარის გვერდზე
- დესერტების ვიტრინა ქვედა კოლონტიტულში

### მენიუს PDF {#menu-pdf}

შექმენით ჩამოსატვირთი PDF მენიუ:

- პროფესიონალური ფორმატირება
- ბეჭდვისთვის მზად ხარისხი
- მოიცავს სურათებსა და აღწერებს
- მარტივია გაზიარება და ელფოსტით გაგზავნა

## კვებითი და ალერგენების ინფორმაცია {#dietary-and-allergen-information}

### კვებითი ატრიბუტები {#dietary-attributes}

მონიშნეთ ელემენტები კვებითი ინფორმაციით:

- **ვეგანური** — ცხოველური პროდუქტების გარეშე
- **ვეგეტარიანული** — ხორცის გარეშე
- **გლუტენის გარეშე** — უსაფრთხოა ცელიაკიისთვის
- **რძის პროდუქტების გარეშე** — რძის პროდუქტების გარეშე
- **თხილეულის გარეშე** — ხის თხილებისა და მიწისთხილის გარეშე
- **დაბალი ნახშირწყლებით** — შემცირებული ნახშირწყლები
- **მაღალი ცილით** — ცილით მდიდარი

### ალერგენების გაფრთხილებები {#allergen-warnings}

ჩართეთ გავრცელებული ალერგენები:

- **თხილეული** — ხის თხილები და მიწისთხილი
- **მოლუსკები და კიბოსნაირები** — კიბოსნაირები და მოლუსკები
- **თევზი** — ყველა სახეობის თევზი
- **რძის პროდუქტები** — რძე და რძის პროდუქტები
- **კვერცხი** — ქათმის კვერცხი
- **სოია** — სოიის პროდუქტები
- **გლუტენი** — ხორბალი და გლუტენის შემცველი მარცვლეული
- **სეზამი** — სეზამის თესლი და ზეთი

## მენიუს მართვა {#menu-management}

### ფასების განახლება {#updating-prices}

მარტივად განაახლეთ მენიუს ფასები:

1. გადადით მენიუს გვერდზე
2. დააწკაპუნეთ "მენიუს რედაქტირება"
3. განაახლეთ ელემენტების ფასები
4. შეინახეთ ცვლილებები
5. ცვლილებები დაუყოვნებლივ გამოჩნდება თქვენს საიტზე

### სეზონური ელემენტების დამატება {#adding-seasonal-items}

შექმენით სეზონური მენიუს ვარიაციები:

1. შექმენით ახალი მენიუს ვერსია
2. დაამატეთ სეზონური ელემენტები
3. მონიშნეთ ელემენტები როგორც "სეზონური"
4. დაგეგმეთ ხელმისაწვდომობის თარიღები
5. ავტომატურად აჩვენეთ სეზონის განმავლობაში

### სპეციალური შეთავაზებების მართვა {#managing-specials}

აჩვენეთ დღიური ან ყოველკვირეული სპეციალური შეთავაზებები:

1. შექმენით "სპეციალური შეთავაზებები" კატეგორია
2. დაამატეთ ელემენტები ხელმისაწვდომობის თარიღებით
3. გამოკვეთეთ სპეციალური შეთავაზებები მთავარ გვერდზე
4. განაახლეთ ყოველდღიურად ან ყოველკვირეულად
5. დააარქივეთ ძველი სპეციალური შეთავაზებები

## ინტეგრაცია Theme Builder-თან {#integration-with-theme-builder}

Theme Builder-ის გამოყენებისას Hospitality საიტებისთვის:

1. **მენიუს ავტომატური ამოცნობა** — ადგენს, ხართ თუ არა საკვების/სასმელების ბიზნესი
2. **მენიუს გვერდის შექმნა** — ქმნის პროფესიონალურ მენიუს გვერდებს
3. **დიზაინის შესაბამისობა** — მენიუები ემთხვევა თქვენი საიტის თემას
4. **მობილური ოპტიმიზაცია** — მენიუები ლამაზად ჩანს ტელეფონებზე
5. **SEO ოპტიმიზაცია** — მენიუები საძიებო სისტემებისთვის მოსახერხებელია

## საუკეთესო პრაქტიკები {#best-practices}

### მენიუს დიზაინი {#menu-design}

- **მკაფიო ორგანიზება** — ლოგიკური კატეგორიის სტრუქტურა
- **წაკითხვადი აღწერები** — მადის აღმძვრელი და ინფორმაციული
- **თანმიმდევრული ფასები** — მკაფიო ვალუტა და ფორმატირება
- **პროფესიონალური ფოტოები** — მაღალი ხარისხის საკვების სურათები
- **ცარიელი სივრცე** — ნუ გადატვირთავთ გვერდს

### შინაარსი {#content}

- **ზუსტი აღწერები** — ზუსტად აღწერეთ ელემენტები
- **გამოკვეთეთ სპეციალური შეთავაზებები** — გამოარჩიეთ სპეციალური შეთავაზებები
- **ჩართეთ ალერგენები** — ყოველთვის ჩამოთვალეთ ალერგენები
- **რეგულარულად განაახლეთ** — ფასები და ელემენტები აქტუალური გქონდეთ
- **გამოიყენეთ მადის აღმძვრელი ენა** — ელემენტები გემრიელად ჟღერდეს

### ხელმისაწვდომობა {#accessibility}

- **წაკითხვადი შრიფტები** — გამოიყენეთ მკაფიო, ადვილად წასაკითხი შრიფტები
- **საკმარისი კონტრასტი** — უზრუნველყავით ტექსტის წაკითხვადობა
- **კვებითი ეტიკეტები** — მკაფიოდ მონიშნეთ კვებითი ვარიანტები
- **ალერგენების გაფრთხილებები** — თვალსაჩინოდ აჩვენეთ ალერგენები
- **მობილურზე მოსახერხებელი** — შეამოწმეთ ყველა მოწყობილობაზე
