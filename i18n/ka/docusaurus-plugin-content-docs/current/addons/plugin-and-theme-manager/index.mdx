# პლაგინებისა და თემების მმართველი {#plugin--theme-manager}

Multisite Ultimate: პლაგინებისა და თემების მმართველი

დაარედაქტირეთ ან დამალეთ დაყენებული პლაგინებისა და თემების მეტა ინფორმაცია (სათაური, ავტორი, მინიატურა და აღწერა). შექმენით კატეგორიები პლაგინებისა და თემებისთვის, რათა თქვენს მომხმარებლებს მათი გაფილტვრის საშუალება მისცეთ და დაამატეთ ლამაზი მორგებული პლაგინების გვერდი თქვენი მომხმარებლების პანელში.

პლაგინის გამოსაყენებლად, უბრალოდ გადადით თქვენი ქსელის ადმინისტრაციაში პლაგინების ან თემების გვერდებზე და დააწკაპუნეთ „პლაგინის/თემის ინფორმაციის რედაქტირებაზე“.

## ინსტალაცია {#installation}

1. ატვირთეთ 'ultimate-multisite-plugin-and-theme-manager' '/wp-content/plugins/' დირექტორიაში
2. გაააქტიურეთ პლაგინი 'Plugins' მენიუდან თქვენს WordPress-ის ქსელის ადმინისტრაციაში

## გამოყენების სახელმძღვანელო {#usage-guide}

Ultimate Multisite: პლაგინებისა და თემების მმართველი არის მძლავრი **premium** Ultimate Multisite დანამატი, რომელიც გაძლევთ სრულ კონტროლს იმაზე, თუ როგორ ხედავენ და იყენებენ თქვენი ქსელის მომხმარებლები პლაგინებისა და თემების პარამეტრებს თქვენს ქსელში.

ამ დანამატს სამი დანიშნულება აქვს:

  1. ის საშუალებას გაძლევთ სრულად შეცვალოთ როგორც პლაგინების, ისე თემების მეტამონაცემები. ეს მოიცავს სახელს, აღწერას, ავტორის სახელს და მინიატურებსაც კი!

  2. ის საშუალებას გაძლევთ დაახარისხოთ პლაგინები და თემები კატეგორიებად, რათა თქვენს მომხმარებლებს მათი მარტივად დალაგება შეეძლოთ.

  3. ის გაძლევთ შესაძლებლობას ძველი, მოსაწყენი პლაგინების გვერდი ჩაანაცვლოთ ახალი, თემების მსგავს გვერდით, კატეგორიების მიხედვით დალაგებით.

მისი ჩამოტვირთვისა და დაყენების შემდეგ Ultimate Multisite პარამეტრების გვერდზე გამოჩნდება ახალი ჩანართი - _დანამატის პარამეტრები_ \-.
