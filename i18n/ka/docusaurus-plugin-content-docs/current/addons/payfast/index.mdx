# PayFast

Ultimate Multisite: Payfast Gateway

ამატებს Payfast-ს — სწრაფ და უსაფრთხო ონლაინ გადახდების gateway-ს სამხრეთ აფრიკაში — Ultimate Multisite-ის გადახდის ვარიანტად.

## ინსტალაცია {#installation}

1. ატვირთეთ 'ultimate-multisite-payfast' '/wp-content/plugins/' დირექტორიაში
2. გაააქტიურეთ plugin WordPress-ის 'Plugins' მენიუდან
3. მიჰყევით ნაბიჯ-ნაბიჯ Wizard-ს plugin-ის დასაყენებლად

## გამოყენების გზამკვლევი {#usage-guide}

PayFast არის წამყვანი ონლაინ გადახდების დამმუშავებელი სამხრეთ აფრიკაში მოქმედი ბიზნესებისთვის. ის ეხმარება ონლაინ მოვაჭრეებს ზრდაში, ტრანზაქციების შეუფერხებელი და უსაფრთხო გახდის გზით. Ultimate Multisite მომხმარებლის მოთხოვნის შესაბამისად, გადავწყვიტეთ შეგვექმნა add-on, რომელიც PayFast-თან ინტეგრაციას შესაძლებელს გახდის. ეს სტატია გაჩვენებთ, როგორ დააყენოთ PayFast თქვენს გადახდის gateway-დ, რათა მიიღოთ გადახდები თქვენს Ultimate Multisite ქსელში.

პირველი ნაბიჯია add-on-ის ინსტალაცია.

# PayFast-ის დაყენება

add-on-ის დაყენების შემდეგ, ახლა შეგიძლიათ გადახვიდეთ Ultimate Multisite პარამეტრებში და დაიწყოთ PayFast-ის კონფიგურაცია.

**Ultimate Multisite settings** გვერდზე შეგიძლიათ იპოვოთ "Payments" ჩანართი. ამ ჩანართში უნდა ნახოთ "**Payfast** " toggle ღილაკთან ერთად, რომ ჩართოთ ან გამორთოთ ეს გადახდის gateway.

![გადახდის gateway-ის პარამეტრები](/img/config/settings-payment-gateways.png)განაგრძეთ და დააწკაპუნეთ toggle ღილაკზე მის ჩასართავად. ამის შემდეგ ველი გაფართოვდება და აჩვენებს სხვადასხვა ველს, რომლებიც უნდა შეავსოთ, როგორიცაა **Mechant ID, Merchant Key და PassPhrase.** ამ ინფორმაციას მიიღებთ თქვენი **Payfast** Account-იდან

<!-- ეკრანის ანაბეჭდი მიუწვდომელია: PayFast-ის გაფართოებული პარამეტრები, რომლებიც აჩვენებს Merchant ID, Merchant Key და PassPhrase ველებს -->

შემდეგი ნაბიჯია თქვენს PayFast Account-ში შესვლა, რათა მიიღოთ საჭირო ინფორმაცია.

შესვლისთანავე შეგიძლიათ დაუყოვნებლივ ნახოთ საჭირო ავტორიზაციის მონაცემები თქვენი **Payfast** Account Dashboard-იდან.

<!-- ეკრანის ანაბეჭდი მიუწვდომელია: PayFast Account Dashboard, რომელიც აჩვენებს Merchant ID და Merchant Key ავტორიზაციის მონაცემებს -->შეგიძლიათ უბრალოდ დააკოპიროთ და ჩასვათ ავტორიზაციის მონაცემები და შეავსოთ დეტალები თქვენს Ultimate Multisite პარამეტრებში. შემდეგ დააწკაპუნეთ **"Save settings** " ღილაკზე ცვლილებების შესანახად.

<!-- ეკრანის ანაბეჭდი მიუწვდომელია: Ultimate Multisite PayFast პარამეტრები შევსებული ავტორიზაციის მონაცემებით და Save ღილაკით -->

სულ ესაა, თქვენი PayFast გადახდის ინტეგრაცია დასრულებულია!
