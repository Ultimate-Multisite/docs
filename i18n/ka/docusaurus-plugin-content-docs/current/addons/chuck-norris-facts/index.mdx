# Chuck Norris-ის ფაქტები {#chuck-norris-facts}

დაამატეთ ცოტა იუმორი თქვენს ყოველდღიურ WordPress-ის ადმინისტრირების ამოცანებს **Ultimate Multisite - Chuck Norris Facts** დამატებით. გააქტიურებისას, ყოველი ადმინისტრირების გვერდის ზედა მარჯვენა კუთხეში გამოჩნდება შემთხვევითი Chuck Norris-ის ფაქტი.

* **ყინულის გალღობა**: შესანიშნავი საუბრის დამწყები ახალი გუნდის წევრებისთვის
* **პროდუქტიულობის ზრდა**: სწრაფ სიცილს შეუძლია გააუმჯობესოს კონცენტრაცია და პროდუქტიულობა
* **მარადიული იუმორი**: Chuck Norris-ის ფაქტები არასდროს ძველდება

## ინსტალაცია {#installation}

1. დარწმუნდით, რომ Ultimate Multisite პლაგინი დაინსტალირებული და გააქტიურებულია თქვენს WordPress ქსელში

3. ქსელურად გააქტიურეთ პლაგინი WordPress-ის 'Plugins' მენიუდან
4. ისიამოვნეთ შემთხვევითი Chuck Norris-ის ფაქტებით ყოველი ადმინისტრირების გვერდზე!
