# Metered Plans ცვლილებების ჟურნალი

ვერსია 1.1.0 - გამოვიდა 2026-05-05
- ახალი: AI token ბილინგი multisite ქვე-საიტებისთვის — აკონტროლეთ და დაარიცხეთ AI token გამოყენება მომხმარებელთა საიტებზე კონფიგურირებადი თითო-token ტარიფებით
- ახალი: Connector-ის აღსრულება ხელახლა აიგო დინამიკური ლიმიტების აღმოჩენითა და write-through-ით, რაც ყველა connector-ში რეალურ დროში სიზუსტეს უზრუნველყოფს
- გამოსწორება: მონაცემთა ბაზის ცხრილების განახლებები ახლა სწორად შეესაბამება BerlinDB schema definitions-ს, რაც ახალ ინსტალაციებზე განახლების ჩავარდნებს თავიდან აგაცილებთ
- გამოსწორება: მონაცემთა ბაზის განახლების callbacks გადაყვანილია სწორ ფორმატში, რაც ჩუმ განახლების ჩავარდნებს აგვარებს
- გამოსწორება: წილადი მნიშვნელობები ახლა მიიღება AI Usage Overage Markup-ის შეყვანის ველში
- გამოსწორება: აღმოფხვრილია fatal errors და ორმაგი ინიციალიზაციის პრობლემები plugin-ის ჩატვირთვისას
- გაუმჯობესდა: დაემატა check-env npm script, რათა developer გარემოებმა პირველ გაშვებაზე თვითკონფიგურაცია მოახდინონ
