# მხარდაჭერის აგენტები

მთელი ქსელის დამოუკიდებლად მართვა მარტივი ამოცანა არ არის. როდესაც თქვენი ქსელი ზრდას დაიწყებს, მხარდაჭერაზე მზარდ მოთხოვნასაც შეამჩნევთ. კარგი იდეა იქნება ისეთი ადამიანების ჩართვა, რომლებიც სამუშაო დატვირთვის შემსუბუქებაში დაგეხმარებიან. თუმცა, სრული წვდომის/ნებართვის მიცემა სარისკოა და შეიძლება თქვენი ქსელი საფრთხის ქვეშ დააყენოს. სწორედ აქ გამოგადგებათ ეს მხარდაჭერის აგენტების დამატება.

ეს დამატება საშუალებას გაძლევთ შექმნათ WordPress მომხმარებლის ახალი ტიპი, სახელწოდებით მხარდაჭერის აგენტი, და სრულ კონტროლს გაძლევთ იმაზე, თუ რა ნებართვები ექნება ამ მომხმარებელს. ნებართვის მინიჭება ისეთივე მარტივია, როგორც გადამრთველი ღილაკის ჩართვა, ხოლო თუ მის გაუქმებას გადაწყვეტთ — მისი გამორთვა.

## ფუნქციები {#features}

- WordPress და Ultimate Multisite ნებართვების დეტალური სია
- წვდომის მარტივად ჩართვა და გამორთვა
- აგენტის ანგარიშების შეუზღუდავი რაოდენობის შექმნა

## ინსტალაცია {#installation}

1. ატვირთეთ 'ultimate-multisite-support-agents' '/wp-content/plugins/' დირექტორიაში
2. გაააქტიურეთ პლაგინი WordPress-ის 'Plugins' მენიუდან
3. მიჰყევით ნაბიჯ-ნაბიჯ ოსტატს პლაგინის დასაყენებლად
