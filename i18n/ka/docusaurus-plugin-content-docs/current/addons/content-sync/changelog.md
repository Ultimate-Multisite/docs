# Content Sync ცვლილებების ჟურნალი

### ვერსია 1.1.0 - გამოვიდა 2025-12-31
* ახალი: ყველა ჩანაწერის ტიპის სინქრონიზაციის მხარდაჭერა (ჩანაწერები, გვერდები, მორგებული ჩანაწერის ტიპები)
* ახალი: ჩანაწერის ტიპის ფილტრაცია შიგთავსის არჩევის ეტაპზე
* ახალი: წყარო საიტის არჩევის ეტაპი - სინქრონიზაცია ნებისმიერი საიტიდან, არა მხოლოდ მთავარი საიტიდან
* ახალი: შვილობილი საიტის წინასწარი არჩევა შაბლონური საიტებისთვის
* ახალი: მედიის ამოღება post_content-იდან ყველა ჩანაწერის ტიპისთვის
* ახალი: ტაქსონომიის სინქრონიზაცია (კატეგორიები, ტეგები, მორგებული ტაქსონომიები)
* შეცვლილია: სახელი შეიცვალა "Elementor Template Sync"-დან "Content Sync"-ად
* შეცვლილია: "Select Templates" ეტაპს დაერქვა "Select Data"
* გაუმჯობესებულია: Elementor-სპეციფიკური კოდი ახლა მხოლოდ elementor_library ჩანაწერის ტიპისთვის ეშვება

### ვერსია 1.0.0 - გამოვიდა 2025-12-19
* საწყისი გამოშვება
* მრავალსაფეხურიანი ოსტატი შაბლონისა და საიტის ასარჩევად
* პაკეტური დამუშავება რეალურ დროში პროგრესის უკუკავშირით
* მედიის სრული სინქრონიზაცია ID-ების ავტომატური ხელახლა მიბმით
* პარამეტრების ინტეგრაცია Ultimate Multisite-თან
* საიტის გამორიცხვის მხარდაჭერა ქსელის პარამეტრების მეშვეობით
* Elementor ქეშის გასუფთავება სინქრონიზაციის შემდეგ
* შაბლონის ტაქსონომიის შენარჩუნება
