# შიგთავსის სინქრონიზაცია {#content-sync}

Multisite Ultimate: შიგთავსის სინქრონიზაცია

ეს დანამატი Multisite Ultimate-სთვის ქსელის ადმინისტრატორებს საშუალებას აძლევს, ნებისმიერი საიტიდან შიგთავსი ქსელის სხვა საიტებზე დაასინქრონონ. იგი მხარს უჭერს ყველა ჩანაწერის ტიპს, მათ შორის ჩანაწერებს, გვერდებს, მორგებულ ჩანაწერის ტიპებს და Elementor-ის შაბლონებს. დანამატი მოიცავს მრავალსაფეხურიან ოსტატის ინტერფეისს შიგთავსისა და სამიზნე საიტების ასარჩევად, პაკეტური დამუშავებითა და პროგრესის რეალურ დროში უკუკავშირით.

## ფუნქციები {#features}

* შიგთავსის არჩევა: ვიზუალური ბარათების ბადე მინიატურებითა და ჩანაწერის ტიპის ფილტრაციით
* ყველა ჩანაწერის ტიპი: ასინქრონებს ჩანაწერებს, გვერდებს, პროდუქტებს, Elementor-ის შაბლონებს და ნებისმიერ მორგებულ ჩანაწერის ტიპს
* საიტის არჩევა: აირჩიეთ, რომელ საიტებზე დასინქრონდეს შიგთავსი, წყარო საიტის გამორიცხვითა და შვილობილი საიტების წინასწარი არჩევით
* პაკეტური დამუშავება: ერთ ჯერზე ასინქრონებს 10 საიტზე, პროგრესის განახლებებით
* მედიის სინქრონიზაცია: ავტომატურად აკოპირებს და ხელახლა აბამს მედია ფაილებს (სურათებს, ვიდეოებს) ჩანაწერის შიგთავსიდან სამიზნე საიტებზე
* ტაქსონომიის სინქრონიზაცია: ინარჩუნებს კატეგორიებს, ჭდეებს და მორგებულ ტაქსონომიებს
* Elementor-ის მხარდაჭერა: Elementor-ის შაბლონების სრული მხარდაჭერა CSS-ის ხელახლა გენერაციითა და შაბლონის ID-ების ხელახლა მიბმით
* კონფლიქტების გადაჭრა: არსებული შიგთავსის ჩანაცვლების ან გამოტოვების ვარიანტი
* ქეშის მართვა: სინქრონიზაციის შემდეგ ავტომატურად ასუფთავებს Elementor-ის ქეშს სამიზნე საიტებზე

## მოთხოვნები {#requirements}

* WordPress 5.3 ან უფრო ახალი
* PHP 7.4 ან უფრო ახალი
* Multisite Ultimate პლაგინი (აქტიური)
* Elementor პლაგინი (არასავალდებულო, საჭიროა Elementor-ის შაბლონების სინქრონიზაციისთვის)

## ინსტალაცია {#installation}

1. ატვირთეთ 'ultimate-multisite-content-sync' '/wp-content/plugins/' დირექტორიაში
2. გაააქტიურეთ პლაგინი 'პლაგინების' მენიუდან თქვენს WordPress Network Admin-ში
3. დააკონფიგურირეთ პარამეტრები Ultimate Multisite → პარამეტრები → შიგთავსის სინქრონიზაცია
4. გადადით Ultimate Multisite → შიგთავსის სინქრონიზაცია სინქრონიზაციის დასაწყებად
