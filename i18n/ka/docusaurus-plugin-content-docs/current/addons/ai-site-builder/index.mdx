# AI საიტის შემქმნელი

ultimate-multisite-ai-site-builder

## მოთხოვნები

- WordPress 5.3 ან უფრო ახალი
- PHP 7.4 ან უფრო ახალი
- Multisite Ultimate პლაგინი (აქტიური)

## ინსტალაცია

1. ატვირთეთ დანამატის ფაილები თქვენს `/wp-content/plugins/` დირექტორიაში
2. გაააქტიურეთ პლაგინი WordPress-ში „პლაგინები“ მენიუს მეშვეობით
3. დააკონფიგურირეთ დანამატის პარამეტრები ქსელის ადმინისტრირების ნაწილში

## განვითარება

### დაყენება

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### პროექტის სტრუქტურა

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP classes and includes
│   ├── class-multisite-ultimate-updater.php  # Update handler
│   └── class-ai_site_builder-main.php             # Main functionality
├── tests/                        # Unit tests
│   ├── bootstrap.php            # Test bootstrap
│   ├── class-*-base.php         # Base test class
│   └── class-*-test.php         # Test cases

│   ├── css/                     # Stylesheets

```

### ტესტირება

დანამატი მოიცავს ყოვლისმომცველ ტესტირების ფრეიმვორკს:

- **ერთეულური ტესტები**: PHPUnit-ზე დაფუძნებული ტესტები WordPress-ის სატესტო ნაკრებით
- **საბაზისო ტესტის კლასი**: უზრუნველყოფს დამხმარე მეთოდებს საერთო სატესტო სცენარებისთვის
- **ტესტების დაფარვა**: კონფიგურირებულია კოდის დაფარვის თვალყურის სადევნებლად
- **მრავალსაიტიანი ტესტირება**: ტესტები ეშვება მრავალსაიტიან გარემოში

### კოდის სტანდარტები

- **WordPress-ის კოდირების სტანდარტები**: სავალდებულოდ მოწმდება PHPCS-ის მეშვეობით
- **PHP 7.4+ თავსებადობა**: მხარდაჭერილია თანამედროვე PHP ფუნქციები
- **სტატიკური ანალიზი**: PHPStan ინტეგრაცია ტიპების შესამოწმებლად
- **კოდის მოდერნიზაცია**: Rector წესები PHP განახლებებისთვის

### აგების სისტემა

- **აქტივების დამუშავება**: CSS/JS ფაილების მინიფიკაცია
- **თარგმნა**: POT ფაილის გენერაცია
- **შეფუთვა**: არქივის შექმნა გავრცელებისთვის
- **განვითარების რეჟიმი**: არამინიფიცირებული აქტივები გამართვისთვის

## კონფიგურაცია

დანამატი მხარს უჭერს სხვადასხვა კონფიგურაციის ვარიანტს, რომლებიც ხელმისაწვდომია ქსელის ადმინისტრირების ინტერფეისიდან.

## ჰუკები და ფილტრები

### მოქმედებები

- `ultimate-multisite-ai-site-builder_init` - ეშვება დანამატის ინიციალიზაციის შემდეგ
- `ultimate-multisite-ai-site-builder_loaded` - ეშვება ყველა დანამატის ფაილის ჩატვირთვის შემდეგ

### ფილტრები

- `ultimate-multisite-ai-site-builder_settings` - დანამატის პარამეტრების შეცვლა
- `ultimate-multisite-ai-site-builder_enabled` - დანამატის ჩართვის/გამორთვის სტატუსის გადაფარვა
