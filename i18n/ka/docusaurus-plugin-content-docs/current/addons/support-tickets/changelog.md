### 1.0.3 - 2026-05-01
* შესწორდა: დასაშვებია null მნიშვნელობები nullable ბილეთის მოდელის setters-ში
* შესწორდა: ყველა ბილეთი ინახება გლობალურ ქსელის მასშტაბის ცხრილებში, სათანადო multisite მხარდაჭერისთვის
* შესწორდა: დამალულია პერსონალის ველები და გასწორებულია ცარიელ-გვერდზე გადამისამართება მომხმარებლის ახალი ბილეთის ფორმაში
* შესწორდა: განუსაზღვრელი add_meta() გამოძახებები ჩანაცვლებულია სწორი update_meta()-ით მეტამონაცემების შესანახად
* შესწორდა: დაურეგისტრირებელი შესაძლებლობის შემოწმება ჩანაცვლებულია სწორი wu_view_all_support_tickets-ით
* შესწორდა: დაემატა გამოტოვებული AJAX handlers ბილეთის სტატუსის, მინიჭებისა და სწრაფი რედაქტირების მოქმედებებისთვის
* შესწორდა: გასწორებულია მეთოდის სახელი ბილეთის ხედებში პერსონალის პასუხის ამოცნობისთვის
* შესწორდა: გაერთიანებულია დუბლირებული პასუხის handlers და გასწორებულია nonce მოქმედებების სახელები
* შესწორდა: დაემატა გამოტოვებული frontend ხედი [wu_submit_ticket] shortcode-ისთვის
* შესწორდა: დაემატა გამოტოვებული user_id სვეტი და გასწორებულია Support_Ticket::get_user_id() მეთოდი
* შესწორდა: ამოღებულია დამატებითი ორმაგი ბრჭყალი პრიორიტეტის ფილტრის option ტეგში
* შესწორდა: დაემატა ქსელის ადმინისტრატორის ბილეთების მართვის პანელი super admins-ისთვის
* გაუმჯობესდა: ადმინისტრატორის CSS გაერთიანებულია ერთ გარე სტილის ფურცლად
* გაუმჯობესდა: Settings ქვემენიუ ამოღებულია ქვე-საიტის ადმინისტრატორის მენიუდან
* გაუმჯობესდა: frontend რესურსები პირობითად იტვირთება მხოლოდ მხარდაჭერის ბილეთების გვერდებზე
* გაუმჯობესდა: plugin autoloader გამოტოვებულია, როდესაც Bedrock root autoloader-ს დამოკიდებულებები უკვე ჩატვირთული აქვს

* გასწორდა: სინტაქსის შეცდომები მხარდაჭერის ბილეთის ფუნქციებში
