# მხარდაჭერის ბილეთები

**მხარდაჭერის ბილეთები** არის მძლავრი addon Ultimate Multisite-ისთვის, რომელიც თქვენს WordPress ქსელს ამატებს სრულყოფილ მხარდაჭერის ბილეთების სისტემას. იდეალურია SaaS პლატფორმებისთვის, WaaS (Website as a Service) მომწოდებლებისთვის და multisite ქსელებისთვის, რომლებსაც სჭირდებათ მომხმარებელთა მხარდაჭერის უზრუნველყოფა როგორც ქსელის, ისე ინდივიდუალური საიტის დონეზე.

## რატომ აირჩიოთ მხარდაჭერის ბილეთები? {#why-choose-support-tickets}

* **მრავალდონიანი მხარდაჭერა**: ქსელის ადმინისტრატორები ამუშავებენ მთელ ქსელზე გავრცელებულ ბილეთებს, ხოლო საიტის მფლობელები მართავენ თავიანთ მომხმარებელთა მხარდაჭერას
* **პროფესიონალური სამუშაო პროცესი**: ბილეთის სრული სასიცოცხლო ციკლის მართვა პრიორიტეტებით, სტატუსებითა და მინიჭებებით
* **მომხმარებლის გამოცდილება**: ლამაზი, ადაპტირებადი ინტერფეისი, რომელიც მომხმარებლებს დახმარების მიღებას უადვილებს
* **უსაფრთხო და საიმედო**: შექმნილია WordPress-ის უსაფრთხოების საუკეთესო პრაქტიკებით და ოპტიმიზებული მონაცემთა ბაზის წარმადობით
* **უწყვეტი ინტეგრაცია**: იდეალურად მუშაობს Ultimate Multisite-ის მომხმარებელთა და საიტების მართვასთან

## ძირითადი ფუნქციები {#key-features}

**ბილეთების სრული მართვა**

* ბილეთების უნიკალური ნუმერაციის სისტემა მარტივი მითითებისთვის
* კონფიგურირებადი პრიორიტეტის დონეები (დაბალი, ნორმალური, მაღალი, კრიტიკული)
* ყოვლისმომცველი სტატუსების სამუშაო პროცესი (ღია, მიმდინარეობს, მოლოდინში, გადაწყვეტილი, დახურული)
* ბილეთების კატეგორიზაცია (ზოგადი, ტექნიკური, გადახდები, ფუნქციის მოთხოვნა, შეცდომის ანგარიში)
* ბილეთების მინიჭება მხარდაჭერის აგენტებზე სურვილისამებრ ავტომატური მინიჭებით

**თემებად დალაგებული საუბრები**

* სრული საუბრის თემებად დალაგება ბილეთის პასუხებისთვის
* განსხვავებული სტილიზაცია თანამშრომლებისა და მომხმარებლების პასუხებისთვის
* თანამშრომლების პირადი შენიშვნები, რომლებსაც მომხმარებლები ვერ ხედავენ
* რეალურ დროში AJAX-ზე დაფუძნებული ინტერფეისი
* თანამედროვე ტექსტური რედაქტორი ფორმატირების ვარიანტებით

**ფაილების დანართები**

* უსაფრთხო ფაილის ატვირთვა drag-and-drop მხარდაჭერით
* კონფიგურირებადი ფაილის ტიპისა და ზომის შეზღუდვები
* გაუმჯობესებული უსაფრთხოება მავნე ფაილების აღმოჩენით
* დაცული ჩამოტვირთვები წვდომის კონტროლებით

**ჭკვიანი შეტყობინებები**

* ავტომატური ელფოსტის შეტყობინებები ახალი ბილეთებისთვის, პასუხებისთვის და სტატუსის ცვლილებებისთვის
* მორგებადი ელფოსტის შაბლონები
* როლებზე დაფუძნებული შეტყობინებები თანამშრომლებისა და მომხმარებლებისთვის
* დინამიკური შაბლონის ცვლადები პერსონალიზებული შეტყობინებებისთვის

**მძლავრი ადმინისტრატორის ინტერფეისი**

* გაუმჯობესებული ბილეთების სია დალაგებითა და გაფილტვრით
* სწრაფი რედაქტირება სტატუსისთვის, პრიორიტეტისთვის და მინიჭებებისთვის
* მასობრივი მოქმედებები ეფექტური მართვისთვის
* სტატისტიკის Dashboard წარმადობის მეტრიკებით
* ყოვლისმომცველი პარამეტრების პანელი

**ლამაზი მომხმარებლის პორტალი**

* სუფთა, ადაპტირებადი ინტერფეისი ყველა მოწყობილობაზე
* მობილურზე ორიენტირებული დიზაინი
* გაუმჯობესებული ძიება და გაფილტვრა
* თანამედროვე რედაქტორი ფორმატირების ხელსაწყოთა ზოლით
* ადვილად გამოსაყენებელი ბილეთის გაგზავნის ფორმები

**მოქნილი ინტეგრაცია**

* გამოიყენეთ shortcodes ან Ultimate Multisite UI ელემენტები
* აჩვენეთ ბილეთის ფორმები და სიები ნებისმიერ ადგილას თქვენს საიტზე
* შეუფერხებლად მუშაობს multisite ქსელის სტრუქტურასთან
* გაფართოებადია hooks-ებითა და filters-ებით

## იდეალურია შემდეგისთვის {#perfect-for}

* SaaS პლატფორმები, რომლებსაც მომხმარებელთა მხარდაჭერა სჭირდებათ
* WaaS მომწოდებლები, რომლებიც მართავენ მრავალი კლიენტის საიტს
* წევრობის საიტები დონეებად დაყოფილი მხარდაჭერით
* ქსელის ადმინისტრატორები, რომლებიც მართავენ მრავალ ბრენდს
* საიტის მფლობელები, რომლებიც მომხმარებელთა მომსახურებას უზრუნველყოფენ

## ინსტალაცია {#installation}

1. დარწმუნდით, რომ Ultimate Multisite plugin დაყენებული და გააქტიურებულია თქვენს WordPress ქსელზე
2. ატვირთეთ addon-ის ფაილები თქვენს `/wp-content/plugins/` დირექტორიაში
3. გაააქტიურეთ plugin ქსელის დონეზე WordPress-ის 'Plugins' მენიუდან
4. გადადით Network Admin > Support Tickets-ზე თქვენი პარამეტრების კონფიგურაციისთვის

## ხშირად დასმული კითხვები {#frequently-asked-questions}

## მოითხოვს თუ არა ეს Ultimate Multisite-ს? {#does-this-require-ultimate-multisite}

დიახ, ამ addon-ს სჭირდება Ultimate Multisite-ის დაყენება და აქტიური მდგომარეობა. ის ღრმად ინტეგრირდება Ultimate Multisite-ის მომხმარებელთა და საიტების მართვის ფუნქციებთან.

## შეუძლიათ თუ არა საიტის მფლობელებს საკუთარი მომხმარებელთა მხარდაჭერის მართვა? {#can-site-owners-handle-their-own-customer-support}

რა თქმა უნდა! საიტის მფლობელებს შეუძლიათ გამოიყენონ ბილეთების სისტემა თავიანთი მომხმარებლებისთვის მხარდაჭერის უზრუნველსაყოფად, ხოლო ქსელის ადმინისტრატორებს შეუძლიათ მართონ მთელ ქსელზე გავრცელებული მხარდაჭერის ბილეთები.

## როგორ აგზავნიან მომხმარებლები ბილეთებს? {#how-do-customers-submit-tickets}

თქვენ შეგიძლიათ დაამატოთ ბილეთის გაგზავნის ფორმები თქვენს საიტზე ნებისმიერ ადგილას shortcodes-ის ან Ultimate Multisite UI ელემენტების გამოყენებით. მომხმარებლები უბრალოდ ავსებენ ფორმას და აგზავნიან თავიანთ მოთხოვნას.

## შემიძლია თუ არა ბილეთის სტატუსებისა და პრიორიტეტების მორგება? {#can-i-customize-the-ticket-statuses-and-priorities}

დიახ! შეგიძლიათ სრულად მოარგოთ პრიორიტეტის დონეები, სტატუსები და ბილეთის ტიპები ქსელის ადმინისტრატორის პარამეტრებში, რათა შეესაბამებოდეს თქვენს მხარდაჭერის სამუშაო პროცესს.

## უჭერს თუ არა მხარს ფაილების დანართებს? {#does-it-support-file-attachments}

დიახ, როგორც მომხმარებლებს, ისე თანამშრომლებს შეუძლიათ ფაილების მიბმა ბილეთებსა და პასუხებზე. უსაფრთხოებისთვის თქვენ გაქვთ სრული კონტროლი დაშვებულ ფაილის ტიპებსა და ზომის ლიმიტებზე.

## როგორ იმართება შეტყობინებები? {#how-are-notifications-handled}

სისტემა ავტომატურად აგზავნის ელფოსტის შეტყობინებებს ახალი ბილეთებისთვის, პასუხებისთვის და სტატუსის ცვლილებებისთვის. შეგიძლიათ პარამეტრებში მოარგოთ ყველა ელფოსტის შაბლონი.

## შემიძლია თუ არა ბილეთების კონკრეტულ მხარდაჭერის აგენტებზე მინიჭება? {#can-i-assign-tickets-to-specific-support-agents}

დიახ, შეგიძლიათ ხელით მიანიჭოთ ბილეთები აგენტებს ან ჩართოთ ავტომატური მინიჭება ბილეთების ავტომატურად გასანაწილებლად.

## არსებობს თუ არა პირადი შენიშვნები თანამშრომლებისთვის? {#are-there-private-notes-for-staff}

დიახ, თანამშრომლებს შეუძლიათ დაამატონ პირადი პასუხები, რომლებიც ხილულია მხოლოდ სხვა თანამშრომლებისთვის და არა მომხმარებლებისთვის.

## არის თუ არა მობილურზე მოსახერხებელი? {#is-it-mobile-friendly}

რა თქმა უნდა! მთელი ინტერფეისი შექმნილია მობილურზე ორიენტირებული მიდგომით და შესანიშნავად მუშაობს ყველა მოწყობილობაზე.

## შემიძლია თუ არა ბილეთების სიებისა და ფორმების ჩვენება shortcodes-ის გამოყენებით? {#can-i-display-ticket-lists-and-forms-using-shortcodes}

დიახ! ხელმისაწვდომი shortcodes:
* `[wu_support_tickets]` - მომხმარებლის ბილეთების სიის ჩვენება
* `[wu_support_ticket_form]` - ახალი ბილეთის გაგზავნის ფორმის ჩვენება
* `[wu_support_ticket_view ticket_id="123"]` - კონკრეტული ბილეთის ჩვენება

## რა ტიპის ფაილების ატვირთვა შეიძლება? {#what-file-types-can-be-uploaded}

დაშვებული ფაილის ტიპების კონფიგურაცია შეგიძლიათ პარამეტრებში. სისტემა მოიცავს გაუმჯობესებულ უსაფრთხოების სკანირებას მავნე ფაილებისგან დასაცავად.

## შემიძლია თუ არა ბილეთების მონაცემების ექსპორტი? {#can-i-export-ticket-data}

დიახ, addon მოიცავს ბილეთების მონაცემებისა და სტატისტიკის ექსპორტის შესაძლებლობებს.
