# Fluent Forms

ეს დამატება შესაძლებელს ხდის Fluent Forms-ისთვის ლიმიტების დამატებას Multisite Ultimate-ში პროდუქტებზე, წევრობებზე ან საიტებზე.
Fluent Forms იყენებს მორგებულ ცხრილებს მორგებული პოსტის ტიპების ნაცვლად, ამიტომ ჩვეულებრივი ლიმიტირების პარამეტრები Fluent Forms-თან არ მუშაობს.

## შესაძლებლობები {#features}

- დაამატეთ ლიმიტები Fluent Forms-ისთვის ან Conversational Fluent Forms-ისთვის
- მომხმარებლები იღებენ მეგობრულ შეტყობინებას, რომელიც მათ გეგმის განახლებისკენ მოუწოდებს, როცა ლიმიტს მიაღწევენ
