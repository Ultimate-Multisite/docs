# Gratis AI Agent

Gratis AI Agent არის ავტონომიური WordPress AI ასისტენტი, რომელსაც შეუძლია სრული WordPress საიტების დაგეგმვა, აგება და მართვა ბუნებრივი ენის ინსტრუქციებით. ის აჩენს **შესაძლებლობების** სტრუქტურირებულ ნაკრებს — ცალკეულ, ტესტირებად მოქმედებებს, რომელთა გამოძახებაც აგენტს შეუძლია — და მოიცავს ყველაფერს კონტენტის მართვიდან დიზაინის სისტემის მორგებამდე. ვერსია 1.9.0 ამატებს კონტენტის შექმნის შესაძლებლობებს, გვერდის ვიზუალურ მიმოხილვას კლიენტის მხარეს გადაღებული ეკრანის ანაბეჭდებით, ხუთ ჩაშენებულ აგენტის პროფილს და ფუნქციების გადამრთველებს წვდომის კონტროლისა და ბრენდინგისთვის.

## ძირითადი ფუნქციები {#key-features}

- **შესაძლებლობების სისტემა** — მოქმედებების მოდულური, გაფართოებადი კატალოგი, რომელთა გამოძახებაც აგენტს შეუძლია თქვენს WordPress ინსტალაციაზე
- **კონტენტის მართვა** — შექმენით ჩანაწერები და გვერდები (გვერდის შაბლონების მინიჭების ჩათვლით), ერთ გამოძახებაში პაკეტურად შექმენით რამდენიმე ჩანაწერი, დააყენეთ გამორჩეული სურათები და ააგეთ საკონტაქტო ფორმები
- **ვიზუალური მიმოხილვა** — გადაიღეთ ცოცხალი გვერდების ეკრანის ანაბეჭდები, შეადარეთ ცვლილებამდე/ცვლილების შემდეგ მდგომარეობები და მიიღეთ AI-ის მიერ გენერირებული დიზაინის მიმოხილვა, რომელიც მოიცავს განლაგებას, ტიპოგრაფიას, ფერსა და ხელმისაწვდომობას
- **ხუთი ჩაშენებული აგენტი** — Content Writer, Site Builder, Design Studio, Plugin Manager და Support Assistant; გადართვადია ჩატის სათაურში მდებარე Agent Picker-ის საშუალებით
- **ფუნქციების გადამრთველები** — წვდომის კონტროლისა და ბრენდინგის გადამრთველები Settings → Feature Flags-ში როლების შეზღუდვებისთვის, თეთრი ეტიკეტის სახელდებისთვის და ვიჯეტის მორგებისთვის
- **მორგებული ჩანაწერის ტიპების მართვა** — დაარეგისტრირეთ, ჩამოთვალეთ და წაშალეთ მორგებული ჩანაწერის ტიპები, გადატვირთვებს შორის შენარჩუნებით
- **მორგებული ტაქსონომიის მართვა** — დაარეგისტრირეთ, ჩამოთვალეთ და წაშალეთ მორგებული ტაქსონომიები, შენარჩუნებით
- **დიზაინის სისტემის შესაძლებლობები** — ჩასვით მორგებული CSS, მართეთ შერჩეული ბლოკის პატერნები, დააყენეთ საიტის ლოგო და გამოიყენეთ theme.json პრესეტები
- **გლობალური სტილების მართვა** — წაიკითხეთ და ჩაწერეთ theme.json მნიშვნელობები ფერთა პალიტრებისთვის, ტიპოგრაფიისთვის, დაშორებებისთვის და სხვა
- **ნავიგაციის მენიუს მართვა** — პროგრამულად შექმენით და განაახლეთ ნავიგაციის მენიუები და მათი ელემენტები
- **პარამეტრების მართვა** — წაიკითხეთ და ჩაწერეთ WordPress პარამეტრები ჩაშენებული უსაფრთხოების ბლოკირების სიით, კრიტიკული პარამეტრების დასაცავად
- **დასაინსტალირებელი შესაძლებლობების რეესტრი** — აღმოაჩინეთ და გაააქტიურეთ დამატებითი შესაძლებლობების პაკეტები ძირითადი plugin-ის ფაილების შეცვლის გარეშე
- **Site Builder ორკესტრაცია v2** — მრავალსაფეხურიანი საიტის აგების გეგმები plugin-ის აღმოჩენით, პროგრესის თვალყურის დევნებით და შეცდომების აღდგენით
- **WP-CLI ბენჩმარკის ბრძანება** — გაუშვით აგენტის შესაძლებლობების ბენჩმარკის ნაკრები ბრძანების სტრიქონიდან CI-სა და მოდელის შეფასებისთვის

## მოთხოვნები {#requirements}

- WordPress 6.0 ან უფრო მაღალი
- PHP 8.1 ან უფრო მაღალი
- WordPress Multisite (ქსელურად გააქტიურებული) ან ერთსაიტიანი ინსტალაცია
- აქტიური AI პროვაიდერის დამაკავშირებელი plugin (Anthropic, OpenAI ან თავსებადი)

## ინსტალაცია {#installation}

1. ატვირთეთ plugin-ის ფაილები `/wp-content/plugins/gratis-ai-agent`-ში
2. გაააქტიურეთ plugin **Plugins** ეკრანიდან WordPress-ში
3. გადადით **Gratis AI Agent → Settings**-ზე და დააკავშირეთ თქვენი AI პროვაიდერი
4. (არასავალდებულო) დაათვალიერეთ **შესაძლებლობების რეესტრი** დამატებითი შესაძლებლობების პაკეტების დასაინსტალირებლად

## სწრაფი დაწყება {#quick-start}

გააქტიურებისა და AI პროვაიდერთან დაკავშირების შემდეგ, გახსენით **Gratis AI Agent** პანელი WordPress-ის ადმინისტრატორის გვერდითა ზოლში. აკრიფეთ ბუნებრივი ენის ინსტრუქცია და აგენტი დაგეგმავს და შეასრულებს საჭირო ნაბიჯებს თავისი ხელმისაწვდომი შესაძლებლობების გამოყენებით.

**ინსტრუქციების მაგალითები:**

```
შექმენი "Portfolio" მორგებული ჩანაწერის ტიპი "Project Category" ტაქსონომიით.
```

```
ჩასვი მუქი რეჟიმის ფერთა პალიტრა თემაში და დააყენე ძირითადი ფერი #1a1a2e-ზე.
```

```
ააგე რესტორნის ვებსაიტი ონლაინ მენიუთი, დაჯავშნის ფორმით და საკონტაქტო გვერდით.
```

## შესაძლებლობების მიმოხილვა {#abilities-overview}

შესაძლებლობები არის ატომური მოქმედებები, რომელთა შესრულებაც აგენტს შეუძლია. თითოეული შესაძლებლობა არის რეგისტრირებული PHP კლასი, რომელიც აჩენს სქემას (პარამეტრებსა და დაბრუნების ტიპებს), რომელსაც აგენტი გაშვების დროს კითხულობს. შესაძლებლობები დაჯგუფებულია ფუნქციურ სფეროებად:

| სფერო | შესაძლებლობები |
|---|---|
| **კონტენტის მართვა** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **ვიზუალური მიმოხილვა** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **მორგებული ჩანაწერის ტიპები** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **მორგებული ტაქსონომიები** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **დიზაინის სისტემა** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **გლობალური სტილები** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **ნავიგაციის მენიუები** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **პარამეტრების მართვა** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **დასაინსტალირებელი შესაძლებლობები** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

სრული პარამეტრების დოკუმენტაციისა და გამოყენების მაგალითებისთვის იხილეთ [შესაძლებლობების ცნობარი](./abilities).

## WP-CLI {#wp-cli}

Gratis AI Agent-ს მოჰყვება `wp gratis-ai-agent` ბრძანებების ოჯახი ბენჩმარკების გაშვებისა და შესაძლებლობების მართვისთვის ბრძანების სტრიქონიდან. ბრძანებების სრული სიისთვის იხილეთ [WP-CLI ცნობარი](./wp-cli).

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## კონფიგურაცია {#configuration}

ყველა პარამეტრი მდებარეობს **Gratis AI Agent → Settings**-ში WordPress-ის ადმინისტრაციაში:

| პარამეტრი | აღწერა |
|---|---|
| **AI პროვაიდერი** | აირჩიეთ და დააკონფიგურირეთ აქტიური AI პროვაიდერის დამაკავშირებელი |
| **მოდელი** | აირჩიეთ ენობრივი მოდელი აგენტის სესიებისთვის გამოსაყენებლად |
| **მაქსიმალური ტოკენები** | ტოკენების ბიუჯეტი თითო მოთხოვნაზე (ნაგულისხმევი 4096) |
| **შესაძლებლობის დროის ლიმიტი** | მაქსიმალური წამები, რამდენ ხანსაც ინდივიდუალური შესაძლებლობა შეიძლება მუშაობდეს (ნაგულისხმევი 30) |
| **გამართვის ლოგირება** | ჩაწერეთ შესაძლებლობების გამოძახებები და აგენტის მსჯელობა WordPress-ის გამართვის ლოგში |

## პარამეტრების მართვის უსაფრთხოების ბლოკირების სია {#options-management-safety-blocklist}

Options Management შესაძლებლობა მოიცავს ჩაშენებულ ბლოკირების სიას, რომელიც აგენტს სენსიტიური WordPress პარამეტრების წაკითხვას ან ჩაწერას უშლის ხელს. ნაგულისხმევი ბლოკირების სია მოიცავს:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL პარამეტრები — ამისთვის გამოიყენეთ სპეციალური მიგრაციის ინსტრუმენტი)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (საიდუმლო გასაღებები)
- `active_plugins`, `template` (plugin/theme აქტივაცია — გამოიყენეთ სპეციალური შესაძლებლობები)
- გადახდის კარიბჭის ავტორიზაციის მონაცემები და API გასაღებები, რომლებიც რეგისტრირებულია დაყენებული plugin-ების მიერ

დამატებითი ჩანაწერების დამატება შესაძლებელია `gratis_ai_agent_options_blocklist` ფილტრის მეშვეობით.

## Hooks და Filters {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — ამოქმედდება შესაძლებლობის რეგისტრაციის შემდეგ, იღებს შესაძლებლობის კლასის სახელს
- `gratis_ai_agent_plan_started` — ამოქმედდება, როდესაც საიტის შემქმნელი გეგმა იწყებს შესრულებას
- `gratis_ai_agent_plan_completed` — ამოქმედდება, როდესაც გეგმა წარმატებით სრულდება
- `gratis_ai_agent_plan_failed` — ამოქმედდება, როდესაც გეგმას გამოუსწორებელი შეცდომა შეხვდება

### Filters {#filters}

- `gratis_ai_agent_abilities` — შეცვალეთ რეგისტრირებული შესაძლებლობების სია, სანამ აგენტი მას ჩატვირთავს
- `gratis_ai_agent_options_blocklist` — გააფართოეთ ან შეამცირეთ პარამეტრების უსაფრთხოების ბლოკირების სია
- `gratis_ai_agent_benchmark_questions` — დაამატეთ მორგებული საორიენტაციო კითხვები შეფასების ნაკრებს
- `gratis_ai_agent_plan_plugins` — შეცვალეთ plugin-ების აღმოჩენის სია საიტის გეგმის გენერირებისას
