# VAT

Multisite Ultimate: ევროპული VAT

Multisite Ultimate-ის ნაგულისხმევი გადასახადების დამუშავებას ამატებს დამატებით ფუნქციებსა და ინსტრუმენტებს, რათა უკეთ უზრუნველყოს ევროპული VAT-ის მხარდაჭერა.
ავტომატურად ამატებს EU VAT განაკვეთებს გადასახადის განაკვეთების გვერდზე

## ინსტალაცია {#installation}

1. ატვირთეთ 'ultimate-multisite-vat' '/wp-content/plugins/' დირექტორიაში
2. გაააქტიურეთ plugin 'პლაგინები' მენიუდან WordPress-ში
3. გადადით Ultimate Multisite > პარამეტრები > გადასახადები და ჩართეთ EU VAT.
4. დააწკაპუნეთ გადასახადის განაკვეთებზე, შემდეგ ქვემოთ განაახლეთ EU VAT განაკვეთები და შეინახეთ გადასახადის განაკვეთები.
5. მზადაა!
