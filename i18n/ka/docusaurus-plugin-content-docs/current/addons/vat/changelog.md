# დღგ-ის ცვლილებების ჟურნალი

ვერსია 1.0.7 - გამოშვებულია 2026-02-03

* შესწორება: გერმანული (DE) დღგ-ის ნომრები ვერ გადიოდა ვალიდაციას VIES SOAP სერვისის MS_MAX_CONCURRENT_REQ შეცდომების გამო. ძველი SOAP API-დან გადავედით ოფიციალურ EU VIES REST API-ზე უფრო საიმედო ვალიდაციისთვის.
* შესწორება: დღგ-ის ნომრები ახლა მიიღება, როცა VIES სერვისი დროებით მიუწვდომელია, ნაცვლად მათი არასწორად უარყოფისა. შემდგომი გადახედვისთვის იქმნება ჟურნალის ჩანაწერი.
* შესწორება: VAT/Tax ID ველი ახლა ჩანს ყველა ქვეყნისთვის და არა მხოლოდ EU წევრებისთვის. არა-EU საგადასახადო ID-ები (მაგ. შვეიცარიული CHE ნომრები) ინახება ინვოისზე გამოსაჩენად VIES ვალიდაციის გარეშე.
* შესწორება: დღგ-ის პრეფიქსის მოძიება საბერძნეთისთვის (GR), მონაკოსთვის (MC) და მენის კუნძულისთვის (IM) არ მუშაობდა მასივში მოძიების არასწორი ლოგიკის გამო.
* გაუმჯობესება: წაიშალა არასაჭირო wp-cli-bundle განვითარების დამოკიდებულება, რომელსაც შეეძლო ფატალური შეცდომების გამოწვევა Site Exporter დამატებასთან ერთად გამოყენებისას.
* გაუმჯობესება: დაემატა ყოვლისმომცველი ერთეულური ტესტების ნაკრები (53 ტესტი).
