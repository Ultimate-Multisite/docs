### ვერსია 1.9.0 - გამოშვებულია 2026-01-18 {#version-190---released-on-2026-01-18}

* გამოსწორებულია: ფატალური შეცდომები აქტივაციისას.
* გადაკეთებულია: კონტენტის წყაროს კლასები Singleton trait-ის გამოსაყენებლად, მშობელი კონსტრუქტორის სათანადო ინიციალიზაციით
* გამოსწორებულია: Vue.js შაბლონის კომპილაციის შეცდომები, რომლებიც კონტენტის ტიპის სელექტორის გამოჩენას უშლიდა ხელს
* გამოსწორებულია: გამოტოვებული Vue მონაცემთა თვისებები (template_id, external_link_url_embedable, external_link_url_checking)
* გამოსწორებულია: TinyMCE რედაქტორი გვერდის ჩატვირთვისას სათანადოდ არ ინიციალიზდებოდა
* გამოსწორებულია: Divi მხარდაჭერის ფაილი Composer-ის მეშვეობით ავტომატურად არ იტვირთებოდა
* გამოსწორებულია: დაზიანებული PHP სინტაქსი Divi მხარდაჭერის კლასში
* გაუმჯობესებულია: JavaScript რესურსების ორგანიზაცია და მინიფიკაცია
* გაუმჯობესებულია: თავსებადობა უახლეს WordPress და გვერდების კონსტრუქტორის ვერსიებთან
