# Ultimate Multisite დოკუმენტაცია

მოგესალმებით Ultimate Multisite დოკუმენტაციაში. Ultimate Multisite გარდაქმნის WordPress Multisite-ს სრულ Website as a Service (WaaS) პლატფორმად.

## დოკუმენტაციის სექციები

### [მომხმარებლის სახელმძღვანელო](/user-guide/getting-started/ultimate-multisite-101)

ყველაფერი, რაც გჭირდებათ თქვენი WaaS პლატფორმის დასაყენებლად, კონფიგურაციისთვის და სამართავად. მოიცავს ინსტალაციას, გადახდის კარიბჭეებს, დომენის მიბმას, კლიენტების მართვას და სხვა საკითხებს.

### [დეველოპერის დოკუმენტაცია](/developer/hooks)

ტექნიკური ცნობარი დეველოპერებისთვის, რომლებიც Ultimate Multisite-ზე ქმნიან. მოიცავს hooks-ის ცნობარს, REST API დოკუმენტაციას და დამატებების განვითარების სახელმძღვანელოებს.

### [დამატებები](/addons)

დოკუმენტაცია ფასიანი დამატებითი plugins-ისთვის, რომლებიც Ultimate Multisite-ს აფართოებს დამატებითი ფუნქციებით, როგორიცაა WooCommerce ინტეგრაცია, ანალიტიკა, დომენების გაყიდვა და სხვა.
