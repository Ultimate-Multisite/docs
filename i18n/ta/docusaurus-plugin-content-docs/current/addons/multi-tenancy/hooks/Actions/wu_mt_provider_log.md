# Action: wu_mt_provider_log {#action-wumtproviderlog}

ஒரு பிரவைடர் (provider) ஒரு செய்தியைப் பதிவு செய்யும் போது இது இயங்கும்.

## அளவுருக்கள் {#parameters}

| Name | Type | விளக்கம் |
|------|------|-------------|
| $message | `string` | பதிவு செய்யப்பட்ட செய்தி. |
| $level | `string` | பதிவு நிலை. |
| $context | `array` | கூடுதல் சூழல்/விவரம். |

- 1.0.0
