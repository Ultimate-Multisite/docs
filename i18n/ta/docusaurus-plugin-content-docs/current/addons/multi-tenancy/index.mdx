# பல-குத்தகை {#multi-tenancy}

செயல்திறன் மற்றும் பாதுகாப்புக்காக துணைத் தளங்களை சேவையகங்களிலும் தரவுத்தளங்களிலும் பகிர்ந்திடுங்கள்

## Version 1.2.0 வெளியீட்டின் முக்கிய அம்சங்கள் {#version-120-release-highlights}

Multi-Tenancy 1.2.0, உற்பத்தி தனிமைப்படுத்தல் பணிச்சூழல்களுக்கான சுயாட்சி குத்தகையாளர் அடித்தளத்தைச் சேர்க்கிறது:

- குறுகிய ஆயுளுடைய, மீள்பயன்பாட்டிலிருந்து பாதுகாக்கப்பட்ட SSO வருகைகளுடன் நிலைச் சார்பற்ற குத்தகையாளர் தானியங்கி உள்நுழைவு.
- சுயாட்சி குத்தகையாளர்களுக்கான ஒவ்வொரு துணைத் தளத்திற்குமான தரவுத்தளம் மற்றும் கோப்பு அமைப்பு தனிமைப்படுத்தல்.
- host-அறிந்த குத்தகையாளர் வழங்கலுக்கான Hostinger multi-tenancy திறன்.
- பழைய தரவு சரிபார்ப்புகள் மற்றும் async push செயலாக்கத்திற்கான சுயாட்சி இடம்பெயர்வு சரிபார்ப்பு வாயில்கள்.
- Bedrock, FrankenPHP, மற்றும் localhost நிறுவல்களுக்கான அமைக்கக்கூடிய அதே-இயந்திர தரவுத்தள host இணைப்பு.
- குத்தகையாளர் bootstrap, தரவுத்தள writer, schema bootstrap, மற்றும் பயனர் வழங்கல் மேம்பாடுகள்.
- குத்தகையாளர் சான்றுகளைச் சுத்தப்படுத்தும் பாதுகாப்பான சுயாட்சி teardown.

[சுயாட்சி இடம்பெயர்வு பணிச்சூழல்](/user-guide/getting-started/sovereign-migration-workflow), [பல-குத்தகை தனிமைப்படுத்தல்](/user-guide/configuration/multi-tenancy-isolation), மற்றும் [சுயாட்சி குத்தகையாளர் செயல்பாடுகள்](/user-guide/administration/sovereign-tenant-operations) ஆகிய பயனர் வழிகாட்டிகளைப் பார்க்கவும்.

## தேவைகள் {#requirements}

- WordPress 5.3 அல்லது அதற்கு மேல்
- PHP 7.4 அல்லது அதற்கு மேல்
- Multisite Ultimate plugin (செயலில்)

## நிறுவல் {#installation}

1. addon கோப்புகளை உங்கள் `/wp-content/plugins/` அடைவில் பதிவேற்றவும்
2. WordPress இல் உள்ள 'Plugins' மெனு மூலம் plugin ஐச் செயல்படுத்தவும்
3. network admin இல் addon அமைப்புகளை உள்ளமைக்கவும்

### சோதனை {#testing}

இந்த addon முழுமையான சோதனை கட்டமைப்பை உள்ளடக்கியுள்ளது:

- **Unit Tests**: WordPress சோதனைத் தொகுப்புடன் PHPUnit-அடிப்படையிலான சோதனைகள்
- **அடிப்படை சோதனை வகுப்பு**: பொதுவான சோதனைச் சூழல்களுக்கான உதவி முறைகளை வழங்குகிறது
- **சோதனை Coverage**: code coverage ஐக் கண்காணிக்க உள்ளமைக்கப்பட்டது
- **Multisite சோதனை**: சோதனைகள் multisite சூழலில் இயங்கும்

### Code தரநிலைகள் {#code-standards}

- **WordPress Coding Standards**: PHPCS மூலம் அமல்படுத்தப்படுகிறது
- **PHP 7.4+ இணக்கத்தன்மை**: நவீன PHP அம்சங்கள் ஆதரிக்கப்படுகின்றன
- **நிலையான பகுப்பாய்வு**: வகைச் சரிபார்ப்பிற்கான PHPStan ஒருங்கிணைப்பு
- **Code நவீனப்படுத்தல்**: PHP மேம்படுத்தல்களுக்கான Rector விதிகள்

### உருவாக்க அமைப்பு {#build-system}

- **சொத்து செயலாக்கம்**: CSS/JS கோப்புகளுக்கான குறுக்கம்
- **மொழிபெயர்ப்பு**: POT கோப்பு உருவாக்கம்
- **பொதியிடல்**: விநியோகத்திற்கான காப்பக உருவாக்கம்
- **மேம்பாட்டு முறை**: பிழைத்திருத்தத்திற்கான குறுக்கப்படாத சொத்துகள்

## உள்ளமைவு {#configuration}

network admin இடைமுகம் மூலம் அணுகக்கூடிய பல்வேறு உள்ளமைவு விருப்பங்களை இந்த addon ஆதரிக்கிறது.

### சுயாட்சி குத்தகையாளர் உள்ளமைவு {#sovereign-tenant-configuration}

சுயாட்சி குத்தகையாளர்களுக்கு, போக்குவரத்தை குத்தகையாளருக்கு வழிமாற்றுவதற்கு முன் குத்தகையாளர் தரவுத்தளம், கோப்பு அமைப்பு root, தரவுத்தள host இணைப்பு, மற்றும் host திறனை உள்ளமைக்கவும். தரவுத்தள சான்றுகள், host இணைப்பு, routing, அல்லது பயனர் வழங்கலைப் பாதிக்கும் ஒவ்வொரு மாற்றத்திற்குப் பிறகும் இடம்பெயர்வு சரிபார்ப்பை இயக்கவும்.

குத்தகையாளர் runtime மற்றும் தரவுத்தள grants உள்ளூர் socket அல்லது அதே-இயந்திர அணுகலை எதிர்பார்க்கும்போது மட்டுமே `localhost` ஐப் பயன்படுத்தவும். உங்கள் provider உள்ளமைத்த grant host அதுவாக இருந்தால் `127.0.0.1` அல்லது ஒரு service hostname ஐப் பயன்படுத்தவும்.

### Hostinger திறன் {#hostinger-capability}

Hostinger multi-tenancy திறன் பகிரப்பட்ட Hostinger ஒருங்கிணைப்பு token உடன் செயல்படுகிறது. சரிபார்ப்புக்கு முன் domain hPanel இல் உள்ளதா, DNS Hostinger account ஐச் சுட்டுகிறதா, மற்றும் குத்தகையாளர் தரவுத்தள அமைப்புகள் hPanel உடன் பொருந்துகிறதா என்பதை உறுதிப்படுத்தவும்.

## Hooks மற்றும் Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-addon-template_init` - addon துவக்கத்திற்குப் பிறகு இயக்கப்படும்
- `ultimate-multisite-addon-template_loaded` - அனைத்து addon கோப்புகளும் ஏற்றப்பட்ட பிறகு இயக்கப்படும்

### Filters {#filters}

- `ultimate-multisite-addon-template_settings` - addon அமைப்புகளை மாற்றவும்
- `ultimate-multisite-addon-template_enabled` - addon enable/disable நிலையை மேலெழுதவும்
