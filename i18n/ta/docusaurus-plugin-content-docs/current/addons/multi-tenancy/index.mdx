# மல்டி-டென்டன்சி (Multi-Tenancy)

செயல்திறன் மற்றும் பாதுகாப்பிற்காக சப்-சைட்களை (subsites) பல சர்வர்கள் மற்றும் டேட்டாபேஸ்களில் பிரித்து நிர்வகித்தல்.

## தேவையானவை (Requirements)

- WordPress 5.3 அல்லது அதற்கு மேல்
- PHP 7.4 அல்லது அதற்கு மேல்
- Multisite Ultimate plugin (செயல்படுத்தப்பட்ட நிலையில்)

## நிறுவுதல் (Installation)

1. ஆட்-ஆன் கோப்புகளை உங்கள் `/wp-content/plugins/` டைரக்டரிக்கு பதிவேற்றவும்.
2. WordPress-ல் உள்ள 'Plugins' மெனு மூலம் பிளகினை செயல்படுத்துங்கள்.
3. நெட்வொர்க் நிர்வாகத்தில் (network admin) ஆட்-ஆன் அமைப்புகளை உள்ளமைக்கவும்.

## மேம்பாடு (Development)

### அமைவு (Setup)

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### ப்ராஜெக்ட் அமைப்பு (Project Structure)

```
ultimate-multisite-addon-template/
├── inc/                          # PHP classes and includes
│   ├── class-multisite-ultimate-updater.php  # Update handler
│   └── class-multi_tenancy-main.php             # Main functionality
├── tests/                        # Unit tests
│   ├── bootstrap.php            # Test bootstrap
│   ├── class-*-base.php         # Base test class
│   └── class-*-test.php         # Test cases
├── views/                       # Template files
├── assets/                      # Static assets
│   ├── css/                     # Stylesheets
│   ├── js/                      # JavaScript files
│   └── images/                  # Images
├── lang/                        # Language files
├── composer.json                # PHP dependencies
├── package.json                 # Node.js dependencies
├── phpunit.xml.dist             # PHPUnit configuration
├── .phpcs.xml.dist              # Code standards configuration
└── rector.php                   # Code modernization rules
```

### சோதனை செய்தல் (Testing)

இந்த ஆட்-ஆன் ஒரு விரிவான சோதனை கட்டமைப்பைக் (testing framework) கொண்டுள்ளது:

- **Unit Tests**: WordPress test suite உடன் PHPUnit அடிப்படையிலான சோதனைகள்.
- **Base Test Class**: பொதுவான சோதனை சூழ்நிலைகளுக்கான உதவி முறைகளை (helper methods) வழங்குகிறது.
- **Test Coverage**: குறியீட்டு கவரேஜை (code coverage) கண்காணிக்க அமைக்கப்பட்டுள்ளது.
- **Multisite Testing**: மல்டி-சைட் சூழலில் சோதனைகள் இயக்கப்படும்.

### குறியீட்டு தரநிலைகள் (Code Standards)

- **WordPress Coding Standards**: PHPCS மூலம் செயல்படுத்தப்படுகிறது.
- **PHP 7.4+ Compatibility**: நவீன PHP அம்சங்கள் ஆதரிக்கப்படுகின்றன.
- **Static Analysis**: டைப் செக்கிங்கிற்காக PHPStan ஒருங்கிணைப்பு.
- **Code Modernization**: PHP மேம்படுத்தல்களுக்கான Rector விதிகள்.

### உருவாக்க அமைப்பு (Build System)

- **Asset Processing**: CSS/JS கோப்புகளுக்கான மினிஃபிகேஷன் (Minification).
- **Translation**: POT கோப்பு உருவாக்கம்.
- **Packaging**: விநியோகத்திற்காக ஆர்கைவ் உருவாக்கம்.
- **Development Mode**: பிழைத்திருத்தத்திற்காக மினிஃபிகே செய்யப்படாத (Unminified) சொத்துக்கள்.

## உள்ளமைவு (Configuration)

இந்த ஆட்-ஆன் நெட்வொர்க் நிர்வாக இடைமுகம் (network admin interface) மூலம் அணுகக்கூடிய பல்வேறு உள்ளமைவு விருப்பங்களை ஆதரிக்கிறது.

## Hooks மற்றும் Filters

### Actions

- `ultimate-multisite-addon-template_init` - ஆட்-ஆன் துவக்கப்பட்ட பிறகு இயக்கப்படும்.
- `ultimate-multisite-addon-template_loaded` - அனைத்து ஆட்-ஆன் கோப்புகளும் ஏற்றப்பட்ட பிறகு இயக்கப்படும்.

### Filters

- `ultimate-multisite-addon-template_settings` - ஆட்-ஆன் அமைப்புகளை மாற்றியமைக்க.
- `ultimate-multisite-addon-template_enabled` - ஆட்-ஆன் செயல்பட/செயல்படாமல் இருக்க நிலையை மாற்றியமைக்க.
