# Plugin மற்றும் Theme Manager மாற்றங்கள் பட்டியல்

Version: 2.0.8 - Released on 2026-05-05
* மேம்படுத்தப்பட்டது: Plugin பட்டியல் இப்போது தளத்தின் கேஷைப் பயன்படுத்துகிறது. இது தரவுத்தள வினவலைக் குறைத்து, செயல்திறனை மேம்படுத்துகிறது.
* மேம்படுத்தப்பட்டது: Bedrock root autoloader ஏற்கனவே dependencies-களை ஏற்றும்போது, Plugin autoloader தவிர்க்கப்படுகிறது.

Version: 2.0.7 - Released on 2026-01-01
* மேம்படுத்தப்பட்டது: இணக்கத்தன்மை புதுப்பிப்புகள்
