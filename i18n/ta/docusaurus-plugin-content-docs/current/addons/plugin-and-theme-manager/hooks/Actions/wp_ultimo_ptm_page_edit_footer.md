# Action: wp_ultimo_ptm_page_edit_footer

| Name | Type | Description |
|------|------|-------------|
| $ | `object` | தகவலை வைத்திருக்கும் பொருள் (Object) |
| $ | `\WU_Page` | Multisite Ultimate Page இன்ஸ்டன்ஸ் (Instance) |
