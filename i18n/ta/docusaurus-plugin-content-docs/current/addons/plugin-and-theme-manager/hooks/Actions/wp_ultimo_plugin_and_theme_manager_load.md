# Action: wp_ultimo_plugin_and_theme_manager_load {#action-wpultimopluginandthememanagerload}

அனைத்து சார்புநிலைகளும் (dependencies) ஏற்றப்பட்ட பிறகு இது தூண்டப்படுகிறது.

இது plugin உருவாக்குநர்கள் புதிய செயல்பாடுகளைச் சேர்க்க அனுமதிக்கிறது. உதாரணமாக, புதிய ஹோஸ்டிங் வழங்குநர்களுக்கான (Hosting providers) ஆதரவைச் சேர்க்கலாம்.
