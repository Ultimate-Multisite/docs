# Plugin & Theme Manager {#plugin--theme-manager}

Multisite Ultimate: Plugin மற்றும் Theme Manager

நிறுவப்பட்ட plugins மற்றும் themes-களின் மெட்டா தகவல்களை (title, author, thumbnail மற்றும் description) திருத்தலாம் அல்லது மறைக்கலாம். உங்கள் பயனர்கள் அவற்றை வடிகட்ட (filter) வசதியையும், உங்கள் பயனர்களின் பேனலுக்கு அழகான தனிப்பயன் Plugins பக்கத்தையும் உருவாக்க plugins மற்றும் themes-களுக்கு வகைகளை (categories) உருவாக்கலாம்.

இந்த plugin-ஐப் பயன்படுத்த, நீங்கள் உங்கள் network admin-ல் உள்ள Plugins அல்லது Themes பக்கங்களுக்குச் சென்று, "Edit Plugin/Theme info" என்பதைக் கிளிக் செய்தால் போதும்.

## Installation {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager'-ஐ '/wp-content/plugins/' என்ற கோப்பகத்தில் (directory) பதிவேற்றவும்
2. உங்கள் WordPress Network Admin-ல் உள்ள 'Plugins' மெனு மூலம் plugin-ஐ Activate செய்யவும்

## Usage Guide {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager என்பது ஒரு சக்திவாய்ந்த **premium** Ultimate Multisite add-on ஆகும். இது உங்கள் network-ல் உள்ள plugin மற்றும் theme விருப்பங்களை உங்கள் network பயனர்கள் பார்க்கும் மற்றும் அதனுடன் தொடர்பு கொள்ளும் விதத்தின் மீது உங்களுக்கு முழு கட்டுப்பாட்டைக் கொடுக்கிறது.

இந்த add-on மூன்று நோக்கங்களுக்காகப் பயன்படுகிறது:

  1. இது plugins மற்றும் themes இரண்டின் மெட்டாடேட்டாவையும் (metadata) முழுமையாக மாற்ற உங்களை அனுமதிக்கிறது. இதில் பெயர், விளக்கம், ஆசிரியர் பெயர் மற்றும் சிறுபடங்கள் (thumbnails) கூட அடங்கும்!

  2. உங்கள் பயனர்கள் அவற்றை எளிதாக வரிசைப்படுத்த (sort) வசதியாக, plugins மற்றும் themes-களுக்கு வகைகளை (categories) உருவாக்க இது அனுமதிக்கிறது.

  3. பழைய, சலிப்பூட்டும் plugins பக்கத்திற்குப் பதிலாக, வகைகளின் (categories) மூலம் வரிசைப்படுத்தக்கூடிய ஒரு புதிய themes போன்ற பக்கத்தை மாற்றும் விருப்பத்தை இது உங்களுக்கு வழங்குகிறது.

இதை பதிவிறக்கம் செய்து நிறுவிய பிறகு, Ultimate Multisite Settings பக்கத்தில் ஒரு புதிய Tab - _Add-on Settings_ - தோன்றும்.

உங்கள் network _Plugins_ பக்கத்தில், இப்போது நீங்கள் அனைத்து plugins-களின் மெட்டா தகவல்களையும் (meta information) திருத்த முடியும். எடுத்துக்காட்டாக, பெயர், ஆசிரியர் மற்றும் plugin-இன் சிறுபடம் (thumbnail) கூட இதில் அடங்கும்.
