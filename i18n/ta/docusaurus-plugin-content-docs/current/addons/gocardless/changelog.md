# GoCardless மாற்றங்கள் பட்டியல்

பதிப்பு: 1.0.3 - வெளியிடப்பட்டது 2026-05-05
* திருத்தம்: GoCardless உறுப்பினர் ஒரு நிலுவையில் அல்லது இடைநிறுத்தப்பட்ட நிலையில் ரத்து செய்யப்படும்போது, கைவிடப்பட்ட நிலுவையில் உள்ள தளங்கள் இப்போது சுத்தம் செய்யப்படுகின்றன.
* மேம்படுத்தப்பட்டது: Bedrock root autoloader ஏற்கனவே சார்புகளை (dependencies) ஏற்றும்போது, Plugin autoloader தவிர்க்கப்படுகிறது.

பதிப்பு: 1.0.1 - வெளியிடப்பட்டது 2025-09-28

* முன்னொட்டை ultimate-multisite என்று பெயர் மாற்றுதல்; உரை டொமைனைப் புதுப்பித்தல்; பதிப்பு அதிகரிப்பு.

2021-09-24 - பதிப்பு 1.0.0-beta.4

* சேர்க்கப்பட்டது: mu-plugins அடிப்படையிலான அமைப்புகளுக்கு filter wp_ultimo_skip_network_active_check சேர்க்கப்பட்டது;

2021-06-16 - பதிப்பு 1.0.0-beta.3

* திருத்தம்: கையொப்பங்களை (signatures) உறுதிப்படுத்தத் தவறும் Webhook handlers சரிசெய்யப்பட்டது;

2021-06-14 - பதிப்பு 1.0.0-beta.2

* மேம்படுத்தப்பட்டது: ஒரு GoCardless கட்டளை (mandate) உருவாக்கப்பட்ட பிறகு தளத்தை வெளியிடுதல்;

பதிப்பு 1.0.0-beta.1 - ஆரம்ப பதிப்பு
