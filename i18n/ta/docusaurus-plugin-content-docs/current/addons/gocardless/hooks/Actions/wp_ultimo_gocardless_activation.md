# Action: wp_ultimo_gocardless_activation {#action-wpultimogocardlessactivation}

பிளகின்-இன் மற்ற பகுதிகளும், இது செயல்படுத்தப்படும்போது (activation) தேவையான செயல்பாடுகளை இணைக்க (attach) இதைப் பயன்படுத்தலாம்.

### Since {#since}

- 1.9.6
### Source {#source}

[`inc/class-hooks.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-gocardless/blob/main/inc/class-hooks.php#L83) என்ற கோப்பில், 83-வது வரியில் வரையறுக்கப்பட்டுள்ளது.
