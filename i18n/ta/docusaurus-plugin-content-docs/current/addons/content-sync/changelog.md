# Content Sync மாற்றங்கள் பட்டியல்

### பதிப்பு 1.1.0 - வெளியிடப்பட்டது 2025-12-31 {#version-110---released-on-2025-12-31}
* புதியது: அனைத்து போஸ்ட் வகைகளையும் (posts, pages, custom post types) ஒத்திசைவு செய்யும் ஆதரவு.
* புதியது: உள்ளடக்கத்தைத் தேர்ந்தெடுக்கும் படிநிலையில் போஸ்ட் வகை வடிகட்டி (Post type filtering).
* புதியது: மூல Site தேர்வு படிநிலை - பிரதான Site-ஐ மட்டும் அல்லாமல் எந்த Site-இலிருந்தும் ஒத்திசைவு செய்யலாம்.
* புதியது: Template Sites-களுக்கு குழந்தை Site-ஐ முன்கூட்டியே தேர்வு செய்யும் வசதி.
* புதியது: அனைத்து போஸ்ட் வகைகளிலும் post_content இலிருந்து மீடியா பிரித்தெடுத்தல் (Media extraction).
* புதியது: வகைப்பாட்டு ஒத்திசைவு (Taxonomy syncing) (categories, tags, custom taxonomies).
* மாற்றப்பட்டது: "Elementor Template Sync" என்பதிலிருந்து "Content Sync" என்று பெயர் மாற்றப்பட்டது.
* மாற்றப்பட்டது: "Select Templates" படிநிலை "Select Data" என்று பெயர் மாற்றப்பட்டது.
* மேம்படுத்தப்பட்டது: Elementor-குறிப்பிட்ட கோட் இப்போது elementor_library போஸ்ட் வகைக்கு மட்டுமே இயங்கும்.

### பதிப்பு 1.0.0 - வெளியிடப்பட்டது 2025-12-19 {#version-100---released-on-2025-12-19}
* ஆரம்ப வெளியீடு.
* Template மற்றும் Site தேர்வுக்கான பல-படி வழிகாட்டி (Multi-step wizard).
* நிகழ்நேர முன்னேற்ற பின்னூட்டத்துடன் தொகுப்பு செயலாக்கம் (Batch processing).
* தானியங்கி ID மறுவரைபடம் (automatic ID remapping) உடன் முழு மீடியா ஒத்திசைவு.
* Ultimate Multisite உடன் அமைப்புகள் ஒருங்கிணைப்பு (Settings integration).
* நெட்வொர்க் அமைப்புகள் மூலம் Site-ஐ தவிர்ப்பதற்கான ஆதரவு (Site exclusion support).
* ஒத்திசைவுக்குப் பிறகு Elementor cache நீக்குதல்.
* Template வகைப்பாட்டுப் பாதுகாப்பு (Template taxonomy preservation).
