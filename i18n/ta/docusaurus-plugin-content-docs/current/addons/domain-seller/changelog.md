# டொமைன் விற்பனையாளர் மாற்றப் பதிவு

பதிப்பு 1.3.0 - 2026-06-02 அன்று வெளியிடப்பட்டது
- புதியது: HostAfrica reseller இருப்பு மிகக் குறைவாகும் போது network-admin எச்சரிக்கை சேர்க்கப்பட்டது
- புதியது: புதிதாகப் பதிவு செய்யப்பட்ட டொமைன்களை network தளத்துக்கு தானாக இணைக்கும் வசதி சேர்க்கப்பட்டது
- சரி செய்யப்பட்டது: புதிய டொமைன் பதிவு செய்யும் போது மட்டுமே பதிவாளர் புலத் தேவைகள் பயன்படுத்தப்பட்டன
- சரி செய்யப்பட்டது: இருப்பைக் கண்காணிக்கும் அறிவிப்புகளை மூடக்கூடியதாக மாற்றப்பட்டது
- சரி செய்யப்பட்டது: WooCommerce பதிவாளர் பில்லிங் விவரங்கள் பாதுகாக்கப்படுவது உறுதி செய்யப்பட்டது
- சரி செய்யப்பட்டது: பதிவு செய்யும் போது பதிவாளர் தொடர்புத் தேவைகள் கட்டாயப்படுத்தப்பட்டன
- சரி செய்யப்பட்டது: டொமைன் பதிவு தயாரிப்புகள் 0% markup உடன் உருவாக்கப்படுவது தடுக்கப்பட்டது
- சரி செய்யப்பட்டது: checkout session ஓட்டம் முழுவதும் டொமைன் தேர்வுகளும் விலைகளும் பாதுகாக்கப்பட்டன
- சரி செய்யப்பட்டது: HostAfrica டொமைன் விலை நாணயக் காட்சி மேம்படுத்தப்பட்டது
- சரி செய்யப்பட்டது: WP-core query-var பொருந்தாமைகளைத் தவிர்க்க checkout form-action நடத்தை மேம்படுத்தப்பட்டது
- மேம்படுத்தப்பட்டது: setup வழிகாட்டலில் HostAfrica reseller அமைப்பு ஆவணங்கள் இணைக்கப்பட்டன

- மேம்படுத்தப்பட்டது: addon readme-இல் plugin compatibility metadata WordPress 7.0 ஆகப் புதுப்பிக்கப்பட்டது
- மேம்படுத்தப்பட்டது: வரவிருக்கும் வெளியீட்டு ஒருங்கிணைப்பிற்குப் பயன்படுத்தப்படும் திட்டமிடல் templates புதுப்பிக்கப்பட்டன

* **புதியது:** டொமைன் தயாரிப்புகளுக்கான அறிமுக விலை ஆதரவு

* **புதியது:** டொமைன் வாழ்க்கைச்சுழற்சி அறிவிப்புகளுக்கான மின்னஞ்சல் வார்ப்புருக்கள்
