# Action: wu_domain_seller_register_integrations

மற்ற பிளகின்கள் (plugins) கூடுதல் டொமைன் விற்பனையாளர் ஒருங்கிணைப்புகளைப் (domain seller integrations) பதிவு செய்ய இது அனுமதிக்கிறது.

## அளவுருக்கள் (Parameters) {#parameters}

| பெயர் (Name) | வகை (Type) | விளக்கம் (Description) |
|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | ஒருங்கிணைப்புப் பதிவேடு (The integration registry). |

### எப்போது இருந்து (Since) {#since}

- 2.1.0
### மூலம் (Source) {#source}

[`inc/class-domain-seller.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-domain-seller/blob/main/inc/class-domain-seller.php#L293) என்ற கோப்பில் 293 வரியில் வரையறுக்கப்பட்டுள்ளது.
