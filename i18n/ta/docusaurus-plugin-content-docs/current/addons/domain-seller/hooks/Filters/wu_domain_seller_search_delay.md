# Filter: wu_domain_seller_search_delay

டொமைன் தேடலின் debounce தாமதத்தை மில்லி விநாடிகளில் (milliseconds) வடிகட்டுகிறது.

மெதுவான இணைப்புகளில் API அழைப்புகளைக் குறைக்க இந்த மதிப்பை அதிகரிக்கவும்.

## அளவுருக்கள் (Parameters) {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $delay | `int` | debounce தாமதம் மில்லி விநாடிகளில். இயல்புநிலை 500. |
