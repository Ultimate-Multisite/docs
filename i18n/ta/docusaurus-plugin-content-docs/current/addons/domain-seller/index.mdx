# டொமைன் விற்பனையாளர்

**Multisite Ultimate: டொமைன் விற்பனையாளர்** உங்கள் WordPress Multisite வலையமைப்பை சக்திவாய்ந்த டொமைன் விற்பனைத் தளமாக மாற்றுகிறது. வாடிக்கையாளர்கள் பதிவு செய்யும் போதே அல்லது தங்கள் Dashboard இலிருந்து நேரடியாக தனிப்பயன் டொமைன்களை பதிவு செய்யலாம். உடனடி கிடைப்புநிலைச் சரிபார்ப்பும் வெளிப்படையான விலையும் கிடைக்கும்.

### ஏன் டொமைன் விற்பனையாளர்?

எந்த WaaS (சேவையாக வலைத்தளம்) தளத்திற்கும் அதிகம் கோரப்படும் கூடுதல் விற்பனைகளில் ஒன்று டொமைன் பதிவு. உங்கள் வாடிக்கையாளர்களை மூன்றாம் தரப்பு பதிவாளரிடம் அனுப்புவதற்குப் பதிலாக, நீங்களே நேரடியாக டொமைன்களை வழங்கலாம் — இதனால் உங்கள் வலையமைப்பில் அதிக வருவாய் தங்கி, வாடிக்கையாளர்களுக்கும் தடையற்ற அனுபவம் கிடைக்கும்.

### முக்கிய அம்சங்கள்

#### பல டொமைன் பதிவாளர் ஆதரவு
* **OpenSRS** — முழுமையான பதிவு, புதுப்பிப்பு, மற்றும் மேலாண்மை
* **Namecheap** — WhoisGuard மற்றும் sandbox சோதனையுடன் முழுமையான ஒருங்கிணைப்பு
* **HostAfrica** — Domains Reseller API மூலம் டொமைன் தேடல், விலை, பதிவு, பரிமாற்றம், DNS, மற்றும் ID பாதுகாப்பு
* **Openprovider** — மறுவிற்பனையாளர் விலை, பதிவு, புதுப்பிப்பு, பரிமாற்றங்கள், DNS, WHOIS தனியுரிமை, மற்றும் TLD ஒத்திசைவு
* **Hostinger** — பகிரப்பட்ட Hostinger ஒருங்கிணைப்பு token-ஐப் பயன்படுத்தி டொமைன் தேடலும் பதிவும்
* **விரிவாக்கக்கூடிய கட்டமைப்பு** — Integration Registry வடிவத்தை அடிப்படையாகக் கொண்டது; மேலும் வழங்குநர்களைச் சேர்ப்பது எளிது
* **உள்ளமைந்த இணைப்பு சோதனை** — நேரலையில் செல்லும் முன் வழங்குநர் சான்றுகளையும் இணைப்பையும் சரிபார்க்கவும்

#### வழிகாட்டும் அமைப்பு வழிகாட்டி
* **ஒரே கிளிக்கில் தொடக்கம்** — வழங்குநர் அமைப்பு, TLD இறக்குமதி, மற்றும் டொமைன் தயாரிப்பு உருவாக்கம் ஆகியவற்றில் உங்களை வழிநடத்தும்
* **தானியங்கி TLD ஒத்திசைவு** — இணைக்கப்பட்ட வழங்குநர்களிடமிருந்து கிடைக்கும் அனைத்து TLD-களையும் மொத்த விலையையும் இறக்குமதி செய்கிறது
* **இயல்புநிலை தயாரிப்பு உருவாக்கம்** — பொருத்தமான லாப உயர்வு இயல்புநிலைகளுடன் உடனே பயன்படுத்தக்கூடிய டொமைன் தயாரிப்பை உருவாக்குகிறது

#### நெகிழ்வான டொமைன் விலை நிர்ணயம்
* **மூன்று லாப உயர்வு முறைகள்** — சதவீத உயர்வு, நிலையான உயர்வு, அல்லது ஒவ்வொரு TLD-க்கும் நிலையான விலை
* **TLD-சார்ந்த தயாரிப்புகள்** — வெவ்வேறு TLD குழுக்களுக்கு (.com/.net, .io, நாடு குறியீடுகள் போன்றவை) தனித்தனி டொமைன் தயாரிப்புகளை உருவாக்கவும்
* **அறிமுக விலை** — புதிய பதிவுகளை ஈர்க்க தள்ளுபடி செய்யப்பட்ட முதல் ஆண்டு விலையை வழங்கவும்
* **உடனடி விலை** — அறிவார்ந்த caching உடன் வழங்குநர்களிடமிருந்து நேரடி மொத்த விலை
* **கூப்பன் ஆதரவு** — Multisite Ultimate தள்ளுபடி குறியீடுகளுடன் முழு இணக்கத்தன்மை

#### WHOIS தனியுரிமை பாதுகாப்பு
* **ஒவ்வொரு தயாரிப்பிற்குமான அமைப்பு** — WHOIS தனியுரிமையை முடக்கப்பட்டது, எப்போதும் சேர்க்கப்பட்டது, அல்லது வாடிக்கையாளர் தேர்வு என அமைக்கவும்
* **வாடிக்கையாளர் விருப்பச் சேர்க்கை** — வாடிக்கையாளர் தேர்வு என அமைக்கப்பட்டால், checkout checkbox மூலம் வாடிக்கையாளர்கள் தனியுரிமை பாதுகாப்பைச் சேர்க்கலாம்
* **அமைக்கக்கூடிய விலை** — தனியுரிமை add-on-க்கு ஆண்டுக்கு ஒரு விலையை அமைக்கவும், அல்லது இலவசமாக வழங்கவும்
* **பதிவாளர் ஒருங்கிணைப்பு** — பதிவு நேரத்தில் தானாகவே WhoisGuard (Namecheap) அல்லது WHOIS தனியுரிமை (OpenSRS) செயல்படுத்தப்படும்

#### ஒருங்கிணைந்த Checkout அனுபவம்
* **மூன்று டொமைன் முறைகள்** — வாடிக்கையாளர்கள் subdomain (இலவசம்), புதிய டொமைன் பதிவு, அல்லது ஏற்கனவே சொந்தமான டொமைனை mapping செய்வது ஆகியவற்றில் ஒன்றைத் தேர்வுசெய்யலாம்
* **உடனடி டொமைன் தேடல்** — பக்கம் மீண்டும் ஏற்றாமல் AJAX மூலம் கிடைப்புநிலைச் சரிபார்ப்பு
* **சிறந்த பரிந்துரைகள்** — விருப்பமான டொமைன் ஏற்கனவே எடுக்கப்பட்டிருந்தால் மாற்று TLD பரிந்துரைகள்
* **வெளிப்படையான விலை** — பதிவு விலை, புதுப்பிப்பு விலை, மற்றும் விருப்பமான WHOIS தனியுரிமை கட்டணம் தெளிவாக காட்டப்படும்
* **பதிவாளர் தொடர்பு fields** — checkout நடக்கும் போதே தேவையான பதிவாளர் தகவல்களை (பெயர், முகவரி, தொலைபேசி) சேகரிக்கும்
* **தானாக உருவாகும் site URL** — தேர்ந்தெடுக்கப்பட்ட டொமைனிலிருந்து site முகவரி தானாக உருவாக்கப்படும்

#### தானியங்கி புதுப்பிப்பு அமைப்பு
* **Membership-இன் அடிப்படையிலான புதுப்பிப்புகள்** — வாடிக்கையாளரின் membership செயலில் இருக்கும் வரை டொமைன்கள் தானாக புதுப்பிக்கப்படும்
* **சிறந்த அட்டவணையிடல்** — டொமைன் காலாவதி தேதிகளின் அடிப்படையில் புதுப்பிப்பு பணிகள் வரிசைப்படுத்தப்படும்
* **மீண்டும் முயற்சி தர்க்கம்** — தோல்வியடைந்த புதுப்பிப்பு முயற்சிகளுக்கு அறிவார்ந்த மீண்டும் முயற்சிகள்
* **மின்னஞ்சல் அறிவிப்புகள்** — வரவிருக்கும் புதுப்பிப்புகள், வெற்றிகரமான புதுப்பிப்புகள், தோல்விகள், மற்றும் காலாவதிகளுக்கான அமைக்கக்கூடிய எச்சரிக்கைகள்

#### வாடிக்கையாளர் DNS மேலாண்மை
* **சுயசேவை DNS** — வாடிக்கையாளர்கள் தங்கள் Dashboard இலிருந்து DNS records-ஐ (A, AAAA, CNAME, MX, TXT) சேர்க்க, திருத்த, மற்றும் நீக்க முடியும்
* **டொமைன் நிலை** — உடனடி பதிவு நிலை, காலாவதி தேதிகள், மற்றும் nameserver தகவல்
* **தானியங்கி டொமைன் mapping** — பதிவு செய்யப்பட்ட அல்லது ஏற்கனவே உள்ள டொமைன்கள் வாடிக்கையாளரின் site-க்கு தானாக mapping செய்யப்படும்

#### நிர்வாக கருவிகள்
* **கைமுறை டொமைன் பதிவு** — admin panel இலிருந்து வாடிக்கையாளர்களின் சார்பில் டொமைன்களை பதிவு செய்யவும்
* **டொமைன் கண்காணிப்பு** — அனைத்து வாடிக்கையாளர் டொமைன்கள், நிலைகள், மற்றும் காலாவதி தேதிகளை ஒரே பக்கத்தில் பார்க்கவும்
* **வழங்குநர்-சார்ந்த logging** — ஒவ்வொரு பதிவாளருக்கும் தனித்தனி log சேனல்கள் (எ.கா., domain-seller-namecheap.log)
* **பதிவு விவரங்கள்** — core domain edit page-இல் பதிவாளர் தகவல் மற்றும் பதிவு தரவைப் பார்க்கவும்

### ஆதரிக்கப்படும் வழங்குநர்கள்

* **OpenSRS** — பதிவு, புதுப்பிப்பு, DNS மேலாண்மை, மற்றும் WHOIS தனியுரிமையுடன் முழு ஒருங்கிணைப்பு
* **Namecheap** — பதிவு, புதுப்பிப்பு, தொகுதி கிடைப்புநிலைச் சரிபார்ப்பு, மற்றும் இலவச WhoisGuard உடன் முழு ஒருங்கிணைப்பு
* **HostAfrica** — பதிவு, புதுப்பிப்பு, பரிமாற்றங்கள், DNS மேலாண்மை, EPP code பெறுதல், registrar lock, மற்றும் ID பாதுகாப்பு
* **Openprovider** — பதிவு, புதுப்பிப்பு, பரிமாற்றங்கள், DNS மேலாண்மை, EPP code பெறுதல், registrar lock, WHOIS தனியுரிமை, மற்றும் மறுவிற்பனையாளர் TLD ஒத்திசைவு
* **Hostinger** — பதிவு, டொமைன் தேடல், nameserver புதுப்பிப்புகள், registrar lock, மற்றும் WHOIS தனியுரிமை; புதுப்பிப்புகள் Hostinger auto-renew மூலம் கையாளப்படும்
* **GoDaddy** — பதிவு மற்றும் புதுப்பிப்பு
* **ResellerClub** — பதிவு, புதுப்பிப்பு, மற்றும் DNS மேலாண்மை
* **NameSilo** — பதிவு மற்றும் புதுப்பிப்பு
* **Enom** — பதிவு, புதுப்பிப்பு, மற்றும் DNS மேலாண்மை

### தொழில்நுட்ப விவரங்கள்

* **தேவைகள்:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **கட்டமைப்பு:** Domain_Selling_Capability interface உடன் Integration Registry வடிவு
* **Autoloading:** PSR-4 class mapping உடன் Jetpack autoloader
* **தரநிலைகள்:** WordPress coding standards, strict typing, prepared statements

## நிறுவல்

1. **Multisite Ultimate v2.4.12+** நிறுவப்பட்டு, வலையமைப்பில் செயல்படுத்தப்பட்டுள்ளதா என்பதை உறுதிசெய்யவும்
2. plugin-ஐ `/wp-content/plugins/ultimate-multisite-domain-seller/` இல் பதிவேற்றவும்
3. WordPress வலையமைப்பு நிர்வாகத்தில் **Plugins** இலிருந்து plugin-ஐ வலையமைப்பில் செயல்படுத்தவும்
4. முதல் செயல்படுத்தலின் போது **அமைப்பு வழிகாட்டி** தானாகத் தொடங்கும் — இந்த படிகளைப் பின்பற்றவும்:
   * உங்கள் domain பதிவாளரை இணைக்கவும்
   * இணைப்பைச் சோதிக்கவும்
   * கிடைக்கும் TLD-கள் மற்றும் மொத்த விலைகளை இறக்குமதி செய்யவும்
   * விலை உயர்வுடன் உங்கள் முதல் domain தயாரிப்பை உருவாக்கவும்
5. **Ultimate Multisite > செலுத்தல் படிவங்கள்** இல் உள்ள உங்கள் செலுத்தல் படிவத்தில் **Domain தேர்வு** புலத்தைச் சேர்க்கவும்

## அடிக்கடி கேட்கப்படும் கேள்விகள்

### Multisite Ultimate-இன் எந்த பதிப்பு தேவை?

பதிப்பு 2.4.12 அல்லது அதற்கு மேல். இந்த addon WP Ultimo v1.x அல்லது 2.4.12-க்கு கீழ் உள்ள Ultimate Multisite பதிப்புகளுடன் இணக்கமாக இல்லை.

### எந்த domain வழங்குநர்கள் ஆதரிக்கப்படுகிறார்கள்?

ஒன்பது பதிவாளர்கள் ஆதரிக்கப்படுகிறார்கள்: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, மற்றும் **Enom**. பதிவாளர் API-கள் வெவ்வேறு செயல்பாடுகளை வழங்குவதால், வழங்குநருக்கு ஏற்ப திறன்கள் மாறுபடும்; ஒவ்வொரு வழங்குநருக்குமான சுருக்கத்துக்கு அமைப்பு வழிகாட்டியைப் பார்க்கவும். addon-இன் Integration Registry கட்டமைப்பு தனிப்பயன் வழங்குநர்களைச் சேர்ப்பதை எளிதாக்குகிறது — [டெவலப்பர் வழிகாட்டி](/developer/integration-guide/custom-registrar) பார்க்கவும்.

### நான் WHOIS தனியுரிமை பாதுகாப்பை வழங்க முடியுமா?

ஆம். ஒவ்வொரு domain தயாரிப்பிலும் மூன்று முறைகளுடன் WHOIS தனியுரிமை அமைப்பு உள்ளது: முடக்கப்பட்டது, எப்போதும் சேர்க்கப்பட்டது, அல்லது வாடிக்கையாளர் தேர்வு. வாடிக்கையாளர் தேர்வாக அமைக்கப்பட்டால், செலுத்தலின்போது வாடிக்கையாளர்கள் விலையுடன் ஒரு தேர்வுப்பெட்டியைப் பார்ப்பார்கள் (அல்லது Namecheap-இன் WhoisGuard-க்கு "இலவசம்"). பதிவு நேரத்தில் தனியுரிமைக் குறி தானாகவே பதிவாளருக்கு அனுப்பப்படும்.

### வெவ்வேறு TLD-களுக்கு வெவ்வேறு விலைகளை அமைக்க முடியுமா?

ஆம். வெவ்வேறு TLD குழுக்களுக்கு தனித்தனி domain தயாரிப்புகளை உருவாக்கவும். ஒவ்வொரு தயாரிப்புக்கும் அதன் சொந்த உயர்வு வகை (சதவீதம், நிலையான உயர்வு, அல்லது நிலையான விலை), ஆதரிக்கப்படும் TLD-கள், மற்றும் அறிமுக விலை இருக்கலாம்.

### வாடிக்கையாளர்கள் செலுத்தலின்போது என்ன விருப்பங்களைப் பார்ப்பார்கள்?

Domain தேர்வு புலம் வாடிக்கையாளர்களுக்கு மூன்று தாவல்களை வழங்குகிறது: **Subdomain** (உங்கள் வலையமைப்பில் இலவச subdomain-ஐப் பயன்படுத்துதல்), **புதிய Domain-ஐ பதிவு செய்** (புதிய domain-ஐத் தேடி பதிவு செய்தல்), மற்றும் **ஏற்கனவே உள்ள Domain** (அவர்கள் ஏற்கனவே வைத்திருக்கும் domain-ஐ வரைபடமிடுதல்). நீங்கள் இயக்கும் முறைகள் மட்டுமே தோன்றும்.

### இது தானியங்கி புதுப்பித்தல்களை ஆதரிக்கிறதா?

ஆம். Domains தானாகப் புதுப்பிக்க அமைக்கப்படலாம் மற்றும் வாடிக்கையாளரின் membership நிலைக்கு இணைக்கப்படும். புதுப்பித்தல் பணப்பரிவர்த்தனைகள் வாடிக்கையாளரின் தற்போதைய payment gateway மூலம் செயலாக்கப்படும்.

### வாடிக்கையாளர்கள் தங்கள் DNS பதிவுகளை நிர்வகிக்க முடியுமா?

ஆம். வாடிக்கையாளர்கள் தங்கள் dashboard-இல் A, AAAA, CNAME, MX, மற்றும் TXT பதிவுகளைச் சேர்க்க, திருத்த, மற்றும் நீக்க DNS நிர்வாக இடைமுகத்தைப் பெறுவார்கள். DNS நிர்வாகம் OpenSRS, ResellerClub, Enom, HostAfrica, மற்றும் Openprovider உடன் கிடைக்கிறது. hosted domains-க்கான Hostinger DNS, மைய Hostinger domain-mapping ஒருங்கிணைப்பால் கையாளப்படுகிறது.

### அமைப்பு வழிகாட்டி உள்ளதா?

ஆம். அமைப்பு வழிகாட்டி முதல் செயல்படுத்தலின் போது தானாக இயங்கி, பதிவாளரை இணைப்பது, TLD-களை இறக்குமதி செய்வது, மற்றும் உங்கள் முதல் domain தயாரிப்பை உருவாக்குவது ஆகியவற்றில் உங்களை வழிநடத்தும். addon அமைப்புகளிலிருந்தும் வழிகாட்டியை மீண்டும் இயக்கலாம்.
