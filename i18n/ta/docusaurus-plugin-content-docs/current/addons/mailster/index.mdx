# Mailster

Ultimate Multisite Mailster Integration addon மூலம் உங்கள் multisite network-ஐ சக்திவாய்ந்த மின்னஞ்சல் பட்டியல் உருவாக்கும் இயந்திரமாக மாற்றுங்கள். புதிய வாடிக்கையாளர்கள் பதிவு செய்யும்போதோ அல்லது பணம் செலுத்தும்போதோ தானாகவே உங்கள் Mailster மின்னஞ்சல் பட்டியல்களில் அவர்களைச் சேர்க்கிறது. இது உங்கள் உறுப்பினர் தளத்திற்கும் (membership platform) மின்னஞ்சல் சந்தைப்படுத்தல் உத்திக்கும் (email marketing strategy) இடையே தடையற்ற இணைப்பை உருவாக்குகிறது.

## முக்கிய அம்சங்கள்

**தானியங்கி சந்தாதாரர் ஒத்திசைவு (Automatic Subscriber Sync)**
* பதிவு செய்யும்போதே வாடிக்கையாளர்களை Mailster பட்டியல்களில் தானாகச் சேர்க்கும்
* பதிவு செய்த உடனேயே ஒத்திசைக்க வேண்டுமா அல்லது பணம் செலுத்துதல் உறுதிப்படுத்தல் வரை காத்திருக்க வேண்டுமா என்பதைத் தேர்வு செய்யலாம்
* கையால் தலையீடு செய்யாமல் உங்கள் மின்னஞ்சல் பட்டியல்களைப் புதுப்பித்த நிலையில் வைக்கவும்

**நெகிழ்வான பட்டியல் மேலாண்மை (Flexible List Management)**
* அனைத்து புதிய வாடிக்கையாளர்களுக்கும் உலகளாவிய இயல்புநிலை பட்டியல்களை அமைக்கவும்
* குறிப்பிட்ட பிரச்சாரங்களுக்காக (targeted campaigns) இயல்புநிலைகளைத் தாண்டி, தயாரிப்பு-குறிப்பிட்ட பட்டியல்களைப் பயன்படுத்தவும்
* Mailster-ன் பழக்கமான இடைமுகத்தைப் பயன்படுத்தி எளிதாகப் பட்டியல் ஒதுக்கீடுகளை நிர்வகிக்கவும்

**GDPR-க்கு இணக்கமான விருப்பங்கள் (GDPR-Compliant Opt-In Options)**
* தானியங்கி சந்தா அல்லது தேர்வுப்பெட்டி அடிப்படையிலான சம்மதம் (checkbox-based consent) ஆகியவற்றுக்கு இடையே தேர்வு செய்யவும்
* உறுதிப்படுத்தப்பட்ட சந்தாக்கள் (confirmed subscriptions) සඳහා இரட்டை விருப்பத்தை (double opt-in) இயக்கவும்
* உங்கள் இணக்கத் தேவைகளின் அடிப்படையில் விருப்பப் தேர்வுப்பெட்டியை (opt-in checkbox) காட்டவும்/மறைக்கவும்

**ஸ்மார்ட் தரவு வரைபடம் (Smart Data Mapping)**
* வாடிக்கையாளர் பெயர்கள் மற்றும் தொடர்புத் தகவல்களை தானாக ஒத்திசைக்கும்
* பில்லிங் முகவரிகளை சந்தாதாரர் புலங்களுடன் (subscriber fields) வரைபடமாக்குதல்
* குறிப்பிட்ட பிரச்சாரங்களுக்காகப் பதிவு செய்த தேதிகளைக் கண்காணிக்கவும்

**தயாரிப்பு-நிலை கட்டுப்பாடு (Product-Level Control)**
* வெவ்வேறு தயாரிப்புகளுக்கு வெவ்வேறு பட்டியல்களை ஒதுக்கலாம்
* வாடிக்கையாளர்கள் என்ன வாங்குகிறார்கள் என்பதன் அடிப்படையில் இலக்கு வைக்கப்பட்ட பிரச்சாரங்களை உருவாக்கவும்
* தேவைப்படும்போது குறிப்பிட்ட தயாரிப்புகளுக்கு Mailster-ஐ முடக்கலாம்

**தொழில்முறை அனுபவம் (Professional Experience)**
* பட்டியல் தேர்வுக்கு Native Mailster UI ஒருங்கிணைப்பு
* சிக்கல் தீர்க்கவும் சரிபார்க்கவும் விரிவான பதிவுசெய்தல் (Detailed logging)
* தற்போதுள்ள சந்தாதாரர்களைத் தடையின்றிப் புதுப்பித்தல்

## யாருக்கு ஏற்றது (Perfect For)

* மின்னஞ்சல் பட்டியல்களை உருவாக்கும் உறுப்பினர் தள உரிமையாளர்கள்
* புதிய வாடிக்கையாளர்களை வளர்க்க விரும்பும் SaaS தளங்கள்
* மாணவர்களைப் பதிவு செய்ததன் அடிப்படையில் பிரிக்கும் (segmenting) பாடப்பிரிவு உருவாக்குபவர்கள்
* Mailster உடன் Ultimate Multisite இயக்கும் எவரும்

## இது எப்படி வேலை செய்கிறது (How It Works)

1. உங்கள் Ultimate Multisite network-ல் addon-ஐ நிறுவி செயல்படுத்தவும்
2. Ultimate Multisite அமைப்புகளில் உலகளாவிய இயல்புநிலை பட்டியல்களை உள்ளமைக்கவும்
3. விருப்பப்பட்டால், தயாரிப்பு-குறிப்பிட்ட பட்டியல்களால் மாற்றவும் (override)
4. உங்கள் விருப்பப் பதிவு முறையையும் (opt-in mode) சந்தா நேரத்தையும் தேர்வு செய்யவும்
5. உங்கள் Mailster பட்டியல்கள் தானாகவே வளரப் பார்க்கவும்!

## நிறுவுதல் (Installation)

1. பி Plugin கோப்புகளை உங்கள் Ultimate Multisite addons கோப்பகத்தில் பதிவேற்றவும்
2. Mailster உங்கள் பிரதான தளத்தில் (main site) நிறுவப்பட்டு செயல்படுவதை உறுதி செய்யவும்
3. Ultimate Multisite addons பக்கத்தின் மூலம் addon-ஐ செயல்படுத்தவும்
4. உள்ளமைக்க Settings > Mailster Integration என்பதற்குச் செல்லவும்
5. உங்கள் இயல்புநிலை பட்டியல்களைத் தேர்ந்தெடுத்து விருப்பப் பதிவு விருப்பங்களை உள்ளமைக்கவும்

## அடிக்கடி கேட்கப்படும் கேள்விகள் (Frequently Asked Questions)

## இந்த addon Ultimate Multisite இல்லாமல் வேலை செய்யுமா?

இல்லை, இந்த addon செயல்பட Ultimate Multisite தேவை. இது Mailster-ஐ Ultimate Multisite-ன் வாடிக்கையாளர் மற்றும் தயாரிப்பு அமைப்புடன் ஒருங்கிணைக்கவே வடிவமைக்கப்பட்டுள்ளது.

## அனைத்து தளங்களிலும் Mailster நிறுவ வேண்டுமா?

இல்லை, Mailster உங்கள் பிரதான network தளத்தில் (main network site) நிறுவப்பட்டு செயல்படுத்தப்பட்டிருந்தால் போதுமானது. addon அனைத்து துணை தளங்களிலிருந்தும் (subsites) தானாகவே அதனுடன் இணைக்கும்.

## வெவ்வேறு தயாரிப்புகளுக்கு வெவ்வேறு பட்டியல்களை ஒதுக்க முடியுமா?

ஆம்! நீங்கள் உலகளாவிய இயல்புநிலை பட்டியல்களை அமைக்கலாம், பின்னர் தயாரிப்பு வாரியாக அவற்றை மாற்றியமைக்கலாம் (override). வாடிக்கையாளர்கள் என்ன வாங்குகிறார்கள் என்பதன் அடிப்படையில் மிகவும் இலக்கு வைக்கப்பட்ட மின்னஞ்சல் பிரச்சாரங்களை உருவாக்க இது அனுமதிக்கிறது.

## ஒரு வாடிக்கையாளர் ஏற்கனவே Mailster-ல் இருந்தால் என்ன நடக்கும்?

இயல்புநிலையாக, addon இருக்கும் சந்தாதாரர்களைப் புதுப்பித்து, அவர்களை புதிய பட்டியல்களிலும் சேர்க்கும். இருக்கும் சந்தாதாரர்களைத் தவிர்க்க விரும்பினால், நீங்கள் இதை அமைப்புகளில் முடக்கலாம்.

## இது GDPR இணக்கமானதா?

ஆம்! இந்த addon-ல் தேர்வுப்பெட்டி அடிப்படையிலான விருப்பப் பதிவு முறை (checkbox-based opt-in mode) உள்ளது. இதில் வாடிக்கையாளர்கள் மின்னஞ்சல் பெற வெளிப்படையாக சம்மதம் அளிக்க வேண்டும். சரியான தனியுரிமைக் கொள்கைகளுடன் (privacy policies) இதை இணைக்கும்போது இது GDPR தேவைகளைப் பூர்த்தி செய்கிறது.

## double opt-in பயன்படுத்த முடியுமா?

நிச்சயமாக! double opt-in அமைப்பை இயக்கவும், சந்தாக்களைச் செயல்படுத்துவதற்கு முன் Mailster உறுதிப்படுத்தல் மின்னஞ்சல்களை அனுப்பும்.
