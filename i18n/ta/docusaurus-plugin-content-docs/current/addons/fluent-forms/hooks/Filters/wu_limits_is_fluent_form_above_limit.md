# Filter: wu_limits_is_fluent_form_above_limit {#filter-wulimitsisfluentformabovelimit}

கொடுக்கப்பட்ட ஃபார்ம் வகை இந்த பிளானில் அனுமதிக்கப்பட்டுள்ளதா என்பதை இது சரிபார்க்கிறது. பிளக்-இன் டெவலப்பர்கள் ரிட்டர்ன் மதிப்பை ஃபில்டர் செய்ய இதை அனுமதிக்கிறது.

## அளவுருக்கள் {#parameters}

| Name | Type | விளக்கம் |
|------|------|-------------|
| $value | `bool` | ஃபார்ம் வகை வரம்பை மீறியதா இல்லையா என்பதைக் குறிக்கும். |
| $form_type | `string` | சரிபார்க்கப்படும் ஃபார்ம் வகை. |
| $form_count | `int` | தற்போதைய ஃபார்ம் எண்ணிக்கை. |
| $quota | `int` | அனுமதிக்கப்பட்ட மொத்த ஒதுக்கீடு. |
