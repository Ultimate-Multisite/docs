# Fluent Forms

இந்த add-on, Multisite Ultimate-ல் உள்ள தயாரிப்புகள் (products), உறுப்பினர் திட்டங்கள் (memberships) அல்லது தளங்களில் (sites) Fluent Forms-க்கு வரம்புகளைச் (limits) சேர்க்க உதவுகிறது.
Fluent Forms ஆனது தனிப்பயன் பதிவு வகைகளுக்குப் (custom post types) பதிலாக தனிப்பயன் அட்டவணைகளைப் (custom tables) பயன்படுத்துகிறது. எனவே, சாதாரண வரம்பு விருப்பங்கள் Fluent Forms உடன் வேலை செய்யாது.

## அம்சங்கள் (Features)

- Fluent Forms அல்லது Conversational Fluent Forms-க்கு வரம்புகளைச் சேர்ப்பது.
- வாடிக்கையாளர்கள் வரம்பை அடைந்தவுடன், தங்கள் திட்டத்தை மேம்படுத்த (upgrade) தூண்டும் வகையில் ஒரு நட்புரீதியான செய்தியைப் பெறுவார்கள்.
