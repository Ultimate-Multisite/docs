### 1.0.1 - 2025-11-03 {#101---2025-11-03}
* சேர்க்கப்பட்டது: Ultimate Multisite update server மூலம் தானியங்கி புதுப்பிப்பு செயல்பாடு.
* மேம்படுத்தப்பட்டது: WordPress 6.6 வரை சோதிக்கப்பட்டது.
* மேம்படுத்தப்பட்டது: composer autoloader உடன் 플러ഗിன் அமைப்பு மேம்படுத்தப்பட்டது.
* மேம்படுத்தப்பட்டது: நவீன WordPress நிறுவல்களுடன் இணக்கத்தன்மை மேம்படுத்தப்பட்டது.

### 1.0.0 - 2024-10-21 {#100---2024-10-21}
* முதல் வெளியீடு.
* 110 க்கும் மேற்பட்ட Chuck Norris உண்மைகள்.
* முழு மொழிபெயர்ப்பு ஆதரவு.
* எல்லா திரை அளவுகளுக்கும் ஏற்ற வடிவமைப்பு (Responsive design).
* RTL மொழி ஆதரவு.
* Block editor உடன் இணக்கத்தன்மை.
* screen reader ஆதரவுடன் அணுகல் அம்சங்கள் (Accessibility features).
