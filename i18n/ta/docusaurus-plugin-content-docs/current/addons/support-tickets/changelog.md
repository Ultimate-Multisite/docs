### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* சரி செய்யப்பட்டது: Null மதிப்புகளை ஏற்கும்படி nullable ticket model setters மாற்றப்பட்டது.
* சரி செய்யப்பட்டது: சரியான multisite ஆதரவுக்காக அனைத்து டிக்கெட்டுகளும் உலகளாவிய network-wide அட்டவணைகளில் சேமிக்கப்படுகின்றன.
* சரி செய்யப்பட்டது: ஊழியர் (staff) புலங்கள் மறைக்கப்பட்டு, வாடிக்கையாளர் new-ticket படிவத்தில் blank-page redirect பிழை சரி செய்யப்பட்டது.
* சரி செய்யப்பட்டது: metadata சேமிப்பிற்காக, undefined add_meta() அழைப்புகள் சரியான update_meta() மூலம் மாற்றப்பட்டன.
* சரி செய்யப்பட்டது: பதிவு செய்யப்படாத capability சோதனையை சரியான wu_view_all_support_tickets மூலம் மாற்றப்பட்டது.
* சரி செய்யப்பட்டது: டிக்கெட் நிலை, ஒதுக்கீடு (assignment) மற்றும் விரைவு-திருத்தம் (quick-edit) செயல்பாடுகளுக்கான காணாமல் போன AJAX handlers சேர்க்கப்பட்டன.
* சரி செய்யப்பட்டது: டிக்கெட் காட்சிகளில் ஊழியர் பதிலைக் கண்டறியும் முறை பெயர் சரி செய்யப்பட்டது.
* சரி செய்யப்பட்டது: நகல் பதில் கையாளுபவர்கள் (reply handlers) ஒருங்கிணைக்கப்பட்டு, nonce action பெயர்கள் சீரமைக்கப்பட்டன.
* சரி செய்யப்பட்டது: [wu_submit_ticket] shortcode க்கான காணாமல் போன frontend காட்சி சேர்க்கப்பட்டது.
* சரி செய்யப்பட்டது: காணாமல் போன user_id நிரல் சேர்க்கப்பட்டு, Support_Ticket::get_user_id() முறை சரி செய்யப்பட்டது.
* சரி செய்யப்பட்டது: முன்னுரிமை (priority) வடிகட்டி விருப்பத் தாளில் உள்ள கூடுதல் இரட்டை மேற்கோள் குறியீடு நீக்கப்பட்டது.
* சரி செய்யப்பட்டது: சூப்பர் நிர்வாகிகளுக்காக (super admins) network நிர்வாக டிக்கெட் மேலாண்மைப் பலகம் சேர்க்கப்பட்டது.
* மேம்படுத்தப்பட்டது: நிர்வாக CSS ஒரு ஒற்றை வெளிப்புற பாணித் தாளில் ஒருங்கிணைக்கப்பட்டது.
* மேம்படுத்தப்பட்டது: துணை தள நிர்வாக மெனுவில் இருந்து Settings துணை மெனு நீக்கப்பட்டது.
* மேம்படுத்தப்பட்டது: ஆதரவு டிக்கெட் பக்கங்களில் மட்டுமே நிபந்தனையின் அடிப்படையில் frontend assets ஏற்றப்படும்.
* மேம்படுத்தப்பட்டது: Bedrock root autoloader ஏற்கனவே சார்புகளை (dependencies) ஏற்றும்போது, plugin autoloader தவிர்க்கப்படும்.

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* சேர்க்கப்பட்டது: நிர்வாகம் (admin) மற்றும் frontend இரண்டிற்கும் முழு டிக்கெட் பார்க்கும் செயல்பாடு.
* சேர்க்கப்பட்டது: டிக்கெட் பதில் சமர்ப்பிப்புகளுக்கான AJAX handler.
* சேர்க்கப்பட்டது: சரியான படிவ கையாளுதலுடன் டிக்கெட் பதில் செயல்பாட்டிற்கு ஆதரவு.
* சேர்க்கப்பட்டது: நிர்வாகத்தில் டிக்கெட் சமர்ப்பிப்புகள் மற்றும் பதில்களுக்கான சரியான அறிவிப்பு காட்சி.
* சேர்க்கப்பட்டது: வாடிக்கையாளர் சுய-சமர்ப்பிப்புகளுக்கு தற்போதைய பயனருடன் தானாகவே டிக்கெட் இணைக்கப்படும்.
* சேர்க்கப்பட்டது: டிக்கெட் உரிமையை வாடிக்கையாளர்கள் மாற்றாமல் தடுக்க பாதுகாப்பு மேம்படுத்தப்பட்டது.
* சேர்க்கப்பட்டது: காணாமல் போன உதவிச் செயல்பாடுகள் (wu_format_date, wu_user_can_view_ticket போன்றவை).
* சேர்க்கப்பட்டது: சரியான கோப்பு இணைப்பு பதிவிறக்கம் மற்றும் கையாளுதல்.
* சேர்க்கப்பட்டது: டிக்கெட் பதில்கள் மற்றும் நிலை மாற்றங்களுக்கான மின்னஞ்சல் அறிவிப்பு அமைப்பு.
* சரி செய்யப்பட்டது: டிக்கெட் பதில் படிவத்தில் இப்போது தேவையான டிக்கெட் ID சேர்க்கப்பட்டுள்ளது.
* சரி செய்யப்பட்டது: நிர்வாக டிக்கெட் காட்சிகளில் அறிவிப்பு தெரிவுப் பிரச்சினைகள் சரி செய்யப்பட்டன.
* சரி செய்யப்பட்டது: ஆதரவு டிக்கெட் செயல்பாடுகளில் உள்ள Syntax பிழைகள்.
* மேம்படுத்தப்பட்டது: பதில்கள் மற்றும் இணைப்புகளுக்கு சரியான query மற்றும் schema வகுப்புகளுடன் தரவுத்தள அமைப்பு.
* மேம்படுத்தப்பட்டது: டிக்கெட் மாடல்களுக்கு பண்புகள் (attributes) என்பதற்குப் பதிலாக உண்மையான பண்புகளுக்கு (real properties) இடம்பெயர்வு.
