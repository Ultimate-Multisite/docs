# VAT மாற்றங்கள் பட்டியல்

Version 1.0.7 - Released on 2026-02-03

* திருத்தம்: VIES SOAP service-ல் ஏற்பட்ட MS_MAX_CONCURRENT_REQ பிழைகள் காரணமாக ஜெர்மன் (DE) VAT எண்கள் சரிபார்ப்பில் தோல்வியடைந்தது. நம்பகமான சரிபார்ப்புக்கு, பழைய SOAP API-யிலிருந்து அதிகாரப்பூர்வ EU VIES REST API-க்கு மாற்றப்பட்டுள்ளது.
* திருத்தம்: VIES சேவை தற்காலிகமாக கிடைக்காதபோது, VAT எண்கள் தவறாக நிராகரிக்கப்படாமல், இப்போது ஏற்றுக்கொள்ளப்படும். இதை பின்னர் மதிப்பாய்வு செய்ய ஒரு பதிவு (log entry) உருவாக்கப்படும்.
* திருத்தம்: VAT/Tax ID ஃபீல்ட் இப்போது அனைத்து நாடுகளுக்கும் தெரியும், EU உறுப்பினர்களுக்கு மட்டும் அல்ல. Non-EU வரி அடையாள எண்கள் (உதாரணமாக, Swiss CHE எண்கள்) VIES சரிபார்ப்பு இல்லாமல் இன்வாய்ஸ் காண்பிக்க சேமிக்கப்படும்.
* திருத்தம்: தவறான array lookup logic காரணமாக கிரீஸ் (GR), மொனாக்கோ (MC), மற்றும் ஐல் ஆஃப் மன் (IM) ஆகியவற்றிற்கான VAT முன்னொட்டு (prefix) தேடல் செயலிழந்திருந்தது.
* மேம்பாடு: Site Exporter addon உடன் பயன்படுத்தப்படும்போது fatal errors-க்கு வழிவகுக்கும் தேவையற்ற wp-cli-bundle dev dependency நீக்கப்பட்டது.
* மேம்பாடு: விரிவான unit test suite சேர்க்கப்பட்டது (53 சோதனைகள்).

Version 1.0.6 - Released on 2026-01-25

* திருத்தம்: உள்நாட்டு B2B பரிவர்த்தனைகள் இப்போது சரியாக VAT வசூலிக்கின்றன. EU VAT விதிகளின்படி, வாடிக்கையாளரின் நாடு நிறுவனத்தின் நாட்டைப் போலவே இருக்கும்போது, ரிவர்ஸ் சார்ஜ் (reverse charge) என்பது எல்லை தாண்டிய B2B பரிவர்த்தனைகளுக்கு மட்டுமே பொருந்தும்.

Version 1.0.5 - Released on 2026-01-22

* திருத்தம்: Addon ஏற்றப்படாமல் அல்லது சரியாக வேலை செய்யாமல் இருந்தது.
* மாற்றம்: மிகவும் நம்பகமான மற்றும் தீவிரமாக பராமரிக்கப்படும் தரவுக்காக, VAT விகித தரவு மூலத்தை euvatrates.com இலிருந்து ibericode/vat-rates repository-க்கு மாற்றப்பட்டுள்ளது.
* திருத்தம்: super_reduced_rates option மதிப்புக்கான தட்டச்சுப் பிழை (typo) சரிசெய்யப்பட்டது.
* மேம்பாடு: பிழைகளைத் தடுக்க country code கையாளுதலுக்கு null checks சேர்க்கப்பட்டன.
* மாற்றம்: Traduttore மூலம் தானியங்கி புதுப்பித்தலுக்கு ஆதரவளிக்கும் வகையில், தொகுக்கப்பட்ட மொழிபெயர்ப்பு கோப்புகள் (bundled translation files) நீக்கப்பட்டன.
