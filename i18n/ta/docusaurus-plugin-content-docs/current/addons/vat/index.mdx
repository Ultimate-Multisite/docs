# VAT

Multisite Ultimate: ஐரோப்பிய VAT

இது Multisite Ultimate-ன் இயல்பான வரி நிர்வாகத்திற்கு கூடுதல் அம்சங்களையும் கருவிகளையும் சேர்க்கிறது. இதன் மூலம் ஐரோப்பிய VAT-ஐ சிறப்பாக ஆதரிக்க முடியும்.
EU VAT விகிதங்களை தானாகவே Tax Rates பக்கத்தில் சேர்க்கிறது.

## நிறுவுதல்

1. 'ultimate-multisite-vat' என்பதை '/wp-content/plugins/' கோப்பகத்தில் (directory) பதிவேற்றவும்.
2. WordPress-ல் உள்ள 'Plugins' மெனு மூலம் இந்த plugin-ஐ Activate செய்யவும்.
3. Ultimate Multisite > Settings > Taxes என்ற பகுதிக்குச் சென்று EU Vat-ஐ Enable செய்யவும்.
4. Tax Rates-ல் கிளிக் செய்து, கீழே உள்ள EU VAT Rates-ஐ Update செய்து, Tax Rates-ஐ Save செய்யவும்.
5. முடிந்தது!
