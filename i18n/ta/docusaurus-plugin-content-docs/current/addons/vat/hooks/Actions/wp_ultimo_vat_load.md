# Action: wp_ultimo_vat_load {#action-wpultimovatload}

அனைத்து சார்புநிலைகளும் (dependencies) ஏற்றப்பட்ட பிறகு இது செயல்படும்.

இது plugin உருவாக்குநர்கள் புதிய செயல்பாடுகளைச் சேர்க்க அனுமதிக்கிறது. உதாரணமாக, புதிய Hosting வழங்குநர்களுக்கான ஆதரவைச் சேர்ப்பது போன்ற விஷயங்களுக்கு இது உதவும்.

### Since {#since}

- 1.0.0
