# மல்டி-நெட்வொர்க் மாற்றங்கள் பட்டியல்

### 1.3.0
* புதியது: நெட்வொர்க் டெம்ப்ளேட் முன்னோட்ட கருவி (Network template previewer) — வாங்குவதற்கு முன், நேரடி பேனலில் நெட்வொர்க் டெம்ப்ளேட்களை உலாவவும் முன்னோட்டமிடுங்கள்.
* புதியது: செக்அவுட்டில் நெட்வொர்க் டெம்ப்ளேட் தேர்வு கருவி (Network template picker at checkout) — வாடிக்கையாளர்கள் பதிவு செய்யும் செக்அவுட் செயல்முறையின் போது நெட்வொர்க் டெம்ப்ளேட்டைத் தேர்ந்தெடுக்கலாம்.
* புதியது: குறுக்கு-நெட்வொர்க் மேஜிக் லிங்க் SSO (Cross-network magic link SSO) — வாடிக்கையாளர்கள் மேஜிக்-லிங்க் அங்கீகாரத்தின் மூலம் துணை-நெட்வொர்க்குகளை தடையின்றி அணுகலாம்.
* புதியது: நெட்வொர்க் மீடியா குளோனிங் (Network media cloning) — டெம்ப்ளேட்டிலிருந்து நெட்வொர்க்குகளை குளோன் செய்யும் போது, தளத்தின் மீடியா சரியாக நகலெடுக்கப்படும்.
* திருத்தம்: நெட்வொர்க் குளோனிங் இப்போது மல்டி-டென்யன்சி 애ட்-ஆன் இல்லாமலும் வேலை செய்கிறது; குளோன் செய்த பிறகு சூப்பர் அட்மின் சலுகைகள் மீண்டும் வழங்கப்படுகின்றன.
* திருத்தம்: நெட்வொர்க் உருவாக்கம் இப்போது WooCommerce மற்றும் அனைத்து கட்டண நுழைவாயில்களுடனும் இணக்கமானது.
* திருத்தம்: ஃபெல்பேக் டொமைன் உருவாக்கத்தில் போர்ட் எண் இப்போது சேர்க்கப்பட்டுள்ளது.
* திருத்தம்: டெம்ப்ளேட் தேர்வு கருவி CSS, அனைத்து தீம்களிலும் சரியான ரெண்டரிங்கிற்காக ஒரு வரிசைப்படுத்தப்பட்ட ஸ்டைல்ஷீட்டிற்கு மாற்றப்பட்டது.
* திருத்தம்: தள வினவல் வரம்பில் வரையறுக்கப்படாத `network_id` விசையைத் தடுக்கும் பாதுகாப்பு சேர்க்கப்பட்டது.
* திருத்தம்: சூழல் மாறுவதற்கு முன் நெட்வொர்க் அனுமதிச் சரிபார்ப்பு, `wpdb` அட்டவணை பண்புப் பிழைகளைத் தடுக்கிறது.
