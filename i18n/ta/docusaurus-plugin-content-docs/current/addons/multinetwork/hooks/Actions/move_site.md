# Action: move_site {#action-movesite}

ஒரு வலைத்தளம் புதிய நெட்வொர்க்கிற்கு மாற்றப்பட்ட பிறகு இது இயங்குகிறது.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | மாற்றப்பட்ட வலைத்தளத்தின் ID. |
| $old_network_id | `int` | வலைத்தளத்தின் அசல் நெட்வொர்க் ID. |
| $new_network_id | `int` | வலைத்தளம் மாற்றப்பட்ட நெட்வொர்க் ID. |

### Since {#since}

- 1.3.0

[`inc/functions/network.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-multinetwork/blob/main/inc/functions/network.php#L1587) என்ற இடத்தில், 1587 வரியில் வரையறுக்கப்பட்டுள்ளது.
