# Action: update_network

ஏற்கனவே உள்ள ஒரு நெட்வொர்க் (network) புதுப்பிக்கப்பட்ட பிறகு இது இயங்கும்.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $network_id | `int` | சேர்க்கப்பட்ட நெட்வொர்க்கின் ID. |
| $args | `array` | நெட்வொர்க் ஆர்குமென்ட்களின் (network arguments) இணைக்கப்பட்ட அணி (Associative array). |
