# Action: wp_ultimo_payfast_load {#action-wpultimopayfastload}

அனைத்து சார்புநிலைகளும் (dependencies) ஏற்றப்பட்ட பிறகு இது செயல்படும்.

இது plugin உருவாக்குநர்கள் புதிய செயல்பாடுகளைச் சேர்க்க அனுமதிக்கிறது. உதாரணமாக, புதிய ஹோஸ்டிங் வழங்குநர்களுக்கான ஆதரவு போன்றவற்றைச் சேர்க்கலாம்.
