# Filter: wp_ultimo_skip_network_active_check

இந்தச் சோதனையைத் தவிர்க்க (short-circuit) டெவலப்பர்களுக்கு இது அனுமதிக்கிறது.

Composer-அடிப்படையிலான மற்றும் Bedrock போன்ற பிற தனிப்பயன் அமைப்புகளைப் பயன்படுத்தும் போது இது மிகவும் பயனுள்ளதாக இருக்கும். ஏனெனில், அத்தகைய அமைப்புகளில் mu-plugins ஆகப் பிளக்இன்களைப் பயன்படுத்துவது இயல்பானது.
