# Loco Translate {#loco-translate}

Ultimate Multisite: Loco Translate Integration addon உங்கள் WordPress multisite network-க்குத் தளத்திற்கு-தனிப்பட்ட மொழிபெயர்ப்பு திறன்களை (site-specific translation capabilities) கொண்டு வருகிறது. உங்கள் நெட்வொர்க்கில் உள்ள வெவ்வேறு தளங்களுக்கு ஒரே plugin அல்லது theme-இன் வெவ்வேறு மொழிபெயர்ப்புகள் தேவைப்பட்டால், இந்த addon மிகச் சரியான தீர்வாகும்.

**இந்த Addon என்ன செய்யும்**

இந்த addon, Ultimate Multisite-ஐ பிரபலமான Loco Translate plugin உடன் தடையின்றி ஒருங்கிணைக்கிறது. இதன் மூலம் உங்கள் நெட்வொர்க்கில் உள்ள ஒவ்வொரு subsite-உக்கும் அதன் சொந்த தனிப்பயன் மொழிபெயர்ப்புகளை (custom translations) பராமரிக்க முடியும். நீங்கள் பல பிராந்தியங்களில் இயங்கும் ஒரு தளத்தை (multi-regional platform) நடத்துகிறீர்களா, வெவ்வேறு மொழி சமூகங்களுக்கு சேவை செய்கிறீர்களா, அல்லது உங்கள் நெட்வொர்க் முழுவதும் மொழிபெயர்ப்பு நெகிழ்வுத்தன்மை தேவைப்படுகிறதா என்பதைப் பொருட்படுத்தாமல், இந்த addon இதை மிகவும் எளிதாக்குகிறது.

**யாருக்கு ஏற்றது:**

* வெவ்வேறு உள்ளூர்மயமாக்கல் (localization) தேவைகள் கொண்ட பல பிராந்திய வணிகங்கள்
* பல்வேறு சமூகங்களுக்கு சேவை செய்யும் கல்வி நிறுவனங்கள்
* பல மொழி ஆதரவை வழங்கும் SaaS தளங்கள்
* பல வாடிக்கையாளர் தளங்களை நிர்வகிக்கும் வெப் ஏஜென்சிகள்
* மொழிபெயர்ப்பு நெகிழ்வுத்தன்மை தேவைப்படும் எந்தவொரு multisite network-உம்

## Installation {#installation}

1. Ultimate Multisite நிறுவப்பட்டு, network செயல்படுத்தப்பட்டுள்ளதை உறுதி செய்யவும்
2. Loco Translate plugin-ஐ நிறுவவும் மற்றும் செயல்படுத்தவும்
3. இந்த addon-ஐ உங்கள் plugins directory-இல் பதிவேற்றவும்
4. Loco Translate Integration addon-ஐ network அளவில் செயல்படுத்தவும்
5. எந்தவொரு subsite-இலும் Loco Translate மூலம் தளத்திற்கு-தனிப்பட்ட மொழிபெயர்ப்புகளை உருவாக்கத் தொடங்கவும்

அவ்வளவுதான்! கூடுதல் அமைப்புகள் எதுவும் செய்யாமல், இந்த ஒருங்கிணைப்பு தானாகவே வேலை செய்யும்.

## Frequently Asked Questions {#frequently-asked-questions}

## நிறுவிய பிறகு நான் எதையாவது கட்டமைக்க வேண்டுமா? {#do-i-need-to-configure-anything-after-installation}

வேண்டாம்! இந்த ஒருங்கிணைப்பு தானியங்குமானது. addon மற்றும் Loco Translate ஆகிய இரண்டும் செயல்படுத்தப்பட்டவுடன், subsite-கள் உடனடியாக அவற்றின் சொந்த தனிப்பயன் மொழிபெயர்ப்புகளை உருவாக்கத் தொடங்கலாம்.

## ஒரு subsite தனிப்பயன் மொழிபெயர்ப்புகளை உருவாக்கவில்லை என்றால் என்ன நடக்கும்? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

அந்தத் தளம் தானாகவே network-wide மொழிபெயர்ப்புகளை மாற்று வழி (fallback) ஆகப் பயன்படுத்தும். எதுவும் உடைந்து போகாது, எல்லாமே தடையின்றி வேலை செய்யும்.

## subsite-கள் network-wide மொழிபெயர்ப்புகளை அணுக முடியுமா? {#can-subsites-access-network-wide-translations}

சாதாரண site admins தங்கள் சொந்த தளத்தின் மொழிபெயர்ப்புகளை மட்டுமே பார்க்கவும் திருத்தவும் முடியும். Super admins, எந்தத் தளத்தைப் பார்க்கிறார்கள் என்பதைப் பொறுத்து, network-wide மற்றும் site-specific மொழிபெயர்ப்புகள் இரண்டையும் அணுக முடியும்.

## இது plugins மற்றும் themes இரண்டிலும் வேலை செய்யுமா? {#does-this-work-with-both-plugins-and-themes}

ஆம்! site-specific மொழிபெயர்ப்புகள் WordPress plugins மற்றும் themes இரண்டிற்கும் வேலை செய்யும்.

## இது எனது multisite network-ஐ மெதுவாக்குமா? {#will-this-slow-down-my-multisite-network}

வேண்டாம். இந்த addon, தேவைப்படும்போது மட்டுமே site-specific மொழிபெயர்ப்புகளைச் சரிபார்க்கும் திறமையான கோப்பு ஏற்றத்தைப் (efficient file loading) பயன்படுத்துகிறது. குறிப்பிடத்தக்க செயல்திறன் பாதிப்பு எதுவும் இல்லை.

## ஒரு தொடக்கப் புள்ளியாக network மொழிபெயர்ப்புகளை ஒரு குறிப்பிட்ட தளத்திற்கு நகலெடுக்க முடியுமா? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

ஆம்! Loco Translate-இல் மொழிபெயர்ப்புகளைத் திருத்தும் போது, நீங்கள் network மொழிபெயர்ப்புகளுடன் எளிதாகத் தொடங்கி, ஒவ்வொரு தளத்திற்கும் அவற்றை தனிப்பயனாக்கலாம்.

## இது WooCommerce மற்றும் பிற plugins உடன் வேலை செய்யுமா? {#does-this-work-with-woocommerce-and-other-plugins}

நிச்சயமாக! நிலையான WordPress மொழிபெயர்ப்பு செயல்பாடுகளைப் (standard WordPress translation functions) பயன்படுத்தும் எந்தவொரு plugin அல்லது theme-உம் site-specific மொழிபெயர்ப்புகளுடன் வேலை செய்யும்.
