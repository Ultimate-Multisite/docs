# AI Site Builder

ultimate-multisite-ai-site-builder

## தேவையானவை (Requirements) {#requirements}

- WordPress 5.3 அல்லது அதற்கு மேல்
- PHP 7.4 அல்லது அதற்கு மேல்
- Multisite Ultimate plugin (செயல்படுத்தப்பட்ட நிலையில்)

## நிறுவுதல் (Installation) {#installation}

1. ஆட்-ஆன் கோப்புகளை உங்கள் `/wp-content/plugins/` கோப்பகத்தில் பதிவேற்றவும்
2. WordPress-ல் உள்ள 'Plugins' மெனு மூலம் பிளகினை செயல்படுத்துங்கள்
3. நெட்வொர்க் நிர்வாகத்தில் (network admin) ஆட்-ஆன் அமைப்புகளை உள்ளமைக்கவும்

## மேம்பாடு (Development) {#development}

### அமைவு (Setup) {#setup}

```bash
# சார்புகளை (dependencies) நிறுவவும்
composer install
npm install

# சோதனைகளை (tests) இயக்கவும்
npm run test

# குறியீடு தரநிலைகளைச் சரிபார்க்கவும்
vendor/bin/phpcs
vendor/bin/phpstan

# குறியீட்டு பாணி சிக்கல்களைச் சரிசெய்யவும்
vendor/bin/phpcbf

# உற்பத்திக்காக உருவாக்கவும் (Build for production)
npm run build
```

### சோதனை செய்தல் (Testing) {#testing}

இந்த ஆட்-ஆன் ஒரு விரிவான சோதனை கட்டமைப்பைக் கொண்டுள்ளது:

- **Unit Tests**: WordPress test suite உடன் PHPUnit அடிப்படையிலான சோதனைகள்
- **Base Test Class**: பொதுவான சோதனை சூழ்நிலைகளுக்கான உதவி முறைகளை வழங்குகிறது
- **Test Coverage**: குறியீட்டு கவரேஜை (code coverage) கண்காணிக்க அமைக்கப்பட்டுள்ளது
- **Multisite Testing**: பல-சைட் சூழலில் (multisite environment) சோதனைகள் இயக்கப்படும்

### குறியீட்டு தரநிலைகள் (Code Standards) {#code-standards}

- **WordPress Coding Standards**: PHPCS மூலம் செயல்படுத்தப்படுகிறது
- **PHP 7.4+ Compatibility**: நவீன PHP அம்சங்கள் ஆதரிக்கப்படுகின்றன
- **Static Analysis**: வகைச் சரிபார்ப்புக்கு (type checking) PHPStan ஒருங்கிணைப்பு
- **Code Modernization**: PHP மேம்படுத்தல்களுக்கான Rector விதிகள்

### உருவாக்க அமைப்பு (Build System) {#build-system}

- **Asset Processing**: CSS/JS கோப்புகளுக்கான சுருக்குதல் (Minification)
- **Translation**: POT கோப்பு உருவாக்கம்
- **Packaging**: விநியோகத்திற்காக ஆர்கைவ் உருவாக்கம்
- **Development Mode**: பிழைத்திருத்தத்திற்காக (debugging) சுருக்குதல் செய்யப்படாத சொத்துக்கள் (Unminified assets)

## உள்ளமைவு (Configuration) {#configuration}

இந்த ஆட்-ஆன், நெட்வொர்க் நிர்வாக இடைமுகம் (network admin interface) மூலம் அணுகக்கூடிய பல்வேறு உள்ளமைவு விருப்பங்களை ஆதரிக்கிறது.

## ஹூக்குகள் மற்றும் ஃபில்டர்கள் (Hooks and Filters) {#hooks-and-filters}

### செயல்பாடுகள் (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - ஆட்-ஆன் துவக்கப்பட்ட பிறகு இயக்கப்படும்
- `ultimate-multisite-ai-site-builder_loaded` - அனைத்து ஆட்-ஆன் கோப்புகளும் ஏற்றப்பட்ட பிறகு இயக்கப்படும்

### ஃபில்டர்கள் (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - ஆட்-ஆன் அமைப்புகளை மாற்றியமைக்க
- `ultimate-multisite-ai-site-builder_enabled` - ஆட்-ஆன் இயக்க/முடக்க நிலையை மாற்றியமைக்க
