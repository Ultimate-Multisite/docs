# Action: wp_ultimo_support_agents_load {#action-wpultimosupportagentsload}

அனைத்து சார்புநிலைகளும் (dependencies) ஏற்றப்பட்ட பிறகு இது தூண்டப்படுகிறது.

இது 플러그인 உருவாக்குநர்கள் புதிய செயல்பாடுகளைச் சேர்க்க அனுமதிக்கிறது. உதாரணமாக, புதிய ஹோஸ்டிங் வழங்குநர்களுக்கான ஆதரவைச் சேர்ப்பது போன்றவை.
