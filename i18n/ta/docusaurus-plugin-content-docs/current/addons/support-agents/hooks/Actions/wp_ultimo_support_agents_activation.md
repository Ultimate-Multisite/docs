# Action: wp_ultimo_support_agents_activation {#action-wpultimosupportagentsactivation}

### Since {#since}

- 1.9.6
### Source {#source}

[`inc/class-hooks.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-support-agents/blob/main/inc/class-hooks.php#L83) என்ற கோப்பில், 83-வது வரியில் வரையறுக்கப்பட்டுள்ளது.
