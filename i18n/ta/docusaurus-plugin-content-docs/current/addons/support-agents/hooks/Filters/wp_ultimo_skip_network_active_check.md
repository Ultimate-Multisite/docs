# Filter: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

டெவலப்பர்கள் இந்தச் சோதனையைத் தவிர்க்க (short-circuit) அனுமதிக்கிறது.

composer-ஐ அடிப்படையாகக் கொண்ட மற்றும் Bedrock போன்ற தனிப்பயன் அமைப்புகளைப் பயன்படுத்தும் போது இது மிகவும் பயனுள்ளதாக இருக்கும். ஏனெனில், அத்தகைய அமைப்புகளில் mu-plugins-ஆக प्लगஇன்களைப் பயன்படுத்துவது இயல்பானது.
