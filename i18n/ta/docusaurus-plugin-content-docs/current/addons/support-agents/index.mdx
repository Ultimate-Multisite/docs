# Support Agents

ஒரு முழு நெட்வொர்க்கையும் நீங்களே நிர்வகிப்பது அவ்வளவு சுலபமான வேலை இல்லை. உங்கள் நெட்வொர்க் வளர ஆரம்பிக்கும்போது, உங்களுக்கு ஆதரவுக்கான தேவை அதிகமாக இருப்பதை நீங்கள் கவனிப்பீர்கள். உங்கள் வேலையின் சுமையைக் குறைக்க உதவும் சிலரை வரவழைப்பது நல்ல யோசனைதான். ஆனால், முழு அணுகல்/அனுமதியை வழங்குவது ஆபத்தானது, மேலும் அது உங்கள் நெட்வொர்க்கைப் பாதிக்கலாம். இங்கதான் இந்த Support Agent add-on தேவைப்படுகிறது.

இந்த add-on மூலம், 'Support Agent' என்று அழைக்கப்படும் ஒரு புதிய வகை WordPress பயனரை உருவாக்க முடியும். மேலும், அந்தப் பயனர் என்னென்ன அனுமதிகளைக் கொண்டிருக்க வேண்டும் என்பதில் உங்களுக்கு முழுமையான கட்டுப்பாடு கிடைக்கும். அனுமதியை வழங்குவது என்பது ஒரு ட Toggle பட்டனை ஆன் செய்வது போல எளிமையானது; அதை நீக்க முடிவு செய்தால், அதை ஆஃப் செய்வது போலவும் எளிமையானது.

## Features {#features}

- WordPress மற்றும் Ultimate Multisite அனுமதிகளின் பட்டியல் (Itemized list of permissions)
- அணுகலை எளிதாக இயக்கவும் (Enable) மற்றும் முடக்கவும் (Disable)
- வரம்பற்ற எண்ணிக்கையிலான Agent கணக்குகளை உருவாக்கலாம்

## Installation {#installation}

1. 'ultimate-multisite-support-agents' என்பதை '/wp-content/plugins/' என்ற கோப்பகத்தில் (directory) பதிவேற்றவும்
2. WordPress-ல் உள்ள 'Plugins' மெனு மூலம் பிளக்கை (plugin) செயல்படுத்துங்கள் (Activate)
3. பிளக்கை அமைக்க, படிப்படியான Wizard-ஐப் பின்பற்றவும்
