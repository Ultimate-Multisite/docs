# Action: wp_ultimo_affiliatewp_load

அனைத்து சார்புநிலைகளும் (dependencies) ஏற்றப்பட்ட பிறகு இது தூண்டப்படுகிறது.

இது plugin உருவாக்குநர்கள் புதிய செயல்பாடுகளைச் சேர்க்க அனுமதிக்கிறது. உதாரணமாக, புதிய Hosting வழங்குநர்களுக்கான ஆதரவைச் சேர்ப்பது போன்றவற்றைச் செய்யலாம்.

- 1.0.0
