# Action: wp_ultimo_affiliatewp_activation

### Version {#since}

- 1.9.6

### மூலம் {#source}

இது [inc/class-hooks.php] என்ற கோப்பில், 83 வரியில் வரையறுக்கப்பட்டுள்ளது.
