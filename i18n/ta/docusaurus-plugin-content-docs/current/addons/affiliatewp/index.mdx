# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

உங்கள் Ultimo Network-இன் வாடிக்கையாளர் தளத்தை வளர்க்க சக்திவாய்ந்த AffiliateWP-ஐப் பயன்படுத்துங்கள்!

## Installation {#installation}

1. 'ultimate-multisite-git ' என்ற கோப்பைப் '/wp-content/plugins/' என்ற கோப்பகத்தில் பதிவேற்றவும்.
2. WordPress-இல் உள்ள 'Plugins' மெனு மூலம் பிளக்இன்-ஐ Activate செய்யவும்.
3. பிளக்இன்-ஐ அமைக்க படிப்படியான Wizard-ஐப் பின்பற்றவும்.

## Usage Guide {#usage-guide}

உங்கள் ஆன்லைன் வணிகத்திற்காக ஒரு affiliate program-ஐ உருவாக்கும்போது, [AffiliateWP](https://affiliatewp.com/) முதன்மையான தீர்வாக மாறியுள்ளது. Ultimate Multisite: AffiliateWP என்பது உங்கள் பிரீமியம் நெட்வொர்க்குவுக்கும் இதேபோன்ற ஒன்றை உருவாக்க உதவும் ஒரு பாலமாக செயல்படும்.

## Getting Started {#getting-started}

எல்லாவற்றிற்கும் மேலாக, நீங்கள் [**AffiliateWP**](https://affiliatewp.com/) **செயல்படுத்தப்பட்ட நிலையில்** இருக்க வேண்டும்! இதற்கு இணையும் **பிளக்இன் உங்களிடம் இல்லையென்றால்**, நீங்கள் எங்கள் add-on-ஐப் பயன்படுத்த முடியாது. உங்கள் வாடிக்கையாளர்கள் தங்கள் தளங்களிலும் இதைப் பயன்படுத்த திட்டமிடவில்லை என்றால், AffiliateWP உங்கள் பிரதான தளத்தில் மட்டுமே செயல்படுத்தப்பட்டிருக்க வேண்டும்.

மேலும், நீங்கள் **Ultimate Multisite** மற்றும் **Ultimate Multisite: AffiliateWP** ஆகிய இரண்டுமே **network-wide** செயல்படுத்தப்பட்ட நிலையில் இருக்க வேண்டும்.

## Adding Ultimate Multisite as an AffiliateWP Integration {#adding-ultimate-multisite-as-an-affiliatewp-integration}

உங்கள் பிரதான தளத்தில், Affiliates Settings-க்குச் சென்று, Integrations tab-இல் Ultimate Multisite-ஐ டிக் செய்யவும். இது [AffiliateWP](https://affiliatewp.com/) இலிருந்து Ultimate Multisite sign-up flow-இல் வரும் வருகைகள் மற்றும் மாற்றங்களைக் (conversions) கண்காணிக்கத் தொடங்கச் சொல்லும்.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_கீழே உள்ள Ultimate Multisite பெட்டியை டிக் செய்து சேமிக்கவும்_

அவ்வளவுதான். பரிந்துரை இணைப்புகள் (referral links) மூலம் வரும் வருகைகள் கண்காணிக்கப்படும், மேலும் அந்தப் பயனர் பின்னர் தனது முதல் கட்டணத்தைச் செய்யும் போது, மாற்றம் (conversion) உருவாக்கப்பட்டு, அந்த மாற்றத்திற்குப் பொறுப்பான affiliate-க்கு ஒரு கமிஷன் ஒதுக்கி வைக்கப்படும்.

இருப்பினும், புதிதாக உருவாக்கப்பட்ட பயனரிடமிருந்து பெறப்படும் முதல் கட்டணம் மட்டுமே கண்காணிக்கப்படும். தொடர்ச்சியான பரிந்துரை கட்டணங்களை (recurring referral payments) செயல்படுத்த, உங்களுக்கு [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) தேவை.

## Enabling Recurring Referrals {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ஐ நிறுவிய பிறகு, Affiliates Settings-இல் உள்ள Recurring Referrals tab-க்குச் சென்று, தொடர்ச்சியான பரிந்துரைகளைப் பயன்படுத்துவதை இயக்கவும்.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_மிகவும் எளிமையானது: Recurring Referrals-ஐ இயக்கினால் போதும், நீங்கள் தயாராகிவிட்டீர்கள்!_

மீண்டும் சொல்கிறேன், அவ்வளவுதான் தேவை!

இப்போது, பரிந்துரைக்கப்பட்ட கணக்கில் (referred account) ஒவ்வொரு முறையும் ஒரு புதிய கட்டணம் பெறப்படும்போது, பொறுப்பான affiliate தானாகவே தனது கமிஷனைப் பெறுவார்!
