# Filter: wu_cap_captcha_config {#filter-wucapcaptchaconfig}

Cap Captcha அமைவு (configuration) ஃபில்டர்.

உருவாக்குநர்கள் (developers) Cap Captcha அமைப்புகளை தனிப்பயனாக்க இது அனுமதிக்கிறது.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Cap அமைவு அணிவரிசை (configuration array). |
| $level | `string` | பாதுகாப்பு நிலை (security level) (fast, medium, max). |

### Since {#since}

- 1.2.0
### Source {#source}

[`inc/class-cap-provider.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-captcha/blob/main/inc/class-cap-provider.php#L153) என்ற இடத்தில், வரி 153-ல் வரையறுக்கப்பட்டுள்ளது.
