# Action: wu_captcha_integrations_init {#action-wucaptchaintegrationsinit}

அனைத்து ஒருங்கிணைப்புகளும் (integrations) துவக்கப்பட்ட பிறகு இது இயங்கும்.

மூன்றாம் தரப்பு ஒருங்கிணைப்புகள் (Third-party integrations) தங்கள் சொந்த செயல்பாடுகளைச் சேர்க்க இந்த இடத்தில் hook செய்யலாம்.
