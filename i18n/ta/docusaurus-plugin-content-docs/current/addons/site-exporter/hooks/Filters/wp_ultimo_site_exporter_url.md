# Filter: wp_ultimo_site_exporter_url {#filter-wpultimositeexporterurl}

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | நீங்கள் அணுக விரும்பும், பிளக்இன் மூலப்பாதைக்கு (plugin root) ஒப்பான பாதை. |

### Since {#since}
