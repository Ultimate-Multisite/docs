# Action: wp_ultimo_site_exporter_load {#action-wpultimositeexporterload}

அனைத்து சார்புநிலைகளும் (dependencies) ஏற்றப்பட்ட பிறகு இது செயல்படும்.

플러ഗിన్ உருவாக்குநர்கள் புதிய செயல்பாடுகளைச் சேர்க்க இது அனுமதிக்கிறது. உதாரணமாக, புதிய ஹோஸ்டிங் வழங்குநர்களுக்கான ஆதரவு போன்றவற்றைச் சேர்க்கலாம்.
