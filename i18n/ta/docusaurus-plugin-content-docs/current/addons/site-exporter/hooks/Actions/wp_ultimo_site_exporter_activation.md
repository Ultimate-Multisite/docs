# Action: wp_ultimo_site_exporter_activation {#action-wpultimositeexporteractivation}

### Since {#since}

- 1.9.6
### Source {#source}

[`inc/class-hooks.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-hooks.php#L82) என்ற கோப்பின் 82-வது வரியில் வரையறுக்கப்பட்டுள்ளது.
