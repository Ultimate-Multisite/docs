# Action: wp_ultimo_site_exporter_deactivation

பிளகின்-இன் மற்ற பகுதிகளும், இது செயலிழக்கச் செய்யப்படும்போது (deactivation) தேவையான செயல்பாடுகளை இணைக்க இதை அனுமதிக்கிறது.

### எப்போது இருந்து {#since}

- 1.9.6
### மூலம் {#source}

இது [inc/class-hooks.php](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-hooks.php#L99) என்ற கோப்பில், 99-வது வரியில் வரையறுக்கப்பட்டுள்ளது.
