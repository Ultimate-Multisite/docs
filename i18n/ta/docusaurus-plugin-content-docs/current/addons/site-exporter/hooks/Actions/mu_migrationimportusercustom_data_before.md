# Action: mu_migration/import/user/custom_data_before

## அளவுருக்கள் (Parameters)

|------|------|-------------|
| $user_data | `array` | பயனர் தரவு அணி (user data array). |
| $user | `\WP_User` | பயனர் பொருள் (user object). |
