# மாற்றப் பதிவு {#changelog}

## 1.18.0 — 2026-06-29 அன்று வெளியிடப்பட்டது {#1180--released-on-2026-06-29}

### புதியவை {#new}

- **Google Calendar கருவிகள்** — அட்டவணை அறிந்த தானியக்கங்களுக்கு அமைக்கப்பட்ட calendars மற்றும் நிகழ்வுகளைப் படிக்கும்
- **தொடர்பு பொருத்துதல் மற்றும் பங்கேற்பாளர் உதவிகள்** — calendar பங்கேற்பாளர்களைத் தளப் பயனர்கள் மற்றும் தொடர்புகளுடன் பொருத்தும்
- **மனித ஒப்புதல் வாயில்கள் மற்றும் நினைவூட்டல் பதிவுகள்** — மதிப்பாய்வுக்காக தானியக்கங்களை இடைநிறுத்தி, மீண்டும் மீண்டும் அறிவிப்புகள் செல்லாமல் தவிர்க்கும்
- **TextBee SMS வழங்குநர்** — TextBee மூலம் அமைக்கப்பட்ட உரைச் செய்தி அறிவிப்புகளை அனுப்பும்
- **மேம்பட்ட துணைத் தொகுப்பு** — WordPress.org build-இலிருந்து தனியாக வழங்கப்படும் நம்பகமான உருவாக்குநர் கோப்பு அமைப்பு, தரவுத்தளம், WP-CLI, REST dispatcher, plugin builder, git snapshot, பயனர் மேலாண்மை, மற்றும் benchmark கருவிகளைச் சேர்க்கும்

### மேம்படுத்தப்பட்டது {#improved}

- **நிர்வகிக்கப்படும் Superdav சேவை அமைப்பு** — ஆதரிக்கப்படும் தளங்களுக்கு hosted service endpoints மற்றும் தானியங்கி இணைப்பு வழங்கலைச் சேர்க்கும்
- **வெளியீட்டு தொகுப்பாக்கம்** — தனித்தனி core மற்றும் Advanced ZIPகளை உருவாக்கி, இரண்டையும் GitHub-இல் வெளியிட்டு, core தொகுப்பை மட்டும் WordPress.org-க்கு அனுப்பும்

### சரிசெய்யப்பட்டது {#fixed}

- **AI கோரிக்கை நம்பகத்தன்மை** — model தேர்வு, கோரிக்கை timeouts, இயல்புநிலை setup model, reasoning-text கையாளுதல், மற்றும் செல்லாத tool calls-க்கு retry வழிகாட்டலை மேம்படுத்தும்
- **Calendar மற்றும் நினைவூட்டல் உறுதிப்படுத்தல்** — Google Calendar tokens மற்றும் நினைவூட்டல் மறுபதிவு தவிர்ப்பை வலுப்படுத்தும்
- **Onboarding மற்றும் approval resumes** — frontend onboarding தொடக்கங்களையும் உறுதிப்படுத்தப்பட்ட திறன் resumes-களையும் சரிசெய்கிறது
- **WordPress.org தொகுப்பாக்க மதிப்பாய்வு சிக்கல்கள்** — core வெளியீட்டிற்கான தொகுப்பாக்க மதிப்பாய்வு கருத்துகளைத் தீர்க்கிறது

## 1.16.0 — 2026-05-20 அன்று வெளியிடப்பட்டது {#1160--released-on-2026-05-20}

### புதியவை {#new-1}

- **Logo SVG உருவாக்கும் திறன்** — Theme Builder இப்போது namespace-safe sanitisation உடன் தனிப்பயன் logo SVGகளை உருவாக்கி உட்பொதிக்க முடியும்
- **Discovery interview-இல் புகைப்படப் பதிவேற்றம்** — Theme Builder discovery interview இப்போது செழுமையான வடிவமைப்பு சூழலுக்காக புகைப்படப் பதிவேற்றப் படியைச் சேர்க்கிறது
- **Palette Contrast சரிபார்ப்பு திறன்** — theme-க்கு பயன்படுத்துவதற்கு முன் WCAG இணக்கத்திற்காக நிற ஜோடிகளைச் சரிபார்க்கும்
- **Hospitality menus** — விருந்தோம்பல் வணிகங்களுக்காக Theme Builder இப்போது கட்டமைக்கப்பட்ட உணவு மற்றும் பான menu பக்கங்களை உருவாக்க முடியும்
- **Desktop மற்றும் mobile preview rendering** — வடிவமைப்பு திசைத் தேர்வின்போது உங்கள் வடிவமைப்பை desktop மற்றும் mobile சாதனங்களில் preview செய்யுங்கள்
- **Navigation label அளவுரு** — Create Menu திறன் இப்போது பக்கத் தலைப்பிலிருந்து தனித்த `navigation_label`-ஐ ஆதரிக்கிறது
- **Tier 1 tool கிடைக்கும் நிலை** — sd-ai-agent/site-scrape இப்போது Theme Builder-இல் இயல்பாகக் கிடைக்கும் Tier 1 tool ஆக உள்ளது

### சரிசெய்யப்பட்டது {#fixed-1}

- **AI Client cache** — இப்போது cross-request persistence-க்காக transients ஆதரவுடன் உள்ளது; நீண்ட நேரம் இயங்கும் agent பணிகளில் தரவு இழப்பைத் தடுக்கிறது
- **Plugin row action links** — தெளிவுக்காகச் சரிசெய்யப்பட்டு மறுபெயரிடப்பட்டது

## 1.10.0 — 2026-05-05 அன்று வெளியிடப்பட்டது {#1100--released-on-2026-05-05}

### புதியவை {#new-2}

- **Tavily இணையத் தேடல்** — Brave Search உடன் மேலும் செழுமையான இணையத் தேடல் முடிவுகளுக்காக Tavily-ஐ தேடல் வழங்குநராகச் சேர்க்கிறது
- **Theme-aware built-in skills** — Block Themes, Classic Themes, Kadence Blocks, மற்றும் Kadence Theme skill வழிகாட்டிகள் இப்போது plugin உடன் வழங்கப்படுகின்றன
- **தள உருவாக்கி தொடர்பு படிவ திறன்** — chat interface-இலிருந்து நேரடியாக எந்தப் பக்கத்திலும் தொடர்பு படிவத்தைச் சேர்க்கும்

### மேம்படுத்தப்பட்டது {#improved-1}

- **WooCommerce ஒருங்கிணைப்பு மறுவடிவமைக்கப்பட்டது** — சிறந்த நம்பகத்தன்மை மற்றும் இணக்கத்திற்காக இப்போது native WooCommerce APIs-ஐப் பயன்படுத்துகிறது
- **வழங்குநர் பட்டியல் தானாக புதுப்பிக்கிறது** — எந்த plugin செயல்படுத்தப்பட்டாலோ செயலிழக்கச் செய்யப்பட்டாலோ

### சரிசெய்யப்பட்டது {#fixed-2}

- **navigate-to திறன்** — சில admin பக்கங்களில் முடிவில்லா reload loop சரிசெய்யப்பட்டது
- **list-posts திறன்** — இப்போது category மற்றும் tag பெயர்களை slugs ஆகச் சரியாகத் தீர்மானிக்கிறது
- **WP-CLI commands** — முந்தைய refactor-க்குப் பிறகு காணாமல் போன namespace aliases மீட்டமைக்கப்பட்டன
- **Event automation** — automation அட்டவணைகள் இன்னும் உருவாக்கப்படாத தளங்களைச் சீராகக் கையாளுகிறது
- **memory-save திறன்** — இப்போது system instruction builder-இல் சரியான namespace prefix-ஐப் பயன்படுத்துகிறது
- **Scalar tool results** — இப்போது AI-க்கு திருப்பி அனுப்புவதற்கு முன் சரியாக wrapped செய்யப்படுகின்றன
- **பயன்பாட்டு புள்ளிவிவரங்கள்** — பழைய பதிப்புகளிலிருந்து மேம்படுத்தும்போது legacy ability key format-ஐ இப்போது சரியாகக் கையாளுகின்றன
