# Changelog

## 1.16.0 — வெளியிடப்பட்டது 2026-05-20

### புதியவை (New)

- **லோகோ SVG உருவாக்குதல் திறன் (Generate Logo SVG ability)** — Theme Builder இப்போது namespace-safe sanitisation உடன் தனிப்பயன் லோகோ SVG-களை உருவாக்கவும் உட்பொதிக்கவும் முடியும்.
- **கண்டுபிடிப்பு நேர்காணலில் புகைப்படப் பதிவேற்றம் (Photo upload in discovery interview)** — Theme Builder discovery interview இப்போது செழுமையான வடிவமைப்பு சூழலுக்காக புகைப்படப் பதிவேற்ற படிவத்தையும் கொண்டுள்ளது.
- **பலகை மாறுபாடு சரிபார்ப்பு திறன் (Validate Palette Contrast ability)** — ஒரு தீம்에ப் பயன்படுத்துவதற்கு முன், வண்ண ஜோடிகளை WCAG இணக்கத்திற்காகச் சரிபார்க்கலாம்.
- **ஹோஸ்பிடாலிட்டி மெனுக்கள் (Hospitality menus)** — Theme Builder இப்போது ஹோஸ்பிடாலிட்டி வணிகங்களுக்காக கட்டமைக்கப்பட்ட உணவு மற்றும் பான மெனுப் பக்கங்களை உருவாக்க முடியும்.
- **டெஸ்க்டாப் மற்றும் மொபைல் முன்னோட்ட ரெண்டரிங் (Desktop and mobile preview rendering)** — வடிவமைப்பு திசையைத் தேர்ந்தெடுக்கும்போது, டெஸ்க்டாப் மற்றும் மொபைல் சாதனங்களில் உங்கள் வடிவமைப்பிற்கான முன்னோட்டத்தை (preview) பார்க்கலாம்.
- **வழிசெலுத்தல் லேபிள் அளவுரு (Navigation label parameter)** — Create Menu திறன் இப்போது பக்கத் தலைப்பிலிருந்து வேறுபட்ட தனித்துவமான `navigation_label`-ஐ ஆதரிக்கிறது.
- **டயர் 1 கருவி கிடைக்கும் தன்மை (Tier 1 tool availability)** — sd-ai-agent/site-scrape இப்போது Theme Builder-இல் இயல்பாகக் கிடைக்கும் ஒரு Tier 1 கருவியாக உள்ளது.

### சரி செய்யப்பட்டவை (Fixed)

- **AI கிளையன்ட் கேச் (AI Client cache)** — இப்போது ட்ரான்சியண்ட்ஸ் (transients) மூலம் ஆதரிக்கப்படுகிறது. இது நீண்ட நேரம் இயங்கும் ஏஜென்ட் பணிகளில் தரவு இழப்பைத் தடுக்கிறது.
- **Plugin வரிசை செயல் இணைப்புகள் (Plugin row action links)** — தெளிவுக்காகச் சரிசெய்யப்பட்டு, பெயர் மாற்றப்பட்டுள்ளது.

## 1.10.0 — வெளியிடப்பட்டது 2026-05-05

### புதியவை (New)

- **Tavily இணைய தேடல் (Tavily internet search)** — Brave Search உடன் சேர்த்து, செழுமையான இணைய தேடல் முடிவுகளுக்காக Tavily-ஐ ஒரு தேடல் வழங்குநராகச் சேர்க்கலாம்.
- **Theme-அறிந்த உள்ளமைக்கப்பட்ட திறன்கள் (Theme-aware built-in skills)** — Block Themes, Classic Themes, Kadence Blocks, மற்றும் Kadence Theme திறன் வழிகாட்டிகள் இப்போது plugin உடன் வருகின்றன.
- **சைட் பில்டர் தொடர்பு படிவம் திறன் (Site builder contact form ability)** — சாட் இடைமுகத்திலிருந்து எந்தப் பக்கத்திலும் நேரடியாக ஒரு தொடர்பு படிவத்தைச் சேர்க்கலாம்.

### மேம்படுத்தப்பட்டவை (Improved)

- **WooCommerce ஒருங்கிணைப்பு மறுசீரமைக்கப்பட்டது (WooCommerce integration refactored)** — இப்போது சிறந்த நம்பகத்தன்மை மற்றும் இணக்கத்தன்மைக்காக native WooCommerce APIs-ஐப் பயன்படுத்துகிறது.
- **சேவை வழங்குநர் பட்டியல் தானாக புதுப்பிக்கப்படும் (Provider list refreshes automatically)** — ஏதேனும் ஒரு plugin செயல்படுத்தப்படும் அல்லது செயலிழக்கப்படும்போது இது நடக்கும்.

### சரி செய்யப்பட்டவை (Fixed)

- **navigate-to திறன் (navigate-to ability)** — சில நிர்வாகப் பக்கங்களில் முடிவற்ற ரீலோட் சுழற்சியை (infinite reload loop) சரிசெய்தது.
- **list-posts திறன் (list-posts ability)** — இப்போது வகை மற்றும் குறிச்சொல் பெயர்களை ஸ்லக்ஸ் (slugs) ஆகச் சரியாகத் தீர்க்கிறது.
- **WP-CLI கட்டளைகள் (WP-CLI commands)** — முந்தைய மறுசீரமைப்புக்குப் பிறகு காணாமல் போன namespace aliases மீட்டெடுக்கப்பட்டன.
- **நிகழ்வு தானியக்கம் (Event automation)** — தானியக்க அட்டவணைகள் இன்னும் உருவாக்கப்படாத தளங்களைச் சிறப்பாகக் கையாளுகிறது.
- **memory-save திறன் (memory-save ability)** — இப்போது system instruction builder-இல் சரியான namespace prefix-ஐப் பயன்படுத்துகிறது.
- **Scalar கருவி முடிவுகள் (Scalar tool results)** — AI-க்குத் திரும்ப அனுப்பப்படுவதற்கு முன் இப்போது சரியாகப் பொதிந்து அனுப்பப்படுகிறது.
- **பயன்பாட்டு புள்ளிவிவரங்கள் (Usage statistics)** — பழைய பதிப்புகளிலிருந்து மேம்படுத்தப்படும்போது, பழைய ability key வடிவத்தை இப்போது சரியாகக் கையாளுகிறது.
