# லோகோ SVG-ஐ உருவாக்குதல்

**Generate Logo SVG** என்ற வசதி, Theme Builder மூலம், தானாகவே namespace-பாதுகாப்பான sanitisation உடன், தனிப்பயன் லோகோ SVG-களை உங்கள் WordPress தளத்தில் நேரடியாக உருவாக்கவும் உட்பொதிக்கவும் அனுமதிக்கிறது.

## கண்ணோட்டம்

இந்த வசதி, உங்கள் தளத்தின் பிராண்டிங் திசை மற்றும் வடிவமைப்பு விருப்பங்களின் அடிப்படையில் அளவிடக்கூடிய வெக்டார் கிராபிக்ஸ் (SVG) லோகோக்களை உருவாக்கும். உருவாக்கப்பட்ட SVG-கள், காட்சித் தரத்தை இழக்காமல், WordPress-ல் பயன்படுத்த பாதுகாப்பானதா என்பதை உறுதிப்படுத்த தானாகவே sanitise செய்யப்படுகின்றன.

## அளவுருக்கள்

| அளவுரு | வகை | தேவை | விளக்கம் |
|-----------|------|----------|-------------|
| `site_name` | string | ஆம் | லோகோவை உருவாக்க வேண்டிய தளத்தின் பெயர் |
| `style` | string | ஆம் | வடிவமைப்பு பாணி (உதாரணமாக, "modern", "classic", "minimalist", "playful") |
| `colors` | array | இல்லை | லோகோவில் பயன்படுத்த வேண்டிய ஹெக்ஸ் வண்ணக் குறியீடுகளின் வரிசை (உதாரணமாக, `["#678233", "#ffffff"]`) |
| `width` | number | இல்லை | பிக்சல்களில் SVG அகலம் (இயல்புநிலை: 200) |
| `height` | number | இல்லை | பிக்சல்களில் SVG உயரம் (இயல்புநிலை: 200) |
| `include_text` | boolean | இல்லை | லோகோவில் தளத்தின் பெயரை உரையாக சேர்க்க வேண்டுமா (இயல்புநிலை: true) |

## வெளியீட்டு வடிவம்

இந்த வசதி பின்வரும் கட்டமைப்பைக் கொண்ட ஒரு SVG சரத்தை (string) திருப்பி அளிக்கும்:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG Sanitisation செயல்பாடு

உருவாக்கப்பட்ட SVG-கள் பின்வரும் காரணங்களுக்காக தானியங்கி namespace-பாதுகாப்பான sanitisation செயல்முறைக்கு உட்படுகின்றன:

- **பாதுகாப்பற்ற பண்புகளை நீக்குதல்** — நிகழ்வு கையாளுதல்கள் (event handlers), ஸ்கிரிப்ட்கள் மற்றும் சாத்தியமான ஆபத்தான பண்புகளை நீக்குகிறது.
- **namespace-களைப் பாதுகாத்தல்** — சரியான ரெண்டரிங்கிற்கு SVG namespace-களை (xmlns, xlink) பராமரிக்கிறது.
- **கட்டமைப்பைச் சரிபார்த்தல்** — SVG ஆனது W3C தரங்களுக்கு இணங்குவதை உறுதி செய்கிறது.
- **அடையாள குறியீடுகளை என்கோட் செய்தல்** — உரை உள்ளடக்கத்தில் உள்ள சிறப்பு எழுத்துக்களைச் சரியாகத் தவிர்க்கிறது (escape).
- **வெளிப்புற குறிப்புகளை நீக்குதல்** — வெளிப்புற ஸ்டைல்ஷீட்கள் மற்றும் பட குறிப்புகளை நீக்குகிறது.

இது, கூடுதல் sanitisation தேவையில்லாமல், SVG-ஐ நேரடியாக WordPress-ல் உட்பொதிக்க பாதுகாப்பானது என்பதை உறுதி செய்கிறது.

## பயன்பாட்டு உதாரணம்

**Prompt:**
```
Generate a modern logo for my tech startup called "CloudSync" using blue and white colors.
```

**Result:**
இந்த வசதி உருவாக்கும் SVG லோகோ பின்வருவனவற்றை உள்ளடக்கியது:
- "CloudSync" என்ற தளத்தின் பெயரை இணைக்கிறது.
- குறிப்பிட்ட நீலம் மற்றும் வெள்ளை வண்ணத் திட்டத்தைப் பயன்படுத்துகிறது.
- நவீன வடிவமைப்பு கொள்கைகளைப் பின்பற்றுகிறது.
- தானாகவே sanitise செய்யப்பட்டு பயன்படுத்தத் தயாராக உள்ளது.

## Theme Builder உடன் ஒருங்கிணைப்பு

Theme Builder-ன் வடிவமைப்பு திசைத் தேர்வைப் பயன்படுத்தும்போது, Generate Logo SVG வசதி:

1. உங்கள் வடிவமைப்பு திசை மற்றும் வண்ணத் தட்டுகளைப் பகுப்பாய்வு செய்கிறது.
2. உங்கள் விருப்பங்களுக்குப் பொருந்தக்கூடிய தனிப்பயன் SVG லோகோவை உருவாக்குகிறது.
3. லோகோவை தானாகவே உங்கள் தளத்தின் தலைப்பு/பிராண்டிங் பகுதிக்குள் உட்பொதிக்கிறது.
4. SVG-ஐ WordPress மீடியாவில் ஒரு தனிப்பயன் லோகோவாகச் சேமிக்கிறது.

## சிறந்த நடைமுறைகள்

- **தெளிவான பாணி விருப்பங்களை வழங்குங்கள்** — நீங்கள் விரும்பும் வடிவமைப்பு பாணியை விவரிக்கவும் (நவீன, கிளாசிக், விளையாட்டுத்தனமான போன்றவை).
- **வண்ணங்களை குறிப்பிடுங்கள்** — நிலைத்தன்மைக்காக உங்கள் பிராண்ட் வண்ணங்களைச் சேர்க்கவும்.
- **ரெண்டரிங்கைச் சோதிக்கவும்** — வெவ்வேறு திரை அளவுகளில் லோகோ சரியாகக் காட்டப்படுகிறதா என்பதைச் சரிபார்க்கவும்.
- **மேலும் தனிப்பயனாக்குங்கள்** — அளவு மற்றும் இடத்தை சரிசெய்ய WordPress-ன் லோகோ தனிப்பயனாக்குதல் கருவிகளைப் பயன்படுத்தவும்.

## வரம்புகள்

- SVG லோகோக்கள் நிலையான கிராபிக்ஸ் (அனிமேஷன் அல்ல) ஆக உருவாக்கப்படுகின்றன.
- பல கூறுகளைக் கொண்ட சிக்கலான லோகோக்களுக்கு கைமுறையான செம்மைப்படுத்தல் தேவைப்படலாம்.
- தனிப்பயன் எழுத்துருக்கள் ஆதரிக்கப்படவில்லை; உரை அமைப்பு எழுத்துருக்களைப் பயன்படுத்துகிறது.
- மிக பெரிய அல்லது மிகச் சிறிய பரிமாணங்கள் தரத்தை பாதிக்கலாம்.
