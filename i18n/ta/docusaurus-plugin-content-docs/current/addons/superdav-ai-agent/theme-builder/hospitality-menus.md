# விருந்தோம்பல் மெனுக்கள்

**Hospitality Menus** அம்சம், Theme Builder மூலம் உங்கள் WordPress தளத்தில் நேரடியாக கட்டமைக்கப்பட்ட உணவு மற்றும் பான மெனு பக்கங்களை உருவாக்கவும் உட்பொதிக்கவும் அனுமதிக்கிறது.

## கண்ணோட்டம் (Overview)

Theme Builder இப்போது உணவகங்கள், கஃபேக்கள், 바ர்கள் மற்றும் கேடரிங் சேவைகள் உள்ளிட்ட விருந்தோம்பல் வணிகங்களுக்கான தொழில்முறை, கட்டமைக்கப்பட்ட மெனு பக்கங்களை உருவாக்க முடியும். இந்த மெனுக்கள் உங்கள் தளத்தின் வடிவமைப்பில் முழுமையாக ஒருங்கிணைக்கப்பட்டு, எளிதாகப் புதுப்பிக்கவும் நிர்வகிக்கவும் முடியும்.

## ஆதரிக்கப்படும் விருந்தோம்பல் வகைகள் (Supported Hospitality Types)

- **Restaurants** — முழு சேவை உணவு மெனுக்கள்
- **Cafes** — காபி மற்றும் லேசான உணவு மெனுக்கள்
- **Bars and Lounges** — பானங்கள் மற்றும் ஸ்டார்ட்டர் மெனுக்கள்
- **Bakeries** — பேஸ்ட்ரி மற்றும் ரொட்டி மெனுக்கள்
- **Catering Services** — நிகழ்வு மெனு விருப்பங்கள்
- **Food Trucks** — மொபைல் உணவு சேவை மெனுக்கள்
- **Breweries and Wineries** — விளக்கங்களுடன் கூடிய பான மெனுக்கள்

## மெனு அமைப்பு (Menu Structure)

### மெனு வகைகள் (Menu Categories)

மெனுக்கள் வகைகளாக ஒழுங்கமைக்கப்பட்டுள்ளன:

- **Appetizers** — ஆரம்ப உணவுகள் மற்றும் சிறிய தட்டுகள்
- **Entrees** — முக்கிய உணவுகள்
- **Sides** — துணை உணவுகள் மற்றும் காய்கறிகள்
- **Desserts** — இனிப்புப் பண்டங்கள்
- **Beverages** — பானங்கள் (ஆல்கஹால் மற்றும் ஆல்கஹால் அல்லாதவை)
- **Specials** — தினசரி அல்லது பருவகால சிறப்பு உணவுகள்

### மெனு உருப்படி வடிவம் (Menu Item Format)

ஒவ்வொரு மெனு உருப்படியிலும் பின்வருவன அடங்கும்:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### உருப்படி புலங்கள் (Item Fields)

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | உணவு அல்லது பானத்தின் பெயர் |
| `description` | string | உருப்படியின் விரிவான விளக்கம் |
| `price` | string | விலை (நாணயத்துடன் வடிவமைக்கப்பட்டது) |
| `dietary_info` | array | உணவுத் தகுதிகள் (Vegan, Gluten-Free, போன்றவை) |
| `allergens` | array | பொதுவான ஒவ்வாமை பொருட்கள் (Nuts, Shellfish, போன்றவை) |
| `availability` | string | எப்போது கிடைக்கும் (Daily, Seasonal, போன்றவை) |

## விருந்தோம்பல் மெனுக்களை உருவாக்குதல் (Creating Hospitality Menus)

### படி 1: மெனு தகவலை வழங்குதல் (Provide Menu Information)

Theme Builder-க்கு உங்கள் மெனுவைப் பற்றி சொல்லுங்கள்:

```
Create a restaurant menu for my Italian restaurant with appetizers,
pasta dishes, main courses, and desserts. Include prices and descriptions.
```

### படி 2: Theme Builder மெனுவை உருவாக்குகிறது (Theme Builder Generates Menu)

Theme Builder:

1. கட்டமைக்கப்பட்ட மெனுப் பக்கத்தை உருவாக்கும்
2. உங்கள் தளத்தின் கருப்பொருளுடன் (theme) பொருந்தும்படி வடிவமைக்கும்
3. உருப்படிகளை வகைகளாக ஒழுங்கமைக்கும்
4. விலைகள் மற்றும் விளக்கங்களை வடிவமைக்கும்
5. உணவுத் தகுதி மற்றும் ஒவ்வாமை தகவல்களைச் சேர்க்கும்

### படி 3: மதிப்பாய்வு செய்து தனிப்பயனாக்குதல் (Review and Customize)

நீங்கள் செய்யக்கூடியவை:

1. மெனு உருப்படிகள் மற்றும் விலைகளைத் திருத்துதல்
2. வகைகளைச் சேர்ப்பது அல்லது நீக்குவது
3. வகைகளுக்குள் உருப்படிகளை மீண்டும் வரிசைப்படுத்துவது
4. விளக்கங்கள் மற்றும் உணவுத் தகவல்களைப் புதுப்பிப்பது
5. ஸ்டைலிங் மற்றும் லேஅவுட்டைச் சரிசெய்வது

## மெனு காட்சி விருப்பங்கள் (Menu Display Options)

### முழு மெனுப் பக்கம் (Full Menu Page)

உங்கள் முழு மெனுவையும் காண்பிக்கும் பிரத்யேகப் பக்கம்:

- வகைகளின் அடிப்படையில் ஒழுங்கமைக்கப்பட்டது
- தேடக்கூடிய மற்றும் வடிகட்டக்கூடியது
- அச்சிட ஏற்ற வடிவமைப்பு
- மொபைலுக்கு ஏற்ற வடிவமைப்பு

### மெனு விட்ஜெட் (Menu Widget)

மற்ற பக்கங்களில் மெனு பிரிவுகளை உட்பொதிக்கவும்:

- முகப்புப் பக்கத்தில் சிறப்பம்சங்கள்
- பக்கவாட்டில் தினசரி சிறப்பு உணவுகள்
- 바ர் பக்கத்தில் பான மெனு
- அடிக்குறிப்பில் இனிப்புப் பண்டங்களின் காட்சி

### மெனு PDF (Menu PDF)

பதிவிறக்கக்கூடிய PDF மெனுவை உருவாக்கவும்:

- தொழில்முறை வடிவமைப்பு
- அச்சிடத் தயாரான தரம்
- படங்கள் மற்றும் விளக்கங்கள் சேர்க்கப்பட்டுள்ளது
- பகிரவும் மின்னஞ்சல் செய்யவும் எளிதானது

## உணவுத் தகுதி மற்றும் ஒவ்வாமை தகவல் (Dietary and Allergen Information)

### உணவுத் தகுதிகள் (Dietary Attributes)

உணவுத் தகவலுடன் உருப்படிகளை குறிக்கவும்:

- **Vegan** — எந்த விலங்குப் பொருட்களும் இல்லை
- **Vegetarian** — இறைச்சி இல்லை
- **Gluten-Free** — சلياக்கு நோயால் பாதிக்கப்பட்டவர்களுக்கு பாதுகாப்பானது
- **Dairy-Free** — பால் பொருட்கள் இல்லை
- **Nut-Free** — மரத்தூள் கொட்டைகள் அல்லது வேர்க்கடலை இல்லை
- **Low-Carb** — குறைவான கார்போஹைட்ரேட்டுகள்
- **High-Protein** — புரதம் நிறைந்த

### ஒவ்வாமை எச்சரிக்கைகள் (Allergen Warnings)

பொதுவான ஒவ்வாமை பொருட்களைச் சேர்க்கவும்:

- **Nuts** — மரத்தூள் கொட்டைகள் மற்றும் வேர்க்கடலை
- **Shellfish** — கிரஸ்டேசியன்கள் மற்றும் மாலஸ்க்கள்
- **Fish** — அனைத்து மீன் இனங்களும்
- **Dairy** — பால் மற்றும் பால் பொருட்கள்
- **Eggs** — கோழி முட்டைகள்
- **Soy** — சோயா பொருட்கள்
- **Gluten** — கோதுமை மற்றும் 글루텐 கொண்ட தானியங்கள்
- **Sesame** — எள் விதைகள் மற்றும் எண்ணெய்

## மெனு மேலாண்மை (Menu Management)

### விலைகளை புதுப்பித்தல் (Updating Prices)

மெனு விலைகளை எளிதாகப் புதுப்பிக்கவும்:

1. மெனுப் பக்கத்திற்குச் செல்லவும்
2. "Edit Menu" என்பதைக் கிளிக் செய்யவும்
3. உருப்படிகளுக்கான விலைகளை புதுப்பிக்கவும்
4. மாற்றங்களைச் சேமிக்கவும்
5. மாற்றங்கள் உடனடியாக உங்கள் தளத்தில் தோன்றும்

### பருவகால உருப்படிகளைச் சேர்த்தல் (Adding Seasonal Items)

பருவகால மெனு மாறுபாடுகளை உருவாக்கவும்:

1. ஒரு புதிய மெனு பதிப்பை உருவாக்கவும்
2. பருவகால உருப்படிகளைச் சேர்க்கவும்
3. உருப்படிகளை "Seasonal" என்று குறிக்கவும்
4. கிடைக்கும் தேதிகளைத் திட்டமிடவும்
5. பருவத்தின் போது தானாகவே காண்பிக்கப்படும்

### சிறப்பு உணவுகளை நிர்வகித்தல் (Managing Specials)

தினசரி அல்லது வாராந்திர சிறப்பு உணவுகளைக் காண்பிக்கவும்:

1. ஒரு "Specials" வகையை உருவாக்கவும்
2. கிடைக்கும் தேதிகளுடன் உருப்படிகளைச் சேர்க்கவும்
3. முகப்புப் பக்கத்தில் சிறப்பு உணவுகளை முன்னிலைப்படுத்தவும்
4. தினசரி அல்லது வாராந்திரமாகப் புதுப்பிக்கவும்
5. பழைய சிறப்பு உணவுகளை ஆர்கைவ் செய்யவும்

## Theme Builder உடன் ஒருங்கிணைப்பு (Integration with Theme Builder)

விருந்தோம்பல் தளங்களுக்காக Theme Builder ஐப் பயன்படுத்தும்போது:

1. **தானியங்கி மெனு கண்டறிதல்** — நீங்கள் உணவு/பான வணிகமாக இருக்கிறீர்களா என்பதைக் கண்டறியும்
2. **மெனுப் பக்க உருவாக்கம்** — தொழில்முறை மெனுப் பக்கங்களை உருவாக்கும்
3. **வடிவமைப்பு பொருத்தம்** — மெனுக்கள் உங்கள் தளத்தின் கருப்பொருளுடன் பொருந்தும்
4. **மொபைல் மேம்படுத்தல்** — மெனுக்கள் தொலைபேசிகளில் அழகாகக் காண்பிக்கப்படும்
5. **SEO மேம்படுத்தல்** — மெனுக்கள் தேடுபொறிக்கு உகந்ததாக இருக்கும்

## சிறந்த நடைமுறைகள் (Best Practices)

### மெனு வடிவமைப்பு (Menu Design)

- **தெளிவான அமைப்பு** — தர்க்கரீதியான வகை அமைப்பு
- **කියවිය හැකි விளக்கங்கள்** — பசியூட்டும் மற்றும் தகவல் நிறைந்தவை
- **நிலையான விலை நிர்ணயம்** — தெளிவான நாணயம் மற்றும் வடிவமைப்பு
- **தொழில்முறை புகைப்படங்கள்** — உயர்தர உணவுப் படங்கள்
- **வெள்ளை இடம் (Whitespace)** — பக்கத்தை நிரம்ப வேண்டாம்

### உள்ளடக்கம் (Content)

- **சரியான விளக்கங்கள்** — உருப்படிகளைத் துல்லியமாக விவரிக்கவும்
- **சிறப்பு உணவுகளை முன்னிலைப்படுத்துதல்** — சிறப்பு உணவுகள் தனித்துத் தெரியும்படி செய்யவும்
- **ஒவ்வாமைகளைச் சேர்த்தல்** — ஒவ்வாமை பொருட்களை எப்போதும் பட்டியலிடவும்
- **தொடர்ந்து புதுப்பித்தல்** — விலைகள் மற்றும் உருப்படிகளைப் புதுப்பி நிலையில் வைக்கவும்
- **பசியூட்டும் மொழியைப் பயன்படுத்துதல்** — உருப்படிகளை சுவையாக ஒலிக்கச் செய்யவும்

### அணுகல்தன்மை (Accessibility)

- **කියවිය හැකි எழுத்துருக்கள்** — தெளிவான, படிக்கக்கூடிய எழுத்துருக்களைப் பயன்படுத்தவும்
- **போதுமான மாறுபாடு** — உரை படிக்கக்கூடியதாக இருப்பதை உறுதி செய்யவும்
- **உணவுத் தラベル்கள்** — உணவுத் தேர்வுகளைத் தெளிவாகக் குறிக்கவும்
- **ஒவ்வாமை எச்சரிக்கைகள்** — ஒவ்வாமை பொருட்களை முக்கியமாக പ്രദർശിപ്പിക്കவும்
- **மொபைலுக்கு ஏற்றது** — அனைத்து சாதனங்களிலும் சோதிக்கவும்
