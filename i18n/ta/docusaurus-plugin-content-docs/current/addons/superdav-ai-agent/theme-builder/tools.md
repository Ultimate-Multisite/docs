# கருவிகள் மற்றும் உள்ளமைவு

Theme Builder உங்கள் WordPress தளத்தை பகுப்பாய்வு செய்யவும், வடிவமைக்கவும், உருவாக்கவும் சில கருவிகளைப் பயன்படுத்துகிறது. பதிப்பு 1.16.0-ல், **sd-ai-agent/site-scrape இப்போது ஒரு நிலை 1 கருவியாக (Tier 1 tool)** உள்ளது, இது Theme Builder-ல் இயல்பாகவே கிடைக்கிறது.

## கிடைக்கும் கருவிகள்

### நிலை 1 கருவிகள் (எப்போதும் கிடைக்கும்)

நிலை 1 கருவிகள் கூடுதல் உள்ளமைவு எதுவும் இல்லாமல் Theme Builder-ல் இயல்பாகவே கிடைக்கின்றன.

#### sd-ai-agent/site-scrape

**நோக்கம்:** வடிவமைப்பு உத்வேகம், உள்ளடக்க அமைப்பு மற்றும் செயல்பாடுகளைப் பிரித்தெடுக்க இருக்கும் வலைத்தளங்களை பகுப்பாய்வு செய்ய உதவுகிறது.

**செயல்பாடுகள்:**

- **வலைத்தள பகுப்பாய்வு** — போட்டியாளர் அல்லது உத்வேகம் தரும் வலைத்தளங்களை ஸ்கிராப் செய்து பகுப்பாய்வு செய்தல்
- **வடிவமைப்பு பிரித்தெடுத்தல்** — வண்ணங்கள், எழுத்துருக்கள் மற்றும் லேஅவுட் வடிவங்களை அடையாளம் காணுதல்
- **உள்ளடக்க அமைப்பு** — பக்க அமைப்பு மற்றும் படிநிலையைப் புரிந்துகொள்ளுதல்
- **அம்ச கண்டறிதல்** — பிளக்இன்கள் மற்றும் செயல்பாடுகளை அடையாளம் காணுதல்
- **செயல்திறன் பகுப்பாய்வு** — பக்க வேகம் மற்றும் உகந்ததாக்குதலைச் சரிபார்த்தல்
- **SEO பகுப்பாய்வு** — மெட்டா டேக்குகள் மற்றும் கட்டமைக்கப்பட்ட தரவை மதிப்பாய்வு செய்தல்

**பயன்பாடு:**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**இது பிரித்தெடுப்பது:**

- வண்ணத் தட்டு மற்றும் வண்ணப் பயன்பாடு
- எழுத்துரு அமைப்பு (fonts மற்றும் அளவுகள்)
- லேஅவுட் அமைப்பு மற்றும் இடைவெளி
- வழிசெலுத்தல் வடிவங்கள்
- உள்ளடக்க அமைப்பு
- படங்கள் மற்றும் மீடியா பயன்பாடு
- ஊடாடும் கூறுகள்
- மொபைல் பதிலளிப்புத் திறன்

**வரம்புகள்:**

- கடவுச்சீர்க் காக்கப்பட்ட தளங்களை ஸ்கிராப் செய்ய முடியாது
- robots.txt மற்றும் தளக் கொள்கைகளை மதிக்கும்
- மாறும் உள்ளடக்கத்தை பிடிக்காமல் போகலாம்
- JavaScript அதிகம் உள்ள தளங்களில் பகுப்பாய்வு குறைவாக இருக்கலாம்
- பெரிய தளங்களை பகுப்பாய்வு செய்ய அதிக நேரம் எடுக்கலாம்

### நிலை 2 கருவிகள் (விரும்பினால்)

நிலை 2 கருவிகள் Theme Builder அமைப்புகளில் வெளிப்படையாக இயக்கப்படும்போது கிடைக்கின்றன.

#### Advanced Analytics

விரிவான செயல்திறன் அளவீடுகளை வழங்குகிறது:

- பக்க ஏற்ற நேரங்கள்
- Core Web Vitals
- SEO மதிப்பெண்
- அணுகல்தன்மை மதிப்பெண்
- சிறந்த நடைமுறைகள் மதிப்பெண்

#### Content Optimizer

பின்வருவனவற்றிற்கான மேம்பாடுகளை பகுப்பாய்வு செய்து பரிந்துரைக்கிறது:

- படிக்க எளிதான தன்மை (Readability)
- SEO உகந்ததாக்குதல்
- முக்கிய வார்த்தை பயன்பாடு
- உள்ளடக்க அமைப்பு
- அழைப்பு-க்கு-செயல் (Call-to-action) வைக்கும் இடம்

### நிலை 3 கருவிகள் (Premium)

நிலை 3 கருவிகளுக்கு கூடுதல் உள்ளமைவு அல்லது பிரீமியம் அம்சங்கள் தேவை.

#### AI Content Generator

பின்வருவனவற்றிற்கான உள்ளடக்கத்தை உருவாக்குகிறது:

- தயாரிப்பு விளக்கங்கள்
- சேவைப் பக்கங்கள்
- வலைப்பதிவு இடுகைகள்
- மெட்டா விளக்கங்கள்
- அழைப்பு-க்கு-செயல் உரை

#### Advanced Design Tools

மேம்பட்ட வடிவமைப்பு திறன்களை வழங்குகிறது:

- தனிப்பயன் CSS உருவாக்கம்
- அனிமேஷன் உருவாக்கம்
- ஊடாடும் கூறு வடிவமைப்பு
- மேம்பட்ட வண்ணக் கோட்பாடு
- எழுத்துரு உகந்ததாக்குதல்

## கருவி உள்ளமைவு

### கருவிகளை இயக்குதல்

Theme Builder-ல் கூடுதல் கருவிகளை இயக்க:

1. **Gratis AI Agent → Settings** என்பதற்குச் செல்லவும்
2. **Theme Builder → Tools** என்பதற்குச் செல்லவும்
3. தேவைக்கேற்ப கருவிகளை ஆன்/ஆஃப் செய்யவும்
4. அமைப்புகளைச் சேமிக்கவும்

### கருவி அனுமதிகள்

சில கருவிகளுக்கு அனுமதிகள் தேவை:

- **Site scraping** — இணைய அணுகல் தேவை
- **Analytics** — Google Analytics ஒருங்கிணைப்பு தேவை
- **Content generation** — API keys தேவை
- **Advanced features** — பிரீமியம் சந்தா தேவைப்படலாம்

### API Keys மற்றும் Credentials

தேவைப்படும் கருவிகளுக்கான API keys-ஐ உள்ளமைக்கவும்:

1. **Gratis AI Agent → Settings → API Keys** என்பதற்குச் செல்லவும்
2. ஒவ்வொரு கருவிக்கும் Credentials-ஐ உள்ளிடவும்
3. இணைப்பைச் சோதிக்கவும்
4. பாதுகாப்பாகச் சேமிக்கவும்

## sd-ai-agent/site-scrape பயன்படுத்துதல்

### அடிப்படைப் பயன்பாடு

ஒரு வலைத்தளத்தை பகுப்பாய்வு செய்ய Theme Builder-க்குக் கேட்கவும்:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### குறிப்பிட்ட பகுப்பாய்வு

குறிப்பிட்ட வகையான பகுப்பாய்வை கோரவும்:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### ஒப்பீட்டு பகுப்பாய்வு

பல தளங்களை ஒப்பிடுக:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## கருவி வரம்புகள் மற்றும் கவனிக்க வேண்டியவை

### விகித வரம்பு (Rate Limiting)

- சர்வர் அதிக சுமையடைவதைத் தடுக்க ஸ்கிராப்பிங் விகித வரம்புக்கு உட்பட்டது
- ஒரே தளத்திற்கு பல கோரிக்கைகள் கட்டுப்படுத்தப்படலாம்
- பெரிய தளங்களை பகுப்பாய்வு செய்ய அதிக நேரம் எடுக்கலாம்

### உள்ளடக்கத் துல்லியம்

- மாறும் உள்ளடக்கத்தை முழுமையாகப் பிடிக்காமல் போகலாம்
- JavaScript மூலம் உருவாக்கப்பட்ட உள்ளடக்கம் முழுமையடையாமல் இருக்கலாம்
- நிகழ்நேர தரவு பிரதிபலிக்காமல் இருக்கலாம்
- சில உள்ளடக்கம் கட்டணச் சுவர்களுக்குப் பின்னால் இருக்கலாம்

### சட்ட மற்றும் நெறிமுறை பரிசீலனைகள்

- robots.txt மற்றும் தளக் கொள்கைகளை மதித்தல்
- பதிப்புரிமை பெற்ற உள்ளடக்கத்தை மீண்டும் உருவாக்க ஸ்கிராப் செய்ய வேண்டாம்
- பகுப்பாய்வை உத்வேகத்திற்காகப் பயன்படுத்துங்கள், நகலெடுக்க அல்ல
- பிரித்தெடுக்கப்பட்ட உள்ளடக்கத்தைப் பயன்படுத்த உங்களுக்கு உரிமைகள் உள்ளதா என்பதைச் சரிபார்க்கவும்
- பகுப்பாய்வு செய்யப்பட்ட தளங்களின் சேவை விதிமுறைகளைப் பின்பற்றவும்

### செயல்திறன் தாக்கம்

- பெரிய தளங்களை ஸ்கிராப் செய்ய நேரம் எடுக்கலாம்
- பல ஒரே நேரத்தில் ஸ்கிராப் செய்வது மெதுவாக இருக்கும்
- நெட்வொர்க் இணைப்பு வேகத்தைப் பாதிக்கிறது
- பெரிய தரவுத் தொகுப்புகள் அதிக செயலாக்கத்தை கோரலாம்

## சிறந்த நடைமுறைகள்

### தளப் பகுப்பாய்வைப் பயன்படுத்துதல்

1. **பல தளங்களை பகுப்பாய்வு செய்யுங்கள்** — பல மூலங்களிலிருந்து உத்வேகத்தை சேகரிக்கவும்
2. **அமைப்பு மீது கவனம் செலுத்துங்கள்** — லேஅவுட் மற்றும் அமைப்பு வடிவங்களைக் கற்றுக்கொள்ளவும்
3. **வண்ணங்களை பிரித்தெடுக்கவும்** — வண்ணத் தட்டுகளை தொடக்கப் புள்ளிகளாகப் பயன்படுத்தவும்
4. **எழுத்துரு அமைப்பைப் படிக்கவும்** — உங்களுக்குப் பிடித்த எழுத்துரு சேர்க்கைகளை அடையாளம் காணவும்
5. **வழிசெலுத்தலைப் பார்க்கவும்** — மெனு அமைப்புப் புரிதலைப் பெறவும்

### நெறிமுறை ஸ்கிராப்பிங்

1. **robots.txt ஐ மதியுங்கள்** — தள வழிகாட்டுதல்களைப் பின்பற்றவும்
2. **உள்ளடக்கத்தை நகலெடுக்க வேண்டாம்** — பகுப்பாய்வை உத்வேகத்திற்காக மட்டுமே பயன்படுத்தவும்
3. **உரிமைகளைச் சரிபார்க்கவும்** — பிரித்தெடுக்கப்பட்ட உள்ளடக்கத்தைப் பயன்படுத்த உங்களால் കഴിയுமா என்பதை உறுதிப்படுத்தவும்
4. **மூலங்களைக் குறிப்பிடவும்** — உத்வேக மூலங்களுக்குப் பெருமை சேர்க்கவும்
5. **விதிமுறைகளைப் பின்பற்றவும்** — தளத்தின் சேவை விதிமுறைகளுக்கு இணங்குங்கள்

### கருவி செயல்திறனை அதிகரித்தல்

1. **குறிப்பாகக் கேளுங்கள்** — குறிப்பிட்ட வகையான பகுப்பாய்வை கோரவும்
2. **சூழலை வழங்கவும்** — உங்கள் தளத்தின் நோக்கத்தை விளக்கவும்
3. **எதிர்பார்ப்புகளை அமைக்கவும்** — உங்கள் வடிவமைப்பு இலக்குகளை விவரிக்கவும்
4. **மீண்டும் செய்யவும்** — முடிவுகளின் அடிப்படையில் செம்மைப்படுத்தவும்
5. **கருவிகளை இணைக்கவும்** — விரிவான பகுப்பாய்வுக்காக பல கருவிகளைப் பயன்படுத்தவும்

### தளம் ஸ்கிராப் செய்யவில்லை

- தளம் பொதுவில் அணுகக்கூடியதா என்று சரிபார்க்கவும்
- robots.txt ஸ்கிராப்பிங்கிற்கு அனுமதிக்கிறதா என்று சரிபார்க்கவும்
- வேறு தளத்தை முயற்சிக்கவும்
- இணைய இணைப்பைச் சரிபார்க்கவும்
- சிக்கல் தொடர்ந்தால் ஆதரவைத் தொடர்பு கொள்ளவும்

### முழுமையற்ற பகுப்பாய்வு

- தளத்தில் மாறும் உள்ளடக்கம் இருக்கலாம்
- JavaScript முழுமையாக ரெண்டர் செய்யப்படாமல் இருக்கலாம்
- பெரிய தளங்கள் காலாவதியாகலாம் (timeout)
- அதற்குப் பதிலாக குறிப்பிட்ட பக்கங்களை பகுப்பாய்வு செய்ய முயற்சிக்கவும்
- குறிப்பிட்ட பகுப்பாய்வு வகைகளைக் கோரவும்

### மெதுவான செயல்திறன்

- பெரிய தளங்களை பகுப்பாய்வு செய்ய அதிக நேரம் எடுக்கும்
- பல ஒரே நேரத்தில் ஸ்கிராப் செய்வது மெதுவாக இருக்கும்
- நெட்வொர்க் இணைப்பு வேகத்தைப் பாதிக்கிறது
- உச்ச நேரத்திற்கு வெளியே முயற்சிக்கவும்
- முதலில் சிறிய பிரிவுகளை பகுப்பாய்வு செய்யவும்
