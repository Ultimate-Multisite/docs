# கருவிகள் மற்றும் உள்ளமைவு

Theme Builder உங்கள் WordPress தளத்தை பகுப்பாய்வு செய்ய, வடிவமைக்க, உருவாக்க பல கருவிகளைப் பயன்படுத்துகிறது. பதிப்பு 1.16.0-இல், **sd-ai-agent/site-scrape இப்போது Tier 1 கருவியாக உள்ளது**. அதனால் இது Theme Builder-இல் இயல்பாகக் கிடைக்கும். Superdav AI Agent v1.18.0 நம்பகமான டெவலப்பர் பணிச்சூழல்களுக்கு தனியாக விநியோகிக்கப்படும் Advanced துணைத் தொகுப்பையும் அறிமுகப்படுத்துகிறது; அந்தக் கருவிகள் WordPress.org மையத் தொகுப்பின் பகுதியாக இல்லை. அவை தனியாக நிறுவப்பட்டு அங்கீகரிக்கப்பட வேண்டும்.

## கிடைக்கும் கருவிகள்

### Tier 1 கருவிகள் (எப்போதும் கிடைக்கும்)

Tier 1 கருவிகள் கூடுதல் உள்ளமைவு இல்லாமல் Theme Builder-இல் இயல்பாகக் கிடைக்கும்.

#### sd-ai-agent/site-scrape

**நோக்கம்:** வடிவமைப்பு ஊக்கம், உள்ளடக்க அமைப்பு, செயல்பாட்டை எடுக்க உள்ள இணையதளங்களைப் பகுப்பாய்வு செய்யும்.

**திறன்கள்:**

- **இணையதளப் பகுப்பாய்வு** — போட்டியாளர் அல்லது ஊக்கமளிக்கும் இணையதளங்களை scrape செய்து பகுப்பாய்வு செய்யும்
- **வடிவமைப்பு எடுத்தல்** — நிறங்கள், எழுத்துருக்கள், தளவமைப்பு முறைகளை அடையாளம் காணும்
- **உள்ளடக்க அமைப்பு** — பக்க ஒழுங்கமைப்பு மற்றும் படிநிலையைப் புரிந்துகொள்ளும்
- **அம்சக் கண்டறிதல்** — plugins மற்றும் செயல்பாட்டை அடையாளம் காணும்
- **செயல்திறன் பகுப்பாய்வு** — பக்க வேகம் மற்றும் மேம்படுத்தலைச் சரிபார்க்கும்
- **SEO பகுப்பாய்வு** — meta குறிச்சொற்கள் மற்றும் கட்டமைக்கப்பட்ட தரவை மதிப்பாய்வு செய்யும்

**பயன்பாடு:**

```
example.com-இன் வடிவமைப்பைப் பகுப்பாய்வு செய்து, அதை என் தளத்திற்கு ஊக்கமாகப் பயன்படுத்துங்கள்.
```

**இது எதை எடுக்கிறது:**

- நிறத் தொகுப்பு மற்றும் நிறப் பயன்பாடு
- Typography (எழுத்துருக்கள் மற்றும் அளவுகள்)
- தளவமைப்பு அமைப்பு மற்றும் இடைவெளி
- வழிசெலுத்தல் முறைகள்
- உள்ளடக்க ஒழுங்கமைப்பு
- படங்கள் மற்றும் ஊடகப் பயன்பாடு
- தொடர்பாடும் கூறுகள்
- மொபைல் பதிலளிக்கும் தன்மை

**வரம்புகள்:**

- கடவுச்சொல்லால் பாதுகாக்கப்பட்ட தளங்களை scrape செய்ய முடியாது
- robots.txt மற்றும் தளக் கொள்கைகளை மதிக்கும்
- மாறும் உள்ளடக்கத்தைப் பிடிக்காமல் இருக்கலாம்
- JavaScript அதிகமாகப் பயன்படுத்தும் தளங்களில் பகுப்பாய்வு வரம்பாக இருக்கலாம்
- பெரிய தளங்களைப் பகுப்பாய்வு செய்ய அதிக நேரம் ஆகலாம்

### Tier 2 கருவிகள் (விருப்பமானவை)

Theme Builder அமைப்புகளில் வெளிப்படையாக இயக்கப்பட்டால் Tier 2 கருவிகள் கிடைக்கும்.

#### மேம்பட்ட பகுப்பாய்வு

விரிவான செயல்திறன் அளவுகோல்களை வழங்குகிறது:

- பக்க ஏற்ற நேரங்கள்
- Core Web Vitals
- SEO மதிப்பெண்
- அணுகல்தன்மை மதிப்பெண்
- சிறந்த நடைமுறைகள் மதிப்பெண்

#### உள்ளடக்க மேம்படுத்தி

இவற்றுக்கான மேம்பாடுகளைப் பகுப்பாய்வு செய்து பரிந்துரைக்கிறது:

- வாசிப்புத் தெளிவு
- SEO மேம்படுத்தல்
- முக்கியச் சொல் பயன்பாடு
- உள்ளடக்க அமைப்பு
- செயல் அழைப்பு அமைவிடம்

### Tier 3 கருவிகள் (Premium)

Tier 3 கருவிகளுக்கு கூடுதல் உள்ளமைவு அல்லது premium அம்சங்கள் தேவை.

#### AI உள்ளடக்க உருவாக்கி

இவற்றுக்கான உள்ளடக்கத்தை உருவாக்குகிறது:

- தயாரிப்பு விளக்கங்கள்
- சேவைப் பக்கங்கள்
- வலைப்பதிவு இடுகைகள்
- Meta விளக்கங்கள்
- செயல் அழைப்பு உரை

#### மேம்பட்ட வடிவமைப்புக் கருவிகள்

மேம்பட்ட வடிவமைப்பு திறன்களை வழங்குகிறது:

- தனிப்பயன் CSS உருவாக்கம்
- Animation உருவாக்கம்
- தொடர்பாடும் கூறு வடிவமைப்பு
- மேம்பட்ட நிறக் கோட்பாடு
- Typography மேம்படுத்தல்

### மேம்பட்ட துணைக் கருவிகள் (தனி தொகுப்பு)

Advanced துணைத் தொகுப்பு, மைய Superdav AI Agent வெளியீட்டிலிருந்து தனியாக விநியோகிக்கப்படுகிறது. அதிக தாக்கமுள்ள கருவிகளை agent பயன்படுத்த நிர்வாகிகள் வெளிப்படையாக அனுமதிக்கும் நம்பகமான டெவலப்பர் மற்றும் தள உரிமையாளர் சூழல்களுக்காக இது உருவாக்கப்பட்டுள்ளது.

Advanced துணைத் திறன்களில் இவை அடங்கலாம்:

- **Filesystem கருவிகள்** — நம்பகமான சூழல்களில் அங்கீகரிக்கப்பட்ட கோப்புகளை ஆய்வு செய்து மாற்றும்
- **Database கருவிகள்** — வெளிப்படையாக அங்கீகரிக்கப்பட்டால் தளத் தரவை query செய்யும் அல்லது புதுப்பிக்கும்
- **WP-CLI கருவிகள்** — WordPress பராமரிப்பு மற்றும் ஆய்வு கட்டளைகளை இயக்கும்
- **REST dispatcher கருவிகள்** — கட்டுப்படுத்தப்பட்ட dispatch பணிச்சூழல்கள் மூலம் பதிவுசெய்யப்பட்ட REST endpoints-ஐ அழைக்கும்
- **Plugin builder கருவிகள்** — plugin code-ஐ scaffold செய்து படிப்படியாக மேம்படுத்தும்
- **Git snapshot கருவிகள்** — ஆபத்தான டெவலப்பர் செயல்பாடுகளுக்கு முன் snapshots உருவாக்கும்
- **பயனர் மேலாண்மை கருவிகள்** — அனுமதிக்கப்பட்ட இடங்களில் பயனர் பதிவுகளை மதிப்பாய்வு செய்ய அல்லது சரிசெய்ய உதவும்
- **Benchmark கருவிகள்** — வளர்ச்சி மதிப்பாய்வுக்காக செயல்திறன் அல்லது திறன் benchmarks இயக்கும்

இவற்றை ஒவ்வொரு நிறுவலுக்கும் பொதுவாகக் கிடைக்கும் Theme Builder கருவிகளாக ஆவணப்படுத்த வேண்டாம். Advanced துணைத் தொகுப்பு நிறுவப்பட்டு, இயக்கப்பட்டு, நம்பகமான நிர்வாகிகளுக்குள் கட்டுப்படுத்தப்பட்டிருந்தால் மட்டுமே இவை கிடைக்கும்.

## கருவி உள்ளமைவு

### கருவிகளை இயக்குதல்

Theme Builder-இல் கூடுதல் கருவிகளை இயக்க:

1. **Gratis AI Agent → Settings** என்பதற்கு செல்லவும்
2. **Theme Builder → Tools** என்பதற்கு செல்லவும்
3. தேவைக்கேற்ப கருவிகளை on/off மாற்றவும்
4. அமைப்புகளைச் சேமிக்கவும்

### கருவி அனுமதிகள்

சில கருவிகளுக்கு அனுமதிகள் தேவை:

- **தள scrape செய்தல்** — இணைய அணுகல் தேவை
- **பகுப்பாய்வு** — Google Analytics ஒருங்கிணைப்பு தேவை
- **உள்ளடக்க உருவாக்கம்** — API keys தேவை
- **மேம்பட்ட அம்சங்கள்** — premium subscription தேவைப்படலாம்
- **மேம்பட்ட துணைக் கருவிகள்** — தனியாக விநியோகிக்கப்படும் Advanced தொகுப்பும் வெளிப்படையான நிர்வாகி நம்பிக்கையும் தேவை

### API Keys மற்றும் சான்றுகள்

அவை தேவைப்படும் கருவிகளுக்கான API keys-ஐ உள்ளமைக்கவும்:

1. **Gratis AI Agent → Settings → API Keys** என்பதற்கு செல்லவும்
2. ஒவ்வொரு கருவிக்கும் சான்றுகளை உள்ளிடவும்
3. இணைப்பைச் சோதிக்கவும்
4. பாதுகாப்பாகச் சேமிக்கவும்

### Advanced துணைத் தொகுப்பை நிறுவுதல்

Advanced துணைத் தொகுப்பு WordPress.org மைய ZIP-இலிருந்து தனியாக வெளியிடப்படுகிறது. வெளியீட்டுக்கான அதிகாரப்பூர்வ திட்ட விநியோக வழியிலிருந்து மட்டுமே அதை நிறுவவும். பின்னர் அணுகலை நம்பகமான நிர்வாகிகளுக்குள் கட்டுப்படுத்தவும். Production தளத்தில் filesystem, database, WP-CLI, REST dispatcher, plugin builder, பயனர் மேலாண்மை, அல்லது benchmark கருவிகளை இயக்குவதற்கு முன் மனித அங்கீகார வாயில்கள் மற்றும் git snapshot நடத்தை மதிப்பாய்வு செய்யவும்.

## sd-ai-agent/site-scrape பயன்படுத்துதல்

### அடிப்படை பயன்பாடு

ஒரு இணையதளத்தைப் பகுப்பாய்வு செய்ய Theme Builder-ஐ கேளுங்கள்:

```
competitor.com-இல் உள்ள என் போட்டியாளர் தளத்தின் வடிவமைப்பைப் பகுப்பாய்வு செய்து,
என் தளத்திற்கு வடிவமைப்பு மேம்பாடுகளைப் பரிந்துரைக்கவும்.
```

### குறிப்பிட்ட பகுப்பாய்வு

குறிப்பிட்ட வகை பகுப்பாய்வுகளை கோருங்கள்:

```
example.com-இலிருந்து நிறத் தொகுப்பை எடுத்து, அதை ஊக்கமாகப் பயன்படுத்தவும்.
```

```
example.com-இன் வழிசெலுத்தல் அமைப்பைப் பகுப்பாய்வு செய்து, அதே போன்ற
ஒழுங்கமைப்பை என் தளத்தில் பயன்படுத்தவும்.
```

```
example.com-இன் மொபைல் பதிலளிக்கும் தன்மையைச் சரிபார்த்து, என் தளமும்
அதே அளவு பதிலளிப்பதைக் உறுதிசெய்யவும்.
```

### ஒப்பீட்டு பகுப்பாய்வு

பல தளங்களை ஒப்பிடுங்கள்:

```
site1.com மற்றும் site2.com-இன் வடிவமைப்புகளை ஒப்பிட்டு, இரண்டின் சிறந்த கூறுகளை
இணைக்கும் ஒரு வடிவமைப்பை உருவாக்கவும்.
```

## கருவி வரம்புகள் மற்றும் கவனிக்க வேண்டியவை

### Rate Limiting

- சர்வர் அதிகச் சுமையைத் தடுக்க Scraping விகித வரம்பில் உள்ளது
- அதே தளத்திற்கான பல கோரிக்கைகள் மெதுவாக்கப்படலாம்
- பெரிய தளங்களை பகுப்பாய்வு செய்ய அதிக நேரம் ஆகலாம்

### உள்ளடக்கத் துல்லியம்

- மாறும் உள்ளடக்கம் முழுமையாகப் பிடிக்கப்படாமல் இருக்கலாம்
- JavaScript மூலம் காட்டப்படும் உள்ளடக்கம் முழுமையற்றதாக இருக்கலாம்
- நேரடி தரவு பிரதிபலிக்கப்படாமல் இருக்கலாம்
- சில உள்ளடக்கம் கட்டணச் சுவர்களுக்குப் பின்னால் இருக்கலாம்

### சட்ட மற்றும் ஒழுக்கக் கருத்துகள்

- robots.txt மற்றும் தளக் கொள்கைகளை மதிக்கவும்
- பதிப்புரிமை உள்ள உள்ளடக்கத்தை மீளுருவாக்கத்திற்காக scrape செய்ய வேண்டாம்
- பகுப்பாய்வை நகலெடுப்பதற்காக அல்ல, ஊக்கத்திற்காகப் பயன்படுத்தவும்
- எடுத்தெடுக்கப்பட்ட உள்ளடக்கத்தைப் பயன்படுத்த உங்களுக்கு உரிமைகள் உள்ளனவா என்பதைச் சரிபார்க்கவும்
- பகுப்பாய்வு செய்யப்பட்ட தளங்களின் சேவை விதிமுறைகளைப் பின்பற்றவும்

### செயல்திறன் தாக்கம்

- பெரிய தளங்களை scrape செய்ய நேரம் ஆகலாம்
- ஒரே நேரத்தில் பல scrapes செய்தால் மெதுவாக இருக்கலாம்
- நெட்வொர்க் இணைப்பு வேகத்தைப் பாதிக்கும்
- பெரிய தரவுத்தொகுப்புகளுக்கு அதிக செயலாக்கம் தேவைப்படலாம்

## சிறந்த நடைமுறைகள்

### தளப் பகுப்பாய்வைப் பயன்படுத்துதல்

1. **பல தளங்களை பகுப்பாய்வு செய்யவும்** — பல மூலங்களிலிருந்து ஊக்கம் சேகரிக்கவும்
2. **அமைப்பில் கவனம் செலுத்தவும்** — தளவமைப்பு மற்றும் ஒழுங்கமைப்பு முறைகளை அறியவும்
3. **நிறங்களை எடுத்தெடுக்கவும்** — நிறத் தொகுப்புகளை தொடக்கப் புள்ளிகளாகப் பயன்படுத்தவும்
4. **எழுத்துருவை ஆய்வு செய்யவும்** — நீங்கள் விரும்பும் எழுத்துரு சேர்க்கைகளை அடையாளம் காணவும்
5. **வழிசெலுத்தலை மதிப்பாய்வு செய்யவும்** — மெனு ஒழுங்கமைப்பைப் புரிந்துகொள்ளவும்

### ஒழுக்கமான Scraping

1. **robots.txt-ஐ மதிக்கவும்** — தள வழிகாட்டுதல்களைப் பின்பற்றவும்
2. **உள்ளடக்கத்தை நகலெடுக்க வேண்டாம்** — பகுப்பாய்வை ஊக்கத்திற்காக மட்டுமே பயன்படுத்தவும்
3. **உரிமைகளைச் சரிபார்க்கவும்** — எடுத்தெடுக்கப்பட்ட உள்ளடக்கத்தைப் பயன்படுத்த முடியுமா என்பதை உறுதிசெய்யவும்
4. **மூலங்களைக் குறிப்பிடவும்** — ஊக்கமளித்த மூலங்களுக்கு புகழளிக்கவும்
5. **விதிமுறைகளைப் பின்பற்றவும்** — தளத்தின் சேவை விதிமுறைகளுக்கு இணங்கவும்

### கருவி செயல்திறனை அதிகப்படுத்துதல்

1. **குறிப்பாக இருங்கள்** — குறிப்பிட்ட வகை பகுப்பாய்வுகளை கோரவும்
2. **சூழலை வழங்கவும்** — உங்கள் தளத்தின் நோக்கத்தை விளக்கவும்
3. **எதிர்பார்ப்புகளை அமைக்கவும்** — உங்கள் வடிவமைப்பு இலக்குகளை விவரிக்கவும்
4. **மீண்டும் மேம்படுத்தவும்** — முடிவுகளின் அடிப்படையில் செம்மைப்படுத்தவும்
5. **கருவிகளை இணைக்கவும்** — முழுமையான பகுப்பாய்வுக்கு பல கருவிகளைப் பயன்படுத்தவும்

## சிக்கல் தீர்ப்பு

### தளத்தை Scrape செய்ய முடியவில்லை

- தளம் பொதுவாக அணுகக்கூடியதா என்பதைச் சரிபார்க்கவும்
- robots.txt scraping-ஐ அனுமதிக்கிறதா என்பதை உறுதிசெய்யவும்
- வேறு தளத்தை முயற்சிக்கவும்
- இணைய இணைப்பைச் சரிபார்க்கவும்
- சிக்கல் தொடர்ந்தால் ஆதரவைத் தொடர்புகொள்ளவும்

### முழுமையற்ற பகுப்பாய்வு

- தளத்தில் மாறும் உள்ளடக்கம் இருக்கலாம்
- JavaScript முழுமையாக render செய்யப்படாமல் இருக்கலாம்
- பெரிய தளங்களுக்கு நேரம் முடிவடையலாம்
- அதற்கு பதிலாக குறிப்பிட்ட பக்கங்களை பகுப்பாய்வு செய்ய முயற்சிக்கவும்
- குறிப்பிட்ட பகுப்பாய்வு வகைகளை கோரவும்

### மெதுவான செயல்திறன்

- பெரிய தளங்களை பகுப்பாய்வு செய்ய அதிக நேரம் ஆகும்
- ஒரே நேரத்தில் பல scrapes மெதுவாக இருக்கும்
- நெட்வொர்க் இணைப்பு வேகத்தைப் பாதிக்கும்
- உச்சமற்ற நேரங்களில் முயற்சிக்கவும்
- முதலில் சிறிய பிரிவுகளை பகுப்பாய்வு செய்யவும்
