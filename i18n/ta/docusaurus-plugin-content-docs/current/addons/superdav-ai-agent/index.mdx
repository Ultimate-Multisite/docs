# Superdav AI Agent

Superdav AI Agent என்பது ஒரு மேம்பட்ட WordPress AI உதவியாளர் ஆகும். இது Gratis AI Agent-ஐ theme-aware திறன்கள், மேம்படுத்தப்பட்ட இணைய தேடல் திறன்கள் மற்றும் улучப்பட்ட site builder அம்சங்களுடன் விரிவாக்குகிறது. Version 1.16.0 மூலம் தனிப்பயன் லோகோ உருவாக்கம், Palette contrast validation, discovery interview-ல் photo upload, structured hospitality menu generation, மற்றும் Theme Builder-ல் desktop/mobile preview rendering போன்ற அம்சங்கள் சேர்க்கப்பட்டுள்ளன.

## முக்கிய அம்சங்கள் (Key Features)

- **Theme-aware skills** — உங்கள் செயலில் உள்ள WordPress theme மற்றும் நிறுவப்பட்ட plugins ஆகியவற்றின் அடிப்படையில் வழிகாட்டுதலையும் திறன்களையும் தானாகவே மாற்றியமைக்கிறது.
- **பல இணைய தேடல் வழங்குநர்கள் (Multiple internet search providers)** — செழுமையான தேடல் முடிவுகள் மற்றும் மேம்பட்ட ஆராய்ச்சி திறன்களுக்காக Brave Search மற்றும் Tavily ஆகியவற்றுக்கு இடையே தேர்வு செய்யலாம்.
- **Site builder contact forms** — உங்கள் நிறுவப்பட்ட form plugin-ஐப் பயன்படுத்தி chat interface-ல் இருந்து நேரடியாக முழுமையாகச் செயல்படும் contact form-களை உருவாக்கலாம்.
- **மேம்படுத்தப்பட்ட site builder** — சிறந்த plugin discovery மற்றும் error recovery உடன் மேம்படுத்தப்பட்ட பல-கட்ட site build திட்டங்கள்.
- **உள்ளமைக்கப்பட்ட theme skills** — Block Themes, Classic Themes, Kadence Blocks, மற்றும் Kadence Theme ஆகியவற்றுக்கான சிறப்பு வழிகாட்டுதல்.
- **மேம்படுத்தப்பட்ட WooCommerce integration** — சிறந்த நம்பகத்தன்மை மற்றும் இணக்கத்தன்மைக்காக native WooCommerce APIs-ஐப் பயன்படுத்த மறுசீரமைக்கப்பட்டுள்ளது.
- **தானியங்கி வழங்குநர் கண்டறிதல் (Automatic provider detection)** — plugins மற்றும் themes செயல்படுத்தப்படும்போதோ அல்லது செயலிழக்கப்படும்போதோ கிடைக்கும்வற்றை தானாகவே கண்டறிந்து புதுப்பிக்கிறது.
- **Custom logo generation** — தானியங்கி namespace-safe sanitisation உடன் தனிப்பயன் SVG லோகோக்களை உருவாக்கவும் உட்பொதிக்கவும்.
- **Palette contrast validation** — உங்கள் theme-ல் பயன்படுத்துவதற்கு முன், WCAG அணுகல் இணக்கத்திற்காக வண்ண ஜோடிகளைச் சரிபார்க்கிறது.
- **Photo-enhanced discovery** — Theme Builder discovery interview-ல் புகைப்படங்களைப் பதிவேற்றலாம், இது வடிவமைப்பு திசையை வடிவமைக்க உதவும்.
- **Hospitality menus** — உணவகங்கள், கஃபேக்கள் மற்றும் விருந்தோம்பல் வணிகங்களுக்கான கட்டமைக்கப்பட்ட உணவு மற்றும் பான மெனுப் பக்கங்களை உருவாக்கலாம்.
- **Desktop மற்றும் mobile previews** — வடிவமைப்பு திசையைத் தேர்ந்தெடுக்கும்போது, desktop மற்றும் mobile சாதனங்களில் உங்கள் வடிவமைப்பைப் பார்க்க முடியும்.
- **Tier 1 site scraping** — sd-ai-agent/site-scrape என்பது இப்போது போட்டியாளர் மற்றும் உத்வேக வலைத்தளங்களை பகுப்பாய்வு செய்ய இயல்பாகவே கிடைக்கிறது.

## தேவைகள் (Requirements)

- WordPress 6.0 அல்லது அதற்கு மேல்
- PHP 8.1 அல்லது அதற்கு மேல்
- WordPress Multisite (network-activated) அல்லது single-site installation
- ஒரு செயலில் உள்ள AI provider connector plugin (Anthropic, OpenAI, அல்லது இணக்கமானது)
- Gratis AI Agent v1.9.0 அல்லது அதற்கு மேல் (Superdav AI Agent, Gratis AI Agent-ஐ விரிவாக்குகிறது)

## நிறுவுதல் (Installation)

1. plugin கோப்புகளை `/wp-content/plugins/superdav-ai-agent` என்ற இடத்தில் பதிவேற்றவும்.
2. WordPress-ல் உள்ள **Plugins** திரையின் மூலம் plugin-ஐ Activate செய்யவும்.
3. Gratis AI Agent-ம் Activate செய்யப்பட்டிருப்பதை உறுதி செய்யவும்.
4. **Gratis AI Agent → Settings** என்பதற்குச் சென்று உங்கள் இணைய தேடல் வழங்குநரை உள்ளமைக்கவும் (விரும்பினால்).

## விரைவான தொடக்கம் (Quick Start)

Activate செய்தவுடன், Superdav AI Agent தானாகவே உங்கள் Gratis AI Agent நிறுவலை theme-aware skills மற்றும் கூடுதல் திறன்களுடன் மேம்படுத்துகிறது. WordPress admin sidebar-ல் உள்ள **Gratis AI Agent** பேனலைத் திறந்து புதிய அம்சங்களைப் பயன்படுத்தத் தொடங்கவும்.

**உதாரண வழிமுறைகள்:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## Theme-Aware Skills

Superdav AI Agent தானாகவே உங்கள் செயலில் உள்ள theme மற்றும் நிறுவப்பட்ட plugins-ஐ கண்டறிந்து, பொருத்தமான theme-aware skill-ஐ உதவியாளரின் சூழலில் ஏற்றுகிறது. கிடைக்கும் திறன்கள் பின்வருமாறு:

| Skill | Applies To |
|---|---|
| **Block Themes** | WordPress block-based (Full Site Editing) architecture பயன்படுத்தும் Themes |
| **Classic Themes** | PHP templates மற்றும் classic editor பயன்படுத்தும் பாரம்பரிய WordPress Themes |
| **Kadence Blocks** | Kadence Blocks plugin செயலில் உள்ள Sites |
| **Kadence Theme** | Kadence theme செயலில் உள்ள Sites |

ஒவ்வொரு skill பற்றிய விரிவான வழிகாட்டுதலுக்கு [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills)-ஐப் பார்க்கவும்.

## Internet Search Providers

Superdav AI Agent v1.10.0 ஆனது இணைய தேடல் வினவல்களுக்கு Brave Search-க்கு மாற்றாக **Tavily** என்பதைச் சேர்க்கிறது. இரண்டு வழங்குநர்களும் நிகழ்நேர தேடல் முடிவுகளை வழங்குகின்றன, மேலும் இவை **Gratis AI Agent → Settings → Advanced** இல் உள்ளமைக்கப்படலாம்.

| Provider | Features | Setup |
|---|---|---|
| **Brave Search** | வேகமான, தனியுரிமை-கவனம் கொண்ட தேடல் முடிவுகள் | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | செழுமையான தேடல் முடிவுகள், மேம்பட்ட ஆராய்ச்சி திறன்கள் | [Tavily API](https://tavily.com/) |

உள்ளமைவு வழிமுறைகளுக்கு [Internet Search](../../user-guide/configuration/internet-search)-ஐப் பார்க்கவும்.

## Site Builder Contact Forms

Site Builder agent இப்போது chat interface-ல் இருந்து நேரடியாக contact form-களை உருவாக்க முடியும். Site Builder-க்கு ஒரு contact form உருவாக்கச் சொன்னால் போதும், அது தானாகவே உங்கள் நிறுவப்பட்ட form plugin (Contact Form 7, WPForms, Fluent Forms, அல்லது Gravity Forms) ஐக் கண்டறிந்து பயன்படுத்தத் தயாரான form-ஐ உருவாக்கும்.

விரிவான வழிகாட்டுதலுக்கு [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms)-ஐப் பார்க்கவும்.

## Theme Builder Features

Superdav AI Agent v1.16.0-ல் Theme Builder ஆனது சக்திவாய்ந்த புதிய திறன்களை உள்ளடக்கியுள்ளது:

### Photo Upload உடன் Discovery Interview

Theme Builder discovery interview-ல் இப்போது ஒரு photo upload படி சேர்க்கப்பட்டுள்ளது, இதில் நீங்கள் காட்சி உத்வேகத்தை வழங்க முடியும். உங்கள் பிராண்ட், போட்டியாளர் தளங்கள் அல்லது வடிவமைப்பு உத்வேகத்தின் புகைப்படங்களைப் பதிவேற்றவும், இது Theme Builder-க்கு உங்கள் வடிவமைப்பு திசையைப் புரிந்துகொள்ள உதவும்.

விரிவான வழிகாட்டுதலுக்கு [Discovery Interview](./theme-builder/discovery-interview.md)-ஐப் பார்க்கவும்.

### Desktop மற்றும் Mobile Previews உடன் Design Direction

Theme Builder உங்கள் முழு theme-ஐ உருவாக்குவதற்கு முன், desktop மற்றும் mobile சாதனங்களில் உங்கள் வடிவமைப்பைப் பார்க்கவும். பல responsive breakpoints-ல் நிகழ்நேர முன்னோட்டங்களைப் பார்க்கவும்.

விரிவான வழிகாட்டுதலுக்கு [Design Direction](./theme-builder/design-direction.md)-ஐப் பார்க்கவும்.

### Hospitality Menus

Theme Builder இப்போது உணவகங்கள், கஃபேக்கள், பார்கள் மற்றும் பிற விருந்தோம்பல் வணிகங்களுக்கான கட்டமைக்கப்பட்ட உணவு மற்றும் பான மெனுப் பக்கங்களை உருவாக்க முடியும். மெனுக்கள் விலை நிர்ணயம், விளக்கங்கள், உணவுமுறைத் தகவல் மற்றும் ஒவ்வாமை எச்சரிக்கைகள் ஆகியவற்றை உள்ளடக்கியது.

விரிவான வழிகாட்டுதலுக்கு [Hospitality Menus](./theme-builder/hospitality-menus.md)-ஐப் பார்க்கவும்.

### புதிய திறன்கள் (New Abilities)

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — தானியங்கி sanitisation உடன் தனிப்பயன் SVG லோகோக்களை உருவாக்கவும்
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — WCAG இணக்கத்திற்காக வண்ண ஜோடிகளைச் சரிபார்க்கவும்
- **[Create Menu](./abilities/create-menu.md)** — தனித்துவமான வழிசெலுத்தல் லேபிள்களுடன் வழிசெலுத்தல் மெனுக்களை உருவாக்கவும்

கிடைக்கக்கூடிய கருவிகள் மற்றும் Tier 1 site-scrape கருவி பற்றிய தகவல்களுக்கு [Tools and Configuration](./theme-builder/tools.md)-ஐப் பார்க்கவும்.

## உள்ளமைவு (Configuration)

அனைத்து அமைப்புகளும் WordPress admin-ல் **Gratis AI Agent → Settings** என்ற பிரிவில் உள்ளன. Superdav AI Agent பின்வரும் உள்ளமைவு விருப்பங்களைச் சேர்க்கிறது:

| Setting | Description |
|---|---|
| **Internet Search Provider** | Brave Search மற்றும் Tavily இடையே தேர்வு செய்யவும் (விரும்பினால்) |
| **Tavily API Key** | Tavily இணைய தேடலுக்கான API key (Tavily தேர்ந்தெடுக்கப்பட்டால்) |

## v1.16.0-ல் மேம்பாடுகள் (Improvements in v1.16.0)

- **AI Client cache** — இப்போது transients மூலம் ஆதரிக்கப்படுகிறது, இது நீண்ட நேரம் இயங்கும் agent பணிகளில் தரவு இழப்பைத் தடுக்கிறது.

- **Plugin row action links** — தெளிவுக்காகச் சரிசெய்யப்பட்டு பெயர் மாற்றப்பட்டுள்ளது.

- வகை மற்றும் tag பெயர்களைச் slugs ஆகச் சரியாகத் தீர்க்காத `list-posts` திறனைச் சரிசெய்தல்.
- முந்தைய refactor-க்குப் பிறகு WP-CLI கட்டளைகள் namespace aliases-ஐ இழப்பதைச் சரிசெய்தல்.
- automation tables இன்னும் உருவாக்கப்படாத தளங்களில் event automation gracefully கையாளப்படுவதைச் சரிசெய்தல்.
- system instruction builder-ல் சரியான namespace prefix-ஐப் பயன்படுத்தும் `memory-save` திறனைச் சரிசெய்தல்.
- scalar tool முடிவுகள் AI-க்குத் திரும்ப அனுப்பப்படுவதற்கு முன் சரியாகப் பொதி செய்யப்படுவதைச் சரிசெய்தல்.
- பழைய பதிப்புகளிலிருந்து மேம்படுத்தும்போது usage statistics சரியாக legacy ability key format-ஐக் கையாள்வதைச் சரிசெய்தல்.
