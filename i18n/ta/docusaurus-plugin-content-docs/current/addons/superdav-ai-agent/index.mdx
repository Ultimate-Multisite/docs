# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent என்பது மேம்பட்ட WordPress AI உதவியாளர். இது Gratis AI Agent-ஐ, தீம்-அறியும் திறன்கள், மேம்பட்ட இணையத் தேடல் திறன்கள், தள உருவாக்கி அம்சங்கள், மற்றும் விருப்பமான தானியக்க ஒருங்கிணைப்புகள் மூலம் விரிவாக்குகிறது. பதிப்பு 1.18.0 Google Calendar வாசிப்பு கருவிகள், தொடர்பு மற்றும் பங்கேற்பாளர் பொருத்துதல், மனித ஒப்புதல் கதவுகள், நினைவூட்டல் நகல் நீக்கம், TextBee SMS அறிவிப்புகள், நிர்வகிக்கப்படும் Superdav சேவை வழங்கல், மற்றும் நம்பகமான டெவலப்பர் பணிமுறைகளுக்காக தனியாக விநியோகிக்கப்படும் Advanced துணை தொகுப்பைச் சேர்க்கிறது.

## முக்கிய அம்சங்கள் {#key-features}

- **தீம்-அறியும் திறன்கள்** — உங்கள் செயலில் உள்ள WordPress தீம் மற்றும் நிறுவப்பட்ட செருகுநிரல்களின் அடிப்படையில் வழிகாட்டலும் திறன்களும் தானாக மாற்றப்படுகின்றன
- **பல இணையத் தேடல் வழங்குநர்கள்** — செழுமையான தேடல் முடிவுகள் மற்றும் மேம்பட்ட ஆய்வு திறன்களுக்காக Brave Search மற்றும் Tavily இடையே தேர்வு செய்யுங்கள்
- **தள உருவாக்கி தொடர்பு படிவங்கள்** — நீங்கள் நிறுவிய படிவச் செருகுநிரலைப் பயன்படுத்தி, அரட்டை இடைமுகத்திலிருந்து நேரடியாக முழுமையாகச் செயல்படும் தொடர்பு படிவங்களை உருவாக்குங்கள்
- **மேம்பட்ட தள உருவாக்கி** — சிறந்த செருகுநிரல் கண்டறிதல் மற்றும் பிழை மீட்புடன் மேம்படுத்தப்பட்ட பலகட்ட தள உருவாக்கத் திட்டங்கள்
- **உள்ளமைக்கப்பட்ட தீம் திறன்கள்** — Block Themes, Classic Themes, Kadence Blocks, மற்றும் Kadence Theme க்கான சிறப்பு வழிகாட்டல்
- **மேம்பட்ட WooCommerce ஒருங்கிணைப்பு** — அதிக நம்பகத்தன்மை மற்றும் இணக்கத்திற்காக சொந்த WooCommerce API-களைப் பயன்படுத்துமாறு மறுவடிவமைக்கப்பட்டது
- **தானியங்கி வழங்குநர் கண்டறிதல்** — செருகுநிரல்கள் மற்றும் தீம்கள் செயல்படுத்தப்படும்போது அல்லது செயலிழக்கப்படும்போது கிடைக்கும் செருகுநிரல்கள் மற்றும் தீம்களை தானாக கண்டறிந்து புதுப்பிக்கும்
- **தனிப்பயன் லோகோ உருவாக்கம்** — தானியங்கி namespace-பாதுகாப்பான சுத்திகரிப்புடன் தனிப்பயன் SVG லோகோக்களை உருவாக்கி உட்பொதிக்கவும்
- **வண்ணத்தட்டு மாறுபாடு சரிபார்ப்பு** — உங்கள் தீமில் பயன்படுத்துவதற்கு முன், நிற ஜோடிகள் WCAG அணுகல்திறன் இணக்கத்துக்கு பொருந்துகிறதா எனச் சரிபார்க்கவும்
- **புகைப்படம் மேம்படுத்திய கண்டறிதல்** — வடிவமைப்பு திசையைத் தீர்மானிக்க Theme Builder கண்டறிதல் நேர்காணலின்போது புகைப்படங்களைப் பதிவேற்றுங்கள்
- **விருந்தோம்பல் மெனுக்கள்** — உணவகங்கள், கஃபேக்கள், மற்றும் விருந்தோம்பல் வணிகங்களுக்காக கட்டமைக்கப்பட்ட உணவு மற்றும் பான மெனு பக்கங்களை உருவாக்குங்கள்
- **டெஸ்க்டாப் மற்றும் மொபைல் முன்னோட்டங்கள்** — வடிவமைப்பு-திசைத் தேர்வின்போது உங்கள் வடிவமைப்பை டெஸ்க்டாப் மற்றும் மொபைல் சாதனங்களில் முன்னோட்டமிடுங்கள்
- **அடுக்கு 1 தளச் சுரண்டல்** — போட்டியாளர் மற்றும் ஊக்கமளிக்கும் இணையதளங்களை பகுப்பாய்வு செய்ய sd-ai-agent/site-scrape இப்போது இயல்பாகக் கிடைக்கிறது
- **Google Calendar வாசிப்பு கருவிகள்** — வரவிருக்கும் அட்டவணை சூழலை தானியக்கங்கள் புரிந்துகொள்ள, அமைக்கப்பட்ட காலண்டர்கள் மற்றும் நிகழ்வுகளை வாசிக்கவும்
- **தொடர்பு மற்றும் பங்கேற்பாளர் பொருத்துதல்** — பின்தொடர் பணியை உருவாக்குவதற்கு முன், காலண்டர் பங்கேற்பாளர்களை அறியப்பட்ட தளப் பயனர்கள் மற்றும் தொடர்புகளுடன் பொருத்துங்கள்
- **மனித ஒப்புதல் கதவுகள்** — செய்திகள் அனுப்புதல், பணியை மீண்டும் தொடங்குதல், அல்லது மாற்றங்கள் செய்வதற்கு முன் மதிப்பாய்வுக்காக உணர்வுபூர்வமான தானியக்கங்களை இடைநிறுத்துங்கள்
- **நினைவூட்டல் பதிவுகள் மற்றும் நகல் நீக்கம்** — ஏற்கனவே அனுப்பப்பட்ட நினைவூட்டல்களைப் பதிவுசெய்து, மீண்டும் இயங்கும் தானியக்கங்கள் அதே பெறுநருக்கு இருமுறை அறிவிக்காமல் இருக்கச் செய்யுங்கள்
- **TextBee SMS அறிவிப்புகள்** — SMS வழங்குநர் இயக்கப்பட்டிருக்கும் போது, அமைக்கப்பட்ட உரைச் செய்தி அறிவிப்புகளை TextBee மூலம் அனுப்புங்கள்
- **நிர்வகிக்கப்படும் Superdav சேவை வழங்கல்** — கையேடு endpoint அமைப்பிற்கு பதிலாக, தானியங்கி அமைப்புடன் ஆதரிக்கப்படும் தளங்களை ஹோஸ்ட் செய்யப்பட்ட Superdav endpoint-களுடன் இணைக்கவும்
- **Advanced துணை தொகுப்பு** — கோப்பு முறைமை, தரவுத்தளம், WP-CLI, REST dispatcher, செருகுநிரல் உருவாக்கி, git snapshot, பயனர்-மேலாண்மை, மற்றும் benchmark பணிமுறைகளுக்காக தனியாக விநியோகிக்கப்படும் நம்பகமான-டெவலப்பர் கருவிகள்

## தேவைகள் {#requirements}

- WordPress 6.0 அல்லது அதற்கு மேல்
- PHP 8.1 அல்லது அதற்கு மேல்
- WordPress Multisite (நெட்வொர்க்-செயல்படுத்தப்பட்டது) அல்லது ஒற்றை-தள நிறுவல்
- செயலில் உள்ள AI வழங்குநர் இணைப்புச் செருகுநிரல் (Anthropic, OpenAI, அல்லது இணக்கமானது)
- Gratis AI Agent v1.9.0 அல்லது அதற்கு மேல் (Superdav AI Agent, Gratis AI Agent-ஐ விரிவாக்குகிறது)

## நிறுவல் {#installation}

1. செருகுநிரல் கோப்புகளை `/wp-content/plugins/superdav-ai-agent` இற்கு பதிவேற்றுங்கள்
2. WordPress இல் **செருகுநிரல்கள்** திரை மூலம் செருகுநிரலைச் செயல்படுத்துங்கள்
3. Gratis AI Agent-யும் செயல்படுத்தப்பட்டிருக்கிறதா என உறுதிசெய்யுங்கள்
4. **Gratis AI Agent → அமைப்புகள்** என்பதிற்குச் சென்று, தேவைக்கேற்ப உங்கள் இணையத் தேடல் வழங்குநர், நிர்வகிக்கப்படும் Superdav சேவை இணைப்பு, Google Calendar, அல்லது TextBee சான்றுகளை அமைக்கவும்

WordPress.org தொகுப்பு மைய Superdav AI Agent அம்சங்களை கொண்டுள்ளது. நம்பகமான டெவலப்பர் கருவிகள் வெளிப்படையாக நிறுவப்பட்டு அங்கீகரிக்கப்படும் சூழல்களுக்காக Advanced துணை தொகுப்பு தனியாக உருவாக்கப்பட்டு விநியோகிக்கப்படுகிறது.

## விரைவு தொடக்கம் {#quick-start}

செயல்படுத்தப்பட்டவுடன், Superdav AI Agent உங்கள் Gratis AI Agent நிறுவலை தீம்-அறியும் திறன்கள் மற்றும் கூடுதல் திறன்களுடன் தானாக மேம்படுத்துகிறது. WordPress நிர்வாக பக்கப்பட்டியில் உள்ள **Gratis AI Agent** பலகையைத் திறந்து புதிய அம்சங்களைப் பயன்படுத்தத் தொடங்குங்கள்.

**எடுத்துக்காட்டு வழிமுறைகள்:**

```
தொடர்பு பக்கத்தில் ஒரு தொடர்பு படிவத்தைச் சேர்க்கவும்.
```

```
சமீபத்திய WordPress பாதுகாப்பு சிறந்த நடைமுறைகளுக்காக இணையத்தில் தேடுங்கள்.
```

```
செயலில் உள்ள தீமின் block patterns-ஐப் பயன்படுத்தி ஒரு hero பிரிவை உருவாக்குங்கள்.
```

```
நாளைய அமைக்கப்பட்ட Google Calendar நிகழ்வுகளை மதிப்பாய்வு செய்து, பங்கேற்பாளர்களுக்கான நினைவூட்டல்களை வரைவு செய்யுங்கள்.
```

```
நான் செய்தியை ஒப்புதல் அளித்த பிறகு SMS நினைவூட்டலை அனுப்புங்கள்.
```

## தீம்-அறியும் திறன்கள் {#theme-aware-skills}

Superdav AI Agent உங்கள் செயலில் உள்ள தீம் மற்றும் நிறுவப்பட்ட செருகுநிரல்களை தானாக கண்டறிந்து, பொருத்தமான தீம்-அறியும் திறனை உதவியாளரின் சூழலுக்கு ஏற்றுகிறது. கிடைக்கும் திறன்கள்:

| திறன் | பொருந்துவது |
|---|---|
| **Block Themes** | WordPress block-அடிப்படையிலான (Full Site Editing) கட்டமைப்பைப் பயன்படுத்தும் தீம்கள் |
| **Classic Themes** | PHP வார்ப்புருக்கள் மற்றும் classic editor-ஐப் பயன்படுத்தும் பாரம்பரிய WordPress தீம்கள் |
| **Kadence Blocks** | Kadence Blocks செருகுநிரல் செயலில் உள்ள தளங்கள் |
| **Kadence Theme** | Kadence theme செயலில் உள்ள தீமாக இருக்கும் தளங்கள் |

ஒவ்வொரு திறனுக்கும் விரிவான வழிகாட்டலுக்கு [தீம்-அறியும் திறன்கள்](../../user-guide/configuration/theme-aware-skills) என்பதைப் பார்க்கவும்.

## இணையத் தேடல் வழங்குநர்கள் {#internet-search-providers}

Superdav AI Agent v1.10.0 இணையத் தேடல் வினவல்களுக்கு Brave Search-க்கு மாற்றாக **Tavily**-ஐச் சேர்க்கிறது. இரு வழங்குநர்களும் நேரடி தேடல் முடிவுகளை வழங்குகின்றனர், மேலும் அவற்றை **Gratis AI Agent → அமைப்புகள் → மேம்பட்டது** இல் அமைக்கலாம்.

| வழங்குநர் | அம்சங்கள் | அமைப்பு |
|---|---|---|
| **Brave Search** | வேகமான, தனியுரிமையை மையப்படுத்திய தேடல் முடிவுகள் | [Brave Search API](https://brave.com/search/api/) |

## காலண்டர், ஒப்புதல், மற்றும் அறிவிப்பு பணிப்பாய்வுகள் {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0, உள்ளடக்கம் மற்றும் தள உருவாக்கப் பணிகளைத் தாண்டி தானியக்க பணிப்பாய்வுகளை விரிவாக்குகிறது:

- **Google Calendar கருவிகள்** காலண்டர் சான்றுகள் கிடைக்கும்போது, அமைக்கப்பட்ட காலண்டர்கள் மற்றும் நிகழ்வு விவரங்களை agent படிக்க அனுமதிக்கின்றன.
- **தொடர்பு மற்றும் பங்கேற்பாளர் உதவிகள்** நினைவூட்டல்கள் அல்லது தொடர்ச்சிப் பணிகளைத் தயாரிப்பதற்கு முன், நிகழ்வு பங்கேற்பாளர்களை WordPress பயனர்கள் அல்லது அறியப்பட்ட தொடர்புகளுடன் பொருத்துகின்றன.
- **மனித ஒப்புதல் வாயில்கள்** அங்கீகரிக்கப்பட்ட பயனர் முன்மொழியப்பட்ட செயலை மதிப்பாய்வு செய்யும் வரை ஒரு தானியக்கத்தை இடைநிறுத்த அனுமதிக்கின்றன.
- **நினைவூட்டல் பதிவுகள்** ஒரு தானியக்கம் மீண்டும் முயற்சிக்கப்படும்போது அல்லது மீண்டும் மீண்டும் திட்டமிடப்படும்போது, நகல் காலண்டர் அல்லது அறிவிப்பு நினைவூட்டல்களைத் தடுக்கின்றன.
- **TextBee SMS ஆதரவு** TextBee வழங்குநர் அமைக்கப்பட்டிருக்கும்போது உரைச் செய்தி அறிவிப்புகளை இயலுமைப்படுத்துகிறது.

- **WooCommerce ஒருங்கிணைப்பு மறுசீரமைக்கப்பட்டது** — மேம்பட்ட நம்பகத்தன்மை மற்றும் இணக்கத்திற்காக இப்போது சொந்த WooCommerce API-களைப் பயன்படுத்துகிறது
- **வழங்குநர் பட்டியல் தானியங்கு புதுப்பிப்பு** — செருகுநிரல்கள் செயல்படுத்தப்படும்போதோ செயலிழக்கச் செய்யப்படும்போதோ தானாகக் கண்டறிந்து, கிடைக்கும் வழங்குநர் பட்டியலைப் புதுப்பிக்கிறது
- **மேம்படுத்தப்பட்ட தள உருவாக்கி** — சிறந்த பிழை மீட்புடன் பல-கட்ட தள உருவாக்கத் திட்டங்கள் மேம்படுத்தப்பட்டன
- **கருப்பொருள்-அறிந்த திறன்கள்** — உங்கள் செயலில் உள்ள கருப்பொருளுக்கு தானாக ஏற்றுக்கொள்ளும் நான்கு புதிய உட்பொதிந்த திறன்கள்
