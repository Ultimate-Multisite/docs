# இணக்கமான எண்ட்பாயிண்டுகளுக்கான Ultimate AI Connector மாற்றங்கள் பட்டியல் {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## பதிப்பு 2.0.0 — வெளியிடப்பட்டது: 2026-04-24 {#version-200--released-on-2026-04-24}

* **புதியது:** பல வழங்குநர் ஆதரவு (Multi-provider support) — பல AI எண்ட்பாயிண்டுகளை கட்டமைக்கலாம் மற்றும் வழங்குநர்கள் முழுவதும் தானியங்கி மாற்று வழி (automatic fallback) மூலம் கோரிக்கைகளை வழிநடத்தலாம்.
* **சரிசெய்தல்:** பல வழங்குநர்களுக்கான SDK ஒருங்கிணைப்பு: சரியான வழங்குநர் IDகள், பதிவு URLகள் மற்றும் ஒவ்வொரு வழங்குநருக்கும் HTTP ஃபில்டர் எல்லை (scope) ஆகியவற்றுடன் சரிசெய்யப்பட்டது.
* **சரிசெய்தல்:** புதிய வழங்குநர் கார்டு (provider card) இப்போது சேர்க்கும்போது தானாகவே விரிவடையும்; plugin புதுப்பித்தலின்போது ஸ்கிரிப்ட் கேச் உடைந்தல் (script cache busting) சேர்க்கப்பட்டது.
* **சரிசெய்தல்:** eval() க்கான டைனமிக் வழங்குநர் வகுப்பு பெயரிடக்கம் (Dynamic provider class namespace) மற்றும் புதிதாக சேர்க்கப்பட்ட வழங்குநர்களுக்கான தானியங்கி விரிவாக்க நடத்தை (auto-expand behaviour) சரிசெய்யப்பட்டது.
* **சரிசெய்தல்:** WordPress 6.9+ இணக்கத்தன்மைக்காக நிலையான Card/CardBody/CardHeader/CardDivider கூறுகள் பயன்படுத்தப்பட்டன (இனி சோதனை நிலையில் இல்லை).
* **சரிசெய்தல்:** வழங்குநரை இழுத்து மீண்டும் வரிசைப்படுத்த (drag-to-reorder) பயன்படுத்தப்பட்ட வரையறுக்கப்படாத DragHandle, யூனிகோட் பிடிமான ஐகானால் மாற்றப்பட்டது.
* **சரிசெய்தல்:** ஒவ்வொரு பக்கத்தின் ஏற்றத்தின்போதும் இயங்கிக் கொண்டிருந்த தடுப்பு HTTP கோரிக்கை (blocking HTTP request) நீக்கப்பட்டது.
* **மேம்படுத்தப்பட்டது:** GitHub Actions பணிப்பாய்வுகள் (workflows) Node.js 24 ஆக மேம்படுத்தப்பட்டன.
