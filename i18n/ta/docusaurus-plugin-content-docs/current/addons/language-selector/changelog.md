# மொழித் தேர்வி மாற்றங்கள் பட்டியல் {#language-selector-changelog}

Version: 1.1.3 - வெளியிடப்பட்டது 2026-05-05
* சரிசெய்தல்: உள்ளீட்டு மாறிகள் இப்போது சுத்திகரிக்கப்படுகின்றன, மேலும் மூன்றாம் தரப்பு 플러ഗിன்களுடன் முரண்பாடுகளைத் தடுக்க தனிப்பயன் ஹூக் பெயர்களுக்கும் முன்னொட்டு சேர்க்கப்பட்டுள்ளது.

Version: 1.1.2 - வெளியிடப்பட்டது 2026-01-01
* மேம்படுத்தப்பட்டது: இணக்கத்தன்மை மேம்படுத்தல்கள் மற்றும் குறியீடு தர மேம்பாடுகள் சேர்க்கப்பட்டுள்ளன.

Version: 1.1.0 - வெளியிடப்பட்டது 2025-09-28
