# Action: wu_page_enqueue_scripts

தேவைக்கேற்ப ஸ்கிரிப்ட்கள் மற்றும் ஸ்டைல்ஸ்களைச் சேர்க்க (enqueue) ஒரு இடத்தை உருவாக்க, இங்கே ஒரு தனிப்பயன் ஆக்‌ஷனை (custom action) சேர்க்கிறோம். மேலும், அவை சரியான பக்கங்களில் மட்டுமே ஏற்றப்படுவதை (load) இது உறுதி செய்கிறது.

## அளவுருக்கள் (Parameters)

| Name | Type | Description |
|------|------|-------------|
| $ | `\WU_Page` | தற்போதைய நிர்வாகப் பக்கம் (The current admin page) |

### எப்போது இருந்து (Since)

- 1.3.0
### ஆதாரம் (Source)

[`inc/class-wu-admin-pages.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-admin-page-creator/blob/main/inc/class-wu-admin-pages.php#L1123) இல் உள்ள 1123 வரியில் வரையறுக்கப்பட்டுள்ளது.
