# Filter: wu_apc_get_meta_content_default_content

டெவலப்பர்கள் இயல்புநிலை மதிப்புகளை (default values) வடிகட்ட (filter) இது அனுமதிக்கிறது.

## அளவுருக்கள் (Parameters)

| Name | Type | Description |
|------|------|-------------|
| $default_value | `string` | இயல்புநிலை மதிப்பு (The default value). |
| $meta_type | `string` | பெறப்பட வேண்டிய மெட்டா டேட்டாவின் (meta data) வகை. |
| $meta_key | `string` | மெட்டா ஃபீல்டின் (meta field) பெயர். |

### எப்போது இருந்து (Since)

- 1.4.0
### மூலம் (Source)

[inc/wu-apc-functions.php](https://github.com/Ultimate-Multisite/ultimate-multisite-admin-page-creator/blob/main/inc/wu-apc-functions.php#L68) என்ற இடத்தில், 68-வது வரியில் வரையறுக்கப்பட்டுள்ளது.
