# Admin Page Creator

## Content {#content}
Admin page creator மூலம், WordPress admin dashboard-க்குள் புதிய admin பக்கங்களையும் (admin pages) அதன் துணைப் பக்கங்களையும் (sub-pages) உங்களுக்கு எளிதாக சேர்க்க முடியும். புதிய பயனர்களுக்கான வரவேற்புப் பக்கமாகவோ (welcome page) அல்லது பயிற்சிப் பக்கமாகவோ (tutorial page) இதை நீங்கள் பயன்படுத்தலாம். Beaver Builder, Elementor, Brizy, மற்றும் Oxygen போன்ற page builder-களைப் பயன்படுத்தி எந்த சிரமமும் இல்லாமல் பக்கங்களை வடிவமைக்கலாம்.

இது தனிப்பயன் HTML, CSS மற்றும் JavaScript ஆதரவை வழங்குகிறது, அத்துடன் plan மற்றும் user role-ஐ அடிப்படையாகக் கொண்ட அணுகல் கட்டுப்பாட்டையும் (access control) வழங்குகிறது!

## Features {#features}
* Custom HTML மற்றும் CSS ஆதரவு
* JavaScript ஆதரவு
* PHP Content Mode
* User role-ஐ அடிப்படையாகக் கொண்ட அனுமதி (Permission)
* Page builders-க்கு ஆதரவு

## Documentation {#documentation}
[Creating Custom Admin Pages and Sub-pages using Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Installation {#installation}

1. **ultimate-multisite-admin-page-creator** என்பதை **/wp-content/plugins/** கோப்பகத்தில் (directory) பதிவேற்றவும்
2. உங்கள் WordPress Network Admin-ல் உள்ள **Plugins** மெனு மூலம் பிளக்இன்-ஐ Activate செய்யவும்

## Usage Guide {#usage-guide}

WordPress ஆனது, தளங்களின் (sites) முன் பக்கத்தில் (front end) உள்ளடக்கத்தைச் சேர்ப்பதை மிகவும் எளிதாக்குகிறது. ஆனால், admin பக்கத்தில் பக்கங்களையும் துணைப் பக்கங்களையும் உருவாக்குவதற்கு அடிப்படை கோடிங் திறன்கள் தேவை. Ultimate Multisite: Admin Page Creator ஆனது, front end-ல் உள்ள அதே பயன்பாட்டு எளிமையை உங்கள் network sites-ன் back end-க்கும் கொண்டு வர இலக்கு வைக்கிறது.

## Why use Admin Page Creator? {#why-use-admin-page-creator}

உங்களுடைய வாடிக்கையாளர்களின் admin panel-ல் பயிற்சிகளுடன் கூடிய தனிப்பயன் பக்கத்தை வழங்க நீங்கள் விரும்பலாம், அல்லது FAQ மற்றும் contact widget உடன் கூடிய தனிப்பயன் பக்கத்தை சேர்க்க நீங்கள் விரும்பலாம். Admin Page Creator மூலம் சாத்தியங்கள் முடிவே இல்லை, மேலும் இன்று நாம் கற்றுக்கொள்ளப் போகும் tutorial-ல் இதைத்தான் ஆராய்வோம்.

## Creating your first Custom Admin Page {#creating-your-first-custom-admin-page}

Ultimate Multisite: Admin Page Creator-ஐ நிறுவி Activate செய்த பிறகு, உங்கள் network admin மெனுவுக்குச் சென்று Admin Pages-க்குச் செல்லவும்.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Network Admin-ல் உள்ள Custom Admin Pages பட்டியல்_

நீங்கள் இதுவரை உருவாக்கிய அனைத்து admin பக்கங்களின் பட்டியலையும் இங்கே காண்பீர்கள் (இது உங்கள் முதல் வருகை என்பதால், பட்டியல் காலியாக இருக்கும்). நீங்கள் **Add new Admin Page** என்பதைக் கிளிக் செய்துவிட்டு, Admin Page editor-க்கு திருப்பி விடப்படுவீர்கள்.

## The elements of a Custom Admin Page {#the-elements-of-a-custom-admin-page}

Admin Page editor-ல், உங்கள் பக்கத்தை தனிப்பயனாக்க பலவிதமான விருப்பங்கள் உங்களுக்குக் கிடைக்கும்.

### Page Title {#page-title}

இது admin மெனு லேபிளாக இருக்காது; மாறாக, பக்கத்தின் மேலே காட்டப்படும் தலைப்பு இதுதான்.

### Content Type Selector {#content-type-selector}

Ultimate Multisite: Admin Page Creator ஆனது, பக்கத்தின் உள்ளடக்கத்தைப் பொறுத்தவரை உங்களுக்கு இரண்டு விருப்பங்களை வழங்குகிறது: நீங்கள் இயல்பான WordPress WYSIWYG editor-ஐப் பயன்படுத்தலாம் அல்லது HTML editor-க்கு மாறலாம்.

உங்களுக்கு HTML பற்றித் தெரியவில்லை என்றால், இயல்பான WordPress editor-ல் தொடங்குங்கள். நீங்கள் ஒரு சாதாரண Post அல்லது Page-ல் உள்ளடக்கத்தை edit செய்வது போலவே, பக்கத்தின் உள்ளடக்கத்தையும் edit செய்ய முடியும்.

## Menu Options {#menu-options}

பக்கவாட்டில், General Options meta-box-ல், நீங்கள் மெனு வகையை (menu type), லேபிளை (label), மற்றும் பலவற்றை உள்ளமை할 (configure) முடியும்.

### Menu Types {#menu-types}

உங்கள் புதிய தனிப்பயன் admin பக்கத்திற்கு இரண்டு வெவ்வேறு மெனு வகைகள் இருக்கலாம்: top-level menu மற்றும் submenu.

ஒரு top-level menu page, அதன் பெயரிலேயே குறிப்பிடுவது போல, நீங்கள் புதிதாக உருவாக்கிய பக்கத்தை இயல்பான WordPress பக்கங்களுடன் சேர்த்து admin மெனுவில் சேர்க்கும்.

நீங்கள் top-level விருப்பத்தைத் தேர்ந்தெடுத்தால், நீங்கள் மெனு வரிசையையும் (menu order) ஒரு மெனு ஐகானையும் (menu icon) தேர்ந்தெடுக்க முடியும்.

இந்த பக்கத்தை ஒரு sub-menu page ஆக மாற்ற விரும்பினால், இந்த பக்கத்துடன் இணைக்க ஒரு parent page-ஐ நீங்கள் தேர்ந்தெடுக்க முடியும். உங்கள் தனிப்பயன் பக்கத்தை இயல்பான WordPress பக்கங்கள் அல்லது உங்கள் top-level தனிப்பயன் admin பக்கங்களில் சேர்க்கலாம்.

_இந்த Custom Page-க்கான மெனு உருப்படியை முழுமையாக தனிப்பயனாக்கலாம்_

<!-- Screenshot unavailable: Admin Page Creator sub-menu parent page selection -->

_உங்கள் சொந்த top-level தனிப்பயன் பக்கங்கள் உட்பட, மற்ற மெனுப் பக்கங்களின் துணை மெனுவாகவும் இதைச் சேர்க்க முடியும்._

## Advanced Options {#advanced-options}

உங்கள் தனிப்பயன் admin பக்கங்களில் எல்லா வகையான உள்ளடக்கங்களையும் நீங்கள் சேர்க்க முடியும். இதில் CSS விதிகள் மற்றும் கோப்புகள், JavaScript, மற்றும் வெளிப்புற நூலகங்கள் (external libraries) ஆகியவை அடங்கும். இதை Advanced Options Tab-ஐப் பயன்படுத்தி நீங்கள் செய்யலாம்.

### Custom CSS {#custom-css}

CSS tab-ல், உங்களால் சொந்தமான தனிப்பயன் CSS கோடைச் சேர்க்க முடியும், அத்துடன் தொலைதூர தளங்களில் இருந்து CSS கோப்புகளை (CDNs, Google Fonts போன்றவை) இறக்குமதி (import) செய்ய முடியும்.

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_தனிப்பயன் CSS-உம் ஆதரிக்கப்படுகிறது. நீங்கள் வெளிப்புற ஸ்டைல் கோப்புகளையும் சேர்க்கலாம்!_

### Custom JavaScript {#custom-javascript}

JavaScript-க்கும் இதுவே பொருந்தும். உங்கள் தனிப்பயன் JavaScript கோடில் பயன்படுத்த வெளிப்புற நூலகங்களை நீங்கள் சேர்க்க முடியும்.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_உங்கள் சொந்த JavaScript கோடைச் சேர்க்கவும்!_

## Permissions {#permissions}

Advanced Options-ல், எந்த துணைத் தளங்கள் (sub-sites) இந்த பக்கத்தை அணுக முடியும் என்பதை நீங்கள் தேர்ந்தெடுக்க முடியும். உதாரணமாக, வெவ்வேறு plan நிலைகளுக்கு வெவ்வேறு பிரத்யேக உள்ளடக்கத்தை வழங்க நீங்கள் திட்டமிட்டால் இது பயனுள்ளதாக இருக்கும்.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_இந்த பக்கம் எங்கு காட்டப்பட வேண்டும் என்பதில் உங்களுக்கு முழு கட்டுப்பாடு உள்ளது_

**முக்கியமானது** : பக்கம் காட்டப்படுவதற்கு இரண்டு நிபந்தனைகளும் பூர்த்தி செய்யப்பட வேண்டும். அதாவது, பயனர் ஒரு குறிப்பிட்ட plan-ன் வாடிக்கையாளராக இருந்தாலும், தேர்ந்தெடுக்கப்பட்ட roll-களில் ஒன்று இல்லையென்றால் (அல்லது இதற்கு நேர்மாறாக), அந்தப் பக்கம் அவர்களின் panel-ல் தோன்றாது.

## Active {#active}

ஒரு தனிப்பயன் admin பக்கத்தை நீக்காமலோ அல்லது அனுமதிகளை குழப்பாமலோ நீங்கள் செயலிழக்கச் செய்ய விரும்பினால், Active விருப்பத்தை மட்டும் Uncheck செய்தால் போதும்.

ஒரு top-level பக்கத்தை Deactivate செய்வது, அந்த parent page- உடன் இணைக்கப்பட்ட அனைத்து sub-pages-களையும் தானாகவே Deactivate செய்யும்.
