# Admin Page Creator மாற்றங்கள் பட்டியல் {#admin-page-creator-changelog}

### Version 1.9.0 - Released on 2026-01-18 {#version-190---released-on-2026-01-18}

* சரி செய்யப்பட்டது: செயல்படுத்தும்போது ஏற்படும் கடுமையான பிழைகள்.
* மறுசீரமைக்கப்பட்டது: Content source classes-ஐ Singleton trait பயன்படுத்தும்படி, சரியான பெற்றோர் constructor துவக்கத்துடன்.
* சரி செய்யப்பட்டது: content type selector ரெண்டர் செய்ய முடியாத Vue.js template compilation பிழைகள்.
* சரி செய்யப்பட்டது: Vue data properties காணாமல் போயிருந்தன (template_id, external_link_url_embedable, external_link_url_checking).
* சரி செய்யப்பட்டது: page load ஆனபோது TinyMCE editor சரியாக துவக்கப்படவில்லை.
* சரி செய்யப்பட்டது: Divi support file Composer மூலம் autoload செய்யப்படவில்லை.
* சரி செய்யப்பட்டது: Divi support class-ல் கெட்டுப்போன PHP syntax.
* மேம்படுத்தப்பட்டது: JavaScript asset அமைப்பு மற்றும் minification.
* மேம்படுத்தப்பட்டது: சமீபத்திய WordPress மற்றும் page builder பதிப்புகளுடன் இணக்கத்தன்மை.
