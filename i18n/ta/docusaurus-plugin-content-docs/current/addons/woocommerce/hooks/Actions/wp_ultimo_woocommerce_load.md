# Action: wp_ultimo_woocommerce_load

அனைத்து கூடுதல் அம்சச் சார்புகளும் (add-on dependencies) ஏற்றப்பட்ட பிறகு இது தூண்டப்படுகிறது.

இது 플러ഗിൻ உருவாக்குநர்கள் புதிய செயல்பாடுகளைச் சேர்க்க அனுமதிக்கிறது.

### Since {#since}

- 2.0.0
### Source {#source}

[`inc/class-wp-ultimo-woocommerce.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/class-wp-ultimo-woocommerce.php#L90) என்ற கோப்பில் 90-வது வரியில் வரையறுக்கப்பட்டுள்ளது.
