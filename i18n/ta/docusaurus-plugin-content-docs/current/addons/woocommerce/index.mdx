# WooCommerce ஒருங்கிணைப்பு {#woocommerce-integration}

WooCommerce ஒருங்கிணைப்பு, எங்கள் Manual Gateway செயல்படும் முறைக்கு மிகவும் ஒத்த முறையில் செயல்படுகிறது. இது உங்கள் பயனர்களுக்கு அதிக நெகிழ்வுத்தன்மையை வழங்குகிறது; உங்கள் WooCommerce நிறுவலில் கிடைக்கும் எந்த கட்டணப் படிவத்தையும் அவர்கள் தேர்வு செய்ய முடியும். மேலும் இது எளிது: பயனரின் பில்லிங் காலம் முடிந்தவுடன், பயனரின் சந்தா நிறுத்திவைக்கப்படும். அதே நேரத்தில், நெட்வொர்க்கின் முதன்மை தளத்தின் WooCommerce நிறுவலில் ஒரு ஆர்டர் உருவாக்கப்படும். பின்னர் பயனர் தனது மின்னஞ்சலில் கட்டணம் செலுத்துவதற்கான இணைப்புடன் ஒரு விலைப்பட்டியலைப் பெறுவார். கட்டணம் பெறப்பட்டதும், சந்தா புதுப்பிக்கப்படும்.

## அம்சங்கள் {#features}

- Ultimate Multisite மற்றும் WooCommerce இடையிலான எளிய ஒருங்கிணைப்பு
- பல கட்டண முறைகள் கிடைக்கும்
- இணைக்கப்பட்ட subscriptions core-ஐப் பயன்படுத்தி தானியங்கும் மீளும் பில்லிங்; விருப்பமாக WooCommerce Subscriptions plugin மேம்பாடுகளுடன்
- Checkout மொத்தங்களில் Ultimate Multisite வரிகள் தனித்தனியான, வரி விதிக்கப்படாத WooCommerce கட்டண வரிகளாக சேர்க்கப்படும். இதனால் வாடிக்கையாளர்கள் கட்டணம் செலுத்துவதற்கு முன் வரித் தொகைகளை மதிப்பாய்வு செய்யலாம்

## ஆவணம் {#documentation}

**WooCommerce ஒருங்கிணைப்பை அமைத்தல்**

**Ultimate Multisite மூலம் உங்கள் கட்டண விருப்பங்களை விரிவாக்குதல்: WooCommerce ஒருங்கிணைப்பு**

## நிறுவல் {#installation}

1. 'ultimate-multisite-woocommerce' ஐ '/wp-content/plugins/' அடைவில் பதிவேற்றவும்
2. WordPress இல் 'Plugins' மெனுவின் மூலம் plugin-ஐ செயல்படுத்தவும்
3. plugin-ஐ அமைக்க படிப்படையான Wizard-ஐப் பின்பற்றவும்

## பயன்பாட்டு வழிகாட்டி {#usage-guide}

_**முக்கிய குறிப்பு: இந்தக் கட்டுரை Ultimate Multisite பதிப்பு 2.x-ஐக் குறிக்கிறது.**_

_**கவனம்:** Ultimate Multisite: WooCommerce Integration உங்கள் முதன்மை தளத்தில் குறைந்தபட்சம் WooCommerce செயல்படுத்தப்பட்டிருக்க வேண்டும்._

சில நாடுகளில் _Stripe_ மற்றும் _PayPal_ கிடைக்காததால், Ultimate Multisite பயனர்கள் எங்கள் plugin-ஐ பயனுள்ள முறையில் பயன்படுத்துவதில் வரம்புகள் அல்லது தடைகள் ஏற்படுகின்றன என்பதை நாங்கள் புரிந்துகொள்கிறோம். ஆகவே, _WooCommerce_ உடன் ஒருங்கிணைக்க ஒரு **add-on** உருவாக்கினோம்.

_WooCommerce_ மிகவும் பிரபலமான eCommerce plugin ஆகும். உலகம் முழுவதும் உள்ள உருவாக்குநர்கள், பல்வேறு கட்டண gateway-களை அதனுடன் ஒருங்கிணைக்க add-on-களை உருவாக்கினர். Ultimate Multisite பில்லிங் அமைப்பில் நீங்கள் பயன்படுத்தக்கூடிய கட்டண gateway-களை விரிவாக்க இதை நாங்கள் பயன்படுத்தினோம்.

## அதை அமைத்தல் {#setting-it-up}

முதலில், **உரிமம் பெற்ற பயனர்களுக்கு இலவசமான** WooCommerce Integration add-on-ஐ நீங்கள் நிறுவ வேண்டும். அதை நிறுவ, உங்கள் super admin Dashboard-இல் Ultimate Multisite-ஐ கிளிக் செய்து **Add-ons** என்பதைத் தேர்ந்தெடுக்கவும்.

![Add-ons பக்கம்](/img/admin/addons-page.png)

இங்கே அனைத்து Ultimate Multisite add-on-களையும் காணலாம். **Ultimate Multisite: WooCommerce Integration** add-on-ஐ கிளிக் செய்யவும்.

![WooCommerce Integration காட்டும் Add-ons பக்கம்](/img/admin/addons-page.png)

### Checkout வரி கட்டண வரிகள் {#checkout-tax-fee-lines}

Ultimate Multisite checkout-இல் வரிகள் இருந்தால், ஒவ்வொரு Ultimate Multisite வரித் தொகையையும் தனித்தனியான, வரி விதிக்கப்படாத கட்டண வரியாக WooCommerce checkout-இல் இந்த ஒருங்கிணைப்பு சேர்க்கும். இதனால் கட்டணம் செலுத்துவதற்கு முன் வரி மொத்தம் தெளிவாக இருக்கும்; அதே நேரத்தில் WooCommerce அந்த வரிக் கட்டணத்திற்கு மீண்டும் வரி விதிப்பது தவிர்க்கப்படும்.

## கட்டணத்தை நிர்வகிப்பது எப்படி {#how-to-manage-the-payment}

WooCommerce ஒருங்கிணைப்பு, எங்கள் **Manual Gateway** செயல்படும் முறைக்கு மிகவும் ஒத்த முறையில் செயல்படுகிறது. குறை என்னவென்றால், ஒவ்வொரு கட்டணமும் பில்லிங் காலத்தின் முடிவில் கைமுறையாக செலுத்தப்பட வேண்டும். ஓட்டம் இவ்வாறு செயல்படுகிறது:

  * பயனரின் பில்லிங் சுழற்சி முடிந்தவுடன், உறுப்பினர் நிலை **நிறுத்திவைக்கப்படும்**. அதே நேரத்தில், உங்கள் நெட்வொர்க்கின் முதன்மை தளத்தில் உள்ள _WooCommerce_ நிறுவலில் ஒரு ஆர்டர் உருவாக்கப்படும்.

  * பின்னர் பயனர்கள் தங்கள் மின்னஞ்சலில் கட்டணம் செலுத்துவதற்கான இணைப்புடன் ஒரு விலைப்பட்டியலைப் பெறுவார்கள் (அந்த இணைப்பு அவர்களின் subsite Dashboard-இல் உள்ள பில்லிங் வரலாற்றிலும் தோன்றும்). அந்த இணைப்பு பயனரை _**WooCommerce**_ **checkout** க்கு திருப்பிவிடும்; அங்கு பயனர்கள் தங்கள் கட்டணத்தை செயலாக்க **கிடைக்கும் _WooCommerce_ gateway-களில்** ஒன்றைத் தேர்வு செய்ய முடியும்.

  * கட்டணம் பெறப்பட்டதும், உறுப்பினர் நிலை **புதுப்பிக்கப்படும்** , கட்டணம் Ultimate Multisite-இல் பதிவு செய்யப்படும், மேலும் ஆர்டர் நிறைவு செய்யப்பட்டதாக அமைக்கப்படும்.

### சில முக்கிய குறிப்புகள்: {#some-important-notes}

  * Ultimate Multisite-இன் இயல்புநிலை செயல்பாடு இந்த ஒருங்கிணைப்பு வடிவத்துடனும் செயல்படும். உதாரணமாக, **Subscription Management** திரையிலிருந்து **பணத்தைத் திருப்பி வழங்குதல்** செய்யும்போது, தேர்ந்தெடுக்கப்பட்ட கட்டணப் படிவத்தைப் பயன்படுத்தி refund வழங்க WooCommerce உடன் தொடர்புகொள்ளும்.

  * Ultimate Multisite: WooCommerce Integration செயல்பட WP-Cron-ஐ சார்ந்துள்ளது. பெரும்பாலும் இது நன்றாக செயல்படும். ஆனால் தளத்தில் குறைந்த போக்குவரத்து இருந்தால், திட்டமிடப்பட்ட நிகழ்வுகளைச் சரிபார்க்க cron தூண்டப்படாது; நிகழ்வு தவறவிடப்படும். இதனால் செயல்படுத்தலில் தாமதங்கள் ஏற்படலாம்.

## WooCommerce Subscriptions Plugin {#woocommerce-subscriptions-plugin}

ஒவ்வொரு மாதமும் கைமுறையாக கட்டணத்தைத் தொடங்குவது இறுதிப் பயனர்களுக்கு சிறந்ததல்ல, ஏனெனில் subscriptions ஒவ்வொரு billing cycle-லும் தானாகவே மீண்டும் நடைபெறும் என்று எதிர்பார்க்கப்படுகிறது. WooCommerce Integration add-on இப்போது core subscriptions code-ஐ இணைத்துள்ளது, அதனால் recurring products எப்போதும் recurring billing-க்கு subscriptions-ஐ பயன்படுத்தும்.

இந்த integration மூலம் recurring products விற்க தனியான [WooCommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/)-ஐ வாங்கவோ நிறுவவோ நீங்கள் **தேவையில்லை**. அந்த plugin வாங்கப்பட்டு, நிறுவப்பட்டு, உங்கள் main site-இல் கிடைக்குமானால், Ultimate Multisite அதை பயன்படுத்தும், மேலும் WooCommerce கூடுதல் subscription மேலாண்மை அம்சங்களை வழங்கும். அது கிடைக்காதபோது, இணைக்கப்பட்ட subscriptions core இன்னும் recurring products-ஐ கையாளும்.

![WooCommerce subscription integration அமைப்புகள்](/img/addons/woocommerce-subscription-integration.png)

அவ்வளவுதான். உங்கள் இறுதிப் பயனர்கள் recurring products-க்கு தங்கள் billing cycle முடிவில் இனி கைமுறையாக கட்டணம் செலுத்த வேண்டியதில்லை.

_WooCommerce_ கீழ் உங்கள் main site-இல் ஒரு order உருவாக்கப்படுவதுடன், ஒவ்வொரு Account-க்கும் ஒரு subscription-ஐயும் உருவாக்கும்.

### தோல்வியடைந்த புதுப்பிப்பு இடைநிறுத்த நடத்தை {#failed-renewal-suspension-behavior}

இயல்பாக, WooCommerce Subscriptions அதன் failed-payment retry schedule-ஐ செயலாக்கும் போது memberships செயலில் இருக்கும். இதனால் வழக்கமான WooCommerce Subscriptions grace window பாதுகாக்கப்படுகிறது; மீளப்பெற்ற renewal, subscription மற்றும் membership-ஐ முன்கூட்டியே அணுகலைத் தடுக்காமல் மீண்டும் ஒத்திசைக்க முடியும்.

கடுமையான அணுகல் கட்டுப்பாட்டை விரும்பும் administrators, WooCommerce Integration settings-இல் **Suspend Memberships Immediately on Failed Renewals**-ஐ இயக்கலாம். இந்த opt-in setting இயக்கப்பட்டால், WooCommerce Subscriptions retry handling முடியும் வரை காத்திருக்காமல், renewal payment தோல்வியடைந்தவுடன் Ultimate Multisite தொடர்புடைய membership-ஐ இடைநிறுத்தும்.

Developers அதே முடிவை `wu_woo_suspend_on_payment_failure` filter மூலம் override செய்யலாம். Hook reference pages add-on source-இலிருந்து உருவாக்கப்படுகின்றன, எனவே இந்த overview custom suspension policies-க்காக filter name-ஐ மட்டும் குறிப்பிடுகிறது.
