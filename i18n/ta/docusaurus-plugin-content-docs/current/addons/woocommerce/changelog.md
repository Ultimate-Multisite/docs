# WooCommerce ஒருங்கிணைப்பு மாற்றப் பதிவு

பதிப்பு 2.2.0 - 2026-07-01 அன்று வெளியிடப்பட்டது
* புதியது: Ultimate Multisite வரித் தொகைகள் இப்போது பணம் செலுத்தும் போது தனி WooCommerce கட்டண வரிகளாகத் தோன்றும். இதனால் பணம் செலுத்துவதற்கு முன் வரி மொத்தங்கள் தெளிவாகத் தெரியும்.
* புதியது: WooCommerce Subscriptions மீண்டும் முயற்சி சாளரத்தின் போது தோல்வியடைந்த புதுப்பிப்புகளுக்கான இடைநிறுத்தத்தை விரும்பும் தளங்களுக்காக, விருப்பமாக இயக்கக்கூடிய "தோல்வியடைந்த புதுப்பிப்புகளில் உறுப்பினர் பதிவுகளை உடனடியாக இடைநிறுத்து" அமைப்பும் `wu_woo_suspend_on_payment_failure` filter-உம் சேர்க்கப்பட்டது.
* சரி செய்யப்பட்டது: தோல்வியடைந்த அல்லது மீட்கப்பட்ட புதுப்பிப்புகளுக்குப் பிறகு Ultimate Multisite உறுப்பினர் பதிவுகளுடன் ஒத்திசைவில் இல்லாமல் இருக்கக்கூடிய WooCommerce Subscription நிலைகள் ஒத்திசைக்கப்பட்டன.
* சரி செய்யப்பட்டது: Ultimate Multisite நாணயப் பட்டியலில் இல்லாதபோது WooCommerce கடை நாணயம் சேர்க்கப்பட்டது.
* சரி செய்யப்பட்டது: சந்தாதாரர்களை WooCommerce பணம் செலுத்தும் பக்கத்திற்கு திருப்பியனுப்பும்போது வாடிக்கையாளர் பில்லிங் விவரங்கள் பாதுகாக்கப்பட்டன.
* மேம்படுத்தப்பட்டது: Jetpack Autoloader 5 உடன் இணக்கத்தன்மை சேர்க்கப்பட்டது.
* மேம்படுத்தப்பட்டது: GitHub மற்றும் சந்தை zip-களில் உள்ளமைந்த staging அடைவுகளும் வளர்ச்சி கோப்புகளும் சேராமல் இருக்க வெளியீட்டு தொகுப்பு உருவாக்கம் சீர்செய்யப்பட்டது.

* சரி செய்யப்பட்டது: WooCommerce பணம் செலுத்தும் பக்கத்தில் ரத்து செய்தாலும் உறுப்பினர் பதிவை இன்னும் மேம்படுத்தக்கூடிய நிலை.

பதிப்பு 2.0.1 - 2023-08-09 அன்று வெளியிடப்பட்டது

* சேர்க்கப்பட்டது: Woocommerce 7.9.0 உடன் இணக்கத்தன்மை.
* சேர்க்கப்பட்டது: Woocommerce Subscriptions 5.3.0 உடன் இணக்கத்தன்மை.
* சேர்க்கப்பட்டது: உறுப்பினர் பதிவு புதுப்பிப்புகளுக்கான ஆதரவு.
* சேர்க்கப்பட்டது: Woocommerce-இல் சோதனை காலங்களும் அமைப்பு கட்டணங்களும் பற்றிய அறிவிப்புகள்.
* சேர்க்கப்பட்டது: meta மதிப்புடன் Ultimate Multisite Woocommerce தயாரிப்புகளை அடையாளம் காணுதல்.
* சேர்க்கப்பட்டது: Ultimate Multisite தொடர்புடைய அனைத்து Woocommerce தயாரிப்புகளையும் குறிக்க ஒருமுறை செய்யும் சரிசெய்தல் சேர்க்கப்பட்டது.
* சேர்க்கப்பட்டது: Ultimate Multisite உருவாக்கிய தயாரிப்புகள் Woocommerce பட்டியலிலிருந்து நீக்கப்பட்டன.
* மேம்பாடு: வண்டியில் பயன்படுத்த மீண்டும் வராத Woocommerce தள்ளுபடி உருவாக்கப்பட்டது.
* மேம்பாடு: மீண்டும் வரும் தள்ளுபடி Woocommerce தயாரிப்பில் மீட்டமைக்கப்பட்டது.
* மேம்பாடு: Woocommerce தயாரிப்பில் மீண்டும் வரும் தள்ளுபடி லேபிள் சேர்க்கப்பட்டது.
* மேம்பாடு: பணம் செலுத்தும் போது தயாரிப்பு வகை உறுதி செய்யப்பட்டது.
* சரி செய்யப்பட்டது: downgrade செயல்முறையின் போது உறுப்பினர் பதிவு நிலை பராமரிக்கப்பட்டது.
* சரி செய்யப்பட்டது: ரத்து செயல்முறையின் போது பிழைகளைத் தவிர்க்க subscription இருக்கிறதா என்று சரிபார்க்கப்பட்டது.
* சரி செய்யப்பட்டது: Woocommerce subscriptions-இல் பயன்படுத்த subscription தொடக்க தேதி சேர்க்கப்பட்டது.
* உள்: புதிய PHP 8.1 build செயல்முறை செயல்படுத்தப்பட்டது.
