* சேர்க்கப்பட்டது: Woocommerce 10.2.1 உடன் இணக்கத்தன்மை சேர்க்கப்பட்டது.

Version 2.0.2 - 2025-07-05 அன்று வெளியிடப்பட்டது
* மாற்றப்பட்டது: Multisite Ultimate: Woocommerce Integration என்று பெயர் மாற்றப்பட்டது.
* சேர்க்கப்பட்டது: Woocommerce 9.8.1 உடன் இணக்கத்தன்மை சேர்க்கப்பட்டது.
* சேர்க்கப்பட்டது: Woocommerce Subscriptions 7.3.0 உடன் இணக்கத்தன்மை சேர்க்கப்பட்டது.
* சரி: வாடிக்கையாளர் ஒரு சந்தாவை ரத்துசெய்வது சரி செய்யப்பட்டது.

* மேம்பாடு: இப்போது Woocommerce high performance custom order tables உடன் இணக்கமானது.
* சரி: WooCommerce checkout-ல் ரத்துசெய்வது, ஒரு உறுப்பினர் நிலையை (membership) மேம்படுத்தும் (upgrade) தன்மையைக் கொண்டிருக்கலாம்.

Version 2.0.1 - 2023-08-09 அன்று வெளியிடப்பட்டது

* சேர்க்கப்பட்டது: Woocommerce 7.9.0 உடன் இணக்கத்தன்மை சேர்க்கப்பட்டது.
* சேர்க்கப்பட்டது: Woocommerce Subscriptions 5.3.0 உடன் இணக்கத்தன்மை சேர்க்கப்பட்டது.
* சேர்க்கப்பட்டது: உறுப்பினர் புதுப்பித்தங்களுக்கான ஆதரவு சேர்க்கப்பட்டது.
* சேர்க்கப்பட்டது: Woocommerce-ல் trial மற்றும் setup fees பற்றிய அறிவிப்புகள் சேர்க்கப்பட்டன.
* சேர்க்கப்பட்டது: ஒரு meta value உடன் Ultimate Multisite Woocommerce தயாரிப்புகளை அடையாளம் காணுதல் சேர்க்கப்பட்டது.
* சேர்க்கப்பட்டது: அனைத்து Ultimate Multisite தொடர்பான Woocommerce தயாரிப்புகளையும் குறிக்க ஒரு முறை சரி சேர்க்கப்பட்டது.
* சேர்க்கப்பட்டது: Woocommerce பட்டியலில் இருந்து Ultimate Multisite உருவாக்கிய தயாரிப்புகள் அகற்றப்பட்டன.
* மேம்பாடு: வண்டியில் (cart) பயன்படுத்த ஒரு முறை மட்டும் (non-recurring) Woocommerce தள்ளுபடி சேர்க்கப்பட்டது.
* மேம்பாடு: Woocommerce தயாரிப்புக்கு மீண்டும் வரும் தள்ளுபடி மீட்டெடுக்கப்பட்டது.
* மேம்பாடு: Woocommerce தயாரிப்புக்கு மீண்டும் வரும் தள்ளுபடி லேபிள் சேர்க்கப்பட்டது.
* மேம்பாடு: checkout-ல் தயாரிப்பு வகை (product type) உறுதி செய்யப்பட்டுள்ளது.
* சரி: downgrade செயல்முறை நடந்தாலும் உறுப்பினர் நிலை பராமரிக்கப்பட்டது.
* சரி: ரத்துசெய்வது (cancellation) செயல்முறைக்கு இடையே பிழைகளைத் தவிர்க்க ஒரு சந்தா உள்ளதா எனச் சரிபார்க்கப்பட்டது.
* சரி: Woocommerce சந்தாக்களில் பயன்படுத்த தொடக்க சந்தா தேதி சேர்க்கப்பட்டது.
* உள்: ஒரு புதிய PHP 8.1 build process செயல்படுத்தப்பட்டது.

Version 2.0.0 - முழுமையாக மீண்டும் எழுதப்பட்டது.

* சேர்க்கப்பட்டது: gateway மாற்றுும்போது அல்லது உறுப்பினர் ரத்துசெய்ும்போது woo சந்தாவை அகற்ற செயல்முறை சேர்க்கப்பட்டது;

* சரி: WU Membership பட்டன் இணைப்பிற்குச் செல்லுதல் சரி செய்யப்பட்டது;
* சரி: Woocommerce சந்தா புதுப்பிப்பு பணம் செலுத்தியிருந்தால், Ultimo ஆர்டர் 'paid' என்று அமைக்கப்பட்டது;
* உருவாக்கப்பட்டது: builder ஆக MPB சேர்க்கப்பட்டது;
