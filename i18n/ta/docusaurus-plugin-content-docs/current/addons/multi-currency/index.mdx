# Multi-Currency {#multi-currency}

Ultimate Multisite: Multi-Currency உங்கள் நெட்வொர்க்கிற்கு பல நாணயங்களில் பணம் ஏற்றுக்கொள்ள உதவுகிறது. இது லைவ் அல்லது கைமுறையான மாற்று விகிதங்களைப் பயன்படுத்தி தானாகவே விலைகளை மாற்றுகிறது, எந்த நாணயத்திற்கும் தயாரிப்பு வாரியான சரியான விலைகளை அமைக்க அனுமதிக்கிறது, மேலும் புவிஇருப்பிடத்தின் (geolocation) மூலம் ஒவ்வொரு பார்வையாளரின் விருப்பமான நாணயத்தையும் கண்டறிகிறது — இவை அனைத்தும் ஒரே நிர்வாக டாஷ்போர்டில் நிர்வகிக்கப்படுகின்றன.

## முக்கிய அம்சங்கள் (Key Features) {#key-features}

- **தானியங்கி மாற்று விகிதம் மாற்றம் (Automatic exchange rate conversion)** — நீங்கள் தேர்ந்தெடுக்கும் வழங்குநரிடமிருந்து (provider) பெறப்பட்ட விகிதங்களைப் பயன்படுத்தி உங்கள் அடிப்படை நாணயத்திலிருந்து விலைகள் மாற்றப்படுகின்றன.
- **தயாரிப்பு வாரியான விலை மாற்றியமைத்தல் (Per-product pricing overrides)** — தானியங்கி மாற்றத்தைத் தவிர்க்க, எந்தவொரு தயாரிப்பிலும் குறிப்பிட்ட நாணயத்திற்கான சரியான விலைகளை அமைக்கலாம்.
- **நாணயம் தேர்வுப் பதிவுப் புலம் (Currency selector signup field)** — வாடிக்கையாளர்கள் checkout/பதிவு செய்யும் போது தங்கள் விருப்பமான நாணயத்தைத் தேர்வு செய்ய இது அனுமதிக்கிறது.
- **பல மாற்று விகித வழங்குநர்கள் (Multiple exchange rate providers)** — ECB, Frankfurter, Stripe, அல்லது முழுமையாக கைமுறை விகிதங்கள்.
- **புவிஇருப்பிடத்தின் அடிப்படையில் நாணயம் கண்டறிதல் (Geolocation-based currency detection)** — பார்வையாளரின் நாட்டிற்குப் பொருந்தக்கூடிய நாணயத்தை தானாகவே முன்னரே தேர்வு செய்கிறது.
- **நிர்வாக டாஷ்போர்டு (Admin dashboard)** — актив நாணயங்கள், மாற்று விகிதங்கள் மற்றும் வழங்குநர் அமைப்புகளை ஒரே இடத்தில் நிர்வகிக்கவும்.

## தேவைகள் (Requirements) {#requirements}

- Ultimate Multisite 2.0.0 அல்லது அதற்கு மேல்
- WordPress Multisite நிறுவல்
- PHP 7.4 அல்லது அதற்கு மேல்

## நிறுவுதல் (Installation) {#installation}

1. Plugin கோப்புகளை `/wp-content/plugins/ultimate-multisite-multi-currency` என்ற இடத்தில் பதிவேற்றவும்.
2. WordPress-ல் உள்ள **Plugins** திரையின் மூலம் Plugin-ஐ செயல்படுத்தவும் (Network Activate).
3. நிர்வாக டாஷ்போர்டைத் திறக்க **Ultimate Multisite → Multi-Currency** என்பதற்குச் செல்லவும்.
4. நீங்கள் ஏற்றுக்கொள்ள விரும்பும் நாணயங்களைச் சேர்த்து, உங்கள் மாற்று விகித வழங்குநரை (exchange rate provider) கட்டமைக்கவும்.

## உள்ளமைவு (Configuration) {#configuration}

### நிர்வாக டாஷ்போர்டு (Admin Dashboard) {#admin-dashboard}

Multi-Currency நிர்வாக டாஷ்போர்டு (**Ultimate Multisite → Multi-Currency**) என்பது பின்வருவனவற்றைச் செய்வதற்கான மைய இடமாகும்:

- நாணயங்களைச் சேர்ப்பது, செயல்படுத்துவது அல்லது முடக்குவது.
- ஒரு மாற்று விகித வழங்குநரைத் தேர்ந்தெடுத்து, தேவைக்கேற்ப விகிதங்களைப் புதுப்பிப்பது.
- அனைத்து актив நாணயங்களுக்கான தற்போதைய விகிதங்களை மதிப்பாய்வு செய்வது.
- தானியங்கி வழங்குநர்கள் பொருத்தமானதாக இல்லாதபோது கைமுறை விகிதங்களை அமைப்பது.

### மாற்று விகித வழங்குநர்கள் (Exchange Rate Providers) {#exchange-rate-providers}

| Provider | Description |
|---|---|
| **ECB** | European Central Bank — EUR அடிப்படையிலான நெட்வொர்க்குகளுக்கு இலவச தினசரி விகிதங்கள். |
| **Frankfurter** | Open-source ECB mirror API — இலவசம், API key தேவையில்லை. |
| **Stripe** | Stripe-இன் லைவ் FX விகிதங்களைப் பயன்படுத்துகிறது — актив Stripe ஒருங்கிணைப்பு தேவை. |
| **Manual** | நீங்கள் நேரடியாக விகிதங்களை உள்ளிடுகிறீர்கள்; வெளிப்புற API அழைப்புகள் இல்லை. |

**Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider** என்ற இடத்தில் உங்கள் வழங்குநரைத் தேர்ந்தெடுக்கவும். விகிதங்களை எந்த நேரத்திலும் கைமுறையாகப் புதுப்பிக்கலாம் அல்லது தானாகவே புதுப்பிக்க திட்டமிடலாம்.

### தயாரிப்பு வாரியான விலை மாற்றியமைத்தல் (Per-Product Pricing Overrides) {#per-product-pricing-overrides}

தானியங்கி மாற்றம் போதுமான துல்லியமாக இல்லாவிட்டால், எந்தவொரு தயாரிப்பிலும் குறிப்பிட்ட நாணயத்திற்கான சரியான விலையை நீங்கள் அமைக்கலாம்:

1. தயாரிப்பு எடிட்டரைத் திறக்கவும் (**Ultimate Multisite → Products → Edit**).
2. **Currency Pricing** பகுதிக்கு ஸ்க்ரோல் செய்யவும்.
3. நீங்கள் மாற்றியமைக்க விரும்பும் ஒவ்வொரு நாணயத்திற்கான சரியான விலையை உள்ளிடவும்.
4. தயாரிப்பைச் சேமிக்கவும்.

மாற்றியமைக்கப்படாத தயாரிப்புகள் உங்கள் அடிப்படை நாணயத்திலிருந்து மாற்றப்பட்ட விலையைப் பயன்படுத்துகின்றன.

### நாணயம் தேர்வுப் பதிவுப் புலம் (Currency Selector Signup Field) {#currency-selector-signup-field}

வாடிக்கையாளர்கள் பதிவு செய்யும்போதே தங்கள் விருப்பமான நாணயத்தைத் தேர்வு செய்ய, நாணயம் தேர்வுப் புலத்தை உங்கள் பதிவு/பணம் செலுத்தும் செயல்முறைக்குச் சேர்க்கலாம்:

1. **Ultimate Multisite → Multi-Currency → Settings** என்பதற்குச் செல்லவும்.
2. **Show currency selector on signup** என்பதை இயக்கவும்.
3. விருப்பப்பட்டால், ஒரு இயல்புநிலை நாணயத்தை அமைக்கலாம் அல்லது புவிஇருப்பிடம் முன்னரே தேர்ந்தெடுக்கப்பட்ட மதிப்பைத் தீர்மானிக்க அனுமதிக்கலாம்.

தேர்ந்தெடுக்கப்பட்ட நாணயம் சந்தாவுடன் சேமிக்கப்பட்டு, அனைத்து அடுத்தடுத்த கட்டணங்களுக்கும் பயன்படுத்தப்படுகிறது.

### புவிஇருப்பிடத்தின் அடிப்படையில் நாணயம் கண்டறிதல் (Geolocation-Based Currency Detection) {#geolocation-based-currency-detection}

புவிஇருப்பிடக் கண்டறிதல் இயக்கப்படும்போது, இந்த addon பார்வையாளரின் நாட்டத்தை ஒரு நாணயத்துடன் இணைத்து, நாணயம் தேர்வுப் புலத்தில் அதை முன்னரே தேர்வு செய்கிறது. இதை இயக்க:

1. **Ultimate Multisite → Multi-Currency → Settings** என்பதற்குச் செல்லவும்.
2. **Detect currency from visitor location** என்பதை இயக்கவும்.
3. **Geolocation Mapping** அட்டவணையில் நாடுகளை நாணயங்களுடன் இணைக்கவும்.

பார்வையாளரின் நாட்டிற்கு எந்த இணைப்பும் இல்லையென்றால், நெட்வொர்க்கின் அடிப்படை நாணயம் fallback ஆகப் பயன்படுத்தப்படும்.

## அடிக்கடி கேட்கப்படும் கேள்விகள் (Frequently Asked Questions) {#frequently-asked-questions}

### மாற்று விகித வழங்குநர் கிடைக்கவில்லை என்றால் என்ன நடக்கும்? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

addon கடைசியாக வெற்றிகரமாகப் பெறப்பட்ட விகிதங்களுக்குத் திரும்பும் (falls back). எந்த விகிதங்களும் இதுவரை பெறப்படவில்லை என்றால், நீங்கள் கட்டமைத்த கைமுறை விகிதங்களுக்குத் திரும்பும். ஒரு பாதுகாப்பு வலையாக கைமுறை விகிதங்களை கட்டமைப்பது பரிந்துரைக்கப்படுகிறது.

### வெவ்வேறு நாணயங்களுக்கு வெவ்வேறு வழங்குநர்களைப் பயன்படுத்த முடியுமா? {#can-i-use-different-providers-for-different-currencies}

முடியாது — அனைத்து நாணயங்களுக்கும் ஒரே வழங்குநர் மட்டுமே செயல்படும். தனிப்பட்ட விகிதங்கள் மீது முழு கட்டுப்பாட்டைக் கொண்டிருக்க நீங்கள் **Manual** பயன்படுத்தவும்.

### புவிஇருப்பிடத்திற்கு தனி Plugin தேவையா? {#does-geolocation-require-a-separate-plugin}

வெளிப்புற புவிஇருப்பிட Plugin தேவையில்லை. addon ஆனது WordPress-இன் உள்ளமைக்கப்பட்ட IP-to-country கண்டறிதலைப் பயன்படுத்துகிறது. அதிக துல்லியத்திற்காக, வழங்கப்பட்ட filter hooks மூலம் பிரத்யேக புவிஇருப்பிட சேவையை ஒருங்கிணைக்க முடியும்.

### தயாரிப்பு வாரியான மாற்றியமைப்புகள் வரிக்கு முன் அல்லது பின் சேர்க்கப்படுகிறதா? {#are-per-product-overrides-applied-before-or-after-tax}

தயாரிப்பு வாரியான மாற்றியமைப்புகள் இலக்கு நாணயத்தில் வரிக்கு முந்தைய விலையை அமைக்கின்றன. உங்கள் இருக்கும் வரி உள்ளமைவைப் பயன்படுத்தி மாற்றியமைக்கப்பட்ட விலையின் மேல் வரி கணக்கிடப்படுகிறது.
