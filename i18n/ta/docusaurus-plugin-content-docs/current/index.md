# Ultimate Multisite ஆவணங்கள்

Ultimate Multisite ஆவணங்களுக்கு உங்களை வரவேற்கிறோம். Ultimate Multisite, WordPress Multisite-ஐ முழுமையான Website as a Service (WaaS) தளமாக மாற்றுகிறது.

## ஆவணப் பிரிவுகள் {#documentation-sections}

### [பயனர் வழிகாட்டி](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

உங்கள் WaaS தளத்தை அமைக்கவும், கட்டமைக்கவும் மற்றும் நிர்வகிக்கவும் உங்களுக்குத் தேவையான அனைத்தும் இதில் உள்ளன. இது நிறுவுதல், பணம் செலுத்தும் நுழைவாயில்கள் (payment gateways), டொமைன் மேப்பிங், வாடிக்கையாளர் மேலாண்மை மற்றும் பலவற்றை உள்ளடக்கியது.

### [டெவலப்பர் ஆவணங்கள்](/developer/hooks) {#developer-documentation}

Ultimate Multisite-ஐ அடிப்படையாகக் கொண்டு உருவாக்கும் உருவாக்குநர்களுக்கான தொழில்நுட்ப குறிப்பு இது. இதில் hooks குறிப்பு, REST API ஆவணங்கள் மற்றும் addon உருவாக்கும் வழிகாட்டிகள் ஆகியவை அடங்கும்.

### [சேர்க்கைகள்](/addons) {#addons}

Ultimate Multisite-ஐ WooCommerce ஒருங்கிணைப்பு, பகுப்பாய்வு (analytics), டொமைன் விற்பனை போன்ற கூடுதல் அம்சங்களுடன் விரிவாக்கும் கட்டண addon plugins-க்கான ஆவணம் இது.
