# Filter: wu_should_redirect_to_primary_domain

திசை திருப்புதல் (redirection) நடக்காமல் இருக்க, இதை பயன்படுத்தி அதைத் தடுக்க (short-circuit) டெவலப்பர்கள் வழிவகை செய்யலாம்.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $should_redirect | `bool` | திசை திருப்புதல் செய்ய வேண்டுமா இல்லையா என்பதை இது குறிக்கும். |

### Since {#since}

- 2.0.0
### Source {#source}

[`inc/domain-mapping/class-primary-domain.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/domain-mapping/class-primary-domain.php#L79) என்ற கோப்பில், 79-வது வரியில் வரையறுக்கப்பட்டுள்ளது.
