# Filter: wp_ultimo_redirect_url_after_signup

திருப்பி அனுப்ப வேண்டிய URL-ஐ அமைக்கவும்.

இது ஒரு பழைய (legacy) filter ஆகும். $site_id போன்ற சில அளவுருக்கள் (parameters) கிடைக்காது.

### இருந்தே {#since}

- 1.1.3: டெவலப்பர்கள் திருப்பி அனுப்ப வேண்டிய URL-ஐ வடிகட்ட (filter) அனுமதிக்கிறது.
