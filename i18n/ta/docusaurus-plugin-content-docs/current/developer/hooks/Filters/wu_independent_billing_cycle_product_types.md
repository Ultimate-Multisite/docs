# Filter: wu_independent_billing_cycle_product_types

தனிப்பட்ட बिलिंग சுழற்சிகளைக் (independent billing cycles) கொண்ட தயாரிப்பு வகைகளை வடிகட்டுகிறது.

தனிப்பட்ட बिलिंग சுழற்சிகளைக் கொண்ட தயாரிப்புகள், கார்ட்டில் உள்ள திட்டத்தின் (plan) बिलिंग காலத்துடன் பொருந்த வேண்டிய கட்டாயம் இல்லை. இந்த அம்சம், திட்டம் மாதாந்திரமாகவோ அல்லது ஆண்டுதோறும் என்றாலும், டொமைன் பதிவு (domain registrations) போன்ற தயாரிப்புகளுக்கு மிகவும் பயனுள்ளதாக இருக்கும்.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | தனிப்பட்ட बिलिंग சுழற்சிகளைக் கொண்ட தயாரிப்பு வகைகளின் (product types) வரிசை (array). |
