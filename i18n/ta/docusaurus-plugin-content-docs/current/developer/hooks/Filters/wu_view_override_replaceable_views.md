# Filter: wu_view_override_replaceable_views

மாற்றக்கூடிய பட்டியலில் (replaceable list) கூடுதல் கோப்புறைகளைச் சேர்க்க இது டெவலப்பர்களுக்கு உதவுகிறது.

கவனமாக இருக்கவும், ஏனெனில் கூடுதல் கோப்புறைகளை அனுமதிப்பது, Ultimate Multisite பதிப்புகளுக்குப் பதிலாக காலாவதியான (out-of-date) பிரதிகளை ஏற்ற வழிவகுக்கக்கூடும்.

## அளவுருக்கள் (Parameters) {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | அனுமதிக்கப்பட்ட கோப்புறைகளின் பட்டியல். |

### இருந்தே (Since) {#since}

- 2.0.0
### மூலம் (Source) {#source}

[`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) என்ற இடத்தில், 50-வது வரியில் வரையறுக்கப்பட்டுள்ளது.
