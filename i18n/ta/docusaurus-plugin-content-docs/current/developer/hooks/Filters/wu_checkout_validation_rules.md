# Filter: wu_checkout_validation_rules

Plugin உருவாக்குபவர்கள் சரிபார்ப்பு விதிகளை (validation rules) ஃபில்டர் செய்ய இது அனுமதிக்கிறது.

## அளவுருக்கள் (Parameters) {#parameters}

| Name | Type | விளக்கம் (Description) |
|------|------|-------------|
| $validation_rules | `array` | பயன்படுத்தப்பட வேண்டிய சரிபார்ப்பு விதிகள். |
| $checkout | `\Checkout` | செக்அவுட் வகுப்பு (checkout class). |

### எப்போது (Since) {#since}

- 2.0.20
### மூலம் (Source) {#source}

[`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L2092) என்ற இடத்தில், 2092 வரியில் வரையறுக்கப்பட்டுள்ளது.
