# Filter: wu_password_require_number

கடவுச்சொற்களில் எண்கள் தேவைப்படுகிறதா என்பதை வடிகட்டுகிறது.

## அளவுருக்கள்

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | எண்கள் தேவையா என்பதை இது குறிக்கிறது. விதிகள் செயல்படுத்தப்பட்டால் இயல்பாகவே உண்மை (true) என இருக்கும். |
| $defender_active | `bool` | Defender Pro Strong Password செயல்படுகிறதா என்பதை இது குறிக்கிறது. |
