# Filter: wu_is_api_enabled

API-க்கு ஒரு குறிப்பிட்ட நிலையை (state) கட்டாயமாக்க, plugin உருவாக்குநர்களுக்கு இது அனுமதிக்கிறது.

### Since

- 1.7.4
### Source

[`inc/class-api.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-api.php#L435) என்ற கோப்பில், 435 வரியில் வரையறுக்கப்பட்டுள்ளது.

## Returns
(இது ஒரு பூலியன் மதிப்பை (boolean value) திரும்ப அளிக்கும்.)
