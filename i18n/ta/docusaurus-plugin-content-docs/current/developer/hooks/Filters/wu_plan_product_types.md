# Filter: wu_plan_product_types

திட்டங்களாகக் (plans) கருதப்பட வேண்டிய தயாரிப்பு வகைகளை வடிகட்டுகிறது.

இந்த ஃபில்டர், சரிபார்ப்பு மற்றும் பிரித்தெடுத்தல் (validation and segregation) ஆகியவற்றில் திட்டங்களாக அங்கீகரிக்கப்பட வேண்டிய கூடுதல் தயாரிப்பு வகைகளை (product types) add-on-களுக்குப் பதிவு செய்ய அனுமதிக்கிறது.

## அளவுருக்கள்

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | திட்டங்களாகக் கருதப்பட வேண்டிய தயாரிப்பு வகைகளின் வரிசை (Array of product types to treat as plans). |

- 2.3.0
