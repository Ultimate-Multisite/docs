# Filter: wu_stripe_existing_product_id {#filter-wustripeexistingproductid}

சரிபார்க்க வேண்டிய தயாரிப்பின் ஐடியை (ID) இது ஃபில்டர் செய்கிறது. இது இருந்தால், புதிய சந்தா இந்த தயாரிப்பைப் பயன்படுத்தும்.

## அளவுருக்கள் {#parameters}

| Name | Type | விளக்கம் |
|------|------|-------------|
| $product_id | `string` | சரிபார்க்க வேண்டிய Stripe தயாரிப்பின் ஐடி. |
| $name | `string` | தயாரிப்பின் பெயர். |

### மூலம் {#source}

இது [`inc/gateways/class-base-stripe-gateway.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/gateways/class-base-stripe-gateway.php#L2741) என்ற இடத்தில், 2741 வரியில் வரையறுக்கப்பட்டுள்ளது.
