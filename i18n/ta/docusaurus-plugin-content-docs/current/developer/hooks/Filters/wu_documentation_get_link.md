# Filter: wu_documentation_get_link

Plugin உருவாக்குநர்கள் இணைப்புகளை வடிகட்ட (filter) அனுமதிக்கிறது.

இது எப்படி பயனுள்ளதாக இருக்கும் என்று எனக்குத் தெரியவில்லை, ஆனால் இதை வைத்திருப்பது தவறில்லை.

## அளவுருக்கள் (Parameters) {#parameters}

| பெயர் (Name) | வகை (Type) | விளக்கம் (Description) |
|------|------|-------------|
| $link | `string` | பதிவு செய்யப்பட்ட இணைப்பு (The link registered) |
| $slug | `string` | இணைப்பை மீட்டெடுக்கப் பயன்படுத்தப்படும் ஸ்லக் (The slug used to retrieve the link) |
| $default_link | `string` | பதிவு செய்யப்பட்ட இயல்புநிலை இணைப்பு (The default link registered) |

### எப்போது (Since) {#since}

- 1.7.0
