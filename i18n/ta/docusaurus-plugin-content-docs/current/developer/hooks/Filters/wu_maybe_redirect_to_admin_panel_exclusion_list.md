# Filter: wu_maybe_redirect_to_admin_panel_exclusion_list {#filter-wumayberedirecttoadminpanelexclusionlist}

தானாக திருப்பிவிடுதலைத் (auto-redirect) தடுக்க உதவும் அளவுருக்களின் (parameters) விலக்கு பட்டியலை (exclusion list) உருவாக்க இது பயன்படுகிறது.

இதைச் செய்ய வேண்டியது என்னவென்றால், இல்லையெனில், ஒருமுறை உள்நுழைவுப் பக்கம் (login page) வரையறுக்கப்பட்ட பிறகு, page builder அதை எடிட் செய்ய முடியாது.

### Since {#since}

- 2.0.4
### Source {#source}

[`inc/checkout/class-checkout-pages.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout-pages.php#L357) என்ற கோப்பில், 357 வரியில் வரையறுக்கப்பட்டுள்ளது.
