# Filter: wu_is_mcp_enabled {#filter-wuismcpenabled}

MCP adapter-க்கு ஒரு குறிப்பிட்ட நிலையை (state) கட்டாயமாக்க, plugin உருவாக்குபவர்களுக்கு இது அனுமதிக்கிறது.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | MCP adapter இயக்கப்பட்டுள்ளதா இல்லையா என்பதைக் குறிக்கிறது. |

### Since {#since}

- 2.5.0
### Source {#source}

[`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L250) என்ற கோப்பில், 250-வது வரியில் வரையறுக்கப்பட்டுள்ளது.
