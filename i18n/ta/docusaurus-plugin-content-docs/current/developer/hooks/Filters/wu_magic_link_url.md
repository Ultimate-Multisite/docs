# Filter: wu_magic_link_url {#filter-wumagiclinkurl}

உருவாக்கப்பட்ட magic link URL-ஐ வடிகட்டுகிறது.

## அளவுருக்கள் {#parameters}

| Name | Type | விளக்கம் |
|------|------|-------------|
| $magic_link | `string` | magic link URL. |
| $user_id | `int` | பயனர் ID. |
| $site_id | `int` | தள ID. |
| $redirect_to | `string` | திருப்பி அனுப்ப வேண்டிய URL. |
