# Filter: wu_add_new_domain_modal_fields

புதிய டொமைனைச் சேர்க்கும் மாடலில் உள்ள புலங்களை (fields) இது வடிகட்டுகிறது.

ஆட்-ஆன்கள் (addons) (உதாரணமாக, Domain Seller) டொமைன் பதிவு புலங்களை மாற்றியமைக்கவோ அல்லது மாற்றீடு செய்யவோ இதை அனுமதிக்கிறது.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | படிவ புலங்கள். |
