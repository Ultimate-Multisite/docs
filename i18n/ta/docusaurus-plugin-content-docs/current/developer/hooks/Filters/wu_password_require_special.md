## அளவுருக்கள் {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | சிறப்பு எழுத்துக்கள் தேவையா என்பதை இது தீர்மானிக்கிறது. விதிகள் செயல்படுத்தப்பட்டால் இயல்பாக உண்மை (true) என இருக்கும். |
