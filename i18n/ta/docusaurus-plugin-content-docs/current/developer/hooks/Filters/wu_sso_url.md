# வடிகட்டி: wu_sso_url

குறுக்கு-டொமைன் வாடிக்கையாளர் செயல்களுக்காக திருப்பி வழங்குவதற்கு முன், உருவாக்கப்பட்ட SSO URLகளை வடிகட்டுகிறது.

ஒரு ஒருங்கிணைப்புக்கு sovereign-tenant SSO இணைப்பில் நம்பகமான சூழலைச் சேர்க்க வேண்டியபோது, அல்லது Ultimate Multisite-இன் token சரிபார்ப்பை பாதுகாத்தபடி broker URL-ஐ மாற்ற வேண்டியபோது இந்த வடிகட்டியைப் பயன்படுத்தவும்.

## அளவுருக்கள்

| பெயர் | வகை | விளக்கம் |
|------|------|-------------|
| $sso_url | `string` | உருவாக்கப்பட்ட SSO URL. |
| $user | `WP_User` | SSO வருகையால் அங்கீகரிக்கப்படவுள்ள பயனர். |
| $site_id | `int` | வருகைக்கான இலக்கு தள ID. |
| $redirect_to | `string` | வெற்றிகரமான SSO சரிபார்ப்புக்குப் பிறகான இலக்கு URL. |

### முதல்

- 2.13.0

### மூலம்

`inc/sso/class-sso.php` இல் வரையறுக்கப்பட்டுள்ளது.


## திரும்பும் மதிப்பு

வடிகட்டப்பட்ட SSO URL.
