# Filter: wu_bypass_checkout_form

டெவலப்பர்கள் வெளியீட்டைத் தாண்டிச் செல்லவும், அதற்குப் பதிலாக புதிய ஒன்றைப் அமைக்கவும் இது அனுமதிக்கிறது.

## அளவுருக்கள் {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $bypass | `string\|bool` | செக்அவுட் படிவத்தைத் தாண்டிச் செல்ல வேண்டுமா என்பதைக் குறிக்க அல்லது படிவத்திற்குப் பதிலாகத் திருப்பி அனுப்ப வேண்டிய ஒரு சரம் (string) கொடுக்கவும். |
| $atts | `array` | செக்அவுட் பிளாக்/ஷார்ட்கோடைப் பயன்படுத்தும் அளவுருக்கள். |
