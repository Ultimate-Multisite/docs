# Filter: wu_upgrade_to_unlock_url

டெவலப்பர்கள் மேம்படுத்தி திறக்க வேண்டிய URL-ஐ மாற்ற இது அனுமதிக்கிறது.

## அளவுருக்கள் {#parameters}

| பெயர் | வகை | விளக்கம் |
|------|------|-------------|
| $url | `string` | மேம்படுத்த வேண்டிய URL. |
| $args | `array` | மேம்படுத்தப்பட வேண்டிய வரம்பின் (limit) மாட்யூல் மற்றும் வகை. |

### மூலம் {#source}

[`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) என்ற கோப்பில் 177 வரியில் வரையறுக்கப்பட்டுள்ளது.
