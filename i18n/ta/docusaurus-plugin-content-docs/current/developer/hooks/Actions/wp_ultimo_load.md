# Action: wp_ultimo_load

அனைத்து சார்புநிலைகளும் (dependencies) ஏற்றப்பட்ட பிறகு இது தூண்டப்படுகிறது (Triggers).

இது பிளக்இன் உருவாக்குநர்கள் புதிய செயல்பாடுகளைச் சேர்க்க அனுமதிக்கிறது. உதாரணமாக, புதிய ஹோஸ்டிங் வழங்குநர்களுக்கான ஆதரவு போன்றவற்றைச் சேர்க்கலாம்.
