# Action: auth_redirect {#action-authredirect}

அங்கீகார திருப்பிவிடுதல் (authentication redirect) நடப்பதற்கு முன்பே இது இயங்கும்.

## அளவுருக்கள் {#parameters}

| Name | Type | விளக்கம் |
|------|------|-------------|
| $user_id | `int` | பயனர் ஐடி (User ID). |

### எப்போது இருந்து {#since}

- 2.8.0
