# Action: wu_domain_mapping_load {#action-wudomainmappingload}

எங்கள் அடிப்படை Domain Mapping ஏற்றப்பட்ட பிறகு இது தூண்டப்படுகிறது (Fired).

எந்த கூடுதல் அம்சங்களை (add-on functionality) கையாள வேண்டுமானால், இதை நீங்கள் பயன்படுத்தலாம் (Hook into this).

### மூலம் (Source) {#source}

[`inc/class-domain-mapping.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-domain-mapping.php#L175) என்ற கோப்பில், 175-வது வரியில் இது வரையறுக்கப்பட்டுள்ளது.
