# Action: wu_{$this->id}_scripts

ஸ்கிரிப்ட்களை இணைக்கும் (enqueue) ஹூக்கை இது தூண்டுகிறது.

இந்த உறுப்பு (element) அதன் `register_scripts` முறையை (method) இணைக்க இதைப் பயன்படுத்துகிறது.
