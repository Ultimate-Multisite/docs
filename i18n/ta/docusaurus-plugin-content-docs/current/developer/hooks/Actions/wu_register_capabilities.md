# Action: wu_register_capabilities

Addon-கள், ஒருங்கிணைப்புகளுக்கு (integrations) திறமை தொகுதிகளை (capability modules) இணைக்க இது உதவுகிறது.

| பெயர் (Name) | வகை (Type) | விளக்கம் (Description) |
|------|------|-------------|
| $registry | `\Integration_Registry` | பதிவேடு நிகழ்வு (The registry instance). |

- 2.5.0
