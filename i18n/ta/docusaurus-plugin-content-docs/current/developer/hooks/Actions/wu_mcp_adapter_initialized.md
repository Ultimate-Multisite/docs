# Action: wu_mcp_adapter_initialized

MCP அடாப்டர் துவக்கப்பட்ட பிறகு இது செயல்படும்.

இதன் மூலம் மற்ற 플러ഗിன்கள் மற்றும் தீம்கள் தங்கள் சொந்த திறன்களைப் பதிவு செய்ய முடியும்.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP அடாப்டர் நிகழ்வு (instance). |

### Since {#since}

- 2.5.0

[`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L98) என்ற கோப்பில், 98 வரியில் வரையறுக்கப்பட்டுள்ளது.
