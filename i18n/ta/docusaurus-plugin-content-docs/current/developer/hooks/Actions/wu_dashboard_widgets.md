# Action: wu_dashboard_widgets

Plugin உருவாக்குநர்கள் (developers) நெட்வொர்க் டாஷ்போர்டு பேனலில் (Network Dashboard Panel) விட்ஜெட்களை (widgets) சேர்க்க இந்த செயல்பாடு அனுமதிக்கிறது.

## அளவுருக்கள் (Parameters)

| Name | Type | விளக்கம் (Description) |
|------|------|-------------|
| $tab | `string` | தற்போதைய Tab. |
| $screen | `\WP_Screen` | ஸ்கிரீன் ஆப்ஜெக்ட். |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite நிர்வாகப் பக்கத்தின் (admin page) நிகழ்வு (instance). |

### எப்போது இருந்து (Since)

- 2.0.0
### மூலம் (Source)

[`inc/admin-pages/class-dashboard-admin-page.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/admin-pages/class-dashboard-admin-page.php#L225) என்ற இடத்தில், 225 வரியில் வரையறுக்கப்பட்டுள்ளது.
