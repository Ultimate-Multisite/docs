# Action: wu_register_rest_routes

கூடுதல் வழித்தடங்களை (routes) பதிவு செய்ய அனுமதிக்கிறது.

இது எங்கள் `/register` எண்ட்பாயிண்ட்டால் பயன்படுத்தப்படுகிறது.

## அளவுருக்கள் {#parameters}

| பெயர் | வகை | விளக்கம் |
|------|------|-------------|
| $api_instance | `self` | தற்போதைய API இன்ஸ்டன்ஸ். |
