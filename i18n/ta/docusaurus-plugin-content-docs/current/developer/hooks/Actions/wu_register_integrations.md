# Action: wu_register_integrations

இன்டெகிரேஷன்கள் (integrations) தங்களை பதிவு செய்ய அனுமதிக்க இந்த செயல்பாடு (action) பயன்படுத்தப்படுகிறது.

கோர் (Core) சிஸ்டம் அதன் இன்டெகிரேஷன்களை இங்கே பதிவு செய்கிறது.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | பதிவேடு நிகழ்வு (The registry instance). |

### Since

- 2.5.0
