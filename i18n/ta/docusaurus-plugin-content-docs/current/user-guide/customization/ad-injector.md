# Ultimate Multisite: Ad Injector மூலம் உங்கள் நெட்வொர்க்கில் விளம்பரங்களை வைத்தல்

Ultimate Multisite: Ad Injector என்பது ஒரு **இலவச add-on** ஆகும், இது உங்கள் plans-ன் edit page-லிருந்தே விளம்பரங்களை வைக்க தேவையான கருவிகளை வழங்குகிறது.

Ultimate Multisite: Ad Injector-ஐ நிறுவிய பிறகு, அதை செயல்படுத்த உங்கள் _Plugins_ page-க்குச் செல்லுங்கள்.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate பட்டனைக் கிளிக் செய்யுங்கள்_

add-on செயல்பட்டவுடன், plan edit screen-ல் உள்ள _Advanced Options_ meta box-ல் _Ad Settings_ என்ற புதிய option தோன்றும்.

![Plan edit screen-ல் Ad Settings](/img/config/product-edit-full.png)

_இப்போது உங்கள் ஒவ்வொரு plan-க்கும் விளம்பர அமைப்புகளை மாற்றலாம்_

அந்த tab-ல், front-end-ல் உள்ள posts மற்றும் pages-க்கு, content-க்கு முன்னும் பின்னும் விளம்பர codes-ஐ சேர்க்கலாம்.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_உங்கள் விளம்பரத்தின் code-ஐ சேர்த்த பிறகு, அது குறிப்பிட்ட plan-ன் posts மற்றும் pages-ல் காட்டப்படும்._

அந்த plan-ன் பயனர்களுக்கான Admin Pages-ன் header-லும் விளம்பரங்களை சேர்க்கலாம்.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_உங்கள் பயனரின் back-end-ல் விளம்பரங்களை காட்ட விரும்பினால் Admin Panel Ad Code option-க்குச் செல்லுங்கள்_

### விளம்பரத்திற்கு அப்பாற்பட்டு

உயர்நிலை plans-க்கு support வழங்க Ultimate Multisite: Ad Injector-ஐ பயன்படுத்தலாம்.

_Admin Panel Ads_ option-க்குச் சென்று உங்கள் support tool-க்கான HTML code-ஐ சேர்க்கவும்.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_குறிப்பிட்ட plans-க்கு customer support சேர்க்கவும்_

இவ்வாறு மக்கள் தங்கள் Admin Panel-லிருந்தே உங்களை தொடர்பு கொள்ளலாம்!
