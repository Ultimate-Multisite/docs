# PRO Theme மூலம் உங்கள் Admin Panel-ஐ தனிப்பயனாக்குதல்

வாடிக்கையாளர் வேலைகள் செய்யும்போது, WordPress-ன் admin dashboard-ல் அவர்களுக்கு தனித்துவமான அனுபவத்தை வழங்க வேண்டிய தேவை அடிக்கடி எழுகிறது. மேலும் தொழில்முறையான தோற்றமும் உணர்வும் தேவைப்பட்டால், PRO Theme உங்களுக்கு உதவியாக இருக்கும்.

PRO Theme என்பது உங்கள் வாடிக்கையாளர்களின் Dashboard-ஐ தனிப்பயனாக்கும்போது உதவுவதற்காக உருவாக்கப்பட்ட ஒரு admin theme ஆகும். இது Ultimate Multisite-ன் premium plugins core-ன் ஒரு பகுதியாகவும் உள்ளது.

இதை நிறுவிய பிறகு, உங்கள் _Plugins_ பக்கத்திற்குச் சென்று உங்கள் Network-ல் செயல்படுத்த கிளிக் செய்யுங்கள்.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_உங்கள் Plugins பக்கத்தில் network-wide ஆக செயல்படுத்துங்கள்_

_**குறிப்பு:**_ _இதை நிறுவி மட்டும் வைத்திருந்தால், உங்கள் வாடிக்கையாளர்களுக்கு இந்த theme-ஐ அந்த பயனருக்கு மட்டும் சேர்க்க ஒரு விருப்பமாக காண்பிக்கும்._

இப்போது, உங்கள் admin interface முழுமையாக மறுவடிவமைப்பு செய்யப்பட்டுள்ளது! இடது பக்கத்தில் PRO Theme என்ற புதிய menu-வும் உள்ளது.
