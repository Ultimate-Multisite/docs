# Ultimate Multisite நிறுவுதல்

:::note
இந்த டுடோரியல் உங்களிடம் ஏற்கனவே WordPress Multisite நிறுவப்பட்டு கட்டமைக்கப்பட்டிருப்பதாக கருதுகிறது. எப்படி செய்வது என்று அறிய, WP Beginner-ன் [இந்த டுடோரியலைப்](https://www.wpbeginner.com/glossary/multisite/) பாருங்கள்.
:::

## Plugin நிறுவுதல்

Ultimate Multisite [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/)-ல் இலவசமாகக் கிடைக்கிறது.

உங்கள் **Network Admin Dashboard**-ல், **Plugins → Add New Plugin** என்பதற்குச் செல்லுங்கள்.

![Network Admin Add New Plugin பக்கம்](/img/installation/add-new-plugin.png)

**"Ultimate Multisite"** (சரியான முடிவுக்கு மேற்கோள் குறிகளுடன்) என்று தேடுங்கள், அது முதல் முடிவாகத் தோன்றும். **Install Now** என்பதைக் கிளிக் செய்யுங்கள்.

![Ultimate Multisite காட்டும் தேடல் முடிவுகள்](/img/installation/search-ultimate-multisite.png)

நிறுவப்பட்டதும், உங்கள் முழு நெட்வொர்க்கிலும் plugin-ஐ செயல்படுத்த **Network Activate** என்பதைக் கிளிக் செய்யுங்கள்.

![Plugin நிறுவப்பட்டது, Network Activate பொத்தானுடன்](/img/installation/plugin-installed.png)

செயல்படுத்தியதும், நீங்கள் தானாகவே Setup Wizard-க்கு திருப்பிவிடப்படுவீர்கள்.

![Plugin செயல்படுத்தப்பட்டு wizard-க்கு திருப்பிவிடப்பட்டது](/img/installation/plugin-activated.png)

## Setup Wizard

Setup Wizard சுமார் 10 நிமிடங்களில் Ultimate Multisite-ஐ கட்டமைக்க உங்களுக்கு வழிகாட்டும்.

### வரவேற்பு

தொடங்க **Get Started** என்பதைக் கிளிக் செய்யுங்கள்.

![Setup Wizard வரவேற்புத் திரை](/img/installation/wizard-welcome.png)

### நிறுவலுக்கு முந்தைய சோதனைகள்

இந்த படி உங்கள் சிஸ்டம் தகவல் மற்றும் WordPress நிறுவலை சோதித்து, Ultimate Multisite-ன் தேவைகளைப் பூர்த்தி செய்கிறதா என்று உறுதி செய்கிறது. எல்லாம் சரியாக இருந்தால், **Go to the Next Step** என்பதைக் கிளிக் செய்யுங்கள்.

![சிஸ்டம் தேவைகளைக் காட்டும் நிறுவலுக்கு முந்தைய சோதனைகள்](/img/installation/wizard-pre-install-checks.png)

### நிறுவல்

Installer தேவையான database tables-ஐ உருவாக்கி, Ultimate Multisite செயல்பட தேவையான `sunrise.php` file-ஐ நிறுவும். தொடர **Install** என்பதைக் கிளிக் செய்யுங்கள்.

![Database tables மற்றும் sunrise.php காட்டும் நிறுவல் படி](/img/installation/wizard-installation.png)

### உங்கள் நிறுவனம்

உங்கள் நிறுவனத் தகவலை நிரப்பி, உங்கள் இயல்பு நாணயத்தை அமைக்கவும். இந்தத் தகவல் உங்கள் WaaS platform முழுவதும் பயன்படுத்தப்படும். முடிந்ததும் **Continue** என்பதைக் கிளிக் செய்யுங்கள்.

![Your Company கட்டமைப்பு படி](/img/installation/wizard-your-company.png)

### இயல்பு உள்ளடக்கம்

இந்த படி முன்வரையறுக்கப்பட்ட templates, products மற்றும் பிற தொடக்க உள்ளடக்கங்களை நிறுவ அனுமதிக்கிறது. Ultimate Multisite-ன் அம்சங்களை அறிந்துகொள்ள இது ஒரு சிறந்த வழி. இயல்பு உள்ளடக்கத்தைச் சேர்க்க **Install** என்பதைக் கிளிக் செய்யுங்கள், அல்லது புதிதாகத் தொடங்க விரும்பினால் இந்த படியைத் தவிர்க்கவும்.

![இயல்பு உள்ளடக்க நிறுவல் படி](/img/installation/wizard-default-content.png)

### பரிந்துரைக்கப்பட்ட Plugins

விருப்பமாக பரிந்துரைக்கப்பட்ட துணை plugins-ஐ நிறுவவும். அவற்றைச் சேர்க்க **Install** என்பதைக் கிளிக் செய்யுங்கள் அல்லது தொடர தவிர்க்கவும்.

![பரிந்துரைக்கப்பட்ட plugins படி](/img/installation/wizard-recommended-plugins.png)

### தயார்!

அவ்வளவுதான்! உங்கள் Ultimate Multisite நிறுவல் முடிந்தது. இப்போது **Network Admin Dashboard**-லிருந்து உங்கள் Website as a Service platform-ஐ உருவாக்கத் தொடங்கலாம்.

![Setup முடிந்தது - Ready திரை](/img/installation/wizard-ready.png)

![Ultimate Multisite செயலில் உள்ள Network Admin Dashboard](/img/installation/network-dashboard.png)

சென்று மகிழுங்கள்!
