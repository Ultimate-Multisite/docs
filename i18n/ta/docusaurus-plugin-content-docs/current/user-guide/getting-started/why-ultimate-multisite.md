# ஏன் Ultimate Multisite

இந்தக் கட்டுரையை நீங்கள் படிக்கிறீர்கள் என்றால், உங்கள் வணிக யோசனையை நனவாக்க எங்கள் plugin உதவும் என்று கேள்விப்பட்டிருக்கலாம். இந்தக் கட்டுரையின் முடிவில் உங்களுக்குத் தேவையான பதில்கள் கிடைக்கும் என்று நம்புகிறோம்.

## Ultimate Multisite என்றால் என்ன?

Ultimate Multisite என்பது ஒரு WordPress Multisite plugin ஆகும், இது உங்கள் வாடிக்கையாளர்களுக்கு WaaS (Website as a Service - இணையதளத்தை ஒரு சேவையாக) வழங்க உதவுகிறது. எங்கள் plugin மூலம், பல்வேறு சந்தா நிலைகளை உருவாக்கி, உங்கள் Multisite நெட்வொர்க்கில் ஒரு தளத்தை ஹோஸ்ட் செய்வதற்கு வாடிக்கையாளர்களிடம் தொடர்ச்சியான கட்டணம் பெறலாம்.

Ultimate Multisite பல்வேறு payment gateway-கள் (WooCommerce அல்லது Stripe போன்றவை) மூலம் பணம் பெற உதவுகிறது. உங்கள் நெட்வொர்க்கில் புதிய பயனர் பதிவு செய்யும்போது site template-கள் உருவாக்கி உங்கள் வேலையை எளிதாக்கலாம், domain-களை map செய்யலாம், மேலும் உங்கள் வாடிக்கையாளர்களுக்கு தனிப்பயனாக்கப்பட்ட அனுபவத்தை வழங்க பதிவு படிவங்களை விருப்பப்படி மாற்றலாம்.

இது ஆரம்பம் மட்டுமே - add-on-கள் மற்றும் webhook-கள் மூலம் இதன் செயல்பாடுகளை மேலும் விரிவுபடுத்தலாம்.

## பிற அம்சங்கள்

Ultimate Multisite இவற்றையும் வழங்குகிறது:

  * Managed WordPress hosting வழங்குநர்களுடன் Domain Mapping ஒருங்கிணைப்பு: புதிய domain map செய்யப்படும்போது, Ultimate Multisite தானாகவே WP Engine, Cloudways, cPanel, RunCloud, Closte மற்றும் பிற admin panel-களில் map செய்யப்பட்ட domain-களைச் சேர்க்கிறது, எனவே நீங்கள் கைமுறையாகச் செய்ய வேண்டியதில்லை. சில சமயங்களில், உங்களுக்காக SSL certificate-ஐயும் தானாகப் பெற்றுத் தருவோம்.

  * Webhooks: Ultimate Multisite-ல் சக்திவாய்ந்த Webhook அமைப்பு உள்ளது, இது உங்கள் நெட்வொர்க்கை பல்வேறு சேவைகளுடன் ஒருங்கிணைக்க உதவுகிறது. Zapier-உடன் நேரடி ஒருங்கிணைப்பும் வழங்குகிறோம், இது அவர்களின் platform-ல் கிடைக்கும் 1000+ app-கள் மற்றும் சேவைகளுக்கு உங்கள் நெட்வொர்க்கைத் திறக்கிறது.

  * உங்கள் விருப்பமான page builder பயன்படுத்தி வாடிக்கையாளர்களுக்கு உதவ சக்திவாய்ந்த Admin Pages: Ultimate Multisite-ல் Admin Pages Creator என்ற எளிமையான add-on உள்ளது, இது Elementor, Beaver Builder மற்றும் Brizy template-களைப் பயன்படுத்தி உங்கள் வாடிக்கையாளர்களின் dashboard-ல் முழுமையாக தனிப்பயனாக்கப்பட்ட Admin Page-களைச் சேர்க்க உதவுகிறது.

  * உங்கள் வாடிக்கையாளர்களின் dashboard அனுபவத்தை தனிப்பயனாக்க இரண்டு வெவ்வேறு Admin Theme-கள்: Material WP மற்றும் PRO பாருங்கள்.

மேலும் மேம்பட்ட விஷயங்கள்:

  * Template Overrides ஆதரவு: Ultimate Multisite மூலம், core plugin கோப்புகளை மாற்றுவது பற்றி கவலைப்படாமல், signup flow-க்கான default HTML template-களை உங்கள் சொந்த பதிப்புகளால் மாற்றலாம்.

  * Search and Replace API: {{site_name}} போன்ற placeholder-களுடன் site template-களை குறிப்பிட்டு, Search and Replace API பயன்படுத்தி, பதிவு செய்த உடனேயே உங்கள் வாடிக்கையாளர் வழங்கிய தகவல்களால் அந்த placeholder-களை தானாக நிரப்பலாம்.

## பயன்பாட்டு நிலைகள்

இப்போது Ultimate Multisite-ன் சாத்தியமான பயன்பாட்டு நிலைகளைப் பார்ப்போம்.

### நிலை 1: ஒரு Agency

பொதுவாக ஒரு agency-யின் முக்கிய திறமைகள் இணையதள வடிவமைப்பில் உள்ளன, hosting அல்லது marketing போன்றவை கூடுதல் சேவைகளாக பட்டியலிடப்படுகின்றன.

Agency-களுக்கு, Ultimate Multisite ஒரே platform-ல் பல இணையதளங்களை host செய்து நிர்வகிக்கும் திறனில் நம்பமுடியாத மதிப்பை வழங்குகிறது. மேலும், GeneratePress, Astra, OceanWP போன்ற குறிப்பிட்ட theme-களில் தங்கள் வடிவமைப்புகளை தரப்படுத்தும் agency-கள், ஒவ்வொரு புதிய தளத்திற்கும் இந்த theme-களை தானாக செயல்படுத்த Ultimate Multisite-ன் திறன்களைப் பயன்படுத்தலாம்.

Ultimate Multisite-ன் அமைப்புகள் மூலம் ஒவ்வொரு தயாரிப்புக்கும் plugin-கள் மற்றும் theme-கள் மீது கடுமையான கட்டுப்பாடு பராமரிக்கப்படுகிறது - plugin-கள் மற்றும் theme-களை கிடைக்கச் செய்யலாம் அல்லது மறைக்கலாம், புதிய தளம் உருவாக்கும்போது அவற்றின் செயல்படுத்தல் நிலையையும் அமைக்கலாம்.

Agency-கள் Ultimate Multisite மூலம் நிம்மதியாக இருக்கலாம், அவர்கள் சிறப்பாகச் செய்வதை - அற்புதமான இணையதளங்களை வடிவமைப்பதை - தொடரலாம்.

### நிலை 2: Niche வழங்குநர்

"ஒரு விஷயத்தைச் செய்யுங்கள், அதை நன்றாகச் செய்யுங்கள்" என்று ஒரு பழைய சொல் உண்டு. பல நிபுணர்களுக்கு, இது ஒரு முக்கிய யோசனையைச் சுற்றி ஒரு தயாரிப்பு அல்லது சேவையை உருவாக்குவதைக் குறிக்கிறது.

குறிப்பிட்ட niche தளங்களின் நெட்வொர்க்கை உருவாக்குவதில் கவனம் செலுத்துகிறீர்கள் என்றால், பொதுவான framework மற்றும் platform அடிப்படையில் சேவைகளை வழங்க விரும்புவீர்கள். அப்போதுதான் site template-கள் உதவும்!

Ultimate Multisite-ன் புதுமையான அம்சங்களில் ஒன்று template site - இதில் theme நிறுவப்பட்டு செயல்படுத்தப்பட்டிருக்கும், தேவையான plugin-கள் நிறுவப்பட்டு செயல்படுத்தப்பட்டிருக்கும், மாதிரி post-கள் அல்லது page-கள் உருவாக்கப்பட்டிருக்கும். ஒரு வாடிக்கையாளர் template அடிப்படையில் புதிய தளத்தை உருவாக்கும்போது, template-ன் உள்ளடக்கங்களும் அமைப்புகளும் புதிதாக உருவாக்கப்பட்ட தளத்திற்கு நகலெடுக்கப்படுகின்றன.

Niche தளங்கள் மற்றும் சேவைகளை வழங்குபவர்களுக்கு, தனிப்பயன் plugin-கள் மற்றும் வடிவமைப்புடன் தயாராக உள்ள தளத்தை உடனடியாக உருவாக்கும் திறனில் இது ஒப்பற்ற நன்மையை வழங்குகிறது. சேவையை முடிக்க வாடிக்கையாளர் குறைந்தபட்ச தகவல்களை மட்டுமே வழங்க வேண்டும்.

### நிலை 3: WordPress Web Hosting

WordPress தளங்களை host செய்ய எண்ணற்ற வழிகள் உள்ளன, ஆனால் WordPress முன்பே நிறுவப்பட்ட நிலையில் வாடிக்கையாளருக்கு web space வழங்குவது மட்டும் எளிமையானது அல்ல. ஏனெனில் அர்த்தமுள்ள சேவையை வழங்க பல முடிவுகளும் பரிசீலனைகளும் ஒன்றிணைய வேண்டும்.

WordPress தளங்களை host செய்வதற்கான விரிவான turnkey தீர்வை வழங்குவதில் Ultimate Multisite சிறந்து விளங்குகிறது. சந்தா சேவைகள், கட்டண வசூல், checkout படிவங்கள், தள்ளுபடி voucher-கள் மற்றும் வாடிக்கையாளர் தொடர்புகளை வழங்குவதற்கான முக்கிய வழிமுறைகள் தீர்வில் சேர்க்கப்பட்டுள்ளன.

WordPress Multisite-ஐ சரியாக நிறுவ, configure செய்ய மற்றும் பராமரிக்க தேவையான பெரும்பாலான ஒருங்கிணைந்த வேலைகளை Ultimate Multisite எளிதாக்குகிறது, எனவே நெட்வொர்க் நிர்வாகிகள் தங்கள் சேவை அல்லது niche தொடர்பான அம்சங்களை மட்டுமே கருத்தில் கொள்ள வேண்டும் - தயாரிப்பு நிலைகள், விலை நிர்ணயம் மற்றும் சேவை சலுகைகள் போன்றவை.

Ultimate Multisite-உடன் ஒருங்கிணைக்க விரும்பும் developer-களுக்கு, விரிவான RESTful API மற்றும் நிகழ்வு அறிவிப்புகளுக்கான Webhook-களும் வழங்கப்படுகின்றன.

பல வெளிப்புற plugin-கள் மற்றும் license-களை சார்ந்திராமல், Ultimate Multisite Wix, Squarespace, [WordPress.com](https://WordPress.com) மற்றும் பிறவற்றுக்கு ஒப்பான அம்சங்கள் நிறைந்த தீர்வை வழங்குகிறது.
