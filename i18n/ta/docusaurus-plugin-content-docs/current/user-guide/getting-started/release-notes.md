# வெளியீட்டு குறிப்புகள்

## பதிப்பு 2.13.0 — 2026-06-05 அன்று வெளியிடப்பட்டது

- புதியது: வாடிக்கையாளர் Account, checkout, பில்லிங், தளம், இன்வாய்ஸ், வார்ப்புரு மாற்றம், மற்றும் domain mapping ஓட்டங்களுக்கு sovereign-tenant ஆதரவு சேர்க்கப்பட்டது. இதனால் tenant networks நிர்வகிக்கப்படும் செயல்களுக்காக வாடிக்கையாளர்களை முதன்மை தளத்துக்கு மீண்டும் அனுப்ப முடியும்.
- புதியது: recurring membership-களுக்கு renewal-credential சரிபார்ப்புகள் சேர்க்கப்பட்டன. சேமிக்கப்பட்ட billing agreement, subscription, அல்லது vault token இல்லையெனில் gateways auto-renewal-ஐ முடக்க முடியும்.
- புதியது: நிலுவையில் உள்ள தள உருவாக்கத்துக்கு HMAC-சரிபார்க்கப்பட்ட loopback publishing சேர்க்கப்பட்டது. பின்னணி பணிகள் தாமதமாகும் hosts-இல் checkout-இலிருந்து தள provision செய்வது மேலும் நம்பகமாகும்.
- புதியது: SSO URL-கள், checkout-form அடிப்படை domains, மற்றும் தானியங்கி domain-record உருவாக்கத்துக்கான developer extension points சேர்க்கப்பட்டன.
- திருத்தம்: mapped domains, anonymous broker வருகைகள், logout, மற்றும் cross-plugin dependency முரண்பாடுகள் முழுவதும் SSO மேலும் நம்பகமாக உள்ளது.
- திருத்தம்: நிலுவையில் உள்ள தள உருவாக்கம் இப்போது பழைய publish flags-இலிருந்து மீள்கிறது. வாடிக்கையாளர்கள் தள உருவாக்கத் திரையில் சிக்கி நிற்பதையும் தவிர்க்கிறது.
- திருத்தம்: பகிரப்பட்ட checkout-form அடிப்படை domains-க்கு domain records இனி உருவாக்கப்படாது. எந்த integration-மும் செயலில் இல்லையெனில் பயன்படுத்தப்படாத host-provider பின்னணி பணிகள் தவிர்க்கப்படும்.
- திருத்தம்: Checkout, பில்லிங் முகவரி, கடவுச்சொல் மீட்டமைப்பு, மின்னஞ்சல் சரிபார்ப்பு, வார்ப்புரு மாற்றம், tours, மற்றும் வாடிக்கையாளர் Dashboard விளிம்பு நிலைகள் இனி சாதாரண வாடிக்கையாளர் ஓட்டங்களைத் தடுக்காது.
- திருத்தம்: பெறுநர் பட்டியல்கள் அல்லது mail transports தோல்வியுற்றால் SMTP/plugin கடுமையான பிழைகளைத் தவிர்க்கும்போது, broadcast மின்னஞ்சல்கள் பெறுநர்களை இப்போது தனிப்பட்டதாக வைத்திருக்கும்.
- திருத்தம்: Membership renewal-கள், காலாவதி காட்சி, மற்றும் கட்டண வசூல் விளிம்பு நிலைகள் இப்போது உடனடி காலாவதி, crashes, அல்லது தவறவிடப்பட்ட கட்டாய கட்டணங்களைத் தவிர்க்கின்றன.
- மேம்படுத்தப்பட்டது: WordPress இணக்கத்தன்மை 7.0 வரை சோதிக்கப்பட்டது, production Vue assets npm sources-இலிருந்து மீண்டும் கட்டப்பட்டன, மேலும் Cypress end-to-end coverage இப்போது அதிக checkout, setup, SSO, மற்றும் gateway ஓட்டங்களைச் சோதிக்கிறது.

## பதிப்பு 2.12.0 — 2026-05-15 அன்று வெளியிடப்பட்டது

- புதியது: domain mapping integration உடன் ஆதரிக்கப்படும் host provider ஆக Hostinger (hPanel) சேர்க்கப்பட்டது
- புதியது: தள ஏற்றுமதியாளர் இப்போது streamlined network-wide தள மீட்டமைப்புக்கான network import bundles-ஐ கையாளுகிறது
- திருத்தம்: பெறுநர் முகவரிகள் வெளிப்படாமல் இருக்க BCC broadcast மின்னஞ்சல்கள் இப்போது undisclosed-recipients header-ஐ பயன்படுத்துகின்றன
- திருத்தம்: தேதி அல்லாத மதிப்புடன் சேமிக்கும் போது membership காலாவதி தேதி இனி சேதமடையாது
- திருத்தம்: Stripe membership புதுப்பிப்புகள் இப்போது deprecated deleteDiscount API-ஐ அழைக்காமல் தள்ளுபடிகளை சரியாக நீக்குகின்றன
- திருத்தம்: domain-mapped தளங்களில் SSO redirects இப்போது முடிவில்லா redirect loops-ஐத் தடுக்க வரையறுக்கப்பட்டுள்ளன
- திருத்தம்: Setup wizard படத் தேர்வியின் தேர்வு இப்போது அடிப்படை data model-ஐ சரியாகப் புதுப்பிக்கிறது
- திருத்தம்: தள ஏற்றுமதியாளர் CLI இப்போது சரியான default network தளத் தேர்வை பாதுகாக்கிறது
- மேம்படுத்தப்பட்டது: plugin தொகுப்பிலிருந்து bundled wp-cli அகற்றப்பட்டது, இதனால் plugin அளவு குறைகிறது
