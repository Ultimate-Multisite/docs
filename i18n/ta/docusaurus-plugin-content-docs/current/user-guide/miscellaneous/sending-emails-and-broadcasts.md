# மின்னஞ்சல்கள் மற்றும் Broadcasts அனுப்புதல் (v2)

_**முக்கிய குறிப்பு: இந்தக் கட்டுரை Ultimate Multisite பதிப்பு 2.x ஐ குறிக்கிறது.**_

Ultimate Multisite ஒரு அம்சத்துடன் வருகிறது, இது குறிப்பிட்ட பயனர் அல்லது பயனர் குழுவிற்கு மின்னஞ்சல் அனுப்புவதன் மூலமும், அவர்களின் admin dashboard-ல் அறிவிப்புகளை broadcast செய்வதன் மூலமும் உங்கள் வாடிக்கையாளர்களுடன் தொடர்பு கொள்ள உதவுகிறது.

## Broadcasts மூலம் உங்கள் வாடிக்கையாளர்களின் dashboard-ல் admin notices சேர்த்தல்

Ultimate Multisite broadcast அம்சத்தைப் பயன்படுத்தி, உங்கள் பயனரின் subsite admin dashboard-ல் **admin notices** சேர்க்கலாம்.

சிஸ்டம் பராமரிப்பு போன்ற அறிவிப்புகள் செய்ய வேண்டியிருந்தாலோ அல்லது உங்கள் தற்போதைய பயனர்களுக்கு புதிய தயாரிப்புகள் அல்லது சேவைகளை வழங்க வேண்டியிருந்தாலோ இது மிகவும் உதவியாக இருக்கும். உங்கள் பயனரின் dashboard-ல் admin notice இப்படித் தோன்றும்.

![வாடிக்கையாளர் dashboard-ல் காண்பிக்கப்படும் Admin notice broadcast](/img/admin/broadcasts-list.png)

Admin notice தொடங்க, உங்கள் network admin dashboard-க்குச் சென்று, **Ultimate Multisite** மெனுவின் கீழ், **Broadcasts** விருப்பத்தைக் காணலாம்.

![Ultimate Multisite admin-ல் Broadcasts மெனு](/img/admin/broadcasts-list.png)

இந்தப் பக்கத்தில், மேலே உள்ள **Add Broadcast** பொத்தானைக் கிளிக் செய்யவும்.

இது Add broadcast modal சாளரத்தைக் கொண்டுவரும், அங்கு எந்த வகையான broadcast அனுப்ப விரும்புகிறீர்கள் என்பதைத் தேர்வு செய்யலாம்.

**Message** என்பதைத் தேர்ந்தெடுத்து, **Next Step** பொத்தானைக் கிளிக் செய்யவும்.

![Message வகை தேர்ந்தெடுக்கப்பட்ட Add broadcast modal](/img/admin/broadcasts-list.png)

அடுத்த சாளரம் **Target customer** அல்லது **Target product** கேட்கும். ஒன்றுக்கு மேற்பட்ட பயனர்கள் அல்லது ஒன்றுக்கு மேற்பட்ட தயாரிப்புகளைத் தேர்ந்தெடுக்கலாம் என்பதை நினைவில் கொள்ளவும்.

பயனர் கணக்கு அல்லது தயாரிப்பைத் தேட, புலத்தில் keyword தட்டச்சு செய்ய ஆரம்பிக்க வேண்டும்.

**Message type** புலத்தின் கீழ், notice-ன் நிறத்தைத் தேர்ந்தெடுக்கலாம். இது உங்கள் செய்தியின் அவசரத்தை வலியுறுத்தும்.

பின்னர் **Next Step** கிளிக் செய்யலாம்.

![Broadcast-க்கான Target customer மற்றும் product தேர்வு](/img/admin/broadcasts-list.png)

அடுத்த சாளரத்தில், பயனர்களுக்கு broadcast செய்ய விரும்பும் subject மற்றும் content/message உள்ளிட்டு உங்கள் செய்தியை எழுத ஆரம்பிக்கலாம்.

![Broadcast message subject மற்றும் content editor](/img/admin/broadcasts-list.png)

உங்கள் செய்தியை உருவாக்கிய பிறகு, **Send** பொத்தானை அழுத்தலாம்.

அவ்வளவுதான். Admin notice உடனடியாக உங்கள் பயனரின் dashboard-ல் தோன்றும்.

## உங்கள் வாடிக்கையாளர்களுக்கு மின்னஞ்சல்கள் அனுப்புதல்

Ultimate Multisite broadcast அம்சத்தைப் பயன்படுத்தி, உங்கள் பயனர்களுக்கு மின்னஞ்சல் அனுப்பலாம். குறிப்பிட்ட பயனர்களுக்கு மட்டும் மின்னஞ்சல் அனுப்ப அல்லது அவர்கள் subscribe செய்துள்ள product அல்லது plan அடிப்படையில் குறிப்பிட்ட பயனர் குழுவை target செய்ய விருப்பம் உள்ளது.

Email broadcast தொடங்க, உங்கள் network admin dashboard-க்குச் சென்று, Ultimate Multisite மெனுவின் கீழ், Broadcast விருப்பத்தைக் காணலாம்.

இது Add broadcast modal சாளரத்தைக் கொண்டுவரும், அங்கு எந்த வகையான broadcast அனுப்ப விரும்புகிறீர்கள் என்பதைத் தேர்வு செய்யலாம். **Email** என்பதைத் தேர்ந்தெடுத்து, **Next Step** பொத்தானைக் கிளிக் செய்யவும்.

![Email வகை தேர்ந்தெடுக்கப்பட்ட Add broadcast modal](/img/admin/broadcasts-list.png)

அடுத்த சாளரம் **Target customer** அல்லது **Target product** கேட்கும். ஒன்றுக்கு மேற்பட்ட பயனர்கள் அல்லது ஒன்றுக்கு மேற்பட்ட தயாரிப்புகளைத் தேர்ந்தெடுக்கலாம் என்பதை நினைவில் கொள்ளவும்.

உங்கள் target audience தேர்ந்தெடுக்கப்பட்டவுடன், **Next Step** கிளிக் செய்யலாம்.

Broadcast அம்சத்தைப் பயன்படுத்தி உங்கள் இறுதிப் பயனர்களுக்கு மின்னஞ்சல் அனுப்புவது இவ்வளவு எளிது.

## System emails

Ultimate Multisite-ல் System emails என்பது registration, payment, domain mapping போன்ற குறிப்பிட்ட செயல்களுக்குப் பிறகு system தானாக அனுப்பும் **automatic notifications** ஆகும். இந்த மின்னஞ்சல்களை Ultimate Multisite settings-ல் இருந்து திருத்தலாம் அல்லது மாற்றலாம். மற்றொரு Ultimate Multisite installation-ல் இருந்து தற்போதைய settings-ஐ reset மற்றும் import செய்ய அனுமதிக்கும் அம்சமும் உள்ளது.

### Resetting & Importing

புதிய Ultimate Multisite பதிப்புகள், add-ons ஆகியவை அவ்வப்போது புதிய மின்னஞ்சல்களைப் பதிவு செய்யும்.

முரண்பாடுகள் மற்றும் பிற சிக்கல்களைத் தவிர்க்க, **புதிய email templates-ஐ உங்கள் install-ல் System Emails-ஆக தானாகச் சேர்க்க மாட்டோம்**, ஒரு குறிப்பிட்ட அம்சம் சரியாக செயல்பட அவை அவசியமானால் தவிர.

எனினும், super admins மற்றும் agents importer tool மூலம் புதிதாகப் பதிவு செய்யப்பட்ட மின்னஞ்சல்களை import செய்யலாம். இந்த செயல்முறை புதிய email template-ன் content மற்றும் configuration-உடன் புதிய system email உருவாக்கும், இது super admin தேவையான மாற்றங்களைச் செய்ய அல்லது அப்படியே வைக்க அனுமதிக்கும்.

#### System emails import செய்வது எப்படி

உங்கள் Ultimate Multisite Settings பக்கத்திற்குச் சென்று **Emails** tab-க்குச் செல்லவும்.

![Ultimate Multisite settings-ல் Emails tab](/img/config/settings-emails.png)

பின்னர், sidebar-ல், **Customize System Emails** பொத்தானைக் கிளிக் செய்யவும்.

![Sidebar-ல் Customize System Emails பொத்தான்](/img/config/settings-emails.png)

System Emails பக்கத்தில், மேலே **Reset & Import** action பொத்தான் இருக்கும். அந்த பொத்தானைக் கிளிக் செய்தால் import மற்றும் reset modal சாளரம் திறக்கும்.

![System Emails பக்கத்தில் Reset and Import action பொத்தான்](/img/config/settings-emails.png)

பின்னர், எந்த system emails import செய்யக் கிடைக்கின்றன என்பதைப் பார்க்க Import Emails விருப்பங்களை toggle செய்யலாம்.

![கிடைக்கும் system emails காட்டும் Import Emails விருப்பங்கள்](/img/config/settings-emails.png)

#### System Emails Reset செய்தல்

சில சமயங்களில், ஒரு குறிப்பிட்ட email template-ல் நீங்கள் செய்த மாற்றங்கள் உங்களுக்கு வேலை செய்யவில்லை என்பதை உணர்வீர்கள், அதை **default state**-க்கு reset செய்ய விரும்புவீர்கள்.

அப்படிப்பட்ட சந்தர்ப்பங்களில், இரண்டு விருப்பங்கள் உள்ளன: system email-ஐ வெறுமனே delete செய்து மீண்டும் import செய்யலாம் (மேலே உள்ள வழிமுறைகளைப் பயன்படுத்தி) - இது send metrics மற்றும் பிற விஷயங்களை அழிக்கும், எனவே இந்த முறை குறைவாகவே விரும்பப்படுகிறது.

அல்லது **Reset & Import tool**-ஐப் பயன்படுத்தி அந்த email template-ஐ reset செய்யலாம்.

Email template-ஐ reset செய்ய, Reset & Import tool-ஐ அடையும் வரை மேலே உள்ள படிகளைப் பின்பற்றலாம், பின்னர், **Reset** விருப்பத்தை toggle செய்து default content-க்கு reset செய்ய விரும்பும் மின்னஞ்சல்களைத் தேர்ந்தெடுக்கலாம்.

![Email templates-ஐ defaults-க்கு restore செய்ய Reset விருப்பம்](/img/config/settings-emails.png)
