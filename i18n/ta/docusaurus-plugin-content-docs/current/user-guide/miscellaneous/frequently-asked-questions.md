# அடிக்கடி கேட்கப்படும் கேள்விகள்

ஆதரவு கோரிக்கைகளின் போது அடிக்கடி எழும் சில **அடிக்கடி கேட்கப்படும் கேள்விகள்** இங்கே உள்ளன.

  * **Ultimate Multisite என்றால் என்ன?**

Ultimate Multisite என்பது WordPress multisite நிறுவல்களுக்காக உருவாக்கப்பட்ட ஒரு WordPress plugin ஆகும், இது உங்கள் WordPress நிறுவலை [WordPress.com](https://WordPress.com) போன்ற பிரீமியம் தளங்களின் வலையமைப்பாக மாற்றுகிறது – இது வாடிக்கையாளர்கள் மாதாந்திர, காலாண்டு அல்லது வருடாந்திர கட்டணங்கள் மூலம் தளங்களை உருவாக்க அனுமதிக்கிறது (நீங்கள் இலவச திட்டங்களையும் உருவாக்கலாம்).

  * **Ultimate Multisite ஆனது Multisite இல்லாமல் வேலை செய்யுமா?**

இல்லை. Ultimate Multisite க்கு WordPress multisite, subdirectory அல்லது subdomain தேவை.

  * **Ultimate Multisite க்கான தேவைகள் என்ன?**

Ultimate Multisite க்கு தேவையானவை:

**WordPress பதிப்பு** : v5.3+ (பரிந்துரை: கடைசி நிலையான பதிப்பு)  
**PHP பதிப்பு** : 7.4.x (> 8.0 ஆதரவு விரைவில் வரும்)  
**MySQL பதிப்பு** : v5+ (பரிந்துரை: பதிப்பு 5.6, உங்கள் hosting வழங்குநரிடம் 8.0 கிடைக்கவில்லை என்றால்)

_Multisite (subdomain அல்லது subdirectory) செயல்படுத்தப்பட வேண்டும்._ [WordPress Multisite ஐ எவ்வாறு நிறுவுவது](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Subdirectory கொண்ட Multisite உடன் Ultimate Multisite வேலை செய்யுமா?**

ஆம். Ultimate Multisite ஆனது subdirectory நிறுவல்கள் மற்றும் subdirectory நிறுவல்கள் இரண்டையும் ஆதரிக்கிறது.

  * **Subdomain கொண்ட Multisite உடன் Ultimate Multisite வேலை செய்யுமா?**

ஆம். Ultimate Multisite ஆனது subdirectory நிறுவல்கள் மற்றும் subdomain நிறுவல்கள் இரண்டையும் ஆதரிக்கிறது.

  * **பயனர்கள் எந்த Plugins மற்றும் Themes பயன்படுத்தலாம் என்பதை கட்டுப்படுத்த Ultimate Multisite ஐ பயன்படுத்தலாமா?**

ஆம். Ultimate Multisite மூலம் உங்கள் வலையமைப்பில் ஒவ்வொரு திட்டத்திற்கும் எந்த plugins மற்றும் themes கிடைக்கும் என்பதை நீங்கள் தேர்வு செய்யலாம்.

  * **ஒரு பயனர் தனது தளத்தில் உருவாக்கக்கூடிய இடுகைகளின் எண்ணிக்கையை கட்டுப்படுத்த Ultimate Multisite ஐ பயன்படுத்தலாமா?**

ஆம். Ultimate Multisite மூலம் உங்கள் வலையமைப்பில் **கிடைக்கும் ஒவ்வொரு post type க்கும்** இடுகை வரம்புகளை நிர்ணயிக்கலாம். நிர்வாகி அழைக்கக்கூடிய பயனர்களின் எண்ணிக்கை, **disk இடத்தின் அளவு** மற்றும் பலவற்றையும் கட்டுப்படுத்தலாம்.

  * **Ultimate Multisite ஆனது Site Templates ஐ ஆதரிக்கிறதா?**

ஆம். Ultimate Multisite மூலம் நீங்கள் விரும்பும் அளவு Site Templates உருவாக்கலாம். Site Templates என்பது themes அல்ல, அவை உங்கள் வலையமைப்பில் உள்ள சாதாரண தளங்கள். நீங்கள் plugins, themes செயல்படுத்தலாம், மற்றும் அந்த Site Template ஐ நீங்கள் விரும்பும் வகையில் தனிப்பயனாக்கலாம். ஒரு பயனர் அந்த வார்ப்புருவை தேர்வு செய்தால், அவரது தளம் அந்த வார்ப்புருவின் நகலாக, அனைத்து தனிப்பயனாக்கங்களுடன் உருவாக்கப்படும்.

ஒரு குறிப்பிட்ட வார்ப்புருவை ஒரு திட்டத்திற்கு அமைக்கலாம். அவ்வாறு செய்வதன் மூலம், அந்த திட்டத்தில் பதிவு செய்யும் ஒவ்வொரு பயனரின் தளமும் அந்த Site Template இன் நகலாக உருவாக்கப்படும்.

  * **Ultimate Multisite ஆனது WooCommerce உடன் ஒருங்கிணைக்கிறதா?**

ஆம்!

உங்கள் வலையமைப்பு பயனர்களுக்கு WooCommerce கிடைக்க வேண்டும் என்றால், அவர்கள் கடைகளை அமைக்க அனுமதிக்க வேண்டும் என்றால், Ultimate Multisite ஐ பயன்படுத்தி அது சாத்தியம். ஒவ்வொரு திட்ட நிலையிலும் உங்கள் வாடிக்கையாளர்கள் உருவாக்கக்கூடிய தயாரிப்புகளின் எண்ணிக்கையை கூட கட்டுப்படுத்தலாம்.

Ultimate Multisite க்கான சந்தா கட்டணங்களைப் பொறுத்தவரை, [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ஐ பயன்படுத்தி உங்கள் Ultimate Multisite வாடிக்கையாளர்கள் உங்கள் முதன்மை தளத்தில் WooCommerce gateways மூலம் கட்டணம் செலுத்த அனுமதிக்கலாம். இந்த வழியில், WooCommerce க்காக உருவாக்கப்பட்ட பெரிய payment gateways நூலகத்தை பயன்படுத்தி உங்கள் Ultimate Multisite வாடிக்கையாளர்களுக்கு கட்டணம் வசூலிக்கலாம்.

  * **WPMUDEV Pro Sites இலிருந்து சந்தாக்களை மாற்ற வழி உள்ளதா?**

ஆம், Ultimate Multisite இப்போது உங்கள் Pro Sites வலையமைப்பை மாற்ற [உதவிகரமான கருவியை](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) வழங்குகிறது.

  * **எனது Plan Edit Page இல் "Allowed Plugins" tab ஏன் காலியாக உள்ளது?**

உங்கள் வலையமைப்பில் network-activated செய்யப்படாத plugins க்கு மட்டுமே அணுகலை அனுமதிக்க அல்லது மறுக்க Ultimate Multisite அனுமதிக்கிறது. ஒரு plugin network-activated செய்யப்படும்போது, அதன் இயல்பிலேயே, அது உங்கள் வலையமைப்பின் ஒவ்வொரு தளத்திலும் இயல்பாக ஏற்றப்படும்.

"Allowed Plugins" tab இல் எந்த plugins ம் பட்டியலிடப்படவில்லை என்றால், விரும்பிய plugin network-active ஆக உள்ளதா இல்லையா என்பதை சரிபார்க்கவும்.

  * **எனது சொந்த Pricing Table வடிவமைப்பை பயன்படுத்தலாமா?**

ஆம். பதிப்பு 1.4.0 முதல், Ultimate Multisite template overrides ஐ ஆதரிக்கிறது.

  * **Ultimate Multisite ஆனது WP Engine உடன் வேலை செய்யுமா?**

ஆம். Ultimate Multisite எப்போதும் WP Engine ஆல் ஆதரிக்கப்பட்டது, ஆனால் domain mapping வேலை செய்ய நிர்வாகியிடமிருந்து கூடுதல் கைமுறை படிகள் தேவைப்பட்டன (WP Engine panel இல் domain ஐ சேர்ப்பது). **பதிப்பு 1.6.0 முதல் இது தேவையில்லை.** Ultimate Multisite நேரடியாக WP Engine API உடன் ஒருங்கிணைக்கிறது மற்றும் mapping க்குப் பிறகு domains தானாகவே panel இல் சேர்க்கப்படும்.

  * **எந்த hosting வழங்குநரை பரிந்துரைக்கிறீர்கள்?**

Ultimate Multisite எந்த hosting வழங்குநருடனும் வேலை செய்ய வேண்டும். ஒரே வேறுபாடு என்னவென்றால், Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane மற்றும் [Runcloud.io](http://Runcloud.io) போன்ற சில hosting நிறுவனங்கள் domain mapping திறன்களுக்கு நெருக்கமான ஒருங்கிணைப்பைக் கொண்டுள்ளன, எனவே அவை உடனடியாக வேலை செய்யும். மற்ற hostings க்கு, நீங்கள் domain mapping ஐ கைமுறையாக அமைக்க வேண்டியிருக்கலாம்.
