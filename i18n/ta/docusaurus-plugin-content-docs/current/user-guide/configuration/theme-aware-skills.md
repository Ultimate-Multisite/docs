# தீம்-அறிவான திறன்கள் (Theme-Aware Skills)

Superdav AI Agent v1.10.0 ஆனது நான்கு புதிய தீம்-அறிவான (theme-aware) உள்ளமைக்கப்பட்ட திறன்களை (built-in skills) வழங்குகிறது. இவை உங்கள் செயலில் உள்ள WordPress தீமுக்கு ஏற்ப தானாகவே மாறிக்கொள்ளும். உங்கள் தீம் கட்டமைப்பிற்கும் அம்சங்களுக்கும் ஏற்றவாறு சிறப்பு வழிகாட்டுதல்களையும் திறன்களையும் இந்தத் திறன்கள் வழங்குகின்றன.

## தீம்-அறிவான திறன்கள் என்றால் என்ன? {#what-are-theme-aware-skills}

தீம்-அறிவான திறன்கள் என்பவை, உங்கள் தளத்தில் தற்போது செயலில் உள்ள WordPress தீம் எது என்பதன் அடிப்படையில் AI உதவியாளர் தானாகவே தேர்ந்தெடுக்கும், முன்பே கட்டமைக்கப்பட்ட அறிவுத் தளங்கள் மற்றும் கருவி தொகுப்புகள் ஆகும். நீங்கள் தீம்களை மாற்றும்போது, உதவியாளரின் கிடைக்கக்கூடிய திறன்கள் தானாகவே புதுப்பிக்கப்படும் — இதற்கு எந்த கைமுறை அமைப்பும் தேவையில்லை.

ஒவ்வொரு திறனும் பின்வருவனவற்றை உள்ளடக்கியது:

- **தீம்-குறிப்பிட்ட ஆவணங்கள் (Theme-specific documentation)** — தீமைப் பயன்படுத்துவது மற்றும் தனிப்பயனாக்குவது குறித்த வழிகாட்டுதல்
- **பிளாக் மற்றும் வடிவமைப்பு குறிப்புகள் (Block and pattern references)** — கிடைக்கக்கூடிய பிளாக்கள், வடிவங்கள் (patterns) மற்றும் வடிவமைப்பு விருப்பங்கள்
- **தனிப்பயனாக்குதல் எடுத்துக்காட்டுகள் (Customization examples)** — பொதுவான பணிகளுக்கான குறியீடு துண்டுகள் (code snippets) மற்றும் அமைப்பு முறைகள்
- **சிறந்த நடைமுறைகள் (Best practices)** — தீம் கட்டமைப்பு மற்றும் பணிப்பாய்வுக்கான பரிந்துரைகள்

## கிடைக்கக்கூடிய தீம்-அறிவான திறன்கள் {#available-theme-aware-skills}

### பிளாக் தீம்கள் (Block Themes) {#block-themes}

**எதற்குப் பொருந்தும்:** WordPress பிளாக் அடிப்படையிலான (Full Site Editing) கட்டமைப்பைப் பயன்படுத்தும் தீம்கள்.

பிளாக் தீம்கள் திறன் பின்வரும் வழிகாட்டுதல்களை வழங்குகிறது:

- பிளாக் எடிட்டரைப் பயன்படுத்தி டெம்ப்ளேட்டுகளை உருவாக்குதல் மற்றும் திருத்துதல்
- பிளாக் வடிவங்கள் (block patterns) மற்றும் மீண்டும் பயன்படுத்தக்கூடிய பிளாக்lerle வேலை செய்தல்
- உலகளாவிய ஸ்டைல்கள் (global styles) மற்றும் theme.json அமைப்புகளை தனிப்பயனாக்குதல்
- தீம் பிளாக்கள் மற்றும் மாறுபாடுகளைப் பயன்படுத்துதல்
- உங்கள் தளத்திற்காக தனிப்பயமான பிளாக் வடிவங்களை உருவாக்குதல்

**எப்போது தானாகச் செயல்படுத்தப்படும்:** உங்கள் செயலில் உள்ள தீம் ஒரு பிளாக் தீமாக இருக்கும்போது (இது `block-templates` அம்சத்தை ஆதரிக்கிறது).

### கிளாசிக் தீம்கள் (Classic Themes) {#classic-themes}

**எதற்குப் பொருந்தும்:** PHP டெம்ப்ளேட்டுகள் மற்றும் கிளாசிக் எடிட்டரைப் பயன்படுத்தும் பாரம்பரிய WordPress தீம்கள்.

கிளாசிக் தீம்கள் திறன் பின்வரும் வழிகாட்டுதல்களை வழங்குகிறது:

- PHP டெம்ப்ளேட் கோப்புகள் மற்றும் ஹூக்குகளுடன் (hooks) வேலை செய்தல்
- கஸ்டமைசரைப் (Customizer) மூலம் தீம் தோற்றத்தை தனிப்பயனாக்குதல்
- 위ட்ஜெட் பகுதிகள் (widget areas) மற்றும் சைட்பார் மூலம் வேலை செய்தல்
- CSS மாற்றுதல் மற்றும் சில்ட் தீம் மேம்பாடு (child theme development)
- தீம் படிநிலை (theme hierarchy) மற்றும் டெம்ப்ளேட் குறிச்சொற்களைப் (template tags) புரிந்துகொள்ளுதல்

**எப்போது தானாகச் செயல்படுத்தப்படும்:** உங்கள் செயலில் உள்ள தீம் ஒரு கிளாசிக் (பிளாக் அல்லாத) தீமாக இருக்கும்போது.

### Kadence Blocks {#kadence-blocks}

**எதற்குப் பொருந்தும்:** மேம்பட்ட பிளாக் அடிப்படையிலான வடிவமைப்பிற்காக Kadence Blocks plugin ஐப் பயன்படுத்தும் தளங்கள்.

Kadence Blocks திறன் பின்வரும் வழிகாட்டுதல்களை வழங்குகிறது:

- Kadence இன் மேம்பட்ட பிளாக் நூலகத்தைப் பயன்படுத்துதல் (Hero, Testimonials, Pricing போன்றவை)
- Kadence பிளாக் அமைப்புகள் மற்றும் பதிலளிக்கும் விருப்பங்களை (responsive options) கட்டமைத்தல்
- Kadence இன் grid மற்றும் container பிளாக்களைப் பயன்படுத்தி தனிப்பயமான லேஅவுட்களை உருவாக்குதல்
- உங்கள் தீமுடன் Kadence பிளாக்களை ஒருங்கிணைத்தல்
- Kadence இன் வடிவமைப்பு அமைப்பு மற்றும் முன்அமைப்புகளைப் (presets) பயன்படுத்துதல்

**எப்போது தானாகச் செயல்படுத்தப்படும்:** Kadence Blocks plugin உங்கள் தளத்தில் செயலில் இருக்கும்போது.

### Kadence Theme {#kadence-theme}

**எதற்குப் பொருந்தும்:** பிளாக் அடிப்படையிலான வடிவமைப்பு மற்றும் தனிப்பயனாக்கத்திற்காக Kadence தீமைப் பயன்படுத்தும் தளங்கள்.

Kadence Theme திறன் பின்வரும் வழிகாட்டுதல்களை வழங்குகிறது:

- உலகளாவிய ஸ்டைல்கள் மற்றும் theme.json மூலம் Kadence தீமை தனிப்பயனாக்குதல்
- Kadence இன் உள்ளமைக்கப்பட்ட பிளாக் வடிவங்கள் மற்றும் டெம்ப்ளேட்டுகளைப் பயன்படுத்துதல்
- Kadence தீம் அமைப்புகள் மற்றும் விருப்பங்களை கட்டமைத்தல்
- Kadence இன் வடிவமைப்பு அமைப்பு மூலம் தனிப்பயமான வடிவமைப்புகளை உருவாக்குதல்
- பிரபலமான பிளக்இன்கள் மற்றும் கருவிகளுடன் Kadence ஐ ஒருங்கிணைத்தல்

**எப்போது தானாகச் செயல்படுத்தப்படும்:** Kadence தீம் உங்கள் செயலில் உள்ள தீமாக இருக்கும்போது.

## திறன்கள் எவ்வாறு தேர்ந்தெடுக்கப்படுகின்றன {#how-skills-are-selected}

உதவியாளர் ஒவ்வொரு செய்தியிலும் உங்கள் செயலில் உள்ள தீம் மற்றும் நிறுவப்பட்ட பிளக்இன்களை தானாகவே கண்டறிகிறது. பொருத்தமான தீம்-அறிவான திறன் கிடைத்தால், அது தானாகவே உதவியாளரின் சூழலில் (context) ஏற்றப்படுகிறது. நீங்கள் திறன்களை கைமுறையாக இயக்கவோ அல்லது மாற்றவோ தேவையில்லை.

### பல திறன்கள் (Multiple Skills) {#multiple-skills}

உங்கள் தளத்திற்கு பல திறன்கள் பொருந்தினால் (உதாரணமாக, உங்களிடம் Kadence Blocks மற்றும் Kadence Theme இரண்டும் செயலில் இருந்தால்), உதவியாளர் அனைத்துப் பொருந்தக்கூடிய திறன்களின் அணுகலைப் பெற்றிருக்கும் மற்றும் ஒவ்வொன்றிலிருந்தும் வழிகாட்டுதல்களை மேற்கோள் காட்ட முடியும்.

### தீம்களை மாற்றுதல் (Switching Themes) {#switching-themes}

நீங்கள் உங்கள் செயலில் உள்ள தீமை மாற்றும்போது, அடுத்த செய்தியில் உதவியாளரின் கிடைக்கக்கூடிய திறன்கள் தானாகவே புதுப்பிக்கப்படும். உதாரணமாக:

1. நீங்கள் **Block Themes** திறன் செயலில் உள்ள ஒரு பிளாக் தீமைப் பயன்படுத்துகிறீர்கள்.
2. நீங்கள் ஒரு கிளாசிக் தீமாக மாறுகிறீர்கள்.
3. உங்கள் அடுத்த செய்தியில், **Classic Themes** திறன் தானாகவே ஏற்றப்பட்டு, **Block Themes** திறன் கிடைக்காமல் போகிறது.

## தீம்-அறிவான திறன்களைப் பயன்படுத்துதல் {#using-theme-aware-skills}

ஒரு தீம்-அறிவான திறனைப் பயன்படுத்த, நீங்கள் என்ன செய்ய விரும்புகிறீர்கள் என்பதை சட் இடைமுகத்தில் (chat interface) விவரித்தால் போதும். பொருத்தமான திறனின் வழிகாட்டுதல்களை உதவியாளர் தானாகவே மேற்கோள் காட்டும்.

### மாதிரி தூண்டுதல்கள் (Example Prompts) {#example-prompts}

**Block Themes க்காக:**
> "பிளாக் வடிவங்களைப் பயன்படுத்தி பின்னணி படத்துடனும் மையப்படுத்தப்பட்ட உரையுடனும் ஒரு ஹீரோ பிரிவை உருவாக்கவும்."

**Classic Themes க்காக:**
> "ஒரு சில்ட் தீத்தைப் பயன்படுத்தி சைட்பார்-க்கு ஒரு தனிப்பயன் 위ட்ஜெட் பகுதியைச் சேர்க்கவும்."

**Kadence Blocks க்காக:**
> "Kadence Testimonials பிளாக்வைப் பயன்படுத்தி ஒரு சாட்சியங்கள் பிரிவை உருவாக்கவும்."

**Kadence Theme க்காக:**
> "தலைப்பு அமைப்பு மற்றும் வழிசெலுத்தல் மெனுவின் ஸ்டைலை தனிப்பயனாக்கவும்."

உங்கள் செயலில் உள்ள தீம் மற்றும் பிளக்இன்களுக்கு ஏற்ற தீம்-குறிப்பிட்ட வழிகாட்டுதலையும் குறியீடு எடுத்துக்காட்டுகளையும் உதவியாளர் வழங்கும்.
