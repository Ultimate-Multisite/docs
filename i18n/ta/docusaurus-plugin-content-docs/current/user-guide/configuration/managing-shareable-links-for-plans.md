# திட்டங்களுக்கான பகிரக்கூடிய இணைப்புகளை நிர்வகித்தல்

Ultimate Multisite உங்கள் வாடிக்கையாளர்களுடன் திட்ட இணைப்புகளை நேரடியாகப் பகிரும் வாய்ப்பை வழங்குகிறது. இதைச் செய்ய, உங்கள் திட்டங்களின் திருத்தும் பக்கங்களில் ஏதேனும் ஒன்றுக்குச் சென்று _Click to copy the Shareable Link_ பொத்தானைக் கிளிக் செய்யுங்கள்.

_திட்டத்தின் திருத்தும் பக்கத்தில், பகிரக்கூடிய இணைப்புகளை உருவாக்குவதற்கான பொத்தானைக் காணலாம்_

இந்த முறை இணைப்பை உருவாக்க இயல்புநிலை பில்லிங் அதிர்வெண்ணைப் பயன்படுத்துகிறது என்பதை கவனியுங்கள். அதாவது, உங்கள் இயல்புநிலை பில்லிங் மாதாந்திரமாக இருந்தால், இறுதியில் நீங்கள் இது போன்ற ஒரு இணைப்பைப் பெறுவீர்கள்: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

பகிரக்கூடிய இணைப்பு காலாண்டு அதிர்வெண்ணை இயல்புநிலையாகக் கொண்டிருக்க வேண்டுமெனில், நீங்கள் இறுதியில் 3 ஐச் சேர்க்க வேண்டும்

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

வெவ்வேறு பில்லிங் அதிர்வெண்களுக்கு ஒரே இணைப்பை வைத்திருக்க முடியாது என்பதை நினைவில் கொள்ளுங்கள். உங்கள் ஒவ்வொரு பில்லிங் அதிர்வெண்ணுக்கும் ஒரு குறிப்பிட்ட இணைப்பைப் பகிர வேண்டும்.

இணைப்பை உங்கள் வாடிக்கையாளர்களுடன் பகிர்ந்தவுடன், அது அவர்களை உங்கள் பதிவு செயல்முறையின் முதல் படிக்கு திருப்பி விடும்.
