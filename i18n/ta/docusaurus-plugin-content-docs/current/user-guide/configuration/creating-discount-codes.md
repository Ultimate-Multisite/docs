# தள்ளுபடி குறியீடுகளை உருவாக்குதல் (v2)

_**முக்கிய குறிப்பு: இந்தக் கட்டுரை Ultimate Multisite பதிப்பு 2.x ஐ குறிக்கிறது.**_

Ultimate Multisite மூலம் உங்கள் வாடிக்கையாளர்களின் சந்தாக்களுக்கு தள்ளுபடிகள் வழங்க தள்ளுபடி குறியீடுகளை உருவாக்கலாம். இதை உருவாக்குவது எளிதானது!

## தள்ளுபடி குறியீடுகளை உருவாக்குதல் மற்றும் திருத்துதல்

தள்ளுபடி குறியீட்டை உருவாக்க அல்லது திருத்த, **Ultimate Multisite > Discount Codes** என்பதற்குச் செல்லுங்கள்.

![](/img/config/discount-codes-list.png)

அங்கு நீங்கள் ஏற்கனவே உருவாக்கிய தள்ளுபடி குறியீடுகளின் பட்டியலைக் காணலாம்.

புதிய கூப்பன் உருவாக்க **Add Discount** **Code** என்பதைக் கிளிக் செய்யலாம் அல்லது ஏற்கனவே உள்ளவற்றின் மீது hover செய்து **Edit** என்பதைக் கிளிக் செய்து திருத்தலாம்.

![Hover செயல்களுடன் கூடிய தள்ளுபடி குறியீடுகள் பட்டியல்](/img/config/discount-codes-list.png)

![Add Discount Code பொத்தான்](/img/config/discount-codes-list.png)

உங்கள் கூப்பன் குறியீட்டை உருவாக்கும் அல்லது திருத்தும் பக்கத்திற்கு நீங்கள் திருப்பி விடப்படுவீர்கள். இந்த உதாரணத்தில் புதிய ஒன்றை உருவாக்குவோம்.

![தள்ளுபடி குறியீடு திருத்தும் பக்கம்](/img/config/discount-codes-list.png)

இங்கு கிடைக்கும் அமைப்புகளைப் பார்ப்போம்:

**Enter Discount Code:** இது உங்கள் தள்ளுபடி குறியீட்டின் பெயர் மட்டுமே. இது checkout படிவத்தில் உங்கள் வாடிக்கையாளர்கள் பயன்படுத்த வேண்டிய குறியீடு அல்ல.

**Description:** இந்த கூப்பன் எதற்கானது என்பதை இங்கு சுருக்கமாக விவரிக்கலாம்.

![தள்ளுபடி குறியீடு பெயர் மற்றும் விளக்கப் புலங்கள்](/img/config/discount-codes-list.png)

**Coupon code:** உங்கள் வாடிக்கையாளர்கள் checkout-இன் போது உள்ளிட வேண்டிய குறியீட்டை இங்கு வரையறுக்கலாம்.

![Coupon code புலம்](/img/config/discount-codes-list.png)

**Discount:** உங்கள் தள்ளுபடி குறியீட்டிற்கு **சதவீதம்** அல்லது **நிலையான தொகை** அமைக்கலாம்.

![தள்ளுபடி சதவீதம் அல்லது நிலையான தொகை அமைப்பு](/img/config/discount-codes-list.png)

**Apply to renewals:** இந்த விருப்பம் அணைக்கப்பட்டிருந்தால், இந்த தள்ளுபடி குறியீடு **முதல் பணம் செலுத்துதலுக்கு** மட்டுமே பொருந்தும். மற்ற எல்லா பணம் செலுத்துதல்களுக்கும் தள்ளுபடி இருக்காது. இந்த விருப்பம் இயக்கப்பட்டிருந்தால், தள்ளுபடி குறியீடு எதிர்கால அனைத்து பணம் செலுத்துதல்களுக்கும் செல்லுபடியாகும்.

**Setup fee discount:** இந்த விருப்பம் அணைக்கப்பட்டிருந்தால், கூப்பன் குறியீடு ஆர்டரின் **setup fee-க்கு எந்த தள்ளுபடியும் தராது**. இந்த விருப்பம் இயக்கப்பட்டிருந்தால், இந்த கூப்பன் குறியீடு உங்கள் திட்டங்களின் setup fee-க்கு பொருந்தும் தள்ளுபடியை (சதவீதம் அல்லது நிலையான தொகை) அமைக்கலாம்.

![Apply to renewals மற்றும் setup fee discount விருப்பங்கள்](/img/config/discount-codes-list.png)

**Active:** இந்த கூப்பன் குறியீட்டை கைமுறையாக இயக்கவும் அல்லது நிறுத்தவும்.

![தள்ளுபடி குறியீட்டுக்கான Active toggle](/img/config/discount-codes-list.png)

**Advanced Options** என்பதன் கீழ், பின்வரும் அமைப்புகள் உள்ளன:

**Limit uses:**

  * **Uses:** தள்ளுபடி குறியீடு எத்தனை முறை பயன்படுத்தப்பட்டது என்பதை இங்கு காணலாம்.

  * **Max uses:** பயனர்கள் இந்த தள்ளுபடி குறியீட்டை எத்தனை முறை பயன்படுத்தலாம் என்பதை இது வரம்பிடும். எடுத்துக்காட்டாக, இங்கு 10 என்று வைத்தால், கூப்பனை 10 முறை மட்டுமே பயன்படுத்த முடியும். இந்த வரம்புக்குப் பிறகு, கூப்பன் குறியீட்டை பயன்படுத்த முடியாது.

![Uses மற்றும் max uses புலங்களுடன் Limit uses அமைப்பு](/img/config/discount-codes-list.png)**Start & expiration dates:** உங்கள் கூப்பனுக்கு தொடக்க தேதி மற்றும்/அல்லது காலாவதி தேதி சேர்க்கும் விருப்பம் இங்கு இருக்கும்.

![தொடக்க மற்றும் காலாவதி தேதி புலங்கள்](/img/config/discount-codes-list.png)

**Limit products:** **Select products** என்பதை இயக்கினால், உங்கள் அனைத்து தயாரிப்புகளும் காட்டப்படும். இந்த கூப்பன் குறியீட்டை எந்த தயாரிப்பு ஏற்கலாம் என்பதை (இயக்கி அல்லது அணைத்து) கைமுறையாகத் தேர்ந்தெடுக்கும் விருப்பம் இருக்கும். இங்கு அணைக்கப்பட்ட தயாரிப்புகளுக்கு உங்கள் வாடிக்கையாளர்கள் இந்த கூப்பன் குறியீட்டைப் பயன்படுத்த முயன்றாலும் எந்த மாற்றமும் காட்டப்படாது.

![Limit products தேர்வு toggleகள்](/img/config/discount-codes-list.png)

இந்த விருப்பங்கள் அனைத்தையும் அமைத்த பின், உங்கள் கூப்பனைச் சேமிக்க **Save Discount Code** என்பதைக் கிளிக் செய்யுங்கள், முடிந்தது!

![Save Discount Code பொத்தான்](/img/config/discount-codes-list.png)

கூப்பன் இப்போது உங்கள் பட்டியலில் உள்ளது, அங்கிருந்து **திருத்த அல்லது நீக்க** கிளிக் செய்யலாம்.

![திருத்து மற்றும் நீக்கு விருப்பங்களுடன் பட்டியலில் தள்ளுபடி குறியீடு](/img/config/discount-codes-list.png)

### URL Parameters பயன்படுத்துதல்:

உங்கள் விலை அட்டவணைகளைத் தனிப்பயனாக்க அல்லது உங்கள் வலைத்தளத்திற்கு அழகான கூப்பன் குறியீடு பக்கத்தை உருவாக்க விரும்பினால், மற்றும் உங்கள் checkout படிவத்தில் தள்ளுபடி குறியீட்டை தானாகப் பொருத்த விரும்பினால், URL parameters மூலம் இதைச் செய்யலாம்.

முதலில், உங்கள் திட்டத்திற்கான shareable link பெற வேண்டும். இதற்கு, **Ultimate Multisite > Products** என்பதற்குச் சென்று ஒரு திட்டத்தைத் தேர்ந்தெடுங்கள்.

**Click to Copy Shareable Link** பொத்தானைக் கிளிக் செய்யுங்கள். இது இந்த குறிப்பிட்ட திட்டத்திற்கான shareable link-ஐ தரும். எங்கள் விஷயத்தில், கிடைத்த shareable link [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_._

![Shareable link பொத்தானுடன் கூடிய Product பக்கம்](/img/config/products-list.png)

இந்த குறிப்பிட்ட திட்டத்திற்கு உங்கள் தள்ளுபடி குறியீட்டைப் பொருத்த, URL-இல் **?discount_code=XXX** என்ற parameter-ஐ சேர்க்கவும். இங்கு **XXX** என்பது கூப்பன் குறியீடு.

இந்த உதாரணத்தில், **50OFF** என்ற கூப்பன் குறியீட்டை இந்த குறிப்பிட்ட தயாரிப்புக்குப் பொருத்துவோம்.

இந்த குறிப்பிட்ட திட்டத்திற்கான URL, 50OFF தள்ளுபடி குறியீடு பொருத்தப்பட்ட நிலையில்: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_.
