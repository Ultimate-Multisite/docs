# இணைய தேடல் (Internet Search) {#internet-search}

Gratis AI Agent v1.5.0 ஆனது **இணைய தேடல்** திறனைச் சேர்க்கிறது. இது AI உதவியாளரால் உரையாடலின் போது இணையத்தில் இருந்து சமீபத்திய தகவல்களைப் பெற உதவுகிறது. இது [Brave Search API](https://brave.com/search/api/) அல்லது [Tavily API](https://tavily.com/) மூலம் இயக்கப்படுகிறது.

## இது எப்படி வேலை செய்கிறது (How It Works) {#how-it-works}

இணைய தேடல் இயக்கப்படும்போது, கேள்விக்கு தற்போதைய அல்லது வெளிப்புறத் தகவல் தேவை என்று உதவியாளர் தீர்மானித்தால், அது தானாகவே நீங்கள் அமைத்த தேடல் வழங்குநரிடம் (search provider) வினவலை அனுப்பும். உதாரணமாக, சமீபத்திய செய்திகள், நேரடி விலை விவரங்கள் அல்லது மாடலின் பயிற்சி நிறுத்தத் தேதிக்குப் பிறகு மாறியிருக்கக்கூடிய ஆவணங்கள் போன்ற தகவல்களுக்கு இது தேவைப்படலாம்.

முடிவுகள் நிகழ்நேரத்தில் பெறப்பட்டு, உதவியாளர் பதிலைத் உருவாக்குவதற்கு முன்பு அதன் சூழலில் (context) செலுத்தப்படும். தேடல் முடிவுகளைப் பயன்படுத்தி ஒரு கேள்விக்கு பதிலளித்தபோது, உதவியாளர் அதைக் குறிப்பிადგენார்.

## இணைய தேடலை இயக்குதல் (Enabling Internet Search) {#enabling-internet-search}

இணைய தேடலுக்கு நீங்கள் தேர்ந்தெடுக்கும் தேடல் வழங்குநரிடமிருந்து ஒரு API key தேவை. இதை அமைக்க, பின்வரும் படிகளைப் பின்பற்றவும்:

1. **Gratis AI Agent → Settings → Advanced** என்பதற்குச் செல்லவும்.
2. **Internet Search Provider** என்ற கீழ்தோன்றும் மெனுவைக் (dropdown) கண்டுபிடித்து, **Brave Search** அல்லது **Tavily** ஒன்றைத் தேர்ந்தெடுக்கவும்.
3. தொடர்புடைய புலத்தில் (field) உங்கள் API key-ஐ உள்ளிடவும். ஒவ்வொரு புலத்தின் அருகிலும் பதிவு செய்வதற்கான இணைப்புகள் (Sign-up links) காட்டப்படும்.
4. **Save Settings** என்பதைக் கிளிக் செய்யவும்.

key சேமிக்கப்பட்டவுடன், இணைய தேடல் திறன் தானாகவே உதவியாளருக்குக் கிடைக்கும்.

## Brave Search {#brave-search}

### Brave Search API Key பெறுதல் {#obtaining-a-brave-search-api-key}

1. [Brave Search API page](https://brave.com/search/api/) என்ற பக்கத்திற்குச் செல்லவும்.
2. ஒரு திட்டத்திற்காக (plan) பதிவு செய்யவும். மாதத்திற்கு ஒரு குறிப்பிட்ட எண்ணிக்கையிலான கோரிக்கைகளுடன் இலவச அடுக்கு (free tier) கிடைக்கிறது.
3. Brave Search developer dashboard இலிருந்து உங்கள் API key-ஐ நகலெடுக்கவும் (Copy).
4. அதை Gratis AI Agent அமைப்புகளில் உள்ள **Brave Search API Key** புலத்தில் ஒட்டவும் (Paste).

### பயன்பாட்டு வரம்புகள் (Usage Limits) {#usage-limits}

பயன்பாடு, அனுப்பப்படும் வினவல் (queries) எண்ணிக்கையின் அடிப்படையில் Brave Search மூலம் கட்டணம் வசூலிக்கப்படுகிறது. தேடலைத் தூண்டும் ஒவ்வொரு AI பதிலும் ஒரு வினவலாகக் கணக்கிடப்படும். எதிர்பாராத கட்டணங்களைத் தவிர்க்க, [Brave Search developer dashboard](https://brave.com/search/api/) இல் உங்கள் பயன்பாட்டைக் கண்காணிக்கவும்.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 ஆனது, மிகவும் செழுமையான தேடல் முடிவுகள் மற்றும் மேம்பட்ட ஆராய்ச்சி திறன்களை வழங்கும் மாற்று இணைய தேடல் வழங்குநராக **Tavily**-யைச் சேர்க்கிறது.

### Tavily API Key பெறுதல் {#obtaining-a-tavily-api-key}

1. [Tavily API page](https://tavily.com/) என்ற பக்கத்திற்குச் செல்லவும்.
2. ஒரு கணக்கிற்காக (account) பதிவு செய்யவும். மாதத்திற்கு ஒரு குறிப்பிட்ட எண்ணிக்கையிலான கோரிக்கைகளுடன் இலவச அடுக்கு கிடைக்கிறது.
3. Tavily dashboard இலிருந்து உங்கள் API key-ஐ நகலெடுக்கவும்.
4. அதை Gratis AI Agent அமைப்புகளில் உள்ள **Tavily API Key** புலத்தில் ஒட்டவும்.

### பயன்பாட்டு வரம்புகள் (Usage Limits) {#usage-limits-1}

பயன்பாடு, அனுப்பப்படும் API அழைவுகளின் (API calls) எண்ணிக்கையின் அடிப்படையில் Tavily மூலம் கட்டணம் வசூலிக்கப்படுகிறது. தேடலைத் தூண்டும் ஒவ்வொரு AI பதிலும் ஒரு அழைவாகக் கணக்கிடப்படும். எதிர்பாராத கட்டணங்களைத் தவிர்க்க, [Tavily dashboard](https://tavily.com/) இல் உங்கள் பயன்பாட்டைக் கண்காணிக்கவும்.

## இணைய தேடலை முடக்குதல் (Disabling Internet Search) {#disabling-internet-search}

செயல்படுத்தப்பட்ட தேடல் வழங்குநரிடமிருந்து API key-ஐ நீக்கிவிட்டு சேமிக்கவும். இணைய தேடல் திறன் இனி உதவியாளருக்கு வழங்கப்படாது.

:::note
உதவியாளர் பதிலைத் தொடங்குவதற்கு முன் தேடல் முடிவுகளுக்காகக் காத்திருக்க வேண்டியிருப்பதால், இணைய தேடல் பதில்களில் தாமதத்தை (latency) சேர்க்கிறது. நேரத்திற்கு முக்கியமான பயன்பாடுகளுக்கு, நிகழ்நேர தேடல் தேவையா அல்லது உதவியாளரின் உள்ளமைக்கப்பட்ட அறிவு போதுமானதா என்பதை நீங்கள் பரிசீலிக்க வேண்டும்.
:::
