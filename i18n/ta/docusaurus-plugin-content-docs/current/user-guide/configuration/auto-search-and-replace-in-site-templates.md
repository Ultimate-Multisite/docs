# தள டெம்ப்ளேட்களில் தானியங்கி தேடல் மற்றும் மாற்றம் (v2)

_**இந்த டுடோரியலுக்கு WP UItimo பதிப்பு 2.x தேவை.**_

Ultimate Multisite-ன் மிகச்சிறந்த அம்சங்களில் ஒன்று, பதிவு படிவத்தில் உரை, நிறம் மற்றும் தேர்வு புலங்களை உங்கள் விருப்பப்படி சேர்க்கும் திறன் ஆகும். இந்தத் தரவுகளைப் பெற்ற பின், தேர்ந்தெடுக்கப்பட்ட தள டெம்ப்ளேட்டின் குறிப்பிட்ட பகுதிகளில் உள்ளடக்கத்தை முன்கூட்டியே நிரப்ப இதைப் பயன்படுத்தலாம். புதிய தளம் வெளியிடப்படும்போது, பதிவின் போது உள்ளிடப்பட்ட உண்மையான தகவல்களுடன் Ultimate Multisite தானாகவே placeholder-களை மாற்றிவிடும்.

உதாரணமாக, உங்கள் டெம்ப்ளேட் தளங்களை placeholder-களுடன் உருவாக்கலாம். Placeholder-கள் இரட்டை சுருள் அடைப்புக்குறிகளால் சூழப்பட்டிருக்க வேண்டும் - {{placeholder_name}}.

பின்னர், அந்தத் தரவைப் பெற ஒரு பொருத்தமான பதிவு புலத்தைச் சேர்க்கலாம்

உங்கள் வாடிக்கையாளர் பதிவின் போது அந்தப் புலத்தை நிரப்ப முடியும்.

வாடிக்கையாளர் வழங்கிய தரவுகளுடன் Ultimate Multisite தானாகவே placeholder-களை மாற்றிவிடும்.

## **"placeholder-கள் நிறைந்த டெம்ப்ளேட்" பிரச்சனைக்கு தீர்வு**

இவை அனைத்தும் சிறப்பானவை, ஆனால் ஒரு அசிங்கமான பிரச்சனை எழுகிறது: இப்போது உங்கள் வாடிக்கையாளர்கள் பார்வையிடக்கூடிய தள டெம்ப்ளேட்கள் அதிகம் விளக்காத அசிங்கமான placeholder-களால் நிரம்பியுள்ளன.

இதைத் தீர்க்க, placeholder-களுக்கு போலி மதிப்புகளை அமைக்கும் வசதியை வழங்குகிறோம். உங்கள் வாடிக்கையாளர்கள் பார்வையிடும்போது, டெம்ப்ளேட் தளங்களில் அவற்றின் உள்ளடக்கங்களைத் தேடி மாற்ற இந்த மதிப்புகளைப் பயன்படுத்துகிறோம்.

**Ultimate Multisite > Settings > Sites** என்பதற்குச் சென்று, பக்கப்பட்டியில் உள்ள Edit Placeholders இணைப்பைக் கிளிக் செய்வதன் மூலம் டெம்ப்ளேட் placeholder-கள் எடிட்டரை அணுகலாம்.

![Sites அமைப்புகள் பக்கப்பட்டியில் உள்ள Edit Placeholders இணைப்பு](/img/config/settings-sites.png)

இது placeholder-கள் உள்ளடக்க எடிட்டருக்கு உங்களை அழைத்துச் செல்லும், அங்கு placeholder-களையும் அவற்றின் தொடர்புடைய உள்ளடக்கங்களையும் சேர்க்கலாம்.

![Placeholder பெயர்கள் மற்றும் மதிப்புகளுடன் கூடிய Placeholders உள்ளடக்க எடிட்டர்](/img/config/settings-sites.png)
