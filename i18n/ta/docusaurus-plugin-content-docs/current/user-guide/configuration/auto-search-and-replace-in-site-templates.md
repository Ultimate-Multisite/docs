# தள வார்ப்புருக்களில் தானியங்கி தேடல் மற்றும் மாற்றம் (v2)

_**இந்த பயிற்சிக்கு WP UItimo பதிப்பு 2.x தேவை.**_

Ultimate Multisite இன் மிகச் சக்திவாய்ந்த அம்சங்களில் ஒன்று, பதிவு படிவத்தில் விருப்பமான உரை, நிறம், மற்றும் தேர்வு புலங்களைச் சேர்க்கும் திறன். அந்த தரவைப் பெற்றவுடன், தேர்ந்தெடுக்கப்பட்ட தள வார்ப்புருவின் சில பகுதிகளில் உள்ள உள்ளடக்கத்தை முன்கூட்டியே நிரப்ப அதைப் பயன்படுத்தலாம். பிறகு, புதிய தளம் வெளியிடப்பட்டதும், Ultimate Multisite பதிவின் போது உள்ளிடப்பட்ட உண்மையான தகவல்களால் இடம்நிரப்பிகளை மாற்றும்.

உதாரணமாக, உங்கள் வார்ப்புரு தளங்களை இடம்நிரப்பிகளுடன் உருவாக்கலாம். இடம்நிரப்பிகள் இரட்டை சுருள் அடைப்புக்குறிகளால் சூழப்பட்டவாறு சேர்க்கப்பட வேண்டும் - {{placeholder_name}}.

பிறகு, அந்த தரவைப் பெற பொருந்தும் பதிவு புலத்தை எளிதாகச் சேர்க்கலாம்

உங்கள் வாடிக்கையாளர் பதிவின் போது அந்த புலத்தை நிரப்ப முடியும்.

Ultimate Multisite வாடிக்கையாளர் வழங்கிய தரவால் இடம்நிரப்பிகளை தானாக மாற்றும்.

## **"இடம்நிரப்பிகளால் நிரம்பிய வார்ப்புரு" சிக்கலைத் தீர்ப்பது**

இவை அனைத்தும் சிறப்பானவை. ஆனால் ஒரு விரும்பத்தகாத சிக்கல் ஏற்படுகிறது: இப்போது எங்கள் வாடிக்கையாளர்கள் பார்வையிடக்கூடிய தள வார்ப்புருக்கள், அதிகம் சொல்லாத அழகில்லாத இடம்நிரப்பிகளால் நிரம்பியுள்ளன.

அதைத் தீர்க்க, இடம்நிரப்பிகளுக்கு போலி மதிப்புகளை அமைக்கும் விருப்பத்தை வழங்குகிறோம். உங்கள் வாடிக்கையாளர்கள் வார்ப்புரு தளங்களைப் பார்வையிடும் போது, அந்த மதிப்புகளைப் பயன்படுத்தி அவற்றின் உள்ளடக்கங்களைத் தேடி மாற்றுகிறோம்.

**Ultimate Multisite > Settings > Sites** என்பதற்கு சென்று, Site Template Options பகுதிவரை உருட்டி, பின்னர் **Edit Placeholders** இணைப்பைக் கிளிக் செய்வதன் மூலம் வார்ப்புரு இடம்நிரப்பிகள் திருத்தியை அணுகலாம்.

![Sites அமைப்புகள் பக்கத்தில் உள்ள தள வார்ப்புரு விருப்பங்கள் பகுதி](/img/config/settings-sites-templates-section.png)

அது உங்களை இடம்நிரப்பிகளின் உள்ளடக்க திருத்திக்குக் கொண்டு செல்லும். அங்கே நீங்கள் இடம்நிரப்பிகளையும் அவற்றின் தொடர்புடைய உள்ளடக்கத்தையும் சேர்க்கலாம்.

![வார்ப்புரு இடம்நிரப்பிகள் திருத்திக்கான நுழைவிடம்](/img/config/settings-sites-templates-section.png)
