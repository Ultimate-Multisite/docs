# உள்ளமைக்கப்பட்ட ஏஜெண்டுகள் (Built-in Agents)

Gratis AI Agent v1.9.0 ஆனது ஐந்து உள்ளமைக்கப்பட்ட ஏஜெண்டுகளை வழங்குகிறது. இவை ஒவ்வொன்றும் ஒரு குறிப்பிட்ட கருவிகளின் தொகுப்பு (tools), ஒரு தனிப்பயனாக்கப்பட்ட சிஸ்டம் ப்ராம்ட் (system prompt), மற்றும் அந்தத் துறையில் பொதுவான பணிகளுக்கான தொடக்க ஆலோசனைகளுடன் (starter suggestions) முன்பே கட்டமைக்கப்பட்டவை. ஏஜெண்டுகளுக்கு இடையே மாறுவது, உங்கள் தரப்பிலிருந்து எந்த உள்ளமைவும் இல்லாமல், உதவியாளர் என்ன செய்ய முடியும் மற்றும் எப்படி பதிலளிப்பார் என்பதை மாற்றுகிறது.

## ஏஜென்ட் என்றால் என்ன? (What Is an Agent?)

ஒவ்வொரு ஏஜென்ட்டும் ஒரு பெயரிடப்பட்ட உள்ளமைவு சுயவிவரமாகும் (named configuration profile). இது பின்வருவனவற்றை ஒருங்கிணைக்கிறது:

- **Tools (கருவிகள்)** — ஏஜென்ட் பயன்படுத்த அனுமதிக்கப்பட்ட திறன்கள் (உதாரணமாக, ஒரு Content Writer-க்கு போஸ்ட் உருவாக்கும் திறன்கள் அணுகல்; ஒரு Design Studio-க்கு CSS மற்றும் theme.json திறன்கள் அணுகல்).
- **System prompt (சிஸ்டம் ப்ராம்ட்)** — ஏஜென்ட்டின் தொனி, முன்னுரிமைகள் மற்றும் கட்டுப்பாடுகளை அமைக்கும் வழிமுறைகள்.
- **Suggestions (ஆலோசனைகள்)** — விரைவாகத் தொடங்க உங்களுக்கு உதவும் வகையில், சாட் இடைமுகத்தில் காட்டப்படும் முன் எழுதப்பட்ட ப்ராம்ட்கள்.

## ஏஜென்ட் பிக்கரை அணுகுவது எப்படி (Accessing the Agent Picker)

1. WordPress admin sidebar-ல் உள்ள **Gratis AI Agent** பேனலைத் திறக்கவும்.
2. சாட் தலைப்பின் மேல் இடது மூலையில் உள்ள **agent icon**-ஐ கிளிக் செய்யவும் (செயல்படுத்தப்பட்ட ஏஜெண்டைக் காட்ட இந்த ஐகான் மாறும்).
3. **Agent Picker** ஒரு படிவ-அட்டச்சுப் (form-table) மேல் அடுக்காகத் திறக்கும். ஒவ்வொரு ஏஜென்ட்டும் அதன் ஐகான், பெயர் மற்றும் ஒரு வரி விளக்கம் ஆகியவற்றுடன் பட்டியலிடப்பட்டுள்ளது.
4. அதைச் செயல்படுத்த ஒரு ஏஜென்ட் வரியை (agent row) கிளிக் செய்யவும். சாட் தலைப்பு உடனடியாகப் புதுப்பிக்கப்படும்.

நீங்கள் உரையாடலின் நடுப்பகுதியிலும் ஏஜெண்டுகளை மாற்றலாம் — புதிய ஏஜெண்டின் சிஸ்டம் ப்ராம்ட் அடுத்த செய்தியிலிருந்து நடைமுறைக்கு வரும்.

## ஐந்து உள்ளமைக்கப்பட்ட ஏஜெண்டுகள் (The Five Built-in Agents)

### Content Writer (உள்ளடக்க எழுத்தாளர்)

**கவனம்:** போஸ்ட்கள், பக்கங்கள் மற்றும் தொடர்பு படிவங்களை உருவாக்குதல் மற்றும் திருத்துதல்.

**கிடைப்பிலுள்ள கருவிகள்:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**சிறப்பாகச் செய்யும்வை:**
- ஒரு சுருக்கத்திலிருந்தோ அல்லது அவுட்லைனிலிருந்தோ வலைப்பதிவு இடுகைகளை வரைந்து வெளியிடுவது.
- ஒரு புதிய தளத்திற்கான பல லேண்டிங் பக்கங்களை உருவாக்குவது.
- தொடர்பு மற்றும் விசாரணை படிவங்களை உருவாக்குவது.
- URL அல்லது தேடலின் மூலம் போஸ்ட்களுக்கு சிறப்பம்சப் படங்களை (featured images) அமைப்பது.

**தொடக்க ஆலோசனைகள்:**
- *WordPress multisite-ன் நன்மைகள் பற்றி 500 வார்த்தைகள் கொண்ட ஒரு வலைப்பதிவு இடுகையை எழுதுங்கள்.*
- *ஒரு 'About', 'Services', மற்றும் 'Contact' பக்கங்களை உருவாக்கி வெளியிடுங்கள்.*
- *தொடர்புப் பக்கத்தில் ஒரு முன்பதிவு விசாரணை படிவத்தைச் சேர்க்கவும்.*

---

### Site Builder (தளம் உருவாக்குபவர்)

**கவனம்:** ஒரே ப்ராம்ட்டிலிருந்து முழுமையான வலைத்தள உருவாக்கம்.

**கிடைப்பிலுள்ள கருவிகள்:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**சிறப்பாகச் செய்யும்வை:**
- விவரிக்கப்பட்ட வணிக வகைக்கு பல கட்டங்கள் கொண்ட தள உருவாக்கத் திட்டத்தை உருவாக்குதல்.
- ஒவ்வொரு கட்டத்தையும் தானாகவே செயல்படுத்துதல் — அமைப்பு, உள்ளடக்கம், வழிசெலுத்தல், வடிவமைப்பு.
- கையேடு தலையீட்டைக் கோராமல், திட்டத்தின் நடுப்பகுதியில் உள்ள பிழைகளிலிருந்து மீண்டு வருதல்.
- உருவாக்கத்தின் ஒரு பகுதியாக பரிந்துரைக்கப்பட்ட plugins-களை நிறுவுதல்.
- சாட் இடைமுகத்திலிருந்தே தொடர்பு படிவங்களை உருவாக்குதல் (Superdav AI Agent v1.10.0+).

**தொடக்க ஆலோசனைகள்:**
- *ஒரு கேலரி போஸ்ட் வகை, ஒரு முன்பதிவுப் பக்கம் மற்றும் ஒரு தொடர்பு படிவத்துடன் கூடிய புகைப்படத் துறை போர்ட்ஃபோலியோ தளத்தை உருவாக்கவும்.*
- *ஆன்லைன் மெனு, திறக்கும் நேரம் மற்றும் மேசை முன்பதிவு விசாரணை படிவத்துடன் கூடிய உணவகம் வலைத்தளத்தை உருவாக்கவும்.*
- *சேவை பக்கங்கள், ஒரு போர்ட்ஃபோலியோ பகுதி மற்றும் ஒரு வலைப்பதிவு கொண்ட ஒரு ஃபிரிலான்ஸ் ஆலோசனை தளத்தை அமைக்கவும்.*
- *தளம் உருவாக்குபவரைப் பயன்படுத்தி தொடர்புப் பக்கத்தில் ஒரு தொடர்பு படிவத்தைச் சேர்க்கவும்.*

### Design Studio (வடிவமைப்பு ஸ்டுடியோ)

**கவனம்:** காட்சி தனிப்பயனாக்கம் — வண்ணங்கள், எழுத்துருக்கள் (typography), CSS மற்றும் பிளாக் வடிவங்கள் (block patterns).

**கிடைப்பிலுள்ள கருவிகள்:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**சிறப்பாகச் செய்யும்வை:**
- பெயரிடப்பட்ட தீம் ப்ரீசெட்களைப் பயன்படுத்துதல் (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog).
- theme.json மூலம் உலகளாவிய எழுத்துரு மற்றும் வண்ணத் தட்டுகளைச் செம்மைப்படுத்துதல்.
- பிராண்ட்-குறிப்பிட்ட மாற்றங்களுக்காக தனிப்பயன் CSS-ஐ செருகுதல்.
- ஒரு பக்கத்தின் ஸ்கிரீன்ஷாட்டை எடுத்து, வடிவமைப்புப் பிரச்சினைகளுக்காக அதை மதிப்பாய்வு செய்தல்.

**தொடக்க ஆலோசனைகள்:**
- *warm-editorial ப்ரீசெட்டைப் பயன்படுத்துங்கள், பின்னர் முதன்மை வண்ணத்தை #2d6a4f என்று அமைக்கவும்.*
- *முகப்புப் பக்கத்தின் ஸ்கிரீன்ஷாட்டை எடுத்து, நீங்கள் என்ன மேம்படுத்தலாம் என்று சொல்லுங்கள்.*
- *முழு அகல பின்னணிப் படத்துடனும் மையப்படுத்தப்பட்ட தலைப்புடனும் ஒரு மீண்டும் பயன்படுத்தக்கூடிய ஹீரோ பிளாக் வடிவத்தை உருவாக்கவும்.*

### Plugin Manager (Plugin மேலாளர்)

**கவனம்:** WordPress plugins-களைக் கண்டறிதல், நிறுவுதல் மற்றும் நிர்வகித்தல்.

**கிடைப்பிலுள்ள கருவிகள்:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**சிறப்பாகச் செய்யும்வை:**
- விவரிக்கப்பட்ட பயன்பாட்டுச் சூழலுக்குச் சிறந்த plugin-ஐப் பரிந்துரைத்தல்.
- பதிவேற்றத்தில் இருந்து ability packs-களை நிறுவுதல்.
- வகையின் அடிப்படையில் கிடைக்கக்கூடிய ability catalogue-ஐ உலாவல்.

**தொடக்க ஆலோசனைகள்:**
- *ஒரு உறுப்பினர் அடைவு (membership directory) க்கான சிறந்த plugin எது?*
- *WooCommerce abilities pack-ஐ நிறுவவும்.*
- *கிடைப்பிலுள்ள அனைத்து ecommerce ability packs-களையும் காட்டுங்கள்.*

### Support Assistant (ஆதரவு உதவியாளர்)

**கவனம்:** தள உள்ளடக்கம், அமைப்புகள் மற்றும் WordPress உள்ளமைவு பற்றிய கேள்விகளுக்குப் பதிலளித்தல்.

**கிடைப்பிலுள்ள கருவிகள்:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**சிறப்பாகச் செய்யும்வை:**
- தற்போதைய தள அமைப்புகள் மற்றும் விருப்பங்களை (settings and options) பார்ப்பது.
- தளத்தில் என்னென்ன post types, taxonomies மற்றும் menus உள்ளமைக்கப்பட்டுள்ளன என்பதை விளக்குவது.
- நேரடி மதிப்புகளைப் படித்து, "இந்த அமைப்பு என்ன செய்கிறது?" என்ற கேள்விகளுக்குப் பதிலளிப்பது.
- மாற்றங்களைச் செய்வதற்கு முன் படிக்க-மட்டும் (read-only) 진டக்னஸ்டிக் அடுக்காகச் செயல்படுவது.

**தொடக்க ஆலோசனைகள்:**
- *இந்த தளத்தில் தற்போது எந்த plugins மற்றும் அமைப்புகள் செயல்படுகின்றன?*
- *இந்த தளத்தில் பதிவுசெய்யப்பட்ட அனைத்து தனிப்பயன் post types-களையும் பட்டியலிடுங்கள்.*
- *என்னென்ன வழிசெலுத்தல் மெனுக்கள் (navigation menus) உள்ளன, அவை எங்கு ஒதுக்கப்பட்டுள்ளது?*

## ஏஜெண்டுகளைத் தனிப்பயனாக்குதல் (Customising Agents)

ஒவ்வொரு உள்ளமைக்கப்பட்ட ஏஜென்ட்டும் `gratis_ai_agent_agents` filter மூலம் நீட்டிக்கப்படலாம் அல்லது மாற்றப்படலாம்.

### ஒரு இருக்கும் ஏஜெண்டிற்கு தனிப்பயன் சிஸ்டம் ப்ராம்ட்டைச் சேர்ப்பது

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### ஒரு புதிய ஏஜெண்டைப் பதிவுசெய்தல்

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

புதிய ஏஜென்ட், filter இயங்கிய உடனேயே Agent Picker-ல் தோன்றும்.

### ஒரு உள்ளமைக்கப்பட்ட ஏஜெண்டைப் நீக்குதல்

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
