# பிற சொருகிகளுடன் அறியப்பட்ட இணக்கமின்மைகள்

Ultimate Multisite-ஐ நிறுவி செயல்படுத்திய பிறகு, பிற சொருகிகளுடன் உள்ள இணக்கமின்மை சிக்கல்கள் காரணமாக நீங்கள் பிரச்சனைகளை எதிர்கொள்ளலாம்.

ஒவ்வொரு புதிய வெளியீட்டிலும் இந்தப் பட்டியலைக் குறைக்க நாங்கள் தீவிரமாக பணியாற்றி வருகிறோம், ஆனால் ஒரு சொருகி இங்கே பட்டியலிடப்பட்டிருந்தால், முழுமையாக செயல்படும் Ultimate Multisite Network-ஐப் பெற அதை செயலிழக்கச் செய்ய வேண்டியிருக்கலாம்.

இதோ அவற்றின் பட்டியலும், இந்தச் சந்தர்ப்பங்களில் நீங்கள் என்ன செய்ய வேண்டும் என்பதும்:

**சில WPMU DEV சொருகிகள்** WPMU DEV-ஆல் _இனி பராமரிக்கப்படாத_ சில சொருகிகள், Ultimate Multisite பதிவு செயல்முறையை கடைசி படியில் நிறுத்துவதாக அறியப்படுகிறது (பயனரால் கணக்கை உருவாக்க முடியாது). இந்தப் பிரச்சனைகளை ஏற்படுத்தும் சொருகிகளின் பட்டியல் இதோ:

  * Signup Language: _Ultimate Multisite-ல் இதே செயல்பாட்டை வழங்கும் ஒரு add-on உள்ளது, அதன் பெயர் Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _இது Ultimate Multisite-ல் உள்ளமைக்கப்பட்டுள்ளது, எனவே கூடுதல் சொருகி தேவையில்லை._

  * New Blog Templates: _இது Ultimate Multisite-ல் உள்ளமைக்கப்பட்டுள்ளது, எனவே கூடுதல் சொருகி தேவையில்லை._

  * Set Password on Multisite Blog Creation: _இது Ultimate Multisite-ல் உள்ளமைக்கப்பட்டுள்ளது, எனவே கூடுதல் சொருகி தேவையில்லை._

  * Signup TOS: _இது Ultimate Multisite-ல் உள்ளமைக்கப்பட்டுள்ளது, எனவே கூடுதல் சொருகி தேவையில்லை._

  * Anti-Splog: குறைந்த சக்தி வாய்ந்ததாக இருந்தாலும், இதுவும் _Ultimate Multisite-ல் உள்ளமைக்கப்பட்டுள்ளது._

**குறிப்பு** : மற்ற அனைத்து WPMU Dev சொருகிகளும் Ultimate Multisite-உடன் சாதாரணமாகப் பயன்படுத்தப்படலாம். எடுத்துக்காட்டுகள்: _Smush_ , _Forminator_ , _Defender,_ போன்றவை.
