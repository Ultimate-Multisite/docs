# Ultimate Multisite சமூகத்தில் இணைவது எப்படி

Ultimate Multisite சமூகம் என்பது மற்ற Ultimate Multisite பயனர்களுடன் தொடர்பில் இருந்து, குறிப்புகள், அனுபவங்கள் மற்றும் பலவற்றைப் பகிர்ந்து கொள்ளக்கூடிய ஒரு இடம். Ultimate Multisite சமூகத்தை அணுக https://community.ultimatemultisite.com/home க்குச் செல்லுங்கள்.

வரவேற்புப் பக்கத்தில் "**creating your account here**" என்ற இணைப்பைக் கிளிக் செய்யுங்கள்.

<!-- Screenshot unavailable: Community welcome page with creating your account link -->

நீங்கள் எங்களது பதிவுப் பக்கத்திற்கு திருப்பி விடப்படுவீர்கள். இப்போது ஒரு பயனர்பெயர், உங்கள் மின்னஞ்சல் மற்றும் கடவுச்சொல்லை உள்ளிட வேண்டும். Captcha-வை நிறைவு செய்து **Register** பொத்தானைக் கிளிக் செய்யுங்கள்.

<!-- Screenshot unavailable: Community register page with username, email, password fields and Captcha -->

பதிவு செய்த பிறகு, உங்கள் சுயவிவரத்தை உருவாக்க வேண்டும். இந்தப் பக்கத்தில் உங்கள் புகைப்படத்தைப் பதிவேற்றலாம், முழுப் பெயரை உள்ளிடலாம், சிறிய அறிமுகக் குறிப்பு எழுதலாம் போன்றவை. மேலும், உங்கள் சமூக வலைதளங்களின் இணைப்புகளைச் சேர்த்து, உங்கள் கணக்கிற்கான அனுமதிகளை அமைக்கலாம்.

<!-- Screenshot unavailable: Community profile creation page with photo upload and bio fields -->

இப்போது நீங்கள் "Introductions" சேனலில் இருக்கிறீர்கள், இங்கே உங்களை அறிமுகப்படுத்திக்கொள்ளலாம், மற்ற உறுப்பினர்களைப் பற்றி கொஞ்சம் தெரிந்துகொள்ளலாம். **New Post** பொத்தானைக் கிளிக் செய்து உங்கள் பதிவை உருவாக்கலாம்.

<!-- Screenshot unavailable: Community Introductions channel with New Post button -->

Lounge பிரிவில் உள்ள சேனல்களும் "Dev Logs" சேனலும் அனைவருக்கும் திறந்திருக்கும். ஆனால் சமூகத்தில் உள்ள மற்ற சேனல்களை அணுக உங்களுக்கு சிறப்பு அனுமதி தேவைப்படும்.

<!-- Screenshot unavailable: Community channels showing Lounge section and Dev Logs channel -->

உங்களிடம் செயலில் உள்ள Ultimate Multisite license key இருப்பதை உறுதிப்படுத்தியவுடன் இந்த அணுகல் வழங்கப்படும். இதைச் செய்ய, ஏதேனும் ஒரு private channel இணைப்பைக் கிளிக் செய்து, பின்னர் **Unlock** பொத்தானைக் கிளிக் செய்யுங்கள்.

<!-- Screenshot unavailable: Community private channel with Unlock button -->

நீங்கள் எங்களது unlock access பக்கத்திற்கு திருப்பி விடப்படுவீர்கள். உங்கள் Ultimate Multisite license key-ஐ உள்ளிட்டு Unlock Access பொத்தானைக் கிளிக் செய்யுங்கள்:

<!-- Screenshot unavailable: Community unlock access page with license key input -->

உங்கள் license சரிபார்க்கப்பட்டு, private channels-ஐ அணுக முடியும். இந்தச் செயல்முறை 24 மணி நேரம் வரை ஆகலாம், எனவே சமூகத்திற்குத் திரும்பிச் செல்ல இணைப்பைக் கிளிக் செய்து, உங்கள் கோரிக்கையை நாங்கள் செயல்படுத்தும் வரை மற்ற இடங்களை அனுபவியுங்கள்.
