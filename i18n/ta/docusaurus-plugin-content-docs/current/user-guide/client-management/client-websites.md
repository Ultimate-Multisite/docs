# உங்கள் வாடிக்கையாளரின் வலைத்தளங்கள் (v2)

உங்கள் நெட்வொர்க்கில் பதிவு செய்த பிறகு, உங்கள் வாடிக்கையாளர்கள் தங்கள் dashboard-ஐ அணுகலாம்.

**Account** பக்கத்தைத் தவிர — அங்கு அவர்கள் தங்கள் கட்டணங்கள், உறுப்பினர்கள், domains, திட்ட வரம்புகள் மற்றும் பிற முக்கியமான தகவல்களைப் பார்க்கலாம் — அவர்கள் **My Sites** பக்கத்தையும் அணுகலாம். அங்கு அவர்கள் தங்கள் அனைத்து வலைத்தளங்களையும் பார்க்கவும் நிர்வகிக்கவும் முடியும். அவர்கள் செய்ய வேண்டியது எல்லாம் தங்கள் dashboard-இல் My Sites என்பதைக் கிளிக் செய்வது மட்டுமே.

![வாடிக்கையாளர் dashboard-இல் My Sites இணைப்பு](/img/admin/customers-list.png)

அவர்களுக்கு அனைத்து வலைத்தளங்களும் காட்டப்படும், ஒவ்வொன்றின் கீழும் **Manage** பொத்தான் இருக்கும். அதைக் கிளிக் செய்தால், அவர்கள் **அந்த குறிப்பிட்ட வலைத்தளத்தின் dashboard**-க்கு திருப்பி விடப்படுவார்கள்.

ஒன்றுக்கு மேற்பட்ட வலைத்தளங்களை அனுமதிக்கும் திட்டங்களில், தங்கள் வலைத்தளங்களில் எது **முதன்மை** வலைத்தளம் என்பதையும் அவர்கள் பார்க்கலாம்.

![வாடிக்கையாளர் வலைத்தளங்களை Manage பொத்தான்களுடன் காட்டும் My Sites பக்கம்](/img/admin/customers-list.png)

தங்கள் முதன்மை வலைத்தளத்தை மாற்ற விரும்பினால், **Account** மெனுவைக் கிளிக் செய்து Account பக்கத்திற்குச் செல்ல வேண்டும்.

![வாடிக்கையாளர் dashboard-இல் Account மெனு இணைப்பு](/img/admin/customers-list.png)

பிறகு, **Change Default Site** என்பதைக் கிளிக் செய்ய வேண்டும்.

![Account பக்கத்தில் Change Default Site பொத்தான்](/img/admin/customers-list.png)

ஒரு pop-up சாளரம் தோன்றும். drop-down பட்டியலைக் கிளிக் செய்து, எந்த வலைத்தளத்தை முதன்மை வலைத்தளமாக மாற்ற விரும்புகிறீர்கள் என்பதைத் தேர்ந்தெடுக்கவும்.

![Change Default Site dropdown தேர்வு](/img/admin/customers-list.png)

பிறகு, **Change Default Site** என்பதைக் கிளிக் செய்தால் முடிந்தது!

![Default site மாற்றத்தை உறுதிப்படுத்துதல்](/img/admin/customers-list.png)
