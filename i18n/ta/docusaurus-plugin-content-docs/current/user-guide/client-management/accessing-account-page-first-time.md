# எனது கணக்கு பக்கத்தை முதல் முறையாக எப்படி அணுகுவது?

Ultimate Multisite-க்கு வரவேற்கிறோம்!

எங்கள் plugin-ஐ வாங்கிய பிறகு உங்கள் மின்னஞ்சலில் பெற்ற இணைப்பு வழியாக உங்கள் கணக்கு பக்கத்தை அணுகலாம்:

<!-- Screenshot unavailable: Account page email with access link -->

வாங்கும்போது நீங்கள் கொடுத்த மின்னஞ்சல் மற்றும் உங்கள் மின்னஞ்சலில் பெற்ற கடவுச்சொல்லைப் பயன்படுத்தவும்.

<!-- Screenshot unavailable: Login page with email and password fields -->

இந்தக் கணக்கில் நீங்கள் முதல் முறையாக உள்நுழைவதால், உங்கள் கடவுச்சொல்லை மாற்றும்படி கேட்கப்படுவீர்கள். அதை மாற்றி _Save_ பொத்தானைக் கிளிக் செய்யவும்.

<!-- Screenshot unavailable: Change password form with Save button -->

உங்கள் கணக்கு பக்கத்தில், பில்லிங், புதுப்பித்தல்கள், ரத்துகள் மற்றும் பல விஷயங்கள் போன்ற அனைத்து கணக்கு விவரங்களையும் நிர்வகிக்கலாம்.
