# எனது சந்தாவை எவ்வாறு ரத்து செய்வது?

நீங்கள் Ultimate Multisite license வைத்திருந்தால், எப்போது வேண்டுமானாலும் அதன் புதுப்பிப்பை ரத்து செய்யலாம். கீழே உள்ள படிகளைப் பின்பற்றுங்கள்:


<!-- Screenshot unavailable: Account page email with access link -->

வாங்கும்போது நீங்கள் கொடுத்த மின்னஞ்சல் மற்றும் கடவுச்சொல்லைப் பயன்படுத்துங்கள்.

<!-- Screenshot unavailable: Login page with email and password fields -->

இது உங்கள் Account Page தொடக்கத் திரை:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

உங்கள் சந்தாவை ரத்து செய்ய, இடது பக்க மெனுவில் _Renewing & Billing_ என்பதற்குச் செல்லுங்கள்.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

ஒரு பக்கவாட்டு சாளரத்தைத் திறக்க வலது புறத்தில் உள்ள அம்புக்குறியைக் கிளிக் செய்யுங்கள். பின்னர், **auto-renew** ஐ ரத்து செய்யும் விருப்பத்தைத் தேர்ந்தெடுக்க வேண்டும்.

கணினி உங்களுக்கு ஒரு உறுதிப்படுத்தல் செய்தியைக் காண்பிக்கும்.

நீங்கள் சந்தாவை ரத்து செய்ய விரும்புவது உறுதியானால், _**Cancel Renewals**_ பொத்தானைக் கிளிக் செய்யுங்கள். இந்தச் செயலுக்குப் பிறகு, ஒரு சிறிய கருத்துக்கணிப்புக்கு பதிலளிக்கும்படி கேட்கப்படுவீர்கள்.

முடிந்தது! உங்கள் சந்தா தானாகப் புதுப்பிக்கப்படாது.

**உங்கள் சந்தா காலாவதியாகும் வரை செல்லுபடியாகும் key உங்களிடம் இருக்கும்**. உங்கள் சந்தாவை மீண்டும் செயல்படுத்த விரும்பினால், அதை நீங்கள் கைமுறையாகச் செய்ய வேண்டும்.
