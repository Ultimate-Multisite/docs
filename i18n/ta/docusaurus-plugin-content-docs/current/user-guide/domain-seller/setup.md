# டொமைன் விற்பனையாளர்: அமைப்பு மற்றும் வழங்குநர் உள்ளமைவு

டொமைன் விற்பனையாளர் addon, தேவையான ஒவ்வொரு படியிலும் உங்களை வழிநடத்தும் அமைப்பு wizard உடன் வருகிறது. இந்தப் பக்கம் wizard ஓட்டத்தையும், அதன் பிறகு வழங்குநர்களை எப்படி உள்ளமைப்பது அல்லது மீண்டும் உள்ளமைப்பது என்பதையும் விளக்குகிறது.

## தேவைகள் {#requirements}

- **Multisite Ultimate** v2.4.12 அல்லது அதற்கு மேல், நெட்வொர்க்-செயல்படுத்தப்பட்டது
- **PHP** 7.4+
- ஆதரிக்கப்படும் குறைந்தது ஒரு பதிவாளருக்கான API சான்றுகள்

## முதல் இயக்க அமைப்பு wizard {#first-run-setup-wizard}

நீங்கள் plugin-ஐ முதல் முறை நெட்வொர்க்-செயல்படுத்தும் போது அமைப்பு wizard தானாகத் தொடங்கும். இது எப்போது வேண்டுமானாலும் **நெட்வொர்க் நிர்வாகம் › Ultimate Multisite › டொமைன் விற்பனையாளர் அமைப்பு** இலிருந்தும் கிடைக்கும்.

### படி 1 — வழங்குநரைத் தேர்வு செய்யவும் {#step-1--choose-a-provider}

நீங்கள் இணைக்க விரும்பும் பதிவாளரைத் தேர்ந்தெடுக்கவும். ஆதரிக்கப்படும் விருப்பங்கள்:

| வழங்குநர் | DNS மேலாண்மை | WHOIS தனியுரிமை |
|---|---|---|
| OpenSRS | ஆம் | ஆம் |
| Namecheap | இல்லை | ஆம் (WhoisGuard, இலவசம்) |
| HostAfrica | ஆம் | ஆம் (ID பாதுகாப்பு) |
| Openprovider | ஆம் | ஆம் |
| Hostinger | ஹோஸ்ட் செய்யப்பட்ட டொமைன்களுக்கு மைய Hostinger டொமைன் மேப்பிங் மூலம் | ஆம் |
| GoDaddy | இல்லை | இல்லை |
| ResellerClub | ஆம் | இல்லை |
| NameSilo | இல்லை | இல்லை |
| Enom | ஆம் | இல்லை |

### படி 2 — சான்றுகளை உள்ளிடவும் {#step-2--enter-credentials}

ஒவ்வொரு வழங்குநருக்கும் வேறுபட்ட சான்று புலங்கள் உள்ளன:

**OpenSRS** — பயனர்பெயர் மற்றும் தனியார் விசை (OpenSRS Reseller Control Panel இலிருந்து)

**Namecheap** — பயனர்பெயர் மற்றும் API விசை (Account › Tools › API Access இலிருந்து)

**HostAfrica** — HostAfrica மறுவிற்பனையாளர் module-இலிருந்து Domains Reseller API endpoint மற்றும் சான்றுகள். தனியான sandbox endpoint தற்போது ஆவணப்படுத்தப்படவில்லை; நேரடி பதிவுகளை இயக்குவதற்கு முன் பாதுகாப்பான வாசிப்பு-மட்டும் சோதனைகளுடன் சோதிக்கவும்.

**Openprovider** — API அணுகல் செயல்படுத்தப்பட்ட பயனர்பெயர் மற்றும் கடவுச்சொல். விருப்ப sandbox mode Openprovider sandbox API-ஐப் பயன்படுத்துகிறது, மேலும் விருப்ப இயல்புநிலை வாடிக்கையாளர் handle-ஐ பதிவுகளுக்கு மீண்டும் பயன்படுத்தலாம்.

**Hostinger** — மைய Hostinger ஒருங்கிணைப்பிலிருந்து பகிரப்பட்ட Hostinger hPanel API token. அதே token மைய டொமைன் மேப்பிங் மற்றும் டொமைன் விற்பனையாளர் பதிவு செயல்பாடுகளையும் இயக்குகிறது.

**GoDaddy** — API விசை மற்றும் ரகசியம் (developer.godaddy.com இலிருந்து)

**ResellerClub** — மறுவிற்பனையாளர் ID மற்றும் API விசை (ResellerClub control panel இலிருந்து)

**NameSilo** — API விசை (namesilo.com › Account › API Manager இலிருந்து)

**Enom** — Account ID மற்றும் API token

நேரலையில் செல்லுவதற்கு முன் வழங்குநரின் சோதனை சூழலில் சோதிக்க, கிடைக்கும் இடங்களில் **Sandbox mode**-ஐத் தேர்வுசெய்யவும்.

### படி 3 — இணைப்பைச் சோதிக்கவும் {#step-3--test-the-connection}

**இணைப்பைச் சோதிக்கவும்** என்பதைக் கிளிக் செய்யவும். சான்றுகள் மற்றும் இணைப்புத்திறனைச் சரிபார்க்க wizard இலகுவான API அழைப்பை அனுப்பும். தொடர்வதற்கு முன் சான்று சிக்கல்களைச் சரிசெய்யவும்.

### படி 4 — TLDகளை இறக்குமதி செய்யவும் {#step-4--import-tlds}

இணைக்கப்பட்ட வழங்குநரிலிருந்து கிடைக்கும் அனைத்து TLDகளையும் மொத்த விலையையும் பெற **TLDகளை இறக்குமதி செய்யவும்** என்பதைக் கிளிக் செய்யவும். இது டொமைன் தயாரிப்புகள் பயன்படுத்தும் TLD பட்டியலை நிரப்பும். பெரிய TLD பட்டியல்கள் உள்ள வழங்குநர்களுக்கு இறக்குமதி 30–60 விநாடிகள் ஆகலாம்.

திட்டமிடப்பட்ட cron பணியின் மூலம் TLDகள் தினமும் ஒருமுறை தானாகவும் மீண்டும் ஒத்திசைக்கப்படும்.

### படி 5 — டொமைன் தயாரிப்பை உருவாக்கவும் {#step-5--create-a-domain-product}

wizard, 10% markup உடன் இயல்புநிலை catch-all டொமைன் தயாரிப்பை உருவாக்கும். இந்த தயாரிப்பை உடனே திருத்தலாம் அல்லது தவிர்த்து, **Ultimate Multisite › தயாரிப்புகள்** கீழ் தயாரிப்புகளை கைமுறையாக உருவாக்கலாம்.

முழு தயாரிப்பு உள்ளமைவு வழிகாட்டிக்கு [டொமைன் தயாரிப்புகள் மற்றும் விலை நிர்ணயம்](./domain-products) பார்க்கவும்.

## வழங்குநரை மீண்டும் உள்ளமைத்தல் {#reconfiguring-a-provider}

**நெட்வொர்க் நிர்வாகம் › Ultimate Multisite › அமைப்புகள் › டொமைன் விற்பனையாளர்** செல்லவும் (அல்லது plugin பட்டியலில் **அமைப்புகள்** என்பதைக் கிளிக் செய்யவும்).

அமைப்புகள் பக்கத்தில் உள்ளவை:

- **டொமைன் விற்பனையை செயல்படுத்து** — முழு அம்சத்தையும் இயக்க/முடக்க மாற்றவும்
- **இயல்புநிலை வழங்குநர்** — டொமைன் தேடல்கள் மற்றும் புதிய தயாரிப்புகளுக்கு பயன்படுத்தப்படும் வழங்குநர்
- **ஒரு தேடலுக்கான அதிகபட்ச TLDகள்** — வாடிக்கையாளர் தேடும் போது எத்தனை TLDகளைச் சரிபார்க்க வேண்டும்; அதிக மதிப்புகள் மேலும் பல விருப்பங்களைக் காட்டும், ஆனால் மெதுவாக இருக்கும்
- **கிடைப்புத் தகவல் cache காலம்** — கிடைப்புத் தகவல் மற்றும் விலை முடிவுகளை எவ்வளவு நேரம் cache செய்ய வேண்டும்; குறைந்த மதிப்புகள் அதிக துல்லியமானவை, ஆனால் API அழைப்புகளை அதிகரிக்கும்
- **டொமைன் தயாரிப்புகளை நிர்வகிக்கவும்** — தயாரிப்புகள் பட்டியலுக்கான விரைவு இணைப்பு
- **வழங்குநர்களை உள்ளமைக்கவும்** — வழங்குநர்களைச் சேர்க்க அல்லது மீண்டும் உள்ளமைக்க Integration Wizard-ஐத் திறக்கும்

### இரண்டாவது வழங்குநரைச் சேர்த்தல் {#adding-a-second-provider}

**வழங்குநர்களை உள்ளமைக்கவும்** என்பதைக் கிளிக் செய்து, புதிய பதிவாளருக்காக wizard-ஐ மீண்டும் இயக்கவும். ஒரே நேரத்தில் பல வழங்குநர்களை உள்ளமைத்திருக்கலாம். ஒவ்வொரு டொமைன் தயாரிப்பையும் ஒரு குறிப்பிட்ட வழங்குநருக்கு ஒதுக்கவும், அல்லது அதை இயல்புநிலையில் விடவும்.

### TLDகளை கைமுறையாக ஒத்திசைத்தல் {#syncing-tlds-manually}

அமைப்புகள் பக்கத்தில், புதிய விலையைப் பெற உள்ளமைக்கப்பட்ட எந்த வழங்குநருக்கும் அடுத்துள்ள **TLDகளை ஒத்திசைக்கவும்** என்பதைக் கிளிக் செய்யவும். வழங்குநர் மொத்த விலையைப் புதுப்பித்த பிறகு அல்லது புதிய TLDகளைச் சேர்த்த பிறகு இது பயனுள்ளதாக இருக்கும்.

## வழங்குநர் திறன் குறிப்புகள் {#provider-capability-notes}

ஒவ்வொரு பதிவாளர் API-யும் ஒரே செயல்பாடுகளை வழங்காது. addon, ஆதரிக்கப்படாத செயல்பாடுகளை அமைதியாக தோல்வியடைய விடாமல், நிர்வாகிகளுக்கு தெளிவான பிழைகளாகக் காட்டும்.

- **HostAfrica** தேடல், TLD/விலை ஒத்திசைவு, பதிவு, புதுப்பிப்பு, பரிமாற்றம், nameserver புதுப்பிப்புகள், DNS பதிவுகள், EPP codes, registrar lock, மற்றும் ID பாதுகாப்பு உட்பட மிக விரிவான நேரடி reseller பணிச்சரத்தை ஆதரிக்கிறது.
- **Openprovider** reseller விலையிலான TLD ஒத்திசைவு, பதிவு, புதுப்பிப்பு, பரிமாற்றங்கள், nameserver புதுப்பிப்புகள், DNS மண்டலங்கள், EPP codes, registrar lock, மற்றும் WHOIS தனியுரிமையை ஆதரிக்கிறது. addon தானாகப் புதுப்பிக்கும் குறுகிய கால bearer token மூலம் இது அங்கீகரிக்கிறது.
- **Hostinger** பகிரப்பட்ட hPanel API token மூலம் கிடைப்புத் தேடல், பதிவு, தொகுப்பு தேடல், nameserver புதுப்பிப்புகள், registrar lock, மற்றும் WHOIS தனியுரிமையை ஆதரிக்கிறது. Hostinger-ன் பொது Domains API reseller/wholesale விலையிடல், உள்வரும் பரிமாற்றம், வெளிப்படையான புதுப்பிப்பு, அல்லது EPP-code பெறுதலை வழங்காது; புதுப்பிப்புகள் auto-renew மட்டும்.
