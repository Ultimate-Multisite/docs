# Ultimate Multisite மொழிபெயர்ப்பு

Ultimate Multisite ஆங்கிலத்தில் உருவாக்கப்பட்டுள்ளது, ஆனால் இந்த plugin முழுமையாக மொழிபெயர்க்கக்கூடியது, மேலும் மொழிபெயர்ப்பு பங்களிப்புகளை நாங்கள் மகிழ்ச்சியுடன் வரவேற்கிறோம்.

## முக்கிய Plugin மொழிபெயர்ப்புகள்

முக்கிய Ultimate Multisite plugin அதிகாரப்பூர்வ **WordPress.org மொழிபெயர்ப்பு தளம்** மூலம் மொழிபெயர்க்கப்படுகிறது:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

இது நிலையான WordPress சமூக மொழிபெயர்ப்பு அமைப்பு. மொழிபெயர்ப்புகள் போதுமான அங்கீகார வரம்பை அடையும்போது, அவை WordPress plugin புதுப்பிப்புகள் வழியாக அனைத்து Ultimate Multisite பயனர்களுக்கும் தானாகவே விநியோகிக்கப்படும்.

**உங்கள் உதவி தேவை!** நீங்கள் ஆங்கிலம் தவிர வேறு மொழி பேசினால், மொழிபெயர்ப்புகளில் பங்களிக்க பரிசீலியுங்கள். சில சொற்றொடர்களை மொழிபெயர்த்தாலும் கூட உதவியாக இருக்கும். தொடங்குவதற்கு:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) பக்கத்திற்குச் செல்லுங்கள்
2. உங்கள் மொழியைத் தேர்ந்தெடுங்கள்
3. "Contribute Translation" என்பதைக் கிளிக் செய்யுங்கள்
4. இலவச WordPress.org கணக்கு தேவை

உங்கள் மொழிக்கான மொழிபெயர்ப்பு ஆசிரியராக (மொழிபெயர்ப்புகளை அங்கீகரிக்கும் திறன்) ஆக விரும்பினால், [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) மூலம் தொடர்பு கொள்ளுங்கள்.

## Addon மொழிபெயர்ப்புகள்

Addon plugins தனித்தனியாக இங்கே மொழிபெயர்க்கப்படுகின்றன:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon மொழிபெயர்ப்புகள் தற்போது AI மூலம் ஆரம்ப நிலையாக உருவாக்கப்பட்டுள்ளன. அவை ஆதரிக்கப்படும் அனைத்து மொழிகளையும் உள்ளடக்கியவை, ஆனால் இயல்பற்ற சொற்றொடர்கள் அல்லது துல்லியமின்மை இருக்கலாம். அவற்றை இயல்பானதாகவும் துல்லியமானதாகவும் மாற்ற மனித மொழிபெயர்ப்பாளர்கள் மதிப்பாய்வு செய்து மேம்பாடுகளை பரிந்துரைப்பதை வரவேற்கிறோம்.

## உள்ளூர் மொழிபெயர்ப்புகள்

முதன்மை திட்டத்திற்கு பங்களிப்பதற்குப் பதிலாக உங்கள் சொந்த திட்டத்திற்காக உள்ளூரில் மொழிபெயர்ப்புகளை உருவாக்க விரும்பினால், இரண்டு வழிகள் உள்ளன:

### Poedit பயன்படுத்துதல்

`.pot` கோப்பை மொழிபெயர்க்க **[Poedit](https://poedit.net)** என்ற desktop பயன்பாட்டைப் பயன்படுத்தலாம்:

1. [poedit.net](https://poedit.net) இலிருந்து Poedit பதிவிறக்கம் செய்யுங்கள்
2. `wp-content/plugins/ultimate-multisite/lang/` இலிருந்து `.pot` கோப்பைத் திறக்கவும்
3. Poedit இடைமுகத்தில் சொற்றொடர்களை மொழிபெயர்க்கவும்
4. சேமிக்கவும் -- Poedit `.po` மற்றும் `.mo` கோப்புகளை உருவாக்கும்
5. வெளியீட்டு கோப்புகளை `lang` கோப்புறையில் வைக்கவும்

### Loco Translate பயன்படுத்துதல்

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin மூலம் எந்த desktop கருவிகளும் இல்லாமல் நேரடியாக admin panel இலிருந்து மொழிபெயர்க்கலாம். இது மொழிபெயர்ப்பு கோப்புகளை உருவாக்குவதையும் தொகுப்பதையும் உங்களுக்காகக் கையாளுகிறது.
