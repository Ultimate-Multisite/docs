# Manual Payments அமைத்தல் (v2)

_**முக்கிய குறிப்பு: இந்தக் கட்டுரை Ultimate Multisite பதிப்பு 2.x-ஐக் குறிக்கிறது.**_

**Stripe** அல்லது **PayPal** உங்கள் பயனர்களுக்கு கிடைக்காதபோது, Manual payments என்பது அவர்களுக்கு பிற கட்டண முறைகளை வழங்கும் ஒரு வழியாகும். அது வயர் அல்லது வங்கி பரிமாற்றமாக இருக்கலாம், அல்லது உங்கள் பயனர்களுக்கு உள்ளூரில் கிடைக்கும் வேறு ஏதேனும் கட்டண முறையாக இருக்கலாம்.

## Manual Payments-ஐ எவ்வாறு இயக்குவது {#how-to-enable-manual-payments}

Manual payment அமைப்பது மிகவும் எளிது. கட்டண நுழைவாயில்களின் கீழ் அதை இயக்கி, பயனர் கட்டணத்தை எவ்வாறு அனுப்ப வேண்டும் என்பதற்கான விரிவான வழிமுறைகளைச் சேர்க்க வேண்டும்.

முதலில், **Ultimate Multisite > Settings > Payments**-க்கு செல்லவும். **Payment Gateways**-ன் கீழ், **Manual**-ஐ இயக்கவும். உங்களுக்காக **Payment Instructions** பெட்டி தோன்றும்.

உங்கள் வாடிக்கையாளர் கட்டணம் செலுத்தத் தேவையான தகவலை இந்தப் பெட்டியில் சேர்க்கவும். எடுத்துக்காட்டாக, அது உங்கள் வங்கி கணக்கு விவரங்களும், வாடிக்கையாளர் கட்டண உறுதிப்படுத்தலை உங்களுக்கு அனுப்ப உங்கள் மின்னஞ்சலும் இருக்கலாம்.

![Payment Instructions உரைப்பகுதியுடன் Manual payment நுழைவாயில் மாற்று](/img/config/manual-gateway-expanded.png)

Manual gateway அமைப்புகள் இடைமுகம் இதோ:

![Manual gateway அமைப்புகள்](/img/config/manual-gateway-settings.png)

அமைத்த பிறகு, **Save Settings**-ஐ கிளிக் செய்தால் போதும்; அது முடிந்துவிடும். பயனர்கள் உங்கள் நெட்வொர்க்கில் பதிவு செய்யும்போது, கொள்முதலை முடிக்க உங்கள் வழிமுறைகளைப் பெறுவார்கள் என்று தெரிவிக்கும் செய்தியைப் பார்ப்பார்கள்.

![பயனருக்கு கட்டண வழிமுறைகள் கிடைக்கும் என்று தெரிவிக்கும் பதிவு உறுதிப்படுத்தல் செய்தி](/img/frontend/registration-manual-notice.png)

மேலும் அவர்கள் உங்கள் **Thank You** பக்கத்திலும் உங்கள் கட்டண வழிமுறைகளுடன் ஒரு செய்தியைப் பெறுவார்கள்.

## Manual payments-ஐ உறுதிப்படுத்துதல் {#confirming-manual-payments}

Manual payment-ஐ உறுதிப்படுத்த, இடது பட்டியில் உள்ள **Payments** மெனுவிற்கு செல்லவும். அங்கு உங்கள் நெட்வொர்க்கில் உள்ள அனைத்து கட்டணங்களையும், அவற்றின் **status** உட்பட விவரங்களையும் காணலாம். நீங்கள் கைமுறையாக மாற்றும் வரை Manual payment எப்போதும் **Pending** நிலையை கொண்டிருக்கும்.

![நிலுவையில் உள்ள Manual payment-ஐக் காட்டும் Payments பட்டியல்](/img/admin/payments-list.png)

**reference code**-ஐ கிளிக் செய்து கட்டணப் பக்கத்திற்குள் செல்லவும். இந்தப் பக்கத்தில் reference ID, தயாரிப்புகள், நேர முத்திரைகள் மற்றும் பல போன்ற நிலுவையில் உள்ள கட்டணத்தின் அனைத்து விவரங்களும் உள்ளன.

![reference code, தயாரிப்புகள் மற்றும் மொத்தங்களைக் காட்டும் கட்டண விவரங்கள் பக்கம்](/img/admin/payment-edit.png)

வலது நெடுவரிசையில், கட்டணத்தின் நிலையை மாற்றலாம். அதை **Completed** ஆக மாற்றி, **Activate Membership** விருப்பத்தை இயக்கினால், உங்கள் வாடிக்கையாளரின் தளம் இயக்கப்படும்; அவர்களின் உறுப்பினர்தன்மை செயலில் இருக்கும்.

![Status Completed ஆக அமைக்கப்பட்டும் Activate Membership மாற்று உள்ள கட்டணத் திருத்தப் பக்கம்](/img/admin/payment-activate-membership.png)
