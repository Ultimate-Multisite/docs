# கைமுறை கொடுப்பனவுகளை அமைத்தல் (v2)

_**முக்கிய குறிப்பு: இந்தக் கட்டுரை Ultimate Multisite பதிப்பு 2.x ஐக் குறிக்கிறது.**_

உங்கள் பயனர்களுக்கு **Stripe** அல்லது **PayPal** கிடைக்காத சூழலில், மற்ற கொடுப்பனவு முறைகளை வழங்க கைமுறை கொடுப்பனவுகள் உதவுகின்றன. இது வங்கிப் பரிமாற்றம் அல்லது உங்கள் பயனர்களுக்கு உள்ளூரில் கிடைக்கும் வேறு எந்த கொடுப்பனவு முறையாகவும் இருக்கலாம்.

## கைமுறை கொடுப்பனவுகளை எவ்வாறு இயக்குவது

கைமுறை கொடுப்பனவை அமைப்பது மிக எளிது. Payment gateways பகுதியில் இதை இயக்கி, பயனர் எவ்வாறு பணம் அனுப்ப வேண்டும் என்பதற்கான விரிவான வழிமுறைகளை எழுத வேண்டும்.

முதலில், **Ultimate Multisite > Settings > Payments** என்பதற்குச் செல்லுங்கள். **Payment Gateways** பகுதியின் கீழ், **Manual** என்பதை இயக்குங்கள். இப்போது **Payment Instructions** பெட்டி தோன்றும்.

இந்தப் பெட்டியில் உங்கள் வாடிக்கையாளர் பணம் செலுத்த தேவையான தகவல்களைச் சேர்க்கவும். உதாரணமாக, உங்கள் வங்கிக் கணக்கு விவரங்கள் மற்றும் மின்னஞ்சல் முகவரியை இங்கே கொடுக்கலாம், இதன் மூலம் வாடிக்கையாளர் பணம் அனுப்பிய உறுதிப்படுத்தலை உங்களுக்கு அனுப்ப முடியும்.

![Manual payment gateway toggle and payment instructions box](/img/config/settings-payment-gateways.png)

அமைப்புகளை முடித்த பிறகு, **Save Settings** என்பதைக் கிளிக் செய்யுங்கள், அவ்வளவுதான். பயனர்கள் உங்கள் நெட்வொர்க்கில் பதிவு செய்யும்போது, கொள்முதலை முடிக்க உங்கள் வழிமுறைகளைப் பெறுவார்கள் என்ற செய்தியைக் காண்பார்கள்.

![Manual payment message shown during registration](/img/config/settings-payment-gateways.png)

மேலும், **Thank You** பக்கத்திலும் உங்கள் கொடுப்பனவு வழிமுறைகளுடன் ஒரு செய்தியைப் பெறுவார்கள்.

![Thank You page showing payment instructions](/img/config/settings-payment-gateways.png)

## கைமுறை கொடுப்பனவுகளை உறுதிப்படுத்துதல்

கைமுறை கொடுப்பனவை உறுதிப்படுத்த, இடது பக்கப்பட்டியில் உள்ள **Payments** மெனுவிற்குச் செல்லுங்கள். அங்கு உங்கள் நெட்வொர்க்கிலுள்ள அனைத்து கொடுப்பனவுகளையும் அவற்றின் **status** உள்ளிட்ட விவரங்களையும் காணலாம். நீங்கள் கைமுறையாக மாற்றும் வரை, கைமுறை கொடுப்பனவு எப்போதும் **Pending** நிலையில் இருக்கும்.

![Payments list showing pending manual payment](/img/admin/payments-list.png)

**reference code** என்பதைக் கிளிக் செய்து கொடுப்பனவு பக்கத்திற்குள் நுழையுங்கள். இந்தப் பக்கத்தில் நிலுவையிலுள்ள கொடுப்பனவின் அனைத்து விவரங்களையும் காணலாம் - reference ID, தயாரிப்புகள், நேர முத்திரைகள் மற்றும் பல.

![Payment details page with reference code and products](/img/admin/payments-list.png)

வலது நெடுவரிசையில், கொடுப்பனவின் நிலையை மாற்றலாம். இதை **Completed** என்று மாற்றி, **Activate Membership** விருப்பத்தை இயக்கினால், உங்கள் வாடிக்கையாளரின் தளம் செயல்படும் மற்றும் அவர்களின் உறுப்பினர் நிலை செயலில் இருக்கும்.

![Payment status change to Completed with Activate Membership toggle](/img/admin/payments-list.png)
