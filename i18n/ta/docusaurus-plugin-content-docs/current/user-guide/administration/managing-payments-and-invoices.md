# பணம் செலுத்துதல் மற்றும் விலைப்பட்டியல் நிர்வகித்தல்

## கட்டண அமைப்புகள்

பணம் பெறத் தொடங்குவதற்கு முன், கட்டணம் தொடர்பான அமைப்புகளை நீங்கள் கட்டமைக்க வேண்டும். **Ultimate Multisite → Settings** என்பதற்குச் சென்று **Payment** தாவலைக் கிளிக் செய்யுங்கள்.

![Payment settings tab](/img/admin/settings-payments-top.png)

### பொதுவான கட்டண விருப்பங்கள்

பொது அமைப்புகளில் நீங்கள் கட்டமைக்கக்கூடியவை:

- **Currency** — பரிவர்த்தனைகளுக்கு பயன்படுத்தப்படும் இயல்புநிலை நாணயம்
- **Currency Position** — நாணய குறியீடு எங்கு தோன்றும் (தொகைக்கு முன்/பின்)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### கட்டண நுழைவாயில்கள்

Ultimate Multisite பல கட்டண நுழைவாயில்களை ஆதரிக்கிறது. Payment அமைப்புகள் தாவலில் இருந்து ஒவ்வொன்றையும் இயக்கி கட்டமைக்கலாம்.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

கிடைக்கும் நுழைவாயில்கள்:

- **Stripe** — Stripe மூலம் கிரெடிட் கார்டு கட்டணங்கள்
- **PayPal** — PayPal கட்டணங்கள்
- **Manual** — ஆஃப்லைன் அல்லது தனிப்பயன் கட்டண செயலாக்கத்திற்கு

ஒவ்வொரு நுழைவாயிலுக்கும் அதன் சொந்த கட்டமைப்பு பகுதி உள்ளது, அங்கு API விசைகள் மற்றும் பிற அமைப்புகளை உள்ளிடலாம்.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### சாண்ட்பாக்ஸ் முறை

நேரடியாகப் பயன்படுத்துவதற்கு முன் உங்கள் கட்டண ஒருங்கிணைப்பைச் சோதிக்க **Sandbox Mode**-ஐ இயக்கலாம். சாண்ட்பாக்ஸ் முறை செயலில் இருக்கும்போது, உண்மையான கட்டணங்கள் எதுவும் வசூலிக்கப்படாது.

## கட்டணங்களைப் பார்த்தல்

உங்கள் நெட்வொர்க் முழுவதும் உள்ள அனைத்து பரிவர்த்தனைகளையும் பார்க்க Ultimate Multisite கீழ் உள்ள **Payments** பக்கத்திற்குச் செல்லுங்கள்.

![Payments list](/img/admin/payments-list.png)

நிலை (completed, pending, failed, refunded) அடிப்படையில் கட்டணங்களை வடிகட்டலாம் மற்றும் குறிப்பிட்ட பரிவர்த்தனைகளைத் தேடலாம்.

ஒரு கட்டணத்தின் முழு விவரங்களைப் பார்க்க அதைக் கிளிக் செய்யுங்கள் - இதில் வரி உருப்படிகள், தொடர்புடைய உறுப்பினர் சந்தா, வாடிக்கையாளர் தகவல்கள் மற்றும் கட்டண நுழைவாயில் தரவு ஆகியவை அடங்கும்.

## விலைப்பட்டியல்கள்

Ultimate Multisite கட்டணங்களுக்கான விலைப்பட்டியல்களை தானாகவே உருவாக்க முடியும். Payment அமைப்புகளில் இருந்து விலைப்பட்டியல் வார்ப்புரு மற்றும் எண்ணிடும் வடிவமைப்பைத் தனிப்பயனாக்கலாம்.

விலைப்பட்டியல் தனிப்பயனாக்க விருப்பங்கள்:

- விலைப்பட்டியல்களில் காட்டப்படும் **நிறுவனத்தின் பெயர் மற்றும் முகவரி**
- **விலைப்பட்டியல் எண்ணிடும்** வடிவமைப்பு மற்றும் வரிசை
- விலைப்பட்டியல் தலைப்பில் காட்டப்படும் **லோகோ**
- விதிமுறைகள், குறிப்புகள் அல்லது சட்டத் தகவல்களுக்கான **தனிப்பயன் அடிக்குறிப்பு உரை**

விலைப்பட்டியல் வார்ப்புருவைத் தனிப்பயனாக்க, **Ultimate Multisite → Settings → Payment** என்பதற்குச் சென்று விலைப்பட்டியல் தொடர்பான அமைப்புகளைக் கண்டறியுங்கள்.
