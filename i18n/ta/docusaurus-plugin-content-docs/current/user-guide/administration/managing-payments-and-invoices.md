# கட்டணங்களையும் விலைப்பட்டியல்களையும் நிர்வகித்தல்

## கட்டண அமைப்புகள் {#payment-settings}

நீங்கள் கட்டணங்களைப் பெறத் தொடங்குவதற்கு முன், கட்டணம் தொடர்பான அமைப்புகளை உள்ளமைக்க வேண்டும். **Ultimate Multisite → Settings**-க்கு சென்று **Payment** தாவலைக் கிளிக் செய்யவும்.

![கட்டண அமைப்புகள் தாவல்](/img/admin/settings-payments-top.png)

கட்டண அமைப்புகள் பக்கத்தின் முழு பார்வை இதோ:

![கட்டண அமைப்புகள் முழுப் பக்கம்](/img/admin/settings-payments-full.png)

### பொதுவான கட்டண விருப்பங்கள் {#general-payment-options}

பொதுவான அமைப்புகளில் நீங்கள் உள்ளமைக்கக்கூடியவை:

- **நாணயம்** — பரிவர்த்தனைகளுக்குப் பயன்படுத்தப்படும் இயல்புநிலை நாணயம்
- **நாணய நிலை** — நாணயச் சின்னம் எங்கு தோன்றும் (தொகைக்கு முன்/பின்)

![கட்டண நுழைவாயில் அமைப்புகள்](/img/admin/settings-payments-options.png)

### கட்டண நுழைவாயில்கள் {#payment-gateways}

Ultimate Multisite பல கட்டண நுழைவாயில்களை ஆதரிக்கிறது. Payment அமைப்புகள் தாவலில் இருந்து ஒவ்வொன்றையும் இயக்கி உள்ளமைக்கலாம்.

![கட்டண நுழைவாயில் உள்ளமைவு](/img/admin/settings-payments-gateways.png)

கிடைக்கும் நுழைவாயில்களில் அடங்குபவை:

- **Stripe** — Stripe வழியாக கிரெடிட் கார்டு கட்டணங்கள்
- **PayPal** — PayPal கட்டணங்கள்
- **Manual** — ஆஃப்லைன் அல்லது தனிப்பயன் கட்டண செயலாக்கத்திற்காக

ஒவ்வொரு நுழைவாயிலுக்கும் தனித்தனி உள்ளமைவு பிரிவு உள்ளது; அங்கு நீங்கள் API keys மற்றும் பிற அமைப்புகளை உள்ளிடலாம்.

![கூடுதல் நுழைவாயில் அமைப்புகள்](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode {#sandbox-mode}

நேரலையில் பயன்படுத்துவதற்கு முன் உங்கள் கட்டண ஒருங்கிணைப்பைச் சோதிக்க **Sandbox Mode**-ஐ இயக்கலாம். Sandbox mode செயலில் இருக்கும்போது, உண்மையான கட்டணங்கள் எதுவும் விதிக்கப்படாது.

## கட்டணங்களைப் பார்வையிடுதல் {#viewing-payments}

உங்கள் வலையமைப்பிலுள்ள அனைத்து பரிவர்த்தனைகளையும் காண Ultimate Multisite-இன் கீழ் உள்ள **Payments** பக்கத்துக்குச் செல்லவும்.

![கட்டணங்கள் பட்டியல்](/img/admin/payments-list.png)

நிலைப்படி (நிறைவடைந்தது, நிலுவையில் உள்ளது, தோல்வியடைந்தது, திருப்பித் தரப்பட்டது) கட்டணங்களை வடிகட்டலாம்; குறிப்பிட்ட பரிவர்த்தனைகளையும் தேடலாம்.

வரி உருப்படிகள், தொடர்புடைய உறுப்பினர் திட்டம், வாடிக்கையாளர் தகவல், மற்றும் கட்டண நுழைவாயில் தரவு உள்ளிட்ட முழு விவரங்களைப் பார்க்க ஒரு கட்டணத்தை கிளிக் செய்யவும்.

## விலைப்பட்டியல்கள் {#invoices}

Ultimate Multisite கட்டணங்களுக்கான விலைப்பட்டியல்களை தானாக உருவாக்க முடியும். Payment அமைப்புகளில் இருந்து விலைப்பட்டியல் வார்ப்புருவையும் எண்ணிடும் வடிவத்தையும் தனிப்பயனாக்கலாம்.

விலைப்பட்டியல் தனிப்பயனாக்க விருப்பங்களில் அடங்குபவை:

- விலைப்பட்டியல்களில் காட்டப்படும் **நிறுவனப் பெயரும் முகவரியும்**
- **விலைப்பட்டியல் எண்ணிடல்** வடிவமும் வரிசையும்
- விலைப்பட்டியல் தலைப்பில் காட்டப்படும் **லோகோ**
- விதிமுறைகள், குறிப்புகள், அல்லது சட்டத் தகவல்களுக்கான **தனிப்பயன் அடிக்குறிப்பு உரை**

விலைப்பட்டியல் வார்ப்புருவைத் தனிப்பயனாக்க, **Ultimate Multisite → Settings → Payment**-க்கு சென்று விலைப்பட்டியல் தொடர்பான அமைப்புகளைத் தேடவும்.
