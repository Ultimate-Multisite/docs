# தளங்களை நிர்வகித்தல்

தளங்கள் (அல்லது துணை தளங்கள்) உங்கள் WaaS வணிகத்தின் மையமாகும். Ultimate Multisite-ல் 3 வகையான தளங்கள் உள்ளன:

- **வாடிக்கையாளர் உரிமையிலான** — குறிப்பிட்ட வாடிக்கையாளர்களுக்கு சொந்தமான தளங்கள்
- **தள வார்ப்புருக்கள்** — வாடிக்கையாளர்கள் தொடக்கப் புள்ளியாக தேர்வு செய்யக்கூடிய முன்வரையறுக்கப்பட்ட தளங்கள்
- **முதன்மை தளம்** — உங்கள் பிரதான நெட்வொர்க் தளம்

## தளங்களைப் பார்த்தல்

உங்கள் நெட்வொர்க்கில் உள்ள அனைத்து துணை தளங்களையும் பார்க்க **Ultimate Multisite → Sites** என்பதற்கு செல்லவும். ஒவ்வொரு தளமும் வகையின்படி லேபிள் இடப்பட்டிருக்கும், மேலும் அனைத்து தளங்கள், வாடிக்கையாளர் உரிமையிலான, வார்ப்புருக்கள் அல்லது நிலுவையில் உள்ளவை என வடிகட்டலாம்.

![தளங்கள் பட்டியல் பக்கம்](/img/admin/sites-list.png)

## புதிய தளத்தைச் சேர்த்தல்

புதிய தளத்தை உருவாக்க **Add Site** பொத்தானைக் கிளிக் செய்யவும். நீங்கள் வழங்க வேண்டியவை:

- **Site Title** — புதிய தளத்தின் பெயர்
- **Site URL** — தளத்திற்கான domain/path
- **Site Type** — இது வாடிக்கையாளர் தளமா, வார்ப்புருவா அல்லது சாதாரண தளமா என்பது

**Copy Site** விருப்பம் ஏற்கனவே உள்ள தள வார்ப்புருவின் அடிப்படையில் புதிய தளத்தை உருவாக்க உதவுகிறது. இதை இயக்கும்போது, தொடக்கப் புள்ளியாக எந்த வார்ப்புருவைப் பயன்படுத்த வேண்டும் என்பதைத் தேர்வு செய்யலாம். மீடியா கோப்புகளையும் சேர்க்க **Copy Media on Duplication** இயக்கப்பட்டிருப்பதை உறுதிசெய்யவும்.

## ஏற்கனவே உள்ள தளத்தை நிர்வகித்தல்

**Edit Site** பக்கத்தைத் திறக்க எந்த தளத்திலும் **Manage** என்பதைக் கிளிக் செய்யவும். இங்கே நீங்கள் காண்பவை:

### அடிப்படை தகவல்கள்

தளத்தின் பெயர், வகை, site ID மற்றும் விளக்கம். மேப் செய்யப்பட்ட domain, தொடர்புடைய membership மற்றும் தளத்தை உரிமையாகக் கொண்ட வாடிக்கையாளர் கணக்கையும் இங்கே காணலாம்.

### தள விருப்பங்கள்

தளத்தின் திறன்கள் மற்றும் வரம்புகளை உள்ளமைக்கவும்:

- **பார்வை வரம்புகள்** — தள பார்வைகளின் அதிகபட்ச எண்ணிக்கை
- **பயனர் கணக்கு வரம்புகள்** — ஒவ்வொரு பயனர் பாத்திரத்திற்கும் வரம்புகள்
- **Disk space** — தளத்திற்கான சேமிப்பு quota
- **Custom domain** — இந்த தளத்திற்கு domain mapping-ஐ இயக்கவும்
- **Plugin மற்றும் theme தெரிவுநிலை** — எந்த plugin-கள் மற்றும் theme-கள் தெரியும், மறைக்கப்படும் அல்லது முன்-செயல்படுத்தப்படும் என்பதைக் கட்டுப்படுத்தவும்

இயல்பாக, தளங்கள் membership நிலையில் அமைக்கப்பட்ட வரம்புகளைப் பின்பற்றும். தள நிலையில் வரம்புகளை அமைப்பது membership அமைப்புகளை மேலெழுதும்.

### தொடர்புகள்

தள விருப்பங்களுக்குக் கீழே இவற்றைப் பற்றிய தகவல்களைக் காணலாம்:

- தளத்துடன் தொடர்புடைய **மேப் செய்யப்பட்ட domain-கள்**
- தளம் உள்ள **Membership**
- தளத்துடன் இணைக்கப்பட்ட **வாடிக்கையாளர் கணக்கு**

### வலது பக்கப்பட்டி

வலது பக்கத்தில் நீங்கள் செய்யக்கூடியவை:

- toggle பொத்தான் மூலம் **தளத்தை இயக்கு/முடக்கு**
- **தள வகையை மாற்று** அல்லது உரிமையை மறுஒதுக்கீடு செய்
- **தளப் படம்/சிறுபடத்தை அமை** (தள வார்ப்புருக்களுக்கு frontend-ல் காட்டப்படும்)
- **தளத்தை நிரந்தரமாக நீக்கு**

:::warning
தளத்தை நீக்குவது மீளமுடியாதது. தளமும் அதன் அனைத்து உள்ளடக்கமும் நிரந்தரமாக அகற்றப்படும்.
:::
