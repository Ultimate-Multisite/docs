# வாடிக்கையாளர்களை நிர்வகித்தல்

**Customers** பக்கம் உங்கள் network-ல் பதிவுசெய்துள்ள உறுப்பினர்களின் பட்டியலைக் காட்டுகிறது. இந்தப் பக்கம் பெயர், மின்னஞ்சல் முகவரி, கடைசி உள்நுழைவு போன்ற அடிப்படை பயனர் தகவல்களையும், **switch to** இணைப்பையும் காட்டுகிறது — இது பயனரின் subsite-ல் site administrator ஆக உள்நுழைய உதவுகிறது.

![Customers list page](/img/admin/customers-list.png)

ஒரு வாடிக்கையாளரின் card-ன் கீழே உள்ள **Manage** என்பதைக் கிளிக் செய்து, அந்த வாடிக்கையாளர் பற்றிய விரிவான தகவல்களைப் பாருங்கள்.

வாடிக்கையாளர் விவரப் பக்கத்தில், **At a Glance** block வாடிக்கையாளர் ID, கடைசி உள்நுழைவு நேரம், மற்றும் இந்த வாடிக்கையாளர் செலவழித்த மொத்த தொகையைக் காட்டுகிறது.

**Customer Options** block-ல் இந்த வாடிக்கையாளரை **VIP** ஆக அமைக்கலாம் (அவர்களை எளிதாகக் கண்டுபிடிக்க உதவும் ஒரு குறிச்சொல்). **Billing Info** tab-ல் மின்னஞ்சல், முகவரி, நாடு போன்ற கட்டணத் தகவல்கள் இருக்கும்.

**Fields** block-ல் வாடிக்கையாளர் username, மின்னஞ்சல் ஆகியவை இருக்கும், மேலும் **Email Verification** நிலையை மாற்றவும் முடியும்.

Customer Options block-க்குக் கீழே நான்கு கூடுதல் block-கள் இருக்கும்:

- **Memberships** — வாடிக்கையாளர் சந்தா செலுத்தியுள்ள தற்போதைய சந்தாக்கள்
- **Payments** — வாடிக்கையாளரின் கட்டண வரலாறு
- **Sites** — இந்த வாடிக்கையாளர் சொந்தமாக வைத்திருக்கும் தளங்கள்
- **Events** — இந்த வாடிக்கையாளர் தொடர்பான நிகழ்வுகள் (மின்னஞ்சல் சரிபார்ப்பு, கட்டண மாற்றங்கள் போன்றவை)

எந்த உருப்படியின் விவரப் பக்கத்திற்கும் செல்ல, அதன் அருகிலுள்ள அம்புக்குறி icon-ஐ (→) கிளிக் செய்யுங்கள். உதாரணமாக, ஒரு தளத்தின் அருகிலுள்ள அம்புக்குறியைக் கிளிக் செய்தால், site edit பக்கத்திற்குச் செல்வீர்கள் — அங்கே membership association, site type, மற்றும் active நிலையை மாற்றலாம்.

Memberships மற்றும் payments-க்கும் இதே போல் பொருந்தும் — ஒவ்வொன்றும் விவரப் பக்கங்களுக்கு இணைக்கப்பட்டிருக்கும், அங்கே membership status, associated customer, products, payment gateway settings ஆகியவற்றைத் திருத்தலாம், refund-களை கூட செயலாக்கலாம்.

வாடிக்கையாளர் பக்கத்தின் வலது பக்கத்தில் இவற்றைக் காணலாம்:

- **Last Login & IPs** — கடைசி உள்நுழைவு நேரம், IP முகவரி, மற்றும் நாடு
- **Timestamps** — வாடிக்கையாளர் கணக்கு எப்போது உருவாக்கப்பட்டது மற்றும் கடைசியாக எப்போது மாற்றப்பட்டது
- **Delete** — வாடிக்கையாளரை நிரந்தரமாக நீக்கும் விருப்பம்

:::warning
வாடிக்கையாளரை நீக்குவது நிரந்தரமானது, மீட்டெடுக்க இயலாது.
:::
