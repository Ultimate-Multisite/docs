# உறுப்பினர்தன்மைகளை நிர்வகித்தல்

Ultimate Multisite இல் உங்கள் நெட்வொர்க்கில் உள்ள உறுப்பினர்தன்மைகளையும் அனைத்து சந்தாக்களையும் நிர்வகிக்கலாம். திட்டங்கள் மற்றும் தயாரிப்புகள், தளங்களின் எண்ணிக்கை, பில்லிங் தொகை மற்றும் அவகாசம், கட்டண வரலாறு, நேரமுத்திரைகள் போன்ற சந்தா விவரங்களைப் பார்த்து திருத்தலாம்.

இந்தப் பக்கத்தை அணுக, Ultimate Multisite மெனுவில் உள்ள **உறுப்பினர்தன்மைகள்** தாவலுக்குச் செல்லவும்.

![உறுப்பினர்தன்மைகள் பட்டியல் பக்கம்](/img/admin/memberships-list.png)

உறுப்பினர்தன்மைகள் பக்கத்தில், உங்கள் வாடிக்கையாளர்களின் பட்டியலையும் அவர்களின் கணக்குகளுடன் தொடர்புடைய தயாரிப்புகளையும் காணலாம். நிலைப்படி உறுப்பினர்தன்மைகளை வடிகட்டலாம்: செயலிலுள்ளது, நிலுவையில் உள்ளது, நிறுத்திவைக்கப்பட்டுள்ளது, காலாவதியானது, அல்லது ரத்து செய்யப்பட்டது.

இலவச உறுப்பினர்தன்மைகள் **ஆயுள் முழுவதும்** நீடிக்கும் உறுப்பினர்தன்மைகளாகக் கருதப்படுகின்றன. அவற்றுக்கு காலாவதி தேதி வழங்கப்படாது, மேலும் அவை தானாக காலாவதியாகாது. ஒரு இலவச திட்டம் சோதனை காலத்திற்குப் பிறகு அல்லது நிர்ணயிக்கப்பட்ட காலத்திற்குப் பிறகு முடிவடைய வேண்டும் என்றால், சோதனை காலத்துடன் கூடிய கட்டண தயாரிப்பைப் பயன்படுத்தவும், உறுப்பினர்தன்மையை கைமுறையாக ரத்து செய்யவும், அல்லது அவர்களின் அணுகல் மாற வேண்டிய நேரத்தில் வாடிக்கையாளரை வேறு திட்டத்துக்கு மாற்றவும்.

ஒரு உறுப்பினர்தன்மையைச் சொடுக்கி, அதன் விவரங்களைத் திருத்தக்கூடிய பக்கத்தை அணுகலாம் — பில்லிங் தகவல், தள அல்லது பார்வை வரம்புகள், மேலும் பல.

![உறுப்பினர்தன்மை திருத்தும் இடைமுகம்](/img/admin/membership-edit.png)

உறுப்பினர்தன்மை திருத்தும் பக்கத்தின் முழுப் பார்வை இங்கே:

![உறுப்பினர்தன்மை திருத்தும் முழு இடைமுகம்](/img/admin/membership-edit-full.png)

உங்கள் நெட்வொர்க்கில் முன்பு பதிவு செய்யப்பட்ட பயனர்களுடன் தொடர்புடைய புதிய உறுப்பினர்தன்மைகளையும் சேர்க்கலாம். **உறுப்பினர்தன்மையைச் சேர்** என்பதைக் கிளிக் செய்து விவரங்களை அமைக்கவும்.

![உறுப்பினர்தன்மையைச் சேர் மாதிரி சாளரம்](/img/admin/membership-add-modal.png)

பின்னர் **உறுப்பினர்தன்மையை உருவாக்கு** என்பதைக் கிளிக் செய்யவும். புதிய உறுப்பினர்தன்மை உறுப்பினர்தன்மைகள் பட்டியலில் தோன்றும்.
