# உறுப்பினர்களை நிர்வகித்தல்

Ultimate Multisite-ல் உங்கள் நெட்வொர்க்கில் உள்ள அனைத்து சந்தாக்களையும் அணுகலாம், உறுப்பினர்களை நிர்வகிக்கலாம். திட்டங்கள் மற்றும் தயாரிப்புகள், தளங்களின் எண்ணிக்கை, பில்லிங் தொகை மற்றும் கால அளவு, பணம் செலுத்திய வரலாறு, நேர முத்திரைகள் போன்ற சந்தா விவரங்களை பார்வையிடலாம், திருத்தலாம்.

இந்தப் பக்கத்தை அணுக, Ultimate Multisite மெனுவில் **Memberships** டேப்பைக் கிளிக் செய்யுங்கள்.

![Memberships list page](/img/admin/memberships-list.png)

Memberships பக்கத்தில் உங்கள் வாடிக்கையாளர்களின் பட்டியலையும், அவர்களின் கணக்குகளுடன் இணைக்கப்பட்ட தயாரிப்புகளையும் காணலாம். active, pending, on hold, expired, அல்லது cancelled என்ற நிலையின் அடிப்படையில் உறுப்பினர்களை வடிகட்டலாம்.

உறுப்பினர் விவரங்களைத் திருத்த — பில்லிங் தகவல், தள வரம்புகள் அல்லது வருகை வரம்புகள் போன்றவற்றை மாற்ற — ஒரு உறுப்பினர் மீது கிளிக் செய்து அந்தப் பக்கத்திற்குச் செல்லுங்கள்.

உங்கள் நெட்வொர்க்கில் ஏற்கனவே பதிவு செய்துள்ள பயனர்களுக்கு புதிய உறுப்பினர்களையும் சேர்க்கலாம். **Add Membership** என்பதைக் கிளிக் செய்து விவரங்களை அமைக்கவும்.

![Add Membership modal](/img/admin/membership-add-modal.png)

பிறகு **Create Membership** என்பதைக் கிளிக் செய்யுங்கள். புதிய உறுப்பினர் memberships பட்டியலில் தோன்றும்.
