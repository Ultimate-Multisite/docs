# ஏற்றுமதி மற்றும் இறக்குமதி (Export & Import)

Ultimate Multisite 2.9.0, **Tools > Export & Import** என்ற இடத்தில் ஒரு ஒற்றை-தள **ஏற்றுமதி மற்றும் இறக்குமதி** கருவியை (tool) சேர்க்கிறது. ஒரு WordPress தளத்தை ZIP கோப்பாகப் பொதி செய்ய வேண்டியிருக்கும்போதும், அந்த ZIP கோப்பை மீட்டெடுக்க வேண்டியிருக்கும்போதும், அல்லது ஒரு தளத்தை இணக்கமான Ultimate Multisite மற்றும் ஒற்றை-தள WordPress நிறுவல்களுக்கு இடையில் மாற்ற வேண்டியிருக்கும்போதும் இதை நீங்கள் பயன்படுத்தலாம்.

## தேவையான அனுமதிகள் (Required permissions) {#required-permissions}

நீங்கள் ஏற்றுமதி செய்யவு அல்லது இறக்குமதி செய்யவு செய்யும் தளத்தில் உள்ள WordPress **Tools** மெனுவை அணுகக்கூடிய நிர்வாகியாக (administrator) உள்நுழைய வேண்டும். ஒரு மல்ட்டிசைட் நெட்வொர்க்கில் (multisite network), நெட்வொர்க் மட்டத்தின் (network-level) Ultimate Multisite கருவிகளைப் பயன்படுத்தி சப்-சைட்களை (subsites) ஏற்றுமதி செய்யும்போது அல்லது இறக்குமதி செய்யும்போது, நெட்வொர்க் நிர்வாகி கணக்கைப் பயன்படுத்தவும்.

ஏற்றுமதி செய்யப்பட்ட ZIP கோப்புகள் அங்கீகரிக்கப்பட்ட பதிவிறக்க இறுதிப் புள்ளியின் (authenticated download endpoint) மூலம் வழங்கப்படுவதால், பதிவிறக்கம் முடியும் வரை நிர்வாக அமர்வை (admin session) செயலில் வைத்திருங்கள், மேலும் உருவாக்கப்பட்ட பதிவிறக்க URL-களை பொதுவில் பகிர வேண்டாம்.

## ஒரு தளத்தை ZIP ஆக ஏற்றுமதி செய்தல் (Exporting a site to a ZIP) {#exporting-a-site-to-a-zip}

1. நீங்கள் நகலெடுக்க விரும்பும் தளத்தின் WordPress நிர்வாகப் பகுதிக்குச் சென்று, **Tools > Export & Import** என்பதற்குச் செல்லவும்.
2. ஏற்றுமதிப் பகுதிக்குள் சென்று, நீங்கள் பொதி செய்ய விரும்பும் தளத்தைத் தேர்ந்தெடுக்கவும்.
3. விருப்பமான உள்ளடக்கத்தை (optional content) சேர்க்கவும். உதாரணமாக, பதிவேற்றங்கள் (uploads), plugins, மற்றும் themes போன்ற விருப்பங்கள் இருந்தால், அவற்றைத் தேர்ந்தெடுக்கவும்.
4. ஏற்றுமதியைத் தொடங்கி, செயல்முறை முடியும் வரை காத்திருக்கவும். பெரிய தளங்களுக்கு ZIP தயாராவதற்கு முன்பு பின்னணியில் (background) முடிக்க வேண்டியிருக்கலாம்.
5. ஏற்றுமதிகள் பட்டியலில் இருந்து முடிக்கப்பட்ட ZIP கோப்பைப் பதிவிறக்கவும்.

இந்த ZIP கோப்பை பாதுகாப்பான இடத்தில் வைக்கவும். இதில் தளத்தின் உள்ளடக்கம், அமைப்புகள், மீடியா கோப்புகள் மற்றும் நீங்கள் தேர்ந்தெடுத்த குறியீட்டு சொத்துக்கள் (code assets) ஆகியவை இருக்கலாம்.

## ஏற்றுமதியில் என்ன உள்ளடங்கும் (What the export includes) {#what-the-export-includes}

ஒரு ஏற்றுமதி ZIP கோப்பில் பின்வருவனவற்றை உள்ளடக்கும்:

- தேர்ந்தெடுக்கப்பட்ட தளத்தின் தரவுத்தள உள்ளடக்கம் மற்றும் அமைப்பு (database content and configuration).
- பதிவேற்றங்கள் சேர்க்கப்பட்டால், பதிவேற்றப்பட்ட மீடியா கோப்புகள் (uploaded media files).
- அந்த விருப்பங்கள் தேர்ந்தெடுக்கப்பட்டால், plugins மற்றும் themes.
- இலக்கு நிறுவலில் (target installation) தளத்தை மீண்டும் உருவாக்க Export & Import கருவி பயன்படுத்தும் இறக்குமதி மெட்டாடேட்டா (import metadata).

சரியான ZIP அளவு, மீடியாவின் அளவு, தேர்ந்தெடுக்கப்பட்ட plugins மற்றும் themes, மற்றும் தளத்தின் தரவுத்தள அட்டவணைகளின் (database tables) அளவைப் பொறுத்து மாறுபடும்.

## ZIP இலிருந்து ஒரு தளத்தை இறக்குமதி செய்தல் (Importing a site from a ZIP) {#importing-a-site-from-a-zip}

1. இலக்கு WordPress தளத்தில் **Tools > Export & Import** என்பதற்குச் செல்லவும்.
2. இறக்குமதிப் பகுதிக்குள் சென்று, Export & Import கருவியால் உருவாக்கப்பட்ட ZIP கோப்பை பதிவேற்றவும்.
3. தளம் ஒரு புதிய முகவரியைப் பயன்படுத்த வேண்டும் என்றால், மாற்றும் URL-ஐ (replacement URL) உள்ளிடவும். அல்லது, அசல் URL-ஐ தக்கவைக்க இந்த புலத்தை காலியாக விடவும்.
4. இறக்குமதி முடிந்த பிறகு பதிவேற்றப்பட்ட ZIP கோப்பை நீக்க வேண்டுமா என்பதைத் தேர்ந்தெடுக்கவும்.
5. **Begin Import** என்பதைக் கிளிக் செய்யவும்.
6. இது முடியும் வரை நிலுவையில் உள்ள இறக்குமதியைக் (pending import) கண்காணிக்கவும். செயல்முறை முடியும் முன்பே நிறுத்த வேண்டியிருந்தால் மட்டுமே **Cancel Import** பயன்படுத்தவும்.
7. சாதாரண போக்குவரத்து அல்லது வாடிக்கையாளர் அணுகலை அனுமதிக்க기 전에 இறக்குமதி செய்யப்பட்ட தளத்தை மதிப்பாய்வு செய்யவும்.

ஒற்றை-தள WordPress நிறுவலில், ஒரு ZIP கோப்பைப் பதிவேற்றுவது, தற்போதைய தளத்தை இறக்குமதி செய்யப்பட்ட தளத்தால் மாற்றுகிறது. நீங்கள் தொடங்குவதற்கு முன், இலக்கு தளத்தின் முழுமையான காப்புப் பிரதியை (full backup) உருவாக்கவும், மேலும் ஒரே நேரத்தில் ஒரே தளத்திற்கான பல இறக்குமதிகளைத் தொடர்வதைத் தவிர்க்கவும்.

## வரம்புகள் மற்றும் இணக்கத்தன்மை குறிப்புகள் (Limitations and compatibility notes) {#limitations-and-compatibility-notes}

- மிக பெரிய பதிவேற்ற கோப்பகங்கள் (uploads directories) அல்லது மீடியா நூலகங்கள் (media libraries) பெரிய ZIP கோப்புகளை உருவாக்கலாம். பெரிய தளங்களை ஏற்றுமதி செய்வதற்கு அல்லது இறக்குமதி செய்வதற்கு முன்பு PHP பதிவேற்ற வரம்புகள் (upload limits), செயலாக்க வரம்புகள் (execution limits), வட்டு இடம் (disk space), நினைவகம் (memory), மற்றும் சர்வர் காலாவதி அமைப்புகள் (server timeout settings) ஆகியவற்றை உறுதிப்படுத்தவும்.
- மிக பெரிய மீடியா நூலகங்கள் குறைந்த போக்குவரத்து உள்ள பராமரிப்பு நேரத்தின் (low-traffic maintenance window) போது மாற்றப்பட வேண்டியிருக்கலாம்.
- இலக்கு WordPress நிறுவல் இணக்கமான WordPress, PHP, Ultimate Multisite, plugin, மற்றும் theme பதிப்புகளை இயக்க வேண்டும்.
- ஒரு ஒற்றை-தள இறக்குமதி இலக்கு தளத்தை மாற்றுகிறது. இது ஒரு இணைக்கும் கருவி (merge tool) அல்ல.
- மல்ட்டிசைட்-க்கு ஒற்றை-தள மற்றும் ஒற்றை-தள-க்கு மல்ட்டிசைட் மாற்றங்கள், இலக்கு சூழல் ஏற்றுமதி செய்யப்பட்ட தளத்தின் plugins, themes, URLs, மற்றும் தேவையான Ultimate Multisite கூறுகளை இயக்க முடிந்தால் மட்டுமே ஆதரிக்கப்படும்.
- ZIP கோப்பை தனிப்பட்டதாக (private) வைத்திருங்கள். இதில் தரவுத்தள உள்ளடக்கம், பதிவேற்றப்பட்ட கோப்புகள் மற்றும் ஏற்றுமதி செய்யப்பட்ட தளத்தின் அமைப்பு விவரங்கள் இருக்கலாம்.

பழைய நெட்வொர்க்-மட்ட ஏற்றுமதி செயல்முறைகளுக்கு, [Site Export](/user-guide/administration/site-export) பார்க்கவும்.
