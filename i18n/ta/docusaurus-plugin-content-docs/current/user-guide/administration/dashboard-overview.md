# Dashboard கண்ணோட்டம்

Ultimate Multisite-ல் Ultimate Multisite மெனுவின் கீழ் ஒரு தனிப்பட்ட **Dashboard** உள்ளது, மேலும் Network Dashboard-க்கும் widgets சேர்க்கிறது. இந்த கட்டுரை இரண்டு இடங்களிலும் நீங்கள் என்ன காண்பீர்கள் என்பதை விளக்குகிறது.

## Network Dashboard

Ultimate Multisite பயனுள்ள widgets-ஐ **My Sites → Network Admin → Dashboard**-ல் காணப்படும் network-level dashboard-ல் சேர்க்கிறது.

![Ultimate Multisite widgets-உடன் கூடிய Network Dashboard](/img/admin/network-dashboard-full.png)

ஒவ்வொரு widget என்ன வழங்குகிறது என்பது இங்கே:

**First Steps:** புதிய நிறுவலில் தோன்றும், payment method அமைத்தல் மற்றும் test account உருவாக்குதல் போன்ற setup பணிகளை முடிக்க நினைவூட்டுகிறது.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** அன்றைய signups மற்றும் வருவாய் எண்ணிக்கையை காட்டுகிறது. MRR (Monthly Recurring Revenue) என்பது வருவாய் உருவாக்கும் memberships கொண்ட வாடிக்கையாளர்கள் முழுவதும் கணிக்கப்பட்ட மொத்தத்தை குறிக்கிறது.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite முழுவதும் நிகழ்வுகளை அறிவிக்கிறது — signups, cancellations, site உருவாக்கங்கள், நீக்குதல்கள் மற்றும் பிற செயல்பாடுகள்.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** புதிய sites அல்லது users உருவாக்க குறுக்குவழிகளுடன் network-ல் உள்ள users மற்றும் sites-ன் சுருக்கமான சுருக்கத்தை காட்டுகிறது.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** updates, முக்கியமான bugs மற்றும் security fixes உள்ளிட்ட சமீபத்திய Ultimate Multisite செய்திகளை பெற்று காட்டுகிறது.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

Network dashboard network பற்றிய முழுமையான தகவல்களை வழங்கும் அதே வேளையில், Ultimate Multisite dashboard (Ultimate Multisite மெனுவின் மேல் நிலையில்) சேவை பற்றிய தகவல்களை வழங்குகிறது.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** 12 மாத காலண்டர் காலத்தில் வருவாயின் மாதாந்திர பிரிவை காட்டுகிறது. புதிய வருவாய் நீல நிறத்திலும், cancellations சிவப்பு நிறத்திலும் தோன்றும்.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** உங்களுக்கு தேவையான தகவல்களில் கவனம் செலுத்த கால அளவு மூலம் dashboard தரவை வடிகட்டுங்கள். Date range கீழே உள்ள widgets-ல் என்ன காட்டப்படுகிறது என்பதை கட்டுப்படுத்துகிறது.

![Date range மற்றும் dashboard stats](/img/admin/um-dashboard-date-range.png)

Dashboard widgets-ல் அடங்குபவை:

- **Revenue** — MRR மற்றும் ஒவ்வொரு product பிரிவுகளுடன் குறிப்பிட்ட காலத்திற்கான மொத்த வருவாய்
- **New Memberships** — தேர்ந்தெடுக்கப்பட்ட காலத்தில் எத்தனை புதிய memberships உருவாக்கப்பட்டன
- **Signups by Countries** — signups-ன் புவியியல் பிரிவு
- **Signups by Forms** — admins சேர்த்தவை vs ஒவ்வொரு checkout form வழியாக எத்தனை வாடிக்கையாளர்கள் signup செய்தனர்
- **Most Visited Sites** — உங்கள் network-ல் அதிகம் பார்வையிடப்பட்ட sites (**Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**-ல் visit counting-ஐ enable செய்ய வேண்டும்)
