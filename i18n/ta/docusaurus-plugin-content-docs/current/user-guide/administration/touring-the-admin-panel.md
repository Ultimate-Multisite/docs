# நிர்வாகப் பேனலைச் சுற்றிப் பார்ப்போம்

Ultimate Multisite முடிந்தவரை எளிமையாகவும் புரிந்துகொள்ள ஏற்றதாகவும் வடிவமைக்கப்பட்டுள்ளது, ஆனால் இவ்வளவு சக்திவாய்ந்த ஒரு கருவியைப் பயன்படுத்தும்போது ஒரு வழிகாட்டுதல் உதவியாக இருக்கும். நிர்வாகப் பக்கங்களை ஒவ்வொன்றாகப் பார்த்து, நீங்கள் எளிதாகப் புரிந்துகொள்ள உதவுவோம்.

## Dashboard

Ultimate Multisite **Dashboard** அடிப்படை அறிக்கைகளையும் பகுப்பாய்வுகளையும் காட்டுகிறது — வருவாய், தள நடவடிக்கைகள், உறுப்பினர் வளர்ச்சி, பார்வையாளர் எண்ணிக்கை மற்றும் புவியியல் தரவுகள் போன்றவை இதில் அடங்கும்.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

## Checkout Forms

**Checkout Forms** மூலம் தனிப்பயன் பதிவுப் பக்கங்களை உருவாக்க முடியும். புலங்களைத் தனிப்பயனாக்கலாம், தனிப்பயன் classes மற்றும் scripts சேர்க்கலாம், பார்வையாளரின் நாடு அல்லது இருப்பிடத்தின் அடிப்படையில் அணுகலைக் கட்டுப்படுத்தலாம்.

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products

**Products** பிரிவில் உங்கள் நெட்வொர்க்கிற்கான வெவ்வேறு திட்டங்கள், பேக்கேஜ்கள் மற்றும் சேவைகளை உருவாக்கலாம். விலைகள், கட்டண அதிர்வெண்கள், வரம்புகள் மற்றும் ஒதுக்கீடுகளை product மட்டத்தில் வரையறுக்கலாம்.

![Products list](/img/admin/products-list.png)

## Memberships

**Memberships** பக்கம் உங்கள் நெட்வொர்க்கில் உள்ள அனைத்து சந்தாக்களையும் காட்டுகிறது. சந்தா விவரங்களான திட்டங்கள், products, கட்டணத் தொகைகள் மற்றும் அதிர்வெண்கள், கட்டண வரலாறு, நேர முத்திரைகள் ஆகியவற்றை இங்கே பார்க்கவும் திருத்தவும் முடியும்.

![Memberships list](/img/admin/memberships-list.png)

## Payments

**Payments** பக்கம் உங்கள் முழு நெட்வொர்க்கிலும் உள்ள கட்டண வரலாற்றை விரைவாகக் காட்டுகிறது — குறிப்பிட்ட products மற்றும் தொகைகள் உள்ளிட்ட விரிவான பரிவர்த்தனைத் தகவல்களும் இதில் அடங்கும்.

![Payments list](/img/admin/payments-list.png)

## Customers

**Customers** பக்கம் உங்கள் நெட்வொர்க்கில் பதிவுசெய்துள்ள அனைத்து உறுப்பினர்களையும் காட்டுகிறது — பெயர், மின்னஞ்சல் முகவரி, கடைசியாக உள்நுழைந்த நேரம், அவர்களின் subsite-க்கு உள்நுழைவதற்கான "switch to" இணைப்பு போன்ற அடிப்படை பயனர் தகவல்கள் இங்கே கிடைக்கும்.

![Customers list](/img/admin/customers-list.png)

## Sites

**Sites** பக்கம் உங்கள் நெட்வொர்க்கில் உள்ள அனைத்து subsites-ஐயும் பட்டியலிடுகிறது — site templates மற்றும் வாடிக்கையாளர்களுக்குச் சொந்தமான sites இதில் அடங்கும். mapped domains, வரம்புகள் மற்றும் ஒதுக்கீடுகள், plugins மற்றும் themes, site thumbnails ஆகியவற்றை எளிதாக நிர்வகிக்கலாம்.

![Sites list](/img/admin/sites-list.png)

## Domains

**Domains** பக்கம் subsites-க்கு map செய்யப்பட்ட தனிப்பயன் domains-க்காக அர்ப்பணிக்கப்பட்டது. Super admin ஆக, தனிப்பயன் domain பெயர்களை கைமுறையாக சேர்க்கலாம் அல்லது map செய்யலாம், DNS records மற்றும் logs-ஐயும் பார்க்கலாம்.

![Domains list](/img/admin/domains-list.png)

## Discount Codes

**Discount Codes** மூலம் உங்கள் இறுதிப் பயனர்களுக்கு தள்ளுபடிகள் வழங்க coupon codes உருவாக்கலாம். தள்ளுபடி மதிப்பைக் குறிப்பிடலாம், குறிப்பிட்ட பயனர்கள் அல்லது குழுக்களை இலக்காகக் கொள்ளலாம்.

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts

**Broadcasts** என்பது உங்கள் subsite பயனர்களுக்கு அறிவிப்புகளை வெளியிடவும் தனிப்பட்ட செய்திகளை அனுப்பவும் உதவும் கருவி. அவர்கள் சந்தா செலுத்தியுள்ள திட்டம் அல்லது product அடிப்படையில் குறிப்பிட்ட பயனர் குழுக்களை இலக்காகக் கொள்ளலாம்.

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings

**Settings** பக்கத்தில் Ultimate Multisite-ஐ உள்ளமைக்கலாம் — பதிவு அமைப்புகள், கட்டணங்கள், API மற்றும் webhooks, domain mapping, மற்றும் பிற ஒருங்கிணைப்புகள்.

![Settings page](/img/admin/settings-general.png)

## Events

**Events** பக்கம் உங்கள் நெட்வொர்க்கில் நடக்கும் அனைத்து events மற்றும் logs-ஐயும் பதிவு செய்கிறது. திட்ட மாற்றங்கள், பதிவுகள் மற்றும் பிற நடவடிக்கைகளைக் கண்காணிக்கிறது — உங்கள் multisite-ஐ மேற்பார்வையிட இது மிகவும் பயனுள்ளது.

![Events list](/img/admin/events-list.png)

## Webhooks

**Webhooks** மூலம் மற்றொரு பயன்பாட்டிற்கு தரவை அனுப்பலாம். Ultimate Multisite-லிருந்து Zapier போன்ற தளங்களுக்கு தரவை அனுப்ப இது பயனுள்ளது.

![Webhooks list](/img/admin/webhooks-list.png)
