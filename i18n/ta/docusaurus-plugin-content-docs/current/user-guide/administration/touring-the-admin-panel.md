# Admin Panel சுற்றுப்பயணம்

Ultimate Multisite முடிந்தவரை எளிமையாகவும் தானாகவே புரிந்துகொள்ளக்கூடியதாகவும் இருக்க நோக்கமிடுகிறது. ஆனால் இவ்வளவு சக்திவாய்ந்த கருவிக்கு வழிகாட்டியுடன் ஒரு சுற்றுப்பயணம் உதவியாக இருக்கும். நீங்கள் எளிதாகப் பழக, admin பக்கங்களைப் பார்ப்போம்.

## Dashboard {#dashboard}

Ultimate Multisite **Dashboard** வருவாய், தளச் செயல்பாடுகள், உறுப்பினர் வளர்ச்சி, பார்வையாளர் எண்ணிக்கை, புவியியல் தரவு உள்ளிட்ட அடிப்படை அறிக்கைகள் மற்றும் பகுப்பாய்வுகளைக் காட்டுகிறது.

Dashboard-ன் கீழ்ப் பகுதி:

![Ultimate Multisite Dashboard கீழ்ப் பகுதி](/img/admin/um-dashboard-bottom.png)

## Checkout Forms {#checkout-forms}

**Checkout Forms** தனிப்பயன் பதிவுப் பக்கங்களை உருவாக்கும் நெகிழ்வுத்தன்மையை வழங்குகின்றன. புலங்களைத் தனிப்பயனாக்கலாம், தனிப்பயன் classes மற்றும் scripts சேர்க்கலாம், மேலும் பார்வையாளரின் நாடு அல்லது இருப்பிடத்தின் அடிப்படையில் அணுகலைக் கட்டுப்படுத்தலாம்.

![Checkout Forms பட்டியல்](/img/admin/checkout-forms-list.png)

## Products {#products}

**Products** பகுதி உங்கள் network-க்கு வெவ்வேறு திட்டங்கள், தொகுப்புகள் மற்றும் சேவைகளை உருவாக்கும் இடமாகும். விலைகள், billing இடைவெளிகள், வரம்புகள் மற்றும் quotas ஆகியவற்றை product நிலையில் வரையறுக்கலாம்.

![Products பட்டியல்](/img/admin/products-list.png)

## Memberships {#memberships}

**Memberships** பக்கம் உங்கள் network-இல் உள்ள அனைத்து subscriptions-ஐக் காட்டுகிறது. திட்டங்கள், products, billing தொகைகள் மற்றும் இடைவெளிகள், கட்டண வரலாறு, timestamps உள்ளிட்ட subscription விவரங்களைப் பார்வையிட்டு திருத்தலாம்.

![Memberships பட்டியல்](/img/admin/memberships-list.png)

## Payments {#payments}

**Payments** பக்கம் உங்கள் முழு network முழுவதிலும் உள்ள கட்டண வரலாற்றை விரைவாகப் பார்க்க உதவுகிறது. குறிப்பிட்ட products மற்றும் தொகைகள் உட்பட விரிவான பரிவர்த்தனைத் தகவல்களையும் காட்டுகிறது.

![Payments பட்டியல்](/img/admin/payments-list.png)

## Customers {#customers}

**Customers** பக்கம் உங்கள் network-இல் பதிவு செய்யப்பட்டுள்ள அனைத்து உறுப்பினர்களையும், பெயர், மின்னஞ்சல் முகவரி, கடைசி உள்நுழைவு, அவர்களின் subsite-இல் உள்நுழைய "switch to" இணைப்பு ஆகிய அடிப்படை பயனர் தகவல்களுடன் காட்டுகிறது.

![Customers பட்டியல்](/img/admin/customers-list.png)

## Sites {#sites}

**Sites** பக்கம் உங்கள் network-இல் உள்ள அனைத்து subsites-ஐயும் பட்டியலிடுகிறது. இதில் தள வார்ப்புருக்கள் மற்றும் வாடிக்கையாளர் சொந்தமான தளங்களும் அடங்கும். Mapped domains, வரம்புகள் மற்றும் quotas, plugins மற்றும் themes, தள thumbnails ஆகியவற்றை எளிதாக நிர்வகிக்கலாம்.

![Sites பட்டியல்](/img/admin/sites-list.png)

## Domains {#domains}

**Domains** பக்கம் subsites-க்கு map செய்யப்பட்ட தனிப்பயன் domains-க்காக அர்ப்பணிக்கப்பட்டது. Super admin ஆக, நீங்கள் தனிப்பயன் domain பெயர்களை கைமுறையாகச் சேர்க்கலாம் அல்லது map செய்யலாம், மேலும் DNS records மற்றும் logs-ஐப் பார்க்கலாம்.

![Domains பட்டியல்](/img/admin/domains-list.png)

## Discount Codes {#discount-codes}

**Discount Codes** உங்கள் இறுதி பயனர்களுக்கு தள்ளுபடிகளை வழங்க coupon codes உருவாக்க அனுமதிக்கின்றன. தள்ளுபடி மதிப்பைக் குறிப்பிடலாம், மேலும் குறிப்பிட்ட பயனர்கள் அல்லது குழுக்களை இலக்காக அமைக்கலாம்.

![Discount Codes பட்டியல்](/img/admin/discount-codes-list.png)

## Broadcasts {#broadcasts}

**Broadcasts** என்பது அறிவிப்புகளை வெளியிட அல்லது உங்கள் subsite பயனர்களுக்கு தனிப்பட்ட செய்திகளை அனுப்பும் கருவி. அவர்கள் subscribed செய்துள்ள plan அல்லது product அடிப்படையில் குறிப்பிட்ட பயனர் குழுக்களை இலக்காக அமைக்கலாம்.

![Broadcasts பட்டியல்](/img/admin/broadcasts-list.png)

## Settings {#settings}

**Settings** பக்கம் Ultimate Multisite-ஐ உள்ளமைக்கும் இடம் — பதிவு அமைப்புகள், payments, API மற்றும் webhooks, domain mapping, மற்றும் பிற integrations.

AI connector அமைப்புகள் தற்போது ஆதரிக்கப்படும் OAuth provider pools-ஐப் பட்டியலிடுகின்றன: Anthropic Max, OpenAI ChatGPT/Codex, மற்றும் Google AI Pro. ஒவ்வொரு provider card-மும் super admins கணக்குகளை இணைக்க, சேமிக்கப்பட்ட கணக்குகளைப் புதுப்பிக்க, மின்னஞ்சல் மூலம் கணக்குகளை அகற்ற, மேலும் sandboxed சூழல் browser redirect-ஐத் தடுக்கும்போது manual OAuth fallback-ஐப் பயன்படுத்த அனுமதிக்கிறது. Cursor Pro setup விருப்பங்கள் admin panel-இலிருந்து அகற்றப்பட்டுள்ளன.

ChatGPT/Codex கணக்குகள், செயல் tools-ஐ அனுமதிக்கும் இடங்களில் connector-backed tool use-ஐ ஆதரிக்கின்றன. எனவே connector-supported operations-ஐ சார்ந்துள்ள admin பணிச்சூழல்கள், OpenAI Account இணைக்கப்பட்ட பிறகு Codex tool நடத்தையைப் பயன்படுத்தலாம்.

## Events {#events}

**Events** பக்கம் உங்கள் network-இல் உள்ள அனைத்து events மற்றும் logs-ன் பதிவை வைத்திருக்கிறது. Plan மாற்றங்கள், signups மற்றும் பிற நடவடிக்கைகள் போன்ற செயல்பாடுகளை இது கண்காணிக்கிறது — உங்கள் multisite-ஐ கண்காணிக்க உதவுகிறது.

![Events பட்டியல்](/img/admin/events-list.png)

## Webhooks {#webhooks}

**Webhooks** தரவை மற்றொரு application-க்கு அனுப்ப அனுமதிக்கின்றன. Ultimate Multisite-இலிருந்து Zapier போன்ற platforms-க்கு தரவை அனுப்ப பயனுள்ளதாக இருக்கும்.

![Webhooks பட்டியல்](/img/admin/webhooks-list.png)
