# Ultimate Multisite டாஷ்போர்டைப் புரிந்துகொள்ளுதல்

மேம்பட்ட கட்டமைப்பு விருப்பங்கள் மற்றும் Ultimate Multisite-ன் செயல்பாட்டு அம்சங்களை ஆராய்வதற்கு முன், டாஷ்போர்டைப் பற்றி பார்ப்போம்.

பல WordPress plugins போலவே, Ultimate Multisite நெட்வொர்க் நிலையில் **Ultimate Multisite** என்ற மெனு உள்ளீட்டை உருவாக்குகிறது, அதன் கீழ் பல துணை-மெனு உள்ளீடுகள் இருக்கும்.

## நெட்வொர்க் டாஷ்போர்ட் விட்ஜெட்கள்

Ultimate Multisite நெட்வொர்க் நிலை டாஷ்போர்டில் சில பயனுள்ள விட்ஜெட்களைச் சேர்க்கிறது. இந்த டாஷ்போர்டை **My Sites → Network Admin → Dashboard** வழியாகச் செல்வதன் மூலம் காணலாம். இந்த விட்ஜெட்கள் நெட்வொர்க் நிர்வாகிக்கு தகவல்களையும் பொதுவான செயல்பாடுகளையும் எளிதாக அணுகுவதற்காக வடிவமைக்கப்பட்டுள்ளன.

![Ultimate Multisite விட்ஜெட்களுடன் கூடிய நெட்வொர்க் டாஷ்போர்ட்](/img/admin/network-dashboard-full.png)

### முதல் படிகள்

![முதல் படிகள் விட்ஜெட்](/img/admin/network-dashboard-first-steps.png)

இந்த விட்ஜெட் Ultimate Multisite-ன் புதிய நிறுவலில் தோன்றும், மேலும் கட்டண முறையை கட்டமைத்தல் மற்றும் சோதனை கணக்கை உருவாக்குதல் போன்ற கூடுதல் பணிகளை முடிக்க நெட்வொர்க் நிர்வாகிக்கு நினைவூட்டுகிறது.

### சுருக்கம்

![சுருக்க விட்ஜெட்](/img/admin/network-dashboard-summary.png)

சுருக்க விட்ஜெட் அன்றைய பதிவுகள் மற்றும் அவற்றின் வருவாயைப் பற்றி அறிக்கை செய்கிறது. MRR (மாதாந்திர தொடர் வருவாய்) வருவாயை ஈட்டும் உறுப்பினர்களைக் கொண்ட வாடிக்கையாளர்களின் மொத்த கணிப்பைக் குறிக்கிறது.

### செயல்பாட்டு நிகழ்வுகள்

![செயல்பாட்டு நிகழ்வுகள் விட்ஜெட்](/img/admin/network-dashboard-activity-stream.png)

இந்த விட்ஜெட் Ultimate Multisite முழுவதும் நடக்கும் நிகழ்வுகளைப் பற்றி அறிக்கை செய்கிறது. நிகழ்வுகளில் பதிவுகள், ரத்துகள், தளம் உருவாக்கங்கள், நீக்குதல்கள் மற்றும் பிற செயல்பாடுகள் அடங்கும்.

மேலும் விரிவான விளக்கத்திற்கு ஆவணத்தின் நிகழ்வுகள் பகுதியைப் பார்க்கவும்.

### இப்போது

![இப்போது விட்ஜெட்](/img/admin/network-dashboard-right-now.png)

இந்த பயன்பாட்டு விட்ஜெட் நெட்வொர்க்கில் உள்ள பயனர்கள் மற்றும் தளங்களின் சுருக்கமான சுருக்கத்தைக் காட்டுகிறது. மேலே உள்ள எளிய குறுக்குவழிகள் புதிய தளங்கள் அல்லது பயனர்களை ஒரே கிளிக்கில் உருவாக்க அனுமதிக்கின்றன.

### செய்திகள் மற்றும் விவாதங்கள்

![செய்திகள் மற்றும் விவாதங்கள் விட்ஜெட்](/img/admin/network-dashboard-news.png)

இந்த விட்ஜெட் சமீபத்திய Ultimate Multisite செய்திகளைப் பெற்று காட்டுகிறது. புதுப்பிப்புகள், முக்கியமான பிழைகள் மற்றும் பாதுகாப்பு திருத்தங்கள் பற்றிய தகவல்களுக்கு இந்த விட்ஜெட்டைக் கவனியுங்கள்.

## Ultimate Multisite டாஷ்போர்ட்

நெட்வொர்க் நிலை டாஷ்போர்ட் நெட்வொர்க் பற்றிய முழுமையான தகவல்களை வழங்கும் இடத்தில், Ultimate Multisite டாஷ்போர்ட் (Ultimate Multisite மெனுவின் மேல் நிலையில் அமைந்துள்ளது) சேவை பற்றிய தகவல்களை வழங்குகிறது.

![Ultimate Multisite டாஷ்போர்ட்](/img/admin/um-dashboard-top.png)

### மாதாந்திர தொடர் வருவாய் வளர்ச்சி

MRR (மாதாந்திர தொடர் வருவாய்) விளக்கப்படம் 12 மாத காலண்டர் காலத்தில் வருவாயின் மாதாந்திர பிரிவை காட்டுகிறது.

![MRR வளர்ச்சி விளக்கப்படம்](/img/admin/um-dashboard-mrr.png)

புதிய வருவாய் நீல நிறத்தில் கண்காணிக்கப்படுகிறது, அதே நேரத்தில் ரத்துகள் சிவப்பு நிறத்தில் பிரதிபலிக்கின்றன.

### தேதி வரம்பு ஆதரவு

வாடிக்கையாளர்கள் மற்றும் செயல்பாடுகளின் எண்ணிக்கையைப் பொறுத்து, புள்ளிவிவரங்கள் அதிகமாக இருக்கலாம். டாஷ்போர்டில் உள்ளமைக்கப்பட்ட தேதி வரம்பு ஆதரவு, நெட்வொர்க் நிர்வாகி தேவையான தகவல்கள் மற்றும் காலகட்டத்தில் மட்டும் கவனம் செலுத்த அனுமதிக்கிறது.

![தேதி வரம்பு தேர்வி மற்றும் புள்ளிவிவரங்கள்](/img/admin/um-dashboard-date-range.png)

தேதி வரம்பு, நேரடியாக கீழே உள்ள விட்ஜெட்களால் காட்டப்படும் தகவல்களின் காலத்தை நிர்வகிக்கிறது, இதில் வருவாய், புதிய உறுப்பினர்கள், நாடு வாரியான பதிவுகள், படிவம் வாரியான பதிவுகள் மற்றும் அதிகம் பார்வையிடப்பட்ட தளங்கள் ஆகியவை அடங்கும்.

![டாஷ்போர்ட் புள்ளிவிவர விட்ஜெட்கள்](/img/admin/um-dashboard-stats.png)

:::tip
**அதிகம் பார்வையிடப்பட்ட தளங்கள்** விட்ஜெட் வேலை செய்ய, **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** இல் பார்வை எண்ணிக்கையை இயக்கவும்.
:::
