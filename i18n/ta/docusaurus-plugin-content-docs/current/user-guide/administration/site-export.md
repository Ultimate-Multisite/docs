# தளத்தை ஏற்றுமதி செய்தல் (Site Export)

**Site Export** நிர்வாகப் பக்கம், நெட்வொர்க் நிர்வாகிகளால் ஒரு தளத்தை அல்லது முழு நெட்வொர்க்கையும் இடம்பெயர்வு (migration), காப்புப் பிரதி (backup) அல்லது ஒப்படைப்புப் பணிகளுக்காக (hand-off work) பதிவிறக்கக்கூடிய ஒரு ஆர்கைவாக (archive) தொகுக்க உதவுகிறது.

## ஒரு தளத்தை ஏற்றுமதி செய்தல் (Exporting one site)

**Ultimate Multisite > Site Export** என்பதற்குச் சென்று, **Generate new Site Export** என்பதைத் தேர்ந்தெடுக்கவும். நீங்கள் ஏற்றுமதி செய்ய விரும்பும் துணைத் தளத்தைத் (subsite) தேர்வு செய்யவும், பின்னர் ஆர்கைவில் பதிவேற்றங்கள் (uploads), plugins மற்றும் themes சேர்க்க வேண்டுமா என்பதைத் தேர்வு செய்யவும்.

ஏற்றுமதி முடிந்ததும், **Existing Exports** பட்டியலில் இருந்து ZIP கோப்பைப் பதிவிறக்கவும். இப்போது ஏற்றுமதி செய்யப்படும் ZIP கோப்புகள் ஒரு சுய-பூட் செய்யும் `index.php` மற்றும் ஒரு `readme.txt` கோப்பைக் கொண்டிருக்கும். இதனால், இந்த ஆர்கைவை ஒரு புதிய ஹோஸ்டில் நிறுவி, தனிப்பட்ட இம்போர்ட்டர் plugin-ஐ முதலில் நிறுவ வேண்டிய அவசியம் இல்லாமல் தொடங்க முடியும்.

## முழு நெட்வொர்க்கையும் ஏற்றுமதி செய்தல் (Exporting the whole network)

நெட்வொர்க்கில் உள்ள அனைத்து துணைத் தளங்களையும் ஒரே ஆர்கைவாகக் கொண்டிருக்க வேண்டியிருக்கும் போது, Site Export பக்கத்தில் **Network Export** பயன்படுத்தவும். ஒவ்வொரு துணைத் தளமும் ஒன்றாகவே பயணிக்க வேண்டியிருக்கும் ஹோஸ்ட் இடம்பெயர்வு (host migrations), பேரழிவு மீட்புப் பயிற்சிகள் (disaster-recovery drills) அல்லது ஸ்டேஜிங் மறுசீரமைப்புகள் (staging rebuilds) போன்ற சமயங்களில் இது மிகவும் பயனுள்ளதாக இருக்கும்.

நெட்வொர்க் ஏற்றுமதி என்பது ஒற்றைத் தள ஏற்றுமதியை விட மிக அதிகமாக இருக்கலாம் என்பதால், குறைந்த போக்குவரத்து நேரத்தின்போது இதை இயக்கவும். மேலும், பதிவேற்றங்கள், plugins, themes மற்றும் உருவாக்கப்பட்ட ZIP கோப்புகளுக்கு இலக்கு சேமிப்பகத்தில் (target storage) போதுமான இலவச இடம் உள்ளதா என்பதை உறுதிப்படுத்தவும்.

### நெட்வொர்க் இம்போர்ட் பண்டில்கள் (Network Import Bundles)

Ultimate Multisite 2.12.0 இலிருந்து, Site Exporter ஆனது **network import bundles** எனப்படும் சிறப்பு ஆர்கைவ்களை உருவாக்க முடியும். இவை நெட்வொர்க் முழுவதும் உள்ள தளங்களை சீராக மீட்டெடுக்க (restore) வடிவமைக்கப்பட்டவை. ஒரு நெட்வொர்க் இம்போர்ட் பண்டில் என்பது பல தளங்களை ஒரு புதிய நெட்வொர்க் நிறுவலில் மீட்டெடுக்கத் தேவையான அனைத்து கோப்புகள் மற்றும் மெட்டாடேட்டாவையும் கொண்டுள்ளது.

#### நெட்வொர்க் இம்போர்ட் பண்டிலை உருவாக்குதல் (Generating a Network Import Bundle)

1. **Ultimate Multisite > Site Export** என்பதற்குச் செல்லவும்
2. **Generate new Network Export** என்பதைக் கிளிக் செய்யவும்
3. ஏற்றுமதி வகையாக **Network Import Bundle** என்பதைத் தேர்ந்தெடுக்கவும்
4. பதிவேற்றங்கள், plugins மற்றும் themes சேர்க்க வேண்டுமா என்பதைத் தேர்வு செய்யவும்
5. **Generate** என்பதைக் கிளிக் செய்யவும்
6. **Existing Exports** பட்டியலில் இருந்து பண்டில் ZIP கோப்பைப் பதிவிறக்கவும்

#### நெட்வொர்க் இம்போர்ட் பண்டிலில் இருந்து மீட்டெடுத்தல் (Restoring from a Network Import Bundle)

ஒரு நெட்வொர்க் இம்போர்ட் பண்டில் இலிருந்து தளங்களை மீட்டெடுக்க:

1. உங்கள் இலக்கு ஹோஸ்டில் Ultimate Multisite-ஐ நிறுவவும்
2. மல்ட்டிசைட் அமைப்பு விஸார்டை (multisite setup wizard) முடிக்கவும்
3. புதிய நெட்வொர்க்கில் **Ultimate Multisite > Site Export** என்பதற்குச் செல்லவும்
4. **Import Network Bundle** என்பதைக் கிளிக் செய்யவும்
5. நெட்வொர்க் இம்போர்ட் பண்டில் ZIP கோப்பை பதிவேற்றவும்
6. திரையில் தோன்றும் இம்போர்ட் வழிமுறைகளைப் பின்பற்றவும்
7. இம்போர்ட் செயல்முறை அனைத்து தளங்களையும், அவற்றின் உள்ளடக்கத்தையும் மற்றும் அமைப்புகளையும் மீட்டெடுக்கும்.

நெட்வொர்க் இம்போர்ட் பண்டில்கள் பின்வருவனவற்றை பாதுகாக்கும்:
- தள உள்ளடக்கம் (posts, pages, custom post types)
- தள அமைப்புகள் மற்றும் விருப்பங்கள் (Site settings and options)
- பயனர் பாத்திரங்கள் மற்றும் அனுமதிகள் (User roles and permissions)
- plugins மற்றும் themes (பண்டிலில் சேர்க்கப்பட்டிருந்தால்)
- மீடியா பதிவேற்றங்கள் (பண்டிலில் சேர்க்கப்பட்டிருந்தால்)
- தனிப்பயன் தரவுத்தள அட்டவணைகள் மற்றும் தரவு (Custom database tables and data)

## சுய-பூட் செய்யும் ஏற்றுமதி ZIP-ஐ நிறுவுதல் (Installing a self-booting export ZIP)

ஒரு புதிய ஹோஸ்டில் சுய-பூட் செய்யும் ZIP-ஐ மீட்டெடுக்க:

1. ஏற்றுமதி செய்யப்பட்ட ZIP உள்ளடக்கங்களை இலக்கு வெப் ரூட்டில் (target web root) பதிவேற்றவும்.
2. பதிவேற்றப்பட்ட `index.php`-ஐ ஒரு உலாவியில் (browser) திறக்கவும்.
3. பண்டில் செய்யப்பட்ட ஏற்றுமதி தொகுப்பிலிருந்து (bundled export package) திரையில் தோன்றும் நிறுவி வழிமுறைகளைப் பின்பற்றவும்.
4. தற்காலிக கோப்புகளை நீக்குவதற்கு முன், பண்டில் செய்யப்பட்ட `readme.txt`-ஐ ஏற்றுமதி தொடர்பான குறிப்புகளுக்காக மதிப்பாய்வு செய்யவும்.

addon-குறிப்பிட்ட நிறுவல் மற்றும் இறக்குமதி விவரங்களுக்கு, [Site Exporter addon documentation](/addons/site-exporter)-ஐப் பார்க்கவும்.
