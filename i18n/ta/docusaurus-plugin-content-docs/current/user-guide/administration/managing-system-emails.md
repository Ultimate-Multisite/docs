# கணினி மின்னஞ்சல்களை நிர்வகித்தல்

பதிவு உறுதிப்படுத்தல்கள், கட்டண ரசீதுகள், மற்றும் உறுப்பினர் மாற்றங்கள் போன்ற நிகழ்வுகளுக்காக Ultimate Multisite பல்வேறு கணினி மின்னஞ்சல்களை அனுப்புகிறது. உங்கள் பிராண்டுக்கு பொருந்தும் வகையில் இந்த மின்னஞ்சல் வார்ப்புருக்களைத் தனிப்பயனாக்கலாம்.

## கணினி மின்னஞ்சல்களைப் பார்வையிடுதல்

மின்னஞ்சல் தொடர்பான அமைப்புகளைப் பார்க்க **Ultimate Multisite → Settings**-க்கு சென்று **Emails** தாவலைக் கிளிக் செய்யவும்.

![Settings - Emails தாவல்](/img/admin/settings-emails.png)

அங்கிருந்து, அனைத்து மின்னஞ்சல் வார்ப்புருக்களையும் பார்க்க **System Emails** பக்கத்தை அணுகலாம்.

![System Emails பட்டியல்](/img/admin/system-emails-list.png)

கணினி மின்னஞ்சல் திருத்தியைப் பயன்படுத்தி தனிப்பட்ட மின்னஞ்சல் வார்ப்புருக்களையும் திருத்தலாம்:

![கணினி மின்னஞ்சல் திருத்தி](/img/admin/system-email-editor.png)

## புதிய மின்னஞ்சல் வார்ப்புருக்களை இறக்குமதி செய்தல்

புதிய Ultimate Multisite பதிப்புகள் மற்றும் add-ons புதிய மின்னஞ்சல் வார்ப்புருக்களைப் பதிவு செய்யலாம். முரண்பாடுகளைத் தவிர்க்க, புதிய வார்ப்புருக்கள் தானாக சேர்க்கப்படாது — அவற்றை கைமுறையாக இறக்குமதி செய்ய வேண்டும்.

புதிய மின்னஞ்சல் வார்ப்புருக்களை இறக்குமதி செய்ய:

1. **System Emails** பக்கத்திற்குச் செல்லவும்
2. பக்கத்தின் மேல்பகுதியில் உள்ள **Reset & Import** பொத்தானைக் கிளிக் செய்யவும்
3. கிடைக்கும் வார்ப்புருக்களைப் பார்க்க **Import Emails** விருப்பத்தை மாற்றவும்
4. இறக்குமதி செய்ய விரும்பும் வார்ப்புருக்களைத் தேர்ந்தெடுத்து உறுதிப்படுத்தவும்

## மின்னஞ்சல் வார்ப்புருக்களை மீட்டமைத்தல்

நீங்கள் ஒரு மின்னஞ்சல் வார்ப்புருவைத் தனிப்பயனாக்கியிருந்தால், அதை இயல்புநிலை உள்ளடக்கத்துக்கு மீண்டும் மாற்ற விரும்பினால், உங்களுக்கு இரண்டு விருப்பங்கள் உள்ளன:

1. **நீக்கி மீண்டும் இறக்குமதி செய்தல்** — கணினி மின்னஞ்சலை நீக்கி அதை மீண்டும் இறக்குமதி செய்யவும் (இது அனுப்பல் அளவீடுகளை அழிக்கும்)
2. **கருவி மூலம் மீட்டமைத்தல்** — **Reset & Import** கருவியைப் பயன்படுத்தி, **Reset** விருப்பத்தை மாற்றி, மீட்டமைக்க விரும்பும் மின்னஞ்சல்களைத் தேர்ந்தெடுக்கவும்

இயல்புநிலை உள்ளடக்கத்தை மீட்டெடுக்கும் போது அனுப்பல் அளவீடுகளையும் பிற மெட்டாடேட்டாவையும் பாதுகாப்பதால் Reset விருப்பம் பரிந்துரைக்கப்படுகிறது.
