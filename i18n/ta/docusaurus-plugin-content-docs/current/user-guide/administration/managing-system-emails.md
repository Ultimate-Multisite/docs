# சிஸ்டம் மின்னஞ்சல்களை நிர்வகித்தல்

Ultimate Multisite பதிவு உறுதிப்படுத்தல்கள், கட்டண ரசீதுகள், மற்றும் membership மாற்றங்கள் போன்ற நிகழ்வுகளுக்கு பல்வேறு சிஸ்டம் மின்னஞ்சல்களை அனுப்புகிறது. இந்த மின்னஞ்சல் வார்ப்புருக்களை உங்கள் brand-க்கு ஏற்றவாறு தனிப்பயனாக்கலாம்.

## சிஸ்டம் மின்னஞ்சல்களைப் பார்த்தல்

**Ultimate Multisite → Settings**-க்கு சென்று, மின்னஞ்சல் தொடர்பான அமைப்புகளைக் காண **Emails** tab-ஐ கிளிக் செய்யவும்.

![Settings - Emails tab](/img/admin/settings-emails.png)

அங்கிருந்து, அனைத்து மின்னஞ்சல் வார்ப்புருக்களையும் பார்க்க **System Emails** பக்கத்தை அணுகலாம்.

![System Emails list](/img/admin/system-emails-list.png)

## புதிய மின்னஞ்சல் வார்ப்புருக்களை இறக்குமதி செய்தல்

புதிய Ultimate Multisite பதிப்புகள் மற்றும் add-on-கள் புதிய மின்னஞ்சல் வார்ப்புருக்களைப் பதிவு செய்யலாம். முரண்பாடுகளைத் தவிர்க்க, புதிய வார்ப்புருக்கள் தானாகச் சேர்க்கப்படாது — நீங்கள் அவற்றை கைமுறையாக இறக்குமதி செய்ய வேண்டும்.

புதிய மின்னஞ்சல் வார்ப்புருக்களை இறக்குமதி செய்ய:

1. **System Emails** பக்கத்திற்குச் செல்லவும்
2. பக்கத்தின் மேலே உள்ள **Reset & Import** பொத்தானை கிளிக் செய்யவும்
3. கிடைக்கும் வார்ப்புருக்களைக் காண **Import Emails** விருப்பத்தை நிலைமாற்றவும்
4. நீங்கள் இறக்குமதி செய்ய விரும்பும் வார்ப்புருக்களைத் தேர்ந்தெடுத்து உறுதிப்படுத்தவும்

## மின்னஞ்சல் வார்ப்புருக்களை மீட்டமைத்தல்

நீங்கள் ஒரு மின்னஞ்சல் வார்ப்புருவைத் தனிப்பயனாக்கியிருந்தால், இயல்புநிலை உள்ளடக்கத்திற்குத் திரும்ப இரண்டு வழிகள் உள்ளன:

1. **நீக்கி மீண்டும் இறக்குமதி செய்தல்** — சிஸ்டம் மின்னஞ்சலை நீக்கிவிட்டு மீண்டும் இறக்குமதி செய்யவும் (இது அனுப்பிய புள்ளிவிவரங்களை அழிக்கும்)
2. **கருவி மூலம் மீட்டமைத்தல்** — **Reset & Import** கருவியைப் பயன்படுத்தி, **Reset** விருப்பத்தை நிலைமாற்றி, நீங்கள் மீட்டமைக்க விரும்பும் மின்னஞ்சல்களைத் தேர்ந்தெடுக்கவும்

Reset விருப்பமே சிறந்தது, ஏனெனில் இது இயல்புநிலை உள்ளடக்கத்தை மீட்டெடுக்கும் போது அனுப்பிய புள்ளிவிவரங்கள் மற்றும் பிற metadata-வைப் பாதுகாக்கிறது.
