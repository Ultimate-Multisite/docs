# WPMU DEV ஒருங்கிணைப்பு

## மேலோட்டப் பார்வை
WPMU DEV என்பது WordPress தளங்களுக்கான ஹோஸ்டிங், plugins மற்றும் சேவைகளை வழங்கும் ஒரு முழுமையான WordPress தளமாகும். இந்த ஒருங்கிணைப்பு Ultimate Multisite மற்றும் WPMU DEV ஹோஸ்டிங் இடையே தானியங்கு domain ஒத்திசைவு மற்றும் SSL சான்றிதழ் மேலாண்மையை செயல்படுத்துகிறது.

## அம்சங்கள்
- தானியங்கு domain ஒத்திசைவு
- SSL சான்றிதழ் மேலாண்மை
- விரிவுபடுத்தப்பட்ட SSL சான்றிதழ் சரிபார்ப்பு முயற்சிகள்

## தேவைகள்
நீங்கள் WPMU DEV-ல் ஹோஸ்ட் செய்கிறீர்களா என்பதை இந்த ஒருங்கிணைப்பு தானாகவே கண்டறிந்து உள்ளமைந்த API-ஐப் பயன்படுத்துகிறது. நீங்கள் WPMU DEV-ல் ஹோஸ்ட் செய்கிறீர்கள் என்றால் கூடுதல் கட்டமைப்பு தேவையில்லை.

இந்த ஒருங்கிணைப்பு `WPMUDEV_HOSTING_SITE_ID` constant இருக்கிறதா என்று சரிபார்க்கிறது, இது WPMU DEV-ல் ஹோஸ்ட் செய்யும்போது தானாகவே வரையறுக்கப்படும்.

## அமைவு வழிமுறைகள்

### 1. WPMU DEV ஹோஸ்டிங்கை சரிபார்க்கவும்

நீங்கள் WPMU DEV-ல் ஹோஸ்ட் செய்கிறீர்கள் என்றால், தேவையான constants ஏற்கனவே வரையறுக்கப்பட்டிருக்கும். பின்வருவனவற்றை சரிபார்க்கவும்:

1. உங்கள் சூழலில் `WPMUDEV_HOSTING_SITE_ID` constant வரையறுக்கப்பட்டுள்ளது
2. API அணுகலுடன் செயலில் உள்ள WPMU DEV உறுப்பினர் தகுதி உங்களிடம் உள்ளது

### 2. ஒருங்கிணைப்பை இயக்கவும்

1. உங்கள் WordPress admin-ல், Ultimate Multisite > Settings என்பதற்குச் செல்லவும்
2. "Domain Mapping" tab-க்கு செல்லவும்
3. "Host Integrations" பகுதிக்கு கீழே உருட்டவும்
4. WPMU DEV ஒருங்கிணைப்பை இயக்கவும்
5. "Save Changes" என்பதைக் கிளிக் செய்யவும்

## இது எவ்வாறு செயல்படுகிறது

### Domain ஒத்திசைவு

Ultimate Multisite-ல் ஒரு domain மேப் செய்யப்படும்போது:

1. இந்த ஒருங்கிணைப்பு WPMU DEV API-ஐப் பயன்படுத்தி உங்கள் ஹோஸ்டிங் கணக்கில் domain-ஐச் சேர்க்கிறது
2. domain-ன் www பதிப்பையும் தானாகவே சேர்க்கிறது
3. WPMU DEV domain கட்டமைப்பு மற்றும் SSL சான்றிதழ் வழங்குதலைக் கையாளுகிறது

### SSL சான்றிதழ் மேலாண்மை

WPMU DEV ஹோஸ்டிங்கிற்கான SSL சான்றிதழ் சரிபார்ப்பு முயற்சிகளின் எண்ணிக்கையை அதிகரிக்க இந்த ஒருங்கிணைப்பு கட்டமைக்கப்பட்டுள்ளது, ஏனெனில் SSL சான்றிதழ்கள் வழங்கப்பட்டு நிறுவப்படுவதற்கு சிறிது நேரம் ஆகலாம். இயல்புநிலையாக, இது SSL சான்றிதழ் சரிபார்ப்புக்கு நிலையான 5 முயற்சிகளுக்கு பதிலாக 10 முறை வரை முயற்சிக்கும்.

## முக்கியமான குறிப்புகள்

### Domain நீக்குதல்

தற்போது, WPMU DEV API domains-ஐ நீக்குவதற்கான வழியை வழங்கவில்லை. Ultimate Multisite-ல் ஒரு domain mapping நீக்கப்படும்போது, அந்த domain உங்கள் WPMU DEV ஹோஸ்டிங் கணக்கில் இருக்கும். தேவைப்பட்டால் WPMU DEV ஹோஸ்டிங் dashboard-லிருந்து அதை கைமுறையாக நீக்க வேண்டும்.

### API அங்கீகாரம்

இந்த ஒருங்கிணைப்பு உங்கள் WordPress தரவுத்தளத்தில் `wpmudev_apikey` option ஆக சேமிக்கப்பட்டுள்ள WPMU DEV API key-ஐப் பயன்படுத்துகிறது. உங்கள் தளத்தை WPMU DEV-உடன் இணைக்கும்போது இது தானாகவே அமைக்கப்படும்.

## சிக்கல் தீர்வு

### API இணைப்பு சிக்கல்கள்
- உங்கள் தளம் WPMU DEV-உடன் சரியாக இணைக்கப்பட்டுள்ளதை சரிபார்க்கவும்
- உங்கள் WordPress தரவுத்தளத்தில் `wpmudev_apikey` option அமைக்கப்பட்டுள்ளதை சரிபார்க்கவும்
- உங்கள் WPMU DEV உறுப்பினர் தகுதி செயலில் உள்ளதை உறுதிசெய்யவும்

### SSL சான்றிதழ் சிக்கல்கள்
- WPMU DEV SSL சான்றிதழ்களை வழங்க சிறிது நேரம் ஆகலாம் (பொதுவாக 5-15 நிமிடங்கள்)
- இந்த ஒருங்கிணைப்பு SSL சான்றிதழ்களுக்கு 10 முறை வரை சரிபார்க்க கட்டமைக்கப்பட்டுள்ளது
- பல முயற்சிகளுக்குப் பிறகும் SSL சான்றிதழ்கள் வழங்கப்படவில்லை என்றால், WPMU DEV ஆதரவைத் தொடர்பு கொள்ளவும்

### Domain சேர்க்கப்படவில்லை
- ஏதேனும் பிழை செய்திகளுக்கு Ultimate Multisite logs-ஐ சரிபார்க்கவும்
- அந்த domain ஏற்கனவே WPMU DEV-ல் சேர்க்கப்படவில்லை என்பதை சரிபார்க்கவும்
- நீங்கள் சேர்க்கும் domains எண்ணிக்கையை உங்கள் WPMU DEV ஹோஸ்டிங் திட்டம் ஆதரிக்கிறதா என்பதை உறுதிசெய்யவும்
