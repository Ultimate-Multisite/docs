# Ultimate Multisite Hooks மற்றும் filters

Ultimate Multisite பல functions மற்றும் database queries கொண்டு உருவாக்கப்பட்டுள்ளது. இதில் hooks மற்றும் filters உள்ளன, அவை ஏற்கனவே உள்ள functions-ஐ மாற்றவோ அல்லது புதிய code சேர்க்கவோ உங்களை அனுமதிக்கும்.

மேம்பட்ட பயனர்கள் தங்கள் network-ஐ மேலும் தனிப்பயனாக்க உதவும் வகையில், இந்த hooks மற்றும் filters-ன் பட்டியலை ஒரு குறிப்பாக நாங்கள் உருவாக்கியுள்ளோம். கீழே உள்ள link-ஐ கிளிக் செய்து Github-ல் இருந்து இந்த பட்டியலை அணுகலாம்.
