# PRO Theme ilə Admin Panelinizi Özelleştirmək

Müştəri işləri zamanı tez-tez WordPress admin panelində müştərilərə özəl bir təcrübə vermək lazım olur. Əgər daha peşəkar görünüş və hisslər axtarırsınızsa, PRO Theme sizə kömək edə bilər.

PRO Theme, müştərilərin Dashboard-larını özelleşdirməyiniz zaman sizə kömək etmək üçün hazırlanmış bir admin theme-dir. O, həmçinin Ultimate Multisite-ın premium pluginlərinin əsas hissəsidir.

Quraşdırdıqdan sonra, _Plugins_ səhifəsinə gedin və onu Network-ünüzdə aktivləşdirmək üçün klikləyin.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Network-də aktiv edin_

_**QEYD:**_ _Əgər sadəcə quraşdırsanız, o sizin müştəriləriniz üçün bir seçim kimi görünəcək ki, bu seçimi yalnız həmin istifadəçiyə tema əlavə etmək üçün işarə edə bilərsiniz._

İndi, admin interfeysiniz tamamilə yenidən dizayn edildi! Həmçinin sol sütunda PRO Theme adlı yeni bir menyu əldə edirsiniz.
