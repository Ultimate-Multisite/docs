# Boshqa Pluginlarga Mümkin Bo'lmaganliklar {#known-incompatibilities-with-other-plugins}

Ultimate Multisite ni o'rnatgan va faollashtirganingizdan so'ng, boshqa pluginlar bilan hal etilmagan mos kelmasliklar tufayli muammolarga duch kelishingiz mumkin.

Biz har bir yangi chiqarishda bu ro'yxatni kamaytirish uchun faol ishlayapmiz, ammo agar biror plugin shu yerda ro'yxatdan o'tgan bo'lsa, to'liq funksional Ultimate Multisite tarmoqini olish uchun uni o'chirish kerak bo'lishi mumkin.

Mana ularning ro'yxati va bunday holatlarda nima qilish kerakligi haqida ma'lumot:

**Ba'zi WPMU DEV pluginlari** Ba'zi pluginlar endi WPMU DEV tomonidan texnik qo'llab-quvvatlanmaydi va ular Ultimate Multisite ro'yxatdan o'tish jarayonini oxirgi bosqichda (foydalanuvchi hisob yarata olmaydi) to'xtatishi mumkinligi ma'lum. Bu muammolarga sabab bo'ladigan pluginlar ro'yxati:

  * Signup Language: _Ultimate Multisite ning shu funksionallikni taqdim etadigan qo'shimcha mavjud, bu Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Bu Ultimate Multisite ichiga kiritilgan, shuning uchun qo'shimcha pluginga ehtiyoj yo'q._

  * New Blog Templates: _Bu Ultimate Multisite ichiga kiritilgan, shuning uchun qo'shimcha pluginga ehtiyoj yo'q._

  * Set Password on Multisite Blog Creation: _Bu Ultimate Multisite ichiga kiritilgan, shuning uchun qo'shimcha pluginga ehtiyoj yo'q._

  * Signup TOS: _Bu Ultimate Multisite ichiga kiritilgan, shuning uchun qo'shimcha pluginga ehtiyoj yo'q._

  * Anti-Splog: Garchi bu kam kuchli bo'lsa ham, u ham _Ultimate Multisite ichiga kiritilgan._

**Eslatma**: Boshqa barcha WPMU Dev pluginlarini Ultimate Multisite bilan birga normal tarzda ishlatish mumkin. Misollar: _Smush_, _Forminator_, _Defender_ va h.k.
