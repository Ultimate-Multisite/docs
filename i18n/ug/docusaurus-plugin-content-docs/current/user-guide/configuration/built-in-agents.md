# ئىچىگە ئورنىتىلغان Agents {#built-in-agents}

Gratis AI Agent v1.9.0 بەش دانە ئىچىگە ئورنىتىلغان agent بىلەن كېلىدۇ؛ ھەر بىرى مەلۇم ساھەدىكى ئادەتتىكى ۋەزىپىلەرگە ماسلاشتۇرۇلغان قوراللار توپلىمى، خاسلاشتۇرۇلغان system prompt ۋە باشلاش تەكلىپلىرى بىلەن ئالدىن تەڭشەلگەن. Agents ئارىسىدا ئالماشتۇرۇش ياردەمچىنىڭ نېمە قىلالايدىغانلىقى ۋە قانداق جاۋاب بېرىدىغانلىقىنى ئۆزگەرتىدۇ — سىز تەرەپتىن ھېچقانداق تەڭشەش تەلەپ قىلىنمايدۇ. Superdav AI Agent v1.18.0 مۇناسىۋەتلىك بىرىكتۈرمىلەر تەڭشەلگەندە بۇ خىزمەت ئېقىملىرىغا ۋاقىت جەدۋىلىدىن خەۋەردار قوراللار، ئەسكەرتىش خاتىرىلىرى، تەستىق دەرۋازىلىرى ۋە SMS ئۇقتۇرۇشلىرىنى قوشالايدۇ.

## Agent دېگەن نېمە؟ {#what-is-an-agent}

ھەر بىر agent تۆۋەندىكىلەرنى بىرلەشتۈرگەن ئاتالغان تەڭشەك ئارخىپىدۇر:

- **قوراللار** — agent چاقىرىشقا رۇخسەت قىلىنغان ئىقتىدارلار (مەسىلەن، مەزمۇن يازغۇچىسى يازما قۇرۇش ئىقتىدارلىرىغا ئېرىشەلەيدۇ؛ لايىھە ستۇدىيىسى CSS ۋە theme.json ئىقتىدارلىرىغا ئېرىشەلەيدۇ)
- **System prompt** — agent نىڭ ئاھاڭى، ئالدىنقى ئورنى ۋە چەكلىمىلىرىنى بەلگىلەيدىغان كۆرسەتمىلەر
- **تەكلىپلەر** — تېز باشلىشىڭىزغا ياردەم بېرىش ئۈچۈن سۆھبەت ئارايۈزىدە كۆرسىتىلىدىغان ئالدىن يېزىلغان prompt لار

## Agent Picker غا كىرىش {#accessing-the-agent-picker}

1. WordPress باشقۇرغۇچ يان ستونىدىن **Gratis AI Agent** تاختىسىنى ئېچىڭ.
2. سۆھبەت باش قىسمىنىڭ سول ئۈستىدىكى **agent سىنبەلگىسى**نى چېكىڭ (سىنبەلگە ئاكتىپ agent نى ئەكس ئەتتۈرۈش ئۈچۈن ئۆزگىرىدۇ).
3. **Agent Picker** فورما-جەدۋەل ئۈستى قەۋىتى سۈپىتىدە ئېچىلىدۇ. ھەر بىر agent ئۆزىنىڭ سىنبەلگىسى، نامى ۋە بىر قۇرلۇق چۈشەندۈرۈشى بىلەن تىزىلىدۇ.
4. بىر agent قۇرىنى چېكىپ ئۇنى ئاكتىپلاڭ. سۆھبەت باش قىسمى دەرھال يېڭىلىنىدۇ.

سۆھبەت جەريانىدا agents نى ئالماشتۇرالايسىز — يېڭى agent نىڭ system prompt ى كېيىنكى ئۇچۇردىن باشلاپ كۈچكە ئىگە بولىدۇ.

## بەش دانە ئىچىگە ئورنىتىلغان Agents {#the-five-built-in-agents}

### مەزمۇن يازغۇچىسى {#content-writer}

**مۇھىم نۇقتا:** يازمىلار، بەتلەر ۋە ئالاقە فورمىلىرىنى قۇرۇش ۋە تەھرىرلەش.

**ئىشلەتكىلى بولىدىغان قوراللار:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Superdav AI Agent v1.18.0 بىرىكتۈرمىلىرى قوزغىتىلغاندا، تەڭشەلگەن كالېندار مەزمۇنى، تەستىق دەرۋازىلىرى، ئەسكەرتىشلەر ۋە SMS ئۇقتۇرۇش قوراللىرى تەستىقلانغان خىزمەت ئېقىملىرىغا ئىشلىتىلىشى مۇمكىن.

**ياخشى قىلالايدىغانلىرى:**
- قىسقىچە چۈشەندۈرۈش ياكى لايىھەدىن بىلوگ يازمىلىرىنى دەسلەپكى نۇسخىدا يېزىش ۋە ئېلان قىلىش
- يېڭى تور بېكەت ئۈچۈن landing page لار توپلاملىرىنى قۇرۇش
- ئالاقە ۋە سوراق فورمىلىرىنى ياساش
- URL ياكى ئىزدەش ئارقىلىق يازمىلارغا featured image تەڭشەش
- تەڭشەلگەن Google Calendar مەزمۇنىدىن پائالىيەتتىن كېيىنكى ئۇچۇرلارنى دەسلەپكى نۇسخىدا يېزىش، ئاندىن ئۇقتۇرۇش ئەۋەتىشتىن بۇرۇن تەستىقنى كۈتۈش

**باشلاش تەكلىپلىرى:**
- *WordPress multisite نىڭ پايدىلىرى ھەققىدە 500 سۆزلۈك بىلوگ يازمىسى يېزىڭ.*
- *ھەققىدە، مۇلازىمەتلەر ۋە ئالاقە بېتىنى قۇرۇپ ئېلان قىلىڭ.*
- *ئالاقە بېتىگە زاكاس سوراش فورمىسى قوشۇڭ.*
- *ئەتىدىكى تەڭشەلگەن كالېندار پائالىيىتىدىكى قاتناشقۇچىلار ئۈچۈن ئەسكەرتىشنى دەسلەپكى نۇسخىدا يېزىڭ ۋە ئەۋەتىشتىن بۇرۇن تەستىقنى كۈتۈڭ.*

### تور بېكەت قۇرغۇچى {#site-builder}

**مۇھىم نۇقتا:** بىرلا prompt دىن باشتىن-ئاخىر تور بېكەت قۇرۇش.

**ئىشلەتكىلى بولىدىغان قوراللار:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Superdav AI Agent v1.18.0 بىلەن، باشقۇرغۇچىلار قوزغىتىدىغان جايلاردا تەڭشەلگەن باشقۇرۇلىدىغان-مۇلازىمەت، تەستىق، ئەسكەرتىش، كالېندار ۋە SMS قوراللىرى ئىشلىتىلىشى مۇمكىن.

**ياخشى قىلالايدىغانلىرى:**
- تەسۋىرلەنگەن سودا تۈرى ئۈچۈن كۆپ باسقۇچلۇق تور بېكەت قۇرۇش پىلانى ھاسىل قىلىش
- ھەر بىر باسقۇچنى ئۆزلۈكىدىن ئىجرا قىلىش — قۇرۇلما، مەزمۇن، يولباشلاش، لايىھە
- پىلان ئوتتۇرىسىدىكى خاتالىقلاردىن قولدا ئارىلىشىش تەلەپ قىلماي ئەسلىگە كېلىش
- قۇرۇشنىڭ بىر قىسمى سۈپىتىدە تەۋسىيە قىلىنغان plugin لارنى ئورنىتىش
- سۆھبەت ئارايۈزىدىن بىۋاسىتە ئالاقە فورمىلىرىنى قۇرۇش (Superdav AI Agent v1.10.0+)
- تەستىق دەرۋازىلىرى ۋە ئەسكەرتىش خاتىرىلىرى قوزغىتىلغاندا، تەكرار ئۇقتۇرۇش بولمايلا ئېلان قىلىش ئەسكەرتىشلىرى ياكى قاتناشقۇچىلارغا كېيىنكى ئالاقىنى ماسلاشتۇرۇش

**باشلاش تەكلىپلىرى:**
- *رەسىمچىلىك پورتفولىيو تور بېكىتىنى gallery post type، زاكاس بېتى ۋە ئالاقە فورمىسى بىلەن قۇرۇڭ.*
- *تور تىزىملىكى، ئېچىلىش سائەتلىرى ۋە ئۈستەل زاكاس سوراش فورمىسى بار رېستوران تور بېكىتى قۇرۇڭ.*
- *مۇلازىمەت بەتلىرى، پورتفولىيو بۆلۈمى ۋە بىلوگى بار ئەركىن مەسلىھەتچىلىك تور بېكىتى تەڭشەڭ.*
- *site builder نى ئىشلىتىپ ئالاقە بېتىگە ئالاقە فورمىسى قوشۇڭ.*
- *تور بېكەت ئېلان قىلىش تەكشۈرۈش تىزىملىكى تەستىقلانغاندىن كېيىن، تەڭشەلگەن ئالاقىدار تەرەپ ئالاقىسىگە SMS ئەسكەرتىشى ئەۋەتىڭ.*

### لايىھە ستۇدىيىسى {#design-studio}

**مۇھىم نۇقتا:** كۆرۈنۈشنى خاسلاشتۇرۇش — رەڭلەر، خەت شەكلى، CSS ۋە block pattern لار.

**ئىشلەتكىلى بولىدىغان قوراللار:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**ياخشى قىلالايدىغانلىرى:**
- ئاتالغان theme preset لارنى قوللىنىش (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- theme.json ئارقىلىق global typography ۋە رەڭ تاختىلىرىنى ئىنچىكە تەڭشەش
- ماركا-خاس قاپلاشلار ئۈچۈن custom CSS كىرگۈزۈش
- بەتنىڭ ئېكران سۈرىتىنى ئېلىپ، لايىھە مەسىلىلىرى بويىچە تەكشۈرۈش

**باشلاش تەكلىپلىرى:**
- *warm-editorial preset نى قوللىنىپ، ئاندىن ئاساسىي رەڭنى #2d6a4f قىلىپ تەڭشەڭ.*
- *باش بەتنىڭ ئېكران سۈرىتىنى ئېلىپ، نېمىنى ياخشىلايدىغانلىقىڭىزنى ئېيتىپ بېرىڭ.*
- *تولۇق كەڭلىكتىكى تەگلىك رەسىمى ۋە مەركەزلەشتۈرۈلگەن ماۋزۇسى بار قايتا ئىشلىتىلىدىغان hero block pattern قۇرۇڭ.*

### Plugin Manager {#plugin-manager}

**مۇھىم نۇقتا:** WordPress plugin لارنى بايقاش، ئورنىتىش ۋە باشقۇرۇش.

**ئىشلەتكىلى بولىدىغان قوراللار:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**ياخشى قىلالايدىغانلىرى:**
- تەسۋىرلەنگەن ئىشلىتىش ئەھۋالى ئۈچۈن ئەڭ ياخشى plugin نى تەۋسىيە قىلىش
- registry دىن ability pack لارنى ئورنىتىش
- بار بولغان ability catalogue نى تۈر بويىچە كۆرۈش

**باشلاش تەكلىپلىرى:**
- *ئەزالىق مۇندەرىجىسى ئۈچۈن ئەڭ ياخشى plugin قايسى؟*
- *WooCommerce abilities pack نى ئورنىتىڭ.*
- *بارلىق ecommerce ability pack لارنى كۆرسىتىڭ.*

### قوللاش ياردەمچىسى {#support-assistant}

**مۇھىم نۇقتا:** تور بېكەت مەزمۇنى، تەڭشەكلەر ۋە WordPress سەپلىمىسى ھەققىدىكى سوئاللارغا جاۋاب بېرىش.

**ئىشلەتكىلى بولىدىغان قوراللار:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**ياخشى قىلالايدىغانلىرى:**
- نۆۋەتتىكى تور بېكەت تەڭشەكلىرى ۋە تاللانمىلىرىنى ئىزدەپ تېپىش
- تور بېكەتتە قانداق يازما تۈرلىرى، تۈرلەندۈرۈش سىستېمىلىرى ۋە تىزىملىكلەر سەپلەنگەنلىكىنى چۈشەندۈرۈش
- نەق قىممەتلەرنى ئوقۇش ئارقىلىق «بۇ تەڭشەك نېمە قىلىدۇ؟» دېگەن سوئاللارغا جاۋاب بېرىش
- ئۆزگەرتىش كىرگۈزۈشتىن بۇرۇن ئوقۇشقىلا بولىدىغان دىياگنوستىكا قەۋىتى سۈپىتىدە خىزمەت قىلىش

**باشلاش تەكلىپلىرى:**
- *بۇ تور بېكەتتە ھازىر قايسى قىستۇرمىلار ۋە تەڭشەكلەر ئاكتىپ؟*
- *بۇ تور بېكەتتە تىزىمغا ئېلىنغان بارلىق خاس يازما تۈرلىرىنى تىزىپ بېرىڭ.*
- *قايسى يولباشلاش تىزىملىكلىرى مەۋجۇت ۋە ئۇلار قەيەرگە تەقسىملەنگەن؟*

## Superdav ئاپتوماتلاشتۇرۇش بىرلەشتۈرۈلۈشى {#superdav-automation-integrations}

Superdav AI Agent v1.18.0 بىرلەشتۈرۈلۈشلىرى سەپلەنگەندە، ئىچىگە ئورنىتىلغان ۋاكالەتچىلەر تېخىمۇ بىخەتەر، ۋاقىت جەدۋىلىنى نەزەردە تۇتىدىغان ئاپتوماتلاشتۇرۇش خىزمەت ئېقىملىرىغا قاتنىشالايدۇ:

- **Google Calendar ئوقۇش قوراللىرى** ۋاكالەتچىلەرگە داۋاملىق خىزمەتنى تەييارلاشتىن بۇرۇن سەپلەنگەن كالېندارلار ۋە پائالىيەتلەرنى تەكشۈرۈشكە يول قويىدۇ.
- **ئالاقىلىشىش كىشىلىرى ۋە قاتناشقۇچىلارنى ماسلاشتۇرۇش** پائالىيەت قاتناشقۇچىلىرىنى WordPress ئىشلەتكۈچىلىرى ياكى تونۇلغان ئالاقىلىشىش كىشىلىرى بىلەن ماسلاشتۇرۇشقا ياردەم بېرىدۇ.
- **ئىنسان تەستىق دەرۋازىلىرى** سەزگۈر ھەرىكەتلەرنى ھوقۇق بېرىلگەن ئىشلەتكۈچى كۆزدىن كەچۈرۈپ جەزملەشتۈرگۈچە توختىتىپ تۇرىدۇ.
- **ئەسكەرتىش خاتىرىلىرى** جەدۋەللىك ۋەزىپىلەر قايتا سىناپ باققاندا ياكى تەكرارلانغاندا تەكرار ئۇقتۇرۇشلارنىڭ ئالدىنى ئالىدۇ.
- **TextBee SMS ئۇقتۇرۇشلىرى** پەقەت SMS كىنىشكىلىرى ۋە خىزمەت ئېقىمى ئىجازەتلىرى قوزغىتىلغاندا سەپلەنگەن قىسقا ئۇچۇرلارنى ئەۋەتىدۇ.

تەۋسىيە قىلىنغان خىزمەت ئېقىمى: ۋاكالەتچىدىن ئۇچۇر ياكى ھەرىكەتنى تەييارلاشنى سوراڭ، تەستىق ئەسكەرتمىسىنى كۆزدىن كەچۈرۈڭ، ئاندىن تەستىقلانغان ھەرىكەتنىڭ داۋاملىشىشىغا يول قويۇڭ. تەكرارلىنىدىغان ئەسكەرتىشلەر ئۈچۈن، ئەسكەرتىش تەكرارلىنىشىنى يوقىتىشنى قوزغىتىپ قويۇڭ، شۇنداق بولغاندا ئوخشاش پائالىيەت ياكى ئالاقىلىشىش كىشىسى قايتا-قايتا ئۇقتۇرۇلمايدۇ.

## ۋاكالەتچىلەرنى خاسلاشتۇرۇش {#customising-agents}

ھەر بىر ئىچىگە ئورنىتىلغان ۋاكالەتچىنى `gratis_ai_agent_agents` filter ئارقىلىق كېڭەيتىشكە ياكى ئالماشتۇرۇشقا بولىدۇ.

### مەۋجۇت ۋاكالەتچىگە خاس سىستېما ئەسكەرتمىسى قوشۇش {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### يېڭى ۋاكالەتچىنى تىزىمغا ئېلىش {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

يېڭى ۋاكالەتچى filter ئىجرا بولغاندىن كېيىن دەرھال ۋاكالەتچى تاللىغۇچتا كۆرۈنىدۇ.

### ئىچىگە ئورنىتىلغان ۋاكالەتچىنى چىقىرىۋېتىش {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
