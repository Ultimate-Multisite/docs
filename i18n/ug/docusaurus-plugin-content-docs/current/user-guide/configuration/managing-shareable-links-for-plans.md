# پلن‌های برای اشتراک‌گذاری لینک‌ها را مدیریت کردن

Ultimate Multisite به شما این امکان را می‌دهد که لینک‌های پلن‌ها را مستقیماً با مشتریانتان به اشتراک بگذارید. برای انجام این کار، به هر یک از صفحات ویرایش پلن‌های خود بروید و دکمه "_Click to copy the Shareable Link_" (کلیک برای کپی کردن لینک قابل اشتراک‌گذاری) را بزنید.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_در صفحه ویرایش پلن، می‌توانید دکمه ایجاد لینک‌های قابل اشتراک‌گذاری را پیدا کنید._

توجه داشته باشید که این روش از فرکانس صورتحساب پیش‌فرض برای ساخت لینک استفاده می‌کند. این بدان معناست که اگر صورتحساب پیش‌فرض شما ماهانه باشد، مثلاً در نهایت شما چنین لینکی خواهید داشت: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

اگر می‌خواهید لینک قابل اشتراک‌گذاری به صورت فصلی (هر سه ماه یکبار) پیش‌فرض باشد، باید ۳ را در انتها اضافه کنید:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

برای انتخاب پلن _premium_ با برنامه سالانه، شما باید ۱ و سپس کلمه "year" (سال) را اضافه کنید:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

به یاد داشته باشید که نمی‌توانید فقط یک لینک برای فرکانس‌های صورتحساب مختلف داشته باشید. شما باید برای هر فرکانس صورتحساب خود، یک لینک خاص به اشتراک بگذارید.

وقتی لینک را با مشتریان خود به اشتراک می‌گذارید، آنها به اولین مرحله از جریان ثبت‌نام شما هدایت خواهند شد.
