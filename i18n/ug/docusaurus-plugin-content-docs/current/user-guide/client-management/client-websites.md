# مۆلّی میزبانی‌larınız (v2)

شێبکه‌تاندا تۆمار کردنی‌سۆزمن، کارمندانه‌تان دەتوانن بچنە داشبۆردی خۆیان.

لە دایەتا **Account** (ڕێکخستنی کاردانەوە)، کە تێیدا دەتوانن زانیاری گرنگ لەبارەی پارەکان، ئەندازیارەکان، دامێنییەکان، سنوورەکانی پلاندانان و زیاتر لەوەش ببینن، دەتوانن بچنە **My Sites** (سایبەری من) بەشی، کە تێیدا هەموو ئەو وێب‌سایتان دەبینن و بەڕێوەدەبرن. هەموو کارێک کە پێویستە ئەوەیە لە داشبۆردەکەت سەرنجی **My Sites** بکەیت.

![Link بۆ My Sites لە داشبۆردی کڕیار](/img/account-page/my-sites-link.png)

ئەوان هەموو وێب‌سایتەکانیان دەبینن، بە چاپی دەکەوێت **Manage** (بەڕێوەبردنی) دۆکۆمێری هەر یەکێک لە خوارەوە. ئەگەر لەسەر ئەو دۆکۆمێرییە کلیک بکەن، دەچنە داشبۆردی ئەو وێب‌سایتە تایبەتە.

ئەوان دەتوانن ببینن کێ چیا وێب‌سایتیانە **primary** (پێشوەختەی) بێت بۆ پلاندانان کە ڕێگە بە هەبوونی زیاتر لە یەک وێب‌سایت دەدات.

![My Sites دۆکۆمێری نیشاندن وێب‌سایتەکانی کڕیار لەگەڵ دۆکۆمێری Manage](/img/account-page/my-sites-page.png)

لێرەدا بینینی تەواوی پێکهاتەی دەستکاری سایتەکە هەیە:

![پێکهاتەی تەواوی دەستکاری سایت](/img/admin/site-edit-full.png)

ئەگەر دەتوانن وێب‌سایتی سەرەکی خۆیان بگۆڕن، پێویستە بە مێنیو **Account** (کاردانەوە) بچن بۆ ئەو ڕێگە.

![Linkی مێنیو Account لە داشبۆردی کڕیار](/img/account-page/account-menu.png)

دەتوانن زانیارییەکانی کڕیاریشان ببینن و دەستکاری بکەن:

![پێکهاتەی دەستکاری کڕیار](/img/admin/customer-edit.png)

ئەو کات، پێویستە لەسەر **Change Default Site** (گۆڕینی سایت سەرەکی) کلیک بکەن.

![دۆکۆمێری Change Default Site لەسەر دۆکۆمێری Account](/img/account-page/change-default-site.png)

پاپ‌آپێک دەردەکەوێت. لە لیستەکەی دراپ‌داون (drop-down list) هەڵبژێرن کە چ وێب‌سایتی دەتانە بگۆڕیت بۆ سایت سەرەکی.

![راوەکەری هەڵبژاردنی دراپ‌داون Change Default Site](/img/account-page/change-default-site-dropdown.png)

پاشان، تەنها لەسەر **Change Default Site** کلیک بکەن و کارتان تەواو دەبێت!

![Default site change confirmation image](/img/account-page/change-default-site-confirm.png)
