# 8-دەرس: خېرىدارنى باشلاشقا يېتەكلەش

خېرىدارنى تىزىملىتىش پەقەت ئىشنىڭ يېرىمىلا. ئەگەر ئۇلار كىرىپ، ئۆزىنى بېسىم ئاستىدا قالغاندەك ھېس قىلىپ، قايتىپ كەلمىسە، سىز ئۇلارنى يوقاتقان بولىسىز. بۇ دەرس يېڭى تىزىملاتقان خېرىدارنى ئاكتىپ، قاتناشقان خېرىدارغا ئايلاندۇرىدىغان تەجرىبىنى لايىھىلەيدۇ.

## قەيەردە توختاپ قالغان ئىدۇق

FitSite ھازىر ئىشلىيەلەيدىغان چىقىم قىلىش ئېقىمى بىلەن تولۇق ماركىلاشتۇرۇلدى. ئەمدى بىز بەدەن چېنىقتۇرۇش ستۇدىيەسىنىڭ ئىگىسى تىزىملىتىشنى تاماملاپ، تۇنجى قېتىم ئۆزىنىڭ يېڭى تور بېكىتىگە كىرگەندىن كېيىن نېمە يۈز بېرىدىغانلىقىغا دىققەت قىلىمىز.

## نېمە ئۈچۈن باشلاشقا يېتەكلەش مۇھىم

تىزىملىتىشتىن كېيىنكى تۇنجى 30 مىنۇت خېرىدارنىڭ قېلىشى ياكى ۋاز كېچىشىنى بەلگىلەيدۇ. بەدەن چېنىقتۇرۇش ستۇدىيەسىنىڭ ئىگىسى:

- كىرىپ، ئاللىقاچان بەدەن چېنىقتۇرۇش تور بېكىتىدەك كۆرۈنىدىغان تور بېكەتنى كۆرسە → قالىدۇ
- كېيىن نېمە قىلىش كېرەكلىكىنى ئېنىق بىلسە → قالىدۇ
- ئادەتتىكى WordPress dashboard ئىچىدە يولىنى يوقاتقاندەك ھېس قىلسا → كېتىدۇ

سىزنىڭ ساھەگە خاس قېلىپلىرىڭىز (4-دەرس) بىرىنچى نۇقتىنى ھەل قىلىدۇ. بۇ دەرس ئىككىنچىسىنى ھەل قىلىدۇ.

## تۇنجى كىرىش تەجرىبىسى

### قارشى ئېلىش Dashboard Widget

يېڭى خېرىدارلارنى قارشى ئېلىپ، ئۇلارنى تەڭشەش جەريانىدا يېتەكلەيدىغان خاس dashboard widget قۇرۇڭ. بۇ ئۇلار تۇنجى قېتىم كىرگەندە كۆزگە كۆرۈنەرلىك ئورۇندا كۆرۈنۈشى كېرەك.

**FitSite تېز باشلاش:**

1. **ستۇدىيە نامىڭىز ۋە لوگوڭىزنى قوشۇڭ** -- Customizer ياكى تور بېكەت كىملىكى تەڭشەكلىرىگە ئۇلانما
2. **دەرس جەدۋىلىڭىزنى يېڭىلاڭ** -- بىۋاسىتە دەرسلەر بېتى تەھرىرلىگۈچىسىگە ئۇلانما
3. **مەشقاۋۇللىرىڭىزنى قوشۇڭ** -- مەشقاۋۇللار بېتى تەھرىرلىگۈچىسىگە ئۇلانما
4. **ئالاقىلىشىش تەپسىلاتلىرىڭىزنى تەڭشەڭ** -- ئالاقە بېتى تەھرىرلىگۈچىسىگە ئۇلانما
5. **تور بېكىتىڭىزنى ئالدىن كۆرۈڭ** -- ئالدى تەرەپكە ئۇلانما

ھەر بىر قەدەم مۇناسىۋەتلىك بەت ياكى تەڭشەككە بىۋاسىتە ئۇلىنىدۇ. تىزىملىكلەردىن ئىزدەپ يۈرۈش يوق.

### Dashboard نى ئاددىيلاشتۇرۇش

يېڭى خېرىدارلار ھەر بىر WordPress تىزىملىك تۈرىنى كۆرۈشىنىڭ ھاجىتى يوق. تۆۋەندىكىلەرنى ئويلىشىڭ:

- بەدەن چېنىقتۇرۇش تور بېكىتىنى باشقۇرۇشقا مۇناسىۋەتسىز تىزىملىك تۈرلىرىنى يوشۇرۇش (مەسىلەن، ئىشلىتىلمىسە ئىنكاسلار)
- ئەڭ كۆپ ئىشلىتىلىدىغان تۈرلەرنى ئالدىغا قويۇش ئۈچۈن تىزىملىكنى قايتا تەرتىپلەش
- ساھەگە ماس كېلىدىغان خاس تىزىملىك ناملىرىنى قوشۇش ("Appearance" نىڭ ئورنىغا "سىزنىڭ ستۇدىيەڭىز")

[Plugin & Theme Manager](/addons/plugin-and-theme-manager) addon خېرىدارلارنىڭ نېمىنى كۆرىدىغانلىقىنى كونترول قىلىشقا ياردەم بېرەلەيدۇ.

## قارشى ئېلىش ئېلېكتىرونلۇق خەت تەرتىپى

بىر دانە قارشى ئېلىش ئېلېكتىرونلۇق خېتى يېتەرلىك ئەمەس. خېرىدارلارنى تۇنجى ھەپتىسىدە يېتەكلەيدىغان بىر تەرتىپ تەڭشەڭ:

### 1-ئېلېكتىرونلۇق خەت: قارشى ئېلىش (تىزىملىتىشتىن دەرھال كېيىن)

- تېما: "FitSite غا خۇش كەپسىز -- ستۇدىيە تور بېكىتىڭىز نەق مەيداندا"
- مەزمۇن: كىرىش ئۇلانمىسى، تېز باشلاش قەدەملىرى، ياردەم مەنبەلىرىگە ئۇلانما
- ئۇسلۇب: ھاياجانلىق، رىغبەتلەندۈرگۈچى، بەدەن چېنىقتۇرۇشقا خاس

### 2-ئېلېكتىرونلۇق خەت: تېز نەتىجىلەر (1-كۈن)

- تېما: "FitSite دا ئالدى بىلەن قىلىدىغان 3 ئىش"
- مەزمۇن: لوگوڭىزنى قوشۇڭ، باش بەت hero رەسىمىنى يېڭىلاڭ، دەرس جەدۋىلىڭىزنى قوشۇڭ
- قەيەردە چېكىش كېرەكلىكىنى ئېنىق كۆرسىتىدىغان ئېكران كۆرۈنۈشلىرىنى ئۆز ئىچىگە ئېلىڭ

### 3-ئېلېكتىرونلۇق خەت: ئۇنى ئۆزىڭىزگە خاس قىلىڭ (3-كۈن)

- تېما: "بەدەن چېنىقتۇرۇش تور بېكىتىڭىزنى كۆزگە كۆرۈنەرلىك قىلىڭ"
- مەزمۇن: رەڭلەرنى خاسلاشتۇرۇڭ، مەشقاۋۇل سۈرەتلىرىنى قوشۇڭ، ستۇدىيە ھېكايىڭىزنى يېزىڭ
- سۇپىدىكى ياخشى بەدەن چېنىقتۇرۇش تور بېكەتلىرىنىڭ مىساللىرىغا ئۇلانما

### 4-ئېلېكتىرونلۇق خەت: نەق مەيدانغا چىقىش (7-كۈن)

- تېما: "FitSite نى دۇنيا بىلەن ھەمبەھىرلەشكە تەييارمۇ؟"
- مەزمۇن: ھەمبەھىرلەشتىن بۇرۇن تەكشۈرۈشكە تېگىشلىك ئىشلار تىزىملىكى، خاس دائىرە نامىنى قانداق ئۇلاش (Growth/Pro دا بولسا)، ئىجتىمائىي تاراتقۇدا ھەمبەھىرلەش ئۇسۇللىرى

:::tip ئېلېكتىرونلۇق خەت ئاپتوماتلاشتۇرۇش
بۇ ئېلېكتىرونلۇق خەتلەرنى ئېلېكتىرونلۇق خەت ماركېتىڭ سۇپىڭىز ئارقىلىق قوزغىتىش ئۈچۈن [Webhooks](/user-guide/integrations/webhooks) ياكى [Zapier](/user-guide/integrations/zapier) نى ئىشلىتىڭ. بۇ ۋاقىتنى كونترول قىلىشتا سىزگە تېخىمۇ كۆپ ئىمكان بېرىدۇ ۋە قاتنىشىشنى ئىز قوغلىشىڭىزغا يول قويىدۇ.
:::

## ياردەم مەنبەلىرى

بەدەن چېنىقتۇرۇش ستۇدىيەسى ئىگىلىرى ئەمەلىيەتتە سورايدىغان سوئاللارغا جاۋاب بېرىدىغان ساھەگە خاس ياردەم مەزمۇنى قۇرۇڭ:

### بىلىم ئامبىرى ماقالىلىرى

- "دەرس جەدۋىلىڭىزنى قانداق يېڭىلاش"
- "مەشقاۋۇل ئارخىپلىرىنى قوشۇش ۋە تەھرىرلەش"
- "ستۇدىيەڭىزنىڭ لوگوسى ۋە رەڭلىرىنى ئۆزگەرتىش"
- "ئۆز دائىرە نامىڭىزنى ئۇلاش" (Growth/Pro خېرىدارلىرى ئۈچۈن)
- "تور بېكىتىڭىزگە زاكاز قىلىش widget نى قوشۇش"

بۇلارنى تېخنىكىلىق بولمىغان ئىشلەتكۈچىلەر ئۈچۈن يېزىڭ. ئېكران كۆرۈنۈشلىرىنى ئىشلىتىڭ. WordPress جارگونىدىن ساقلىنىڭ.

### سىن ئارقىلىق چۈشەندۈرۈشلەر

تۆۋەندىكىلەرنى كۆرسىتىدىغان قىسقا (2-3 مىنۇتلۇق) ئېكران خاتىرىلىرى:

- تۇنجى كىرىش ۋە تونۇشتۇرۇش
- باش بەتنى تەھرىرلەش
- دەرس جەدۋىلىنى يېڭىلاش
- يېڭى مەشقاۋۇل قوشۇش

بۇلارنىڭ ناھايىتى سىلىق ئىشلەنگەن مەھسۇلات بولۇشى ھاجەت ئەمەس. ئېنىق، پايدىلىق ۋە ساھەگە خاس بولۇشى مۇھىم.

## Account بېتى

Ultimate Multisite خېرىدارلار مۇشتەرىلىكىنى باشقۇرىدىغان خېرىدارغا قارىتىلغان [Account بېتى](/user-guide/client-management/account-page) نى ئۆز ئىچىگە ئالىدۇ. بۇنى تۆۋەندىكىلەر ئۈچۈن خاسلاشتۇرۇڭ:

- ئۇلارنىڭ نۆۋەتتىكى FitSite پىلانىنى كۆرسىتىش
- بەدەن چېنىقتۇرۇشقا خاس پايدىلار بىلەن يېڭىلاش تاللاشلىرىنى كۆرسىتىش
- ھەق تۆلەش تارىخى ۋە ئىنۋويىس چۈشۈرۈشنى تەمىنلەش
- ياردەم مەنبەلىرىگە ئۇلاش

## باشلاشقا يېتەكلەش مۇۋەپپەقىيىتىنى ئۆلچەش

باشلاشقا يېتەكلەشنىڭ ئىشلەۋاتقان-ئىشلىمەيۋاتقانلىقىنى بىلىش ئۈچۈن بۇ كۆرسەتكۈچلەرنى ئىز قوغلاڭ:

- **ئاكتىپلىشىش نىسبىتى**: تىزىملاتقانلارنىڭ قانچە پىرسەنتى تۇنجى ھەپتە ئىچىدە ئەمەلىيەتتە تور بېكىتىنى خاسلاشتۇرىدۇ؟
- **تۇنجى ھەپتىدىكى كىرىشلەر**: يېڭى خېرىدار تۇنجى ھەپتىسىدە قانچە قېتىم كىرىدۇ؟
- **يېڭى خېرىدارلاردىن كەلگەن قوللاش بېلەتلىرى**: كۆپ بولسا، باشلاشقا يېتەكلەشتە بوشلۇقلار بار دېگەنلىك
- **سىناقتىن ھەقلىققا ئۆتۈش**: ئەگەر سىناق تەمىنلىسىڭىز، قانچە پىرسەنتى ئۆتىدۇ؟

## ھازىرغىچە بولغان FitSite تورى

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured + branded)
├── Platform Domain (fitsite.com + wildcard SSL)
├── Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products (Starter, Growth, Pro + Order Bumps)
├── Checkout Flow (niche-specific, tested)
├── Branding (login, dashboard, emails, invoices, marketing site)
├── Customer Onboarding
│   ├── Quick Start dashboard widget
│   ├── Simplified dashboard for fitness site management
│   ├── 4-email welcome sequence
│   ├── Niche-specific knowledge base
│   ├── Video walkthroughs
│   └── Customized account page
└── Ready for pricing strategy (next lesson)
```

## بۇ دەرسدە قۇرغانلىرىمىز

- **يېتەكلەنگەن تۇنجى كىرىش تەجرىبىسى**، Quick Start widget بىلەن
- **ئاددىيلاشتۇرۇلغان dashboard**، بەدەن چېنىقتۇرۇش تور بېكىتىنى باشقۇرۇش ۋەزىپىلىرىگە مەركەزلەشكەن
- **قارشى ئېلىش ئېلېكتىرونلۇق خەت تەرتىپى**، خېرىدارلارنى تۇنجى ھەپتىسىدە يېتەكلەيدىغان
- **ساھەگە خاس ياردەم مەنبەلىرى**، تېخنىكىلىق بولمىغان بەدەن چېنىقتۇرۇش ستۇدىيەسى ئىگىلىرى ئۈچۈن يېزىلغان
- **باشلاشقا يېتەكلەش ئۆلچەملىرى**، تەجرىبىنى ئىز قوغلاش ۋە ياخشىلاش ئۈچۈن
