# دەرس 10: ئېلان قىلىش كۈنى

ھەممە نەرسە قۇرۇلۇپ بولدى. ئىشىكلەرنى ئېچىشتىن بۇرۇن، بۇ دەرس ئالدىن ئېلان قىلىش تەكشۈرۈش تىزىملىكىنى بېسىپ ئۆتۈپ، ھېچ نەرسىنىڭ بۇزۇلغان، كەم قالغان ياكى نومۇس قىلغۇدەك ئەمەسلىكىگە كاپالەتلىك قىلىدۇ.

## بىز قەيەردە توختىغان ئىدۇق

FitSite نىڭ قېلىپلىرى، پىلانلىرى، چىقىم قىلىش جەريانى، ماركا كۆرۈنۈشى، باشلىنىش يېتەكلەش جەريانى ۋە باھالىرىنىڭ ھەممىسى سەپلەندى. ئەمدى ھەممىنىڭ ئىشلەيدىغانلىقىنى تەكشۈرۈپ، رەسمىي ئىشقا كىرىشتۈرىمىز.

## ئالدىن ئېلان قىلىش تەكشۈرۈش تىزىملىكى

ھەر بىر تۈرنى بىر-بىرلەپ ئىشلەپ چىقىڭ. ھېچبىرىنى ئاتلاپ كەتمەڭ.

### سۇپا ئۇل ئەسلىھەسى

- [ ] Hosting مۇقىم بولۇپ، يۈك بېسىمى ئاستىدا ياخشى ئىشلەۋاتىدۇ
- [ ] Wildcard SSL ئاكتىپ، بارلىق تارماق دومېنلار HTTPS ئارقىلىق تەمىنلىنىۋاتىدۇ
- [ ] دومېن خەرىتىلەش ئىشلەيدۇ -- بىر تور بېكەت قۇرۇپ، خاس دومېن خەرىتىلەشنى سىناپ بېقىڭ
- [ ] زاپاسلاش سەپلەنگەن ۋە سىناق قىلىنغان (دەلىللەش ئۈچۈن ئاز دېگەندە بىرىنى ئەسلىگە كەلتۈرۈڭ)
- [ ] كۆزىتىش يولغا قويۇلغان -- سۇپا چۈشۈپ كەتسە، بۇنى بىلىسىز

### قېلىپلار

- [ ] ئۈچ قېلىپنىڭ ھەممىسى يېڭى تور بېكەتلەردە توغرا يۈكلىنىدۇ
- [ ] ۋاقىتلىق مەزمۇن پايدىلىق ۋە ئىملا خاتالىرىدىن خالىي
- [ ] بارلىق رەسىملەرنىڭ ئىجازىتى مۇۋاپىق (ھېچقانداق stock photo سۇ بەلگىسى يوق)
- [ ] ھەر بىر قېلىپ بېتىدە كۆچمە ئۈسكۈنىگە ماسلىشىش ئىشلەيدۇ
- [ ] بەت يۈكلىنىش سۈرئىتى قوبۇل قىلغۇدەك (GTmetrix ياكى PageSpeed Insights غا ئوخشاش قورال بىلەن سىناڭ)
- [ ] ھېچقانداق قېلىپتا بۇزۇلغان ئۇلىنىش ياكى كەم قالغان مەنبە يوق

### پىلانلار ۋە مەھسۇلاتلار

- [ ] ئۈچ پىلاننىڭ ھەممىسى ئاكتىپ ۋە كۆرۈنەرلىك
- [ ] پىلان چۈشەندۈرۈشلىرى توغرا ۋە نىشقا خاس
- [ ] باھا توغرا (ئايلىق ۋە يىللىق)
- [ ] تەڭشەش ھەقلىرى توغرا پىلانلاردا سەپلەنگەن
- [ ] Starter پىلانىدا سىناق مۇددىتى ئىشلەيدۇ
- [ ] چىقىم قىلىش جەريانىدا زاكاز قوشۇمچىلىرى توغرا كۆرۈنىدۇ
- [ ] قىستۇرما ۋە تېما چەكلىمىلىرى ھەر پىلان بويىچە توغرا ئىجرا قىلىنىدۇ

### چىقىم قىلىش جەريانى

- [ ] ھەر بىر پىلاندا تولۇق سىناق تىزىملىتىشنى تاماملاڭ (سىناق پۇل تۆلەش ھالىتىنى ئىشلىتىڭ)
- [ ] قېلىپ تاللاش ھەر پىلانغا ماس توغرا قېلىپلارنى كۆرسىتىدۇ
- [ ] پۇل تۆلەش مۇۋەپپەقىيەتلىك بىر تەرەپ قىلىنىدۇ
- [ ] خېرىدار تىزىملاتقاندىن كېيىن قارشى ئېلىش ئېلخېتىنى تاپشۇرۇۋالىدۇ
- [ ] يېڭى تور بېكەت توغرا قېلىپ بىلەن قۇرۇلىدۇ
- [ ] خېرىدار يېڭى تور بېكىتىگە دەرھال كىرەلەيدۇ
- [ ] ئېتىبار كودلىرى توغرا ئىشلەيدۇ

### ماركا كۆرۈنۈشى

- [ ] كىرىش بېتى FitSite ماركا كۆرۈنۈشىنى كۆرسىتىدۇ
- [ ] باشقۇرۇش Dashboardى FitSite ماركا كۆرۈنۈشىنى كۆرسىتىدۇ
- [ ] بارلىق سىستېما ئېلخەتلىرى FitSite ماركا كۆرۈنۈشى ۋە بەدەن چېنىقتۇرۇشقا خاس تىلنى ئىشلىتىدۇ
- [ ] ھېسابات تالونلىرى سودا ئۇچۇرلىرىڭىز بىلەن توغرا كۆرسىتىلىدۇ
- [ ] بازارلاشتۇرۇش تور بېكىتى جانلىق بولۇپ، چىقىم قىلىش جەدۋىلىگە ئۇلىنىدۇ

### باشلىنىش يېتەكلەش

- [ ] تېز باشلاش ۋىجىتى يېڭى خېرىدار Dashboardلىرىدا كۆرۈنىدۇ
- [ ] بارلىق تېز باشلاش ئۇلىنىشلىرى توغرا بەتلەرگە كۆرسىتىدۇ
- [ ] قارشى ئېلىش ئېلخەت تەرتىپى سەپلەنگەن ۋە سىناق قىلىنغان
- [ ] بىلىم ئامبىرى ماقالىلىرى ئېلان قىلىنغان ۋە زىيارەت قىلغىلى بولىدۇ
- [ ] Account بېتى توغرا پىلان ئۇچۇرى ۋە يۇقىرىلىتىش تاللانمىلىرىنى كۆرسىتىدۇ

### قانۇنىي ۋە سودا

- [ ] مۇلازىمەت شەرتلىرى ئېلان قىلىنغان ۋە چىقىم قىلىشتىن ئۇلانغان
- [ ] مەخپىيەتلىك سىياسىتى ئېلان قىلىنغان ۋە زىيارەت قىلغىلى بولىدۇ
- [ ] پۇل قايتۇرۇش سىياسىتى بەلگىلەنگەن ۋە ھۆججەتلەنگەن
- [ ] پۇل تاپشۇرۇۋېلىش ئۈچۈن سودا گەۋدىسى قۇرۇلغان
- [ ] پۇل تۆلەش دەرۋازىسى جانلىق ھالەتتە (سىناق ھالىتى ئەمەس)
- [ ] باج سەپلىمىسى سىزنىڭ قانۇن رايونىڭىزغا ماس توغرا

## يۇمشاق ئېلان قىلىش بىلەن قاتتىق ئېلان قىلىش

ئىككى باسقۇچلۇق ئېلان قىلىشنى ئويلىشىپ بېقىڭ:

### باسقۇچ 1: يۇمشاق ئېلان قىلىش

ئاممىۋى ئېلان قىلىشتىن بۇرۇن 5-10 بەدەن چېنىقتۇرۇش ستۇدىيىسى ئىگىلىرىنى تىزىملىتىشقا تەكلىپ قىلىڭ. بۇلار سىزنىڭ beta خېرىدارلىرىڭىز. تۆۋەندىكىلەر بەدىلىگە ئۇلارغا كۆرۈنەرلىك ئېتىبار (ئۆمۈرلۈك 50% چۈشۈرۈم ياكى 3 ئاي ھەقسىز) تەكلىپ قىلىڭ:

- تىزىملىتىش ۋە باشلىنىش يېتەكلەش تەجرىبىسى توغرىسىدا سەمىمىي پىكىر
- ئۇلارنىڭ تور بېكىتىنى كۆرگەزمە مىسالى قىلىپ ئىشلىتىشكە رۇخسەت
- ئۇچرىغان ھەر قانداق خاتالىق ياكى مەسىلىلەرنى دوكلات قىلىش

بۇ سىزگە ئاممىغا ئېچىشتىن بۇرۇن ھەقىقىي خېرىدار پىكرى ۋە كۆرسىتىدىغان جانلىق تور بېكەتلەرنى بېرىدۇ.

### باسقۇچ 2: ئاممىۋى ئېلان قىلىش

يۇمشاق ئېلان قىلىش پىكىرلىرى كىرگۈزۈلگەندىن كېيىن:

- پۇل تۆلەش دەرۋازىسىنى جانلىق ھالەتكە ئالماشتۇرۇڭ
- بازارلاشتۇرۇش تور بېكىتىڭىزنى ئېلان قىلىڭ
- خېرىدار تېپىشنى باشلاڭ (دەرس 11)
- مۇناسىۋەتلىك بەدەن چېنىقتۇرۇش كەسپى قاناللىرىدا ئېلان قىلىڭ

## ئېلان قىلىش كۈنىدىكى ھەرىكەتلەر

ئاممىغا ئېلان قىلىدىغان كۈندە:

1. **جانلىق پۇل تۆلەشكە ئالماشتۇرۇڭ** -- پۇل تۆلەش دەرۋازىڭىزدىكى سىناق ھالىتىنى چەكلەڭ
2. **يەنە بىر قېتىم دەلىللەڭ** -- ھەقىقىي پۇل تۆلەش بىلەن تولۇق سىناق تىزىملىتىشنى قىلىڭ (كېيىن ئۆزىڭىزگە قايتۇرۇڭ)
3. **يېقىندىن كۆزىتىڭ** -- خاتالىقلار، مەغلۇپ بولغان تىزىملىتىشلەر ياكى پۇل تۆلەش مەسىلىلىرىنى كۆزىتىڭ
4. **تەييار تۇرۇڭ** -- تۇنجى ھەقىقىي خېرىدارلىرىڭىز ياردەمگە ئېھتىياجلىق بولۇشى مۇمكىن، تېز جاۋاب ئىشەنچ بەرپا قىلىدۇ
5. **قىسقىچە تەبرىكلەڭ** -- ئاندىن قايتا ئىشقا كىرىشىڭ

## نېمە خاتا كېتىشى مۇمكىن

تۆۋەندىكىلەرگە تەييار بولۇڭ:

- **پۇل تۆلەش دەرۋازىسى مەسىلىلىرى**: دەرۋازا تەمىنلىگۈچىڭىزنىڭ قوللاش ئالاقىسىنى تەييار تۇتۇڭ
- **SSL گۇۋاھنامە مەسىلىلىرى**: گۇۋاھنامىلەرنى قانداق تەكشۈرۈش ۋە يېڭىلاشنى بىلىڭ
- **ئېلخەت يەتكۈزۈش مەغلۇبىيىتى**: ئېلخەتلەرنىڭ راستىنلا يېتىپ بارىدىغانلىقىنى سىناڭ (ئەخلەت خەت قىسقۇچلىرىنى تەكشۈرۈڭ)
- **يۈك بېسىمى ئاستىدىكى ئىقتىدار**: قاتناش كۆپىيىپ كەتسە، hostingىڭىزنى قانداق كېڭەيتىشنى بىلىڭ
- **خېرىدار چۈشىنىكسىزلىكى**: بىلىم ئامبىرىڭىز ۋە قوللاش قاناللىرىڭىزنى تەييار تۇتۇڭ

## بۇ دەرسلىكتە نېمىلەرنى قۇردۇق

- **كەڭ دائىرىلىك ئالدىن ئېلان قىلىش تەكشۈرۈش تىزىملىكى** ئۇل ئەسلىھە، مەزمۇن، پۇل تۆلەش ۋە قانۇنىي تەرەپلەرنى قاپلايدۇ
- **يۇمشاق ئېلان قىلىش ئىستراتېگىيەسى** ئاممىغا ئېلان قىلىشتىن بۇرۇن ھەقىقىي پىكىر ئېلىش ئۈچۈن
- **ئېلان قىلىش كۈنى تەرتىپلىرى** ئىشەنچ بىلەن جانلىق ھالەتكە ئۆتۈش ئۈچۈن
- **زاپاس پىلان** ئادەتتىكى ئېلان قىلىش كۈنى مەسىلىلىرى ئۈچۈن
