# Action: wu_page_\{$this->id\}_register_widgets

بۇ بەت ئۈچۈن widgets تىزىمغا ئېلىنغاندىن كېيىن ئىجرا بولىدۇ.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## پارامېتىرلار {#parameters}

| ئىسمى | تۈرى | چۈشەندۈرۈشى |
|------|------|-------------|
| $id | `string` | بەت idسى. |
| $page_hook | `string` | بەت hookى. |
| $page | `object` | بەت ئوبيېكتى. |
