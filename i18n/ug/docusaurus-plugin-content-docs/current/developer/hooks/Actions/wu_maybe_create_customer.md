# مەشغۇلات: wu_maybe_create_customer

قىستۇرما ئىجادكارلىرىغا خېرىدار قوشۇلغاندا قوشۇمچە ئىشلارنى قىلىشقا يول قويىدۇ.

مەسىلەن، خېرىدار-&gt;ئىشلەتكۈچىنى ئاساسىي تور بېكەتكە قوشۇش ئۈچۈن hookلارنى قوشىدىغان يەر مۇشۇ.

## پارامېتىرلار {#parameters}

| ئىسمى | تۈرى | چۈشەندۈرۈشى |
|------|------|-------------|
| $customer | `\Customer` | بەلكىم قۇرۇلغان خېرىدار. |
| $checkout | `\Checkout` | نۆۋەتتىكى چىقىم قىلىش سىنىپى. |
