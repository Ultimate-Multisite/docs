# مەشغۇلات: wu_before_light_ajax {#action-wubeforelightajax}

بەزى ئەھۋاللاردا، مەشغۇلاتلارنى بىر تەرەپ قىلىش ئۈچۈن قوشۇمچە كۈچ يۈكلىشىمىز كېرەك بولىدۇ.

بۇ مەشغۇلات زۆرۈر بولغاندا قوشۇمچە بېقىنىشلارنى يۈكلەشكە ئىشلىتىلىدۇ.
