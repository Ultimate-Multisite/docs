# سۈزگۈچ: wp_ultimo_render_vars

قىستۇرما ئاچقۇچىلارغا كۆرسىتىش مۇھىتىغا پۈتۈن دائىرىدە قوشۇمچە ئۆزگەرگۈچى قوشۇشقا يول قويىدۇ.

## پارامېتىرلار {#parameters}

| ئىسمى | تىپى | چۈشەندۈرۈش |
|------|------|-------------|
| $args | `array` | كۆرسىتىش چاقىرىقى ئارقىلىق يوللانغان ئۆزگەرگۈچىلەرنى ئۆز ئىچىگە ئالغان گۇرۇپپا. |
| $view | `string` | كۆرسىتىلىدىغان كۆرۈنۈشنىڭ ئىسمى. |
| $default_view | `string` | fallback_view نىڭ ئىسمى |
