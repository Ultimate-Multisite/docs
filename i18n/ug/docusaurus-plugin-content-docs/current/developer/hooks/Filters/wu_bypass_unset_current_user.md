# Filter: wu_bypass_unset_current_user

ئىجادكارلارغا نۆۋەتتىكى ئىشلەتكۈچىنى بىكارلاش كودىنى ئايلىنىپ ئۆتۈشكە رۇخسەت قىلىدۇ.

null دىن باشقا ھەرقانداق نەرسە قايتۇرۇلسا، كىرگەن نۆۋەتتىكى ئىشلەتكۈچىنى بىكارلاش ئايلىنىپ ئۆتۈلىدۇ. بۇ بەزى ئەھۋاللاردا پايدىلىق بولۇشى مۇمكىن، مەسىلەن، باشقۇرۇش تاختىسى سۈپىتىدە ئىشلىتىلىۋاتقان تارماق تور بېكەتلەرنى بىر تەرەپ قىلغاندا.

## پارامېتىرلار

| ئىسمى | تۈرى | چۈشەندۈرۈش |
|------|------|-------------|
| $null_or_bypass | `mixed` | داۋاملاشتۇرۇش ئۈچۈن Null، باشقا ھەرقانداق نەرسە ئۇنى ئايلىنىپ ئۆتىدۇ. |
| $current_user | `false\|\WP_User` | نۆۋەتتىكى ئىشلەتكۈچى ئوبيېكتى. |
