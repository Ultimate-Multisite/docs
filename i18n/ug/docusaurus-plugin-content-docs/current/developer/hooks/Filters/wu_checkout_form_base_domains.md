# سۈزگۈچ: wu_checkout_form_base_domains

ئورتاق checkout-form ئاساسىي دائىرىلىرىنى سۈزۈپ، ئۇلارنىڭ mapped-domain خاتىرىلىرىگە ئايلىنىپ كەتمەسلىكىنى كاپالەتلەندۈرىدۇ.

بىر integration checkout form **Site URL** بۆلەكلىرى ئۈچۈن قوشۇمچە ئاساسىي دائىرىلەرنى تەمىنلىگەندە، بۇ سۈزگۈچنى ئىشلىتىڭ. بۇ سۈزگۈچ قايتۇرغان دائىرىلەر ھەر بىر بېكەتنىڭ خاس دائىرىسىنىڭ ئورنىغا، ئورتاق تىزىملىتىش ساھىبلىرى دەپ قارىلىدۇ.

## پارامېتىرلار

| Name | Type | Description |
|------|------|-------------|
| $domains | `array` | checkout form سەپلىمىسىدىن توپلانغان ئورتاق ئاساسىي دائىرىلەر. |

### باشلاپ

- 2.13.0

### مەنبە

`inc/functions/domain.php` دا بېكىتىلگەن.


## قايتۇرۇلىدۇ

نورماللاشتۇرۇلغان checkout-form ئاساسىي دائىرىلىرىنىڭ array سى.
