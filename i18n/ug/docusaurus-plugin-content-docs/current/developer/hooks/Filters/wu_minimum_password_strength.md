# سۈزگۈچ: wu_minimum_password_strength

تەلەپ قىلىنىدىغان ئەڭ تۆۋەن پارول كۈچىنى سۈزۈڭ (zxcvbn نومۇرى).

كۈچ دەرىجىلىرى: - 0, 1: ئىنتايىن ئاجىز - 2: ئاجىز - 3: ئوتتۇرا - 4: كۈچلۈك (كۆڭۈلدىكى)

## پارامېتىرلار

| نامى | تۈرى | چۈشەندۈرۈش |
|------|------|-------------|
| $min_strength | `int` | تەلەپ قىلىنىدىغان ئەڭ تۆۋەن كۈچ دەرىجىسى. |
| $strength_setting | `string` | باشقۇرغۇچى تەڭشىكىنىڭ قىممىتى (medium, strong, super_strong). |
