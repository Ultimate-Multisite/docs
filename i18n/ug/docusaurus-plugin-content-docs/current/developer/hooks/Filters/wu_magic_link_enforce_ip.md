# سۈزگۈچ: wu_magic_link_enforce_ip

IP ئادرېسىنى دەلىللەشنى ئىجرا قىلىش-قىلماسلىقنى سۈزۈڭ.

tokens نىڭ ئوخشىمىغان تورلاردىن ئىشلىشىگە يول قويۇش ئۈچۈن false قىلىپ تەڭشەڭ. بۇ بىخەتەرلىكنى تۆۋەنلىتىدۇ، ئەمما ئىشلىتىشچانلىقىنى ئاشۇرىدۇ (مەسىلەن، تور ئالماشتۇرىدىغان كۆچمە ئۈسكۈنە ئىشلەتكۈچىلىرى ئۈچۈن).

## پارامېتىرلار {#parameters}

| ئىسمى | تىپى | چۈشەندۈرۈش |
|------|------|-------------|
| $enforce | `bool` | IP ئادرېسىنىڭ ماس كېلىشىنى ئىجرا قىلىش-قىلماسلىق. |

- 2.0.0
