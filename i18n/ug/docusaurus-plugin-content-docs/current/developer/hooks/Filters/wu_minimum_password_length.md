# سۈزگۈچ: wu_minimum_password_length

ئەڭ تۆۋەن پارول ئۇزۇنلۇقىنى سۈزۈڭ.

پەقەت wu_enforce_password_rules true بولغاندا ئىجرا قىلىنىدۇ.

## پارامېتىرلار {#parameters}

| ئىسمى | تۈرى | چۈشەندۈرۈش |
|------|------|-------------|
| $min_length | `int` | ئەڭ تۆۋەن پارول ئۇزۇنلۇقى. كۆڭۈلدىكىسى 12 (Defender Pro بىلەن ماس كېلىدۇ). |
| $defender_active | `bool` | Defender Pro Strong Password ئاكتىپ ياكى ئەمەسلىكى. |

### باشلاپ {#since}

- 2.4.0
