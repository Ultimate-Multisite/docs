# سۈزگۈچ: wu_use_domain_mapping

خەرىتە ئىشلىتىلىشى كېرەكمۇ-يوق، شۇنى بەلگىلەيدۇ

ئادەتتە، پەقەت ئاكتىپ خەرىتىلەرنىڭ ئىشلىتىلىشىگە يول قويۇشنى خالايسىز. ئەمما، تېخىمۇ ئىلغار لوگىكا ئىشلىتىشنى ياكى ئاكتىپ بولمىغان دائىرە ناملىرىنىڭمۇ خەرىتىلىنىشىگە يول قويۇشنى خالىسىڭىز، بۇ يەردە سۈزۈڭ.
