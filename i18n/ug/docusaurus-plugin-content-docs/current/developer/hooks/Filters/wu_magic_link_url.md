# سۈزگۈچ: wu_magic_link_url {#filter-wumagiclinkurl}

## پارامېتىرلار {#parameters}

| ئىسمى | تۈرى | چۈشەندۈرۈشى |
|------|------|-------------|
| $magic_link | `string` | سېھىرلىك ئۇلانما URL. |
| $user_id | `int` | ئىشلەتكۈچى ID سى. |
| $site_id | `int` | تور بېكەت ID سى. |
| $redirect_to | `string` | قايتا نىشانلاش URL. |
