# Filter: get_site_url_for_previewer {#filter-getsiteurlforpreviewer}

## پارامېتىرلار {#parameters}

| ئىسمى | تىپى | چۈشەندۈرۈش |
|------|------|-------------|
| $domain | `string` | ھازىر ئىشلىتىلىۋاتقان كۆڭۈلدىكى domain، ئۆزگەرتىشلەر ئۈچۈن پايدىلىق |
| $domain_options | `array` | Ultimate Multisite Settings -&gt; Network Settings -&gt; Domain Options دا كىرگۈزۈلگەن بارلىق domain تاللانمىلىرىنىڭ تىزىملىكى |
