# سۈزگۈچ: wu_independent_billing_cycle_product_types

مۇستەقىل ھەق ئېلىش دەۋرى بار مەھسۇلات تۈرلىرىنى سۈزۈڭ.

مۇستەقىل ھەق ئېلىش دەۋرى بار مەھسۇلاتلار ھارۋىدا پىلاننىڭ ھەق ئېلىش مۇددىتىگە ماسلىشىشقا مەجبۇرلانمايدۇ. بۇ، پىلان ئايلىق ياكى يىللىق بولۇشىدىن قەتئىينەزەر ھەمىشە يىللىق ھەق ئېلىنىدىغان دومېن تىزىملىتىشقا ئوخشاش مەھسۇلاتلار ئۈچۈن پايدىلىق.

## پارامېتىرلار

| ئىسمى | تۈرى | چۈشەندۈرۈلۈشى |
|------|------|-------------|
| $types | `array` | مۇستەقىل ھەق ئېلىش دەۋرى بار مەھسۇلات تۈرلىرىنىڭ array. |
