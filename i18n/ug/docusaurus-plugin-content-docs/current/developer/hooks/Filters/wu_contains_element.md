# سۈزگۈچ: wu_contains_element {#filter-wucontainselement}

ئىجادكارلارنىڭ دەسلەپكى ئىزدەش نەتىجىلىرىنى ئۆزگەرتىشىگە يول قويىدۇ.

بۇ ئۈچىنچى تەرەپ قۇرغۇچىلار ۋە شۇنىڭغا ئوخشاشلار ئۈچۈن پايدىلىق.

## پارامېتىرلار {#parameters}

| ئىسمى | تۈرى | چۈشەندۈرۈش |
|------|------|-------------|
| $contains_elements | `bool` | ئەگەر ئېلېمېنت مەزمۇن ئىچىدە بولسا. |
| $content | `string` | تەكشۈرۈلۈۋاتقان مەزمۇن. |
| $element | `self` | نۆۋەتتىكى ئېلېمېنت. |
| $post | `null\|\WP_Post` | تەكشۈرۈلىدىغان post. |
