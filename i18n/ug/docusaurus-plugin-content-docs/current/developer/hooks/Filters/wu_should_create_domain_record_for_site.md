# سۈزگۈچ: wu_should_create_domain_record_for_site {#filter-wushouldcreatedomainrecordforsite}

Ultimate Multisite يېڭى قۇرۇلغان تور بېكەت ئۈچۈن دائىرە خاتىرىسى قۇرۇش-قۇرماسلىقىنى سۈزۈپ بېرىدۇ.

بۇ سۈزگۈچنى ئورتاق چىقىم قىلىش جەدۋىلى ئاساسىي دائىرىسى، ئىچكى host ياكى باشقا بىر بىرلەشتۈرۈش ئايرىم باشقۇرىدىغان دائىرە ئىشلىتىدىغان تور بېكەتلەر ئۈچۈن ئاپتوماتىك دائىرە-خاتىرىسى قۇرۇشنى توسۇش ياكى كېچىكتۈرۈش ئۈچۈن ئىشلىتىڭ.

## پارامېتىرلار {#parameters}

| ئىسىم | تىپ | چۈشەندۈرۈش |
|------|------|-------------|
| $create | `bool` | دائىرە خاتىرىسى قۇرۇلۇشى كېرەك-كېرەك ئەمەسلىكى. |
| $site | `WP_Site` | يېڭى قۇرۇلغان تور بېكەت ئوبيېكتى. |

### باشلاپ {#since}

- 2.13.0

### مەنبە {#source}

`inc/functions/domain.php` دا ئېنىقلانغان.


## قايتۇرىدۇ {#returns}

دائىرە خاتىرىسى قۇرۇش-قۇرماسلىقنى كۆرسىتىدىغان Boolean.
