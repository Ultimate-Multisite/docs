# سۈزگۈچ: wu_view_override_replaceable_views

ئىجادكارلارنىڭ ئالماشتۇرغىلى بولىدىغان تىزىملىككە قوشۇمچە قىسقۇچلارنى قوشۇشىغا يول قويىدۇ.

ئېھتىيات قىلىڭ، چۈنكى قوشۇمچە قىسقۇچلارغا يول قويۇش Ultimate Multisite نەشرىنىڭ ئورنىغا ۋاقتى ئۆتكەن كۆپەيتىلگەن نۇسخىلارنىڭ يۈكلىنىشىنى كەلتۈرۈپ چىقىرىشى مۇمكىن.

## پارامېتىرلار {#parameters}

| ئىسمى | تۈرى | چۈشەندۈرۈش |
|------|------|-------------|
| $replaceable_views | `array` | يول قويۇلغان قىسقۇچلار تىزىملىكى. |

- 2.0.0
