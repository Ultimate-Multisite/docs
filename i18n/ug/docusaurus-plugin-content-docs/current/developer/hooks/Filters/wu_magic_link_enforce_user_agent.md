# سۈزگۈچ: wu_magic_link_enforce_user_agent

ئىشلەتكۈچى ۋاكالەتچىسىنى دەلىللەشنى مەجبۇرىي ئىجرا قىلىش-قىلماسلىقنى سۈزۈش.

tokens نىڭ ئوخشىمىغان توركۆرگۈچ/ئۈسكۈنىلەردە ئىشلىشىگە يول قويۇش ئۈچۈن false قىلىپ تەڭشەڭ. بۇ بىخەتەرلىكنى تۆۋەنلىتىدۇ، ئەمما ئىشلىتىشكە قۇلايلىقلىقنى ئاشۇرىدۇ.

## پارامېتىرلار

| نامى | تۈرى | چۈشەندۈرۈشى |
|------|------|-------------|
| $enforce | `bool` | ئىشلەتكۈچى ۋاكالەتچىسى ماس كېلىشىنى مەجبۇرىي ئىجرا قىلىش-قىلماسلىق. |
