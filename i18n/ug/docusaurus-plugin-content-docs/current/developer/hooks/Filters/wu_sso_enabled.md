# سۈزگۈچ: wu_sso_enabled

بۇ قىممەتنى سۈزۈپ، يەككە كىرىشنى پۈتۈنلەي تاقاڭ ياكى ئۇنى شەرتلىك قوزغىتىڭ.

|------|------|-------------|
| $enabled | `bool` | SSO قوزغىتىلسۇنمۇ؟ ئېچىش ئۈچۈن True، تاقاشقا يېقىن ھالەت ئۈچۈن false-ish. |
