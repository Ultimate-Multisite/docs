# سۈزگۈچ: wu_sso_url

ھاسىل قىلىنغان SSO URL لارنى cross-domain خېرىدار مەشغۇلاتلىرى ئۈچۈن قايتۇرۇشتىن بۇرۇن سۈزگۈچتىن ئۆتكۈزىدۇ.

بىر بىرلەشتۈرۈش sovereign-tenant SSO ئۇلانمىسىغا ئىشەنچلىك مەزمۇن قوشۇش ياكى Ultimate Multisite نىڭ token دەلىللەش ئىقتىدارىنى ساقلاپ قالغان ھالدا broker URL نى ئالماشتۇرۇشقا ئېھتىياجلىق بولغاندا، بۇ سۈزگۈچنى ئىشلىتىڭ.

## پارامېتىرلار {#parameters}

| ئىسمى | تۈرى | چۈشەندۈرۈش |
|------|------|-------------|
| $sso_url | `string` | ھاسىل قىلىنغان SSO URL. |
| $user | `WP_User` | SSO زىيارىتى ئارقىلىق دەلىللىنىدىغان ئىشلەتكۈچى. |
| $site_id | `int` | زىيارەت ئۈچۈن نىشان تور بېكەت ID سى. |
| $redirect_to | `string` | SSO دەلىللەش مۇۋەپپەقىيەتلىك بولغاندىن كېيىنكى نىشان URL. |

### باشلاپ {#since}

- 2.13.0

### مەنبە {#source}

`inc/sso/class-sso.php` دا ئېنىقلانغان.


## قايتۇرىدۇ {#returns}

سۈزگۈچتىن ئۆتكۈزۈلگەن SSO URL.
