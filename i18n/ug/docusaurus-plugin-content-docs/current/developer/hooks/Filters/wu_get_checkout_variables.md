# سۈزگۈچ: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

قىستۇرما ئىجادكارلىرىنىڭ چىقىم قىلىش بېتىنىڭ ئالدىن بەلگىلەنگەن قىممەتلىرىنى سۈزۈشىگە يول قويىدۇ.

ئېھتىيات قىلىڭ، كەم قالغان ئاچقۇچلار ئالدى يۈزدىكى چىقىم قىلىش جەريانىنى پۈتۈنلەي بۇزۇۋېتىشى مۇمكىن.

## پارامېتىرلار {#parameters}

| نامى | تۈرى | چۈشەندۈرۈش |
|------|------|-------------|
| $variables | `array` | يەرلىكلەشتۈرۈلگەن ئۆزگەرگۈچىلەر. |
| $checkout | `\Checkout` | چىقىم قىلىش سىنىپى. |

### باشلاپ {#since}

- 2.0.0

يېڭى ئۆزگەرگۈچىلەر arrayى.
