# سۈزگۈچ: wu_admin_notices

ئىجادكارلارغا Ultimate Multisite تەرىپىدىن قوشۇلغان admin ئۇقتۇرۇشلىرىنى سۈزۈشكە يول قويىدۇ.

## پارامېتىرلار

| Name | Type | Description |
|------|------|-------------|
| $notices | `array` | شۇ مۇئەييەن panel ئۈچۈن ئۇقتۇرۇشلار تىزىملىكى. |
| $all_notices | `array` | قوشۇلغان ئۇقتۇرۇشلار تىزىملىكى، panel بويىچە ئايرىلغان. |
| $panel | `string` | ئۇقتۇرۇشلارنى ئېلىش ئۈچۈن panel. |
| $filter | `string` | ئەگەر ئۆچۈرۈشكە بولىدىغان ئۇقتۇرۇشلار سۈزۈۋېتىلگەن بولسا. |
| $dismissed_messages | `array` | ئۆچۈرۈلگەن ئۇقتۇرۇش ئاچقۇچلىرى تىزىملىكى. |
