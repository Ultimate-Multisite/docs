# سۈزگۈچ: wu_apply_plan_limits

قىستۇرما ئىجادكارلىرىغا چەكلىمىلەرنى قىسقا يول بىلەن ئايلىنىپ ئۆتۈشكە يول قويىدۇ.

بۇ سۈزگۈچنى چەكلىمىلەرنىڭ ھېچقايسىسى باشلىنىشتىن بۇرۇن خالىغان كودنى ئىجرا قىلىشقا ئىشلىتەلەيسىز. ئەگەر سۈزگۈچىڭىز ھەرقانداق راست دەپ قارىلىدىغان قىممەتنى قايتۇرسا، جەريان داۋاملىشىدۇ؛ ئەگەر ھەرقانداق يالغان دەپ قارىلىدىغان قىممەتنى قايتۇرسا، كود قايتىدۇ ۋە تۆۋەندىكى hookلارنىڭ ھېچقايسىسى ئىجرا قىلىنمايدۇ.

### نەشرىدىن باشلاپ

- 1.7.0
