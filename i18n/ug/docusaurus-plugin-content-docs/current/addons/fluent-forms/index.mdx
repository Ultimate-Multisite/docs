# Fluent Forms

بۇ قوشۇمچە دېتال مەھسۇلاتلار، ئەزالىقلار ياكى تور بېكەتلەردە Multisite Ultimate ئۈستىدىكى Fluent Forms غا چەك قوشۇشنى مۇمكىن قىلىدۇ.
Fluent Forms ئادەتتىكى يازما تۈرلىرىنىڭ ئورنىغا مەخسۇس جەدۋەللەرنى ئىشلىتىدۇ، شۇڭا ئادەتتىكى چەكلەش تاللانمىلىرى Fluent Forms بىلەن ئىشلىمەيدۇ.

## ئىقتىدارلار

- Fluent Forms ياكى Conversational Fluent Forms ئۈچۈن چەك قوشۇش
- خېرىدارلار چەككە يەتكەندە، پىلانىنى يۇقىرى كۆتۈرۈشكە ئۈندەيدىغان دوستانە ئۇچۇرغا ئېرىشىدۇ
