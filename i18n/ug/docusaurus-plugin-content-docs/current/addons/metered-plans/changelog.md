# Metered Plans ئۆزگىرىش خاتىرىسى {#metered-plans-changelog}

Version 1.1.0 - 2026-05-05 ئېلان قىلىندى
- يېڭى: multisite تارماق تور بېكەتلىرى ئۈچۈن AI token ھېسابات قىلىش — خېرىدار تور بېكەتلىرى بويىچە AI token ئىشلىتىلىشىنى ئىز قوغلاپ ھېسابلايدۇ، تەڭشەشكە بولىدىغان ھەر بىر token نىسبىتى بىلەن
- يېڭى: Connector ئىجرا قىلىش ھەرىكەتچان چەك بايقاش ۋە write-through بىلەن قايتا قۇرۇلدى، بارلىق connector لار بويىچە ھەقىقىي ۋاقىتتىكى توغرىلىققا كاپالەتلىك قىلىدۇ
- تۈزىتىش: Database جەدۋەل يېڭىلاشلىرى ئەمدى BerlinDB schema ئېنىقلىمىلىرى بىلەن توغرا ماسلاشتۇرۇلدى، يېڭى قاچىلاشلاردا يېڭىلاش مەغلۇبىيەتلىرىنىڭ ئالدىنى ئالىدۇ
- تۈزىتىش: Database يېڭىلاش callbacks توغرا فورماتقا ئايلاندۇرۇلدى، سۈكۈتتىكى يېڭىلاش مەغلۇبىيەتلىرىنى ھەل قىلىدۇ
- تۈزىتىش: پارچە قىممەتلەر ئەمدى AI Usage Overage Markup كىرگۈزۈش مەيدانىدا قوبۇل قىلىنىدۇ
- تۈزىتىش: plugin يۈكلىنىشىدىكى ئېغىر خاتالىقلار ۋە قوش باشلىنىش مەسىلىلىرى ھەل قىلىندى
- ياخشىلاندى: developer مۇھىتلىرى تۇنجى ئىجرا قىلىشتا ئۆز-ئۆزىنى تەڭشىشى ئۈچۈن check-env npm script قوشۇلدى
