# قوللاش ۋاكالەتچىلىرى {#support-agents}

پۈتكۈل تورنى ئۆزىڭىزلا باشقۇرۇش ئاسان ئىش ئەمەس. تورىڭىز چوڭىيىشقا باشلىغانسېرى، قوللاشقا بولغان ئېھتىياجنىڭ ئېشىۋاتقانلىقىنى كۆرۈشكە باشلايسىز. خىزمەت يۈكىڭىزنى يېنىكلىتىشكە ياردەم بېرىدىغان كىشىلەرنى قوشۇش ياخشى پىكىر بولىدۇ. ئەمما، تولۇق زىيارەت/ئىجازەت بېرىش خەتەرلىك بولۇپ، تورىڭىزغا زىيان يەتكۈزۈشى مۇمكىن. دەل مۇشۇ جايدا بۇ قوللاش ۋاكالەتچىسى قوشۇلمىسى ئىشقا كىرىدۇ.

بۇ قوشۇلما Support Agent دەپ ئاتىلىدىغان يېڭى تۈردىكى WordPress ئىشلەتكۈچىسىنى قۇرۇشىڭىزغا يول قويىدۇ ۋە شۇ ئىشلەتكۈچىنىڭ قانداق ئىجازەتلىرى بارلىقىنى تولۇق كونترول قىلىشىڭىزغا ئىمكان بېرىدۇ. ئىجازەت بېرىش بىر ئالماشتۇرۇش كۇنۇپكىسىنى ئاچقاندەكلا ئاسان، ئۇنى چىقىرىۋېتىشنى قارار قىلسىڭىز ئۆچۈرۈپ قويسىڭىزلا بولىدۇ.

## ئىقتىدارلار {#features}

- WordPress ۋە Ultimate Multisite ئىجازەتلىرىنىڭ تۈرلەنگەن تىزىملىكى
- زىيارەتنى ئاسانلا قوزغىتىش ۋە چەكلەش
- چەكسىز ساندا ۋاكالەتچى ھېساباتلىرىنى قۇرۇش

## ئورنىتىش {#installation}

1. 'ultimate-multisite-support-agents' نى '/wp-content/plugins/' مۇندەرىجىسىگە يۈكلەڭ
2. WordPress دىكى 'قىستۇرمىلار' تىزىملىكى ئارقىلىق قوشۇلمىنى ئاكتىپلاڭ
3. قوشۇلمىنى تەڭشەش ئۈچۈن قەدەممۇ-قەدەم يېتەكچىگە ئەگىشىڭ
