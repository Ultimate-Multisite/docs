# Multi-Network

Ultimate Multisite: Multinetwork قوشۇمچىسى Ultimate Multisite سۇپىڭىزغا كۈچلۈك كۆپ تور ئىقتىدارىنى قوشىدۇ. بۇ قوشۇمچە خېرىدارلىرىڭىزنىڭ پەقەت ئايرىم تور بېكەتلەرنىلا ئەمەس، بەلكى پۈتۈن WordPress كۆپ بېكەت تورلىرىنى سېتىۋېلىشى ۋە باشقۇرۇشىغا ئىمكان بېرىدۇ.

## ئاساسلىق ئىقتىدارلار {#key-features}

* **تور تەمىنلەش**: خېرىدارلىرىڭىز ئۈچۈن تولۇق WordPress كۆپ بېكەت تورلىرىنى ئاپتوماتىك قۇرۇپ ۋە سەپلەپ بېرىدۇ
* **تور قېلىپلىرى**: خېرىدارلارغا تىزىملىتىشتىن بۇرۇن ئالدىن ياسالغان تور قېلىپىنى كۆرۈش، ئالدىن كۆرۈپ بېقىش ۋە تاللاش ئىمكانىنى بېرىدۇ
* **تور باشقۇرۇش**: كۆپ تور مۇھىتلىرى ئۈچۈن تولۇق باشقۇرۇش ئىقتىدارلىرى
* **ئايرىۋېتىلگەن تورلار**: ھەر بىر خېرىدار تورى ئۆزىنىڭ سانلىق مەلۇمات ئامبىرى جەدۋەللىرى ۋە سەپلىمىسى بىلەن مۇستەقىل ئىشلەيدۇ
* **تورلار ئارا SSO**: خېرىدارلارغا magic-link دەلىللەش ئارقىلىق تارماق تورلىرىغا توسالغۇسىز كىرىش ئىمكانىنى بېرىدۇ
* **كېڭەيتكىلى بولىدىغان قۇرۇلما**: ئەلالاشتۇرۇلغان بايلىق تەقسىماتى بىلەن كۆپ تورنى ئۈنۈملۈك بىر تەرەپ قىلىدۇ
* **بىرلەشتۈرۈش**: Ultimate Multisite نىڭ مۇشتەرىلىك ۋە ھېساب قىلىش سىستېمىلىرى بىلەن توسالغۇسىز بىرلىشىدۇ

## ئىشلىتىش ئەھۋاللىرى {#use-cases}

* **ئاق ماركا ساھىبخانلىقى**: خېرىدارلارغا ئۆزىنىڭ ماركىسىدىكى كۆپ بېكەت تورلىرىنى تەمىنلەڭ
* **ئاگېنتلىق ھەل قىلىش چارىلىرى**: ئاگېنتلىقلارغا بىرلا سۇپىدىن كۆپ خېرىدار تورلىرىنى باشقۇرۇش ئىمكانىنى بېرىڭ
* **كارخانا ئورۇنلاشتۇرۇشلىرى**: مۇرەككەپ كۆپ تور تەلەپلىرى بار چوڭ تەشكىلاتلارنى قوللاڭ
* **مائارىپ سۇپىلىرى**: ئوخشىمىغان مەكتەپلەر ياكى تارماقلار ئۈچۈن ئايرىم تورلارنى قۇرۇڭ

## تەلەپلەر {#requirements}

* Ultimate Multisite 2.0.0 ياكى ئۇنىڭدىن يۇقىرى
* WordPress كۆپ بېكەت ئورنىتىلىشى
* PHP 7.4 ياكى ئۇنىڭدىن يۇقىرى
* MySQL 5.6 ياكى ئۇنىڭدىن يۇقىرى

## ئورنىتىش {#installation}

1. plugin ھۆججەتلىرىنى `/wp-content/plugins/ultimate-multisite-multinetwork` مۇندەرىجىسىگە يۈكلەڭ
2. WordPress ئىچىدىكى 'Plugins' ئېكرانى ئارقىلىق plugin نى ئاكتىپلاڭ
3. كۆپ تور تەڭشەكلىرىنى Ultimate Multisite → Settings ئارقىلىق سەپلەڭ

## تور قېلىپلىرى {#network-templates}

1.3.0 نەشرى يېڭى تور سېتىۋالىدىغان خېرىدارلار ئۈچۈن قېلىپ بايقاش ۋە تاللاش ئېقىمىنى قوشىدۇ. ھەر بىر يېڭى خېرىدار تورىنىڭ تەييارلانغان قۇرۇلما، لايىھە، مەزمۇن كۈتۈپخانىسى ياكى ساھەگە خاس سەپلىمىدىن باشلىشىنى خالىغاندا تور قېلىپلىرىنى ئىشلىتىڭ.

### قېلىپنى ئالدىن كۆرگۈچ {#template-previewer}

تور قېلىپىنى ئالدىن كۆرگۈچ خېرىدارلارغا checkout دىن بۇرۇن بار قېلىپلارنى كۆرۈش ئىمكانىنى بېرىدۇ. ھەر بىر قېلىپنى جانلىق ئالدىن كۆرۈش تاختىسىدا ئاچقىلى بولىدۇ، شۇڭا خېرىدارلار تور سېتىۋېلىشتىن بۇرۇن ئورۇنلاشتۇرۇش ۋە مەزمۇنلارنى سېلىشتۇرالايدۇ.

ئالدىن كۆرگۈچنى خېرىدارلارغا ياردەم بېرىش ئۈچۈن ئىشلىتىڭ:

* ھەر بىر بار تور قېلىپىنىڭ كۆرۈنۈشى ۋە قۇرۇلمىسىنى تەكشۈرۈش
* سېتىۋېلىش ئېقىمىدىن ئايرىلمايلا كۆپ قېلىپنى سېلىشتۇرۇش
* تىزىملىتىشتىن بۇرۇن ئۆزىنىڭ ئىشلىتىش ئەھۋالىغا ئەڭ ماس كېلىدىغان قېلىپنى تاللاش

قېلىپ ناملىرى ۋە چۈشەندۈرۈشلەرنى خېرىدارغا چۈشىنىشلىك ساقلاڭ. خېرىدارلار كۆزدە تۇتۇلغان ئىشلىتىش ئەھۋالىنى، مەسىلەن ئاگېنتلىق تورلىرى، مەكتەپ تورلىرى، فرانشزا تورلىرى ياكى ئاق ماركا ساھىبخانلىق بولاقلىرىنى چۈشىنەلەيدىغان بولغاندا، ئالدىن كۆرگۈچ ئەڭ پايدىلىق بولىدۇ.

### checkout تا قېلىپ تاللىغۇچ {#template-picker-at-checkout}

قېلىپ تاللىغۇچنى تىزىملىتىش checkout ئېقىمىغا بىر سۆز بۆلىكى سۈپىتىدە قوشقىلى بولىدۇ. سەپلەنگەندە، خېرىدارلار checkout جەريانىدا ئۆزلىرى خالايدىغان تور قېلىپىنى تاللايدۇ، تاللانغان قېلىپ ئۇلارنىڭ يېڭى تورى تەمىنلەنگەندە ئىشلىتىلىدۇ.

تاللىغۇچنى checkout ئېقىمىدا ئىشلىتىش ئۈچۈن:

1. قېلىپ سۈپىتىدە تەمىنلىمەكچى بولغان تورلارنى تەييارلاڭ.
2. مۇناسىۋەتلىك تىزىملىتىش ياكى checkout جەدۋىلىدە Multinetwork قېلىپ تاللاش سۆز بۆلىكىنى قوزغىتىڭ.
3. شۇ checkout ئېقىمى ئۈچۈن قايسى قېلىپلارنىڭ بار بولۇشى كېرەكلىكىنى تاللاڭ.
4. checkout ئېقىمىنى ئېلان قىلىپ، تاللانغان قېلىپنىڭ يېڭى تورغا قوللىنىلغانلىقىنى جەزملەشتۈرۈش ئۈچۈن بىر تىزىملىتىشنى سىناڭ.

ئەگەر checkout ئېقىمى پەقەت بىرلا تور مەھسۇلاتىنى تەمىنلىسە، تاللىغۇچ يەنىلا يېتەكلەش خاراكتېرلىك باشلاش قەدىمى سۈپىتىدە پايدىلىق بولالايدۇ. كۆپ پىلانلىق تونىللاردا، پەقەت تاللانغان مەھسۇلات ياكى پىلان بىلەن ماس كېلىدىغان قېلىپلارنىلا كۆرسىتىڭ.

## تورلار ئارا Magic Link SSO {#cross-network-magic-link-sso}

تورلار ئارا magic link SSO خېرىدارلارنىڭ ئاساسىي Account رايونى بىلەن تارماق تورلار ئارىسىدا قايتا قولدا كىرىش قىلماي يۆتكىلىشىگە ئىمكان بېرىدۇ. خېرىدار تورلار ئارا كىرىش ئۇلانمىسىغا ئەگەشكەندە، Ultimate Multisite ۋاقىت چەكلىمىسى بار magic link ھاسىل قىلىپ، خېرىدارنى نىشان تورغا دەلىللەپ كىرگۈزىدۇ.

بۇ بىر نەچچە تورنى باشقۇرىدىغان ياكى سېتىۋالغاندىن كېيىن يېڭى كۆچۈرۈلگەن قېلىپ تورلىرىغا دەرھال كىرىشى كېرەك بولغان خېرىدارلارنىڭ تەجرىبىسىنى ياخشىلايدۇ.

باشقۇرغۇچى تەكشۈرۈش تىزىملىكى:

* Multinetwork قوشۇمچىسىنىڭ WordPress كۆپ بېكەت ئورنىتىلىشى بويىچە ئاكتىپ ئىكەنلىكىنى جەزملەشتۈرۈڭ.
* خېرىدارلارغا تورلار ئارا ئۇلانمىلار تەمىنلىنىشتىن بۇرۇن ئۇلارنىڭ توغرا تور كىرىش ھوقۇقى بارلىقىغا كاپالەتلىك قىلىڭ.
* تور دائىرە ناملىرى ۋە خەرىتىلەنگەن دائىرە ناملىرىنى توغرا سەپلەپ ساقلاڭ، شۇنداق قىلىپ magic link لار كۆزدە تۇتۇلغان نىشان تورغا ھەل قىلىنسۇن.
* تور، دائىرە نامى ياكى checkout تەڭشەكلىرىنى ئۆزگەرتكەندىن كېيىن، خېرىدار Dashboard ى ۋە checkout دىن كېيىنكى ھەر قانداق كىرىش كۇنۇپكىلىرىنى سىناڭ.

Magic link لارغا كىرىش ئۇلانمىلىرى سۈپىتىدە مۇئامىلە قىلىش كېرەك. ئۇلار خېرىدارلارنىڭ توسالغۇسىز كىرىشى ئۈچۈن لايىھەلەنگەن بولۇپ، كۆزدە تۇتۇلغان سىرتقى ئولتۇرۇشتىن باشقا يەرگە يوللانماسلىقى ياكى قايتا ئىشلىتىلمەسلىكى كېرەك.

## دائىم سورىلىدىغان سوئاللار {#frequently-asked-questions}

### Multinetwork دېگەن نېمە؟ {#what-is-multinetwork}

Multinetwork بولسا بىرلا WordPress ئورنىتىلىشىدا كۆپ مۇستەقىل كۆپ بېكەت تورلىرىنى ئىجرا قىلىشقا ئىمكان بېرىدىغان WordPress ئىقتىدارىدۇر.

### بۇ ئادەتتىكى كۆپ بېكەتتىن قانداق پەرقلىنىدۇ؟ {#how-is-this-different-from-regular-multisite}

ئادەتتىكى كۆپ بېكەت بىر تور ئىچىدە كۆپ تور بېكەتكە ئىمكان بېرىدۇ. Multinetwork بولسا بىرلا WordPress ئورنىتىلىشىنىڭ ئىچىدە، ھەر بىرىنىڭ ئۆزىگە خاس تور بېكەتلىرى بار كۆپ تورغا ئىمكان بېرىدۇ.

### بۇ ئالاھىدە مۇلازىمېتىر سەپلىمىسىنى تەلەپ قىلامدۇ؟ {#does-this-require-special-server-configuration}

ھەئە، Multinetwork مەخسۇس مۇلازىمېتىر تەڭشەش ۋە سەپلىمىنى تەلەپ قىلىدۇ. تەپسىلىي تەڭشەش كۆرسەتمىلىرى ئۈچۈن ھۆججەتلەرگە مۇراجىئەت قىلىڭ.
