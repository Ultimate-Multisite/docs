# كۆپ-تور ئۆزگىرىش خاتىرىسى

### 1.3.0 {#130}
* يېڭى: تور قېلىپى ئالدىن كۆرگۈچى — سېتىۋېلىشتىن بۇرۇن جانلىق تاختا ئارقىلىق تور قېلىپلىرىنى كۆرۈپ چىقىڭ ۋە ئالدىن كۆرۈڭ.
* يېڭى: چىقىمداندا تور قېلىپى تاللىغۇچ — خېرىدارلار تىزىملىتىش چىقىمدان ئېقىمىدا تور قېلىپىنى تاللىيالايدۇ.
* يېڭى: تورلار ئارا سېھىرلىك ئۇلانما SSO — خېرىدارلار سېھىرلىك-ئۇلانما كىملىك دەلىللەش ئارقىلىق تارماق-تورلارغا ئۈزۈلۈكسىز كىرەلەيدۇ.
* يېڭى: تور مېدىياسىنى كۆپەيتىش — تورلارنى قېلىپتىن كۆپەيتكەندە بېكەت مېدىياسى توغرا نۇسخىلىنىدۇ.
* تۈزىتىش: تور كۆپەيتىش ئەمدى كۆپ-ئىجارىدارلىق addon بولمىسىمۇ ئىشلەيدۇ؛ كۆپەيتىشتىن كېيىن super admin ھوقۇقلىرى قايتا بېرىلىدۇ.
* تۈزىتىش: تور قۇرۇش ئەمدى WooCommerce ۋە بارلىق ھەق تۆلەش كىرىش يوللىرى بىلەن ماس كېلىدۇ.
* تۈزىتىش: پورت نومۇرى ئەمدى زاپاس دائىرە ھاسىل قىلىشقا قوشۇلىدۇ.
* تۈزىتىش: قېلىپ تاللىغۇچ CSS بارلىق ئۇسلۇبلاردا توغرا كۆرسىتىلىشى ئۈچۈن قاتارغا قوشۇلغان stylesheet قا يۆتكەلدى.
* تۈزىتىش: بېكەت سۈرۈشتۈرۈش دائىرىسىدىكى ئېنىقلانمىغان `network_id` ئاچقۇچىدىن ساقلىنىدۇ.
* تۈزىتىش: مۇھىت ئالماشتۇرۇشتىن بۇرۇنقى تور ئىجازەت تەكشۈرۈشى `wpdb` جەدۋەل خاسلىقى خاتالىقلىرىنىڭ ئالدىنى ئالىدۇ.
