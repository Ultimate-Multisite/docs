# Gratis AI Agent

Gratis AI Agent تەبىئىي تىل كۆرسەتمىلىرى ئارقىلىق تولۇق WordPress تور بېكەتلىرىنى پىلانلىيالايدىغان، قۇرالايدىغان ۋە باشقۇرالايدىغان ئاپتونوم WordPress AI ياردەمچىسىدۇر. ئۇ **قابىلىيەتلەر**نىڭ قۇرۇلمىلىق توپلىمىنى ئاشكارىلايدۇ — بۇلار ۋاكالەتچى چاقىرالايدىغان ئايرىم، سىناق قىلغىلى بولىدىغان ھەرىكەتلەر بولۇپ، مەزمۇن باشقۇرۇشتىن لايىھە سىستېمىسىنى خاسلاشتۇرۇشقىچە بولغان ھەممىنى ئۆز ئىچىگە ئالىدۇ. 1.9.0 نەشرى مەزمۇن قۇرۇش قابىلىيەتلىرىنى، خېرىدار تەرەپتىكى ئېكران سۈرەتلىرى ئارقىلىق كۆرۈنۈشلۈك بەت تەكشۈرۈشىنى، بەش دانە ئىچىگە قۇرۇلغان ۋاكالەتچى ئارخىپىنى، شۇنداقلا زىيارەت كونتروللۇقى ۋە ماركا نامى ئۈچۈن ئىقتىدار بايراقلىرىنى قوشىدۇ.

## ئاساسلىق ئىقتىدارلار {#key-features}

- **قابىلىيەتلەر سىستېمىسى** — ۋاكالەتچى WordPress ئورنىتىلمىڭىزدا چاقىرالايدىغان ھەرىكەتلەرنىڭ مودۇللۇق، كېڭەيتكىلى بولىدىغان كاتالوگى
- **مەزمۇن باشقۇرۇش** — يازمىلار ۋە بەتلەرنى قۇرۇش (بەت قېلىپلىرىنى تەقسىملەشنىمۇ ئۆز ئىچىگە ئالىدۇ)، بىر چاقىرىشتا كۆپ يازمىنى توپ ھالەتتە قۇرۇش، ئالاھىدە سۈرەتلەرنى بەلگىلەش ۋە ئالاقە فورمىلىرىنى قۇرۇش
- **كۆرۈنۈشلۈك تەكشۈرۈش** — نەق مەيدان بەتلىرىنىڭ ئېكران سۈرەتلىرىنى تۇتۇش، ئالدىن/كېيىنكى ھالەتلەرنى سېلىشتۇرۇش، ھەمدە ئورۇنلاشتۇرۇش، خەت نۇسخىسى، رەڭ ۋە زىيارەتچانلىقنى ئۆز ئىچىگە ئالغان AI ھاسىل قىلغان لايىھە تەكشۈرۈشىگە ئېرىشىش
- **بەش دانە ئىچىگە قۇرۇلغان ۋاكالەتچى** — Content Writer، Site Builder، Design Studio، Plugin Manager ۋە Support Assistant؛ سۆھبەت بېشىدىكى Agent Picker ئارقىلىق ئالماشتۇرغىلى بولىدۇ
- **ئىقتىدار بايراقلىرى** — Settings → Feature Flags ئىچىدىكى رول چەكلىمىلىرى، white-label نام بېرىش ۋە widget خاسلاشتۇرۇش ئۈچۈن زىيارەت كونتروللۇقى ۋە ماركا نامى ئالماشتۇرغۇچلىرى
- **خاس يازما تۈرىنى باشقۇرۇش** — قايتا قوزغىتىشلاردىن كېيىنمۇ ساقلىنىش بىلەن خاس يازما تۈرلىرىنى تىزىملاش، تىزىش ۋە ئۆچۈرۈش
- **خاس تۈرگە ئايرىشنى باشقۇرۇش** — ساقلىنىش بىلەن خاس تۈرگە ئايرىشلارنى تىزىملاش، تىزىش ۋە ئۆچۈرۈش
- **لايىھە سىستېمىسى قابىلىيەتلىرى** — خاس CSS قىستۇرۇش، تاللانغان بۆلەك ئەندىزىلىرىنى باشقۇرۇش، تور بېكەت لوگوسىنى بەلگىلەش ۋە theme.json ئالدىن تەڭشەكلىرىنى قوللىنىش
- **ئومۇمىي ئۇسلۇبلارنى باشقۇرۇش** — رەڭ پالېتلىرى، خەت نۇسخىسى، ئارىلىق ۋە باشقىلار ئۈچۈن theme.json قىممەتلىرىنى ئوقۇش ۋە يېزىش
- **يولباشچى تىزىملىكىنى باشقۇرۇش** — يولباشچى تىزىملىكلىرى ۋە ئۇلارنىڭ تۈرلىرىنى پروگراممىلىق ھالدا قۇرۇش ۋە يېڭىلاش
- **تاللانمىلارنى باشقۇرۇش** — مۇھىم تەڭشەكلەرنى قوغداش ئۈچۈن ئىچىگە قۇرۇلغان بىخەتەرلىك چەكلەش تىزىملىكى بىلەن WordPress تاللانمىلىرىنى ئوقۇش ۋە يېزىش
- **ئورنىتىلىدىغان قابىلىيەتلەر تىزىملىكى** — يادرولۇق plugin ھۆججەتلىرىنى ئۆزگەرتمەيلا قوشۇمچە قابىلىيەت بوغچىلىرىنى بايقاش ۋە ئاكتىپلاش
- **تور بېكەت قۇرغۇچى ئوركېستىرلاش v2** — plugin بايقاش، ئىلگىرىلەشنى ئىز قوغلاش ۋە خاتالىقتىن ئەسلىگە كەلتۈرۈش بىلەن كۆپ باسقۇچلۇق تور بېكەت قۇرۇش پىلانلىرى
- **WP-CLI ئۆلچەش بۇيرۇقى** — CI ۋە مودېل باھالاش ئۈچۈن بۇيرۇق قۇرىدىن ۋاكالەتچى قابىلىيىتى ئۆلچەش توپلىمىنى ئىجرا قىلىش

## تەلەپلەر {#requirements}

- WordPress 6.0 ياكى ئۇنىڭدىن يۇقىرى
- PHP 8.1 ياكى ئۇنىڭدىن يۇقىرى
- WordPress Multisite (تور بويىچە ئاكتىپلانغان) ياكى يەككە تور بېكەت ئورنىتىلمىسى
- ئاكتىپ AI تەمىنلىگۈچى ئۇلىغۇچ plugin (Anthropic، OpenAI ياكى ماس كېلىدىغان)

## ئورنىتىش {#installation}

1. plugin ھۆججەتلىرىنى `/wp-content/plugins/gratis-ai-agent` غا يۈكلەڭ
2. WordPress ئىچىدىكى **Plugins** ئېكرانى ئارقىلىق pluginنى ئاكتىپلاڭ
3. **Gratis AI Agent → Settings** غا يۆتكىلىپ، AI تەمىنلىگۈچىڭىزنى ئۇلاڭ
4. (ئىختىيارىي) قوشۇمچە قابىلىيەت بوغچىلىرىنى ئورنىتىش ئۈچۈن **قابىلىيەتلەر تىزىملىكى**نى كۆرۈپ چىقىڭ

## تېز باشلاش {#quick-start}

ئاكتىپلىنىپ AI تەمىنلىگۈچىگە ئۇلانغاندىن كېيىن، WordPress باشقۇرغۇچى يان بالداقىدىكى **Gratis AI Agent** تاختىسىنى ئېچىڭ. تەبىئىي تىلدىكى كۆرسەتمىنى كىرگۈزۈڭ، ۋاكالەتچى بار بولغان قابىلىيەتلىرىدىن پايدىلىنىپ زۆرۈر باسقۇچلارنى پىلانلايدۇ ۋە ئىجرا قىلىدۇ.

**مىسال كۆرسەتمىلەر:**

```
"Portfolio" ناملىق خاس يازما تۈرىنى "Project Category" تۈرگە ئايرىشى بىلەن قۇرۇڭ.
```

```
theme ئىچىگە قاراڭغۇ ھالەت رەڭ پالېتىنى قىستۇرۇپ، ئاساسىي رەڭنى #1a1a2e قىلىپ بەلگىلەڭ.
```

```
توردا تىزىملىك، زاكاز فورمىسى ۋە ئالاقە بېتى بار ئاشخانا تور بېكىتى قۇرۇڭ.
```

## قابىلىيەتلەر ئومۇمىي كۆرۈنۈشى {#abilities-overview}

قابىلىيەتلەر ۋاكالەتچى ئىجرا قىلالايدىغان ئاتوملۇق ھەرىكەتلەردۇر. ھەر بىر قابىلىيەت ۋاكالەتچى ئىجرا ۋاقتىدا ئوقۇيدىغان schema (پارامېتىرلار ۋە قايتۇرۇش تۈرلىرى)نى ئاشكارىلايدىغان تىزىملىتىلغان PHP classتۇر. قابىلىيەتلەر ئىقتىدار ساھەلىرى بويىچە گۇرۇپپىلىنىدۇ:

| ساھە | قابىلىيەتلەر |
|---|---|
| **مەزمۇن باشقۇرۇش** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **كۆرۈنۈشلۈك تەكشۈرۈش** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **خاس يازما تۈرلىرى** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **خاس تۈرگە ئايرىشلار** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **لايىھە سىستېمىسى** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **ئومۇمىي ئۇسلۇبلار** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **يولباشچى تىزىملىكلىرى** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **تاللانمىلارنى باشقۇرۇش** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **ئورنىتىلىدىغان قابىلىيەتلەر** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **تور بېكەت قۇرغۇچى** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

تولۇق پارامېتىر ھۆججەتلىرى ۋە ئىشلىتىش مىساللىرى ئۈچۈن [قابىلىيەتلەر قوللانمىسى](./abilities)نى كۆرۈڭ.

## WP-CLI {#wp-cli}

Gratis AI Agent بۇيرۇق قۇرىدىن ئۆلچەشلەرنى ئىجرا قىلىش ۋە قابىلىيەتلەرنى باشقۇرۇش ئۈچۈن `wp gratis-ai-agent` بۇيرۇق ئائىلىسى بىلەن كېلىدۇ. تولۇق بۇيرۇق تىزىملىكى ئۈچۈن [WP-CLI قوللانمىسى](./wp-cli)نى كۆرۈڭ.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## سەپلىمە {#configuration}

بارلىق تەڭشەكلەر WordPress باشقۇرغۇچىسىدىكى **Gratis AI Agent → Settings** ئاستىدا:

| تەڭشەك | چۈشەندۈرۈش |
|---|---|
| **AI تەمىنلىگۈچى** | ئاكتىپ AI تەمىنلىگۈچى ئۇلىغۇچنى تاللاڭ ۋە سەپلەڭ |
| **مودېل** | ۋاكالەتچى سۆھبەتلىرىدە ئىشلىتىلىدىغان تىل مودېلىنى تاللاڭ |
| **ئەڭ كۆپ Tokens** | ھەر بىر تەلەپ ئۈچۈن token خامچوتى (سۈكۈتتىكى 4096) |
| **قابىلىيەت ۋاقتى توشۇش** | بىر قابىلىيەت ئىجرا بولالايدىغان ئەڭ كۆپ سېكۇنت (سۈكۈتتىكى 30) |
| **Debug خاتىرىلەش** | قابىلىيەت چاقىرىشلىرى ۋە ۋاكالەتچىنىڭ مۇھاكىمىسىنى WordPress debug خاتىرىسىگە يېزىش |

## تاللانمىلارنى باشقۇرۇش بىخەتەرلىك چەكلەش تىزىملىكى {#options-management-safety-blocklist}

Options Management ئىقتىدارىدا agent نىڭ سەزگۈر WordPress تاللانمىلىرىنى ئوقۇشى ياكى يېزىشىنىڭ ئالدىنى ئالىدىغان ئىچكى توسۇش تىزىملىكى بار. كۆڭۈلدىكى توسۇش تىزىملىكى تۆۋەندىكىلەرنى ئۆز ئىچىگە ئالىدۇ:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL تاللانمىلىرى — بۇلار ئۈچۈن مەخسۇس كۆچۈرۈش قورالى ئىشلىتىڭ)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (مەخپىي ئاچقۇچلار)
- `active_plugins`, `template` (قىستۇرما/ئۇسلۇبنى ئاكتىپلاش — مەخسۇس ئىقتىدارلارنى ئىشلىتىڭ)
- قاچىلانغان قىستۇرمىلار تەرىپىدىن تىزىمغا ئېلىنغان چىقىم ئېغىزى كىرگۈزۈش ئۇچۇرلىرى ۋە API ئاچقۇچلىرى

قوشۇمچە تۈرلەرنى `gratis_ai_agent_options_blocklist` filter ئارقىلىق قوشقىلى بولىدۇ.

## Hooks ۋە Filters {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — بىر ئىقتىدار تىزىمغا ئېلىنغاندىن كېيىن قوزغىتىلىدۇ، ئىقتىدار class نامىنى تاپشۇرۇۋالىدۇ
- `gratis_ai_agent_plan_started` — تور بېكەت قۇرغۇچى پىلانى ئىجرا قىلىنىشنى باشلىغاندا قوزغىتىلىدۇ
- `gratis_ai_agent_plan_completed` — پىلان مۇۋەپپەقىيەتلىك تاماملانغاندا قوزغىتىلىدۇ
- `gratis_ai_agent_plan_failed` — پىلان ئەسلىگە كەلتۈرگىلى بولمايدىغان خاتالىققا يولۇققاندا قوزغىتىلىدۇ

### Filters {#filters}

- `gratis_ai_agent_abilities` — agent ئۇنى يۈكلەشتىن بۇرۇن تىزىمغا ئېلىنغان ئىقتىدار تىزىملىكىنى ئۆزگەرتىڭ
- `gratis_ai_agent_options_blocklist` — تاللانمىلار بىخەتەرلىك توسۇش تىزىملىكىنى كېڭەيتىڭ ياكى قىسقارتىڭ
- `gratis_ai_agent_benchmark_questions` — باھالاش توپلىمىغا خاس benchmark سوئاللىرىنى قوشۇڭ
- `gratis_ai_agent_plan_plugins` — تور بېكەت پىلانى ھاسىل قىلىش جەريانىدا قىستۇرما بايقاش تىزىملىكىنى ئۆزگەرتىڭ
