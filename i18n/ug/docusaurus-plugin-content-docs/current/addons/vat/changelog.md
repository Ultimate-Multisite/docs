# VAT ئۆزگىرىش خاتىرىسى {#vat-changelog}

نەشرى 1.0.7 - 2026-02-03 دە ئېلان قىلىنغان

* تۈزىتىش: گېرمانىيە (DE) VAT نومۇرلىرى VIES SOAP مۇلازىمىتىدىكى MS_MAX_CONCURRENT_REQ خاتالىقلىرى سەۋەبىدىن دەلىللەشتىن ئۆتەلمەيتتى. تېخىمۇ ئىشەنچلىك دەلىللەش ئۈچۈن كونا SOAP API دىن رەسمىي EU VIES REST API غا ئالماشتۇرۇلدى.
* تۈزىتىش: VIES مۇلازىمىتى ۋاقىتلىق ئىشلىمەي قالغاندا VAT نومۇرلىرى ئەمدى خاتا رەت قىلىنماي، قوبۇل قىلىنىدۇ. كېيىن تەكشۈرۈش ئۈچۈن log خاتىرىسى قۇرۇلىدۇ.
* تۈزىتىش: VAT/Tax ID خانىسى ئەمدى پەقەت EU ئەزالىرىدىلا ئەمەس، بارلىق دۆلەتلەر ئۈچۈن كۆرۈنىدۇ. EU سىرتىدىكى باج ID لىرى (مەسىلەن، شۋېيتسارىيە CHE نومۇرلىرى) VIES دەلىللەشسىز invoice كۆرسىتىشى ئۈچۈن ساقلىنىدۇ.
* تۈزىتىش: گىرېتسىيە (GR)، موناكو (MC) ۋە مەن ئارىلى (IM) ئۈچۈن VAT ئالدى قوشۇلغۇچىسىنى ئىزدەش خاتا array ئىزدەش لوگىكىسى سەۋەبىدىن بۇزۇلغان ئىدى.
* ياخشىلاش: Site Exporter قوشۇلمىسى بىلەن بىللە ئىشلىتىلگەندە ئېغىر خاتالىقلارنى كەلتۈرۈپ چىقىرىشى مۇمكىن بولغان زۆرۈر بولمىغان wp-cli-bundle dev تايىنىشى چىقىرىۋېتىلدى.
* ياخشىلاش: كەڭ دائىرىلىك unit test يۈرۈشلۈكى قوشۇلدى (53 سىناق).

نەشرى 1.0.6 - 2026-01-25 دە ئېلان قىلىنغان

* تۈزىتىش: دۆلەت ئىچىدىكى B2B سودىلىرى ئەمدى VAT نى توغرا ئالىدۇ. EU VAT قائىدىلىرىگە ئاساسەن، reverse charge پەقەت چېگرا ھالقىغان B2B سودىلىرىغا قوللىنىلىدۇ، خېرىدارنىڭ دۆلىتى شىركەتنىڭ دۆلىتى بىلەن ئوخشاش بولغاندا قوللىنىلمايدۇ.

نەشرى 1.0.5 - 2026-01-22 دە ئېلان قىلىنغان

* تۈزىتىش: قوشۇلما يۈكلەنمەيتتى ياكى توغرا ئىشلىمەيتتى.
* ئۆزگەرتىش: تېخىمۇ ئىشەنچلىك ۋە ئاكتىپ ساقلىنىۋاتقان سانلىق مەلۇمات ئۈچۈن VAT نىسبىتى سانلىق مەلۇمات مەنبەسى euvatrates.com دىن ibericode/vat-rates repository غا ئالماشتۇرۇلدى.
* تۈزىتىش: super_reduced_rates تاللانما قىممىتىدىكى يېزىق خاتالىقى تۈزىتىلدى.
* ياخشىلاش: خاتالىقلارنىڭ ئالدىنى ئېلىش ئۈچۈن دۆلەت كودىنى بىر تەرەپ قىلىشقا null تەكشۈرۈشلىرى قوشۇلدى.
* ئۆزگەرتىش: Traduttore ئارقىلىق ئاپتوماتىك يېڭىلانمىلارنى قوللاش ئۈچۈن ئىچىگە باغلانغان تەرجىمە ھۆججەتلىرى چىقىرىۋېتىلدى.

نەشرى: 1.0.3 - 2025-09-28 دە ئېلان قىلىنغان

* ئالدى قوشۇلغۇچى ultimate-multisite قىلىپ ئۆزگەرتىلدى؛ text domain يېڭىلاندى؛ نەشرى ئۆستۈرۈلدى.
