# VAT

Multisite Ultimate: ياۋروپا VAT

Multisite Ultimate نىڭ كۆڭۈلدىكى باج بىر تەرەپ قىلىشىغا قوشۇمچە ئىقتىدار ۋە قوراللارنى قوشۇپ، ياۋروپا VAT نى تېخىمۇ ياخشى قوللايدۇ.
EU VAT نىسبەتلىرىنى ئاپتوماتىك ھالدا باج نىسبەتلىرى بېتىگە قوشىدۇ

## قاچىلاش {#installation}

1. 'ultimate-multisite-vat' نى '/wp-content/plugins/' مۇندەرىجىسىگە يۈكلەڭ
2. WordPress دىكى 'قىستۇرمىلار' تىزىملىكى ئارقىلىق قىستۇرمىنى ئاكتىپلاڭ
3. Ultimate Multisite > تەڭشەكلەر > باجلار غا بېرىپ EU VAT نى قوزغىتىڭ.
4. باج نىسبەتلىرىنى چېكىپ، ئاستىدىكى EU VAT نىسبەتلىرىنى يېڭىلاڭ، ئاندىن باج نىسبەتلىرىنى ساقلاڭ.
5. تامام!
