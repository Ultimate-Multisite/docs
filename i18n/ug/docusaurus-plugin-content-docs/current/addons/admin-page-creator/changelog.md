### نەشرى 1.9.0 - 2026-01-18 دە ئېلان قىلىندى

* تۈزىتىلدى: ئاكتىپلىغاندا كۆرۈلىدىغان ئېغىر خاتالىقلار.
* قايتا قۇرۇلدى: مەزمۇن مەنبەسى classلىرى توغرا parent constructor باشلىنىشى بىلەن Singleton trait ئىشلىتىدىغان قىلىندى
* تۈزىتىلدى: مەزمۇن تۈرى تاللىغۇچنىڭ كۆرسىتىلىشىنى توسۇۋاتقان Vue.js قېلىپ compile خاتالىقلىرى
* تۈزىتىلدى: كەم قالغان Vue سانلىق مەلۇمات خاسلىقلىرى (template_id, external_link_url_embedable, external_link_url_checking)
* تۈزىتىلدى: بەت يۈكلەنگەندە TinyMCE تەھرىرلىگۈچنىڭ توغرا باشلانماسلىقى
* تۈزىتىلدى: Divi قوللاش ھۆججىتىنىڭ Composer ئارقىلىق ئاپتوماتىك يۈكلەنمەسلىكى
* تۈزىتىلدى: Divi قوللاش classىدىكى بۇزۇلغان PHP syntax
* ياخشىلاندى: JavaScript asset تەشكىللىنىشى ۋە كىچىكلىتىلىشى
* ياخشىلاندى: ئەڭ يېڭى WordPress ۋە بەت قۇرغۇچ نەشرلىرى بىلەن ماسلىشىشچانلىق
