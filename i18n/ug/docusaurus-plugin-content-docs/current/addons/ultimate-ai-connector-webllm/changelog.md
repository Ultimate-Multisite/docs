# Ultimate AI Connector for WebLLM ئۆزگىرىش خاتىرىسى

## نەشرى 1.1.0 — 2026-04-09 ئېلان قىلىندى

* يېڭى: admin-bar ھالەت كۆرسەتكۈچى بار لەيلىمە سۆھبەت widgetى — تىزىمغا كىرگەن ھەر قانداق ئىشلەتكۈچى ئالدى تەرەپتىن بىۋاسىتە توركۆرگۈچ تەرەپتىكى LLMگە prompt يوللىيالايدۇ.
* يېڭى: SharedWorker ئىجرا مۇھىتى — MLC ماتورى ھازىر SharedWorker ئىچىدە ئىشلەيدۇ، شۇڭا كۆپ بەتكۈچلەر بىر GPU سىستېمىسىنى تالىشىشنىڭ ئورنىغا ئورتاق ئىشلىتىدۇ.
* يېڭى: apiFetch middleware توسقۇچى — AI Client SDK ئەندىزىسىگە ماس كېلىدىغان WordPress REST تەلەپلىرى يەرلىك WebLLM ۋاستىچىسىغا ئاشكارا ھالدا يوللىنىدۇ، loopback HTTP قايتىپ-كېلىش سەپىرى ھاجەت ئەمەس.
* يېڭى: سۆھبەت widgetىنى ئېچىش-تاقاش ۋە auto-prompt ھەرىكىتىنى سەپلەش ئۈچۈن Connector تاختىسىدىكى widget تەڭشەك UIسى.
* تۈزىتىش: مودېل ئېغىرلىق چۈشۈرۈشلىرى سۈكۈتتىكى Cache API يولىنى بۇزىدىغان HuggingFace xet CDN قايتا يۆنىلىشلەردىن كېيىنمۇ ساقلىنىشى ئۈچۈن IndexedDB cache backendىنى مەجبۇرىي ئىشلىتىش.
* تۈزىتىش: embedding مودېللىرى ئۈچۈن context_window KV-cache قاپلاشنى ئاتلاپ ئۆتۈش (ئۇلاردا decoder يوق، قاپلاش ئىجرا خاتالىقى پەيدا قىلغان).
* تۈزىتىش: worker بەتكۈچى تېخى يۈكلەنمىگەن بولسىمۇ، SDK ئىستېمالچىلىرى بىر مودېلنى دەرھال كۆرسۇن دەپ /webllm/v1/models ئىچىدە cold-start نامزات مودېلىنى ئېلان قىلىش.
* تۈزىتىش: WebLLMنى wpai_preferred_text_models filterىغا ئۇلاش، شۇنداق قىلىپ مودېل سەپلەنگەندە AI Experiments ئىقتىدارى توركۆرگۈچ ماتورىغا يوللىنىدۇ.
* ياخشىلاندى: باشتىن-ئاخىر سىناش جەريانىدا ئوتتۇرىغا چىققان cache-busting، مەزمۇن نورماللاشتۇرۇش ۋە ئۈسكۈنە-پايدىلانما تۈزىتىشلىرى.
