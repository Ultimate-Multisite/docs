# ماس كېلىدىغان ئاخىرقى نۇقتىلار ئۈچۈن Ultimate AI ئۇلىغۇچىنىڭ ئۆزگىرىش خاتىرىسى

## 2.0.0 نەشرى — 2026-04-24 كۈنى ئېلان قىلىندى {#version-200--released-on-2026-04-24}

* يېڭى: كۆپ تەمىنلىگۈچى قوللىشى — كۆپ AI ئاخىرقى نۇقتىسىنى سەپلەپ، تەمىنلىگۈچىلەر ئارىسىدا ئاپتوماتىك fallback بىلەن تەلەپلەرنى يوللاڭ.
* تۈزىتىش: توغرا تەمىنلىگۈچى IDلىرى، تىزىملاش URLلىرى ۋە ھەر بىر تەمىنلىگۈچى بويىچە HTTP filter دائىرىسى بىلەن كۆپ تەمىنلىگۈچى SDK بىرلەشتۈرۈشى.
* تۈزىتىش: يېڭى تەمىنلىگۈچى كارتىسى قوشۇلغاندا ئەمدى ئاپتوماتىك كېڭىيىدۇ؛ plugin يېڭىلانغاندا script cache busting.
* تۈزىتىش: eval() ئۈچۈن دىنەمىك تەمىنلىگۈچى class namespace ۋە يېڭى قوشۇلغان تەمىنلىگۈچىلەر ئۈچۈن ئاپتوماتىك كېڭىيىش ھەرىكىتى.
* تۈزىتىش: WordPress 6.9+ ماسلىشىشچانلىقى ئۈچۈن مۇقىم Card/CardBody/CardHeader/CardDivider componentلىرى ئىشلىتىلىدۇ (ئەمدى تەجرىبە خاراكتېرلىك ئەمەس).
* تۈزىتىش: تەمىنلىگۈچىنى سۆرەپ قايتا تەرتىپلەش ئۈچۈن ئېنىقلانمىغان DragHandle unicode grip سىنبەلگىسى بىلەن ئالماشتۇرۇلدى.
* تۈزىتىش: ھەر بىر بەت يۈكلەنگەندە ئىجرا بولىدىغان توسقۇنلۇق قىلىدىغان HTTP تەلىپى يوقىتىلدى.
* ياخشىلاندى: GitHub Actions workflows Node.js 24 كە كۆتۈرۈلدى.
