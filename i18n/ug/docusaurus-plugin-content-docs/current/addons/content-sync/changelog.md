# Content Sync ئۆزگىرىش خاتىرىسى {#content-sync-changelog}

### نەشرى 1.1.0 - 2025-12-31 ئېلان قىلىندى {#version-110---released-on-2025-12-31}
* يېڭى: بارلىق يازما تۈرلىرىنى ماسقەدەملەشنى قوللايدۇ (يازمىلار، بەتلەر، خاس يازما تۈرلىرى)
* يېڭى: مەزمۇن تاللاش باسقۇچىدا يازما تۈرى بويىچە سۈزۈش
* يېڭى: مەنبە تور بېكىتىنى تاللاش باسقۇچى - پەقەت ئاساسىي تور بېكەتتىنلا ئەمەس، ھەرقانداق تور بېكەتتىن ماسقەدەملەش
* يېڭى: قېلىپ تور بېكەتلىرى ئۈچۈن تارماق تور بېكەتنى ئالدىن تاللاش
* يېڭى: بارلىق يازما تۈرلىرى ئۈچۈن post_content دىن مېدىيا چىقىرىش
* يېڭى: تاكسونومىيە ماسقەدەملەش (كاتېگورىيەلەر، خەتكۈچلەر، خاس تاكسونومىيەلەر)
* ئۆزگەرتىلدى: "Elementor Template Sync" دىن "Content Sync" غا قايتا نام بېرىلدى
* ئۆزگەرتىلدى: "Select Templates" باسقۇچى "Select Data" دەپ قايتا نام بېرىلدى
* ياخشىلاندى: Elementor غا خاس كود ھازىر پەقەت elementor_library يازما تۈرى ئۈچۈنلا ئىجرا بولىدۇ

### نەشرى 1.0.0 - 2025-12-19 ئېلان قىلىندى {#version-100---released-on-2025-12-19}
* دەسلەپكى ئېلان
* قېلىپ ۋە تور بېكەت تاللاش ئۈچۈن كۆپ باسقۇچلۇق يېتەكلىگۈچ
* ھەقىقىي ۋاقىتتىكى ئىلگىرىلەش ئىنكاسى بىلەن توپ بىر تەرەپ قىلىش
* ئاپتوماتىك ID قايتا خەرىتىلەش بىلەن تولۇق مېدىيا ماسقەدەملەش
* Ultimate Multisite بىلەن تەڭشەكلەر بىرلەشتۈرۈلۈشى
* تور تەڭشەكلىرى ئارقىلىق تور بېكەتنى چىقىرىۋېتىشنى قوللاش
* ماسقەدەملەشتىن كېيىن Elementor كەش تازىلاش
* قېلىپ تاكسونومىيىسىنى ساقلاپ قېلىش
