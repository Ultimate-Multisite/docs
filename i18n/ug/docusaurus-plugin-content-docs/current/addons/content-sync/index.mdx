# مەزمۇن ماسقەدەملەش {#content-sync}

Multisite Ultimate: مەزمۇن ماسقەدەملەش

Multisite Ultimate ئۈچۈن بۇ قوشۇلما تور باشقۇرغۇچىلىرىغا ھەرقانداق تور بېكەتتىن توردىكى باشقا تور بېكەتلەرگە مەزمۇن ماسقەدەملەشكە ئىمكان بېرىدۇ. ئۇ يازمىلار، بەتلەر، خاس يازما تۈرلىرى ۋە Elementor قېلىپلىرىنى ئۆز ئىچىگە ئالغان بارلىق يازما تۈرلىرىنى قوللايدۇ. بۇ قوشۇلما مەزمۇن ۋە نىشان تور بېكەتلەرنى تاللاش ئۈچۈن كۆپ باسقۇچلۇق يېتەكچى كۆرۈنمە يۈزىنى ئۆز ئىچىگە ئالىدۇ، توپلاملىق بىر تەرەپ قىلىش ۋە ھەقىقىي ۋاقىتلىق ئىلگىرىلەش ئىنكاسى بار.

## ئىقتىدارلار {#features}

* مەزمۇن تاللاش: كىچىك رەسىملەر ۋە يازما تۈرى بويىچە سۈزۈش بار كۆرۈنمە كارتا تورى
* بارلىق يازما تۈرلىرى: يازمىلار، بەتلەر، مەھسۇلاتلار، Elementor قېلىپلىرى ۋە ھەرقانداق خاس يازما تۈرىنى ماسقەدەملەش
* تور بېكەت تاللاش: مەزمۇننى قايسى تور بېكەتلەرگە ماسقەدەملەشنى تاللاڭ، مەنبە تور بېكەتنى چىقىرىۋېتىش ۋە بالا تور بېكەتلەرنى ئالدىن تاللاش بار
* توپلاملىق بىر تەرەپ قىلىش: ئىلگىرىلەش يېڭىلانمىلىرى بىلەن بىر ۋاقىتتا 10 تور بېكەتكە ماسقەدەملەيدۇ
* مېدىيا ماسقەدەملەش: مېدىيا ھۆججەتلىرىنى (رەسىملەر، سىنلار) يازما مەزمۇنىدىن نىشان تور بېكەتلەرگە ئاپتوماتىك كۆچۈرۈپ، قايتا خەرىتىلەيدۇ
* تۈرلىمە ماسقەدەملەش: سەھىپىلەر، خەتكۈچلەر ۋە خاس تۈرلىمىلەرنى ساقلاپ قالىدۇ
* Elementor قوللاش: CSS قايتا ھاسىل قىلىش ۋە قېلىپ ID قايتا خەرىتىلەش بىلەن Elementor قېلىپلىرىنى تولۇق قوللايدۇ
* توقۇنۇشنى ھەل قىلىش: مەۋجۇت مەزمۇننى ئالماشتۇرۇش ياكى ئاتلاپ كېتىش تاللانمىسى
* غەملەك باشقۇرۇش: ماسقەدەملەشتىن كېيىن نىشان تور بېكەتلەردىكى Elementor غەملەكنى ئاپتوماتىك تازىلايدۇ

## تەلەپلەر {#requirements}

* WordPress 5.3 ياكى ئۇنىڭدىن يۇقىرى
* PHP 7.4 ياكى ئۇنىڭدىن يۇقىرى
* Multisite Ultimate قىستۇرمىسى (ئاكتىپ)
* Elementor قىستۇرمىسى (ئىختىيارىي، Elementor قېلىپ ماسقەدەملەش ئۈچۈن تەلەپ قىلىنىدۇ)

## ئورنىتىش {#installation}

1. 'ultimate-multisite-content-sync' نى '/wp-content/plugins/' مۇندەرىجىسىگە يۈكلەڭ
2. WordPress تور باشقۇرغۇچىڭىزدىكى «قىستۇرمىلار» تىزىملىكى ئارقىلىق قىستۇرمىنى ئاكتىپلاڭ
3. Ultimate Multisite → تەڭشەكلەر → مەزمۇن ماسقەدەملەش ئىچىدە تەڭشەكلەرنى سەپلەڭ
4. ماسقەدەملەشنى باشلاش ئۈچۈن Ultimate Multisite → مەزمۇن ماسقەدەملەش كە يۆتكىلىڭ
