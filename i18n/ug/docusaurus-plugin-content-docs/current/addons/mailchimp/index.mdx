# Mailchimp

Multisite Ultimate: Mailchimp بىرلەشتۈرۈش

Multisite Ultimate تورىڭىزنى Mailchimp بىلەن بىرلەشتۈرۈڭ. مەلۇم پىلانلارنى Mailchimp تىزىملىكلىرىگە باغلاپ، تىزىملاتقاندا مۇشتەرىلەرنى ئاپتوماتىك ھالدا شۇ تىزىملىكلەرگە قوشالايسىز.

## قاچىلاش {#installation}

1. 'ultimate-multisite-mailchimp' نى '/wp-content/plugins/' مۇندەرىجىسىگە يۈكلەڭ
2. WordPress تور باشقۇرغۇچىڭىزدىكى 'قىستۇرمىلار' تىزىملىكى ئارقىلىق قىستۇرمىنى ئاكتىپلاڭ

## ئىشلىتىش قوللانمىسى {#usage-guide}

مۇشتەرىلەرنىڭ ئېلخەتلىرىنى MailChimp تىزىملىكلىرىڭىز بىلەن ماسقەدەملەش بىزنىڭ Ultimate Multisite: MailChimp Integration قوشۇلمىسى ئارقىلىق ناھايىتى ئاسان. بۇ دەرسلىك ھەممىنى تەڭشەپ چىقىشىڭىزغا ياردەم بېرىشنى مەقسەت قىلىدۇ!

## API ئاچقۇچى ئېلىش {#getting-an-api-key}

ئىشلەتكىلى بولىدىغان ئوخشىمىغان تەڭشەكلەرگە كىرىشتىن بۇرۇن، بىزگە MailChimp API ئاچقۇچى لازىم. بۇ Ultimate Multisite نىڭ MailChimp API بىلەن ئالاقىلىشىپ، تىزىملىكلەرنى، گۇرۇپپىلارنى ئېلىشىغا، ھەمدە ئىشلەتكۈچىلەر تىزىملاتقاندا ئېلخەت ئادرېسلىرىنى شۇ تىزىملىك ۋە گۇرۇپپىلارغا قوشۇشىغا يول قويىدۇ.

ئۆزىڭىزنىڭ MailChimp API ئاچقۇچىنى ئېلىش ئۈچۈن بۇ دەرسلىككە ئەگىشىڭ: [API ئاچقۇچلىرى ھەققىدە | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- ئېكران كۆرۈنۈشى يوق: Ultimate Multisite قوشۇلما تەڭشەكلىرىدىكى MailChimp API ئاچقۇچى تەڭشەكلىرى -->

_MailChimp API ئاچقۇچىڭىزنى_ _**Ultimate Multisite تەڭشەكلىرى - > قوشۇلما تەڭشەكلىرى -> MailChimp**_ _غا قوشۇڭ_

## بىرلەشتۈرۈش ھالىتى {#integration-mode}

MailChimp Integration قوشۇلمىسى ئىككى “بىرلەشتۈرۈش ھالىتى” تەمىنلەيدۇ، بۇ دېگەنلىك ئېلخەت ئادرېسلىرىنى ماسقەدەملەش ئۈچۈن قوشۇلمىنى ئىشلىتىشنىڭ ئىككى خىل ئۇسۇلى بارلىقىنى بىلدۈرىدۇ.

<!-- ئېكران كۆرۈنۈشى يوق: كۆپ تىزىملىك ۋە يەككە تىزىملىك تاللانمىلىرى بار MailChimp بىرلەشتۈرۈش ھالىتى تاللىشى -->

_تېخىمۇ چوڭ جانلىقلىق ئۈچۈن ئىككى بىرلەشتۈرۈش ھالىتى!_

### ھالەت 1: كۆپ تىزىملىك {#mode-1-multiple-lists}

سۈكۈتتىكى ھالەت **كۆپ تىزىملىك** نى ئىشلىتىدۇ. بۇ ھالەت ھەر بىر پىلان ئۈچۈن كۆپ MailChimp تىزىملىكىنى تاللىشىڭىزغا يول قويىدۇ. ئەگەر ئىشلەتكۈچىلىرىڭىزنى ئوخشىمىغان تىزىملىكلەرگە بۆلسىڭىز، بۇ ئەڭ ياخشى تاللاش.

مەسىلەن، MailChimp ھېساباتىڭىزدا 3 ئوخشىمىغان تىزىملىك بار دەپ پەرەز قىلايلى: **خەۋەرنامە** ، **پىلان A** ، ۋە **پىلان B**. سىز بارلىق ئىشلەتكۈچىلەرنىڭ، ئۇلارنىڭ پىلانى قانداق بولۇشىدىن قەتئىينەزەر، خەۋەرنامە تىزىملىكىگىمۇ كىرگۈزۈلۈشىنى خالايسىز، شۇنداق بولغاندا ئۇلار يېڭىلىنىشلىرىڭىزنى قولدىن بېرىپ قويمايدۇ.

بۇنى ئەمەلگە ئاشۇرۇش ئۈچۈن، كۆپ تىزىملىك ھالىتىنى ئىشلىتىپ **خەۋەرنامە** ۋە **پىلان A** دىن ئىبارەت ئىككى تىزىملىكنى پىلان A MailChimp تەڭشەكلىرىگە، **خەۋەرنامە** ۋە **پىلان B** نى پىلان B MailChimp تەڭشەكلىرىگە قوشالايسىز.

<!-- ئېكران كۆرۈنۈشى يوق: يېڭى ھېساباتلار ۋە ئەمەلدىن قالدۇرۇلغان ھېساباتلار ئۈچۈن MailChimp سۈكۈتتىكى تىزىملىكلەر تاللىشى -->

_يېڭى ھېساباتلار ۋە ئەمەلدىن قالدۇرۇلغان ھېساباتلار ئۈچۈن سۈكۈتتىكى تىزىملىكلەرنى تاللاڭ_

### ھالەت 2: يەككە تىزىملىك + كۆپ گۇرۇپپا {#mode-2-single-list--multiple-groups}

MailChimp تىزىملىكىڭىزنى **گۇرۇپپىلار** دەپ ئاتىلىدىغان كىچىكرەك ئىشلەتكۈچى گۇرۇپپىلىرىغا بۆلۈشىڭىزگە يول قويىدۇ. بۇ سىزگە تېخىمۇ كۆپ كونتروللۇق بېرىدۇ، چۈنكى MailChimp دا ئوخشاش كامپانىيەنىڭ كۆپ نۇسخىسىنى قۇرمايلا مۇشتەرىلىرىڭىزنىڭ مەلۇم بىر كىچىك توپىغا كامپانىيە ئەۋەتەلەيسىز (بۇ كېيىن ئۇ كامپانىيەنىڭ ئۆلچەملىرىنى تەھلىل قىلىشنى قىيىنلاشتۇرىدۇ).

مەسىلەن، **تورۇمدىكى مۇشتەرىلەر،** دەپ ئاتىلىدىغان بىرلا تىزىملىكىڭىز بولالايدۇ، ئۇنىڭ ئىچىدە **پىلان A** ۋە **پىلان B** ئۈچۈن گۇرۇپپىلار بولىدۇ.

<!-- ئېكران كۆرۈنۈشى يوق: سۈكۈتتىكى تىزىملىكلەر ئۈچۈن MailChimp يەككە تىزىملىك ۋە گۇرۇپپىلار تاللىشى -->

_ئەگەر “يەككە تىزىملىك + كۆپ گۇرۇپپا” بىرلەشتۈرۈش ھالىتىنى ئىشلەتسىڭىز، سۈكۈتتىكى تىزىملىكلەر ۋە گۇرۇپپىلارنى تاللاڭ_

## سۈكۈتتىكى تىزىملىكلەر ۋە گۇرۇپپىلار {#default-lists-and-groups}

قايسى بىرلەشتۈرۈش ھالىتىنىڭ سىزنىڭ مەلۇم ئىشلىتىش ئەھۋالىڭىزغا تېخىمۇ ماس كېلىدىغانلىقىنى تاللىغاندىن كېيىن، سۈكۈتتىكى تىزىملىكلەرنى (ھالەتكە قاراپ گۇرۇپپىلارنىمۇ) تاللايدىغان ۋاقىت كەلدى. مەلۇم پىلاندا مەخسۇس تىزىملىك تاللانمىغان بولسا، سۈكۈتتىكى تىزىملىك ئىشلىتىلىدۇ.

ئەمەلدىن قالدۇرۇلغان ھېساباتلار ئۈچۈنمۇ سۈكۈتتىكى تىزىملىكنى تاللىيالايسىز. ھەر قېتىم ئىشلەتكۈچى مۇشتەرىلىكىنى ئاخىرلاشتۇرۇپ، ھېساباتىنى تورىڭىزدىن ئۆچۈرگەندە، ئۇنىڭ ئېلخەت ئادرېسى ئەمەلدىن قالدۇرۇلغان ھېساباتلار ئۈچۈن بېكىتىلگەن تىزىملىككە يۆتكىلىدۇ.

## قوش مۇشتەرىلىكنى قوزغىتىش {#enabling-double-subscription}

ئوخشاش تەڭشەكلەر بېتىدە، **قوش مۇشتەرىلىك** نى قوزغىتالايسىز. بۇ تاللانما ئىشلەتكۈچىنى تاللانغان سۈكۈتتىكى تىزىملىككە ھەم شۇ ئىشلەتكۈچىنىڭ پىلانى ئۈچۈن تاللانغان تىزىملىككە قوشىدۇ. ئەگەر بۇ تاللانما قوزغىتىلمىغان بولسا، ئىشلەتكۈچى پەقەت ئىشلەتكۈچىنىڭ پىلانى ئۈچۈن تاللانغان تىزىملىككە قوشۇلىدۇ. ئەگەر شۇ مەلۇم پىلان ئۈچۈن ھېچنېمە بېكىتىلمىگەن بولسا، ئۇنداقتا ئىشلەتكۈچى تاللانغان سۈكۈتتىكى تىزىملىككە قوشۇلىدۇ.

<!-- ئېكران كۆرۈنۈشى يوق: MailChimp قوش مۇشتەرىلىك ئالماشتۇرۇش تەڭشىكى -->

_تىزىملىتىشتا ئىشلەتكۈچىلىرىڭىزنى سۈكۈتتىكى تىزىملىككە ھەم پىلاننىڭ تىزىملىكىگە مۇشتەرى قىلالايسىز_

## ھەر بىر پىلان ئۈچۈن تىزىملىكلەر ۋە گۇرۇپپىلارنى تاللاش {#selecting-lists-and-groups-for-each-of-the-plans}

**پىلانلار** غا بېرىپ، تەھرىرلىمەكچى بولغان پىلاننى تاللاڭ. **ئىلغار تاللانمىلار** meta-box ئىچىدە MailChimp دەپ ئاتىلىدىغان يېڭى بەتكۈچ بولىدىغانلىقىغا دىققەت قىلىڭ.

شۇ بەتكۈچتە، شۇ پىلان ئۈچۈن كۆپ تىزىملىك ياكى بىر يەككە تىزىملىك ۋە كۆپ گۇرۇپپىنى (بىرلەشتۈرۈش ھالىتىگە قاراپ) تاللىيالايسىز.

<!-- ئېكران كۆرۈنۈشى يوق: پىلان تەھرىرلەش بېتىدىكى MailChimp بەتكۈچىدە تىزىملىكلەر ۋە گۇرۇپپىلار تاللىشى كۆرسىتىلگەن -->

_ھەر بىر پىلانىڭىز ئۈچۈن ئىشلەتمەكچى بولغان تىزىملىكلەر ۋە گۇرۇپپىلارنى تاللاڭ_

يېڭى ئىشلەتكۈچى شۇ پىلانغا تىزىملاتقاندا، ئۇنىڭ ئېلخەت ئادرېسى تاللانغان تىزىملىك ۋە گۇرۇپپىلارغا قوشۇلىدۇ!
