# Filter: wu_domain_seller_search_delay

دومېن ئىزدەشنىڭ debounce كېچىكىش ۋاقتىنى مىللىسېكۇنت بويىچە سۈزگۈچلەيدۇ.

ئاستا ئۇلىنىشلاردا API چاقىرىشلىرىنى ئازايتىش ئۈچۈن بۇ قىممەتنى ئاشۇرۇڭ.

## پارامېتىرلار

| ئىسمى | تىپى | چۈشەندۈرۈش |
|------|------|-------------|
| $delay | `int` | مىللىسېكۇنت بويىچە debounce كېچىكىش ۋاقتى. كۆڭۈلدىكى قىممىتى 500. |
