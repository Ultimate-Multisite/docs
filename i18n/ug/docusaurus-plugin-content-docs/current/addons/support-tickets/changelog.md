# قوللاش بېلەتلىرى ئۆزگىرىش خاتىرىسى {#support-tickets-changelog}

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* ياخشىلاندى: vendor/ مۇندەرىجىسى Git ئىز قوغلاشتىن چىقىرىۋېتىلدى (.gitignore ئارقىلىق ئاللىقاچان قاپلانغان)، repository چوڭلۇقى كىچىكلىتىلدى
* ياخشىلاندى: WordPress 7.0 غىچە سىناقتىن ئۆتكۈزۈلدى

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* تۈزىتىلدى: nullable بېلەت model setters ئىچىدە null قىممەتلىرىنى قوبۇل قىلىش
* تۈزىتىلدى: مۇۋاپىق multisite قوللىشى ئۈچۈن بارلىق بېلەتلەرنى پۈتۈن تور مىقياسىدىكى ئومۇمىي جەدۋەللەردە ساقلاش
* تۈزىتىلدى: خادىم fields نى يوشۇرۇش ۋە خېرىدار يېڭى بېلەت جەدۋىلىدىكى بوش بەت قايتا يوللاشنى تۈزىتىش
* تۈزىتىلدى: مېتا سانلىق مەلۇمات ساقلاش ئۈچۈن ئېنىقسىز add_meta() چاقىرىقلىرىنى توغرا update_meta() بىلەن ئالماشتۇرۇش
* تۈزىتىلدى: تىزىمغا ئېلىنمىغان capability تەكشۈرۈشىنى توغرا wu_view_all_support_tickets بىلەن ئالماشتۇرۇش
* تۈزىتىلدى: بېلەت ھالىتى، تەقسىملەش ۋە تېز تەھرىرلەش مەشغۇلاتلىرى ئۈچۈن كەم قالغان AJAX handlers نى قوشۇش
* تۈزىتىلدى: بېلەت كۆرۈنۈشلىرىدە خادىم جاۋابىنى بايقاش ئۈچۈن method نامىنى توغرىلاش
* تۈزىتىلدى: تەكرار reply handlers نى بىرلەشتۈرۈش ۋە nonce مەشغۇلات ناملىرىنى ماسلاشتۇرۇش
* تۈزىتىلدى: [wu_submit_ticket] shortcode ئۈچۈن كەم قالغان ئالدى تەرەپ كۆرۈنۈشىنى قوشۇش
* تۈزىتىلدى: كەم قالغان user_id ئىستونىنى قوشۇش ۋە Support_Ticket::get_user_id() method نى تۈزىتىش
* تۈزىتىلدى: مۇھىملىق filter option tag ئىچىدىكى ئارتۇق قوش پەشنى چىقىرىۋېتىش
* تۈزىتىلدى: super admins ئۈچۈن تور باشقۇرۇش بېلەت باشقۇرۇش تاختىسىنى قوشۇش
* ياخشىلاندى: باشقۇرۇش CSS نى بىرلا سىرتقى ئۇسلۇب جەدۋىلىگە بىرلەشتۈرۈش
* ياخشىلاندى: تارماق تور بېكەت باشقۇرۇش تىزىملىكىدىن تەڭشەكلەر تارماق تىزىملىكىنى چىقىرىۋېتىش
* ياخشىلاندى: ئالدى تەرەپ assets نى پەقەت قوللاش بېلەت بەتلىرىدىلا شەرتلىك يۈكلەش
* ياخشىلاندى: Bedrock root autoloader ئاللىقاچان dependencies نى يۈكلىگەن بولسا plugin autoloader دىن ئاتلاپ ئۆتۈش

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* قوشۇلدى: باشقۇرغۇچى ۋە ئالدى تەرەپ ئۈچۈن تولۇق بېلەت كۆرۈش ئىقتىدارى
* قوشۇلدى: بېلەت جاۋاب يوللاشلىرى ئۈچۈن AJAX handler
* قوشۇلدى: مۇۋاپىق جەدۋەل بىر تەرەپ قىلىش بىلەن بېلەت جاۋاب ئىقتىدارىنى قوللاش
* قوشۇلدى: باشقۇرۇش بۆلىكىدە بېلەت يوللاش ۋە جاۋابلار ئۈچۈن مۇۋاپىق ئۇقتۇرۇش كۆرسىتىش
* قوشۇلدى: خېرىدارنىڭ ئۆزى يوللاشلىرى ئۈچۈن نۆۋەتتىكى ئىشلەتكۈچى بىلەن ئاپتوماتىك بېلەت باغلاش
* قوشۇلدى: خېرىدارلارنىڭ بېلەت ئىگىدارچىلىقىنى ئۆزگەرتىۋېتىشىنىڭ ئالدىنى ئېلىش ئۈچۈن بىخەتەرلىك كۈچەيتىشى
* قوشۇلدى: كەم قالغان ياردەمچى functions (wu_format_date, wu_user_can_view_ticket, قاتارلىقلار)
* قوشۇلدى: مۇۋاپىق ھۆججەت قوشۇمچىسى چۈشۈرۈش ۋە بىر تەرەپ قىلىش
* قوشۇلدى: بېلەت جاۋابلىرى ۋە ھالەت ئۆزگىرىشلىرى ئۈچۈن ئېلخەت ئۇقتۇرۇش سىستېمىسى
* تۈزىتىلدى: بېلەت جاۋاب جەدۋىلى ھازىر زۆرۈر بېلەت ID نى ئۆز ئىچىگە ئالىدۇ
* تۈزىتىلدى: باشقۇرۇش بېلەت كۆرۈنۈشلىرىدىكى ئۇقتۇرۇش كۆرۈنۈش مەسىلىلىرى
* تۈزىتىلدى: قوللاش بېلەت functions ئىچىدىكى syntax خاتالىقلىرى
* ياخشىلاندى: جاۋاب ۋە قوشۇمچىلار ئۈچۈن مۇۋاپىق query ۋە schema classes بىلەن ساندان قۇرۇلمىسى
* ياخشىلاندى: بېلەت models ئۈچۈن attributes ئورنىغا ھەقىقىي properties كە كۆچۈش
