# Tutor Multisite Compatibility

كۆپ تور بېكەتلىك مۇھىتلاردا Tutor LMS ئۈچۈن ماسلىشىش تۈزىتىشلىرىنى قوشىدىغان WordPress قىستۇرمىسى.

## چۈشەندۈرۈش {#description}

بۇ قىستۇرما WordPress كۆپ تور بېكەتلىك تورلىرىدا Tutor LMS نى ئىجرا قىلىش ئۈچۈن زۆرۈر ماسلىشىش تۈزىتىشلىرىنى تەمىنلەيدۇ. ئۇ ئىشلەتكۈچى تىزىملىتىش، ئوقۇتقۇچى باشقۇرۇش، دەرس چەكلىمىلىرى ۋە كۆپ تور بېكەتكە خاس باشقا ئىقتىدارلارنى بىر تەرەپ قىلىدۇ.

## ئىقتىدارلار {#features}

- **ساندان جەدۋىلىنى باشقۇرۇش**: كۆپ تور بېكەتلىك تاشقى ئاچقۇچ چەكلىمىلىرى سەۋەبىدىن مەغلۇپ بولغان، كەم قالغان Tutor LMS جەدۋەللىرىنى ئاپتوماتىك بايقايدۇ ۋە قۇرىدۇ
- **كۆپ تور بېكەتلىك ئىشلەتكۈچى تىزىملىتىش**: تور بېكەتلىرى ئارىسىدا ئىشلەتكۈچى تىزىملىتىشنى باشقۇرۇش ئۈچۈن خاس AJAX بىر تەرەپ قىلغۇچلار
- **ئوقۇتقۇچى رولىنى باشقۇرۇش**: كۆپ تور بېكەتلىك مۇھىتلاردا ئوقۇتقۇچى رولىنى ئاپتوماتىك تەقسىملەش ۋە باشقۇرۇش
- **دەرس چەكلىمىلىرى**: تور بېكەت پىلانلىرىغا ئاساسەن دەرس ۋە funnel قۇرۇش چەكلىمىلىرىنى باشقۇرۇش ئۈچۈن Ultimate Multisite بىلەن بىرلەشتۈرۈش
- **خاس باشقۇرغۇچى كۆرۈنۈشلىرى**: كۆپ تور بېكەتكە ماس ئوقۇتقۇچى تىزىملىكى ۋە باشقۇرۇش ئارايۈزى
- **باشقۇرغۇچى قوراللىرى**: تور ۋە تور بېكەت باشقۇرغۇچىلىرى ئۈچۈن ساندان باشقۇرۇش ئارايۈزى

## تەلەپلەر {#requirements}

- WordPress 5.0+
- PHP 7.4+
- Tutor LMS قىستۇرمىسى (`Requires Plugins` بېشى ئارقىلىق ئاپتوماتىك مەجبۇرىي قىلىنىدۇ)
- WordPress كۆپ تور بېكەتلىك ئورنىتىش
- Ultimate Multisite قىستۇرمىسى (دەرس چەكلىمىلىرى ئىقتىدارى ئۈچۈن)

## ئورنىتىش {#installation}

1. قىستۇرما ھۆججەتلىرىنى `/wp-content/plugins/tutor-multisite-compatibility/` غا يۈكلەڭ
2. Tutor LMS نىڭ ئورنىتىلغان ۋە ئاكتىپلاشتۇرۇلغانلىقىغا كاپالەتلىك قىلىڭ
3. WordPress دىكى «قىستۇرمىلار» تىزىملىكى ئارقىلىق قىستۇرمىنى ئاكتىپلاشتۇرۇڭ
4. قىستۇرما Tutor LMS بىلەن ئاپتوماتىك بىرلىشىدۇ

## MU plugin دىن يۆتكەلگەن ئىقتىدارلار {#functionality-moved-from-mu-plugin}

بۇ قىستۇرما ئىلگىرى `wp-content/mu-plugins/daves.php` ئىچىدە بولغان ئىقتىدارلارنى ئۆز ئىچىگە ئالىدۇ:

- دەرس ھالىتىنى ئۆزگەرتىش چەكلىمىلىرى
- دەرس قۇرۇش چەكلىمىلىرى
- CartFlows بىرلەشتۈرۈشى ۋە چەكلىمىلىرى
- Tutor ئوقۇتقۇچى رولىنى باشقۇرۇش
- كۆپ تور بېكەتلىك ئىشلەتكۈچى تىزىملىتىشنى بىر تەرەپ قىلىش
- دەرس مەھسۇلات ID كۆرسىتىش

## ساندان باشقۇرۇش {#database-management}

### مەسىلە {#the-problem}

WordPress كۆپ تور بېكەتلىك مۇھىتلاردا، ئىشلەتكۈچىلەر جەدۋىلىگە تاشقى ئاچقۇچ چەكلىمىلىرى بار جەدۋەللەر ئۈچۈن Tutor LMS جەدۋەل قۇرۇشى مەغلۇپ بولىدۇ. بۇ تۆۋەندىكىلەر سەۋەبىدىن يۈز بېرىدۇ:

- تور بېكەتكە خاس جەدۋەللەر `wp_2_tutor_carts` دېگەندەك ئالدى قوشۇمچىلارنى ئىشلىتىدۇ
- تاشقى ئاچقۇچلار `wp_2_users` نى كۆرسىتىدۇ (ئۇ مەۋجۇت ئەمەس)
- ھەقىقىي ئىشلەتكۈچىلەر جەدۋىلى `wp_users` (تور بويىچە ئورتاق) بولىدۇ

### ھەل قىلىش چارىسى {#the-solution}

بۇ قىستۇرما ئاپتوماتىك ھالدا:

1. **كەم جەدۋەللەرنى بايقايدۇ**: تاشقى ئاچقۇچ چەكلىمىلىرى سەۋەبىدىن قۇرۇلماي قالغان جەدۋەللەرنى تەكشۈرىدۇ
2. **توغرا كۆرسەتمىلەر بىلەن جەدۋەل قۇرىدۇ**: مۇۋاپىق ئىشلەتكۈچى جەدۋەل نامىنى ئىشلىتىدۇ (كۆپ تور بېكەتتە `wp_users`)
3. **كۆپ خىل ئەھۋاللارنى بىر تەرەپ قىلىدۇ**:
   - قىستۇرمىنى ئاكتىپلاشتۇرۇش/قايتا ئاكتىپلاشتۇرۇش
   - توردا يېڭى تور بېكەت قۇرۇش
   - باشقۇرغۇچى ئارايۈزى ئارقىلىق قولدا جەدۋەل تەكشۈرۈش

### تەسىرگە ئۇچرايدىغان جەدۋەللەر {#affected-tables}

- `tutor_coupon_usages` - ئىشلەتكۈچىلەر جەدۋىلىگە تاشقى ئاچقۇچنى ئۆز ئىچىگە ئالىدۇ
- `tutor_carts` - ئىشلەتكۈچىلەر جەدۋىلىگە تاشقى ئاچقۇچنى ئۆز ئىچىگە ئالىدۇ
- `tutor_notification_preferences` - ئىشلەتكۈچىلەر جەدۋىلىگە تاشقى ئاچقۇچ بار TutorPro جەدۋىلى

### باشقۇرغۇچى ئارايۈزى {#admin-interface}

ساندان باشقۇرۇش قوراللىرىغا تۆۋەندىكى يەردىن كىرىڭ:

- **تور باشقۇرغۇچىسى**: تەڭشەكلەر → Tutor Multisite DB
- **تور بېكەت باشقۇرغۇچىسى**: قوراللار → Tutor Multisite DB

بۇ ئارايۈز سىزگە تۆۋەندىكىلەرنى قىلىشقا يول قويىدۇ:
- بارلىق تور بېكەتلەردىكى كەم جەدۋەللەرنى كۆرۈش
- جەدۋەل قۇرۇشنى قولدا قوزغىتىش
- ساندان سەپلىمىسىنى تەكشۈرۈش
- تەپسىلىي ھالەت ئۇچۇرلىرىنى كۆرۈش

## قوللاش {#support}

بۇ قىستۇرما مەخسۇس KursoPro سۇپىسى ئۈچۈن لايىھەلەنگەن. تېخنىكىلىق قوللاش ئۈچۈن، تەرەققىيات ئەترىتى بىلەن ئالاقىلىشىڭ.
