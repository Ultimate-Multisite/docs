# تىل تاللىغۇچ {#language-selector}

Ultimate Multisite: تىزىملىتىشتە تىل تاللىغۇچ

تىزىملىتىش ئېقىمىغا تىل تاللاش بۆلىكى قوشۇڭ.

## ئورنىتىش {#installation}

1. 'ultimate-multisite-language-selector' نى '/wp-content/plugins/' مۇندەرىجىسىگە يۈكلەڭ
2. WordPress Network Admin دىكى «قىستۇرمىلار» تىزىملىكى ئارقىلىق قىستۇرمىنى قوزغىتىڭ

## ئىشلىتىش قوللانمىسى {#usage-guide}

بۇ ھەقسىز قوشۇمچە ئارقىلىق، ئىشلەتكۈچىلىرىڭىزنىڭ تىزىملىتىش ئېقىمى ۋە كىرىش بېتىدە قايسى تىلنى كۆرۈشنى خالايدىغانلىقىنى تاللىشىغا يول قويالايسىز.

[ئىجازەتنامىڭىزنى قوزغاتقاندىن](../../user-guide/getting-started/installing-ultimate-multisite.md) كېيىن، ئىجازەتنامىڭىز زىيارەت ھوقۇقى بېرىدىغان Ultimate Multisite قوشۇمچىلىرىنى ئورنىتالايسىز.

Ultimate Multisite تەڭشەك بېتىڭىزگە كىرىڭ، ئاندىن سول تەرەپتىكى تىزىملىكتىن _قوشۇمچىلار_ بەلگىسىنى چېكىڭ.

![قوشۇمچىلار بېتى](/img/admin/addons-page.png)

_Language Selector نى ئورنىتىشنى باشلاش ئۈچۈن قوشۇمچىلار بېتىگە كىرىڭ_

ئۇ يەردە، بىز تەمىنلەيدىغان بارلىق قوشۇمچىلارنى تاپالايسىز.

ھەقسىز قوشۇمچىلار مۇشتەرىلىك پىلانى قانداق بولۇشىدىن قەتئىينەزەر، بارلىق خېرىدارلىرىمىزغا تەمىنلىنىدۇ.

قوشۇمچىنىڭ كىچىك سۈرىتىنىڭ ئاستىدىكى _ھازىرلا ئورنىتىڭ!_ كۇنۇپكىسىنى چېكىڭ. ئاندىن ئۇنى قىستۇرمىلار بېتىدە قوزغىتىڭ.

![ئورنىتىش كۇنۇپكىسى بار قوشۇمچىلار بېتى](/img/admin/addons-page.png)

_قوشۇمچىنى ئورنىتىش ئۈچۈن كۇنۇپكىنى چېكىڭ_

<!-- ئېكران كۆرۈنۈشى يوق: WP قىستۇرمىلار بېتى Language Selector قوشۇمچىسى ئۈچۈن تور بويىچە قوزغىتىش كۇنۇپكىسىنى كۆرسىتىدۇ -->

_ئۇنى قىستۇرمىلار بېتىڭىزدە قوزغىتىڭ_

ئەمدى، Ultimate Multisite نىڭ تەڭشەكلەر تىزىملىكىگە كىرىپ، _تور تەڭشەكلىرى_ بەتكۈچىگە كىرىڭ. ئۇ يەردە، بەتنىڭ ئاستى قىسمىدا _تىللار_ دەپ ئاتىلىدىغان يېڭى بىر بۆلەك تاپالايسىز.

![تەڭشەكلەر بېتى](/img/config/settings-general.png)

_پۈتۈنلەي يېڭى بىر بۆلەك كۆرۈنىدۇ!_

بارلىق تىللارنى تاللاش سۈپىتىدە كۆرسىتىش ئۈچۈن ئۇنى بوش قالدۇرالايسىز ياكى قايسىلىرىنى تەمىنلەشنى خالايدىغانلىقىڭىزنى تاللىيالايسىز. تىللار WordPress تەڭشىكىڭىزدە ئورنىتىلغان تىللاردۇر.

<!-- ئېكران كۆرۈنۈشى يوق: WP ئورنىتىشىدا بار تىللارنى كۆرسىتىدىغان تىل تاللاش چۈشۈرۈلمە تىزىملىكى -->

_بۇ WP ئورنىتىشىدا بار تىللار تىزىملىكى_

قىلغان ئۆزگەرتىشلىرىڭىزنى ساقلىغاندىن كېيىن، سىز تاللىغان تىللار بار تاللاش تىزىملىكى كىرىش بېتىدە شۇنداقلا تىزىملىتىش ئېقىمىدا كۆرسىتىلىدۇ.

![كىرىش ۋە تىزىملىتىش تەڭشەكلىرى](/img/config/settings-login-registration.png)

_كىرىش بېتىدىكى Language Selector_

![تىزىملىتىش بېتى](/img/config/registration-page.png)

_تىزىملىتىش ئېقىمىدىكى Language Selector_

## URL ئارقىلىق تىلنى تەڭشەش {#setting-the-language-via-the-url}

سىز يەنە locale پارامېتىرىنى URL غا يوللاش ئارقىلىق بەتنىڭ تىلىنى ئۆزگەرتەلەيسىز. مەسىلەن، ئىتاليانچە تىزىملىتىش بېتىگە قايتا يۆنىمەكچى بولسىڭىز، URL غا ?locale=it_IT قوشۇڭ.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

بۇنىڭ ئىشلىشى ئۈچۈن تىل WordPress تورىڭىزدا ئورنىتىلغان بولۇشى كېرەك.
