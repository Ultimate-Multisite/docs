# سۈزگۈچ: signup_get_available_languages

ئالدى تەرەپتىكى تور بېكەت تىزىملىتىشلىرى ئۈچۈن ئىشلەتكىلى بولىدىغان تىللار تىزىملىكىنى سۈزۈپ بېرىدۇ.

بۇ hook قا بوش array يوللانسا، تىزىملىتىش جەدۋىلىدە بۇ تەڭشەكنىڭ چىقىرىلىشى چەكلىنىدۇ، تور بېكەت قۇرۇلغاندا كۆڭۈلدىكى تىل ئىشلىتىلىدۇ. ئالدىن ئورنىتىلمىغان تىللار چىقىرىۋېتىلىدۇ.

## پارامېتىرلار {#parameters}

| نامى | تۈرى | چۈشەندۈرۈش |
|------|------|-------------|
| $available_languages | `array` | ئىشلەتكىلى بولىدىغان تىللار. |

- 4.4.0
