# Multi-Currency {#multi-currency}

Ultimate Multisite: Multi-Currency تورىڭىزنىڭ كۆپ خىل پۇل بىرلىكىدە ھەق قوبۇل قىلىشىغا ئىمكان بېرىدۇ. ئۇ جانلىق ياكى قولدا تەڭشەلگەن ئالماشتۇرۇش نىسبىتى ئارقىلىق باھالارنى ئاپتوماتىك ئايلاندۇرىدۇ، ھەرقانداق پۇل بىرلىكى ئۈچۈن ھەر مەھسۇلاتقا ئېنىق باھا بەلگىلىشىڭىزگە يول قويىدۇ، ھەمدە جۇغراپىيەلىك ئورۇن ئارقىلىق ھەر بىر زىيارەتچىنىڭ ياخشى كۆرىدىغان پۇل بىرلىكىنى بايقايدۇ — ھەممىسى بىرلا باشقۇرغۇچى Dashboard دىن باشقۇرۇلىدۇ.

## ئاساسلىق ئىقتىدارلار {#key-features}

- **ئاپتوماتىك ئالماشتۇرۇش نىسبىتى ئايلاندۇرۇشى** — باھالار تاللىغان تەمىنلىگۈچىڭىزدىن ئېلىنغان نىسبەتلەر ئارقىلىق ئاساسىي پۇل بىرلىكىڭىزدىن ئايلاندۇرۇلىدۇ
- **ھەر مەھسۇلات باھاسىنى قاپلاش** — ئاپتوماتىك ئايلاندۇرۇشنى ئايلىنىپ ئۆتۈش ئۈچۈن ھەرقانداق مەھسۇلاتقا پۇل بىرلىكىگە خاس ئېنىق باھالارنى بەلگىلەڭ
- **پۇل بىرلىكى تاللىغۇچ تىزىملىتىش بۆلىكى** — خېرىدارلارنىڭ checkout/تىزىملىتىش جەريانىدا ئۆزى ياخشى كۆرىدىغان پۇل بىرلىكىنى تاللىشىغا يول قويىدۇ
- **كۆپ خىل ئالماشتۇرۇش نىسبىتى تەمىنلىگۈچىلەر** — ECB، Frankfurter، Stripe ياكى تولۇق قولدا تەڭشىلىدىغان نىسبەتلەر
- **جۇغراپىيەلىك ئورۇنغا ئاساسلانغان پۇل بىرلىكىنى بايقاش** — زىيارەتچىنىڭ دۆلىتىگە ماس كېلىدىغان پۇل بىرلىكىنى ئاپتوماتىك ئالدىن تاللايدۇ
- **باشقۇرغۇچى Dashboard** — ئاكتىپ پۇل بىرلىكلىرى، ئالماشتۇرۇش نىسبەتلىرى ۋە تەمىنلىگۈچى تەڭشەكلىرىنى بىر جايدىن باشقۇرۇڭ

## تەلەپلەر {#requirements}

- Ultimate Multisite 2.0.0 ياكى ئۇنىڭدىن يۇقىرى
- WordPress Multisite ئورنىتىلمىسى
- PHP 7.4 ياكى ئۇنىڭدىن يۇقىرى

## ئورنىتىش {#installation}

1. plugin ھۆججەتلىرىنى `/wp-content/plugins/ultimate-multisite-multi-currency` غا يۈكلەڭ
2. WordPress دىكى **Plugins** ئېكرانى ئارقىلىق plugin نى ئاكتىپلاڭ (تور بويىچە ئاكتىپلاش)
3. باشقۇرغۇچى Dashboard نى ئېچىش ئۈچۈن **Ultimate Multisite → Multi-Currency** غا يۆتكىلىڭ
4. قوبۇل قىلماقچى بولغان پۇل بىرلىكلىرىنى قوشۇڭ ۋە ئالماشتۇرۇش نىسبىتى تەمىنلىگۈچىڭىزنى سەپلەڭ

## سەپلەش {#configuration}

### باشقۇرغۇچى Dashboard {#admin-dashboard}

Multi-Currency باشقۇرغۇچى Dashboard (**Ultimate Multisite → Multi-Currency**) تۆۋەندىكىلەر ئۈچۈن مەركىزىي ئورۇن:

- پۇل بىرلىكلىرىنى قوشۇش، قوزغىتىش ياكى چەكلەش
- ئالماشتۇرۇش نىسبىتى تەمىنلىگۈچىنى تاللاش ۋە ئېھتىياجغا قاراپ نىسبەتلەرنى يېڭىلاش
- بارلىق ئاكتىپ پۇل بىرلىكلىرىنىڭ نۆۋەتتىكى نىسبەتلىرىنى تەكشۈرۈش
- ئاپتوماتىك تەمىنلىگۈچىلەر ماس كەلمىگەندە قولدا نىسبەت بەلگىلەش

### ئالماشتۇرۇش نىسبىتى تەمىنلىگۈچىلەر {#exchange-rate-providers}

| تەمىنلىگۈچى | چۈشەندۈرۈش |
|---|---|
| **ECB** | ياۋروپا مەركىزى بانكىسى — EUR ئاساس قىلىنغان تورلار ئۈچۈن ھەقسىز كۈندىلىك نىسبەتلەر |
| **Frankfurter** | ئوچۇق مەنبەلىك ECB ئەينەك API — ھەقسىز، API ئاچقۇچى تەلەپ قىلىنمايدۇ |
| **Stripe** | Stripe نىڭ جانلىق FX نىسبەتلىرىنى ئىشلىتىدۇ — ئاكتىپ Stripe بىرلەشتۈرۈشىنى تەلەپ قىلىدۇ |
| **Manual** | نىسبەتلەرنى بىۋاسىتە ئۆزىڭىز كىرگۈزىسىز؛ تاشقى API چاقىرىقلىرى يوق |

تەمىنلىگۈچىڭىزنى **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider** ئاستىدىن تاللاڭ. نىسبەتلەرنى ھەر ۋاقىت قولدا يېڭىلىغىلى ياكى ئاپتوماتىك يېڭىلىنىدىغان قىلىپ ۋاقىتلاشتۇرغىلى بولىدۇ.

### ھەر مەھسۇلات باھاسىنى قاپلاش {#per-product-pricing-overrides}

ئاپتوماتىك ئايلاندۇرۇش يېتەرلىك دەرىجىدە ئېنىق بولمىغاندا، ھەرقانداق مەھسۇلاتتا مەلۇم بىر پۇل بىرلىكى ئۈچۈن ئېنىق باھا بەلگىلىيەلەيسىز:

1. مەھسۇلات تەھرىرلىگۈچىنى ئېچىڭ (**Ultimate Multisite → Products → Edit**)
2. **Currency Pricing** بۆلىكىگە سىيرىڭ
3. قاپلىماقچى بولغان ھەر بىر پۇل بىرلىكى ئۈچۈن ئېنىق باھانى كىرگۈزۈڭ
4. مەھسۇلاتنى ساقلاڭ

قاپلاش بەلگىلەنمىگەن مەھسۇلاتلار ئاساسىي پۇل بىرلىكىڭىزدىن ئايلاندۇرۇلغان باھانى ئىشلىتىدۇ.

### پۇل بىرلىكى تاللىغۇچ تىزىملىتىش بۆلىكى {#currency-selector-signup-field}

پۇل بىرلىكى تاللىغۇچ بۆلىكىنى تىزىملىتىش/checkout ئېقىمىڭىزغا قوشۇپ، خېرىدارلارنىڭ تىزىملىتىش ۋاقتىدا ئۆزى ياخشى كۆرىدىغان پۇل بىرلىكىنى تاللىشىغا ئىمكان بەرگىلى بولىدۇ:

1. **Ultimate Multisite → Multi-Currency → Settings** غا بېرىڭ
2. **Show currency selector on signup** نى قوزغىتىڭ
3. ئىختىيارىي ھالدا كۆڭۈلدىكى پۇل بىرلىكىنى بەلگىلەڭ ياكى جۇغراپىيەلىك ئورۇننىڭ ئالدىن تاللانغان قىممەتنى بەلگىلىشىگە يول قويۇڭ

تاللانغان پۇل بىرلىكى مۇشتەرىلىك بىلەن بىللە ساقلىنىدۇ ۋە كېيىنكى بارلىق ھېساب قىلىشتا ئىشلىتىلىدۇ.

### جۇغراپىيەلىك ئورۇنغا ئاساسلانغان پۇل بىرلىكىنى بايقاش {#geolocation-based-currency-detection}

جۇغراپىيەلىك ئورۇننى بايقاش قوزغىتىلغاندا، addon زىيارەتچىنىڭ دۆلىتىنى بىر پۇل بىرلىكىگە ماسلاشتۇرىدۇ ۋە ئۇنى پۇل بىرلىكى تاللىغۇچ بۆلىكىدە ئالدىن تاللايدۇ. قوزغىتىش ئۈچۈن:

1. **Ultimate Multisite → Multi-Currency → Settings** غا بېرىڭ
2. **Detect currency from visitor location** نى قوزغىتىڭ
3. **Geolocation Mapping** جەدۋىلىدە دۆلەتلەرنى پۇل بىرلىكلىرىگە ماسلاشتۇرۇڭ

ئەگەر زىيارەتچىنىڭ دۆلىتى ئۈچۈن ماسلاشتۇرۇش مەۋجۇت بولمىسا، تورنىڭ ئاساسىي پۇل بىرلىكى fallback سۈپىتىدە ئىشلىتىلىدۇ.

## كۆپ سورىلىدىغان سوئاللار {#frequently-asked-questions}

### ئەگەر ئالماشتۇرۇش نىسبىتى تەمىنلىگۈچى ئىشلەتكىلى بولمىسا نېمە بولىدۇ؟ {#what-happens-if-an-exchange-rate-provider-is-unavailable}

addon ئاخىرقى قېتىم مۇۋەپپەقىيەتلىك ئېلىنغان نىسبەتلەرگە قايتىدۇ. ئەگەر ئەزەلدىن ھېچقانداق نىسبەت ئېلىنمىغان بولسا، ئۇ سىز سەپلىگەن قولدا نىسبەتلەرگە قايتىدۇ. بىخەتەرلىك تورى سۈپىتىدە قولدا نىسبەتلەرنى سەپلەش تەۋسىيە قىلىنىدۇ.

### ئوخشىمىغان پۇل بىرلىكلىرى ئۈچۈن ئوخشىمىغان تەمىنلىگۈچىلەرنى ئىشلىتەمدىم؟ {#can-i-use-different-providers-for-different-currencies}

ياق — بارلىق پۇل بىرلىكلىرى ئۈچۈن بىرلا تەمىنلىگۈچى بىر ۋاقىتتا ئاكتىپ بولىدۇ. ئايرىم نىسبەتلەر ئۈستىدە تولۇق كونترولغا ئېھتىياجىڭىز بولسا **Manual** نى ئىشلىتىڭ.

### جۇغراپىيەلىك ئورۇن ئايرىم plugin تەلەپ قىلامدۇ؟ {#does-geolocation-require-a-separate-plugin}

تاشقى جۇغراپىيەلىك ئورۇن plugin تەلەپ قىلىنمايدۇ. addon WordPress نىڭ ئىچىگە قۇرۇلغان IP دىن دۆلەتكە بايقاش ئىقتىدارىنى ئىشلىتىدۇ. تېخىمۇ يۇقىرى ئېنىقلىق ئۈچۈن، مەخسۇس جۇغراپىيەلىك ئورۇن مۇلازىمىتى تەمىنلەنگەن filter hook لار ئارقىلىق بىرلەشتۈرۈلسە بولىدۇ.

### ھەر مەھسۇلات قاپلاشلىرى باجدىن بۇرۇن قوللىنىلامدۇ ياكى كېيىنمۇ؟ {#are-per-product-overrides-applied-before-or-after-tax}

ھەر مەھسۇلات قاپلاشلىرى نىشان پۇل بىرلىكىدىكى باجدىن بۇرۇنقى باھانى بەلگىلەيدۇ. باج ھازىرقى باج سەپلىمىڭىزنى ئىشلىتىپ قاپلاش باھاسىنىڭ ئۈستىگە ھېسابلىنىدۇ.
