# لوگو SVG ھاسىل قىلىش

**لوگو SVG ھاسىل قىلىش** ئىقتىدارى Theme Builder نىڭ خاس لوگو SVG لىرىنى بىۋاسىتە WordPress تور بېكىتىڭىزگە قۇرۇپ قىستۇرۇشىغا، شۇنداقلا ئاپتوماتىك ئات بوشلۇقى بىخەتەر تازىلاش ئېلىپ بېرىشىغا يول قويىدۇ.

## ئومۇمىي چۈشەنچە {#overview}

بۇ ئىقتىدار تور بېكىتىڭىزنىڭ ماركا يۆنىلىشى ۋە لايىھە تاللاشلىرىغا ئاساسەن كېڭەيتكىلى بولىدىغان ۋېكتورلۇق گرافىك (SVG) لوگولارنى ھاسىل قىلىدۇ. ھاسىل قىلىنغان SVG لار كۆرۈنۈش پۈتۈنلۈكىنى ساقلىغان ھالدا WordPress تا ئىشلىتىشكە بىخەتەر بولۇشى ئۈچۈن ئاپتوماتىك تازىلىنىدۇ.

## پارامېتىرلار {#parameters}

| پارامېتىر | تۈرى | زۆرۈر | چۈشەندۈرۈش |
|-----------|------|----------|-------------|
| `site_name` | string | ھەئە | لوگو ھاسىل قىلىنىدىغان تور بېكەتنىڭ نامى |
| `style` | string | ھەئە | لايىھە ئۇسلۇبى (مەسىلەن، "modern"، "classic"، "minimalist"، "playful") |
| `colors` | array | ياق | لوگودا ئىشلىتىلىدىغان ئون ئالتىلىق رەڭ كودلىرىنىڭ تىزىقى (مەسىلەن، `["#678233", "#ffffff"]`) |
| `width` | number | ياق | پىكسېل بويىچە SVG كەڭلىكى (سۈكۈتتىكى: 200) |
| `height` | number | ياق | پىكسېل بويىچە SVG ئېگىزلىكى (سۈكۈتتىكى: 200) |
| `include_text` | boolean | ياق | تور بېكەت نامىنى لوگودا تېكىست سۈپىتىدە ئۆز ئىچىگە ئالامدۇ-يوق (سۈكۈتتىكى: true) |

## چىقىرىش فورماتى {#output-format}

بۇ ئىقتىدار تۆۋەندىكى قۇرۇلما بىلەن SVG تىزمىسى قايتۇرىدۇ:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG تازىلاش ھەرىكىتى {#svg-sanitisation-behaviour}

ھاسىل قىلىنغان SVG لار تۆۋەندىكىلەر ئۈچۈن ئاپتوماتىك ئات بوشلۇقى بىخەتەر تازىلاشتىن ئۆتىدۇ:

- **بىخەتەر بولمىغان خاسلىقلارنى چىقىرىۋېتىش** — ھادىسە بىر تەرەپ قىلغۇچلار، قوليازمىلار ۋە ئېھتىمال خەتەرلىك خاسلىقلارنى چىقىرىۋېتىدۇ
- **ئات بوشلۇقلىرىنى ساقلاش** — توغرا كۆرسىتىش ئۈچۈن SVG ئات بوشلۇقلىرىنى (xmlns, xlink) ساقلايدۇ
- **قۇرۇلمىنى دەلىللەش** — SVG نىڭ W3C ئۆلچىمىگە ئۇيغۇن بولۇشىغا كاپالەتلىك قىلىدۇ
- **گەۋدىلەرنى كودلاش** — تېكىست مەزمۇنىدىكى ئالاھىدە ھەرپلەرنى توغرا قېچىرىدۇ
- **سىرتقى پايدىلىنىشلارنى چىقىرىۋېتىش** — سىرتقى ئۇسلۇب جەدۋەللىرى ۋە سۈرەت پايدىلىنىشلىرىنى چىقىرىۋېتىدۇ

بۇ SVG نىڭ قوشۇمچە تازىلاش تەلەپ قىلمايلا WordPress قا بىۋاسىتە قىستۇرۇشقا بىخەتەر بولۇشىغا كاپالەتلىك قىلىدۇ.

## ئىشلىتىش مىسالى {#usage-example}

**ئەسكەرتىش:**
```
"CloudSync" دەپ ئاتىلىدىغان تېخنىكا ئىگىلىك باشلىمىم ئۈچۈن كۆك ۋە ئاق رەڭلەرنى ئىشلىتىپ زامانىۋى لوگو ھاسىل قىلىڭ.
```

**نەتىجە:**
بۇ ئىقتىدار تۆۋەندىكىدەك SVG لوگو قۇرىدۇ:
- "CloudSync" تور بېكەت نامىنى ئۆز ئىچىگە ئالىدۇ
- بەلگىلەنگەن كۆك ۋە ئاق رەڭ لايىھىسىنى ئىشلىتىدۇ
- زامانىۋى لايىھە پىرىنسىپلىرىغا ئەگىشىدۇ
- ئاپتوماتىك تازىلانغان بولۇپ، ئىشلىتىشكە تەييار

## Theme Builder بىلەن بىرلەشتۈرۈش {#integration-with-theme-builder}

Theme Builder نىڭ لايىھە يۆنىلىشى تاللاش ئىقتىدارىنى ئىشلەتكەندە، لوگو SVG ھاسىل قىلىش ئىقتىدارى:

1. لايىھە يۆنىلىشىڭىز ۋە رەڭ پالېتىڭىزنى تەھلىل قىلىدۇ
2. تاللاشلىرىڭىزغا ماس كېلىدىغان خاس SVG لوگو ھاسىل قىلىدۇ
3. لوگونى تور بېكىتىڭىزنىڭ بەت بېشى/ماركا رايونىغا ئاپتوماتىك قىستۇرىدۇ
4. SVG نى WordPress ۋاسىتىلىرىدە خاس لوگو سۈپىتىدە ساقلايدۇ

## ئەڭ ياخشى ئەمەلىيەتلەر {#best-practices}

- **ئېنىق ئۇسلۇب تاللاشلىرىنى تەمىنلەڭ** — خالايدىغان لايىھە ئۇسلۇبىنى تەسۋىرلەڭ (زامانىۋى، كلاسسىك، شوخ قاتارلىقلار)
- **رەڭلەرنى بەلگىلەڭ** — ئىزچىللىق ئۈچۈن ماركا رەڭلىرىڭىزنى كىرگۈزۈڭ
- **كۆرسىتىلىشىنى سىناڭ** — لوگونىڭ ئوخشىمىغان ئېكران چوڭلۇقلىرىدا توغرا كۆرسىتىلىدىغانلىقىنى دەلىللەڭ
- **تېخىمۇ خاسلاشتۇرۇڭ** — WordPress نىڭ لوگو خاسلاشتۇرۇش قوراللىرىنى ئىشلىتىپ چوڭلۇق ۋە ئورۇنلاشتۇرۇشنى تەڭشەڭ

## چەكلىمىلەر {#limitations}

- SVG لوگولار تۇراقلىق گرافىك سۈپىتىدە ھاسىل قىلىنىدۇ (ھەرىكەتلىك ئەمەس)
- نۇرغۇن ئېلېمېنتلىرى بار مۇرەككەپ لوگولار قولدا ئىنچىكە تەڭشەشنى تەلەپ قىلىشى مۇمكىن
- خاس خەت نۇسخىلىرى قوللانمايدۇ؛ تېكىست سىستېما خەت نۇسخىلىرىنى ئىشلىتىدۇ
- بەك چوڭ ياكى بەك كىچىك ئۆلچەملەر سۈپەتكە تەسىر كۆرسىتىشى مۇمكىن
