# تىزىملىك قۇرۇش {#create-menu}

**تىزىملىك قۇرۇش** ئىقتىدارى بەت ماۋزۇلىرىدىن ئايرىم بولغان پەرقلىق يولباشلاش بەلگىلىرىنى قوللايدىغان WordPress يولباشلاش تىزىملىكلىرىنى قۇرىدۇ.

## ئومۇمىي چۈشەنچە {#overview}

بۇ ئىقتىدار ئۆلچەملىك تىزىملىك قۇرۇش فۇنكسىيەسىنى `navigation_label` پارامېتىرىنى بەلگىلەش ئىقتىدارى بىلەن كېڭەيتىدۇ. بۇ، يولباشلاشتا كۆرسىتىلىدىغان بەلگە بەت ماۋزۇسىدىن پەرقلىنىدىغان تىزىملىكلەرنى قۇرۇشقا يول قويۇپ، تور بېكەت قۇرۇلمىسى ۋە ئىشلەتكۈچى تەجرىبىسىدە تېخىمۇ كۆپ جانلىقلىق تەمىنلەيدۇ.

## پارامېتىرلار {#parameters}

| پارامېتىر | تۈرى | زۆرۈر | چۈشەندۈرۈش |
|-----------|------|----------|-------------|
| `name` | string | ھەئە | تىزىملىك نامى، مەسىلەن `Primary Navigation` |
| `location` | string | ياق | بۇ تىزىملىكنى تەقسىملەيدىغان theme ئورنى، مەسىلەن `primary` |
| `navigation_label` | string | ياق | يولباشلاشتا كۆرسىتىلىدىغان بەلگە (بەت ماۋزۇسىدىن پەرقلىق) |

## قايتۇرۇلغان قىممەت {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## يولباشلاش بەلگىسى بىلەن بەت ماۋزۇسى {#navigation-label-vs-page-title}

`navigation_label` پارامېتىرى ئىچكى تىزىملىك نامىنى ئىشلەتكۈچىلەرگە كۆرسىتىلىدىغان بەلگىدىن ئايرىشىڭىزغا يول قويىدۇ:

- **`name`** — WordPress ئىشلىتىدىغان ئىچكى تىزىملىك پەرقلەندۈرگۈچىسى (مەسىلەن، "Primary Navigation")
- **`navigation_label`** — تور بېكەت زىيارەتچىلىرىگە يولباشلاشتا كۆرسىتىلىدىغان بەلگە (مەسىلەن، "Main Menu")

بۇ تۆۋەندىكى ئەھۋاللاردا پايدىلىق:
- ئىچكى نام قويۇش قائىدىڭىز ئىشلەتكۈچىگە كۆرۈنىدىغان بەلگىلەردىن پەرقلىق بولسا
- يولباشلاشتا باشقۇرۇش تاختىسىدىكىدىن قىسقىراق بەلگىلەرنى خالىسىڭىز
- ئوخشىمىغان بەلگە ئۇزۇنلۇقلىرى بار كۆپ تىلنى قوللاشقا ئېھتىياجلىق بولسىڭىز
- مەخسۇس رايونلار ياكى ئىشلەتكۈچى گۇرۇپپىلىرى ئۈچۈن تىزىملىكلەر قۇرۇۋاتقان بولسىڭىز

## ئىشلىتىش مىساللىرى {#usage-examples}

### مىسال 1: يولباشلاش بەلگىسى بار ئاددىي تىزىملىك {#example-1-simple-menu-with-navigation-label}

**Prompt:**
```
"Main Navigation" دەپ ئاتىلىدىغان، يولباشلاش بەلگىسى "Menu" بولغان ئاساسىي يولباشلاش تىزىملىكى قۇرۇڭ.
```

**نەتىجە:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### مىسال 2: مەخسۇس Theme ئورنى ئۈچۈن تىزىملىك {#example-2-menu-for-specific-theme-location}

**Prompt:**
```
"Footer Links" دەپ ئاتىلىدىغان، يولباشلاش بەلگىسى "Quick Links" بولغان footer تىزىملىكى قۇرۇڭ ۋە ئۇنى footer ئورنىغا تەقسىملەڭ.
```

**نەتىجە:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Theme Builder بىلەن بىرلەشتۈرۈش {#integration-with-theme-builder}

Theme Builder ئىشلىتىلگەندە، تىزىملىك قۇرۇش ئىقتىدارى:

1. مەۋجۇت theme تىزىملىك ئورۇنلىرىنى ئاپتوماتىك بايقايدۇ
2. لايىھىڭىزگە ماس يولباشلاش بەلگىلىرى بىلەن تىزىملىكلەرنى قۇرىدۇ
3. تىزىملىكلەرنى توغرا theme ئورۇنلىرىغا تەقسىملەيدۇ
4. قۇرۇلغاندىن كېيىن تىزىملىك تۈرلىرىنى قوشۇشنى قوللايدۇ

## مۇناسىۋەتلىك ئىقتىدارلار {#related-abilities}

- **`add_menu_item`** — مەۋجۇت تىزىملىككە تۈرلەر قوشۇش
- **`update_menu`** — تىزىملىكنىڭ نامىنى ئۆزگەرتىش ياكى ئۇنى theme ئورنىغا قايتا تەقسىملەش
- **`delete_menu`** — تور بېكىتىڭىزدىن تىزىملىكنى ئۆچۈرۈش

## ئەڭ ياخشى ئەمەلىيەتلەر {#best-practices}

- **ئېنىق يولباشلاش بەلگىلىرىنى ئىشلىتىڭ** — بەلگىلەرنى ئىشلەتكۈچىلەر ئۈچۈن ئىخچام ۋە تەسۋىرلىك قىلىڭ
- **Theme ئورۇنلىرىغا ماسلاشتۇرۇڭ** — توغرا كۆرسىتىش ئۈچۈن تىزىملىكلەرنى توغرا theme ئورنىغا تەقسىملەڭ
- **تىزىملىك قۇرۇلمىسىنى پىلانلاڭ** — تۈرلەرنى قۇرۇشتىن بۇرۇن تىزىملىك قاتلام قۇرۇلمىڭىزنى بەلگىلەڭ
- **جاۋابچانلىقنى سىناڭ** — تىزىملىكلەرنىڭ كۆچمە ئۈسكۈنىلەردە توغرا كۆرسىتىلىدىغانلىقىنى دەلىللەڭ
- **بەلگىلەرنى يەرلىكلەشتۈرۈڭ** — ئوخشىمىغان تىل نەشرلىرى ئۈچۈن ئوخشىمىغان يولباشلاش بەلگىلىرىنى ئىشلىتىڭ

## چەكلىمىلەر {#limitations}

- يولباشلاش بەلگىلىرى پەقەت كۆرسىتىش ئۈچۈندۇر؛ ئىچكى `name` يەنىلا WordPress پەرقلەندۈرۈشى ئۈچۈن ئىشلىتىلىدۇ
- Theme قوللىشى پەرقلىنىدۇ؛ بارلىق theme لار بارلىق تىزىملىك ئورۇنلىرىنى قوللىمايدۇ
- تىزىملىك تۈرلىرى تىزىملىك قۇرۇلغاندىن كېيىن ئايرىم قوشۇلۇشى كېرەك
- يولباشلاش بەلگىسىنى ئۆزگەرتىش تىزىملىكنى يېڭىلاشنى تەلەپ قىلىدۇ
