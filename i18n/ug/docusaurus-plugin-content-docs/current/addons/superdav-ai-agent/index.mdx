# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent بولسا Gratis AI Agent نى تېما-خەۋەردار ماھارەتلەر، كۈچەيتىلگەن ئىنتېرنېت ئىزدەش ئىقتىدارلىرى، تور بېكەت قۇرغۇچ ئىقتىدارلىرى ۋە ئىختىيارىي ئاپتوماتلاشتۇرۇش بىرلەشتۈرۈلۈشلىرى بىلەن كېڭەيتىدىغان ئىلغار WordPress AI ياردەمچىسىدۇر. 1.18.0 نەشرى Google Calendar ئوقۇش قوراللىرى، ئالاقىداش ۋە قاتناشقۇچى خەرىتىلەش، ئىنسان تەستىق دەرۋازىلىرى، ئەسكەرتىش تەكرارىنى يوقىتىش، TextBee SMS ئۇقتۇرۇشلىرى، باشقۇرۇلىدىغان Superdav مۇلازىمەت تەمىنلەش، ۋە ئىشەنچلىك ئاچقۇچى خىزمەت ئېقىملىرى ئۈچۈن ئايرىم تارقىتىلىدىغان Advanced ھەمراھ بولاقىنى قوشىدۇ.

## ئاساسلىق ئىقتىدارلار {#key-features}

- **تېما-خەۋەردار ماھارەتلەر** — ئاكتىپ WordPress تېمىڭىز ۋە قاچىلانغان plugin لىرىڭىزغا ئاساسەن يېتەكچىلىك ۋە ئىقتىدارلارنى ئاپتوماتىك ماسلاشتۇرىدۇ
- **كۆپ خىل ئىنتېرنېت ئىزدەش تەمىنلىگۈچىلەر** — تېخىمۇ مول ئىزدەش نەتىجىلىرى ۋە ئىلغار تەتقىقات ئىقتىدارلىرى ئۈچۈن Brave Search بىلەن Tavily ئارىسىدىن تاللاڭ
- **تور بېكەت قۇرغۇچ ئالاقە جەدۋەللىرى** — قاچىلانغان جەدۋەل plugin ىڭىزنى ئىشلىتىپ، سۆھبەت يۈزىدىن بىۋاسىتە تولۇق ئىشلەيدىغان ئالاقە جەدۋەللىرىنى قۇرۇڭ
- **كۈچەيتىلگەن تور بېكەت قۇرغۇچ** — plugin بايقاش ۋە خاتالىقتىن ئەسلىگە كېلىش تېخىمۇ ياخشى بولغان ياخشىلانغان كۆپ باسقۇچلۇق تور بېكەت قۇرۇش پىلانلىرى
- **ئىچىگە قۇرۇلغان تېما ماھارەتلىرى** — Block Themes، Classic Themes، Kadence Blocks ۋە Kadence Theme ئۈچۈن مەخسۇس يېتەكچىلىك
- **ياخشىلانغان WooCommerce بىرلەشتۈرۈلۈشى** — تېخىمۇ ياخشى ئىشەنچلىكلىك ۋە ماسلىشىشچانلىق ئۈچۈن ئەسلى WooCommerce API لىرىنى ئىشلىتىشكە قايتا قۇرۇلدى
- **تەمىنلىگۈچىنى ئاپتوماتىك بايقاش** — plugin لار ۋە تېمىلار ئاكتىپلانغان ياكى ئاكتىپسىزلانغاندا بارلىرىنى ئاپتوماتىك بايقاپ يېڭىلايدۇ
- **خاس لوگو ھاسىل قىلىش** — ئاپتوماتىك namespace-بىخەتەر تازىلاش بىلەن خاس SVG لوگولارنى ھاسىل قىلىپ قىستۇرىدۇ
- **رەڭ تاختىسى سېلىشتۇرمىسىنى دەلىللەش** — تېمىڭىزغا قوللىنىشتىن بۇرۇن رەڭ جۈپلىرىنىڭ WCAG قولايلىقلىق ماسلىقىنى تەكشۈرۈڭ
- **سۈرەت بىلەن كۈچەيتىلگەن بايقاش** — Theme Builder بايقاش سۆھبىتى جەريانىدا سۈرەت يۈكلەپ، لايىھە يۆنىلىشىگە تەسىر كۆرسىتىڭ
- **مېھماندارچىلىق تىزىملىكلىرى** — رېستوران، قەھۋەخانا ۋە مېھماندارچىلىق كارخانىلىرى ئۈچۈن قۇرۇلمىلىق يېمەكلىك ۋە ئىچىملىك تىزىملىك بەتلىرىنى ھاسىل قىلىڭ
- **ئۈستەلئۈستى ۋە كۆچمە ئالدىن كۆرۈنۈشلىرى** — لايىھە يۆنىلىشىنى تاللاش جەريانىدا لايىھىڭىزنى ئۈستەلئۈستى ۋە كۆچمە ئۈسكۈنىلەردە ئالدىن كۆرۈڭ
- **1-دەرىجە تور بېكەت قىرىش** — sd-ai-agent/site-scrape ھازىر رىقابەتچى ۋە ئىلھام تور بېكەتلىرىنى تەھلىل قىلىش ئۈچۈن كۆڭۈلدىكى ھالەتتە ئىشلىتىشكە بولىدۇ
- **Google Calendar ئوقۇش قوراللىرى** — سەپلەنگەن كالېندار ۋە پائالىيەتلەرنى ئوقۇپ، ئاپتوماتلاشتۇرۇشلارنىڭ كېلىدىغان ۋاقىت جەدۋىلى مەزمۇنى ھەققىدە مۇلاھىزە قىلىشىغا ياردەم بېرىدۇ
- **ئالاقىداش ۋە قاتناشقۇچى خەرىتىلەش** — كېيىنكى خىزمەتنى ھاسىل قىلىشتىن بۇرۇن كالېندار قاتناشقۇچىلىرىنى تونۇلغان تور بېكەت ئىشلەتكۈچىلىرى ۋە ئالاقىداشلارغا ماسلاشتۇرىدۇ
- **ئىنسان تەستىق دەرۋازىلىرى** — ئۇچۇر ئەۋەتىش، خىزمەتنى داۋاملاشتۇرۇش ياكى ئۆزگەرتىش كىرگۈزۈشتىن بۇرۇن سەزگۈر ئاپتوماتلاشتۇرۇشلارنى تەكشۈرۈش ئۈچۈن توختىتىدۇ
- **ئەسكەرتىش خاتىرىلىرى ۋە تەكرارنى يوقىتىش** — قايسى ئەسكەرتىشلەرنىڭ ئاللىقاچان ئەۋەتىلگەنلىكىنى ئىز قوغلاپ، تەكرار ئاپتوماتلاشتۇرۇش ئىجراسىنىڭ ئوخشاش قوبۇللىغۇچىغا ئىككى قېتىم ئۇقتۇرۇش ئەۋەتمەسلىكىگە كاپالەتلىك قىلىدۇ
- **TextBee SMS ئۇقتۇرۇشلىرى** — SMS تەمىنلىگۈچى قوزغىتىلغاندا TextBee ئارقىلىق سەپلەنگەن قىسقا ئۇچۇر ئۇقتۇرۇشلىرىنى ئەۋەتىدۇ
- **باشقۇرۇلىدىغان Superdav مۇلازىمەت تەمىنلەش** — قوللايدىغان تور بېكەتلەرنى قولدا endpoint سەپلەشنىڭ ئورنىغا ئاپتوماتىك تەڭشەش بىلەن ساھىبخانىلىق قىلىنغان Superdav endpoint لىرىغا ئۇلايدۇ
- **Advanced ھەمراھ بولاقى** — ھۆججەت سىستېمىسى، ساندان، WP-CLI، REST تارقاتقۇچ، plugin قۇرغۇچ، git snapshot، ئىشلەتكۈچى-باشقۇرۇش ۋە benchmark خىزمەت ئېقىملىرى ئۈچۈن ئايرىم تارقىتىلىدىغان ئىشەنچلىك-ئاچقۇچى قوراللىرى

## تەلەپلەر {#requirements}

- WordPress 6.0 ياكى ئۇنىڭدىن يۇقىرى
- PHP 8.1 ياكى ئۇنىڭدىن يۇقىرى
- WordPress Multisite (تور-ئاكتىپلانغان) ياكى يەككە-تور بېكەت ئورنىتىلىشى
- ئاكتىپ AI تەمىنلىگۈچى ئۇلىغۇچ plugin ى (Anthropic، OpenAI ياكى ماس كېلىدىغان)
- Gratis AI Agent v1.9.0 ياكى ئۇنىڭدىن يۇقىرى (Superdav AI Agent بولسا Gratis AI Agent نى كېڭەيتىدۇ)

## ئورنىتىش {#installation}

1. plugin ھۆججەتلىرىنى `/wp-content/plugins/superdav-ai-agent` غا يۈكلەڭ
2. WordPress ئىچىدىكى **Plugins** ئېكرانى ئارقىلىق plugin نى ئاكتىپلاڭ
3. Gratis AI Agent نىڭمۇ ئاكتىپلانغانلىقىغا كاپالەتلىك قىلىڭ
4. **Gratis AI Agent → Settings** قا يۆتكىلىپ، ئېھتىياجغا ئاساسەن ئىنتېرنېت ئىزدەش تەمىنلىگۈچىڭىزنى، باشقۇرۇلىدىغان Superdav مۇلازىمەت ئۇلىنىشىنى، Google Calendar ياكى TextBee كىنىشكىلىرىنى سەپلەڭ

WordPress.org بولاقى Superdav AI Agent نىڭ يادرو ئىقتىدارلىرىنى ئۆز ئىچىگە ئالىدۇ. Advanced ھەمراھ بولاقى ئىشەنچلىك ئاچقۇچى قوراللىرى ئېنىق ئورنىتىلغان ۋە ھوقۇق بېرىلگەن مۇھىتلار ئۈچۈن ئايرىم قۇرۇلۇپ تارقىتىلىدۇ.

## تېز باشلاش {#quick-start}

ئاكتىپلانغاندىن كېيىن، Superdav AI Agent تېما-خەۋەردار ماھارەتلەر ۋە قوشۇمچە ئىقتىدارلار بىلەن Gratis AI Agent ئورنىتىلىشىڭىزنى ئاپتوماتىك كۈچەيتىدۇ. WordPress باشقۇرغۇچ يان بالداقىدىكى **Gratis AI Agent** تاختىسىنى ئېچىپ، يېڭى ئىقتىدارلارنى ئىشلىتىشنى باشلاڭ.

**مىسال كۆرسەتمىلەر:**

```
ئالاقە بېتىگە ئالاقە جەدۋىلى قوشۇڭ.
```

```
ئەڭ يېڭى WordPress بىخەتەرلىك ئەڭ ياخشى ئەمەلىيەتلىرىنى توردا ئىزدەڭ.
```

```
ئاكتىپ تېمىنىڭ block ئەندىزىلىرىنى ئىشلىتىپ hero بۆلىكى قۇرۇڭ.
```

```
ئەتە ئۈچۈن سەپلەنگەن Google Calendar پائالىيەتلىرىنى كۆزدىن كەچۈرۈپ، قاتناشقۇچىلار ئۈچۈن ئەسكەرتىش لايىھەسىنى تەييارلاڭ.
```

```
مەن ئۇچۇرنى تەستىقلىغاندىن كېيىن SMS ئەسكەرتىش ئەۋەتىڭ.
```

## تېما-خەۋەردار ماھارەتلەر {#theme-aware-skills}

Superdav AI Agent ئاكتىپ تېمىڭىز ۋە قاچىلانغان plugin لىرىڭىزنى ئاپتوماتىك بايقاپ، مۇۋاپىق تېما-خەۋەردار ماھارەتنى ياردەمچىنىڭ مەزمۇنىغا يۈكلەيدۇ. ئىشلەتكىلى بولىدىغان ماھارەتلەر تۆۋەندىكىلەرنى ئۆز ئىچىگە ئالىدۇ:

| ماھارەت | قوللىنىلىدىغانلار |
|---|---|
| **Block Themes** | WordPress block ئاساسلىق (Full Site Editing) قۇرۇلمىسىنى ئىشلىتىدىغان تېمىلار |
| **Classic Themes** | PHP قېلىپلىرى ۋە كلاسسىك تەھرىرلىگۈچنى ئىشلىتىدىغان ئەنئەنىۋى WordPress تېمىلىرى |
| **Kadence Blocks** | Kadence Blocks plugin ى ئاكتىپ بولغان تور بېكەتلەر |
| **Kadence Theme** | Kadence تېمىسى ئاكتىپ تېما بولغان تور بېكەتلەر |

ھەر بىر ماھارەت توغرىسىدىكى تەپسىلىي يېتەكچىلىك ئۈچۈن [تېما-خەۋەردار ماھارەتلەر](../../user-guide/configuration/theme-aware-skills) نى كۆرۈڭ.

## ئىنتېرنېت ئىزدەش تەمىنلىگۈچىلەر {#internet-search-providers}

Superdav AI Agent v1.10.0 ئىنتېرنېت ئىزدەش سوئاللىرى ئۈچۈن Brave Search نىڭ ئورنىغا تاللاش سۈپىتىدە **Tavily** نى قوشىدۇ. ھەر ئىككى تەمىنلىگۈچى ھەقىقىي ۋاقىتتىكى ئىزدەش نەتىجىلىرىنى تەمىنلەيدۇ ۋە **Gratis AI Agent → Settings → Advanced** ئىچىدە سەپلىگىلى بولىدۇ.

| تەمىنلىگۈچى | ئىقتىدارلار | تەڭشەش |
|---|---|---|
| **Brave Search** | تېز، مەخپىيەتلىككە مەركەزلەشكەن ئىزدەش نەتىجىلىرى | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | تېخىمۇ مول ئىزدەش نەتىجىلىرى، ئىلغار تەتقىقات ئىقتىدارلىرى | [Tavily API](https://tavily.com/) |

تەڭشەش كۆرسەتمىلىرى ئۈچۈن [ئىنتېرنېت ئىزدەش](../../user-guide/configuration/internet-search) نى كۆرۈڭ.

## كالېندار، تەستىقلاش ۋە ئۇقتۇرۇش خىزمەت ئېقىملىرى {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 ئاپتوماتلاشتۇرۇش خىزمەت ئېقىملىرىنى مەزمۇن ۋە تور بېكەت قۇرۇش ۋەزىپىلىرىدىن ھالقىتىپ كېڭەيتىدۇ:

- **Google Calendar قوراللىرى** كالېندار كىرىش ئۇچۇرلىرى بار بولغاندا ۋاكالەتچىنىڭ تەڭشەلگەن كالېندارلار ۋە ھادىسە تەپسىلاتلىرىنى ئوقۇشىغا يول قويىدۇ.
- **ئالاقىداش ۋە قاتناشقۇچى ياردەمچىلىرى** ئەسكەرتىش ياكى كېيىنكى قەدەم ۋەزىپىلىرىنى تەييارلاشتىن بۇرۇن ھادىسە قاتناشقۇچىلىرىنى WordPress ئىشلەتكۈچىلىرى ياكى تونۇلغان ئالاقىداشلارغا ماسلاشتۇرىدۇ.
- **ئىنسان تەستىق دەرۋازىلىرى** تەكلىپ قىلىنغان ھەرىكەتنى ھوقۇق بېرىلگەن ئىشلەتكۈچى تەكشۈرگۈچە ئاپتوماتلاشتۇرۇشنى توختىتىپ تۇرۇشقا يول قويىدۇ.
- **ئەسكەرتىش خاتىرىلىرى** ئاپتوماتلاشتۇرۇش قايتا سىناق قىلىنغاندا ياكى قايتا-قايتا جەدۋەلگە قويۇلغاندا تەكرار كالېندار ياكى ئۇقتۇرۇش ئەسكەرتىشلىرىنىڭ ئالدىنى ئالىدۇ.
- **TextBee SMS قوللىشى** TextBee تەمىنلىگۈچى تەڭشەلگەندە قىسقا ئۇچۇر ئۇقتۇرۇشلىرىنى قوزغىتىدۇ.

بۇ قوراللارنى ۋاقىت جەدۋىلىدىن خەۋەردار ياردەم، قاتناشقۇچىلارغا كېيىنكى ئالاقە ۋە سەزگۈر ئۇچۇرلار ئەۋەتىلىشتىن بۇرۇن يەنىلا ئىنسان تەكشۈرۈشىنى تەلەپ قىلىدىغان ئەسكەرتىش خىزمەت ئېقىملىرى ئۈچۈن ئىشلىتىڭ.

## تور بېكەت قۇرغۇچى ئالاقە جەدۋەللىرى {#site-builder-contact-forms}

تور بېكەت قۇرغۇچى ۋاكالەتچىسى ھازىر پاراڭلىشىش كۆرۈنمە يۈزىدىن بىۋاسىتە ئالاقە جەدۋەللىرىنى قۇرالايدۇ. پەقەت تور بېكەت قۇرغۇچىدىن ئالاقە جەدۋىلى قۇرۇشنى سوراڭ، ئۇ ئورنىتىلغان جەدۋەل pluginىڭىزنى (Contact Form 7، WPForms، Fluent Forms ياكى Gravity Forms) ئاپتوماتىك بايقاپ، ئىشلىتىشكە تەييار جەدۋەل ھاسىل قىلىدۇ.

تەپسىلىي يېتەكچىلىك ئۈچۈن [تور بېكەت قۇرغۇچى ئالاقە جەدۋەللىرى](../../user-guide/administration/site-builder-contact-forms) نى كۆرۈڭ.

## Theme Builder ئىقتىدارلىرى {#theme-builder-features}

Superdav AI Agent v1.16.0 دىكى Theme Builder كۈچلۈك يېڭى ئىقتىدارلارنى ئۆز ئىچىگە ئالىدۇ:

### سۈرەت يۈكلەش بىلەن بايقاش سۆھبىتى {#discovery-interview-with-photo-upload}

Theme Builder بايقاش سۆھبىتى ھازىر كۆرۈنمە ئىلھام بېرىدىغان سۈرەت يۈكلەش قەدىمىنى ئۆز ئىچىگە ئالىدۇ. ماركىڭىز، رىقابەتچى تور بېكەتلەر ياكى لايىھە ئىلھامىغا ئائىت سۈرەتلەرنى يۈكلەپ، Theme Builderنىڭ لايىھە يۆنىلىشىڭىزنى چۈشىنىشىگە ياردەم بېرىڭ.

تەپسىلىي يېتەكچىلىك ئۈچۈن [بايقاش سۆھبىتى](./theme-builder/discovery-interview.md) نى كۆرۈڭ.

### ئۈستەل كومپيۇتېرى ۋە كۆچمە ئالدىن كۆرۈش بىلەن لايىھە يۆنىلىشى {#design-direction-with-desktop-and-mobile-previews}

Theme Builder تولۇق themeىڭىزنى ھاسىل قىلىشتىن بۇرۇن لايىھەڭىزنى ئۈستەل كومپيۇتېرى ۋە كۆچمە ئۈسكۈنىلەردە تەكشۈرۈڭ. كۆپ خىل ماسلىشىشچان بۆلەك نۇقتىلىرىدا ھەقىقىي ۋاقىتلىق ئالدىن كۆرۈشلەرنى كۆرۈڭ.

تەپسىلىي يېتەكچىلىك ئۈچۈن [لايىھە يۆنىلىشى](./theme-builder/design-direction.md) نى كۆرۈڭ.

### مېھمان كۈتۈش تىزىملىكلىرى {#hospitality-menus}

Theme Builder ھازىر ئاشخانىلار، قەھۋەخانىلار، قاۋاقخانىلار ۋە باشقا مېھمان كۈتۈش كارخانىلىرى ئۈچۈن قۇرۇلمىلىق يېمەك-ئىچمەك تىزىملىك بەتلىرىنى ھاسىل قىلالايدۇ. تىزىملىكلەر باھا، چۈشەندۈرۈش، يېمەك-ئىچمەك ئۇچۇرى ۋە ئاللېرگېن ئاگاھلاندۇرۇشلىرىنى ئۆز ئىچىگە ئالىدۇ.

تەپسىلىي يېتەكچىلىك ئۈچۈن [مېھمان كۈتۈش تىزىملىكلىرى](./theme-builder/hospitality-menus.md) نى كۆرۈڭ.

### يېڭى قابىلىيەتلەر {#new-abilities}

- **[Logo SVG ھاسىل قىلىش](./abilities/generate-logo-svg.md)** — ئاپتوماتىك تازىلاش بىلەن خاس SVG لوگولارنى قۇرۇش
- **[رەڭ تاختىسى سېلىشتۇرمىسىنى دەلىللەش](./abilities/validate-palette-contrast.md)** — رەڭ جۈپلىرىنىڭ WCAG ماسلىشىشچانلىقىنى تەكشۈرۈش
- **[تىزىملىك قۇرۇش](./abilities/create-menu.md)** — پەرقلىق يولباشلاش بەلگىلىرى بىلەن يولباشلاش تىزىملىكلىرىنى قۇرۇش

ئىشلەتكىلى بولىدىغان قوراللار ۋە Tier 1 تور بېكەت قىرىش قورالى توغرىسىدىكى ئۇچۇر ئۈچۈن [قوراللار ۋە تەڭشەش](./theme-builder/tools.md) نى كۆرۈڭ.

## تەڭشەش {#configuration}

بارلىق تەڭشەكلەر WordPress باشقۇرۇشىدىكى **Gratis AI Agent → Settings** ئاستىدا. Superdav AI Agent تۆۋەندىكى تەڭشەش تاللانمىلىرىنى قوشىدۇ:

| تەڭشەك | چۈشەندۈرۈش |
|---|---|
| **Internet Search Provider** | Brave Search بىلەن Tavily ئارىسىدىن تاللاڭ (ئىختىيارىي) |
| **Tavily API Key** | Tavily ئىنتېرنېت ئىزدەش ئۈچۈن API ئاچقۇچى (Tavily تاللانسا) |
| **Managed Superdav Service** | قوللىنىدىغان تور بېكەتلەرنى ئاپتوماتىك endpoint تەمىنلەش بىلەن ساھىبخانىلىق قىلىنغان Superdav مۇلازىمىتىگە ئۇلاڭ |
| **Google Calendar Credentials** | ۋاكالەتچى كالېندارلار ۋە ھادىسىلەرنى ئوقۇيالىشى ئۈچۈن كالېندار زىيارىتىنى تەڭشەڭ |
| **TextBee SMS Provider** | تەستىقلانغان SMS ئۇقتۇرۇش خىزمەت ئېقىملىرى ئۈچۈن TextBee كىرىش ئۇچۇرلىرىنى تەڭشەڭ |

## ئىلغار ھەمراھ بوغچا {#advanced-companion-package}

Superdav AI Agent v1.18.0 ئىشەنچلىك ئاچقۇچى خىزمەت ئېقىملىرى ئۈچۈن ئايرىم تارقىتىلىدىغان ئىلغار ھەمراھ بوغچىنى قوشىدۇ. بۇ ئىقتىدارلار WordPress.org يادرو بوغچىسىغا بىرلەشتۈرۈلمىگەن. ھەمراھ بوغچىنى پەقەت باشقۇرغۇچىلار ۋاكالەتچىنىڭ ئاچقۇچى قوراللىرىغا كىرىشىگە ئېنىق ئىشەنگەن تور بېكەتلەرگىلا ئورنىتىڭ.

ئىلغار قوراللار ھۆججەت سىستېمىسى مەشغۇلاتلىرى، ساندان زىيارىتى، WP-CLI، REST تارقاتقۇچ خىزمەت ئېقىملىرى، plugin قۇرغۇچى قوراللىرى، git سۈرەتكە ئېلىشلىرى، ئىشلەتكۈچى باشقۇرۇش ياردەمچىلىرى ۋە benchmark قوراللىرىنى ئۆز ئىچىگە ئالىدۇ. بۇ قوراللارنى ئىشەنچلىك باشقۇرغۇچىلار بىلەنلا چەكلىڭ ۋە ئىشلەپچىقىرىش تور بېكەتلىرىدە قوزغىتىشتىن بۇرۇن تەستىق دەرۋازىسى تەڭشەكلىرىنى تەكشۈرۈڭ.

## v1.18.0 دىكى ياخشىلىنىشلار {#improvements-in-v1180}

- **Google Calendar قوللىشى** — ۋاقىت جەدۋىلىدىن خەۋەردار ئاپتوماتلاشتۇرۇشلار ئۈچۈن تەڭشەلگەن كالېندارلار ۋە ھادىسىلەرنى ئوقۇش
- **ئالاقىداش ۋە قاتناشقۇچى ماسلاشتۇرۇش** — كالېندار قاتناشقۇچىلىرىنى تور بېكەت ئىشلەتكۈچىلىرى ۋە ئالاقىداشلار بىلەن ماسلاشتۇرۇش
- **ئىنسان تەستىق دەرۋازىلىرى** — داۋاملاشتۇرۇشتىن بۇرۇن سەزگۈر ئاپتوماتلاشتۇرۇشلارنى تەكشۈرۈش ئۈچۈن توختىتىپ تۇرۇش
- **ئەسكەرتىش تەكرارلىقىنى يوقىتىش** — قايتا سىناقلار ياكى تەكرار ئىجرا قىلىنىشلاردا تەكرار ئۇقتۇرۇشلارنىڭ ئالدىنى ئېلىش ئۈچۈن ئەسكەرتىشلەرنى خاتىرىلەش
- **TextBee SMS تەمىنلىگۈچىسى** — زۆرۈر تەستىقلاردىن كېيىن تەڭشەلگەن قىسقا ئۇچۇر ئۇقتۇرۇشلىرىنى ئەۋەتىش
- **باشقۇرۇلىدىغان Superdav مۇلازىمىتىنى تەڭشەش** — قوللىنىدىغان تور بېكەتلەر ئۈچۈن ساھىبخانىلىق قىلىنغان مۇلازىمەت endpointلىرىنى ئاپتوماتىك تەمىنلەش
- **ئىلغار ھەمراھ تارقىتىلىشى** — ئىشەنچلىك ئاچقۇچى قوراللىرىنى WordPress.org يادرو قۇرۇلمىسىدىن ئايرىم ئېلان قىلىش
- **ئىشەنچلىكلىك تۈزىتىشلىرى** — model تاللاش، تەلەپ ۋاقتى ئېشىپ كېتىش، كۆڭۈلدىكى setup model بىر تەرەپ قىلىش، reasoning-text بىر تەرەپ قىلىش، قايتا سىناش يېتەكچىلىكى، كالېندار token مۇستەھكەملەش، ئەسكەرتىش مۇستەھكەملەش، onboarding قوزغىتىشلىرى، جەزملەشتۈرۈلگەن قابىلىيەت داۋاملاشتۇرۇشلىرى ۋە WordPress.org بوغچىلىرىنى تەكشۈرۈش مەسىلىلىرىنى ياخشىلاش

## v1.16.0 دىكى ياخشىلىنىشلار {#improvements-in-v1160}

- **AI خېرىدار ساقلىغۇچ** — ھازىر تەلەپلەر ئارىسىدا داۋاملىق ساقلىنىش ئۈچۈن transients ئارقىلىق قوللىنىدۇ، ئۇزۇن ۋاقىت ئىجرا بولىدىغان agent ۋەزىپىلىرىدە سانلىق مەلۇمات يوقاپ كېتىشىنىڭ ئالدىنى ئالىدۇ
- **1-دەرىجىلىك تور بېكەت قىرىپ-ئېلىش قورالى** — sd-ai-agent/site-scrape ھازىر رىقابەتچى ۋە ئىلھام تور بېكەتلىرىنى تەھلىل قىلىش ئۈچۈن Theme Builder دا سۈكۈتتىكى ھالەتتە ئىشلىتىلىدۇ

## v1.10.0 دىكى ياخشىلىنىشلار {#improvements-in-v1100}

- **WooCommerce بىرلەشتۈرۈشى قايتا قۇرۇلدى** — ھازىر تېخىمۇ ياخشى ئىشەنچلىكلىك ۋە ماسلىشىشچانلىق ئۈچۈن يەرلىك WooCommerce APIs نى ئىشلىتىدۇ
- **تەمىنلىگۈچى تىزىملىكىنى ئاپتوماتىك يېڭىلاش** — قىستۇرمىلارنىڭ قوزغىتىلغان ياكى توختىتىلغانلىقىنى ئاپتوماتىك بايقاپ، ئىشلىتىشكە بولىدىغان تەمىنلىگۈچى تىزىملىكىنى يېڭىلايدۇ
- **كۈچەيتىلگەن تور بېكەت قۇرغۇچ** — تېخىمۇ ياخشى خاتالىقتىن ئەسلىگە كېلىش ئىقتىدارى بىلەن كۆپ باسقۇچلۇق تور بېكەت قۇرۇش پىلانلىرى ياخشىلاندى
- **ئۆرنەككە ماسلىشىدىغان ماھارەتلەر** — ئاكتىپ ئۆرنىكىڭىزگە ئاپتوماتىك ماسلىشىدىغان تۆت يېڭى ئىچىگە ئورنىتىلغان ماھارەت
