# مېھماندارچىلىق تىزىملىكلىرى

**مېھماندارچىلىق تىزىملىكلىرى** ئىقتىدارى Theme Builder غا قۇرۇلمىلىق يېمەكلىك ۋە ئىچىملىك تىزىملىك بەتلىرىنى بىۋاسىتە WordPress تور بېكىتىڭىزگە ھاسىل قىلىپ قىستۇرۇش ئىمكانىنى بېرىدۇ.

## ئومۇمىي چۈشەنچە

Theme Builder ھازىر رېستوران، قەھۋەخانا، بار ۋە يېمەكلىك تەمىنلەش مۇلازىمىتى قاتارلىق مېھماندارچىلىق كارخانىلىرى ئۈچۈن كەسپىي، قۇرۇلمىلىق تىزىملىك بەتلىرىنى قۇرالايدۇ. بۇ تىزىملىكلەر تور بېكىتىڭىزنىڭ لايىھىسىگە تولۇق بىرلەشتۈرۈلگەن بولۇپ، ئاسان يېڭىلىغىلى ۋە باشقۇرغىلى بولىدۇ.

## قوللايدىغان مېھماندارچىلىق تۈرلىرى

- **رېستورانلار** — تولۇق مۇلازىمەتلىك تاماق تىزىملىكلىرى
- **قەھۋەخانىلار** — قەھۋە ۋە يېنىك يېمەكلىك تىزىملىكلىرى
- **بارلار ۋە ئارامخانىلار** — ئىچىملىك ۋە ئىشتىھا ئېچىش تىزىملىكلىرى
- **ناۋايخانىلار** — پېچىنە-پىرەنىك ۋە نان تىزىملىكلىرى
- **يېمەكلىك تەمىنلەش مۇلازىمەتلىرى** — پائالىيەت تىزىملىك تاللاشلىرى
- **كۆچمە يېمەكلىك ماشىنىلىرى** — كۆچمە يېمەكلىك مۇلازىمىتى تىزىملىكلىرى
- **پىۋا زاۋۇتلىرى ۋە شارابخانىلار** — چۈشەندۈرۈشلۈك ئىچىملىك تىزىملىكلىرى

## تىزىملىك قۇرۇلمىسى

### تىزىملىك كاتېگورىيەلىرى

تىزىملىكلەر كاتېگورىيەلەر بويىچە تەشكىللىنىدۇ:

- **ئىشتىھا ئېچىشلار** — باشلىنىش تاماقلىرى ۋە كىچىك تەخسىلەر
- **ئاساسىي تاماقلار** — ئاساسىي تائاملار
- **قوشۇمچىلار** — قوشۇمچە يېمەكلىكلەر ۋە كۆكتاتلار
- **تاتلىقلار** — تاتلىق يېمەكلىكلەر
- **ئىچىملىكلەر** — ئىچىملىكلەر (ئىسپىرتلىق ۋە ئىسپىرتسىز)
- **ئالاھىدە تۈرلەر** — كۈندىلىك ياكى پەسىللىك ئالاھىدە تۈرلەر

### تىزىملىك تۈرى فورماتى

ھەر بىر تىزىملىك تۈرى تۆۋەندىكىلەرنى ئۆز ئىچىگە ئالىدۇ:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### تۈر سەھەلىرى

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | تاماق ياكى ئىچىملىك ئىسمى |
| `description` | string | تۈرنىڭ تەپسىلىي چۈشەندۈرۈشى |
| `price` | string | باھا (پۇل بىرلىكى بىلەن فورماتلانغان) |
| `dietary_info` | array | يېمەك-ئىچمەك خاسلىقلىرى (Vegan، Gluten-Free قاتارلىقلار) |
| `allergens` | array | كۆپ ئۇچرايدىغان ئاللېرگېنلار (ياڭاقلار، قۇلۇلە دېڭىز مەھسۇلاتلىرى قاتارلىقلار) |
| `availability` | string | قاچان تەمىنلىنىدىغانلىقى (كۈندىلىك، پەسىللىك قاتارلىقلار) |

## مېھماندارچىلىق تىزىملىكلىرىنى قۇرۇش

### 1-قەدەم: تىزىملىك ئۇچۇرىنى تەمىنلەڭ

Theme Builder غا تىزىملىكىڭىز ھەققىدە سۆزلەپ بېرىڭ:

```
ئىتالىيە رېستورانىم ئۈچۈن ئىشتىھا ئېچىشلار،
ماكارون تاماقلىرى، ئاساسىي تاماقلار ۋە تاتلىقلار بار رېستوران تىزىملىكى قۇرۇڭ. باھا ۋە چۈشەندۈرۈشلەرنى قوشۇڭ.
```

### 2-قەدەم: Theme Builder تىزىملىكنى ھاسىل قىلىدۇ

Theme Builder:

1. قۇرۇلمىلىق تىزىملىك بېتى قۇرىدۇ
2. ئۇنى تور بېكىتىڭىزنىڭ themeىغا ماس كېلىدىغان قىلىپ لايىھىلەيدۇ
3. تۈرلەرنى كاتېگورىيەلەرگە تەشكىللەيدۇ
4. باھا ۋە چۈشەندۈرۈشلەرنى فورماتلايدۇ
5. يېمەك-ئىچمەك ۋە ئاللېرگېن ئۇچۇرلىرىنى قوشىدۇ

### 3-قەدەم: كۆزدىن كەچۈرۈش ۋە خاسلاشتۇرۇش

سىز قىلالايسىز:

1. تىزىملىك تۈرلىرى ۋە باھالارنى تەھرىرلەش
2. كاتېگورىيەلەرنى قوشۇش ياكى چىقىرىۋېتىش
3. كاتېگورىيە ئىچىدىكى تۈرلەرنىڭ تەرتىپىنى قايتا رەتلەش
4. چۈشەندۈرۈشلەر ۋە يېمەك-ئىچمەك ئۇچۇرلىرىنى يېڭىلاش
5. ئۇسلۇب ۋە ئورۇنلاشتۇرۇشنى تەڭشەش

## تىزىملىك كۆرسىتىش تاللاشلىرى

### تولۇق تىزىملىك بېتى

تولۇق تىزىملىكىڭىزنى كۆرسىتىدىغان مەخسۇس بەت:

- كاتېگورىيە بويىچە تەشكىللەنگەن
- ئىزدەشكە ۋە سۈزۈشكە بولىدىغان
- بېسىپ چىقىرىشقا ماس ئورۇنلاشتۇرۇش
- كۆچمە ئۈسكۈنىلەرگە ماس لايىھە

### تىزىملىك Widgetى

تىزىملىك بۆلەكلىرىنى باشقا بەتلەرگە قىستۇرۇڭ:

- باش بەتتىكى تاللانغان تۈرلەر
- يانبالداقتىكى كۈندىلىك ئالاھىدە تۈرلەر
- بار بېتىدىكى ئىچىملىك تىزىملىكى
- ئاستى بۆلەكتىكى تاتلىق كۆرسەتمىسى

### تىزىملىك PDFى

چۈشۈرگىلى بولىدىغان PDF تىزىملىك ھاسىل قىلىڭ:

- كەسپىي فورماتلاش
- بېسىپ چىقىرىشقا تەييار سۈپەت
- رەسىملەر ۋە چۈشەندۈرۈشلەرنى ئۆز ئىچىگە ئالىدۇ
- ھەمبەھىرلەش ۋە ئېلخەت بىلەن ئەۋەتىش ئاسان

## يېمەك-ئىچمەك ۋە ئاللېرگېن ئۇچۇرى

### يېمەك-ئىچمەك خاسلىقلىرى

تۈرلەرگە يېمەك-ئىچمەك ئۇچۇرى بەلگىلەڭ:

- **Vegan** — ھايۋان مەھسۇلاتلىرى يوق
- **Vegetarian** — گۆش يوق
- **Gluten-Free** — سېلىياك كېسىلى بارلار ئۈچۈن بىخەتەر
- **Dairy-Free** — سۈت مەھسۇلاتلىرى يوق
- **Nut-Free** — دەرەخ ياڭاقلىرى ياكى يەر ياڭىقى يوق
- **Low-Carb** — كاربون سۇ بىرىكمىلىرى ئازايتىلغان
- **High-Protein** — ئاقسىلغا باي

### ئاللېرگېن ئاگاھلاندۇرۇشلىرى

كۆپ ئۇچرايدىغان ئاللېرگېنلارنى قوشۇڭ:

- **ياڭاقلار** — دەرەخ ياڭاقلىرى ۋە يەر ياڭىقى
- **قۇلۇلە دېڭىز مەھسۇلاتلىرى** — قىسقۇچپاقا تۈرىدىكىلەر ۋە قۇلۇلە تۈرىدىكىلەر
- **بېلىق** — بارلىق بېلىق تۈرلىرى
- **سۈت مەھسۇلاتلىرى** — سۈت ۋە سۈت مەھسۇلاتلىرى
- **تۇخۇم** — توخۇ تۇخۇمى
- **پۇرچاق** — پۇرچاق مەھسۇلاتلىرى
- **Gluten** — بۇغداي ۋە Gluten بار دانلار
- **كۈنجۈت** — كۈنجۈت ئۇرۇقى ۋە مېيى

## تىزىملىك باشقۇرۇش

### باھالارنى يېڭىلاش

تىزىملىك باھالارنى ئاسان يېڭىلاڭ:

1. تىزىملىك بېتىگە يۆتكىلىڭ
2. "تىزىملىكنى تەھرىرلەش" نى چېكىڭ
3. تۈرلەرنىڭ باھاسىنى يېڭىلاڭ
4. ئۆزگەرتىشلەرنى ساقلاڭ
5. ئۆزگەرتىشلەر تور بېكىتىڭىزدە دەرھال كۆرۈنىدۇ

### پەسىللىك تۈرلەرنى قوشۇش

پەسىللىك تىزىملىك ۋارىيانتى قۇرۇڭ:

1. يېڭى تىزىملىك نەشرى قۇرۇڭ
2. پەسىللىك تۈرلەرنى قوشۇڭ
3. تۈرلەرنى "پەسىللىك" دەپ بەلگىلەڭ
4. تەمىنلىنىش ۋاقىتلىرىنى جەدۋەللەڭ
5. پەسىل مەزگىلىدە ئاپتوماتىك كۆرسىتىڭ

### ئالاھىدە تۈرلەرنى باشقۇرۇش

كۈندىلىك ياكى ھەپتىلىك ئالاھىدە تۈرلەرنى كۆرسىتىڭ:

1. "ئالاھىدە تۈرلەر" كاتېگورىيەسى قۇرۇڭ
2. تەمىنلىنىش ۋاقىتلىرى بار تۈرلەرنى قوشۇڭ
3. ئالاھىدە تۈرلەرنى باش بەتتە گەۋدىلەندۈرۈڭ
4. كۈندىلىك ياكى ھەپتىلىك يېڭىلاڭ
5. كونا ئالاھىدە تۈرلەرنى ئارخىپلاڭ

## Theme Builder بىلەن بىرلەشتۈرۈش

مېھماندارچىلىق تور بېكەتلىرى ئۈچۈن Theme Builder ئىشلەتكەندە:

1. **ئاپتوماتىك تىزىملىك بايقاش** — سىزنىڭ يېمەكلىك/ئىچىملىك كارخانىسى ئىكەنلىكىڭىزنى پەرقلەندۈرىدۇ
2. **تىزىملىك بېتى قۇرۇش** — كەسپىي تىزىملىك بەتلىرىنى ھاسىل قىلىدۇ
3. **لايىھە ماسلاشتۇرۇش** — تىزىملىكلەر تور بېكىتىڭىزنىڭ themeىغا ماس كېلىدۇ
4. **كۆچمە ئۈسكۈنىگە ئەلالاشتۇرۇش** — تىزىملىكلەر تېلېفونلاردا چىرايلىق كۆرۈنىدۇ
5. **SEO ئەلالاشتۇرۇش** — تىزىملىكلەر ئىزدەش ماتورلىرىغا ماس

## ئەڭ ياخشى ئەمەلىيەتلەر

### تىزىملىك لايىھىسى

- **ئېنىق تەشكىللەش** — لوگىكىلىق كاتېگورىيە قۇرۇلمىسى
- **ئوقۇشقا قولاي چۈشەندۈرۈشلەر** — ئىشتىھا ئاچقۇچى ۋە ئۇچۇرلۇق
- **بىردەك باھا كۆرسىتىش** — ئېنىق پۇل بىرلىكى ۋە فورماتلاش
- **كەسپىي سۈرەتلەر** — يۇقىرى سۈپەتلىك يېمەكلىك رەسىملىرى
- **بوش ئورۇن** — بەتنى بەك قىستىۋەتمەڭ

### مەزمۇن

- **توغرا چۈشەندۈرۈشلەر** — تۈرلەرنى توغرا تەسۋىرلەڭ
- **ئالاھىدە تۈرلەرنى گەۋدىلەندۈرۈش** — ئالاھىدە تۈرلەرنى كۆزگە كۆرۈنەرلىك قىلىڭ
- **ئاللېرگېنلارنى قوشۇش** — ئاللېرگېنلارنى ھەمىشە تىزىڭ
- **دائىم يېڭىلاش** — باھا ۋە تۈرلەرنى نۆۋەتتىكى ھالەتتە ساقلاڭ
- **ئىشتىھا ئاچقۇچى تىل ئىشلىتىش** — تۈرلەرنى مەززىلىك ئاڭلىتىڭ

### قولايلىق

- **ئوقۇشقا قولاي خەت نۇسخىلىرى** — ئېنىق، ئوقۇشقا ئاسان خەت شەكىللىرىنى ئىشلىتىڭ
- **يېتەرلىك سېلىشتۇرما** — تېكىستنىڭ ئوقۇلۇشىغا كاپالەتلىك قىلىڭ
- **يېمەك-ئىچمەك بەلگىلىرى** — يېمەك-ئىچمەك تاللاشلىرىنى ئېنىق بەلگىلەڭ
- **ئاللېرگېن ئاگاھلاندۇرۇشلىرى** — ئاللېرگېنلارنى كۆزگە كۆرۈنەرلىك كۆرسىتىڭ
- **كۆچمە ئۈسكۈنىگە ماس** — بارلىق ئۈسكۈنىلەردە سىناڭ
