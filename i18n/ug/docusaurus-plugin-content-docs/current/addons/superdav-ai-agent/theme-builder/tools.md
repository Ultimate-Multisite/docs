# قوراللار ۋە سەپلىمە

Theme Builder سىزنىڭ WordPress تور بېكىتىڭىزنى تەھلىل قىلىش، لايىھەلەش ۋە قۇرۇش ئۈچۈن بىر يۈرۈش قوراللارنى ئىشلىتىدۇ. 1.16.0 نەشرىدە، **sd-ai-agent/site-scrape ھازىر Tier 1 قورالى** بولۇپ، Theme Builder دا سۈكۈتتىكى ھالەتتە ئىشلىتىشكە بولىدۇ. Superdav AI Agent v1.18.0 ئىشەنچلىك ئىجادىيەتچى خىزمەت ئېقىملىرى ئۈچۈن ئايرىم تارقىتىلىدىغان Advanced ھەمراھ بولىقىنىمۇ تونۇشتۇرىدۇ؛ بۇ قوراللار WordPress.org يادرو بولىقىنىڭ بىر قىسمى ئەمەس، ئايرىم ئورنىتىلىشى ۋە ھوقۇق بېرىلىشى كېرەك.

## ئىشلىتىشكە بولىدىغان قوراللار

### Tier 1 قوراللار (ھەمىشە ئىشلىتىشكە بولىدۇ)

Tier 1 قوراللار ھېچقانداق قوشۇمچە سەپلىمەسىز Theme Builder دا سۈكۈتتىكى ھالەتتە ئىشلىتىشكە بولىدۇ.

#### sd-ai-agent/site-scrape

**مەقسىتى:** مەۋجۇت تور بېكەتلەرنى تەھلىل قىلىپ، لايىھە ئىلھامى، مەزمۇن قۇرۇلمىسى ۋە ئىقتىدارنى چىقىرىۋېلىش.

**ئىقتىدارلىرى:**

- **تور بېكەت تەھلىلى** — رىقابەتچى ياكى ئىلھام بېرىدىغان تور بېكەتلەرنى يىغىش ۋە تەھلىل قىلىش
- **لايىھە چىقىرىۋېلىش** — رەڭلەر، خەت نۇسخىلىرى ۋە ئورۇنلاشتۇرۇش ئەندىزىلىرىنى بايقاش
- **مەزمۇن قۇرۇلمىسى** — بەت تەشكىللىنىشى ۋە قاتلامىنى چۈشىنىش
- **ئىقتىدار بايقاش** — plugin لار ۋە ئىقتىدارلارنى بايقاش
- **ئىقتىدار ئۈنۈمى تەھلىلى** — بەت سۈرئىتى ۋە ئەلالاشتۇرۇشنى تەكشۈرۈش
- **SEO تەھلىلى** — meta بەلگىلىرى ۋە قۇرۇلمىلىق سانلىق مەلۇماتلارنى كۆزدىن كەچۈرۈش

**ئىشلىتىلىشى:**

```
example.com نىڭ لايىھەسىنى تەھلىل قىلىپ، ئۇنى مېنىڭ تور بېكىتىم ئۈچۈن ئىلھام سۈپىتىدە ئىشلىتىڭ.
```

**ئۇ چىقىرىۋالىدىغانلار:**

- رەڭ پالېتىسى ۋە رەڭ ئىشلىتىلىشى
- تىپوگرافىيە (خەت نۇسخىلىرى ۋە چوڭلۇقى)
- ئورۇنلاشتۇرۇش قۇرۇلمىسى ۋە ئارىلىق
- يول باشلاش ئەندىزىلىرى
- مەزمۇن تەشكىللىنىشى
- رەسىملەر ۋە مېدىيا ئىشلىتىلىشى
- ئۆزئارا تەسىرلىك ئېلېمېنتلار
- كۆچمە ماسلىشىشچانلىق

**چەكلىمىلەر:**

- پارول بىلەن قوغدالغان تور بېكەتلەرنى يىغالمايدۇ
- robots.txt ۋە تور بېكەت سىياسەتلىرىگە ھۆرمەت قىلىدۇ
- ھەرىكەتچان مەزمۇننى تۇتۇۋالماسلىقى مۇمكىن
- JavaScript كۆپ بولغان تور بېكەتلەردە تەھلىل چەكلىك بولۇشى مۇمكىن
- چوڭ تور بېكەتلەرنى تەھلىل قىلىشقا ئۇزۇنراق ۋاقىت كېتىشى مۇمكىن

### Tier 2 قوراللار (ئىختىيارىي)

Tier 2 قوراللار Theme Builder تەڭشەكلىرىدە ئېنىق قوزغىتىلغاندا ئىشلىتىشكە بولىدۇ.

#### Advanced Analytics

تەپسىلىي ئىقتىدار ئۈنۈمى ئۆلچەملىرىنى تەمىنلەيدۇ:

- بەت يۈكلىنىش ۋاقتى
- Core Web Vitals
- SEO نومۇرى
- زىيارەتچانلىق نومۇرى
- ئەڭ ياخشى ئەمەلىيەتلەر نومۇرى

#### مەزمۇن ئەلالاشتۇرغۇچ

تۆۋەندىكىلەرنى تەھلىل قىلىپ ياخشىلاش تەكلىپلىرىنى بېرىدۇ:

- ئوقۇشچانلىق
- SEO ئەلالاشتۇرۇش
- ئاچقۇچلۇق سۆز ئىشلىتىلىشى
- مەزمۇن قۇرۇلمىسى
- ھەرىكەتكە چاقىرىش ئورۇنلاشتۇرۇشى

### Tier 3 قوراللار (Premium)

Tier 3 قوراللار قوشۇمچە سەپلىمە ياكى premium ئىقتىدارلارنى تەلەپ قىلىدۇ.

#### AI مەزمۇن ھاسىللىغۇچ

تۆۋەندىكىلەر ئۈچۈن مەزمۇن ھاسىل قىلىدۇ:

- مەھسۇلات چۈشەندۈرۈشلىرى
- مۇلازىمەت بەتلىرى
- بلوگ يازمىلىرى
- Meta چۈشەندۈرۈشلىرى
- ھەرىكەتكە چاقىرىش نۇسخىسى

#### Advanced لايىھە قوراللىرى

Advanced لايىھە ئىقتىدارلىرىنى تەمىنلەيدۇ:

- خاس CSS ھاسىل قىلىش
- ئانىماتسىيە يارىتىش
- ئۆزئارا تەسىرلىك ئېلېمېنت لايىھەسى
- Advanced رەڭ نەزەرىيەسى
- تىپوگرافىيە ئەلالاشتۇرۇش

### Advanced ھەمراھ قوراللىرى (ئايرىم بولاق)

Advanced ھەمراھ بولىقى يادرو Superdav AI Agent تارقىتىلمىسىدىن ئايرىم تارقىتىلىدۇ. ئۇ باشقۇرغۇچىلار agent نىڭ يۇقىرى تەسىرلىك قوراللارنى ئىشلىتىشىگە ئېنىق رۇخسەت قىلىدىغان ئىشەنچلىك ئىجادىيەتچى ۋە تور بېكەت ئىگىسى مۇھىتلىرى ئۈچۈن نىشانلانغان.

Advanced ھەمراھ ئىقتىدارلىرى تۆۋەندىكىلەرنى ئۆز ئىچىگە ئېلىشى مۇمكىن:

- **ھۆججەت سىستېمىسى قوراللىرى** — ئىشەنچلىك مۇھىتلاردا تەستىقلانغان ھۆججەتلەرنى تەكشۈرۈش ۋە ئۆزگەرتىش
- **ساندان قوراللىرى** — ئېنىق ھوقۇق بېرىلگەندە تور بېكەت سانلىق مەلۇماتلىرىنى سۈرۈشتۈرۈش ياكى يېڭىلاش
- **WP-CLI قوراللىرى** — WordPress ئاسراش ۋە تەكشۈرۈش بۇيرۇقلىرىنى ئىجرا قىلىش
- **REST تارقاتقۇچ قوراللىرى** — كونترول قىلىنغان تارقىتىش خىزمەت ئېقىملىرى ئارقىلىق تىزىملاتقان REST endpoint لارنى چاقىرىش
- **Plugin قۇرغۇچ قوراللىرى** — plugin كودىنى قېلىپلاشتۇرۇش ۋە تەكرار ياخشىلاش
- **Git snapshot قوراللىرى** — خەتەرلىك ئىجادىيەتچى مەشغۇلاتلىرىدىن بۇرۇن snapshot لارنى يارىتىش
- **ئىشلەتكۈچى باشقۇرۇش قوراللىرى** — رۇخسەت قىلىنغان جايلاردا ئىشلەتكۈچى خاتىرىلىرىنى كۆزدىن كەچۈرۈش ياكى تەڭشەشكە ياردەم بېرىش
- **Benchmark قوراللىرى** — ئىجادىيەت تەكشۈرۈشى ئۈچۈن ئىقتىدار ئۈنۈمى ياكى قابىلىيەت benchmark لىرىنى ئىجرا قىلىش

بۇلارنى ھەر بىر ئورنىتىش ئۈچۈن ئومۇميۈزلۈك ئىشلىتىلىدىغان Theme Builder قوراللىرى سۈپىتىدە ھۆججەتلىمەڭ. ئۇلار پەقەت Advanced ھەمراھ بولىقى ئورنىتىلغان، قوزغىتىلغان ۋە ئىشەنچلىك باشقۇرغۇچىلارغا چەكلەنگەن چاغدىلا ئىشلىتىشكە بولىدۇ.

## قورال سەپلىمىسى

### قوراللارنى قوزغىتىش

Theme Builder دا قوشۇمچە قوراللارنى قوزغىتىش ئۈچۈن:

1. **Gratis AI Agent → Settings** غا يۆتكىلىڭ
2. **Theme Builder → Tools** غا كىرىڭ
3. ئېھتىياجغا ئاساسەن قوراللارنى ئېچىڭ/تاقاڭ
4. تەڭشەكلەرنى ساقلاڭ

### قورال ئىجازەتلىرى

بەزى قوراللار ئىجازەت تەلەپ قىلىدۇ:

- **تور بېكەت يىغىش** — ئىنتېرنېت زىيارىتىنى تەلەپ قىلىدۇ
- **Analytics** — Google Analytics بىرلەشتۈرۈشىنى تەلەپ قىلىدۇ
- **مەزمۇن ھاسىل قىلىش** — API ئاچقۇچلىرىنى تەلەپ قىلىدۇ
- **Advanced ئىقتىدارلار** — premium ئەزالىق پىلانىنى تەلەپ قىلىشى مۇمكىن
- **Advanced ھەمراھ قوراللىرى** — ئايرىم تارقىتىلغان Advanced بولىقى ۋە ئېنىق باشقۇرغۇچى ئىشەنچىنى تەلەپ قىلىدۇ

### API ئاچقۇچلىرى ۋە كىنىشكىلىرى

ئۇلارنى تەلەپ قىلىدىغان قوراللار ئۈچۈن API ئاچقۇچلىرىنى سەپلەڭ:

1. **Gratis AI Agent → Settings → API Keys** غا كىرىڭ
2. ھەر بىر قورال ئۈچۈن كىنىشكىلەرنى كىرگۈزۈڭ
3. باغلىنىشنى سىناڭ
4. بىخەتەر ساقلاڭ

### Advanced ھەمراھ بولىقىنى ئورنىتىش

Advanced ھەمراھ بولىقى WordPress.org يادرو ZIP دىن ئايرىم تارقىتىلىدۇ. ئۇنى پەقەت شۇ تارقىتىلمىنىڭ رەسمىي تۈر تارقىتىش قانىلىدىن ئورنىتىڭ، ئاندىن زىيارەتنى ئىشەنچلىك باشقۇرغۇچىلارغا چەكلەڭ. ئىشلەپچىقىرىش تور بېكىتىدە ھۆججەت سىستېمىسى، ساندان، WP-CLI، REST تارقاتقۇچ، plugin قۇرغۇچ، ئىشلەتكۈچى باشقۇرۇش ياكى benchmark قوراللىرىنى قوزغىتىشتىن بۇرۇن، ئادەم تەستىق دەرۋازىلىرى ۋە git snapshot ھەرىكىتىنى كۆزدىن كەچۈرۈڭ.

## sd-ai-agent/site-scrape نى ئىشلىتىش

### ئاساسىي ئىشلىتىش

Theme Builder دىن بىر تور بېكەتنى تەھلىل قىلىشنى سوراڭ:

```
competitor.com دىكى رىقابەتچىمنىڭ تور بېكىتىنىڭ لايىھەسىنى تەھلىل قىلىپ،
مېنىڭ تور بېكىتىم ئۈچۈن لايىھە ياخشىلاش تەكلىپلىرىنى بېرىڭ.
```

### كونكرېت تەھلىل

كونكرېت تۈردىكى تەھلىللەرنى تەلەپ قىلىڭ:

```
example.com دىن رەڭ پالېتىسىنى چىقىرىۋېلىپ، ئۇنى ئىلھام سۈپىتىدە ئىشلىتىڭ.
```

```
example.com نىڭ يول باشلاش قۇرۇلمىسىنى تەھلىل قىلىپ، ئوخشاش
تەشكىللىنىشنى مېنىڭ تور بېكىتىمگە قوللىنىڭ.
```

```
example.com نىڭ كۆچمە ماسلىشىشچانلىقىنى تەكشۈرۈپ، مېنىڭ تور بېكىتىمنىڭ
ئوخشاش دەرىجىدە ماسلىشىشچان بولۇشىغا كاپالەتلىك قىلىڭ.
```

### سېلىشتۇرما تەھلىل

كۆپ تور بېكەتنى سېلىشتۇرۇڭ:

```
site1.com ۋە site2.com نىڭ لايىھەلىرىنى سېلىشتۇرۇپ، ھەر ئىككىسىنىڭ ئەڭ ياخشى ئېلېمېنتلىرىنى
بىرلەشتۈرگەن لايىھە يارىتىڭ.
```

## قورال چەكلىمىلىرى ۋە ئويلىنىشلار

### سۈرئەت چەكلىمىسى

- Scraping مۇلازىمېتىرغا زىيادە يۈك چۈشۈشنىڭ ئالدىنى ئېلىش ئۈچۈن سۈرئەت چەكلىمىسىگە ئۇچرايدۇ
- ئوخشاش تور بېكەتكە كۆپ قېتىملىق تەلەپلەر ئاستىلىتىلىشى مۇمكىن
- چوڭ تور بېكەتلەرنى تەھلىل قىلىشقا تېخىمۇ ئۇزۇن ۋاقىت كېتىشى مۇمكىن

### مەزمۇن توغرىلىقى

- ھەرىكەتچان مەزمۇن تولۇق تۇتۇلماسلىقى مۇمكىن
- JavaScript ئارقىلىق كۆرسىتىلگەن مەزمۇن تولۇق بولماسلىقى مۇمكىن
- دەل ۋاقتىدىكى سانلىق مەلۇمات ئەكس ئەتمەسلىكى مۇمكىن
- بەزى مەزمۇنلار ھەقلىق توسۇقنىڭ ئارقىسىدا بولۇشى مۇمكىن

### قانۇنىي ۋە ئەخلاقىي ئويلىنىشلار

- robots.txt ۋە تور بېكەت سىياسەتلىرىگە ھۆرمەت قىلىڭ
- نەشر قىلىش ھوقۇقى بار مەزمۇننى قايتا كۆپەيتىش ئۈچۈن scrape قىلماڭ
- تەھلىلنى كۆچۈرۈش ئۈچۈن ئەمەس، ئىلھام ئېلىش ئۈچۈن ئىشلىتىڭ
- چىقىرىۋېلىنغان مەزمۇننى ئىشلىتىش ھوقۇقىڭىز بارلىقىنى دەلىللەڭ
- تەھلىل قىلىنغان تور بېكەتلەرنىڭ مۇلازىمەت شەرتلىرىگە ئەمەل قىلىڭ

### ئىقتىدار تەسىرى

- چوڭ تور بېكەتلەرنى scrape قىلىشقا ۋاقىت كېتىشى مۇمكىن
- بىرلا ۋاقىتتا كۆپ scrape قىلىش ئاستىراق بولۇشى مۇمكىن
- تور ئۇلىنىشى سۈرئەتكە تەسىر قىلىدۇ
- چوڭ سانلىق مەلۇمات توپلاملىرى تېخىمۇ كۆپ بىر تەرەپ قىلىشنى تەلەپ قىلىشى مۇمكىن

## ئەڭ ياخشى ئەمەلىيەتلەر

### تور بېكەت تەھلىلىنى ئىشلىتىش

1. **كۆپ تور بېكەتنى تەھلىل قىلىڭ** — بىر قانچە مەنبەدىن ئىلھام توپلاڭ
2. **قۇرۇلمىغا دىققەت قىلىڭ** — ئورۇنلاشتۇرۇش ۋە تەشكىللەش ئەندىزىلىرىنى ئۆگىنىڭ
3. **رەڭلەرنى چىقىرىڭ** — رەڭ پالېتلىرىنى باشلىنىش نۇقتىسى سۈپىتىدە ئىشلىتىڭ
4. **خەت نۇسخىسىنى ئۆگىنىڭ** — ئۆزىڭىز ياقتۇرىدىغان خەت بىرىكمىلىرىنى ئېنىقلاڭ
5. **يولباشلاشنى كۆرۈپ چىقىڭ** — تىزىملىك تەشكىللىنىشىنى چۈشىنىڭ

### ئەخلاقىي Scraping

1. **robots.txt غا ھۆرمەت قىلىڭ** — تور بېكەت كۆرسەتمىلىرىگە ئەمەل قىلىڭ
2. **مەزمۇننى كۆچۈرمەڭ** — تەھلىلنى پەقەت ئىلھام ئۈچۈنلا ئىشلىتىڭ
3. **ھوقۇقلارنى دەلىللەڭ** — چىقىرىۋېلىنغان مەزمۇننى ئىشلىتەلەيدىغانلىقىڭىزغا كاپالەتلىك قىلىڭ
4. **مەنبەلەرنى كۆرسىتىڭ** — ئىلھام مەنبەلىرىگە ئىناۋەت بېرىڭ
5. **شەرتلەرگە ئەمەل قىلىڭ** — تور بېكەتنىڭ مۇلازىمەت شەرتلىرىگە بويسۇنۇڭ

### قورال ئۈنۈمدارلىقىنى ئەڭ يۇقىرى چەككە يەتكۈزۈش

1. **ئېنىق بولۇڭ** — كونكرېت تەھلىل تۈرلىرىنى تەلەپ قىلىڭ
2. **مەزمۇن-ئارقا كۆرۈنۈش تەمىنلەڭ** — تور بېكىتىڭىزنىڭ مەقسىتىنى چۈشەندۈرۈڭ
3. **كۈتۈلمىنى بەلگىلەڭ** — لايىھە نىشانلىرىڭىزنى تەسۋىرلەڭ
4. **قايتا-قايتا ياخشلاڭ** — نەتىجىلەرگە ئاساسەن ئىنچىكىلەشتۈرۈڭ
5. **قوراللارنى بىرلەشتۈرۈڭ** — ئەتراپلىق تەھلىل ئۈچۈن كۆپ قورال ئىشلىتىڭ

## مەسىلە ھەل قىلىش

### تور بېكەت Scrape قىلىنمايدۇ

- تور بېكەتنىڭ ئاممىۋى زىيارەت قىلىنىدىغانلىقىنى تەكشۈرۈڭ
- robots.txt نىڭ scraping غا رۇخسەت قىلىدىغانلىقىنى دەلىللەڭ
- باشقا بىر تور بېكەتنى سىناپ بېقىڭ
- تور ئۇلىنىشىنى تەكشۈرۈڭ
- مەسىلە داۋاملاشسا قوللاش بىلەن ئالاقىلىشىڭ

### تولۇق بولمىغان تەھلىل

- تور بېكەتتە ھەرىكەتچان مەزمۇن بولۇشى مۇمكىن
- JavaScript تولۇق كۆرسىتىلمەسلىكى مۇمكىن
- چوڭ تور بېكەتلەردە ۋاقىت ئېشىپ كېتىشى مۇمكىن
- ئۇنىڭ ئورنىغا كونكرېت بەتلەرنى تەھلىل قىلىپ بېقىڭ
- كونكرېت تەھلىل تۈرلىرىنى تەلەپ قىلىڭ

### ئاستا ئىقتىدار

- چوڭ تور بېكەتلەرنى تەھلىل قىلىشقا تېخىمۇ ئۇزۇن ۋاقىت كېتىدۇ
- بىرلا ۋاقىتتا كۆپ scrape قىلىش ئاستىراق بولىدۇ
- تور ئۇلىنىشى سۈرئەتكە تەسىر قىلىدۇ
- يۇقىرى پەللە ۋاقتىدىن باشقا ۋاقىتتا سىناپ بېقىڭ
- ئالدى بىلەن كىچىكرەك بۆلەكلەرنى تەھلىل قىلىڭ
