# ئۆزگىرىش خاتىرىسى

## 1.18.0 — 2026-06-29 دا ئېلان قىلىندى {#1180--released-on-2026-06-29}

### يېڭى {#new}

- **Google Calendar قوراللىرى** — جەدۋەلنى تونۇيدىغان ئاپتوماتلاشتۇرۇشلار ئۈچۈن تەڭشەلگەن كالېندارلار ۋە ۋەقەلەرنى ئوقۇش
- **ئالاقە خەرىتىلەش ۋە قاتناشقۇچى ياردەمچىلىرى** — كالېندار قاتناشقۇچىلىرىنى بېكەت ئىشلەتكۈچىلىرى ۋە ئالاقە ئادەملىرى بىلەن ماسلاشتۇرۇش
- **ئادەم تەستىق دەرۋازىلىرى ۋە ئەسكەرتىش خاتىرىلىرى** — تەكشۈرۈش ئۈچۈن ئاپتوماتلاشتۇرۇشلارنى توختىتىپ تۇرۇش ۋە تەكرار ئۇقتۇرۇشلارنىڭ ئالدىنى ئېلىش
- **TextBee SMS تەمىنلىگۈچىسى** — TextBee ئارقىلىق تەڭشەلگەن قىسقا ئۇچۇر ئۇقتۇرۇشلىرىنى يوللاش
- **ئىلغار ھەمراھ بولاق** — WordPress.org قۇرۇلمىسىدىن ئايرىم تارقىتىلىدىغان ئىشەنچلىك ئىجادىيەتچى ھۆججەت سىستېمىسى، ساندان، WP-CLI، REST تارقاتقۇچ، plugin قۇرغۇچ، git سۈرەت نۇسخىسى، ئىشلەتكۈچى باشقۇرۇش ۋە benchmark قوراللىرىنى قوشۇش

### ياخشىلاندى {#improved}

- **باشقۇرۇلىدىغان Superdav مۇلازىمەت تەڭشىكى** — قوللايدىغان بېكەتلەر ئۈچۈن ساھىبخانىلىق قىلىنغان مۇلازىمەت endpoint لىرى ۋە ئاپتوماتىك ئۇلىنىش تەمىنلەشنى قوشۇش
- **ئېلان قىلىش بولاقلىشى** — ئايرىم يادرو ۋە Advanced ZIP لىرىنى قۇرۇش، ھەر ئىككىسىنى GitHub دا ئېلان قىلىش، ھەمدە پەقەت يادرو بولىقىنى WordPress.org غا يوللاش

### تۈزىتىلدى {#fixed}

- **AI ئىلتىماس ئىشەنچلىكلىكى** — model تاللاش، ئىلتىماس ۋاقىت چەكلىمىسى، كۆڭۈلدىكى تەڭشەك modeli، reasoning-text بىر تەرەپ قىلىش ۋە ئىناۋەتسىز قورال چاقىرىشلىرى ئۈچۈن قايتا سىناش يېتەكچىلىكىنى ياخشىلاش
- **كالېندار ۋە ئەسكەرتىش مۇستەھكەملەش** — Google Calendar token لىرى ۋە ئەسكەرتىش تەكرارسىزلاشتۇرۇشنى كۈچەيتىش
- **باشلاش يېتەكچىلىكى ۋە تەستىق داۋاملاشتۇرۇشلىرى** — frontend باشلاش يېتەكچىلىكىنى قوزغىتىش ۋە تەستىقلانغان قابىلىيەت داۋاملاشتۇرۇشلىرىنى تۈزىتىش
- **WordPress.org بولاق تەكشۈرۈش مەسىلىلىرى** — يادرو ئېلانى ئۈچۈن بولاق تەكشۈرۈش ئىنكاسلىرىنى ھەل قىلىش

## 1.16.0 — 2026-05-20 دا ئېلان قىلىندى {#1160--released-on-2026-05-20}

### يېڭى {#new-1}

- **Logo SVG ھاسىل قىلىش قابىلىيىتى** — Theme Builder ئەمدى namespace-بىخەتەر تازىلاش بىلەن خاس logo SVG لىرىنى ھاسىل قىلىپ قىستۇرالايدۇ
- **بايقاش سۆھبىتىدە سۈرەت يۈكلەش** — Theme Builder بايقاش سۆھبىتى ئەمدى تېخىمۇ مول لايىھە مەزمۇنى ئۈچۈن سۈرەت يۈكلەش باسقۇچىنى ئۆز ئىچىگە ئالىدۇ
- **Palette Contrast دەلىللەش قابىلىيىتى** — theme غا قوللىنىشتىن بۇرۇن رەڭ جۈپلىرىنىڭ WCAG ماسلىقىنى تەكشۈرۈش
- **مېھماندارچىلىق تىزىملىكلىرى** — Theme Builder ئەمدى مېھماندارچىلىق كارخانىلىرى ئۈچۈن قۇرۇلمىلىق يېمەكلىك ۋە ئىچىملىك تىزىملىك بەتلىرىنى ھاسىل قىلالايدۇ
- **ئۈستەلئۈستى ۋە كۆچمە ئالدىن كۆرۈش رەندېرلىشى** — لايىھە يۆنىلىشى تاللاش جەريانىدا لايىھىڭىزنى ئۈستەلئۈستى ۋە كۆچمە ئۈسكۈنىلەردە ئالدىن كۆرۈڭ
- **يولباشچى بەلگە پارامېتىرى** — Create Menu قابىلىيىتى ئەمدى بەت ماۋزۇسىدىن ئايرىم بولغان `navigation_label` نى قوللايدۇ
- **Tier 1 قورال ئىشلىتىشچانلىقى** — `sd-ai-agent/site-scrape` ئەمدى Theme Builder دا كۆڭۈلدىكى ھالەتتە ئىشلىتىلىدىغان Tier 1 قورال

### تۈزىتىلدى {#fixed-1}

- **AI Client cache** — ئەمدى ئىلتىماسلار ئارا ساقلىنىش ئۈچۈن transients بىلەن قوللىنىدۇ، ئۇزاق ئىجرا بولىدىغان agent ۋەزىپىلىرىدە سانلىق مەلۇمات يوقاپ كېتىشنىڭ ئالدىنى ئالىدۇ
- **Plugin قۇر ھەرىكەت ئۇلانمىلىرى** — توغرىلاندى ۋە ئېنىقلىق ئۈچۈن قايتا نام بېرىلدى

## 1.10.0 — 2026-05-05 دا ئېلان قىلىندى {#1100--released-on-2026-05-05}

### يېڭى {#new-2}

- **Tavily ئىنتېرنېت ئىزدەش** — Brave Search بىلەن بىللە تېخىمۇ مول ئىنتېرنېت ئىزدەش نەتىجىلىرى ئۈچۈن Tavily نى ئىزدەش تەمىنلىگۈچىسى سۈپىتىدە قوشۇش
- **Theme نى تونۇيدىغان ئىچىگە قاچىلانغان ماھارەتلەر** — Block Themes، Classic Themes، Kadence Blocks ۋە Kadence Theme ماھارەت يېتەكچىلىرى ئەمدى plugin بىلەن بىللە كېلىدۇ
- **بېكەت قۇرغۇچ ئالاقە جەدۋىلى قابىلىيىتى** — پاراڭلىشىش كۆرۈنمە يۈزىدىن بىۋاسىتە ھەر قانداق بەتكە ئالاقە جەدۋىلى قوشۇش

### ياخشىلاندى {#improved-1}

- **WooCommerce بىرلەشتۈرۈشى قايتا قۇرۇلدى** — تېخىمۇ ياخشى ئىشەنچلىكلىك ۋە ماسلىشىشچانلىق ئۈچۈن ئەمدى native WooCommerce API لىرىنى ئىشلىتىدۇ
- **تەمىنلىگۈچى تىزىملىكى ئاپتوماتىك يېڭىلىنىدۇ** — ھەر قانداق plugin ئاكتىپلانغان ياكى ئاكتىپسىزلانغاندا

### تۈزىتىلدى {#fixed-2}

- **navigate-to قابىلىيىتى** — بەزى admin بەتلىرىدىكى چەكسىز قايتا يۈكلەش دەۋرى تۈزىتىلدى
- **list-posts قابىلىيىتى** — ئەمدى category ۋە tag ناملىرىنى slugs قا توغرا يېشىدۇ
- **WP-CLI بۇيرۇقلىرى** — ئالدىنقى refactor دىن كېيىن يوقاپ كەتكەن namespace alias لىرى ئەسلىگە كەلتۈرۈلدى
- **ۋەقە ئاپتوماتلاشتۇرۇشى** — ئاپتوماتلاشتۇرۇش جەدۋەللىرى تېخى قۇرۇلمىغان بېكەتلەرنى يۇمشاق بىر تەرەپ قىلىدۇ
- **memory-save قابىلىيىتى** — ئەمدى system instruction قۇرغۇچىسىدا توغرا namespace prefix نى ئىشلىتىدۇ
- **Scalar قورال نەتىجىلىرى** — ئەمدى AI غا قايتۇرۇلۇشتىن بۇرۇن توغرا ئورالغان
- **ئىشلىتىش ئىستاتىستىكىسى** — ئەمدى كونا نەشرلەردىن يېڭىلىغاندا legacy قابىلىيەت ئاچقۇچى فورماتىنى توغرا بىر تەرەپ قىلىدۇ
