# GoCardless ئۆزگىرىش خاتىرىسى

نەشرى: 1.0.3 - 2026-05-05 دا ئېلان قىلىندى
* تۈزىتىلدى: GoCardless ئەزالىقى كۈتۈۋاتقان ياكى توختىتىپ قويۇلغان ھالەتتە ئەمەلدىن قالدۇرۇلغاندا، يېتىم قالغان كۈتۈۋاتقان تور بېكەتلەر ھازىر تازىلىنىدۇ
* ياخشىلاندى: Bedrock يىلتىز autoloader ئاللىقاچان تايىنىشچانلىقلارنى يۈكلىگەن بولسا، قىستۇرما autoloader ئاتلاپ ئۆتۈلىدۇ

نەشرى: 1.0.1 - 2025-09-28 دا ئېلان قىلىندى

* ئالدى قوشۇمچىنى ultimate-multisite قىلىپ ئۆزگەرتىش؛ تېكىست دائىرىسىنى يېڭىلاش؛ نەشرىنى ئۆستۈرۈش.



2021-09-24 - نەشرى 1.0.0-beta.4

* قوشۇلدى: mu-plugins ئاساسىدىكى سەپلەشلەر ئۈچۈن filter wp_ultimo_skip_network_active_check؛

2021-06-16 - نەشرى 1.0.0-beta.3

* تۈزىتىلدى: Webhook بىر تەرەپ قىلغۇچلىرى ئىمزىلارنى جەزملەشتۈرەلمەيۋاتقان مەسىلە؛

2021-06-14 - نەشرى 1.0.0-beta.2

* ياخشىلاندى: GoCardless mandate قۇرۇلغاندىن كېيىن تور بېكەتنى ئېلان قىلىش؛

نەشرى 1.0.0-beta.1 - دەسلەپكى نەشرى
