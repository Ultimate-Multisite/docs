# AffiliateWP ئۆزگىرىش خاتىرىسى

نەشرى: 2.0.1 - 2025-09-28 كۈنى ئېلان قىلىندى

* prefix نى ultimate-multisite غا ئۆزگەرتىش؛ text domain نى يېڭىلاش؛ نەشر نومۇرىنى ئۆستۈرۈش.



نەشرى: 2.0.0 - 2022-08-05 كۈنى ئېلان قىلىندى

* plugin نى ultimo چىقىملىرى ئاساسىدا ئىشلەيدىغان قىلىپ قايتا قۇرۇش

نەشرى: 2.0.0-beta.2 - 2022-01-21 كۈنى ئېلان قىلىندى

* ئىچكى: hooks ۋە filters ھاسىللىغۇچ قوشۇلدى؛
* ئىچكى: تەرەققىياتچىلارنىڭ ئىشلىشىنى قولايلاشتۇرۇش ئۈچۈن Multisite Ultimate stubs قوشۇلدى؛
* تۈزىتىلدى: بېرىلگەن سېۋەتتە ئېتىبار code يوق بولغاندا يۈز بېرىدىغان ئېغىر خاتالىق؛
* تۈزىتىلدى: زىيارەتلەرنىڭ توغرا ئايلاندۇرۇلماسلىقى؛

نەشرى 2.0.0-beta.1 - 2021-12-29 كۈنى ئېلان قىلىندى

* 2.0 ئاساسىدىكى نەشرىنىڭ دەسلەپكى ئېلان قىلىنىشى؛
