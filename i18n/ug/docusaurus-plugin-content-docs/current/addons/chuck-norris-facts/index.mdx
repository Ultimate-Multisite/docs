# چاك نوررىس پاكىتلىرى {#chuck-norris-facts}

كۈندىلىك WordPress باشقۇرۇش ۋەزىپىلىرىڭىزگە ئازراق يۇمۇر قوشۇڭ، بۇنى **Ultimate Multisite - چاك نوررىس پاكىتلىرى** قوشۇمچىسى بىلەن قىلالايسىز. ئاكتىپلانغاندا، ھەر بىر باشقۇرۇش بېتىنىڭ ئۈستى ئوڭ بۇرجىكىدە ئىختىيارىي بىر چاك نوررىس پاكىتى كۆرسىتىلىدۇ.

گۇرۇپپىڭىزغا كۈلۈمسىرەش ئېلىپ كېلىش، ئۇزۇن باشقۇرۇش ئولتۇرۇشلىرىدا كەيپىياتنى يېنىكلىتىش ياكى پەقەتلا چاك نوررىسنىڭ رىۋايەتتەك قالتىسلىقىنى تەبرىكلەش ئۈچۈن ناھايىتى ماس كېلىدۇ.

## نېمىشقا چاك نوررىس پاكىتلىرى؟ {#why-chuck-norris-facts}

* **گۇرۇپپا روھىنى كۆتۈرۈش**: تورىڭىزنى باشقۇرۇشقا قىزىقارلىق بىر ئېلېمېنت قوشۇڭ
* **مۇزنى ئېرىتىش**: يېڭى گۇرۇپپا ئەزالىرى ئۈچۈن ياخشى سۆھبەت باشلىغۇچ
* **ئۈنۈمدارلىقنى ئاشۇرۇش**: قىسقا بىر كۈلكە دىققەت ۋە ئۈنۈمدارلىقنى ياخشىلايدۇ
* **ۋاقىتتىن ھالقىغان يۇمۇر**: چاك نوررىس پاكىتلىرى ھېچقاچان كونىرىپ قالمايدۇ

## ئاساسلىق ئىقتىدارلار {#key-features}

**110+ رىۋايەتتەك پاكىت**
رىۋايەتنىڭ ئۆزىنى تەبرىكلەيدىغان كلاسسىك چاك نوررىس چاقچاقلىرى ۋە پاكىتلىرىنىڭ كەڭ توپلىمى.

**دەخلىسىز كۆرسىتىش**
پاكىتلار باشقۇرۇش بالدىقىدا خىزمەت ئېقىمىڭىزغا دەخلى قىلمايدىغان كۆرۈنەر-كۆرۈنمەس ئۇسۇلدا پەيدا بولىدۇ.

**تەرجىمىگە تەييار**
بارلىق پاكىتلار تەرجىمە فۇنكسىيەلىرى ئىچىگە ئورالغان بولۇپ، تىلىڭىزغا يەرلىكلەشتۈرۈش ياكى گۇرۇپپىڭىز ئۈچۈن خاس پاكىتلارنى قۇرۇشنى ئاسانلاشتۇرىدۇ.

**تولۇق ماسلىشىشچان**
پاكىتلار ئۈستەل كومپيۇتېرى، تاختا كومپيۇتېر ۋە يانفون ئېكرانلىرىدا توغرا كۆرسىتىلىدۇ - چاك نوررىس بارلىق ئۈسكۈنىلەردە قالتىس.

**RTL قوللىشى**
ئوڭدىن سولغا يېزىلىدىغان تىللارنى تولۇق قوللايدۇ، چۈنكى چاك نوررىس تىل توسالغۇلىرىدىن ھالقىپ كېتىدۇ.

**بۆلەك تەھرىرلىگۈچ بىلەن ماسلىشىدۇ**
ئارايۈز توقۇنۇشلىرىدىن ساقلىنىش ئۈچۈن پاكىتلار بۆلەك تەھرىرلىگۈچتە ئەقلىي ھالدا يوشۇرۇلىدۇ.

**زىيارەتچانلىققا مەركەزلەشكەن**
تولۇق زىيارەتچانلىق تەلەپلىرىگە ماس كېلىش ئۈچۈن مۇۋاپىق ئېكران ئوقۇغۇچ تېكىستىنى ئۆز ئىچىگە ئالىدۇ.

## كىملەرگە ماس {#perfect-for}

* خاسلىق قوشماقچى بولغان multisite تور باشقۇرغۇچىلىرى
* ياخشى يۇمۇرنى قەدىرلەيدىغان تەرەققىيات گۇرۇپپىلىرى
* كۈندىلىك چاك نوررىس قالتىسلىقىدىن بىر ئۈلۈش لازىم بولغان ھەر قانداق كىشى
* قىزىقارلىق، ئىجابىي خىزمەت مۇھىتى ياراتماقچى بولغان بېكەتلەر

## ئورنىتىش {#installation}

1. WordPress تورىڭىزدا Ultimate Multisite قىستۇرمىسىنىڭ ئورنىتىلغان ۋە ئاكتىپلانغانلىقىغا كاپالەتلىك قىلىڭ
2. قوشۇمچە ھۆججەتلەرنى `/wp-content/plugins/` مۇندەرىجىڭىزگە يۈكلەڭ
3. WordPress ئىچىدىكى 'Plugins' تىزىملىكى ئارقىلىق قىستۇرمىنى تور بويىچە ئاكتىپلاڭ
4. ھەر بىر باشقۇرۇش بېتىدە ئىختىيارىي چاك نوررىس پاكىتلىرىدىن ھۇزۇرلىنىڭ!

سەپلىمە كېرەك ئەمەس - پەقەت ئاكتىپلاڭ ۋە كۈلۈمسىرەڭ!

## دائىم سورىلىدىغان سوئاللار {#frequently-asked-questions}

## بۇ Ultimate Multisite تەلەپ قىلامدۇ؟ {#does-this-require-ultimate-multisite}

Ultimate Multisite قوشۇمچىسى سۈپىتىدە لايىھەلەنگەن بولسىمۇ، بۇ قىستۇرما مۇستەقىل ئىشلەيدۇ ۋە ھەر قانداق WordPress ئورنىتىلمىسىدا يۈرگۈزۈلسە بولىدۇ.

## ئۆز پاكىتلىرىمنى قوشالايمەنمۇ؟ {#can-i-add-my-own-facts}

ھەئە! پاكىتلار تەرجىمىگە تەييار. خاس پاكىتلار بىلەن ئۆزىڭىزنىڭ `.po` ھۆججىتىنى قۇرۇپ، ئۇنى `/lang` مۇندەرىجىسىگە قويالايسىز. ياكى پروگرامما ئارقىلىق پاكىتلار تىزىقىنى سۈزەلەيسىز.

## بۇ مېنىڭ باشقۇرۇشۇمنى ئاستىلىتىۋېتەمدۇ؟ {#does-this-slow-down-my-admin}

ھەرگىز ئۇنداق ئەمەس! بۇ قىستۇرما ئىنتايىن يېنىك بولۇپ، ئىقتىدارغا ھېچقانداق تەسىر كۆرسەتمەيدۇ. ئۇ پەقەت تېكىست كۆرسىتىدۇ - سىرتقى API چاقىرىشلىرى ياكى ئېغىر بىر تەرەپ قىلىش يوق.

## ئۇنى مەلۇم بەتلەردە چەكلەيەلەيمەنمۇ؟ {#can-i-disable-it-on-certain-pages}

پاكىتلار دەخلى قىلىشى مۇمكىن بولغان بەتلەردە (مەسىلەن بۆلەك تەھرىرلىگۈچتە) ئاپتوماتىك يوشۇرۇلىدۇ. پاكىتلارنىڭ قەيەردە كۆرۈنىدىغانلىقىنى خاسلاشتۇرۇش ئۈچۈن WordPress hooks نىمۇ ئىشلىتەلەيسىز.

## پاكىتلار بارلىق تاماشىبىنلارغا ماس كېلەمدۇ؟ {#are-the-facts-appropriate-for-all-audiences}

ھەئە! بارلىق پاكىتلار پاكىز، ئائىلىگە ماس يۇمۇر بولۇپ، چاك نوررىسنىڭ رىۋايەتتەك ئورنىنى تەبرىكلەيدۇ.

## پاكىتلارنى باشقا تىلغا تەرجىمە قىلالايمەنمۇ؟ {#can-i-translate-the-facts-to-another-language}

ئەلۋەتتە! بارلىق پاكىتلار WordPress تەرجىمە فۇنكسىيەلىرىنى ئىشلىتىدۇ. تىلىڭىز ئۈچۈن تەرجىمە ھۆججىتى قۇرۇپ، ئۇنى `/lang` مۇندەرىجىسىگە قويۇڭ. تېكىست دائىرىسى: `ultimate-multisite-chuck-norris-facts`

## ئۇ multisite بىلەن ئىشلەمدۇ؟ {#does-it-work-with-multisite}

ھەئە! ئۇ مەخسۇس multisite تورلىرى ئۈچۈن لايىھەلەنگەن بولۇپ، تور بويىچە ئاكتىپلانسا بولىدۇ.
