# Loco Translate

Ultimate Multisite: Loco Translate Integration addon سىزنىڭ WordPress كۆپ تور بېكىتىڭىزگە بېكەت-خاس تەرجىمە ئىقتىدارلىرىنى ئېلىپ كېلىدۇ. ئەگەر تورىڭىزدىكى ئوخشىمىغان بېكەتلەرگە ئوخشاش بىر plugin ياكى theme نىڭ ئوخشىمىغان تەرجىمىلىرى لازىم بولغان بولسا، بۇ addon ئەڭ مۇكەممەل ھەل قىلىش چارىسىدۇر.

**بۇ Addon نېمە قىلىدۇ**

بۇ addon Ultimate Multisite نى ئاممىباب Loco Translate plugin بىلەن يوچۇقسىز بىرلەشتۈرۈپ، تورىڭىزدىكى ھەر بىر تارماق بېكەتنىڭ ئۆزىگە خاس تەرجىمىلىرىنى ساقلىشىغا ئىمكان بېرىدۇ. مەيلى سىز كۆپ رايونلۇق سۇپا يۈرگۈزۈۋاتقان، ئوخشىمىغان تىل جەمئىيەتلىرىگە مۇلازىمەت قىلىۋاتقان ياكى پەقەت تورىڭىز بويىچە تەرجىمە جانلىقلىقىغا ئېھتىياجلىق بولغان بولۇڭ، بۇ addon بۇ ئىشنى ناھايىتى ئاسانلاشتۇرىدۇ.

**تۆۋەندىكىلەر ئۈچۈن ناھايىتى ماس:**

* ئوخشىمىغان يەرلىكلەشتۈرۈش ئېھتىياجى بار كۆپ رايونلۇق كارخانىلار
* كۆپ خىل جەمئىيەتلەرگە مۇلازىمەت قىلىدىغان مائارىپ ئاپپاراتلىرى
* كۆپ تىللىق قوللاش تەمىنلەيدىغان SaaS سۇپىلىرى
* كۆپلىگەن خېرىدار بېكەتلىرىنى باشقۇرىدىغان تور ئاگېنتلىقلىرى
* تەرجىمە جانلىقلىقىنى تەلەپ قىلىدىغان ھەرقانداق كۆپ تور بېكەت تورى
