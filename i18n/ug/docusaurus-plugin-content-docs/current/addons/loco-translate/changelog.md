# Loco Translate ئۆزگىرىش خاتىرىسى

### 1.0.1 {#101}
* ئەڭ يېڭى Loco Translate نەشرى بىلەن ماسلىشىشچانلىقى ياخشىلاندى
* تېخىمۇ ياخشى بىخەتەرلىك ئۈچۈن زىيارەت كونتروللۇقى كۈچەيتىلدى
* ئىقتىدار ئەلالاشتۇرۇشلىرى

### 1.0.0 {#100}
* دەسلەپكى ئېلان قىلىنىش
* تور بېكەتكە خاس تەرجىمە مۇندەرىجىلىرى
* تەرجىمە ئالدىنلىق سىستېمىسى
* زىيارەت كونتروللۇقىنى ئەمەلىيلەشتۈرۈش
* Loco Translate بىلەن يوچۇقسىز كۆرۈنمە يۈز بىرلەشتۈرۈلۈشى
