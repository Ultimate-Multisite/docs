# تور بېكەت چىقارغۇچ ئۆزگىرىش خاتىرىسى

نەشرى: 1.0.6 - 2026-05-11 دە ئېلان قىلىنغان
* يېڭى: تور بېكەت چىقىرىشلىرى ھازىر ئۆزى قوزغىلىدىغان index.php نى بىرگە باغلايدۇ، شۇڭا ZIP نى ئايرىم plugin ئورنىتىشسىز يېڭى مۇلازىمېتىرغا ئورنىتىشقا بولىدۇ
* يېڭى: تور چىقىرىشى باشقۇرغۇچىلارنىڭ Site Export باشقۇرۇش بېتىدىن بارلىق تارماق تور بېكەتلەرنى بىرلا ئارخىپتا چىقىرىشىغا يول قويىدۇ

نەشرى: 1.0.5 - 2026-05-05 دە ئېلان قىلىنغان
* تۈزىتىلدى: WP_CLI polyfill stub غا __callStatic قوشۇلدى، get_config غا ئوخشاش تەقلىد قىلىنمىغان ئۇسۇللارنى بىر تەرەپ قىلىپ، سىناق bootstrap مەغلۇبىيەتلىرىنىڭ ئالدىنى ئالىدۇ

نەشرى: 1.0.4 - 2025-11-25 دە ئېلان قىلىنغان

* Ultimate Multisite 2.4 بىلەن ماسلىشىشچانلىقى تۈزىتىلدى.

نەشرى: 1.0.2 - 2025-09-28 دە ئېلان قىلىنغان

* ئالدى قوشۇلغۇچى نام ultimate-multisite غا ئۆزگەرتىلدى؛ تېكىست دائىرىسى يېڭىلاندى؛ نەشر نومۇرى كۆتۈرۈلدى.

### نەشرى 1.0.1 - 2023-08-09 دە ئېلان قىلىنغان {#version-101---released-on-2023-08-09}

* ياخشىلاندى: ھازىر ئىمپورت قىلغۇچ ھۆججىتىنىڭ توغرا ھۆججەت يولىنىڭ ئېلىنىشىغا كاپالەتلىك قىلىدۇ.
* تۈزىتىلدى: ئىمپورت جەريانىدا خاتالىقلارنىڭ ئالدىنى ئېلىش ئۈچۈن object cache چەكلەندى.
* تۈزىتىلدى: ئىمپورت قىلغۇچ plugin چۈشۈرۈش ئۇلىنىشى ئىشلىمەسلىك مەسىلىسى تۈزىتىلدى.

### نەشرى 1.0.0 - 2022-12-23 دە ئېلان قىلىنغان {#version-100---released-on-2022-12-23}

* تۈزىتىلدى: چىقىرىش جەدۋىلىنىڭ popup ى يۈكلەنمەسلىك؛
* تۈزىتىلدى: چىقىرىش بېتىدە Javascript ھۆججەتلىرى يۈكلەنمەسلىك؛
* ئىچكى: يېڭى قۇرۇش جەريانى؛

### نەشرى 1.0.0-beta.4 - 2022-01-13 دە ئېلان قىلىنغان {#version-100-beta4---released-on-2022-01-13}

* ئىچكى: hooks ۋە filters ھاسىللىغۇچ قوشۇلدى؛
* ئىچكى: تەرەققىياتچىلارنىڭ تۇرمۇش سۈپىتى ئۈچۈن WP Ultimo stubs قوشۇلدى؛
