# WooCommerce بىرىكتۈرۈش ئۆزگىرىش خاتىرىسى {#woocommerce-integration-changelog}

نەشرى 2.2.0 - 2026-07-01 دە ئېلان قىلىندى
* يېڭى: Ultimate Multisite باج سوممىلىرى ئەمدى چىقىم قىلىش جەريانىدا ئايرىم WooCommerce ھەق قۇرلىرى سۈپىتىدە كۆرۈنىدۇ، بۇ پۇل تۆلەشتىن بۇرۇن باج ئومۇمىي سوممىسىنى تېخىمۇ ئېنىق قىلىدۇ.
* يېڭى: WooCommerce Subscriptions قايتا سىناش كۆزنىكىدە مەغلۇپ بولغان يېڭىلاش سەۋەبىدىن ئەزالىقنى توختىتىشنى خالايدىغان تور بېكەتلەر ئۈچۈن تاللاپ قوزغىتىلىدىغان "مەغلۇپ بولغان يېڭىلاشلاردا ئەزالىقلارنى دەرھال توختىتىش" تەڭشىكى ۋە `wu_woo_suspend_on_payment_failure` سۈزگۈچى قوشۇلدى.
* تۈزىتىش: مەغلۇپ بولغان ياكى ئەسلىگە كەلگەن يېڭىلاشلاردىن كېيىن Ultimate Multisite ئەزالىقلىرى بىلەن ماس قەدەمدىن چىقىپ كېتىشى مۇمكىن بولغان WooCommerce Subscription ھالەتلىرى ماسلاشتۇرۇلدى.
* تۈزىتىش: كەم بولغاندا WooCommerce دۇكان پۇل بىرلىكى Ultimate Multisite پۇل بىرلىكى تىزىملىكىگە قوشۇلدى.
* تۈزىتىش: مۇشتەرىلەرنى WooCommerce چىقىم قىلىش بېتىگە قايتا يوللىغاندا خېرىدارنىڭ ھېسابات تالونى تەپسىلاتلىرى ساقلاپ قېلىندى.
* ياخشىلاندى: Jetpack Autoloader 5 بىلەن ماسلىشىشچانلىق قوشۇلدى.
* ياخشىلاندى: ئېلان بولىقى ھاسىل قىلىش تازىلاندى، شۇڭا GitHub ۋە بازار zip ھۆججەتلىرى ئىچكى باسقۇچ مۇندەرىجىلىرى ۋە تەرەققىيات ھۆججەتلىرىدىن ساقلىنىدۇ.

نەشرى 2.0.6 - 2026-01-16 دە ئېلان قىلىندى
* ياخشىلىنىش: قوشۇمچە ئىقتىدارغا يادرولۇق مۇشتەرىلىكلەر كىرگۈزۈلدى. ئەمدى Woocommerce Subscriptinos كېڭەيتىلمىسىنى تەلەپ قىلمايدۇ.

نەشرى 2.0.5 - 2026-01-09 دە ئېلان قىلىندى

نەشرى 2.0.1 - 2023-08-09 دە ئېلان قىلىندى

* قوشۇلدى: Woocommerce 7.9.0 بىلەن ماسلىشىشچانلىق.
* قوشۇلدى: Woocommerce Subscriptions 5.3.0 بىلەن ماسلىشىشچانلىق.

* قوشۇلدى: Ultimate Multisite قۇرغان مەھسۇلاتلار Woocommerce تىزىملىكىدىن چىقىرىۋېتىلدى.

* ياخشىلىنىش: تەكرارلىنىدىغان ئېتىبار Woocommerce مەھسۇلاتىغا ئەسلىگە كەلتۈرۈلدى.
* ياخشىلىنىش: Woocommerce مەھسۇلاتىغا تەكرارلىنىدىغان ئېتىبار بەلگىسى قوشۇلدى.
* ياخشىلىنىش: چىقىم قىلىشتا مەھسۇلات تۈرى كاپالەتكە ئىگە قىلىندى.
* تۈزىتىش: تۆۋەن دەرىجىگە چۈشۈرۈش جەريانىدا ئەزالىق ھالىتى ساقلاپ قېلىندى.
* تۈزىتىش: بىكار قىلىش جەريانىدا خاتالىقلاردىن ساقلىنىش ئۈچۈن مۇشتەرىلىكنىڭ مەۋجۇتلۇقى تەكشۈرۈلدى.
* تۈزىتىش: Woocommerce مۇشتەرىلىكلىرىدە ئىشلىتىش ئۈچۈن مۇشتەرىلىك باشلىنىش ۋاقتى قوشۇلدى.
* ئىچكى: يېڭى PHP 8.1 قۇرۇش جەريانى يولغا قويۇلدى.

نەشرى 2.0.0 - تولۇق قايتا يېزىلدى.

* قوشۇلدى: چىقىم دەرۋازىسىنى ئۆزگەرتكەندە ياكى ئەزالىقنى بىكار قىلغاندا woo مۇشتەرىلىكىنى چىقىرىۋېتىش ئۈچۈن بىكار قىلىش ئۇسۇلىنى بىر تەرەپ قىلىش;
* قوشۇلدى: ئەزالىقلارنى تۆۋەن دەرىجىگە چۈشۈرۈش ۋە يۇقىرى دەرىجىگە كۆتۈرۈش بىر تەرەپ قىلغۇچىسى;
* ياخشىلىنىش: ھېسابات يېڭىلاشقا يول قويۇش ئۈچۈن تارماق تور بېكەتلەردىكى خېرىدار يېڭىلاش جەدۋىلىدە woocommerce بېقىندىلىقلىرىنى يۈكلەش;
* ياخشىلىنىش: مەۋجۇت بولمىسا Woocommerce ھارۋىسىنى توغرا يۈكلەش;
* ياخشىلىنىش: چىقىم قىلىشنى بىر تەرەپ قىلغاندا ئاساسىي تور بېكەت جەدۋەللىرىدە ئىكەنلىكىمىزنى كاپالەتكە ئىگە قىلىش;
* ياخشىلىنىش: Ultimo يېڭىلاش زاكازىنى ئاخىرقى پۇل تۆلەشتىن ئەمەس، Woocommerce مۇشتەرىلىك زاكاز قىممىتىگە ئاساسلاندۇرۇش;
* تۈزىتىش: WU Membership كۇنۇپكا ئۇلانمىسىغا بېرىش;
* تۈزىتىش: Woocommerce مۇشتەرىلىكلىرىنىڭ يېڭىلىنىشى تۆلەنگەندە Ultimo زاكازىنى تۆلەنگەن دەپ بەلگىلەش;
* قۇرۇش: MPB نى قۇرغۇچى سۈپىتىدە قوشۇش;
