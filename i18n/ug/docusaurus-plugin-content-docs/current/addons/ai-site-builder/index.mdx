# AI تور بېكەت قۇرغۇچىسى

ultimate-multisite-ai-site-builder

## تەلەپلەر {#requirements}

- WordPress 5.3 ياكى ئۇنىڭدىن يۇقىرى
- PHP 7.4 ياكى ئۇنىڭدىن يۇقىرى
- Multisite Ultimate قىستۇرمىسى (ئاكتىپ)

## ئورنىتىش {#installation}

1. قوشۇمچە ھۆججەتلەرنى `/wp-content/plugins/` مۇندەرىجىڭىزگە يۈكلەڭ
2. WordPress ئىچىدىكى 'Plugins' تىزىملىكى ئارقىلىق قىستۇرمىنى ئاكتىپلاڭ
3. تور باشقۇرغۇچتا قوشۇمچە تەڭشەكلىرىنى سەپلەڭ

## تەرەققىيات {#development}

### تەڭشەش {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP تۈرلىرى ۋە قوشۇلمىلار

│   └── class-ai_site_builder-main.php             # ئاساسلىق ئىقتىدار
├── tests/                        # بىرلىك سىناقلىرى

│   ├── class-*-base.php         # ئاساسىي سىناق تۈرى

├── views/                       # قېلىپ ھۆججەتلىرى
├── assets/                      # تۇراقلىق مەنبەلەر
│   ├── css/                     # ئۇسلۇب جەدۋەللىرى
│   ├── js/                      # JavaScript ھۆججەتلىرى

```

### سىناق قىلىش {#testing}

بۇ قوشۇمچە مۇكەممەل سىناق رامكىسىنى ئۆز ئىچىگە ئالىدۇ:

- **بىرلىك سىناقلىرى**: WordPress سىناق يۈرۈشلۈكى بىلەن PHPUnit ئاساسىدىكى سىناقلار
- **ئاساسىي سىناق تۈرى**: ئورتاق سىناق سىنارىيەلەر ئۈچۈن ياردەمچى ئۇسۇللارنى تەمىنلەيدۇ
- **سىناق قاپلىنىشى**: كود قاپلىنىشىنى ئىز قوغلاشقا سەپلەنگەن
- **Multisite سىناق قىلىش**: سىناقلار multisite مۇھىتىدا ئىجرا قىلىنىدۇ

### كود ئۆلچەملىرى {#code-standards}

- **WordPress كود ئۆلچەملىرى**: PHPCS ئارقىلىق ئىجرا قىلىنىدۇ
- **PHP 7.4+ ماسلىشىشچانلىقى**: زامانىۋى PHP ئىقتىدارلىرى قوللىنىدۇ
- **تۇراقلىق تەھلىل**: تىپ تەكشۈرۈش ئۈچۈن PHPStan بىرلەشتۈرۈلۈشى
- **كودنى زامانىۋىلاشتۇرۇش**: PHP يېڭىلاشلىرى ئۈچۈن Rector قائىدىلىرى

### قۇرۇش سىستېمىسى {#build-system}

- **مەنبە بىر تەرەپ قىلىش**: CSS/JS ھۆججەتلىرى ئۈچۈن كىچىكلىتىش
- **تەرجىمە**: POT ھۆججىتى ھاسىل قىلىش
- **بوغچىلىش**: تارقىتىش ئۈچۈن ئارخىپ يارىتىش
- **تەرەققىيات ھالىتى**: سازلاش ئۈچۈن كىچىكلىتىلمىگەن مەنبەلەر

## سەپلىمە {#configuration}

بۇ قوشۇمچە تور باشقۇرغۇچ ئارايۈزى ئارقىلىق زىيارەت قىلغىلى بولىدىغان ھەر خىل سەپلىمە تاللانمىلىرىنى قوللايدۇ.

## Hookلار ۋە Filterلار {#hooks-and-filters}

### مەشغۇلاتلار {#actions}

- `ultimate-multisite-ai-site-builder_init` - قوشۇمچە دەسلەپلەشتۈرۈلگەندىن كېيىن قوزغىتىلىدۇ
- `ultimate-multisite-ai-site-builder_loaded` - بارلىق قوشۇمچە ھۆججەتلەر يۈكلەنگەندىن كېيىن قوزغىتىلىدۇ

### Filterلار {#filters}

- `ultimate-multisite-ai-site-builder_settings` - قوشۇمچە تەڭشەكلىرىنى ئۆزگەرتىدۇ
- `ultimate-multisite-ai-site-builder_enabled` - قوشۇمچىنىڭ قوزغىتىش/چەكلەش ھالىتىنى بېسىپ ئۆتىدۇ
