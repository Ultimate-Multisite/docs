# قىستۇرما ۋە ئۇسلۇب باشقۇرغۇچىسى

Multisite Ultimate: قىستۇرما ۋە ئۇسلۇب باشقۇرغۇچىسى

قاچىلانغان قىستۇرمىلار ۋە ئۇسلۇبلارنىڭ مېتا ئۇچۇرلىرىنى (ماۋزۇ، ئاپتور، كىچىك سۈرەت ۋە چۈشەندۈرۈش) تەھرىرلەڭ ياكى يوشۇرۇڭ. قىستۇرمىلار ۋە ئۇسلۇبلار ئۈچۈن تۈرلەر قۇرۇپ، ئىشلەتكۈچىلىرىڭىزنىڭ ئۇلارنى سۈزۈشىگە يول قويۇڭ ھەمدە ئىشلەتكۈچىلىرىڭىزنىڭ تاختىسىغا چىرايلىق خاس قىستۇرمىلار بېتىنى قوشۇڭ.

قىستۇرمىنى ئىشلىتىش ئۈچۈن، تور باشقۇرغۇچىڭىزدىكى قىستۇرمىلار ياكى ئۇسلۇبلار بېتىگە بېرىپ، "قىستۇرما/ئۇسلۇب ئۇچۇرىنى تەھرىرلەش" نى چېكىشلا كۇپايە.

## قاچىلاش

1. 'ultimate-multisite-plugin-and-theme-manager' نى '/wp-content/plugins/' مۇندەرىجىسىگە يۈكلەڭ
2. WordPress تور باشقۇرغۇچىڭىزدىكى 'قىستۇرمىلار' تىزىملىكى ئارقىلىق قىستۇرمىنى ئاكتىپلاڭ

## ئىشلىتىش قوللانمىسى

Ultimate Multisite: قىستۇرما ۋە ئۇسلۇب باشقۇرغۇچىسى كۈچلۈك **ھەقلىق** Ultimate Multisite قوشۇمچىسى بولۇپ، تورىڭىزدىكى ئىشلەتكۈچىلەرنىڭ قىستۇرما ۋە ئۇسلۇب تاللانمىلىرىنى قانداق كۆرۈشى ۋە ئۇلار بىلەن قانداق ئالاقە قىلىشى ئۈستىدىن تولۇق كونترول بېرىدۇ.

بۇ قوشۇمچە ئۈچ مەقسەتكە خىزمەت قىلىدۇ:

  1. ئۇ قىستۇرمىلار ۋە ئۇسلۇبلارنىڭ مېتا سانلىق مەلۇماتلىرىنى پۈتۈنلەي ئۆزگەرتىشىڭىزگە يول قويىدۇ. بۇ نام، چۈشەندۈرۈش، ئاپتور نامى، ھەتتا كىچىك سۈرەتلەرنىمۇ ئۆز ئىچىگە ئالىدۇ!

  2. ئۇ قىستۇرمىلار ۋە ئۇسلۇبلارنى تۈرگە ئايرىشىڭىزغا يول قويۇپ، ئىشلەتكۈچىلىرىڭىزنىڭ ئۇلارنى ئاسانلا رەتلىشىگە ئىمكان بېرىدۇ.

  3. ئۇ سىزگە كونا، زېرىكىشلىك قىستۇرمىلار بېتىنى تۈرلەر بويىچە رەتلەش ئىقتىدارى بار يېڭى ئۇسلۇبقا ئوخشاش بەت بىلەن ئالماشتۇرۇش تاللانمىسىنى بېرىدۇ.

ئۇنى چۈشۈرۈپ قاچىلىغاندىن كېيىن، Ultimate Multisite تەڭشەكلىرى بېتىدە يېڭى بەتكۈچ - _قوشۇمچە تەڭشەكلىرى_ \- كۆرۈنىدۇ.

_بۇ يەردە Ultimate Multisite: قىستۇرما ۋە ئۇسلۇب باشقۇرغۇچىسى قوشقان ئومۇمىي تاللانمىلارنى تاپالايسىز_

تورىڭىزنىڭ _قىستۇرمىلار_ بېتىدە ئەمدى بارلىق قىستۇرمىلارنىڭ نامى، ئاپتورى، ھەتتا قىستۇرمىنىڭ كىچىك سۈرىتى قاتارلىق مېتا ئۇچۇرلىرىنى تەھرىرلىيەلەيسىز.

<!-- ئېكران سۈرىتى يوق: بەلگىلىك ساھەلەر ئۈچۈن كۆرسىتىش/يوشۇرۇش ئالماشتۇرغۇچىسى بار قىستۇرما مېتا ئۇچۇر تەھرىرلىگۈچىسى -->

_سىز يەنە بەلگىلىك قىستۇرما ئۇچۇرلىرىنى كۆرسىتىش ياكى كۆرسەتمەسلىكنىمۇ تاللىيالايسىز_

قىستۇرما ۋە ئۇسلۇب باشقۇرغۇچىسى ئىككى كۆرسىتىش تۈرىنى تەمىنلەيدۇ: قىستۇرما ئۇسلۇبى ۋە ئۇسلۇب ئۇسلۇبى. شۇڭا، ئاخىرىدا، لازىملىق تەڭشەشلەرنى قىلغاندىن كېيىن، خېرىدارلىرىڭىز ئۆزلىرىنىڭ _قىستۇرمىلار_ بېتىنى مۇنداق كۆرىدۇ:

_خېرىدارنىڭ قىستۇرمىلار بېتىدە قىستۇرما ئۇسلۇبى كۆرسىتىلگەن_
