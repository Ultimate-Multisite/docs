# Material WP {#material-wp}

WordPress باشقۇرۇش ئارايۈزى ئۇزۇن يىللاردىن بۇيان ئوخشاش ھالەتتە بولۇپ كەلدى. ئەمما باشقۇرۇش ئارايۈزىنىڭ Material Design نۇسخىسى قانداق كۆرۈنىدىغانلىقىنى ئويلاپ باققانمۇسىز؟ 

Material WP سىزگە شۇنداق كۆرۈنۈش ۋە تۇيغۇ بېرىش ئۈچۈن ياسالغان باشقۇرۇش theme بولۇپ، خېرىدارلىرىڭىزغا تېخىمۇ خاسلاشتۇرۇلغان Dashboard تەقدىم قىلىشىڭىزغا ياردەم بېرىدۇ.

## ئىقتىدارلار {#features}

- Custom CSS قوللاش
- ئاسان ئۇسلۇب تەڭشەكلىرى
- تىزىملىكنى قايتا رەتلەش
- ئاسان ئەكىرىش/چىقىرىش

## ھۆججەتلەر {#documentation}

**Material WP ئارقىلىق Admin Panel نى خاسلاشتۇرۇش**

## ئورنىتىش {#installation}

1. 'material-wp' نى '/wp-content/plugins/' مۇندەرىجىسىگە يۈكلەڭ
2. قىستۇرمىنى WordPress ئىچىدىكى 'قىستۇرمىلار' تىزىملىكى ئارقىلىق ئاكتىپلاشتۇرۇڭ

## ئىشلىتىش قوللانمىسى {#usage-guide}

WordPress باشقۇرۇش ئارايۈزى ئۇزۇن يىللاردىن بۇيان ئوخشاش ھالەتتە بولۇپ كەلدى. ئەمما باشقۇرۇش ئارايۈزىنىڭ Material Design نۇسخىسى قانداق كۆرۈنىدىغانلىقىنى ئويلاپ باققانمۇسىز؟

Material WP سىزگە شۇنداق كۆرۈنۈش ۋە تۇيغۇ بېرىش ئۈچۈن ياسالغان باشقۇرۇش theme بولۇپ، خېرىدارلىرىڭىزغا تېخىمۇ خاسلاشتۇرۇلغان Dashboard تەقدىم قىلىشىڭىزغا ياردەم بېرىدۇ. Material WP يەنە Ultimate Multisite نىڭ premium قىستۇرمىلار يادروسىنىڭ بىر قىسمى.

**ئۇنى ئورناتقاندىن** كېيىن، _قىستۇرمىلار_ بېتىڭىزگە بېرىپ، ئۇنى تورىڭىزدا ئاكتىپلاشتۇرۇش ئۈچۈن چېكىڭ.
