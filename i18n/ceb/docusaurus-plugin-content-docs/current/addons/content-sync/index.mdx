# Content Sync

Multisite Ultimate: Content Sync

Kining addon para sa Multisite Ultimate nagtugot sa mga network administrator nga mag-synchronize og content gikan sa bisan unsang site ngadto sa ubang sites sulod sa network. Suportahan niini ang tanang post types, lakip na ang posts, pages, custom post types, ug Elementor templates. Naglakip usab ang addon og multi-step wizard interface para sa pagpili og content ug target nga mga site, uban ang batch processing ug real-time progress feedback.

## Features {#features}

* Content Selection: Visual card grid nga adunay thumbnails ug filtering base sa post type
* All Post Types: I-sync ang posts, pages, products, Elementor templates, ug bisan unsang custom post type
* Site Selection: Pagpili kung unsang sites ang pag-synchronizehan og content, uban ang paglimita sa source site ug pre-selection sa child site
* Batch Processing: Nag-sync ngadto sa 10 ka sites matag higayon nga adunay update sa progreso
* Media Sync: Awto nitabangkop ug nag-remap sa media files (hulagway, video) gikan sa post content ngadto sa target sites
* Taxonomy Sync: Gipreserbar ang mga categories, tags, ug custom taxonomies
* Elementor Support: Bug-os nga suporta alang sa Elementor templates uban ang CSS regeneration ug template ID remapping
* Conflict Resolution: Kapilian nga ilisan ang kasamtangang content o i-skip kini
* Cache Management: Awto kining maglinaw sa Elementor cache sa target sites pagkahuman sa sync

## Requirements {#requirements}

* WordPress 5.3 or higher
* PHP 7.4 or higher
* Multisite Ultimate plugin (active)
* Elementor plugin (optional, required for Elementor template sync)

## Installation {#installation}

1. I-upload ang 'ultimate-multisite-content-sync' ngadto sa direktoryo nga '/wp-content/plugins/'
2. I-activate ang plugin pinaagi sa 'Plugins' menu sa imong WordPress Network Admin
3. I-configure ang mga setting sa Ultimate Multisite → Settings → Content Sync
4. Pag-navigate sa Ultimate Multisite → Content Sync aron magsugod sa pag-sync
