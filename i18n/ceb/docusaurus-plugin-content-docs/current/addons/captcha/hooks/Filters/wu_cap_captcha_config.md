# Pansala: wu_cap_captcha_config

Salain ang configuration sa Cap Captcha.

Gitugotan ang mga developer nga i-customize ang mga setting sa Cap Captcha.

## Mga Parameter

| Ngalan | Tipo | Deskripsyon |
|------|------|-------------|
| $config | `array` | Ang array sa configuration sa Cap. |
| $level | `string` | Ang lebel sa seguridad (fast, medium, max). |
