# AffiliateWP {#affiliatewp}

Multisite Ultimate: AffiliateWP Integration

Gamita ang gamhanang AffiliateWP aron palapad pa ang base sa mga kustomer sa imong Ultimo Network!

## Installation {#installation}

1. I-upload ang 'ultimate-multisite-git' ngadto sa direktoryo nga '/wp-content/plugins/'
2. I-activate ang plugin pinaagi sa 'Plugins' menu sa WordPress
3. Sunda ang step by step Wizard aron ma-set up ang plugin

## Usage Guide {#usage-guide}

Ang [AffiliateWP](https://affiliatewp.com/) nahimong pinaka-kasaligan nga solusyon kon magtukod ka og affiliate program para sa imong online business. Ang Ultimate Multisite: AffiliateWP mao ang tulay nga makatabang nimo pagtukod niini mismo alang sa imong premium network.

## Getting Started {#getting-started}

Una sa tanan, **kinahanglan gyud nimo i-activate** ang [**AffiliateWP**](https://affiliatewp.com/)! Dili nimo magamit ang atong add-on kung wala kay **ang plugin nga makakonektar niini**. Kung dili ka plano nga ipagamit kini sa imong mga client sa ilang sites, ang AffiliateWP kinahanglan lang i-activate sa imong main site.

Dugang pa, kinahanglan pud nimo nga **i-activate** ang **Ultimate Multisite** ug **Ultimate Multisite: AffiliateWP** **sa tibuok network**.

## Adding Ultimate Multisite as an AffiliateWP Integration {#adding-ultimate-multisite-as-an-affiliatewp-integration}

Sa imong main site, adto sa Affiliates Settings, ug sa Integrations tab, i-check ang Ultimate Multisite. Kini magpahibalo kang [AffiliateWP](https://affiliatewp.com/) nga magsugod na pag-track sa mga bisita ug conversions gikan sa Ultimate Multisite sign-up flow.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_I-check ang Ultimate Multisite box sa ubos ug i-save_

Mao ra gyud na. Ang mga bisita nga naggikan sa referral links ma-track, ug kung mohatag pa og unang bayad kana nga user, ang conversion mahimo ug komisyon ang ibutang ngadto sa affiliate nga responsable niini nga conversion.

Apan, ang unang bayad lang gikan sa bag-o nilalang user ang ma-track. Aron ma-enable ang recurring referral payments, kinahanglan nimo ang [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Enabling Recurring Referrals {#enabling-recurring-referrals}

Pagkahuman i-install sa [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), adto sa Affiliates Settings Recurring Referrals tab, ug i-enable ang paggamit sa recurring referrals.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_Kasimple lang ni: I-Enable Recurring Referrals ug andam na ka!_

Mao ra gyud to!

Karon, matag mohatag og bag-ong bayad sa usa ka gi-refer nga account, ang responsable nga affiliate makakuha dayon sa iyang parte automatically!
