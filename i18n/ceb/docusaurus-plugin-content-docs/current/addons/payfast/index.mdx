# PayFast {#payfast}

Multisite Ultimate: Payfast Gateway

Nagdugang kini og Payfast – usa ka paspas ug luwas nga online payment gateway sa South Africa – isip opsyon sa pagbayad sa Multisite Ultimate.

## Pag-instalar (Installation) {#installation}

1. I-upload ang 'ultimate-multisite-payfast' sa direktoryo nga '/wp-content/plugins/'
2. I-activate ang plugin pinaagi sa 'Plugins' menu sa WordPress
3. Sunda ang step by step Wizard aron i-set up ang plugin

## Giya sa Paggamit (Usage Guide) {#usage-guide}

Ang PayFast usa ka nanguna nga online payment processor para sa mga negosyo sa South Africa. Gitumong niini ang pagtabang sa paglambo sa mga online merchant pinaagi sa paghimo sa transaksyon nga seamless ug luwas. Sumala sa hangyo sa user sa Ultimate Multisite, nagdesisyon kami sa pagmugna og usa ka addon nga makapahimo sa integration uban sa PayFast. Kini nga artikulo magpakita kung unsaon pag-set up sa PayFast isip imong payment gateway aron makadawat og bayad sa imong Ultimate Multisite network.

Ang unang lakang mao ang pag-install sa add-on.

# Giunsa Pag-install sa Payfast Add-on: {#how-to-install-the-payfast-add-on}

Adto sa add-ons page ug pilia ang Ultimate Multisite: Payfast Gateway add-on.

![Add-ons page](/img/admin/addons-page.png)

Mo-pop up ang bintana sa add-on. I-klik ang Install Now.

![Add-ons page with install option](/img/admin/addons-page.png)

Pagkahuman sa pag-install, imong direktahan ngadto sa plugins page. Dinhi, i-klik lang ang Network Activate ug ma-activate ang _Payfast_ add-on sa imong network.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Pag-set up sa PayFast {#setting-up-payfast}

Human i-install ang add-on, makadto na ka karon sa Ultimate Multisite settings ug magsugod pag-configure sa PayFast.

Sa ilalom sa pahina nga **Ultimate Multisite settings**, makita nimo ang "Payments" tab. Niilalom niining tab, makita nimo ang "**Payfast** " kauban ang usa ka toggle button aron ma-enable o ma-disable kining payment gateway.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Ipadayon ug i-klik ang toggle button aron ma-enable kini. Dayon mo-expand ang box aron ipakita ang nagkalain-laing fields nga kinahanglan nimong pun-an sama sa **Mechant ID, Merchant Key, ug PassPhrase.** Makakuha ka niining impormasyon gikan sa imong **Payfast** account

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Ang sunod nga lakang mao ang pag-login sa imong PayFast account aron makuha ang impormasyon nga imong gikinahanglan.

Upon logging in, makita dayon nimo ang credentials nga kinahanglan nimo gikan sa **Payfast** account dashboard.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Mahimo nimong i-copy ug i-paste ang credential ug pun-an ang mga detalye sa imong Ultimate Multisite settings. Dayon i-klik ang **"Save settings** " button aron ma-save ang mga kausaban.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Mao na! Kumpleto na ang imong PayFast payment integration!
