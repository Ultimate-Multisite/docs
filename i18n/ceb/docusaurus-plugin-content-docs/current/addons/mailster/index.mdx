# Mailster

I-usab ang imong multisite network ngadto sa usa ka gamhanang makina sa paghimo og email list gamit ang Ultimate Multisite Mailster Integration addon. Awtomatikong idugang ang mga bag-ong kustomer sa imong Mailster email lists kon sila mo-sign up o magpalit, nga nagmugna og walay putol nga koneksyon tali sa imong membership platform ug email marketing strategy.

## Key Features

**Automatic Subscriber Sync**
* Awtomatikong pagdugang sa mga kustomer ngadto sa Mailster lists kon sila mo-sign up
* Pagpili kon gusto ba nimo nga mag-sync dayon pagka-sign up o maghulat pa sa confirmation sa bayad
* Tipigan ang imong email lists nga fresh ug updated nga dili na kinahanglan og mano-manong buhat

**Flexible List Management**
* Magtakda og global default lists alang sa tanang bag-ong kustomer
* Mahimong i-override ang defaults gamit ang product-specific lists para sa mas gitutukan nga mga kampanya
* Dali ra kaayo pagdumala sa list assignment gamit ang pamilyar nga interface ni Mailster

**GDPR-Compliant Opt-In Options**
* Pagpili tali sa awtomatikong subscription o consent base sa checkbox
* Pagpahigayon og double opt-in para sa kumpirmadong mga subscription
* Ipakita/Itago ang opt-in checkbox base sa imong compliance requirements

**Smart Data Mapping**
* Awtomatikong pag-sync sa ngalan ug contact information sa kustomer
* Pag-map sa billing addresses ngadto sa subscriber fields
* Pagsubay sa signup dates para sa mga gitutukan nga kampanya

**Product-Level Control**
* Pag-assign og lainlaing lists sa nagkalain-laing produkto
* Pagmugna og targeted campaigns base sa unsa ang gipalit sa kustomer
* Pag-disable ni Mailster alang sa piho nga mga produkto kon kinahanglanon

**Professional Experience**
* Native Mailster UI integration para sa pagpili sa listahan
* Detalyado nga logging para sa troubleshooting ug verification
* Walay putol nga update sa kasamtangang subscribers

## Perfect For

* Mga tag-iya sa membership site nga nagtukod og email lists
* SaaS platforms nga nangandoy pag-atiman sa mga bag-ong kustomer
* Course creators nga nagsegmentar sa estudyante base sa ilang enrollment
* Bisag kinsa nga naggamit og Ultimate Multisite uban ang Mailster

## How It Works

1. I-install ug i-activate ang addon sa imong Ultimate Multisite network
2. I-configure ang global default lists sa Ultimate Multisite settings
3. Opsyonal nga i-override gamit ang product-specific lists
4. Pagpili sa imong opt-in mode ug subscription timing
5. Tan-awa pagdako og awtomatikong imong Mailster lists!

## Installation

1. I-upload ang plugin files ngadto sa imong Ultimate Multisite addons directory
2. Siguroha nga naka-install ug active si Mailster sa imong main site
3. I-activate ang addon pinaagi sa Ultimate Multisite addons page
4. Adto sa Settings > Mailster Integration para i-configure
5. Pagpili sa imong default lists ug pag-configure sa opt-in preferences

## Frequently Asked Questions

## Does this addon work without Ultimate Multisite?

Dili, kining addon nagkinahanglan og Ultimate Multisite aron molihok. Kini gidesinyo nga espesipiko aron i-integrate ang Mailster uban sa customer ug product system ni Ultimate Multisite.

## Do I need Mailster installed on all sites?

Dili, kinahanglan lang nga naka-install ug active si Mailster sa imong main network site. Ang addon awtomatikong mo-konektar niini gikan sa tanang subsites.

## Can I assign different lists to different products?

Oo! Mahimo kang magtakda og global default lists, dayon i-override kini base sa matag produkto. Kini nagtugot kanimo sa pagmugna og highly targeted nga email campaigns base sa unsa ang gipalit sa kustomer.

## What happens if a customer already exists in Mailster?

Sa default, ang addon mag-update sa kasamtangang subscribers ug idugang sila sa bag-ong lists. Mahimo nimo kining i-disable sa settings kon mas gusto nimo nga paliton ang mga existing subscribers.

## Is this GDPR compliant?

Oo! Ang addon naglakip og checkbox-based opt-in mode diin kinahanglan gayud nga mohatag og eksaktong pagtugot ang mga kustomer aron makadawat og emails. Kini nakatubos sa mga requirements sa GDPR kon gi-kombinar uban sa saktong privacy policies.

## Can I use double opt-in?

Sigurado! I-enable lang ang double opt-in setting ug magpadala si Mailster og confirmation emails sa dili pa ma-activate ang subscriptions.
