# Multi-Network Changelog

### 1.3.0 {#130}
* Bag-o: Network template previewer — makasusi ug makapre-view sa mga network templates gamit ang live panel sa dili pa mopalit.
* Bag-o: Network template picker at checkout — ang mga kustomer makapili og network template samtang nag-sign up o nag-checkout.
* Bag-o: Cross-network magic link SSO — ang mga kustomer makasulod sa sub-networks nga walay kaulaw (seamlessly) pinaagi sa magic-link authentication.
* Bag-o: Network media cloning — tama nga gikopya ang media sa site kung mag-clone og networks gikan sa usa ka template.
* Giayo: Ang network cloning karon molihok bisan wala ang multi-tenancy addon; ibalik ang super admin privileges pagkahuman sa clone.
* Giayo: Ang paghimo og network karon compatible na sa WooCommerce ug tanang payment gateways.
* Giayo: Ang port number karon apil na sa fallback domain generation.
* Giayo: Gipadala ang Template picker CSS ngadto sa usa ka enqueued stylesheet aron husto kining ma-render sa tanan nga themes.
* Giayo: Nagpanalipod batok sa undefined `network_id` key sa site query scope.
* Giayo: Ang pag-check sa network permission sa dili pa ang context switching makapugong sa mga error sa property sa `wpdb` table.
