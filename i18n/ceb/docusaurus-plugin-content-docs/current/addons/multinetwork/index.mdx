# Multi-Network {#multi-network}

Ang Ultimate Multisite: Multinetwork addon nagpalapad sa imong Ultimate Multisite platform pinaagi sa gamhanang functionality sa multinetwork. Kini nga addon nagtugot sa imong mga kustomer nga makapalit ug mo-manage og tibuok WordPress multisite networks, dili lang ang matag usa ka site.

## Key Features {#key-features}

* **Network Provisioning**: Awtomatikong pagmugna ug pag-configure sa kompleto nga WordPress multisite networks alang sa imong mga kustomer
* **Network Templates**: Tugotan ang mga kustomer nga mag-browse, makakita og preview, ug mopili og prebuilt network template sa dili pa sila mo-signup
* **Network Management**: Kompleto nga kakayahan sa pagdumala sa multinetwork environments
* **Isolated Networks**: Ang matag customer network naglihok nga independente, uban ang kaugalingon nilang database tables ug configuration
* **Cross-Network SSO**: Paghatag og seamless access sa mga kustomer ngadto sa ilang sub-networks gamit ang magic-link authentication
* **Scalable Architecture**: Maayo pagdumala sa daghang networks aron episyente ang paggamit sa resources
* **Integration**: Seamless nga koneksyon sa subscription ug billing systems sa Ultimate Multisite

## Use Cases {#use-cases}

* **White Label Hosting**: Paghatag sa mga kustomer og ilang kaugalingong branded multisite networks
* **Agency Solutions**: Pagtugot sa mga ahensya sa pagdumala og daghang client networks gikan sa usa ka platform
* **Enterprise Deployments**: Pagsuporta sa dagkong organisasyon nga adunay komplikadong multi-network requirements
* **Education Platforms**: Pagmugna og lahi nga networks alang sa nagkalain-laing eskwelahan o departamento

## Requirements {#requirements}

* Ultimate Multisite 2.0.0 or higher
* WordPress Multisite installation
* PHP 7.4 or higher
* MySQL 5.6 or higher

## Installation {#installation}

1. I-upload ang mga plugin files ngadto sa `/wp-content/plugins/ultimate-multisite-multinetwork` directory
2. I-activate ang plugin pinaagi sa 'Plugins' screen sa WordPress
3. I-configure ang multinetwork settings pinaagi sa Ultimate Multisite → Settings

## Network Templates {#network-templates}

Ang Version 1.3.0 nagdugang og template discovery ug selection flow alang sa mga kustomer nga mopalit og bag-ong networks. Gamita ang network templates kung gusto nimo nga ang matag bag-ong customer network magsugod gikan sa usa ka giandam nang structure, design, content library, o vertical-specific setup.

### Template previewer {#template-previewer}

Ang network template previewer nagtugot sa mga kustomer nga mag-browse og available templates sa dili pa sila mo-checkout. Ang matag template mahimong ablihan sa live preview panel, aron makatandi ang mga kustomer sa layout ug content sa wala pa nila mopalit og network.

Gamita ang previewer aron tabangan ang mga kustomer:

* Pagreview sa hitsura ug structure sa matag available nga network template
* Pagtandi sa daghang templates nga dili kinahanglan i-biya sa purchase flow
* Pagpili sa template nga labing haom sa ilang use case sa wala pa sila mo-signup

Ipadayon ang mga ngalan ug deskripsyon sa template aron dali sabton sa kustomer. Mas mapuslanon ang previewer kung ang mga kustomer makasabot sa gitumong nga paggamit, sama sa agency networks, school networks, franchise networks, o white-label hosting bundles.

### Template picker at checkout {#template-picker-at-checkout}

Ang template picker mahimong idugang sa signup checkout flow isip usa ka field. Kung ma-configure, ang mga kustomer mopili og network template nga ilang gusto panahon sa pag-checkout, ug ang napili nga template mao ang gamiton kung ang ilang bag-ong network ma-provision.

Aron magamit ang picker sa checkout flow:

1. Pag-andam sa mga networks nga gusto nimong i-offer isip templates.
2. I-enable ang Multinetwork template selection field sa relevant signup o checkout form.
3. Pagpili kung unsang templates ang kinahanglan nga available para nianang checkout flow.
4. I-publish ang checkout flow ug mag-test og pag-signup aron makumpirma nga ang napili nga template maapil sa bag-ong network.

Kung ang usa ka checkout flow nag-offer lang og usa ka network product, mapuslanon gihapon ang picker isip usa ka giya nga onboarding step. Para sa multi-plan funnels, ipakita lang ang mga templates nga compatible sa napili nga produkto o plan.

## Cross-Network Magic Link SSO {#cross-network-magic-link-sso}

Ang cross-network magic link SSO nagtugot sa mga kustomer nga molihok tali sa ilang primary account area ug sub-networks nga dili na kinahanglan mag-sign in pag-usab mano-mano. Kung ang usa ka kustomer mo-follow og cross-network access link, ang Ultimate Multisite makamugna og time-limited magic link ug ma-authenticate ang kustomer ngadto sa destinasyon network.

Kini nagpaayo sa experience alang sa mga kustomer nga nagdumala og labaw pa sa usa ka network o nanginahanglan og pag-access dayon sa bag-ong gi-clone template networks human mopalit.

Admin checklist:

* Kumpirmaha nga ang Multinetwork addon active sa tibuok WordPress multisite installation.
* Siguroha nga ang mga kustomer adunay saktong network access sa dili pa sila mohatag og cross-network links.
* Ipadayon ang pagka-configure sa network domains ug mapped domains aron ang magic links mag-resolve ngadto sa gitumong nga destinasyon network.
* I-test ang customer dashboard ug bisan unsang post-checkout access buttons human nipausab og network, domain, o checkout settings.

Ang mga magic links kinahanglan itrato isip login links. Gipalami kini alang sa seamless customer access ug dili kini dapat i-forward o gamiton pag-usab gawas sa gitumong nga session.

## Frequently Asked Questions {#frequently-asked-questions}

### Unsa ang multinetwork? {#what-is-multinetwork}

Ang Multinetwork usa ka feature sa WordPress nga nagtugot kanimo sa pagpadagan og daghang independente nga multisite networks sa usa ka single WordPress installation.

### Giunsa kini kalainan gikan sa regular nga multisite? {#how-is-this-different-from-regular-multisite}

Ang regular nga multisite nagtugot og daghang sites sulod sa usa ka network. Ang Multinetwork nagtugot og daghang networks, ang matag usa adunay kaugalingon nilang sites, tanan sulod sa usa ka WordPress installation.

### Nagkinahanglan ba kini og espesyal nga server configuration? {#does-this-require-special-server-configuration}

Oo, ang multinetwork nagkinahanglan og specific nga server setup ug configuration. Palihog konsultaha ang dokumentasyon alang sa detalyado nga mga instruksyon sa pag-setup.
