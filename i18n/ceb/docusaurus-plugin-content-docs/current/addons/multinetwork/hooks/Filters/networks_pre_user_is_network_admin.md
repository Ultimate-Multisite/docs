# Pagsala: networks_pre_user_is_network_admin {#filter-networkspreuserisnetworkadmin}

Gisala ang mga network diin ang user mao ang administrador, aron mapamub-an ang proseso.

## Mga Parameter {#parameters}

| Ngalan | Matang | Deskripsyon |
|------|------|-------------|
| $ | `array\|bool\|null` | Lista sa mga network ID o false. Bisan unsa gawas sa null mopamubo sa proseso. |
| $ | `int` | User ID diin ang mga network kinahanglan ibalik. |
