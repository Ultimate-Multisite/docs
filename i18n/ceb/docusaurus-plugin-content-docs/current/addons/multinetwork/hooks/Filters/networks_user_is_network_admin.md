| Ngalan | Matang | Deskripsyon |
|------|------|-------------|
| $ | `array\|bool` | Lista sa mga ID sa network o false kung walay mga network alang sa tiggamit. |
| $ | `int` | ID sa tiggamit diin ang mga network kinahanglang ibalik. |
