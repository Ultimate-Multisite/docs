# Action: move_site

Moandar human ang site mabalhin ngadto sa bag-ong network.

## Mga Parameter

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | ID sa site nga nabalhin. |
| $old_network_id | `int` | ID sa orihinal nga network alang sa site. |
| $new_network_id | `int` | ID sa network diin gibalhin ang site. |

### Sukad

- 1.3.0
