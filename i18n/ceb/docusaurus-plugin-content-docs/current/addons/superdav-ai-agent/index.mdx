# Superdav AI Agent

Ang Superdav AI Agent usa ka abanteng WordPress AI assistant nga nagpalapad sa Gratis AI Agent gamit ang mga kahanas nga nakasabot sa tema, gipalambo nga mga kapabilidad sa pagpangita sa internet, mga bahin sa paghimo og site, ug opsyonal nga mga integrasyon sa awtomasyon. Ang bersyon 1.18.0 nagdugang og mga himan sa pagbasa sa Google Calendar, pagmapa sa kontak ug tigtambong, mga ganghaan sa pag-apruba sa tawo, pagwagtang sa doble nga mga pahinumdom, TextBee SMS notifications, pinadumala nga Superdav service provisioning, ug usa ka bulag nga giapod-apod nga Advanced companion package para sa kasaligang mga workflow sa developer.

## Pangunang mga Bahin {#key-features}

- **Mga kahanas nga nakasabot sa tema** — awtomatikong mopahiangay sa giya ug mga kapabilidad base sa imong aktibong WordPress tema ug na-install nga mga plugin
- **Daghang tagapanghatag sa pagpangita sa internet** — pagpili tali sa Brave Search ug Tavily para sa mas dato nga mga resulta sa pagpangita ug abanteng mga kapabilidad sa panukiduki
- **Mga contact form sa site builder** — paghimo og bug-os nga magamit nga mga contact form direkta gikan sa chat interface gamit ang imong na-install nga form plugin
- **Gipalambo nga site builder** — gipaayo nga daghang-yugto nga mga plano sa paghimo og site nga adunay mas maayong pagdiskobre sa plugin ug pagbawi gikan sa sayop
- **Built-in nga mga kahanas sa tema** — espesyal nga giya para sa Block Themes, Classic Themes, Kadence Blocks, ug Kadence Theme
- **Gipaayo nga WooCommerce integration** — gi-refactor aron gamiton ang native nga WooCommerce APIs para sa mas maayong kasaligan ug compatibility
- **Awtomatikong pag-ila sa provider** — awtomatikong makaila ug mo-refresh sa anaa nga mga plugin ug mga tema kung kini gi-activate o gi-deactivate
- **Custom nga paghimo og logo** — paghimo ug pag-embed og custom nga SVG logos nga adunay awtomatikong namespace-safe sanitisation
- **Pag-validate sa contrast sa palette** — susihon ang mga pares sa kolor para sa pagsunod sa WCAG accessibility sa dili pa i-apply sa imong tema
- **Photo-enhanced discovery** — pag-upload og mga litrato atol sa discovery interview sa Theme Builder aron maimpluwensiyahan ang direksyon sa disenyo
- **Mga menu sa hospitality** — paghimo og structured nga mga panid sa menu sa pagkaon ug ilimnon para sa mga restawran, cafe, ug mga negosyo sa hospitality
- **Mga preview sa desktop ug mobile** — i-preview ang imong disenyo sa desktop ug mobile nga mga device atol sa pagpili sa direksyon sa disenyo
- **Tier 1 nga site scraping** — sd-ai-agent/site-scrape anaa na karon isip default para sa pag-analisar sa mga website sa kakompetensya ug inspirasyon
- **Mga himan sa pagbasa sa Google Calendar** — pagbasa sa na-configure nga mga kalendaryo ug mga event aron ang mga awtomasyon makapangatarungan bahin sa umaabot nga konteksto sa iskedyul
- **Pagmapa sa kontak ug tigtambong** — ipares ang mga partisipante sa kalendaryo sa nailhang mga user ug kontak sa site sa dili pa maghimo og follow-up nga trabaho
- **Mga ganghaan sa pag-apruba sa tawo** — ipahunong ang sensitibong mga awtomasyon para sa pagribyu sa dili pa magpadala og mga mensahe, mopadayon sa trabaho, o mohimo og mga kausaban
- **Mga rekord sa pahinumdom ug pagwagtang sa doble** — subaya ang mga pahinumdom nga napadala na aron ang balik-balik nga pagpadagan sa awtomasyon dili magpahibalo sa parehas nga tigdawat kaduha
- **TextBee SMS notifications** — magpadala og na-configure nga text-message notifications pinaagi sa TextBee kung gi-enable ang usa ka SMS provider
- **Pinadumala nga Superdav service provisioning** — ikonektar ang suportadong mga site sa hosted nga Superdav endpoints gamit ang awtomatikong setup imbes nga manwal nga endpoint configuration
- **Advanced companion package** — bulag nga giapod-apod nga mga himan sa kasaligang developer para sa filesystem, database, WP-CLI, REST dispatcher, plugin builder, git snapshot, user-management, ug benchmark workflows

## Mga Kinahanglanon {#requirements}

- WordPress 6.0 o mas taas
- PHP 8.1 o mas taas
- WordPress Multisite (network-activated) o single-site nga installation
- Usa ka aktibong AI provider connector plugin (Anthropic, OpenAI, o compatible)
- Gratis AI Agent v1.9.0 o mas taas (ang Superdav AI Agent nagpalapad sa Gratis AI Agent)

## Pag-install {#installation}

1. I-upload ang mga file sa plugin ngadto sa `/wp-content/plugins/superdav-ai-agent`
2. I-activate ang plugin pinaagi sa **Plugins** screen sa WordPress
3. Siguroa nga ang Gratis AI Agent gi-activate usab
4. Adto sa **Gratis AI Agent → Settings** ug i-configure ang imong internet search provider, pinadumala nga Superdav service connection, Google Calendar, o TextBee credentials sumala sa gikinahanglan

Ang WordPress.org package naglangkob sa mga core nga bahin sa Superdav AI Agent. Ang Advanced companion package gitukod ug giapod-apod nga bulag para sa mga environment diin ang kasaligang developer tools hayag nga gi-install ug gi-awtorisar.

## Mga Tagapanghatag sa Pagpangita sa Internet {#internet-search-providers}

Ang Superdav AI Agent v1.10.0 nagdugang sa **Tavily** isip alternatibo sa Brave Search para sa mga pangutana sa pagpangita sa internet. Ang duha ka provider nagtanyag og real-time nga mga resulta sa pagpangita ug mahimong i-configure sa **Gratis AI Agent → Settings → Advanced**.

| Provider | Mga Feature | Setup |
|---|---|---|
| **Brave Search** | Paspas, nakapokus sa pribasiya nga mga resulta sa pagpangita | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Mas daghang resulta sa pagpangita, abanteng mga kapabilidad sa panukiduki | [Tavily API](https://tavily.com/) |

Tan-awa ang [Internet Search](../../user-guide/configuration/internet-search) para sa mga instruksiyon sa configuration.

## Mga Workflow sa Kalendaryo, Pag-apruba, ug Abiso {#calendar-approval-and-notification-workflows}

Ang Superdav AI Agent v1.18.0 nagpalapad sa mga workflow sa automation lapas sa mga buluhaton sa sulod ug paghimo og site:

- Ang **Google Calendar tools** nagtugot sa agent nga mobasa sa naka-configure nga mga kalendaryo ug mga detalye sa event kung anaa ang mga credential sa kalendaryo.

- Ang **mga gate sa pag-apruba sa tawo** nagtugot sa automation nga mohunong hangtod ang usa ka awtorisadong user makarebyu sa gisugyot nga aksiyon.
- Ang **mga rekord sa pahinumdom** mopugong sa doble nga mga pahinumdom sa kalendaryo o abiso kung ang automation sulayan pag-usab o balik-balik nga i-iskedyul.

Gamita kini nga mga himan para sa tabang nga nahibalo sa eskedyul, follow-up sa attendee, ug mga workflow sa pahinumdom nga nagkinahanglan gihapon og rebyu sa tawo sa dili pa ipadala ang sensitibong mga mensahe.

## Configuration {#configuration}

Ang tanang setting naa sa ilawom sa **Gratis AI Agent → Settings** sa WordPress admin. Ang Superdav AI Agent nagdugang sa mosunod nga mga opsiyon sa configuration:

| Setting | Deskripsiyon |
|---|---|
| **Internet Search Provider** | Pagpili tali sa Brave Search ug Tavily (opsiyonal) |
| **Tavily API Key** | API key para sa Tavily internet search (kung gipili ang Tavily) |
| **Managed Superdav Service** | Ikonektar ang gisuportahang mga site ngadto sa hosted Superdav service nga adunay awtomatikong endpoint provisioning |
| **Google Calendar Credentials** | I-configure ang access sa kalendaryo aron makabasa ang agent sa mga kalendaryo ug event |
| **TextBee SMS Provider** | I-configure ang TextBee credentials para sa giaprobahang mga workflow sa SMS notification |

## Abanteng Companion Package {#advanced-companion-package}

Ang Superdav AI Agent v1.18.0 nagdugang og gilain nga gipang-apod-apod nga Advanced companion package para sa kasaligang mga workflow sa developer. Kini nga mga kapabilidad wala gi-bundle sa WordPress.org core package. I-install ang companion package lamang sa mga site diin ang mga administrator hayagang misalig sa agent nga maka-access sa developer tooling.

Ang Advanced tools naglakip sa mga operasyon sa filesystem, access sa database, WP-CLI, REST dispatcher workflows, plugin builder utilities, git snapshots, user-management helpers, ug benchmark tooling. Ibutang kini nga mga himan nga limitado lamang sa kasaligang mga administrator ug rebyuha ang approval-gate configuration sa dili pa kini i-enable sa production nga mga site.
