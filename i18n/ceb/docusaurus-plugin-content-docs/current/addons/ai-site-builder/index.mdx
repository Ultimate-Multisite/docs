# AI Site Builder

ultimate-multisite-ai-site-builder

## Mga Kinahanglanon (Requirements)

- WordPress 5.3 o mas taas pa
- PHP 7.4 o mas taas pa
- Multisite Ultimate plugin (aktibo)

## Pag-instalar (Installation)

1. I-upload ang mga addon files sa imong `/wp-content/plugins/` directory
2. I-activate ang plugin pinaagi sa 'Plugins' menu sa WordPress
3. I-configure ang settings sa addon didto sa network admin

## Pagpalambo (Development)

### Setup

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Pag-test (Testing)

Ang addon naglakip og kumpleto nga testing framework:

- **Unit Tests**: Mga test base sa PHPUnit uban ang WordPress test suite
- **Base Test Class**: Naghatag og helper methods para sa kasagarang mga scenario sa pag-test
- **Test Coverage**: Gi-set up aron masubay ang code coverage
- **Multisite Testing**: Ang mga test moandar sa multisite environment

### Mga Pamatarungan sa Code (Code Standards)

- **WordPress Coding Standards**: Gipahamtangan pinaagi sa PHPCS
- **PHP 7.4+ Compatibility**: Gisuportahan ang modernong features sa PHP
- **Static Analysis**: Integrasyon sa PHPStan para sa type checking
- **Code Modernization**: Mga rules sa Rector alang sa pag-upgrade sa PHP

### Build System

- **Asset Processing**: Minification para sa CSS/JS files
- **Translation**: Paghimo og POT file
- **Packaging**: Pagmugna og archive para sa pag-distribute
- **Development Mode**: Walay minified assets para sa debugging

## Configuration (Pag-configure)

Gisuportahan sa addon ang nagkalain-laing configuration options nga ma-access pinaagi sa network admin interface.

## Hooks ug Filters

### Actions

- `ultimate-multisite-ai-site-builder_init` - Ma-trigger pagkahuman ma-initialize ang addon
- `ultimate-multisite-ai-site-builder_loaded` - Ma-trigger pagkahuman ma-load tanan nga files sa addon

### Filters

- `ultimate-multisite-ai-site-builder_settings` - Pag-modify sa settings sa addon
- `ultimate-multisite-ai-site-builder_enabled` - Pag-override sa status kung ang addon naka-enable o disabled
