# Filter: wu_apc_should_display_admin_menu

Tugoti ang mga admin sa pagtago sa mga pahina sa admin menu, kon nahuman na nila ang proseso sa pag-customize.

## Mga Parameter {#parameters}

| Ngalan | Type | Deskripsyon |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Kon kinahanglan ba natong ipakita ang mga pahina. |
| $network_active | `bool` | Susiha kon ang plugin ba aktibo sa network admin. |
