# Site Exporter

## Content {#content}

Nakasabot mi sa panginahanglan nga makalihok o makabalhin og mga subsites sulod sa imong network ngadto sa usa ka single site installation, ug vice versa kung ang imong kustomer magdesisyon nga i-migrate ang ilang website. Ang Site exporter and importer add-on nagpalalom niining buluhaton. Kini nga tool nag-export ug nag-import og mga sites gamit ang zip files, nga mahimong magsilbing backup usab.

## Features {#features}

* I-export ug I-import og Sites

* Sayon gamiton nga interface

* Paghimo og site backup

## Documentation {#documentation}
**Giunsa pag-Export ug Import sa usa ka Site**

## Installation {#installation}

1. I-upload ang **ultimate-multisite-site-exporter** ngadto sa direktoryo nga **/wp-content/plugins/**
2. I-activate ang plugin pinaagi sa **Plugins** menu sa WordPress
3. Sunda ang step by step Wizard aron i-set up ang plugin

Uban ni Ultimate Multisite, mahimo nimong i-export ang usa ka site gikan sa imong network o i-import ang usa ka site ngadto sa imong multisite o single-site installation.

## Giunsa pag-export og usa ka site {#how-to-export-a-site}

Aron i-export ang usa ka site gikan sa imong network, adto sa **Export & Import** sa Ultimate Multisite menu ug, sa **Export Sites** tab, i-klik ang **Generate new Site Export**.

<!-- Screenshot unavailable: Export Sites tab with Generate new Site Export button -->

Mangita og site nga gusto nimong i-export ug pilia ang mga butang nga gusto nimo nga ilakip sa gi-export nga file. Mahimo kang maglakip og themes, plugins, ug uploads. Kung dako kaayo ang imong site, mahimo kang mopili nga ipadayon ang export process sa background.<!-- Screenshot unavailable: Site export options with site search, include themes/plugins/uploads checkboxes -->

I-klik ang **Export Site** button. Ang imong gi-export nga site makita sa **Existing Exports** list. I-klik ang **Download ZIP** button aron ma-download ang file. Ang Export ZIPs naglakip og self-booting `index.php` ug `readme.txt`, busa mahimo kining i-install sa usa ka bag-ong host nga wala pa makainstall og separate importer plugin.<!-- Screenshot unavailable: Existing Exports list with Download ZIP button -->

## Giunsa pag-export sa tibuok network {#how-to-export-the-whole-network}

Aron ma-export ang tanang subsites nga magkuyog, adto sa **Export & Import** sa Ultimate Multisite menu ug gamita ang **Network Export** option sa export screen. Ang gihimo nga archive naglakip og tanang subsites sa usa ka ZIP alang sa host migrations, staging rebuilds, o network-level backups.

Ang network exports mahimong dako. Pagpadagan niini kung ubos ang traffic ug kumpirmaha nga igo pa ang storage sa server para sa archive ug bisan unsay gilakip nga uploads, plugins, ug themes.

## Giunsa pag-import og usa ka site {#how-to-import-a-site}

Aron i-import ang usa ka site, adto sa **Import a Site** tab ug i-upload ang .zip file sa site nga gusto nimong i-import. Mahimo nimo usbon ang URL ngadto sa bag-o o mapadayon ang daan nga URL pinaagi sa pagbiya sa space blangko. Makasusi ka usab sa box aron ma-delete ang .zip file gikan sa imong installation human sa import. Dayon i-klik ang **Begin Import**.

<!-- Screenshot unavailable: Import a Site tab with zip upload field and Begin Import button -->

Ang proseso mahimong mokuha og gamay nga panahon. Makita nimo ang usa ka pending import sa listahan ubos sa importer.

<!-- Screenshot unavailable: Pending import in the list below the importer -->

Mahimo nimong hunongon ang proseso pinaagi sa pag-klik sa **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Pagkahuman sa proseso, andam na ang imong site sa imong network.

## Giunsa pag-import og usa ka site sa single install {#how-to-import-a-site-on-a-single-install}

Aron i-import ang usa ka site ngadto sa imong single install, una i-install ug i-activate ang importer sa imong site. Makita nimo ang link aron ma-download kini sa exporter tab.

<!-- Screenshot unavailable: Exporter tab with link to download single-site importer -->

Makita nimo ang importer sa **Tools** menu, sa wala, sa subitem nga **Export & Import**.

<!-- Screenshot unavailable: Single-site Tools menu showing Export and Import submenu -->

I-upload ang .zip file sa site nga gusto nimong i-import. Mahimo nimo usbon ang URL ngadto sa bag-o o mapadayon ang daan nga URL pinaagi sa pagbiya sa space blangko. Makasusi ka usab sa box aron ma-delete ang .zip file gikan sa imong installation human sa import. Dayon i-klik ang **Begin Import**.

<!-- Screenshot unavailable: Single-site import form with zip upload and Begin Import button -->

Ang proseso mahimong mokuha og gamay nga panahon. Makita nimo ang usa ka pending import sa listahan ubos sa importer.

<!-- Screenshot unavailable: Single-site pending import in the list -->

Mahimo nimong hunongon ang proseso pinaagi sa pag-klik sa **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Pagkahuman sa proseso, mapuli ang imong site sa site nga imong gi-import.
