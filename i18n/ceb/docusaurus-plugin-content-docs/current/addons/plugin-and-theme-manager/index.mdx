# Plugin & Theme Manager {#plugin--theme-manager}

Ultimate Multisite: Plugin and Theme Manager

I-edit o itago ang metadata (title, author, thumbnail, ug description) sa mga na-install nga plugins ug themes. Makahimo ka og mga kategorya para sa mga plugin ug theme aron mahimo sa imong mga user nga ma-filter kini, ug makadugang og nindot nga custom Plugins page sa panel sa imong mga user.

Aron gamiton ang plugin, pwede lang kang moadto sa imong Plugins o Themes pages sa network admin ug i-klik ang "Edit Plugin/Theme info".

## Installation {#installation}

1. I-upload ang 'ultimate-multisite-plugin-and-theme-manager' ngadto sa direktoryo nga '/wp-content/plugins/'
2. I-activate ang plugin pinaagi sa 'Plugins' menu sa imong WordPress Network Admin

## Usage Guide {#usage-guide}

Ang Ultimate Multisite: Plugin and Theme Manager usa ka gamhanang **premium** add-on para sa Ultimate Multisite nga naghatag kanimo og kompleto nga kontrol kon giunsa pagtan-aw ug paggamit sa mga network user sa plugin ug theme options sa imong network.

Kini nga add-on adunay tulo ka katuyoan:

  1. Gitugotan ka niini nga hingpit nga usbon ang metadata sa plugins ug themes. Naglakip kini sa ngalan, description, pangalan sa author, ug bisan pa sa mga thumbnail!

  2. Makatabang kini kanimo pagkategorya sa mga plugin ug theme, aron dali kining ma-sort sa imong mga user.

  3. Naghatag kini kanimo og kapilian nga ipuli ang daan, makapangutana nga plugins page og usa ka bag-ong pages nga sama sa themes—nga adunay pag-sort base sa kategorya.

Pagkahuman nimo kining i-download ug i-install, motan-aw og bag-ong tab—_Add-on Settings_—sa Ultimate Multisite Settings page.
