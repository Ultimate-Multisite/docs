# Material WP

Ang WordPress admin interface pareho na kaayo sulod sa daghang tuig. Pero naghunahuna ka na ba kung unsaon pagtan-aw ang usa ka Material Design nga bersyon sa admin interface?

Ang Material WP usa ka admin theme nga gihimo aron ihatag kanimo kana nga hitsura ug pagbati, makatabang nimo sa paghatag og mas personalized nga Dashboard sa imong mga kliyente.

## Features

- Suporta sa Custom CSS
- Sayon gamiton nga setting sa estilo (styling)
- Pag-usab sa pagka-order sa menu
- Sayon nga pag-import/export

## Documentation

**Pagpa-customize sa Imong Admin Panel Gamit ang Material WP**

## Installation

1. I-upload ang 'material-wp' ngadto sa direktoryo nga '/wp-content/plugins/'
2. I-activate ang plugin pinaagi sa 'Plugins' menu sa WordPress

## Usage Guide

Ang WordPress admin interface pareho na kaayo sulod sa daghang tuig. Pero naghunahuna ka na ba kung unsaon pagtan-aw ang usa ka Material Design nga bersyon sa admin interface?

Pagkahuman nimo ni **i-install**, adto sa imong _Plugins_ page ug i-klik aron ma-activate kini sa imong Network.
