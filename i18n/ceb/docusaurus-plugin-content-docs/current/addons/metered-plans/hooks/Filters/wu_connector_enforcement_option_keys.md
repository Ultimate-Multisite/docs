# Filter: wu_connector_enforcement_option_keys

Salaa ang lista sa mga option key sa AI provider nga gipatuman gikan sa main site.

Pagdugang og mga option key alang sa custom o third-party nga AI provider plugins aron ang ilang mga setting mapanunod usab gikan sa main site sa mga subsite.

## Mga Parameter

| Ngalan | Type | Deskripsyon |
|------|------|-------------|
| $keys | `string[]` | Mga ngalan sa option key nga ipatuman (naglakip na sa dinamikong nadiskobrehan nga mga connector key ug EXTRA_PROVIDER_OPTIONS). |
