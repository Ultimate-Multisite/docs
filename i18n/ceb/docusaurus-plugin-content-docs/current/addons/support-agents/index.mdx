# Support Agents

Ang pag-manage sa tibuok network by sarili nimo dili sayon nga buluhaton. Samtang nagtubo ang imong network, magsugod kang makakita og dako nga panginahanglan para sa suporta. Maayo nga magdala ka og mga tawo nga motabang kanimo aron mapagaan ang imong trabaho. Apan, ang paghatag og bug-os nga access/permission kay risgoso ug mahimong makompromiso sa imong network. Mao na diin moagi kining Support Agent add-on.

Kini nga add-on nagtugot kanimo sa paghimo og bag-o nga klase sa WordPress user nga gitawag og Support Agent, ug gihatag ni nimo ang bug-os nga kontrol sa unsa nga mga permission ang naa anang user. Ang paghatag og permiso kay sayon ra sama sa pag-on o pag-off sa toggle button kung magdesisyon kang tangtang ni kini.

## Mga Tampo (Features) {#features}

- Detalyado nga listahan sa WordPress & Ultimate Multisite permission
- Pag-enable ug pag-disable sa access sa dali nga paagi
- Paghimo og walay limit nga kantidad sa Agent accounts

## Instalasyon (Installation) {#installation}

1. I-upload ang 'ultimate-multisite-support-agents' sa directory nga '/wp-content/plugins/'
2. I-activate ang plugin pinaagi sa 'Plugins' menu sa WordPress
3. Sunda ang step-by-step Wizard aron ma-set up ang plugin
