# Loco Translate {#loco-translate}

Ang Ultimate Multisite: Loco Translate Integration addon nagdala og mga kapabilidad sa paghubad nga piho sa site ngadto sa imong WordPress multisite network. Kung kaniadto kinahanglan nimo nga ang lain-laing site sa imong network adunay lahi nga hubad alang sa parehas nga plugin o theme, kini nga addon mao ang perpekto nga solusyon.

**Unsay Buhaton Niining Addon**

Kini nga addon nagkonektar pagkahiusa (seamlessly) sa Ultimate Multisite uban sa sikat nga Loco Translate plugin, nga nagtugot sa matag subsite sa imong network nga magpabilin sa ilang kaugalingong custom translations. Bisan pa kung nagpadagan ka og multi-regional platform, nagserbisyo og lain-laing komunidad sa lenggwahe, o kinahanglan lang nimo ang flexibility sa paghubad sa tibuok network, kini nga addon makapadali niini.

**Perpekto Para Kan:**

* Mga negosyo nga naglihok sa daghang rehiyon ug adunay lain-laing panginahanglan sa localization
* Mga institusyong edukasyonal nga nagserbisyo og nagkalain-laing komunidad
* Mga SaaS platform nga nagtanyag og multilingual support
* Mga web agency nga nagdumala og daghang site sa kustomer
* Bisan unsang multisite network nga nanginahanglan og flexibility sa paghubad

## Pag-instalar (Installation) {#installation}

1. Siguroha nga ang Ultimate Multisite gi-install ug na-activate na ang network
2. I-install ug i-activate ang Loco Translate plugin
3. I-upload kining addon ngadto sa imong plugins directory
4. Network activate ang Loco Translate Integration addon
5. Sugdi paghimo og site-specific translations pinaagi sa Loco Translate sa bisan unsang subsite

Mao ra na! Ang integration awtomatikong molihok nga walay dugang nga configuration.

## Kasagarang Pangutana (Frequently Asked Questions) {#frequently-asked-questions}

## Kinahanglan ba nako mag-configure pa pagkahuman i-install? {#do-i-need-to-configure-anything-after-installation}

Dili! Ang pag-integrate kay awtomatiko. Sa higayon nga active na ang addon ug ang Loco Translate, ang mga subsite makasugod dayon sa paghimo sa ilang kaugalingong custom translations.

## Unsay mahitabo kung ang usa ka subsite dili makahimo og custom translations? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Ang site awtomatikong mogamit sa network-wide translations isip fallback (alternatibo). Walay babagsak, ug tanan molihok nga hapsay.

## Makaka-access ba ang mga subsite sa network-wide translations? {#can-subsites-access-network-wide-translations}

Ang regular site admins makakita ug makabag-o lang sa translations sa ilang kaugalingong site. Ang Super admins mahimong mo-access sa network-wide ug site-specific translations depende kung unsang site ang ilang gitan-aw.

## Naglihok ba kini sa mga plugin ug themes? {#does-this-work-with-both-plugins-and-themes}

Oo! Ang site-specific translations molihok alang sa WordPress plugins ug themes.

## Makapahinay ba niini ang akong multisite network? {#will-this-slow-down-my-multisite-network}

Dili. Ang addon naggamit og episyente nga pag-load sa file nga magsususi lang sa site-specific translations kung kinahanglanon ra kini. Walay mapansarang epekto sa performance.

## Mahimo ba nako kopyahon ang network translations ngadto sa usa ka piho nga site isip sinugdanan? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Oo! Samtang nag-edit og translations sa Loco Translate, dali kang magsugod gamit ang network translations ug i-customize kini alang sa matag site.

## Naglihok ba kini uban sa WooCommerce ug ubang plugins? {#does-this-work-with-woocommerce-and-other-plugins}

Sigurado! Bisan unsang plugin o theme nga naggamit og standard WordPress translation functions molihok uban sa site-specific translations.
