# Language Selector

Multisite Ultimate: Language Selector sa Sign-up

Magdugang og language selector field sa sign-up flow.

## Installation (Pag-instalar) {#installation}

1. I-upload ang 'ultimate-multisite-language-selector' ngadto sa '/wp-content/plugins/' directory
2. I-activate ang plugin pinaagi sa 'Plugins' menu sa imong WordPress Network Admin

## Usage Guide (Giya sa Paggamit) {#usage-guide}

Niining libreng add-on, mahimo nimo tugotan ang imong mga user nga mopili unsang lenggwahe ang ilang gusto makita sa sign-up flow ug sa login page.

Human [pag-activate sa imong license](../../user-guide/getting-started/installing-ultimate-multisite.md), makahimo ka na og i-install ang Ultimate Multisite add-ons nga gihatag sa imong license.

Pangadto sa imong Ultimate Multisite settings page ug dayon, sa menu sa wala, i-klik ang _Add-ons_ label.

![Add-ons page](/img/admin/addons-page.png)

_Pangadto sa Add-ons page aron magsugod og i-install ang Language Selector_

Didto nimo makita tanan namong gi-offer nga mga add-on.

Ang free add-ons available sa tanan namong kustomer, bisag unsay subscription plan ang ilang gipili.

I-klik ang _Install it Now!_ button sa ilalom sa thumbnail sa add-on. Pagkahuman, i-activate kini sa Plugins page.

![Add-ons page with install button](/img/admin/addons-page.png)

_I-klik ang button aron i-install ang add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_I-activate kini sa imong Plugins page_

Karon, pangadto sa Ultimate Multisite's Settings menu ug suldi ang _Network Settings_ tab. Didto, sa ubos sa page, makakita ka og bag-ong seksyon nga gitawag og _Languages_.

![Settings page](/img/config/settings-general.png)

_Naghitsit ang usa ka bag-ong seksyon!_

Mahimo nimo kining biyaan blank aron ipakita tanang lenggwahe isip mga kapilian, o makapili unsang gusto nimong himuong available. Ang mga lenggwahe mao kadtong na-install nimo sa imong WordPress set up.

_List of languages available on this WP install_

Pagkahuman nimo i-save ang mga pagbag-o, usa ka selection menu nga adunay mga lenggwahe nga imong gipili maipakita sa Login page ingon man sa Sign-up Flow.

## Setting the Language via the URL (Pagbutang sa Lenggwahe pinaagi sa URL) {#setting-the-language-via-the-url}

Mahimo usab nimong usbon ang lenggwahe sa usa ka pahina pinaagi sa pag-bypass sa 'locale' parameter sa URL. Kung gusto nimo mag-redirect ngadto sa register page nga Italian, pananglitan, dugangi og ?locale=it_IT sa URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Kinahanglan kining na-install sa imong WordPress network aron kini molihok.
