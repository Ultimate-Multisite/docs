# VAT

Multisite Ultimate: European VAT

Naghatag og dugang mga features ug tools sa default nga pag-handle sa tax handling sa Multisite Ultimate aron mas maayo suportahan ang European VAT.
Automated nga nagdadag og EU VAT rates sa Tax Rates page gikan sa

## Installation

1. I-upload ang 'ultimate-multisite-vat' sa directory nga '/wp-content/plugins/'
2. I-activate ang plugin pinaagi sa menu nga 'Plugins' sa WordPress
3. Padulong sa Ultimate Multisite > Settings > Taxes and Enable EU Vat.
4. Adto ug i-click ang Tax Rates ug Update EU VAT Rates sa ubos ug Save Tax Rates.
5. Tapos na!
