# Fluent Forms {#fluent-forms}

Kini nga add-on nagtugot sa pagdugang og limitasyon (limits) sa Fluent Forms sa Multisite Ultimate, ilabi na sa mga produkto, membership, o site.
Naggamit ang Fluent Forms og *custom tables* imbes nga *custom post types*, mao nga dili molihok ang kasagarang options sa paglimita (limiting options) kung gamiton kini uban ni Fluent Forms.

## Features {#features}

- Pagdugang og limitasyon alang sa Fluent Forms o Conversational Fluent Forms
- Makadawat ang mga kustomer og maayo nga mensahe nga nag-aghat kanila nga i-upgrade ang ilang plano kung moabot sila sa limitasyon
