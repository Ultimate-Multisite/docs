# Pag-customize sa imong Admin Panel gamit ang PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Kanunay nga kung nagtrabaho ta para sa mga kliyente, naa tay panginahanglan nga maghatag kanila og custom experience sa admin dashboard sa WordPress. Kung nagpangita ka og mas propesyonal nga hitsura ug pakarampara, ang PRO Theme makatabang nimo.

Ang PRO Theme usa ka admin theme nga gihimo aron motabang nimo kung mag-customize sa Dashboard sa imong mga kliyente. Ug kini parte usab sa core sa premium plugins sa Ultimate Multisite.

Human i-install kini, adto sa imong _Plugins_ page ug i-click para ma-activate kini sa imong Network.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_I-Activate kini sa tibuok network sa imong Plugins page_

_**TANDAAN:**_ _Kung i-install lang nimo, ipakita kini sa imong mga kliyente isip option nga mahimong i-mark para dugangan ang theme sa usa ka user lang._

Karon, naa na ang imong admin interface nga kompleto nga gi-redesign! Naa usab kay bag-ong menu sa left column nga gitawag og PRO Theme.
