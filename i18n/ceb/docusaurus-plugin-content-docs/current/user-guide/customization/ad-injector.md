# Pagbutang sa Ads sa imong network gamit ang Ultimate Multisite: Ad Injector

Ang Ultimate Multisite: Ad Injector usa ka **libre nga add-on** nga naghatag kanimo og mga gamit aron magbutang og ads sa imong mga plano direkta gikan sa ilang edit page.

Human sa pag-install sa Ultimate Multisite: Ad Injector, adto sa imong _Plugins_ page para i-activate kini.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_I-click ang Network Activate button_

Pagkahuman nga aktibo na ang add-on, ang _Advanced Options_ meta box sa plan edit screen adunay bag-ong option nga gitawag og _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Karon pwede nimo i-edit ang ad settings para sa matag usa sa imong mga plano_

Sa tab nga mao, mahimo nimong magdugang og ad codes alang sa posts ug pages sa front-end, sa dili pa ug human sa content.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Human sa pagbutang sa imong ad's code, kini modama sa mga posts ug pages niadto nga plano._

Mahimo usab nimong maglakip og ads sa header sa Admin Pages para sa mga user sa plano.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Adto sa Admin Panel Ad Code option kung gusto nimo ipakita ang ads sa back-end sa imong user_

### Labaw pa sa Advertising {#beyond-advertising}

Mahimo usab nimong gamiton ang Ultimate Multisite: Ad Injector aron maghatag og suporta sa high-tier plans.

Adto lang sa _Admin Panel Ads_ option ug dugangi ang HTML code para sa imong support tool.
