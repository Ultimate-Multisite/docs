# Unsaon nako pagkansela sa akong subscription?

Kung naa kay Ultimate Multisite license, pwede nimo kanselahon ang renewal bisan kanus-a gusto nimo. Sunda lang ni mga lakang:

<!-- Screenshot unavailable: Account page email with access link -->

Gamita ang email nga imong gihatag panahon sa pagpalit ug ang imong password.

<!-- Screenshot unavailable: Login page with email and password fields -->

Kini mao ang imong unang screen sa Account Page:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Aron kanselahon ang imong subscription, sa menu sa wala nga bahin, adto sa _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

I-click ang arrow sa tuig para buksan ang usa ka side window. Unya, kinahanglan nimo piliha ang option nga mosunod aron kanselahon ang **auto-renew**.

Ang sistema magpakita kanimo og confirmation message.

Kung sigurado ka nga gusto nimong kanselahon ang imong subscription, i-click lang ang _**Cancel Renewals**_ button. Human niini nga aksyon, ipangutana ka og dali nga survey.

Tapos na! Ang imong subscription dili na awtomatikong ma-renew.

**Aduna kay valid key hangtod mawala ang imong subscription**. Kung gusto nimo i-reactivate ang imong subscription, kinahanglan nimo kini buhaton manually.
