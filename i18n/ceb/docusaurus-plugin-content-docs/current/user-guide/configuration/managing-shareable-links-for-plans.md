# Pag-manage sa mga Shareable Link para sa Plans {#managing-shareable-links-for-plans}

Ang Ultimate Multisite naghatag kanimo og option nga i-share ang plan links direkta sa imong mga kliyente. Aron magbuhat niana, pumunta lang sa bisan unsang edit page sa imong plans ug i-click ang _Click to copy the Shareable Link_ button.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Sa Plan's Edit Page, makit-an nimo ang button para maghimo og shareable links_

Ablihi nga kini nga mekanismo naggamit sa default billing frequency para buhaton ang link. Nagpasabot ni nga kung ang imong default billing kay monthly, pananglitan, sa katapusan, naa kay link nga ingon niini: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Kung gusto nimo nga ang shareable link mag-default sa quarterly, kinahanglan nimong magdugang og 3 sa katapusan

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Aron mapili ang plan _premium_ sa yearly schedule, magdugang ka og 1 day ug ang pulong:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Ayaw kalimti nga dili ikaw makahigugma lang og usa ka link para sa lain-laing billing frequencies. Kinahanglan nimo i-share ang usa ka espesipikong link para sa matag frequency sa imong billing.

Pagkahuman nimong i-share ang link sa imong mga customer, kini magdadala kanila sa unang lakang sa imong signup flow.
