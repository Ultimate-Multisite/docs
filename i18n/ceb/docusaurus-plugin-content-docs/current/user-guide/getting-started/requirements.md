# Mga Kinahanglanon sa Ultimate Multisite

Ang una nga butang nga mahimong pangutana nimo sa dili pa mogasto para sa Ultimate Multisite kay unsa ang mga kinahanglanon aron ma-run nato ang plugin. Sa maayo, gamay ra gyud ang basic requirements para sa pag-run og Ultimate Multisite: usa ka **domain name**, usa ka maayong **hosting provider** ug usa ka WordPress **Multisite** installation. Mao na lang ang kinahanglan nimo!

## Domain Name

Una, atong hisgutan ang domain name.

Kung magpili ka og domain name, hunahunaa kung unsa man ang imong specific niche ug unsa ang imong himuon gamit ang Ultimate Multisite: Gusto ba nimo maghimo og agency? Magbaligya og mga website nga sama sa shop? O basin WordPress web hosting? Ang tanan mahimong buhaton uban ang Ultimate Multisite.

Ang among plugin naghatag og subscription services, pag-collect og bayad, checkout forms, discount vouchers ug customer communications. Dako kaayo ang trabaho nga kinahanglan buhaton para i-install, i-configure ug i-maintain ang WordPress Multisite, apan gihimo kini sa Ultimate Multisite.

Bisan unsa man imong niche, mahimong ma-manage gamit ang among plugin basta nagbaligya ka og mga website isip serbisyo! Siguraduhon nga mopili ka og domain name nga tugma sa imong business plan.

## Hosting Provider

Karon nga naa kay usa ka nindot nga domain name nga nagrepresentar sa imong niche ug negosyo, kinahanglan nimo magpili og maayong web hosting company aron i-host ang imong network uban sa Ultimate Multisite. Naa pipila ka maayong mga opsyon sa merkado. Ang ubang hosting provider maayo nga mag-integrate uban sa Ultimate Multisite, nga mahimong usa ka importante nga butang depende kung unsa ang imong plano nga himoon.

Ang Ultimate Multisite mahimo gamiton sa bisan unsang hosting provider. Ang nag-inusara lang nga kalainan kay ang pipila ka hosting companies sama sa _WPMU DEV Hosting_, _WP Engine_, _Closte_, _Cloudways_, _Gridpane_ ug _Runcloud_ adunay mas lig-on nga integrasyon para sa domain mapping. Para sa ubang hosting, kinahanglan nimo i-map manually ang custom domains. Maayo usab kung makakuha ka og hosting provider nga nagsuporta sa AutoSSL. Ang _Closte_, _Runcloud_, ug _cPanel_ nagsuporta niini.

Kon bahin sa server specifications, walay espesipikong pormula kay kini magdepende sa mga features ug functions nga imong i-offer ug sa gidak-on sa imong network. Ang shared hosting mahimong maayo nga pagsugod, apan samtang nagsugod ang imong network nga molambo, siguro kinahanglan nimo og dedicated VPS.

## WordPress Multisite

Karon, atong hisgutan gamay bahin sa usa ka WordPress Multisite installation.

Ang WordPress Multisite nagtugot kanimo nga maghimo og network sa mga site sulod sa usa lang ka installation. Kini usa ka built-in feature, apan dili kini aktibo sa default. Kinahanglan nimo i-enable kining feature sulod sa imong WordPress installation ug web hosting control panel.

Ang pagtabang sa WordPress nga adunay Multisite nga enabled **mao ang pundasyon sa Ultimate Multisite**. Tungod kay ang pangunang gamit sa Ultimate Multisite mao ang pagtabang sa mga user sa pagdumala ug pagbaligya og websites isip serbisyo, kini kinahanglan buhaton sa WordPress nga adunay Multisite nga enabled.

Aduna kitay **usa ka in-depth article** nga nagtudlo step by step kung unsa ang kinahanglan buhaton aron ma-enable ang Multisite feature sa imong WordPress installation.

Human nimo i-enable ang Multisite sa WordPress, panahon na para mag-install og Ultimate Multisite. Aron sa hustong pag-install sa Ultimate Multisite, kinahanglan nimo kini:

  * WordPress: v5.3+ (Recommended: Last stable version)

  * PHP: 7.4.x (support to > 8.0 coming soon)

  * MySQL: v5+ (Recommended: 5.6, in case you don't have 8.0 available on your hosting provider)

Kini ra ang mga software requirements para sa Ultimate Multisite.

Hunahunaa nga mahimong molihok ang Ultimate Multisite uban sa PHP 8.0 pero kami nagrekomenda nga gamiton kini uban sa PHP 7.4.x.

Gidawat usab namo nga i-install kini sa **main domains**, dili sa subdomains o subdirectories. Mahimong molihok kini sa mga subdomain/subdirectory, pero mahimong maghatag og pipila ka error.

Para sa usa ka detalyadong giya kung unsaon pag-install ang Ultimate Multisite, **klik diri**.
