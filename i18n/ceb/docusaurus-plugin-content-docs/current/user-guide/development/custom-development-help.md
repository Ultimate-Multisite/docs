# Unsaon Pagkuha og Custom Development Help gikan sa Codeable?

Usahay, kinahanglan gyud nato og maayo nga customization para mahimong perpekto ang among network flows sama sa gusto nato. Kung mao kana ang imong gipangita, kami nagrekomenda sa [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), usa ka lugar diin makakita ka og mga experienced developer nga adunay Ultimate Multisite expertise.

Ang [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) usa ka platform nga nagkonektar sa mga customer sa mga qualified WordPress freelance developers, nga makahimo og mga proyekto bisan unsa pa kini kadako. Aduna sila usa ka detalyado nga proseso sa aplikasyon aron masiguro nga ikaw ma-match sa mga eksperto nga espesyalista para sa trabaho nga imong gihunahuna.

### Unsaon Paghimo og Project sa Codeable? {#how-to-create-a-project-in-codeable}

Una, pumunta sa [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ug i-click ang **Start a Project** button.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Karon, aron magsumite og project, i-click sa wala nga bahin para sa **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Sa "I need help with", piliha ang **Customizing**:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Sa "My", piliha ang **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Dayon i-click ang **Continue** button.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Karon, panahon na sa paghulagway sa imong project ug panginahanglan sa pinakma nga paagi nimo. Sulayi nga hatag og kadaghan nga impormasyon!

<!-- Screenshot unavailable: Codeable project description form -->

Tubaga ang mosunod nga mga pangutana aron makatabang sa pricing.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Pagkahuman niana, kung wala pa kay [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) account, maghimo na ug usa ug i-click ang **Publish My Project** button.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Pagkahuman nimo sa pag-publish sa imong [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) project, konektahan ka sa mga developer nga haom sa skill set nga gikinahanglan para sa imong project.

Kini nga proseso makatabang nimo sa pagpangita sa saktong tawo alang sa imong trabaho, usa nga adunay kasinatian sa WordPress ug Ultimate Multisite!

Ang minimum rate sa Codeable kay $70–$120 USD kada oras. Ang mga estimasyon magbag-o depende sa scope of work, kalisod (complexity), ug urgency sa project.

Pagkahuman nimo makakuha sa imong estimate, mahimo na nimong bayran sa developer nga labing haom sa imong panginahanglan ug sugdan ang inyong project!
