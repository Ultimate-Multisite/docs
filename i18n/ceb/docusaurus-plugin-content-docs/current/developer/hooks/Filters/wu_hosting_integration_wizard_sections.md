# Filter: wu_hosting_integration_wizard_sections {#filter-wuhostingintegrationwizardsections}

Nagsala sa mga seksyon sa wizard alang sa setup sa hosting integration.

Nagtugot sa mga addon nga modugang, motangtang, o mousab sa mga seksyon sa wizard.

## Mga Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | Mga seksyon sa wizard. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Ang integration nga gi-configure. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Ang instance sa panid sa wizard. |
