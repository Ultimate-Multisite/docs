|------|------|-------------|
| $require | `bool` | Kon ipangayo ba ang espesyal nga mga karakter. Default kay true kon gipatuman ang mga lagda. |
| $defender_active | `bool` | Kon aktibo ba ang Defender Pro Strong Password. |
