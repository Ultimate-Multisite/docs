# Salain: wu_use_domain_mapping

Tinoa kung gamiton ba ang usa ka pagmapa

Kasagaran, gusto nimo nga tugotan lamang ang aktibong mga pagmapa nga gamiton. Apan, kung gusto nimo mogamit og mas abanteng lohika, o tugotan usab ang dili aktibong mga domain nga mapamapa, salain lang dinhi.
