# Filter: wu_post_count_statuses {#filter-wupostcountstatuses}

Tugoti ang mga developer sa plugin nga usbon kung unsang estado sa post ang angay ihapon. Sa default, ihapon ang mga published ug private nga post

## Mga Parameter {#parameters}

| Ngalan | Matang | Deskripsyon |
|------|------|-------------|
| $post_status | `array` | Ang lista sa mga estado sa post |
| $post_type | `string` | Ang slug sa matang sa post |

### Sukad {#since}

- 1.9.1
### Tinubdan {#source}

Gidefine sa [`inc/limitations/class-limit-post-types.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/limitations/class-limit-post-types.php#L119) sa linya 119


## Mga Gibalik {#returns}
Bag-ong array sa estado sa post
