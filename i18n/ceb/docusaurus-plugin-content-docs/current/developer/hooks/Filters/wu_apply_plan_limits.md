# Filter: wu_apply_plan_limits

Tugoti ang mga plugin developer nga i-short-circuit ang mga limitasyon.

Mahimo nimong gamiton kini nga filter aron modagan og bisan unsang code sa wala pa masugdan ang bisan unsang mga limitasyon. Kung ang imong filter mobalik og bisan unsang truthy nga value, mopadayon ang proseso; kung mobalik kini og bisan unsang falsy nga value, mobalik ang code ug walay bisan usa sa mga hook sa ubos ang modagan.
