# Pagsala: wu_password_require_number

Salaa kung kinahanglan ba ang mga numero sa mga password.

## Mga Parameter

| Ngalan | Matang | Deskripsyon |
|------|------|-------------|
| $require | `bool` | Kung kinahanglan ba ang mga numero. Default true kung gipatuman ang mga lagda. |
| $defender_active | `bool` | Kung aktibo ba ang Defender Pro Strong Password. |

### Sukad

- 2.4.0
