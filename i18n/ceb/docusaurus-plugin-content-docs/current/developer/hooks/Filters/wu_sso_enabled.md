# Salain: wu_sso_enabled {#filter-wussoenabled}

I-enable/i-disable ang kapabilidad nga cross-domain single-sign-on.

Sal-a kini nga kantidad aron hingpit nga mapalong ang single-sign-on, o kondisyonal nga i-enable hinuon kini.

## Mga Parameter {#parameters}

| Ngalan | Matang | Deskripsyon |
|------|------|-------------|
| $enabled | `bool` | Angay bang i-enable ang SSO? True para sa naka-on, murag false para sa naka-off. |

### Sukad {#since}

- 2.0.11
### Tinubdan {#source}

Gihubit sa [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) sa linya 110


## Ibalik {#returns}
Kung ang SSO naka-enable o wala.
