# Filter: wu_view_override_replaceable_views

Nagtugot sa mga developer nga makadugang ug dugang nga mga folder sa replaceable list.

Pag-amping, kay ang pagtugot sa dugang nga mga folder mahimong makapahinabo nga ma-load ang mga karaan nga kopya imbis sa mga bersyon sa Ultimate Multisite.

## Mga Parameter

| Ngalan | Tipo | Deskripsyon |
|------|------|-------------|
| $replaceable_views | `array` | Lista sa gitugot nga mga folder. |

### Sukad

- 2.0.0
### Gigikanan

Gidefine sa [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) sa linya 50
