# Filter: wu_get_site_domain_and_path

Tugoti ang mga developer nga manipulahon ang mga pares sa domain/path.

Mahimo kini nga mapuslanon alang sa daghang butang, sama sa pagpatuman og usa ka matang sa staging solution, lain-laing servers, ug uban pa.

## Mga Parameter

| Name | Type | Description |
|------|------|-------------|
| $d | `object` | Ang kasamtangang object nga adunay mga key sa domain ug path. |
| $path_or_subdomain | `string` | Ang orihinal nga path/subdomain nga gipasa sa function. |

### Sukad

- 2.0.0
### Tinubdan

Gihubit sa [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) sa linya 235


## Mga Ibalik
Usa ka object nga adunay mga key sa domain ug path.
