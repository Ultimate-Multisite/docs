| Ngalan | Type | Deskripsyon |
|------|------|-------------|
| $domain | `string` | Default nga domain nga gigamit karon, mapuslanon alang sa mga pag-usab |
| $domain_options | `array` | Lista sa tanang mga opsyon sa domain nga gisulod sa Mga Setting sa Ultimate Multisite -&gt; Mga Setting sa Network -&gt; Mga Opsyon sa Domain |
