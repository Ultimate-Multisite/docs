# Filter: wu_get_checkout_variables

Tugoti ang mga developer sa plugin sa pag-filter sa mga pre-set sa panid sa checkout.

Pag-amping, ang nawala nga mga key mahimong hingpit nga makaguba sa checkout sa front-end.

## Mga Parameter {#parameters}

| Ngalan | Type | Deskripsyon |
|------|------|-------------|
| $variables | `array` | Mga localized variable. |
| $checkout | `\Checkout` | Ang checkout class. |

### Sukad {#since}

- 2.0.0
### Tinubdan {#source}

Gihubit sa [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) sa linya 1970


## Mga Balik {#returns}
Ang bag-ong variables array.
