# Action: wu_register_integrations {#action-wuregisterintegrations}

Mopalihok aron tugotan ang mga integration nga magparehistro sa ilang kaugalingon.

Ang kinauyokan nagparehistro sa iyang mga integration dinhi.

## Mga Parameter {#parameters}

| Ngalan | Matang | Deskripsyon |
|------|------|-------------|
| $registry | `\Integration_Registry` | Ang instance sa registry. |
