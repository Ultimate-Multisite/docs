# Aksyon: wu_dashboard_widgets

Tugoti ang mga developer sa plugin nga makadugang og mga widget sa Network Dashboard Panel.

## Mga Parameter

| Ngalan | Type | Deskripsyon |
|------|------|-------------|
| $tab | `string` | Ang kasamtangang tab. |
| $screen | `\WP_Screen` | Ang screen object. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instance sa admin page sa Ultimate Multisite. |
