# Action: wu_page_\{$this->id\}_register_widgets

Modagan human marehistro ang mga widget alang niining panid.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Mga Parameter

| Ngalan | Matang | Deskripsyon |
|------|------|-------------|
| $id | `string` | Ang id sa panid. |
| $page_hook | `string` | Ang hook sa panid. |
| $page | `object` | Ang object sa panid. |
