# Ultimate Multisite Documentation

Malipayon ka namo sa Ultimate Multisite documentation. Ang Ultimate Multisite makabag-o sa WordPress Multisite ngadto sa usa ka kompleto nga Website as a Service (WaaS) platform.

## Documentation Sections

### [User Guide](/user-guide/getting-started/ultimate-multisite-101)

Kini ang tanan nimong kinahanglan aron i-set up, ikonegro, ug pamahalaon ang imong WaaS platform. Naglakip kini sa pag-install, payment gateways, domain mapping, client management, ug uban pa.

### [Developer Documentation](/developer/hooks)

Teknikal nga reference alang sa mga developer nga nagtukod og butang gamit ang Ultimate Multisite. Naglakip kini sa hooks reference, REST API documentation, ug giya sa paghimo og addon.

### [Addons](/addons)

Dokumentasyon para sa bayad nga addon plugins nga nagpalapad sa Ultimate Multisite uban ang dugang nga mga features sama sa WooCommerce integration, analytics, domain selling, ug uban pa.
