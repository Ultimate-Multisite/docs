# Tárgáil Ultimate Multisite {#translating-ultimate-multisite}

Tá Ultimate Multisite dénachd in Gaeilge, ach tá an plugin go léir fada tradháil agus táimid ag féachaint ar chontrait tradáiste.

## Tárgála Plugin Core {#core-plugin-translations}

Tá an plugin core Ultimate Multisite tradáiste tríd an **fomrá tradáiste oifige** de WordPress:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Is é seo an siomradh tradáiste comhrá WordPress. Nuair a bhfaigheann na tradáistí an chéad chuid athbhreithniú, tá siad ag déanamh díreach ar an gcuid úsáidheor Ultimate Multisite tríd athbhreithniú plugin WordPress.

**Táimid ag teacht ar fáil!** Má tá tú ar luach eile de Bhéarla, le ghnáthach tradáistí a bheith agat. Tá sé go maith an t-tradáiste chuid atá. Chun an t-amhailt a dhéanamh:

1. Roinn [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Déan chun an luach a bheith agat
3. Cliom ar "Contribute Translation" (Tárgáil a chontraithe)
4. Deatach tú ar chontrait WordPress.org le free

Má tá tú ag iarraidh a bheith mar editor tradáiste do luach (a bheith ag fhéadú tradáistí), féach ar [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

Má tá tú a fáil ar an talamh chun traslaithe a chur in áit chomh maith le do tháirge, mar chuid féin is báid an tairiscint upstream, tá de Dhá chuntas:

### Aistriú le Poedit {#addon-translations}

Is féidir leat a bheith ag úsáid an application desktop **[Poedit](https://poedit.net)** chun an file `.pot` a traslaigh:

1. Déan download Poedit ó [poedit.net](https://poedit.net)
2. Lán an file `.pot` ó `wp-content/plugins/ultimate-multisite/lang/`
3. Traslaigh na strings (teangailí) i bhfíne Poedit
4. Save -- Poedit a gnáimhneann files `.po` agus `.mo`
5. Slán an file output i folder `lang`

### Aistriú le Loco Translate {#local-translations}

Leann an plugin WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** leat traslaigh go díreach ó chuid admin panel gan aon rud desktop. Tá sé ag cur isteach agus a gnáimhneann files translation chun cinn duit.
