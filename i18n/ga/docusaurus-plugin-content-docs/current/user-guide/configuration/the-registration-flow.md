# An Faocháir Registration (v2)

_**NOTE CRUCIAL: Tá an curte seo ag curte ar Ultimate Multisite version 2.x.**_

Is féidir leis na súiliteoirí san áit a chur inregistri go dtí do ngrúpa. Is féidir leo a bheith ag úsáid an form inregistréad atá agat nó link iadúil chun plan ar chuid a bhfháil a bheith ag curte. Má tá email confirmation ag cur síos ar fáil, deicinn leis na email inbox a bheith agus le clic ar an link féachfaidh siad. Ndiúint an website gan a bheith oibríoch mar níl é an t-aistriú an email.

## Aistriú le Link Shareable:

Tá an t-aistriú ag cur síos ag úsáid link shareable ag cur isteach ar fáil mar atá an chéad aistriú, ach tá an t-aistriú ag úsáid link shareable, is é an t-aistriú a bheith ag cur chluainn nó template website ar fáil ar an form checkout (cur le cineál seo ar an seachtain Pre-selecting products and templates via URL parameters) nó beidh coupon code ag cur isteach (cur le cineál seo ar an seachtain Using URL Parameters).

Beidh an t-aistriú ag cur síos ag cur chluainn, username, email address, ainm website agus titíl agus eile... ach beidh an plan nó template site atá ag cur chluainn ar fáil go léir.

### Aistriú le Payante Manual:

Má níl tú ag cur isteach ar PayPal, Stripe nó aon ghlúin payante eile a chur ar fáil de do Ultimate Multisite nó na haghaidh is é, leat a bheith ag úsáid payante manual don chluainneacha. Mar chuid seo, leat an invoice a sh fóil chun siad a ghairt ar an phroiseas payante a bhfháil ar do chur-bhfháil payante ar fáil tar éis a bheith registered ar do ngrú.

Beidh an t-aistriú ag cur síos mar atá an chéad aistriú, ach beidh an chluainneacha a fheiceáil ar an seachtain a chur in iúl go bhfuil siad ag fháil ar email le cúisí eile chun an payante a deán.

![Message de paiement manuel pendant l'inscription](/img/frontend/registration-manual-notice.png)

Éirinn leat, tar éifeachtach ar an bhfáil. Agus tar éifeachtach tar éifeachtach an iscáil, beidh siad ag fheiceáil an curteíocht a shollátht tú (agus beidh siad ag tógáil é ar email).

![Curteíochta de bhainneacha pochtáil leata sonraí](/img/frontend/registration-payment-instructions.png)

Is féidir an curteíocht a thaisrúnadh ar **Ultimate Multisite > Settings > Payments** tar éifeachtach tar éifeachtach ar chuid a bheith ag tógáil an option de bhainneacha pochtála: **Manual**.

![Tógáil manual payment le fhaothair curteíochta](/img/config/manual-gateway-settings.png)

Tar éifeachtach tar éifeachtach nach bhfuil an iscáil manach ar chuid a bheith ag tógáil an confirmation, bíonn tú ag iarraidh **a chuid a bheith ag tógáil an bhainne** chun an fhoireann agus an website a bheith ag tógáil.

Chun é sin a dhéanamh, rag thar **Ultimate Multisite > Payments** agus is é ar an iscáil a bheith ag tógáil. Ba cheart sé ag tairiscint **Pending**.

![Líon pochtála le bhainneacha manual pending](/img/admin/payments-list.png)

Cliceáil ar núm pochtála agus beidh tú ag cur an stádas go **Completed**.

![Lástáil curteíochta de bhainneacha](/img/admin/payment-edit.png)

![Chang an stádas pochtála go Completed](/img/admin/payment-status-completed.png)

Tar éifeachtach tar éifeachtach ar an stádas a bheith ag tógáil go **Completed**, beidh tú ag fheiceáil an meán "Activate membership". Tógáil an option seo **on** chun an fhoireann agus an website a bheith ag tógáil a bhaineann le an iscáil seo. Ansin, cliceáil chun **Save Payment**.

![Tógáil activate membership agus Save Payment](/img/admin/payment-activate-membership.png)

Ba cheart sé ar an iscáil a bheith ag cur fáil ar an dashboard agus ar an gcuid feichar a bhaineann leat a bheith ag tógáil.
