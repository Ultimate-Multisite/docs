# Ceardlacha Intirithe

Is é AI Agent Gratis v1.5.0 níos adda **Internet Search** (Ceardlacha Intirithe) a leas an AI assistant a fháil ina chuid féin inforamach níos nua ó theibhiú go dtí an cuid. Is é a bhfáil tríd [Brave Search API](https://brave.com/search/api/) nó [Tavily API](https://tavily.com/).

## Conas Tá É seo ag Raid

Nuair a bheith ceardlacha intirithe ag fáil, is féidir leis an assistant a ghlúint go dtí chuid féin ar an cungaí a bhaint as tú nuair a bhíonn sé ag féachaint go bhfuil curtha nó inforamach oifríomhach atá ag curtha - mar shampla, news nua, prisiú laimhseallach, nó daccúiní a bhí athrú ansin atá ag curtha an t-aonracha an model.

Bíonn na rathacha ag fáil go dtí chuid féin agus ag curtha i gcaint an assistant ar bhealach réel-time before sé a chur in ar fháil. Bíonn an assistant ag teacht ina lá máireann sé go bhfuil sé ag úsáid rathacha intirithe chun curtha ach an cuid.

## Féachaint Internet Search

Bíonn ceardlacha intirithe ag fáil ag iarraidh key API ó chuid cungaí a bhaint as tú. Chun cúintiú é:

1. Déan curte ar **Gratis AI Agent → Settings → Advanced**.
2. Cuir isteach i **Internet Search Provider** (Cungaint Internet Search) agus déan sé ag fáil ar **Brave Search** nó **Tavily**.
3. Cuir an key API a bhaineann tú i gcurtha féin. Bíonn linkí curtha in aís chun cinn ag curte ar chuid féin.
4. Cliom ar **Save Settings** (Cúintiú Cuid).

Agairm go bhfuil an key curtha, bíonn an fhiachán Internet Search ag fáil go dtí chuid féin.

## Brave Search

### Fáil Key API Brave Search

1. Déan curte ar [Brave Search API page](https://brave.com/search/api/).
2. Cuir isteach i plan. Tá curtha leat ar fáil an t-amach (free tier) le curtha curtha amhaireachta ar chuid curtha.
3. Cliom an key API a bhaineann tú ó dháta deóirí Brave Search.
4. Cuir isteach é i **Brave Search API Key** i gcurtha féin ag curte.

### Liacht Uirlisí

An túsáid a bhaineann le Brave Search ag cur ar chuid na gníomhartha. Is é a bhreithneacht ar an tadhaun a dhéanamh ar an n-uair a dhéanann tú. Tá mar gheall ar chomhthéacs na héifearcháin AI atá ag cur ar chuid search, tá sé ag cheanglaíonn sé caillte as ceadúil é seo. Monitóraigh an úsáid agat i [Brave Search developer dashboard](https://brave.com/search/api/) chun gach chuid a bhaint a bheith go héifeachtach agus chun spriocanna n-uair a bhaint a bheith go héifeachtach a shábháil.

## Tavily

Tá Superdav AI Agent v1.10.0 ag cur **Tavily** ar fáil mar chuid alternatív search internet, ag teacht le rialta a chanachaire agus cúhse ar fhunghúint.

### Tógadh Clé API de Tavily

1. Roinn an [Tavily API page](https://tavily.com/).
2. Déan seirbhís (sign up) chun rachmasán a laigh a fháil. Tá cur chuig free tier ar fáil le liúchán rachmasáin míshínteacha.
3. Clionaigh an chéad clé API a bhaineann le Tavily dashboard.
4. Clonaigh é i ardán **Tavily API Key** ag cur isteach i ardán Gratis AI Agent.

### Liúcháin Úsáid

Tá úsáid a bhaineann le Tavily ar an chuid chuid call API a dhéanann tú. Tá gach chuid response AI a thabhairt a tháir search, ag cheanglaíonn sé caillte call. Monitóraigh an úsáid agat i [Tavily dashboard](https://tavily.com/) chun spriocanna n-uair a bhaint a bheith go héifeachtach a shábháil.

## An tsláthadh search internet

Riaigh an clé API ó chuid provider search activ i ardán agus síligh. Beidh cur chuig search internet ag cur ar fáil ar an astú.

:::note
Tá search internet ag cur leathrú ar na response because iarraidh an astú a bheith ag teacht ar na rialta search primaill chun an t-aonach a dhéanamh. Chun caillte as amach, déan cinnte go bhfuil tú ag iarraidh search real-time nó a bheith fhios agat má tá eolais a bhaineann leat ag cur ar fáil iarracht na héifearcháin AI.
:::
