# Déanamh ar Linkí Shareable do Plananna

Caithfidh Ultimate Multisite leat an fáil an tairiscint a bheith agat chun linkí plananna a chur ar fáil díreach do do chliantacht. Chun cheannach, rag thar éiteach ar aon dearcais plananna agus cliceáil ar an nútair _Click to copy the Shareable Link_.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Ar an cur chuile plan, is féidir leat a bheith ag iarraidh an nútair chun linkí shareable a chur ar fáil._

An bhfuil tú ag iarraidh an tairiscint a bheith ag quarterly (trí chuid) deiridh? Is féidir leat ag iarraidh 3 ag cur isteach an deiridh.

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Chuir ar iúl an plan _premium_ ar chur chuile chuid, is féidir leat ag cur isteach 1 agus an fhoimne:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Béal é a bheith agat ar aon link amháin do chuid feicí bhunaithe ar fhéideachas breiseadh ag féachracha eile. Is féidir leat an tairiscint a chur ar fáil link spriocailte do chuid feicí breiseadh.

Agus leanaor, nuair a chur ar fáil an link do do chliantacht, beidh sé ag cur iomlán do chéad áit dearcais chun an t-amachas (signup flow) a dhéanamh.
