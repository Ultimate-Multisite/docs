# Export & Import

Céadú Ultimate Multisite 2.9.0 curte **Export & Import** (Export & Import) céadach ar an bhfoghla **Tools > Export & Import**. Use é seo nuair tá tú ag iarraidh pácaidh site WordPress atá ag cur isteach mar ZIP file, aistriú é sin ZIP, nó siúite site ó chuid eile leat ag cur isteach idir na WordPress installations Ultimate Multisite agus single-site.

## Permisiúní ag cur fáil

Is féidir leat a bheith ag login mar administrator atá ag cur féin ar an menu **Tools** (Foghla) ar an site atá ag cur fáil nó ag cur isteach. Ar na ngrúpa multisite, usaigh account administrator network when export nó import subsites ó tool Ultimate Multisite level network.

Tá download ZIP a chur ar fáil tríd endpoint download authenticated, mar sin déan seisiún admin (admin session) go léir go dtí an t-download a deireann agus ní a chur ar fáil na URL download a bhfoghla publiquint.

## Exportú site go ZIP

1. Ag admin WordPress do site atá ag cur isteach, rag thar **Tools > Export & Import**.
2. Lán an áireacht export (export area) agus chun an site atá ag pácaidh tú a chosadh.
3. Déan chun cinn na conatáta optional a bheith ag cur isteach, mar uploads, plugins, agus themes, nuair atá sin na oifrí ag cur fáil.

Is é an tairm a bheith ag fáil ar na leimeanna seo:

*   Leimeanna muyr (uploads directories) nó librar media mór leataigh leatúin ZIP mór. Déan cinnte go bhfuil liomha PHP, liomha eisteachtach (execution limits), spás disc, meagair agus cur chuig (timeout settings) an server ag fáil ar chuid seo primaí chun sitiú nó importáil siopaí mór.
*   Leimeanna media muyr leataigh leatúin leataigh mar gheall ar amach na gceolaisceana (low-traffic maintenance window).
*   Is féidir an càion WordPress a bhaint as (target WordPress installation) a bheith i gcónaí WordPress, PHP, Ultimate Multisite, plugin agus theme comhionachtaí.
*   An import siopaí aon-siopa (single-site import) leatúin an siopa a bhaint as. Níl sé tool chun siopaí a chomhcheangail (merge tool).
*   Tá cur chuig Multisite-to-single-site agus single-site-to-multisite ag fáil ar chuid seo má tá an chuid a bhaint as ag eoláint an plugin, na themeanna, na URL-a agus na comhionachtaí Ultimate Multisite ag an chuid a bhaint as.
*   Déan ZIP leatúin seachainn (private). Is féidir leis an tairm a bheith ag cur chuig cur chuig deacais (database content), fáil leataigh leataigh, agus cur chuig cur chuig deacais comhionachtaí ó chuid a bhaint as.

For workflows leatúin ar leithreach-leimeanna (network-level export workflows) a bhí anseo inniu, léiri [Site Export](/user-guide/administration/site-export).
