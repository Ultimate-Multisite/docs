# Déanamh ar Oileúin

Leonaíonn an lús **Customers** an liosta oileáin a bhfostaithe i gceoladh duit. Taispeann an cur chuig níos éasca faoi bhealach, email, láithreachas achainn, agus link **switch to** a leagan duit logáil isteach i subsite an oileán mar site administrator.

![Customers list page](/img/admin/customers-list.png)

Cliceáil ar **Manage** leathanach ar chuid oileán chun thôngas ardchaidh faoi bhealach sin a fheiceáil.

Ar leathanach cur chuig an oileán, tá an bloc **At a Glance** ag tairiscint ID an oileán, amach láithreachas achainn, agus cur táirge a chluainte ar an oileán.

Leaganach **Customer Options** leaganann duit a bheith mar **VIP** (tag chun siad a bheith níos éasca a fháil). Tá an tab **Billing Info** ag tairiscint thôngas achainn mar email, áit, agus na hiontaí.

Tá an bloc **Fields** ag tairiscint an username an oileán, email, agus leaganann duit athrú an stát **Email Verification**.

Leathanach ar an chuid Customer Options, beidh tú leat seachain chéad bloc eile:

- **Memberships** — An gcurraisteacha a bhaineann leis an oileán
- **Payments** — An scéal achainn an oileán
- **Sites** — Na sitea a bhaineann leis an oileán
- **Events** — Eagraíochtaí a bhaineann leis an oileán (email verification, athrú thôngas, atáir le chéile, etc.)

Cliceáil ar an chuid icon (→) ag cur chuig aon cheann chun an leathanach ardchaidh sin a bheith. Mar shampla, cliceáil ar an chuid icon next to site leagann tú go dtí leathanach éidite site beag mar a féidir leat athrú an gcurraisteacht, an tairisc site, agus an stát activ.

Tá an chúram seo ag cur chuig memberships agus payments — each link go dtí leathanach ardchaidh a bhfuil tú ag éidite an stát membership, an oileán a bhaineann leis, na tairisc, curteacha gateway achainn, agus até go haghaidh athrú achainn.

Ar an chuid de chéad leathanach oileán, beidh tú leat:

- **An Lábhairt Áirítear & IP's** — Am leas an láithreacha, an chuid IP agus na cosúil (country)
- **Timestamps** — Cad an t-acaite an contaacht a bhcreasta agus a athrú an lást
- **Deileadh** — Olg an t-acaite go hiomlán
