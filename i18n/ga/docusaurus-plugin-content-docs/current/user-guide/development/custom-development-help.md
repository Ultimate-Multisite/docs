# Conas an tairm do chuidiúcháin de dhuit ó Codeable? {#how-to-get-custom-development-help-from-codeable}

Caithfidh méidí ar thuir, caithfidh méidí ar thuir chun na fáilteanna a bheith linn i gcónaí is lán mar atá ag méidí. Má tá sin é seo a bhfuil tú ag iarraidh, tá Codeable an t-amach agam. Is é seo curte áit a bhfaidh tú a fháil ar d'fhéanteach dévelperí le tuairiscí ar Ultimate Multisite.

Is Codeable platform a chiallaíonn siad le oileú le chéile custaimerí le dévelperí WordPress freelance a bheith chuidaithe, ag eolais ar projéct de chuid méidí. Tá siad ar chur chun cinn a bheith scéal leat chun an t-amach leat a bheith mar chuid fhéanteacha is mhaith leat a bheith ag an t-amach leat.

### Conas a dhéanamh ar projéct i Codeable? {#how-to-create-a-project-in-codeable}

Ar anois, rag méidí ar [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) agus cliceáil ar **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Anois, chun projéct a thaisrúnadh, cliceáil ar an chéad leat go lá ar **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

I "I need help with", déan **Customizing**:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

I "My", déan **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Ansin, cliceáil ar an t-amach **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Anois, tá an am ag cur sonra ar do projéct agus is féidir leat a bheith mar chuid fhéanteacha is mhaith. Déan an gcur áit méidí a bheith agat!

<!-- Screenshot unavailable: Codeable project description form -->

Answán na ciste seo chun cabhrú leat ag cur sonra ar an t-amach.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Ansin, má tá thuairisc [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) agam ach anois, creai athbhreithniú agus cliceáil ar an nútair **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

B'fhéidir go bhfuil an platform ag cur isteach tú le deabhacaithe atá ag tairisc a bhtréimhseann leis an chuid foirme a oscaíonn.

Beidh an chéad cheangailte seo ag cur tú in áit an duine ceart chun an oibrí a dhéanamh, a bhfuil taithí le WordPress agus ar Ultimate Multisite!

An táirge míniú Codeable is é astú USD 70–120 ar amach. Beidh an tairisc ag athrú mar a bhfearann an scopi oibríochta, an chuid foirme, agus an ghráid an oibrí.

Gach níghníomh atá agat leat, beidh tú in áit a bheith leat chun an deabhacaithe is fearr do chuid raonachtaí a bheith leis agus chun an tairisc a dhéanamh ina láthair!
