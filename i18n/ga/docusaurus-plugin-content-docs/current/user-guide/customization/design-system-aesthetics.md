# Fíneadh Skill Design System Aesthetics {#design-system-aesthetics-skill}

Is **Fíneadh Skill Design System Aesthetics** cur chuig an t-amhrán a chur ar fáil chun do chuid deashcaíocht site a fheabhsú. Tá sé ag cur chun cinn túidh ar chuid cúrama ar fáil faoi chuid téacs, colur, spás agus tokenacha motion a bhfaidh siad a dhéanamh ar chuid deashcaíocht system atá agat.

## Cad é Design System Aesthetics? {#what-is-design-system-aesthetics}

Is Design System Aesthetics skill struchtúrtha a bhfuil sé ag cur chun cinn ar:

- **Typography**: Fáil téacs, méidí, gáirí agus leathra line
- **Color**: Palataí príomh, deacairt, accent, agus neamhchuid
- **Spacing**: Padding, margin, agus scale gapacha
- **Borders**: Tokenacha radius agus meitheal
- **Shadows**: Tokenacha elevation agus depth
- **Motion**: Animationacha agus transitions

Tá na cúrama seo a bhaint amach i do `theme.json` dearcais, ag cur cúram deashcaíocht chomhúil ar fáil.

## Cad a bheith ag éile Design System Aesthetics? {#why-use-design-system-aesthetics}

### Cúram {#consistency}

Taispeán design system a thaispeann:

- Tá an t-amhrán téacs a bheith ag úsáid iad
- Tá colurí a úsáid go cúram ar chuid site
- Tá spás a bhaint amach ar fáil ar chuid pattern
- Tá animationacha ag teacht ar chuid cúram

### Eifeachtas {#efficiency}

An t-amhrán a dhéanamh ar chuid page ar chuid deashcaíocht, tú:

- Déan tokenacha anois
- Áirbhriú iad ar fáil ar fhisc
- Athchunnamh go láidir ag athrú adhrán amháin

### Flexibilidade {#flexibility}

Is féidir leat:

- Aistriú do chuid design system go tapa
- Teacht ar chuid deashcaíocht eile
- Déan cúram branda a bheith chomhúil ag teacht ar fáil mar thuairim

## Anailís na Fócas Áisiú {#the-design-system-aesthetics-process}

### Step 1: Typografaíocht {#step-1-typography}

Nfháil an agent ar chuid fáilteacht agat ar do chohraitheacht:

```
Cad é do chohraitheacht typografaíochta ar fáil?
- Serif (traditionál, éileamhach)
- Sans-serif (nóta, clean)
- Monospace (teicnical, ag cur isteach ar code)

Do chohraitheacht heading:
- An bhfuil tú ag iarraidh chohraitheacht heading a bheith ag teacht, nó a úsáid an sin mar téacs leat?
- Fáilteacht: bold, éileamhach, playful, minimal?

Do chohraitheacht body:
- Is é an fáilteacht ar fáil. An bhfuil tú ag iarraidh:
  - Téacs láidir, spásaithe níos mó
  - Téacs beag, éifeachtach
  - Cuidéad téacs stiúradúil
```

Ansin a chur in uradh:

- **Chohraitheacht heading**: Chohraitheacht príomhaidh do titlaí agus heading.
- **Chohraitheacht body**: Chohraitheacht ar fáil paraínta agus téacs body.
- **Chohraitheacht monospace**: Chohraitheacht ar fáil ar code agus curteacha teicnicalta.
- **Scale cothrom:** Cuidéadanna preagda (beag, base, láidir, XL, etc.).
- **Scale chuid:** Scale chuid (regular, medium, bold, etc.).
- **Line height:** Spás ar chuid leat chun fáilteacht a bheith ag teacht.

### Step 2: Palat de Colur {#step-2-color-palette}

```
Lón é do palat de colur a chur in uradh.

Chomhlaíocht príomhaidh (do chomhlaíocht branda):
- An t-am: [táin an chomhlaíocht a bheith]
- Chang ar: [cloc de colur nó code hex]

Chomhlaíocht seaca (chomhlaíocht a thugann cúram):
- An t-am: [táin an chomhlaíocht a bheith]
- Chang ar: [cloc de colur nó code hex]

Chomhlaíocht accent (fócas agus CTAs):
- An t-am: [táin an chomhlaíocht a bheith]
- Chang ar: [cloc de colur nó code hex]

Palat nua (grays ar fáil do téacs, borderaí, spásanna):
- Léineach: [colur]
- Meánach: [colur]
- Dubhairt: [colur]
```

Ansin a chur in uradh palat leat go léiríonn sé:

- Chomhlaíocht príomhaidh, seaca, agus accent.
- Grays nua (léineach, meánach, dubhairt).
- Colur semantic (suasach, athbhreithniú, éiríse).
- Státa hover agus active.

### Step 3: Spásanna {#step-3-spacing}

Cénn mé chomh mór spás a bhfuil tú ag iarraidh?

Páiste spás (Spacing scale preference):
- Compact (layoutaite, éifeachtach)
- Normal (spás beartaithe)
- Spacious (spás mór)

Is é seo ag athrú:
- Padding sa bhfiontanna agus ar charde
- Margins a bheith ina mbainneacha
- Spás ach an t-amach ina chuiditeacha grid

Tá an agent ag déanamh cinnte deatóg spáiste (spacing tokens):

- Unit bunaithe (tiomá 4px nó 8px)
- Scale: xs, sm, md, lg, xl, 2xl
- Spásanna mhaith chun padding, margin, gap a chur i bhfeidhm

### Step 4: Borders agus Sháileanna (Shadows) {#step-4-borders-and-shadows}

```
Déithíocht chuid agus deimhneacht:

Páiste radius ar chuid (Border radius preference):
- Sharp (nó leatadh)
- Subtle (radius beag)
- Rounded (radius meánach)
- Very rounded (radius mór)

Spás sháileán (Shadow depth):
- Flat (nó leatadh)
- Subtle (sháileanna beaga)
- Pronounced (sháileanna go leor)**
```

Tá an agent ag déanamh:

- Tokens radius ar chuid (do fiontanna, charde, inputaí)
- Tokens sháileán do leagan tábhachta (elevation levels)
- Tokens width ar chuid

### Step 5: Motion agus Animation {#step-5-motion-and-animation}

```
Conas thuig an site a bheith ag iarraidh gointinn?

Páiste animation:
- Minimal (nó leatadh animation)
- Subtle (transíshioin beag)
- Playful (animation a bheith cinnte)

Animationa spécifike:
- Transíshioin page: fade, slide, nó ní?
- Hover ar fiontanna: scale, change color, nó de bhad?
- Loading states: spinner, skeleton, nó progress bar?
```

Tá an agent ag déanamh:

- Duradan transíshioin (fast, normal, slow)
- Functions easing (ease-in, ease-out, ease-in-out)
- Keyframes animationa ar chuid éifeachtach

## Áirítear na Aistriú Design System {#applying-design-system-aesthetics}

### Áirítear Authiúil (Automatic Application) {#automatic-application}

Tar comhthais an skill, níl an agent:

1. Athchun an `theme.json` dearg an theme leat ag an t-tokens
2. A chur ar an design system ar an theme a bhfuil tú ag úsáid
3. Athchun style blockaí chun a bheith ag cur i gceart leis an nua system
4. A chruinneadh an theme athrúlaite

### Áirítear Múir (Manual Application) {#manual-application}

Is féidir leat hefyd a athrú `theme.json` go díreach:

```json
{
  "version": 3,
  "settings": {
    "color": {
      "palette": [
        {
          "color": "#0066CC",
          "name": "Primary",
          "slug": "primary"
        }
      ]
    },
    "typography": {
      "fontFamilies": [
        {
          "fontFamily": "Inter, sans-serif",
          "name": "Body",
          "slug": "body"
        }
      ]
    },
    "spacing": {
      "spacingSizes": [
        {
          "size": "0.5rem",
          "name": "Small",
          "slug": "sm"
        }
      ]
    }
  }
}
```

## Bealach Mhaith do Design System (Design System Best Practices) {#design-system-best-practices}

### Cúramh (Consistency) {#consistency-1}

- Use an tairgse amach i gcónaí
- Ná déan colachtaí nó cothromacha a bhaineann leat
- Déan ar tairgse go bhfuil siad ag cur isteach chuid atá ar chuid fáil.

### Oileadh (Naming) {#naming}

Use léine, semantic names:

- ✓ `primary`, `secondary`, `accent`
- ✗ `blue`, `color-1`, `main`

- ✓ `heading-lg`, `body-sm`
- ✗ `font-size-24`, `text-large`

### Scalability (Scalability) {#scalability}

Design an systeem at scale:

- Use units relatíva (rem, em) in féin ann am píxel
- Creach scáileanna (xs, sm, md, lg, xl) instead of chuidéadach chuidéadacha
- Plan éiríseacht chun curthaí níos leathanach

### Documentation {#documentation}

Dúlamhaigh an systéim deasctha:

- Cad a chur iadach na colúir a bhfuil siad ag iarraidh
- Conas a bheith ag úsáid cadacháin (token) chéad
- Éisteachtacha agus casaí éifeachtach

## Cuidéadacha deasctha common {#common-design-system-patterns}

### Modern Minimalist {#modern-minimalist}

- Typógraf sans-serif (Inter, Helvetica)
- Colúir colúir líne (2-3 colúir)
- Spéisiúcháin mór
- Sháileanna beag
- Aonáisiúna mhaith agus leathúil

### Warm and Friendly {#warm-and-friendly}

- Chuidí de serif agus sans-serif
- Colúir colúir áitreach (orange, gráta áitreach)
- Ciorcail chruinne
- Sháileanna beag
- Aonáisiúna spraoiúil

### Professional Corporate {#professional-corporate}

- Sans-serif clean (Roboto, Open Sans)
- Colúir nua le colúir a bhaint as
- Spéisiúcháin strátaigh
- Sháileanna beag go léir
- Aonáisiúna beag

### Creative and Bold {#creative-and-bold}

- Typógraf chuidéadach
- Colúir colúir spraoiúil
- Spéisiúcháin ar fáil
- Sháileanna stróga
- Aonáisiúna a bheith ag teacht

## Treoíocht (Troubleshooting) {#troubleshooting}

**Ndiúlaim na méidí deasctha agam níl ag tairt**
- Glacadh an cache do chuid féin
- Rebuild an site agat má tá tú ag úsáid generator statach
- Chléir an theme.json go bhfuil sé JSON vala
- Féachaint ar an theme a bheith ag obair

**Tá na colúir ag déanamh éifeachtach ar chuid leathanacha eile**
- Féachaint ar CSS a bhíonn ag curthaí i plugins
- Féachaint ar an gcuid leathanach a bheith ag úsáid an theme atá ag obair
- Glacadh aon plugin cache

**Tá mé ag iarraidh do chuid deasctha a bheith ag curthaigh ar fáil**
- Cuir iadach an agent: "Show me my design system history"
- Edit manualmente an theme.json le chuid chuidéadacha a bhí ag obair
- Ríomh séilíocht leis an chuid féin le chuid chuidéadacha eile

## Taisceanna (Next Steps) {#next-steps}

Tar éis tríúchláir do chuid deasctha:

1. **Anail an gsite seo**: Gúir go lá i mhaoinseacht na hionn a bhfuil an fhoinn seo ar an spiorail nua.
2. **Féigh an t-aon**: Déan séasú síos a úig agam le haghaidh an skill seo.
3. **Creoliad templatí**: Déan bloc templatí a bhaint as a chur i gcomhthéacs a bhaineann leis na token deasú (design tokens) atá agat.
4. **Dúlamach**: Cionn an siostam deasú seo le do chuid oibre.
