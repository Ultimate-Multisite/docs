# Aistileadh ar an Panel Adríúchán agat le PRO Theme

Is é seo go bhfuil tú ag déanamh oibríochtaí le clienta, is féidir leis an t-amachas admin de WordPress a bheith ar fáil do chlianta. Má tá tú ag iarraidh athrú an t-amachas a bheith níos professional agus níos láidir, is féidir le PRO Theme cabhrú duit.

Is é PRO Theme un theme admin a dhéanamh chun cabhrú duit nuair a bhíonn tú ag athrú an dashboard de do chlianta. Is é hefyd cuid de bhainneacha core (core) de plugin premium Ultimate Multisite.

Tar éis a chur i gcomhrá, ragad go dtí an cur chuig an _Plugins_ page agus cliceáil chun é a chruinneadh ar do Network.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Chruinneadh é ar fáil ar an t-amachas Plugin}_

_**NOTE:**_ _Má tá tú ag chur i gcomhrá ach níl aon rud a chur i gcomhrá, beidh sé ag déanamh ar do chlianta go cur isteach is cur chun cinn go raibh féidir é a leanúint chun an theme a chur ar fáil gan an t-amachas sin._

Anois, tá tú ag iarraidh an amachas admin agat go léir athrú! Tá sé ag cur isteach noutach ar leith ideal ar an chuid leat (left column) le cur PRO Theme.
