# Pléad Iarann Áiríse ar do ngrúda le Ultimate Multisite: Ad Injector

Is é **add-on grátis** an Ultimate Multisite: Ad Injector a thug tú na fáilteanna a bheith ag cur ads ar do plána tríchtain.

Tar éis a chur in gluach Ultimate Multisite: Ad Injector, rag méid _Plugins_ go dtí athbhreithniú é.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Click an nút Athbhreithniú Ngrúda_

Tar éis a bheith an add-on acuánach, beidh an _Advanced Options_ meta box ar scéal tríchtain ag cur oiriún nua le cur _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Anois is féidir leat éidadh ad settings do chomhthéacsíocht chomh maith agat ar chomhthéacsíocht chomh maith agat._

Ar an tab sin, beidh tú ag cur ad codes ar postanna agus pages i bhfíneán (front-end), ar fáil (before) agus ar fhágáil (after) an cur.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Tar éis a chur i bhfíneán an codad ad, beidh sé ag tairgseáil ar na postanna agus pages sin tríchtainacha._

Is féidir leat hefyd a bheith ads curte ag an héad (header) deacháin Admin Pages do chuid úsáidheoracht sin.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Go dtí an option Ad Code ar an Admin Panel mar thuair tú ads a chur i bhfíneán chuid úsáidheoracht._

### An áiríse ar an tairgseáil {#beyond-advertising}

Is féidir leat hefyd a bheith ag úsáid Ultimate Multisite: Ad Injector chun cur fáil ar na plána chomh maith agat.

An bhfuil tú ag cur i bhfíneán an option Admin Panel Ads agus is féidir leat codad HTML a chur ar do fáil tool.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot n-a mhaith: Widget cúrama i bhfostála ar an panel admin do plan áitiúil -->

_Add cúramh cúrama do plan áitiúil_

Is a chuid seo, is féidir le daoine go díreach os caillteacht (Admin Panel) a roinnt agat!
