# Conas éim féidir liom rud éifeachtúil mo shláin containt a fáil anois?

Fáilte do Ultimate Multisite!

Fáilte ar an shláin containt agat tríd an link atá ag teacht i bhfáil iad an t-e-mail tar éis a gcuid plugin a bhrónach:

<!-- Screenshot unavailable: Account page email with access link -->

Uile an e-mail atá agat ag curtha go leim ar chuid a chur in áit agus an ciste (password) atá agat iad.

<!-- Screenshot unavailable: Login page with email and password fields -->

Nuair atá tú ag logáil ina chuid chéadú seo, beidh tú ag ceart a thaispeáint an ciste (password) a athrú. Athrúch é agus cliceáil ar an nútair _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

Ar shláin containt agat, is féidir leat teastú go léir do chuid ardán containt mar billings, athbhreithniú, athchúrtú, agus go leor eile.
