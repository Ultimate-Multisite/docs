# Inkompatibilitáirí a Plugins eile {#known-incompatibilities-with-other-plugins}

Tar éastaúchadh agus activaitheor Ultimate Multisite, is féidir leat ghlúin a bheith ag fáil eolas ar inkompatibilitáirí leat a bhaint as leatais eile.

Tá méid ar chuid na liosta seo a gréimneamh go léir leat ag an t-aisríocht nua, ach má tá plugin ag cur isteach anseo, is féidir leat é a deactiváid chun Ultimate Multisite Network a bheith go léir foirmiúil.

Seo liosta na hiontaí agus conas atá thuiginn iarrachtaí seo:

**Céad sin miontaí WPMU DEV** Tá roinnt pluginí _níl é ar chuidaithe ag WPMU DEV a bheith ag cur isteach inbhallaí an t-aisríocht Ultimate Multisite ar an áit isfear a chur i bhfostála (ní féidir leis an oifig accdeantóir a fháil). Seo liosta na pluginí a dhéanamh cinnte seo:

  * Signup Language: _Tá add-on ag Ultimate Multisite a thaispeáiríonn an aon foirmiú seo, a cheart go leim: Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Tá sé cur isteach i Ultimate Multisite, mar níl aon bhaint ag plugin eile.*

  * New Blog Templates: _Tá sé cur isteach i Ultimate Multisite, mar níl aon bhaint ag plugin eile.*

  * Set Password on Multisite Blog Creation: _Tá sé cur isteach i Ultimate Multisite, mar níl aon bhaint ag plugin eile.*

  * Signup TOS: _Tá sé cur isteach i Ultimate Multisite, mar níl aon bhaint ag plugin eile.*

  * Anti-Splog: Mara níos foirmiúil é, tá sé cur isteach i Ultimate Multisite.

**Note**: Is féidir na pluginí WPMU Dev eile a úsáid go nárdach le Ultimate Multisite. Déanann seo an smuaghdaí isé: _Smush_ , _Forminator_ , _Defender,_ agus sin mairt.
