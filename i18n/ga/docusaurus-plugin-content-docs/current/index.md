# Doiciméadacht Ultimate Multisite {#ultimate-multisite-documentation}

Fáilte chuig doiciméadacht Ultimate Multisite. Athraíonn Ultimate Multisite WordPress Multisite ina ardán iomlán Suíomh Gréasáin mar Sheirbhís (WaaS).

## Rannáin Doiciméadachta {#documentation-sections}

### [Treoir Úsáideora](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Gach rud atá uait chun d’ardán WaaS a shocrú, a chumrú agus a bhainistiú. Clúdaíonn sé suiteáil, geataí íocaíochta, mapáil fearainn, bainistíocht cliant, agus tuilleadh.

### [Doiciméadacht Forbróra](/developer/hooks) {#developer-documentation}

Tagairt theicniúil d’fhorbróirí atá ag tógáil ar Ultimate Multisite. Áirítear léi tagairt hooks, doiciméadacht REST API, agus treoracha forbartha breiseán.

### [Breiseáin](/addons) {#addons}

Doiciméadacht do bhreiseáin íoctha a leathnaíonn Ultimate Multisite le gnéithe breise amhail comhtháthú WooCommerce, anailísíocht, díol fearann, agus tuilleadh.
