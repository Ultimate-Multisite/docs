# Gníomh: wu_domain_became_primary

Spreagtar é nuair a éiríonn fearann mar phríomhfhearann do shuíomh.

Spreagtar an gníomh seo nuair a shocraítear bratach primary_domain fearainn go true, bíodh sin agus príomhfhearann nua á chruthú nó agus fearann atá ann cheana á nuashonrú le bheith ina phríomhfhearann.

## Paraiméadair

| Ainm | Cineál | Cur síos |
|------|------|-------------|
| $domain | `\WP_Ultimo\Models\Domain` | An fearann a d'éirigh príomhúil. |
| $blog_id | `int` | ID blag an tsuímh lena mbaineann. |
| $was_new | `bool` | Cibé acu an fearann nuachruthaithe é seo. |
