# Scagaire: wu_minimum_password_strength {#filter-wuminimumpasswordstrength}

Scag an neart íosta pasfhocail atá riachtanach (scór zxcvbn).

Leibhéil neart: - 0, 1: An-lag - 2: Lag - 3: Meánach - 4: Láidir (réamhshocrú)

| $min_strength | `int` | An leibhéal neart íosta atá riachtanach. |
| $strength_setting | `string` | Luach socraithe an riarthóra (medium, strong, super_strong). |
