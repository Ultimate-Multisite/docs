# Scagaire: wu_sso_url

Scagann sé URLanna SSO ginte sula gcuirtear ar ais iad le haghaidh gníomhartha custaiméara trasfhearainn.

Úsáid an scagaire seo nuair is gá d’intégrú comhthéacs iontaofa a chur le nasc SSO do thionónta ceannasach nó URL an bhróicéara a athsholáthar agus bailíochtú token Ultimate Multisite á chaomhnú.

## Paraiméadair

| Ainm | Cineál | Cur síos |
|------|------|-------------|
| $sso_url | `string` | URL SSO ginte. |
| $user | `WP_User` | Úsáideoir a fhíordheimhneofar leis an gcuairt SSO. |
| $site_id | `int` | ID an tsuímh sprice don chuairt. |
| $redirect_to | `string` | URL sprice tar éis bailíochtú SSO rathúil. |

### Ó

- 2.13.0

### Foinse

Sainithe in `inc/sso/class-sso.php`.


## Filleadh

URL SSO scagtha.
