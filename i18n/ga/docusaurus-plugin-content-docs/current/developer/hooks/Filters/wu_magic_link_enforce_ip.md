# Scagaire: wu_magic_link_enforce_ip

Scag an gcuirfear fíorú seoladh IP i bhfeidhm.

Socraigh go false chun ligean do tokens oibriú ó líonraí éagsúla. Laghdaíonn sé seo slándáil ach méadaíonn sé inúsáidteacht (m.sh., d’úsáideoirí soghluaiste a athraíonn líonraí).

| Ainm | Cineál | Cur síos |
|------|------|-------------|
| $enforce | `bool` | An gcuirfear meaitseáil seoladh IP i bhfeidhm. |
