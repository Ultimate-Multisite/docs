# Scagaire: wu_enforce_password_rules

Scag cé acu rialacha breise pasfhocail a fhorfheidhmiú.

Nuair atá sé true, forfheidhmítear íosfhad agus riachtanais charachtar. Cumasaítear go huathoibríoch é don socrú "Super Strong" nó nuair atá gné Strong Password Defender Pro gníomhach.

## Paraiméadair

| Ainm | Cineál | Cur síos |
|------|------|-------------|
| $enforce_rules | `bool` | Cé acu rialacha breise a fhorfheidhmiú. |
| $strength_setting | `string` | Luach an tsocraithe riaracháin. |
| $defender_active | `bool` | Cé acu atá Defender Pro Strong Password gníomhach. |
