# Scagaire: wu_minimum_password_length

Scag fad íosta an phasfhocail.

Ní chuirtear i bhfeidhm ach nuair atá wu_enforce_password_rules fíor.

## Paraiméadair {#parameters}

| Ainm | Cineál | Cur síos |
|------|------|-------------|
| $min_length | `int` | Fad íosta an phasfhocail. Réamhshocrú 12 (meaitseálann Defender Pro). |
| $defender_active | `bool` | Cibé an bhfuil Defender Pro Strong Password gníomhach. |
