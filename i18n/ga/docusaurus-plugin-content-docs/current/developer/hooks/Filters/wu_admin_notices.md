# Scagaire: wu_admin_notices

Lig d’fhorbróirí fógraí riaracháin a chuireann Ultimate Multisite leis a scagadh.

## Paraiméadair {#parameters}

| Ainm | Cineál | Cur síos |
|------|------|-------------|
| $notices | `array` | Liosta fógraí don phainéal áirithe sin. |
| $all_notices | `array` | Liosta fógraí a cuireadh leis, scartha de réir painéil. |
| $panel | `string` | Painéal chun na fógraí a aisghabháil. |
| $filter | `string` | Más rud é gur scagadh amach na fógraí indíbeartha. |
| $dismissed_messages | `array` | Liosta d’eochracha fógra a díbheadh. |
