# Scagaire: wu_sso_enabled

Cumasaigh/díchumasaigh cumas single-sign-on trasfhearainn.

Scag an luach seo chun single-sign-on a mhúchadh go hiomlán, nó chun é a chumasú go coinníollach ina ionad sin.

## Paraiméadair

| Ainm | Cineál | Cur síos |
|------|------|-------------|
| $enabled | `bool` | Ar cheart SSO a bheith cumasaithe? True le haghaidh ar siúl, luach cosúil le false le haghaidh múchta. |
