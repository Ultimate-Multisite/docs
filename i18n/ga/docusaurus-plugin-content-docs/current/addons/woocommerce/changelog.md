# Loga Athruithe Chomhtháthú WooCommerce {#woocommerce-integration-changelog}

Leagan 2.2.0 - Eisithe ar 2026-07-01
* Nua: Taispeántar méideanna cánach Ultimate Multisite anois mar línte táille WooCommerce ar leith ag an checkout, rud a dhéanann iomláin chánach níos soiléire roimh íocaíocht.
* Nua: Cuireadh socrú roghnach "Cuir Ballraíochtaí ar Fionraí Láithreach ar Athnuachana Teipthe" agus scagaire `wu_woo_suspend_on_payment_failure` leis do shuíomhanna ar mian leo fionraí i gcás athnuachan teipthe le linn fhuinneog athiarrachta WooCommerce Subscriptions.
* Ceartúchán: Réitíodh stádais WooCommerce Subscription a d’fhéadfadh fanacht as sioncronú le ballraíochtaí Ultimate Multisite tar éis athnuachana teipthe nó aisghafa.
* Ceartúchán: Cuireadh airgeadra siopa WooCommerce le liosta airgeadraí Ultimate Multisite nuair a bhí sé ar iarraidh.
* Ceartúchán: Caomhnaíodh sonraí billeála custaiméirí agus síntiúsóirí á n-atreorú chuig checkout WooCommerce.
* Feabhsaithe: Cuireadh comhoiriúnacht le Jetpack Autoloader 5 leis.
* Feabhsaithe: Glanadh giniúint an phacáiste eisithe ionas go seachnaíonn zipanna GitHub agus margaidh comhadlanna stáitsithe neadaithe agus comhaid forbartha.
