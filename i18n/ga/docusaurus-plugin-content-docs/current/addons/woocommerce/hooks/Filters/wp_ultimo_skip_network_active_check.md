# Scagaire: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Ceadaigh d’fhorbróirí seiceáil ghníomhachtaithe an líonra a ghearrchiorcadú.

Tá sé seo úsáideach agus socruithe bunaithe ar composer agus socruithe saincheaptha eile á n-úsáid, mar shampla Bedrock, áit a mbíonn úsáid plugins mar mu-plugins ina gnáthchleachtas.

## Paraiméadair {#parameters}

| Ainm | Cineál | Cur síos |
|------|------|-------------|
| $skip_network_activation_check | `bool` | An mba cheart dúinn an tseiceáil a scipeáil nó nár cheart; is é false an réamhshocrú. |

### Ó {#since}

- 2.0.0
### Foinse {#source}

Sainithe in [`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/class-requirements.php#L272) ag líne 272


## Tuairisceáin {#returns}
true más mian leat an tseiceáil a scipeáil, false murach sin.
