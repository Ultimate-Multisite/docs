# Fluent Forms {#fluent-forms}

Déanann an breiseán seo indéanta teorainneacha a chur le Fluent Forms ar Multisite Ultimate i dtáirgí, ballraíochtaí nó suíomhanna.
Úsáideann Fluent Forms táblaí saincheaptha in ionad cineálacha postála saincheaptha, mar sin ní oibríonn na gnáthroghanna teorannaithe le Fluent Forms.

## Gnéithe {#features}

- Cuir teorainneacha le Fluent Forms nó Conversational Fluent Forms
- Faigheann custaiméirí teachtaireacht chairdiúil a spreagann iad chun a bplean a uasghrádú nuair a shroicheann siad an teorainn
