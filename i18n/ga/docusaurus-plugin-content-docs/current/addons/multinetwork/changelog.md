# Loga Athruithe Il-Líonra {#multi-network-changelog}

### 1.3.0 {#130}
* Nua: Réamhamharcóir teimpléad líonra — brabhsáil agus réamhamharc ar theimpléid líonra le painéal beo roimh cheannach.
* Nua: Roghnóir teimpléad líonra ag checkout — is féidir le custaiméirí teimpléad líonra a roghnú le linn shreabhadh signup checkout.
* Nua: SSO le magic link traslíonra — is féidir le custaiméirí rochtain a fháil ar fho-líonraí gan dua trí fhíordheimhniú magic-link.
* Nua: Clónáil meán líonra — déantar meáin suímh a dhúbailt i gceart agus líonraí á gclónáil ó theimpléad.
* Ceartú: Oibríonn clónáil líonra anois gan an addon ilthionóntachta; déantar pribhléidí super admin a dheonú arís tar éis clónála.
* Ceartú: Tá cruthú líonra comhoiriúnach anois le WooCommerce agus le gach geata íocaíochta.
* Ceartú: Cuirtear uimhir an phoirt san áireamh anois i nginiúint fearainn fallback.
* Ceartú: Bogadh CSS an roghnóra teimpléid go stylesheet enqueued le haghaidh rindreáil cheart ar gach theme.
* Ceartú: Cosaint ar eochair `network_id` neamhshainithe i raon feidhme fiosraithe suímh.
* Ceartú: Coscann seiceáil ceadanna líonra roimh athrú comhthéacs earráidí airí tábla `wpdb`.
