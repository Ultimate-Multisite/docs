# Tógálaí Suímh AI {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Riachtanais {#requirements}

- WordPress 5.3 nó níos airde
- PHP 7.4 nó níos airde
- Breiseán Multisite Ultimate (gníomhach)

## Suiteáil {#installation}

1. Uaslódáil comhaid an addon chuig do chomhadlann `/wp-content/plugins/`
2. Gníomhachtaigh an breiseán tríd an roghchlár 'Breiseáin' in WordPress
3. Cumraigh socruithe an addon sa riarthóir líonra

## Forbairt {#development}

### Socrú {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

ultimate-multisite-ai-site-builder/

### Tástáil {#testing}

Áirítear creat cuimsitheach tástála leis an addon:

- **Tástálacha Aonaid**: Tástálacha bunaithe ar PHPUnit le sraith tástála WordPress
- **Bunrang Tástála**: Soláthraíonn sé modhanna cabhracha do chásanna coitianta tástála
- **Clúdach Tástála**: Cumraithe chun clúdach cóid a rianú
- **Tástáil Multisite**: Ritheann tástálacha i dtimpeallacht multisite

### Caighdeáin Chóid {#code-standards}

- **Caighdeáin Chódaithe WordPress**: Curtha i bhfeidhm trí PHPCS
- **Comhoiriúnacht PHP 7.4+**: Tacaítear le gnéithe nua-aimseartha PHP
- **Anailís Statach**: Comhtháthú PHPStan le haghaidh seiceáil cineáil
- **Nuachóiriú Cóid**: Rialacha Rector le haghaidh uasghráduithe PHP

### Córas Tógála {#build-system}

- **Próiseáil Sócmhainní**: Íoslaghdú do chomhaid CSS/JS
- **Aistriúchán**: Giniúint comhaid POT
- **Pacáistiú**: Cruthú cartlainne le haghaidh dáileacháin
- **Mód Forbartha**: Sócmhainní neamhíoslaghdaithe le haghaidh dífhabhtaithe

## Cumraíocht {#configuration}

Tacaíonn an addon le roghanna cumraíochta éagsúla atá inrochtana trí chomhéadan riarthóra an líonra.

## Hooks agus Scagairí {#hooks-and-filters}

### Gníomhartha {#actions}

- `ultimate-multisite-ai-site-builder_init` - Spreagtar tar éis thúsú an addon
- `ultimate-multisite-ai-site-builder_loaded` - Spreagtar tar éis gach comhad addon a bheith luchtaithe

### Scagairí {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Athraigh socruithe an addon
- `ultimate-multisite-ai-site-builder_enabled` - Sáraigh stádas cumasaithe/díchumasaithe an addon
