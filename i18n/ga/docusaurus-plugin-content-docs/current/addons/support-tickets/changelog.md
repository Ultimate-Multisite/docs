# Loga Athruithe Ticéad Tacaíochta

### 1.0.4 - 2026-05-05
* Feabhsaithe: Baineadh an chomhadlann vendor/ ó rianú Git (clúdaithe cheana ag .gitignore), ag laghdú mhéid an taisclainne
* Feabhsaithe: Tástáilte suas go WordPress 7.0

### 1.0.3 - 2026-05-01
* Ceartú: Glac le luachanna null i socraitheoirí samhla ticéid atá in-nullaithe
* Ceartú: Stóráil gach ticéad i dtáblaí domhanda ar fud an líonra le haghaidh tacaíocht cheart multisite
* Ceartú: Folaigh réimsí foirne agus ceartaigh atreorú leathanaigh bháin ar fhoirm ticéid nua an chustaiméara
* Ceartú: Cuir update_meta() ceart in ionad glaonna add_meta() neamhshainithe le haghaidh stóráil meiteashonraí
* Ceartú: Cuir wu_view_all_support_tickets ceart in ionad seiceáil cumais neamhchláraithe
* Ceartú: Cuir láimhseálaithe AJAX atá ar iarraidh leis le haghaidh stádas ticéid, sannadh, agus gníomhartha mear-eagarthóireachta
* Ceartú: Ceartaigh ainm an mhodha le haghaidh braite freagra foirne i radhairc ticéid
* Ceartú: Comhdhlúthaigh láimhseálaithe freagra dúblacha agus ailínigh ainmneacha gníomhartha nonce
* Ceartú: Cuir radharc frontend atá ar iarraidh leis don shortcode [wu_submit_ticket]
* Ceartú: Cuir colún user_id atá ar iarraidh leis agus ceartaigh modh Support_Ticket::get_user_id()
* Ceartú: Bain comhartha athfhriotail dúbailte breise i gclib rogha scagaire tosaíochta
* Ceartú: Cuir painéal bainistíochta ticéad riaracháin líonra leis do shár-riarthóirí
* Feabhsaithe: Comhdhlúthaigh CSS riaracháin i stílbhileog sheachtrach aonair
* Feabhsaithe: Bain fo-roghchlár Settings ó roghchlár riaracháin an fho-láithreáin
* Feabhsaithe: Luchtaigh sócmhainní frontend go coinníollach ar leathanaigh ticéad tacaíochta amháin
* Feabhsaithe: Scipeáil uathlódálaí plugin nuair atá uathlódálaí fréimhe Bedrock tar éis spleáchais a luchtú cheana

### 1.0.2 - 2025-12-11
* Curtha leis: Feidhmiúlacht iomlán féachana ticéad don riarachán agus don frontend

* Curtha leis: Tacaíocht d’fheidhmiúlacht freagra ticéid le láimhseáil cheart foirme
* Curtha leis: Taispeáint cheart fógraí le haghaidh cur isteach ticéad agus freagraí sa riarachán
* Curtha leis: Ceangal uathoibríoch ticéid leis an úsáideoir reatha le haghaidh féin-chur isteach custaiméara
* Curtha leis: Feabhsú slándála chun custaiméirí a chosc ó úinéireacht ticéid a shárú
* Curtha leis: Feidhmeanna cúnta atá ar iarraidh (wu_format_date, wu_user_can_view_ticket, etc.)
* Curtha leis: Íoslódáil agus láimhseáil cheart iatán comhaid
* Curtha leis: Córas fógraí ríomhphoist le haghaidh freagraí ticéid agus athruithe stádais
* Ceartaithe: Tá ID ticéid riachtanach san áireamh anois i bhfoirm freagra ticéid
* Ceartaithe: Fadhbanna infheictheachta fógraí i radhairc ticéid riaracháin
* Ceartaithe: Earráidí comhréire i bhfeidhmeanna ticéad tacaíochta
* Feabhsaithe: Struchtúr bunachair sonraí le ranganna ceiste agus scéime cearta le haghaidh freagraí agus iatán
* Feabhsaithe: Imirce chuig airíonna fíora in ionad tréithe do shamhlacha ticéid
