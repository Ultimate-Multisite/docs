# VAT

Multisite Ultimate: VAT Eorpach

Cuireann sé gnéithe agus uirlisí breise le láimhseáil cánach réamhshocraithe Multisite Ultimate chun tacú níos fearr le VAT Eorpach.
Cuireann sé rátaí VAT an AE go huathoibríoch leis an leathanach Rátaí Cánach ó

## Suiteáil

1. Uaslódáil 'ultimate-multisite-vat' chuig an gcomhadlann '/wp-content/plugins/'
2. Gníomhachtaigh an breiseán tríd an roghchlár 'Breiseáin' in WordPress
3. Téigh chuig Ultimate Multisite > Socruithe > Cánacha agus cumasaigh VAT an AE.
4. Cliceáil ar Rátaí Cánach agus Nuashonraigh Rátaí VAT an AE ag an mbun agus Sábháil Rátaí Cánach.
5. Déanta!
