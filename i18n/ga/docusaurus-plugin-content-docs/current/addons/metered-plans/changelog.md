# Loga Athruithe Metered Plans {#metered-plans-changelog}

Leagan 1.1.0 - Eisithe ar 2026-05-05
- Nua: billeáil chomharthaí AI do fho-láithreáin multisite — rianaigh agus billeáil úsáid chomharthaí AI thar láithreáin chustaiméirí le rátaí in aghaidh an chomhartha atá inchumraithe
- Nua: athtógadh forfheidhmiú nascóirí le haimsiú teorainneacha dinimiciúil agus scríobh-tríd, rud a chinntíonn cruinneas fíor-ama thar gach nascóir
- Deisiú: tá uasghráduithe táblaí bunachair sonraí ailínithe i gceart anois le sainmhínithe scéime BerlinDB, rud a chuireann cosc ar theipeanna uasghrádaithe ar shuiteálacha nua
- Deisiú: tiontaíodh glaonna ar ais uasghrádaithe bunachair sonraí go dtí an fhormáid cheart, ag réiteach teipeanna uasghrádaithe ciúine
- Deisiú: glactar anois le luachanna codánacha i réimse ionchuir AI Usage Overage Markup
- Deisiú: réitíodh earráidí marfacha agus fadhbanna tosaithe dúbailte ar luchtú plugin
- Feabhsaithe: cuireadh script npm check-env leis ionas go ndéanann timpeallachtaí forbróirí iad féin a chumrú ar an gcéad rith
