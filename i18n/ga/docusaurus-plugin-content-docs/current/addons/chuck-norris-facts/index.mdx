# Fíricí Chuck Norris

## Suiteáil

1. Cinntigh go bhfuil plugin Ultimate Multisite suiteáilte agus gníomhachtaithe ar do líonra WordPress
2. Uaslódáil comhaid an bhreiseáin chuig d’eolaire `/wp-content/plugins/`
3. Gníomhachtaigh an plugin ar fud an líonra tríd an roghchlár 'Plugins' in WordPress
4. Bain taitneamh as fíricí randamacha Chuck Norris ar gach leathanach riaracháin!

Níl aon chumraíocht de dhíth - gníomhachtaigh é agus déan meangadh gáire!

## Ceisteanna Coitianta

## An dteastaíonn Ultimate Multisite uaidh seo?

Cé gur dearadh é mar bhreiseán Ultimate Multisite, oibríonn an plugin go neamhspleách agus is féidir leis rith ar aon suiteáil WordPress.

## An féidir liom mo chuid fíricí féin a chur leis?

Is féidir! Tá na fíricí réidh le haistriú. Is féidir leat do chomhad `.po` féin a chruthú le fíricí saincheaptha agus é a chur san eolaire `/lang`. Nó is féidir leat eagar na bhfíricí a scagadh go ríomhchláraithe.

## An gcuireann sé seo mo riarachán ar moill?

Ní chuireann ar chor ar bith! Tá an plugin thar a bheith éadrom agus níl aon tionchar aige ar fheidhmíocht. Ní thaispeánann sé ach téacs - gan glaonna seachtracha API ná próiseáil throm.

## An féidir liom é a dhíchumasú ar leathanaigh áirithe?

Cuirtear na fíricí i bhfolach go huathoibríoch ar leathanaigh ina bhféadfadh siad cur isteach (cosúil leis an eagarthóir bloc). Is féidir leat crúcaí WordPress a úsáid freisin chun an áit a mbíonn fíricí le feiceáil a shaincheapadh.

## An bhfuil na fíricí oiriúnach do gach lucht féachana?

Tá! Is greann glan, oiriúnach don teaghlach iad na fíricí go léir a cheiliúrann stádas finscéalach Chuck Norris.

## An féidir liom na fíricí a aistriú go teanga eile?

Cinnte! Úsáideann na fíricí go léir feidhmeanna aistriúcháin WordPress. Cruthaigh comhad aistriúcháin do do theanga agus cuir é san eolaire `/lang`. Fearann téacs: `ultimate-multisite-chuck-norris-facts`

## An oibríonn sé le multisite?

Oibríonn! Dearadh é go sonrach do líonraí multisite agus is féidir é a ghníomhachtú ar fud an líonra.
