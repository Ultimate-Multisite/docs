# Ríomhphoist {#emails}

Ligeann breiseán Ultimate Multisite: Emails d’úinéirí líonra cuntais ríomhphoist custaiméirí a sholáthar agus a bhainistiú in éineacht le suíomhanna agus ballraíochtaí. Cuireann leagan 1.1.0 bainistíocht cuntas ríomhphoist don chustaiméir, síniú isteach aonair ar ríomhphost gréasáin, tacaíocht d’atreoraithe CyberPanel, agus feabhsuithe ar sholáthar ag am checkout leis.

## Príomhghnéithe {#key-features}

* **Giuirléid cuntais ríomhphoist custaiméara**: Is féidir le custaiméirí boscaí poist a chruthú agus a bhainistiú óna leathanach cuntais tar éis dóibh ballraíocht ríomhphoist ghníomhach a bheith acu.
* **SSO ríomhphoist gréasáin agus naisc dhraíochta**: Is féidir le custaiméirí ríomhphost gréasáin a oscailt go díreach ón leathanach cuntais le fíordheimhniú aon chliceáil nuair a thacaíonn an soláthraí leis.
* **Athruithe pasfhocail**: Is féidir le custaiméirí agus riarthóirí pasfhocail bosca poist a nuashonrú ón ngiuirléid bainistíochta ríomhphoist.
* **Soláthar teidlíochta checkout**: Cruthaíonn checkout an bhallraíocht ríomhphoist, nascann sé í le fearann shuíomh an chustaiméara, agus cuireann sé uirlisí bosca poist leathanach an chuntais ar fáil.
* **Atreoraithe CyberPanel**: Is féidir le líonraí a fhaigheann tacaíocht ó CyberPanel atreoraithe ríomhphoist a chruthú agus a bhainistiú mar chuid de shreabhadh na seirbhíse ríomhphoist.
* **Socruithe atá feasach ar sholáthraí**: Scagann an leathanach socruithe soláthraithe go comhtháthuithe gníomhacha cumraithe ionas nach bhfeiceann riarthóirí ach roghanna inúsáidte.

## Riachtanais {#requirements}

* Ultimate Multisite 2.0.0 nó níos airde
* Suiteáil WordPress Multisite
* Comhtháthú soláthraí ríomhphoist cumraithe amháin ar a laghad
* Dintiúir API soláthraí le cead boscaí poist a chruthú, pasfhocail a athrú, agus naisc rochtana ríomhphoist gréasáin a eisiúint nuair a thacaítear leis

## Suiteáil {#installation}

1. Uaslódáil comhaid an plugin chuig `/wp-content/plugins/ultimate-multisite-emails`.
2. Gníomhachtaigh an breiseán ar fud an líonra ó **Plugins** i riarachán líonra WordPress.
3. Téigh go **Ultimate Multisite** > **Settings** agus oscail socruithe ríomhphoist an bhreiseáin.
4. Cumasaigh na comhtháthuithe soláthraí ríomhphoist ar mhaith leat go n-úsáidfeadh custaiméirí iad.
5. Sábháil socruithe, ansin tástáil cruthú cuntais le custaiméir neamhtháirgthe nó suíomh stáitse.

## Conas a Oibríonn Sreabhadh Díolacháin Ríomhphoist {#how-the-email-seller-flow-works}

Tá dhá shraith ar leith ag díol ríomhphoist:

1. **Teidlíocht** — táirge/ballraíocht ríomhphoist Ultimate Multisite a deir go bhfuil cead ag an gcustaiméir ríomhphost a úsáid le haghaidh fearainn shuímh, soláthraí, cuóta, agus teorann cuntas ar leith.
2. **Cruthú bosca poist** — bosca poist iarbhír an tsoláthraí, amhail `info@example.com`, a chruthaítear nuair a roghnaíonn an custaiméir nó riarthóir líonra an chuid áitiúil agus an pasfhocal.

Le linn checkout, cruthaíonn nó gníomhachtaíonn an breiseán an **bhallraíocht ríomhphoist** agus taifeadann sé fearann shuíomh an chustaiméara agus an soláthraí ar an mballraíocht sin. Ní dhéanann sé buille faoi thuairim faoi ainm bosca poist ná pasfhocal. Tar éis don cheannach a bheith críochnaithe agus don chomhthéacs suíomh/fearann a bheith ann, is féidir leis an gcustaiméir an leathanach cuntais a oscailt agus na hainmneacha bosca poist atá uathu a chruthú.

Ciallaíonn sé seo gurb é gnáthshreabhadh custaiméara:

1. Ceannaíonn an custaiméir plean suímh agus cuireann sé táirge ríomhphoist leis le linn checkout.
2. Cruthaíonn Ultimate Multisite ballraíocht an tsuímh agus an bhallraíocht ríomhphoist thánaisteach.
3. Nascann an breiseán Emails an bhallraíocht ríomhphoist le fearann shuíomh an chustaiméara agus leis an soláthraí.
4. Osclaíonn an custaiméir **Account** ina Dashboard.
5. Taispeántar an painéal **Cuntais Ríomhphoist** don bhallraíocht ríomhphoist ghníomhach sin.
6. Cliceálann an custaiméir **Cruthaigh Cuntas**, cuireann sé isteach an chuid áitiúil amhail `info`, socraíonn sé pasfhocal, agus cruthaíonn an breiseán `info@their-domain.com` ag an soláthraí cumraithe.

## Cumraigh Táirge Ríomhphoist {#configure-an-email-product}

Cruthaigh táirge ríomhphoist amháin do gach pacáiste ríomhphoist ar mhaith leat a dhíol nó a thabhairt saor in aisce. Má tá socrú táirgí nua duit, tosaigh le [Do Chéad Táirge Síntiúis a Chruthú](../../user-guide/configuration/creating-your-first-subscription-product) agus ansin fill anseo do na socruithe a bhaineann go sonrach le ríomhphost.

1. Téigh go **Network Admin > Ultimate Multisite > Products > Add Product**.
2. Socraigh **Product Type** go **Email**.
3. Cumraigh praghsáil. Úsáid praghas athfhillteach íoctha do phlean díolta, nó socraigh an táirge mar shaor in aisce/ar phraghas nialasach más plean ríomhphoist comhlántach ba cheart dó a bheith.
4. Oscail an rannán **Email Settings**.
5. Roghnaigh an **Email Provider** cumraithe.
6. Socraigh **Number of Accounts**, **Storage Quota (MB)**, teorainneacha ailiasanna, atreoraithe, agus uathfhreagróirí de réir mar is gá.
7. Sábháil an táirge.

Ní bhíonn ach soláthraithe atá cumasaithe agus cumraithe go hiomlán ar an leathanach **Integrations** le feiceáil sa roghnóir soláthraí. Má tá an roghnóir folamh, cumraigh agus tástáil an soláthraí ar dtús.

## Cuir Ríomhphost le Checkout {#add-email-to-checkout}

De ghnáth tairgtear táirgí ríomhphoist mar bhreiseáin le plean suímh. Féach [Foirmeacha Checkout](../../user-guide/configuration/checkout-forms) don sreabhadh oibre ginearálta d’eagarthóir checkout.

1. Cruthaigh nó cuir an plean suímh a cheannaíonn custaiméirí ar dtús in eagar.
2. Cuir an táirge ríomhphoist leis mar bhreiseán/order bump atá ar fáil don sreabhadh checkout sin.
3. Deimhnigh go bhfuil gnáthréimsí riachtanacha an tsuímh fós san fhoirm checkout, go háirithe réimse URL/fearainn an tsuímh.
4. Rith checkout tástála le custaiméir stáitse.
5. Tar éis checkout, oscail [leathanach cuntais](../../user-guide/client-management/account-page) an chustaiméara agus deimhnigh go bhfuil an painéal **Cuntais Ríomhphoist** le feiceáil.

Tógtar an fearann ríomhphoist ó fhearann shuíomh an chustaiméara. Ní chuireann custaiméirí an fearann ríomhphoist isteach i gcéim an táirge ríomhphoist.

## Bainistíocht Ríomhphoist Cuntais Custaiméara {#customer-account-email-management}

Cuireann leagan 1.1.0 giuirléid bainistíochta ríomhphoist le leathanach cuntais an chustaiméara. Úsáid í nuair ba cheart do chustaiméirí cruthú agus cothabháil boscaí poist dá suíomhanna féin a dhéanamh mar fhéinseirbhís.

Ón leathanach cuntais, is féidir le custaiméirí:

1. Cuntais ríomhphoist atá ceangailte lena mballraíocht ríomhphoist/fearann suímh reatha a fheiceáil.
2. Bosca poist nua a chruthú nuair a cheadaíonn a dtáirge ríomhphoist cuntais bhreise.
3. Pasfhocal bosca poist atá ann cheana a athrú.
4. Soláthraithe ríomhphoist gréasáin a dtacaítear leo a oscailt trí SSO nó logáil isteach le nasc draíochta.

Úsáideann an ghiuirléid córas reatha leathanach cuntais Ultimate Multisite, mar sin tá gníomhartha teoranta don chomhthéacs cuntais atá an custaiméir ag féachaint air. Cuireann sé seo cosc ar chustaiméirí cuntais ríomhphoist a bhaineann le ballraíocht nó suíomh eile a bhainistiú.

Mura bhfeiceann an custaiméir an painéal ríomhphoist, seiceáil go bhfuil **active Email product membership** acu, go bhfuil an membership nasctha le fearann suímh, agus go bhfuil soláthraí cumraithe ag an táirge.

## Plean Ríomhphoist Saor in Aisce a Thabhairt do Chustaiméir {#giving-a-customer-a-free-email-plan}

Chun ríomhphost a thabhairt do chustaiméir gan táille a ghearradh orthu, deonaigh an **email entitlement** ar dtús, ansin cruthaigh an bosca poist nó lig don chustaiméir é a chruthú.

Roghanna molta:

* **Cruthaigh táirge ríomhphoist saor in aisce**: socraigh praghas an táirge ríomhphoist mar shaor in aisce/nialas, ansin cuir membership gníomhach leis don chustaiméir sin ó **Ultimate Multisite > Memberships > Add Membership**. Roghnaigh an táirge ríomhphoist agus an custaiméir. Maireann memberships saor in aisce ar feadh an tsaoil go dtí go n-athraítear nó go gcuirtear ar ceal iad. Má tá sé i gceist go leanfadh an email membership suíomh atá ann cheana, cinntigh go bhfuil sé nasctha leis an gcomhthéacs suímh/fearainn sin.
* **Úsáid lascaine 100% nó nasc checkout saor in aisce**: lig don chustaiméir dul trí checkout ionas go gcruthóidh Ultimate Multisite an email membership agus an taifead íocaíochta go huathoibríoch.
* **Cruthaigh bosca poist mar riarthóir**: téigh go **Ultimate Multisite > Email Accounts > Add Email Account** chun bosca poist sonrach a chruthú ag an soláthraí do chustaiméir. Tá sé seo úsáideach don tacaíocht, ach ní thagann sé in áit an email product membership a chumasaíonn uirlisí féinseirbhíse leathanach cuntais an chustaiméara.

Más mian leat go mbainisteodh an custaiméir an bosca poist iad féin ón leathanach cuntais, cinntigh go bhfuil email membership gníomhach acu don fhearann. Is féidir le bosca poist soláthraí a cruthaíodh de láimh gan an membership sin a bheith ann, ach b’fhéidir nach bhfeicfidh an custaiméir an painéal ríomhphoist féinseirbhíse dó.

## Cuntais a Chruthaigh Riarthóir agus "Gan Uimhir Ordaithe" {#admin-created-accounts-and-no-order-number}

Cruthaíonn nó cuireann leathanach riaracháin **Email Accounts** bosca poist soláthraí in eagar. Ní hé an chéad chéim é chun plean ríomhphoist a dhíol nó a dheonú.

Agus cuntas ríomhphoist á chur in eagar, cuardaíonn an réimse **Membership** memberships Ultimate Multisite atá ann cheana de réir a n-uimhir thagartha/ordaithe. Má dhéanann tú iarracht cuntas a shannadh sula bhfuil email membership ag an gcustaiméir, b’fhéidir nach mbeidh aon uimhir ordaithe le roghnú.

Úsáid an t-ord oibríochtaí seo ina ionad:

1. Cruthaigh nó aimsigh email membership an chustaiméara. Maidir le ceannacháin, déanann checkout é seo. Maidir le plean saor in aisce, cruthaigh an membership de láimh ó [Memberships](../../user-guide/administration/managing-memberships) nó seol an custaiméir trí checkout saor in aisce.
2. Deimhnigh go bhfuil an membership gníomhach agus go n-úsáideann sé táirge Ríomhphoist.
3. Deimhnigh go bhfuil an email membership nasctha leis an bhfearann suímh/soláthraí.
4. Cruthaigh an bosca poist ó leathanach cuntais an chustaiméara nó ó **Ultimate Multisite > Email Accounts**.
5. Más gá, cuir taifead an bhosca poist in eagar agus sann an custaiméir, an membership, agus an suíomh.

Mura gá duit ach bosca poist soláthraí a chruthú don chustaiméir agus mura bhfuil bainistíocht féinseirbhíse ar an leathanach cuntais ag teastáil uait, d’fhéadfadh an cuntas a chruthaigh an riarthóir a bheith sách maith. Más mian leat go bhfeicfeadh agus go mbainisteodh an custaiméir é in Ultimate Multisite, cruthaigh an email membership freisin.

## Webmail SSO agus Logáil Isteach Magic-Link {#webmail-sso-and-magic-link-login}

Tugann Webmail SSO gníomh díreach **Oscail webmail** do chustaiméirí ón leathanach cuntais. Nuair a thacaíonn an soláthraí gníomhach leis, iarrann an addon URL rochtana gearrshaolach agus atreoraíonn sé an custaiméir isteach i webmail gan iarraidh orthu dintiúir an bhosca poist a chóipeáil.

Is féidir le hiompar soláthraí athrú:

* Is féidir le soláthraithe a bhfuil APIanna SSO dúchasacha acu seisiún webmail fíordheimhnithe díreach a chruthú.
* Is féidir le soláthraithe a bhfuil tacaíocht magic-link acu nasc logála isteach faoi theorainn ama a ghiniúint.
* Ba cheart do sholáthraithe gan tacaíocht SSO ná magic-link sonraí an bhosca poist agus gníomhartha bainistíochta pasfhocail a thaispeáint fós, ach gan cnaipe webmail aon-chliceáil.

Cumasaigh rochtain webmail amháin do sholáthraithe ina bhfuil na dintiúir API riachtanacha cumraithe agat. Má tá an gníomh ar iarraidh do sholáthraí, deimhnigh ar dtús go bhfuil an soláthraí gníomhach, cumraithe, agus go dtacaíonn sé le naisc logála isteach webmail.

## Soláthar Checkout {#checkout-provisioning}

Soláthraíonn an addon an **membership context** ríomhphoist le linn checkout. Taifeadann sé an soláthraí, teorainn cuntais, cuóta, agus fearann suímh ionas gur féidir le custaiméirí a n-ainmneacha bosca poist féin a chruthú ón leathanach cuntais tar éis ceannacháin.

Cur i bhfeidhm molta:

1. Cumraigh teorainneacha an tsoláthraí agus an táirge sula dtairgtear an táirge ríomhphoist.
2. Déan tástáil ar cheannach le táirge staging agus deimhnigh go gcruthaítear an email membership.
3. Athbhreithnigh leathanach cuntais an chustaiméara chun a dheimhniú go bhfeictear an giuirléid **Email Accounts** don fhearann.
4. Cruthaigh bosca poist tástála ón ngiuirléid.
5. Deimhnigh gníomhartha athraithe pasfhocail agus webmail sula gcumasaítear an sreabhadh do tháirgí táirgeachta.

Má theipeann ar sholáthar, is féidir leis an gcuntas custaiméara agus an membership a bheith ann fós agus leanúint riarthóra ag teastáil don email membership nó don bhosca poist soláthraí. Seiceáil logaí gníomhaíochta Ultimate Multisite agus logaí an tsoláthraí sula ndéanann tú iarracht arís.

## Atreoraithe Ríomhphoist CyberPanel {#cyberpanel-email-forwarders}

Maidir le líonraí atá tacaithe ag CyberPanel, cuireann leagan 1.1.0 tacaíocht atreoraithe ríomhphoist leis. Úsáid atreoraithe nuair a bhíonn seoltaí ar nós `info@customer-domain.test` ag teastáil ó chustaiméirí chun post a atreorú chuig bosca isteach eile gan bosca poist iomlán a chruthú.

Sula dtairgeann tú atreoraithe, deimhnigh go bhfuil comhtháthú CyberPanel cumraithe agus inrochtana. Féach an treoir [Comhtháthú CyberPanel](../../user-guide/administration/cyberpanel-integration) don óstach, ainm úsáideora, pasfhocal, agus tairisigh phacáiste riachtanacha.

## Ceisteanna Coitianta {#frequently-asked-questions}

### An féidir le custaiméirí a gcuntais ríomhphoist féin a chruthú? {#can-customers-create-their-own-email-accounts}

Is féidir. Nuair atá membership ríomhphoist ghníomhach ag an gcustaiméir do dhomain agus nuair a cheadaíonn an táirge cuntais bhreise, is féidir le custaiméirí a mboscaí poist féin a chruthú agus a bhainistiú ón leathanach Account.

### An dtacaíonn gach soláthraí le webmail SSO? {#does-every-provider-support-webmail-sso}

Ní thacaíonn. Braitheann tacaíocht SSO agus magic-link ar an soláthraí ríomhphoist gníomhach. Is féidir le soláthraithe gan API nasc-rochtana tacú fós le cruthú boscaí poist agus athruithe pasfhocail.

### An féidir le riarthóirí cuntais ríomhphoist a chruthú do chustaiméirí? {#can-administrators-create-email-accounts-for-customers}

Is féidir. Is féidir le riarthóirí cuntais ríomhphoist a chruthú ón gcomhéadan riaracháin, lena n-áirítear an pasfhocal tosaigh a shocrú agus an Account á chruthú. Maidir le bainistiú féinfhreastail custaiméirí, cinntigh freisin go bhfuil membership ríomhphoist ghníomhach ag an gcustaiméir don dámhain.

### An féidir liom Account ríomhphoist a thabhairt saor in aisce? {#can-i-give-an-email-account-away-for-free}

Is féidir. Cruthaigh táirge Ríomhphoist saor in aisce/ar phraghas nialasach agus cuir leis mar membership ghníomhach don chustaiméir é, nó lig don chustaiméir checkout a chríochnú le lascaine 100%. Tar éis don membership ríomhphoist a bheith ann, is féidir leis an gcustaiméir nó le riarthóir an bosca poist iarbhír a chruthú.

### Cén fáth a n-iarrann sannadh Account ríomhphoist uimhir ordaithe? {#why-does-assigning-an-email-account-ask-for-an-order-number}

Féachann an réimse Membership suas membership Ultimate Multisite atá ann cheana de réir a huimhreach tagartha/ordaithe. Murar cheannaigh an custaiméir táirge ríomhphoist agus mura bhfuil membership ríomhphoist cruthaithe agat de láimh, níl aon membership/ordú ríomhphoist le sannadh. Cruthaigh an membership ríomhphoist ar dtús, ansin cruthaigh nó sann an bosca poist.

### An ionann seoltóirí agus boscaí poist? {#are-forwarders-the-same-as-mailboxes}

Ní hionann. Stórálann bosca poist post agus tá dintiúir logála isteach aige. Atreoraíonn seoltóir post chuig seoladh eile agus ní sholáthraíonn sé bosca isteach ar leith.
