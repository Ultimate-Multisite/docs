# Loga Athruithe Ultimate AI Connector for Chríochphointí Comhoiriúnacha

## Leagan 2.0.0 — Eisithe ar 2026-04-24 {#version-200--released-on-2026-04-24}

* Nua: Tacaíocht ilsholáthraí — cumraigh ilchríochphointí AI agus stiúir iarratais le cúlú uathoibríoch thar sholáthraithe.
* Ceartú: Comhtháthú SDK ilsholáthraí le IDanna soláthraí cearta, URLanna cláraithe, agus scóip scagaire HTTP in aghaidh an tsoláthraí.
* Ceartú: Leathnaíonn cárta soláthraí nua go huathoibríoch anois nuair a chuirtear leis é; neamhbhailíochtú taisce scripte ar nuashonrú plugin.
* Ceartú: Ainmspás aicme soláthraí dinimiciúil le haghaidh eval() agus iompar uathleathnaithe do sholáthraithe nua-churtha leis.
* Ceartú: Úsáideann comhpháirteanna cobhsaí Card/CardBody/CardHeader/CardDivider (ní turgnamhach a thuilleadh) le haghaidh comhoiriúnacht WordPress 6.9+.
* Ceartú: Cuireadh deilbhín greim unicode in ionad DragHandle neamhshainithe le haghaidh tarraingt chun soláthraithe a athordú.
* Ceartú: Cuireadh deireadh le hiarratas blocála HTTP a lasadh ar gach luchtú leathanaigh.
* Feabhsaithe: Uasghrádaíodh sreafaí oibre GitHub Actions go Node.js 24.
