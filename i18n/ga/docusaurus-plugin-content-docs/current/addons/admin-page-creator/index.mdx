# Admin Page Creator

## Ábhar
Ligfidh Admin Page Creator duit leathanaigh riaracháin nua agus fo-leathanaigh a chur leis go héasca laistigh de dashboard riaracháin WordPress. Is féidir é a úsáid chun leathanach fáilte nó leathanach teagaisc a chruthú a d’fhéadfadh feidhmiú mar leathanach tuirlingthe d’úsáideoirí nua. Is féidir leat tógálaí leathanaigh cosúil le Beaver Builder, Elementor, Brizy, agus Oxygen a úsáid chun na leathanaigh a dhearadh gan stró.

Cuireann sé tacaíocht shaincheaptha HTML, CSS agus JavaScript ar fáil freisin, chomh maith le rialú rochtana bunaithe ar phlean agus ar ról úsáideora!

## Gnéithe
* Tacaíocht do HTML agus CSS saincheaptha
* Tacaíocht JavaScript
* Mód Ábhair PHP
* Cead bunaithe ar ról Úsáideora
* Tacaíocht do thógálaithe leathanaigh

## Doiciméadú
[Leathanaigh Riaracháin Shaincheaptha agus Fo-leathanaigh a chruthú le Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Suiteáil

1. Uaslódáil **ultimate-multisite-admin-page-creator** chuig an gcomhadlann **/wp-content/plugins/**
2. Gníomhachtaigh an breiseán tríd an roghchlár **Breiseáin** i do WordPress Network Admin

## Treoir Úsáide

Cé go ndéanann WordPress an-éasca ábhar a chur le ceann tosaigh suíomhanna, teastaíonn roinnt bunscileanna códaithe chun leathanaigh agus fo-leathanaigh a chruthú ar thaobh an riarthóra. Tá sé mar aidhm ag Ultimate Multisite: Admin Page Creator an éascaíocht úsáide chéanna atá ag an gceann tosaigh a thabhairt chuig ceann cúil do shuíomhanna líonra.

## Cén fáth Admin Page Creator a úsáid?

B’fhéidir gur mhaith leat leathanach saincheaptha a thairiscint ar phainéal riaracháin do chliaint le teagaisc, nó b’fhéidir gur mhaith leat leathanach saincheaptha a chur leis le Ceisteanna Coitianta agus giuirléid teagmhála. Tá na féidearthachtaí gan teorainn le Admin Page Creator, agus sin é a fhiosróimid i dteagasc an lae inniu.

## Do chéad Leathanach Riaracháin Saincheaptha a chruthú

Tar éis Ultimate Multisite: Admin Page Creator a shuiteáil agus a ghníomhachtú, téigh chuig do roghchlár riaracháin líonra agus téigh chuig Leathanaigh Riaracháin.

![Liosta Leathanaigh Riaracháin Saincheaptha ar Network Admin](/img/admin/admin-pages-list.png)

_Liosta Leathanaigh Riaracháin Saincheaptha ar an Network Admin_

Feicfidh tú liosta de na leathanaigh riaracháin go léir atá cruthaithe agat go dtí seo (ós é seo do chéad chuairt, beidh an liosta folamh). Lean ar aghaidh agus cliceáil **Cuir Leathanach Riaracháin nua leis**. Athdhíreofar thú chuig eagarthóir an Leathanaigh Riaracháin.

## Roghanna Roghchláir

Ar an taobh, sa mheiteabhosca Roghanna Ginearálta, beidh tú in ann cineál an roghchláir, an lipéad, agus i bhfad níos mó a chumrú.

### Cineálacha Roghchláir

Is féidir dhá chineál roghchláir éagsúla a bheith ag do leathanach riaracháin saincheaptha nua: roghchlár barrleibhéil agus fo-roghchlár.

Mar a thugann an t-ainm le fios, cuirfidh leathanach roghchláir barrleibhéil do leathanach nua-chruthaithe leis an roghchlár riaracháin in éineacht leis na leathanaigh réamhshocraithe WordPress.

Má roghnaíonn tú an rogha barrleibhéil, beidh tú in ann ord an roghchláir agus deilbhín roghchláir a roghnú freisin.

Más mian leat an leathanach seo a dhéanamh ina leathanach fo-roghchláir, beidh tú in ann leathanach tuismitheora a roghnú chun an leathanach seo a cheangal leis. Is féidir leat do leathanach saincheaptha a chur le leathanaigh réamhshocraithe WordPress nó le do leathanaigh riaracháin shaincheaptha barrleibhéil.

## Ardroghanna

Ba cheart go mbeifeá in ann gach cineál ábhair a chur le do leathanaigh riaracháin shaincheaptha. Áirítear leis sin rialacha agus comhaid CSS, JavaScript, agus leabharlanna seachtracha freisin. Is féidir leat é sin a dhéanamh ag úsáid an Táb Ardroghanna.

### CSS Saincheaptha

Ar an táb CSS, is féidir leat do chód CSS saincheaptha féin a chur leis, chomh maith le comhaid CSS a iompórtáil ó shuíomhanna cianda (cosúil le CDNanna, Google Fonts, srl.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Tacaítear le CSS Saincheaptha freisin. Is féidir leat comhaid stíle sheachtracha a chur san áireamh freisin!_

### JavaScript Saincheaptha

Tá an rud céanna bailí do JavaScript. Is féidir leat leabharlanna seachtracha a chur leis le húsáid ar do chód JavaScript saincheaptha.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Cuir do chód JavaScript féin leis!_

## Ceadanna

Sna hArdroghanna freisin, beidh tú in ann a roghnú cé na fo-shuíomhanna a mbeidh rochtain acu ar an leathanach seo. Tá sé seo úsáideach má tá sé ar intinn agat ábhar eisiach éagsúil a chur ar fáil do shraitheanna plean éagsúla, mar shampla.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Tá smacht iomlán agat ar an áit a mbeidh an leathanach seo le feiceáil_

**Tábhachtach** : Ní mór an dá choinníoll a chomhlíonadh chun an leathanach a thaispeáint. Ciallaíonn sé sin má tá an t-úsáideoir ina chliant de phlean ar leith, ach mura bhfuil ceann de na róil roghnaithe aige (nó a mhalairt), ní bheidh an leathanach le feiceáil ar a phainéal.

## Gníomhach

Más mian leat leathanach riaracháin saincheaptha a dhíchumasú gan é a scriosadh ná cur isteach ar na ceadanna, níl le déanamh ach an rogha Gníomhach a dhíthiceáil.

Má dhíghníomhachtaítear leathanach barrleibhéil, díghníomhachtófar go huathoibríoch na fo-leathanaigh go léir atá ceangailte leis an máthairleathanach sin.
