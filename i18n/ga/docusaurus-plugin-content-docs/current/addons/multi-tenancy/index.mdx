# Iltionóntacht

Dáil fo-shuíomhanna thar fhreastalaithe agus bunachair sonraí ar mhaithe le feidhmíocht agus slándáil

## Buaicphointí eisiúna leagan 1.2.0 {#version-120-release-highlights}

Cuireann Multi-Tenancy 1.2.0 an bhunchloch tionónta cheannasaigh leis do shreafaí oibre leithlisithe táirgthe:

- Uathlogáil isteach tionónta gan staid le cuairteanna SSO gearrshaolacha atá cosanta ar athsheinm.
- Leithlisiú bunachair sonraí agus córais comhad de réir fo-shuíomh do thionóntaí ceannasacha.
- Cumas iltionóntachta Hostinger do sholáthar tionónta atá feasach ar óstach.
- Geataí fíoraithe imirce ceannasaí do sheiceálacha sonraí oidhreachta agus do phróiseáil bhrú neamhshioncronach.
- Ceangal óstaigh bunachair sonraí inchumraithe ar an meaisín céanna do shuiteálacha Bedrock, FrankenPHP, agus localhost.
- Feabhsuithe ar thúsú tionónta, scríbhneoir bunachair sonraí, túsú scéime, agus soláthar úsáideoirí.
- Díchoimisiúnú ceannasach níos sábháilte a ghlanann dintiúir tionónta.

Féach na treoracha úsáideora don [sreabhadh oibre imirce ceannasaí](/user-guide/getting-started/sovereign-migration-workflow), [leithlisiú iltionóntachta](/user-guide/configuration/multi-tenancy-isolation), agus [oibríochtaí tionónta cheannasaigh](/user-guide/administration/sovereign-tenant-operations).

## Riachtanais {#requirements}

- WordPress 5.3 nó níos airde
- PHP 7.4 nó níos airde
- breiseán Multisite Ultimate (gníomhach)

## Suiteáil {#installation}

1. Uaslódáil comhaid an bhreiseáin go dtí d’eolaire `/wp-content/plugins/`
2. Gníomhachtaigh an breiseán tríd an roghchlár 'Plugins' in WordPress
3. Cumraigh socruithe an bhreiseáin i riarachán an líonra

## Forbairt {#development}

### Socrú {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Tástáil {#testing}

Cuimsíonn an breiseán creat tástála cuimsitheach:

- **Tástálacha Aonaid**: Tástálacha bunaithe ar PHPUnit le sraith tástála WordPress
- **Bunrang Tástála**: Soláthraíonn sé modhanna cabhracha do chásanna tástála coitianta
- **Clúdach Tástála**: Cumraithe chun clúdach cóid a rianú
- **Tástáil Multisite**: Ritheann tástálacha i dtimpeallacht multisite

### Caighdeáin Chóid {#code-standards}

- **Caighdeáin Chódaithe WordPress**: Forfheidhmithe trí PHPCS
- **Comhoiriúnacht PHP 7.4+**: Tacaítear le gnéithe nua-aimseartha PHP
- **Anailís Statach**: Comhtháthú PHPStan do sheiceáil cineáil
- **Nuachóiriú Cóid**: Rialacha Rector d’uasghráduithe PHP

### Córas Tógála {#build-system}

- **Próiseáil Sócmhainní**: Mionlaghdú do chomhaid CSS/JS
- **Aistriúchán**: Giniúint comhaid POT
- **Pacáistiú**: Cruthú cartlainne le haghaidh dáileacháin
- **Mód Forbartha**: Sócmhainní neamh-mhionlaghdaithe le haghaidh dífhabhtaithe

## Cumraíocht {#configuration}

Tacaíonn an breiseán le roghanna cumraíochta éagsúla atá inrochtana trí chomhéadan riaracháin an líonra.

### Cumraíocht tionónta cheannasaigh {#sovereign-tenant-configuration}

Do thionóntaí ceannasacha, cumraigh bunachar sonraí an tionónta, fréamh an chórais comhad, ceangal óstaigh an bhunachair sonraí, agus cumas an óstaigh sula ndéantar trácht a ródú chuig an tionónta. Rith fíorú imirce tar éis gach athrú a théann i bhfeidhm ar dhintiúir bunachair sonraí, ceangal óstaigh, ródú, nó soláthar úsáideoirí.

Úsáid `localhost` amháin nuair a bhíonn am rite an tionónta agus deontais an bhunachair sonraí ag súil le soicéad áitiúil nó rochtain ar an meaisín céanna. Úsáid `127.0.0.1` nó óstainm seirbhíse nuair is é sin an t-óstach deontais atá cumraithe ag do sholáthraí.

### Cumas Hostinger {#hostinger-capability}

Oibríonn cumas iltionóntachta Hostinger leis an gcomhartha comhtháthaithe comhroinnte Hostinger. Deimhnigh go bhfuil an fearann i láthair in hPanel, go ndíríonn DNS ar an gcuntas Hostinger, agus go meaitseálann socruithe bhunachar sonraí an tionónta hPanel roimh fhíorú.

## Hooks agus Scagairí {#hooks-and-filters}

### Gníomhartha {#actions}

- `ultimate-multisite-addon-template_init` - Spreagtar é tar éis thúsú an bhreiseáin
- `ultimate-multisite-addon-template_loaded` - Spreagtar é tar éis gach comhad den bhreiseán a luchtú

### Scagairí {#filters}

- `ultimate-multisite-addon-template_settings` - Athraigh socruithe an bhreiseáin
- `ultimate-multisite-addon-template_enabled` - Sáraigh stádas cumasaithe/díchumasaithe an bhreiseáin
