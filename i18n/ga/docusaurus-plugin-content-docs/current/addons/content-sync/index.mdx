# Sioncronú Ábhair {#content-sync}

Multisite Ultimate: Sioncronú Ábhair

Ligeann an breiseán seo do Multisite Ultimate do riarthóirí líonra ábhar a shioncronú ó aon suíomh chuig suíomhanna eile sa líonra. Tacaíonn sé le gach cineál postála, lena n-áirítear postálacha, leathanaigh, cineálacha postála saincheaptha, agus teimpléid Elementor. Áirítear leis an mbreiseán comhéadan draoi ilchéime chun ábhar agus suíomhanna sprice a roghnú, le próiseáil i mbaisceanna agus aiseolas dul chun cinn i bhfíor-am.

## Gnéithe {#features}

* Roghnú Ábhair: Greille amhairc cártaí le mionsamhlacha agus scagadh de réir cineáil postála
* Gach Cineál Postála: Sioncronaigh postálacha, leathanaigh, táirgí, teimpléid Elementor, agus aon chineál postála saincheaptha
* Roghnú Suímh: Roghnaigh cé na suíomhanna ar cheart ábhar a shioncronú leo, le suíomh foinseach eisiata agus suíomhanna páiste réamhroghnaithe
* Próiseáil i mBaisceanna: Sioncronaíonn sé le 10 suíomh ag an am le nuashonruithe dul chun cinn
* Sioncronú Meán: Cóipeálann agus athmhapálann sé comhaid mheán (íomhánna, físeáin) go huathoibríoch ó ábhar postála chuig suíomhanna sprice
* Sioncronú Tacsanomaíochta: Caomhnaíonn sé catagóirí, clibeanna, agus tacsanomaíochtaí saincheaptha
* Tacaíocht Elementor: Tacaíocht iomlán do theimpléid Elementor le CSS a athghiniúint agus IDanna teimpléid a athmhapáil
* Réiteach Coimhlinte: Rogha chun ábhar atá ann cheana a ionadú nó é a scipeáil
* Bainistíocht Taisce: Glanann sé taisce Elementor go huathoibríoch ar shuíomhanna sprice tar éis sioncronaithe

## Riachtanais {#requirements}

* WordPress 5.3 nó níos airde
* PHP 7.4 nó níos airde
* breiseán Multisite Ultimate (gníomhach)
* breiseán Elementor (roghnach, riachtanach le haghaidh sioncronú teimpléad Elementor)

## Suiteáil {#installation}

1. Uaslódáil 'ultimate-multisite-content-sync' chuig an gcomhadlann '/wp-content/plugins/'
2. Gníomhachtaigh an breiseán tríd an roghchlár 'Breiseáin' i do Riarachán Líonra WordPress
3. Cumraigh socruithe in Ultimate Multisite → Socruithe → Sioncronú Ábhair
4. Téigh chuig Ultimate Multisite → Sioncronú Ábhair chun tosú ar shioncronú
