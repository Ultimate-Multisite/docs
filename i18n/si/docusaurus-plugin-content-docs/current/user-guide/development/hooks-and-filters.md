# Ultimate Multisite Hooks සහ Filters {#ultimate-multisite-hooks-and-filters}

Ultimate Multisite කියන්නේ ගොඩක් ෆන්ක්ෂන් (functions) සහ දත්ත සමුදා ප්‍රශ්න (database queries) වලින් සමන්විත දෙයක්. මේකේ hook's සහ filters තියෙනවා, ඒ හරහා දැනට පවතින ෆන්ක්ෂන් වල වෙනස්කම් කරන්න හෝ අලුත් කේත (code) එකක් එකතු කරන්න පුළුවන්.

ඔබේ ජාලය (network) තවත් වෙනස් කරගන්න උදව් වෙන්න අපි මේ hook's සහ filters ලැයිස්තුවක් සාදා තිබෙනවා. දියුණු පරිශීලකයින්ට ඔවුන්ගේ ජාලය තවදුරටත් කස්ටමයිස් කරන්න මේකෙන් ප්‍රයෝජන ගන්න පුළුවන්. මේ ලැයිස්තුව Github එකේ පහත link එකෙන් බලන්න පුළුවන්.
