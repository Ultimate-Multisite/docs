# සැලසුම් සඳහා බෙදාගත හැකි සබැඳි කළමනාකරණය කිරීම {#managing-shareable-links-for-plans}

Ultimate Multisite ඔබට ඔබේ ප්ලෑන් (plan) සබැඳි සෘජුවම ඔබේ පාරිභෝගිකයින් සමඟ බෙදා ගැනීමට විකල්පයක් ලබා දෙයි. මෙය කිරීමට, ඔබගේ ඕනෑම ප්ලෑන් එකක සංස්කරණ පිටුවට ගොස් '_Shareable Link එක පිටපත් කරන්න' (Click to copy the Shareable Link) බොත්තම ක්ලික් කරන්න.

![shareable link button on product edit page](/img/config/product-edit-full.png)

_ප්ලෑන් සංස්කරණ පිටුවේ, බෙදාගත හැකි සබැඳි නිර්මාණය කිරීමට ඔබට බොත්තම සොයාගත හැකිය._

මෙම ක්‍රියාවලිය සබැඳිය ගොඩනැගීමට පෙරනිමි බිල් කිරීමේ වාරික කාලය (default billing frequency) භාවිතා කරන බව කරුණාවෙන් සලකන්න. මෙයින් අදහස් කරන්නේ ඔබේ පෙරනිමි බිල් කිරීම මාසික නම්, උදාහරණයක් ලෙස, අවසානයේ ඔබට මෙවැනි සබැඳියක් ලැබෙනු ඇත: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

ඔබට බෙදාගත හැකි සබැඳිය කාර්තර් (quarterly) සඳහා පෙරනිමිය කිරීමට අවශ්‍ය නම්, අවසානයේ 3 එකක් එක් කළ යුතුය.

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

වසරකට _premium_ ප්ලෑන් තෝරා ගැනීමට, ඔබ 1 එක සහ 'year' යන වචනය එකතු කළ යුතුය:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

විවිධ බිල් කිරීමේ වාරික කාලයන් සඳහා එක් සබැඳියක් පමණක් තිබිය හැකි බව මතක තබා ගන්න. ඔබේ සෑම බිල් කිරීමේ වාරික කාලයක් සඳහාම නිශ්චිත සබැඳියක් බෙදා ගත යුතුය.

ඔබ පාරිභෝගිකයින් සමඟ සබැඳිය බෙදාගත් පසු, එය ඔවුන්ව ඔබගේ ලියාපදිංචීමේ ප්‍රවාහයේ (signup flow) පළමු පියවරට යොමු කරනු ඇත.
