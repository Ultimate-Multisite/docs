# වෙබ් අඩවි සැකිලිවල ස්වයංක්‍රීය සෙවීම සහ ආදේශ කිරීම (v2) {#auto-search-and-replace-in-site-templates-v2}

_**මෙම මාර්ගෝපදේශයට WP UItimo version 2.x අවශ්‍ය වේ.**_

Ultimate Multisite හි ඇති වඩාත්ම බලවත් විශේෂාංග වලින් එකක් නම්, ලියාපදිංචි කිරීමේ පෝරමය සඳහා ඕනෑම පෙළක්, වර්ණයක් සහ තෝරාගත හැකි ක්ෂේත්‍ර (fields) එක් කිරීමට ඇති හැකියාවයි. අපට එම දත්ත ලබාගත් පසු, එය තෝරාගත් වෙබ් අඩවි සැකිල්ලේ ඇතැම් කොටස්වල අන්තර්ගතය පෙර පිරවීමට (pre-populate) භාවිතා කළ හැකිය. ඉන්පසු නව වෙබ් අඩවිය ප්‍රකාශයට පත් කරන විට, Ultimate Multisite විසින් ලියාපදිංචි කිරීමේදී ඇතුළත් කරන ලද සැබෑ තොරතුරු භාවිතයෙන් එම placeholders (තනතුරු) ප්‍රතිස්ථාපනය කරනු ඇත.

උදාහරණයක් ලෙස, ඔබට ඔබේ සැකිලි වෙබ් අඩවිවල placeholder (තනතුරු) භාවිතා කළ හැකිය. Placeholder (තනතුරු) ද්විත්ව වරලින් වරින් වට囲ා තිබිය යුතුය - {{placeholder_name}}.

ඉන්පසු, එම දත්ත ලබා ගැනීමට ගැලපෙන ලියාපදිංචි කිරීමේ ක්ෂේත්‍රයක් සරලව එකතු කළ හැකිය.

ඔබේ පාරිභෝගිකයාට ලියාපදිංචි කිරීමේදී එම ක්ෂේත්‍රය පුරවා ගත හැකිය.

ඉන්පසු Ultimate Multisite විසින් පාරිභෝගිකයා විසින් සපයන දත්ත භාවිතයෙන් placeholders (තනතුරු) ස්වයංක්‍රීයව ප්‍රතිස්ථාපනය කරනු ඇත.

## **"placeholder වලින් පිරුණු සැකිල්ලක්" යන ගැටලුව විසඳීම** {#solving-the-template-full-of-placeholders-problem}

සියල්ල විශිෂ්ටයි, නමුත් අපට නරක ප්‍රශ්නයකට මුහුණ දීමට සිදු වේ: දැන් අපගේ වෙබ් අඩවි සැකිලි - අපගේ පාරිභෝගිකයින්ට පිවිසීමට හැකි ඒවා - තොරතුරු නොදෙන නරක placeholders වලින් පිරී ඇත.

එය විසඳීම සඳහා, අපි placeholders (තනතුරු) සඳහා අසත්‍ය අගයන් (fake values) සැකසීමේ විකල්පයක් ලබා දෙන්නෙමු, එම අගයන් භාවිතයෙන් පාරිභෝගිකයින් වෙබ් අඩවි භාවිතා කරන අතරතුර සැකිලි වෙබ් අඩවිවල ඒවායේ අන්තර්ගතය සෙවීමට සහ ප්‍රතිස්ථාපනය කිරීමට (search and replace) අපට හැකිය.

ඔබට template placeholders editor වෙත ප්‍රවේශ වීමට **Ultimate Multisite > Settings > Sites** වෙත ගොස්, Site Template Options ප්‍රදේශයට ස්ක්‍රෝල් කර, ඉන්පසු **Edit Placeholders** සබැඳිය ක්ලික් කිරීමෙන් ප්‍රවේශ විය හැකිය.

![Sites settings page හි Site Template Options ප්‍රදේශය](/img/config/settings-sites-templates-section.png)

එයින් ඔබව placeholders වල අන්තර්ගත සංස්කරණ (content editor) වෙත ගෙන යනු ඇත, එහිදී ඔබට placeholders සහ ඒවායේ අදාළ අන්තර්ගතය එකතු කළ හැකිය.
