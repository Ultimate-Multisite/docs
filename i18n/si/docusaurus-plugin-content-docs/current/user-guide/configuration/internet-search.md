# අන්තර්ජාල සෙවුම

Gratis AI Agent v1.5.0 එකේ **Internet Search** හැකියාව එකතු කර තිබෙනවා. මේකෙන් කතා කරන අතරතුර AI සහායකයාට වෙබ් අඩවියෙන් යාවත්කාලීන තොරතුරු ලබාගන්න පුළුවන්. මේක ක්‍රියාත්මක වෙන්නේ [Brave Search API](https://brave.com/search/api/) හෝ [Tavily API](https://tavily.com/) මගින්.

## එය ක්‍රියා කරන ආකාරය

අන්තර්ජාල සෙවුම සක්‍රිය කළ විට, ප්‍රශ්නයකට වත්මන් හෝ බාහිර තොරතුරු අවශ්‍ය බව AI විසින් තීරණය කරන විට, එය ඔබ සැකසූ සෙවුම් සපයන්නා (search provider) වෙත ස්වයංක්‍රීයව විමසීම් (query) යොමු කළ හැකියි. උදාහරණයක් ලෙස, මෑත පුවත්, සජීවී මිල ගණන් හෝ මාදිලියේ පුහුණු කිරීමේ කාලය අවසන් වී ඇති නිසා වෙනස් විය හැකි ලේඛන වැනි දේවල් සඳහා මෙය භාවිතා වේ.

ප්‍රතිඵල තත්‍ය කාලයේදී (real time) ලබාගෙන, පිළිතුරක් ජනනය කිරීමට පෙර AI සහායකයාගේ context එකට ඇතුළත් කරනු ලැබේ. ප්‍රශ්නයකට පිළිතුරු දීමට සෙවුම් ප්‍රතිඵල භාවිතා කළ බව AI දැනුම් දෙයි.

## අන්තර්ජාල සෙවුම සක්‍රිය කිරීම

අන්තර්ජාල සෙවුමට ඔබ තෝරාගත් සෙවුම් සපයන්නා (search provider) වෙතින් API key එකක් අවශ්‍ය වේ. එය වින්‍යාස කිරීමට:

1. **Gratis AI Agent → Settings → Advanced** වෙත යන්න.
2. **Internet Search Provider** පහළ ඇති dropdown එකෙන් **Brave Search** හෝ **Tavily** තෝරන්න.
3. අදාළ ක්ෂේත්‍රයේ ඔබේ API key එක ඇතුළත් කරන්න. සෑම ක්ෂේත්‍රයක් අසලම ලියාපදිංචි වීමේ සබැඳි (Sign-up links) පෙන්වනු ලැබේ.
4. **Save Settings** ක්ලික් කරන්න.

Key එක සුරැකූ පසු, Internet Search හැකියාව ස්වයංක්‍රීයව AI සහායකයාට ලබා ගත හැක.

## Brave Search

### Brave Search API Key ලබා ගැනීම

1. [Brave Search API page](https://brave.com/search/api/) වෙත පිවිසෙන්න.
2. සැලැස්මක් (plan) සඳහා ලියාපදිංචි වන්න. මාසික ඉල්ලීම් සීමාවකින් යුත් නොමිලේ භාවිතා කළ හැකි මට්ටමක් (free tier) තිබේ.
3. Brave Search developer dashboard එකෙන් ඔබේ API key එක පිටපත් කරන්න (copy).
4. Gratis AI Agent settings වල **Brave Search API Key** ක්ෂේත්‍රයට එය අලවන්න (paste).

### භාවිත සීමාවන්

භාවිතාව බ්‍රේව් සර්ච් (Brave Search) මගින් සිදු කරන ප්‍රශ්න ගණන මත බිල දාශනය කරනු ලැබේ. සෙවුම් කිරීමක් ඇති කරන සෑම AI ප්‍රතිචාරයක්ම එක් ප්‍රශ්නයක් ලෙස ගණන් ගනී. අනපේක්ෂිත අය කිරීම් වළක්වා ගැනීම සඳහා [Brave Search developer dashboard](https://brave.com/search/api/) හි ඔබේ භාවිතය නිරීක්ෂණය කරන්න.

## Tavily

Superdav AI Agent v1.10.0 මගින් වඩාත් පොහොසත් සෙවුම් ප්‍රතිඵල සහ දියුණු පර්යේෂණ හැකියාවන් ලබා දෙන වෙනත් අන්තර්ජාල සෙවුම් සපයන්නෙකු ලෙස **Tavily** එක එක් කරයි.

### Tavily API Key ලබා ගැනීම

1. [Tavily API page](https://tavily.com/) වෙත පිවිසෙන්න.
2. ගිණුමක් සඳහා ලියාපදිංචි වන්න. මාසික ඉල්ලීම් සීමාවකින් යුත් නොමිලේ මට්ටමක් තිබේ.
3. Tavily dashboard එකෙන් ඔබේ API key පිටපත් කරන්න.
4. Gratis AI Agent සැකසුම් වල **Tavily API Key** ක්ෂේත්‍රයට එය අලවන්න (Paste).

### භාවිත සීමාවන්

භාවිතාව Tavily මගින් සිදු කරන API ඇමතුම් ගණන මත බිල දාශනය කරනු ලැබේ. සෙවුමක් ඇති කරන සෑම AI ප්‍රතිචාරයක්ම එක් ඇමතුමක් ලෙස ගණන් ගනී. අනපේක්ෂිත අය කිරීම් වළක්වා ගැනීම සඳහා [Tavily dashboard](https://tavily.com/) හි ඔබේ භාවිතය නිරීක්ෂණය කරන්න.

## අන්තර්ජාල සෙවීම අක්‍රිය කිරීම

API key එක ක්‍රියාකාරී සෙවුම් සපයන්නා ක්ෂේත්‍රයෙන් ඉවත් කර සුරකින්න (Save). පසුව, සහායකයාට ප්‍රතිචාර ජනනය කිරීමට පෙර සෙවුම් ප්‍රතිඵල බලා සිටීම නිසා ප්‍රතිචාරවල ප්‍රමාදයක් (latency) ඇති වේ. කාලය මත රඳා පවතින භාවිත අවස්ථා සඳහා, තත්‍ය කාලීන සෙවීම අවශ්‍යද නැද්ද යන්න හෝ සහායකයාගේ අභ්‍යන්තර දැනුම ප්‍රමාණවත් දැයි ඔබ සලකා බැලිය යුතුය.
