# තේමාව මත පදනම් වූ කුසලතා {#theme-aware-skills}

Superdav AI Agent v1.10.0 හි නව තේමාව මත අවධානය යොමු කරන (theme-aware) අභ්‍යන්තර කුසලතා හතරක් ඇතුළත් කර ඇත. මෙම කුසලතා ඔබේ WordPress තේමාවට ස්වයංක්‍රීයව ගැලපෙන පරිදි විශේෂිත මඟ පෙන්වීම් සහ හැකියාවන් සපයයි.

## තේමාව මත පදනම් වූ කුසලතා යනු මොනවාද? {#what-are-theme-aware-skills}

තේමාව මත පදනම් වූ කුසලතා කියන්නේ ඔබේ වෙබ් අඩවියේ දැනට ක්‍රියාත්මක වන WordPress තේමාව අනුව AI සහායකයා ස්වයංක්‍රීයව තෝරා ගන්නා පෙර සැකසූ දැනුම් දත්ත සමුදායන් (knowledge bases) සහ මෙවලම් කට්ටල (tool sets) වේ. ඔබ තේමා මාරු කරන විට, AI සහායකයාට තිබෙන කුසලතා ස්වයංක්‍රීයව යාවත්කාලීන වේ — අතින් කිසිදු සැකසුමක් අවශ්‍ය නොවේ.

එක් එක් කුසලතාවට ඇතුළත් වන්නේ:

- **තේමාව සඳහා විශේෂිත ලේඛන** — තේමාව භාවිතා කර වෙනස් කිරීමට මඟ පෙන්වීම්
- **Block සහ pattern යොමු කිරීම්** — ලබා ගත හැකි blocks, patterns සහ නිර්මාණ විකල්ප
- **වෙනස් කිරීමේ උදාහරණ** — පොදු කාර්යයන් සඳහා කේත කොටස් (code snippets) සහ සැකසුම් රටා
- **හොඳම භාවිතයන් (Best practices)** — තේමාවේ ගෘහ නිර්මාණ ශිල්පය සහ වැඩ ප්‍රවාහය පිළිබඳ නිර්දේශ

## ලබා ගත හැකි තේමාව මත පදනම් වූ කුසලතා {#available-theme-aware-skills}

### Block Themes (බ්ලොක් තේමා) {#block-themes}

**අදාළ වන්නේ:** WordPress block-based (Full Site Editing) ගෘහ නිර්මාණ ශිල්පය භාවිතා කරන තේමා සඳහා.

Block Themes කුසලතාව මගින් පහත දේ පිළිබඳ මඟ පෙන්වීම් ලබා දෙයි:

- block editor භාවිතයෙන් template නිර්මාණය කිරීම සහ සංස්කරණය කිරීම
- block patterns සහ නැවත භාවිතා කළ හැකි blocks සමඟ වැඩ කිරීම
- ගෝලීය විලාසිතා (global styles) සහ theme.json සැකසුම් වෙනස් කිරීම
- theme blocks සහ variations භාවිතා කිරීම
- ඔබේ වෙබ් අඩවිය සඳහා custom block patterns ගොඩනැගීම

**ස්වයංක්‍රීයව සක්‍රිය වන විට:** ඔබේ ක්‍රියාකාරී තේමාව block theme එකක් වූ විට ( `block-templates` විශේෂාංගය සඳහා සහාය වේ).

### Classic Themes (ප්‍රාථමික තේමා) {#classic-themes}

**අදාළ වන්නේ:** PHP templates සහ සම්භාව්‍ය editor භාවිතා කරන සාම්ප්‍රදායික WordPress තේමා සඳහා.

Classic Themes කුසලතාව මගින් පහත දේ පිළිබඳ මඟ පෙන්වීම් ලබා දෙයි:

PHP template ගොනු සහ hooks සමඟ වැඩ කිරීම
කස්ටම් මෝස්තරකරණය (Customizer) හරහා තේමයේ පෙනුම වෙනස් කිරීම
widget areas සහ sidebars භාවිතා කිරීම
CSS සංශෝධනය සහ child theme සංවර්ධනය
theme hierarchy සහ template tags අවබෝධ කර ගැනීම

**මෙය ස්වයංක්‍රීයව සක්‍රිය වේ:** ඔබගේ ක්‍රියාකාරී තේමාව classic (non-block) තේමාවක් වූ විට.

### Kadence Blocks {#kadence-blocks}

**අදාළ වන්නේ:** උසස් block-based නිර්මාණ සඳහා Kadence Blocks plugin භාවිතා කරන වෙබ් අඩවි වලට.

Kadence Blocks කුසලතාව මගින් පහත දේ පිළිබඳ මඟ පෙන්වීමක් ලබා දෙයි:

- Kadence හි උසස් block library (Hero, Testimonials, Pricing වැනි) භාවිතා කිරීම
- Kadence block සැකකිරීම් සහ responsive විකල්ප වින්‍යාස කිරීම
- Kadence ග්‍රිඩ් සහ container blocks භාවිතයෙන් කස්ටම් layout නිර්මාණය කිරීම
- ඔබේ තේමාව සමඟ Kadence blocks ඒකාබද්ධ කිරීම
- Kadence හි design system සහ presets ප්‍රයෝජනයට ගැනීම

**මෙය ස්වයංක්‍රීයව සක්‍රිය වේ:** Kadence Blocks plugin ඔබගේ වෙබ් අඩවිය මත ක්‍රියාත්මක වූ විට.

### Kadence Theme {#kadence-theme}

**අදාළ වන්නේ:** block-based නිර්මාණය සහ කස්ටම් කිරීම සඳහා Kadence theme භාවිතා කරන වෙබ් අඩවි වලට.

Kadence Theme කුසලතාව මගින් පහත දේ පිළිබඳ මඟ පෙන්වීමක් ලබා දෙයි:

- global styles සහ theme.json හරහා Kadence theme සංවර්ධනය කිරීම
- Kadence හි built-in block patterns සහ templates භාවිතා කිරීම
- Kadence theme සැකකිරීම් සහ විකල්ප වින්‍යාස කිරීම
- Kadence design system භාවිතයෙන් කස්ටම් නිර්මාණ ගොඩනැගීම
- ජනප්‍රිය plugins සහ tools සමඟ Kadence ඒකාබද්ධ කිරීම

**මෙය ස්වයංක්‍රීයව සක්‍රිය වේ:** Kadence theme ඔබගේ ක්‍රියාකාරී තේමාව වූ විට.

## කුසලතා තෝරා ගැනීම (How Skills Are Selected) {#how-skills-are-selected}

සෑම පණිවිඩයකදීම සහ ඔබගේ ක්‍රියාකාරී තේමාව සහ ස්ථාපනය කර ඇති plugins ස්වයංක්‍රීයව හඳුනා ගනී. ගැළපෙන theme-aware කුසලතාවක් තිබේ නම්, එය ස්වයංක්‍රීයව assistant හි context එකට පටවා ඇත. ඔබට අතින් කුසලතා සක්‍රිය කිරීමට හෝ මාරු කිරීමට අවශ්‍ය නොවේ.

### බහු කුසලතා (Multiple Skills) {#multiple-skills}

ඔබේ වෙබ් අඩවියට විවිධ කුසලතා (උදාහරණයක් ලෙස, ඔබට Kadence Blocks සහ Kadence Theme දෙකම ක්‍රියාත්මක නම්) අදාළ වන විට, සහායකයාට සියලුම අදාළ කුසලතා වලට ප්‍රවේශ වීමට හැකි අතර ඒවායින් මඟපෙන්වීම් උපුටා ගත හැකිය.

### තේමා මාරු කිරීම (Switching Themes) {#switching-themes}

ඔබ ක්‍රියාත්මක වන තේමාව වෙනස් කරන විට, ඊළඟ පණිවිඩයේදී සහායකයාගේ ලබා ගත හැකි කුසලතා ස්වයංක්‍රීයව යාවත්කාලීන වේ. උදාහරණයක් ලෙස:

1. ඔබ **Block Themes** කුසලතාව ක්‍රියාත්මක වන බ්ලොක් තේමාවක් භාවිතා කරයි.
2. ඔබ සම්භාව්‍ය (classic) තේමා වෙත මාරු වේ.
3. ඊළඟ පණිවිඩයේදී, **Classic Themes** කුසලතාව ස්වයංක්‍රීයව පැටවනු ලබන අතර **Block Themes** කුසලතාව තවදුරටත් ලබා ගත නොහැක.

## තේමා-අවබෝධී කුසලතා භාවිතා කිරීම (Using Theme-Aware Skills) {#using-theme-aware-skills}

තේමා-අවබෝධී කුසලතාවක් භාවිතා කිරීමට, ඔබට අවශ්‍ය දේ චැට් අතුරුමුහුණතේ විස්තර කිරීම ප්‍රමාණවත් වේ. සහායකයා ස්වයංක්‍රීයව සුදුසු කුසලතාවේ මඟපෙන්වීම් උපුටා ගනු ඇත.

### උදාහරණ ප්‍රොම්ප්ට් (Example Prompts) {#example-prompts}

**Block Themes සඳහා:**
> "බ්ලොක් රටා භාවිතා කර පසුබිම් රූපයක් සහ මැද පෙළ සහිත හීරෝ කොටසක් සාදන්න."

**Classic Themes සඳහා:**
> "child theme එකක් භාවිතා කර පැති තීරුවේ (sidebar) අමතර වයිජට් ප්‍රදේශයක් එක් කරන්න."

**Kadence Blocks සඳහා:**
> "Kadence Testimonials බ්ලොක් එක භාවිතා කර අත්දැකීම් කොටසක් ගොඩනඟන්න."

**Kadence Theme සඳහා:**
> "හෙඩර් සැකසුම් සහ නාවික මෙනු (navigation menu) විලාසිතා වෙනස් කරන්න."

ක්‍රියාත්මක වන ඔබේ තේමාවට සහ ප්ලගිනවලට ගැලපෙන තේමා-විශේෂිත මඟපෙන්වීම් සහ කේත උදාහරණ සහායකයා විසින් ලබා දෙනු ඇත.
