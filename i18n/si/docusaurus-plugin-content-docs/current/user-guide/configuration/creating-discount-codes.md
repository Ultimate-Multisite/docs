# වට්ටම් කේත නිර්මාණය කිරීම (v2)

_**වැදගත් සටහන: මෙම ලිපිය Ultimate Multisite version 2.x සම්බන්ධ වේ.**_

Ultimate Multisite සමඟ, ඔබට ඔබේ පාරිභෝගිකයින්ගේ දායකත්වයන් සඳහා ඔවුන්ට වට්ටම් ලබා දීමට වට්ටම් කේත (discount codes) නිර්මාණය කළ හැකිය. ඒවා නිර්මාණය කිරීම ඉතා පහසුයි!

## වට්ටම් කේත නිර්මාණය කිරීම සහ සංස්කරණය කිරීම

වට්ටම් කේතයක් නිර්මාණය කිරීමට හෝ සංස්කරණය කිරීමට, **Ultimate Multisite > Discount Codes** වෙත යන්න.

![Discount codes list — empty state before any codes exist](/img/config/discount-codes-empty.png)

එහිදී ඔබ දැනටමත් නිර්මාණය කර ඇති වට්ටම් කේතවල ලැයිස්තුවක් ඔබට පෙනෙනු ඇත.

නව කුපන් එකක් (coupon) නිර්මාණය කිරීමට **Add Discount Code** ක්ලික් කරන්න, නැතිනම් ඒවා සංස්කරණය කිරීමට ඒවා මත මවුස් එක ගෙන ගොස් **Edit** ක්ලික් කරන්න.

![Discount codes list with hover actions showing Edit and Delete links](/img/config/discount-codes-list-hover.png)

![Add Discount Code button in the page header](/img/config/discount-codes-add-button.png)

ඔබ ඔබේ කුපන් කේතය නිර්මාණය කිරීමට හෝ සංස්කරණය කිරීමට යොමු කරන පිටුවට යාවත්කාලීන වනු ඇත. මෙම උදාහරණයේදී අපි අලුත් එකක් සාදමු.

![Discount code edit page with all sections visible](/img/config/discount-code-edit.png)

මෙහි ඇති සැකසුම් බලමු:

**Enter Discount Code:** මෙය ඔබේ වට්ටම් කේතයේ නම පමණි. පාරිභෝගිකයින්ට ගෙවීම් පිටුව (checkout form) වල භාවිතා කිරීමට අවශ්‍ය කේතය මෙය නොවේ.

**Description:** මෙහිදී, මෙම කුපන් එක කුමක් සඳහාද යන්න කෙටියෙන් විස්තර කළ හැකිය.

![Discount code name and description fields at the top of the edit page](/img/config/discount-code-description.png)

ඔබට වට්ටම් කේතය ක්‍රියාත්මක (active) හෝ අක්‍රිය (inactive) ලෙස ද දැකගත හැකිය:

![Discount code active status](/img/config/discount-code-active.png)

**Coupon code:** පාරිභෝගිකයින් ගෙවීම් පිටුවේ ඇතුළත් කිරීමට අවශ්‍ය කේතය මෙහිදී ඔබ නිර්වචනය කරයි.

![පාරිභෝගිකයන් ගෙවීමේදී කේතය ඇතුළත් කරන කූපන් කේත ක්ෂේත්‍රය](/img/config/discount-code-coupon-field.png)

**අමතර වට්ටම් (Discount):** මෙහිදී, ඔබේ කූපන් කේත සඳහා **ප්‍රතිශතයක් (percentage)** හෝ **ස්ථාවර මුදලක් (fixed amount)** තීරණය කළ හැකියි.

![ප්‍රතිශත හෝ ස්ථාවර මුදල සහිත අමතර වට්ටම් සැකසුම](/img/config/discount-code-amount.png)

**නැවත ලබා ගැනීම් සඳහා යෙදීම (Apply to renewals):** මෙම විකල්පය ක්‍රියාත්මක නොකරන්නේ නම්, මෙම කූපන් කේත පළමු ගෙවීම සඳහා පමණක් යොදා ගැනේ. අනෙක් සියලුම ගෙවීම්වලට වට්ටමක් නොලැබේ. මෙම විකල්පය ක්‍රියාත්මක කර ඇත්නම්, එම කූපන් කේත අනාගත සියලුම ගෙවීම් සඳහා වලංගු වේ.

**සැකසුම් ගාස්තු වට්ටම් (Setup fee discount):** මෙම විකල්පය ක්‍රියාත්මක නොකරන්නේ නම්, කූපන් කේත මගින් ඇණවුමේ **සැකසුම් ගාස්තුවට කිසිදු වට්ටමක් ලබා නොදේ**. මෙම විකල්පය ක්‍රියාත්මක කර ඇත්නම්, ඔබේ සැලසුම්වල සැකසුම් ගාස්තු සඳහා මෙම කූපන් කේත යොදන ප්‍රතිශතය හෝ ස්ථාවර මුදල තීරණය කළ හැකියි.

![නැවත ලබා ගැනීම් සහ සැකසුම් ගාස්තු වට්ටම් විකල්පයන් යෙදීම](/img/config/discount-code-renewals.png)

**ක්‍රියාත්මක (Active):** මෙම කූපන් කේත අතින් ක්‍රියාත්මක කිරීමට හෝ අක්‍රිය කිරීමට හැකිය.

![අමතර වට්ටම් කේත අතින් සක්‍රීය කිරීම හෝ අක්‍රීය කිරීම සඳහා විකල්පය](/img/config/discount-code-active.png)

**උසස් විකල්ප (Advanced Options)** යටතේ, අපට පහත සැකසුම් තිබේ:

![අමතර වට්ටම් කේත උසස් විකල්පයන්](/img/config/discount-code-advanced.png)

**භාවිතා සීමාව (Limit uses):**

  * **භාවිතා කිරීම් (Uses):** මෙහිදී, එම කූපන් කේත කොපමණ වාරයක් භාවිතා කර ඇත්දැයි ඔබට බැලිය හැකියි.

  * **උපරිම භාවිතයන් (Max uses):** මෙය පරිශීලකයින්ට මෙම කූපන් කේත භාවිතා කළ හැකි වාර ගණන සීමා කරයි. උදාහරණයක් ලෙස, ඔබ මෙහි 10 යැවුවහොත්, එම කූපන් කේත උපරිම වරක් පමණක් භාවිතා කළ හැකිය. මෙම සීමාව ඉක්මවා ගිය පසු, කූපන් කේත තවත් භාවිතා කළ නොහැක.

![වත්මන් භාවිතයන් ගණන සහ උපරිම භාවිතයන් ක්ෂේත්‍ර සහිත සීමා කිරීම සැකසුම](/img/config/discount-code-limit-uses.png)

**ආරම්භක සහ අවසන් දිනයන්:** මෙහිදී ඔබට ඔබේ කූපන් එකට ආරම්භක දිනයක් සහ/හෝ අවසන් දිනයක් එක් කිරීමට විකල්පයක් ලැබෙනවා.

![දැන්වීම් කේත කාලසටහන සඳහා ආරම්භක සහ අවසන් දිනය ක්ෂේත්‍ර](/img/config/discount-code-dates.png)

**නිෂ්පාදන සීමා කිරීම:** ඔබ **Select products** එක ඔන් (on) කරන්නේ නම්, ඔබේ සියලුම නිෂ්පාදන ඔබට පෙන්වනු ඇත. මෙම කූපන් කේත පිළිගන්නා නිෂ්පාදනය තෝරා ගැනීමට (ඔබට අවශ්‍ය පරිදි ඔන් හෝ ඕෆ් කරමින්) ඔබට හැකියාව ලැබේ. මෙහි ඔෆ් කර ඇති නිෂ්පාදන සඳහා, ඔබේ පාරිභෝගිකයින් මෙම කූපන් කේත ඔවුන්ට භාවිතා කිරීමට උත්සාහ කළද කිසිදු වෙනසක් සිදු නොවේ.
