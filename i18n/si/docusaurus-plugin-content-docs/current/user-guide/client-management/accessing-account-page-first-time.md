# මගේ ගිණුම් පිටුවට පළමු වතාවේ මට පිවිසෙන්නේ කොහොමද?

Ultimate Multisite වෙත ඔබව සාදරයෙන් පිළිගනිමු!

ඔබ මිලදී ගැනීමේදී ලබා දුන් විද්‍යුත් තැපෑලේ තිබූ විද්‍යුත් තැපෑල සහ ඔබට ලැබුණු මුරපදය භාවිතා කරන්න.

මෙය ඔබගේ ගිණුමට ලොග් වන පළමු වතාව බැවින්, ඔබගේ මුරපදය වෙනස් කිරීමට ඉල්ලා සිටිනු ඇත. එය වෙනස් කර _Save_ බොත්තම ක්ලික් කරන්න.

ඔබේ ගිණුම් පිටුවේදී ඔබට බිලකරණය (billings), අලුත් කිරීම් (renewals), අවලංගු කිරීම් (cancellations) සහ වෙනත් බොහෝ දේ ඇතුළු සියලුම ගිණුම් විස්තර කළමනාකරණය කළ හැකිය.
