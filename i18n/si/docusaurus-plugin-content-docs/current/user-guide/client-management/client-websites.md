# ඔබේ සේවාදායකයන්ගේ වෙබ් අඩවි (v2) {#your-clients-websites-v2}

ඔබ ඔබේ ජාලයට ලියාපදිංචි වූ පසු, ඔබේ පාරිභෝගිකයින්ට ඔවුන්ගේ dashboard එකට ප්‍රවේශ විය හැකිය.

ගෙවීම්, සාමාජිකත්වයන්, ડોමේන් (domains), සැලැස්ම සීමාවන් සහ තවත් බොහෝ දේ පිළිබඳ වැදගත් තොරතුරු බැලිය හැකි **Account** පිටුව හැර, ඔවුන්ට සියලු වෙබ් අඩවි බැලීමට සහ කළමනාකරණය කිරීමට හැකි වන **My Sites** පිටුවට ද ප්‍රවේශ විය හැකිය. ඔවුන් කළ යුත්තේ තම dashboard එකේ My Sites මත ක්ලික් කිරීම පමණි.

![පාරිභෝගික dashboard එකේ My Sites සබැඳිය](/img/account-page/my-sites-link.png)

ඔවුන්ගේ සියලුම වෙබ් අඩවි පෙන්වනු ලැබේ, සෑම එකකම පහළින් **Manage** බොත්තමක් ඇත. ඔවුන් එය ක්ලික් කළහොත්, එම නිශ්චිත වෙබ් අඩවියේ **dashboard** එකට යොමු වනු ඇත.

එය එක් එක් සැලැස්මට වඩා වැඩි වෙබ් අඩවි තිබීමට ඉඩ දෙන විට, ඔවුන්ගේ වෙබ් අඩවි වලින් කුමන එක ප්‍රාථමික (primary) එකද යන්න ද ඔවුන්ට දැකගත හැකිය.

![පාරිභෝගික වෙබ් අඩවි සහ Manage බොත්තම් සහිත My Sites පිටුව](/img/account-page/my-sites-page.png)

මෙන්න වෙබ් අඩවිය සංස්කරණය කිරීමේ සම්පූර්ණ දර්ශනය:

![වෙබ් අඩවිය සංස්කරණ සම්පූර්ණ අතුරුමුහුණත](/img/admin/site-edit-full.png)

ඔවුන් තම ප්‍රාථමික වෙබ් අඩවිය වෙනස් කිරීමට අවශ්‍ය නම්, ඔවුන් **Account** මෙනුව ක්ලික් කිරීමෙන් Account පිටුවට යා යුතුය.

![පාරිභෝගික dashboard එකේ Account මෙනු සබැඳිය](/img/account-page/account-menu.png)

ඔවුන්ගේ පාරිභෝගික විස්තර ද බැලීමට සහ සංස්කරණය කිරීමට ද ඔවුන්ට හැකිය:

![පාරිභෝගික සංස්කරණ අතුරුමුහුණත](/img/admin/customer-edit.png)

ඉන්පසු, ඔවුන් **Change Default Site** ක්ලික් කළ යුතුය.

![Account පිටුවේ Change Default Site බොත්තම](/img/account-page/change-default-site.png)

එකක් pop-up කවුළුවක් ලෙස දිස්වේ. 드롭-다운 ලැයිස්තුව මත ක්ලික් කර ඔබ ප්‍රාථමික වෙබ් අඩවිය කිරීමට කැමති කුමන වෙබ් අඩවිය තෝරාගත යුතුද යන්න තෝරන්න.

![Change Default Site 드롭다운 තේරීම](/img/account-page/change-default-site-dropdown.png)

ඉන්පසු, **Change Default Site** ක්ලික් කරන්න, ඔබට වැඩ අවසන්!

![ප්‍රධාන වෙබ් අඩවිය වෙනස් කිරීම තහවුරු කිරීම](/img/account-page/change-default-site-confirm.png)
