# සිදුවීම: Zapier හරහා ගිණුමක් ලියාපදිංචි කිරීම

[Integrating Ultimate Multisite with Zapier](zapier.md) ලිපියේදී, ට්‍රිගර් සහ සිදුවීම් මත පදනම්ව Ultimate Multisite තුළ Zapier භාවිතා කර විවිධ ක්‍රියාවන් සිදු කළ හැකි ආකාරය අපි සාකච්ඡා කළෙමු. මෙම ලිපියෙන්, අපට තෙවන පාර්ශවීය යෙදුම් (3rd party applications) ඒකාබද්ධ කරගත හැකි ආකාරය පෙන්වන්නෙමු. මෙහිදී, දත්ත මූලාශ්‍රයක් ලෙස Google Sheets භාවිතා කර ගිණුමක් ලියාපදිංචි කිරීමට Ultimate Multisite වෙත තොරතුරු යොමු කරනු ලැබේ.

Zapier සම්බන්ධ කිරීමට පෙර, **Ultimate Multisite > Settings > API & Webhooks** වෙත ගොස් API සක්‍රීය කර ඇති බව තහවුරු කරන්න. Zapier ඔබෙන් Ultimate Multisite ගිණුම් විස්තර ඉල්ලා සිටින විට මෙම තිරයෙන් API Key සහ API Secret පිටපත් කරන්න.

![API and Webhooks settings with API Key, API Secret, and Enable API options](/img/admin/settings-api-webhooks.png)

පළමුව, ඔබේ Google Drive යටතේ **Google Sheet** එකක් නිර්මාණය කළ යුතුය. පසුව දත්ත පහසුවෙන් සිතියම්ගත කිරීමට හැකි වන පරිදි සෑම තීරුවක්ම නිසි ලෙස නිර්වචනය කර ඇති බවට වග බලා ගන්න.

Google Sheet එකක් නිර්මාණය කිරීමෙන් පසු, ඔබට ඔබේ Zapier ගිණුමට ලොග් වී zap එකක් නිර්මාණය කිරීම ආරම්භ කළ හැකිය.

**"App event"** සඳහා වන සෙවුම් ක්ෂේත්‍රයේ **"Google Sheets"** තෝරන්න.

ඉන්පසු "**Event** " ක්ෂේත්‍රය සඳහා **"New spreadsheet row"** (නව පැතුරුම් පේළිය) තෝරා "**Continue**" (අඛණ්ඩව යන්න) ක්ලික් කරන්න.

ඊළඟ පියවර වන්නේ දත්ත ලැබෙන Google Sheet එක සහ worksheet එක තෝරා ගැනීමට අවශ්‍ය වන **Google Account** එක තෝරා ගැනීමයි. එබැවින් නිවැරදි ගූගල් ගිණුම සඳහන් කර ඇති බවට වග බලා ගන්න.

**"Set up trigger"** යටතේ, දත්ත ලැබෙන Google Sheet එක සහ worksheet එක තෝරාගත යුතු අතර එය සඳහන් කළ යුතුය. ඉදිරියට යාමට එම තොරතුරු පුරවා "**Continue**" ක්ලික් කරන්න.

ඊළඟට ඔබේ Google Sheet නිසි ලෙස සම්බන්ධ වී ඇත්දැයි බැලීමට **"test your trigger"** (ඔබේ ට්‍රිගර් පරීක්ෂා කරන්න) කළ යුතුය.

ඔබේ පරීක්ෂණය සාර්ථක නම්, ඔබේ spreadhseets වලින් සමහර අගයන් පෙන්වන ප්‍රතිඵලයක් ඔබට දැකගත හැකිය. ඉදිරියට යාමට "**Continue**" ක්ලික් කරන්න.

Ultimate Multisite එකේ ගිණුමක් නිර්මාණය කිරීමට හෝ ලියාපදිංචි කිරීමට අවශ්‍ය දෙවන ක්‍රියාව සැකසීම මෙහිදී සිදු කෙරේ. සෙවුම් ක්ෂේත්‍රයේ "**Ultimate Multisite(2.0.2)** " තෝරන්න.

"**Event**" යන ක්ෂේත්‍රය යටතේ, "**Register an Account in Ultimate Multisite** " තෝරාගෙන පසුව "**Continue** " බොත්තම ක්ලික් කරන්න.

"**Set up an action**" යටතේ, ඔබට ගනුදෙනුකරුවන්ගේ දත්ත, සාමාජිකත්වයන්, නිෂ්පාදන වැනි විවිධ ක්ෂේත්‍ර සඳහා වෙනස් කළ හැකි ක්ෂේත්‍ර කිහිපයක් දැකගත හැක. ඔබ ඔබේ Google Sheet එකේ ඇති අගයන් ඒවා නිවැරදිව ඇතුළත් විය යුතු ස්ථානවලට පැවරීමට (map) හැකිය. එය පහත ස්ක්‍රීන්ෂොට් එකේ පෙන්වා ඇති පරිදි, එම ක්ෂේත්‍රවලට අදාළ වන්න.

අගයන් මාරු කිරීමෙන් පසු, ඔබට එම ක්‍රියාව පරීක්ෂා කළ හැකිය.
