# V1 සිට සංක්‍රමණය කිරීම

## Ultimate Multisite එහි මුල් 1.x අනුවාද මාලාවෙන් 2.x අනුවාද මාලාවට වෙනස් කර ඇත.

Ultimate Multisite අනුවාදය 2.0 සහ ඉහළ ඒවා යනු කේතය සම්පූර්ණයෙන්ම නැවත ලියූ එකක් (complete rewrite) වන අතර, එයින් පැරණි අනුවාදය හා නව එක අතර බෙදාගන්නා දේවල් ඉතා අඩුවීමයි. ඒ නිසා, 1.x සිට 2.x දක්වා උත්ප්‍රේරණය කිරීමේදී, ඔබගේ දත්ත නව අනුවාදයන්ට තේරුම් ගත හැකි ආකෘතියකට සංක්‍රමණය කළ යුතු වෙනවා.

වාසනාවකට මෙන්, Ultimate Multisite 2.0+ හි **පරිවර්තකය (migrator)** තිබෙනවා. එය ප්‍රධාන කොටසෙහිම ගොඩනඟා ඇති අතර පැරණි අනුවාදයේ දත්ත හඳුනාගෙන නව ආකෘතියට පරිවර්තනය කිරීමට හැකියාව තිබෙනවා. මෙම සංක්‍රමණය සිදු වන්නේ 2.0+ හි **Setup Wizard** (සැකසුම් මඟ) තුළයි.

මෙම පාඩමෙන් කරන්නේ එම migrator ක්‍රියා කරන ආකාරය, අසාර්ථක වුවහොත් කළ යුතු දේ සහ මෙම ක්‍රියාවලිය අතරතුර ඇතිවිය හැකි ගැටළු විසඳා ගන්නා ආකාරය ගැනයි.

_**වැදගත්: ඔබ 1.x සිට 2.0 දක්වා උත්ප්‍රේරණය කිරීමට පෙර, කරුණාකර ඔබේ වෙබ් අඩවි දත්ත සමුදායේ (site database) පිටපතක් (backup) නිර්මාණය කර ඇති බවට වග බලා ගන්න.**_

## පළමු පියවර

පළමු පියවර වන්නේ plugin .zip ගොනුව බාගත කර ඔබේ network admin dashboard එකේ version 2.0 ස්ථාපනය කිරීමයි.

ඔබ [install and activate version 2.0](../getting-started/installing-ultimate-multisite.md) කළ පසු, ඔබේ Multisite පැරණි අනුවාදයේ (legacy version) ක්‍රියාත්මක වන බව පද්ධතිය ස්වයංක්‍රීයව හඳුනා ගනු ඇති අතර plugin පිටුවේ ඉහළින් ඔබට මෙම පණිවිඩය දැකගත හැකිය.

_**සටහන: ඔබ ඔබේ Multisite එකේ Ultimate Multisite 1.x ස්ථාපනය කර ඇත්නම්, ඔබ බාගත කළ අනුවාදය සමඟ plugin එක ප්‍රතිස්ථාපනය කිරීමට (replace) ඔබට විකල්පයක් ලැබෙනු ඇත. කරුණාකර, "Replace current with uploaded" මත ක්ලික් කරන්න.**_

## දැන්, සංක්‍රමණය (Migration)

මෙන්න migrator එකේ ඇතුළත් ආරක්ෂිත විශේෂාංගයක්. ඒ කියන්නේ ඔබේ Ultimate Multisite දත්ත සියල්ලම කිසිදු ගැටලුවකින් තොරව සංක්‍රමණය කළ හැකිද කියලා පරීක්ෂා කරන්න එය මුළු multisite එකම පරීක්ෂා කරනවා. ක්‍රියාවලිය ආරම්භ කිරීමට **Run Check** බොත්තම ක්ලික් කරන්න.

පරීක්ෂාව කර අවසන් වූ පසු, ඔබට විකල්ප දෙකක් තිබෙනවා: ප්‍රතිඵලය **දෝෂයක් සමඟ (with an error)** හෝ **දෝෂයකින් තොරව (without an error)** විය හැකියි.

### දෝෂයක් සමඟ (With Error)

ඔබට දෝෂ පණිවිඩයක් ලැබුණොත්, එම දෝෂය නිවැරදි කිරීමට ඔවුන්ට උදව් කළ හැකි නිසා අපගේ support team එකෙන් සම්බන්ධ කර ගැනීමට ඔබට සිදු වේ. ඔබ ටිකට් එකක් නිර්මාණය කරන විට **error log එක ලබා දීමට** වග බලා ගන්න. ඔබට ලොග් එක බාගත කළ හැකියි නැත්නම් අපගේ support team එක සම්බන්ධ කර ගැනීමට කියන link එක ක්ලික් කළ හැකියි. එය ඔබේ පිටුවේ දකුණු පස ඇති help widget එක විවෘත කරනු ඇත, එහි description යට error logs ඇතුළුව ඔබ වෙනුවෙන් කලින් සකස් කරන ලද fields තිබේ.

### දෝෂයකින් තොරව (Without Error)

පද්ධතියට කිසිදු දෝෂයක් නොමැති නම්, ඔබට සාර්ථක පණිවිඩයක් සහ සංක්‍රමණය ඉදිරියට ගෙන යාමට හැකි පහළ ඇති **Migrate** බොත්තමක් දැකගත හැකිය. මෙම පිටුවේදී, ඉදිරියට යාමට පෙර ඔබේ database එකේ backup එකක් නිර්මාණය කිරීමට මතක් කරනු ඇත, එය අපි දැඩි ලෙස නිර්දේශ කරමු. ඔබට දැනටමත් backup එකක් තිබේ නම් **Migrate** ක්ලික් කරන්න.

මෙයින් සියල්ල අවසන්!

ඔබට ඔබේ ජාලයේ ලෝගෝ සහ වෙනත් දේවල් යාවත්කාලීන කිරීමට වizard (Wizard) සැකසුම දිගටම කරගෙන යා හැකියි, නැතිනම් ඔබේ Ultimate Multisite version 2.0 මෙනුව සහ එහි නව අතුරුමුහුණත හරහා ගමන් කිරීම ආරම්භ කළ හැකියි. දැන් විනෝද වෙන්න!
