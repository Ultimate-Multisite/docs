# අපනයන සහ ආනයන (Export & Import)

Ultimate Multisite 2.9.0 හි **Tools > Export & Import** යටතේ තනි-සයිට් සඳහා එක් මෙවලමක් වන **Export & Import** එකක් එකතු කර ඇත. ඔබ එක් WordPress සයිට් එක ZIP ගොනුවක් ලෙස පැකේජ් කිරීමට, එම ZIP ප්‍රතිස්ථාපනය කිරීමට හෝ අනුකූල Ultimate Multisite සහ තනි-සයිට් WordPress ස්ථාපනයන් අතර සයිට් එකක් මාරු කිරීමට අවශ්‍ය වූ විට එය භාවිතා කරන්න.

## අවශ්‍ය අවසරයන් (Required permissions)

ඔබ අපනයනය කරන ලද හෝ ආනයනය කරන ලද සයිට් එකේ WordPress **Tools** මෙනුවට ප්‍රවේශ විය හැකි පරිපාලකයක් ලෙස ලොග් විය යුතුය. බහු-සයිට් ජාලයකදී, ජාල මට්ටමේ Ultimate Multisite මෙවලම් භාවිතයෙන් උප-සයිට් අපනයනය කිරීමේදී හෝ ආනයනය කිරීමේදී ජාල පරිපාලක ගිණුමක් භාවිතා කරන්න.

අපනයන ZIP downloads අනුමත කරන ලද බාගත කිරීමේ අවසානයක් හරහා සපයනු ලැබේ, එබැවින් බාගත කිරීම අවසන් වන තෙක් පරිපාලක සැසිය ක්‍රියාකාරීව තබා ගන්න, සහ ජනනය කරන ලද බාගත කිරීමේ URL ලොකු ප්‍රසිද්ධියේ බෙදා නොගන්න.

## සයිට් එකක් ZIP ලෙස අපනයනය කිරීම (Exporting a site to a ZIP)

1. ඔබ පිටපත් කිරීමට අවශ්‍ය සයිට් එකේ WordPress පරිපාලනයේ, **Tools > Export & Import** වෙත යන්න.
2. අපනයන ප්‍රදේශය විවෘත කර පැකේජ් කිරීමට අවශ්‍ය සයිට් එක තෝරන්න.
3. එම විකල්ප තිබේ නම්, uploads (ගොනු), plugins (ප්ලග්ඉන්) සහ themes (තේමා) වැනි අභ්‍යන්තර දත්ත ඇතුළත් කිරීමට අවශ්‍ය අන්තර්ගතය තෝරන්න.
4. අපනයනය ආරම්භ කර ක්‍රියාවලිය අවසන් වන තුරු බලා සිටින්න. විශාල සයිට් සඳහා ZIP එක සූදානම් වීමට පෙර පසුබිමින් අවසන් විය හැක.
5. exports (අපනයන) ලැයිස්තුවෙන් අවසන් වූ ZIP ගොනුව බාගත කරන්න.

ZIP ගොනුව ආරක්ෂිත ස්ථානයක තබා ගන්න. එය සයිට් අන්තර්ගතය, සැකසුම්, මීඩියා ගොනු සහ තෝරාගත් කේත දත්ත ඇතුළත් විය හැකිය.

## අපනයනයේ ඇතුළත් දේ (What the export includes)

අපනයන ZIP එකට පහත දෑ ඇතුළත් කළ හැකිය:

- තෝරාගත් සයිට් එකේ දත්ත සමුදා අන්තර්ගතය සහ සැකසුම්.
- uploads (ගොනු) ඇතුළත් කර ඇත්නම්, upload කරන ලද මීඩියා ගොනු.
- එම විකල්ප තෝරාගෙන ඇත්නම්, plugins (ප්ලග්ඉන්) සහ themes (තේමා).
- ඉලක්ක ස්ථාපනය මත සයිට් එක නැවත ගොඩනැගීමට Export & Import මෙවලමට භාවිතා කරන import metadata (ආනයන දත්ත).

ZIP එකේ හරියටම ප්‍රමාණය තීරණය වෙන්නේ මීඩියා දත්ත ප්‍රමාණය, තෝරාගත් plugins සහ themes වල ප්‍රමාණය සහ වෙබ් අඩවියේ database tables වල ප්‍රමාණය අනුවයි.

## ZIP එකකින් site එකක් import කිරීම

1. ගමනාන්ත WordPress site එකේ **Tools > Export & Import** වෙත යන්න.
2. import කරන ප්‍රදේශය විවෘත කර, Export & Import tool මගින් නිර්මාණය කළ ZIP එක upload කරන්න.
3. site එකට අලුත් ලිපිනයක් භාවිතා කිරීමට අවශ්‍ය නම් replacement URL එක ඇතුළත් කරන්න, නැතිනම් මුල් URL එක තබා ගැනීමට එම ක්ෂේත්‍රය හිස්ව තබන්න.
4. import අවසන් වූ පසු upload කළ ZIP එක මකා දැමීමට හෝ නොදැමීමට තීරණය කරන්න.
5. **Begin Import** මත ක්ලික් කරන්න.
6. import අවසන් වන තුරු පවතින import එක නිරීක්ෂණය කරන්න. සම්පූර්ණ කිරීමෙන් පසුව ක්‍රියාවලිය නැවැත්වීමට අවශ්‍ය නම් පමණක් **Cancel Import** භාවිතා කරන්න.
7. සාමාන්‍ය ගමනාගාර හෝ පාරිභෝගික ප්‍රවේශයන් ලබා දීමට පෙර import කළ site එක සමාලෝචනය කරන්න.

එක-site WordPress ස්ථාපනයකදී, ZIP එකකින් import කිරීම මගින් වත්මන් site එක import කරන ලද site එකෙන් ප්‍රතිස්ථාපනය කරයි. ඔබ ආරම්භ කිරීමට පෙර ඉලක්කගත site එකේ සම්පූර්ණ backup එකක් සාදා ගන්න, සහ එකම site එක සඳහා එකවර බහු imports ආරම්භ කිරීමෙන් වළකින්න.

## සීමාවන් සහ අනුකූලතා සටහන්

വളരെ വലിയ അപ്‌ലോഡ് ഡയറക്‌ടറിയോ മീഡിയ ലൈബ്രറിയിലോ ഉള്ള ഫയലുകൾക്ക് വലിയ ZIP ഫയലുകൾ ഉണ്ടാകാൻ സാധ്യതയുണ്ട്. വലിയ സൈറ്റുകൾ എക്സ്പോർട്ട് ചെയ്യാനോ ഇംപോർട്ട് ചെയ്യാനോ മുമ്പ് PHP അപ്‌ലോഡ് ലിമിറ്റുകൾ, എക്സിക്യൂഷൻ ലിമിറ്റുകൾ, ഡിസ്ക് സ്പേസ്, മെമ്മറി, സെർവർ ടൈമാउट ക്രമീകരണങ്ങൾ എന്നിവ ഉറപ്പുവരുത്തുക.
കുറഞ്ഞ ട്രാഫിക്കുള്ള മെയിന്റനൻസ് സമയങ്ങളിൽ വളരെ വലിയ മീഡിയ ലൈബ്രറികൾ മാറ്റേണ്ടി വന്നേക്കാം.
ലക്ഷ്യമായ WordPress ഇൻസ്റ്റാളേഷൻ അനുയോജ്യമായ WordPress, PHP, Ultimate Multisite, plugin, theme പതിപ്പുകൾ പ്രവർത്തിപ്പിക്കണം.
ഒരു സിംഗിൾ-സൈറ്റ് ഇംപോർട്ട് ലക്ഷ്യമിടുന്ന സൈറ്റിനെ മാറ്റിസ്ഥാപിക്കുന്നു. ഇത് ഒരു മെർജ് ടൂൾ അല്ല.
Multisite-to-single-site അല്ലെങ്കിൽ single-site-to-multisite മാറ്റങ്ങൾ, ലക്ഷ്യപ്രാവ 환경 എക്സ്പോർട്ട് ചെയ്ത സൈറ്റിലെ plugin-കൾ, theme-കൾ, URL-കൾ, ആവശ്യമായ Ultimate Multisite ഘടകങ്ങൾ എന്നിവ പ്രവർത്തിപ്പിക്കാൻ കഴിയുമെങ്കിൽ മാത്രമേ പിന്തുണയ്ക്കുകയുള്ളൂ.
ZIP ഫയൽ സ്വകാര്യമായി സൂക്ഷിക്കുക. ഇതിൽ ഡാറ്റാബേസ് ഉള്ളടക്കം, അപ്‌ലോഡ് ചെയ്ത ഫയലുകൾ, എക്സ്പോർട്ട് ചെയ്ത സൈറ്റിന്റെ കോൺഫിഗറേഷൻ വിവരങ്ങൾ എന്നിവ ഉണ്ടാകാൻ സാധ്യതയുണ്ട്.

പഴയ നെറ്റ്‌വർക്ക്-ലെവൽ എക്സ്പോർട്ട് വർക്ക്ഫ്ലോകൾക്കായി [Site Export](/user-guide/administration/site-export) കാണുക.
