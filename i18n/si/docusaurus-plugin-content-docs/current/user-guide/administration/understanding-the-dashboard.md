# Ultimate Multisite Dashboard එක තේරුම් ගැනීම

Ultimate Multisite වල වඩාත් උසස් සැකසුම් සහ ක්‍රියාකාරී අංශ ගැන ගැඹුරට යෑමට පෙර, අපි මුලින්ම දාශබෝඩ් එක (dashboard) බලමු.

WordPress plugin ගොඩක් වගේ, Ultimate Multisite මගින් **Ultimate Multisite** නමින් ජාල මට්ටමේ මෙනුව ඇතුළත් කිරීමක් නිර්මාණය කරන අතර එහි පහළින් උප-මෙනු ඇතුළත් කිරීම් කිහිපයක් තිබේ.

## Network Dashboard Widgets (ජාල දත්ත පුවරුවේ වයිජට්)

Ultimate Multisite මගින් ජාල මට්ටමේ දත්ත පුවරුවේ (network-level dashboard) ප්‍රයෝජනවත් වයිජට් කිහිපයක් ඇතුළත් කරයි. මෙම දත්ත පුවරුව වෙත යාමට **My Sites → Network Admin → Dashboard** වෙත යන්න. මෙම වයිජට් නිර්මාණය කර ඇත්තේ ජාල පරිපාලකයාට තොරතුරු සහ පොදු මෙහෙයුම් සඳහා පහසුවෙන් ප්‍රවේශ විය හැකි වන පරිදි ය.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### පළමු පියවර (First Steps)

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

මෙම වයිජට් Ultimate Multisite හි අලුත් ස්ථාපනයක දිස්වන අතර, ගෙවීම් ක්‍රමයක් සැකසීම සහ පරීක්ෂණ ගිණුමක් (test account) නිර්මාණය කිරීම වැනි අමතර කාර්යයන් සම්පූර්ණ කිරීමට ජාල පරිපාලකයා මතක් කරයි.

### සාරාංශය (Summary)

![Summary widget](/img/admin/network-dashboard-summary.png)

සාරාංශ වයිජට් එකෙන් දවසට ලැබුණු ලියාපදිංචි ගණන සහ ඔවුන්ගේ ආදායම පිළිබඳ වාර්තා කරයි. MRR (Monthly Recurring Revenue) යනු ආදායම් උපයන සාමාජිකයින් සමඟ ඇති පවතින මුළු ආදායම ඇස්තමේන්තුගත කිරීමයි.

### ක්‍රියාකාරකම් ප්‍රවාහය (Activity Stream)

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

මෙම වයිජට් එක Ultimate Multisite පුරා සිදුවන සිදුවීම් වාර්තා කරයි. සිදුවීම් අතර ලියාපදිංචි වීම්, අවලංගු කිරීම්, වෙබ් අඩවි නිර්මාණය කිරීම්, මකා දැමීම් සහ වෙනත් ක්‍රියාකාරකම් ඇතුළත් වේ.

වැඩි විස්තර සඳහා කරුණාකර ලේඛනවල Events කොටස බලන්න.

### දැන් (Right Now)

![Right Now widget](/img/admin/network-dashboard-right-now.png)

මෙම utility widget එක මගින් ජාලයේ සිටින පරිශීලකයන් සහ වෙබ් අඩවි පිළිබඳ කෙටි සාරාංශයක් පෙන්වයි. ඉහළ ඇති කෙටි මාර්ග (shortcuts) මගින් නව අඩවි හෝ පරිශීලකයින් නිර්මාණය කිරීමට ක්ලික් එකකින් ප්‍රවේශ විය හැකියි.

### පුවත් සහ සාකච්ඡා

![News and Discussions widget](/img/admin/network-dashboard-news.png)

මෙම widget එකෙන් Ultimate Multisite හි නවතම පුවත් ලබාගෙන පෙන්වයි. යාවත්කාලීන කිරීම්, බරපතල දෝෂ සහ ආරක්ෂක නිවැරදි කිරීම් පිළිබඳ තොරතුරු සඳහා මෙම widget එක නිරන්තරයෙන් නරඹන්න.

## Ultimate Multisite Dashboard

ජාල මට්ටමේ dashboard එක මගින් ජාලය පිළිබඳ සමස්ත තොරතුරු ඉදිරිපත් කරන අතර, Ultimate Multisite dashboard එක (Ultimate Multisite මෙනුවේ මුල් මට්ටමේ පිහිටා ඇති) සේවාව පිළිබඳ තොරතුරු ඉදිරිපත් කරයි.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### මාසික නැවත සිදුවන ආදායම වර්ධනය (Monthly Recurring Revenue Growth)

MRR (මාසික නැවත සිදුවන ආදායම) ප්‍රස්ථාරය මගින් දින 12ක දින දර්ශන කාල පරිච්ඡේදයක් තුළ මාසිකව ලැබෙන ආදායම පිළිබඳ විස්තර පෙන්වයි.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

නව ආදායම් නිල් පැහැයෙන් සලකුණු කර ඇති අතර අවලංගු කිරීම් රතු පැහැයෙන් පිළිබිඹු වේ.

### දිනයක පරාසයන්ට සහාය (Date-range Support)

පාරිභෝගිකයින්ගේ සංඛ්‍යාව සහ ක්‍රියාකාරකම් අනුව, සංඛ්‍යා දත්ත අධික විය හැකියි. dashboard එක තුළ ඇති දිනයක පරාසයන් (date-range support) මගින් ජාල පරිපාලකයාට අවශ්‍ය තොරතුරු සහ කාල පරිච්ඡේදය පමණක් කෙරෙහි අවධානය යොමු කිරීමට ඉඩ සලසයි.

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

මෙම දිනයක පරාසයන්, ආදායම, නව සාමාජිකත්වයන්, රට අනුව ලියාපදිංචි වීම්, පෝරමය අනුව ලියාපදිංචි වීම් සහ වැඩිපුරම පිවිසෙන අඩවි ඇතුළු widget වල පහළින් පෙන්වන තොරතුරු සඳහා කාල පරිච්ඡේදය නියම කරයි.

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

**Most Visited Sites** widget එක ක්‍රියාත්මක කිරීමට, **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** වෙත ගොස් visit counting සක්‍රීය කරන්න.
