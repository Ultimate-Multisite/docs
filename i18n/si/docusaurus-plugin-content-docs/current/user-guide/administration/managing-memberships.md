# සාමාජිකත්වයන් කළමනාකරණය කිරීම {#managing-memberships}

Ultimate Multisite එකේ, ඔබට ඔබේ ජාලයේ ඇති සියලුම දායකත්වයන් (subscriptions) සහ ප්‍රවේශයන් කළමනාකරණය කළ හැකියි. සැලසුම් (plans) සහ නිෂ්පාදන (products), වෙබ් අඩවි ගණන, බිල්පත් මුදල සහ සංඛ්‍යාතය, ගෙවීම් ඉතිහාසය සහ කාල සටහන් වැනි දේ සමාලෝචනය කර සංස්කරණය කළ හැකියි.

මෙම පිටුවට පිවිසීමට, Ultimate Multisite මෙනුවේ ඇති **Memberships** ටැබයට යන්න.

![Memberships list page](/img/admin/memberships-list.png)

Memberships පිටුවේදී ඔබට ඔබේ පාරිභෝගිකයින්ගේ ලැයිස්තුව සහ ඔවුන්ගේ ගිණුම් සමඟ සම්බන්ධ නිෂ්පාදන දැකගත හැකියි. තත්ත්වය අනුව (active, pending, on hold, expired, or cancelled) සාමාජිකත්වයන් පෙරහනක් (filter) ලෙස භාවිතා කර ඔබට ඒවා පෙරහනය කළ හැකියි.

නිදහස් සාමාජිකත්වයන් **සදාකාලික** සාමාජිකත්වයන් ලෙස සැලකේ. ඒවාට අවසන් දිනයක් නොලැබෙන අතර ස්වයංක්‍රීයව අවසන් නොවේ. නිදහස් සැලැස්මක් අත්හිටුවීම හෝ නියමිත කාලයක් අවසානයේ එය අවසන් විය යුතු නම්, ප්‍රසාද දීමනා සහිත (trial) නිෂ්පාදනයක් සමඟ ගෙවන ලද නිෂ්පාදනයක් භාවිතා කරන්න, සාමාජිකත්වය අතින් අවලංගු කරන්න, නැතිනම් ඔවුන්ගේ ප්‍රවේශය වෙනස් විය යුතු විට පාරිභෝගිකයා තවත් සැලැස්මක් වෙත මාරු කරන්න.

සාමාජිකත්වයකට ක්ලික් කිරීමෙන් ඔබට එහි විස්තර සංස්කරණය කළ හැකි පිටුවට යා හැකිය — ගෙවීම් තොරතුරු, වෙබ් අඩවි හෝ පිවිසුම් සීමාවන් සහ තවත් බොහෝ දේ.

![Membership edit interface](/img/admin/membership-edit.png)

සාමාජිකත්ව සංස්කරණ පිටුවේ සම්පූර්ණ දර්ශනය මෙන්න:

![Membership edit full interface](/img/admin/membership-edit-full.png)

ඔබට ඔබේ ජාලයේ කලින් ලියාපදිංචි වූ පරිශීලකයන් සමඟ සම්බන්ධ නව සාමාජිකත්වයන් එක් කළ හැකිය. **Add Membership** ක්ලික් කර විස්තර වින්‍යාස කරන්න.

![Add Membership modal](/img/admin/membership-add-modal.png)

ඉන්පසු **Create Membership** ක්ලික් කරන්න. නව සාමාජිකත්වය Memberships ලැයිස්තුවේ දිස්වේ.
