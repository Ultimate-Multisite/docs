# Gratis AI Agent සැකසුම් {#gratis-ai-agent-settings}

Gratis AI Agent තුළ ඇති **Settings → Advanced** තිරය, පසුපස ඒකාබද්ධ කිරීම් සඳහා පරිපාලක මට්ටමේ වින්‍යාසය සපයයි. මෙම පිටුව ප්‍රතිචාර ඉදිරියට යැවීම, සෙවුම් සපයන්නාගේ යතුරු, කළමනාකරණය කළ Superdav සේවා සැකසුම, Google Calendar පාලන, TextBee SMS සැකසුම්, සහ ජාලය පුරා ක්‍රියාත්මක විශේෂාංග සලකුණු ලේඛනගත කරයි.

## සැකසුම් වෙත ප්‍රවේශ වීම {#accessing-settings}

1. WordPress පරිපාලකයේ, **Gratis AI Agent → Settings** වෙත යන්න.
2. **Advanced** ටැබය ක්ලික් කරන්න.

## ප්‍රතිචාර අන්ත ලක්ෂ්‍ය වින්‍යාසය {#feedback-endpoint-configuration}

පරිශීලකයෙකු thumbs-down බොත්තම, ස්වයං-ඉඟි බැනරය, හෝ `/report-issue` විධානය හරහා ප්‍රතිචාරයක් යොමු කරන සෑම විටම, ප්‍රතිචාර අන්ත ලක්ෂ්‍යය AI නියෝජිතයාගෙන් POST ඉල්ලීම් ලබා ගනී.

| ක්ෂේත්‍රය | විස්තරය |
|---|---|
| **ප්‍රතිචාර අන්ත ලක්ෂ්‍ය URL** | JSON ශරීරයක් සමඟ HTTP POST ඉල්ලීම් ලෙස ප්‍රතිචාර යොමුකිරීම් ලබා ගන්නා URL. |
| **Feedback API Key** | සෑම ප්‍රතිචාර ඉල්ලීමකම `Authorization` header තුළ යවන bearer token එකකි. ඔබගේ අන්ත ලක්ෂ්‍යයට සත්‍යාපනය අවශ්‍ය නොවේ නම් හිස්ව තබන්න. |

### අපේක්ෂිත JSON භාරය {#expected-json-payload}

ඔබගේ ප්‍රතිචාර අන්ත ලක්ෂ්‍යය අවම වශයෙන් පහත ක්ෂේත්‍ර සහිත JSON ශරීරයක් පිළිගත යුතුය:

```json
{
  "message_id": "msg_abc123",
  "conversation_id": "conv_xyz789",
  "feedback_text": "The answer was incorrect about pricing.",
  "triage_category": "factual_error"
}
```

සංවාද සන්දර්භය අනුව භාරය තුළ අමතර ක්ෂේත්‍ර තිබිය හැක.

### `triage_category` අගයන් {#triagecategory-values}

AI වර්ගීකරණ ස්ථරය, භාරය ඉදිරියට යැවීමට පෙර `triage_category` සඳහා පහත අගයන්ගෙන් එකක් නියම කරයි:

| අගය | අර්ථය |
|---|---|
| `factual_error` | සහායකයා වැරදි සත්‍ය තොරතුරු සපයා ඇත. |
| `unhelpful_answer` | ප්‍රතිචාරය තාක්ෂණිකව නිවැරදි වූ නමුත් ප්‍රයෝජනවත් නොවීය. |
| `inappropriate_content` | ප්‍රතිචාරයේ පරිශීලකයන්ට පෙන්විය නොයුතු අන්තර්ගතයක් තිබුණි. |
| `other` | ප්‍රතිචාරය දන්නා කාණ්ඩයකට නොගැළපුණි. |

### සත්‍යාපනය {#authentication}

ඔබගේ අන්ත ලක්ෂ්‍යයට සත්‍යාපනය අවශ්‍ය නම්, **Feedback API Key** ක්ෂේත්‍රය ඔබගේ bearer token වෙත සකසන්න. නියෝජිතයා යවන්නේ:

```
Authorization: Bearer <your-api-key>
```

**Feedback API Key** ක්ෂේත්‍රය හිස් නම්, `Authorization` header එකක් නොයවයි.

### ප්‍රතිචාර එකතු කිරීම අක්‍රීය කිරීම {#disabling-feedback-collection}

**ප්‍රතිචාර අන්ත ලක්ෂ්‍ය URL** සහ **Feedback API Key** ක්ෂේත්‍ර දෙකම හිස්ව තබන්න. thumbs-down බොත්තම සහ ප්‍රතිචාර UI පරිශීලකයන්ට පෙනේ, නමුත් යොමුකිරීම් කිසිදු බාහිර සේවාවකට ඉදිරියට නොයවයි.

## Brave Search API Key {#brave-search-api-key}

**Advanced** ටැබයෙහිම, **Brave Search API Key** ක්ෂේත්‍රය [අන්තර්ජාල සෙවුම්](../configuration/internet-search) හැකියාව සක්‍රීය කරයි.

| ක්ෂේත්‍රය | විස්තරය |
|---|---|
| **Brave Search API Key** | Brave Search සංවර්ධක Dashboard වෙතින් ඔබගේ API යතුර. AI සහායකයේ අන්තර්ජාල සෙවුම සක්‍රීය කිරීමට අවශ්‍ය වේ. |

ක්ෂේත්‍ර ලේබලයේ Brave Search API ලියාපදිංචි වීමේ පිටුවට ක්ලික් කළ හැකි සබැඳියක් ඇත. අන්තර්ජාල සෙවුම අක්‍රීය කිරීමට හිස්ව තබන්න.

මෙම විශේෂාංගය පිළිබඳ අවසන් පරිශීලක ලේඛන සඳහා [අන්තර්ජාල සෙවුම්](../configuration/internet-search) බලන්න.

## කළමනාකරණය කළ Superdav සේවාව {#managed-superdav-service}

Superdav AI Agent v1.18.0, සහය දක්වන අඩවි සඳහා කළමනාකරණය කළ Superdav සේවා අන්ත ලක්ෂ්‍ය සහ ස්වයංක්‍රීය සම්බන්ධතා සැපයීම එක් කරයි. ඔබගේ අඩවිය අතින් වින්‍යාස කළ සේවා අන්ත ලක්ෂ්‍යයක් වෙනුවට සත්කාරක සපයන්නාට සම්බන්ධ විය යුතු විට මෙම පාලන භාවිතා කරන්න.

| ක්ෂේත්‍රය | විස්තරය |
|---|---|
| **කළමනාකරණය කළ Superdav සේවාව** | සහය දක්වන අඩවි සඳහා සත්කාරක Superdav සේවා සම්බන්ධතාව සක්‍රීය කරයි. |
| **සම්බන්ධතාව සැපයීම** | ස්වයංක්‍රීය අන්ත ලක්ෂ්‍ය සහ අක්තපත්‍ර සැපයීම ආරම්භ කරයි. අඩවිය කළමනාකරණය කළ සපයන්නා භාවිතා කළ යුතු බව තහවුරු කිරීමෙන් පසු මෙය භාවිතා කරන්න. |

සැපයීමෙන් පසු, සැකසුම් සුරකින්න සහ කළමනාකරණය කළ සේවා වැඩප්‍රවාහ මත රඳා සිටීමට පෙර සම්බන්ධතා තත්ත්වය සත්‍යාපනය කරන්න. සැපයීම අසාර්ථක නම්, පෙන්වන නැවත උත්සාහ මඟපෙන්වීම පරීක්ෂා කර, සත්කාරක සපයන්නා භාවිතා කිරීමට අඩවියට අවසර ඇති බව තහවුරු කරන්න.
