# Ultimate Multisite පරිවර්තනය කිරීම

Ultimate Multisite කියන්නේ ඉංග්‍රීසියෙන් දියුණු කළ ප්ලගිනක් වුණත්, ඒක සම්පූර්ණයෙන්ම පරිවර්තනය කරන්න පුළුවන් ហើយ අපි පරිවර්තන සඳහා සහභාගී වීම ගැන සතුටු වෙනවා.

## ප්‍රධාන ප්ලගින් පරිවර්තන

ප්‍රධාන Ultimate Multisite ප්ලගින් එක පරිවර්තනය කරන්නේ නිල **WordPress.org පරිවර්තන වේදිකාව** හරහායි:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

මේක තමයි WordPress ප්‍රජාවේ සම්මත පරිවර්තන පද්ධතිය. පරිවර්තන ප්‍රමාණවත් අනුමැතියක් ලැබුණු විට, ඒවා WordPress ප්ලගින් යාවත්කාලීන කිරීම් හරහා Ultimate Multisite භාවිතා කරන සියලුම පරිශීලකයින්ට ස්වයංක්‍රීයව බෙදා හරිනවා.

**අපේ උදව් ඕනේ!** ඔබ ඉංග්‍රීසි හැර වෙනත් භාෂාවක් කතා කරනවා නම්, කරුණාකර පරිවර්තන දායක වීමට සලකා බලන්න. කිහිපයක් පරිවර්තනය කිරීම පවා ආරම්භ කිරීමට උපකාරී වේ. ආරම්භ කිරීමට:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) වෙත පිවිසෙන්න
2. ඔබේ භාෂාව තෝරන්න
3. "Contribute Translation" (පරිවර්තනය දායක වන්න) ක්ලික් කරන්න
4. ඔබට නොමිලේ WordPress.org ගිණුමක් අවශ්‍ය වනු ඇත

ඔබේ භාෂාව සඳහා පරිවර්තන සංස්කාරකවරයෙකු (පරිවර්තන අනුමත කළ හැකි අයෙක්) වීමට කැමති නම්, [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) හරහා සම්බන්ධ වන්න.

## Addon පරිවර්තන

Addon ප්ලගින් පරිවර්තනය කරන්නේ වෙනම සිදු කරනවා:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

පัจจุบัน add-on පරිවර්තන ආරම්භක අවස්ථාවක් ලෙස AI මගින් ජනනය කර ඇත. ඒවා සහාය වන සියලුම භාෂාවන් ආවරණය කළත්, සමහර විට අපැහැදිලි වාක්‍ය රටා හෝ වැරදි තොරතුරු තිබිය හැකියි. ඒවා වඩාත් ස්වාභාවික හා නිවැරදි කිරීමට මිනිස් පරිවර්තකයන්ට සමාලෝචනය කර වැඩිදියුණු කිරීම් යෝජනා කිරීමට අපි සාදරයෙන් පිළිගනිමු.

## දේශීය පරිවර්තන

ඔබට උසස් මට්ටමේ (upstream) දායකත්වය දීමට වඩා ඔබේම ව්‍යාපෘතිය සඳහා දේශීය පරිවර්තන නිර්මාණය කිරීමට කැමති නම්, ඔබට විකල්ප දෙකක් තිබේ:

### Poedit භාවිතා කිරීම

`.pot` ගොනුව පරිවර්තනය කිරීමට ඔබ **[Poedit](https://poedit.net)** ડેස්ක්ටොප් යෙදුම භාවිතා කළ හැකිය:

1. [poedit.net](https://poedit.net) වෙතින් Poedit බාගත කරන්න
2. `wp-content/plugins/ultimate-multisite/lang/` හි ඇති `.pot` ගොනුව විවෘත කරන්න
3. Poedit අතුරුමුහුණතේ ඇති string (පෙළ) පරිවර්තනය කරන්න
4. සුරකින්න -- Poedit මගින් `.po` සහ `.mo` ගොනු ජනනය කරයි
5. ප්‍රතිදාන ගොනු `lang` ෆෝල්ඩරයේ තබන්න

### Loco Translate භාවිතා කිරීම

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin මගින් ඔබට ડેස්ක්ටොප් මෙවලම් කිසිවක් නොමැතිව, admn panel එකෙන් කෙලින්ම පරිවර්තනය කළ හැකියි. එය ඔබ වෙනුවෙන් පරිවර්තන ගොනු නිර්මාණය කිරීම සහ සම්පාදනය (compile) කිරීම සිදු කරයි.
