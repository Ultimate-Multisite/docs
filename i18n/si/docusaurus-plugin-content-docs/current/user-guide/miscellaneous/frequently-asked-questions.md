# မကြာခဏ မေးလေ့ရှိသော မေးခွန်းများ

အထောက်အပံ့ တောင်းဆိုမှုများတွင် မကြာခဏ ပေါ်လာတတ်သည့် **မကြာခဏ မေးလေ့ရှိသော မေးခွန်းများ** ကို အောက်တွင် ဖော်ပြထားပါသည်။

  * **Ultimate Multisite ဆိုတာ ဘာလဲ?**

Ultimate Multisite ဟာ WordPress multisite install တွေအတွက် ရည်ရွယ်တဲ့ WordPress plugin တစ်ခုဖြစ်ပြီး၊ သင့်ရဲ့ WordPress install ကို [WordPress.com](https://WordPress.com) လိုမျိုး ပရီမီယံ ဝဘ်ဆိုက်ကွန်ရက်တစ်ခုအဖြစ် ပြောင်းလဲပေးပါတယ်။ ဒါမှ အသုံးပြုသူတွေဟာ လစဉ်၊ သုံးလတစ်ကြိမ်၊ သုံးနှစ်စာ စတဲ့ အခကြေးငွေနဲ့ ဆိုက်တွေ ဖန်တီးနိုင်မှာ ဖြစ်ပါတယ် (အခမဲ့ ပလန်တွေလည်း ဖန်တီးနိုင်ပါတယ်)။

  * **Multisite မရှိဘဲ Ultimate Multisite နဲ့ အလုပ်လုပ်မလား?**

မလုပ်နိုင်ပါဘူး။ Ultimate Multisite ဟာ WordPress multisite၊ subdirectory ဒါမှမဟုတ် subdomain လိုအပ်ပါတယ်။

  * **Ultimate Multisite အတွက် လိုအပ်ချက်တွေက ဘာတွေလဲ?**

Ultimate Multisite မှာ အောက်ပါအရာတွေ လိုအပ်ပါတယ်-

**WordPress Version** : v5.3+ (အကြံပြုသည်: နောက်ဆုံးတည်ငြိမ်သော ဗားရှင်း)
**PHP Version** : 7.4.x (8.0 ထက်ကျော်ရင်လည်း မကြာခင် ပံ့ပိုးပေးမှာပါ)
**MySQL Version** : v5+ (အကြံပြုသည်: Hosting provider မှာ 8.0 မရှိသေးရင် Version 5.6 ကို အသုံးပြုဖို့ အကြံပြုပါတယ်)

_Multisite (subdomain ဒါမှမဟုတ် subdirectory) ကိုလည်း အသက်ဝင်အောင် လုပ်ထားဖို့ လိုအပ်ပါတယ်._ [WordPress Multisite ကို ဘယ်လို တင်ရမလဲ](/user-guide/getting-started/how-to-install-wordpress-multisite)။

  * **Subdirectories နဲ့ Multisite နဲ့ Ultimate Multisite နဲ့ အလုပ်လုပ်နိုင်မလား?**

လုပ်နိုင်ပါတယ်။ Ultimate Multisite ဟာ subdirectory install တွေနဲ့ subdomain install နှစ်မျိုးစလုံးကို ပံ့ပိုးပေးပါတယ်။

  * **Subdomains နဲ့ Multisite နဲ့ Ultimate Multisite နဲ့ အလုပ်လုပ်နိုင်မလား?**

လုပ်နိုင်ပါတယ်။ Ultimate Multisite ဟာ subdirectory install တွေနဲ့ subdomain install နှစ်မျိုးစလုံးကို ပံ့ပိုးပေးပါတယ်။

  * **အသုံးပြုသူတွေ ဘယ် Plugin တွေနဲ့ Theme တွေကို ကန့်သတ်ဖို့ Ultimate Multisite ကို သုံးလို့ရပါသလား?**

သုံးလို့ရပါတယ်။ Ultimate Multisite နဲ့ဆိုရင် သင့်ကွန်ရက်ပေါ်က အစီအစဉ် (plan) တစ်ခုချင်းစီအတွက် ဘယ် plugin တွေနဲ့ theme တွေကို ရွေးချယ်ပေးနိုင်တယ်ဆိုတာကို သင်ရွေးချယ်နိုင်ပါတယ်။

  * **အသုံးပြုသူတစ်ဦးရဲ့ ဆိုက်ပေါ်မှာ ဖန်တီးနိုင်တဲ့ post အရေအတွက်ကို ကန့်သတ်ဖို့ Ultimate Multisite ကို သုံးလို့ရပါသလား?**

ඔව්. Ultimate Multisite එකෙන් ඔබේ ජාලයේ ඇති **සෑම පෝස් වර්ගයකටම (every available post type)** සඳහා පෝස් ප්‍රමාණයන් (post quotas) සකසා ගත හැකියි. පරිපාලකයෙකු ආරාධනා කළ හැකි පරිශීලකයන් ගණන, **ดิස්ක් ඉඩ ප්‍රමාණය (amount of disk space)** සහ තවත් බොහෝ දේ සීමා කිරීමට ඔබට හැකියාව තිබෙනවා.

  * **Ultimate Multisite Site Templates සඳහා සහාය දක්වයිද?**

ඔව්. Ultimate Multisite එකෙන් ඔබට අවශ්‍ය පරිදි Site Templates ගණනක් නිර්මාණය කළ හැකියි. Site Templates කියන්නේ themes (තේමා) නෙවෙයි, ඒ වෙනුවට ඔබේ ජාලයේ ඇති සාමාන්‍ය වෙබ් අඩවි (normal sites) වගේ ඒවා. ඔබට ප්ලග්ඉන් (plugins), තේමා සහ එම Site Template එක ඕනෑම ආකාරයකින් කස්ටමයිස් (customize) කරන්න පුළුවන්. පරිශීලකයෙකු එම template එක තෝරා ගන්නේ නම්, ඔහුගේ හෝ ඇයගේ අඩවිය එම template එකේ පිටපතක් ලෙස නිර්මාණය වනු ඇති අතර සියලුම කස්ටමයිසි කරුණු එහි තිබිය හැකියි.

ඔබට එක් සැලැස්මක් සඳහා විශේෂිත template එකක් සකසා දිය හැකියි. එසේ කිරීමෙන්, එම සැලැස්මට ලියාපදිංචි වන සෑම පරිශීලකයෙකුගේම අඩවිය එම Site Template එකේ පිටපතක් ලෙස නිර්මාණය වනු ඇත.

ඔව්!

* **මගේ Plan Edit පිටුවේ “Allowed Plugins” ටැබය හිස් ඇත්තේ ඇයි?**

Ultimate Multisite ඔබට ඔබේ ජාලයේ ක්‍රියාත්මක නොවන ප්ලග්ඉන් වලට ප්‍රවේශය ලබා දීමට හෝ ප්‍රතික්ෂේප කිරීමට පමණක් ඉඩ දෙයි. ප්ලග්ඉනයක් ජාලයේ සක්‍රීය කර ඇති විට, එහි ස්වභාවය අනුව එය ඔබේ ජාලයේ සෑම වෙබ් අඩවියකම පෙරනිමඟින් (default) පැටවීම සිදු වේ.

“Allowed Plugins” ටැබයේ කිසිදු ප්ලග්ඉන් ලැයිස්තුවක් නොපෙනේ නම්, ඔබ අවශ්‍ය ප්ලග්ලය ජාලයේ ක්‍රියාත්මක දැයි පරීක්ෂා කරන්න.

ඔව්. Ultimate Multisite සැමවිටම WP Engine මගින් සහාය දක්වා තිබුණි, නමුත් ડોමේන් මැප කිරීම (domain mapping) ක්‍රියාත්මක වීමට පරිපාලකයාගෙන් අතින් අමතර පියවරක් ගැනීමට සිදු විය (WP Engine පැනලයට ડોමේනය එකතු කිරීම). **මෙය 1.6.0 අනුවායෙන් පසු වෙනස් වී ඇත.** Ultimate Multisite WP Engine API සමඟ සෘජුව සම්බන්ධ වන අතර මැප කිරීමෙන් පසු ડોමේන් ස්වයංක්‍රීයව පැනලයට එක් වේ.
