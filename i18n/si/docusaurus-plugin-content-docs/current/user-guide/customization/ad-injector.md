# ඔබේ ජාලයේ දැන්වීම් තැබීම Ultimate Multisite සමඟ: Ad Injector

Ultimate Multisite: Ad Injector කියන්නේ **නිදහස් add-on එකක්** වන අතර, ඒ හරහා ඔබට ඔබේ සැලසුම්වල (plans) සංස්කරණ පිටුවෙන්ම දැන්වීම් තැබීමට අවශ්‍ය මෙවලම් ලබා දෙනවා.

Ultimate Multisite: Ad Injector ස්ථාපනය කිරීමෙන් පසු, එය සක්‍රිය කිරීමට ඔබේ _Plugins_ පිටුවට යන්න.

add-on එක ක්‍රියාත්මක වූ පසු, සැලසුම් සංස්කරණ තිරයේ ඇති _Advanced Options_ meta box එකේ _Ad Settings_ නමින් නව විකල්පයක් දිස්වනු ඇත.

_දැන් ඔබට ඔබේ සෑම සැලසුමක් සඳහාම දැන්වීම් සැකසුම් සංස්කරණය කළ හැකියි_

එම ටැබ් එකේ, ඔබ ඉදිරිපෙළ (front-end) හි පෝස්ට් සහ පිටු වලට, අන්තර්ගතයට පෙර සහ පසු දැන්වීම් කේත (ad codes) එක් කිරීමට හැකි වනු ඇත.

_ඔබ ඔබේ දැන්වීමේ කේතය ඇතුළත් කළ පසු, එය එම නිශ්චිත සැලසුමේ පෝස්ට් සහ පිටු වල දිස් වනු ඇත._

එම සැලසුමේ පරිශීලකයන් සඳහා පරිපාලන පිටු (Admin Pages) හි හෙඩර් එකේ දැන්වීම් ද ඇතුළත් කිරීමට ඔබට හැකියාව තිබේ.

_ඔබට ඔබේ පරිශීලකයාගේ පසුපස පුවරුවේ (back-end) දැන්වීම් ප්‍රදර්ශනය කිරීමට අවශ්‍ය නම්, Admin Panel Ad Code විකල්පයට යන්න._

### දැන්වීම් වලට ඔබ්බට {#beyond-advertising}

ඉහළ මට්ටමේ සැලසුම් සඳහා සහාය ලබා දීමටත් ඔබට Ultimate Multisite: Ad Injector භාවිතා කළ හැකියි.

_Admin Panel Ads_ විකල්පයට ගොස් ඔබේ සහාය මෙවලම සඳහා HTML කේතය එක් කරන්න.

මේ ආකාරයෙන් අයට ඔවුන්ගේ Admin Panel එකෙන් ඔබට කෙළින්ම සම්බන්ධ විය හැකිය!
