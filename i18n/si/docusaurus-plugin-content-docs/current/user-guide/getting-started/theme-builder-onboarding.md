# තේමා ගොඩනැගීමේ ආරම්භක ක්‍රියාවලිය (Theme Builder Onboarding Flow)

Superdav AI Agent v1.12.0 සමඟ, ඔබ මුලින්ම සැකසීමේදී කස්ටම් බ්ලොක් තේමක් නිර්මාණය කිරීමට උපකාරී වන මඟ පෙන්වන **තේමා ගොඩනැගීමේ ආරම්භක ක්‍රියාවලිය (Theme Builder onboarding flow)** හඳුන්වා දී ඇත. මෙය පැරණි Site Builder මාදිලිය වෙනුවට වඩාත් නම්‍යශීලී, ඒජන්ට් සහාය සහිත ප්‍රවේශයක් බවට පත්වේ.

## තේමා ගොඩනැගීමේ ආරම්භක ක්‍රියාවලිය යනු කුමක්ද?

තේමා ගොඩනැගීමේ ආරම්භක ක්‍රියාවලිය යනු අන්තර්ක්‍රියාකාරී සැකසුම් මඟ පෙන්වන්නෙකු (interactive setup wizard) වන අතර එය පහත දේ කරයි:

- නිර්මාණ තීරණ හරහා ඔබව මඟ පෙන්වීම (වර්ණ, අකුරු වර්ග, සැකිල්ල - layout)
- ඔබේ වෙබ් අඩවියේ දෘශ්‍ය අනන්‍යතා කැමැත්ත ලබා ගැනීම
- ඔබේ අවශ්‍යතා අනුව සකස් කරන ලද කස්ටම් බ්ලොක් තේමක් ජනනය කිරීම
- සම්පූර්ණ වූ පසු තේමාව ස්වයංක්‍රීයව ක්‍රියාත්මක කිරීම

මෙම ක්‍රියාවලිය **Setup Assistant agent** මගින් බල ගැන්වේ. එය පැහැදිලි කරන ප්‍රශ්න අසමින් ඔබේ තේමාව පියවරෙන් පියවර ගොඩනඟයි.

## තේමා ගොඩනැගීම ආරම්භ කිරීම

### පළමු වරට සැකසීම (First-Run Setup)

ඔබ නව WordPress ස්ථාපනයක් මත Superdav AI Agent මුලින්ම දියත් කරන විට, ඔබට මෙවැනි දෙයක් පෙනෙනු ඇත:

```
Superdav AI Agent වෙත සාදරයෙන් පිළිගනිමු!

ඔබට කුමක් කිරීමට අවශ්‍යද?
1. කස්ටම් තේමක් ගොඩනැගීම (Theme Builder)
2. දැනට පවතින තේමාවක් භාවිතා කිරීම
3. මේ සඳහා දැන් සැකසීම මඟ හැරීම
```

තේමා ගොඩනැගීමේ ආරම්භක ක්‍රියාවලියට ඇතුළු වීමට **"Build a custom theme"** තෝරන්න.

### අතින් ක්‍රියාත්මක කිරීම (Manual Activation)

ඔබ ඕනෑම වේලාවක පහත සඳහන් දේ ඉල්ලා සිටීමෙන් තේමා ගොඩනැගීමේ ආරම්භක ක්‍රියාවලිය ආරම්භ කළ හැකිය:

```
"Start the Theme Builder onboarding"
```

හෝ

```
"Help me create a custom theme"
```

## ආරම්භක පියවර (The Onboarding Steps)

### පියවර 1: මාදිලිය තෝරා ගැනීම (Mode Selection)

Setup Assistant agent ඔබගේ කැමැත්ත ගැන විමසයි:

```
ඔබේ තේමාව ගොඩනැගීමට ඔබට අවශ්‍ය වන්නේ කෙසේද?
- මඟ පෙන්වීම සහිතව (මට ප්‍රශ්න අසමින් එය ඔබ වෙනුවෙන් ගොඩනඟා දෙන්නම්)
- අතින් ක්‍රියා කිරීම (මට විකල්ප පෙන්වා ඔබ තීරණය කරන්න)
```

බොහෝ පරිශීලකයින් සඳහා **Guided mode** නිර්දේශ කරනු ලැබේ; ඔබේ කර්මාන්තය සහ ඉලක්ක මත පදනම්ව ඒජන්ට් නිර්මාණ නිර්දේශ කරයි.

### පියවර 2: වෙබ් අඩවි විස්තරය (Site Specification)

ඔබේ වෙබ් අඩවිය ගැන ඔබට ප්‍රශ්න ඇසනු ඇත:

- **සයිට් අරමුණ**: E-commerce, බ්ලොග්, 포트폴리오, SaaS වැනි දේ.
- **ඉලක්කගත ප්‍රේක්ෂකයින්**: ඔබේ වෙබ් අඩවියට පැමිණෙන්නේ කවුද?
- **වෙළඳ නාම වර්ණ**: ප්‍රාථමික සහ ද්විතීයික වර්ණ (හෝ "මට විශ්වාස නැහැ").
- **ස්වරය**: වෘත්තීය, නිර්මාණශීලී, විනෝදජනක, අවම, ආදිය.

මෙම තොරතුරු ඔබේ **site_brief** මතකයේ ගබඩා කර ඇත, එය අනාගත සැසිවලදී ඒකාබද්ධයන් විසින් යොමු කෙරේ.

### පියවර 3: නිර්මාණ පද්ධති තීරණ (Design System Decisions)

ඒකාබද්ධයා ඔබට නිර්මාණ ටෝකන් (design tokens) තෝරා ගැනීම පිළිබඳ මඟ පෙන්වයි:

- **අකුරු සැකැස්ම (Typography)**: අකුරු පවුල (serif, sans-serif, monospace) සහ ප්‍රමාණය පරිමාණය.
- **වර්ණ පලස (Color palette)**: ප්‍රාථමික, ද්විතීයික, අවධාරණ (accent), සහ මධ්‍යස්ථ වර්ණ.
- **අතරමැදි (Spacing)**: සංයුක්ත (compact), සාමාන්‍ය (normal), හෝ විශාල (spacious) සැකැස්ම.
- **චලනය (Motion)**: අනුරූපීකරණයන් සහ සංක්‍රාන්ති (desired නම්).

### පියවර 4: තේමා ජනනය (Theme Generation)

Setup Assistant ඒකාබද්ධයා ඔබේ කස්ටම් බ්ලොක් තේමාව පහත දේ සමඟ සැකසීමට උපකාරී වේ:

- සියලු නිර්මාණ ටෝකන් අඩංගු `theme.json`
- පොදු සැකැස්මක් සඳහා බ්ලොක් සැකිලි (homepage, blog, contact)
- ඔබේ නිර්මාණ පද්ධතියට ගැලපෙන කස්ටම් බ්ලොක් විලාසිතා
- තේමා 메ටා දත්ත සහ WordPress සහය ප්‍රකාශන

### පියවර 5: සක්‍රීය කිරීම සහ තහවුරු කිරීම (Activation and Verification)

තේමාව ස්වයංක්‍රීයව සක්‍රීය වන අතර ඔබට පහත දේ දැකගත හැකිය:

```
✓ ඔබේ කස්ටම් තේමාව දැන් ක්‍රියාත්මකයි!
  තේමා නාමය: [ඔබේ වෙබ් අඩවි නාමය] Theme
  වර්ණ: [ප්‍රාථමික] / [ද්විතීයික]
  අකුරු සැකැස්ම: [අකුරු පවුල]

  නව නිර්මාණය දැකීමට ඔබේ වෙබ් අඩවියට පිවිසෙන්න.
```

ඉන්පසු තේමාව නිවැරදිව පෙන්වන බව තහවුරු කර ගැනීමට ඔබට ඔබේ වෙබ් අඩවියට පිවිසිය හැකිය.

## වෙබ් අඩවි විස්තරය සහ site_brief මතකය (Site Specification and site_brief Memory)

ආරම්භ කිරීමේදී, ඒකාබද්ධයා ඔබේ වෙබ් අඩවියේ විස්තර **site_brief** මතක කාණ්ඩයට ග්‍රහණය කර ගනී. මෙයට ඇතුළත් වන්නේ:

- වෙබ් අඩවි අරමුණ සහ ඉලක්ක
- ඉලක්කගත ප්‍රේක්ෂකයින්
- වෙළඳ නාම වර්ණ සහ ස්වරය
- නිර්මාණ මනාපයන්
- අන්තර්ගත ව්‍යුහය

### site_brief වැදගත් වන්නේ ඇයි?

අනාගත සැසිවලදී, ඒකාබද්ධයන් ඔබේ site_brief වෙත යොමු කරන්නේ:

### ඔබේ වෙබ් අඩවියේ සාරාංශය බැලීම

ඔබට පහත විධාන භාවිතා කළ හැකිය:

```
"Show me my site brief"
```

හෝ

```
"What do you know about my site?"
```

එවිට ඒ ඇගෙන් ඔබ ගබඩා කර ඇති ඔබේ වෙබ් අඩවි විස්තරය (site specification) පෙන්වනු ඇත.

## ආරම්භ කිරීමෙන් පසු වෙනස්කම් සිදු කිරීම

Theme Builder එකේ ආරම්භ කිරීම අවසන් වූ පසු, ඔබට පහත දෑ කළ හැකිය:

### Design System Aesthetics skill භාවිතා කිරීම

නිර්මාණයේ සුළු වෙනස්කම් ඉල්ලා සිටින්න:

```
"Refine the typography to be more modern"
```

හෝ

```
"Adjust the color palette to be warmer"
```

**Design System Aesthetics skill** මගින් ඔබට නිශ්චිත නිර්මාණ යාවත්කාලීන කිරීම් සඳහා මඟ පෙන්වනු ඇත.

### theme.json සෘජුව සංස්කරණය කිරීම

උසස් පරිශීලකයින් සඳහා, පහත දේ වෙනස් කිරීමට `/wp-content/themes/[theme-name]/theme.json` ගොනුව සංස්කරණය කරන්න:

- Color tokens (වර්ණ කේත)
- Typography scales (අකුරු ප්‍රමාණ මට්ටම්)
- Spacing values (පරතර වටිනාකම්)
- Border and shadow definitions (සීමාවන් සහ සෙවනැලි නිර්වචන)

### අභිරුචි බ්ලොක් සැකිලි (Custom Block Templates) නිර්මාණය කිරීම

WordPress block editor එක භාවිතා කර පහත දේ සඳහා අභිරුචි සැකිලි සාදන්න:

- Homepage layouts (මුල් පිටුවේ සැකසුම්)
- Blog post pages (බ්ලොග් පෝස් පිටු)
- Product pages (නිෂ්පාදන පිටු)
- Contact forms (සම්බන්ධතා පෝරම)

## ගැටළු නිරාකරණය (Troubleshooting)

**ආරම්භ කිරීමේ ක්‍රියාවලිය සම්පූර්ණ නොවීය**
- ක්‍රියාවලිය නැවත ආරම්භ කරන්න: "Start the Theme Builder onboarding"
- ඔබේ WordPress ස්ථාපනය යාවත්කාලීන කර ඇති බව තහවුරු කරන්න.
- Setup Assistant agent සක්‍රීය කර ඇත්දැයි පරීක්ෂා කරන්න

**මගේ site_brief එක භාවිතා නොවේ**
- ඒ ඇගෙන් මතකය (memory) වලට ප්‍රවේශ වීමට ඇති හැකියාව තහවුරු කරන්න
- ඇගෙන් "මගේ site brief එක මතක් කරပါ" කියලා අසන්න
- ඔබේ සැකසුම් වල මතකය සක්‍රීය කර තිබේදැයි පරීක්ෂා කරන්න

**උත්පාදිත තේමාව මගේ කැමැත්තට ගැලපෙන්නේ නැත**
- එය වැඩිදියුණු කිරීමට Design System Aesthetics කුසලතාව (skill) භාවිතා කරන්න
- ඇගෙන් "[විශේෂ වෙනස්කම්] සමඟ තේමාව නැවත ජනනය කරන්න" කියලා අසන්න
- නිවැරදි පාලනය සඳහා theme.json එක කෙලින්ම සංස්කරණය කරන්න

## ඊළඟ පියවර

Theme Builder onboarding අවසන් කිරීමෙන් පසු:

1. **ඔබේ වෙබ් අඩවිය තහවුරු කරන්න**: නව තේමාව බැලීමට ඔබේ වෙබ් අඩවියට පිවිසෙන්න
2. **නිර්මාණය වැඩිදියුණු කරන්න**: වෙනස්කම් සඳහා Design System Aesthetics කුසලතාව භාවිතා කරන්න
3. **අන්තර්ගතය එක් කරන්න**: ඔබේ වෙබ් අඩවියේ අන්තර්ගතය ගොඩනැගීම ආරම්භ කරන්න
4. **හැකියාවන් ගවේෂණය කරන්න**: scaffold-block-theme සහ activate-theme වැනි වෙනත් agent හැකියාවන් ගැන ඉගෙන ගන්න
