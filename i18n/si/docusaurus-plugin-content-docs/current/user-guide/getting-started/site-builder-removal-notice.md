# સાઇટ બિલ્ડર મોડ દૂર કરવા અંગેની સૂચના

**Superdav AI Agent v1.12.0 માં Site Builder mode દૂર કરવામાં આવ્યું છે.** જો તમે Site Builder mode નો ઉપયોગ કરતા હતા, તો થીમ બનાવવાની અને સાઇટ સેટઅપ માટે **Setup Assistant agent** પર માઈગ્રેટ (Migration) કરવું જોઈએ.

## શું થયું?

### Site Builder Mode (Legacy)

Site Builder mode એ નીચેની બાબતો માટે એક વિઝાર્ડ-આધારિત ઇન્ટરફેસ હતું:

- ટેમ્પલેટ્સમાંથી સાઇટ્સ બનાવવી
- મૂળભૂત સેટિંગ્સ ગોઠવવી
- થીમ પસંદ કરવી
- પ્રારંભિક સામગ્રી સેટ કરવી

### તેની જગ્યાએ શું આવ્યું છે?

**Setup Assistant agent** હવે નીચેની બાબતો સાથે તમામ Site Builder કાર્યક્ષમતા સંભાળે છે:

- વધુ લવચીક, એજન્ટ-ગાઇડેડ સેટઅપ
- થીમ કસ્ટમાઇઝેશન માટે વધુ સારા વિકલ્પો
- Theme Builder ઓનબોર્ડિંગ સાથે એકીકરણ (Integration)
- ભવિષ્યના સત્રો માટે સ્થાયી site_brief મેમરી

## જો તમે Site Builder Mode નો ઉપયોગ કરતા હોવ તો

### તમારી સાઇટ્સ સુરક્ષિત છે

- Site Builder mode માં બનાવેલી હાલની સાઇટ્સ ચાલુ રહેશે
- કોઈ ડેટા ગુમાવવો કે સાઇટમાં ખલેલ પહોંચાડવી નહીં
- તમે તમારી સાઇટ્સનું સામાન્ય રીતે સંચાલન ચાલુ રાખી શકો છો

### Setup Assistant Agent પર માઈગ્રેટ કરો

નવી સાઇટ સેટઅપ અથવા થીમ ફેરફારો માટે, Setup Assistant agent નો ઉપયોગ કરો:

```
"Help me set up a new site"
```

અથવા

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent વધુ લવચીકતા સાથે સમાન કાર્યક્ષમતા પ્રદાન કરે છે.

## તુલના: Site Builder વિરુદ્ધ Setup Assistant

| લક્ષણ | Site Builder (હեռացેલ) | Setup Assistant (નવું) |
|---|---|---|
| સેટઅપ પદ્ધતિ | વિઝાર્ડ ફોર્મ | એજન્ટ વાતચીત |
| થીમ પસંદગી | વ્યાખ્યાયિત ટેમ્પલેટ્સ | કસ્ટમ જનરેશન |
| કસ્ટમાઇઝેશન | મર્યાદિત વિકલ્પો | સંપૂર્ણ ડિઝાઇન સિસ્ટમ |
| સાઇટ બ્રીફ | સંગ્રહિત નથી | સ્થાયી મેમરી |
| ભવિષ્યના સત્રો | પુનરાવર્તિત સેટઅપ | સ્ટોર્ડ site_brief નો ઉપયોગ કરો |
| લવચીકતા | નિશ્ચિત વર્કફ્લો | અનુકૂલનશીલ વાતચીત |

## Setup Assistant Agent પર માઈગ્રેશન

### નવી સાઇટ્સ માટે

සයිට් බිලඩර් මාදිලිය වෙනුවට භාවිතා කිරීම:

1. ඉල්ලීම: "මට අලුත් වෙබ් අඩවියක් සකස් කර ගැනීමට උදව් කරන්න"
2. සැකසුම් සහායක නියෝජිතයා ඔබව මෙහෙයවනු ඇත:
   - වෙබ් අඩවියේ අරමුණ සහ ඉලක්ක
   - ඉලක්කගත ප්‍රේක්ෂකයින්
   - වෙළඳ නාම අනන්‍යතාවය (Brand identity)
   - තේමා ජනනය කිරීම
   - ආරම්භක සැකසුම්

### දැනට පවතින අඩවි සඳහා

ඔබ සයිට් බිලඩර් මාදිලියේ තිබූ දැනට පවතින අඩවියක් තිබේ නම්:

1. ඔබට එය එලෙසම භාවිතා කළ හැකිය
2. තේමා යාවත්කාලීන කිරීමට, ඉල්ලීම කරන්න: "මගේ වෙබ් අඩවිය නැවත නිර්මාණය කරන්න"
3. සැකසුම් සහායක නියෝජිතයා නව තේමක් නිර්මාණය කිරීමට ඔබට උදවු කරයි
4. ඔබේ අඩවි දත්ත වෙනස් නොවේ

### තේමා වෙනස් කිරීම් සඳහා

සයිට් බිලඩර් මාදිලියේ තේමා තේරීම වෙනුවට:

1. ඉල්ලීම: "මගේ තේමා මාරු කරන්න"
2. සැකසුම් සහායක නියෝජිතයා කරනු ඇත:
   - ඔබේ නිර්මාණ කැමැත්ත ගැන විමසනු ඇත
   - අභිරුචි තේමක් ජනනය කරනු ඇත
   - එය ඔබේ වෙබ් අඩවිය මත සක්‍රීය කරනු ඇත

## ප්‍රධාන වෙනස්කම්

### සයිට් බිලඩර් මාදිලිය (Site Builder Mode)

```
1. සැකිල්ලක් තෝරන්න (Choose a template)
2. තේමක් තෝරන්න (Select a theme)
3. මූලික සැකසුම් වින්‍යාස කරන්න (Configure basic settings)
4. අවසන් (Done)
```

### සැකසුම් සහායක නියෝජිතයා (Setup Assistant Agent)

```
1. ඔබේ වෙබ් අඩවියේ අරමුණ විස්තර කරන්න (Describe your site's purpose)
2. ඔබේ ඉලක්කගත ප්‍රේක්ෂකයින් නිර්වචනය කරන්න (Define your target audience)
3. නිර්මාණ කැමැත්ත තෝරන්න (Choose design preferences)
4. නියෝජිතයා අභිරුචි තේමක් ජනනය කරයි (Agent generates custom theme)
5. නියෝජිතයා තේමාව සක්‍රීය කරයි (Agent activates theme)
6. අනාගත සැසි සඳහා වෙබ් කෙටි සටහන සුරැ둡니다 (Site brief stored for future sessions)
```

## සැකසුම් සහායක නියෝජිතයාගේ ප්‍රතිලාභ

### වඩාත් නම්‍යශීලී වීම (More Flexible)

- ඔබේ අඩවිය ස්වභාවික භාෂාවෙන් විස්තර කරන්න
- අභිරුචි නිර්දේශ ලබා ගන්න
- ඔබේ නිශ්චිත අවශ්‍යතා වලට අනුව හැඩගැසීමට හැකි වේ

### වඩා හොඳ අභිරුචය (Better Customization)

- අභිරුචි තේමාවක් ජනනය කිරීම
- නිර්මාණ පද්ධති තීරණ ගැනීම (Design system decisions)
- ස්ථිර නිර්මාණ ටෝකන් (Persistent design tokens)

### ස්ථිර මතකය (Persistent Memory)

- ඔබේ `site_brief` සුරැකේ
- අනාගත නියෝජිතයන්ට ඔබේ වෙබ් අඩවිය තේරුම් ගැනීමට හැකි වේ
- සැකසුම් තොරතුරු නැවත කිරීමට අවශ්‍ය නොවේ

### ඒකාබද්ධ වැඩ ප්‍රවාහය (Integrated Workflow)

- තේමා ගොඩනැගීම සඳහා ආරම්භ කිරීම (Theme Builder onboarding)
- නිර්මාණ පද්ධති අලංකාර කුසලතාව (Design System Aesthetics skill)
- දෘශ්‍යතා පාලනයන් ඇති හැකියාව (Ability Visibility controls)
- සියල්ල සුමටව එක්ව ක්‍රියා කරයි

## ගැටළු නිරාකරණය (Troubleshooting)

### මට සයිට් බිලඩර් මාදිලිය සොයාගත නොහැක නම්

සයිට් බිලඩර් මාදිලිය ඉවත් කර ඇත. ඒ වෙනුවට සැකසුම් සහායක නියෝජිතයා භාවිතා කරන්න:

"මට අලුත් වෙබ් අඩවියක් සකස් කරගන්න උදව් කරන්න"

### මම Site Builder එකකින් අඩවියක් නැවත නිර්මාණය කිරීමට කැමතියි

Setup Assistant agent (උපකාරක සහායක නියෝජිතයා) භාවිතා කර ඔබට එය නැවත නිර්මාණය කළ හැකිය:

1. ඉල්ලීම: "මට අලුත් වෙබ් අඩවියක් සකස් කරගන්න උදව් කරන්න"
2. ඔබේ මුල් වෙබ් අඩවියේ අරමුණ සහ මෝස්තරය විස්තර කරන්න
3. ඒ අනුව Agent එක සමාන තේමාවක් (theme) ජනනය කරයි
4. ඔබේ `site_brief` (අඩවි කෙටි සටහන) අනාගතයක් සඳහා ගබඩා කරනු ලැබේ

### මගේ පවතින Site Builder වෙබ් අඩවිය වැඩ කරන්නේ නැහැ

Site Builder මාදිලියේ නිර්මාණය කරන ලද පැරණි වෙබ් අඩවි තවමත් ක්‍රියාත්මක වේ. ඔබට ගැටළු ඇති වුවහොත්:

1. ඔබේ තේමාව (theme) ක්‍රියාත්මක (active) කර ඇත්දැයි පරීක්ෂා කරන්න
2. ඔබේ plugins සක්‍රිය (enabled) කර තිබේදැයි තහවුරු කරන්න
3. WordPress error logs පරීක්ෂා කරන්න
4. ගැටළු දිගටම පවතින්නේ නම් support එක අමතන්න

### මට මගේ පැරණි Site Builder template (ආකෘති) භාවිතා කළ හැකිද?

Site Builder template දැන් ලබා ගත නොහැක. කෙසේ වෙතත්:

- ඔබේ දැනට පවතින වෙබ් අඩවි තවමත් ක්‍රියාත්මක වේ
- Setup Assistant agent භාවිතයෙන් සමාන වෙබ් අඩවි නැවත නිර්මාණය කළ හැකිය
- Setup Assistant agent මගින් වැඩි tùy කිරීම් (customization) විකල්ප ලබා දෙයි
