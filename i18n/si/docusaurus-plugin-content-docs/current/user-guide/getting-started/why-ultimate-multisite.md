# Ultimate Multisite ඇයි

ඔබ මේ ලිපියට ආවේ නම්, අපේ plugin එක ඔබේ ව්‍යාපාරික අදහස සැබෑ කර ගැනීමට විසඳුමක් විය හැකි බව ඔබ අසා ඇති. මෙම ලිපියේ අවසානය වන විට ඔබට අවශ්‍ය පිළිතුරු ලැබෙනු ඇතැයි අපි බලාපොරොත්තු වෙමු.

## Ultimate Multisite යනු කුමක්ද? {#what-is-ultimate-multisite}

Ultimate Multisite යනු WordPress Multisite plugin එකක් වන අතර, එය ඔබට WaaS (Website as a Service) සේවා සැපයීමට ඉඩ සලසයි. අපගේ plugin එක සමඟ, ඔබට විවිධ දායකත්ව මට්ටම් තිබිය හැකි අතර, නව පරිශීලකයින් ඔබේ ජාලයට ලියාපදිංචි වූ විට ඔබේ ජීවිතය වඩාත් පහසු කිරීමට site templates (site සැකිලි) නිර්මාණය කළ හැකිය. එමෙන්ම, WooCommerce හෝ Stripe වැනි විවිධ gateways හරහා ගෙවීම් ලබා ගැනීමට, වෙබ් අඩවි මූලයන් (domains) සිතියම්ගත කිරීමට සහ පාරිභෝගිකයින් සඳහා කේන්ද්‍රීය අත්දැකීමක් නිර්මාණය කිරීම සඳහා ඔබේ ලියාපදිංචි පෝරම පුද්ගලීකරණය කිරීමට Ultimate Multisite ඔබට ඉඩ සලසයි.

මෙය ආරම්භය පමණි, මන්ද add-ons සහ webhooks සමඟ එහි ක්‍රියාකාරිත්වයන් පුළුල් කළ හැකිය.

## වෙනත් විශේෂාංග {#other-features}

Ultimate Multisite පහත දේ ද පිරිනමයි:

  * Managed WordPress hosting providers සමඟ Domain Mapping integration: නව domain එකක් සිතියම්ගත කරන විට, WP Engine, Cloudways, cPanel, RunCloud, Closte සහ තවත් බොහෝ ඒවායේ admin panels වලට එම mapped domains ස්වයංක්‍රීයව එකතු කරයි. එබැවින් ඔබට එය අතින් කිරීමට අවශ්‍ය නොවේ. සමහර අවස්ථාවලදී, SSL certificate fetching ඔබ වෙනුවෙන් ක්‍රියාත්මක කරනු ඇත.

  * Webhooks: Ultimate Multisite හි බලවත් Webhook system එකක් තිබේ, එය ඔබේ ජාලය විවිධ සේවාවන් සමඟ සම්බන්ධ කිරීමට ඔබට ඉඩ දෙයි. Zapier සමඟ සෘජු ඒකාබද්ධ කිරීම ද අපි ලබා දෙන්නෙමු, එමඟින් ඔවුන්ගේ වේදිකාවේ ඇති 1000+ යෙදුම් සහ සේවා හරහා ඔබේ ජාලය විවෘත කරයි.

* ඔබේ පාරිභෝගිකයින්ට ඔවුන්ගේ ප්‍රියතම පිටු ගොඩනැගීමේ මෙවලම් (page builder) භාවිතා කිරීමට උපකාර වන බලවත් Admin Pages සඳහා: Ultimate Multisite හි Admin Pages Creator නම් පහසු add-on එකක් තිබෙනවා. එය Elementor, Beaver Builder සහ Brizy template භාවිතයෙන් ඔබේ පාරිභෝගිකයින්ගේ dashboard වල සම්පූර්ණයෙන්ම කස්ටමයිස් කළ Admin Pages එක් කිරීමට ඔබට ඉඩ සලසනවා.

* ඔබේ පාරිභෝගිකයින්ගේ dashboard අත්දැකීම කස්ටමයිස් කිරීමට වෙනස් Admin Themes දෙකක්: Check Material WP සහ PRO.

ඊට වඩා දියුණු දේවල්:

* Template Overrides සඳහා සහාය: Ultimate Multisite සමඟ, ඔබ ප්‍රධාන plugin ගොනු වෙනස් කිරීම ගැන කරදර නොවී ඔබේම අනුවාදවලින් signup flow සඳහා පෙරනිමිය HTML template වෙනුවට ආදේශ කළ හැකියි.

* Search and Replace API: ඔබට {{site_name}} වැනි placeholders (තනතුරු) සහිත site templates සලකුණු කිරීමටත්, පාරිභෝගිකයා විසින් signup කිරීමෙන් පසු ලබා දෙන තොරතුරු එම placeholders ස්වයංක්‍රීයව පුරවා ගැනීමට Search and Replace API භාවිතා කළ හැකියි.

## භාවිත අවස්ථා (Use cases) {#use-cases}

Ultimate Multisite සඳහා ඇති විය හැකි භාවිත අවස්ථා කිහිපයක් දැන් බැලීමට කාලයයි.

### අවස්ථාව 1: an Agency (ඒජන්සියක්) {#case-1-an-agency}

සාමාන්‍යයෙන් ඒජන්සියක ප්‍රධාන කුසලතා වන්නේ ඔවුන්ගේ hosting හෝ marketing වැනි දේ අමතර සේවාවන් ලෙස ලැයිස්තුගත කිරීම වැනි website නිර්මාණය කිරීමයි.

ඒජන්සි සඳහා, Ultimate Multisite එක මගින් තනි වේදිකාවක් මත වෙබ් අඩවි කිහිපයක් host කිරීමට සහ කළමනාකරණය කිරීමට ඇති හැකියාව නිසා විශ්මයජනක වටිනාකමක් ඉදිරිපත් කරයි. GeneratePress, Astra, OceanWP හෝ වෙනත් විශේෂිත themes වල නිර්මාණ ප්‍රමිතිකරණය කරන ඒජන්සිවලට, නව site එකක් සඳහා මෙම themes ස්වයංක්‍රීයව සක්‍රිය කිරීමට Ultimate Multisite හි හැකියාවන් භාවිතා කළ හැකියි.

Ultimate Multisite හි සැකසුම් හරහා plugins සහ themes එක් එක් site එක සඳහා ස්ථාපනය කිරීමේදී ලබා ගත හැකි හෝ සැඟවිය හැකි ආකාරය සහ ඒවායේ ක්‍රියාත්මක වන තත්ත්වය පාලනය කිරීම තුළින්, නිෂ්පාදන අනුව (per-product basis) plugin සහ themes පිළිබඳ දැඩි පාලනයක් පවත්වා ගනී.

एजेंसीಗಳಿಗೆ Ultimate Multisite ഉപയോഗിച്ച് ਉਨ੍ਹਾਂ ਜੋ શ્રેષ્ઠ ਕਰਦੇ ਹਨ - અસાધારણ વેબસાઇટ્સ ડિઝાઇન કરવા માટે - સંપૂર્ણ શાંતિ મળશે.

### કેસ ૨: નિશ (Niche) પ્રદાતા {#case-2-niche-provider}

એક જૂની કહેવત છે કે, "એક જ વસ્તુ કરો અને તેને સારી રીતે કરો". ઘણા નિષ્ણાતો માટે, આનો અર્થ એક મુખ્ય વિચારની આસપાસ ઉત્પાદન અથવા સેવા બનાવવાનો થાય છે.

જો તમે સાઇટ્સનું એક વિશિષ્ટ નેટવર્ક વિકસાવવા પર ધ્યાન કેન્દ્રિત કરી રહ્યા હોવ, તો તમારી પાસે સામાન્ય ફ્રેમવર્ક અને પ્લેટફોર્મ પર આધારિત સેવાઓ પૂરી પાડવી જોઈએ. આ સમયે સાઇટ ટેમ્પલેટ્સ (site templates) મદદ કરી શકે છે!

Ultimate Multisite ની એક નવીન સુવિધા એ છે કે ટેમ્પલેટ સાઇટ એટલે એવી સાઇટ જ્યાં થીમ ઇન્સ્ટોલ અને સક્રિય કરવામાં આવી હોય, જરૂરી પ્લગિન્સ ઇન્સ્ટોલ અને સક્રિય કરવામાં આવ્યા હોય, અને નમૂનાના પોસ્ટ્સ અથવા પેજ બનાવવામાં આવ્યા હોય. જ્યારે કોઈ ગ્રાહક ટેમ્પલેટ પર આધારિત નવી સાઇટ બનાવે છે, ત્યારે ટેમ્પલેટની સામગ્રી અને સેટિંગ્સ નવી બનાવેલી સાઇટમાં કોપી થઈ જાય છે.

નિશ સાઇટ્સ અને સેવાઓ પ્રદાતા માટે, આ તમને કસ્ટમ પ્લગિન્સ અને ડિઝાઇન સાથે તરત જ તૈયાર સાઇટ બનાવવાની અજોડ છૂટ આપે છે. ગ્રાહકને સેવા પૂર્ણ કરવા માટે માત્ર ન્યૂનતમ ઇનપુટ આપવાની જરૂર હોય છે.

### કેસ ૩: WordPress વેબ હોસ્ટિંગ {#case-3-wordpress-web-hosting}

WordPress સાઇટ્સ હોસ્ટ કરવાની ઘણી રીતો છે, પરંતુ ગ્રાહકને પ્રી-ઇન્સ્ટોલ કરેલા વર્ઝન સાથે વેબ સ્પેસ પૂરી પાડવી એ હંમેશા આટલી સરળ નથી. કારણ કે એક અર્થપૂર્ણ સેવા આપવા માટે ઘણા નિર્ણયો અને વિચારણાઓ એકસાથે આવવાની જરૂર હોય છે.

Ultimate Multisite WordPress સાઇટ્સ હોસ્ટ કરવા માટે સંપૂર્ણ ટર્નકી (turnkey) સોલ્યુશન પ્રદાન કરીને આ ક્ષેત્રમાં શ્રેષ્ઠ પ્રદર્શન કરે છે. સોલ્યુશનમાં સબ્સ્ક્રિપ્શન સેવાઓ, ચુકવણી એકત્રીકરણ, ચેકઆઉટ ફોર્મ્સ, ડિસ્કાઉન્ટ વાઉચર્સ અને ગ્રાહક સંચાર આપવા માટેના મુખ્ય મિકેનિઝમ્સનો સમાવેશ થાય છે.

WordPress Multisite නිවැරදිව ස්ථාපනය කිරීමට, වින්‍යාස කිරීමට සහ නඩත්තු කිරීමට අවශ්‍ය බොහෝ ප්‍රධාන වැඩ සඳහා Ultimate Multisite උපකාරී වේ. එය ජාල පරිපාලකයින්ට ඔවුන්ගේ සේවාව හෝ විශේෂිත ක්ෂේත්‍රයන් (උදාහරණයක් ලෙස නිෂ්පාදන මට්ටම්, මිල ගණන් සහ සේවා දීමනා) හා සම්බන්ධ අංශ පමණක් සලකා බැලීමට ඉඩ සලසයි.

Ultimate Multisite සමඟ ඒකාබද්ධ කිරීමට කැමති සංවර්ධකයින් සඳහා, මෙම විසඳුම පුළුල් RESTful API එකක් සහ සිදුවීම් දැනුම් දීම සඳහා Webhooks ද ලබා දෙයි.

බොහෝ බාහිර plugin (ප්ලග්ඉන්) සහ බලපත්‍ර මත රඳා නොසිටින පරිදි, Ultimate Multisite Wix, Squarespace, [WordPress.com](https://WordPress.com) සහ වෙනත් ඒවාට සමාන විශේෂාංග බහුල විසඳුමක් සපයයි.
