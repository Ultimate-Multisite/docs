# පාඩම 8: පාරිභෝගික ආරම්භක මඟපෙන්වීම

පාරිභෝගිකයෙකු ලියාපදිංචි කරවා ගැනීම වැඩේ අඩක් පමණයි. ඔවුන් ලොග් ඉන් වී, අධික ලෙස සංකීර්ණ බවක් දැනී, නැවත නොඑන්නේ නම්, ඔබ ඔවුන් අහිමි කරගෙන ඇත. මෙම පාඩම නව ලියාපදිංචියක් සක්‍රිය, සම්බන්ධ වී සිටින පාරිභෝගිකයෙකු බවට පත් කරන අත්දැකීම සැලසුම් කරයි.

## අප නතර වූ තැන {#where-we-left-off}

FitSite දැන් ක්‍රියාත්මක checkout ක්‍රියාවලියක් සමඟ සම්පූර්ණයෙන්ම සන්නාමකරණය කර ඇත. දැන් අපි අවධානය යොමු කරන්නේ fitness studio හිමිකරුවෙකු ලියාපදිංචිය සම්පූර්ණ කර ඔවුන්ගේ නව අඩවියට පළමු වරට පැමිණි පසු සිදුවන දේ වෙතයි.

## ආරම්භක මඟපෙන්වීම වැදගත් ඇයි {#why-onboarding-matters}

ලියාපදිංචියෙන් පසු පළමු මිනිත්තු 30 පාරිභෝගිකයෙකු රැඳී සිටිනවාද නැතිනම් ඉවත් වෙනවාද යන්න තීරණය කරයි. Fitness studio හිමිකරුවෙකු:

- ලොග් ඉන් වී දැනටමත් fitness website එකක් වගේ පෙනෙන අඩවියක් දකී නම් → රැඳී සිටී
- ඊළඟට කළ යුතු දේ පැහැදිලිව දනී නම් → රැඳී සිටී
- සාමාන්‍ය WordPress dashboard එකක අතරමං වූ බවක් දැනේ නම් → ඉවත් වේ

ඔබේ niche templates (පාඩම 4) පළමු කරුණ හසුරුවයි. මෙම පාඩම දෙවැනි කරුණ හසුරුවයි.

Ultimate Multisite තුළ පාරිභෝගිකයන් ඔවුන්ගේ subscription කළමනාකරණය කරන customer-facing [Account පිටුව](/user-guide/client-management/account-page) ඇතුළත් වේ. මෙය customize කරන්නේ:

- ඔවුන්ගේ වත්මන් FitSite plan පෙන්වීමට
- fitness-specific ප්‍රතිලාභ සමඟ upgrade විකල්ප පෙන්වීමට
- billing history සහ invoice downloads ලබා දීමට
- help resources වෙත සබැඳීමට

## ආරම්භක මඟපෙන්වීමේ සාර්ථකත්වය මැනීම {#measuring-onboarding-success}

ඔබේ onboarding ක්‍රියා කරනවාද යන්න දැන ගැනීමට මෙම දර්ශක track කරන්න:

- **Activation rate**: ලියාපදිංචි වූවන්ගෙන් කී ප්‍රතිශතයක් පළමු සතිය තුළ ඔවුන්ගේ අඩවිය customize කරන්නේද?
- **පළමු සතියේ logins**: නව පාරිභෝගිකයෙකු ඔවුන්ගේ පළමු සතිය තුළ කී වරක් ලොග් ඉන් වේද?
- **නව පාරිභෝගිකයන්ගෙන් support tickets**: ඉහළ ප්‍රමාණයක් තිබීම ඔබේ onboarding හි හිස්තැන් තිබෙන බව පෙන්වයි
- **Trial-to-paid conversion**: ඔබ trials ලබා දෙන්නේ නම්, කී ප්‍රතිශතයක් convert වේද?

## මෙතෙක් FitSite Network {#the-fitsite-network-so-far}

```
FitSite ජාලය
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured + branded)
├── Platform Domain (fitsite.com + wildcard SSL)
├── අඩවි Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products (Starter, Growth, Pro + Order Bumps)
├── Checkout ක්‍රියාවලිය (niche-specific, tested)
├── සන්නාමකරණය (login, dashboard, emails, invoices, marketing site)
├── පාරිභෝගික ආරම්භක මඟපෙන්වීම
│   ├── Quick Start dashboard widget
│   ├── fitness site කළමනාකරණය සඳහා සරල කළ dashboard
│   ├── email 4ක පිළිගැනීමේ අනුක්‍රමය
│   ├── Niche-specific knowledge base
│   ├── Video walkthroughs
│   └── customize කළ account පිටුව
└── pricing strategy සඳහා සූදානම් (ඊළඟ පාඩම)
```

## මෙම පාඩමේදී අප සෑදූ දේ {#what-we-built-this-lesson}
