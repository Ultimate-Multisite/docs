# Action: wu_page_\{$this->id\}_register_widgets

මෙම පිටුව සඳහා widgets ලියාපදිංචි කිරීමෙන් පසු ක්‍රියාත්මක වේ.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## පරාමිතීන් {#parameters}

| නම | වර්ගය | විස්තරය |
|------|------|-------------|
| $id | `string` | පිටුවේ හැඳුනුම. |
| $page_hook | `string` | පිටුවේ hook එක. |
| $page | `object` | පිටුවේ object එක. |
