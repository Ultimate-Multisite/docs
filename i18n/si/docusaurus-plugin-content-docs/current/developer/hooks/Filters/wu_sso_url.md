# පෙරහන: wu_sso_url

හරස්-වසම් පාරිභෝගික ක්‍රියා සඳහා ආපසු ලබා දීමට පෙර සාදන ලද SSO URLs පෙරහන් කරයි.

ඒකාබද්ධ කිරීමකට ස්වාධීන-කුලීකරු SSO සබැඳියකට විශ්වාසදායක සන්දර්භයක් එක් කිරීමට හෝ Ultimate Multisite හි token වලංගු කිරීම රැකගෙන broker URL ප්‍රතිස්ථාපනය කිරීමට අවශ්‍ය විට මෙම පෙරහන භාවිත කරන්න.

## පරාමිතීන්

| නම | වර්ගය | විස්තරය |
|------|------|-------------|
| $sso_url | `string` | සාදන ලද SSO URL. |
| $user | `WP_User` | SSO පිවිසුම මඟින් සත්‍යාපනය කෙරෙන පරිශීලකයා. |
| $site_id | `int` | පිවිසුම සඳහා ඉලක්ක අඩවි ID. |
| $redirect_to | `string` | සාර්ථක SSO වලංගු කිරීමෙන් පසු ගමනාන්ත URL. |
