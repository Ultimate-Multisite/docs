# පෙරහන: wu_bypass_unset_current_user

unset current user කේතය මඟහැරීමට සංවර්ධකයන්ට ඉඩ දෙයි.

null නොවන ඕනෑම දෙයක් ආපසු දුන්නොත්, පිවිසී සිටින වත්මන් පරිශීලකයා unset කිරීම මඟහැරේ. මෙය සමහර අවස්ථාවල ප්‍රයෝජනවත් විය හැක. උදාහරණයක් ලෙස, පරිපාලන පැනල් ලෙස භාවිත වන උප-අඩවි සමඟ කටයුතු කරන විට.

## පරාමිතීන්

| $current_user | `false\|\WP_User` | වත්මන් පරිශීලක object එක. |
