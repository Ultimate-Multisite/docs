# පෙරහන: wu_use_domain_mapping {#filter-wuusedomainmapping}

මැපිං එකක් භාවිත කළ යුතුද යන්න තීරණය කරන්න

සාමාන්‍යයෙන්, භාවිත කිරීමට ඉඩ දිය යුත්තේ සක්‍රිය මැපිං පමණක් විය යුතුය. එහෙත්, ඔබට වඩා උසස් තර්කයක් භාවිත කිරීමට හෝ සක්‍රිය නොවන වසම්ද මැප් කිරීමට ඉඩ දීමට අවශ්‍ය නම්, මෙහිදී පෙරහන් කරන්න.
