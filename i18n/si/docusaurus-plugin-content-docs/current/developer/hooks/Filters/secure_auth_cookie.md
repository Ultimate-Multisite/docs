# පෙරහන: secure_auth_cookie {#filter-secureauthcookie}

සත්‍යාපන cookie එක HTTPS හරහා පමණක් යැවිය යුතුද යන්න පෙරහන් කරයි.

## පරාමිතීන් {#parameters}

| නම | වර්ගය | විස්තරය |
|------|------|-------------|
| $secure | `bool` | cookie එක HTTPS හරහා පමණක් යැවිය යුතුද යන්න. |
| $user_id | `int` | පරිශීලක හැඳුනුම් අංකය. |
