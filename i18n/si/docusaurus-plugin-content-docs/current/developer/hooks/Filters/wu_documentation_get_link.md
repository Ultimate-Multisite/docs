# පෙරහන: wu_documentation_get_link

plugin සංවර්ධකයන්ට සබැඳි පෙරීමට ඉඩ දෙන්න.

එය ප්‍රයෝජනවත් විය හැක්කේ කෙසේදැයි නිශ්චිත නැත, නමුත් එය තිබීම හානියක් නොවේ

## පරාමිතීන් {#parameters}

| නම | වර්ගය | විස්තරය |
|------|------|-------------|
| $link | `string` | ලියාපදිංචි කළ සබැඳිය |
| $slug | `string` | සබැඳිය ලබා ගැනීමට භාවිත කළ slug |
| $default_link | `string` | ලියාපදිංචි කළ පෙරනිමි සබැඳිය |
