# පෙරහන: wu_magic_link_enforce_ip

IP ලිපිනය සත්‍යාපනය ක්‍රියාත්මක කළ යුතුද යන්න පෙරහන්න.

වෙනස් ජාලවලින් tokens ක්‍රියා කිරීමට ඉඩ දීමට false ලෙස සකසන්න. මෙය ආරක්ෂාව අඩු කරයි, නමුත් භාවිත පහසුව වැඩි කරයි (උදා., ජාල මාරු කරන ජංගම පරිශීලකයන් සඳහා).

## පරාමිතීන් {#parameters}

| නම | වර්ගය | විස්තරය |
|------|------|-------------|
| $enforce | `bool` | IP ලිපිනය ගැලපීම ක්‍රියාත්මක කළ යුතුද යන්න. |
