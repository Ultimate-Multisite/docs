# Ultimate Multisite ලේඛන

Ultimate Multisite ලේඛන වෙත සාදරයෙන් පිළිගනිමු. Ultimate Multisite, WordPress Multisite පූර්ණ වෙබ් අඩවියක් සේවාවක් ලෙස (WaaS) වේදිකාවක් බවට පරිවර්තනය කරයි.

## ලේඛන කොටස් {#documentation-sections}

### [පරිශීලක මාර්ගෝපදේශය](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

ඔබේ WaaS වේදිකාව පිහිටුවීමට, වින්‍යාස කිරීමට, සහ කළමනාකරණය කිරීමට අවශ්‍ය සියල්ල. ස්ථාපනය, ගෙවීම් ද්වාර, වසම් සිතියම් කිරීම, සේවාදායක කළමනාකරණය, සහ තවත් දේ ආවරණය කරයි.

### [සංවර්ධක ලේඛන](/developer/hooks) {#developer-documentation}

Ultimate Multisite මත ගොඩනඟන සංවර්ධකයින් සඳහා තාක්ෂණික යොමුව. hooks යොමුව, REST API ලේඛන, සහ ඇඩෝන සංවර්ධන මාර්ගෝපදේශ ඇතුළත් වේ.

### [ඇඩෝන](/addons) {#addons}

WooCommerce ඒකාබද්ධ කිරීම, විශ්ලේෂණ, වසම් අලෙවිය, සහ තවත් දේ වැනි අමතර විශේෂාංග සමඟ Ultimate Multisite පුළුල් කරන ගෙවීම් ඇඩෝන ප්ලගින සඳහා ලේඛන.
