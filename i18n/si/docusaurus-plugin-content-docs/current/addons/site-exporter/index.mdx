## අඩවියක් අපනයනය කරන්නේ කෙසේද {#how-to-export-a-site}

## සම්පූර්ණ ජාලය අපනයනය කරන්නේ කෙසේද {#how-to-export-the-whole-network}

සියලුම උපඅඩවි එකට අපනයනය කිරීමට, Ultimate Multisite මෙනුවේ **අපනයනය සහ ආනයනය** වෙත ගොස්, අපනයන තිරයේ **ජාල අපනයනය** විකල්පය භාවිත කරන්න. ජනනය වූ archive එක host මාරු කිරීම්, staging නැවත ගොඩනැගීම්, හෝ ජාල මට්ටමේ උපස්ථ සඳහා සියලුම උපඅඩවි එක ZIP එකක අඩංගු කරයි.

ජාල අපනයන විශාල විය හැක. traffic අඩු වේලාවක ඒවා ධාවනය කරන්න, සහ archive එකටත් ඇතුළත් කළ උඩුගත කිරීම්, ප්ලගින, සහ තේමා වලටත් server තුළ ප්‍රමාණවත් ගබඩා ඉඩ ඇති බව තහවුරු කරන්න.

## තනි ස්ථාපනයක අඩවියක් ආනයනය කරන්නේ කෙසේද {#how-to-import-a-site-on-a-single-install}
