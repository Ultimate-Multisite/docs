# සහාය නියෝජිතයන්

සම්පූර්ණ ජාලයක් ඔබම කළමනාකරණය කිරීම පහසු කාර්යයක් නොවේ. ඔබේ ජාලය වර්ධනය වීමට පටන් ගත් විට, සහාය සඳහා ඇති ඉල්ලුම වැඩි වන බව ඔබට පෙනෙනු ඇත. ඔබේ වැඩ බර අඩු කර ගැනීමට උපකාරී වන පුද්ගලයන් එක් කර ගැනීම හොඳ අදහසකි. කෙසේ වෙතත්, සම්පූර්ණ ප්‍රවේශය/අවසරය ලබා දීම අවදානම් වන අතර ඔබේ ජාලය අනාරක්ෂිත කළ හැක. මෙම Support Agent add-on පැමිණෙන්නේ එතැනටය.

මෙම add-on ඔබට Support Agent ලෙස හැඳින්වෙන නව WordPress පරිශීලක වර්ගයක් සෑදීමට ඉඩ දෙයි, සහ එම පරිශීලකයාට ඇති අවසර මොනවාද යන්න පිළිබඳ සම්පූර්ණ පාලනය ඔබට ලබා දෙයි. අවසර ලබා දීම toggle බොත්තමක් සක්‍රිය කිරීම තරම් පහසු වන අතර, එය ඉවත් කිරීමට ඔබ තීරණය කළහොත් එය අක්‍රිය කිරීමත් එසේම පහසුය.

## විශේෂාංග

- WordPress සහ Ultimate Multisite අවසරවල අයිතමගත ලැයිස්තුව
- පහසුවෙන් ප්‍රවේශය සක්‍රිය හා අක්‍රිය කරන්න
- අසීමිත නියෝජිත Account සංඛ්‍යාවක් සාදන්න

## ස්ථාපනය

1. 'ultimate-multisite-support-agents' '/wp-content/plugins/' නාමාවලියට උඩුගත කරන්න
2. WordPress හි 'ප්ලගින' මෙනුව හරහා plugin සක්‍රිය කරන්න
3. plugin සැකසීමට පියවරෙන් පියවර Wizard අනුගමනය කරන්න
