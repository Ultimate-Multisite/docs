# අන්තර්ගත සමමුහුර්තකරණය {#content-sync}

Multisite Ultimate: අන්තර්ගත සමමුහුර්තකරණය

Multisite Ultimate සඳහා මෙම ඇඩෝනය, ජාල පරිපාලකයන්ට ඕනෑම අඩවියකින් ජාලයේ වෙනත් අඩවි වෙත අන්තර්ගතය සමමුහුර්ත කිරීමට ඉඩ දෙයි. එය ලිපි, පිටු, අභිරුචි ලිපි වර්ග, සහ Elementor ටෙම්ප්ලේට් ඇතුළු සියලුම ලිපි වර්ග සඳහා සහය දක්වයි. අන්තර්ගතය සහ ඉලක්ක අඩවි තෝරා ගැනීමට බහු-පියවර විශාර්ඩ් අතුරුමුහුණතක්, කාණ්ඩ සැකසීම සහ සජීවී ප්‍රගති ප්‍රතිපෝෂණය මෙම ඇඩෝනයට ඇතුළත් වේ.

## විශේෂාංග {#features}

* අන්තර්ගත තේරීම: සිඟිති රූ සහ ලිපි වර්ග පෙරහන සහිත දෘශ්‍ය කාඩ් ජාලකය
* සියලුම ලිපි වර්ග: ලිපි, පිටු, නිෂ්පාදන, Elementor ටෙම්ප්ලේට්, සහ ඕනෑම අභිරුචි ලිපි වර්ගයක් සමමුහුර්ත කරයි
* අඩවි තේරීම: මූලාශ්‍ර අඩවිය බැහැර කිරීම සහ ළමා අඩවි පෙර-තේරීම සමඟ අන්තර්ගතය සමමුහුර්ත කළ යුතු අඩවි තෝරන්න
* කාණ්ඩ සැකසීම: ප්‍රගති යාවත්කාලීන සමඟ එකවර අඩවි 10කට සමමුහුර්ත කරයි
* මාධ්‍ය සමමුහුර්තකරණය: ලිපි අන්තර්ගතයෙන් මාධ්‍ය ගොනු (රූප, වීඩියෝ) ඉලක්ක අඩවි වෙත ස්වයංක්‍රීයව පිටපත් කර නැවත සිතියම්ගත කරයි
* වර්ගීකරණ සමමුහුර්තකරණය: ප්‍රවර්ග, ටැග්, සහ අභිරුචි වර්ගීකරණ ආරක්ෂා කරයි
* Elementor සහාය: CSS නැවත ජනනය සහ ටෙම්ප්ලේට් හැඳුනුම් නැවත සිතියම්ගත කිරීම සමඟ Elementor ටෙම්ප්ලේට් සඳහා සම්පූර්ණ සහාය
* ගැටුම් විසඳීම: පවතින අන්තර්ගතය ප්‍රතිස්ථාපනය කිරීමට හෝ එය මඟ හැරීමට විකල්පය
* කෑෂ් කළමනාකරණය: සමමුහුර්ත කිරීමෙන් පසු ඉලක්ක අඩවිවල Elementor කෑෂ් ස්වයංක්‍රීයව හිස් කරයි

## අවශ්‍යතා {#requirements}

* WordPress 5.3 හෝ ඉහළ
* PHP 7.4 හෝ ඉහළ
* Multisite Ultimate ප්ලගිනය (සක්‍රීය)
* Elementor ප්ලගිනය (විකල්ප, Elementor ටෙම්ප්ලේට් සමමුහුර්තකරණය සඳහා අවශ්‍යයි)

## ස්ථාපනය {#installation}

1. 'ultimate-multisite-content-sync' '/wp-content/plugins/' නාමාවලියට උඩුගත කරන්න
2. ඔබේ WordPress ජාල පරිපාලනයේ 'ප්ලගින' මෙනුව හරහා ප්ලගිනය සක්‍රීය කරන්න
3. Ultimate Multisite → සැකසුම් → අන්තර්ගත සමමුහුර්තකරණය තුළ සැකසුම් වින්‍යාස කරන්න
4. සමමුහුර්ත කිරීම ආරම්භ කිරීමට Ultimate Multisite → අන්තර්ගත සමමුහුර්තකරණය වෙත යන්න
