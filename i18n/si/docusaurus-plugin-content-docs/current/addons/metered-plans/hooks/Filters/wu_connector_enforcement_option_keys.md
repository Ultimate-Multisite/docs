# පෙරහන: wu_connector_enforcement_option_keys

ප්‍රධාන අඩවියෙන් බලාත්මක කරන AI සැපයුම්කරු option key ලැයිස්තුව පෙරහන් කරන්න.

අභිරුචි හෝ තෙවන පාර්ශ්ව AI සැපයුම්කරු දිගු සඳහා option key එක් කරන්න. එවිට ඒවායේ සැකසුම්ද උපඅඩවිවල ප්‍රධාන අඩවියෙන් උරුම වේ.

## පරාමිතීන්

| නම | වර්ගය | විස්තරය |

| $keys | `string[]` | බලාත්මක කළ යුතු option key නම් (දැනටමත් ගතිකව සොයාගත් connector key සහ EXTRA_PROVIDER_OPTIONS ඇතුළත් වේ). |
