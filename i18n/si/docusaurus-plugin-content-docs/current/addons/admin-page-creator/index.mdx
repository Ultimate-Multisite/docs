### මෙනු වර්ග

ඔබේ නව අභිරුචි පරිපාලක පිටුවට වෙනස් මෙනු වර්ග දෙකක් තිබිය හැක: ඉහළ මට්ටමේ මෙනුව සහ උපමෙනුව.

නමෙන්ම කියවෙන පරිදි, ඉහළ මට්ටමේ මෙනු පිටුවක් ඔබ නවයෙන් සාදන ලද පිටුව පෙරනිමි WordPress පිටු සමඟ පරිපාලක මෙනුවට එක් කරයි.

ඔබ ඉහළ මට්ටමේ විකල්පය තෝරන්නේ නම්, මෙනු අනුපිළිවෙළ සහ මෙනු නිරූපකයක්ද තෝරා ගත හැක.

මෙම පිටුව උපමෙනු පිටුවක් කිරීමට ඔබට අවශ්‍ය නම්, මෙම පිටුව සම්බන්ධ කිරීමට මව් පිටුවක් තෝරා ගත හැක. ඔබේ අභිරුචි පිටුව පෙරනිමි WordPress පිටු වෙත හෝ ඔබේ ඉහළ මට්ටමේ අභිරුචි පරිපාලක පිටු වෙත එක් කළ හැක.

_මෙම අභිරුචි පිටුව සඳහා මෙනු අයිතමය සම්පූර්ණයෙන් අභිරුචිකරණය කරන්න_

_ඔබේම අභිරුචි ඉහළ මට්ටමේ පිටු ඇතුළුව, වෙනත් මෙනු පිටු වල උපමෙනුවක් ලෙසද මෙම පිටුව එක් කළ හැක._

## උසස් විකල්ප

ඔබේ අභිරුචි පරිපාලක පිටු වෙත සියලු වර්ගවල අන්තර්ගත එක් කිරීමට ඔබට හැකි විය යුතුය. එයට CSS නීති සහ ගොනු, JavaScript, සහ බාහිර පුස්තකාලද ඇතුළත් වේ. උසස් විකල්ප ටැබය භාවිතයෙන් එය කළ හැක.

### අභිරුචි CSS

CSS ටැබය තුළ, ඔබේම අභිරුචි CSS කේතය එක් කළ හැකි අතර, දුරස්ථ අඩවි වලින් CSS ගොනු ආනයනය කළ හැක (CDN, Google Fonts, ආදිය වැනි).

<!-- තිර රූපය නොමැත: බාහිර stylesheet ආනයනය සහිත පරිපාලක පිටු නිර්මාපකයේ අභිරුචි CSS සංස්කාරකය -->

_අභිරුචි CSS සඳහාද සහාය ඇත. බාහිර style ගොනුද ඇතුළත් කළ හැක!_

### අභිරුචි JavaScript

JavaScript සඳහාද ඒ දේම වලංගු වේ. ඔබේ අභිරුචි JavaScript කේතය තුළ භාවිතා කිරීමට බාහිර පුස්තකාල එක් කළ හැක.

<!-- තිර රූපය නොමැත: බාහිර පුස්තකාල ආනයනය සහිත පරිපාලක පිටු නිර්මාපකයේ අභිරුචි JavaScript සංස්කාරකය -->

_ඔබේම JavaScript කේතය එක් කරන්න!_

## අවසර

උසස් විකල්ප තුළම, මෙම පිටුවට ප්‍රවේශය ඇති උප-අඩවි තෝරා ගත හැක. උදාහරණයක් ලෙස, වෙනස් plan මට්ටම් සඳහා වෙනස් සුවිශේෂී අන්තර්ගත සැපයීමට ඔබ සැලසුම් කරන්නේ නම් මෙය ප්‍රයෝජනවත් වේ.

<!-- තිර රූපය නොමැත: plan සහ භූමිකාව මත පදනම් වූ ප්‍රවේශ පාලනය සඳහා පරිපාලක පිටු නිර්මාපකයේ අවසර සැකසුම් -->

_මෙම පිටුව කොතැන පෙන්විය යුතුද යන්න පිළිබඳ සම්පූර්ණ පාලනය ඔබට ඇත_

**වැදගත්** : පිටුව පෙන්වීමට කොන්දේසි දෙකම සපුරාලිය යුතුය. එනම්, පරිශීලකයා දී ඇති සැලැස්මක සේවාදායකයෙකු වුවත්, තෝරාගත් භූමිකාවලින් එකක් නොමැති නම් (හෝ එහි විරුද්ධව) පිටුව ඔවුන්ගේ පැනලයේ නොපෙන්වයි.

## සක්‍රීය

අභිරුචි පරිපාලක පිටුවක් මකා නොදමා හෝ අවසර වෙනස් නොකර අබල කිරීමට අවශ්‍ය නම්, සක්‍රීය විකල්පයේ සලකුණ ඉවත් කරන්න.

ඉහළ මට්ටමේ පිටුවක් අක්‍රීය කිරීමෙන්, එම මව් පිටුවට සම්බන්ධ සියලු උප-පිටු ස්වයංක්‍රීයව අක්‍රීය වේ.
