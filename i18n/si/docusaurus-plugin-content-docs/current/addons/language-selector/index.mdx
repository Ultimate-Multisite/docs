# භාෂා තේරීම

Multisite Ultimate: ලියාපදිංචි වීමේදී භාෂා තේරීම

ලියාපදිංචි වීමේ ප්‍රවාහයට භාෂා තේරීමේ ක්ෂේත්‍රයක් එක් කරන්න.

## ස්ථාපනය {#installation}

1. 'ultimate-multisite-language-selector' '/wp-content/plugins/' නාමාවලියට උඩුගත කරන්න
2. ඔබගේ WordPress ජාල පරිපාලකයේ 'ප්ලගින' මෙනුව හරහා plugin එක සක්‍රිය කරන්න

## භාවිත මාර්ගෝපදේශය {#usage-guide}

මෙම නොමිලේ add-on එකෙන්, ඔබේ පරිශීලකයන්ට ලියාපදිංචි වීමේ ප්‍රවාහයේ සහ පිවිසුම් පිටුවේ දැකීමට කැමති භාෂාව තෝරාගැනීමට ඉඩ දිය හැක.

[ඔබගේ බලපත්‍රය සක්‍රිය කිරීමෙන්](../../user-guide/getting-started/installing-ultimate-multisite.md) පසු, ඔබගේ බලපත්‍රය ඔබට ප්‍රවේශය ලබාදෙන Ultimate Multisite add-ons ස්ථාපනය කළ හැක.

ඔබගේ Ultimate Multisite සැකසුම් පිටුවට ගොස්, වම්පස ඇති මෙනුවේ _Add-ons_ ලේබලය ක්ලික් කරන්න.

![Add-ons පිටුව](/img/admin/addons-page.png)

_Language Selector ස්ථාපනය ආරම්භ කිරීමට Add-ons පිටුවට යන්න_

එහිදී, අපි ලබාදෙන සියලු add-ons ඔබට හමුවේ.

නොමිලේ add-ons, ඔවුන් ලියාපදිංචි වූ subscription plan එක කුමක් වුවත්, අපගේ සියලු පාරිභෝගිකයන්ට ලබාගත හැක.

add-on එකේ thumbnail එකට පහළින් ඇති _දැන්ම ස්ථාපනය කරන්න!_ බොත්තම ක්ලික් කරන්න. ඉන්පසු Plugins පිටුවේ එය සක්‍රිය කරන්න.

![ස්ථාපන බොත්තම සහිත Add-ons පිටුව](/img/admin/addons-page.png)

_add-on එක ස්ථාපනය කිරීමට බොත්තම ක්ලික් කරන්න_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_ඔබගේ Plugins පිටුවේ එය සක්‍රිය කරන්න_

දැන්, Ultimate Multisite හි සැකසුම් මෙනුවට ගොස් _ජාල සැකසුම්_ ටැබයට ඇතුළු වන්න. එහි, පිටුවේ පහළින්, _භාෂා_ නමින් නව අංශයක් ඔබට හමුවේ.

![සැකසුම් පිටුව](/img/config/settings-general.png)

_අලුත්ම අංශයක් දිස්වේ!_

සියලු භාෂා විකල්ප ලෙස පෙන්වීමට එය හිස්ව තැබිය හැක, නැතහොත් ලබාගත හැකි ලෙස කිරීමට ඔබ කැමති ඒවා තෝරාගත හැක. භාෂා වන්නේ ඔබගේ WordPress සැකසුමේ ස්ථාපනය කර ඇති ඒවාය.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_මෙම WordPress ස්ථාපනයේ ලබාගත හැකි භාෂා ලැයිස්තුව_

ඔබ කළ වෙනස්කම් සුරැකීමෙන් පසු, ඔබ තෝරාගත් භාෂා සහිත තේරීම් මෙනුවක් පිවිසුම් පිටුවේ මෙන්ම ලියාපදිංචි වීමේ ප්‍රවාහයේද පෙන්වනු ඇත.

![පිවිසුම් සහ ලියාපදිංචි සැකසුම්](/img/config/settings-login-registration.png)

_පිවිසුම් පිටුවේ භාෂා තේරීම_

![ලියාපදිංචි පිටුව](/img/config/registration-page.png)

_ලියාපදිංචි වීමේ ප්‍රවාහයේ භාෂා තේරීම_

## URL එක හරහා භාෂාව සැකසීම {#setting-the-language-via-the-url}

URL එකට locale පරාමිතිය යොමු කිරීමෙන් පිටුවක භාෂාවත් වෙනස් කළ හැක. උදාහරණයක් ලෙස, ඉතාලි භාෂාවෙන් ලියාපදිංචි පිටුවකට යොමු කිරීමට අවශ්‍ය නම්, URL එකට ?locale=it_IT එක් කරන්න.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

මෙය ක්‍රියා කිරීමට, එම භාෂාව ඔබගේ WordPress ජාලයේ ස්ථාපනය කර තිබිය යුතුය.
