# AI Site Builder {#ai-site-builder}

ultimate-multisite-ai-site-builder

## අවශ්‍යතා {#requirements}

- WordPress 5.3 හෝ ඊට වැඩි
- PHP 7.4 හෝ ඊට වැඩි
- Multisite Ultimate ප්ලගිනය (සක්‍රිය)

## ස්ථාපනය {#installation}

1. ඇඩෝන ගොනු ඔබගේ `/wp-content/plugins/` නාමාවලියට උඩුගත කරන්න
2. WordPress තුළ 'Plugins' මෙනුව හරහා ප්ලගිනය සක්‍රිය කරන්න
3. ජාල පරිපාලකය තුළ ඇඩෝන සැකසුම් වින්‍යාස කරන්න

## සංවර්ධනය {#development}

### සැකසීම {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### ව්‍යාපෘති ව්‍යුහය {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP classes and includes
│   ├── class-multisite-ultimate-updater.php  # Update handler
│   └── class-ai_site_builder-main.php             # Main functionality
├── tests/                        # Unit tests
│   ├── bootstrap.php            # Test bootstrap
│   ├── class-*-base.php         # Base test class
│   └── class-*-test.php         # Test cases
├── views/                       # Template files
├── assets/                      # Static assets
│   ├── css/                     # Stylesheets
│   ├── js/                      # JavaScript files
│   └── images/                  # Images
├── lang/                        # Language files
├── composer.json                # PHP dependencies
├── package.json                 # Node.js dependencies
├── phpunit.xml.dist             # PHPUnit configuration
├── .phpcs.xml.dist              # Code standards configuration
└── rector.php                   # Code modernization rules
```

### පරීක්ෂා කිරීම {#testing}

ඇඩෝනයට සම්පූර්ණ පරීක්ෂණ රාමුවක් ඇතුළත් වේ:

- **ඒකක පරීක්ෂණ**: WordPress පරීක්ෂණ කට්ටලය සමඟ PHPUnit මත පදනම් වූ පරීක්ෂණ
- **මූලික පරීක්ෂණ පන්තිය**: පොදු පරීක්ෂණ අවස්ථා සඳහා උපකාරක ක්‍රම සපයයි
- **පරීක්ෂණ ආවරණය**: කේත ආවරණය අනුගමනය කිරීමට වින්‍යාස කර ඇත
- **Multisite පරීක්ෂා කිරීම**: පරීක්ෂණ multisite පරිසරය තුළ ධාවනය වේ

### කේත ප්‍රමිතීන් {#code-standards}

- **WordPress කේත ප්‍රමිතීන්**: PHPCS හරහා බලාත්මක කරයි
- **PHP 7.4+ අනුකූලතාව**: නවීන PHP විශේෂාංග සඳහා සහාය දක්වයි
- **ස්ථිතික විශ්ලේෂණය**: වර්ග පරීක්ෂා කිරීම සඳහා PHPStan ඒකාබද්ධ කිරීම
- **කේත නවීකරණය**: PHP උත්ශ්‍රේණි සඳහා Rector නීති

### ගොඩනැගීමේ පද්ධතිය {#build-system}

- **සම්පත් සැකසීම**: CSS/JS ගොනු සඳහා කුඩා කිරීම
- **පරිවර්තනය**: POT ගොනු උත්පාදනය
- **ඇසුරුම් කිරීම**: බෙදාහැරීම සඳහා ලේඛනාගාරයක් සෑදීම
- **සංවර්ධන මාදිලිය**: දෝෂ සොයාගැනීම සඳහා කුඩා නොකළ සම්පත්

## වින්‍යාසය {#configuration}

ඇඩෝනය ජාල පරිපාලක අතුරුමුහුණත හරහා ප්‍රවේශ විය හැකි විවිධ වින්‍යාස විකල්ප සඳහා සහාය දක්වයි.

## Hooks සහ Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - ඇඩෝනය ආරම්භ කිරීමෙන් පසු ක්‍රියාත්මක වේ
- `ultimate-multisite-ai-site-builder_loaded` - සියලුම ඇඩෝන ගොනු පූරණය වූ පසු ක්‍රියාත්මක වේ

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - ඇඩෝන සැකසුම් වෙනස් කරන්න
- `ultimate-multisite-ai-site-builder_enabled` - ඇඩෝනය සක්‍රිය/අක්‍රිය තත්ත්වය අතික්‍රමණය කරන්න
