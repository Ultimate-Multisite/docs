# Multi-Network වෙනස්කම් සටහන

### 1.3.0
* නව: ජාල අච්චු පෙරදසුන්කරු — මිලදී ගැනීමට පෙර සජීවී පැනලයක් සමඟ ජාල අච්චු පිරික්සා පෙරදසුන් බලන්න.
* නව: checkout හි ජාල අච්චු තෝරකය — ලියාපදිංචි checkout ප්‍රවාහය තුළ පාරිභෝගිකයින්ට ජාල අච්චුවක් තෝරාගත හැක.
* නව: ජාල-අතර magic link SSO — පාරිභෝගිකයින්ට magic-link සත්‍යාපනය හරහා උප-ජාල වෙත බාධාවකින් තොරව ප්‍රවේශ විය හැක.
* නව: ජාල මාධ්‍ය පිටපත් කිරීම — අච්චුවකින් ජාල පිටපත් කරන විට site මාධ්‍ය නිවැරදිව පිටපත් වේ.
* නිවැරදි කිරීම: multi-tenancy addon නොමැතිව දැන් ජාල පිටපත් කිරීම ක්‍රියා කරයි; පිටපත් කිරීමෙන් පසු super admin වරප්‍රසාද නැවත ලබා දේ.
* නිවැරදි කිරීම: ජාල නිර්මාණය දැන් WooCommerce සහ සියලු ගෙවීම් දොරටු සමඟ අනුකූල වේ.
* නිවැරදි කිරීම: fallback domain නිර්මාණයට දැන් port number ඇතුළත් වේ.
* නිවැරදි කිරීම: සියලු themes මත නිවැරදිව පෙන්වීම සඳහා template picker CSS enqueued stylesheet එකකට ගෙන ගොස් ඇත.
* නිවැරදි කිරීම: site විමසුම් scope තුළ අර්ථ දක්වා නොමැති `network_id` key එකට එරෙහිව ආරක්ෂා කරයි.
* නිවැරදි කිරීම: context switching පෙර ජාල අවසර පරීක්ෂාව `wpdb` table property දෝෂ වළක්වයි.
