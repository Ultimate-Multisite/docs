# Gravity Forms

**Gravity Forms බලයෙන් ඔබේ පරිශීලක ලියාපදිංචි අත්දැකීම පරිවර්තනය කරන්න**

ලෝකයේ වඩාත්ම උසස් form builder එක Multisite Ultimate සමඟ බාධාවකින් තොරව ඒකාබද්ධ කර, ඔබේ membership ලියාපදිංචි වීම වැඩි කරන සහ ආදායම ඉහළ නංවන ආකර්ෂණීය, ඉහළ පරිවර්තන ලියාපදිංචි forms සාදන්න.

## 🚦 එය ක්‍රියා කරන ආකාරය {#-how-it-works}

### 1. **ඔබේ Form එක සැලසුම් කරන්න** {#1-design-your-form}
පරිපූර්ණ ලියාපදිංචි අත්දැකීම සාදීමට Gravity Forms' drag-and-drop builder භාවිතා කරන්න.

### 2. **ඔබේ Fields Map කරන්න** {#2-map-your-fields}
අපගේ දෘශ්‍ය mapping interface එකෙන් form fields Multisite Ultimate data points වෙත සම්බන්ධ කරන්න.

### 3. **ගෙවීම් වින්‍යාස කරන්න** {#3-configure-payment}

### 4. **සජීවී කරන්න** {#4-go-live}
ඔබේ form එක ප්‍රකාශයට පත් කර අමුත්තන් ගෙවන පාරිභෝගිකයන් බවට පරිවර්තනය කිරීම ආරම්භ කරන්න!
