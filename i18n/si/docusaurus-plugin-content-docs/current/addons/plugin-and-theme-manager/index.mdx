# ප්ලගිනය සහ තේමාව කළමනාකරු

Multisite Ultimate: ප්ලගිනය සහ තේමාව කළමනාකරු

ස්ථාපනය කර ඇති ප්ලගින සහ තේමා වල මෙටා තොරතුරු (මාතෘකාව, කතුවරයා, කුඩා රූපය සහ විස්තරය) සංස්කරණය කරන්න හෝ සඟවන්න. ඔබේ පරිශීලකයන්ට ඒවා පෙරීමට හැකි වන පරිදි ප්ලගින සහ තේමා සඳහා ප්‍රවර්ග සාදන්න, සහ ඔබේ පරිශීලක පුවරුවට අලංකාර අභිරුචි ප්ලගින පිටුවක් එක් කරන්න.

ප්ලගිනය භාවිතා කිරීමට, ඔබේ network admin හි ඔබේ ප්ලගින හෝ තේමා පිටුවලට ගොස් "ප්ලගිනය/තේමා තොරතුරු සංස්කරණය කරන්න" මත ක්ලික් කළ හැකිය.

## ස්ථාපනය {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' '/wp-content/plugins/' නාමාවලියට උඩුගත කරන්න
2. ඔබේ WordPress Network Admin හි 'ප්ලගින' මෙනුව හරහා ප්ලගිනය සක්‍රිය කරන්න

## භාවිත මාර්ගෝපදේශය {#usage-guide}

Ultimate Multisite: ප්ලගිනය සහ තේමාව කළමනාකරු යනු ඔබේ ජාල පරිශීලකයන් ඔබේ ජාලයේ ප්ලගිනය සහ තේමා විකල්ප දකින සහ ඒවා සමඟ අන්තර්ක්‍රියා කරන ආකාරය පිළිබඳ සම්පූර්ණ පාලනය ලබාදෙන බලවත් **ගෙවීම් සහිත** Ultimate Multisite add-on එකකි.

මෙම add-on එක අරමුණු තුනක් ඉටු කරයි:

  1. එය ඔබට ප්ලගින සහ තේමා දෙවර්ගයේම මෙටාදත්ත සම්පූර්ණයෙන්ම වෙනස් කිරීමට ඉඩ දෙයි. ඒයට නම, විස්තරය, කතුවර නම, සහ කුඩා රූප පවා ඇතුළත් වේ!

  2. එය ඔබට ප්ලගින සහ තේමා ප්‍රවර්ග කිරීමට ඉඩ දෙයි, එවිට ඔබේ පරිශීලකයන්ට ඒවා පහසුවෙන් වර්ග කරගත හැකිය.

  3. එය පැරණි, කම්මැලි ප්ලගින පිටුව වෙනුවට ප්‍රවර්ග අනුව වර්ග කිරීම සහිත නව තේමා-වැනි පිටුවක් යෙදවීමට ඔබට විකල්පය ලබා දෙයි.

එය බාගත කර ස්ථාපනය කළ පසු, Ultimate Multisite Settings පිටුවේ නව ටැබ් එකක් - _Add-on සැකසුම්_ \- පෙනෙනු ඇත.

_මෙහි ඔබට Ultimate Multisite: ප්ලගිනය සහ තේමාව කළමනාකරු විසින් එක් කළ ගෝලීය විකල්ප සොයාගත හැකිය_

ඔබේ ජාල _ප්ලගින_ පිටුවේ දැන් ඔබට නම, කතුවරයා, සහ ප්ලගිනයේ කුඩා රූපය පවා වැනි සියලුම ප්ලගිනවල මෙටා තොරතුරු සංස්කරණය කළ හැකිය.

ප්ලගිනය සහ තේමාව කළමනාකරු ප්‍රදර්ශන වර්ග දෙකක් ලබා දෙයි: ප්ලගින රටාව සහ තේමා රටාව. එබැවින්, අවසානයේ, ඔබට අවශ්‍ය සකස් කිරීම් කළ පසු, ඔබේ පාරිභෝගිකයන්ට ඔවුන්ගේ _ප්ලගින_ පිටුව මෙසේ පෙනෙනු ඇත:
