# Material WP {#material-wp}

WordPress පරිපාලක අතුරුමුහුණත දිගු කාලයක සිට එකම ආකාරයෙන් තිබුණි. නමුත් පරිපාලක අතුරුමුහුණතේ Material Design අනුවාදයක් කෙසේ පෙනෙනු ඇත්දැයි ඔබ කවදා හෝ සිතා තිබේද? 

Material WP යනු ඔබට එම පෙනුම සහ හැඟීම ලබා දීමට සංවර්ධනය කළ පරිපාලක තේමාවකි. එය ඔබේ සේවාලාභීන්ට වඩාත් අභිරුචිකරණය කළ Dashboard එකක් ලබා දීමට උදව් කරයි.

## විශේෂාංග {#features}

- අභිරුචි CSS සහාය
- පහසු ශෛලි සැකසුම්
- මෙනු නැවත අනුපිළිවෙළ කිරීම
- පහසු ආනයනය/අපනයනය

## ලේඛන {#documentation}

**Material WP සමඟ ඔබේ පරිපාලක පැනලය අභිරුචිකරණය කිරීම**

## ස්ථාපනය {#installation}

1. 'material-wp' '/wp-content/plugins/' නාමාවලියට උඩුගත කරන්න
2. WordPress තුළ 'ප්ලගින' මෙනුව හරහා plugin එක සක්‍රීය කරන්න

## භාවිත මාර්ගෝපදේශය {#usage-guide}

WordPress පරිපාලක අතුරුමුහුණත දිගු කාලයක සිට එකම ආකාරයෙන් තිබුණි. නමුත් පරිපාලක අතුරුමුහුණතේ Material Design අනුවාදයක් කෙසේ පෙනෙනු ඇත්දැයි ඔබ කවදා හෝ සිතා තිබේද?

Material WP යනු ඔබට එම පෙනුම සහ හැඟීම ලබා දීමට සංවර්ධනය කළ පරිපාලක තේමාවකි. එය ඔබේ සේවාලාභීන්ට වඩාත් අභිරුචිකරණය කළ Dashboard එකක් ලබා දීමට උදව් කරයි. Material WP යනු Ultimate Multisite හි ප්‍රිමියම් plugins core හි කොටසක් ද වේ.

**එය ස්ථාපනය කිරීමෙන්** පසු, ඔබේ _ප්ලගින_ පිටුවට ගොස් ඔබේ Network මත එය සක්‍රීය කිරීමට ක්ලික් කරන්න.

_ඔබේ ප්ලගින පිටුවේ එය ජාලය පුරා සක්‍රීය කරන්න_

_**සටහන:**_ _ඔබ එය පමණක් සක්‍රීය කළහොත්, එම පරිශීලකයාට පමණක් තේමාව එක් කිරීමට සලකුණු කළ හැකි විකල්පයක් ලෙස එය ඔබේ සේවාලාභියාගේ පැත්තේ පෙන්වනු ඇත._

දැන්, ඔබේ පරිපාලක අතුරුමුහුණත සම්පූර්ණයෙන්ම නැවත නිර්මාණය වී ඇත! වම් තීරුවේ Material WP නම් නව මෙනුවක්ද ඔබට ඇත
