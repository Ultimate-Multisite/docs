# Gratis AI Agent

Gratis AI Agent යනු ස්වාභාවික භාෂා උපදෙස් හරහා සම්පූර්ණ WordPress අඩවි සැලසුම් කිරීමට, ගොඩනැගීමට, සහ කළමනාකරණය කිරීමට හැකි ස්වයංක්‍රීය WordPress AI සහායකයකි. එය නියෝජිතයාට කැඳවිය හැකි **හැකියාවන්** — වෙන්වූ, පරීක්ෂා කළ හැකි ක්‍රියා — ව්‍යුහගත කට්ටලයක් ලෙස ලබා දෙයි. ඒවා අන්තර්ගත කළමනාකරණයේ සිට සැලසුම් පද්ධති අභිරුචිකරණය දක්වා සියල්ල ආවරණය කරයි. අනුවාදය 1.9.0 අන්තර්ගත නිර්මාණ හැකියාවන්, client-පාර්ශ්ව තිරරූ හරහා දෘශ්‍ය පිටු සමාලෝචනය, ඇතුළත් නියෝජිත පැතිකඩ පහක්, සහ ප්‍රවේශ පාලනය හා සන්නාමකරණය සඳහා විශේෂාංග ධජ එක් කරයි.

## ප්‍රධාන විශේෂාංග {#key-features}

- **හැකියාවන් පද්ධතිය** — ඔබේ WordPress ස්ථාපනය මත නියෝජිතයාට කැඳවිය හැකි ක්‍රියාවන්ගේ මොඩියුලර්, පුළුල් කළ හැකි නාමාවලියක්
- **අන්තර්ගත කළමනාකරණය** — ලිපි සහ පිටු සාදන්න (පිටු අච්චු පැවරීම ඇතුළුව), එක් ඇමතුමකින් ලිපි කිහිපයක් සමූහ වශයෙන් සාදන්න, විශේෂාංග රූප සකසන්න, සහ සම්බන්ධතා පෝරම ගොඩනගන්න
- **දෘශ්‍ය සමාලෝචනය** — සජීවී පිටු වල තිරරූ ලබාගන්න, පෙර/පසු තත්ත්වයන් වෙනස්කම් ලෙස සසඳන්න, සහ පිරිසැලසුම, අකුරු රටාව, වර්ණ, සහ ප්‍රවේශනීයතාව ආවරණය කරන AI-ජනිත සැලසුම් සමාලෝචනයක් ලබාගන්න
- **ඇතුළත් නියෝජිතයන් පහක්** — අන්තර්ගත ලේඛක, අඩවි ගොඩනගන්නා, සැලසුම් ස්ටුඩියෝ, Plugin කළමනාකරු, සහ සහාය සහකරු; චැට් ශීර්ෂයේ නියෝජිත තේරීම හරහා මාරු කළ හැක
- **විශේෂාංග ධජ** — භූමිකා සීමා, white-label නම්කරණය, සහ widget අභිරුචිකරණය සඳහා සැකසුම් → විශේෂාංග ධජ තුළ ප්‍රවේශ පාලන සහ සන්නාමකරණ ටොගල්
- **අභිරුචි ලිපි වර්ග කළමනාකරණය** — නැවත ආරම්භ කිරීම් අතර පවත්වාගැනීම සමඟ අභිරුචි ලිපි වර්ග ලියාපදිංචි කරන්න, ලැයිස්තුගත කරන්න, සහ මකන්න
- **අභිරුචි වර්ගීකරණ කළමනාකරණය** — පවත්වාගැනීම සමඟ අභිරුචි වර්ගීකරණ ලියාපදිංචි කරන්න, ලැයිස්තුගත කරන්න, සහ මකන්න
- **සැලසුම් පද්ධති හැකියාවන්** — අභිරුචි CSS ඇතුළු කරන්න, තෝරාගත් block patterns කළමනාකරණය කරන්න, අඩවි ලාංඡනය සකසන්න, සහ theme.json presets යොදන්න
- **ගෝලීය ශෛලි කළමනාකරණය** — වර්ණ පුවරු, අකුරු රටා, ඉඩකඩ, සහ තවත් දේ සඳහා theme.json අගයන් කියවන්න සහ ලියන්න
- **සංචාලන මෙනු කළමනාකරණය** — සංචාලන මෙනු සහ ඒවායේ අයිතම වැඩසටහන්මය ලෙස සාදන්න සහ යාවත්කාලීන කරන්න
- **විකල්ප කළමනාකරණය** — වැදගත් සැකසුම් ආරක්ෂා කිරීමට ඇතුළත් ආරක්ෂිත අවහිර ලැයිස්තුවක් සමඟ WordPress විකල්ප කියවන්න සහ ලියන්න
- **ස්ථාපනය කළ හැකි හැකියාවන් රෙජිස්ට්‍රිය** — මූලික plugin ගොනු වෙනස් නොකර අතිරේක හැකියාවන් පැකේජ සොයාගන්න සහ සක්‍රිය කරන්න
- **අඩවි ගොඩනැගීමේ සම්බන්ධීකරණය v2** — plugin සොයාගැනීම, ප්‍රගති නිරීක්ෂණය, සහ දෝෂ ප්‍රතිසාධනය සමඟ බහු-පියවර අඩවි ගොඩනැගීමේ සැලසුම්
- **WP-CLI benchmark විධානය** — CI සහ model ඇගයීම සඳහා නියෝජිත හැකියාවන් benchmark කට්ටලය command line වෙතින් ධාවනය කරන්න

## අවශ්‍යතා {#requirements}

- WordPress 6.0 හෝ ඊට ඉහළ
- PHP 8.1 හෝ ඊට ඉහළ
- WordPress Multisite (network-activated) හෝ single-site ස්ථාපනය
- සක්‍රිය AI සැපයුම්කරු connector plugin එකක් (Anthropic, OpenAI, හෝ ගැළපෙන එකක්)

## ස්ථාපනය {#installation}

1. plugin ගොනු `/wp-content/plugins/gratis-ai-agent` වෙත උඩුගත කරන්න
2. WordPress හි **Plugins** තිරය හරහා plugin එක සක්‍රිය කරන්න
3. **Gratis AI Agent → සැකසුම්** වෙත ගොස් ඔබේ AI සැපයුම්කරු සම්බන්ධ කරන්න
4. (විකල්ප) අතිරේක හැකියාවන් පැකේජ ස්ථාපනය කිරීමට **හැකියාවන් රෙජිස්ට්‍රිය** බ්‍රවුස් කරන්න

## ඉක්මන් ආරම්භය {#quick-start}

සක්‍රිය කර AI සැපයුම්කරු සමඟ සම්බන්ධ කළ පසු, WordPress පරිපාලක පැති තීරුවේ **Gratis AI Agent** පැනලය විවෘත කරන්න. ස්වාභාවික භාෂා උපදෙසක් ටයිප් කරන්න. නියෝජිතයා තම පවතින හැකියාවන් භාවිතයෙන් අවශ්‍ය පියවර සැලසුම් කර ක්‍රියාත්මක කරයි.

**උදාහරණ උපදෙස්:**

## හැකියාවන් සාරාංශය {#abilities-overview}

## WP-CLI {#wp-cli}

Gratis AI Agent command line වෙතින් benchmarks ධාවනය කිරීමට සහ හැකියාවන් කළමනාකරණය කිරීමට `wp gratis-ai-agent` විධාන පවුලක් සමඟ පැමිණේ. සම්පූර්ණ විධාන ලැයිස්තුව සඳහා [WP-CLI යොමුව](./wp-cli) බලන්න.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

wp gratis-ai-agent abilities install <slug>
```

## විකල්ප කළමනාකරණ ආරක්ෂිත අවහිර ලැයිස්තුව {#options-management-safety-blocklist}

Options Management හැකියාවට, සංවේදී WordPress විකල්ප කියවීම හෝ ලිවීම agent වෙතින් වළක්වන අන්තර්ගත blocklist එකක් ඇතුළත් වේ. පෙරනිමි blocklist එක ආවරණය කරන්නේ:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL විකල්ප — මේවා සඳහා කැපවූ සංක්‍රමණ මෙවලමක් භාවිත කරන්න)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (රහස් යතුරු)
- `active_plugins`, `template` (plugin/theme සක්‍රීය කිරීම — කැපවූ හැකියාවන් භාවිත කරන්න)
- ස්ථාපිත plugins මඟින් ලියාපදිංචි කර ඇති ගෙවීම් gateway අක්තපත්‍ර සහ API යතුරු

අමතර ඇතුළත් කිරීම් `gratis_ai_agent_options_blocklist` filter හරහා එක් කළ හැක.
