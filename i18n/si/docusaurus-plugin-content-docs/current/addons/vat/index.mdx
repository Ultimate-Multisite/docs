# VAT {#vat}

Multisite Ultimate: යුරෝපීය VAT

යුරෝපීය VAT සඳහා වඩා හොඳ සහාය ලබා දීමට, Multisite Ultimate හි පෙරනිමි බදු හැසිරවීමට අමතර විශේෂාංග සහ මෙවලම් එක් කරයි.
Tax Rates පිටුවට EU VAT අනුපාත ස්වයංක්‍රීයව එක් කරයි

## ස්ථාපනය {#installation}

1. 'ultimate-multisite-vat' '/wp-content/plugins/' නාමාවලියට උඩුගත කරන්න
2. WordPress තුළ 'Plugins' මෙනුව හරහා plugin සක්‍රීය කරන්න
3. Ultimate Multisite > සැකසුම් > බදු වෙත ගොස් EU VAT සක්‍රීය කරන්න.
4. බදු අනුපාත මත ක්ලික් කර පහළ ඇති EU VAT අනුපාත යාවත්කාලීන කර බදු අනුපාත සුරකින්න.
5. අවසන්!
