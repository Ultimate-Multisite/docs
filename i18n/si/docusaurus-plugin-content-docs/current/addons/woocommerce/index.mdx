# WooCommerce ඒකාබද්ධකරණය {#woocommerce-integration}

WooCommerce ඒකාබද්ධකරණය, අපගේ අතින් ගෙවීම් ද්වාරය ක්‍රියා කරන ආකාරයට ඉතා සමානව ක්‍රියා කරයි. එය ඔබගේ පරිශීලකයින්ට විශාල නම්‍යශීලීත්වයක් ලබා දෙයි. එමඟින් ඔවුන්ට ඔබගේ WooCommerce ස්ථාපනයේ ඇති ඕනෑම ගෙවීම් ආකෘතියක් තෝරාගත හැක. එය සරලයි: පරිශීලකගේ බිල්කරණ කාලසීමාව අවසන් වූ පසු, පරිශීලක දායකත්වය නවතා තබයි. ඒ සමඟම, ජාලයේ ප්‍රධාන අඩවියේ WooCommerce ස්ථාපනය තුළ ඇණවුමක් සාදනු ලැබේ. පසුව පරිශීලකයාට ගෙවීම සිදු කිරීමට සබැඳියක් සහිත ඉන්වොයිසියක් ඊමේල් මඟින් ලැබේ. ගෙවීම ලැබුණු පසු, දායකත්වය නැවත අලුත් කරනු ලැබේ.
