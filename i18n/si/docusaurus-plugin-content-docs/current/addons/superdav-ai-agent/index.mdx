# Superdav AI Agent

Superdav AI Agent යනු Gratis AI Agent පුළුල් කරන උසස් WordPress AI සහායකයකි. එය theme-සචේතන කුසලතා, වැඩිදියුණු කළ අන්තර්ජාල සෙවුම් හැකියාවන්, අඩවි නිර්මාණ විශේෂාංග, සහ විකල්ප ස්වයංක්‍රීයකරණ ඒකාබද්ධතා සපයයි. 1.18.0 අනුවාදය Google Calendar කියවීමේ මෙවලම්, සම්බන්ධතා සහ සහභාගිවන්නන් සිතියම්ගත කිරීම, මානව අනුමැති දොරටු, මතක්කිරීම් අනුපිටපත් ඉවත් කිරීම, TextBee SMS දැනුම්දීම්, කළමනාකරණය කළ Superdav සේවා සැපයීම, සහ විශ්වාසනීය සංවර්ධක වැඩප්‍රවාහ සඳහා වෙනම බෙදාහරින Advanced සහායක පැකේජයක් එක් කරයි.

## ප්‍රධාන විශේෂාංග {#key-features}

- **Theme-සචේතන කුසලතා** — ඔබේ සක්‍රිය WordPress theme එක සහ ස්ථාපිත plugins මත පදනම්ව මඟපෙන්වීම සහ හැකියාවන් ස්වයංක්‍රීයව ගැළපේ
- **බහු අන්තර්ජාල සෙවුම් සැපයුම්කරුවන්** — වඩා සරු සෙවුම් ප්‍රතිඵල සහ උසස් පර්යේෂණ හැකියාවන් සඳහා Brave Search සහ Tavily අතරින් තෝරන්න
- **අඩවි නිර්මාණ සම්බන්ධතා forms** — ඔබ ස්ථාපිත කර ඇති form plugin භාවිතයෙන් chat අතුරුමුහුණතෙන්ම සම්පූර්ණයෙන් ක්‍රියාකාරී සම්බන්ධතා forms සාදන්න
- **වැඩිදියුණු කළ අඩවි නිර්මාණකරු** — වඩා හොඳ plugin සොයාගැනීම සහ දෝෂ ප්‍රතිසාධනය සමඟ වැඩිදියුණු කළ බහු-අදියර අඩවි ගොඩනැගීමේ සැලසුම්
- **ඉදි කළ theme කුසලතා** — Block Themes, Classic Themes, Kadence Blocks, සහ Kadence Theme සඳහා විශේෂිත මඟපෙන්වීම
- **වැඩිදියුණු කළ WooCommerce ඒකාබද්ධතාව** — වඩා හොඳ විශ්වසනීයතාව සහ ගැළපීම සඳහා ස්වදේශීය WooCommerce API භාවිතයට නැවත සැකසූවකි
- **ස්වයංක්‍රීය සැපයුම්කරු හඳුනාගැනීම** — plugins සහ themes සක්‍රිය හෝ අක්‍රිය කරන විට ලබාගත හැකි ඒවා ස්වයංක්‍රීයව හඳුනාගෙන නැවුම් කරයි
- **අභිරුචි logo ජනනය** — ස්වයංක්‍රීය namespace-ආරක්ෂිත පිරිසිදුකරණය සමඟ අභිරුචි SVG logos ජනනය කර ඇතුළත් කරන්න
- **Palette contrast වලංගු කිරීම** — ඔබේ theme එකට යෙදීමට පෙර WCAG ප්‍රවේශනීයතා අනුකූලතාව සඳහා වර්ණ යුගල පරීක්ෂා කරන්න
- **ඡායාරූප-වැඩිදියුණු කළ සොයාගැනීම** — සැලසුම් දිශාවට බලපෑම් කිරීමට Theme Builder සොයාගැනීමේ සම්මුඛ සාකච්ඡාව අතරතුර ඡායාරූප උඩුගත කරන්න
- **ආගන්තුක සත්කාර මෙනු** — ආපනශාලා, කැෆේ, සහ ආගන්තුක සත්කාර ව්‍යාපාර සඳහා ව්‍යුහගත ආහාර සහ පාන මෙනු පිටු ජනනය කරන්න
- **Desktop සහ mobile පෙරදසුන්** — සැලසුම් දිශාව තේරීම අතරතුර desktop සහ mobile උපාංගවල ඔබේ සැලසුම පෙරදසුන් කරන්න
- **Tier 1 අඩවි scraping** — තරඟකාරී සහ ප්‍රේරණාත්මක වෙබ් අඩවි විශ්ලේෂණය සඳහා sd-ai-agent/site-scrape දැන් පෙරනිමියෙන් ලබාගත හැක
- **Google Calendar කියවීමේ මෙවලම්** — ඉදිරියේ එන කාලසටහන් සන්දර්භය ගැන ස්වයංක්‍රීයකරණවලට තර්ක කළ හැකි වන පරිදි සකසා ඇති calendars සහ events කියවන්න
- **සම්බන්ධතා සහ සහභාගිවන්නන් සිතියම්ගත කිරීම** — පසු-ක්‍රියා වැඩ ජනනය කිරීමට පෙර calendar සහභාගිවන්නන් දන්නා අඩවි පරිශීලකයන් සහ සම්බන්ධතා සමඟ ගැළපන්න
- **මානව අනුමැති දොරටු** — පණිවිඩ යැවීමට, වැඩ නැවත ආරම්භ කිරීමට, හෝ වෙනස්කම් කිරීමට පෙර සමාලෝචනය සඳහා සංවේදී ස්වයංක්‍රීයකරණ විරාම කරන්න
- **මතක්කිරීම් වාර්තා සහ අනුපිටපත් ඉවත් කිරීම** — දැනටමත් යවා ඇති මතක්කිරීම් නිරීක්ෂණය කර, නැවත ක්‍රියාත්මක වන ස්වයංක්‍රීයකරණ එකම ලබන්නාට දෙවරක් දැනුම් නොදෙන බව සහතික කරයි
- **TextBee SMS දැනුම්දීම්** — SMS සැපයුම්කරු සක්‍රිය කර ඇති විට TextBee හරහා සකසා ඇති text-message දැනුම්දීම් යවන්න
- **කළමනාකරණය කළ Superdav සේවා සැපයීම** — අතින් endpoint සැකසීම වෙනුවට ස්වයංක්‍රීය සැකසීම සමඟ සහාය දක්වන අඩවි hosted Superdav endpoints වෙත සම්බන්ධ කරන්න
- **Advanced සහායක පැකේජය** — filesystem, database, WP-CLI, REST dispatcher, plugin builder, git snapshot, user-management, සහ benchmark වැඩප්‍රවාහ සඳහා වෙනම බෙදාහරින විශ්වාසනීය-සංවර්ධක මෙවලම්

## අවශ්‍යතා {#requirements}

- WordPress 6.0 හෝ ඉහළ
- PHP 8.1 හෝ ඉහළ
- WordPress Multisite (network-activated) හෝ single-site ස්ථාපනය
- සක්‍රිය AI සැපයුම්කරු connector plugin එකක් (Anthropic, OpenAI, හෝ ගැළපෙන එකක්)
- Gratis AI Agent v1.9.0 හෝ ඉහළ (Superdav AI Agent Gratis AI Agent පුළුල් කරයි)

## ස්ථාපනය {#installation}

1. plugin ගොනු `/wp-content/plugins/superdav-ai-agent` වෙත උඩුගත කරන්න
2. WordPress තුළ **Plugins** තිරය හරහා plugin එක සක්‍රිය කරන්න
3. Gratis AI Agent ද සක්‍රිය කර ඇති බව සහතික කරන්න
4. **Gratis AI Agent → Settings** වෙත ගොස් අවශ්‍ය පරිදි ඔබේ අන්තර්ජාල සෙවුම් සැපයුම්කරු, කළමනාකරණය කළ Superdav සේවා සම්බන්ධතාව, Google Calendar, හෝ TextBee credentials සකසන්න

WordPress.org පැකේජයේ මූලික Superdav AI Agent විශේෂාංග අඩංගු වේ. විශ්වාසනීය සංවර්ධක මෙවලම් පැහැදිලිව ස්ථාපනය කර අනුමත කරන පරිසර සඳහා Advanced සහායක පැකේජය වෙනම ගොඩනගා බෙදාහරිනු ලැබේ.
