# Logo SVG ජනනය කිරීම {#generate-logo-svg}

**Logo SVG ජනනය කිරීම** හැකියාව මඟින් Theme Builder ට ස්වයංක්‍රීය namespace-ආරක්ෂිත පිරිසිදු කිරීම සමඟ, ඔබේ WordPress අඩවිය තුළට අභිරුචි logo SVG සෘජුවම සාදා ඇතුළත් කළ හැක.

## සාරාංශය {#overview}

මෙම හැකියාව ඔබේ අඩවියේ සන්නාම දිශාව සහ සැලසුම් කැමැත්ත අනුව පරිමාණය කළ හැකි vector graphics (SVG) logo ජනනය කරයි. දෘශ්‍ය ගුණාත්මක බව රැක ගනිමින් WordPress තුළ භාවිතයට ආරක්ෂිත බව තහවුරු කිරීමට ජනනය කළ SVG ස්වයංක්‍රීයව පිරිසිදු කරනු ලැබේ.

## පරාමිතීන් {#parameters}

| පරාමිතිය | වර්ගය | අවශ්‍යයි | විස්තරය |
|-----------|------|----------|-------------|
| `site_name` | string | ඔව් | logo ජනනය කළ යුතු අඩවියේ නම |
| `style` | string | ඔව් | සැලසුම් ශෛලිය (උදා., "modern", "classic", "minimalist", "playful") |
| `colors` | array | නැත | logo තුළ භාවිතා කිරීමට hex වර්ණ කේත array එක (උදා., `["#678233", "#ffffff"]`) |
| `width` | number | නැත | SVG පළල pixel වලින් (පෙරනිමිය: 200) |
| `height` | number | නැත | SVG උස pixel වලින් (පෙරනිමිය: 200) |
| `include_text` | boolean | නැත | අඩවියේ නම logo තුළ පෙළ ලෙස ඇතුළත් කළ යුතුද යන්න (පෙරනිමිය: true) |

## ප්‍රතිදාන ආකෘතිය {#output-format}

මෙම හැකියාව පහත ව්‍යුහය සහිත SVG string එකක් ආපසු ලබා දෙයි:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG පිරිසිදු කිරීමේ හැසිරීම {#svg-sanitisation-behaviour}

ජනනය කරන SVG සඳහා ස්වයංක්‍රීය namespace-ආරක්ෂිත පිරිසිදු කිරීම සිදුවන්නේ මෙ සඳහාය:

- **අනාරක්ෂිත ගුණාංග ඉවත් කිරීම** — event handlers, scripts, සහ අවදානම් විය හැකි ගුණාංග ඉවත් කරයි
- **namespaces රැක ගැනීම** — නිසි rendering සඳහා SVG namespaces (xmlns, xlink) පවත්වා ගනී
- **ව්‍යුහය වලංගු කිරීම** — SVG එක W3C ප්‍රමිතීන්ට අනුකූල බව තහවුරු කරයි
- **entities encode කිරීම** — පෙළ අන්තර්ගතයේ විශේෂ අක්ෂර නිසි ලෙස escape කරයි
- **බාහිර යොමු ඉවත් කිරීම** — බාහිර stylesheets සහ පින්තූර යොමු ඉවත් කරයි

මෙමඟින් අතිරේක පිරිසිදු කිරීමක් අවශ්‍ය නොවී SVG එක WordPress තුළ සෘජුවම embed කිරීමට ආරක්ෂිත බව තහවුරු වේ.

## භාවිත උදාහරණය {#usage-example}

**යොමු පණිවිඩය:**
```
"CloudSync" ලෙස හැඳින්වෙන මගේ tech startup සඳහා නිල් සහ සුදු වර්ණ භාවිතා කර modern logo එකක් ජනනය කරන්න.
```

**ප්‍රතිඵලය:**
මෙම හැකියාව පහත ලක්ෂණ සහිත SVG logo එකක් සාදයි:
- "CloudSync" අඩවි නාමය ඇතුළත් කරයි
- සඳහන් කළ නිල් සහ සුදු වර්ණ සැකසුම භාවිතා කරයි
- modern සැලසුම් මූලධර්ම අනුගමනය කරයි
- ස්වයංක්‍රීයව පිරිසිදු කර භාවිතයට සූදානම් කර ඇත

## Theme Builder සමඟ ඒකාබද්ධ කිරීම {#integration-with-theme-builder}

Theme Builder හි සැලසුම්-දිශා තේරීම භාවිතා කරන විට, Logo SVG ජනනය කිරීමේ හැකියාව:

1. ඔබේ සැලසුම් දිශාව සහ වර්ණ palette විශ්ලේෂණය කරයි
2. ඔබේ කැමැත්තට ගැළපෙන අභිරුචි SVG logo එකක් ජනනය කරයි
3. logo එක ඔබේ අඩවියේ header/branding ප්‍රදේශයට ස්වයංක්‍රීයව embed කරයි
4. SVG එක WordPress media තුළ අභිරුචි logo ලෙස ගබඩා කරයි

## හොඳම භාවිතයන් {#best-practices}

- **පැහැදිලි ශෛලි කැමැත්ත ලබා දෙන්න** — ඔබට අවශ්‍ය සැලසුම් ශෛලිය විස්තර කරන්න (modern, classic, playful, ආදිය)
- **වර්ණ සඳහන් කරන්න** — එකඟතාවය සඳහා ඔබේ brand වර්ණ ඇතුළත් කරන්න
- **rendering පරීක්ෂා කරන්න** — විවිධ තිර ප්‍රමාණවල logo එක නිවැරදිව පෙන්වන බව තහවුරු කරන්න
- **තවදුරටත් අභිරුචිකරණය කරන්න** — ප්‍රමාණය සහ ස්ථානගත කිරීම සකස් කිරීමට WordPress හි logo අභිරුචිකරණ මෙවලම් භාවිතා කරන්න

## සීමාවන් {#limitations}

- SVG logo static graphics ලෙස ජනනය වේ (animated නොවේ)
- මූලිකාංග බොහොමයක් සහිත සංකීර්ණ logo සඳහා අතින් refinement අවශ්‍ය විය හැක
- අභිරුචි fonts සහාය නොදක්වයි; පෙළ system fonts භාවිතා කරයි
- ඉතා විශාල හෝ ඉතා කුඩා මානයන් ගුණාත්මකභාවයට බලපාිය හැක
