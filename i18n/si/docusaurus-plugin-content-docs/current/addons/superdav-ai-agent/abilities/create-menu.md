# මෙනුව සාදන්න

**මෙනුව සාදන්න** හැකියාව පිටු මාතෘකා වලින් වෙනස් වූ නාවික ලේබල් සඳහා සහාය සමඟ WordPress නාවික මෙනු සාදයි.

## සාරාංශය {#overview}

මෙම හැකියාව `navigation_label` පරාමිතියක් සඳහන් කිරීමේ හැකියාව සමඟ සම්මත මෙනු සාදන ක්‍රියාවලිය පුළුල් කරයි. මෙය නාවිකයේ පෙන්වන ලේබලය පිටු මාතෘකාවෙන් වෙනස් වන මෙනු සෑදීමට ඉඩ දෙයි. එමඟින් අඩවි ව්‍යුහය සහ පරිශීලක අත්දැකීම සඳහා වැඩි නම්‍යශීලිතාවක් ලැබේ.

## පරාමිතීන් {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | මෙනු නම, උදා. `Primary Navigation` |
| `location` | string | No | මෙම මෙනුවට පැවරීමට තේමා ස්ථානය, උදා. `primary` |
| `navigation_label` | string | No | නාවිකයේ පෙන්වීමට ලේබලය (පිටු මාතෘකාවෙන් වෙනස්) |

## ආපසු ලැබෙන අගය {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## නාවික ලේබලය සහ පිටු මාතෘකාව {#navigation-label-vs-page-title}

`navigation_label` පරාමිතිය අභ්‍යන්තර මෙනු නම පරිශීලකයන්ට පෙන්වන ලේබලයෙන් වෙන් කිරීමට ඉඩ දෙයි:

- **`name`** — WordPress භාවිතා කරන අභ්‍යන්තර මෙනු හඳුනාගැනීම (උදා., "Primary Navigation")
- **`navigation_label`** — අඩවි නැරඹුම්කරුවන්ට නාවිකයේ පෙන්වන ලේබලය (උදා., "Main Menu")

මෙය ප්‍රයෝජනවත් වන්නේ:
- ඔබේ අභ්‍යන්තර නම් කිරීමේ රටාව පරිශීලකයන්ට පෙනෙන ලේබල්වලින් වෙනස් වන විට
- පරිපාලන පුවරුවට වඩා නාවිකයේ කෙටි ලේබල් අවශ්‍ය වන විට
- වෙනස් ලේබල් දිග ඇති භාෂා කිහිපයකට සහාය දීමට අවශ්‍ය වන විට
- ඔබ නිශ්චිත කලාප හෝ පරිශීලක කණ්ඩායම් සඳහා මෙනු සාදන විට

## භාවිත උදාහරණ {#usage-examples}

### උදාහරණය 1: නාවික ලේබලයක් සහිත සරල මෙනුව {#example-1-simple-menu-with-navigation-label}

**ප්‍රොම්ප්ට්:**
```
"Main Navigation" නමින් ප්‍රධාන නාවික මෙනුවක්, "Menu" නාවික ලේබලය සමඟ සාදන්න.
```

**ප්‍රතිඵලය:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### උදාහරණය 2: නිශ්චිත තේමා ස්ථානයක් සඳහා මෙනුව {#example-2-menu-for-specific-theme-location}

**ප්‍රොම්ප්ට්:**
```
"Footer Links" නමින් පාදක මෙනුවක්, "Quick Links" නාවික ලේබලය සමඟ සාදා, එය පාදක ස්ථානයට පවරන්න.
```

**ප්‍රතිඵලය:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Theme Builder සමඟ ඒකාබද්ධ කිරීම {#integration-with-theme-builder}

Theme Builder භාවිතා කරන විට, මෙනුව සාදන හැකියාව:

1. ලබාගත හැකි තේමා මෙනු ස්ථාන ස්වයංක්‍රීයව හඳුනාගනී
2. ඔබේ සැලසුමට ගැළපෙන නාවික ලේබල් සහිත මෙනු සාදයි
3. මෙනු නිවැරදි තේමා ස්ථානවලට පවරයි
4. සාදීමෙන් පසු මෙනු අයිතම එකතු කිරීමට සහාය දක්වයි

## සම්බන්ධිත හැකියාවන් {#related-abilities}

- **`add_menu_item`** — පවතින මෙනුවකට අයිතම එකතු කරන්න
- **`update_menu`** — මෙනුවක් නැවත නම් කරන්න හෝ එය තේමා ස්ථානයකට නැවත පවරන්න
- **`delete_menu`** — ඔබේ අඩවියෙන් මෙනුවක් ඉවත් කරන්න

## හොඳම භාවිතයන් {#best-practices}

- **පැහැදිලි නාවික ලේබල් භාවිතා කරන්න** — පරිශීලකයන් සඳහා ලේබල් කෙටි සහ විස්තරාත්මකව තබා ගන්න
- **තේමා ස්ථාන ගැළපෙන්න** — නිසි පෙන්වීම සඳහා මෙනු නිවැරදි තේමා ස්ථානයට පවරන්න
- **මෙනු ව්‍යුහය සැලසුම් කරන්න** — අයිතම සෑදීමට පෙර ඔබේ මෙනු ධූරාවලිය තීරණය කරන්න
- **ප්‍රතිචාරාත්මක බව පරීක්ෂා කරන්න** — ජංගම උපාංගවල මෙනු නිවැරදිව පෙන්වෙන බව තහවුරු කරන්න
- **ලේබල් දේශීයකරණය කරන්න** — වෙනස් භාෂා අනුවාද සඳහා වෙනස් නාවික ලේබල් භාවිතා කරන්න

## සීමාවන් {#limitations}

- නාවික ලේබල් පෙන්වීම සඳහා පමණි; අභ්‍යන්තර `name` තවමත් WordPress හඳුනාගැනීම සඳහා භාවිතා වේ
- තේමා සහාය වෙනස් වේ; සියලු තේමා සියලු මෙනු ස්ථාන සඳහා සහාය නොදක්වයි
- මෙනුව සාදීමෙන් පසු මෙනු අයිතම වෙනම එකතු කළ යුතුය
- නාවික ලේබලයක් වෙනස් කිරීමට මෙනුව යාවත්කාලීන කිරීම අවශ්‍ය වේ
