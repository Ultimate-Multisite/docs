## ලබා ගත හැකි මෙවලම්

### ස්ථර 1 මෙවලම් (සැමවිටම ලබා ගත හැක)

ස්ථර 1 මෙවලම් අමතර වින්‍යාසයක් නැතිව Theme Builder තුළ පෙරනිමියෙන් ලබා ගත හැක.

#### sd-ai-agent/site-scrape

**අරමුණ:** නිර්මාණ ආභාසය, අන්තර්ගත ව්‍යුහය, සහ ක්‍රියාකාරීත්වය ලබා ගැනීමට පවතින වෙබ් අඩවි විශ්ලේෂණය කිරීම.

**හැකියාවන්:**

- **වෙබ් අඩවි විශ්ලේෂණය** — තරගකාරී හෝ ආභාස වෙබ් අඩවි scrape කර විශ්ලේෂණය කරන්න
- **නිර්මාණ උපුටා ගැනීම** — වර්ණ, අකුරු, සහ සැකසුම් රටා හඳුනා ගන්න
- **අන්තර්ගත ව්‍යුහය** — පිටු සංවිධානය සහ ධුරාවලිය තේරුම් ගන්න
- **විශේෂාංග හඳුනාගැනීම** — plugins සහ ක්‍රියාකාරීත්වය හඳුනා ගන්න
- **කාර්ය සාධන විශ්ලේෂණය** — පිටු වේගය සහ ප්‍රශස්තකරණය පරීක්ෂා කරන්න
- **SEO විශ්ලේෂණය** — meta tags සහ ව්‍යුහගත දත්ත සමාලෝචනය කරන්න

**භාවිතය:**

```
example.com හි නිර්මාණය විශ්ලේෂණය කර එය මගේ අඩවිය සඳහා ආභාසයක් ලෙස භාවිතා කරන්න.
```

**එය උපුටා ගන්නා දේ:**

- වර්ණ පුවරුව සහ වර්ණ භාවිතය
- අකුරුකරණය (අකුරු සහ ප්‍රමාණ)
- සැකසුම් ව්‍යුහය සහ ඉඩ තැබීම
- සංචාලන රටා
- අන්තර්ගත සංවිධානය
- රූප සහ මාධ්‍ය භාවිතය
- අන්තර්ක්‍රියාකාරී අංග
- ජංගම ප්‍රතිචාරශීලී බව

**සීමාවන්:**

- මුරපදයෙන් ආරක්ෂිත අඩවි scrape කළ නොහැක
- robots.txt සහ අඩවි ප්‍රතිපත්ති ගරු කරයි
- ගතික අන්තර්ගතය සම්පූර්ණයෙන් අල්ලා නොගත හැක
- JavaScript බහුල අඩවි සඳහා විශ්ලේෂණය සීමිත විය හැක
- විශාල අඩවි විශ්ලේෂණයට වැඩි කාලයක් ගත විය හැක

### ස්ථර 2 මෙවලම් (විකල්ප)

- උසස් වර්ණ සිද්ධාන්තය

### උසස් සහායක මෙවලම් (වෙනම පැකේජය)

උසස් සහායක පැකේජය මූලික Superdav AI Agent නිකුතුවෙන් වෙනම බෙදාහරිනු ලැබේ. පරිපාලකයන් ඉහළ බලපෑමක් ඇති මෙවලම් භාවිතා කිරීමට agent වෙත පැහැදිලිව අවසර දෙන විශ්වාසදායක සංවර්ධක සහ අඩවි හිමි පරිසර සඳහා එය අදහස් කර ඇත.

උසස් සහායක හැකියාවන්ට ඇතුළත් විය හැක්කේ:

- **ගොනු පද්ධති මෙවලම්** — විශ්වාසදායක පරිසරවල අනුමත ගොනු පරීක්ෂා කර වෙනස් කරන්න
- **දත්ත සමුදා මෙවලම්** — පැහැදිලිව අවසර දුන් විට අඩවි දත්ත විමසන්න හෝ යාවත්කාලීන කරන්න
- **WP-CLI මෙවලම්** — WordPress නඩත්තු සහ පරීක්ෂණ විධාන ධාවනය කරන්න
- **REST බෙදාහැරීම් මෙවලම්** — පාලිත බෙදාහැරීම් කාර්ය ප්‍රවාහ හරහා ලියාපදිංචි REST endpoints අමතන්න
- **Plugin ගොඩනැගීමේ මෙවලම්** — plugin code සැකිලිගත කර පියවරෙන් පියවර වැඩිදියුණු කරන්න
- **Git snapshot මෙවලම්** — අවදානම් සහිත සංවර්ධක මෙහෙයුම් පෙර snapshots සාදන්න

1. **robots.txt ගරු කරන්න** — site මාර්ගෝපදේශ අනුගමනය කරන්න
2. **අන්තර්ගතය පිටපත් නොකරන්න** — විශ්ලේෂණය ප්‍රේරණය සඳහා පමණක් භාවිත කරන්න
3. **අයිතිවාසිකම් තහවුරු කරන්න** — ලබාගත් අන්තර්ගතය භාවිත කළ හැකි බව සහතික කරන්න
4. **මූලාශ්‍ර සඳහන් කරන්න** — ප්‍රේරණ මූලාශ්‍රවලට ගෞරවය දෙන්න
5. **කොන්දේසි අනුගමනය කරන්න** — site සේවා කොන්දේසි පිළිපදින්න

### මෙවලම් කාර්යක්ෂමතාව උපරිම කිරීම

1. **නිශ්චිත වන්න** — නිශ්චිත විශ්ලේෂණ වර්ග ඉල්ලා සිටින්න
2. **සන්දර්භය ලබා දෙන්න** — ඔබේ site හි අරමුණ පැහැදිලි කරන්න
3. **අපේක්ෂා සකසන්න** — ඔබේ සැලසුම් ඉලක්ක විස්තර කරන්න
4. **පුනරාවර්තනය කරන්න** — ප්‍රතිඵල මත පදනම්ව ශෝධනය කරන්න
5. **මෙවලම් එක් කරන්න** — පුළුල් විශ්ලේෂණයක් සඳහා මෙවලම් කිහිපයක් භාවිත කරන්න
