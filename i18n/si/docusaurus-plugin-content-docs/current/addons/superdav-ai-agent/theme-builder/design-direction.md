# නිර්මාණ දිශාව

**නිර්මාණ දිශාව** පියවරෙන්, Theme Builder සම්පූර්ණ theme එක ජනනය කිරීමට පෙර ඔබේ theme එකේ දෘශ්‍ය නිර්මාණය සමාලෝචනය, වැඩිදියුණු කිරීම, සහ අවසන් කිරීම කළ හැක.

## දළ විශ්ලේෂණය

Discovery Interview සම්පූර්ණ කිරීමෙන් පසු, Theme Builder **desktop සහ mobile පෙරදසුන් ප්‍රදර්ශනය** සමඟ ඔබේ නිර්මාණ දිශාව පෙන්වයි. මෙය theme එකට බැඳීමට පෙර, වෙනස් උපාංගවල ඔබේ නිර්මාණය නිවැරදිව කෙසේ දිස්වනු ඇත්දැයි දැකීමට ඉඩ දෙයි.

## නිර්මාණ දිශාවේ සංරචක

### 1. වර්ණ පුවරුව

ඔබේ theme එකේ වර්ණ සැලැස්ම, ඇතුළත්ව:

- **ප්‍රාථමික වර්ණය** — බොත්තම්, සබැඳි, සහ අවධාරණ සඳහා භාවිත කරන ප්‍රධාන සන්නාම වර්ණය
- **ද්විතීයික වර්ණය** — විවිධත්වය සඳහා යෙදෙන පූරක වර්ණය
- **අවධාරණ වර්ණය** — වැදගත් අංග සඳහා යෙදෙන ඉස්මතු කිරීමේ වර්ණය
- **නිෂ්පක්ෂ වර්ණ** — පසුබිම් සහ පෙළ සඳහා අළු සහ සුදු වර්ණ
- **පෙළ වර්ණය** — කියවීමට පහසුකම සඳහා ප්‍රධාන පෙළ වර්ණය

### 2. අකුරු රටාව

අකුරු තේරීම්:

- **ශීර්ෂ අකුර** — පිටු මාතෘකා සහ කොටස් ශීර්ෂ සඳහා භාවිත වේ
- **ප්‍රධාන පෙළ අකුර** — ඡේද පෙළ සහ අන්තර්ගතය සඳහා භාවිත වේ
- **අකුරු ප්‍රමාණ** — වෙනස් තිර ප්‍රමාණ සඳහා ප්‍රතිචාරාත්මක ප්‍රමාණකරණය
- **පේළි උස** — කියවීමට පහසුකම සඳහා පේළි අතර ඉඩ
- **අකුරු බර** — තද, සාමාන්‍ය, සහ සැහැල්ලු වෙනස්කම්

### 3. සැලසුම සහ ඉඩකඩ

- **බහාලුම් පළල** — අන්තර්ගතය සඳහා උපරිම පළල
- **ඇතුළත සහ පිටත ඉඩ** — අංග වටා ඇති ඉඩ
- **ජාල පද්ධතිය** — ප්‍රතිචාරාත්මක නිර්මාණය සඳහා තීරු සැලසුම
- **සංරචක ඉඩකඩ** — UI අංග අතර දුර

### 4. දෘශ්‍ය අංග

- **බොත්තම් ශෛලීන්** — ප්‍රාථමික, ද්විතීයික, සහ තෘතීයික බොත්තම් නිර්මාණ
- **කාඩ් නිර්මාණ** — අන්තර්ගත කොටස් සඳහා සැලසුම්
- **අයිකන** — අයිකන ශෛලිය සහ ප්‍රමාණය
- **සීමා සහ සෙවනැලි** — සියුම් දෘශ්‍ය ගැඹුර
- **Hover effects** — අන්තර්ක්‍රියාත්මක තත්ත්ව ශෛලීකරණය

## පෙරදසුන් ප්‍රදර්ශනය

### Desktop පෙරදසුන

Desktop පෙරදසුන ඔබේ නිර්මාණය සම්පූර්ණ පළලින් පෙන්වයි (සාමාන්‍යයෙන් 1200px හෝ ඊට වැඩි):

- සම්පූර්ණ navigation මෙනුව
- සියලු තීරු සමඟ සම්පූර්ණ සැලසුම
- විශාල අකුරු රටාව සහ ඉඩකඩ
- සියලු දෘශ්‍ය අංග සම්පූර්ණ ප්‍රමාණයෙන්

### Mobile පෙරදසුන

Mobile පෙරදසුන ඔබේ නිර්මාණය mobile පළලින් පෙන්වයි (සාමාන්‍යයෙන් 375px):

- ප්‍රතිචාරාත්මක navigation (hamburger menu)
- එක්-තීරු සැලසුම
- සකස් කළ අකුරු රටාව සහ ඉඩකඩ
- ස්පර්ශයට හිතකර බොත්තම් ප්‍රමාණ

### ප්‍රතිචාරාත්මක කඩඉම්

Theme Builder මෙම කඩඉම්වල පෙරදසුන් ප්‍රදර්ශනය කරයි:

| උපාංගය | පළල | පෙරදසුන |
|--------|-------|---------|
| **Mobile** | 375px | සිරස් දුරකථනය |
| **Tablet** | 768px | තිරස් tablet |
| **Desktop** | 1200px | සම්පූර්ණ පළල desktop |
| **විශාල Desktop** | 1920px | අති-පළල් තිරය |

## ඔබේ නිර්මාණය වැඩිදියුණු කිරීම

### සකස්කිරීම් කිරීම

ඔබට ඔබේ නිර්මාණ දිශාව මෙසේ වැඩිදියුණු කළ හැක:

1. **වර්ණ සකස් කිරීම** — පුවරුවේ ඕනෑම වර්ණයක් වෙනස් කරන්න
2. **අකුරු වෙනස් කිරීම** — වෙනස් අකුරු මුහුණත් තෝරන්න
3. **ඉඩකඩ වෙනස් කිරීම** — ඇතුළත සහ පිටත ඉඩ සකස් කරන්න
4. **සැලසුම යාවත්කාලීන කිරීම** — බහාලුම් පළල සහ ජාල තීරු වෙනස් කරන්න
5. **අංග අභිරුචිකරණය** — බොත්තම් ශෛලීන්, කාඩ් නිර්මාණ ආදිය වෙනස් කරන්න

### පෙරදසුන් යාවත්කාලීන

වෙනස්කම් සජීවීව පිළිබිඹු වේ:

- Desktop පෙරදසුන වහාම යාවත්කාලීන වේ
- Mobile පෙරදසුන වහාම යාවත්කාලීන වේ
- සියලු ප්‍රතිචාරාත්මක කඩඉම් යාවත්කාලීන වේ
- වෙනස්කම් තහවුරු කිරීමට පෙරදසුන් අතර මාරු විය හැක

### සැසඳුම් දසුන

ඔබේ නිර්මාණ දිශාව මෙවැනි දේ සමඟ සැසඳන්න:

- **මුල් නිර්මාණය** — වෙනස් වූ දේ බලන්න
- **තරඟකරුවන්ගේ නිර්මාණ** — ආභාසය ලබාගත් අඩවි සමඟ සැසඳන්න
- **පෙර අනුවාද** — කලින් නිර්මාණ දිශාවලට නැවත යන්න

## නිර්මාණ දිශාව අනුමත කිරීම

ඔබේ නිර්මාණය ගැන සෑහීමකට පත්වූ පසු:

1. **පෙරදසුන් දෙකම සමාලෝචනය කරන්න** — desktop සහ mobile හොඳින් දිස්වන බව තහවුරු කරන්න
2. **සියලු වර්ණ පරීක්ෂා කරන්න** — විරුද්ධතාව සහ ප්‍රවේශකම සහතික කරන්න
3. **අකුරු රටාව පරීක්ෂා කරන්න** — සියලු ප්‍රමාණවල කියවීමට පහසු බව තහවුරු කරන්න
4. **සැලසුම තහවුරු කරන්න** — ඉඩකඩ සහ පෙළගැස්වීම පරීක්ෂා කරන්න
5. **නිර්මාණය අනුමත කරන්න** — theme ජනනයට ඉදිරියට යන්න

## ඊළඟ පියවර

ඔබේ නිර්මාණ දිශාව අනුමත කිරීමෙන් පසු:

4. [ආගන්තුක සත්කාර මෙනු](./hospitality-menus.md) හෝ වෙනත් විශේෂාංග වෙත ඉදිරියට යන්න

## හොඳම ක්‍රම

- **සැබෑ උපාංගවල පරීක්ෂා කරන්න** — හැකි නම් සැබෑ දුරකථන සහ tablet භාවිත කරන්න
- **කියවීමට පහසුකම පරීක්ෂා කරන්න** — සියලු ප්‍රමාණවල පෙළ කියවීමට පහසු බව සහතික කරන්න
- **විරුද්ධතාව තහවුරු කරන්න** — ප්‍රවේශකම සඳහා [වර්ණ පුවරු විරුද්ධතාව වලංගු කරන්න](../abilities/validate-palette-contrast.md) භාවිත කරන්න
- **කාර්යසාධනය සලකා බලන්න** — වේගය සඳහා පින්තූර සහ අකුරු ප්‍රශස්ත කරන්න
- **අන්තර්ගතය සඳහා සැලසුම් කරන්න** — ඔබේ සැබෑ අන්තර්ගතය සමඟ සැලසුම ක්‍රියා කරන බව සහතික කරන්න

## ගැටලු නිරාකරණය

### පෙරදසුන යාවත්කාලීන නොවීම

- වෙනත් browser එකක් උත්සාහ කරන්න
- අන්තර්ජාල සම්බන්ධතාව පරීක්ෂා කරන්න

- වෙනස් උපාංගවල උත්සාහ කරන්න
- ප්‍රවේශකම මෙවලම් සමඟ වර්ණ විරුද්ධතාව තහවුරු කරන්න
- වර්ණ අන්ධතාව අනුකරණ මෙවලම් සලකා බලන්න

### අකුරු රටා ගැටලු

- වෙනස් කඩඉම්වල අකුරු ප්‍රමාණය පරීක්ෂා කරන්න
- සැබෑ අන්තර්ගතය සමඟ පරීක්ෂා කරන්න
- කියවීමට පහසුකම සඳහා පේළි දිග සලකා බලන්න
