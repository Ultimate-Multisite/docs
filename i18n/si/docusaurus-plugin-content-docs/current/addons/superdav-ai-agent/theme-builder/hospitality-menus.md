# ආගන්තුක සේවා මෙනු {#hospitality-menus}

**ආගන්තුක සේවා මෙනු** විශේෂාංගය Theme Builder හට ව්‍යුහගත ආහාර සහ පාන මෙනු පිටු සෘජුවම ඔබේ WordPress අඩවියට ජනනය කර ඇතුළත් කිරීමට ඉඩ දෙයි.

## දළ විශ්ලේෂණය {#overview}

Theme Builder හට දැන් අවන්හල්, කැෆේ, බාර් සහ catering සේවා ඇතුළු ආගන්තුක සේවා ව්‍යාපාර සඳහා වෘත්තීය, ව්‍යුහගත මෙනු පිටු නිර්මාණය කළ හැක. මෙම මෙනු ඔබේ අඩවියේ නිර්මාණයට සම්පූර්ණයෙන් ඒකාබද්ධ වන අතර පහසුවෙන් යාවත්කාලීන කර කළමනාකරණය කළ හැක.

## සහාය දක්වන ආගන්තුක සේවා වර්ග {#supported-hospitality-types}

- **අවන්හල්** — පූර්ණ සේවා ආහාර ගැනීමේ මෙනු
- **කැෆේ** — කෝපි සහ සැහැල්ලු ආහාර මෙනු
- **බාර් සහ ලවුන්ජ්** — පාන සහ ආරම්භක කෑම මෙනු
- **බේකරි** — පේස්ට්‍රි සහ පාන් මෙනු
- **Catering සේවා** — උත්සව මෙනු විකල්ප
- **Food Trucks** — චලන ආහාර සේවා මෙනු
- **බෲවරි සහ වයිනරි** — විස්තර සහිත පාන මෙනු

## මෙනු ව්‍යුහය {#menu-structure}

### මෙනු කාණ්ඩ {#menu-categories}

මෙනු කාණ්ඩවලට සංවිධානය කර ඇත:

- **ආරම්භක කෑම** — ආරම්භක සහ කුඩා පිඟන්
- **ප්‍රධාන කෑම** — ප්‍රධාන ආහාර
- **අතුරු කෑම** — සහායක කෑම සහ එළවළු
- **අතුරුපස** — පැණි රස කෑම
- **පාන** — බීම වර්ග (මධ්‍යසාර සහ මධ්‍යසාර නොවන)
- **විශේෂ කෑම** — දෛනික හෝ කාලානුරූපී විශේෂ

### මෙනු අයිතම ආකෘතිය {#menu-item-format}

එක් එක් මෙනු අයිතමයට ඇතුළත් වන්නේ:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### අයිතම ක්ෂේත්‍ර {#item-fields}

| ක්ෂේත්‍රය | වර්ගය | විස්තරය |
|-------|------|-------------|
| `name` | string | ආහාරයේ හෝ පාන වර්ගයේ නම |
| `description` | string | අයිතමයේ සවිස්තර විස්තරය |
| `price` | string | මිල (මුදල් ඒකකය සමඟ ආකෘතිගත) |
| `dietary_info` | array | ආහාරමය ගුණාංග (Vegan, Gluten-Free, ආදිය) |
| `allergens` | array | පොදු අසාත්මිකකාරක (ගෙඩි වර්ග, කවච මත්ස්‍ය, ආදිය) |
| `availability` | string | ලබා ගත හැකි කාලය (දෛනික, කාලානුරූපී, ආදිය) |

## ආගන්තුක සේවා මෙනු නිර්මාණය කිරීම {#creating-hospitality-menus}

### පියවර 1: මෙනු තොරතුරු ලබා දෙන්න {#step-1-provide-menu-information}

ඔබේ මෙනුව ගැන Theme Builder ට කියන්න:

```
මගේ ඉතාලි අවන්හල සඳහා ආරම්භක කෑම,
පැස්ටා ආහාර, ප්‍රධාන කෑම සහ අතුරුපස සහිත අවන්හල් මෙනුවක් සාදන්න. මිල සහ විස්තර ඇතුළත් කරන්න.
```

### පියවර 2: Theme Builder මෙනුව ජනනය කරයි {#step-2-theme-builder-generates-menu}

Theme Builder:

1. ව්‍යුහගත මෙනු පිටුවක් නිර්මාණය කරයි
2. එය ඔබේ අඩවියේ theme එකට ගැළපෙන ලෙස නිර්මාණය කරයි
3. අයිතම කාණ්ඩවලට සංවිධානය කරයි
4. මිල සහ විස්තර ආකෘතිගත කරයි
5. ආහාරමය සහ අසාත්මිකකාරක තොරතුරු එක් කරයි

### පියවර 3: සමාලෝචනය කර අභිරුචිකරණය කරන්න {#step-3-review-and-customize}

ඔබට හැකිය:

1. මෙනු අයිතම සහ මිල සංස්කරණය කිරීම
2. කාණ්ඩ එක් කිරීම හෝ ඉවත් කිරීම
3. කාණ්ඩ තුළ අයිතම නැවත අනුපිළිවෙළට සැකසීම
4. විස්තර සහ ආහාරමය තොරතුරු යාවත්කාලීන කිරීම
5. හැඩගැන්වීම සහ පිටු සැකැස්ම සකස් කිරීම

## මෙනු ප්‍රදර්ශන විකල්ප {#menu-display-options}

### සම්පූර්ණ මෙනු පිටුව {#full-menu-page}

ඔබේ සම්පූර්ණ මෙනුව පෙන්වන කැපවූ පිටුවක්:

- කාණ්ඩ අනුව සංවිධානය කර ඇත
- සෙවිය හැකි සහ පෙරහන් කළ හැකි
- මුද්‍රණයට පහසු පිටු සැකැස්ම
- ජංගම උපාංගවලට ප්‍රතිචාර දක්වන නිර්මාණය

### මෙනු Widget {#menu-widget}

වෙනත් පිටු මත මෙනු කොටස් ඇතුළත් කරන්න:

- මුල් පිටුවේ තෝරාගත් අයිතම
- පැති තීරුවේ දෛනික විශේෂ
- බාර් පිටුවේ පාන මෙනුව
- footer හි අතුරුපස ප්‍රදර්ශනය

### මෙනු PDF {#menu-pdf}

බාගත කළ හැකි PDF මෙනුවක් ජනනය කරන්න:

- වෘත්තීය ආකෘතිකරණය
- මුද්‍රණයට සූදානම් ගුණාත්මකභාවය
- රූප සහ විස්තර ඇතුළත් වේ
- බෙදාගැනීමට සහ විද්‍යුත් තැපෑලෙන් යැවීමට පහසුය

## ආහාරමය සහ අසාත්මිකකාරක තොරතුරු {#dietary-and-allergen-information}

### ආහාරමය ගුණාංග {#dietary-attributes}

ආහාරමය තොරතුරු සමඟ අයිතම සලකුණු කරන්න:

- **Vegan** — සත්ත්ව නිෂ්පාදන නැත
- **Vegetarian** — මස් නැත
- **Gluten-Free** — සීලියැක් රෝගය ඇති අයට ආරක්ෂිතයි
- **Dairy-Free** — කිරි නිෂ්පාදන නැත
- **Nut-Free** — ගස් ගෙඩි හෝ රටකජු නැත
- **Low-Carb** — අඩු කළ කාබෝහයිඩ්‍රේට්
- **High-Protein** — ප්‍රෝටීන් බහුල

### අසාත්මිකකාරක අනතුරු ඇඟවීම් {#allergen-warnings}

පොදු අසාත්මිකකාරක ඇතුළත් කරන්න:

- **ගෙඩි වර්ග** — ගස් ගෙඩි සහ රටකජු
- **කවච මත්ස්‍ය** — ක්‍රස්ටේසියන් සහ මොලුස්කාවන්
- **මාළු** — සියලුම මාළු විශේෂ
- **කිරි ආහාර** — කිරි සහ කිරි නිෂ්පාදන
- **බිත්තර** — කුකුළු බිත්තර
- **සෝයා** — සෝයා නිෂ්පාදන
- **ග්ලූටන්** — තිරිඟු සහ ග්ලූටන් අඩංගු ධාන්‍ය
- **තල** — තල ඇට සහ තෙල්

## මෙනු කළමනාකරණය {#menu-management}

### මිල යාවත්කාලීන කිරීම {#updating-prices}

මෙනු මිල පහසුවෙන් යාවත්කාලීන කරන්න:

1. මෙනු පිටුවට යන්න
2. "මෙනුව සංස්කරණය කරන්න" ක්ලික් කරන්න
3. අයිතම සඳහා මිල යාවත්කාලීන කරන්න
4. වෙනස්කම් සුරකින්න
5. වෙනස්කම් ඔබේ අඩවියේ වහාම දිස් වේ

### කාලානුරූපී අයිතම එක් කිරීම {#adding-seasonal-items}

කාලානුරූපී මෙනු වෙනස්කම් සාදන්න:

1. නව මෙනු අනුවාදයක් සාදන්න
2. කාලානුරූපී අයිතම එක් කරන්න
3. අයිතම "කාලානුරූපී" ලෙස සලකුණු කරන්න
4. ලබා ගත හැකි දිනයන් සැලසුම් කරන්න
5. කාලය තුළ ස්වයංක්‍රීයව පෙන්වන්න

### විශේෂ කෑම කළමනාකරණය {#managing-specials}

දෛනික හෝ සතිපතා විශේෂ පෙන්වන්න:

1. "විශේෂ කෑම" කාණ්ඩයක් සාදන්න
2. ලබා ගත හැකි දිනයන් සමඟ අයිතම එක් කරන්න
3. මුල් පිටුවේ විශේෂ කෑම ඉස්මතු කරන්න
4. දෛනිකව හෝ සතිපතා යාවත්කාලීන කරන්න
5. පැරණි විශේෂ කෑම සංරක්ෂණය කරන්න

## Theme Builder සමඟ ඒකාබද්ධ කිරීම {#integration-with-theme-builder}

ආගන්තුක සේවා අඩවි සඳහා Theme Builder භාවිතා කරන විට:

1. **ස්වයංක්‍රීය මෙනු හඳුනාගැනීම** — ඔබ ආහාර/පාන ව්‍යාපාරයක්දැයි හඳුනාගනී
2. **මෙනු පිටු නිර්මාණය** — වෘත්තීය මෙනු පිටු ජනනය කරයි
3. **නිර්මාණ ගැළපීම** — මෙනු ඔබේ අඩවියේ theme එකට ගැළපේ
4. **ජංගම ප්‍රශස්තිකරණය** — මෙනු දුරකථනවල අලංකාර ලෙස පෙන්වයි
5. **සෙවුම් යන්ත්‍ර ප්‍රශස්තිකරණය** — මෙනු සෙවුම් යන්ත්‍ර හිතකාමීය

## හොඳම භාවිතයන් {#best-practices}

### මෙනු නිර්මාණය {#menu-design}

- **පැහැදිලි සංවිධානය** — තාර්කික කාණ්ඩ ව්‍යුහය
- **කියවිය හැකි විස්තර** — රසවත් සහ තොරතුරුදායක
- **සංගත මිලකරණය** — පැහැදිලි මුදල් ඒකකය සහ ආකෘතිකරණය
- **වෘත්තීය ඡායාරූප** — උසස් තත්ත්වයේ ආහාර රූප
- **හිස් ඉඩ** — පිටුව අධික ලෙස පුරවන්න එපා

### අන්තර්ගතය {#content}

- **නිවැරදි විස්තර** — අයිතම නිවැරදිව විස්තර කරන්න
- **විශේෂ කෑම ඉස්මතු කරන්න** — විශේෂ කෑම කැපී පෙනෙන ලෙස කරන්න
- **අසාත්මිකකාරක ඇතුළත් කරන්න** — අසාත්මිකකාරක සැමවිටම ලැයිස්තුගත කරන්න
- **නිතර යාවත්කාලීන කරන්න** — මිල සහ අයිතම වර්තමානව තබා ගන්න
- **රසවත් භාෂාව භාවිතා කරන්න** — අයිතම රසවත් ලෙස හැඟෙන සේ කරන්න

### ප්‍රවේශවීමේ හැකියාව {#accessibility}

- **කියවිය හැකි අකුරු** — පැහැදිලි, කියවීමට පහසු අකුරු මුහුණත් භාවිතා කරන්න
- **ප්‍රමාණවත් ප්‍රතිවිරෝධය** — පෙළ කියවිය හැකි බව තහවුරු කරන්න
- **ආහාරමය ලේබල්** — ආහාරමය විකල්ප පැහැදිලිව සලකුණු කරන්න
- **අසාත්මිකකාරක අනතුරු ඇඟවීම්** — අසාත්මිකකාරක ප්‍රමුඛව පෙන්වන්න
- **ජංගම-හිතකාමී** — සියලු උපාංගවල පරීක්ෂා කරන්න

### අවන්හල් මෙනු ව්‍යුහය {#restaurant-menu-structure}

```
ආරම්භක කෑම

├── කැලමාරි ෆ්‍රිටි
└── කප්‍රේසේ සලාදය

├── ස්පැගෙටි කාර්බොනාරා
├── ෆෙටුචිනි ඇල්ෆ්‍රෙඩෝ

└── සල්ටිම්බොකා අල්ලා රොමානා

├── ටිරමිසු
├── පැන්නා කොට්ටා
└── ජෙලාටෝ තේරීම
```
