# බහු-මුදල්

Ultimate Multisite: බහු-මුදල් ඔබේ ජාලයට බහු මුදල් ඒකකවලින් ගෙවීම් භාර ගැනීමට ඉඩ දෙයි. එය සජීවී හෝ අතින් සකසන විනිමය අනුපාත භාවිතයෙන් මිල ස්වයංක්‍රීයව පරිවර්තනය කරයි, ඕනෑම මුදල් ඒකකයකට නිශ්චිත නිෂ්පාදන-අනුව මිල සකසීමට ඉඩ දෙයි, සහ භූ-ස්ථානගත කිරීම මගින් එක් එක් අමුත්තන්ගේ කැමති මුදල් ඒකකය හඳුනා ගනී — මේ සියල්ල එක් පරිපාලන Dashboard එකකින් කළමනාකරණය කෙරේ.

## ප්‍රධාන විශේෂාංග {#key-features}

- **ස්වයංක්‍රීය විනිමය අනුපාත පරිවර්තනය** — ඔබ තෝරාගත් සපයන්නාගෙන් ලබාගන්නා අනුපාත භාවිතයෙන් ඔබේ මූලික මුදල් ඒකකයෙන් මිල පරිවර්තනය කෙරේ
- **නිෂ්පාදන-අනුව මිල අභිබවා යෙදවීම්** — ස්වයංක්‍රීය පරිවර්තනය මඟ හැරීමට ඕනෑම නිෂ්පාදනයකට නිශ්චිත මුදල් ඒකක-විශේෂිත මිල සකසන්න
- **මුදල් ඒකක තේරීමේ ලියාපදිංචි ක්ෂේත්‍රය** — ගෙවීමේ/ලියාපදිංචි වීමේදී පාරිභෝගිකයන්ට තමන් කැමති මුදල් ඒකකය තෝරා ගැනීමට ඉඩ දෙයි
- **බහු විනිමය අනුපාත සපයන්නන්** — ECB, Frankfurter, Stripe, හෝ සම්පූර්ණයෙන්ම අතින් සකසන අනුපාත
- **භූ-ස්ථානගත කිරීම මත පදනම් වූ මුදල් ඒකක හඳුනාගැනීම** — අමුත්තන්ගේ රටට ගැළපෙන මුදල් ඒකකය ස්වයංක්‍රීයව පෙර-තෝරයි
- **පරිපාලන Dashboard** — සක්‍රිය මුදල් ඒකක, විනිමය අනුපාත, සහ සපයන්නා සැකසුම් එක් තැනකින් කළමනාකරණය කරන්න

## අවශ්‍යතා {#requirements}

- Ultimate Multisite 2.0.0 හෝ ඉහළ
- WordPress Multisite ස්ථාපනය
- PHP 7.4 හෝ ඉහළ

## ස්ථාපනය {#installation}

1. දිගු ගොනු `/wp-content/plugins/ultimate-multisite-multi-currency` වෙත උඩුගත කරන්න
2. WordPress හි **දිගු** තිරය හරහා දිගුව සක්‍රිය කරන්න (ජාලය සක්‍රිය කරන්න)
3. පරිපාලන Dashboard විවෘත කිරීමට **Ultimate Multisite → බහු-මුදල්** වෙත යන්න
4. ඔබ භාර ගැනීමට කැමති මුදල් ඒකක එක් කර, ඔබේ විනිමය අනුපාත සපයන්නා වින්‍යාස කරන්න

## වින්‍යාසය {#configuration}

### පරිපාලන Dashboard {#admin-dashboard}

බහු-මුදල් පරිපාලන Dashboard (**Ultimate Multisite → බහු-මුදල්**) යනු පහත දේවල් සඳහා මධ්‍යස්ථ ස්ථානයයි:

- මුදල් ඒකක එක් කිරීම, සක්‍රිය කිරීම, හෝ අක්‍රිය කිරීම
- විනිමය අනුපාත සපයන්නා තෝරා, අවශ්‍ය විට අනුපාත නැවුම් කිරීම
- සියලු සක්‍රිය මුදල් ඒකක සඳහා වත්මන් අනුපාත පරීක්ෂා කිරීම
- ස්වයංක්‍රීය සපයන්නන් සුදුසු නොවන විට අතින් අනුපාත සකසීම

### විනිමය අනුපාත සපයන්නන් {#exchange-rate-providers}

| සපයන්නා | විස්තරය |
|---|---|
| **ECB** | යුරෝපීය මධ්‍යම බැංකුව — EUR-මූලික ජාල සඳහා නොමිලේ දෛනික අනුපාත |
| **Frankfurter** | විවෘත-මූලාශ්‍ර ECB mirror API — නොමිලේ, API key අවශ්‍ය නැත |
| **Stripe** | Stripe හි සජීවී විදේශ විනිමය අනුපාත භාවිතා කරයි — සක්‍රිය Stripe ඒකාබද්ධ කිරීමක් අවශ්‍යයි |
| **අතින්** | ඔබ අනුපාත සෘජුව ඇතුළත් කරයි; බාහිර API ඇමතුම් නැත |

**Ultimate Multisite → බහු-මුදල් → සැකසුම් → විනිමය අනුපාත සපයන්නා** යටතේ ඔබේ සපයන්නා තෝරන්න. අනුපාත ඕනෑම වේලාවක අතින් නැවුම් කළ හැකිය, නැතහොත් ස්වයංක්‍රීයව යාවත්කාලීන වීමට සැලසුම් කළ හැකිය.

### නිෂ්පාදන-අනුව මිල අභිබවා යෙදවීම් {#per-product-pricing-overrides}

ස්වයංක්‍රීය පරිවර්තනය ප්‍රමාණවත් තරම් නිවැරදි නොවන විට, ඕනෑම නිෂ්පාදනයක නිශ්චිත මුදල් ඒකකයක් සඳහා නිශ්චිත මිලක් සකසා හැකිය:

1. නිෂ්පාදන සංස්කාරකය විවෘත කරන්න (**Ultimate Multisite → නිෂ්පාදන → සංස්කරණය**)
2. **මුදල් ඒකක මිලකරණය** කොටසට පහළට අනුචලනය කරන්න
3. ඔබට අභිබවා යෙදවීමට අවශ්‍ය එක් එක් මුදල් ඒකකය සඳහා නිශ්චිත මිල ඇතුළත් කරන්න
4. නිෂ්පාදනය සුරකින්න

අභිබවා යෙදවීමක් නැති නිෂ්පාදන ඔබේ මූලික මුදල් ඒකකයෙන් පරිවර්තනය කළ මිල භාවිතා කරයි.

### මුදල් ඒකක තේරීමේ ලියාපදිංචි ක්ෂේත්‍රය {#currency-selector-signup-field}

පාරිභෝගිකයන් ලියාපදිංචියේදී තමන් කැමති මුදල් ඒකකය තෝරාගැනීමට, මුදල් ඒකක තේරීමේ ක්ෂේත්‍රය ඔබේ ලියාපදිංචි/ගෙවීම් ප්‍රවාහයට එක් කළ හැකිය:

1. **Ultimate Multisite → බහු-මුදල් → සැකසුම්** වෙත යන්න
2. **ලියාපදිංචියේදී මුදල් ඒකක තේරීම පෙන්වන්න** සක්‍රිය කරන්න
3. අවශ්‍ය නම් පෙරනිමි මුදල් ඒකකයක් සකසන්න, නැතහොත් භූ-ස්ථානගත කිරීමෙන් පෙර-තෝරාගත් අගය තීරණය වීමට ඉඩ දෙන්න

තෝරාගත් මුදල් ඒකකය දායකත්වය සමඟ ගබඩා කෙරෙන අතර ඉන්පසු ඇති සියලු බිල් කිරීම සඳහා භාවිතා කෙරේ.
