# Wie Sie Hilfe bekommen {#how-to-get-help}

Sie haben Ihre gültige Lizenzschlüssel und damit Zugang zu unserem Team, das Ihnen helfen kann, Probleme zu lösen, die Sie beim Einrichten von Ultimate Multisite auftreten könnten.

Diese Lektion behandelt die verfügbaren Support-Kanäle für Lizenzinhaber und welche Art von Informationen Sie zur Kontaktaufnahme mit unserem Support-Team griffbereit haben müssen.

## Hilfe-Seite {#help-site}

Sie können auf unsere Hilfeseite zugreifen, wo Sie verschiedene Artikel und Tutorials finden, indem Sie auf den Link unten klicken.

[)

Wir ermutigen unsere Nutzer dazu, sich mit den Grundlagen und Funktionen von Ultimate Multisite vertraut zu machen. Es gibt "Getting Started"-Artikel auf unserer Hilfeseite und Antworten auf häufig gestellte Fragen finden Sie dort auch, wie man Add-ons installiert oder Zahlungsgateways integriert usw.

## Support kontaktieren {#contact-support}

Wenn Sie feststecken und die Antwort nicht auf unserer Hilfeseite finden können, können Sie sich entweder über das Sprechblasen-Symbol klicken, das sich unten rechts auf unserer Website befindet, ODER indem Sie eine E-Mail direkt an support@ultimatemultisite.com senden, an unser Support-Team wenden.

Wenn Sie sich wegen technischer Probleme bei der Unterstützung melden, stellen Sie bitte sicher, dass Sie eine detaillierte Beschreibung Ihres Problems zusammen mit einem Screenshot, Fehlermeldungen oder allen Informationen bereitstellen, die unseren Support-Mitarbeitern helfen, Ihr Problem leicht zu verstehen und einzuschätzen. Sie müssen auch Ihren Lizenzschlüssel angeben. Nur aktive Lizenzinhaber haben Anspruch auf Unterstützung.

## Community-Seite {#community-site}

Unsere [community site](https://community.ultimatemultisite.com/) ist ein Treffpunkt für Ultimate Multisite-Nutzer, um miteinander zu interagieren. Sie dient als Informationssystem, in dem Mitglieder Beiträge posten, an Diskussionen kommentieren, Ratschläge geben, zusammenarbeiten oder gemeinsam Lösungen finden können. Um private Bereiche/Seiten freizuschalten, müssen Sie Ihren Ultimate Multisite Lizenzschlüssel angeben.
