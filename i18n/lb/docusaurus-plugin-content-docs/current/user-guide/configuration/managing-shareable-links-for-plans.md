# Het Verwalten vo Teilbare Links für Plannen

Ultimate Multisite biedt Ihnen d'Möglichkeit, Plan-Links direkt mit Ihre Kunde z'teile. Um das zu tun, gahts uf jede vo de Edit-Seiten vo Ihre Plän und klickt uf de _Click to copy the Shareable Link_ Knopf.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_U de Edit-Seite vom Plan finded Sie de Knopf, um teilbare Links z'erstelle._

Merksch, dass die Mechanik d'Standard-Abrechnigsfrequenz brucht, um de Link ufzbaue. Das heisst, wenn Ihre Standardabrechnung zum Bischpil monatlich isch, händ Sie am Schluss en Link wie dä: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Wenn Sie wend, dass de teilbare Link uf vierteljährlich standardmässig isch, müesse Sie 3 am End dezue füege:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Um de Plan _premium_ für en jährliche Zyklus z'wähle, füeget Sie 1 und s'Wort:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Halted dra, dass Sie nöd nur en Link für verschiedeni Abrechnigsfrequenze händ. Sie söllt en spezifische Link für jede vo Ihre Abrechnigsfrequenze teile.

Sobald Sie de Link mit Ihre Kunde teiled, wird er sie zur erschte Schritt vom Iischtiegsablauf wiiterleite.
