# Iires Kunde-Websites (v2) {#your-clients-websites-v2}

Nachdem Sie sich in unserem Netzwerk registriert händ, chönne Ihre Kunde ihri Dashboard ufrufe.

Neben de **Account**-Siite, wo sie wichtigi Infos zu ihre Zahligä, Mitgliedschafte, Domäne, Planlimitatione und meh gseh chönne, händ sie au Zuegang zur Siite **My Sites**, wo sie alli ihri Websites gseh und verwalte chönne. Alles, was sie müesse mache, isch druf dr "My Sites" uf ihrem Dashboard z'klicke.

![Link zu My Sites i de Kunde-Dashboard](/img/account-page/my-sites-link.png)

Dort werdet alli ihri Websites azeigt, mit eme **Manage**-Button direkt unde jede. Wenn sie druf klicke, wärdet sie zur **dashboard vo dere spezifische Website** umgleitet.

Sie chönne au gseh, weli vo ihre Websites d'**primäri** isch für Plän, wo ihne meh als e Website zuelönd.

![My Sites Siite mit Kunde-Websites und Manage-Buttons](/img/account-page/my-sites-page.png)

Hier händ Sie en ganz Überblick über d'Site-Bearbeitigs-Oberflächi:

![Vollbild-Interface zur Website-Bearbeitig](/img/admin/site-edit-full.png)

Wenn sie ihri primäri Website ändere wend, müesse sie uf de Account-Siite druf klicke und d'**Account**-Menü wähle.

![Link zum Account-Menü i de Kunde-Dashboard](/img/account-page/account-menu.png)

Sie chönne au Kundedetails aaluege und bearbeite:

![Kunde-Bearbeitigs-Oberflächi](/img/admin/customer-edit.png)

Dänn müesse sie uf **Change Default Site** klicke.

![Change Default Site Button uf de Account-Siite](/img/account-page/change-default-site.png)

Es wird es Pop-up-Fänschter uftauche. Klicke uf d'Drop-down-Liste und wähle, weli Website Sie als primäri Website mache wend.

![Auswahl im Change Default Site Dropdown](/img/account-page/change-default-site-dropdown.png)

Dänn klicke sie eifach uf **Change Default Site** und fertig!
