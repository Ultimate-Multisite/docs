# Werbemittel uf Ihrem Netzwerk mit Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector isch en **kostenlose Add-on**, wo dir d'Werkzüg git, um Werbemittel direkt vo de Edit-Seite dini Plän z'plaziere.

Nachdem du Ultimate Multisite: Ad Injector installiert hesch, gaasch zu dinem _Plugins_ Page und aktiviere ihn.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Klick uf d'Network Activate-Schaltfläche_

Sobald s'Add-on aktiv isch, wirds im _Advanced Options_ Meta Box uf em Plan-Edit-Bildschirm e neui Option heisse _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Jetzt chasch d'Werbe-Einstellungen für jede vo dine Plän bearbeite._

U dere Tab chasch du d'Ad-Codes für Posts und Pages im Front-End, vor und nach em Inhalt, hinzufüge.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Nachdem du de Code vo dinere Werbig iigfüegt hesch, wird er uf de Posts und Pages vo däm spezifische Plan angezeigt._

Du chasch au Werbemittel im Header vo Admin-Pages für Benutzer vo däm Plan iibinde.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Ga zu de Option Admin Panel Ad Code, wenn du Werbemittel uf de Back-End vo dine Benutzer aazeige wetsch._

### Über d'Werbig use

Du chasch Ultimate Multisite: Ad Injector au bruche, um Support für hochrangigi Plän anzbiete.

Ga eifach zu de Option _Admin Panel Ads_ und füeg de HTML-Code für dini Support-Tool hinzu.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot nöd verfügbar: Kundestütz-Widget uf em Admin Panel für en spezifische Plan -->

_Kundestütz zu bestimmte Pläne hinzufüge_

So chönd d'Lüüt direkt vo ihrem Admin Panel uf Sie zuegriffe!
