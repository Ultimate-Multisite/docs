# Filter: wu_contains_element

Erlaabt Entwéckler, d'Resultater vun der initialer Sich ze änneren.

Dëst ass nëtzlech fir Drëttubidder-Builder an Änleches.

## Parameteren {#parameters}

| Numm | Typ | Beschreiwung |
|------|------|-------------|
| $contains_elements | `bool` | Wann d'Element am Inhalt enthale ass. |
| $content | `string` | Den Inhalt, deen ënnersicht gëtt. |
| $element | `self` | Dat aktuellt Element. |
| $post | `null\|\WP_Post` | post fir ze kontrolléieren. |
