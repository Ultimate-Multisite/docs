# Filter: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Filtert d'Produktzorten, déi onofhängeg Ofrechnungszyklen hunn.

Produkter mat onofhängegen Ofrechnungszyklen ginn net gezwongen, der Ofrechnungsperiod vum Plang am Wuerekuerf ze entspriechen. Dat ass nëtzlech fir Produkter wéi Domain-Registréierungen, déi ëmmer jäerlech verrechent ginn, egal ob de Plang monatlech oder jäerlech ass.

## Parameteren {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Array vu Produktzorte mat onofhängegen Ofrechnungszyklen. |
