# Filter: wu_apply_plan_limits

Erlaabt Plugin-Entwéckler, d'Limitatiounen ze ëmgoen.

Dir kënnt dëse Filter benotzen, fir belibegen Code auszeféieren, ier eng vun de Limitten initiéiert gëtt. Wann Äre Filter iergende Wäert zeréckgëtt, deen als wouer gëllt, geet de Prozess weider; wann en iergende Wäert zeréckgëtt, deen als falsch gëllt, gëtt de Code zeréckginn a keng vun den Hooks hei drënner gëtt ausgeféiert.

### Zënter

- 1.7.0
