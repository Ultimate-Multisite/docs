# Filter: wu_minimum_password_length {#filter-wuminimumpasswordlength}

Filtert déi minimal Passwuertlängt.

Nëmmen duerchgesat, wann wu_enforce_password_rules true ass.

## Parameteren {#parameters}

| Numm | Typ | Beschreiwung |
|------|------|-------------|
| $min_length | `int` | Minimal Passwuertlängt. Standard 12 (entsprécht Defender Pro). |
| $defender_active | `bool` | Ob Defender Pro Strong Password aktiv ass. |
