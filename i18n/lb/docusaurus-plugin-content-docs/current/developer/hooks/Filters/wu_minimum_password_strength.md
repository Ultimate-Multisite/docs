# Filter: wu_minimum_password_strength

Filter déi erfuerderlech minimal Passwuertstäerkt (zxcvbn-Score).

Stäerktniveauen: - 0, 1: Ganz schwaach - 2: Schwaach - 3: Mëttel - 4: Staark (Standard)

## Parameteren

| Numm | Typ | Beschreiwung |
|------|------|-------------|
| $min_strength | `int` | Den erfuerderleche minimale Stäerktniveau. |
| $strength_setting | `string` | De Wäert vun der Admin-Astellung (medium, strong, super_strong). |
