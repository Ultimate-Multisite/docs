# Filter: wu_use_domain_mapping

Bestëmmt, ob eng Mapping benotzt soll ginn

Normalerweis wëlls du nëmmen erlaben, datt aktiv Mappings benotzt ginn. Wann s du awer méi fortgeschratt Logik benotze wëlls, oder och net-aktiv Domänen zouloosse wëlls, datt se gemappt ginn, filter einfach hei.
