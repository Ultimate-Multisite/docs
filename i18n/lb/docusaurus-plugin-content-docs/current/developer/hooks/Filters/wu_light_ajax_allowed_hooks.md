# Filter: wu_light_ajax_allowed_hooks

Aus Sécherheetsgrënn limitéiere mir d'Zuel vun den Aktiounen, déi fir d'Abannen iwwer Hooks verfügbar sinn. Dëse Filter erlaabt Entwéckler, dës Lëscht ze erweideren, wann néideg.
