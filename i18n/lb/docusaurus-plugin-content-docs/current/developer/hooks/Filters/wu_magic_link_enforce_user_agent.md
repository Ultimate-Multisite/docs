# Filter: wu_magic_link_enforce_user_agent

Filter, ob d'User-Agent-Verifizéierung duerchgesat gëtt.

Op false setzen, fir Tokens iwwer verschidde Browser/Apparater funktionéieren ze loossen. Dat reduzéiert d'Sécherheet, mee erhéicht d'Benotzerfrëndlechkeet.

## Parameteren

| Numm | Typ | Beschreiwung |
|------|------|-------------|
| $enforce | `bool` | Ob d'Ofstëmmung mam User Agent duerchgesat gëtt. |
