# Filter: wp_ultimo_woocommerce_checkout_fields_to_clean

Erlaabt Entwéckler, Standard-WooCommerce-Rechnungsadressfelder ze filteren.

Well Ultimate Multisite Rechnungsadressfelder huet, kéint et Sënn maachen, déi Standardfelder ze läschen, déi vu WooCommerce um Checkout-Formular ugewise ginn. Standardméisseg gëtt kee Wäert un dës Lëscht weiderginn, sou datt all Felder, déi a WooCommerce fir d'Uwiese konfiguréiert sinn, bäibehale ginn.

### Zënter {#since}

- 2.0.0
### Quell {#source}

Definéiert an [`inc/gateways/class-woocommerce-gateway.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/gateways/class-woocommerce-gateway.php#L277) op der Linn 277


## Gëtt zréck {#returns}
D'Lëscht vun de Felder, déi geläscht solle ginn. Disponibel Wäerter enthalen billing_first_name, billing_last_name, billing_company, billing_country, billing_address_1, billing_address_2, billing_city, billing_state, billing_postcode, billing_email, an billing_phone.
