# Material WP

D'WordPress-Admin-Interface ass zanter Joren d'selwecht. Mee hues du dech jeemools gefrot, wéi eng Material Design-Versioun vun der Admin-Interface ausgesi géif? 

Material WP ass en Admin-Theme, dat entwéckelt gouf, fir dir dee Look and Feel ze ginn an dir ze hëllefen, dengen Clienten en nach méi personaliséierte Dashboard unzebidden.

## Funktiounen {#features}

- Ënnerstëtzung fir benotzerdefinéiert CSS
- Einfach Astellunge fir de Styling
- Nei Uerdnung vum Menü
- Einfachen Import/Export

## Dokumentatioun {#documentation}

**Däi Admin-Panel mat Material WP personaliséieren**

## Installatioun {#installation}

1. Lued 'material-wp' an den '/wp-content/plugins/' Verzeechnes erop
2. Aktivéier d'Erweiderung iwwer de Menü 'Erweiderungen' a WordPress
