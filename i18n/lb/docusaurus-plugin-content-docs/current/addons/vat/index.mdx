# VAT

Multisite Ultimate: Europäesch VAT

Füügt zousätzlech Funktiounen an Tools der Standard-Steierbehandlung vu Multisite Ultimate derbäi, fir europäesch VAT besser z'ënnerstëtzen.
Füügt automatesch EU-VAT-Sätz op d'Steiersätz-Säit bäi vun

## Installatioun {#installation}

1. Luet 'ultimate-multisite-vat' an de Verzeechnes '/wp-content/plugins/' erop
2. Aktivéiert de Plugin iwwer de Menü 'Plugins' a WordPress
3. Gitt op Ultimate Multisite > Astellungen > Steieren an aktivéiert EU Vat.
4. Gitt op Steiersätz a klickt op EU-VAT-Sätz aktualiséieren ënnen, a späichert d'Steiersätz.
5. Fäerdeg!
