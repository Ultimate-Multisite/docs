# Support Agents

E ganzt Netzwierk eleng ze verwalten ass keng einfach Aufgab. Wann däin Netzwierk ufänkt ze wuessen, gesäis du eng ëmmer méi grouss Nofro no Support. Et ass eng gutt Iddi, Leit dobäizehuelen, déi dir hëllefen, deng Aarbechtsbelaaschtung ze reduzéieren. Allerdéngs ass voll Zougang/Berechtegung ze ginn riskant a kann däin Netzwierk a Gefor bréngen. Genee do kënnt dëse Support Agent Add-on an d'Spill.

Dëse Add-on erlaabt dir, en neien Typ vu WordPress-Benotzer mam Numm Support Agent ze erstellen, a gëtt dir voll Kontroll doriwwer, wéi eng Berechtegungen dëse Benotzer huet. Berechtegung ze ginn ass esou einfach wéi e Toggle-Button unzeschalten, an en erëm auszeschalten, wanns du dech entscheets, se ewechzehuelen.

## Funktiounen {#features}

- Detailléiert Lëscht vu WordPress- & Ultimate Multisite-Berechtegungen
- Zougang einfach aktivéieren an deaktivéieren
- Onlimitéiert Zuel vun Agent-Accounten erstellen

## Installatioun {#installation}

1. Luet 'ultimate-multisite-support-agents' an den '/wp-content/plugins/'-Verzeechnes erop
2. Aktivéier de Plugin iwwer de 'Plugins'-Menü a WordPress
3. Folleg dem Schrëtt-fir-Schrëtt-Wizard, fir de Plugin anzestellen
