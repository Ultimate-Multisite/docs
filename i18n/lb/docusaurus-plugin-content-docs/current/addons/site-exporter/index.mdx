# Site-Exporter

## Inhalt

Mir verstinn de Besoin, Subsiten an Ärem Netzwierk op eng eenzel Site-Installatioun ze réckelen an ëmgedréint, falls Äre Client decidéiert, seng Websäit ze migréieren. Den Site-Exporter- an Importer-Add-on mécht dës Aufgab méi einfach. Dëst Tool exportéiert an importéiert Siten mat ZIP-Dateien, déi och als Backup dénge kënnen.

## Funktiounen

* Siten exportéieren an importéieren

* Einfach ze benotzend Interface

* Site-Backup erstellen

## Dokumentatioun
**Wéi een eng Site exportéiert an importéiert**

## Installatioun

1. Luet **ultimate-multisite-site-exporter** an den **/wp-content/plugins/** Verzeechnes erop
2. Aktivéiert de Plugin iwwer de **Plugins**-Menü am WordPress
3. Follegt dem Schrëtt-fir-Schrëtt-Wizard, fir de Plugin anzestellen

## Wéi een eng Site exportéiert

Fir eng Site aus Ärem Netzwierk ze exportéieren, gitt op **Export & Import** am Ultimate Multisite-Menü a klickt am Tab **Siten exportéieren** op **Neie Site-Export generéieren**.

<!-- Screenshot net verfügbar: Tab Siten exportéieren mam Button Neie Site-Export generéieren -->

Sicht no der Site, déi Dir exportéiere wëllt, a wielt d'Elementer aus, déi Dir an déi exportéiert Datei ophuele wëllt. Dir kënnt Designen, Plugins an Eropluede mat abannen. Wann Är Site ze grouss ass, kënnt Dir wielen, den Exportprozess am Hannergrond lafen ze loossen.<!-- Screenshot net verfügbar: Site-Exportoptioune mat Site-Sich, Kontrollkëschte fir Designen/Plugins/Eroplueden anzebannen -->

Klickt op de Button **Site exportéieren**. Är exportéiert Site steet an der Lëscht **Bestoend Exporter** zur Verfügung. Klickt op de Button **ZIP eroflueden**, fir d'Datei erofzelueden. Export-ZIP-Dateie enthalen eng selwer startend `index.php` an `readme.txt`, sou datt se op engem frëschen Host installéiert kënne ginn, ouni fir d'éischt e separaten Importer-Plugin ze installéieren.<!-- Screenshot net verfügbar: Lëscht Bestoend Exporter mam Button ZIP eroflueden -->

## Wéi een dat ganzt Netzwierk exportéiert

Fir all Subsiten zesummen ze exportéieren, gitt op **Export & Import** am Ultimate Multisite-Menü a benotzt d'Optioun **Netzwierk-Export** um Exportbildschierm. Den generéierten Archiv enthält all Subsiten an enger eenzeger ZIP-Datei fir Host-Migratiounen, Staging-Neiopbauen oder Backups um Netzwierkniveau.

Netzwierk-Exporter kënne grouss sinn. Féiert se aus, wann de Verkéier niddereg ass, a bestätegt, datt de Server genuch Späicherplaz fir den Archiv an all abegraff Eroplueden, Plugins an Designen huet.

## Wéi een eng Site importéiert

Fir eng Site ze importéieren, gitt op den Tab **Eng Site importéieren** a lued d'.zip-Datei vun der Site erop, déi Dir importéiere wëllt. Dir kënnt d'URL op eng nei änneren oder déi al URL behalen, andeems Dir d'Feld eidel loosst. Ausserdeem kënnt Dir d'Këscht unhaken, fir d'.zip-Datei no dem Import aus Ärer Installatioun ze läschen. Da klickt op **Import ufänken**.

<!-- Screenshot net verfügbar: Tab Eng Site importéieren mat ZIP-Eropluedfeld an Import ufänken-Button -->

De Prozess kann e bëssen daueren. Dir gesitt en ofwaardenden Import an der Lëscht ënner dem Importer.

<!-- Screenshot net verfügbar: Ofwaardenden Import an der Lëscht ënner dem Importer -->

Dir kënnt de Prozess stoppen, andeems Dir op **Import ofbriechen** klickt.

<!-- Screenshot net verfügbar: Import ofbriechen-Button fir den Importprozess ze stoppen -->

Nodeems de Prozess ofgeschloss ass, ass Är Site schonn an Ärem Netzwierk verfügbar.

## Wéi een eng Site op enger eenzeler Installatioun importéiert

<!-- Screenshot net verfügbar: Exporter-Tab mam Link fir den Single-Site-Importer erofzelueden -->

Dir fannt den Importer am Menü **Tools**, lénks, am Ënnerpunkt **Export & Import**.

<!-- Screenshot net verfügbar: Single-Site-Tools-Menü weist den Ënnermenü Export an Import -->

Luet d'.zip-Datei vun der Site erop, déi Dir importéiere wëllt. Dir kënnt d'URL op eng nei änneren oder déi al URL behalen, andeems Dir d'Feld eidel loosst. Ausserdeem kënnt Dir d'Këscht unhaken, fir d'.zip-Datei no dem Import aus Ärer Installatioun ze läschen. Da klickt op **Import ufänken**.

<!-- Screenshot net verfügbar: Single-Site-Importformular mat ZIP-Eropluedung an Import ufänken-Button -->

De Prozess kann e bëssen daueren. Dir gesitt en ofwaardenden Import an der Lëscht ënner dem Importer.

<!-- Screenshot net verfügbar: Single-Site ofwaardenden Import an der Lëscht -->

<!-- Screenshot net verfügbar: Import ofbriechen-Button fir den Importprozess ze stoppen -->
