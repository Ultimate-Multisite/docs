# Erweiderungs- & Design-Thema-Manager

Multisite Ultimate: Erweiderungs- an Design-Thema-Manager

Ännert oder verstoppt d’Metainformatiounen (Titel, Auteur, Miniaturbild a Beschreiwung) vun installéierten Erweiderungen an Design-Themen. Erstellt Kategorië fir Erweiderungen an Design-Themen, fir Äre Benotzer z’erméiglechen, se ze filteren, a füügt eng schéin personaliséiert Erweiderungs-Säit an de Panel vun Äre Benotzer derbäi.

Fir d’Erweiderung ze benotzen, kënnt Dir einfach op Är Erweiderungs- oder Design-Thema-Säiten an Ärem Netzwierk-Admin goen an op "Erweiderungs-/Design-Thema-Info änneren" klicken.

## Installatioun

1. Luet 'ultimate-multisite-plugin-and-theme-manager' an den '/wp-content/plugins/' Verzeechnes erop
2. Aktivéiert d’Erweiderung iwwer de Menü 'Erweiderungen' an Ärem WordPress Netzwierk-Admin

## Benotzungsguide

Ultimate Multisite: Erweiderungs- an Design-Thema-Manager ass eng mächteg **premium** Ultimate Multisite Zousaz-Erweiderung, déi Iech voll Kontroll doriwwer gëtt, wéi Är Netzwierk-Benotzer d’Erweiderungs- an Design-Thema-Optiounen op Ärem Netzwierk gesinn a mat hinnen interagéieren.

Dës Zousaz-Erweiderung erfëllt dräi Zwecker:

  1. Si léisst Iech d’Metadate vu béiden, Erweiderungen an Design-Themen, komplett änneren. Dat enthält Numm, Beschreiwung, Auteurnumm a souguer Miniaturbiller!

  2. Si léisst Iech Erweiderungen an Design-Themen kategoriséieren, fir Äre Benotzer z’erméiglechen, se einfach ze sortéieren.

  3. Si gëtt Iech d’Méiglechkeet, déi al, langweileg Erweiderungs-Säit duerch eng nei, design-thema-änlech Säit mat Sortéierung no Kategorien ze ersetzen.

Nodeems Dir se erofgelueden an installéiert hutt, erschéngt en neien Tab - _Zousaz-Erweiderungs-Astellungen_ \- op der Ultimate Multisite-Astellungssäit.
