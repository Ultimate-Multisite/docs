# Inhaltssynchroniséierung

Multisite Ultimate: Inhaltssynchroniséierung

Dësen Addon fir Multisite Ultimate erlaabt Netzwierkadministrateuren, Inhalt vun all Site op aner Sitten am Netzwierk ze synchroniséieren. En ënnerstëtzt all Bäitragstypen, inklusiv Bäiträg, Säiten, personaliséiert Bäitragstypen an Elementor-Schablounen. Den Addon enthält eng Wizard-Interface mat méi Schrëtt fir Inhalt an Zil-Sitten auszewielen, mat Batch-Veraarbechtung a Fortschrëttsfeedback an Echtzäit.

## Funktiounen {#features}

* Inhaltsauswiel: Visuellt Kaarte-Gitter mat Miniaturbiller a Filterung no Bäitragstyp
* All Bäitragstypen: Synchroniséiert Bäiträg, Säiten, Produkter, Elementor-Schablounen an all personaliséierte Bäitragstyp
* Site-Auswiel: Wielt aus, op wéi eng Sitten den Inhalt synchroniséiert gëtt, mat Ausschloss vum Quell-Site a Virauswiel vu Kanner-Sitten
* Batch-Veraarbechtung: Synchroniséiert op 10 Sitten gläichzäiteg mat Fortschrëttsaktualiséierungen
* Medie-Synchroniséierung: Kopéiert a verknäppt Medie-Fichieren (Biller, Videoen) automatesch nei aus dem Bäitragsinhalt op d'Zil-Sitten
* Taxonomie-Synchroniséierung: Erhält Kategorien, Tags a personaliséiert Taxonomien
* Elementor-Ënnerstëtzung: Voll Ënnerstëtzung fir Elementor-Schabloune mat CSS-Neigeneréierung an Neiverknëppung vun Template-IDen
* Konfliktléisung: Optioun fir bestoenden Inhalt ze ersetzen oder ze iwwersprangen
* Cache-Gestioun: Läscht automatesch den Elementor-Cache op den Zil-Sitten no der Synchroniséierung

## Ufuerderungen {#requirements}

* WordPress 5.3 oder méi héich
* PHP 7.4 oder méi héich
* Multisite Ultimate plugin (aktiv)
* Elementor plugin (optional, erfuerderlech fir d'Synchroniséierung vun Elementor-Schablounen)

## Installatioun {#installation}

1. Luet 'ultimate-multisite-content-sync' an den Dossier '/wp-content/plugins/' erop
2. Aktivéiert de plugin iwwer de Menü 'Plugins' an Ärem WordPress Network Admin
3. Konfiguréiert d'Astellungen ënner Ultimate Multisite → Settings → Content Sync
4. Navigéiert op Ultimate Multisite → Content Sync fir d'Synchroniséierung unzefänken
