# Superdav AI Agent

Superdav AI Agent ass en erweiderten WordPress AI-Assistent, deen Gratis AI Agent mat thema-bewosste Fäegkeeten, verbesserte Fäegkeete fir Internetsich, Websäit-Builder-Funktiounen an optionalen Automatiséierungsintegratiounen erweidert. Versioun 1.18.0 füügt Google Calendar-Lies-Tools, Kontakt- an Participanten-Zouuerdnung, mënschlech Geneemegungsschleisen, Erënnerungs-Deduplikatioun, TextBee SMS-Benoriichtegungen, verwaltete Superdav-Service-Bereetstellung an e separat verdeelten Advanced-Begleetpaket fir vertrauenswierdeg Entwéckler-Workflows dobäi.

## Haaptfunktiounen

- **Thema-bewosst Fäegkeeten** — passt Uleedung a Fäegkeeten automatesch un däin aktivt WordPress-Thema an installéiert Erweiderungen un
- **Méi Internetsich-Ubidder** — wielt tëscht Brave Search an Tavily fir méi räich Sichresultater an erweidert Fuerschungsfäegkeeten
- **Websäit-Builder-Kontaktformulairen** — erstellt voll funktionell Kontaktformulairen direkt aus der Chat-Uewerfläch mat denger installéierter Formulaire-Erweiderung
- **Verbesserte Websäit-Builder** — verbessert méi-phasëg Websäit-Baupläng mat besserer Erweiderungs-Entdeckung a Feeler-Erhuelung
- **Agebaute Thema-Fäegkeeten** — spezialiséiert Uleedung fir Block-Themen, klassesch Themen, Kadence Blocks a Kadence Theme
- **Verbessert WooCommerce-Integratioun** — ëmgebaut fir nativ WooCommerce-APIs ze benotzen, fir besser Zouverlässegkeet a Kompatibilitéit
- **Automatesch Ubidder-Erkennung** — erkennt an aktualiséiert verfügbar Erweiderungen an Themen automatesch, wann se aktivéiert oder deaktivéiert ginn
- **Benotzerdefinéiert Logo-Generéierung** — generéiert a bett benotzerdefinéiert SVG-Logoen an, mat automatescher namespace-sécherer Sanitiséierung
- **Palette-Kontrastvalidatioun** — iwwerpréift Faarfpuer op WCAG-Zougänglechkeetskonformitéit, ier se op däin Thema ugewannt ginn
- **Foto-verstäerkt Entdeckung** — lued Fotoe wärend dem Theme Builder-Entdeckungsinterview erop, fir d'Design-Richtung ze beaflossen
- **Hospitality-Menüen** — generéiert strukturéiert Iess- a Gedrénksmenü-Säite fir Restauranten, Caféen an Hospitality-Betriber
- **Desktop- a mobil Virschauen** — kuck däin Design um Desktop an op mobilen Apparater wärend der Auswiel vun der Design-Richtung vir
- **Tier-1-Websäit-Scraping** — sd-ai-agent/site-scrape ass elo standardméisseg verfügbar fir Konkurrenten- an Inspiratiouns-Websäiten ze analyséieren
- **Google Calendar-Lies-Tools** — lies konfiguréiert Kalenneren an Eventer, sou datt Automatiséierunge kommende Plang-Kontext berécksiichtege kënnen
- **Kontakt- an Participanten-Zouuerdnung** — vergläich Kalennerparticipanten mat bekannte Websäit-Benotzer a Kontakter, ier Nofollg-Aarbecht generéiert gëtt
- **Mënschlech Geneemegungsschleisen** — pauséiert sensibel Automatiséierunge fir Iwwerpréiwung, ier Messagen geschéckt, Aarbechte weidergefouert oder Ännerunge gemaach ginn
- **Erënnerungsopzeechnungen an Deduplikatioun** — verfollegt Erënnerungen, déi scho geschéckt goufen, sou datt widderholl Automatiséierungslaf net dee selwechten Empfänger zweemol informéieren
- **TextBee SMS-Benoriichtegungen** — schéckt konfiguréiert Textnoriichte-Benoriichtegungen iwwer TextBee, wann en SMS-Ubidder aktivéiert ass
- **Verwaltete Superdav-Service-Bereetstellung** — verbënnt ënnerstëtzt Websäite mat gehoste Superdav-Endpunkten, mat automatescher Ariichtung amplaz manueller Endpunkt-Konfiguratioun
- **Advanced-Begleetpaket** — separat verdeelt Tools fir vertrauenswierdeg Entwéckler fir Dateisystem, Datebank, WP-CLI, REST-Dispatcher, Erweiderungs-Builder, git-Snapshot, Benotzerverwaltung a Benchmark-Workflows

## Ufuerderungen

- WordPress 6.0 oder méi héich
- PHP 8.1 oder méi héich
- WordPress Multisite (netzaktivéiert) oder Single-Site-Installatioun
- Eng aktiv AI-Ubidder-Connector-Erweiderung (Anthropic, OpenAI oder kompatibel)
- Gratis AI Agent v1.9.0 oder méi héich (Superdav AI Agent erweidert Gratis AI Agent)

## Installatioun

1. Lued d'Erweiderungsdateien op `/wp-content/plugins/superdav-ai-agent` erop
2. Aktivéier d'Erweiderung iwwer den **Erweiderungen**-Bildschierm a WordPress
3. Stell sécher, datt Gratis AI Agent och aktivéiert ass
4. Navigéier op **Gratis AI Agent → Astellungen** a konfiguréier däin Internetsich-Ubidder, déi verwaltete Superdav-Service-Verbindung, Google Calendar oder TextBee-Zouganksdaten no Bedarf

D'WordPress.org-Paket enthält d'Kärfunktioune vum Superdav AI Agent. D'Advanced-Begleetpaket gëtt separat gebaut a verdeelt fir Ëmfeld, an deenen Tools fir vertrauenswierdeg Entwéckler ausdrécklech installéiert an autoriséiert sinn.

## Kalenner-, Geneemegungs- an Notifikatiouns-Workflows

Superdav AI Agent v1.18.0 erweidert Automatiséierungs-Workflows iwwer Aufgaben fir Inhalt a site-Opbau eraus:

- **Google Calendar-Tools** erlaben dem Agent, konfiguréiert Kalenneren an Eventdetailer ze liesen, wann Kalenner-Umeldungsdaten disponibel sinn.
- **Kontakt- an Participant-Hëllefsfunktiounen** ordnen Event-Participanten WordPress-Benotzer oder bekannte Kontakter zou, éier Erënnerungen oder Nofassungsaufgabe virbereet ginn.
- **Mënschlech Geneemegungspaarten** erlaben enger Automatiséierung ze pauséieren, bis en autoriséierte Benotzer déi proposéiert Aktioun iwwerpréift.
- **Erënnerungsopzeechnungen** verhënneren duebel Kalenner- oder Notifikatiounserënnerungen, wann eng Automatiséierung nach eng Kéier probéiert oder widderholl geplangt gëtt.
- **TextBee SMS-Ënnerstëtzung** erméiglecht SMS-Notifikatiounen, wann den TextBee-Ubidder konfiguréiert ass.

Benotzt dës Tools fir zäitplangbewosst Ënnerstëtzung, Nofassung mat Participanten an Erënnerungs-Workflows, déi nach ëmmer mënschlech Iwwerpréiwung erfuerderen, ier sensibel Messagë geschéckt ginn.

## Fortgeschrattene Begleetpak

Superdav AI Agent v1.18.0 füügt e separat verdeelte fortgeschrattene Begleetpak fir vertrauenswierdeg Entwéckler-Workflows derbäi. Dës Méiglechkeete sinn net am WordPress.org-Kärpak gebündelt. Installéiert de Begleetpak nëmmen op sites, wou Administrateuren dem Agent ausdrécklech vertrauen, Zougang zu Entwéckler-Tooling ze kréien.

Fortgeschratten Tools enthalen Dateisystem-Operatiounen, Datebankzougang, WP-CLI, REST-Dispatcher-Workflows, plugin-Builder-Utilities, Git-Snapshots, Benotzerverwaltungs-Hëllefsfunktiounen a Benchmark-Tooling. Limitéiert dës Tools op vertrauenswierdeg Administrateuren an iwwerpréift d'Konfiguratioun vun de Geneemegungspaarten, ier Dir se op Produktiouns-sites aktivéiert.
