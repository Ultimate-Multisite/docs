# AI-Site-Ersteller

ultimate-multisite-ai-site-builder

## Ufuerderungen {#requirements}

- WordPress 5.3 oder méi héich
- PHP 7.4 oder méi héich
- Multisite Ultimate-Plugin (aktiv)

## Installatioun {#installation}

1. Luet d'Erweiderungsdateien an Ären `/wp-content/plugins/`-Dossier erop
2. Aktivéiert de Plugin iwwer de 'Plugins'-Menü am WordPress
3. Konfiguréiert d'Astellunge vun der Erweiderung am Netzwierk-Admin

## Entwécklung {#development}

### Ariichtung {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

## Hooks a Filteren {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - Ausgeléist no der Initialiséierung vun der Erweiderung
- `ultimate-multisite-ai-site-builder_loaded` - Ausgeléist nodeems all Erweiderungsdateie geluede sinn

### Filteren {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Astellunge vun der Erweiderung änneren
- `ultimate-multisite-ai-site-builder_enabled` - Aktivéieren/Desaktivéieren-Status vun der Erweiderung iwwerschreiwen
