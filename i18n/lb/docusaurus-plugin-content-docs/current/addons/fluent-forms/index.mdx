# Fluent Forms

Dës Erweiderung mécht et méiglech, Limitte fir Fluent Forms op Multisite Ultimate a Produkter, Memberschaften oder Websäiten derbäizesetzen.
Fluent Forms benotzt eegen Tabellen amplaz vun eegene Post-Typen, dofir funktionéieren déi normal Limit-Optiounen net mat Fluent Forms.

## Funktiounen {#features}

- Limitte fir Fluent Forms oder Conversational Fluent Forms derbäisetzen
- Clientë kréien eng frëndlech Noriicht, déi si dozou opfuerdert, hire Plang eropzesetzen, wann si d'Limit erreechen
