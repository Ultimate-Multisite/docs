# Multi-Tenancy {#multi-tenancy}

Verdeelt Ënnersäiten iwwer Serveren an Datebanken fir Leeschtung a Sécherheet

## Héichpunkte vun der Verëffentlechung vun der Versioun 1.2.0 {#version-120-release-highlights}

Multi-Tenancy 1.2.0 füügt d'souverän Tenant-Grondlag fir Produktiouns-Isolatiouns-Workflows derbäi:

- Zoustandslos Tenant-Autologin mat kuerzliewegen, géint Replay geschützte SSO-Visitten.
- Datebank- a Dateiesystem-Isolatioun pro Ënnersäit fir souverän Tenant-en.
- Hostinger Multi-Tenancy-Fäegkeet fir host-bewosst Tenant-Bereetstellung.
- Souverän Migratiouns-Verifikatiounsgaten fir Legacy-Datenkontrollen an asynchron Push-Veraarbechtung.
- Konfiguréierbar Datebank-Host-Bindung op der selwechter Maschinn fir Bedrock, FrankenPHP an localhost-Installatiounen.
- Verbesserunge beim Tenant-Bootstrap, Datebank-Writer, Schema-Bootstrap a Benotzer-Bereetstellung.
- Méi sécher souverän Ofrappen, dat Tenant-Umeldungsdaten botzt.

Kuckt d'Benotzerguiden fir [souverän Migratiouns-Workflow](/user-guide/getting-started/sovereign-migration-workflow), [Multi-Tenancy-Isolatioun](/user-guide/configuration/multi-tenancy-isolation) an [souverän Tenant-Operatiounen](/user-guide/administration/sovereign-tenant-operations).

## Konfiguratioun {#configuration}

Den Addon ënnerstëtzt verschidde Konfiguratiounsoptiounen, déi iwwer d'Netzwierk-Admin-Interface zougänglech sinn.

### Souverän Tenant-Konfiguratioun {#sovereign-tenant-configuration}

Fir souverän Tenant-en konfiguréiert d'Tenant-Datebank, d'Dateiesystem-Wuerzel, d'Datebank-Host-Bindung an d'Host-Fäegkeet, ier Dir Traffic un den Tenant rout. Féiert no all Ännerung, déi Datebank-Umeldungsdaten, Host-Bindung, Routing oder Benotzer-Bereetstellung beaflosst, eng Migratiouns-Verifikatioun aus.

Benotzt `localhost` nëmmen dann, wann d'Tenant-Runtime an d'Datebank-Grants lokale Socket- oder Zougrëff op der selwechter Maschinn erwaarden. Benotzt `127.0.0.1` oder e Service-Hostname, wann dat de Grant-Host ass, deen vun Ärem Provider konfiguréiert gouf.

### Hostinger-Fäegkeet {#hostinger-capability}

D'Hostinger Multi-Tenancy-Fäegkeet funktionéiert mam gedeelt Hostinger-Integratiouns-Token. Confirméiert, datt d'Domain an hPanel disponibel ass, DNS op den Hostinger account weist, an d'Tenant-Datebank-Astellunge mat hPanel iwwereneestëmmen, ier Dir verifizéiert.
