# Sprooch-Auswieler

Multisite Ultimate: Sprooch-Auswieler bei der Umeldung

Füügt e Feld fir d'Sprooch-Auswiel an den Umeldungs-Oflaf derbäi.

## Installatioun {#installation}

1. Luet 'ultimate-multisite-language-selector' an den '/wp-content/plugins/' Verzeechnes erop
2. Aktivéiert d'Erweiderung iwwer de Menü 'Erweiderungen' an Ärem WordPress Network Admin

## Benotzungsuleedung {#usage-guide}

Mat dësem gratis Zousaz kënnt Dir Är Benotzer auswiele loossen, wéi eng Sprooch si am Umeldungs-Oflaf an op der Login-Säit gesi wëllen.

Nodeems Dir [Är Lizenz aktivéiert hutt](../../user-guide/getting-started/installing-ultimate-multisite.md), kënnt Dir Ultimate Multisite Zousätz installéieren, op déi Är Lizenz Iech Zougang gëtt.

Gitt op Är Astellungssäit vun Ultimate Multisite a klickt dann am Menü lénks op d'Beschrëftung _Zousätz_.

![Zousätz-Säit](/img/admin/addons-page.png)

_Gitt op d'Zousätz-Säit, fir mam Installéiere vum Sprooch-Auswieler unzefänken_

Do fannt Dir all Zousätz, déi mir ubidden.

Gratis Zousätz si fir all eis Clienten disponibel, onofhängeg vum Abonnementsplang, fir dee si sech ugemellt hunn.

Klickt op de Knäppchen _Elo installéieren!_ ënner der Miniatur vum Zousaz. Aktivéiert en duerno op der Erweiderungen-Säit.

![Zousätz-Säit mat Installatiounsknäppchen](/img/admin/addons-page.png)

_Klickt op de Knäppchen, fir de Zousaz ze installéieren_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Aktivéiert en op Ärer Erweiderungen-Säit_

Gitt elo an den Astellungsmenü vun Ultimate Multisite a maacht den Tab _Netzwierk-Astellungen_ op. Do fannt Dir um Enn vun der Säit eng nei Sektioun mam Numm _Sproochen_.

![Astellungssäit](/img/config/settings-general.png)

_Eng ganz nei Sektioun erschéngt!_

Dir kënnt et eidel loossen, fir all d'Sproochen als Optiounen unzeweisen, oder auswielen, wéi eng Dir disponibel maache wëllt. D'Sprooche sinn déi, déi Dir an Ärer WordPress Installatioun installéiert hutt.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Lëscht vun de Sproochen, déi an dëser WP Installatioun disponibel sinn_

Nodeems Dir d'Ännerunge gespäichert hutt, déi Dir gemaach hutt, gëtt e Selektiounsmenü mat de Sproochen, déi Dir gewielt hutt, op der Login-Säit souwéi am Umeldungs-Oflaf ugewisen.

![Login- a Registréierungsastellungen](/img/config/settings-login-registration.png)

_Sprooch-Auswieler op der Login-Säit_

![Registréierungssäit](/img/config/registration-page.png)

_Sprooch-Auswieler am Umeldungs-Oflaf_

## D'Sprooch iwwer d'URL festleeën {#setting-the-language-via-the-url}

Dir kënnt och d'Sprooch vun enger Säit änneren, andeems Dir de locale-Parameter un d'URL weidergitt. Wann Dir zum Beispill op eng Registréierungssäit op Italienesch weiderleede wëllt, füügt ?locale=it_IT un d'URL derbäi.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

D'Sprooch muss an Ärem WordPress Netzwierk installéiert sinn, fir datt dëst funktionéiert.
