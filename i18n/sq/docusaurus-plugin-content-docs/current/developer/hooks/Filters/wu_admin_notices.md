# Filtër: wu_admin_notices

Lejon zhvilluesit të filtrojnë njoftimet e adminit të shtuara nga Ultimate Multisite.

## Parametrat {#parameters}

| Emri | Lloji | Përshkrimi |
|------|------|-------------|
| $notices | `array` | Lista e njoftimeve për atë panel të veçantë. |
| $all_notices | `array` | Lista e njoftimeve të shtuara, të ndara sipas panelit. |
| $panel | `string` | Paneli për të marrë njoftimet. |
| $filter | `string` | Nëse njoftimet që mund të hiqen janë filtruar jashtë. |
| $dismissed_messages | `array` | Lista e kyçeve të njoftimeve të hequra. |

### Që nga {#since}

- 2.0.0
