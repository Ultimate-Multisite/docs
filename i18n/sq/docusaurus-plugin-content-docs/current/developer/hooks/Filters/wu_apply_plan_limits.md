# Filter: wu_apply_plan_limits

Lejoni zhvilluesit e shtojcave të anashkalojnë kufizimet.

Mund ta përdorni këtë filter për të ekzekutuar kod arbitrar përpara se të niset ndonjë nga kufizimet. Nëse filter-i juaj kthen ndonjë vlerë truthy, procesi do të vazhdojë; nëse kthen ndonjë vlerë falsy, kodi do të kthehet dhe asnjë nga hooks më poshtë nuk do të ekzekutohet.
