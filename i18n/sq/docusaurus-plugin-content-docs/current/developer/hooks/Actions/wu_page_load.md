# Action: wu_page_load

Lejo zhvilluesit e plugin të shtojnë hooks shtesë në faqet tona.

## Parametrat {#parameters}

| Emri | Tipi | Përshkrimi |
|------|------|-------------|
| $id | `string` | ID-ja e kësaj faqeje. |
| $page_hook | `string` | Page hook i kësaj faqeje. |
| $admin_page | `self` | Instanca e faqes. |

### Që prej {#since}

- 1.8.2
- 2.0.4: U shtua parametri i tretë: instanca e faqes.
