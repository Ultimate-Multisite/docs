|------|------|-------------|
| $tab | `string` | Skeda aktuale. |
| $screen | `\WP_Screen` | Objekti i ekranit. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instanca e faqes së administrimit të Ultimate Multisite. |
