# Action: wu_page_\{$this->id\}_load

Lejo zhvilluesit e plugin-eve të shtojnë hooks shtesë në faqet tona.

## Parametrat {#parameters}

| Emri | Lloji | Përshkrimi |
|------|------|-------------|
| $id | `string` | ID-ja e kësaj faqeje. |
| $page_hook | `string` | Hook-u i faqes së kësaj faqeje. |
| $admin_page | `self` | Instanca e faqes. |
