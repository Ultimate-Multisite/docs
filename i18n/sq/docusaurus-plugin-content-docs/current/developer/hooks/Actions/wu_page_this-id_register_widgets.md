# Action: wu_page_\{$this->id\}_register_widgets

Niset pasi widgets regjistrohen për këtë faqe.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Parametrat {#parameters}

| Emri | Lloji | Përshkrimi |
|------|------|-------------|
| $id | `string` | ID-ja e faqes. |
| $page_hook | `string` | Hook-u i faqes. |
| $page | `object` | Objekti i faqes. |
