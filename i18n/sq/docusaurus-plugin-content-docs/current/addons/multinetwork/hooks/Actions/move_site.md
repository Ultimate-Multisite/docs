# Action: move_site

Aktivizohet pasi një sajt është zhvendosur në një rrjet të ri.

## Parametrat {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | ID-ja e sajtit që është zhvendosur. |
| $old_network_id | `int` | ID-ja e rrjetit origjinal për sajtin. |
| $new_network_id | `int` | ID-ja e rrjetit ku është zhvendosur sajti. |

- 1.3.0
