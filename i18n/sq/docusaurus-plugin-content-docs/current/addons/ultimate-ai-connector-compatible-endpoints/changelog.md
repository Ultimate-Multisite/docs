# Ultimate AI Connector për Compatible Endpoints Changelog

## Versioni 2.0.0 — Publikuar më 2026-04-24

* E re: Mbështetje për shumë ofrues — konfiguroni disa endpoint-e AI dhe drejtoni kërkesat me fallback automatik midis ofruesve.
* Rregullim: Integrim i SDK-së për shumë ofrues me ID të sakta të ofruesve, URL regjistrimi dhe kufizim të filtrit HTTP për secilin ofrues.
* Rregullim: Karta e ofruesit të ri tani zgjerohet automatikisht gjatë shtimit; pastrim i cache-it të skriptit gjatë përditësimit të plugin-it.
* Rregullim: Namespace dinamik i klasës së ofruesit për eval() dhe sjellje e zgjerimit automatik për ofruesit e saposhtuar.
* Rregullim: Përdor komponentë të qëndrueshëm Card/CardBody/CardHeader/CardDivider (jo më eksperimentalë) për pajtueshmëri me WordPress 6.9+.
* Rregullim: Zëvendësoi DragHandle të papërcaktuar me ikonë unicode grip për tërheqje-për-rirenditje të ofruesit.
* Rregullim: Eliminoi kërkesën bllokuese HTTP që aktivizohej në çdo ngarkim faqeje.
* Përmirësuar: GitHub Actions workflows u përditësuan në Node.js 24.
