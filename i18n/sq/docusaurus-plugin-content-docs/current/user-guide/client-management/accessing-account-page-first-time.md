# Si mund të aksesoj faqen e kontos sime për herë të parë? {#how-can-i-access-my-account-page-for-the-first-time}

Mirë se jeni mirë se jeni në Ultimate Multisite!

Për të aksesuar faqen e kontos tuaj përmes linkut që keni marrë në email pas blerjes së pluginit tonë:

<!-- Screenshot unavailable: Account page email with access link -->

Përdorni email-in që dhatë gjatë blerjes dhe fjalëkalimin që keni marrë në email.

<!-- Screenshot unavailable: Login page with email and password fields -->

Meqenëse ky është herët e parë që hyeni në këtë llogari, do t'ju kërkohet të ndryshoni fjalëkalimin tuaj. Ndryshojeni dhe klikoni butonin _Save_ (Ruaj).

<!-- Screenshot unavailable: Change password form with Save button -->

Në faqen e kontos suaj, mund të menaxhoni të gjitha detajet e llogarisë tuaj si faturat, rinovimet, anullimet dhe shumë gjëra të tjera.
