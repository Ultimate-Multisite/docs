# Personalizimi i Paneli të Administratorit me PRO Theme

Shpesh kur bëjmë punë për klientët, kemi nevojën t'u ofrojmë një përvojë të personalizuar në dashboard-in e WordPress. Nëse kërkoni një pamje dhe ndjeshmëri më profesionale, PRO Theme mund t'ju ndihmojë.

PRO Theme është një temë admin e zhvilluar për t'ju ndihmuar kur personalizoni Dashboard-in e klientëve tuaj. Gjithashtu, ai është pjesë e bazës (core) të plugin-eve premium të Ultimate Multisite.

Pas instalimit, shkoni te faqja juaj _Plugins_ dhe klikoni për ta aktivizuar në rrjetën tuaj.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktivizojeni atë në të gjithë rrjetin tuaj në faqen e Plugin-eve._

_**SHËNJSË:**_ _Nëse thjesht e instaloni, do të shfaqet tek klientët tuaj si një opsion që mund të markoni për t'i dhënë temën vetëm atij përdoruesit._

Tani, keni interfejsin tuaj admin i redizajnuar plotësisht! Keni edhe një meniu të ri në kolonën e majtë të quajtur PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Një pamje e re për Panelin tuaj Admin._
