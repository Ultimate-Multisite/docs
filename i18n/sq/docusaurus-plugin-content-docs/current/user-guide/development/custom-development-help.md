# Siç si mund të marr ndihmë për zhvillim nga Codeable? {#how-to-get-custom-development-help-from-codeable}

Ndonjëherë, ne vërtet kemi nevojë për ndonjë personalizim fantastik që të bëjmë rrjetat tona pikërisht ashtu si dëshirojmë. Nëse kjo është ajo që kërkoni, ju rekomandojmë [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), një vend ku mund të gjeni zhvillues me përvojë me ekspertizën e Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) është një platformë që lidh klientët me zhvillues freelance WordPress të kualifikuar, të aftë të punojnë në projekte të çdo madhësie. Ata kanë një proces aplikimi shumë të rregullt për të siguruar që ju të përshtatet me ekspertë specifikisht kualifikuar për detyrën që keni parasysh.

### Si të krijoni një projekt në Codeable? {#how-to-create-a-project-in-codeable}

Para se ndonjëherë, shkoni te [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) dhe klikoni butonin **Start a Project** (Ndrysho emrin e butonit në shqip nëse dëshironi të jetë më i qartë, p.sh., "Filloj një Projekt").

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Tani, për të dorëzuar një projekt, klikoni në majtë te **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Në seksionin "I need help with" (Kam nevojë për ndihmë me), zgjidhni **Customizing** (Personalizim):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Në seksionin "My" (Mja), zgjidhni **Plugin** (Plugin):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Pastaj klikoni butonin **Continue** (Prekoni).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Tani, është koha për të përshkruar projektin tuaj dhe nevojat e më të mira që mundeni. Provo të jepni sa më shumë informacion!

<!-- Screenshot unavailable: Codeable project description form -->

Përgjigjuni pyetjeve më poshtë për të ndihmuar me çmimin.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Pas kështu, nëse nuk ke ende një konton [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), krijoj një dhe kliko butonin **Publish My Project** (Publiko Projektim Timin).

<!-- Ekran i shfaqur jo i disponueshëm: Krijimi i kontosit Codeable dhe butoni Publish My Project -->

Pasi të publikosh projektin tënd në [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), platforma do të lidhë ty me zhvillues që përshtaten me aftësitë e nevojshme për projektin tënd.

Ky proces do të ndihmojë në gjetjen e personit të duhur për punën tënde, që ka përvojë me WordPress dhe Ultimate Multisite!

Rata minimale e Codeable është $70–$120 USD në orë. Vlerat do të ndryshojnë sipas rajonit të punës, kompleksitetit dhe urgjencës së projektit.

Pasi të marrësh vlerën (estimate), do të jesh në gjendje të punosh zhvilluesin që i përshtatet më mirë nevojave të tua dhe të nisë së bashku projektin!
